# Exact 4He Spectral Function in a Semirealistic NN Potential Model

> The spectral function of 4He is calculated with the Lorentz integral transform method in a large energy and momentum range. The excitation spectrum of the residual 3N-system is fully taken into account. The obtained spectral function is used to calculate the quasi elastic longitudinal (e,e') response R_l of 4He for q=300, 400, and 500 MeV/c. Comparison with the exact R_l shows a rather sizeable disagreement except in the quasi elastic peak, where the differences reduce to about 10% at q=500 MeV/c. It is shown as well that very simple momentum distribution approximations for the spectral function provide practically the same results for R_l as the exact spectral function.

# Exact 4He Spectral Function in a Semirealistic NN Potential Model

## Key claim
The full spectral function of 4He is calculated for the first time using the Lorentz integral transform method, but the results are limited to a semirealistic central NN potential (Trento potential) that lacks tensor forces, and the quasi-elastic longitudinal response computed from this spectral function shows sizeable disagreement with the exact response except near the quasi-elastic peak where differences reduce to about 10% at q=500 MeV/c.

## Method
The spectral function S(k,E) for 4He is obtained by solving a three-body inhomogeneous equation with hyperspherical harmonics expansion, using the Lorentz integral transform method to invert the integral equation relating the transform to the spectral function. The proton spectral function is then used within the plane wave impulse approximation (PWIA) to compute the longitudinal (e,e') response R_L at momentum transfers q=300, 400, and 500 MeV/c, with comparison to exact calculations from Ref. [ELO97] using the same NN potential.

## Key numbers
- Norm sum rule satisfied to within 0.9% relative difference
- Mean kinetic energy sum rule satisfied to within 0.2% relative difference
- Koltun sum rule satisfied to within 0.8% relative difference
- Spectral factor for TN potential: 0.89 (compared to 0.84 for Argonne v18 + Urbana IX realistic potential)
- At q=300 MeV/c: PWIA quasi-elastic peak shifted by 15 MeV to higher energy, peak height overestimated by more than 40%
- At q=400 MeV/c: PWIA peak shifted by 12 MeV, peak height overestimated by 25%
- At q=500 MeV/c: PWIA peak shifted by 12 MeV, peak height overestimated by 13%, differences at peak reduce to about 10%
- Neutron electric form factor contribution is about 1% at q_mu^2 = (500 MeV/c)^2

## Context
This work extends exact spectral function calculations previously performed only for three-body nuclei (Dieperink et al., Phys. Lett. B 63, 261 (1976); Ciofi degli Atti et al., Phys. Rev. C 21, 805 (1980)) to the four-body system 4He, where only approximate evaluations neglecting final state interaction in the residual 3N-system existed (Morita and Suzuki, Prog. Theor. Phys. 86, 671 (1991)). The results enable comparison of the spectral function ansatz for inclusive (e,e') scattering within a nonrelativistic framework, and show that simple momentum distribution approximations for the spectral function provide practically identical results for R_L as the full spectral function. Similar findings for the three-body systems 3H and 3He were reported by Ishikawa et al. (Phys. Lett. B 339, 293 (1994)) at momentum transfers of 300 and 400 MeV/c.
