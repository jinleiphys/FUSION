# New CTEQ global analysis of quantum chromodynamics with high-precision data from the LHC

> We present the new parton distribution functions (PDFs) from the CTEQ-TEA collaboration, obtained using a wide variety of high-precision Large Hadron Collider (LHC) data, in addition to the combined HERA I+II deep-inelastic scattering data set, along with the data sets present in the CT14 global QCD analysis. New LHC measurements in single-inclusive jet production with the full rapidity coverage, as well as production of Drell-Yan pairs, top-quark pairs, and high-$p_T$ $Z$ bosons, are included to achieve the greatest sensitivity to the PDFs. The parton distributions are determined at NLO and NNLO, with each of these PDFs accompanied by error sets determined using the Hessian method. Fast PDF survey techniques, based on the Hessian representation and the Lagrange Multiplier method, are used to quantify the preference of each data set to quantities such as $α_s(m_Z)$, and the gluon and strange quark distributions. We designate the main resulting PDF set as CT18. The ATLAS 7 TeV precision $W/Z$ data are not included in CT18, due to their tension with other data sets in the global fit. Alternate PDF sets are generated including the ATLAS precision 7 TeV $W/Z$ data (CT18A), a new scale choice for low-$x$ DIS data (CT18X), or all of the above with a slightly higher choice for the charm mass (CT18Z). Theoretical calculations of standard candle cross sections at the LHC (such as the $gg$ fusion Higgs boson cross section) are presented.

## Key claim

The CT18 parton distribution functions (PDFs) are obtained from a global QCD analysis that includes high-precision LHC data on jet, Drell-Yan, top-quark pair, and high-pT Z boson production, but the ATLAS 7 TeV precision W/Z data are excluded from the main CT18 set due to systematic tension with other data sets in the global fit, and are instead provided only in an alternate PDF set (CT18A).

## Method

The CTEQ-TEA collaboration performed next-to-leading order (NLO) and next-to-next-to-leading order (NNLO) global QCD fits to an expanded set of experimental measurements, including the combined HERA I+II deep-inelastic scattering data, legacy data from CT14, and eleven new LHC data sets at 7 and 8 TeV on W/Z, jet, and top-antitop production. Fast interpolation tables (fastNLO, ApplGrid, fastNNLO) were used to compute NNLO cross sections, and the Hessian method with a two-tier criterion was employed to determine PDF uncertainties at the 90% confidence level, with fast survey techniques (PDFSense, ePump) used to quantify the preference of each data set for quantities such as alpha_s(M_Z) and the gluon and strange quark distributions.

## Key numbers

- Total of 3681 data points in the CT18 ensemble, with chi^2/N_pt = 1.17 at NNLO
- Total of 3493 data points in the CT18Z ensemble, with chi^2/N_pt = 1.19 at NNLO
- QCD coupling constant assumed: alpha_s(M_Z) = 0.118 (world-average value)
- Charm quark mass: Q_0 = 1.3 GeV for CT18, 1.4 GeV for CT18Z
- Factorization scale for DIS in CT18X and CT18Z: mu^2_F,x = 0.8^2 (Q^2 + 0.3 GeV^2 / x_B^0.3)
- The HERA I+II inclusive DIS data set shows S_E values as high as five units or more in the effective Gaussian variable distribution

## Context

This work updates the CT14 PDFs from the CTEQ-TEA collaboration by incorporating high-precision LHC Run-1 data, while retaining legacy data from HERA, fixed-target experiments, and the Tevatron. The analysis extends previous CTEQ-TEA global fits and provides four PDF ensembles (CT18, CT18A, CT18X, CT18Z) that address tensions between data sets, particularly the ATLAS 7 TeV W/Z data which favor an increased strange quark distribution at low x. The results enable more precise theoretical predictions for standard candle cross sections at the LHC, such as Higgs boson production via gluon fusion, and provide a foundation for future PDF analyses incorporating additional LHC measurements.
