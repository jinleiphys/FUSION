# Inclusive breakup calculations in angular momentum basis: application to $^7$Li+$^{58}$Ni

> The angular momentum basis method is introduced to solve the inclusive breakup within the model proposed by Ichimura, Austern, and Vincent [Phys. Rev. C 32, 431 (1985)]. This method is based on the geometric transformation between Jacobi coordinates, thus it is easy to corporate with particle spins. To test the validity of this partial wave expansion method, a benchmark calculation is done comparing with the one given in [Phys. Rev. C 92, 044616 (2015)]. Using the distorted-wave Born approximation (DWBA) version of IAV model, some applications to $^7$Li reactions are presented and compared with available data.

## Key claim

The angular momentum basis method for inclusive breakup calculations is introduced as a more efficient and general alternative to previous implementations, particularly for projectiles with internal orbital angular momentum \(\ell \ge 1\) (such as \(^7\)Li with \(\ell=1\)), where the earlier method becomes numerically difficult due to additional angular momentum couplings.

## Method

The authors implement the Ichimura-Austern-Vincent (IAV) model using a partial wave expansion in angular momentum basis, based on geometric transformation between Jacobi coordinates. The nonelastic breakup (NEB) cross section is computed within the distorted-wave Born approximation (DWBA) version of the IAV model, while the elastic breakup (EBU) component is calculated using the CDCC formalism with the coupled-channels code FRESCO. The method is benchmarked against the earlier expansion method from Phys. Rev. C 92, 044616 (2015), showing numerical differences less than 1% for the \(\ell=0\) case.

## Key numbers

- The numerical difference between the new angular momentum basis method and the earlier expansion method (Ref. Jin15) is less than 1% for the \(\ell=0\) benchmark calculation using the same input parameters
- Breakup threshold energies: \(^6\)Li \(\alpha+d\) breakup at 1.474 MeV, \(^7\)Li \(\alpha+t\) breakup at 2.468 MeV
- For the \(^4\)He-target interaction: \(V = 49.5\) MeV, \(R_0 = 5.88\) fm, \(a_0 = 0.5\) fm, \(W = 11.0\) MeV, \(R_w = 5.69\) fm, \(a_w = 0.5\) fm (fitted to 12 MeV \(^4\)He + \(^{58}\)Ni data)
- For convergence of \(^7\)Li CDCC calculations, \(\alpha+t\) partial waves up to \(\ell = 3\) were required, including \(\ell=3\) resonant states at 4.63 MeV (\(7/2^{-}\)) and 6.68 MeV (\(5/2^{-}\))

## Context

This work extends the IAV model, originally proposed by Ichimura, Austern, and Vincent [Phys. Rev. C 32, 431 (1985)], which had fallen into disuse due to computational limitations but has been re-examined by several groups (Jin15, Jin15b, Pot15, Pot17, Carlson2016, Moro2016). The method enables calculations for projectiles with \(\ell>0\) configurations, such as \(^7\)Li (\(\alpha+t\) cluster with \(\ell=1\)), which was previously challenging compared to the \(\ell=0\) cases of deuterons and \(^6\)Li. The results enable systematic studies of inclusive alpha production in reactions induced by weakly-bound projectiles, with possible applications to knockout reactions at intermediate energies currently under study.
