# Application of Absorbing Boundary Condition to Nuclear Breakup Reactions

> Absorbing boundary condition approach to nuclear breakup reactions is investigated. A key ingredient of the method is an absorbing potential outside the physical area, which simulates the outgoing boundary condition for scattered waves. After discretizing the radial variables, the problem results in a linear algebraic equation with a sparse coefficient matrix, to which efficient iterative methods can be applicable. No virtual state such as discretized continuum channel needs to be introduced in the method. Basic aspects of the method are discussed by considering a nuclear two-body scattering problem described with an optical potential. We then apply the method to the breakup reactions of deuterons described in a three-body direct reaction model. Results employing the absorbing boundary condition are found to accurately coincide with those of the existing method which utilizes discretized continuum channels.

# Application of Absorbing Boundary Condition to Nuclear Breakup Reactions

## Key claim

The absorbing boundary condition (ABC) method provides an alternative fully quantum mechanical description of nuclear breakup reactions that accurately reproduces results from the coupled discretized continuum channels (CDCC) method, but without requiring the introduction of virtual states or discretized continuum channels. The method is demonstrated for deuteron breakup reactions, with the caveat that careful parameter selection for the absorbing potential, guided by a WKB-derived condition, is necessary to achieve accurate differential cross sections, particularly at large scattering angles.

## Method

The ABC method replaces the outgoing-wave boundary condition with an absorbing potential placed outside the physical region (beyond radii \(r_c\) and \(R_c\) for the internal and relative coordinates, respectively), imposing a vanishing boundary condition at the outer edge of the absorbing region. After discretizing the radial variables using a discrete variable representation (DVR) with uniform grid spacing, the three-body Schrödinger-like equation is converted into a large sparse linear algebraic equation (dimension ~10^5), which is solved using the Bi-conjugate gradient method with preconditioning. The method is applied to the \(d + ^{58}\text{Ni}\) breakup reaction at 80 MeV incident deuteron energy, using the same Hamiltonian and optical potentials as the CDCC method of Yahiro et al.

## Key numbers

- Incident deuteron energy: \(E_d = 80\) MeV for the \(d + ^{58}\text{Ni}\) reaction
- Absorbing potential parameters: \(W_{PT} = 50\) MeV, \(R_c = 25\) fm, \(\Delta R = 25\) fm; \(W_{nC} = 20\) MeV, \(r_c = 25\) fm, \(\Delta r = 25\) fm
- Radial grid spacings: \(H = 0.2\) fm for \(R\) (projectile-target coordinate), \(h = 0.5\) fm for \(r\) (neutron-core coordinate)
- Total angular momentum \(J\) up to 100, including \(l = 0\) and \(l = 2\) for the \(p\)-\(n\) relative angular momentum
- Matrix dimension: \(N_{\text{dim}} = 10^5\) (product of \(N_J = 4\) angular momentum channels, \(N_R = 250\) grid points for \(R\), and \(N_r = 100\) grid points for \(r\))
- Deuteron binding energy: 2.2177 MeV (DVR calculation with 0.5 fm spacing gives same value as the exact 2.2177 MeV)
- For the two-body test case (\(^{16}\text{O} + ^{12}\text{C}\) at \(E_{\text{lab.}} = 139.2\) MeV): absorbing potential parameters \(r_c = 20\) fm, \(\Delta r = 10\) fm, with \(W_{\text{abs}} = 200\) MeV satisfying the WKB condition

## Context

This work extends the absorbing boundary condition method, previously developed for chemical reactions (Kosloff, Neuhauser, Seideman), to nuclear breakup reactions described as three-body direct reactions. It provides an alternative to the established coupled discretized continuum channels (CDCC) method of Yahiro, Austern, and Sakuragi, which requires discretizing continuum states. The ABC method avoids introducing discretized continuum channels and enables efficient iterative solution of the scattering problem. The results for \(d + ^{58}\text{Ni}\) at 80 MeV are compared directly with CDCC calculations from Yahiro et al., showing accurate agreement. This approach is particularly relevant for reactions involving weakly-bound projectiles such as halo nuclei, where breakup processes are significant.
