# PLATYPUS: a code for fusion and breakup in reactions induced by weakly-bound nuclei within a classical trajectory model with stochastic breakup

> A self-contained Fortran-90 program based on a classical trajectory model with stochastic breakup is presented, which should be a powerful tool for quantifying complete and incomplete fusion, and breakup in reactions induced by weakly-bound two-body projectiles near the Coulomb barrier. The code calculates complete and incomplete fusion cross sections and their angular momentum distribution, as well as breakup observables (angle, kinetic energy and relative energy distributions).

## Key claim

The PLATYPUS code provides a classical trajectory model with stochastic breakup that can quantify complete fusion, incomplete fusion, and breakup observables for reactions induced by weakly-bound two-body projectiles near the Coulomb barrier, but it is not a breakup model itself; rather it requires a stochastically sampled breakup function as input, which must be determined from sub-barrier breakup measurements.

## Method

The code uses a three-dimensional classical dynamical model combined with Monte Carlo sampling of probability-density distributions to simulate the trajectories of the projectile fragments and target after stochastic breakup. For each partial wave of the incident projectile, the code samples initial conditions including excitation energy, relative angular momentum between fragments, separation distance, breakup radius, and fragment orientation, then integrates the classical equations of motion using a fourth-order Runge-Kutta method. During time propagation, the code determines capture of breakup fragments by the target to classify events as complete fusion, incomplete fusion, or no-capture breakup, and calculates the corresponding cross sections and angular momentum distributions.

## Key numbers

- Total incomplete fusion cross section (SIGMA_ICF): 0.2883E+03 mb for the test case of pseudo-8Be projectile with 208Pb target
- Total complete fusion cross section (SIGMA_CF(TOT)): 0.2986E+03 mb for the same system
- Complete fusion cross section for inert projectile (SIGMA_CF(INERT PROJ)): 0.6521E+03 mb
- No-capture breakup cross section (SIGMA_BU): 0.2653E+03 mb
- Running time: approximately 75 minutes on a PC with 1.5 GHz processor for the provided input
- RAM requirement: 10 MB
- Sampling: 1000 breakup events per partial wave in the example calculation

## Context

This work extends classical trajectory methods previously applied to multi-fragmentation and incomplete fusion in heavy-ion reactions at energies well above the Coulomb barrier (greater than 10 MeV/nucleon) as in Refs. Moehring and Bondorf, adapting them to near-barrier energies. The approach addresses limitations of the continuum-discretised coupled-channels (CDCC) model, which cannot calculate integrated incomplete and complete fusion cross sections unambiguously, and provides a tool for interpreting fusion and breakup measurements of weakly-bound nuclei such as those by Rafiei, Luong, and Santra. The code enables future studies of isomer ratios and gamma-ray spectroscopy applications as discussed in Ref. Alexis1.
