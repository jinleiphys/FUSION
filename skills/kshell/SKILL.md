# Driving KSHELL

KSHELL diagonalizes the nuclear shell-model Hamiltonian in an M-scheme basis with
the thick-restart (block) Lanczos method. Given a valence space and an effective
interaction, it returns the lowest eigenstates: their energies, spins, parities,
isospins, occupations and quadrupole moments, and, through the companion `transit`
executable, transition strengths between them.

Upstream: the maintained GaffaSnobb fork on GitHub (Python 3.10+ tooling), Fortran 90
+ OpenMP, needs LAPACK/BLAS and Python 3.10+. It ships 25 interaction files. See
`references/failure-modes.md` for the license situation.

## Prime rules (do not skip)

1. **Content is the verdict, never the exit status.** `kshell.exe` writes an
   eigenvalue summary (`N <H>: E <JJ>: JJ J: 2J/2 prty P`). Success means a zero
   exit and a finite negative ground state, ascending energies, and the requested
   number of states in that summary. `run_kshell.sh` asserts these from the log.
2. **This is a TIER 2 skill.** KSHELL ships no reference eigenvalue for a fixed
   case, so verify is input alignment + cross-build agreement + a physics anchor,
   honestly not a shipped-reference reproduction. The 20Ne USDA benchmark g.s. is
   -40.46689 MeV, identical to all five printed decimals across gfortran 13.3 and
   15.2. See `references/verification.md`.
3. **Valence numbers are relative to the interaction's CORE, not Z and N.** For
   20Ne with USDA (16O core) it is 2 valence protons and 2 valence neutrons, not
   10 and 10. Pick an interaction whose model space contains the nucleus.
4. **Two inputs plus a namelist.** An interaction `.snt` (shipped in `snt/`) and a
   partition `.ptn` generated by `gen_partition.py` (Python 3.10+, prompts on stdin,
   feed `0` for the full space), tied by the `&input ... &end` namelist that
   `kshell.exe` reads as its argument. Field reference: `references/input-format.md`.
5. **`mtot` is 2*M** (0 for even-A, 1 for odd-A), not the spin J; J comes out of
   the diagonalization. Only `fn_int`, `fn_ptn`, `mtot`, `n_eigen` change the
   eigenvalues; `hw_type`, `eff_charge`, `gl`, `gs` affect only transitions.
6. **No em-dashes in any prose you write** (user's flat rule).

## Environment (auto-install)

`scripts/install_kshell.sh` clones the GaffaSnobb fork, builds it with gfortran
(overriding LIBS per platform: Accelerate on macOS, `-llapack -lblas` on Linux,
plus `-fallow-argument-mismatch` for gfortran 10+), runs the 20Ne probe, and
prints four lines:

```
KSHELL=<path to kshell.exe>
KSHELL_ROOT=<repo root>
KSHELL_SNT=<snt/ interaction directory>
KSHELL_GENPTN=<bin/gen_partition.py>
```

About 20 s from a cold clone. Requires `git`, `gfortran`, `python3` (3.10+), and LAPACK/BLAS
(Accelerate on macOS provides it).

## Command line

```
gen_partition.py <interaction.snt> <case.ptn> <valence_p> <valence_n> <parity>
kshell.exe <case.input>
```

## Workflow

```bash
bash scripts/install_kshell.sh                       # prints KSHELL=, KSHELL_SNT=, ...
bash scripts/run_kshell.sh                            # 20Ne USDA, 5 lowest M=0 states
bash scripts/run_kshell.sh usda.snt 4 4 1 0 5         # 24Mg (4 valence p + 4 n above 16O)
bash scripts/run_kshell.sh kb3g.snt 4 4 1 0 3         # 48Cr in the fp shell, KB3G
bash scripts/verify_kshell.sh                         # tier-2 benchmark
bash scripts/selftest_kshell.sh                       # test the harness guards (13 cases)
```

## Verified benchmark

**^20Ne in the sd shell, USDA interaction, five lowest M=0 states:**

| state | E (MeV) | J^pi | Ex (MeV) |
|---|---|---|---|
| 1 | -40.46689 | 0+ | 0.0000 |
| 2 | -38.77105 | 2+ | 1.6958 |
| 3 | -36.37577 | 4+ | 4.0911 |
| 4 | -33.91870 | 0+ | 6.5482 |
| 5 | -32.88208 | 2+ | 7.5848 |

Tier 2: no shipped reference eigenvalue, so L1 is a cross-build integrity check
(the five eigenvalues match to all five printed decimals on gfortran 13.3 and
15.2) and L2 is the physics (the 20Ne rotational band: 0+, 2+ at Ex = 1.70 MeV
against experiment 1.63, 4+; USDA is a fitted interaction). Full account:
`references/verification.md`.

## Failure modes

See `references/failure-modes.md`. The ones that bite: use the GaffaSnobb fork
(the jorgenem mirror is Python 2); the build needs `-fallow-argument-mismatch` on
gfortran 10+ and Accelerate (not `-llapack`) on macOS; valence numbers are
relative to the core; `gen_partition` can leave an empty `.ptn`; and success is
the eigenvalue summary, not the exit status.
