# Pointlike constituent quarks and scattering equivalences

> In this paper scattering equivalences are used to simplify current operators in constituent quark models. The simplicity of the method is illustrated by applying it to a relativistic constituent quark model that fits the meson mass spectrum. This model requires a non-trivial constituent quark current operator to fit the pion form factor data. A model with a different confining interaction, that has the identical spectrum and can reproduce the measured pion form factor using only point-like constituent quark impulse currents is constructed. Both the original and transformed models are relativistic direct-interaction models with a light-front kinematic subgroup.

## Key claim

Scattering equivalences (unitary transformations that are finite-rank perturbations of the identity) can map a relativistic constituent quark model that fits the meson mass spectrum but requires non-trivial quark current operators to fit the pion form factor, into a different model with the identical spectrum that reproduces the measured pion form factor using only point-like constituent quark impulse currents.

## Method

The authors start with the relativistic constituent quark-antiquark model of Carlson, Kogut, and Pandharipande (1995), which fits the meson mass spectrum but fails to reproduce the pion form factor when using point-like impulse currents. They construct a rank-one unitary scattering equivalence operator \(A = I + \Delta\) that maps the original pion ground state \(|m_\pi\rangle\) to a new pion ground state \(|\bar{m}_\pi\rangle\) obtained from a mass operator with different confining interaction parameters. The transformed mass operator \(M' = M_0 + V_c'\) has the same spectrum as the original but uses the new pion wave function, allowing the pion form factor to be computed with point-like quark currents; the transformation simultaneously generates exchange current contributions in the original representation.

## Key numbers

* Overlap between original and transformed pion ground states: \(\cos(\theta) = \langle m_\pi | \bar{m}_\pi \rangle = 0.731\), giving \(\rho = 3.71\).
* Original Carlson-Kogut-Pandharipande model parameters: quark mass \(m = 0.360\) GeV, \(\lambda_1 = -0.777\) GeV, \(\lambda_2 = -0.5\), \(\lambda_3 = 0.197\) GeV\(^2\), \(\lambda_4 = 0.66\) GeV\(^{-1}\).
* Transformed interaction \(\bar{V}_c\) parameters: \(m = 0.360\) GeV, \(\lambda_1 = -0.911\) GeV, \(\lambda_2 = -0.14\), \(\lambda_3 = 0.049\) GeV\(^2\), \(\lambda_4 = 0.35\) GeV\(^{-1}\).
* Pion decay constant in original model: 193 MeV; after transformation: 131 MeV (experimental value: 92 MeV).
* Overlap probability of original and transformed ground states: \(P = \cos^2(\theta) = 0.53\), meaning 0.47 probability for the transformed ground state to be in excited states of the original model.

## Context

This work extends the scattering equivalence formalism of Polyzou (2003) and applies it to the relativistic constituent quark model of Carlson, Kogut, and Pandharipande (1995). It directly addresses the representation dependence of current operators noted by Cardarelli et al. (1995), who needed single-quark form factors to fit pion data. The method enables construction of equivalent models with simplified current operators, and the authors note that scattering equivalences also exist between models with different kinematic subgroups (light-front, Euclidean, Lorentz), as discussed by Bakamjian and Thomas (1953) and Foldy (1956). Future work could apply more general (compact) \(\Delta\) operators or lower constituent quark masses to improve the pion decay constant prediction.
