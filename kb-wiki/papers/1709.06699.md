# A practical scheme for constructing the minimum weight states of the su(n)-Lipkin model in arbitrary fermion number

> With the aim of performing an argument supplement to the previous paper by the present authors, in this paper, a practical scheme for constructing the minimum weight states of the su(n)-Lipkin model in arbitrary fermion number is discussed. The idea comes from the following two points : (i) consideration on the property of one-fermion transfer induced by the su(n)-generators in the Lipkin model and (ii) use of the auxiliary su(2)-algebra presented by the present authors. The form obtained under the points (i) and (ii) is simple.

# A practical scheme for constructing the minimum weight states of the su(n)-Lipkin model in arbitrary fermion number

## Key claim

The paper presents a practical scheme for constructing the minimum weight states of the su(n)-Lipkin model for arbitrary fermion number, supplementing the authors' earlier work (Tsue et al., 2017) which gave a form that was "not practical" due to complicated commutation relations among generators with different n'. The new scheme avoids the normalization difficulties of the earlier approach by using an auxiliary su(2)-algebra that commutes with all su(n)-generators, allowing the minimum weight state to be expressed as a normalized raising-operator product acting on a simpler base state.

## Method

The authors construct the minimum weight state |m⟩ of the su(M+1)-Lipkin model by first defining a base state |m₀⟩ that satisfies both the minimum weight conditions and the auxiliary su(2)-algebra conditions Λ₋|m₀⟩=0 and Λ₀|m₀⟩=−λ|m₀⟩. The full minimum weight state is then obtained as |m⟩ = (Λ₊)^{λ+λ₀}|m₀⟩, where Λ₊ is the raising operator of the auxiliary su(2)-algebra that commutes with all su(M+1)-generators. The base state |m₀⟩ is constructed explicitly as a product of fermion creation operators acting on the vacuum, with occupation numbers ν_{i,μ} that satisfy ν₀ ≥ ν₁ ≥ ... ≥ ν_L > 0 and ν_{L+1}=...=ν_M=0, reflecting the property that one-fermion transfer via the su(n)-generators preserves the quantum number μ.

## Key numbers

- The auxiliary su(2)-algebra operator Λ₀ has eigenvalue λ = Ω − (1/2)ν₀ for the base state |m₀⟩, where Ω is half the number of single-particle states per level and ν₀ is the fermion number in level 0.
- The total fermion number for the base state |m₀⟩ is N_min = ν₀ + ν(L), where ν(L) = Σ_{i=1}^{L} ν_i.
- The maximum fermion number is N_max = 2Ω·(M+1) − Mν₀ − ν(L) = N_min + (M+1)·2λ.
- The eigenvalue of the operator ℒ (which commutes with all su(n)-generators) is ℒ = Ω − ((M−1)/(2(M+1)))ν₀ + (1/(M+1))ν(L).
- The parameter λ₀ for arbitrary fermion number N is λ₀ = N/(M+1) − (Ω − ((M−1)/(2(M+1)))ν₀ + (1/(M+1))ν(L)), with the inequality −λ ≤ λ₀ ≤ λ.
- For the case L=0, the allowed domains are: (i) 0 ≤ N ≤ 2Ω with 0 ≤ ν₀ ≤ N, and (ii) 2Ω ≤ N ≤ 2(M+1)Ω with 0 ≤ ν₀ ≤ (2(M+1)Ω−N)/M.

## Context

This work extends the authors' previous treatment of the su(n)-Lipkin model (Tsue et al., 2017, referred to as (I)) which gave minimum weight states in terms of raising operators of auxiliary su(2)-algebras for su(n')-Lipkin models with n' ≤ n, but with complicated commutation relations that made normalization tedious. The present scheme builds on the introduction of an auxiliary su(2)-algebra into the su(2)-Lipkin model (Tsue et al., 2017) and generalizes it to su(n). The approach enables practical construction of normalized minimum weight states for arbitrary fermion number, which is necessary for algebraic many-body treatments; the authors indicate that in a forthcoming paper they will apply this to a random phase approximation and discuss phase changes.
