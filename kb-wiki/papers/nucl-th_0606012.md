# Simultaneous description of four positive and four negative parity bands

> The extended coherent state model is further extended in order to describe two dipole bands of different parities. The formalism provides a consistent description of eight rotational bands. A unified description for spherical, transitional and deformed nuclei is possible. Projecting out the angular momentum and parity from a sole state, the $K^π=1^+$ band acquires a magnetic character, while the electric properties prevail for the other band. Signatures for a static octupole deformation in some states of the dipole bands are pointed out. Some properties which distinguish between the dipole band states and states of the same parity but belonging to other bands are mentioned. Interesting features concerning the decay properties of the two bands are found. Numerical applications are made for $^{158}$Gd, $^{172}$Yb, $^{228,232}$Th, $^{226}$Ra, $^{238}$U and $^{238}$Pu, and the results are compared with the available data.

## Key claim

The extended coherent state model (ECSM) is further extended to describe two dipole bands of opposite parity ($K^\pi=1^+$ and $K^\pi=1^-$) in addition to the six previously described bands, but the description relies on the condition that the new Hamiltonian terms affect only diagonal matrix elements of the dipole states, with off-diagonal couplings to other bands amounting to only a few keV.

## Method

The authors construct intrinsic states for the dipole bands using an octupole boson excitation operator $\Omega^\dagger_3 b^\dagger_{31}$ acting on parity-projected octupole coherent states, ensuring orthogonality to the intrinsic states of the six previously defined bands ($g^\pm$, $\beta^\pm$, $\gamma^\pm$). Angular momentum and parity are projected from these intrinsic states to generate the rotational band members. Two new terms, ${\cal C}_1\Omega^\dagger_3\Omega_3$ and ${\cal C}_2\Omega^\dagger_3\hat{N}_2\Omega_3$, are added to the model Hamiltonian; ${\cal C}_2$ is fixed to cancel the ${\cal B}_1$ contribution in the negative dipole band, and ${\cal C}_1$ is fixed to reproduce the measured $1^-$ excitation energy. Numerical applications are performed for seven nuclei: $^{158}$Gd, $^{172}$Yb, $^{226}$Ra, $^{228}$Th, $^{232}$Th, $^{238}$U, and $^{238}$Pu.

## Key numbers

- For $^{158}$Gd: the $1^+$ state is calculated at 2.531 MeV (experiment: 2.534 MeV); the $1^-$ state is calculated at 1.856 MeV (experiment: 1.856 MeV).
- For $^{172}$Yb: the $1^+$ state is calculated at 1.880 MeV (experiment: 2.010 MeV); the $1^-$ state is calculated at 1.155 MeV (experiment: 1.155 MeV).
- For $^{226}$Ra: the $1^-$ state is calculated at 1.049 MeV (experiment: 1.080 MeV); the $3^+$ state is calculated at 1.420 MeV (experiment: 1.422 MeV).
- For $^{228}$Th: the $1^-$ state is calculated at 0.952 MeV (experiment: 0.952 MeV); the $1^+$ state is calculated at 1.247 MeV (no experimental value listed).
- For $^{232}$Th: the $1^+$ state is calculated at 1.508 MeV (experiment: 1.489 MeV); the $1^-$ state is calculated at 1.078 MeV (experiment: 1.078 MeV).
- For $^{238}$U: the $1^+$ state is calculated at 1.367 MeV (experiment: 1.354 MeV); the $1^-$ state is calculated at 0.967 MeV (experiment: 0.967 MeV).
- For $^{238}$Pu: the $1^+$ state is calculated at 1.343 MeV (experiment: 1.310 MeV); the $1^-$ state is calculated at 0.863 MeV (experiment: 0.963 MeV).
- Deformation parameters: $d$ ranges from 3.0 ($^{158}$Gd, $^{226}$Ra, $^{228}$Th) to 3.9 ($^{238}$U, $^{238}$Pu); $f$ ranges from 0.3 (most nuclei) to 0.8 ($^{226}$Ra).
- Structure coefficients (in keV): ${\cal A}_J$ ranges from 0.49 ($^{226}$Ra) to 4.72 ($^{172}$Yb); ${\cal A}_{J23}$ ranges from 4.70 ($^{172}$Yb) to 15.0 ($^{158}$Gd).

## Context

This work extends the Extended Coherent State Model (ECSM) previously developed by Raduta and collaborators (Raduta et al., various years), which described six rotational bands ($g^\pm$, $\beta^\pm$, $\gamma^\pm$) by projecting angular momentum and parity from intrinsic states with both quadrupole and octupole deformations. The new addition of $K^\pi=1^\pm$ dipole bands allows a consistent description of eight rotational bands within a single framework, applicable to spherical, transitional, and deformed nuclei. The results enable further investigation of signatures for static octupole deformation, such as the interleaved structure of moments of inertia and vanishing energy displacement functions observed in $^{172}$Yb, $^{226}$Ra, $^{238}$U, and $^{238}$Pu.
