# A numerical implementation of the NLO DIS structure functions in the dipole picture

> We present a numerical program that evaluates deep inelastic scattering (DIS) structure functions at next-to-leading order (NLO) accuracy in the dipole picture. In this numerical implementation the NLO DIS impact factors with massive quarks are written in a form that ensures a stable numerical evaluation of the DIS cross sections.

## Key claim

The authors present a numerical program that evaluates deep inelastic scattering (DIS) structure functions at next-to-leading order (NLO) accuracy in the dipole picture, but the implementation is limited to the case where the impact parameter dependence of the dipole-target scattering amplitude is factorized out, and the program does not support impact-parameter-dependent dipole amplitudes for nuclear targets without additional approximations.

## Method

The program implements the NLO DIS impact factors with massive quarks calculated by Beuf et al. (2021, 2022), separating the total cross section into three contributions: the lowest-order quark-antiquark term evaluated at the initial condition of the Balitsky-Kovchegov evolution, the NLO dipole term containing loop corrections to the quark-antiquark system, and the quark-antiquark-gluon term where a three-parton state interacts with the target shockwave. The dipole-target scattering amplitude is provided externally as a function of dipole size and evolution rapidity, and the multidimensional integrals are performed using the Vegas algorithm from the Cuba library with the GNU Scientific Library.

## Key numbers

* The program computes structure functions at NLO accuracy; an example calculation yields F2 = 0.875895 at Q2 = 8.5 GeV2 and xbj = 0.001 using the provided dipole amplitude datafile.
* The proton transverse area is set to 9.08 mb (23.3192 GeV-2) in the example configuration.
* The charm quark mass is set to 1.24 GeV and light quark mass to 0.005 GeV in the example.
* The running coupling scale parameter C2 is set to 1.74, with the coupling evaluated at scale 4C2/r2.
* The non-perturbative scale Q02 is set to 1 GeV2 for the evolution rapidity calculation.
* The maximum dipole size parameter (maxr) is set to 30 GeV-1 in the example configuration.

## Context

This work extends the NLO DIS calculations of Beuf et al. (2021, 2022) by providing a publicly available numerical implementation that can be used with BK-evolved dipole amplitudes from existing fits such as those by Casuga et al. (2025) and Hanninen et al. (2022). The program enables precision comparisons with HERA data and predictions for the Electron-Ion Collider, including nuclear structure functions through the optical Glauber model approach of Lappi et al. (2013). The code is available on GitHub and Zenodo.
