# Realistic Ghost State : Pauli Forbidden State from Rigorous Solution of the Alpha Particle

> The antisymmetrization of the composite particles in cluster model calculations manifests itself in Pauli forbidden states (ghost states), if one restricts oneself on undeformed cluster in the low energy region. The resonating group method and the generating coordinate method rely on a property of the norm kernel, which introduces some of the ghost states. The norm kernel has been usually calculated under the assumption that the inner wavefunctions have a simple Gaussian form. It is the first time that this assumption is tested by the rigorous way. In the 4He+N system, we demonstrate a ghost state, which is calculated from a rigorous solution of Yakubovsky equations for the alpha particle. The ghost states calculated by rigorous and approximate methods turn out to have a very similar form. It is analytically proved that the trace of the norm kernel does not depend on the inner wavefunction we choose.

## Key claim

The paper demonstrates for the first time that the Pauli forbidden state (ghost state) in the 4He+N system, calculated from a rigorous solution of the Yakubovsky equations for the alpha particle, has a very similar form to the ghost state obtained using the conventional Gaussian approximation for the inner cluster wavefunction.

## Method

The authors solved the Yakubovsky equations rigorously to obtain the exact four-nucleon wavefunction of the alpha particle. They then used this realistic wavefunction to compute the norm kernel for the 4He+N system and diagonalized it to extract the Pauli forbidden state (ghost state) with eigenvalue gamma_n = 1. The resulting ghost state was compared with the one obtained using the standard Gaussian approximation for the alpha particle internal wavefunction.

## Key numbers

* The ghost state eigenvalue for the norm kernel is exactly gamma_n = 1, by definition of the Pauli forbidden state.
* The momentum-space ghost state u_0(q) was computed over a range from q = 0 to q = 5 fm^{-1}, as shown in the first figure.
* The overlap ratio R between the rigorous and approximate ghost states was computed as a function of Omega (a Gaussian width parameter) from approximately 0.16 to 0.30 fm^{-2}, with R values ranging from about 90% to 100%.
* The coordinate-space ghost state u_0(r) was computed over a range from r = 0 to r = 7 fm, as shown in the third figure.
* The coordinate-space ghost state u_1(r) was computed over a range from r = 0 to r = 7 fm, as shown in the fourth figure.

## Context

This work tests the long-standing assumption in the resonating group method (RGM) and generating coordinate method (GCM), dating back to Wheeler (1937), that the inner cluster wavefunctions can be approximated by simple Gaussian forms. The authors extend prior cluster model studies of 8Be (Tang, Kircher), 12C (Kamimura, Fujiwara, Kamada), and 16O (Oryu) by providing the first rigorous test of this Gaussian assumption using exact solutions of the Yakubovsky equations. The analytical proof that the trace of the norm kernel does not depend on the choice of inner wavefunction provides a theoretical justification for why the Gaussian approximation works well, and this result enables future cluster model calculations to be performed with confidence in the Gaussian approximation for the alpha particle internal structure.
