# Coarse graining hadronic scattering

> We show that it makes sense to coarse grain hadronic interactions such as $ππ$ and $πN$ reactions following previous work on NN scattering. Moreover, if the interaction is taken to be given by chiral dynamics at long distances above a given value $r > r_c$ larger than the elementary radii of the interaction hadrons the unknown short distance region $r< r_c$ is characterized by a {\it finite} number of fitting parameters. This number of independent parameters needed for a presumably complete description of scattering data for a CM energy below $\sqrt{s}$ has been found to be given by $N_{\rm Par} = N_S \times N_I \times (p r_c )^2 /2 $ with $N_S$ and $N_I$ the number of spin and isospin channels, and $p$ the CM momentum respectively. Therefore, for an experiment (or sets of experiments) with a total number of data $N_{\rm Dat}$ the number of degrees of freedom involved in a $χ^2$-fit is given by $ν= N_{\rm Dat}-N_{\rm Par}$ and confidence levels can be obtained accordingly by standard means. Namely a $1 σ$ confidence level corresponds to $χ_{\rm min}^2/ν\in (1- \sqrt{2/ν},1+\sqrt{2/ν})$. We discuss the approach for $ππ$ and $πN$ with an eye put on a data selection program and the eventual validation of chiral symmetry.

# Coarse graining hadronic scattering

## Key claim
The authors argue that hadronic scattering data for ππ, πN, and NN reactions can be described by a finite number of independent parameters determined solely by the maximum center-of-mass momentum and a cutoff radius, independent of experimental precision, provided the long-distance interaction (r > r_c) is given by chiral dynamics and the short-distance region (r < r_c) is coarse-grained. This claim is conditional on the assumption that all published experiments are correct and that the cutoff radius r_c is chosen larger than the elementary radii of the interacting hadrons (approximately 1.5 fm).

## Method
The authors apply a coarse-graining approach in configuration space, using a Schrodinger-type equation with relativistic kinematics where the interaction potential is split into a long-distance part from chiral perturbation theory (one- and two-pion exchange) and a short-distance part represented by delta-shell potentials at discrete radial grid points. The number of independent fitting parameters is estimated as N_Par = N_S × N_I × (p r_c)^2 / 2, where N_S and N_I are the number of spin and isospin channels, p is the CM momentum, and r_c is the cutoff radius. The method is applied to ππ, πN, and NN scattering, with the goal of validating chiral symmetry through data selection and statistical consistency checks.

## Key numbers
- The number of independent parameters for a complete description of scattering data below CM energy √s is given by N_Par = N_S × N_I × (p r_c)^2 / 2, where p is the CM momentum and r_c is the cutoff radius.
- For NN scattering at maximum CM momentum corresponding to T_LAB = 3000 MeV, the SAID database contains 25188 pp data points with χ² = 48225.0 (χ²/N_Dat = 1.9, p-value = 4 × 10⁻¹⁴⁵³) and 12962 np data points with χ² = 26079.9 (χ²/N_Dat = 2.0, p-value = 1 × 10⁻⁸⁸³).
- For πN scattering at T_LAB = 3000 MeV, the SAID database contains 41926 data points with χ² = 166585.05 (χ²/N_Dat = 4.1, p-value = 4 × 10⁻¹⁴⁵¹³); at T_LAB = 300 MeV, there are 2599 data points with χ² = 4586.2 (χ²/N_Dat = 1.8, p-value = 4 × 10⁻¹¹³).
- The cutoff radius for effective elementarity is taken as r_c ≈ 1.5 fm, based on electromagnetic form factors using the Vector Dominance form factor for the pion (m_ρ = 770 MeV) and the dipole form factor for the proton (Λ = 700 MeV).
- The Compton wavelength for one-pion exchange is λ_C = ħ/m_π c = 1.4 fm (for NN); for two-pion exchange it is λ_C = ħ/2m_π c = 0.7 fm (for πN and ππ).
- For ππ scattering below K̅K threshold, the estimated number of parameters is N_Par ≈ 9 = 3 × (S₀₀) + 3 × (S₂₀) + 3 × (P₁₁).
- For πN scattering with √s ≤ m_π + M_Δ = 1.38 GeV, the estimated number of parameters is about 18 = 3 × (S₁₁+S₃₁) + 3 × (P₁₁+P₃₁+P₁₃+P₃₃).
- For NN scattering, N_Par ∼ 50, which explains the traditional number of fitting parameters for high-quality NN analyses.

## Context
This work extends the coarse-graining approach previously applied to NN scattering (Perez, 2013) to ππ and πN reactions, with the goal of arbitrating between inconsistent experimental data sets. The approach contrasts with standard partial wave analyses such as the SAID database, which show poor p-values (e.g., 4 × 10⁻¹⁴⁵¹³ for πN at 3000 MeV) indicating either faulty models or mutually inconsistent data. The method enables a data selection program that can validate chiral perturbation theory predictions, similar to how the Granada-2013 database for NN scattering (Perez, 2013) paved the way for ChPT tests at N5LO (Reinert, 2017). The authors note that if ChPT had been used to select NN scattering data, many more data would have been rejected than in the Granada-2013 database.
