# Cronin momentum behavior in saturation model for $p+A$, $d+A$, $A+A$ collisions

> In this paper we consider Cronin momentum behaviour for $p+A$, $d+A$ and $A+A$ collisions in saturation model. Our analysis shows that Cronin momentum behavior at different rapidities and energies, can be related with scaling law using simple dimensional consideration. Using exact numerical solution of Balitsky-Kovchegov equation we show that although this dependence is slightly different for McLerran-Venugopalan and Balitsky-Kovchegov definition of gluon distribution function in simple model in this case dependencies is almost the same (i.e ratio of Cronin momentum calculated using these gluon distribution functions is big constant). This can be used to experimentally distinguish this two variant of gluon distribution function definition in saturation model and choose the right one.

# Cronin momentum behavior in saturation model for $p+A$, $d+A$, $A+A$ collisions

## Key claim
The simple scaling law for Cronin momentum previously proposed by Abramovsky (2003) does not hold exactly when the dipole forward scattering amplitude is defined via the full Balitsky-Kovchegov evolution equation, but is asymptotically restored only at high energy. The ratio of Cronin momentum calculated using the McLerran-Venugopalan and Balitsky-Kovchegov definitions of the unintegrated gluon distribution function is approximately constant, which could be used to experimentally distinguish between the two definitions.

## Method
The authors numerically solved the Balitsky-Kovchegov evolution equation for the dipole forward scattering amplitude in coordinate space using a second-order implicit integration method, with an initial condition given at $x_0=0.01$ and initial saturation momentum $Q^2_{0s}=2$ GeV. They computed the unintegrated gluon distribution function using both the Balitsky-Kovchegov and McLerran-Venugopalan definitions, then calculated the gluon production cross-section in $k_t$-factorized form for $p+A$, $d+A$, and $A+A$ collisions. The strong coupling constant was fixed by requiring the linear component of the scaling law to match $\lambda=0.3$ at $x=0.01$, consistent with deep inelastic scattering data.

## Key numbers
- The ratio of Cronin momentum calculated using the Balitsky-Kovchegov definition versus the McLerran-Venugopalan definition is approximately constant and equal to $\exp(0.37)$.
- The geometric scaling parameter $\lambda=0.3$ was used, fixed by deep inelastic scattering data at $x=0.01$.
- The slope parameter $b$ in the simple scaling law $\ln(q_C)=a+b\ln(\sqrt{s})$ was calculated as $b=\lambda/(2+\lambda)=0.1304$.
- The initial saturation momentum was set to $Q^2_{0s}=2$ GeV at $x_0=0.01$ with regularization parameter $\epsilon=0.1$.
- The critical anomalous dimension $\gamma_c=0.6275$ solves the equation $\chi(\gamma_c)=\gamma_c\chi'(\gamma_c)$ for the BFKL kernel.

## Context
This work extends the earlier saturation model analysis of Abramovsky (2003), which found a simple scaling law for Cronin momentum in the central rapidity region using an approximate expression for the cross-section and a simpler model for the dipole forward scattering amplitude. The present paper replaces that approximate treatment with the full Balitsky-Kovchegov evolution equation, which belongs to the Fisher-Kolmogorov-Petrovsky-Piscounov class as shown in the BK-FKPP papers. The results enable experimental tests to distinguish between the Balitsky-Kovchegov and McLerran-Venugopalan definitions of the unintegrated gluon distribution function, and provide a baseline for future studies incorporating running coupling effects and finite nucleus size as discussed in the BK numerical solution literature.
