# Analysis of the Daya Bay Reactor Antineutrino Flux Changes with Fuel Burnup

> We investigate the recent Daya Bay results on the changes in the antineutrino flux and spectrum with the burnup of the reactor fuel. We find that the discrepancy between current model predictions and the Daya Bay results can be traced to the original measured $^{235}$U/$^{239}$Pu ratio of the fission beta spectra that were used as a base for the expected antineutrino fluxes. An analysis of the antineutrino spectra that is based on a summation over all fission fragment beta-decays, using nuclear database input, explains all of the features seen in the Daya Bay evolution data. However, this summation method still predicts an anomaly. Thus, we conclude that there is currently not enough information to use the antineutrino flux changes to rule out the possible existence of sterile neutrinos.

## Key claim

The discrepancy between the Daya Bay measured antineutrino flux evolution and the Huber-Mueller model predictions originates from the original measured $^{235}$U/$^{239}$Pu ratio of the fission beta spectra used as a base for the expected antineutrino fluxes; however, a summation method using nuclear database input explains all features of the evolution data but still predicts an anomaly, so there is currently not enough information to use the antineutrino flux changes to rule out the possible existence of sterile neutrinos.

## Method

The authors refitted the original 1980s Institute Laue-Langevin (ILL) aggregate beta spectra for $^{235}$U and $^{239}$Pu, varying assumptions about allowed versus first-forbidden beta transitions and the effective charge and mass assignments ($Z_{\rm eff}$, $A_{\rm eff}$) used in the Fermi function. They also performed a summation calculation using JEFF-3.1 cumulative fission yields combined with a preliminary version of the ENDF/B-VIII.0 decay data sub-library to predict antineutrino spectra and inverse beta decay (IBD) yields, comparing these to the Daya Bay measurements of the IBD yield as a function of the $^{239}$Pu fission fraction $F_{239}$.

## Key numbers

- Daya Bay measured average IBD yield: $\overline{\sigma}_f = (5.9 \pm 0.13) \times 10^{-43}$ cm$^2$/fission
- Daya Bay measured change in IBD yield with $^{239}$Pu fraction: $\frac{d\sigma_f}{dF_{239}} = (-1.86 \pm 0.18) \times 10^{-43}$ cm$^2$/fission
- Daya Bay measured IBD yield for $^{235}$U: $\sigma_{235} = (6.17 \pm 0.17) \times 10^{-43}$ cm$^2$/fission
- Daya Bay measured IBD yield for $^{239}$Pu: $\sigma_{239} = (4.27 \pm 0.26) \times 10^{-43}$ cm$^2$/fission
- Daya Bay measured ratio $\sigma_{235}/\sigma_{239} = 1.445 \pm 0.097$
- Huber-Mueller model ratio $\sigma_{235}/\sigma_{239} = 1.53 \pm 0.05$
- Huber-Mueller model value for $\frac{d\sigma_f}{dF_{239}} = (-2.46 \pm 0.06) \times 10^{-43}$ cm$^2$/fission, differing from Daya Bay by 3.1 sigma
- Summation method predicts $\sigma_{235}/\sigma_{239} = 1.445$ and an average IBD yield 3.5% higher than observed
- Summation predictions for $\overline{S}^{-1}dS_j/dF_{239}$ in four prompt energy ranges: (-0.143, -0.273, -0.521, -0.678), compared to experimental values (-0.16$\pm$0.07, -0.23$\pm$0.04, -0.49$\pm$0.05, -0.69$\pm$0.12)

## Context

This work extends the analysis of the Daya Bay reactor neutrino experiment (Daya Bay Collaboration, 2017) which measured changes in antineutrino flux and spectrum with fuel burnup, confirming the "reactor neutrino anomaly" first identified by Mention et al. (2011). The authors show that the Huber-Mueller model (Huber, 2011; Mueller et al., 2011) predicts a $\sigma_{235}/\sigma_{239}$ ratio constrained by the original ILL beta spectra measurements of Schreckenbach et al. (1985), while the summation method using nuclear databases provides a better description of the evolution data but still predicts an anomaly. The work enables future very short baseline neutrino experiments to resolve the sterile neutrino question and motivates a re-measurement of the aggregate fission beta spectra of $^{235}$U and $^{239}$Pu.
