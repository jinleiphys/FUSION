# A First-Landau-Level Laughlin/Jain Wave Function for the Fractional Quantum Hall Effect

> We show that the introduction of a more general closed-shell operator allows one to extend Laughlin's wave function to account for the richer hierarchies (1/3, 2/5, 3/7 ...; 1/5, 2/9, 3/13, ..., etc.) found experimentally. The construction identifies the special hierarchy states with condensates of correlated electron clusters. This clustering implies a single-particle (ls)j algebra within the first Landau level (LL) identical to that of multiply filled LLs in the integer quantum Hall effect. The end result is a simple generalized wave function that reproduces the results of both Laughlin and Jain, without reference to higher LLs or projection.

## Key claim

The paper shows that Laughlin's wave function for the fractional quantum Hall effect can be generalized by replacing a closed $\ell$-shell with a closed $\ell s$-shell, where $2s+1$ counts the number of electrons in correlated clusters; this construction reproduces the results of both Laughlin and Jain without reference to higher Landau levels or projection, but only for those special hierarchy states where the ground state is filled by a condensate of clusters.

## Method

The authors confine electrons to the surface of a sphere with a magnetic monopole at its center, following Haldane (1983). They construct a generalized wave function by replacing Laughlin's closed-shell operator $\prod_{i<j} u(i)\cdot u(j)$ with a closed $\ell s$ shell operator that incorporates derivatives $d(i)\cdot d(j)$ to reduce magnetic flux and antisymmetrization to separate clusters of $N_c$ electrons. The resulting wave function is recast in an $(\ell s)j$ angular momentum algebra, where $2\ell+1 = N/N_c$ is the number of clusters and $2s+1 = N_c$ is the cluster size.

## Key numbers

- Overlaps $|\langle \Psi_{exact}|\Psi_{m,N_c}\rangle|$ for the generalized wave function with exact numerical diagonalization are $\geq 0.993$ for all tested cases with $N_c \geq 2$, for systems up to $N=10$ electrons and $2S=24$ flux quanta.
- For the 2/5 filling state at $(2S,N) = (11,6)$, the overlap is 0.9998; for $(2S,N) = (16,8)$, the overlap is 0.9996.
- For the 2/7 filling state at $(2S,N) = (19,6)$, the overlap is 0.9964, differing from Jain's projected wave function by 0.0008.
- The original Laughlin wave function ($N_c=1$) gives overlaps ranging from 0.9741 (for 4 electrons at 1/7 filling) to 0.9991 (for 5 electrons at 1/3 filling).
- The fractional filling for large $N$ and fixed cluster size $N_c$ is $N_c/((m-1)N_c+1)$, yielding the series 1/3, 2/5, 3/7 for $m=3$ and 1/5, 2/9, 3/13 for $m=5$.

## Context

This work extends Laughlin's (1983) original wave function for fillings $1/m$ and provides a physical interpretation for Jain's (1989) construction, which required excitations into higher Landau levels followed by numerical projection. The clustering mechanism implies an $(\ell s)j$ algebra within the first Landau level that is identical to multiply filled Landau levels in the integer quantum Hall effect, but with shell gaps determined by Coulomb pairing energy rather than magnetic energy. This enables a unified description of the fractional quantum Hall effect hierarchy and provides a starting point for exploring excited state spectroscopy, extending the work of Rezayi and Read (1994) on half-filled systems.
