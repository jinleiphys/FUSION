# Longitudinal Spin Physics at RHIC and a Future eRHIC

> Recent highlights from the spin program at the Relativistic Heavy Ion Collider (RHIC), focusing on the gluon contribution to the proton spin and the polarization of the light flavor sea, are presented. The impact of these data on recent global fits by the DSSV and NNPDF groups is also discussed. Finally, this note examines the constraints on the longitudinal spin structure of the proton which would be possible with a proposed eRHIC facility.

## Key claim

The RHIC spin program has provided evidence for the first non-zero gluon contribution to the proton spin, but only over a restricted Bjorken-x range (0.05 ≤ x ≤ 1.0), while the low-x region (x < 0.05) remains largely unconstrained.

## Method

The paper presents results from the STAR and PHENIX experiments at RHIC, which measured longitudinal double-spin asymmetries (ALL) in polarized proton-proton collisions at √s = 200 GeV using inclusive jets (STAR) and inclusive π0 production (PHENIX). These data were incorporated into next-to-leading order (NLO) global analyses by the DSSV and NNPDF collaborations to extract the gluon helicity distribution Δg(x,Q²). Additionally, single-spin asymmetries (AL) from W boson production at √s = 500-510 GeV were used to constrain flavor-separated polarized sea quark distributions.

## Key numbers

- DSSV global analysis including 2009 RHIC data yields Δg(x,Q²) = 0.20^{+0.06}_{-0.07} in the range 0.05 ≤ x ≤ 1.0 at Q² = 10 GeV², the first non-zero extraction
- NNPDF re-weighted analysis gives Δg(x,Q²) = 0.23 ± 0.06 in the same x range (0.05 ≤ x ≤ 1.0), consistent with DSSV
- The original DSSV extraction (without 2009 data) gave Δg(x,Q²) = 0.005^{+0.129}_{-0.164} in the range 0.05 ≤ x ≤ 0.2
- STAR inclusive jet ALL results from 2009 data are a factor of 3 to 4 more precise than previous results from the 2006 data set
- Fixed-target polarized DIS experiments show quark and anti-quark helicities contribute roughly 30% to the proton spin in the range 0.001 ≤ x ≤ 1.0

## Context

This work extends the global analyses of polarized parton distribution functions (PDFs) by the DSSV collaboration (deFlorian 2008, 2009) and the NNPDF collaboration (Nocera 2014), incorporating new RHIC data that significantly improve constraints on the gluon helicity. The results enable future precision measurements at the proposed eRHIC facility (Aschenauer 2014), which would use polarized electron-proton collisions to access a much wider kinematic range in x and Q² than current fixed-target DIS experiments, potentially resolving the low-x gluon polarization and providing flavor-separated PDFs through both SIDIS and charged-current DIS measurements.
