# Quark-Exchange Mechanism of $γd \to np$ Reaction At 2-6 GeV

> Within the constituent quark model, we examine the extent to which the deuteron photo-disintegration at 2-6 GeV can be described by the quark-exchange mechanism. With the parameters constrained by the $np$ scattering, the calculated differential cross sections disagree with the data in both magnitude and energy-dependence. The results can be improved if we use a smaller size parameter for quark wavefunctions. We also find that the on-shell approximation used in a previous investigation is not accurate

## Key claim

Within the constituent quark model, the quark-exchange mechanism alone cannot describe the deuteron photo-disintegration data at 2-6 GeV when parameters are constrained by np scattering; the calculated differential cross sections disagree with data in both magnitude and energy-dependence, though results can be improved by using a smaller size parameter for the quark wavefunctions.

## Method

The authors performed an exact calculation of the quark-exchange amplitude for the gamma d to np reaction, using Gaussian wavefunctions for the three-quark clusters with a range parameter b, a gluon-exchange interaction V(q) with parameters from a pi N scattering study, and a quark propagator G(q_p) with constituent quark mass m_q = 330 MeV. The multi-dimensional integration was carried out by expanding the gluon-exchange interaction and deuteron wavefunction in Gaussians (N_V=35 and N_G=50 terms), then performing all integrations analytically except for the propagating quark momentum, which was integrated numerically. The np scattering wavefunction was generated from a parameterized t-matrix fitted to np elastic scattering data in the 4-12 GeV region using exchanges of pi, sigma, rho, omega, and Pomeron.

## Key numbers

* With the standard quark model size parameter b = 0.52 fm, the predicted cross sections are an order of magnitude smaller than the data.
* To get reasonable magnitudes at 90 degrees, a much smaller value b = 0.25 fm was needed.
* The gluon-exchange interaction parameters were alpha_s = 1 and Lambda_g = 1.087 GeV, determined from fitting pi N scattering data.
* The constituent quark mass was set to m_q = 330 MeV, with the quark propagator using M_5 = 5 m_q.
* The np elastic scattering data used for fitting covered the 4-12 GeV region.

## Context

This work extends the quark-exchange model of Frankfurt et al. (2000) for deuteron photo-disintegration, who succeeded in describing data at 90 degrees but failed at other angles without a phenomenological factor. The present study finds that the on-shell approximation used in that previous investigation is not accurate, as the principal-value part of the quark propagator gives sizable contributions. The results suggest that several improvements are needed: including momentum-dependent quark mass as suggested by QCD model calculations, improving the gluon-exchange interaction, using a deuteron wavefunction valid at high energies, and choosing the quark cluster size parameter from more fundamental considerations.
