# Holographic decays of large-spin mesons

> We study the decay process of large-spin mesons in the context of the gauge/string duality, using generic properties of confining backgrounds and systems with flavour branes. In the string picture, meson decay corresponds to the quantum-mechanical process in which a string rotating on the IR "wall" fluctuates, touches a flavour brane and splits into two smaller strings. This process automatically encodes flavour conservation as well as the Zweig rule. We show that the decay width computed in the string picture is in remarkable agreement with the decay width obtained using the phenomenological Lund model.

## Key claim

The decay width of large-spin mesons computed using the gauge/string duality is in remarkable agreement with the decay width obtained from the phenomenological Lund model, but the holographic description automatically encodes flavour conservation and the Zweig rule which must be added by hand in the Lund model.

## Method

The authors model a high-spin meson as a U-shaped open string in a confining supergravity background, with endpoints on flavour D6-branes and a horizontal section rotating along the infrared "wall". They compute the semi-classical decay rate by determining the probability that quantum fluctuations cause the horizontal part of the string to touch a flavour brane, using the string wave function constructed via semi-classical quantisation near the wall, combined with the flat-space string splitting probability from Dai and Polchinski (1989) and Mitchell and Turok (1987).

## Key numbers

- Effective string tension at the wall: \( T_{\text{eff}} = \frac{1}{2\pi\alpha'} \left(\frac{U_\Lambda}{R}\right)^{3/2} = \frac{2}{27\pi} M_\Lambda^2 (g^2_{\text{YM}} N) \)
- For light quarks in the relativistic limit (\(x \rightarrow 1\)): \( J \rightarrow \frac{1}{2\pi T_{\text{eff}}} E^2 \), recovering the standard Regge trajectory
- For heavy quarks in the non-relativistic limit (\(x \rightarrow 0\)): \( E \rightarrow 2 m_q + T_{\text{eff}} L \) and \( J \rightarrow 2\sqrt{T_{\text{eff}} m_q} L^{3/2} \)
- Mass scale of high-spin mesons: \( M_{\text{high}} \sim M_\Lambda \sqrt{g_{\text{YM}}^2 N} \)
- Low-spin meson masses: \( M_{\text{low}} \sim M_\Lambda \sim M_{KK} \)
- In the Casher-Neuberger-Nussinov model, the decay width ratio for rotating flux tubes: \( (\Gamma/M)_{\text{rot}} = (0.6-8.5)\times 10^{-2} \), using constituent quark masses of 75 MeV for light quarks and 400 MeV for heavy quarks with \( T_{\text{eff}} \approx 0.177~\text{GeV}^2 \)

## Context

This work extends the holographic description of mesons from the kinematical regime of spectra, studied by Kruczenski (2004) and Paredes (2004), into the dynamical regime of decay rates. It builds on the flavour brane approach of Karch and Katz (2002) and the low-spin meson decay calculations of Sakai and Sugimoto (2004), while providing a string-theoretic foundation for the phenomenological Lund fragmentation model of Sjostrand (1982) and Andersson (1983). The results enable future comparison with experimental data for very high-spin mesons, where deviations from the Lund model are predicted.
