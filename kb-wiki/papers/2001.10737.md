# Confronting Experimental Data with Heavy-Ion Models: Rivet for Heavy Ions

> The Rivet library is an important toolkit in particle physics, and serves as a repository for analysis data and code. It allows for comparisons between data and theoretical calculations of the final state of collision events. This paper outlines several recent additions and improvements to the framework to include support for analysis of heavy ion collision simulated data. The paper also presents examples of these recent developments and their applicability in implementing concrete physics analyses.

## Key claim

The Rivet toolkit has been extended to support heavy-ion collision analyses, enabling systematic comparisons between Monte Carlo event generators and experimental data from proton-nucleus (pA) and nucleus-nucleus (AA) collisions, though the framework requires careful calibration of centrality estimators because generator-level quantities often do not directly match experimental detector-level measurements.

## Method

The authors implemented new Rivet features including a centrality framework with four estimation modes (reference observable, generator-calibrated, impact parameter, and HepMC record), a generic framework for flow measurements using Q-vector based multi-particle cumulants, event mixing capabilities, reentrant finalization for merging multiple runs, and experimental primary-particle definitions (e.g., ALICE's requirement of proper lifetime tau >= 1 cm/c). These additions were validated by implementing concrete physics analyses comparing Monte Carlo generators such as Angantyr, EPOS-LHC, and Herwig+Pista to experimental data.

## Key numbers

- Rivet contains 949 analysis routines at the time of writing
- ALICE primary particle definition requires proper lifetime tau >= 1 cm/c
- ALICE V0 detector covers pseudorapidity ranges -3.7 < eta < -1.7 and 2.8 < eta < 5.1
- ATLAS forward calorimeters cover |eta| between 3.2 and 4.9
- STAR centrality uses charged tracks in pseudorapidity interval -0.5 < eta < 0.5
- Q-vector based correlator calculation achieves O(M log M) complexity compared to O(M^m) for naive m-particle correlators, crucial for heavy-ion events where M can reach 10,000 tracks

## Context

This work extends the Rivet framework (Buckley 2010) from proton-proton physics to heavy-ion collisions, addressing validation challenges similar to those faced earlier for hadron collisions. The development enables systematic model comparisons for generators like JETSCAPE (Putschke 2019) and supports the growing need for cross-system comparisons following the discovery of quark-gluon plasma-like behavior in small systems (Khachatryan 2010, ALICE 2017). The framework enables future studies of similarities between proton-proton, proton-nucleus, and nucleus-nucleus collisions as outlined in community planning documents (Citron 2018).
