# TMDlib2 and TMDplotter: a platform for 3D hadron structure studies

> A common library, TMDlib2, for Transverse-Momentum-Dependent distributions (TMDs) and unintegrated parton distributions (uPDFs) is described, which allows for easy access of commonly used TMDs and uPDFs, providing a three-dimensional (3D) picture of the partonic structure of hadrons. The tool TMDplotter allows for web-based plotting of distributions implemented in TMDlib2, together with collinear pdfs as available in LHAPDF.

## Key claim

TMDlib2 provides a common C++ library and web-based plotting tool (TMDplotter) for accessing transverse-momentum-dependent distributions (TMDs) and unintegrated parton distributions (uPDFs), now including uncertainty information for the first time in such a library tool, though the specific prescriptions for calculating uncertainties must be found in the original publications for each TMD set.

## Method

The library stores TMDs as momentum-weighted distributions x A_j(x, kt, mu) on multidimensional grids, with interpolation methods varying by TMD type (linear for PB and CCFM types, Lagrange polynomials for Pavia type, cubic spline for KS type). Grids are read into memory as separate C++ class instances for each TMD set or uncertainty replica, with metadata stored in YAML info files and grid files downloaded separately via the TMDlib-getdata tool. The library provides interfaces for both C++ and Fortran access, and TMDplotter enables web-based visualization of distributions alongside collinear PDFs from LHAPDF.

## Context

This work extends the original TMDlib version 1 (Hautmann 2014, Connor 2016) which first established a common standard for accessing TMD sets analogous to LHAPDF for collinear PDFs. The new version enables systematic TMD phenomenology at colliders and fixed-target machines by making TMD uncertainties accessible for the first time in a library tool, supporting both Hessian and Monte Carlo replica methods as used in prior work (Pumplin 2002, Giele 1998). The library enables comparisons between different TMD approaches including Parton Branching method (Martinez 2018), Pavia-type TMDs (Bacchetta 2017), and small-x approaches (Kutak 2012), and facilitates studies of the relationship between integrated TMDs and collinear PDFs as investigated by Catani (1993), Ebert (2020), and others.
