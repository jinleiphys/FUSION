# Nuclear Clustering in Fermionic Molecular Dynamics

> Clustering plays an important role in the structure of nuclei, especially for light nuclei in the $p$-shell. In nuclear cluster models these degrees of freedom are introduced explicitly. In the Resonating Group Method or in the Generator Coordinate Method the clusters are built from individual nucleons interacting via an effective nucleon-nucleon interaction; the total wave function is antisymmetrized. Fermionic Molecular Dynamics (FMD) goes beyond pure cluster models. It is a microscopic many-body approach using a Gaussian wave packet basis that includes the harmonic oscillator shell model and Brink-type cluster model wave functions as special cases. Clustering is not imposed but appears dynamically in the calculations. The importance of clustering for the understanding of bound states, resonances and scattering states is illustrated with examples discussing the charge radii of the Neon isotopes, the $^3$He($α$,$γ$)$^7$Be capture reaction and the cluster states in the $^{12}$C continuum.

## Key claim

Fermionic Molecular Dynamics (FMD) goes beyond pure cluster models by using a Gaussian wave packet basis that includes both harmonic oscillator shell model and Brink-type cluster model wave functions as special cases, so that clustering is not imposed but appears dynamically in the calculations. The authors illustrate that this approach can describe bound states, resonances, and scattering states, but the examples shown are limited to specific cases (Neon isotopes, the 3He(α,γ)7Be capture reaction, and cluster states in the 12C continuum).

## Method

The authors use Fermionic Molecular Dynamics (FMD), a microscopic many-body approach where single-particle states are described by Gaussian wave packets localized in phase space, with parameters for width and complex vector encoding mean position and momentum. Slater determinants of these single-particle states form the intrinsic many-body basis, and symmetries (parity, angular momentum, total linear momentum) are restored via projection operators. The method is applied to study clustering in light nuclei, with the effective nucleon-nucleon interaction not tuned to particular systems.

- 4He ground state binding energy: E(0+1) = -28.30 MeV
- 4He charge radius: about 1.67 fm
- 4He first excited state: E(0+2) = 20.21 MeV, width Γ = 0.5 MeV
- 4He second excited state: E(0-1) = 21.01 MeV, width Γ = 0.84 MeV
- 3He + n threshold relative to 4He: 20.58 MeV
- 8Be ground state is 91.8 keV above the 4He + 4He threshold, with width Γ = 5.6 eV
- 8Be next threshold (7Li + p) at 17.26 MeV
- Maximum probability for α-cluster relative motion in 8Be ground state: around r = 3.5 fm
- Probability for α-clusters drops practically to zero below r ≈ 2.5 fm

## Context

This work extends the Resonating Group Method (RGM) introduced by Wheeler (1937) and the Generator Coordinate Method (GCM) using Brink-type cluster states (Brink, 1965), as well as the THSR ansatz (Funaki, 2002). The FMD approach addresses limitations of traditional cluster models where cluster wave functions are restricted to simple harmonic oscillator functions and effective interactions are tuned for each nucleus. The method enables a consistent description of clustering across all nuclei, including those that are not α-nuclei, and can incorporate polarization and admixture effects that pure cluster models neglect.
