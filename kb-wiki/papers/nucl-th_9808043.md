# Bose-Einstein Final State Symmetrization for Event Generators of Heavy Ion Collisions

## Key claim

The paper presents two algorithms (a "quantum" and a "classical" interpretation) for calculating identical two-particle Bose-Einstein correlations from event generator output, but notes that the quantum algorithm introduces a free wavepacket width parameter sigma whose physical interpretation remains unclear, while the classical algorithm can produce unphysical correlation functions if the underlying phase-space distribution violates the Heisenberg uncertainty relation.

## Method

The authors start from event generator output consisting of sets of phase-space emission points (position, momentum, time) for identical particles. They derive two algorithms: a "quantum" interpretation that associates each emission point with a Gaussian wavepacket of width sigma, and a "classical" interpretation that treats the emission points as a discrete approximation to the Wigner phase-space density, using Gaussian bin functions of width epsilon to smooth the momentum distributions. Both algorithms are tested analytically and numerically against the Zajc model, a toy model with tunable position-momentum correlation parameter s, which allows controlled comparison of the numerical results to exact analytical expressions.

## Key numbers

- The Zajc model phase-space volume is given by V = 2^3 R_0^3 P_0^3 (1-s^2)^(3/2), which goes to zero as s approaches 1.
- The classical algorithm yields a critical correlation parameter s_crit = sqrt(1 - 1/(2R_0P_0)^2); for s > s_crit the HBT radius squared becomes negative, producing an unphysical rise of the correlation function with increasing relative momentum q^2.
- The bin width epsilon must satisfy epsilon << sqrt(2) P_0 to properly approximate the epsilon -> 0 limit in the classical algorithm.
- Numerical tests used 50 events of multiplicity 1000, showing good agreement between analytical and numerical results.
- The quantum algorithm produces an HBT radius that is always positive regardless of s, with the radius depending on sigma, R_0, P_0, and s via R_qm^2 = sigma^2/(1+2sigma^2 P_0^2) * (sigma^2 P_0^2 + R_0^2/sigma^2 + 2R_0^2 P_0^2 (1-s^2)).

## Context

This work extends earlier approaches to incorporating Bose-Einstein correlations into event generators, addressing the need to test simulated space-time structure rather than just momentum distributions. The authors build on the standard relation between two-particle correlations and the Wigner phase-space density (Heinz 1996), and on the Zajc model (Zajc 1993) which provides an analytically solvable toy model with position-momentum correlations. The algorithms enable future application to realistic event generators of relativistic heavy ion collisions, as reported in a subsequent work (Ellis, Geiger, Heinz, Wiedemann 1998), allowing systematic comparison of different dynamical scenarios against experimental HBT data.
