# Discrete non-orthogonal shell model for nuclear structure: Towards heavy elements

> We present recent developments of the Discrete Non-Orthogonal Shell Model (DNO-SM) for nuclear structure studies far from stability. Exact shell-model solutions are obtained for typical open-shell light sd and pf nuclei using non-orthogonal Slater determinants consistently derived from the variation after projection approach. The latter represents a powerful method to include correlations from particle-hole excitations. Applications to proton-rich nuclei at the N$\sim$Z line show the important role of these correlations to probe the structure transition in the $^{84,86}$Mo isotopes. We finally present a first complete description of low-lying spectroscopy in the superheavy 254 No, reproducing excellently various band structures and isomers in this challenging nucleus.

# Discrete Non-Orthogonal Shell Model for Nuclear Structure: Towards Heavy Elements

## Key claim
The Discrete Non-Orthogonal Shell Model (DNO-SM) using variation-after-projection (VAP) with non-orthogonal Slater determinants can recover exact shell-model solutions for open-shell nuclei using dramatically fewer basis states than conventional approaches, but this efficiency is demonstrated only for specific light nuclei (sd and pf shells) and one superheavy nucleus (²⁵⁴No), with the method's general applicability to arbitrary heavy systems remaining to be established.

## Method
The authors construct nuclear wave functions as superpositions of non-orthogonal Slater determinants, each derived from a variation-after-projection (VAP) minimization of the projected energy functional. Starting from an initial intrinsic state, they use Thouless parametrization to generate new non-orthogonal states, solve the Hill-Wheeler-Griffin equation for angular-momentum-projected states, and iteratively minimize the projected energy to obtain a discrete set of optimal Slater determinants. The method is benchmarked against exact shell-model diagonalization in the sd and pf valence spaces using the USDB and KB3 effective interactions, then applied to ⁸⁴,⁸⁶Mo and ²⁵⁴No.

## Key numbers
- For ²⁴Mg ground state binding energy: DNO-SM(VAP) gives −87.10405 MeV with 16 basis states, compared to exact shell-model value of −87.10445 MeV (difference of 0.4 keV); the PAV approach required 975 states to reach −87.10428 MeV.
- For ⁴⁸Cr ground state binding energy: DNO-SM(VAP) gives −32.950 MeV with 40 basis states, compared to exact shell-model value of −32.953 MeV (difference of 3 keV); the full pf diagonalization requires 1,963,461 basis states.
- B(E2; 2₁⁺ → 0₁⁺) for ⁸⁴Mo: DNO-SM gives 1512 e²fm⁴, experimental value is 1740⁺⁵⁸⁰₋₄₃₀ e²fm⁴.
- B(E2; 2₁⁺ → 0₁⁺) for ⁸⁶Mo: DNO-SM gives 893 e²fm⁴, shell-model diagonalization gives 731 e²fm⁴, experimental value is 707(71) e²fm⁴.
- For ⁸⁴Mo, calculations using only two-nucleon (2N) interaction produce B(E2) of 174 e²fm⁴, more than one order of magnitude below the experimental value.
- For ⁸⁶Mo, calculations using only 2N interaction produce B(E2) of 0.63 e²fm⁴.

## Context
This work extends the variational shell-model approach pioneered by Schmid (1984, 2004) and builds on the Broeckhove-Deumens theorem (Deumens, 1979) which guarantees that a discrete set of non-orthogonal Slater determinants can span the shell-model valence space. The DNO-SM(VAP) method provides a practical solution to the secular diagonalization problem that has limited conventional shell-model calculations in heavy nuclei, as reviewed by Caurier (2005). The application to ²⁵⁴No represents the first complete shell-model description of a superheavy nucleus, reproducing the yrast band, 3⁺ and 8⁻ isomers, and associated K-bands using a Kuo-Herling-based effective interaction without adaptation to this mass region. The results for ⁸⁴,⁸⁶Mo suggest a termination of the island of deformation around A∼80, with three-body forces governing the evolution of g9/2-d5/2 valence orbitals along the N∼Z line. This work enables future shell-model studies of superheavy nuclei that were previously considered computationally intractable.
