# Kaon Photoproduction and the $Λ$ Decay Parameter $α_-$

> The weak decay parameter $α_-$ of the $Λ$ is an important quantity for the extraction of polarization observables in various experiments. Moreover, in combination with $α_+$ from $\barΛ$ decay it provides a measure for matter-antimatter asymmetry. The weak decay parameter also affects the decay parameters of the $Ξ$ and $Ω$ baryons and, in general, any quantity in which the polarization of the $Λ$ is relevant. The recently reported value by the BESIII collaboration of $0.750(9)(4)$ is significantly larger than the previous PDG value of $0.642(13)$ that had been accepted and used for over 40 years. In this work we make an independent estimate of $α_-$, using an extensive set of polarization data measured in kaon photoproduction in the baryon resonance region and constraints set by spin algebra. The obtained value is 0.721(6)(5). The result is corroborated by multiple statistical tests as well as a modern phenomenological model, showing that our new value yields the best description of the data in question. Our analysis supports the new BESIII finding that $α_-$ is significantly larger than the previous PDG value. Any experimental quantity relying on the value of $α_-$ should therefore be re-considered.

## Key claim

The authors independently estimate the Lambda weak decay parameter alpha_- to be 0.721(6)(5), which is significantly larger than the previous PDG value of 0.642(13) but noticeably smaller than the BESIII value of 0.750(9)(4); the result depends on the choice of prior for beam polarization calibration parameters, with the quoted central value coming from the assumption of uniform priors within the quoted systematic uncertainties.

## Method

The authors use an extensive set of polarization data from kaon photoproduction measured by the CLAS collaboration, comprising seven observables (O_x, O_z, T, Sigma, P, C_x, C_z) at 314 kinematic points in the baryon resonance region. They exploit two Fierz identities (spin algebra constraints) that relate these observables to construct a likelihood function for the scaling parameters a, l, c, where a scales alpha_- relative to the old PDG value, and l and c scale the linear and circular photon beam polarizations. Markov Chain Monte Carlo sampling with different priors for l and c is used to obtain posterior distributions, and the Julich-Bonn dynamical coupled-channel model is refitted to data scaled by different alpha_- values as a cross-check.

## Key numbers

* The authors' extracted value of alpha_- is 0.721 with statistical uncertainty 0.006 and systematic uncertainty 0.005 (from the uniform prior analysis).
* The previous PDG value (2018) was 0.642(13), and the BESIII value is 0.750(9)(4).
* The Julich-Bonn refit using alpha_- = 0.721 yields the best chi-squared per data point of 1.59 for all 2254 data points, compared to 1.67 for alpha_- = 0.642 and 1.66 for alpha_- = 0.750.
* The analysis uses 314 kinematic points in the common region of the three CLAS measurements, with systematic uncertainties for beam polarizations taken as delta_l = 0.05 and delta_c = 0.02.

## Context

This work provides an independent estimate of the Lambda decay parameter alpha_- following the BESIII measurement (Ablikim 2018) that disagreed with the PDG value used for over 40 years. The result supports the BESIII finding that alpha_- is significantly larger than the old PDG value, though with a central value 0.721 rather than 0.750. The analysis extends a similar strategy previously applied to pion-induced reactions (Astbury 1975) and demonstrates that all experimental quantities relying on alpha_- (including polarization observables from CLAS, STAR, ATLAS, and PS185, as well as decay parameters for Xi and Omega baryons) should be re-evaluated. The Julich-Bonn model refits show that the new value improves the description of photoproduction data used in baryon resonance studies.
