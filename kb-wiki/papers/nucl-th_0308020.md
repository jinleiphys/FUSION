# KEWPIE: a dynamical cascade code for decaying exited compound nuclei

> A new dynamical cascade code for decaying hot nuclei is proposed and specially adapted to the synthesis of super-heavy nuclei. For such a case, the interesting channel is the tiny fraction that will decay through particles emission, thus the code avoids classical Monte-Carlo methods and proposes a new numerical scheme. The time dependence is explicitely taken into account in order to cope with the fact that fission decay rate might not be constant. The code allows to evaluate both statistical and dynamical observables. Results are successfully compared to experimental data.

# KEWPIE: a dynamical cascade code for decaying excited compound nuclei

## Key claim
The KEWPIE code provides a deterministic (non-Monte-Carlo) numerical solution for the multi-channel decay of excited compound nuclei that explicitly includes time-dependent fission decay widths, enabling evaluation of the tiny evaporation residue cross sections relevant to superheavy element synthesis. The authors claim successful comparison to experimental data for nobelium isotopes, but note that the only unconstrained free parameter is the reduced friction coefficient beta, which is set to 5 x 10^20 s^-1 based on agreement with data rather than from first principles.

## Method
The code solves a system of coupled differential equations for nuclear populations as functions of time, using an increasing time step scheme (delta t_{i+1} = alpha delta t_i with alpha typically 1.2) to cover processes with very different time scales. Evaporation widths are calculated using the Hauser-Feshbach approach, fission widths use the Bohr formula with Kramers and Strutinsky correction factors, and gamma emission widths include L=1 and L=2 transitions. The energy spectra of daughter nuclei are fully calculated and processed by discretizing the excitation energy integral into N bins, avoiding Monte-Carlo methods entirely.

## Key numbers
- Initial time step delta t_0 = 1 hbar/MeV, chosen to be longer than the mean collision time (approximately 0.1 hbar/MeV) and shorter than the mean evaporation time (few tens of hbar/MeV)
- Time step increase parameter alpha = 1.2, which permits coverage of a wide range of excitation energy
- Reduced friction coefficient beta set to 5 x 10^20 s^-1; setting beta to 5 x 10^21 s^-1 overestimates residue cross sections by roughly one order of magnitude
- Gamma emission integration ranges: L=1 from E^1_gamma_m = 1 MeV to E^1_gamma_M = 7 MeV; L=2 from E^2_gamma_m = 1 MeV to E^2_gamma_M = 4 MeV
- Gamma emission parameters: xi_1 = 4.24 x 10^-9 A^(2/3), xi_2 = 3.00 x 10^-12 A^(5/3)
- Level density parameter: a = A/14.61 (1 + 3.114 F_2/A^(1/3) + 5.626 F_3/A^(2/3)) from Toke and Swiatecki
- Shell correction damping parameter E_damp = 18.5 MeV
- Fission barrier formula: B_f0 = (0.7332 x 10^u Z^2)/A^(1/3) with u = 0.368 - 5.057x + 8.93x^2 - 8.71x^3, where x is fissility
- Fissility formula: x = Z^2 / (49.22A(1 - 0.3803I^2) - 20.489I^4) where I = N - Z
- hbar*omega_gs and hbar*omega_sd both set to 1 MeV

## Context
This work extends the Bateman equations (Bateman, original work) for radioactive decay chains to include time-dependent fission widths, addressing the known finite time required to build up quasi-stationary probability flow over the fission barrier (reviewed by Hil and Abe). The code is designed to replace Monte-Carlo methods (Gav, Cole) which are unsuitable for evaluating the tiny evaporation residue fractions in superheavy element synthesis. The code can accommodate fusion probabilities from the two-step model (Abe, Shen, Kos) and uses mass tables from Moller et al. and optionally Koura, with fission barrier parametrization from Schmidt et al. based on data compiled by Dahlinger et al. The code enables future calculations of evaporation residue cross sections for superheavy elements where fusion hindrance is significant.
