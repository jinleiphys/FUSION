# Density matrix renormalization group and wave function factorization for nuclei

> We employ the density matrix renormalization group (DMRG) and the wave function factorization method for the numerical solution of large scale nuclear structure problems. The DMRG exhibits an improved convergence for problems with realistic interactions due to the implementation of the finite algorithm. The wave function factorization of fpg-shell nuclei yields rapidly converging approximations that are at the present frontier for large-scale shell model calculations.

# Density matrix renormalization group and wave function factorization for nuclei

## Key claim
The density matrix renormalization group (DMRG) with the finite algorithm and improved orbital ordering yields fairly well-converged results for large-scale nuclear structure problems with realistic interactions, but for the computationally larger problem of ⁵⁶Ni the method fails to fully converge (missing the true ground state by almost 400 keV) despite converging exponentially for ²⁸Si. The wave function factorization produces accurate approximations for fpg-shell nuclei using only a few hundred correlated proton and neutron states, with errors of order 100 keV.

## Method
The authors implement the DMRG using a partition of neutron orbitals versus proton orbitals, with conjugate pairs of single-particle orbitals as the equivalent of lattice sites, ordered so that orbitals near the Fermi surface lie at the center of the chain. They abandon the infinite algorithm and instead start the finite algorithm from the spherical Hartree-Fock configuration and its 1p-1h excitations. For the wave function factorization, they expand the ground state as a sum over Ω products of correlated proton and neutron states, solve coupled eigenvalue equations iteratively, and compute excited states as a by-product of the ground state factorization.

## Key numbers
- For ²⁸Si using the USD interaction: the DMRG ground state energy converges exponentially as the number m of kept states increases; the full eigenvalue problem has dimension d = 93710.
- For ⁵⁶Ni using the KB3 interaction: the DMRG results appear converged but the true ground state energy is almost 400 keV lower; the exact diagonalization of this problem involves about one billion Slater determinants.
- For ⁷⁸Kr in the fpg-shell: the model space contains about 320,000 Slater determinants each for protons and neutrons, combining to about 1.4 × 10⁹ dimensional configuration space of definite parity; the largest eigenvalue problem within the factorization uses about 400 correlated states, with dimension 3.5 × 10⁶ and about 2.6 × 10⁹ nonzero matrix elements.
- Ground state shell-model energies from wave function factorization: ⁷⁶Ge at -57.23 MeV, ⁷⁶Se at -75.74 MeV, ⁷⁸Kr at -96.06 MeV.
- Excitation energies of the first 2⁺ state: ⁷⁶Ge at 0.70 MeV (experiment 0.563 MeV), ⁷⁶Se at 0.46 MeV (experiment 0.559 MeV), ⁷⁸Kr at 0.44 MeV (experiment 0.455 MeV); theoretical uncertainty about 150 keV.

## Context
This work extends earlier DMRG studies by Dukelsky and coworkers (2002) who obtained rapidly converging results for schematic pairing models but encountered slow convergence for sd-shell nuclei with realistic interactions. The wave function factorization builds on the authors' prior work (2003, 2004) showing that singular value decomposition yields exponentially decreasing singular values for ground states from realistic shell-model interactions. The fpg-shell calculations are at the frontier of large-scale shell-model work, comparable to exact diagonalizations for double beta-decay studies in ⁷⁶Ge and ⁷⁶Se by Caurier and coworkers (1996, 1999) and the determination of T=1 two-body matrix elements by Lisetskiy and coworkers (2004). The method enables accurate structure calculations for fpg-shell nuclei and tests of effective interactions at a fraction of the effort of exact diagonalizations.
