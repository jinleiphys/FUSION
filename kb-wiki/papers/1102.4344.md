# The Soft Function for Exclusive N-Jet Production at Hadron Colliders

> The N-jettiness event shape divides phase space into N+2 regions, each containing one jet or beam. Using a geometric measure these regions correspond to jets with circular boundaries. We give a factorization theorem for the cross section fully differential in the (transverse) mass of each jet, and compute the corresponding soft function at next-to-leading order (NLO). The ultraviolet divergences are analytically extracted by exploiting hemispheres for interactions between each pair of hard partons, leaving only convergent integrals that are sensitive to the precise boundaries. This method can be applied in general to N-jet soft functions, including other observables. For N-jettiness, the final result for the soft function involves stable one-dimensional numerical integrals, and all ingredients are now available to extend NLO cross sections to resummed predictions at next-to-next-to-leading logarithmic order.

## Method

The authors use soft-collinear effective theory (SCET) to factorize the N-jet cross section into hard, beam, jet, and soft functions, with the N-jettiness event shape \(\Tau_N\) dividing phase space into N+2 regions. They compute the one-loop soft function \(S_N\) by analytically extracting ultraviolet divergences via a hemisphere decomposition for each pair of hard partons, reducing the remaining non-hemisphere contributions to stable one-dimensional numerical integrals. The calculation is performed keeping the normalization factors \(Q_i\) arbitrary, allowing for different distance measures such as the geometric measure where jet regions have circular boundaries.

## Key numbers

* The cusp anomalous dimension at leading order: \(\Gamma_\text{cusp}(\alpha_s) = \frac{\alpha_s}{4\pi} \cdot 4 + \mathcal{O}(\alpha_s^2)\)
* The non-cusp anomalous dimension for quark jet functions at one loop: \(\gamma_J^q(\alpha_s) = \frac{\alpha_s}{4\pi} \cdot 6 C_F + \mathcal{O}(\alpha_s^2)\)
* The non-cusp anomalous dimension for gluon jet functions at one loop: \(\gamma_J^g(\alpha_s) = \frac{\alpha_s}{4\pi} \cdot 2\beta_0 + \mathcal{O}(\alpha_s^2)\)
* The non-cusp anomalous dimension for the hard Wilson coefficient for quarks at one loop: \(\gamma_C^q(\alpha_s) = -\frac{\alpha_s}{4\pi} \cdot 3 C_F + \mathcal{O}(\alpha_s^2)\)
* The non-cusp anomalous dimension for the hard Wilson coefficient for gluons at one loop: \(\gamma_C^g(\alpha_s) = -\frac{\alpha_s}{4\pi} \cdot \beta_0 + \mathcal{O}(\alpha_s^2)\)

## Context

This work extends the beam thrust formalism for 0-jet exclusive cross sections (Stewart 2009) to the general N-jet case using N-jettiness, which was introduced by the same authors in Stewart (2010). The factorization approach avoids complications present in jet algorithm calculations, such as non-global logarithms (Dasgupta 2001, Banfi 2002) and additional soft scales (Ellis 2009, Ellis 2010). The computed soft function provides the missing ingredient for next-to-next-to-leading logarithmic (NNLL) resummation of exclusive N-jet cross sections at hadron colliders, enabling more precise theoretical predictions for Higgs and new-physics searches at the LHC and Tevatron.
