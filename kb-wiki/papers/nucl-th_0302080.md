# Determination of the chiral coupling constants c3 and c4 in new pp and np partial-wave analyses

> As a first result of two new partial-wave analyses, one of the pp and another one of the np scattering data below 500 MeV, we report a study of the long-range chiral two-pion exchange interaction which contains the chiral coupling constants c1, c3, and c4. By using as input a theoretical value for c1 we are able to determine in pp as well as in np scattering accurate values for c3 and c4. The values determined from the pp data and independently from the np data are in very good agreement, indicating the correctness of the chiral two-pion exchange interaction. The weighted averages are c3 = -4.78(10) / GeV and c4 = 3.96(22) / GeV, where the errors are statistical. The value of c3 is best determined from the pp data and that of c4 from the np data.

## Key claim

The chiral coupling constants c3 and c4 can be determined accurately and independently from both pp and np scattering data below 500 MeV, and the values agree very well, but only when the value of c1 is fixed to a theoretical input of -0.76/GeV (derived from a low sigma term of 35(5) MeV) because c1 cannot be determined precisely from the NN data alone.

## Method

Energy-dependent partial-wave analyses (PWA) of 5109 pp data and 4786 np data below 500 MeV were performed by solving the Schrodinger equation with an energy-dependent boundary condition at r = 1.6 fm. The long-range interaction for r > b included the electromagnetic interaction, one-pion exchange (OPE), and the long-range part of the chiral two-pion exchange (chiTPE) potential, with the short-range interaction parametrized by analytic functions of energy. The chiral coupling constants c3 and c4 were fitted as free parameters in the chiTPE potential, while c1 was fixed to -0.76/GeV using the relation to the pion-nucleon sigma term.

## Key numbers

- Weighted averages from combined pp and np data: c3 = -4.78(10)/GeV and c4 = 3.96(22)/GeV, with statistical errors only.
- From pp data alone (5109 data, 33 boundary condition parameters, chi2_min = 5184.3): c3 = [-4.78(11) + 80(f^2 - 0.0755)]/GeV, c4 = [3.92(52) + 260(f^2 - 0.0755)]/GeV, correlation rho = -0.47.
- From np data alone (4786 data, 40 boundary condition parameters, chi2_min = 4806.2): c3 = [-4.77(22) + 100(f^2 - 0.0755)]/GeV, c4 = [3.97(24) + 40(f^2 - 0.0755)]/GeV, correlation rho = 0.22.
- The value of c3 from pp data is more than twice as accurate as from np data; the value of c4 from np data is twice as accurate as from pp data.
- Input value for c1 was fixed at -0.76/GeV, derived from sigma term = 35(5) MeV using the relation c1 = -[sigma/(4 m_pi^2) + (9/16)(f^2/m_s^2) m_pi].

## Context

This work extends the earlier Nijmegen pp PWA study of Rentmeester et al. (1999) from below 350 MeV to below 500 MeV, and for the first time performs a fully independent np PWA without taking I=1 phase shifts from the pp analysis, allowing tests of charge-independence breaking. The results are compared to determinations from pion-nucleon scattering analyses by Bernard et al. (1997) and Buttiker et al. (2000), which used chiral perturbation theory fits to piN amplitudes; the NN-derived values agree well with the piN range (c3 between -4.70 and -6.19/GeV, c4 between 3.25 and 4.12/GeV). The accurate c3 and c4 values are important input for calculations of the two-pion exchange three-nucleon force, and the agreement between pp and np results is presented as experimental evidence for the correctness of the chiral two-pion exchange interaction.
