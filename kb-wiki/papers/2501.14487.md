# Automated NLO calculations for asymmetric hadron-hadron collisions in MadGraph5_aMC@NLO

> We have extended {\tt MadGraph5\_aMC@NLO} capabilities by implementing computations for asymmetric hadron-hadron collisions, including proton-nucleus, pion-hadron or nucleus-nucleus collisions in order to obtain a tool for automated perturbative computations of cross sections (or ratios of cross sections) in asymmetric reactions at next-to-leading (NLO) order in $α_S$ in collinear factorisation. This tool, like the original symmetric version of \texttt{MadGraph5\_aMC@NLO}, automatically computes cross sections for different factorisation and renormalisation scales and PDFs provided by the {\tt LHAPDF} library, thereby allowing for an easy assessment of the associated theoretical uncertainties. In this paper, we present the validation of our code using $W$ and $Z$ boson production in $p$Pb collisions and Drell-Yan-pair production in $πW$ collisions. We also illustrate the capabilities of the framework by providing cross section and nuclear modification factors with their uncertainties for the production of $W$ and $Z$ bosons, charm and bottom quarks as well as associated production of $b\bar b+H^0$ in $p$Pb collisions at the LHC.

## Key claim
The authors extended MadGraph5_aMC@NLO to handle asymmetric hadron-hadron collisions (proton-nucleus, pion-hadron, nucleus-nucleus) at next-to-leading order in α_S within collinear factorisation, but note that for most asymmetric cases (except πp) QCD factorisation has not been formally proven and power-suppressed terms may be enhanced.

## Method
The implementation modifies the standard MadGraph5_aMC@NLO framework to accept two different LHAPDF sets for the two incoming hadrons via a new `asymm_choice` flag in the run card. The code reuses the existing NLO fixed-order infrastructure (Event and Counterevent decomposition with scale/PDF reweighting) and automatically computes both symmetric and asymmetric cross sections in a single run. Validation was performed by comparing W/Z boson production in pPb collisions and Drell-Yan pair production in πW collisions against published results.

## Key numbers
- Validation performed using W and Z boson production in pPb collisions at the LHC and Drell-Yan pair production in πW collisions
- The code supports multiple LHAIDs simultaneously; example configuration uses nCTEQ15 PDF sets with IDs 102000 (proton), 102400 (Ne), and 103100 (Pb)
- The HwU output format stores per bin: 2 bin edges, 1 cross section, 1 MC error, plus (3+N_μ) entries for scale variations and Σ(3+N_mem^i) entries for PDF uncertainties across N_LHAID sets
- The tool works only in NLO QCD fixed-order mode (no parton shower matching tested)

## Context
This work extends the symmetric MadGraph5_aMC@NLO framework (Alwall 2014) to asymmetric collisions, which previously required custom, often non-public codes. It enables NLO-accurate predictions for pA, AB, πp, and πA collisions, supporting nPDF fits (nCTEQ, EPPS, nNNPDF collaborations) and heavy-ion phenomenology at the LHC and RHIC. The tool is publicly available through the NLOAccess platform and is intended for Run 3 LHC data analysis and feasibility studies for future experiments.
