# Non-linear flow response and reaction plane correlations

> We apply the non-linear flow response formalism to the recently measured event plane correlations. We find that as a result of the combined effects of linear and non-linear flow response, the observed event plane correlations can be understood as an effective average of the 'linear limit' and 'non-linear limit'.

## Key claim

The observed event plane correlations can be understood as an effective average of the "linear limit" (where the correlation reduces to the initial participant plane correlation) and the "non-linear limit" (where the event planes would be perfectly correlated), with the actual value controlled by the relative magnitudes of the linear and quadratic response coefficients.

## Method

The authors applied a quadratic response formalism using cumulant expansions (rather than moment expansions) to characterize the initial geometry of heavy-ion collisions. They calculated the medium response to a given cumulant at linear and quadratic order by deforming a Gaussian distribution and computing response coefficients with ideal and viscous hydrodynamics in "single-shot" simulations. The event plane correlations were then expressed using the complex form of harmonic flow, where each harmonic receives contributions from both linear response (proportional to the cumulant) and non-linear quadratic response (proportional to products of lower-order eccentricities).

## Key numbers

- Freeze-out temperature used: T_fo = 150 MeV
- Shear viscosity to entropy density ratio used: eta/s = 1/(4 pi)
- The response formalism qualitatively reproduces observed two-plane and three-plane correlations, with one exception: the correlator <cos(2*Psi_2 - 6*Psi_3 + 4*Psi_4)>
- The non-linear flow response dominates correlation for non-central collisions and for larger values of eta/s

## Context

This work extends the non-linear flow response formalism of Teaney (2012) to explain event plane correlations recently measured by ATLAS (Jia, 2012). It provides an alternative to full event-by-event hydrodynamic simulations (Qiu, 2011; Gardim, 2011) by offering a simpler analytic framework that still captures the essential physics. The formalism enables systematic study of how shear viscosity affects event plane correlations and can be compared to event-by-event hydrodynamics in future work.
