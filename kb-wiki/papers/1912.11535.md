# In search of the best nuclear data file for proton induced reactions: varying both models and their parameters

> A lot of research work has been carried out in fine tuning model parameters to reproduce experimental data for neutron induced reactions. This however is not the case for proton induced reactions where large deviations still exist between model calculations and experiments for some cross sections. In this work, we present a method for searching both the model and model parameter space in order to identify the 'best' nuclear reaction models with their parameter sets that reproduces carefully selected experimental data. Three sets of experimental data from EXFOR are used in this work: (1) cross sections of the target nucleus (2) cross sections of the residual nuclei and (3) angular distributions. Selected models and their parameters were varied simultaneously to produce a large set of random nuclear data files. The goodness of fit between our adjustments and experimental data was achieved by computing a global reduced chi square which took into consideration the above listed experimental data. The method has been applied for the adjustment of proton induced reactions on Co-59 between 1 to 100 MeV. The adjusted files obtained are compared with available experimental data and evaluations from other nuclear data libraries.

## Key claim

The authors present a method for simultaneously varying both nuclear reaction models and their parameters to identify the "best" model combination for proton-induced reactions, but the final "best" file was based only on the first generation of parameter resampling due to computational resource constraints, and the method performed poorly for residual production cross sections compared to TENDL-2017.

## Method

The authors generated 200 random model combinations by varying selected nuclear reaction models within the TALYS code (version 1.9), producing 682 random nuclear data files for the parent generation. They computed a global reduced chi-square combining three experimental categories from EXFOR (reaction cross sections, residual production cross sections, and elastic angular distributions) and selected the model combination with the minimum chi-square as the "best" model set. This selected model combination was then used as the nominal file for re-sampling model parameters to produce a first generation of TALYS outputs, which were again compared with experimental data to select a new "best" file.

## Key numbers

- 200 random model combinations were generated by varying selected nuclear reaction models within TALYS
- 682 random nuclear data files were produced for the parent generation
- 169 experimental data points were used for reaction cross sections, 141 for residual production cross sections, and 185 for elastic angular distributions
- The 1st generation achieved global chi-square of 22.17, compared to 35.60 for the parent file
- The 1st generation achieved chi-square values of 21.65 for reaction cross sections, 22.43 for residual production cross sections, and 22.42 for angular distributions
- The parent file had chi-square values of 51.07 for reaction cross sections, 24.43 for residual production cross sections, and 22.42 for angular distributions
- The method was applied for proton-induced reactions on Co-59 from 1 to 100 MeV

## Context

This work extends the approach of Koning (2019) which demonstrated that simultaneous variation of models and their parameters induces prior correlations affecting nuclear data adjustments, but did not emphasize model selection and adjustment to fit differential experimental data. The authors note that the TENDL-2017 and JEFF-3.3 libraries contain evaluated nuclear data for 2804 isotopes for proton-induced reactions, but all evaluations were carried out with default TALYS models and parameters. The method enables future iterative improvement of evaluations for proton-induced reactions, which have been relatively neglected compared to neutron-induced reactions (49 isotopes in ENDF/B-VIII.0 proton sub-library versus 557 in the neutron sub-library).
