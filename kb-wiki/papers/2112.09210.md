# Concerning pion parton distributions

> Analyses of the pion valence-quark distribution function (DF), ${u}^π(x;ζ)$, which explicitly incorporate the behaviour of the pion wave function prescribed by quantum chromodynamics (QCD), predict ${u}^π(x\simeq 1;ζ) \sim (1-x)^{β(ζ)}$, $β(ζ\gtrsim m_p)>2$, where $m_p$ is the proton mass. Nevertheless, more than forty years after the first experiment to collect data suitable for extracting the $x\simeq 1$ behaviour of ${u}^π$, the empirical status remains uncertain because some methods used to fit existing data return a result for ${u}^π$ that violates this constraint. Such disagreement entails one of the following conclusions: the analysis concerned is incomplete; not all data being considered are a true expression of qualities intrinsic to the pion; or QCD, as it is currently understood, is not the theory of strong interactions. New, precise data are necessary before a final conclusion is possible. In developing these positions, we exploit a single proposition, viz. there is an effective charge which defines an evolution scheme for parton DFs that is all-orders exact. This proposition has numerous corollaries, which can be used to test the character of any DF, whether fitted or calculated.

## Key claim

The paper argues that the pion valence-quark distribution function must behave as (1-x)^β with β>2 at large x if QCD is correct, but existing data analyses remain ambiguous because some methods return β<2, which would imply either incomplete analysis, contaminated data, or that QCD is not the theory of strong interactions.

## Method

The authors exploit the proposition that a process-independent effective charge defines an all-orders exact evolution scheme for parton distribution functions, which they use to test the consistency of existing Drell-Yan data from the E615 experiment. They compare the large-x behavior predicted by QCD with results from different next-to-leading-logarithm threshold resummation methods (Mellin-Fourier-cosine, Mellin-Fourier-expansion, and double-Mellin) applied to the same E615 data set.

## Key numbers

- The QCD prediction for the pion valence-quark distribution at large x is (1-x)^β with β(ζ≳m_p)>2, where m_p is the proton mass
- The original E615 analysis (Conway 1989) found β=1.26(4) at ζ=5.2 GeV, contradicting the QCD constraint
- A subsequent NLO analysis with threshold resummation (Aicher 2010) found β=2.34(7) at ζ=4 GeV, consistent with QCD
- The double-Mellin method (Barry 2021) returned an effective exponent β_eff(ζ_c)~1.2 at ζ_c=1.27 GeV, with parameters α=-0.40±0.05, β=1.23±0.05, γ=0.62±0.22
- The hadron scale is defined as ζ_H=m_G=0.331(2) GeV, determined from QCD's process-independent effective charge
- A contact-interaction model yields a valence-quark distribution with ⟨x u^π(x;ζ_H)⟩=1/2, meaning each valence quark carries half the pion's light-front momentum

## Context

This work extends the long-standing debate about the large-x behavior of the pion valence-quark distribution, which began with the E615 experiment (Conway 1989) and was revisited by Aicher (2010) using threshold resummation. The paper directly addresses the conflicting results from different analysis methods in Barry (2021) and argues that new, precise data from upcoming experiments (JLab, COMPASS, EIC) are necessary to resolve the ambiguity. The all-orders evolution scheme based on the process-independent effective charge (Binosi 2016, Cui 2019) provides a framework to test whether any fitted or calculated distribution is consistent with QCD.
