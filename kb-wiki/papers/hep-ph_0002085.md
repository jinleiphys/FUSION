# How precisely can the difference method determine the pion-nucleon coupling constant?

> The Coulomb-like backward peak of the neutron-proton scattering differential cross section is due to one-pion exchange. Extrapolation to the pion pole of precise data should allow to obtain the value of the charged pion-nucleon coupling constant. This was classically attempted by the use of a smooth physical function, the Chew function, built from the cross section. To improve accuracy of such an extrapolation one has introduced a difference method. It consists of extrapolating the difference between the Chew function based on experimental data and that built from a model where the pion-nucleon coupling is exactly known. Here we cross-check to which precision can work this novel extrapolation method by applying it to differences between models and between data and models. With good reference models and for the 162 MeV neutron-proton Uppsala single energy precise data with a normalisation error of 2.3 %, the value of the charged pion-nucleon coupling constant is obtained with an accuracy close to 1.8 %.

## Key claim

The difference method can determine the charged pion-nucleon coupling constant from 162 MeV neutron-proton charge-exchange data with an accuracy close to 1.8%, provided a good reference model is used; the method's model dependence is small, contrary to earlier criticisms by the Nijmegen group.

## Method

The authors applied the Chew function extrapolation, the Ashmore method, and the difference method to pseudo-data generated from energy-dependent partial-wave analyses (PWA) of nucleon-nucleon scattering, where the coupling constant was exactly known, to calibrate systematic uncertainties. They then applied the difference method to the 162 MeV Uppsala neutron-proton charge-exchange data, using five different PWA reference models (A12.83g, A12.83, A13.75, A14.28, A14.28g) from Arndt (1999), fitting the difference in the Chew function between model and data with polynomial expansions over reduced (0 to 4 m_pi^2) and full (0 to 10.1 m_pi^2) momentum-transfer ranges.

## Key numbers

* For the model DA99 (unknown coupling, true value 14.28), the difference method over the full range gave sqrt(N) g^2_pi±/4pi = 14.14 ± 0.25 (statistical + extrapolation) ± 0.08 (systematic) ± 0.16 (normalisation) = 14.14 ± 0.31, an accuracy of 2.2%.
* For the same model over the reduced range, the result was 14.07 ± 0.32 (statistical + extrapolation) ± 0.11 (systematic) ± 0.16 (normalisation) = 14.07 ± 0.37, an accuracy of 2.6%.
* Using the Uppsala data with the A13.75 reference model in the reduced range (3-term fit, chi^2/N_df = 1.06), the extracted coupling was 14.50 ± 0.12.
* Using the Uppsala data with the A13.75 reference model in the full range (4-term fit, chi^2/N_df = 1.12), the extracted coupling was 14.38 ± 0.14.
* The Uppsala data have a normalisation error of 2.3%.

## Context

This work extends the earlier application of the difference method by Ericson et al. (1995) and Rahm et al. (1998) to the 162 MeV Uppsala data, which gave g^2_pi±/4pi = 14.52 ± 0.26. It directly addresses the criticism by the Nijmegen group (Stoks et al., 1993; SWA97) that the difference method has a very strong model dependence, demonstrating instead that with good reference models the systematic shift is below 1%. The results enable a more reliable determination of the charged pion-nucleon coupling constant from single-energy backward neutron-proton data, which can be compared with values from partial-wave analyses (e.g., Nijmegen 13.58 ± 0.05, VPI 13.73 ± 0.10) and the Goldberger-Miyazawa-Oehme sum-rule (Ericson et al., 1999, 14.17 ± 0.17).
