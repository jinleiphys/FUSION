# Harmonic-oscillator excitations of precise few-body wave functions

> A method for calculating the occupation probability of the number of harmonic oscillator (HO) quanta is developed for a precise few-body wave function obtained in a correlated Gaussian basis. The probability distributions of two- to four-nucleon wave functions obtained using different nucleon- nucleon (NN) interactions are analyzed to gain insight into the characteristic behavior of the various interactions. Tensor correlations as well as short-range correlations play a crucial role in enhancing the probability of high HO excitations. For the excited states of 4He, the interaction dependence is much less because high HO quanta are mainly responsible for describing the relative motion function between the 3N+N (3H+p and 3He+n) clusters.

# Harmonic-oscillator excitations of precise few-body wave functions

## Key claim

The occupation probability distribution of harmonic oscillator (HO) quanta in precise few-body wave functions reveals that tensor correlations dominate high-Q components below Q=20, while short-range repulsion produces a distinct bump and tail beyond Q=10; for excited states of 4He, the distribution is largely independent of the interaction model because high HO quanta primarily describe the relative motion between 3N+N clusters rather than details of the nuclear force.

## Method

The authors formulate a projection-operator method to compute the occupation probability P_Q of total HO quanta Q for wave functions expanded in a correlated Gaussian basis with global vectors. Ground and excited states of 2H, 3H, and 4He are obtained via the stochastic variational method using four different NN interactions: Minnesota (central only), Afnan-Tang S3 (central with short-range repulsion), G3RS+3NF, and AV8'+3NF (both with tensor, spin-orbit, and short-range components). The HO frequency is fixed by requiring that the lowest (0s)^N configuration reproduces the root-mean-square matter radius of the precise wave function.

## Key numbers

- For 4He ground state with AV8'+3NF: P_0 = 79.1%, D-state probability = 14.07%, M_Q = 2.59, sigma_Q = 7.31
- For 4He ground state with Minnesota potential: P_0 = 95.4%, D-state probability = 0.00%, M_Q = 0.263, sigma_Q = 1.48
- For 4He ground state with AV8'+3NF: probability still 1.6% at Q=10 and 0.7% at Q=20
- For 4He first excited 0^+0 state with AV8'+3NF: M_Q = 15.3, sigma_Q = 13.3
- For 4He 0^-0 state (21.01 MeV) with AV8'+3NF: P_1 = 37.9%, M_Q = 5.42, sigma_Q = 6.36
- For 4He 2^-0 state (21.84 MeV) with AV8'+3NF: P_1 = 20.5%, M_Q = 8.72, sigma_Q = 8.00
- HO frequencies determined for 4He: 23.2 MeV (MN), 23.4 MeV (ATS3), 22.2 MeV (G3RS+3NF), 21.6 MeV (AV8'+3NF)

## Context

This work extends the formulation of Suzuki (1996) for calculating HO quanta occupation probabilities from multicluster wave functions to the correlated Gaussian basis, enabling analysis of precise ab initio few-body wave functions. The results directly inform truncation strategies for no-core shell model calculations (Barrett 2013), importance-truncated NCSM (Roth 2007, Forssen 2013), and symmetry-adapted no-core shell model (Dytrych 2007). The finding that cluster states require extremely high HO quanta beyond practical computational limits echoes earlier observations for the Hoyle state of 12C (Suzuki 1996, Neff 2009) and the first excited state of 16O (Suzuki 1976, Horiuchi 2014), and supports the use of methods like the resonating group method to improve the asymptotic behavior of HO bases.
