# A combined mathematica--fortran program package for analytical calculation of the matrix elements of the microscopic cluster model

> We present a computer code that analytically evaluates the matrix elements of the microscopic nuclear Hamiltonian and unity operator between Slater determinants of displaced gaussian single particle orbits. Such matrix elements appear in the generator coordinate model and the resonating group model versions of the microscopic multicluster calculations.

## Key claim

The paper presents a combined Mathematica-Fortran program package that analytically evaluates matrix elements of the microscopic nuclear Hamiltonian and unity operator between Slater determinants of displaced Gaussian single-particle states, specifically for the generator coordinate model (GCM) and resonating group model versions of microscopic multicluster calculations. The method is restricted to systems of 0s clusters with a common oscillator width, though the author notes generalization to nonequal width parameters or higher harmonic oscillator orbits is straightforward.

## Method

The Mathematica component analytically evaluates the determinants, cofactors, and subdeterminants appearing in the overlap, one-body, and two-body matrix elements of Slater determinants of nonorthogonal single-particle states, producing symbolic expressions as sums of products of spatial overlap integrals. The Fortran component then collects identical terms (reducing the number of terms by about an order of magnitude) and substitutes the concrete spatial form of the single-particle matrix elements for displaced Gaussian functions with equal oscillator parameters, using a Gaussian two-body potential and the kinetic energy operator as the one-body example.

## Key numbers

- For the test run of 6He (alpha+n+n), the Mathematica output for the two-body kernel contains 560 terms before simplification.
- For the 8He (alpha+n+n+n+n) five-cluster system, the potential energy kernel contains about 50000 terms, of which only 3528 are different after collecting identical terms.
- Typical running time for the test run example is approximately 300 seconds.
- The program was developed for an IBM PC 386DX running DOS 3.2, using Mathematica v.1.2 and Fortran.

## Context

This work extends the microscopic nuclear cluster model calculations of Tang and Langanke (both cited without year) by providing an automated tool for evaluating the analytically complex matrix elements that arise in multicluster systems. The program makes the technique of the nuclear cluster model more accessible to nuclear physicists, and the output kernels are intended as building blocks for subsequent angular momentum projection following the formalism of Brink and Horiuchi (both cited without year). The approach enables systematic studies of light multicluster systems such as 6He and 8He.
