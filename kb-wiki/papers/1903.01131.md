# Multiple $SU(3)$ algebras in shell model and \\ interacting boson model

> Rotational $SU(3)$ algebraic symmetry continues to generate new results in the shell model (SM). Interestingly, it is possible to have multiple $SU(3)$ algebras for nucleons occupying an oscillator shell $η$. Several different aspects of the multiple $SU(3)$ algebras are investigated using shell model and also deformed shell model based on Hartree-Fock single particle states with nucleons in $sdg$ orbits giving four $SU(3)$ algebras. Results show that one of the $SU(3)$ algebra generates prolate shapes, one oblate shape and the other two also generate prolate shape but one of them gives quiet small quadrupole moments for low-lying levels. These are inferred by using the standard form for the electric quadrupole transition operator and using quadrupole moments and $B(E2)$ values in the ground $K=0^+$ band in three different examples. Multiple $SU(3)$ algebras extend to interacting boson model and using $sdg$IBM, the structure of the four $SU(3)$ algebras in this model are studied by coherent state analysis and asymptotic formulas for $E2$ matrix elements. The results from $sdg$IBM further support the conclusions from the $sdg$ shell model examples.

## Key claim

For nucleons occupying an oscillator shell, there exist multiple distinct $SU(3)$ algebras (specifically $2^{\lfloor \eta/2 \rfloor}$ of them) that all generate the same rotational energy spectrum but produce different wavefunctions, leading to qualitatively different quadrupole moments and $B(E2)$ values; the paper demonstrates this for the $\eta=4$ ($sdg$) shell where four $SU(3)$ algebras exist, with one generating prolate shapes, one oblate shapes, and two others also prolate but with one giving very small quadrupole moments for low-lying levels.

## Method

The authors used the shell model (SM) with the Antoine code and the deformed shell model (DSM) based on Hartree-Fock single-particle states, both in the full $sdg$ ($\eta=4$) oscillator space, to compute spectra, quadrupole moments, and $B(E2)$ values for the four Hamiltonians $H_Q^{\alpha} = -Q^2(\alpha) \cdot Q^2(\alpha)$ corresponding to the four phase choices $\alpha = (\alpha_{sd}, \alpha_{dg}) = (\pm 1, \pm 1)$. They also performed a coherent-state analysis and derived asymptotic formulas for $E2$ matrix elements in the $sdg$ interacting boson model (IBM) to further support the shell-model conclusions.

## Key numbers

* For the $(sdg)^{6_p}$ system (6 protons), the ground-state energy from all four $H_Q$ Hamiltonians is $E_{gs}=378$ (dimensionless), matching the $SU(3)$ analytical result for the $(18,0)$ irrep.
* For $H_Q^{(-,-)}$ in the $(sdg)^{6_p}$ system, the quadrupole moment of the $2^+_1$ state is $-49.18\; e\,fm^2$ and the $B(E2; 2^+_1 \rightarrow 0^+_1)$ is $585.97\; e^2fm^4$; for $H_Q^{(+,+)}$ the corresponding values are $+13.44\; e\,fm^2$ and $42.20\; e^2fm^4$, indicating prolate versus oblate shapes.
* For the $(sdg)^{6_p,6_n}$ system (6 protons + 6 neutrons), the intrinsic quadrupole moments (in units of $b^2$) from DSM are $+71.95$ for $H_Q^{(-,-)}$, $+47.43$ for $H_Q^{(+,-)}$, $+5.06$ for $H_Q^{(-,+)}$, and $-19.45$ for $H_Q^{(+,+)}$.
* The correlation coefficient $\zeta$ between $Q\cdot Q$ operators for $(\alpha_{sd},\alpha_{dg}) = (-,-)$ and $(+,-)$ ranges from $0.35$ to $0.39$ across $m=4$ to $56$ nucleons, while for $(-,-)$ and $(+,+)$ it is $0.00$ to $0.07$.

## Context

This work extends the early recognition by Parikh (year not given) of two $SU(3)$ algebras in the $sd$ shell and the known two $SU(3)$ algebras in $sd$IBM (Iachello, 1987) to the full multiplicity $2^{\lfloor \eta/2 \rfloor}$ for any oscillator shell, with the first detailed investigation of the four algebras in the $sdg$ shell. It parallels a recent study of multiple pairing algebras in shell model and IBM by Kota (year not given) and enables future work using $SU(3)$ Wigner-Racah algebra to compute matrix elements of the standard $E2$ operator in bases built from non-standard $SU(3)$ algebras, as well as applications to shape coexistence and phase transition studies in heavier nuclei.
