# Exploring percolation phase transition in the three-dimensional Ising model with machine learning

> The percolation study offers valuable insights into the characteristics of phase transition, shedding light on the underlying mechanisms that govern the formation of global connectivity within the system. We explore the percolation phase transition in the 3D cubic Ising model by employing two machine learning techniques. Our results demonstrate the capability of machine learning methods in distinguishing different phases during the percolation transition. Through the finite-size scaling analysis on the output of the neural networks, the percolation temperature and a correlation length exponent in the geometrical percolation transition are extracted and compared to those in the thermal magnetization phase transition within the 3D Ising model. These findings provide a valuable way essential for enhancing our understanding of the property of the QCD critical point, which belongs to the same universality class as the 3D Ising model.

## Key claim

The paper demonstrates that unsupervised (PCA) and semi-supervised (DANN) machine learning methods can identify the percolation phase transition of geometrical spin clusters in the 3D Ising model and extract the percolation temperature and correlation length exponent, but the extracted percolation temperature is slightly lower than the thermal magnetization critical temperature, and the correlation length exponent for geometrical clusters is larger than that of the thermal transition.

## Method

The authors generated 2000 independent spin configurations of the 3D cubic Ising model using the Wolff Monte-Carlo algorithm for lattice sizes L = 12, 16, 20, 24, 28 over a temperature range from T = 3.52 to T = 5.52. They used only the spins belonging to the largest cluster as input data for two machine learning architectures: a Principal Component Analysis (PCA) network with one principal component, and a Domain Adversarial Neural Network (DANN) with a convolutional feature extractor, label predictor, and domain classifier. Finite-size scaling analysis was applied to the neural network outputs to extract the percolation temperature and correlation length exponent.

## Key numbers

- Percolation temperature from PCA: T_p = 4.475 ± 0.004
- Percolation temperature from DANN: T_p = 4.466 ± 0.005
- Correlation length exponent from DANN data collapse: ν = 0.72 ± 0.03
- Thermal magnetization critical temperature for comparison: T_c = 4.5115 ± 0.0001 (from references NewTc1, NewTc2)
- Pearson correlation coefficient between average largest cluster size and first principal component: 1.0
- System sizes used: L = 12, 16, 20, 24, 28

## Context

This work extends previous studies of percolation transitions in the Ising model, specifically the finding by Perco3D that the geometrical spin cluster percolation temperature in 3D differs from the thermal magnetization critical temperature, unlike the 2D case where they coincide (Perco2D). The results are qualitatively consistent with the earlier cluster size heterogeneity result of ν ≈ 0.75 for geometric clusters (3DIsingnu), and the extracted percolation temperatures are slightly lower than the well-established thermal critical temperature. The machine learning approach enables future application to experimental heavy-ion collision data to investigate the QCD critical point, which belongs to the same Z(2) universality class as the 3D Ising model.
