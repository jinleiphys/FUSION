# On invariants for particle propagation in non-Abelian fields

> Characterising the propagation of particles in an external non-Abelian field only in terms of invariants constructed from its field tensor is not always sufficient, especially, in many analytically tractable and phenomenologically interesting cases.

## Key claim
Characterising particle propagation in external non-Abelian fields using only invariants constructed from the field tensor is insufficient when the field tensor satisfies det 𝔽 = 0, because gauge-inequivalent gauge field configurations can produce the same field tensor and lead to different physical observables.

## Method
The paper constructs explicit gauge field configurations in SU(2) (or an SU(2) subgroup) that yield the same constant or time-dependent non-Abelian field tensor but are not gauge equivalent, as demonstrated by computing Wilson loops, Yang–Mills currents, and the Klein–Gordon and Dirac propagators in mixed representation. The propagators are solved analytically using Whittaker functions for the covariantly constant configuration and exponential ansätze for the genuinely non-Abelian configuration, with the gauge group structure handled via projection operators and eigenvalue decompositions.

## Key numbers
- For the constant electric field configuration with rescaling parameter c, the Yang–Mills current invariant is J^a_μ J^{aμ} = (c^{-2} - c^2) |E_3|^3, with the c=1 case distinguished from J^a_μ=0 by J^a_μ J^{bμ} J^a_ν J^{bν} = (c^{-4} + c^4) |E_3|^6.
- For the Klein–Gordon propagator in the covariantly constant configuration, the homogeneous solutions are Whittaker functions M_{−i m_⊥^2/(4e_n), −(−1)^l/4}(i e_n t^2) with l∈{1,2}, and the denominator (Wronskian) evaluates to i.
- For the Klein–Gordon propagator in the genuinely non-Abelian static configuration, the exponential ansatz yields four eigenvalues λ^2_± = −[ω^2 + (a_0)^2 + (a_3)^2] ± 2√[(a_0)^2(a_3)^2 + (a_0)^2ω^2 + (p_3)^2(a_3)^2], giving exponentially growing/decaying modes when [ω^2 − (a_0)^2 + (a_3)^2]^2 < 4(p_3)^2(a_3)^2.
- For the Dirac propagator in the genuinely non-Abelian static configuration, the exponential ansatz yields μ^2_± = −[ω^2 + (a_0)^2 + (a_3)^2] ± √{4[(a_0)^2(a_3)^2 + (a_0)^2ω^2 + (p_3)^2(a_3)^2] − (E_3)^2}, leading to purely oscillatory behaviour.

## Context
This work extends the Wu–Yang ambiguity (Wu and Yang, 1975) by showing that the covariant derivative, not the field tensor, is the elementary building block for gauge invariants, and that for det 𝔽 = 0 the Yang–Mills current J^a_μ provides a distinguishing observable. The results contradict the assumption that field-tensor invariants alone suffice for effective actions in non-Abelian backgrounds, and enable distinguishing between flux-tube realisations in ultrarelativistic collisions (Casher, 1978) and between colour glass condensate field configurations (McLerran and Venugopalan, 1993).
