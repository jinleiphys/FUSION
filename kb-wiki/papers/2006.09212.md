# Impact of single string structure and multiple string interaction on strangeness production in Pb+Pb collisions at $\sqrt{S_{NN}} = 2.76$ TeV

> We present a systematic study on the strange and multi-strange particles production in Pb+Pb collisions at $\sqrt{S_{NN}} = 2.76$ TeV based on the PACIAE model simulations. Two different mechanisms, namely the single string structure variations and multiple string interactions, are implemented in the simulations. These modifications give rise to an enhancement of the string tension value involved in the Lund string fragmentation framework and generate more strange particles in the hadronic final state. By comparing the simulation results with the ALICE experimental data, it turns out that the inclusion of the variable effective string tension in the PACIAE model is capable to reach an improved agreement between theory and experiment on the strangeness production in Pb+Pb collisions.

## Key claim

The inclusion of a variable effective string tension, derived from both single-string structure variations and multiple-string interactions, in the PACIAE model improves agreement with ALICE data for strangeness production in Pb+Pb collisions at 2.76 TeV, but the model requires an additional 1.5-times enlargement of the default strange-quark suppression parameters (PARJ(2) and PARJ(3)) to fully describe the most extreme multi-strange baryon (Omega) yields.

## Method

The authors extended the PACIAE event generator, which uses PYTHIA for parton initial states and a Glauber model for nuclear geometry, by implementing two mechanisms: (1) a string-by-string effective string tension based on the hardest gluon's transverse momentum within each string (single-string structure), and (2) an event-wise effective string tension that scales with the number of multiple parton interactions (MPI) weighted by the ratio of binary collisions to participant nucleons (Ncoll/Npart) for nuclear collisions (multiple-string interaction). These effective tensions modify the Lund string fragmentation parameters (diquark suppression, strange-quark suppression, and transverse-momentum width) via a scaling relation derived from the tunneling probability. The model was tuned to reproduce mid-rapidity charged pion yields from ALICE, then used to compute yields of K_S^0, Lambda, Xi, and Omega as a function of mean number of participants.

## Key numbers

* Mid-rapidity pion yields for 0-5% centrality: PACIAE gives pi+ = 722.5 and pi- = 722.2, consistent with ALICE data pi+ = 733 ± 54 and pi- = 732 ± 52.
* The effective string tension from multiple-string interaction uses exponent r = 0.2 and p_T_ref^2 / p_0^2 = 1, same as in pp collisions.
* Default Lund fragmentation parameters in vacuum: PARJ(1) = 0.1, PARJ(2) = 0.3, PARJ(3) = 0.4, PARJ(21) = 0.85.
* For the Omega baryon, the combined single+multiple string tension scenario (kappa_eff^{s+m}) required PARJ(2) and PARJ(3) to be enlarged by a factor of 1.5 to describe ALICE data.
* The impact on pion or proton multiplicity can be as large as 30% when varying from the default string tension (kappa_0) to the combined single+multiple scenario (kappa_eff^{s+m}).

## Context

This work extends the authors' previous study of variable effective string tension in pp collisions at 7 TeV (Zhou et al., 2020) to the Pb+Pb system at 2.76 TeV. It builds on the color ropes mechanism used in the DIPSY model (Bierlich, 2014) and the thermodynamic string features implemented in PYTHIA8 (Fischer, 2016), but provides a microscopic parametrization based on gluon structure within strings and string overlap density. The results suggest that the strangeness enhancement traditionally attributed to quark-gluon plasma formation can be qualitatively reproduced within a string fragmentation framework, enabling further tests of whether such mechanisms contribute to the observed effects in heavy-ion collisions.
