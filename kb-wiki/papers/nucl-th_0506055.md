# Single particle calculations for a Woods-Saxon potential with triaxial deformations, and large Cartesian oscillator basis

> We present a computer program which solves the Schrodinger equation of the stationary states for an average nuclear potential of Woods-Saxon type. In this work, we take specifically into account triaxial (i.e. ellipsoidal) nuclear surfaces. The deformation is specified by the usual Bohr parameters. The calculations are carried out in two stages. In the first, one calculates the representative matrix of the Hamiltonian in the cartesian oscillator basis. In the second stage one diagonalizes this matrix with the help of subroutines of the EISPACK library. If it is wished, one can calculate all eigenvalues, or only the part of the eigenvalues that are contained in a fixed interval defined in advance. In this latter case the eigenvectors are given conjointly. The program is very rapid, and the run-time is mainly used for the diagonalization. Thus, it is possible to use a significant number of the basis states in order to insure a best convergence of the results.

## Key claim

The paper presents a computer program that solves the Schrödinger equation for a Woods-Saxon potential with triaxial (ellipsoidal) deformations, using a Cartesian harmonic oscillator basis expansion and diagonalization via the EISPACK library; the abstract's claim of a "very rapid" program is qualified by the fact that run-time is dominated by diagonalization and that for the largest basis sizes (Nmax ~23-26), diagonalization takes the major part of the running time, though the global run-time remains "reasonable."

## Method

The Hamiltonian includes a central Woods-Saxon potential, a spin-orbit interaction, and a Coulomb potential for protons, with triaxial deformation specified by Bohr parameters (β, γ) applied to three distinct but homothetic ellipsoidal surfaces for the central, spin-orbit, and Coulomb fields. The representative matrix of the Hamiltonian is constructed in a Cartesian anisotropic harmonic oscillator basis truncated by either a spherical criterion (n_x + n_y + n_z ≤ Nmax) or a deformed criterion based on oscillator frequencies, with matrix elements computed via Gauss-Hermite quadrature (30×30×30 points) and the Coulomb potential evaluated via Gauss-Legendre quadrature (48×48 points). The matrix is then diagonalized using subroutines from the EISPACK library, exploiting parity and Kramers degeneracy (signature quantum number) to reduce the problem to four blocks (two parities for neutrons and protons, each with signature +1/2).

## Key numbers

- Basis truncation limit: Nmax ≤ 26 (maximum number of major shells)
- For Nmax = 23, neutrons case, both parities, all bound state eigenenergies and eigenfunctions: running time ~80 seconds on an AMD Athlon 1 GHz computer
- Matrix element calculation for Nmax = 23 takes ~20 seconds of the total run time
- Quadrature mesh: 30×30×30 points for Gauss-Hermite integration of matrix elements
- Coulomb potential quadrature: 48×48 points for Gauss-Legendre integration
- Deformation range tested: 0 ≤ β ≤ 0.6
- Proton mass-energy: m_p c² = 938.2592 MeV
- Neutron mass-energy: m_n c² = 939.553 MeV
- ħc = 197.32879 MeV·fm
- t_p = 0.0240958315 MeV⁻¹fm⁻² (proton conversion factor)
- t_n = 0.0241290571 MeV⁻¹fm⁻² (neutron conversion factor)
- ħω₀ ≈ 41·A⁻¹/³ MeV (Nilsson model estimate)
- ħω₀ ≈ 0.979·|V₀|·A⁻¹/³ MeV (refined estimate from square well comparison)

## Context

This work extends the approach of Cwiok et al. (1987) and other authors who solved the Woods-Saxon problem with deformed potentials, by providing a documented Fortran program (named "Triaxial") specifically for triaxial (ellipsoidal) shapes using a Cartesian oscillator basis with explicit treatment of parity and Kramers degeneracy. The program enables systematic single-particle energy calculations for triaxially deformed nuclei, with the option to use either user-defined parameters or the Myers droplet model parameters, and can serve as a foundation for more complete nuclear structure calculations such as the Strutinsky method or quasiparticle calculations.
