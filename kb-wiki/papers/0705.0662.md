# Manifestly covariant current matrix elements in the Point Form Relativistic Hamiltonian Dynamics

> A manifestly covariant expression for the current matrix elements of three quark bound systems is derived in the framework of the Point Form Relativistic Hamiltonian Dynamics. The relativistic impulse approximation is assumed in the model. A critical comparison is made with other expressions usually given in the literature.

## Key claim

The paper derives a manifestly covariant expression for current matrix elements of three-quark bound systems within Point Form Relativistic Hamiltonian Dynamics, under the assumption of the relativistic impulse approximation; the key hedge is that the expression is manifestly covariant only when using Dirac boosts and covariant integrations over spectator quark momenta, in contrast to the standard formalism which employs Wigner rotation matrices and velocity states that are not manifestly covariant.

## Method

The author constructs the Poincaré algebra generators for a three-quark system using the Bakamjian-Thomas construction, introducing the four-velocity operator \(V^\mu\) and the free mass operator \(M_f\) as functions of the quark momenta. The wave functions are defined in the rest frame using Jacobi momenta and then boosted to a generic frame using Dirac boost operators, yielding a Dirac wave function for the bound system. The current matrix elements are then computed under the relativistic impulse approximation, integrating over the spectator quark momenta while keeping the interacting quark on-shell, resulting in the manifestly covariant expression given in eq.(5.3c).

## Key numbers

- The free mass operator \(M_f\) is defined as \(M_f = [\sum_{ij} p_i^\mu p_j^\nu g_{\mu\nu}]^{1/2}\) (eq. 2.7), a Lorentz invariant quantity.
- The total mass operator is \(M = M_f + W\) (eq. 2.16), where \(W\) is a Lorentz invariant interaction operator.
- The four-momentum operator is \(P^\mu = M \cdot V^\mu\) (eq. 2.19), with \(V^\mu V_\mu = 1\) (eq. 2.8b).
- The charge normalization factor for the electromagnetic current is \(\mathcal{N}_G = [m / \epsilon(\mathbf{p}_{1G}^*)]^{1/2}\) (eq. 5.9), which automatically satisfies total charge normalization for proton (+1) and neutron (0) at zero momentum transfer.
- The quark charge operator for electromagnetic interactions is \(e_1^{em} = \frac{1}{2}\tau_1^3 + \frac{1}{6}\) (eq. 5.5), and for axial-vector interactions is \(e_1^{ax} = \tau_1^+\) (eq. 5.11).

## Context

This work extends the Point Form Relativistic Hamiltonian Dynamics formalism developed in refs. [1-3] and implemented in ref. [6], providing an alternative manifestly covariant formulation that avoids the use of Wigner rotation matrices and velocity states. The approach enables the calculation of nucleon electromagnetic and axial form factors using constituent quark models, with previous numerical calculations showing good agreement with experimental data (refs. [2,7] for electromagnetic, refs. [3,4] for axial). The method also allows for future developments including the definition of a dynamically conserved electromagnetic current via minimal coupling and the introduction of virtual negative energy states in the current matrix elements.
