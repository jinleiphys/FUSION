# One fluid to rule them all: viscous hydrodynamic description of event-by-event central p+p, p+Pb and Pb+Pb collisions at $\sqrt{s}=5.02$ TeV

> The matter created in central p+p, p+Pb and Pb+Pb collisions at sqrt{s}=5.02 TeV is simulated event-by-event using the superSONIC model, which combines pre-equilibrium flow, viscous hydrodynamic evolution and late-stage hadronic rescatterings. Employing a generalization of the Monte Carlo Glauber model where each nucleon possesses three constituent quarks, superSONIC describes the experimentally measured elliptic and triangular flow at central rapidity in all systems using a single choice for the fluid parameters, such as shear and bulk viscosities. This suggests a common hydrodynamic origin of the experimentally observed flow patterns in all high energy nuclear collisions, including p+p.

# One fluid to rule them all: viscous hydrodynamic description of event-by-event central p+p, p+Pb and Pb+Pb collisions at √s=5.02 TeV

## Key claim

Using a single set of fluid parameters (shear viscosity η/s=0.08 and bulk viscosity ζ/s=0.01) with the superSONIC model, the experimentally measured elliptic and triangular flow at central rapidity in p+p, p+Pb, and Pb+Pb collisions at √s=5.02 TeV can be simultaneously described, suggesting a common hydrodynamic origin for flow patterns in all these systems. The authors note that no fine-tuning of superSONIC parameters was attempted, so no precision fit of experimental data can be expected, and that more work will be needed to corroborate this conclusion.

## Method

The superSONIC hybrid model combines pre-equilibrium flow from gauge/gravity simulations, 2+1 dimensional viscous hydrodynamic evolution starting at τ₀=0.25 fm using a lattice QCD equation of state, and late-stage hadronic rescatterings with the B3D cascade code. Initial conditions are generated using a Monte Carlo Glauber framework modified with the OSU (Ohio State) constituent quark model, where each nucleon contains three valence quarks with Gaussian gluon clouds, and entropy deposition occurs through wounded nucleons. For each system, 100 initial conditions per centrality class were simulated on 100×100 grids with lattice spacings adapted to the collision system size.

## Key numbers

- Shear viscosity to entropy density ratio: η/s = 0.08 (fixed for all systems)
- Bulk viscosity to entropy density ratio: ζ/s = 0.01 (fixed for all systems)
- Hydrodynamic starting time: τ₀ = 0.25 fm
- Nucleon size parameter: B = (0.52 fm)²
- Gluon cloud width: σ_g = 0.46 fm
- Inelastic nucleon-nucleon cross section: σ_NN^{inel} = 70 mb
- Woods-Saxon radius for lead nucleus: 6.62 fm with skin depth 0.546 fm
- Hadronic cascade freeze-out temperature: T = 0.17 GeV
- B3D hadronic cascade constant cross section: 10 mb
- For p+Pb and Pb+Pb collisions at √s=5.02 TeV, model results for dN/dy for 0-5% highest multiplicity events are within five percent of experimental values at midrapidity
- Integrated elliptic flow v₂ saturates at approximately 6 percent for high multiplicities in p+p collisions

## Context

This work extends the Panta Rhei ("Everything Flows") approach to relativistic ion collisions, following previous hydrodynamic descriptions of small systems by Romatschke (2015) and others, and directly addresses the criticism that hydrodynamic descriptions of individual systems could be coincidental. The superSONIC model was previously used to successfully predict experimental flow results in p+Au, d+Au, and ³He+Au collisions at √s=0.2 TeV. The key advance is demonstrating that a single set of fluid parameters works across system sizes from p+p to Pb+Pb, which enables future work to constrain the minimum size for a QCD fluid droplet and to investigate whether bulk viscous corrections to particle spectra can achieve quantitative matching of pion mean transverse momenta in p+p collisions.
