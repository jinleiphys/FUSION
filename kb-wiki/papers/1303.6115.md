# Uncertainty study of nuclear model parameters for the n+ ^{56}Fe reactions in the fast neutron region below 20 MeV

> In this work, we study the uncertainty of nuclear model parameters for neutron induced ^{56}Fe reactions in fast neutron region by using the Total Monte Carlo method. We perform a large number of TALYS runs and compare the calculated results with the experimental data of the cross sections to obtain the uncertainties of the model parameters. Based on the derived uncertainties another 1000 TALYS runs have been performed to create random cross section files. For comparison with the experimental data we calculate a weighted χ^2 value for each random file as well as the ENDF/B-VII.1, JEFF3.1, JENDL4.0 and CENDL3.1 data libraries. Furthermore, we investigate the optical model parameters correlation obtained by way of this procedure.

## Key claim
The study demonstrates that the Total Monte Carlo method, using a binary reject/accept scheme in the TASMAN code, can produce random nuclear data files for n+⁵⁶Fe reactions that perform as well as or better than existing evaluated libraries (ENDF/B-VII.1, JEFF3.1, JENDL4.0, CENDL3.1) when judged by a weighted χ² metric, but the resulting χ² distribution is asymmetric with a tail toward high values.

## Method
The authors used the TALYS nuclear model code with the TASMAN uncertainty package, starting from default parameters and manually searching for a good parameter set. They then made rough assumptions on parameter uncertainties, sampled parameter space in many TALYS runs, and accepted or rejected each run based on agreement with experimental data from EXFOR within typical experimental uncertainties. Using the derived parameter uncertainties, they performed 1000 TALYS runs to create random cross section files and calculated a weighted χ² value for each file compared to experimental data across multiple reaction channels.

## Key numbers
- Parameter uncertainties for ⁵⁶Fe range from 2% (e.g., rⁿ_V, vⁿ_1, rᵖ_V) to 35% (wⁿ_so2) of the absolute value
- The (n,tot) channel has 34 energy points from 8-20 MeV with weight 49 in the χ² calculation
- The (n,p) channel has 300 energy points from 5.46-20 MeV with weight 4.5
- The (n,γ) channel has only 1 energy point at 14.2 MeV with weight 0.04
- Level density parameter a(⁵⁶Fe) has uncertainty 6.5%, a(⁵⁷Fe) has 4.5%, a(⁵⁵Fe) has 5%
- The optical model parameter vⁿ_3 has uncertainty 9%, dⁿ_1 has 9.4%, dⁿ_2 has 10%, dⁿ_3 has 9.7%
- The spin-orbit parameter wⁿ_so1 has uncertainty 30%, wⁿ_so2 has 35%

## Context
This work extends the Total Monte Carlo method developed by Koning (2008) and implemented in the TASMAN code by Koning (2012) to the specific case of neutron-induced reactions on ⁵⁶Fe in the fast neutron region below 20 MeV. The study verifies the reliability of the TASMAN reject/accept method and its ability to produce parameter correlations, and it enables future production of ENDF random files including resonance parts for reactor code simulations on burn-up and safety parameters, as previously studied by Henrik (2013) and Erwin (2013).
