# A new approach to handling factorial moment correlations through principal component analysis

> Intermittency analysis of factorial moments is a promising method used for the detection of power-law scaling in high-energy collision data. In particular, it has been employed in the search of fluctuations characteristic of the critical point (CP) of strongly interacting matter. However, intermittency analysis has been hindered by the fact that factorial moments measurements corresponding to different scales are correlated, since the same data are conventionally used to calculate them. This invalidates many assumptions involved in fitting data sets and determining the best fit values of power-law exponents. We present a novel approach to intermittency analysis, employing the well-established statistical and data science tool of Principal Component Analysis (PCA). This technique allows for the proper handling of correlations between scales without the need for subdividing the data sets available.

## Key claim

Principal Component Analysis (PCA) can properly handle the correlations between factorial moments at different momentum scales in intermittency analysis, eliminating the need to subdivide data sets, but the statistical independence of the resulting principal components is limited to linear correlations and must be empirically verified against a standard chi-squared distribution.

## Method

The authors compute second scaled factorial moments (SSFMs) \(F_2(M)\) for proton transverse momenta using the correlation integral method rather than a grid-based approach, which eliminates grid artifacts and is computationally faster. They estimate the covariance matrix of \(F_2(M)\) across different bin numbers \(M\) via bootstrap resampling of events, then apply Principal Component Analysis to diagonalize this covariance matrix and define uncorrelated linear combinations (principal components) of the original \(M\)-bins. The chi-squared statistic for fitting a power-law model is then calculated in the principal component space, where the assumption of independent residuals is valid, and the resulting likelihood distribution over model parameters \((\alpha_0, \varphi_2)\) is obtained by scanning a grid of parameter values.

## Key numbers

- The theoretically predicted intermittency index for protons in the 3D-Ising universality class is \(\varphi_{2,cr}^{(p)} = 5/6\).
- For a typical analysis window of total width \(\Delta p_{x,y} \simeq 3\) GeV/\(c\), a power-law is sought in the range \(32^2 \leq M^2 \leq 150^2\), corresponding to momentum scales \(95 \geq \delta p_{x,y} \geq 20\) MeV/\(c\).
- The CMC event generator can produce intermittency indices in the range \(\phi_2 \in [0.1, 1]\).
- CMC simulations used \(\sim 400\)K events with 1.60% critical protons, and the simulation was repeated independently for \(\sim 8\)K iterations.
- The PCA transformation retained 38 most significant principal components out of the original number of \(M\)-bins.

## Context

This work extends the intermittency analysis methodology pioneered by Bialas, Peschanski and others (Bialas 1986, Bialas 1988, Bialas 1990, Antoniou 2006) for detecting power-law scaling in high-energy nuclear collisions, specifically in the search for the QCD critical point. It addresses the known problem of correlated factorial moment bins, which invalidates simple chi-squared fitting; prior attempts to solve this include correlated chi-squared fits using the full correlation matrix (Wosiek 1990) which proved unstable, and using different event subsets for different scales (NA61/SHINE 2023) which severely reduces statistics. The PCA approach enables unbiased parameter estimation and confidence interval determination without sacrificing event statistics, and can be applied to analyses in proton, dipion, or all-charged channels.
