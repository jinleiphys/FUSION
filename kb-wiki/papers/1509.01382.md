# Heavy quark potential from QCD-related effective coupling

> We implement our past investigations in the quark-antiquark interaction through a non-perturbative running coupling defined in terms of a gluon mass function, similar to that used in some Schwinger-Dyson approaches. This coupling leads to a quark-antiquark potential, which satisfies not only asymptotic freedom but also describes linear confinement correctly. From this potential, we calculate the bottomonium and charmonium spectra below the first open flavor meson-meson thresholds and show that for a small range of values of the free parameter determining the gluon mass function an excellent agreement with data is attained.

## Key claim

The authors show that a quark-antiquark potential derived from a modified non-perturbative running coupling, which incorporates a Gribov singularity into a Schwinger-Dyson-based gluon mass function, reproduces the heavy quarkonium spectra with only one free parameter (Λ ≈ 320 MeV), provided the gluon mass function parameter m₀ is set equal to Λ.

## Method

The authors start from a Schwinger-Dyson running coupling defined via a gluon mass function and modify it by multiplying by (q² + m_g²(q²))/q², which cancels the propagator singularity and introduces a 1/q⁴ Gribov singularity. The resulting potential in coordinate space is obtained by Fourier transform and regularized by subtracting the divergent small-q expansion terms. The Schrödinger equation is then solved numerically for bottomonium (n_f = 4) and charmonium (n_f = 3) to compute the mass spectra, with the quark masses m_b and m_c adjusted for best fit at each Λ.

## Key numbers

- Λ = 320 MeV is the single free parameter that gives good spectral description for both bottomonium and charmonium
- Bottomonium 1s mass: calculated 9489 MeV vs PDG 9460.30 ± 0.26 MeV (with m_b = 4450 MeV)
- Charmonium 1s mass: calculated 3090 MeV vs PDG 3096.916 ± 0.011 MeV (with m_c = 1030 MeV)
- Bottomonium equivalent Cornell string tension: σ_{b\bar{b}} = 800 MeV·fm⁻¹
- Charmonium equivalent Cornell string tension: σ_{c\bar{c}} = 735 MeV·fm⁻¹
- Equivalent Cornell Coulomb strength: χ = 100 MeV·fm for both systems
- Schwinger-Dyson mass parameters fixed at ℳ = 436 MeV and p = 0.15
- Relative 1s quark velocities: (v²_{1s})_{c\bar{c}} = 0.3c², (v²_{1s})_{b\bar{b}} = 0.09c²

## Context

This work extends the authors' previous investigations (Gonzalez 2011, Vento 2012) on deriving the quark-antiquark potential from non-perturbative QCD couplings. It contrasts with the Schwinger-Dyson approach of Aguilar (2006, 2009, 2013) which gave a non-linearly rising potential, and instead produces a Cornell-like potential for intermediate distances (0.1–4 fm). The resulting Λ = 320 MeV agrees better with the QCD scale in the MSbar scheme (Λ ≈ 358 ± 22 MeV for n_f = 3, Λ ≈ 303 ± 21 MeV for n_f = 4 from Yndurain 1999 and Kataev 2001, 2015) than Richardson's (1978) potential which required Λ = 398 MeV. The approach enables future applications to other processes at moderate to high energies using the same effective coupling.
