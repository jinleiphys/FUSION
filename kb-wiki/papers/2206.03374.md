# Nuclear $\text{C}(e,e'p)$ Transparencies in a Relativistic Glauber Model

> In light of the recent Jefferson Laboratory (JLab) data for the nuclear $^{12}\text{C}(e,e'p)$ transparencies, calculations, obtained in a relativistic multiple scattering Glauber approximation, are discussed. The shell-separated $^{12}$C transparencies are shown and it is concluded that the $p$-shell nucleons are 75\% more transparent than the $s$-shell ones. The presented comparisons between the calculations made here and the current $^{12}\text{C}(e,e'p)$ data show no clear indication for the onset of color transparency when implemented within the color diffusion model with standard parameters.

## Key claim
The relativistic multiple scattering Glauber calculations for $^{12}\text{C}(e,e'p)$ transparencies show no clear indication of color transparency onset when implemented within the color diffusion model with standard parameters, despite the abstract's focus on comparing calculations to recent JLab data.

## Method
The authors performed relativistic multiple scattering Glauber approximation calculations for nuclear transparencies in $^{12}\text{C}(e,e'p)$ reactions. They computed shell-separated transparencies for the $p$-shell and $s$-shell nucleons, and compared these results to recent Jefferson Laboratory data. The color transparency effect was implemented using the color diffusion model with standard parameters to test for any signal of the phenomenon.

## Key numbers
- $p$-shell nucleons in $^{12}\text{C}$ are 75% more transparent than $s$-shell nucleons, based on the relativistic Glauber calculations.
- The calculations were compared to recent JLab $^{12}\text{C}(e,e'p)$ transparency data, with no clear indication of color transparency onset.

## Context
This work extends prior Glauber model calculations for nuclear transparencies in $(e,e'p)$ reactions, specifically addressing the shell dependence of transparency in $^{12}\text{C}$. The results contradict expectations from color transparency models that predict a rise in transparency with momentum transfer; instead, the standard color diffusion model parameters fail to describe the data. The findings enable future refinements of the color diffusion model or alternative approaches to color transparency, and provide benchmark calculations for comparison with upcoming higher-energy measurements at JLab.
