# Update on nuclear structure effects in light muonic atoms

> We present calculations of the nuclear structure corrections to the Lamb shift in light muonic atoms, using state-of-the-art nuclear potentials. We outline updated results on finite nucleon size contributions.

# Update on nuclear structure effects in light muonic atoms

## Key claim
The authors present updated calculations of nuclear structure corrections to the Lamb shift in light muonic atoms (A=2-4), incorporating a previously omitted term related to the 3rd Zemach moments of the nucleons and a corrected expression for the neutron-proton second-order nucleon size correction; these updates supersede some results from their earlier work (Ji 2013, Hernandez 2014, Nevo Dinur 2016).

## Method
The two-photon exchange (TPE) contribution to the Lamb shift is separated into elastic Zemach and inelastic polarization terms, each further divided into nuclear and nucleonic components. The nuclear polarizability is calculated using a multipole expansion formalism with state-of-the-art nuclear potentials, while nucleon size corrections are derived in coordinate-space following Friar (2013). The neutron-proton second-order nucleon size correction is re-derived using the isoscalar property of the dipole operators, yielding an expression that supersedes earlier momentum-space treatments.

## Key numbers
- For muonic deuterium (μD): total TPE contribution δ_TPE = -1.727(20) meV, with nuclear Zemach δ^A_Zem = -0.424(3) meV and nuclear polarizability δ^A_pol = -1.245(19) meV
- For muonic tritium (μ³H): total TPE δ_TPE = -0.767(25) meV, with δ^A_Zem = -0.227(6) meV and δ^A_pol = -0.473(17) meV
- For muonic helium-3 (μ³He⁺): total TPE δ_TPE = -15.46(39) meV, with δ^A_Zem = -10.49(24) meV and δ^A_pol = -4.17(17) meV
- For muonic helium-4 (μ⁴He⁺): total TPE δ_TPE = -9.58(38) meV, with δ^A_Zem = -6.29(28) meV and δ^A_pol = -2.36(14) meV
- The nucleon Zemach term for μD is δ^N_Zem = -0.030(2) meV, scaled from muonic hydrogen using the factor (Z m_r(μA)/m_r(μH))^4

## Context
This work extends the authors' previous ab initio calculations (Ji 2013, Hernandez 2014, Ji 2014, Nevo Dinur 2016) for muonic systems with A=2-4 by incorporating a correct implementation of the nucleon Zemach term and an updated neutron-proton second-order correction. The results are intended to support the CREMA collaboration's measurements at PSI, which aim to extract nuclear charge radii from Lamb shift measurements in multiple muonic atoms to investigate the proton radius puzzle. The updated expressions coincide with the special case of deuterium given by Pachucki (2015) and were used in a recent theory summary for the μD experiment (Krauth 2016).
