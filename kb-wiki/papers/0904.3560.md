# String Theory at LHC Using Higgs Production and Decay From String Balls

> We study Higgs production and decay from TeV scale string balls in $pp$ collisions at $\sqrt{s}$ = 14 TeV at LHC. We present the results of total cross section of diphotons, invariant mass distribution of the diphotons and $p_T$ distribution of the diphotons and $ZZ$ pairs from Higgs from string balls at LHC. We find that the invariant mass distribution of diphotons from Higgs from string balls is not very sensitive to the increase in diphoton invariant mass. We find that for string mass scale $M_s$=2.5 TeV, which is the lower limit of the string mass scale set by the recent CMS collaboration at LHC, the $\frac{dσ}{dp_T}$ of high $p_T$ ($\ge$ 450 GeV) diphotons and $ZZ$ pairs produced from Higgs from string balls is larger than that from standard model Higgs. Hence in the absence of black hole production at LHC an enhancement of high $p_T$ diphotons and $ZZ$ pairs at LHC can be useful signatures for string theory at LHC. Since the matrix element for Higgs production in parton-parton collisions via string regge excitations is not available we compute $\frac{dσ}{dp_T}$ of photon production from string regge excitations and make a comparison with that from string balls at LHC. We find that for string mass scale $M_s$ = 2.5 TeV the $\frac{dσ_{photon}}{dp_T}$ from string regge excitations is larger than that from string balls and black holes at LHC.

## Key claim

For a string mass scale \(M_s = 2.5\) TeV (the lower limit set by CMS), the differential cross section \(\frac{d\sigma}{dp_T}\) of high-\(p_T\) (\(\ge 450\) GeV) diphotons and \(ZZ\) pairs produced from Higgs decay from string balls is larger than that from standard model Higgs, but only if black hole production is absent; the invariant mass distribution of diphotons from Higgs from string balls is not very sensitive to the increase in diphoton invariant mass.

## Method

The author computed the differential cross section \(\frac{d\sigma}{dp_T}\) for diphoton and \(ZZ\) pair production from Higgs bosons (\(M_H = 120\) GeV for diphotons, \(M_H = 200\) GeV for \(ZZ\)) produced via TeV-scale string balls in \(pp\) collisions at \(\sqrt{s} = 14\) TeV, using parton distribution functions (CTEQ6M) and the string ball production cross section from Dimopoulos and Emparan (2001). For comparison, the same observables were computed for Higgs from black holes and from standard model parton fusion processes at NLO, as well as for direct diphoton and \(ZZ\) production from parton fusion. Additionally, because the matrix element for Higgs production via string Regge excitations is unavailable, the author computed \(\frac{d\sigma}{dp_T}\) for photon production from string Regge excitations using tree-level four-particle open-string amplitudes for \(q\bar{q} \rightarrow \gamma\gamma\) and \(gg \rightarrow \gamma\gamma\) in the low-energy limit (\(\hat{s} < M_s^2\)).

## Key numbers

*   For \(M_s = 2.5\) TeV, \(\frac{d\sigma}{dp_T}\) of diphotons from Higgs from string balls is larger than that from standard model Higgs for \(p_T \ge 450\) GeV.
*   For \(M_s = 2.5\) TeV, \(\frac{d\sigma}{dp_T}\) of \(ZZ\) pairs from Higgs from string balls is larger than that from standard model Higgs for \(p_T \ge 400\) GeV.
*   Diphoton cross section from Higgs from string balls at LHC: 110 fb for \(M_s = 1\) TeV; 0.56 fb for \(M_s = 2.5\) TeV.
*   Diphoton cross section from Higgs from black holes at LHC: 616 fb for \(M_s = 1\) TeV; 0.0015 fb for \(M_s = 2.5\) TeV.
*   For \(M_s = 2.5\) TeV, \(\frac{d\sigma_{photon}}{dp_T}\) from string Regge excitations is larger than that from string balls and black holes at LHC.
*   The Hagedorn temperature of a string ball is \(T_{SB} = M_s / (\sqrt{8} \pi)\).
*   The string coupling used is \(g_s^2 = 1/3\).

## Context

This work extends the study of Higgs production from black holes via Hawking radiation (Giddings and Thomas, 2002; Landsberg, 2002; Argyres et al., 2005) to the case of string balls, which are lighter than black holes and thus expected to be more abundant at LHC. The analysis uses the string ball production scheme of Dimopoulos and Emparan (2001) and compares results with standard model Higgs production via parton fusion at NLO and with direct diphoton and \(ZZ\) production from QCD processes. The paper enables experimental searches for string theory signatures at LHC by proposing that an enhancement of high-\(p_T\) diphotons and \(ZZ\) pairs, in the absence of black hole production, could serve as a distinctive signal for string balls, and that photon production from string Regge excitations could provide an additional complementary signature.
