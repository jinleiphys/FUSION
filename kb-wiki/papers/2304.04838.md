# Multi-nucleon structure and dynamics via quantum computing

> We propose a framework for computing the structure and dynamics for second-quantized many-nucleon Hamiltonians on quantum computers. We develop an oracle-based Hamiltonian input model that computes the many-nucleon states and nonzero Hamiltonian matrix elements of the many-nucleon system. With our Fock-state based input model, we show how to implement the sparse matrix simulation algorithms to calculate the dynamics of the second-quantized many-nucleon Hamiltonian. Based on the dynamics simulation methods, we also present the methodology for structure calculations of the many-nucleon system. In this work, we provide an explicit circuit design of our input model of the second-quantized Hamiltonian within a direct encoding scheme that maps the occupation of each available single-particle state in the many-nucleon state to the state of specific qubit in a quantum register. We analyze our method and provide the asymptotic cost in computing resources for structure and dynamics calculations of many-nucleon systems. For pedagogical purposes, we demonstrate our input model with two model problems in restricted model spaces.

## Key claim

The authors propose a framework for computing the structure and dynamics of second-quantized many-nucleon Hamiltonians on quantum computers, but the demonstration is limited to two model problems in restricted model spaces for pedagogical purposes; the method has not yet been applied to realistic many-nucleon systems with actual inter-nucleon interactions.

## Method

The authors develop an oracle-based Hamiltonian input model that operates directly on Fock states encoded via the direct encoding (Jordan-Wigner) scheme, where each qubit represents the occupation of a single-particle basis state. The oracles compute the output Fock state, the active single-nucleon bases that are annihilated and created, and the few-nucleon kernel contributing to the Hamiltonian matrix element. Using these oracle queries, the authors construct a block encoding of the Hamiltonian, enabling the application of sparse matrix simulation algorithms for dynamics calculations and adaptation of those algorithms for structure calculations.

## Key numbers

* The number of qubits required for the direct encoding scheme equals the number of single-particle basis states \(N_{\rm sp}\), independent of the nucleon number \(A\).
* The compact encoding scheme would require \(A \lceil \log_2 N_{\rm sp} \rceil\) qubits, which scales better than \(N_{\rm sp}\) for the direct encoding scheme.
* For an \(A\)-nucleon system with \(N_{\rm sp}\) available single-particle basis states, the number of many-nucleon basis states is \(N_{\rm mp} = \binom{N_{\rm sp}}{A}\).
* The Hamiltonian matrix element \(H_{pqrs}\) is composed of three two-body terms: \(T^{\rm rel}_{pqrs} + V^{\rm NN}_{pqrs} + H^{\rm CM}_{pqrs}\).

## Context

This work extends the line of research applying quantum computing to nuclear many-body problems, following prototype calculations on real quantum hardware by Dumitrescu (2018), Roggero (2020), and Kiss (2022). The authors complement the work by Kirby et al. (2021) on efficient quantum algorithms for second-quantized Hamiltonians, but adopt the direct encoding scheme with primitive gate operations for straightforward prototype calculations on near-term noisy intermediate scale quantum (NISQ) devices, rather than the compact encoding scheme preferred for long-term quantum computers. The framework enables future dynamics and structure calculations for simple many-nucleon systems with realistic inter-nucleon interactions within restricted model spaces.
