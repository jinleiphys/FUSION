# Exchange Currents for Hypernuclear Magnetic Moments

> The meson(K and $π$) exchange currents for the hypernuclear magnetic moments are calculated using the effective Lagrangian method. The seagull diagram, the mesonic diagram and the $Σ^0$-excitation diagram are considered. The $Λ$-N exchange magnetic moments for the ${}^5_ΛHe$, ${}^6_ΛHe$ and ${}^6_ΛLi$ are calculated employing the harmonic oscillator shell model. It is found that the two-body correction is about -9% of the single particle value for ${}^5_ΛHe$. The $π$ exchange current, induced only in the $Σ^0$-excitation diagram, is found to give dominant contribution for the isovector magnetic moments of hypernuclei with A=6.

## Key claim

The meson exchange current (kaon and pion) correction to hypernuclear magnetic moments is calculated to be about -9% of the single-particle value for \({}^{5}_{\Lambda}\mathrm{He}\), but this result depends on the harmonic oscillator shell model and neglects core polarization effects, which the authors argue are small for this specific nucleus.

## Method

The authors employed an effective Lagrangian approach with pseudovector coupling to derive one-kaon and one-pion exchange current operators between \(\Lambda\) and nucleons, considering seagull, mesonic, and \(\Sigma^{0}\)-excitation diagrams. They then evaluated matrix elements of these operators using harmonic oscillator shell model wave functions for \({}^{5}_{\Lambda}\mathrm{He}\), \({}^{6}_{\Lambda}\mathrm{He}\), and \({}^{6}_{\Lambda}\mathrm{Li}\), with coupling constants taken from the Nijmegen meson exchange potential model F.

## Key numbers

- Two-body correction for \({}^{5}_{\Lambda}\mathrm{He}\): \(\delta\mu = 0.054\,\mu_{N}\) (-8.8% of the single-particle value of -0.613 \(\mu_{N}\))
- Single-particle magnetic moment for A=6 isodoublet: isoscalar 1.090 \(\mu_{N}\), isovector 2.377 \(\mu_{N}\)
- \(\Lambda\)-N exchange contribution to A=6 isovector moment: 0.044 \(\mu_{N}\) (1.9% of single-particle value)
- \(\pi\)-exchange (\(\Sigma^{0}\)-excitation) contribution to \(\delta\mu(\Lambda(s_{1/2})\)-N(p\({}_{3/2})\)) isovector part: 0.041 \(\mu_{N}\)
- Total \(\delta\mu\) for \({}^{6}_{\Lambda}\mathrm{He}\): -0.021 \(\mu_{N}\) (1.6% of single-particle value of -1.288 \(\mu_{N}\))
- Total \(\delta\mu\) for \({}^{6}_{\Lambda}\mathrm{Li}\): -0.040 \(\mu_{N}\) (-1.2% of single-particle value of 3.467 \(\mu_{N}\))
- Gaussian oscillator parameter: b = 1.35 fm for both \(\Lambda\) and nucleons

## Context

This work extends the study of meson exchange currents, previously applied to nuclear magnetic moments by Chemtob (1971) and Mathiot (1989), to the hypernuclear sector using kaon and pion exchanges. The isovector \(\Sigma^{0}\)-excitation contribution supports the earlier analysis of \(\Sigma\) mixing effects by Dover, Feshbach, and Gal, while the isoscalar kaon exchange current provides a new prediction that could be tested by proposed experiments such as Nakai's measurement of the \({}^{5}_{\Lambda}\mathrm{He}\) magnetic moment using hyperfine magnetic fields.
