# Dispersive analysis of the scalar form factor of the nucleon

> Based on the recently proposed Roy-Steiner equations for pion-nucleon scattering, we derive a system of coupled integral equations for the pi pi --> N-bar N and K-bar K --> N-bar N S-waves. These equations take the form of a two-channel Muskhelishvili-Omnes problem, whose solution in the presence of a finite matching point is discussed. We use these results to update the dispersive analysis of the scalar form factor of the nucleon fully including K-bar K intermediate states. In particular, we determine the correction Delta_sigma=sigma(2M_pi^2)-sigma_{pi N}, which is needed for the extraction of the pion-nucleon sigma term from pi N scattering, as a function of pion-nucleon subthreshold parameters and the pi N coupling constant.

## Key claim

The paper updates the dispersive analysis of the nucleon's scalar form factor by fully including kaon-antikaon intermediate states in a coupled-channel framework, determining the correction Delta_sigma = sigma(2 M_pi^2) - sigma_{pi N} as a function of pion-nucleon subthreshold parameters and the pion-nucleon coupling constant, rather than providing a single fixed value.

## Method

The authors derive a system of coupled integral equations for the pi pi -> N-bar N and K-bar K -> N-bar N S-waves based on Roy-Steiner equations for pion-nucleon scattering, formulated as a two-channel Muskhelishvili-Omnes problem with a finite matching point. They construct the corresponding Omnes matrix numerically using the infinite-matching-point solution as a starting point, then apply this formalism to compute the scalar form factor of the nucleon including K-bar K intermediate states in both the unitarity relation and the determination of the pi pi -> N-bar N partial wave.

## Key numbers

- The previous estimate for Delta_sigma was (15.2 +/- 0.4) MeV from Gasser, Leutwyler, and Sainio (1990), which neglected K-bar K contributions in the unitarity relation for sigma(t) and used a pion-nucleon coupling constant g^2/4pi = 14.28 that cannot be reconciled with more recent determinations.
- The matching point t_m is set at the energy where the two-channel model breaks down due to the effective onset of 4-pi inelasticities.
- The Omnes matrix determinant behaves as det Omega(t) ~ |t_m - t|^x near the matching point, with x = psi(t_m)/pi, where psi is the scattering phase.
- The exponents x_ij for the Omnes matrix components satisfy x_11 = x_21, x_12 = x_22, and x_ij = delta_ij/pi, with x_11 + x_12 = x.
- The ratio of Omnes matrix components at the matching point is given by bar_Omega_21 / bar_Omega_11 = N / (2|g| sigma_{t_m}^K) and bar_Omega_12 / bar_Omega_22 = -N / (2|g| sigma_{t_m}^pi), where N = sqrt(1 - eta^2 cos^2 pi(2y - x)) - eta sin pi(2y - x).

## Context

This work extends the dispersive analysis of the nucleon scalar form factor by Gasser, Leutwyler, and Sainio (1990), which neglected K-bar K intermediate states in the unitarity relation for sigma(t) and used older input for the pi pi -> N-bar N S-wave from Hoehler (1983). The authors build on the Roy-Steiner equations for pion-nucleon scattering developed by Hoferichter, Ditsche, Kubis, and Meissner (2012) and the two-channel Muskhelishvili-Omnes formalism applied to scalar form factors by Moussallam (1999) and others. The updated analysis enables a more consistent extraction of the pion-nucleon sigma term from pi N scattering data by providing Delta_sigma as a function of modern pi N subthreshold parameters and coupling constants.
