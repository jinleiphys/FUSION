# Parameter optimisation using Bayesian inference for spallation models

> The accuracy and precision of high-energy spallation models are key issues for the design and development of new applications and experiments. We present a method to estimate model parameters and associated uncertainties by leveraging the Bayesian version of the Generalised Least Squares method, which enables us to incorporate prior knowledge on the parameter values. This approach is designed to adjust parameters based on experimental data, accounting for experimental uncertainty information, and providing uncertainties for all adjusted parameters. This approach is designed in order both to improve the accuracy of models through the modification of free parameters of these models, which results in a better reproduction of experimental data, and to estimate the uncertainties of these parameters and, by extension, their impacts on the model output. We aim at demonstrating the Generalised Least Square method can be applied in the case of Monte Carlo models. We present a proof-of-concept for Monte Carlo models in the specific case of nuclear physics with the model combination INCL/ABLA. We discuss the challenges in the application of this method to high-energy spallation models, notably the large runtime and the stochasticity of the models. Our results indicate this framework can also be applied to analogous situations where parameters of a computationally expensive Monte Carlo code should be inferred/improved.

## Key claim

The paper demonstrates that a Bayesian Generalised Least Squares method, combined with an approximative Gibbs sampling scheme, can be applied to optimise parameters and estimate their uncertainties for computationally expensive Monte Carlo spallation models like INCL/ABLA, but only under the assumption that the model is perfect (i.e., that the "correct" parameter values would lead to predictions coinciding with true values), which the authors acknowledge is never the case.

## Method

The authors developed a two-phase algorithm: first, an iterative Generalised Least Squares (GLS) phase that linearises the INCL/ABLA model around the current parameter set using a Jacobian matrix, then updates parameters to minimise the difference between model predictions and experimental data while incorporating prior knowledge on parameter values. Second, an approximative Gibbs sampling phase alternates between evaluating the posterior covariance matrix using GLS formulae and sampling new parameter sets from a multivariate normal distribution, producing a posterior distribution from which parameter uncertainties and correlations are extracted. The Jacobian is only recalculated when the Taylor approximation deviates from the model output by more than twice the best relative prediction accuracy previously achieved.

## Key numbers

- The MYRRHA project operates at energies up to 600 MeV, motivating the need for high-energy spallation model improvements.
- The minimum number of iterations required ranges from a few tens to a hundred for the GLS phase and from a few hundreds to a few thousands for the Gibbs sampling phase.
- The Jacobian is kept valid as long as the Taylor approximation differs by less than twice the best relative prediction ever obtained (e.g., if best prediction was within 20% accuracy, the Jacobian is conserved until the Taylor approximation differs by more than 40%).
- Some experimental data included in the study had relative uncertainties below 1%, which the authors identify as problematic for the Bayesian procedure.

## Context

This work extends earlier efforts within the CHANDA project (2013-2017) and the SANDA project, where a methodology for quantifying uncertainties linked to parameters in high-energy models was first investigated. The authors specifically build on a previous study by Schnabel (2018) that addressed model defect estimation, while the present work focuses on parameter optimisation and uncertainty quantification for the INCL/ABLA model combination, which had been evaluated as the best among tested configurations by the IAEA. The approximative Gibbs sampling approach is described as new in nuclear modelling. The method is designed to be applicable to other computationally expensive Monte Carlo codes beyond nuclear spallation models.
