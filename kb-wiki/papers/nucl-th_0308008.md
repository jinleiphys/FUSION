# Ladder Dyson-Schwinger calculation of the anomalous gamma-3pi form factor

> The anomalous processes, γ\to 3 πand γπ\to ππ, are investigated within the Dyson-Schwinger framework using the rainbow-ladder approximation. Calculations reveal that a complete set of ladder diagrams beyond the impulse approximation are necessary to reproduce the fundamental low-energy theorem for the anomalous form factor. Higher momentum calculations also agree with the limited form factor data and exhibit the same resonance behavior as the phenomenological vector meson dominance model.

## Key claim

The rainbow-ladder truncation of Dyson-Schwinger equations, when including a complete set of ladder diagrams beyond the generalized impulse approximation (GIA), reproduces the low-energy theorem for the anomalous gamma-3pi form factor, whereas the GIA alone fails when using the full pion Bethe-Salpeter amplitude.

## Method

The authors solved the Dyson-Schwinger equation for the quark propagator and the homogeneous Bethe-Salpeter equation for the pion bound state in the rainbow-ladder truncation, using an effective interaction fitted to pion and rho meson properties. To compute the gamma-3pi form factor, they solved an inhomogeneous Bethe-Salpeter equation that sums all planar ladder diagrams with gluon exchange in the s, t, and u channels, going beyond the generalized impulse approximation used in prior DSE studies.

## Key numbers

- Low-energy theorem prediction: F^{3pi}(0,0,0) = e/(4 pi^2 f_pi^3) = 9.72 GeV^{-3} for physical pion mass (f_pi = 92.4 MeV)
- Chiral limit value: hat{F}^{3pi}(0,0,0) = e/(4 pi^2 hat{f}_pi^3) approximately 10.5 GeV^{-3} (hat{f}_pi = 90 MeV)
- Model parameters: omega = 0.4 GeV, D = 0.93 GeV^2, degenerate u/d quark mass = 5.5 MeV at mu = 1 GeV
- Calculated rho meson mass: 742 MeV (experimental: 771 MeV)
- Calculated omega meson mass: 742 MeV (experimental: 783 MeV)
- Extracted omega-rho-pi coupling from DSE fit: g_{omega rho pi} = 10.3 GeV^{-1} (phenomenological VMD value: 11.8 GeV^{-1})
- Threshold for three-pion production: Q^2 = -9 m_pi^2 approximately -0.172 GeV^2
- Experimental data point at Q^2 = 0: 12.9 +/- 0.9 +/- 0.5 GeV^{-3} (corrected value 11.9 GeV^{-3} from Holstein 1996)

## Context

This work extends prior DSE studies of the gamma-3pi process by Alkofer (1996) and Bistrovic (1999) that used the generalized impulse approximation, which the authors show is insufficient for four-body amplitudes. The calculation is analogous to the authors' earlier work on pi-pi scattering (Cotanch 2002) and uses the same model interaction developed by Maris (1999) that successfully describes pion and rho meson properties. The results enable comparison with ongoing JLab experiments (Miskimen) measuring the form factor in the range 0.27 GeV^2 < s < 0.72 GeV^2, and provide a microscopic foundation for vector meson dominance phenomenology.
