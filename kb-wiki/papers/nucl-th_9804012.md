# Method of integral transforms for calculating few-body reactions

> A non-conventional approach to calculating reactions in quantum mechanics is presented. Reaction observables are obtained with bound state calculation techniques. The accuracy of the method to calculate few-nucleon response functions is discussed.

# Method of integral transforms for calculating few-body reactions

## Key claim

The paper presents a method to calculate reaction observables using bound-state techniques, avoiding explicit treatment of continuum wave functions. The authors claim the method is accurate for few-nucleon response functions, but note that the inversion stability depends critically on the choice of a Lorentzian kernel and on regularization parameters, with the 4-nucleon case showing a stability region of only ΔN ≈ 3-4 basis functions.

## Method

The method proceeds in two steps: first, an integral transform of the response function R(ε) is computed using a Lorentzian kernel K(σ) = 1/[(σ_R - ε)² + σ_I²], where the transform Φ(σ) is obtained by solving a Schrödinger-like equation with a source term using bound-state techniques (inhomogeneous Faddeev-type equations or Jastrow-correlated hyperspherical harmonic basis). Second, the response function R(ε) is recovered by inverting the integral transform, expanding R(ε) in a basis of known functions χ_n and fitting the transformed basis to the computed Φ(σ) values, with the number of basis functions N serving as the regularization parameter.

## Key numbers

- σ_I values taken from the interval 5 MeV ≤ σ_I ≤ 20 MeV for three- and four-nucleon calculations
- In the 4-nucleon calculations, the stability region for the regularization parameter N was ΔN ≈ 3-4
- Non-energy-weighted sum rule differences were about 1% or less; energy-weighted sum rule differences were less than 2%
- K_max values governing the number of CHH basis functions were substantially lower than those required in bound-state calculations with the same NN potential

## Context

This work extends the original Stieltjes transform approach introduced by Efros (1985), which suffered from stability problems as noted in Efros, Leidemann, Orlandini (1993). The Lorentzian kernel was introduced in Efros, Leidemann, Orlandini (1994) to cure those instabilities. The method enabled accurate calculations of three- and four-nucleon longitudinal (e,e') response functions and total photoabsorption cross sections, as demonstrated in subsequent works by the same group (1995, 1997). The approach opens the possibility of treating exclusive and strong interaction induced reactions using bound-state techniques.
