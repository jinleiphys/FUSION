# Application of the gradient method to Hartree-Fock-Bogoliubov theory

> A computer code is presented for solving the equations of Hartree-Fock-Bogoliubov (HFB) theory by the gradient method, motivated by the need for efficient and robust codes to calculate the configurations required by extensions of HFB such as the generator coordinate method. The code is organized with a separation between the parts that are specific to the details of the Hamiltonian and the parts that are generic to the gradient method. This permits total flexibility in choosing the symmetries to be imposed on the HFB solutions. The code solves for both even and odd particle number ground states, the choice determined by the input data stream. Application is made to the nuclei in the $sd$-shell using the USDB shell-model Hamiltonian.

## Key claim

The gradient method provides a robust and flexible algorithm for solving Hartree-Fock-Bogoliubov (HFB) equations that can handle multiple constraints and both even- and odd-particle-number systems without requiring modifications to the Hamiltonian or iteration process, unlike matrix-diagonalization methods which need additional constraining fields for odd-A nuclei.

## Method

The authors implement an iterative gradient minimization of the HFB energy functional using the generalized Thouless transformation, where the update direction is given by the skew-symmetric matrix \(H_c^{20}\) derived from the constrained Hamiltonian. The code separates Hamiltonian-specific modules from generic gradient-method routines, allowing flexible imposition of symmetries (axial, parity, neutron-proton blocks) via input parameters. For odd-A systems, the initial \(U,V\) configuration is converted to odd number parity by interchanging a column between the \(U\) and \(V\) matrices, and unwanted symmetries in the starting configuration are broken by applying a random \(Z\) transformation.

## Key numbers

- For \(^{24}\)Mg with constraints \(N=4\), \(Z=4\), and \(\langle Q_Q\rangle = 10\) \(\hbar/m\omega_0\), convergence criterion \(|H_c^{20}| < 1.0\times 10^{-2}\) MeV required approximately 40 to 250 iterations depending on deformation.
- For \(^{21}\)Ne, the HFB energy from randomized starting configurations converged to \(-41.715\) MeV for all column exchanges, compared to \(-40.837\) MeV (neutron exchanges) and \(-40.176\) to \(-40.215\) MeV (proton exchanges) without randomization.
- For \(^{32}\)Mg (\(N=12, Z=4\)), the HFB ground state energy was \(E_{HFB} = -135.641\) MeV with \(\Delta Z^2 = 2.93\) and \(Q_Q^{00} = 0.00\); the Hartree-Fock minimum was \(E_{HF} = -134.460\) MeV, giving a correlation energy of 1.18 MeV.
- The number of iterations to convergence for \(^{21}\)Ne ranged from 140 (fixed gradient, \(\eta = 0.10\) MeV\(^{-1}\)) to 34 (variable preconditioned gradient, \(\eta\) from 0.7 to 2.0).
- For \(^{24}\)Mg, the converged HFB minimum had quadrupole moment \(\langle Q_Q\rangle = 12.8\) \(\hbar/m\omega_0\) with vanishing pairing condensate.

## Context

This work extends the gradient method previously described by Ring and Schuck (1980) for HFB theory, building on earlier implementations by Robledo and collaborators (2002) and the HF gradient method discussion by Reinhard (1982). The code enables systematic calculation of the multiple constrained configurations needed for the generator coordinate method (GCM), and provides a testing ground for the \(sd\)-shell using the USDB Hamiltonian of Brown and Richter (2006). The approach contrasts with matrix-diagonalization HFB codes (such as those by Pozzi et al. 1997, Bonche et al. 2005, and others) that require additional constraints for odd-A systems and cannot easily handle neutron-proton mixing.
