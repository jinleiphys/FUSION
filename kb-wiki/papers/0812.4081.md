# Determination of the effective parameters of proton-$^{3}$He scattering on the basis of the neutron-triton scattering data

> We have studied the relations between the neutron-triton scattering lengths and effective ranges and the corresponding quantities for the p --$^{3}$He scattering in the framework of the potential model with an effective nucleon-nucleus interaction in the form of a $δ$-shell potential. It is shown that the Coulomb renormalization of the pure nuclear scattering lengths does not change the relation well established for the n + $^{3}$H system between the lengths: $A^{1} < A^{0}$. We have predicted the p--$^{3}$He scattering lengths which give preference to set I of the phase analysis performed by E.A. George et al. (2003), which corresponds to the inequality $A^{1}_{nc} < A^{0}_{nc}$ for the scattering lengths.

# Determination of the effective parameters of proton-³He scattering on the basis of the neutron-triton scattering data

## Key claim

Using a δ-shell potential model, the Coulomb renormalization of pure nuclear scattering lengths does not change the well-established relation A¹ < A⁰ from the n+³H system, and the predicted p-³He scattering lengths favor set I of the phase analysis by George et al. (2003), which corresponds to the inequality A¹_nc < A⁰_nc for the scattering lengths.

## Method

The author employs a δ-shell potential model for the effective nucleon-nucleus interaction, with analytic expressions relating the pure nuclear scattering lengths and effective ranges for n-t scattering to the corresponding nuclear-Coulomb quantities for p-³He scattering. Using the most reliable experimental and theoretical n-t scattering data as input, the potential parameters (range R and intensity λ) are determined from formulas (20) and (21), and then applied to predict the p-³He scattering lengths and effective ranges via formulas (22) and (23). Both attractive and repulsive δ-shell potentials are considered, with the repulsive case motivated by Pauli repulsion in the nucleon-three-nucleon system.

## Key numbers

- Triplet n-t scattering length: A¹_n = (3.6 ± 0.1) fm (experimental value from reference [10])
- Singlet n-t scattering length (averaged theoretical): A⁰_n = (4.0 ± 0.1) fm
- Singlet n-t scattering length (R-matrix result): A⁰_n = (4.453 ± 0.100) fm from reference [16]
- Triplet n-t effective range (theoretical): r¹_n = (1.75 ± 0.05) fm
- Singlet n-t effective range (theoretical): r⁰_n = (1.95 ± 0.05) fm
- Predicted triplet p-³He scattering length (repulsive potentials): A¹_nc = (8.88 ± 0.48) fm
- Predicted triplet p-³He effective range (repulsive potentials): r¹_nc = (1.68 ± 0.04) fm
- Predicted singlet p-³He scattering length (repulsive potentials, using A⁰_n = 4.0 fm): A⁰_nc = (10.63 ± 0.52) fm
- Predicted singlet p-³He effective range (repulsive potentials, using A⁰_n = 4.0 fm): r⁰_nc = (1.85 ± 0.04) fm
- Predicted singlet p-³He scattering length (repulsive potentials, using A⁰_n = 4.453 fm): A⁰_nc = (13.05 ± 0.62) fm
- Predicted singlet p-³He effective range (repulsive potentials, using A⁰_n = 4.453 fm): r⁰_nc = (1.97 ± 0.04) fm
- Set I from George et al. (2003): A¹_nc = 7.9 fm, A⁰_nc = 15.1 fm
- Set II from George et al. (2003): A¹_nc = 10.4 fm, A⁰_nc = 7.2 fm
- Ratio A¹_n / A⁰_n = 0.895 from theoretical averaging
- Ratio r¹_n / r⁰_n = 0.897 from theoretical averaging

## Context

This work extends the use of δ-shell potentials, previously applied to NN scattering and n-t scattering in four-particle Faddeev-Yakubovsky calculations by the author and collaborators (Levashev, references [18, 26, 27]), to the charge-symmetric p-³He system. The predictions resolve the ambiguity between two competing sets of p-³He scattering lengths from the phase analysis of George et al. (2003), favoring set I (A¹_nc < A⁰_nc). The results agree with four-particle calculations using the Kohn variational principle and hyperspherical harmonics with three-nucleon forces by reference [21] (potential AV18UR), which gave A¹_nc = 9.13 fm and A⁰_nc = 11.5 fm, and with Monte Carlo calculations by reference [32] (potential AV18UR) giving A¹_nc = (10.1 ± 0.5) fm. The method is also validated by application to the p+α and n+α systems, where predictions match experimental data from reference [33].
