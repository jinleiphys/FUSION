# Collectivity in Heavy Nuclei in the Shell Model Monte Carlo Approach

> The microscopic description of collectivity in heavy nuclei in the framework of the configuration-interaction shell model has been a major challenge. The size of the model space required for the description of heavy nuclei prohibits the use of conventional diagonalization methods. We have overcome this difficulty by using the shell model Monte Carlo (SMMC) method, which can treat model spaces that are many orders of magnitude larger than those that can be treated by conventional methods. We identify a thermal observable that can distinguish between vibrational and rotational collectivity and use it to describe the crossover from vibrational to rotational collectivity in families of even-even rare-earth isotopes. We calculate the state densities in these nuclei and find them to be in close agreement with experimental data. We also calculate the collective enhancement factors of the corresponding level densities and find that their decay with excitation energy is correlated with the pairing and shape phase transitions.

## Key claim

The shell model Monte Carlo (SMMC) method can microscopically describe the crossover from vibrational to rotational collectivity in families of even-even rare-earth isotopes within a truncated spherical shell model space, using the thermal observable ⟨J²⟩_T to distinguish between the two types of collectivity.

## Method

The authors used the SMMC method, based on the Hubbard-Stratonovich transformation, to compute thermal expectation values in a configuration-interaction shell model space consisting of six proton orbitals and eight neutron orbitals with a Hamiltonian that includes monopole pairing and multipole-multipole (quadrupole, octupole, hexadecupole) interactions. They calculated ⟨J²⟩_T as a function of temperature for even-even samarium (¹⁴⁸–¹⁵⁴Sm) and neodymium (¹⁴⁴–¹⁵²Nd) isotopes, and extracted state densities via saddle-point inversion of the canonical partition function. The collective enhancement factor K was defined as the ratio of SMMC state densities to finite-temperature Hartree-Fock-Bogoliubov (HFB) state densities.

## Key numbers

- For ¹⁴⁸Sm (vibrational), the extracted E_{2⁺} from SMMC is 0.538 ± 0.031 MeV, compared to the experimental value of 0.550 MeV.
- For ¹⁵⁴Sm (rotational), the extracted E_{2⁺} from SMMC is 0.087 ± 0.006 MeV, compared to the experimental value of 0.082 MeV.
- For ¹⁴⁴Nd (vibrational), the extracted E_{2⁺} from SMMC is 0.702 ± 0.062 MeV, compared to the experimental value of 0.697 MeV.
- For ¹⁵⁰Nd (rotational), the extracted E_{2⁺} from SMMC is 0.132 ± 0.012 MeV, compared to the experimental value of 0.130 MeV.
- For ¹⁵²Nd (rotational), the extracted E_{2⁺} from SMMC is 0.107 ± 0.006 MeV, compared to the experimental value of 0.073 MeV.

## Context

This work extends earlier SMMC studies of collectivity in ¹⁶²Dy by Alhassid (2008) to families of samarium and neodymium isotopes, demonstrating a microscopic description of the vibrational-to-rotational crossover that was previously only accessible via phenomenological models. The calculated state densities agree closely with experimental level counting data and neutron resonance data, and the collective enhancement factors show that vibrational collectivity is lost above pairing transition energies while rotational collectivity persists until the shape phase transition, enabling future microscopic calculations of level densities for heavy nuclei.
