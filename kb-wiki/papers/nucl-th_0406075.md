# Axially Deformed Solution of the Skyrme-Hartree-Fock-Bogolyubov Equations using The Transformed Harmonic Oscillator Basis. The program HFBTHO (v1.66p)

> We describe the program HFBTHO for axially deformed configurational Hartree-Fock-Bogoliubov calculations with Skyrme-forces and zero-range pairing interaction using Harmonic-Oscillator and/or Transformed Harmonic-Oscillator states. The particle-number symmetry is approximately restored using the Lipkin-Nogami prescription, followed by an exact particle number projection after the variation. The program can be used in a variety of applications, including systematic studies of wide ranges of nuclei, both spherical and axially deformed, extending all the way out to nucleon drip lines.

## Key claim

The paper describes the HFBTHO program for axially deformed Skyrme-Hartree-Fock-Bogoliubov calculations, claiming that using the Transformed Harmonic Oscillator (THO) basis allows accurate description of deformation effects and pairing correlations in nuclei arbitrarily close to the particle drip lines, but only for even-even nuclei with axial-, time-reversal-, and space-inversion symmetries assumed.

## Method

The program expands quasiparticle wave functions in an axially deformed Transformed Harmonic Oscillator (THO) single-particle basis, generated by applying a local-scale coordinate transformation to standard harmonic oscillator states. It iteratively diagonalizes the Hartree-Fock-Bogoliubov Hamiltonian based on Skyrme forces and zero-range pairing interaction until self-consistency is achieved, with the particle-number symmetry approximately restored using the Lipkin-Nogami prescription followed by exact particle number projection after variation.

## Key numbers

- Memory required: 59 MB when using N_sh=20
- Typical running time: 4 seconds per iteration on an Intel Xeon 2.8 GHz processor when using N_sh=20
- Number of lines in distributed program: 7876 lines
- Integration uses 22 Gauss-Hermite points in the z>0 direction and 22 Gauss-Laguerre points in the r direction, giving sufficient accuracy for calculations up to N_sh=40
- The basis consists of M_0 = (N_sh+1)(N_sh+2)(N_sh+3)/6 states having the lowest HO energies for given frequencies

## Context

This work extends the configuration-space HFB approach by introducing a new prescription for the THO basis that requires no variational optimization of the local-scale transformation function, unlike earlier implementations by Stoitsov (1998, 1999, 2003). The method enables systematic large-scale studies of nuclear properties across the periodic table, including nuclei near the drip lines, and is intended to be faster than coordinate-space HFB methods such as the two-basis method of Gall (1994) or the basis-spline method of Terasaki (2003).
