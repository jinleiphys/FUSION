# Estimating Elliptic Flow Coefficient in Heavy Ion Collisions using Deep Learning

> Machine Learning (ML) techniques have been employed for the high energy physics (HEP) community since the early 80s to deal with a broad spectrum of problems. This work explores the prospects of using Deep Learning techniques to estimate elliptic flow ($v_2$) in heavy-ion collisions at the RHIC and LHC energies. A novel method is developed to process the input observables from particle kinematic information. The proposed DNN model is trained with Pb-Pb collisions at $\sqrt{s_{\rm NN}} = 5.02$ TeV minimum bias events simulated with AMPT model. The predictions from the ML technique are compared to both simulation and experiment. The Deep Learning model seems to preserve the centrality and energy dependence of $v_2$ for the LHC and RHIC energies. The DNN model is also quite successful in predicting the $p_{\rm T}$ dependence of $v_2$. When subjected to event simulation with additional noise, the proposed DNN model still keeps the robustness and prediction accuracy intact up to a reasonable extent.

## Key claim

The authors claim that a Deep Neural Network (DNN) trained only on minimum bias Pb-Pb collisions at \(\sqrt{s_{\rm NN}} = 5.02\) TeV from the AMPT model can predict the elliptic flow coefficient \(v_2\) for different centrality classes and collision energies at both RHIC and LHC, with the caveat that the model's performance degrades for events with true \(v_2\) values above 0.2 due to limited training statistics in that region.

## Method

The authors simulated minimum bias Pb-Pb collisions at \(\sqrt{s_{\rm NN}} = 5.02\) TeV using the AMPT model in string melting mode (version 2.26t9b) to generate 200K events for training. They constructed input features by binning the \((\eta, \phi)\) space into a \(32 \times 32\) grid for charged hadrons with \(0.2 < p_{\rm T} < 5.0\) GeV/c and \(|\eta| < 0.8\), and added three weighted layers based on transverse momentum, particle mass, and \(\log(\sqrt{s_{\rm NN}}/s_0)\) to produce 3072 input features per event. A feed-forward DNN with three hidden layers of 256 nodes each and an input layer of 128 nodes was trained using the Adam optimizer with mean-squared-error loss and early stopping to predict the event-by-event elliptic flow coefficient \(v_2\).

## Key numbers

* The DNN achieved a mean-absolute-error (MAE) of \(\Delta v_2 = 0.0073\) when predicting \(v_2\) for 10K minimum bias Pb-Pb events at \(\sqrt{s_{\rm NN}} = 5.02\) TeV from AMPT.
* With a \(32 \times 32\) bin input space (3072 input neurons), the testing MAE was 0.0102 for 50K training events and 5K testing events, with 558,209 trainable parameters and 2.684 seconds per epoch.
* The maximum relative error (\(\Delta v_2 / v_2^{\rm true}\)) was less than 6.0% for the region \(v_2^{\rm true} < 0.2\) and between 6-10% for the region \(v_2^{\rm true} > 0.2\) when applying the model to RHIC and LHC energies.
* The model was trained on 200K minimum bias Pb-Pb events at \(\sqrt{s_{\rm NN}} = 5.02\) TeV and validated on 20% of events, with a maximum of 60 epochs and early stopping patience of 10 epochs.

## Context

This work extends the application of machine learning techniques in high-energy physics, which have been used since the early 1980s, to the specific problem of estimating elliptic flow in heavy-ion collisions. The authors note that traditional methods for estimating \(v_2\) include the complex reaction plane identification method (Poskanzer and Voloshin, 1998), the cumulant method (Borghini, Dinh, and Ollitrault, 2000), and the Lee-Yang zeroes method (Bhalerao, 2003), as well as Principal Component Analysis approaches (Bhalerao, 2014; Hippert, 2019; CMS Collaboration, 2017; Gardim, 2019). The proposed DNN approach enables faster estimation of \(v_2\) once trained, and the authors suggest that future work could explore convolutional neural network or PointNet based approaches for handling image-like inputs directly.
