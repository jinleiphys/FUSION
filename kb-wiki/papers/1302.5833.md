# Novel and simple description for a smooth transition from $α$-cluster wave functions to $jj$-coupling shell model wave functions

> We propose an improved version of Antisymmetrized Quasi-Cluster Model (AQCM) to describe a smooth transition from the $α$-cluster wave function to the $jj$-coupling shell model wave function and apply it to the ground state of $^{12}$C. The cluster-shell transition in $^{12}$C is characterized in AQCM by only two parameters: $R$ representing the distance between $α$ clusters and the center of mass, and $Λ$ describing the break of $α$ clusters. The optimal AQCM wave function for the ground state of $^{12}$C is an intermediate state between the three-$α$ cluster state and the shell model state with the $p_{3/2}$ subshell closure configuration. The result is consistent with that of the Antisymmetrized Molecular Dynamics (AMD), and the optimal AQCM wave function quantitatively agrees with the AMD one, although the number of degrees of freedom in AQCM is significantly fewer.

# Novel and simple description for a smooth transition from α-cluster wave functions to jj-coupling shell model wave functions

## Key claim

The improved Antisymmetrized Quasi-Cluster Model (AQCM) can describe a smooth transition from three-α cluster wave functions to the jj-coupling shell model configuration with p3/2 subshell closure using only two parameters (R and Λ), but the optimal ground state of 12C is an intermediate state between these limits rather than either pure configuration, with squared overlaps of 63.6% to the three-α state and 47.3% to the p3/2 subshell closure state.

## Method

The authors place three quasi-α clusters in an equilateral triangular configuration and describe each nucleon with a Gaussian wave packet whose center has a complex parameter ζ = R + iΛ e_spin × R, where R is the distance between clusters and the center of mass, and Λ controls the breaking of α clusters via the spin-orbit interaction. Unlike the previous AQCM version which restricted spin orientations to the z-axis, the improved version rotates both spatial and spin parts of all three quasi-α clusters simultaneously using Euler angles, allowing all clusters to become quasi-clusters. The total wave function is projected onto parity and angular momentum eigenstates, and the Hamiltonian uses the Volkov No.2 central force and the G3RS spin-orbit potential.

## Key numbers

- The minimum of the energy surface for the 0+ state of 12C appears at (R, Λ) = (0.9 fm, 0.2) with energy -89.6 MeV
- Squared overlap between the optimal AQCM wave function and the three-α state (R = 0.9 fm, Λ = 0.0) is 63.6%
- Squared overlap between the optimal AQCM wave function and the p3/2 subshell closure state (R = 0.01 fm, Λ = 1.0) is 47.3%
- The spin-orbit interaction contributes approximately -16 MeV to the total energy at the minimum
- The expectation value of the one-body spin-orbit operator at the optimal point (R = 0.9 fm, Λ = 0.2) is 2.38, intermediate between 0 (cluster limit) and 4 (p3/2 subshell closure limit)
- The squared overlap between the optimal AQCM wave function and the AMD wave function is 98.6%
- The squared overlap between the optimal AQCM wave function and the AMD wave function with threefold rotational symmetry is 99.9%
- The energy of the 0+1 state converges at -91.8 MeV after GCM calculation, close to the experimental value -92.2 MeV
- The three-α cluster threshold energy in the model is -82.9 MeV

## Context

This work extends the AQCM approach previously applied to C, Ne, and Mg isotopes (Suhara, Itagaki, and collaborators, 2010-2012) by removing the restriction that only one α cluster could be converted into a quasi-cluster, which prevented description of subshell closure configurations. The improved AQCM reproduces the results of Antisymmetrized Molecular Dynamics (AMD) calculations for 12C (En'yo, 1998; Suhara, 2010) with far fewer degrees of freedom (2 versus approximately 100). The approach enables quantitative study of cluster-shell competition in light nuclei and can be extended to describe the Hoyle state (0+2) near the α-decay threshold, though the authors note that a closed quantum system framework cannot fully capture the coupling to decay channels important for such near-threshold states.
