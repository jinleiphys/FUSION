# Classification of States in O(8) Proton-Neutron Pairing Model

> Isoscalar (T=0) plus isovector (T=1) pairing hamiltonian in LS-coupling, which is important for heavy N=Z nuclei, is solvable in terms of a O(8) algebra for some special values of the mixing parameter that measures the competition between T=0 and T=1 pairing. The O(8) algebra is generated, amongst others, by the $S=1,T=0$ and $S=0,T=1$ pair creation and annihilation operators . Shell model algebras, with only number conserving operators, that are complementary to the $O(8) \supset O_{ST}(6) \supset O_S(3) \otimes O_T(3)$, $O(8) \supset [ O_S(5) \supset O_S(3) ] \otimes O_T(3)$ and $O(8) \supset [ O_T(5) \supset O_T(3)] \otimes O_S(3)$ sub-algebras are identified. The problem of classification of states for a given number of nucleons (called `plethysm' problem in group theory), for these group chains is solved explicitly for states with O(8) seniority $v=0, 1, 2, 3$ and 4. Using them, band structures in isospin space are identified for states with $v=0, 1, 2$ and 3.

## Key claim

The paper provides a complete classification of many-nucleon states in the O(8) proton-neutron pairing model for O(8) seniority v = 0, 1, 2, 3, and 4, but only for the special mixing parameter values x = 0, 1, and -1 where the pairing Hamiltonian becomes exactly solvable via the O(8) algebra and its three subgroup chains.

## Method

The authors identify three number-conserving shell-model group chains within U(4Ω) that are complementary to the three O(8) subgroup chains: (A) U(4Ω) ⊃ [U(Ω) ⊃ O(Ω)] ⊗ [O_ST(6) ⊃ O_S(3) ⊗ O_T(3)], (B) U(4Ω) ⊃ [U(2Ω) ⊃ Sp(2Ω) ⊃ O(Ω) ⊗ SU_T(2)] ⊗ SU_S(2), and (C) U(4Ω) ⊃ [U(2Ω) ⊃ Sp(2Ω) ⊃ O(Ω) ⊗ SU_S(2)] ⊗ SU_T(2). Using plethysm methods in group theory, they solve the branching rules for irreducible representations of these chains for states with O(8) seniority v = 0 through 4, and identify band structures in isospin space for v = 0, 1, 2, and 3.

## Key numbers

- O(8) seniority quantum numbers v = 0, 1, 2, 3, and 4 are explicitly classified for all three group chains
- For v = 0, the O_ST(6) irreps [P] = [s, 0, 0] are obtained, where s = 0, 2, 4, ..., 2k for m = 4k and s = 1, 3, 5, ..., 2k+1 for m = 4k+2
- For v = 1 with [μ] = [1], the O_ST(6) irreps are [P] = [s-1/2, 1/2, -1/2] + [s+1/2, 1/2, 1/2]
- For v = 2 with [μ] = [2], the O_ST(6) irreps are [P] = [s-1, 0, 0] + [s, 1, 0] + [s+1, 0, 0]
- For v = 2 with [μ] = [1²], the O_ST(6) irreps are [P] = [s-1, 1, -1] + [s, 1, 0] + [s+1, 1, 1]
- For v = 4 with [μ] = [4], the O_ST(6) irrep is [P] = [s, 0, 0]
- For v = 4 with [μ] = [2²], the O_ST(6) irreps are [P] = [s-2, 0, 0] + [s-1, 1, 0] + [s, 2, 0] + [s, 0, 0] + [s+1, 1, 0] + [s+2, 0, 0]

## Context

This work extends the earlier O(8) pairing model results of Flowers and Szpikowski (1964) and subsequent studies by Pang (1969), Evans (1981, 1986), and Kota (2004), which had only classified states for O(8) seniority v = 0 (and partially for v = 1, 2 in chain (i)). The complete classification for v ≤ 4 solves a long-standing problem and provides the basis states needed for shell-model calculations in heavy N=Z nuclei such as ⁶²Ga, ⁶⁶As, ⁷⁰Br, and ⁷⁴Rb, where isoscalar (T=0) and isovector (T=1) pairing competition is important. The results enable future spectroscopic studies of these nuclei using the three symmetry limits of the O(8) model.
