# Improved linear Boltzmann transport model for hadron and jet suppression in ultra-relativistic heavy-ion collisions

> Jets serve as powerful tomographic probes of the quark-gluon plasma (QGP) created in relativistic heavy-ion collisions. While the expanding landscape of jet observables reveals multi-faceted aspects of jet-medium interactions, a precise and simultaneous description of the nuclear modification factors of hadrons and full jets still remains a challenge for theoretical models. In this work, we present two essential improvements to the linear Boltzmann transport (LBT) model to bridge this gap. First, instead of implementing in-medium parton transport after vacuum parton showers complete, we introduce a medium scale at which the parton transport is inserted into the vacuum parton showers, providing a more physical picture of parton-QGP interactions. Second, we incorporate color flow information into the LBT model, enabling string connections between partons whose configurations are correlated with the medium-modified parton showers before hadronization. We demonstrate that both improvements alter the predicted ratio of hadron to jet quenching, leading to a satisfactory description of the nuclear modification factors of hadrons and jets with different flavors within a unified framework.

# Improved linear Boltzmann transport model for hadron and jet suppression in ultra-relativistic heavy-ion collisions

## Key claim

The authors demonstrate that introducing a medium virtuality scale (around 1-2 GeV) at which in-medium parton transport interrupts vacuum parton showers, together with color flow tracking for string hadronization, enables a simultaneous description of hadron and jet nuclear modification factors across different flavors within a single framework, but they explicitly state that the goal is identifying essential physics ingredients rather than precise parameter tuning to data.

## Method

The authors modify the linear Boltzmann transport (LBT) model by inserting a medium scale Q_M into Pythia vacuum showers, so that highly virtual jet partons first evolve via vacuum splittings to Q_M, then scatter with the quark-gluon plasma (QGP) at that scale, and finally resume vacuum showers to the hadronization scale after exiting the medium. They also implement color flow tracking for both elastic (2→2) and inelastic (1→2) scatterings using the large-N_c color scheme, enabling Pythia string fragmentation of medium-modified partons. The QGP medium is modeled using the (3+1)-dimensional CLVisc hydrodynamic model, and the strong coupling α_s is treated as a parameter adjusted to fit jet R_AA data for each choice of Q_M.

## Key numbers

- Medium scale Q_M values tested: 0.5 GeV, 1 GeV, and 2 GeV; Q_M = 0.5 GeV corresponds to earlier LBT versions where partons evolved to hadronization scale Q_h = 0.5 GeV before entering LBT
- For Q_M = 2 GeV, α_s values between 0.34 and 0.44 produce the uncertainty bands shown in the results
- Charm quark mass set to m_c = 1.3 GeV; bottom quark mass set to m_b = 4.2 GeV
- QGP initial time τ_0 = 0.6 fm; hadronization temperature T_pc = 165 MeV
- Non-perturbative K_p factor parameters: amplitude A_p = 5, width σ_p = 5 GeV
- Debye screening mass: μ_D^2 = 4πα_s T^2 (N_c + N_f/2)/3 with N_c = 3
- Jet transport coefficient q̂_a defined as d⟨p_⊥a^2⟩/dt, extracted from elastic scattering rates

## Context

This work extends the LBT model previously developed by the group (Cao 2016, Luo 2023) to address the persistent challenge of simultaneously describing hadron and jet R_AA, which earlier studies (Xing 2019 for hadrons, He 2018 and He 2022 for jets) required different α_s values for each observable. The approach contrasts with the JETSCAPE framework (2024) which still encounters tension between hadron and jet R_AA even with Bayesian calibration, and with the Hybrid model (Casalderrey-Solana 2018) and Lido model (Ke 2020) which achieved simultaneous descriptions through global fitting. The improved model enables future quantitative studies of hadron-triggered jet acoplanarity observed at RHIC (STAR 2025) and the LHC (ALICE 2023), and the source code is publicly available.
