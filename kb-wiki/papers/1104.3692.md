# Finite amplitude method for the quasi-particle-random-phase approximation

> We present the finite amplitude method (FAM) for superfluid systems. A Hartree-Fock-Bogoliubov code may be transformed into a code of the quasi-particle-random-phase approximation (QRPA) with simple modifications. This technique has advantages over the conventional QRPA calculations, such as coding feasibility and computational cost. We perform the fully self-consistent linear-response calculation for a spherical neutron-rich nucleus 174Sn, modifying the HFBRAD code, to demonstrate the accuracy, feasibility, and usefulness of the FAM.

# Finite Amplitude Method for the Quasi-Particle-Random-Phase Approximation

## Key claim

The finite amplitude method (FAM) can be generalized to superfluid systems, allowing a static Hartree-Fock-Bogoliubov (HFB) code to be transformed into a quasi-particle-random-phase approximation (QRPA) code with simple modifications, but the demonstration is limited to spherical nuclei and the method requires careful choice of the finite difference parameter η in the range 10⁻⁵ to 10⁻⁹ to achieve accurate convergence.

## Method

The authors start from the time-dependent HFB equations and take the small-amplitude limit to derive the QRPA linear-response equations. Instead of constructing the full QRPA matrix explicitly, they compute the induced fields δh(ω) and δΔ(±)(ω) using numerical finite differences of the static HFB Hamiltonian evaluated on perturbed wave functions (Uη, Vη) defined in terms of the forward and backward amplitudes X and Y. They solve the resulting linear system iteratively using the generalized conjugate residual (GCR) method, applying the method to the spherical HFBRAD code for the neutron-rich nucleus ¹⁷⁴Sn with the SkM* Skyrme functional and a delta pairing interaction.

## Key numbers

- The two-quasi-particle space for Jπ = 0⁺ in ¹⁷⁴Sn amounts to 12,632 states within the HFB model space (quasi-particle energy cutoff E_qp^c = 200 MeV, maximum angular momenta j_max^n = 21/2 for neutrons and j_max^p = 15/2 for protons).
- The box size used in the HFBRAD calculation is R_box = 20 fm.
- The delta pairing interaction strength is V₀ = -77.5 MeV fm³.
- The calculated energy-weighted strengths summed up to 300 MeV exhaust 99.6% of the theoretical sum-rule value given by (2/m) A ⟨r²⟩.
- For η values between 10⁻⁵ and 10⁻⁹, the GCR iterations converge to accuracy ε < 10⁻⁵; at ω = 4 MeV convergence requires 161 iterations, at ω = 12 MeV it requires 439 iterations, and at ω = 20 MeV it requires 469 iterations.
- A smearing width γ = 1.0 MeV is used to smear the strength function at discrete eigenenergies.

## Context

This work extends the finite amplitude method, originally proposed for the RPA without pairing correlations (Nakatsukasa 2007), to superfluid systems described by the QRPA. It enables fully self-consistent linear-response calculations using existing HFB codes with minor modifications, avoiding the tedious construction of the full QRPA matrix. The authors compare their results for the isoscalar monopole strength in ¹⁷⁴Sn with those of Terasaki (2005), finding good agreement for the first two peaks at approximately 5 and 8.5 MeV, with slight differences in peak heights between 11 and 18 MeV. This work is a first step toward constructing a fast computer program for fully self-consistent and triaxially deformed QRPA calculations.
