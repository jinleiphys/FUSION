# Transvese momentum dependent parton distributions of pion at leading twist

> We calculate the leading twist pion unpolarized transverse momentum distribution $f_1(x,k_T^2)$ and the Boer-Mulders function $h_1^\perp(x,k_T^2)$, using leading Fock-state light front wave functions (LF-LFWFs) based on Dyson-Schwinger and Bethe-Salpeter equations. These DS-BSEs based LF-LFWFs provide dynamically generated s- and p-wave components, which are indispensable in producing chirally odd Boer-Mulders function that has one parton spin flipped. Employing a non-perturbative SU(3) gluon rescattering kernel to treat the gauge link of the Boer-Mulders function, we thus obtain both TMDs at hadronic scale and then evolve them to the scale of $μ^2=4.0$ GeV$^2$. We finally calculate the generalized Boer-Mulders shift and find it to be in agreement with the lattice prediction.

## Key claim

The paper calculates the leading-twist unpolarized transverse momentum distribution and Boer-Mulders function for the pion using light-front wave functions derived from Dyson-Schwinger and Bethe-Salpeter equations, finding that the generalized Boer-Mulders shift agrees with lattice predictions, though the positivity bound is violated at small x and moderate-to-large transverse momentum, indicating limitations of the leading Fock-state truncation in that region.

## Method

The authors construct leading Fock-state light-front wave functions (LF-LFWFs) for the pion from Dyson-Schwinger and Bethe-Salpeter equations (DS-BSEs), which dynamically generate both s-wave and p-wave components. They compute the unpolarized TMD f1(x,kT^2) directly from the LF-LFWFs, and obtain the Boer-Mulders function h1^perp(x,kT^2) by incorporating a non-perturbative SU(3) gluon rescattering kernel (chromodynamic lensing function) to treat the gauge link. The TMDs are calculated at the hadronic scale mu0=0.5 GeV and then evolved to mu^2=4.0 GeV^2 using DGLAP evolution for the collinear PDF and Collins-Soper-Sterman evolution for the TMDs, including Sudakov factors and non-perturbative contributions.

## Key numbers

- Pion mass M_pi = 130 MeV used throughout the calculation
- Initial hadronic scale set to mu0 = 0.5 GeV (mu0^2 = 0.25 GeV^2)
- TMDs evolved to final scale mu^2 = 4.0 GeV^2 (mu = 2.0 GeV) for comparison with lattice QCD
- Non-perturbative Sudakov parameter g2 = 0.13 used in TMD evolution
- Strong coupling constant alpha_s(M_Z^2) = 0.118 adopted for DGLAP evolution
- Parameters for the Dyson-Schwinger gluon propagator: Lambda = 0.71 GeV, a1=1.106, a2=2.324, b1=0.004, b2=3.169
- Color function expansion coefficients for SU(3): c1=5.333, c2=6.222, c3=3.951, c4=1.934, c5=0.680, c6=0.198, c7=0.047, c8=0.00967

## Context

This work extends the authors' previous calculation of the unpolarized pion TMD in the chiral limit (Shi, 2018) to physical pion mass and to the Boer-Mulders function, using DS-BSE-based LF-LFWFs that were also successfully applied to rho and J/psi mesons (Shi, 2021). It contrasts with earlier model calculations of the pion Boer-Mulders function that used perturbative one-gluon exchange approximations, such as the antiquark spectator model (Lu, 2004; Meissner, 2008), light-front constituent quark model (Pasquini, 2014; Wang, 2017), NJL model (Noguera, 2015), and light-front holographic approach (Ahmady, 2019), by employing a non-perturbative SU(3) gluon rescattering kernel following the chromodynamic lensing approach of Gamberg (2009). The results enable comparison with lattice QCD predictions for the generalized Boer-Mulders shift (Engelhardt, 2015) and provide input for phenomenological studies of pion-induced Drell-Yan processes at COMPASS.
