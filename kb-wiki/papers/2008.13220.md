# Shell model structure of proxy-SU(3) pairs of orbitals

> The Nilsson orbitals used in the substitutions occurring in the proxy-SU(3) scheme, which are the orbitals bearing the maximum value of total angular momentum in each shell, have an extremely simple structure in the shell model basis |N l j Omega>, with each Nilsson orbital corresponding to a single shell model eigenvector. This simple structure is valid at all deformations for these orbitals, while in other orbitals it is valid only at small deformations. Nilsson 0[110] pairs are found to correspond to |1 1 1 0> pairs in the spherical shell model basis, paving the way for using the proxy-SU(3) approximation within the shell model.

# Shell model structure of proxy-SU(3) pairs of orbitals

## Key claim

The Nilsson orbitals used in the proxy-SU(3) substitution, specifically those bearing the maximum total angular momentum in each shell, correspond to a single shell model eigenvector in the |N l j Ω⟩ basis, and this one-to-one correspondence holds at all deformations, not only at small deformations as is the case for other orbitals. This means the proxy-SU(3) replacement rule in the Nilsson asymptotic basis (0[110] pairs) translates into a |1 1 1 0⟩ replacement rule in the spherical shell model basis.

## Method

The authors expanded Nilsson orbitals from standard Nilsson model calculations (using the Hamiltonian with harmonic oscillator, spin-orbit, and l² terms) into the spherical shell model basis |N l Λ Σ⟩ and then transformed into the |N l j Ω⟩ basis using Clebsch-Gordan coefficients. They computed expansions for orbitals in the proton 50-82 shell and neutron 82-126 shell at deformation ε=0.22 (and checked ε=0.05 and 0.30), using standard Nilsson parameters κ=0.0637 and μ=0.60 for protons and μ=0.42 for neutrons. They compared the coefficients of 0[110] partner orbitals to identify which pairs exhibit a dominant single-component structure.

## Context

This work extends the proxy-SU(3) scheme introduced by Bonatsos et al. (2017), which had provided parameter-free predictions for collective variables β and γ and a solution to the prolate-over-oblate dominance puzzle. The authors connect proxy-SU(3) to the shell model by showing that the 0[110] pairs used in the proxy substitution correspond to |1 1 1 0⟩ pairs in the spherical shell model basis, enabling direct testing of the proxy-SU(3) approximation within shell model calculations. The 0[110] proton-neutron pairs were first experimentally identified by Cakirli et al. as corresponding to maximum proton-neutron interaction from binding energy differences, and later shown by Karampagia et al. to have large spatial overlaps. This work also relates to the Federman-Pittel mechanism for the onset of deformation, where 0[020] pairs are important at the beginning of mass regions while 0[110] pairs become important further inside each region.
