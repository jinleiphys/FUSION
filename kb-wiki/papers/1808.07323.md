# Study of the double Gamow-Teller transitions using the shell model approach

> The double Gamow-Teller strength distributions in the lightest double beta-decay candidate $^{48}$Ca and its isotope $^{46}$Ca were calculated using the nuclear shell model by applying the single Gamow-Teller operator two times sequentially on the ground state of parent nucleus. The nuclear matrix element of the double Gamow-Teller transition from the ground state to the ground state that goes into the double beta decay calculation was shown as a small fraction of the total transition.

## Key claim

The double Gamow-Teller (DGT) strength distributions in the lightest double beta-decay candidate ⁴⁸Ca and its isotope ⁴⁶Ca were calculated using the nuclear shell model by applying the single Gamow-Teller operator two times sequentially on the ground state of the parent nucleus; the nuclear matrix element of the DGT transition from ground state to ground state that enters double beta decay calculations was shown to be a very tiny fraction (3 × 10⁻⁴) of the total DGT strength.

## Method

The authors used the nuclear shell model code NuShellX@MSU with the FPD6 interaction in the complete fp-model space to obtain wave functions for the initial ground state, intermediate states, and final states. They applied the single Gamow-Teller operator sequentially: first computing all transitions from the parent 0⁺ ground state to all 1⁺ intermediate states, then computing all transitions from those 1⁺ states to each 0⁺ or 2⁺ final state in the daughter nucleus. For ⁴⁶Ti, all 2343 possible J_f = 0⁺ states were included, while for J_f = 2⁺ in ⁴⁶Ti and J_f = 0⁺ in ⁴⁸Ti, calculations were limited to 5000 states out of 9884 and 14177 states respectively, with 500 intermediate states used throughout.

## Key numbers

- Total DGT strength for ⁴⁶Ti J_f = 0⁺: calculated value 223.7, sum rule upper limit ≤ 252
- Total DGT strength for ⁴⁶Ti J_f = 2⁺: calculated value 752.6, sum rule lower limit ≥ 720
- Total DGT strength for ⁴⁸Ti J_f = 0⁺: calculated value 385.0, sum rule upper limit ≤ 432
- DGT strength from ground state to ground state in ⁴⁸Ti: 0.109, which is 3 × 10⁻⁴ of the total strength
- DGT strength from ground state to first excited state in ⁴⁶Ti J_f = 0⁺: 0.201
- DGT strength from ground state to first excited state in ⁴⁶Ti J_f = 2⁺: 0.017
- Average energy of DGT strength in ⁴⁶Ti for J = 0⁺: 21.2 MeV
- Average energy of DGT strength in ⁴⁶Ti for J = 2⁺: 18.0 MeV
- Average energy of DGT strength in ⁴⁸Ti for J = 0⁺: 24.6 MeV

## Context

This work extends earlier DGT calculations for even-A Neon isotopes by Muto (1992) and a recent calculation for ⁴⁸Ca by PhysRevLett.120.142502, both of which used the Lanczos method, whereas the present work uses the shell model code NuShellX@MSU with the FPD6 interaction. The results are compared to DGT sum rules derived by Vogel (1988), PhysRevC.40.936, Muto (1992), and PhysRevC.94.064325, and the authors note that the DGT strength distributions show multiple peaks (at least two, up to four) rather than a single resonance, similar to the structure seen in single Gamow-Teller resonances as described by Goodman (1982). The work aims to provide theoretical DGT strength distributions that can be compared with experimental double charge-exchange reactions using light ions, such as those reported by Cappuzzello (2018) and Ejiri (2017), to help calibrate nuclear matrix elements for double beta decay.
