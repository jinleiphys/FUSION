# Qcombo: A Python Package for Automated Commutator Calculations of Quantum Many-Body Operators

> qcombo is a Python package for the symbolic evaluation of commutators between general quantum many-body operators expressed in normal-ordered form using the generalized Wick theorem. The package provides an automated and systematic framework for generating the corresponding algebraic expressions, significantly reducing the risk of human error in lengthy and complex analytical derivations. It is designed to assist the development and implementation of modern many-body methods in nuclear physics, quantum chemistry, and related fields. The functionality and workflow of the package are demonstrated through an application to the in-medium similarity renormalization group (IMSRG) method, which has been widely used for nuclear ab initio calculations. As a representative example, qcombo is employed to automatically generate the complete set of multi-reference IMSRG flow equations with operators truncated at the normal-ordered three-body level.

## Key claim

The paper presents qcombo, a Python package that automates commutator calculations for normal-ordered many-body operators using the generalized Wick theorem, but the demonstration is limited to generating flow equations for the multi-reference IMSRG method with operators truncated at the normal-ordered three-body level, and the package's correctness depends on the user's ability to specify operator indices and apply appropriate simplification rules.

## Method

The package implements a symbolic framework based on the generalized Wick theorem to evaluate commutators between normal-ordered many-body operators. The workflow consists of five sequential modules: input (specifying operator indices), commutator (applying generalized Wick theorem to generate all contractions), regularization (filtering terms and enforcing canonical index ordering), simplification (exploiting antisymmetry of matrix elements and dummy index renaming), and output (exporting results in LaTeX format and as input for the amc package). The package interfaces with the existing amc package (Tichai 2020) to enable export of expressions in angular-momentum-coupled representation for nuclear structure calculations.

## Key numbers

- The SR-IMSRG(3) method scales as O(N^9) where N denotes the number of single-particle states, compared with O(N^6) scaling for SR-IMSRG(2)
- The package requires Python version >=3.12
- The package depends on sympy and IPython as required additional packages

## Context

This work extends the automated commutator calculation capability needed for advanced many-body methods, specifically targeting the multi-reference IMSRG(3) framework which goes beyond the normal-ordered two-body (NO2B) approximation commonly used in IMSRG applications (Hergert 2016, Stroberg 2019). The package enables systematic derivation of flow equations for MR-IMSRG(3) and MR-CCSDT with full triplet truncations, which have been challenging due to the rapid growth of terms in high-rank commutators. Prior work by Heinz (2021) and Stroberg (2024) developed approximate IMSRG(3) schemes, and this package provides a tool to generate the complete set of equations for the full treatment.
