# Effect of Sigma-beam Asymmetry Data on Fits to Single Pion Photoproduction off Neutron

> We investigate the influence of new GRAAL Sigma-beam asymmetry measurements on the neutron in multipole fits to the single-pion photoproduction database. Results are compared to those found with the addition of a double-polarization quantity associated with the sum rule.

## Key claim

The inclusion of new GRAAL Sigma-beam asymmetry measurements for the neutron significantly alters multipole fits to single-pion photoproduction, particularly for the S11 multipole amplitude, but the effect is conditional on the specific final state and energy range; the new data reduce the chi-squared per data point dramatically but also reveal that previous pi- p measurements provided better constraints than the pi0 n data.

## Method

The authors added 216 Sigma-beam asymmetry measurements for gamma n -> pi0 n (E_gamma = 703-1475 MeV, theta = 53-164 degrees) and 99 Sigma measurements for gamma n -> pi- p (E_gamma = 753-1439 MeV, theta = 33-163 degrees) from the GRAAL collaboration to the existing SAID database for gamma N -> pi N up to E_gamma = 2.7 GeV. They performed new partial-wave analyses (SAID-MA09) including these data and compared the results to the previous SAID-SP09 fit (without GRAAL data) and to MAID2007 predictions, evaluating the impact on multipole amplitudes and on sum-rule integrals (GDH, Baldin, forward spin polarizability) for the neutron.

## Key numbers

- 216 Sigma-beam asymmetries for gamma n -> pi0 n at E_gamma = 703-1475 MeV and theta = 53-164 degrees were added to the database
- 99 Sigma-beam asymmetries for gamma n -> pi- p at E_gamma = 753-1439 MeV and theta = 33-163 degrees were added
- The GRAAL pi0 n contribution doubled the world database for that reaction
- The best fit (MA09) reduced initial chi-squared per data point from 223 (SP09) to 3.1 for pi0 n, and from 89 to 4.9 for pi- p
- For the GDH sum rule (in microbarns): gamma n -> pi- p gave 21/21/20 (MA09/SP09/MAID2007); gamma n -> pi0 n gave -159/-157/-151
- For the Baldin sum rule (in 10^-4 fm^3): gamma n -> pi- p gave 8.4/8.4/8.7; gamma n -> pi0 n gave 4.8/4.8/4.6
- For the forward spin polarizability gamma_0 (in 10^-4 fm^4): gamma n -> pi- p gave 1.4/1.5/1.4; gamma n -> pi0 n gave -1.5/-1.5/-1.5

## Context

This work extends the SAID partial-wave analysis program (Arndt et al., 2006) for pion photoproduction by incorporating new GRAAL polarization data on the neutron, which had been largely missing compared to the extensive proton-target database. The results are compared with the MAID2007 model (Drechsel et al.) and with previous SAID fits (SP09 from Dugger et al., 2009). The improved neutron data enable better determination of isoscalar and isovector electromagnetic couplings of baryon resonances, particularly for states like N(1675)5/2- and N(1720)3/2+ where neutron couplings are poorly known, and allow evaluation of sum rules (GDH, Baldin, forward spin polarizability) for the neutron with single-pion contributions converging by 1 GeV.
