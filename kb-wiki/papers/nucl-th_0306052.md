# Semiclassical treatment of the ground state rotational band and of the isovector angle vibration for deformed superfluid nuclei

> This work presents an older, but still interesting description of a superfluid condensate with two components (protons and neutrons), confined by time-dependent deformed harmonic oscillator potentials either in collective uniform rotation, or opposite angular oscillation.

## Key claim

This paper presents a semiclassical, mainly analytic model for describing both the ground-state rotational band and the isovector angle vibration (giant angle dipole resonance) in deformed superfluid nuclei, using the cranked anisotropic harmonic oscillator and a pairing interaction defined via the time-reversal operator. The model's validity is conditional on the adiabatic approximation and the restriction to a small number of valence nucleons (10–12) treated with Hartree-Fock-Bogolyubov (HFB) coherent states, while the remaining nucleons are treated in a Hartree-Fock (HF) state.

## Method

The single-particle Hamiltonian is taken as the cranked anisotropic harmonic oscillator, which is diagonalized analytically. The pairing interaction is defined in a basis-independent form using the time-reversal operator, and the many-body ground state is constructed as an SO(2n) coherent state (HFB trial wave function) parameterized by an antisymmetric complex matrix Z. Variational equations are solved for the matrix t (real and symmetric) separately for protons and neutrons, yielding ω-dependent intrinsic states; the ground-state rotational band is then obtained by imposing equal angular frequencies for protons and neutrons and quantizing the total angular momentum via the semiclassical condition ⟨L_xp⟩ + ⟨L_xn⟩ = √[I(I+1)]. For the isovector angle vibration, a classical Lagrange function with a restoring force between proton and neutron ellipsoids is introduced, and the time-dependent wave function is quantized by requiring periodicity of the phase, leading to a stationary state via Fourier projection.

## Key numbers

- For ¹⁵⁶Gd: deformation δ = 0.239 (from Lamm); pairing constant G = 23/A MeV; ω_sp = 39.8 A^(−1/3) MeV, ω_sn = 44.8 A^(−1/3) MeV.
- For ¹⁵⁸Gd: δ = 0.250; for ¹⁶⁸Er: δ = 0.273.
- For ¹⁵⁶Gd (HFB): proton angular momentum L_p(ω) = 19.8 ω + 20 ω² (ħ); neutron L_n(ω) = 33.7 ω + 6.2 ω² (ħ); proton energy ε_p(ω) = (37.9/2) ω² + 0.31|ω| (MeV); neutron ε_n(ω) = (44.1/2) ω² + 0.54|ω| (MeV).
- For ¹⁵⁶Gd (HF): L_p(ω) = 34.4 ω (ħ); L_n(ω) = 43.2 ω (ħ); ε_p(ω) = (34.2/2) ω² (MeV); ε_n(ω) = (43.2/2) ω² (MeV).
- For the isovector angle vibration in ¹⁵⁶Gd: B(M1)_iv = 1.87 μ_N²; B(M1)_is = 0.03 μ_N²; oscillation amplitudes a_n = 7.33 deg, a_p = 8.57 deg; elastic constant C = 79 MeV; ω₀ = 0.55 MeV; E_AV = 1.57 Ω (with Ω determined by fitting the experimental dominant line at 3.1 MeV).
- Limit of small amplitude: B(M1)_iv → 1.66 μ_N², B(M1)_is → 0.009 μ_N².
- Experimental total strength of 1⁺ fragments: B(M1)_iv,exp = 2.3 ± 0.5 μ_N² (Palumbo, 1980; Richter, 1984).

## Context

This work extends the cranking model treatment of pairing in rotating nuclei, following earlier approaches by the author (1988) and others (e.g., the use of the Nilsson basis with Coriolis terms in refs. [3]–[5]). It provides a semiclassical microscopic description of the isovector angle vibration (giant dipole resonance), which was initially proposed as two ellipsoids in opposite angular oscillation (ref. [17]) and later criticized (refs. [18], [19]) but revived in the classical limit of the IBA-2 model (ref. [22]). The model enables calculation of B(M1) strengths for the 1⁺ state in rare-earth nuclei and suggests a framework for further studies of pairing rotation and vibration modes, including pair transfer (ref. [23]).
