# An Update of B(E2) Evaluation for 0+1 -> 2+1 Transitions in Even-Even Nuclei near N~Z~28

> An update of B(E2) evaluation for even-even Cr, Fe, Ni and Zn nuclei has been presented. Current update is a continuation of S. Raman work on B(E2) values and was motivated by large numbers of new measurements. It extends the previous evaluation from 20 to 38 nuclei and includes comprehensive shell model analysis. Evaluation policies for analysis of experimental data have been discussed. Future plans for complete B(E2;0+1 -> 2+1) evaluation of even-even nuclei are outlined.

## Key claim

This evaluation updates the B(E2) values for 0+1 to 2+1 transitions in even-even Cr, Fe, Ni, and Zn nuclei from 20 to 38 nuclei, but the authors caution that model-independent or traditional measurement types are the most reliable, with combined and model-dependent values used only when no other data exist.

## Method

The authors compiled experimental B(E2) values, lifetimes, and deformation parameters from 114 primary and 13 secondary references published before April 2011, using the Nuclear Science References, Evaluated Nuclear Structure Data File, and Experimental Unevaluated Nuclear Data List databases. They converted all values to B(E2) in e2b2, applied a minimum 5% uncertainty to older results, and used the AveTools software package with limitation of relative statistical weight, normalized residual, and Rajeval technique statistical methods to produce recommended values. Shell-model calculations were performed using the GXPF1A effective interaction in the pf-shell valence space for lighter isotopes and the JUN45 effective interaction in the f5/2, p3/2, p1/2, g9/2 valence space for heavier Ni and Zn isotopes.

## Key numbers

- 18 new recommended values were added for 46,56,58,60,62Cr, 50,52,62,64,66Fe, 54,70,74Ni, and 72,74,76,78,80Zn
- B(E2) value for 46Cr: 0.093(20) e2b2 from intermediate-energy Coulomb excitation
- B(E2) value for 50Fe: 0.140(30) e2b2 from intermediate-energy Coulomb excitation
- B(E2) value for 54Ni: 0.059(17) e2b2 from intermediate-energy Coulomb excitation
- B(E2) value for 52Fe: 0.082(10) e2b2 from intermediate-energy Coulomb excitation at Michigan State University
- B(E2) value for 70Ni: 0.086(14) e2b2 from Coulomb excitation on a 208Pb target at intermediate energy
- B(E2) value for 72Zn: 0.174(21) e2b2 from Coulomb excitation at intermediate energy
- B(E2) values for 76,78,80Zn: 0.145(18), 0.077(19), and 0.073(9) e2b2 from low-energy Coulomb excitation at REX-ISOLDE
- Lifetimes for 62,64Fe: 7.4(9) and 7.4(26) ps from recoil-distance Doppler shift method at GANIL
- Lifetimes for 62,64,66Fe: 8.0(10), 10.3(10), and 39.0(40) ps from MSU measurements using plunger method
- Lifetime for 74Zn: 27.6(43) ps from recoil distance Doppler-shift measurement at GANIL

## Context

This work continues the B(E2) evaluation program originally established by Raman (1987, 2001) at the Oak Ridge Nuclear Data Project, extending the previous evaluation from 20 to 38 nuclei in the N approximately Z approximately 28 region. The update was motivated by new measurements from rare isotope facilities such as the RISING collaboration, GANIL, MSU, and REX-ISOLDE, and includes comprehensive shell-model analysis using GXPF1A and JUN45 effective interactions. The results enable future complete B(E2) evaluation for all even-even nuclei and provide benchmark data for nuclear model calculations, with the authors planning to extend the evaluation to other Z regions based on user feedback.
