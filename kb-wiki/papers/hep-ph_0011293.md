# The S-wave Λπphase shift is not large

> We study the strong interaction S-wave Λπphase shift in the region of the Ξmass in the framework of a relativistic chiral unitary approach based on coupled channels. All parameters have been previously determined in a fit to strangeness S= -1 S-wave kaon-nucleon data. We find 0^\circ \le δ_0 \le 1.1^\circ in agreement with previous chiral perturbation theory calculations (or extensions thereof). We also discuss why a recent coupled channel K-matrix calculation gives a result for δ_0 that is negative and much bigger in magnitude. We argue why that value should not be trusted.

## Key claim

The S-wave Λπ phase shift at the Ξ mass is small and positive, bounded between 0° and 1.1°, contradicting a recent K-matrix calculation that found a large negative value near -7°; the authors argue that the K-matrix result is unreliable because it used an insufficient approximation for the scalar loop function.

## Method

The authors used a relativistic chiral unitary approach based on coupled channels, where the inverse of the partial wave T-matrix satisfies a dispersion relation that resums the lowest-order chiral perturbation theory (CHPT) amplitudes. The input parameters (baryon octet mass in the chiral limit m0, three-flavor meson decay constant F0, and a subtraction constant a(μ)) were previously fixed by fitting to strangeness S = -1 S-wave kaon-nucleon scattering data, including cross sections, threshold ratios, and the Λ(1405) mass distribution. The Λπ → Λπ amplitude was then projected out from the coupled-channel solutions to extract the S-wave phase shift δ0 at the Ξ mass.

## Key numbers

- For parameter set I (best fit): δ0(mΞ0) = 0.10°, δ0(mΞ-) = 0.16°
- For parameter set II (natural values): δ0(mΞ0) = 0.92°, δ0(mΞ-) = 1.11°
- The overall bound from the calculation: 0° ≤ δ0 ≤ 1.1°
- Set I parameters: m0 = 1.286 GeV, F0 = 74.1 MeV, a(μ) = -2.23 at scale μ = 630 MeV
- Set II parameters: m0 = 1.151 GeV, F0 = 86.4 MeV, a(μ) = -2 at scale μ = 630 MeV

## Context

This work extends earlier chiral perturbation theory calculations by Liu, Wise (1994) and Donoghue, Pakvasa (1995) that found δ0 ~ 0.5°, and contradicts the K-matrix calculation of Tandean, Tibbetts, Valencia (2000) which reported δ0 ≈ -7°. The authors show that the K-matrix result arises from using only the imaginary part of the scalar loop function (the approximation g(s)i = -i ρ(s)i) instead of the full relativistic expression, and that a proper dispersion relation treatment is essential. The result enables reliable extraction of CP-violating phases from Ξ → Λπ → pππ decays, as the small S-wave phase shift is now precisely bounded.
