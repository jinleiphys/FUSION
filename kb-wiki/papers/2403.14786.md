# Strong Coupling of Hydrodynamics and Reactions in Nuclear Statistical Equilibrium for Modeling Convection in Massive Stars

> We build on the simplified spectral deferred corrections (SDC) coupling of hydrodynamics and reactions to handle the case of nuclear statistical equilibrium (NSE) and electron/positron captures/decays in the cores of massive stars. Our approach blends a traditional reaction network on the grid with a tabulated NSE state from a very large, O(100) nuclei, network. We demonstrate how to achieve second-order accuracy in the simplified-SDC framework when coupling NSE to hydrodynamics, with the ability to evolve the star on the hydrodynamics timestep. We discuss the application of this method to convection in massive stars leading up to core-collapse. We also show how to initialize the initial convective state from a 1D model in a self-consistent fashion. All of these developments are done in the publicly available Castro simulation code and the entire simulation methodology is fully GPU accelerated.

## Key claim

The authors achieve second-order temporal accuracy when coupling hydrodynamics with nuclear statistical equilibrium (NSE) using a simplified spectral deferred corrections (SDC) framework, but only when using tricubic interpolation of the NSE table; the method allows evolving massive star cores on the hydrodynamics timestep rather than restricting the timestep based on energy release from reactions.

## Method

The method blends a 19-isotope reaction network (based on aprox19) advected on the grid with a tabulated NSE state from a 96-nuclei network, using the simplified-SDC iterative coupling approach from Zingale (2022). Within each SDC iteration, the advective update uses CTU-PPM with interface states that include a reactive source term, then the system is evolved as an ODE using VODE, with the NSE table providing weak-rate evolution of electron fraction, average binding energy per nucleon, and neutrino losses. The NSE condition is determined by heuristic thresholds on density, temperature, helium-plus-iron-group mass fraction, and silicon mass fraction, and a Newton-Raphson procedure (nse_eos) inverts the equation of state to find temperature consistent with NSE composition.

## Key numbers

- NSE table contains 96 nuclei, mapped to the 19 on-grid isotopes
- NSE condition thresholds: density > 10^7 g cm^-3, temperature > 3 x 10^9 K, helium + iron-group mass fraction > 0.88, silicon mass fraction < 0.02
- Maximum silicon mass fraction in NSE table is 0.012
- Newton-Raphson tolerance for temperature inversion: 10^-6
- Two SDC iterations required for second-order temporal convergence

## Context

This work extends the simplified-SDC coupling method of Zingale (2022) to handle NSE and weak interactions in massive star cores, building on the NSE table approach of Seitenzahl (2009) and the composition advection method of Ma (2013). Prior multidimensional simulations of massive star convection (e.g., Fields 2020, 2021; Yoshida 2021) used smaller networks like aprox21 with operator splitting and reaction-based timestep constraints, or cut out the iron core entirely. The method enables self-consistent evolution of the core on the hydrodynamics timestep in the Castro code, with full GPU acceleration, and is designed for application to convection in massive stars leading to core-collapse.
