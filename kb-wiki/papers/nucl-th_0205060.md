# Absence of anomalous stopping in heavy ion collisions

> We show that the baryon stopping observed in heavy ion collisions both at CERN-SPS and at RHIC can be derived from the one observed in proton-proton collisions. No increase in the size of the baryon junction component is required between small size $(pp)$ and large size (AA) systems.

## Key claim

The baryon stopping observed in heavy ion collisions at both CERN-SPS and RHIC can be reproduced from proton-proton data using the same strength of the string junction component, with no need to increase the coupling of that component when going from small (pp) to large (AA) systems.

## Method

The author constructs a two-component model for net baryon production: a diquark-preserving (DP) component from fragmentation of a valence diquark, and a diquark-breaking (DB) component where the string junction follows a valence quark. The rapidity distributions are derived from Regge intercepts, with the DB component using intercept α_{SJ+q_v}(0) = 1/2 and the DP component using α_{SJ+(qq)_v}(0) = -1. The model is applied to pp, pA, and AA collisions using the same free parameter a = 0.4 (the fraction of DB component) determined from pp data, with the number of inelastic collisions per participant scaled appropriately for nuclear systems.

## Key numbers

- In pp collisions at √s = 17.2 GeV, the calculated dN/dy for p - p̅ at y* = 0 is 9.2 (scaled by n_A = 175); at y* = 2 it is 47.1, compared to data 45.4 ± 1.4.
- In pp collisions at √s = 27.4 GeV, the calculated dN/dy for p - p̅ at y* = 1 is 9.3, compared to data 9.6 ± 0.9.
- In central Pb Pb collisions at √s = 17.2 GeV, the calculated dN/dy for p - p̅ at y* = 0 is 23.0, compared to data 26.7 ± 3.7; for B - B̅ at y* = 0 it is 58.5, compared to data 67.7 ± 7.3.
- In central Au Au collisions at √s = 130 GeV, the calculated dN/dy for p - p̅ at y* = 0 is 8.0, compared to data 5.6 ± 0.9 ± 24%; for B - B̅ at y* = 0 it is 20.9.
- The DB component fraction a = 0.4 is used for all systems; the strangeness suppression factor S = 0.1 (S/L = 0.22).

## Context

This work extends the Dual Parton Model (DPM) and Quark Gluon String Model (QGSM) frameworks for net baryon production, directly addressing the claim by several authors (Kopeliovich and others, 1990s-2000s) that an anomalous increase in the baryon junction component was needed for heavy ion collisions. The paper contradicts the findings of Shabelski (2002) and earlier models by Kopeliovich et al. that suggested anomalous stopping. It enables a unified description of pp and AA net baryon production without invoking new physics for large systems, and provides a baseline for future studies of final state interactions that modify strange baryon yields.
