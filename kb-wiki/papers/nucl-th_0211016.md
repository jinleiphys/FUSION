# Scattering Calculations with Wavelets

> We show that the use of wavelet bases for solving the momentum-space scattering integral equation leads to sparse matrices which can simplify the solution. Wavelet bases are applied to calculate the K-matrix for nucleon-nucleon scattering with the s-wave Malfliet-Tjon V potential. We introduce a new method, which uses special properties of the wavelets, for evaluating the singular part of the integral. Analysis of this test problem indicates that a significant reduction in computational size can be achieved for realistic few-body scattering problems.

## Key claim

The paper demonstrates that Daubechies wavelets can produce sparse matrix representations of momentum-space scattering integral equations, but the authors hedge that this advantage is demonstrated only for a test problem (s-wave Malfliet-Tjon V potential) and that "analysis of this test problem indicates that a significant reduction in computational size can be achieved for realistic few-body scattering problems", the claim is conditional on extrapolation from this single test case.

## Method

The authors solve the momentum-space K-matrix integral equation for nucleon-nucleon scattering using the s-wave Malfliet-Tjon V potential, transforming the half-infinite momentum interval to a finite interval with the singularity mapped to the origin. They expand the unknown function in a Daubechies scaling basis on a fine scale, compute matrix elements using a one-point quadrature rule (which exploits the property that scaling functions integrate quadratic polynomials exactly), and treat the singular principal-value integral using a new method based on the scaling equation to relate singular integrals to known moments and partial moments. The resulting linear system is solved in the scaling basis representation.

## Key numbers

- The Malfliet-Tjon V potential parameters: $\lambda_1 = -570.316$ MeV fm, $\mu_1 = 1.55$ fm$^{-1}$, $\lambda_2 = 1438.4812$ MeV fm, $\mu_2 = 3.11$ fm$^{-1}$, with $1/m = 41.47$ MeV fm$^2$
- Test calculations are performed for half on-shell K-matrix at on-shell energies $p_0^2/m = 10$ MeV and $80$ MeV
- For the Daubechies K=2 wavelet, singular integrals $I^{\pm}_{-1} = -2.779949550280 \mp i 4.291495373146$ and $I^{\pm}_{-2} = -0.269952669589 \pm i 1.149902719556$
- For the Daubechies K=3 wavelet, singular integrals include $I^{\pm}_{-1} = -0.1717835441734 \mp i 4.041140804162$, $I^{\pm}_{-2} = -1.7516314066967 \pm i 1.212142562305$, $I^{\pm}_{-3} = -0.3025942645356 \mp i 0.299291822651$, and $I^{\pm}_{-4} = -0.3076858066180 \mp i 0.013302589081$

## Context

This work extends the application of Daubechies wavelets (1988) from data compression problems (FBI fingerprint archive, JPEG files) to nuclear scattering calculations, specifically addressing the challenge that conventional methods produce dense matrix representations of momentum-space integral equations. The authors introduce a new technique for evaluating singular integrals using the scaling equation, which they note is a key element not present in prior wavelet treatments of integral equations by Shann and others. The method enables future application to relativistic few-body scattering problems, which are naturally formulated in momentum space where the sparsification advantage is most significant.
