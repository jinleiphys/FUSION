# JAM small-$x$ helicity phenomenology

> We present the first-ever description the world data on the $g_1^{p,n}$ structure function at small Bjorken $x$ using evolution equations in $x$ derived from first principles QCD. This is a Monte-Carlo analysis within the JAM global framework that allows us to fit all existing polarized DIS data below $x<0.1$ as well as predict future measurements of small $x$ $g_1^{p,n}$ at the EIC. This is a necessary step in determining the quark helicity PDFs and, ultimately, the quark contribution to the proton spin.

# JAM small-$x$ helicity phenomenology

## Key claim
The paper presents the first description of world polarized deep-inelastic scattering (DIS) data on the $g_1^{p,n}$ structure functions at small Bjorken $x$ using evolution equations derived from first-principles QCD (KPS evolution), but only for data with $x<0.1$ and $Q^2>1.69$ GeV$^2$, and the extraction of individual helicity parton distribution functions (hPDFs) requires either an artificial constraint ($\Delta s^+=0$) or future parity-violating DIS data from the Electron-Ion Collider (EIC).

## Method
The authors implement the double-logarithmic-approximation (DLA) KPS evolution equations for the polarized dipole amplitude $G_q$ in the large-$N_c$ limit, using a Born-inspired initial condition $G_q^{(0)}(s_{10},\eta)=a_q\eta+b_q s_{10}+c_q$ with flavor-dependent parameters. They fit the $g_1^{p,n}$ structure functions (extracted from double-spin asymmetries $A_{||}$ via $A_1\propto g_1/F_1$, with $F_1$ taken from the JAM global analysis) to existing polarized DIS data from SLAC, EMC, SMC, COMPASS, and HERMES using a Monte-Carlo Bayesian framework within the JAM global analysis infrastructure.

## Key numbers
- The fit achieves $\chi^2/N_{\rm pts}=1.01$ for the description of longitudinal double-spin asymmetry data on proton, deuteron, and $^3$He targets at $x<0.1$ and $Q^2\in[1.73,19.70]$ GeV$^2$.
- The strong coupling is fixed at $\alpha_s=0.3$, a typical value for the $Q^2$ range studied.
- The formalism applies for $x<0.1$ (compared to $x<0.01$ for unpolarized small-$x$ evolution), with data restricted to $Q^2>m_c^2\approx1.69$ GeV$^2$ and $s>4$ GeV$^2$.
- Predictions for $g_1^p$ are extended down to $x=10^{-5}$.

## Context
This work extends the KPS evolution framework developed by Kovchegov, Pitonyak, Sievert, and collaborators (2015-2019) to a global phenomenological analysis of existing polarized DIS data, which had previously only been described by DGLAP-based parameterizations such as DSSV (de Florian et al., 2014, 2019). The analysis demonstrates that first-principles small-$x$ evolution can describe existing data and provides predictions for future EIC measurements, enabling a path toward determining the quark helicity contribution to the proton spin via the Jaffe-Manohar sum rule.
