# Multi-boson effects and the normalization of the two-pion correlation function

> The two-pion correlation function can be defined as a ratio of either the measured momentum distributions or the normalized momentum space probabilities. We show that the first alternative avoids certain ambiguities since then the normalization of the two-pion correlator contains important information on the multiplicity distribution of the event ensemble which is lost in the second alternative. We illustrate this explicitly for specific classes of event ensembles.

## Key claim

The two-pion correlation function defined as a ratio of measured momentum distributions (definition I) retains information about the pion multiplicity distribution through its normalization, whereas the alternative definition using normalized momentum-space probabilities (definition II) discards this information; the authors show this explicitly for specific classes of event ensembles, including a model where multi-boson symmetrization effects cause the two definitions to differ dramatically.

## Method

The authors compare two definitions of the two-pion correlation function: \(C^I = N_2/(N_1 N_1)\) based on inclusive distributions normalized to factorial moments, and \(C^{II} = P_2/(P_1 P_1)\) based on normalized probabilities. They analyze a class of model ensembles using density matrices that incorporate multi-pion Bose-Einstein symmetrization effects, with a Gaussian source emission function \(g(x,\vec{p})\) characterized by spatial width \(R\) and momentum width \(\Delta\). They compute the single-pion spectrum, spatial distribution, multiplicity distribution, and the normalization factor \(\langle n \rangle^2 / \langle n(n-1) \rangle\) as functions of the average pion phase-space density \(d = \langle n \rangle / (2R\Delta)^3\).

## Key numbers

- For the \(m=2\) state (non-Poissonian multiplicity distribution), \(C^I(\vec{p}_1,\vec{p}_2) = 2 + 1/(4n_0^2)\) while \(C^{II}(\vec{p}_1,\vec{p}_2) = 1\).
- In the high-density limit (\(d \to \infty\)), the multiplicity distribution becomes Bose-Einstein: \(P(n) = \langle n \rangle^n / (\langle n \rangle + 1)^{n+1}\), with \(C^I = 2\) and \(C^{II} = 1\).
- In the high-density limit, the effective momentum width becomes \(\Delta_{\rm eff}^2 = \Delta / (2R) \leq \Delta^2\).
- For the model with Gaussian source parameters \(R=3\) fm and \(\Delta=140\) MeV, the ratio \(\langle n \rangle^2 / \langle n(n-1) \rangle\) decreases from near 1 at low phase-space density to 0.5 as \(d \to \infty\).
- The critical input multiplicity for the Bose condensation limit is \(n_c = (R\Delta + 1/2)^3 \geq 1\).
- For small systems with \(2R\Delta = 1\), even in the limit \(n_0 \to 0\), \(\langle n(n-1) \rangle / \langle n \rangle^2 = 1 + 1/(2R\Delta)^3 \leq 2\).

## Context

This work extends the argument of Miskowiec and Voloshin (1997) that definition I of the correlation function is preferable because it is based directly on measured quantities and consistent with the standard theoretical expression involving the source emission function. The authors show that definition I additionally provides information about the pion multiplicity distribution that is lost in definition II, and that this can be used to search for multi-pion symmetrization effects. The model class studied here was previously developed by Pratt (1993), Csorgo and collaborators, and Zhang and collaborators; the results enable experimental extraction of the average pion phase-space density from the normalization of the correlation function, which can serve as an alternative method for detecting strong multi-boson effects in high-multiplicity hadronic and heavy-ion collisions.
