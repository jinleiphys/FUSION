# Gradient Symplectic Algorithms for Solving the Schroedinger Equation with Time-Dependent Potentials

> We show that the method of factorizing the evolution operator to fourth order with purely positive coefficients, in conjunction with Suzuki's method of implementing time-ordering of operators, produces a new class of powerful algorithms for solving the Schroedinger equation with time-dependent potentials. When applied to the Walker-Preston model of a diatomic molecule in a strong laser field, these algorithms can have fourth order error coefficients that are three orders of magnitude smaller than the Forest-Ruth algorithm using the same number of Fast Fourier Transforms. When compared to the second order split-operator method, some of these algorithms can achieve comparable convergent accuracy at step sizes 50 times as large. Morever, we show that these algorithms belong to a one-parameter family of algorithms, and that the parameter can be further optimized for specific applications.

## Key claim
The paper claims that a new class of fourth-order symplectic algorithms, which use the gradient of the potential and purely positive factorization coefficients, can solve the time-dependent Schroedinger equation with far greater efficiency than existing fourth-order algorithms with negative coefficients. The hedge is that this efficiency is demonstrated specifically for the Walker-Preston model of a diatomic molecule in a strong laser field, and the algorithms belong to a one-parameter family that can be further optimized for specific applications.

## Method
The authors factorize the time-evolution operator using Suzuki's method of implementing time-ordering via a forward time derivative operator, then apply gradient symplectic factorization schemes that include the double commutator [V,[T,V]] (the square of the gradient of the potential). They transcribe four previously derived gradient algorithms (4A, 4B, 4C, 4D) to time-dependent form and test them on the one-dimensional Walker-Preston model, discretizing the wave function on 64 uniform points with spacing 0.08 atomic units. The algorithms are compared by fitting convergence errors to monomials in (Delta t)^2 or (Delta t)^4 and by computing normalized equal-effort error coefficients.

## Key numbers
- Algorithm 4A requires only 4 Fast Fourier Transforms (FFT) per iteration, twice the computational effort of the second-order split-operator method.
- For the Walker-Preston model, the Forest-Ruth algorithm's convergence error fits 9.7 x 10^-4 (Delta t)^4; algorithm 4A fits 4.8 x 10^-6 (Delta t)^4.
- The second-order split-operator method (SO) has error coefficient b_SO = 1.1; the projected equal-effort curve SO' has coefficient b_SO/9 = 0.122.
- The natural time scale of the laser oscillation is tau = 2 pi / omega = 351.6 atomic units; the convergent range for SO is below Delta t < tau/100 = 3.52 atomic units.
- The converged total energy at 1000 tau is E_con = 5.029155 E_0, where E_0 is the Morse oscillator ground state energy.
- The optimized algorithm 4ACB (t_0 = 0.144) achieves an effective time step factor tau_eff = 8.5 relative to Forest-Ruth; algorithm 4BDA (t_1 = 0.35) achieves tau_eff = 8.4.
- Both 4ACB and 4BDA achieve 10^-4 accuracy in total energy at a time step size 50 times as large as the equal-effort second-order method SO'.

## Context
This work extends the authors' prior development of gradient symplectic algorithms for classical and quantum dynamical problems (Chin 1997, Chin and Chen 2002) by incorporating Suzuki's method for time-ordering. It directly competes with the Forest-Ruth algorithm (Forest 1990) and McLachlan's algorithm (McLachlan 1995), both fourth-order algorithms with negative coefficients, and with the second-order split-operator method (Feit 1982, Gray and Verosky 1994, Blanes 1999). The Walker-Preston model (Walker and Preston 1990) is used as a standard test problem, previously employed by Sanz-Serna and Portillo (2000) and others. The work enables long-time quantum simulations on large 3D meshes (e.g., 256^3 grid points) where FFT-based algorithms are essential, and it opens the question of whether sixth-order positive-coefficient factorization schemes exist.
