# Learning Hadron Emitting Sources with Deep Neural Networks

> The correlation function observed in high-energy collision experiments encodes critical information about the emitted source and hadronic interactions. While the proton-proton interaction potential is well constrained by nucleon-nucleon scattering data, these measurements offer a unique avenue to investigate the proton-emitting source, reflecting the dynamical properties of the collisions. In this Letter, we present an unbiased approach to reconstruct proton-emitting sources from experimental correlation functions. Within an automatic differentiation framework, we parameterize the source functions with deep neural networks, to compute correlation functions. This approach achieves a lower chi-squared value compared to conventional Gaussian source functions and captures the long-tail behavior, in qualitative agreement with simulation predictions.

## Key claim
The authors present an unbiased, data-driven method to reconstruct proton-emitting source functions from experimental correlation functions using deep neural networks, achieving a lower chi-squared value than conventional Gaussian source functions and capturing the long-tail behavior of the source. The hedge is that the approach assumes an isotropic source approximation for proton-proton collisions and relies on the proton-proton interaction potential being already well-constrained by scattering data.

## Method
The authors solve the Schrödinger equation with the Reid and Av-18 nucleon-nucleon potentials using the CATS framework to obtain proton-proton scattering wavefunctions for spin-singlet and spin-triplet states. They parameterize the source function using a deep neural network with 4 layers and width 64, trained via automatic differentiation to minimize the chi-squared loss between the computed correlation function (convolution of source with scattering wavefunctions) and experimental data from ALICE at √s = 7 TeV. The Softplus activation function on the output layer enforces positive definiteness of the source function.

## Key numbers
- Gaussian source fit to ALICE pp data at √s = 7 TeV yields r₀ = 1.32 fm with χ² = 90.72 for the Reid potential, and r₀ = 1.26 fm with χ² = 89.12 for the Av-18 potential
- Neural network source achieves χ² = 34.55 for the Reid potential and χ² = 35.81 for the Av-18 potential, significantly lower than the Gaussian fits
- The neural network uses a default architecture of width = 64 and L = 4 layers, trained with Adam optimizer at learning rate 10⁻³ over 10,000 epochs
- The scattering wavefunction expansion uses l_max = 3, with convergence verified

## Context
This work extends femtoscopy methodology beyond the Lednický and Lyuboshits model (Lednicky 1981) and the common Gaussian source assumption, which prior studies (ALICE 2018, Mihaylov 2018) have shown to be inadequate due to resonance contributions. The extracted non-Gaussian source with long-tail behavior qualitatively agrees with simulation predictions from Mihaylov (2018) and ALICE (2023), and this is the first data-driven extraction of a hadron-emitting source from experimental measurements. The method enables more precise probing of hadronic interactions for other particle pairs, demonstrated here for p-Λ correlations using the Usmani potential form, and can be extended to three-dimensional source extraction in heavy-ion collisions.
