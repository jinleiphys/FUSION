# Source Function Determined from HBT Correlations by the Maximum Entropy Principle

> We study the reconstruction of the source function in space-time directly from the measured HBT correlation function using the Maximum Entropy Principle. We find that the problem is ill-defined without at least one additional theoretical constraint as input. Using the requirement of a finite source lifetime for the latter we find a new Gaussian parametrization of the source function directly in terms of the measured HBT radius parameters and its lifetime, where the latter is a free parameter which is not directly measurable by HBT. We discuss the implications of our results for the remaining freedom in building source models consistent with a given set of measured HBT radius parameters.

## Key claim

The reconstruction of the source function in space-time from measured HBT correlation functions using the Maximum Entropy Principle is ill-defined without at least one additional theoretical constraint; requiring a finite source lifetime yields a well-defined Gaussian source parametrization, but the lifetime parameter itself remains a free parameter not directly measurable by HBT.

## Method

The authors applied the Maximum Entropy Principle to the problem of reconstructing the four-dimensional source function from HBT correlation data, using the measured HBT radius parameters as constraints. They demonstrated that with only the experimentally accessible constraints from standard or Yano-Koonin-Podgoretskii parametrizations, the entropy maximization yields a source with divergent lifetime and normalization. They then added the theoretical constraint of a finite source lifetime (specifying the variance of the emission time distribution) and derived the resulting Gaussian source function analytically in terms of the measured HBT radii and the lifetime parameter.

## Key numbers

- The standard HBT radius parameters are related to source variances as: \(R_s^2 = \langle \tilde y^2\rangle\), \(R_o^2 = \langle (\tilde x-\beta_o \tilde t)^2\rangle\), \(R_l^2 = \langle (\tilde z-\beta_l \tilde t)^2\rangle\), \(R_{lo}^2 = \langle (\tilde x-\beta_o \tilde t)(\tilde z-\beta_l \tilde t)\rangle\)
- With the finite lifetime constraint \(\langle \tilde t^2\rangle = T^2(K)\), the resulting source variances become: \(\langle \tilde x^2\rangle = R_o^2 + \beta_\perp^2 T^2\), \(\langle \tilde y^2\rangle = R_s^2\), \(\langle \tilde z^2\rangle = R_l^2 + \beta_l^2 T^2\), \(\langle \tilde t^2\rangle = T^2\), \(\langle \tilde z\tilde t\rangle = \beta_l T^2\), \(\langle \tilde x\tilde t\rangle = \beta_\perp T^2\), \(\langle \tilde x\tilde z\rangle = R_{ol}^2 + \beta_\perp \beta_l T^2\)
- The normalization constant for the resulting source function is \(4 \pi^2 T\,R_s \sqrt{\Delta}\) where \(\Delta = R_o^2 R_l^2 - (R_{ol}^2)^2\)

## Context

This work extends the theoretical framework of HBT interferometry by applying the Maximum Entropy Principle, a method from information theory, to the problem of source reconstruction. It addresses the fundamental limitation identified in earlier work by Chapman, Nix, and Heinz (1995) that the HBT correlator provides insufficient information to reconstruct the four-dimensional source structure, even in the Gaussian approximation. The result provides a new Gaussian parametrization of the source function that explicitly demonstrates the remaining freedom in building source models consistent with a given set of measured HBT radius parameters, and enables systematic exploration of the range of possible source geometries compatible with experimental data.
