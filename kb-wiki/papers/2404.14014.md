# Fission barriers with Weizsäcker-Skyrme mass model

> Based on Weizsäcker-Skyrme (WS4) mass model, the fission barriers of nuclei are systematically studied. Considering the shell corrections, the macroscopic deformation energy and a phenomenological residual correction, the fission barrier heights for nuclei with $Z\geqslant 82$ can be well described, with an rms deviation of 0.481 MeV with respect to 71 empirical barrier heights. In addition to the shell correction at the ground state, the shell correction at the saddle point and its relative value are also important for both deformed nuclei and spherical nuclei. The fission barriers for nuclei far from the $β$-stability line and super-heavy nuclei are also predicted with the proposed approach.

## Key claim

The Weizsäcker-Skyrme (WS4) mass model, when augmented with shell corrections at both ground state and saddle point plus a phenomenological residual correction, reproduces 71 empirical fission barrier heights for nuclei with Z ≥ 82 with an rms deviation of 0.481 MeV, but the residual correction is empirically fitted to the same data, so the quoted accuracy is not a purely predictive test.

## Method

The authors compute fission barrier heights as the difference between saddle-point and ground-state energies using the WS4 macroscopic-microscopic model. The macroscopic barrier follows Cohen-Swiatecki's formula with WS4 coefficients; shell corrections are obtained from Strutinsky's method using an axially deformed Woods-Saxon potential with β₂ and β₄ deformations. A phenomenological residual correction ΔB is fitted as a function of the relative shell correction δU² to improve agreement with empirical barriers from Capote (2009).

## Key numbers

- rms deviation of 0.481 MeV for 71 empirical fission barrier heights (Z ≥ 82) using WS4 + residual correction ΔB
- Without saddle-point shell correction and deformation energy, rms deviation is 2.410 MeV; with them (B_f^WS4) it reduces to 0.873 MeV
- For ²⁰⁸Pb: B_f^(0) = 25.1 MeV (without saddle correction), B_f^WS4 = 27.3 MeV (with U_sad = 2.2 MeV), empirical value ≈ 27.4 MeV
- For ²³⁸U: B_f^(0) = 9.2 MeV, B_f^WS4 = 6.8 MeV (with B_def = 3.4 MeV, U_sad = 1.0 MeV), empirical value ≈ 6.3 MeV
- For ²¹⁰Ra: predicted B_f^WS4+ΔB = 8.78 MeV, measured value 8.54 MeV (5% uncertainty) from Veselsky (2024)
- For ²⁸⁶Cn: B_f^WS4 = 4.20 MeV, comparable to FRDM result multiplied by 0.7
- For ²⁹⁷(119): FRLDM predicts 7.94 MeV, WS4 predicts about 2 MeV lower

## Context

This work extends the WS4 mass model (Wang 2014), which reproduces known masses with ~0.3 MeV rms error, to fission barrier predictions. It improves upon the finite-range liquid-drop model (FRLDM) of Moller (2015) by incorporating saddle-point shell corrections and a residual correction, achieving lower rms deviation (0.481 MeV vs. FRLDM's comparable but slightly larger scatter). The predicted barriers for super-heavy nuclei (Z=119, 120) and neutron-deficient isotopes (e.g., ²¹⁰Ra) enable more accurate calculations of evaporation-residue cross sections for fusion reactions searching for new isotopes and super-heavy elements.
