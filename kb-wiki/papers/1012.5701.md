# Reanalysis of the (J = 5) state at 592 keV in 180Ta and its role in the neutrino-process nucleosynthesis of 180Ta in supernovae

> We analyze the production and freeze-out of the isomer 180Tam in the neutrino-process. We consider the influence of a possible low-lying intermediate (J = 5) state at 592 keV using a transition width estimated from the measured half-life. This more realistic width is much smaller than the previous estimate.We find that the 592-keV state leads only to a small reduction of the residual isomer population ratio from the previous result; that is, considering this better estimate for the transition width, the isomer population ratio changes from R = 0.39 to R = 0.38, whereas previously it was estimated that this transition could reduce the ratio to R = 0.18. This finding strengthens the evidence that 138La and 180Ta are coproduced by neutrino nucleosynthesis with an electron neutrino temperature of kT~4 MeV.

## Key claim

Using a more realistic transition width for the 592 keV state in 180Ta, estimated from its measured half-life rather than an exponential extrapolation, the isomer population ratio changes only slightly from R = 0.39 to R = 0.38, contrary to the previous estimate that this state could reduce the ratio to R = 0.18.

## Method

The authors performed a time-dependent calculation of the freeze-out of 180Ta in the neutrino-process, incorporating the 592 keV intermediate state (IMS) with a transition width of Gamma_tr = 1.6 x 10^-10 eV estimated from the measured half-life of 16.1 ns and an assumed 1% branch for the 235-keV gamma-ray. They compared this result to their previous calculation using only experimentally confirmed IMSs at higher energies (1087, 1297, and 1507 keV) and to an earlier estimate using an extrapolated width of 3 x 10^-6 eV.

## Key numbers

- Isomer population ratio R = 0.39 +/- 0.01 using experimentally confirmed IMSs only (from photoactivation data)
- Isomer population ratio R = 0.38 using the more realistic transition width for the 592 keV state (Gamma_tr = 1.6 x 10^-10 eV)
- Previous extrapolated width of Gamma_tr = 3 x 10^-6 eV would give R = 0.18, but this width is an overestimate by more than two orders of magnitude
- Measured half-life of the 592 keV state: T_1/2 = 16.1 +/- 1.9 ns
- The 235-keV gamma-ray transition strength is estimated at 7.5 x 10^-3 Weisskopf units (W.u.)

## Context

This work extends the authors' previous study (Hayakawa 2010) of 180Ta nucleosynthesis in the neutrino-process, which found R = 0.39 using experimentally confirmed IMSs from photoactivation experiments (Bel99, Bel02). It contradicts an earlier suggestion (Mohr 2007) that the 592 keV state could significantly reduce the isomer ratio. The finding strengthens the evidence that 138La and 180Ta are coproduced by neutrino nucleosynthesis with an electron neutrino temperature of kT approximately 4 MeV, and motivates future experimental measurements of the weak decay branches of the 592 keV state.
