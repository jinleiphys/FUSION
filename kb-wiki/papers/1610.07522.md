# Scalable Nuclear Density Functional Theory with Sky3D

> In nuclear astrophysics, quantum simulations of large inhomogeneous dense systems as they appear in the crusts of neutron stars present big challenges. The number of particles in a simulation with periodic boundary conditions is strongly limited due to the immense computational cost of the quantum methods. In this paper, we describe techniques for an efficient and scalable parallel implementation of Sky3D, a nuclear density functional theory solver that operates on an equidistant grid. Presented techniques allow Sky3D to achieve good scaling and high performance on a large number of cores, as demonstrated through detailed performance analysis on a Cray XC40 supercomputer.

# Scalable Nuclear Density Functional Theory with Sky3D

## Key claim

The paper demonstrates that a carefully designed MPI-only parallelization of the Sky3D nuclear DFT code, using a 2D decomposition for matrix operations and a 1D decomposition for wave function computations, can achieve good scaling to moderately large core counts (256-1024 processors) for nuclear pasta simulations, though the MPI-only implementation unexpectedly outperforms the hybrid MPI/OpenMP implementation due to poor performance of ScaLAPACK's eigendecomposition routines in the hybrid case.

## Method

The authors parallelized the static (ground-state) version of Sky3D, which operates on an equidistant 3D grid and uses the damped gradient iteration method with FFTs for derivatives. They employed a 1D block-cyclic distribution of wave functions for computations on individual wave functions, and a 2D block-cyclic distribution for matrix construction (overlap and Hamiltonian matrices) and eigendecompositions using ScaLAPACK routines (PZGEMM for matrix multiplication, PZHEEVR for Hermitian eigenvalue decomposition). Processors were split into neutron and proton groups with sizes determined by quadratically weighting particle numbers (P_N = N^2/(N^2+Z^2) * C, P_P = Z^2/(N^2+Z^2) * C), with constraints to ensure near-square process grids. The diagonalization and orthogonalization steps were overlapped by using half the available MPI ranks within each group for each operation, and the post-processing matrix-vector multiplications were combined to reduce computational cost from 2I^2V to I^3 + I^2V.

## Key numbers

- 256 to 1024 processors are reported as reasonably efficient for nuclear pasta simulations and considered for production runs, depending on the exact calculation
- The paper estimates that the new MPI-parallel Sky3D code can calculate pasta phases for over 10,000 nucleons in a fairly large box using a quantum mechanical treatment, representing approximately an order of magnitude increase over the previous limit of about 1000 nucleons for quantum methods
- For a system with 5000 neutrons and 1000 protons on 256 processors, the quadratic weighting scheme gives P_N = 246 and P_P = 10, but requiring processor counts to be multiples of 16 yields P_N = 240 and P_P = 16, resulting in a 16x15 process grid for neutrons and a 4x4 grid for protons
- The row and column block sizes for 2D block-cyclic distribution were empirically set to NB = MB = 32 for large computations, with experiments using values 2, 4, 8, 32, 64 showing negligible performance differences for problems with over 1000 particles on at least a few hundred cores

## Context

This work extends the Sky3D code (Maruhn et al., 2014) from a shared-memory OpenMP-only parallel version to a distributed-memory MPI implementation, enabling quantum mechanical simulations of nuclear pasta phases in neutron star crusts at significantly larger scales than previously possible. Prior quantum mechanical methods (Magierski et al., 2002; Goedecker et al., 2007; Newton and Stone, 2009; Sonoda et al., 2008; Schuetrumpf et al., 2014, 2015; Sagert et al., 2016; Fattoyev et al., 2017) were limited to about 1000 nucleons, while semi-classical methods such as Molecular Dynamics (Schneider et al., 2013, 2014; Horowitz et al., 2015; Watanabe et al., 2002-2009) could handle 50,000 or more particles. The new parallel implementation enables investigation of more complicated pasta shapes and defects in pasta structures that only manifest in large boxes, which had previously been observed in MD simulations (Horowitz et al., 2015) but not in quantum simulations.
