# Vud from Nuclear Decays

> The current value of Vud is determined from superallowed beta-decay experiments. Other methods, briefly summarised here, have to overcome specific experimental hurdles before they are competitive. However, the nuclear results do depend on a nuclear-structure calculation of isospin-symmetry breaking, which is often a cause of some concern. We show here, by adopting the Conserved Vector Current (CVC) hypothesis, that these theoretical corrections can be tested for consistency. In this test, calculations based on the shell model with Saxon-Woods radial functions perform the best.

## Key claim

The current most precise value of the CKM matrix element Vud is determined from superallowed nuclear beta decays, but this determination depends on a nuclear-structure calculation of isospin-symmetry breaking (δC) whose theoretical uncertainty now dominates over experimental uncertainty; the authors show that by adopting the Conserved Vector Current (CVC) hypothesis, the various theoretical δC corrections can be tested for consistency, and only the shell-model calculation with Saxon-Woods radial functions (SM-SW) passes this test.

## Method

The authors combine experimental ft values (derived from measured Q-values, lifetimes, and branching ratios) with radiative corrections (δR', δNS, ΔRV) and isospin-symmetry breaking corrections (δC) to compute corrected Ft values for superallowed 0+ → 0+ beta decays. They then apply a consistency test based on the CVC hypothesis: for any set of δC calculations to be acceptable, the Ft values must be nucleus-independent, so they treat the average Ft as a single adjustable parameter and compute the normalized χ2 for five different δC model calculations (SM-SW, SM-HF, RHF-RPA, RH-RPA, and IVMR) to see which best satisfies the CVC requirement.

## Key numbers

- Average corrected Ft value: 3071.81 ± 0.83 s (from 2009 survey)
- Vud from superallowed decays: 0.97425 ± 0.00022
- Vud from neutron decay: 0.9743 ± 0.0015
- Vud from T=1/2 mirror transitions: 0.9719 ± 0.0017
- Vud from pion beta decay: 0.9742 ± 0.0026
- CKM unitarity sum: |Vud|^2 + |Vus|^2 + |Vub|^2 = 0.99990 ± 0.00060 (precision of 0.06%)
- Normalized χ2/nd for SM-SW model (statistical errors only): 1.2 with confidence level 26%
- Normalized χ2/nd for SM-HF model (statistical errors only): 8.3 with confidence level 0%
- Normalized χ2/nd for RHF-RPA model (statistical errors only): 7.2 with confidence level 0%
- Normalized χ2/nd for RH-RPA model (statistical errors only): 6.0 with confidence level 0%
- Normalized χ2/nd for IVMR model (statistical errors only): 48.0 with confidence level 0%

## Context

This work extends the 2009 survey by Hardy and Towner (2009) and the 2010 survey by Towner and Hardy (2010) on superallowed beta decay, providing a systematic test of competing δC calculations from Ormand and Brown (1995), Sagawa et al. (1996), Liang et al. (2009), and Auerbach (2009). The test confirms that only the shell-model with Saxon-Woods radial functions (SM-SW) satisfies the CVC hypothesis, while shell-model with Hartree-Fock radial functions (SM-HF) fails for high-Z cases like 62Ga and 74Rb. This enables future work to focus on improving the SM-SW approach and resolving the discrepancy with claims by Miller and Schwenk (2008) about neglected radial excitation terms, as well as incorporating recent confirmations from Grinyer et al. (2010) using semi-empirical analysis and from Satula et al. (2010) using Skyrme-density-functional theory.
