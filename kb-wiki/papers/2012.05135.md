# N$^3$LO extraction of the Sivers function from SIDIS, Drell-Yan, and $W^\pm/Z$ data

> We perform the global analysis of polarized Semi-Inclusive Deep Inelastic Scattering (SIDIS), pion-induced polarized Drell-Yan (DY), and $W^\pm/Z$ boson production data and extract the Sivers function for $u$, $d$, $s$ and for sea-quarks. We use the framework of transverse momentum dependent factorization at N$^3$LO accuracy. The Qiu-Sterman function is determined in a model-independent way from the extracted Sivers function. We also evaluate the significance of the predicted sign change of Sivers function in DY with respect to SIDIS.

# N$^3$LO extraction of the Sivers function from SIDIS, Drell-Yan, and $W^\pm/Z$ data

## Key claim
The Sivers function is extracted at N$^3$LO perturbative precision from a global fit of SIDIS, pion-induced Drell-Yan, and $W^\pm/Z$ boson production data, using unpolarized TMDs from the same order and scheme for the first time. However, the predicted sign change of the Sivers function between SIDIS and Drell-Yan cannot be conclusively confirmed with current data: a fit without the sign change yields $\chi^2/N_{pt} = 1.0$, and the sign of the sea-quark Sivers function alone can compensate the overall sign-flip effect at the cross-section level.

## Method
The authors perform a global analysis of transverse spin asymmetries using TMD factorization at N$^3$LO accuracy within the $\zeta$-prescription, which selects a reference scale on the equipotential line of the field anomalous dimension. The Sivers function is parameterized directly in $b$-space with a 12-parameter ansatz for $u$, $d$, $s$, and sea quarks, without matching to the Qiu-Sterman function at small $b$; instead, the Qiu-Sterman function is derived afterward from the extracted Sivers function via the operator product expansion at NLO. Unpolarized TMDs for proton and pion are taken from the SV19 extraction (Scimemi and Vladimirov, 2019) and Vladimirov (2019) at the same perturbative order. The fit uses 76 experimental data points after kinematic cuts ($\langle Q\rangle > 2$ GeV, $\delta < 0.3$), and uncertainties are estimated via the replica method with 500 replicas, including propagation of unpolarized TMD uncertainties.

## Key numbers
- 76 experimental data points used in the global fit: 63 from SIDIS (HERMES, COMPASS, JLab) and 13 from Drell-Yan (STAR $W^\pm/Z$, COMPASS pion-induced DY).
- Combined SIDIS+DY fit yields $\chi^2/N_{pt} = 0.88$ for all 76 points, with $\chi^2/N_{pt} = 0.88$ for SIDIS and $\chi^2/N_{pt} = 0.90$ for DY.
- SIDIS-only fit has $\chi^2/N_{pt} = 0.87$ and describes DY data with $\chi^2/N_{pt} = 1.23$ without retuning.
- Fit without the sign change (same sign in SIDIS and DY) yields $\chi^2/N_{pt} = 1.0$.
- The Qiu-Sterman function is evaluated at $\mu = 10$ GeV using $b \simeq 0.11$ GeV$^{-1}$.
- The Collins-Soper kernel non-perturbative parameter $c_0$ is a free parameter in the form $\mathcal{D}(b,\mu) = \mathcal{D}_{\text{resum}}(b^*,\mu) + c_0 b b^*$, with $b^* = b/\sqrt{1+(b/(2\ \text{GeV}^{-1}))^2}$.

## Context
This work extends previous Sivers function extractions (Efremov 2004, Vogelsang 2005, Anselmino 2005-2016, Collins 2005, Bacchetta 2011, Gamberg 2013, Sun 2013, Echevarria 2014, Boglione 2018, Bacchetta 2020, Echevarria 2020, Cammarota 2020) by achieving N$^3$LO perturbative precision and consistently using unpolarized TMDs from the same order and scheme (SV19). The model-independent relation between the Sivers function and the Qiu-Sterman function enables a new determination of the latter, with an estimate of the unknown gluon QS function contribution. The results set a benchmark for future studies at the Electron Ion Collider, Jefferson Lab 12 GeV Upgrade, RHIC, and COMPASS, and are important for testing the predicted sign change of the Sivers function between SIDIS and Drell-Yan processes.
