# Charmonium decay in the C3P0 Model

> In this work, we use the $C^3P_{\,0}$ model to calculate the decay widths of the low lying charmonium $J^{PC}=1^{--}$ states, nominally $J/ψ(1S)$ and $ψ(2S)$, in the following common channels: $ρ\,π$, $ω\,η$, $ω\,η^\prime$, $K^{\ast +}\,K^-$, $K^{\ast 0}\,\bar{K}^0$, $ϕ\,η$, $ϕ\,η^\prime$.

## Key claim

The C³P₀ model, which incorporates bound-state corrections to the leading-order ³P₀ pair-creation mechanism via the Fock-Tani formalism, can reproduce the OZI-suppressed decay widths of J/ψ(1S) and ψ(2S) into light meson channels, but only when a flavor-mixing scheme is introduced to account for the small c̄c components in the light vector and pseudoscalar mesons.

## Method

The authors start from a microscopic qq̄ pair-creation Hamiltonian and apply the Fock-Tani unitary transformation to obtain an effective meson decay Hamiltonian. The leading-order term reproduces the standard ³P₀ model, while next-to-leading-order terms introduce bound-state corrections via the bound-state kernel Δ, which modifies the quark-antiquark interaction strength. Decay widths are computed from the resulting C³P₀ amplitudes combined with relativistic phase space, using harmonic-oscillator spatial wave functions and a flavor-mixing scheme for ω, φ, η, η′, and J/ψ.

## Key numbers

- For J/ψ(1S) decays: ρπ channel Γ = 1.55 keV (exp. 1.57 keV); ωη channel Γ = 0.16 keV (exp. 0.16 keV); K*+K− channel Γ = 0.47 keV (exp. 0.46 keV); φη′ channel Γ = 0.04 keV (exp. 0.04 keV); overall fit quality R = 0.41
- For ψ(2S) decays: ρπ channel Γ = 9.47 eV (exp. 9.40 eV); ωη channel Γ = 3.67 eV (exp. 3.23 eV); K*+K− channel Γ = 5.06 eV (exp. 5.12 eV); φη′ channel Γ = 3.28 eV (exp. 4.52 eV); overall fit quality R = 1.44
- Best-fit parameters for J/ψ(1S): γ = 0.33, βJ/ψ = 0.182 GeV, βρ = 0.55 GeV, βπ = 0.55 GeV, βω = 0.328 GeV, βη = 0.348 GeV, βη′ = 0.170 GeV, βK = 0.4 GeV, βK* = 0.315 GeV, βφ = 0.340 GeV
- Best-fit parameters for ψ(2S): γ = 0.33, βψ = 0.042 GeV, βρ = 0.55 GeV, βπ = 0.55 GeV, βω = 0.7 GeV, βη = 0.7 GeV, βη′ = 0.8 GeV, βK = 0.47 GeV, βK* = 0.8 GeV, βφ = 0.7 GeV

## Context

This work extends the C³P₀ model, previously applied to φ meson decays (da Silva et al., 2020, Physical Review C), into the charmonium sector. The approach improves upon the leading-order ³P₀ model (Barnes et al., various) by including bound-state corrections that modify the pair-creation strength; for φ mesons these corrections reduced the fit quality parameter R. The results serve as a baseline for future studies of charmed meson spectroscopy at the PANDA experiment at FAIR, particularly for searches for exotic states such as the c̄cg charmonium hybrid and multiquark states like χc1(3872) and Zc(4430).
