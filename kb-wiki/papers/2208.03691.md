# An effective and efficient algorithm for the Wigner rotation matrix at high angular momenta

> The Wigner rotation matrix ($d$-function), which appears as a part of the angular-momentum-projection operator, plays a crucial role in modern nuclear-structure models. However, it is a long-standing problem that its numerical evaluation suffers from serious errors and instability, which hinders precise calculations for nuclear high-spin states. Recently, Tajima [Phys. Rev. C 91, 014320 (2015)] has made a significant step toward solving the problem by suggesting the high-precision Fourier method, which however relies on formula-manipulation softwares. In this paper we propose an effective and efficient algorithm for the Wigner $d$ function based on the Jacobi polynomials. We compare our method with the conventional Wigner method and the Tajima Fourier method through some testing calculations, and demonstrate that our algorithm can always give stable results with similar high-precision as the Fourier method, and in some cases (for special sets of $j, m, k$ and $θ$) ours are even more accurate. Moreover, our method is self-contained and less memory consuming. A related testing code and subroutines are provided as Supplemental Material in the present paper.

## Key claim

The paper proposes a Jacobi-polynomial-based algorithm for evaluating the Wigner \(d\)-function that achieves stable high precision (around \(10^{-14}\) to \(10^{-15}\) for \(j \leq 100\)) without relying on external formula-manipulation software, unlike the Tajima Fourier method which requires such software and large memory storage (about 70 MB for \(j \leq 50\) and 1.2 GB for \(j \leq 100\)).

## Method

The authors express the Wigner \(d\)-function \(d^j_{mk}(\theta)\) in terms of Jacobi polynomials \(P^{(\mu,\nu)}_s(\cos\theta)\) with parameters \(\mu = |m-k|\), \(\nu = |m+k|\), and \(s = j - (\mu+\nu)/2\). They compute the Jacobi polynomial using recurrence relations rather than its explicit summation formula, which avoids the accumulation of numerical errors from factorials of large numbers. The algorithm is implemented in a FORTRAN90 code and compared against the conventional Wigner formula and the Tajima Fourier method for various \(j\), \(m\), \(k\), and \(\theta\) values.

## Key numbers

- Maximum error from the conventional Wigner method exceeds \(10^{-10}\) for \(j \geq 25\), growing exponentially with \(j\).
- The Jacobi algorithm (with recurrence relations) achieves precision of about \(10^{-14}\) to \(10^{-15}\) for \(j \leq 100\), comparable to the Fourier method.
- The Fourier method requires approximately 70 MB of memory for \(j \leq 50\) and 1.2 GB for \(j \leq 100\); the Jacobi method avoids this memory consumption.
- For the integral test \(I^j_{mk} = \int_0^\pi d\theta \sin\theta \, d^j_{mk}(\theta) \, d^{j+1}_{mk}(\theta) = 0\), the Jacobi algorithm achieves errors as low as \(10^{-16}\) for \(j \leq 100\) with 200 Gauss-Legendre mesh points, while the Wigner method exceeds \(10^{-8}\) error for \(j \gtrsim 35\).

## Context

This work extends the effort by Tajima (2015) who introduced a Fourier-series method for the Wigner \(d\)-function that significantly improved numerical stability but required formula-manipulation software (MAXIMA or Mathematica) and large memory storage. The Jacobi algorithm provides a self-contained alternative that matches or exceeds the Fourier method's precision while being more memory-efficient and user-friendly. The algorithm enables precise angular-momentum projection calculations for nuclear high-spin states, which are essential in beyond-mean-field methods such as the projected shell model and applications to beta-decay, neutrinoless double-beta decay, and nuclear fission.
