# Distribution of matrix elements of random operators

> It is shown that an operator can be defined in the abstract space of random matrices ensembles whose matrix elements statistical distribution simulates the behavior of the distribution found in real physical systems. It is found that the key quantity that determines these distribution is the commutator of the operator with the Hamiltonian. Application to symmetry breaking in quantum many-body systems is discussed.

# Distribution of Matrix Elements of Random Operators

## Key claim

The authors show that a random operator can be defined in the abstract space of random matrix ensembles whose matrix elements reproduce the statistical distributions observed in real physical systems, with the key quantity controlling these distributions being the commutator of the operator with the Hamiltonian. The claim is hedged by the fact that the operator is constructed specifically to simulate behavior in intermediate situations between chaos and order, and the results depend on the choice of the chaoticity parameter λ and the operator parameter q.

## Method

The authors construct random Hamiltonians using the Maximum Entropy Principle in the form H = H₀ + λH₁, where λ controls the chaoticity, and define a random operator O from the same GOE matrix elements. They study two transitions: GOE→Poisson (using projection operators Pᵢ = |i⟩⟨i|) and GOE→two coupled GOE's (using block projectors P and Q). For each case, they diagonalize the Hamiltonian, compute normalized matrix elements Tₖₗ = ⟨Eₗ|αₖ⟩ where |αₖ⟩ = O|Eₖ⟩/⟨Eₖ|O²|Eₖ⟩, perform local averaging with a Gaussian filter of variance 2, and histogram the logarithm of the scaled intensities yₖₗ = |Tₖₗ|²/⟨|Tₖₗ|²⟩.

## Key numbers

- Matrix dimension N = 100 for the GOE→Poisson transition calculations
- Matrix dimension N = 120 with block sizes M₁ = 40 and M₂ = 80 for the GOE→2GOE's transition calculations
- Chaoticity parameter λ = 0.032 was fixed for the GOE→2GOE's transition study, with q varied
- The parameter λ is defined in the domain 0 ≤ λ ≤ 1, where λ = 1 gives H = H^GOE and λ = 0 gives a reduced chaotic situation
- The Gaussian filter used for local averaging has variance equal to 2

## Context

This work extends the deformed Gaussian orthogonal ensemble (DGOE) framework developed by Hussein and Pato (1993) and Pato et al. (1994) to the distribution of matrix elements, which had previously only been studied in the fully chaotic limit where the Porter-Thomas law applies (Porter and Thomas, 1956). The authors contradict the suggestion by Alhassid and Levine (1986) that a single χ² distribution suffices for intermediate situations, finding instead that a linear combination of two χ² distributions is needed. The theory enables application to symmetry breaking in quantum many-body systems, with isospin symmetry violation in light nuclei (Adams, Mitchell, and Shriner, 1998) cited as an ongoing application.
