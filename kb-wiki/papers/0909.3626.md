# HFODD (v2.40h) User's Guide

## Key claim

This user guide consolidates and supersedes six earlier independent publications describing the HFODD code, providing a single comprehensive reference for input data, installation procedures, and usage of version 2.40h of the code that solves the nuclear Hartree-Fock (HF) or Hartree-Fock-Bogoliubov (HFB) equations with no symmetry restrictions imposed on the nuclear mean field.

## Method

The paper documents the complete input-data structure, keyword-driven parameter specification, and installation workflow for the HFODD code, which performs self-consistent mean-field calculations using Skyrme energy density functionals or Nilsson potentials in a deformed harmonic-oscillator basis. It includes detailed indexes of input-data keywords, subroutines, files, and variables, along with example input and output files for nuclei such as Dy-152, Ge-064, La-132, Pb-208, Sn-120, and Tb-151.

## Key numbers

* Example input/output files are provided for Dy-152 (files dy152-a through dy152-f), Ge-064 (files ge064-a through ge064-c), La-132 (files la132-a and la132-b), Pb-208 (file pb208-a), Sn-120 (files sn120-a and sn120-b), and Tb-151 (files tb151-a through tb151-d), referenced to pages 73-80 of the guide.
* The code supports basis sizes controlled by the keyword `BASIS_SIZE` (page 48, 69) and cutoff energies via `CUTOFF` (page 21) and `ENECUT` (page 48).
* Convergence criteria are set by `ITERAT_EPS` and `EPSITE` (pages 10-11, 74), with a recommended value for `EPSPNG` to control ping-pong oscillations (pages 11-12, 74).
* The code can perform angular momentum projection (AMP) in 1D and 3D (page 30), using Gauss-Legendre and Gauss-Tchebyschev nodes (pages 30-31, 68, 80).

## Context

This work extends the series of HFODD code publications by Dobaczewski and collaborators, consolidating documentation that was previously scattered across six separate papers. The code enables calculations of nuclear structure properties for both even-even and odd-mass nuclei, including superdeformed states, tilted-axis cranking, and symmetry-unrestricted mean fields. It provides the foundation for subsequent studies of nuclear fission, high-spin states, and nuclear moments using the same computational framework.
