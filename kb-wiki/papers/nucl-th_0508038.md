# Exactly Solvable Model for the Decay of Superdeformed Nuclei

> The history and importance of superdeformation in nuclei is briefly discussed. A simple two-level model is then employed to obtain an elegant expression for the branching ratio for the decay via the E1 process in the normal-deformed band of superdeformed nuclei. From this expression, the spreading width Gamma^downarrow for superdeformed decay is found to be determined completely by experimentally known quantities. The accuracy of the two-level approximation is verified by considering the effects of other normal-deformed states. Furthermore, by using a statistical model of the energy levels in the normal-deformed well, we can obtain a probabilistic expression for the tunneling matrix element V.

# Exactly Solvable Model for the Decay of Superdeformed Nuclei

## Key claim

The two-level model yields an exact expression for the spreading width Gamma^downarrow that is determined completely by experimentally measurable quantities, but this Gamma^downarrow is not equal to its ensemble average (the two are equal only in the continuum limit of overlapping resonances, which is not appropriate for most nuclei of interest).

## Method

The authors model superdeformed (SD) nuclear decay as a two-level mixing problem between one SD state and the nearest normal-deformed (ND) state of the same angular momentum, using a retarded Green's function formalism solved exactly via Dyson's equation. They then extend the analysis to a three-level system to test the validity of the two-level approximation, and apply a Gaussian Orthogonal Ensemble (GOE) statistical model to the ND level distribution to derive a probability density for the tunneling matrix element V.

- The discrepancy between Gamma^downarrow and its ensemble average can be as much as three to four orders of magnitude in real nuclei
- In the mass-150 region, three-level corrections to two-state branching ratios can reach 25% (I=28) to 40% (I=26) for V/D_N greater than approximately 0.15

## Context

This work extends the two-level model introduced by Stafford and Barrett (1999) and provides a rigorous connection between experimental observables and the tunneling matrix element V, which had previously been treated via ensemble averaging in work by Vigezzi and collaborators. The authors show that the two-level approximation is sufficient for the mass-190 region but requires corrections in the mass-150 region, consistent with the infinite-band exact solution of Dzyublik and Utyuzh (2003). The results enable direct extraction of the spreading width from measured branching ratios and provide a probabilistic framework for determining the tunneling matrix element, which can be related to the barrier height as shown by Wilson and Davidson (2004).
