# Study of the semileptonic decay Λ_b^0 \to Λ_c^+ l^- \barν_l

> Within the framework of a nonrelativistic quark model we evaluate the six form factors associated to the Λ_b^0 \to Λ_c^+ l^- \barν_l semileptonic decay. The baryon wave functions were evaluated using a variational approach applied to a family of trial functions constrained by Heavy Quark Symmetry (HQS). We use a spectator model with only one-body current operators. For these operators we keep up to first order terms on the internal (small) heavy quark momentum, but all orders on the transferred (large) momentum. Our result for the partially integrated decay width is in good agreement with lattice calculations. Comparison of our total decay width to experiment allows us to extract the V_{cb} Cabbibo-Kobayashi-Maskawa matrix element for which we obtain a value of |V_{cb}|=0.047\pm 0.005 in agreement with a recent determination by the DELPHI Collaboration. Furthermore, we obtain the universal Isgur-Wise function with a slope parameter ρ^2=0.98 in agreement with lattice results.

## Key claim

Within a nonrelativistic quark model that incorporates Heavy Quark Symmetry (HQS) constraints on the baryon wave functions, the authors evaluate the six form factors for the semileptonic decay Lambda_b^0 to Lambda_c^+ l- nu_l, keeping terms up to first order in the internal heavy quark momentum but all orders in the transferred momentum, and find that the extracted |V_cb| value of 0.047 plus or minus 0.005 agrees with a recent DELPHI determination.

## Method

The authors solve the nonrelativistic three-body problem for the Lambda_b and Lambda_c baryons using a variational ansatz for the spatial wave function, which consists of single-particle wave functions for each light quark relative to the heavy quark multiplied by a Jastrow-type correlation function. They employ a spectator model with one-body current operators, keeping terms up to first order in the internal heavy quark momentum but all orders in the transferred momentum, to compute the six form factors F1, F2, F3, G1, G2, G3 that parametrize the weak charged current matrix element. The differential decay width is then evaluated as a function of the four-velocity product omega, and the total decay width is compared to experiment to extract the CKM matrix element |V_cb|.

## Key numbers

* Extracted CKM matrix element: |V_cb| = 0.047 plus or minus 0.005, obtained by comparing the calculated total decay width to experimental data.
* Isgur-Wise function slope parameter: rho^2 = 0.98, obtained from the universal Isgur-Wise function derived from the form factors.
* Partially integrated decay width shows good agreement with lattice calculations (no specific numerical value given for the width itself).

## Context

This work extends the authors' previous HQS-constrained variational method for heavy baryon wave functions (Albertus, Hernandez, Nieves, 2003) to the calculation of semileptonic decay form factors, providing a simpler alternative to more sophisticated Faddeev calculations (Silvestre-Brac, 1996). The results enable a determination of |V_cb| that agrees with the DELPHI Collaboration measurement and provides the universal Isgur-Wise function with a slope parameter consistent with lattice QCD results, which can be used for interpolations at low momentum transfer in lattice calculations.
