# Chiral Three-Nucleon Forces from p-wave Pion Production

> Production of p-wave pions in nucleon-nucleon collisions is studied according to an improved power counting that embodies the constraints of chiral symmetry. Contributions from the first two non-vanishing orders are calculated. We find reasonable convergence and agreement with data for a spin-triplet cross section in $pp\to ppπ^0$,with no free parameters. Agreement with existing data for a spin-singlet cross section in $pp\to pnπ^+$ constrains a short-range operator shown recently to contribute significantly to the three-nucleon potential.

# Chiral Three-Nucleon Forces from p-wave Pion Production

## Key claim

The production of p-wave pions in nucleon-nucleon collisions near threshold (eta ~ 1) shows reasonable convergence in chiral perturbation theory, and data on the spin-singlet amplitude in pp -> pn pi+ can constrain the short-range operator parameters d_i that determine a novel three-nucleon force, provided the same cutoff and parameter set are used consistently in both calculations.

## Method

The authors apply chiral perturbation theory with an improved power counting scheme that treats the different momentum scales (initial nucleon momenta of order sqrt(m_N m_pi) versus final-state momenta of order m_pi) by separating the production amplitude into a kernel K and initial/final state wave functions. They calculate contributions at leading order (direct nucleon and delta production from the chiral Lagrangian at chiral index zero) and next-to-leading order (recoil corrections, rescattering through seagull vertices with parameters c_3 and c_4, and short-range pi(N^dagger N)^2 interactions with parameters d_1 and d_2), using the phenomenological CCF coupled-channel model for the initial-state wave function.

## Key numbers

- The chiral index of an interaction is defined as Delta = d + f/2 - 2, where d counts derivatives, m_pi, and delta m (the Delta-N mass splitting), and f is the number of fermion field operators; chiral symmetry implies Delta >= 0.
- The c_3 parameter from tree-level pi N scattering is c_3^(tree) = -3.90 GeV^{-1}; from one-loop order it is c_3^(loop) = -5.29 GeV^{-1}.
- The c_4 parameter from tree-level pi N scattering is c_4^(tree) = 2.25 GeV^{-1}; from one-loop order it is c_4^(loop) = 3.63 GeV^{-1}.
- The short-range operator combination d = (1/5)(d_1 + 4d_2) is expected on dimensional grounds to be d = delta/(f_pi^2 M_QCD) with delta = O(1).
- A value of delta = -0.2 is consistent with both the pion production data for the amplitude a_0 in pp -> pn pi+ and with the contribution to the Ay asymmetry in Nd scattering at energies of a few MeV.
- The lowest-order loop contribution in p-wave production enters at O((m_pi/m_N)^(3/2)), one order higher than the rescattering terms.

## Context

This work extends the chiral perturbation theory approach to pion production (Cohen, Friar, Miller, van Kolck 1996) from the s-wave region (eta < 0.4) into the p-wave region (eta ~ 1), where convergence is expected to be better because the production proceeds through leading-order operators better determined from other processes. The short-range three-nucleon force involving pion exchange between one nucleon and two others interacting via short-ranged forces (van Kolck 1994) depends on the same parameters d_i that enter p-wave pion production, providing a direct connection between IUCF pion production experiments at ~350 MeV and Nd scattering experiments at ~10 MeV (Madison, TUNL). The results enable extraction of the three-nucleon force parameter d from improved pion production data, which could then be used to predict three-nucleon observables and potentially resolve remaining discrepancies in Ay in Nd scattering (Friar and Huber 1998) and excited levels in bound states (Wiringa 1998).
