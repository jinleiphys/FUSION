# Prediction of Nuclear Charge Density Distribution with Feedback Neural Network

> The nuclear charge density distribution plays an important role in nuclear physics and atomic physics. As one of the most frequently used models to obtain charge density distribution, the two-parameter fermi (2pF) model has been widely applied in both nuclear physics and atomic physics. Currently, the feedforward neural network has been employed to study the available 2pF model parameters for 86 nuclei, and it is found that by introducing A^{1/3} into the input parameter of the neural network, the accuracy and precision of the parameter learning effect are improved. Furthermore, the average result of multiple predictions is more reliable than the best result of a single prediction, and there is no significant difference between the average result of density value and of parameter value for the average charge density distribution. In addition, 2pF parameters of 284 (near) stable nuclei are also predicted in this work, which provides a reference for the experiment.

## Key claim

The feedforward neural network (FNN) approach can predict two-parameter Fermi (2pF) model parameters for nuclear charge density distributions, but only when the input includes \(A^{1/3}\) in addition to \(Z, N, Z^{1/3}\) does the accuracy improve significantly; the method's extrapolation ability is limited, as predictions far from the learning set (e.g., calcium isotopes far from \(^{40}\)Ca) become unreliable.

## Method

The authors trained a feedforward neural network with two hidden layers (five neurons each) using the Levenberg-Marquardt back-propagation algorithm and stochastic gradient descent, with the hyperbolic tangent activation function. The input variables were either \((Z, N, Z^{1/3})\) (FNN-I3) or \((Z, N, Z^{1/3}, A^{1/3})\) (FNN-I4), and the outputs were the 2pF parameters \(c\) (half-density radius) and \(z\) (diffuseness). The network was trained on experimental 2pF data for 86 nuclei from DeJager (1974), DeVries (1987), and Fricke (1995), with 80% of the data for training and 20% for validation, repeated 1000 times with random initialization and data splits.

## Key numbers

* For parameter \(c\) with FNN-I4 (including \(A^{1/3}\)), the mean squared error (MSE) on the training set was \(0.02408\ \mathrm{fm}^2\) and on the validation set was \(0.02286\ \mathrm{fm}^2\), compared to FNN-I3 values of \(0.03913\ \mathrm{fm}^2\) and \(0.03655\ \mathrm{fm}^2\) respectively.
* For parameter \(z\) with FNN-I4, the MSE on the training set was \(0.00115\ \mathrm{fm}^2\) and on the validation set was \(0.00188\ \mathrm{fm}^2\).
* The Pearson correlation coefficient between \(A^{1/3}\) and parameter \(c\) was 0.9953, and between \(Z^{1/3}\) and \(c\) was 0.9969; for parameter \(z\), the correlations were \(-0.1625\) and \(-0.1609\) respectively.
* The rms deviation between experimental charge radii and FNN-I4 predictions for the 230 nuclei in the prediction set with available data was \(0.07693\ \mathrm{fm}\).
* The rms deviations for the second and fourth moments of the charge density distribution on the learning set were \(0.7045\ \mathrm{fm}^2\) and \(42.83\ \mathrm{fm}^4\) respectively.

## Context

This work extends the application of feedforward neural networks, previously used for nuclear masses, charge radii, and other ground-state properties (e.g., Niu 2018, Utama 2016), to the prediction of 2pF charge density parameters. It directly builds on the experimental 2pF data compilations of DeJager (1974), DeVries (1987), and Fricke (1995), and provides predicted 2pF parameters for 284 (near) stable nuclei that lack experimental data. The results enable future experiments to have reference charge density distributions and highlight the need for more training data to improve extrapolation to exotic nuclei, as demonstrated by the calcium isotope chain predictions.
