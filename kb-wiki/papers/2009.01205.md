# Finite-temperature mean-field approximations for shell model Hamiltonians: the code HF-SHELL

> We present the code HF-SHELL for solving the self-consistent mean-field equations for configuration-interaction shell model Hamiltonians in the proton-neutron formalism. The code can calculate both ground-state and finite-temperature properties in the Hartree-Fock (HF), HF+Bardeen-Cooper-Schrieffer (HF+BCS), and the Hartree-Fock-Bogoliubov (HFB) mean-field approximations. Particle-number projection after variation is incorporated to reduce the grand-canonical ensemble to the canonical ensemble, making the code particularly suitable for the calculation of nuclear state densities. The code does not impose axial symmetry and allows for triaxial quadrupole deformations. The self-consistency cycle is particularly robust through the use of the heavy-ball optimization technique and the implementation of different options to constrain the quadrupole degrees of freedom.

## Key claim

The code HF-SHELL solves self-consistent mean-field equations for configuration-interaction shell model Hamiltonians at both zero and finite temperature, using the heavy-ball optimization technique to achieve a particularly robust self-consistency cycle; the abstract does not specify that the code assumes time-reversal symmetry and is therefore suitable only for even-even nuclei.

## Method

The code iterates the Hartree-Fock (HF), HF+Bardeen-Cooper-Schrieffer (HF+BCS), or Hartree-Fock-Bogoliubov (HFB) mean-field equations for a proton-neutron formalism, using the heavy-ball algorithm of Ryssens (2019) supplemented by the two-basis method of Gall (1994) for HFB. Particle-number projection after variation, following Fanto (2017), reduces the grand-canonical ensemble to the canonical ensemble. The code does not impose axial symmetry, allowing triaxial quadrupole deformations, and assumes time-reversal, z-signature, and reflection symmetries with real matrix elements.

## Key numbers

- For the examples provided, a single calculation on a laptop takes from much less than a second for 24Mg up to a few seconds for 162Dy.
- A complete calculation of the state density of 162Dy as a function of excitation energy (involving roughly 500 individual mean-field calculations) takes about ten minutes.
- The code uses less than 300 MB of memory in all examples shown.

## Context

This code extends the unpublished finite-temperature mean-field code used in Alhassid (2016) and Ozen (2013) and the HF code HFGRAD of Bertsch (2016) by adding pairing correlations, triaxial deformations, and quadrupole constraints at finite temperature. It enables microscopic calculation of nuclear level densities with collective effects included, complementing exact shell model Monte Carlo methods such as those of Alhassid (2017) and providing a benchmark for global mean-field-based level density models like those of Hilaire (2012).
