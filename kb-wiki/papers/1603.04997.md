# Event shape analysis in ultrarelativistic nuclear collisions

> We present a novel method for sorting events. So far, single variables like flow vector magnitude were used for sorting events. Our approach takes into account the whole azimuthal angle distribution rather than a single variable. This method allows us to determine the good measure of the event shape, providing a multiplicity-independent insight. We discuss the advantages and disadvantages of this approach, the possible usage in femtoscopy, and other more exclusive experimental studies.

## Key claim

The authors propose a novel iterative algorithm for sorting heavy-ion collision events by the full shape of their azimuthal angle distribution rather than by a single flow observable, claiming it provides a multiplicity-independent measure of event shape. However, the method's success depends on the arbitrary choice of initial event rotation, and in realistic simulations with multiple harmonic orders the sorting is not dominated by any single flow harmonic.

## Method

The algorithm takes each event's azimuthal angle histogram (20 bins), initially sorts events by a chosen variable, divides them into deciles, then iteratively reassigns events to bins based on the probability that an event with a given histogram belongs to each decile. The process repeats until convergence, yielding a final ordering where events with similar angular distributions are grouped together. The authors tested the method on 5000 toy-model events generated with multiplicity-dependent flow coefficients up to fifth order, with multiplicities M between 300 and 3000.

## Key numbers

- 5000 events were generated from a toy model with azimuthal distribution P5(phi) including harmonics n=1 to 5
- Multiplicity range: M in (300, 3000)
- Azimuthal histograms used 20 bins per event
- Events were sorted into 10 quantiles (deciles) by the algorithm

## Context

This work extends the "Event Shape Engineering" approach (Kopecna, citing the ESE method) which sorts events by single observables like the flow vector magnitude q2. The authors show that in simple cases with only v1 and v2, the true v2 correlates better with their sorting variable than q2 does, but in more realistic simulations with harmonics up to fifth order, neither v2 nor q2 dominates the sorting. The method is intended to enable femtoscopic studies using mixed-event techniques on samples of events with similar momentum distributions, potentially approaching single-event femtoscopy.
