# Non-identical particle femtoscopy in models with single freeze-out

> We present femtoscopic results from hydrodynamics-inspired thermal models with single freeze-out. Non-identical particle femtoscopy is studied and compared to results of identical particle correlations. Special emphasis is put on shifts between average space-time emission points of non-identical particles of different masses. They are found to be sensitive to both the spatial shift coming from radial flow, as well as average emission time difference coming from the resonance decays. The Therminator Monte-Carlo program was chosen for this study because it realistically models both of these effects. In order to analyze the results we present and test the methodology of non-identical particle correlations.

## Key claim

Non-identical particle femtoscopy in the single freeze-out model is sensitive to both spatial shifts from radial flow and average emission time differences from resonance decays, and these two effects are of comparable magnitude in the laboratory frame for pion-kaon, pion-proton, and kaon-proton systems.

## Method

The Therminator Monte-Carlo program, implementing a hydrodynamics-inspired single freeze-out model with a blast-wave parametrization and a quasi-linear velocity profile, was used to generate particles including all resonances from the Particle Data Group. Unstable particles were decayed with proper propagation, and the resulting stable particles were used to construct femtoscopic correlation functions via the two-particle weight method, which exactly accounts for Coulomb interactions in the pair rest frame. The correlation functions were then fitted using a numerical Monte-Carlo procedure that assumes a Gaussian source with a shift parameter, mimicking experimental analysis.

## Key numbers

- Freeze-out hypersurface parameters from fit to STAR central AuAu data: proper time τ = 8.55 fm, maximum transverse radius ρ_max = 8.92 fm, velocity parameter v_T = 1.41, average radial velocity 0.31.
- Average emission times from Therminator: pions 12.3 fm/c, kaons 10.7 fm/c, protons 7.9 fm/c.
- Bohr radii for Coulomb interaction: pion-kaon 248.5 fm, pion-proton 222.6 fm, kaon-proton 83.6 fm.
- Spatial shifts in the source frame for pair transverse velocity β_t = 0.5–0.65: r_πK = -2.4 fm, r_πp = -3.4 fm, r_Kp = -0.9 fm.
- Time shifts in the source frame for β_t = 0.5–0.65: Δt_πK = 3.6 fm/c, Δt_πp = 5.7 fm/c, Δt_Kp = 2.1 fm/c.
- Agreement between fitted shifts and input model asymmetries within 1.0 fm.

## Context

This work extends the single freeze-out model calculations from identical particle femtoscopy (Kisiel 2006) to non-identical particle correlations, which had very few theoretical predictions (Lisa and Retiere). It provides the first self-consistent estimate of both spatial and temporal emission asymmetries in a model with full resonance treatment, enabling experimental extraction of the average emission point difference between particle species, a unique probe of collective flow and resonance decay dynamics. The sum rule for shifts between three particle species and the consistency check between identical and non-identical femtoscopic sizes are proposed as data quality tests.
