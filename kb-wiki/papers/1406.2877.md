# Diffractive vector meson production in ultraperipheral heavy ion collisions from the Color Glass Condensate

> We compute cross sections for incoherent and coherent diffractive J/$Ψ$ and $Ψ(2S)$ production in ultraperipheral heavy ion collisions. The dipole models used in these calculations are obtained by fitting the HERA deep inelastic scattering data and compared with available electron-proton diffraction measurements. We obtain a reasonably good description of the available ALICE data. We find that the normalization of the ultraperipheral cross section has large model dependence, but the rapidity dependence is more tightly constrained.

## Key claim

The authors compute cross sections for coherent and incoherent diffractive J/Ψ and Ψ(2S) production in ultraperipheral heavy ion collisions using dipole models fitted to HERA deep inelastic scattering data, finding that the absolute normalization has large model dependence but the rapidity dependence is more tightly constrained.

## Method

The authors use two phenomenological dipole-proton scattering amplitude parametrizations (IIM and a factorized version of IPsat called fIPsat) that include realistic impact parameter dependence, fitted to HERA data. They extend the dipole cross section from protons to nuclei using the independent scattering approximation with a Woods-Saxon distribution for nucleon positions, then compute coherent diffraction by averaging the amplitude over nucleon configurations and incoherent diffraction as the variance. The cross sections for ultraperipheral heavy ion collisions are obtained by convoluting the photon flux from one nucleus with the photon-nucleus scattering cross section.

## Key numbers

- Coherent Ψ(2S) production at midrapidity in lead-lead collisions at √s = 2.76 TeV: dσ/dy = 0.83 ± 0.19 mb (ALICE data), compared to theoretical values of 0.64–0.65 mb using fIPsat or IIM dipole cross sections with Boosted Gaussian wave function
- Ψ(2S) Boosted Gaussian wave function parameters for transverse polarization: R₂s = 1.851 GeV⁻¹, α₂s,₁ = -0.55816, N₂ = 0.7394

## Context

This work extends the Color Glass Condensate framework for diffractive vector meson production from electron-proton collisions (where HERA data from H1 and ZEUS experiments provide constraints) to heavy ion collisions, where no small-x nuclear DIS data exists. The authors use the optical Glauber model as done in Lappi (2013) to obtain dipole-nucleus cross sections, and their results can be tested against ALICE data from Abbas (2013). Future electron-ion colliders such as LHeC and eRHIC will provide direct lepton-nucleus DIS measurements to further constrain these models.
