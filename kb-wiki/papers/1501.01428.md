# On Fermionic Shadow Wave Functions for strongly-correlated multi-reference systems based on a single Slater determinant

> We demonstrate that extending the Shadow Wave Function to fermionic systems facilitates to accurately calculate strongly-correlated multi-reference systems such as the stretched H2 molecule. This development considerably extends the scope of electronic structure calculations and enables to efficiently recover the static correlation energy using just a single Slater determinant.

## Key claim

The authors demonstrate that the Fermionic Shadow Wave Function (FSWF) and its antisymmetric variant (ASWF) can recover both dynamic and static correlation energy for strongly-correlated multi-reference systems using only a single Slater determinant, overcoming the limitation that standard Jastrow-Slater trial wave functions fail for stretched bonds.

## Method

The authors apply Variational Monte Carlo (VMC) with several trial wave functions to compute the binding energy curve of the H2 molecule across a range of proton-proton distances. They compare a standard Jastrow-Slater (JS) trial wave function using either plane-wave orbitals or DFT orbitals (from Quantum Espresso with PBE functional) against the ASWF and FSWF, which introduce auxiliary shadow coordinates integrated over with a Gaussian coupling. The variational parameters are optimized using a modified Stochastic Reconfiguration method.

## Key numbers

- The JS trial wave function with DFT orbitals substantially overestimates the potential energy of the H2 molecule upon dissociation, while the ASWF and FSWF quantitatively reproduce the exact full configuration interaction reference data.
- The FSWF energy difference from the exact full CI reference is approximately constant across the bond stretching coordinate, indicating correct recovery of static correlation.
- DFT orbitals were computed with a plane-wave cutoff of just 2 Ry using the bare Coulomb pseudopotential and PBE exchange-correlation functional.
- The Yukawa-Jastrow pseudopotential used for electron-electron and electron-proton correlations has the form u(r) = A (1 - e^{-Fr})/r, with both A and F treated as variational parameters.

## Context

This work extends the Shadow Wave Function formalism, originally introduced by Kalos and coworkers (1988), from bosonic to fermionic systems for electronic structure calculations. It addresses the well-known multi-reference problem of the stretched H2 molecule, which Hartree-Fock and DFT methods fail to describe correctly (Cohen, 2008) and which typically requires multi-determinant wave functions (as in full configuration interaction QMC methods). The FSWF enables efficient single-determinant VMC calculations for strongly-correlated systems while retaining size extensivity, unlike truncated configuration interaction approaches.
