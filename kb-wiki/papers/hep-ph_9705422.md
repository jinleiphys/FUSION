# Higher Order Bose-Einstein Correlations for Systems with Large Halo

> A generalization of the core/halo model to the Bose-Einstein correlation function of any number of particles is presented. In particular, a simple prediction is obtained for the intercept parameter of the n-particle inclusive Bose-Einstein correlation functions for arbitrary large values of n.

## Key claim
The paper presents a generalization of the core/halo model to Bose-Einstein correlation functions of any number of particles, obtaining a simple prediction for the intercept parameter of the n-particle correlation function for arbitrarily large n, under the condition that the emission function can be separated into a core and a halo part and that the source is completely chaotic with no partial coherence.

## Method
The author uses the Wigner-function formalism to describe boson emission, assuming the emission function separates into a central core (resolvable by intensity interferometry) and a surrounding halo (unresolvable due to large spatial scales). The n-particle Bose-Einstein correlation function is expressed in terms of Fourier-transformed Wigner-functions and momentum-dependent core fractions, with the summation over all permutations of n indices. A recurrence relation for the number of fully mixing permutations is derived to compute the intercept parameter for arbitrary n.

## Key numbers
- For the NA44 data on central S + Pb reactions at 200 A GeV, the measured two-particle intercept parameter is λ*₂ ≈ 0.52 ± 0.02, approximately independent of transverse mass.
- From λ*₂ = 0.52, the core fraction is extracted as f_c = 0.72.
- Predicted higher-order intercept parameters for the NA44 acceptance: λ*₃ = 2.3, λ*₄ = 8.6, λ*₅ = 33.4, λ*₆ = 148.0.
- The number of fully mixing permutations for j indices: α₁ = 0, α₂ = 1, α₃ = 2, α₄ = 9, α₅ = 44, α₆ = 265.

## Context
This work extends the core/halo model, previously formulated for two-particle Bose-Einstein correlations in Csorgo (1995), to arbitrary order n. It builds on earlier numerical simulations of resonance halo effects by Pratt and Csorgo (1990) and analytical approaches by Bowler (1991) and others. The results enable experimental tests of the core/halo picture using higher-order correlation functions, particularly with NA44 data, and show formal analogy to correlation functions from completely chaotic sources with particle mis-identification as studied by Cramer and Kadija. The paper also notes that the separation of core from halo depends on the experimental momentum resolution Q_min, typically around 10-15 MeV.
