# Coordinate-space solution of the Skyrme-Hartree-Fock-Bogolyubov equations within spherical symmetry. The program HFBRAD (v1.0)

> We describe the first version (v1.00) of the code HFBRAD which solves the Skyrme-Hartree-Fock or Skyrme-Hartree-Fock-Bogolyubov equations in the coordinate representation within the spherical symmetry. A realistic representation of the quasiparticle wave functions on the space lattice allows for performing calculations up to the particle drip lines. Zero-range density-dependent interactions are used in the pairing channel. The pairing energy is calculated by either using a cut-off energy in the quasiparticle spectrum or the regularization scheme proposed by A. Bulgac and Y. Yu.

## Key claim

The paper describes the first official release (v1.00) of the HFBRAD code, which solves the Skyrme-Hartree-Fock-Bogolyubov equations in coordinate representation under spherical symmetry, enabling calculations up to the particle drip lines; however, the authors caution that the code must be used carefully for open-shell nuclei because deformation effects not included in the spherical assumption are essential for those systems.

## Method

The program determines the two-component Hartree-Fock-Bogolyubov quasiparticle wave functions on a lattice of equidistant points in the radial coordinate by solving the eigensystem of two second-order differential equations using the Numerov method, then applying a standard iterative procedure to find self-consistent solutions for the nuclear product wave functions and densities. Zero-range density-dependent interactions are used in the pairing channel, and the pairing energy is calculated either by using a cut-off energy in the quasiparticle spectrum or the regularization scheme proposed by Bulgac and Yu (2002).

## Key numbers

- One Hartree-Fock iteration takes about 0.4 seconds for a medium mass nucleus; convergence is achieved in about 40 seconds.
- The code requires approximately 30 MBytes of memory to execute with typical data.
- The distributed program contains 5164 lines, of which 1635 are comments and separators, totaling 400 kbytes.
- In the illustrative calculation for Sn-174, the neutron Fermi energy is lambda_n = -0.345 MeV and the lowest l=0 quasiparticle state has E = 0.429 MeV for a box radius of 30 fm.

## Context

This work extends the coordinate-space spherical HFB approach first described by Dobaczewski (1984) and provides the first officially published version of a code that had been distributed informally for years. The code complements the published HFBCS code for Skyrme interactions by Reinhard (1991) and the relativistic mean-field Hartree-Bogolyubov code by Poschl et al. (1997), while also implementing the pairing regularization scheme of Bulgac and Yu (2002). The availability of this fast spherical code (tens of seconds per nucleus) enables quick estimates of nuclear properties across the nuclear chart, including near drip lines, which can precede much more time-consuming calculations beyond spherical symmetry.
