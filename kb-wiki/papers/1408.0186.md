# Heavy Ion Collision evolution modeling with ECHO-QGP

> We present a numerical code modeling the evolution of the medium formed in relativistic heavy ion collisions, ECHO-QGP. The code solves relativistic hydrodynamics in $(3+1)-$D, with dissipative terms included within the framework of Israel-Stewart theory; it can work both in Minkowskian and in Bjorken coordinates. Initial conditions are provided through an implementation of the Glauber model (both Optical and Monte Carlo), while freezeout and particle generation are based on the Cooper-Frye prescription. The code is validated against several test problems and shows remarkable stability and accuracy with the combination of a conservative (shock-capturing) approach and the high-order methods employed. In particular it beautifully agrees with the semi-analytic solution known as Gubser flow, both in the ideal and in the viscous Israel-Stewart case, up to very large times and without any ad hoc tuning of the algorithm.

## Key claim

ECHO-QGP reproduces the semi-analytic Gubser flow solution (both ideal and viscous Israel-Stewart) with high accuracy (discrepancy on average of order 0.1%) and up to late times (verified up to 10 fm/c) without any ad hoc tuning of the algorithm or modifications of the reconstruction algorithm.

## Method

The code solves (3+1)-D relativistic hydrodynamics with dissipative terms in the Israel-Stewart framework, using either Minkowskian or Bjorken coordinates. Initial conditions are provided through an implementation of the Glauber model (both Optical and Monte Carlo), while freezeout and particle generation use the Cooper-Frye prescription. The code employs a conservative (shock-capturing) approach combined with high-order numerical methods, and was validated against shock-tube problems and the Gubser flow solution.

## Key numbers

- Gubser flow test: discrepancy between ECHO-QGP and semi-analytic solution is on average of order 0.1% for temperature and shear stress tensor components
- Gubser flow test: agreement verified up to 10 fm/c in evolution time
- Shock tube test: performed on a 201-point square grid of size 10 fm along x and y, evolved from t=1 to t=4 fm/c, with initial temperatures T^L = 0.4 GeV and T^R = 0.2 GeV
- Gubser flow simulation: grid spacing of 0.025 fm in space and 0.001 fm in time; shear viscosity to entropy density ratio set to eta/s = 0.2; shear relaxation time tau_R = 5 eta/(e+P); energy scale q = 1 fm^{-1}

## Context

This work extends the validation of the ECHO-QGP code (originally presented by Del Zanna (2013)) by demonstrating agreement with the Gubser flow solution, which was derived for ideal fluids by Gubser (2010) and extended to viscous Israel-Stewart hydrodynamics by Marrochio (2013). The code has been used to investigate fluctuation propagations (Floerchinger, 2013, 2014) and is being applied to study vorticity effects on directed flow. This enables reliable modeling of heavy-ion collision evolution with dissipative effects, including shear viscosity and relaxation time, without requiring parameter fine-tuning.
