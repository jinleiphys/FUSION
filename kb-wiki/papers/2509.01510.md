# Dirac states from the 't Hooft model

> The dynamics of a light fermion bound to a heavy one is expected to be described by the Dirac equation with an external potential. The potential breaks translation invariance, whereas the bound state momentum is well defined. Boosting the bound state determines the frame dependence of the light fermion dynamics. I study the Dirac limit of QCD$_2$ in the limit of $N_c \to \infty$. The light quark wave function turns out to be independent of the frame of the bound state, up to an irrelevant Lorentz contraction. The discrete bound state spectrum determines corresponding discrete energies of the Dirac equation, which for a linear potential allows a continuous spectrum.

## Key claim

The light quark wave function in QCD$_2$ at $N_c \to \infty$ reduces to the Dirac equation with a linear potential in all frames, but the discrete bound state spectrum of the $q\bar q$ system determines a discrete set of energies for the Dirac equation, which for a linear potential would otherwise allow a continuous spectrum.

## Method

The author studies QCD in 1+1 dimensions (QCD$_2$) in the large-$N_c$ limit using canonically quantized $q\bar q$ states in temporal gauge, following the method of Hoyer (2025). The bound state equation for quarks with masses $m_1$ and $m_2$ is solved analytically using confluent hypergeometric functions, and the limit $m_2 \to \infty$ is taken to extract the Dirac equation for the light quark in a linear potential $V = V'|x|$. The approach is verified for both the rest frame and for bound states with non-zero center-of-mass momentum $P$.

## Key numbers

- For $m_1 = 0.14\sqrt{V'}$ and $P=0$, the Dirac energy extracted from the $m_2 \to \infty$ limit is $M_D = 1.532\sqrt{V'}$, obtained by fitting a fourth-order polynomial in $1/m_2$ to numerical values for $m_2 \leq 100\sqrt{V'}$.
- The difference between the $q\bar q$ and Dirac wave functions converges proportionally to $1/m_2$, as shown at $x = 4.0/\sqrt{V'}$.
- At $m_2 = 20\sqrt{V'}$, the deviation between bound state and Dirac wave functions at $x = 5.0/\sqrt{V'}$ is commensurate with the ratio $V'x/m_2 = 0.25$.

## Context

This work extends the study of QCD$_2$ at large $N_c$ originally pioneered by 't Hooft (1974) using light-front quantization, and later studied by Bars and Green (1977) using equal-time quantization with modified propagator prescriptions. The author uses a recent canonical quantization method from Hoyer (2025) that avoids singular propagators and gives wave functions that do not vanish at zero momentum fraction. The results contradict a previous study by Kalashnikova (1997) which found non-local interactions with scalar and space-like vector potentials in the heavy-quark limit. This work enables future extension to $D=3+1$ dimensions using the relativistic bound state framework described in Hoyer (2021).
