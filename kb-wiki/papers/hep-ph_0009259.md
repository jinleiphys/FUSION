# The gamma -> 3 pi form factor as a constraint on Schwinger-Dyson modeling of light quarks

> The form factor for gamma pi^+ -> pi^+ pi^0 was calculated in a simple-minded constituent model with a constant quark mass parameter, as well as in the Schwinger-Dyson approach. The comparative discussion of these and various other theoretical results on this anomalous process, as well as the scarce already available data (hopefully to be supplemented by more accurate CEBAF data soon), seem to favor Schwinger-Dyson modeling which would yield relatively small low-momentum values of the constituent (dynamically dressed) quark mass function.

## Key claim

The form factor for the anomalous process γπ⁺ → π⁺π⁰, calculated in both a constant-mass constituent quark model and a Schwinger-Dyson (SD) approach, suggests that SD modeling yielding relatively small low-momentum values of the dynamically dressed quark mass function is favored by existing data and theoretical comparisons, though the only available experimental point (Serpukhov) lies above all predictions and is likely an overestimate.

## Method

The form factor was computed as a quark "box" amplitude in two approaches: first, using a simple constituent quark model with a constant quark mass parameter M; second, using the Schwinger-Dyson approach with a momentum-dependent quark mass function M(k²) from the dressed quark propagator, employing the Ball-Chiu Ansatz for the dressed quark-photon vertex. The kinematics considered the photon and all three pions on shell, with q² = 0 and all pion masses at m_π = 138.5 MeV (or in the chiral limit m_π = 0).

## Key numbers

- The constant-mass quark model predictions for the normalized γ3π form factor are shown for M = 330 MeV (long-dashed curve), M = 400 MeV (line of empty boxes), and M = 580 MeV (line of crosses).
- The Schwinger-Dyson approach yields a solid curve for the empirical pion mass m_π = 138.5 MeV and a dashed curve for the chiral limit m_π = 0.
- The momentum-dependent mass function in the adopted SD model has a typical value for light u,d quarks of M(k² ≈ 0) ≈ 360 MeV.
- Some other SD models obtain significantly higher values, M(k² ≈ 0) ≈ 600 MeV and more.
- The Ball-Chiu Ansatz for the dressed quark-photon vertex is adequate for the real photon case (q² = 0), as it approximates the true solution well for -0.4 GeV² < q² < 0.2 GeV².

## Context

This work extends earlier calculations of the γ → 3π form factor using the quark loop model (Klabucar and Bistrovic, 1999) and the Schwinger-Dyson approach (Klabucar and Bistrovic, 1999), comparing them with vector meson dominance (Rudaz, 1984) and chiral perturbation theory (Holstein, 1996). The results indicate that accurate CEBAF data, expected soon (Miskimen et al., 1994), should constrain the infrared behavior of the quark mass function M(k²) and thus the whole SD modeling, potentially requiring reformulation if the experimental form factor is measured to be higher than the SD predictions.
