# Application of radial basis functions neutral networks in spectral functions

> The reconstruction of spectral function from correlation function in Euclidean space is a challenging task. In this paper, we employ the Machine Learning techniques in terms of the radial basis functions networks to reconstruct the spectral function from a finite number of correlation data. To test our method, we first generate one type of correlation data using a mock spectral function by mixing several Breit-Wigner propagators. We found that compared with other traditional methods, TSVD, Tikhonov, and MEM, our approach gives a continuous and unified reconstruction for both positive definite and negative spectral function, which is especially useful for studying the QCD phase transition. Moreover, our approach has considerably better performance in the low frequency region. This has advantages for the extraction of transport coefficients which are related to the zero frequency limit of the spectral function. With the mock data generated through a model spectral function of stress energy tensor, we find our method gives a precise and stable extraction of the transport coefficients.

## Key claim

The paper claims that a radial basis function network (RBFN) approach provides a continuous and unified reconstruction of both positive definite and negative spectral functions from Euclidean correlation data, with considerably better performance in the low frequency region compared to traditional methods (TSVD, Tikhonov, MEM), but this claim is conditioned on tests using only mock data generated from known model spectral functions (Breit-Wigner propagators and a stress-energy tensor model), not on actual lattice QCD or experimental data.

## Method

The authors discretize the Fredholm integral equation relating the Euclidean correlation function G(tau) to the spectral function rho(omega) and represent rho(omega) as a linear combination of radial basis functions (Gaussian or multiquadric) centered at each frequency grid point. The weights of the RBF expansion are then obtained by solving the resulting linear system using truncated singular value decomposition (TSVD) on the transformed kernel matrix, with 30 mock correlation data points and 500 frequency grid points. The method is tested on two types of mock spectral functions: a mixture of Breit-Wigner propagators and a model spectral function of the stress-energy tensor, with Gaussian noise added to the correlation data at levels epsilon = 0.001, 0.0001, and 0.00001.

## Key numbers

*   The RBFN method uses 30 correlation data points (widehat{N} = 30) and 500 frequency grid points (M = N = 500) for the spectral function reconstruction.
*   Gaussian noise levels added to the mock correlation data are epsilon = 0.001, 0.0001, and 0.00001 for testing noise sensitivity.
*   The singular values s_i of the kernel matrix decrease exponentially, motivating the use of TSVD truncation.

## Context

This work extends the application of machine learning techniques to the ill-posed inverse problem of reconstructing spectral functions from Euclidean correlation functions, a challenge central to lattice QCD and functional QCD methods. It directly compares its RBFN approach against three established methods: truncated singular value decomposition (TSVD) as developed by Hansen (1990), Tikhonov regularization as described by Groetsch (1984) and Dudal (2013), and the maximum entropy method (MEM) as applied by Asakawa (2000) and others. The paper aims to overcome limitations of these traditional methods, particularly their reliance on smoothness assumptions and their difficulty in handling negative spectral functions, which are relevant for studying QCD phase transitions and extracting transport coefficients like shear viscosity via the Kubo formula.
