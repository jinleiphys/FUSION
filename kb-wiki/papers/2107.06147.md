# QTRAJ 1.0: A Lindblad equation solver for heavy-quarkonium dynamics

> We introduce an open-source package called QTraj that solves the Lindblad equation for heavy-quarkonium dynamics using the quantum trajectories algorithm. The package allows users to simulate the suppression of heavy-quarkonium states using externally-supplied input from 3+1D hydrodynamics simulations. The code uses a split-step pseudo-spectral method for updating the wave-function between jumps, which is implemented using the open-source multi-threaded FFTW3 package. This allows one to have manifestly unitary evolution when using real-valued potentials. In this paper, we provide detailed documentation of QTraj 1.0, installation instructions, and present various tests and benchmarks of the code.

## Key claim

The paper introduces QTraj 1.0, a CPU-based open-source package that solves the Lindblad equation for heavy-quarkonium dynamics using the quantum trajectories algorithm, achieving computational efficiency comparable to previous GPU-based implementations through a split-step pseudo-spectral method combined with FFTW3.

## Method

The code implements the Monte-Carlo wave-function (quantum trajectories) method with a waiting-time approach to solve the Lindblad equation for the reduced density matrix of heavy quarkonium in a quark-gluon plasma. Between quantum jumps, the wave-function is evolved using a non-Hermitian effective Hamiltonian via a split-step pseudo-spectral method that uses FFTW3 for Fourier transforms, providing manifestly unitary evolution when real-valued potentials are used. The package supports both first-principles potentials from pNRQCD (potential non-relativistic QCD) and phenomenological KMS-type potentials with isotropic or anisotropic variants.

## Key numbers

- The code supports lattice sizes of `num` grid points where `2^n` gives best performance, with box size `L` in units of GeV$^{-1}$
- Time step `dt` is specified in units of GeV$^{-1}$, with maximum number of time steps set by `maxSteps`
- The QGP lifetime is stated as $t_\text{QGP} \lesssim 15\ \text{fm/c}$
- The temperature threshold for deconfinement is given as $T \gtrsim 2 \times 10^{12}$ K
- The code outputs masses of the 1S, 2S, 1P, 3S, 2P, and 1D vacuum eigenstates when using the built-in eigensolver with `initType = 100`

## Context

This work extends the quantum trajectories approach previously applied to quarkonium phenomenology in Brambilla (2020) using GPUs, by providing a CPU-based implementation that is more accessible to practicing scientists. The code solves the Lindblad equations derived from the OQS+pNRQCD formalism developed by Brambilla (2016, 2017), and includes phenomenological KMS-type potentials from Karsch (1987) and Strickland (2011). The package enables systematic studies of in-medium quarkonium suppression including collisional dissociation, color screening, and recombination effects, and can be interfaced with external 3+1D hydrodynamics simulations.
