# Correct Definition of Color Singlet P-Wave Non-Perturbative Matrix Element of Heavy Quarkonium Production

> Recently we have proved factorization of infrared divergences in NRQCD S-wave heavy quarkonium production at high energy colliders at all orders in coupling constant. One of the problem which still exists in the higher order pQCD calculation of color singlet P-wave heavy quarkonium production/anihillation is the appearance of non-canceling infrared divergences due to real soft gluons exchange, although no such infrared divergences are present in the color singlet S-wave heavy quarkonium. In this paper we find that since the non-perturbative matrix element of the color singlet P-wave heavy quarkonium production contains derivative operators, the gauge links are necessary to make it gauge invariant and be consistent with the factorization of such non-canceling infrared divergences at all orders in coupling constant.

## Key claim

The paper claims that the non-canceling infrared divergences in higher-order pQCD calculations of color singlet P-wave heavy quarkonium production/annihilation arise because the conventional non-perturbative matrix element containing ordinary derivative operators is not gauge invariant; the correct gauge-invariant definition requires the insertion of gauge links (Wilson lines) to absorb these divergences at all orders in the coupling constant.

## Method

The author uses the path integral formulation of QCD and the background field method to analyze the infrared behavior of color singlet P-wave heavy quarkonium production. By introducing a pure gauge background field generated by light-like eikonal currents, the paper demonstrates that the non-perturbative matrix element with ordinary derivatives can be factorized into a gauge-invariant form containing gauge links. The derivation proceeds by changing integration variables in the path integral and applying type I gauge transformations to show equivalence between the original and gauge-linked definitions.

## Key numbers

- The S-wave non-perturbative matrix element is given by \(|R(0)|^2 = \frac{2\pi}{3} \langle 0|\mathcal{O}_H|0\rangle\) (equation for S-wave, no numerical value provided).
- The conventional P-wave non-perturbative matrix element is given by \(|R'(0)|^2 = \frac{2\pi}{27} \langle 0|\mathcal{O}_H|0\rangle\) (equation for P-wave, no numerical value provided).
- The evolution equation relating color octet and color singlet matrix elements in NRQCD contains the factor \(\frac{4C_F \alpha_s(\Lambda)}{3N_c\pi M^2}\) (equation 6.5 of NRQCD reference, no numerical value provided).

## Context

This work extends the author's previous proof of infrared divergence factorization in NRQCD S-wave heavy quarkonium production (Nayak, 2017) to the P-wave case. It addresses the long-standing problem of non-canceling infrared divergences in color singlet P-wave heavy quarkonium production/decay found by Barbieri et al. (1976) and others, which had previously been handled in NRQCD by introducing color octet matrix elements (Bodwin, Braaten, Lepage, 1995). The correct gauge-invariant definition enables consistent factorization of these divergences without invoking color octet mechanisms, and the result is independent of the light-like vector used to define the gauge link.
