# Lattice determination of the Batalin-Vilkovisky function and the strong running interaction

> The Batalin-Vilkovisky function is a central component in the modern formulation of the background field method and the physical applications derived from it. In the present work we report on novel lattice results for this particular quantity, obtained by capitalizing on its equality with the Kugo-Ojima function in the Landau gauge. The results of the lattice simulation are in very good agreement with the predictions derived from a continuum analysis based on the corresponding Schwinger-Dyson equations. In addition, we show that an important relation connecting this function with the ghost propagator is fulfilled rather accurately. With the aid of these results, we carry out the first completely lattice-based determination of the process-independent strong running interaction, employed in a variety of phenomenological studies.

## Key claim

The authors report the first purely lattice-based determination of the process-independent strong running interaction, obtained by capitalizing on the equality between the Batalin-Vilkovisky (BV) function and the Kugo-Ojima function in the Landau gauge, and they find that the lattice results for the BV function are in very good agreement with predictions from Schwinger-Dyson equations.

## Method

The BV function was computed on the lattice by exploiting its equality with the Kugo-Ojima function, which is defined through a two-point function of composite operators involving the covariant derivative, ghost fields, and gluon fields. Quenched lattice ensembles were generated with the Wilson gauge action at beta = 6.0 using two volumes (64^4 and 80^4), and the Landau gauge was fixed by identifying configurations belonging to the Gribov region via Fourier Accelerated Steepest Descent. The bare lattice results were renormalized using the Taylor scheme by imposing the exact relation between the BV function, the ghost dressing function, and the form factor L(q), with the parameters determined through a chi-squared minimization for momenta below 0.5 GeV.

## Key numbers

* The lattice spacing was a = 0.0962 fm (a^{-1} = 2.05 GeV) for the quenched ensembles at beta = 6.0, determined via the Sommer parameter method.
* The renormalization point was set at mu = 4.3 GeV, where the perturbative one-loop value gave 1 + G(mu) = 0.974 and the SDE result gave 1 + G(mu) = 0.973.
* The Taylor coupling at mu = 4.3 GeV was alpha_s = 0.216.
* The subtractive renormalization constant was determined as s = 0.121 from the combined data of both volumes using 18 points in the fitting window below 0.5 GeV.
* The fit for L(q) used Lambda_T = 425 MeV as the value of Lambda_QCD in the Taylor scheme, with anomalous dimension d_L = 35/44.

## Context

This work extends the continuum analysis of the BV function based on Schwinger-Dyson equations (Aguilar 2009) by providing the first lattice verification of those predictions. The results enable the construction of the process-independent strong running interaction introduced by Binosi (2014) and Cui (2019), which has been used in phenomenological studies of hadron observables. The lattice determination of the BV function and the running interaction strength provides an ab initio check on the nonperturbative continuum QCD methods used to describe basic hadron properties.
