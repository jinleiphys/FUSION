# Spin dependent parton distributions and structure functions

> Nuclear parton distributions and structure functions are determined in an effective chiral quark theory. We also discuss an extension of our model to fragmentation functions.

## Key claim

The paper predicts that the polarized EMC effect (the modification of spin-dependent quark distributions in nuclei compared to free nucleons) is larger than the unpolarized EMC effect for light nuclei, based on calculations in an effective chiral quark theory where the nucleon is described as a quark-diquark bound state.

## Method

The authors describe the nucleon as a bound state of a quark and a diquark using the Faddeev framework in the Nambu-Jona-Lasinio (NJL) model, including scalar and axial-vector diquark channels with confinement effects implemented via proper-time regularization. Finite nuclei are described in an independent particle picture using depth parameters of scalar and vector potentials from earlier self-consistent nuclear matter calculations, which are translated into average fields for quarks via the Faddeev equation. The quark distributions in bound nucleons are then calculated, and nuclear structure functions are obtained using a convolution formalism with nucleon momentum distributions.

## Key numbers

- For $^{7}$Li, $^{11}$B, and $^{27}$Al, the polarized EMC ratio is predicted to be larger than the unpolarized EMC ratio, corresponding to a reduction of the quark spin sum in the nucleus.
- The polarized structure function $g_{1A}^H$ is of order $1/A$ relative to $F_{2A}$.
- The EMC ratios are defined as $R(x) = F_{2A}(x_A) / [Z F_{2p}(x) + N F_{2n}(x)]$ and $R_s^H(x) = g^H_{1A}(x_A) / [P_p^H g_{1p}(x) + P_n^H g_{1n}(x)]$, where $x_A$ is $A$ times the Bjorken variable for the nucleus of mass number $A$.

## Context

This work extends the authors' earlier calculations of the unpolarized EMC effect (Cloet 2006) to the spin-dependent case, using the same effective chiral quark theory framework. The paper also discusses an extension to fragmentation functions via the Drell-Levy-Yan (DLY) relation (Drell 1969), which connects distribution and fragmentation functions through crossing and charge conjugation symmetries. The results enable future experimental searches for the polarized EMC effect in light nuclei such as $^{7}$Li, $^{11}$B, and $^{27}$Al, where the polarization is dominated by protons.
