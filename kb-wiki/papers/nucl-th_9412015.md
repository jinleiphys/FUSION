# Two neutrino double beta decay within the $ξ$-approximation

> We examine the contributions of odd-parity nuclear operators to the two-neutrino double beta decay $0^+\rightarrow 0^+$ amplitude, which come from the $P$-wave Coulomb corrections to the electron wave functions and the recoil corrections to the nuclear currents. Although they are formally of higher order in $αZ/2$ or $v/c$ of the nucleon than the usual Fermi and Gamow-Teller matrix elements, explicit calculations performed within the QRPA show that they are significant when confronted with the experimental data.

## Key claim

The paper argues that odd-parity (first-forbidden) nuclear operators, which are formally higher-order corrections, contribute significantly to the two-neutrino double beta decay amplitude within the ξ-approximation, and that these contributions can be comparable to the measured matrix elements for some nuclei, particularly when the allowed and forbidden matrix elements cancel.

## Method

The authors derived the half-life expression for 0⁺→0⁺ two-neutrino double beta decay within the ξ-approximation, which retains P-wave Coulomb corrections to electron wave functions and recoil corrections to nuclear currents. They evaluated the first-forbidden matrix elements using the quasiparticle random phase approximation (QRPA) with the same residual interaction and configuration space as in their previous work, employing the conserved-vector-current (CVC) relation for the <v> matrix element and renormalizing the <σ·v> matrix element by one-pion exchange current contributions.

## Key numbers

- The ξ parameter is given by ξ = αZ/(2R) ≈ 1.18 Z A^{-1/3} (equation 7)
- For the transitions ⁸²Se → ⁸²Kr and ¹³⁰Te → ¹³⁰Xe, the Williams and Haxton calculation yields M_{2ν}^{FF}(0⁻) ≈ 0.010, while neglecting the <iσ·r> matrix element gives M_{2ν}^{FF}(0⁻) ≈ 0.006 in the present QRPA evaluation
- The suppression factor for the allowed matrix element |M_{2ν}| ranges from 1/2 for ¹⁰⁰Mo to 1/15 for ¹³⁰Te when compared with the naive closure estimate M_{2ν} ∼ -0.3
- The oscillator frequency is ω₀ ≈ 80 A^{-1/3} MeV
- The ratio of velocity-dependent to position-dependent matrix elements is approximately -68/Z, of order unity for medium-heavy nuclei

## Context

This work extends the earlier study by Williams and Haxton (1988), who first pointed out the potential significance of the <σ·v> matrix element for two-neutrino double beta decay but did not simultaneously examine all five first-forbidden moments. The paper builds on the QRPA framework developed by Krmpotić (1993), which showed that the allowed matrix element M_{2ν}^A passes through zero near the natural value of the particle-particle strength (t ∼ 1.5), and demonstrates that the first-forbidden contributions can be significant when the allowed matrix element is suppressed. The results enable future studies of 2⁻ virtual states, which might modify the electron spectrum shape and the phase space factor predicted by the allowed approximation.
