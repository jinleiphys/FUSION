# Sum rule of the correlation function

> We discuss a sum rule satisfied by the correlation function of two particles with small relative momenta. The sum rule, which results from the completeness condition of the quantum states of the two particles, is first derived and then we check how it works in practice. The sum rule is shown to be trivially satisfied by free particle pair, and then there are considered three different systems of interacting particles. We discuss a pair of neutron and proton in the s-wave approximation and the case of the so-called hard spheres with the phase shifts taken into account up to l=4. Finally, the Coulomb system of two charged particles is analyzed.

## Key claim

The sum rule for the two-particle correlation function, which relates the momentum integral of the correlation function to the source function at zero separation and bound-state formation rates, is formally exact but fails in practice for the most important case of Coulomb-interacting particles because the integral diverges; for neutron-proton pairs and hard spheres, the sum rule is violated when using realistic approximations, suggesting it is either impractical or meaningless.

## Method

The authors first derive the sum rule from the quantum-mechanical completeness relation for two-particle wave functions, obtaining an expression that relates the integral of the correlation function minus unity to the source function at zero separation and bound-state formation rates. They then test the sum rule by computing correlation functions for four systems: free particles (analytically), neutron-proton pairs in the s-wave approximation using scattering lengths and effective ranges with a Gaussian source, hard spheres with phase shifts up to l=4, and Coulomb-interacting particles using the exact Coulomb wave function. For each interacting system, they compute the function S(q_max) = 4π ∫₀^{q_max} dq q² (R(q)-1) and compare its asymptotic value to the sum rule prediction.

## Key numbers

- Neutron-proton singlet scattering length a^s = -23.7 fm, effective range d^s = 2.7 fm; triplet scattering length a^t = 5.4 fm, effective range d^t = 1.7 fm (from McCarthy (1968))
- Deuteron wave function Hulthén parameters: α = 0.23 fm⁻¹, β = 1.61 fm⁻¹ (from Hodgson (1971))
- Hard sphere diameter a = 1 fm, source size parameters r₀ = 4 fm and r₀ = 6 fm
- For free particles with Gaussian source of width r₀, the sum rule integral saturates at q_max not much exceeding 1/(2r₀)
- For Coulomb-interacting non-identical particles, (R(q)-1) ∼ 1/q² as q → ∞, causing the sum rule integral to diverge linearly

## Context

This work extends the sum rule first derived by Mrowczynski (1994) and tests its practical applicability, contradicting the expectation that it could be used to extract particle phase-space density without removing Coulomb interactions (Bertsch (year), Brown (2000)). The paper shows that the sum rule fails for Coulomb systems, which is problematic because the method of determining phase-space density (Bertsch (year), Brown (2000)) relies on this integral. The results also constrain the imaging method of Brown (1997, 2000) and affect the interpretation of correlation functions for exotic systems like antiproton-lambda (Kisiel (2004)).
