# Coupled-cluster calculations of neutrinoless double-beta decay in $^{48}$Ca

> We use coupled-cluster theory and nuclear interactions from chiral effective field theory to compute the nuclear matrix element for the neutrinoless double-beta decay of $^{48}$Ca. Benchmarks with the no-core shell model in several light nuclei inform us about the accuracy of our approach. For $^{48}$Ca we find a relatively small matrix element. We also compute the nuclear matrix element for the two-neutrino double-beta decay of $^{48}$Ca with a quenching factor deduced from two-body currents in recent ab-initio calculation of the Ikeda sum-rule in $^{48}$Ca [Gysbers et al., Nature Physics 15, 428-431 (2019)].

## Key claim

The coupled-cluster calculation yields a neutrinoless double-beta decay nuclear matrix element for ⁴⁸Ca in the range 0.25 ≤ M⁰ν ≤ 0.75 (CCSDT-1 approximation), which is relatively small and consistent with other ab initio results, but the uncertainty bracket arises from the choice of spherical versus deformed reference state and the treatment of triples correlations.

## Method

The authors employ the intrinsic Hamiltonian with chiral nucleon-nucleon and three-nucleon potentials (1.8/2.0 (EM) interaction) in a harmonic-oscillator basis with frequency ℏΩ = 16 MeV and model space up to Nmax = 10. They use coupled-cluster theory at the CCSD and CCSDT-1 levels, transforming the Hamiltonian to a natural-orbital basis, and compute the 0νββ matrix element via equation-of-motion coupled-cluster techniques starting from either a spherical ⁴⁸Ca reference or a deformed ⁴⁸Ti reference. Benchmarks against no-core shell model results in light nuclei (A = 6, 8, 10, 14, 22) validate the approach, and the 2νββ matrix element is computed using a Lanczos continued-fraction method with the similarity-transformed Gamow-Teller operator.

## Key numbers

- 0νββ nuclear matrix element for ⁴⁸Ca → ⁴⁸Ti: 0.25 ≤ M⁰ν ≤ 0.75 (CCSDT-1 approximation)
- 2νββ nuclear matrix element for ⁴⁸Ca → ⁴⁸Ti: M²ν = 0.065 ± 0.002 (converged CCSDT-1 result)
- Quenched 2νββ matrix element: q²M²ν = 0.042 ± 0.001, using quenching factor q² = 0.81² from two-body currents (Gysbers 2019)
- Experimental 2νββ matrix element for comparison: M²ν = 0.035 ± 0.003 (Barabash 2019, Stoica 2013)
- Computed energy difference (EI − EF)/2 = 1.32 MeV; experimental value = 1.35 MeV
- Energy of first 1⁺ state in ⁴⁸Sc: ΔEμ=0 = 2.93 MeV (computed) versus 3.02 MeV (experimental)
- Closure energy for ⁴⁸Ca → ⁴⁸Ti: Ecl = 7.72 MeV
- With contact term coupling strength g = ±1 fm², NME range becomes 0.15 ≤ M⁰ν ≤ 1.02

## Context

This work extends ab initio coupled-cluster calculations to the 0νββ decay of ⁴⁸Ca, a candidate nucleus for next-generation experiments, and provides the first coupled-cluster result for this decay using chiral EFT interactions. The results are consistent with the very recent IMSRG+GCM calculation (Yao 2020) and with other ab initio work (Belley 2020), while differing from more phenomenological approaches such as the realistic shell model (Coraggio 2020), QRPA (Simkovic 2013), IBM (Barea 2015), and various energy-density functional methods (Vaquero 2013, Yao 2015). The 2νββ calculation uses a quenching factor from two-body currents in the Ikeda sum-rule (Gysbers 2019), and the benchmarks against NCSM in light nuclei establish the accuracy of the coupled-cluster approach for these matrix elements. This work enables future calculations in heavier candidate nuclei such as ⁷⁶Ge, ¹³⁰Te, and ¹³⁶Xe, though those will require larger model spaces and inclusion of triaxial deformation and symmetry projection.
