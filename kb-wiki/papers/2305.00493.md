# Estimation of collision centrality in terms of the number of participating nucleons in heavy-ion collisions using deep learning

> The deep learning technique has been applied for the first time to investigate the possibility of centrality determination in terms of the number of participants ($N_{\mathrm{part}}$) in high-energy heavy-ion collisions. For this purpose, supervised learning using both deep neural network (DNN) and convolutional neural network (CNN) is performed with labeled data obtained by modeling relativistic heavy-ion collisions utilizing A Multi-phase Transport Model (AMPT). Event-by-event distributions of pseudorapidity and azimuthal angle of charged hadrons weighted by their transverse momentum are used as input to train the DL models. The DL models did remarkably well in predicting $N_{\mathrm{part}}$ values with CNN slightly outperforming the DNN model. The Mean Squared Logarithmic Error (MSLE) for the CNN model (Model-4) is determined to be 0.0592 for minimum bias collisions and 0.0114 for 0-60\% centrality class, indicating that the model performs better for semi-central and central collisions. Furthermore, the studied DL model is proven to be robust to changes in energy as well as model parameters of the input. The current study demonstrates that the data-driven technique has a distinct potential for determining centrality in terms of the number of participants in high-energy heavy-ion collision experiments.

## Key claim

Deep learning models (DNN and CNN) can predict the number of participating nucleons ($N_{\mathrm{part}}$) in heavy-ion collisions from simulated $p_T$-weighted $\eta$-$\phi$ spectra, with the caveat that performance is significantly better for semi-central and central collisions (0-60% centrality) than for peripheral collisions, and the models were trained and tested exclusively on AMPT-generated events rather than experimental data.

## Method

Minimum bias Au+Au collisions at $\sqrt{s}=200$ GeV were simulated using the AMPT model (string melting version) to generate 180,000 events. Event-by-event $p_T$-weighted pseudorapidity ($\eta$) and azimuthal angle ($\phi$) distributions of charged hadrons within $|\eta|<1$ and $p_T>0.2$ GeV/c were used as input to train four deep learning models: one deep neural network (DNN) and three convolutional neural networks (CNN) with varying architectures. The models performed supervised regression to predict $N_{\mathrm{part}}$ values, with 150,000 events used for training/validation and 30,000 held for testing.

## Key numbers

- Mean Squared Logarithmic Error (MSLE) for the best CNN model (Model-4): 0.0592 for minimum bias collisions and 0.0114 for 0-60% centrality class
- $R^2$ value of approximately 0.98 for all four DL models on the test dataset
- Mean relative error (MRE) for Model-4: 0.1970 for minimum bias, 0.0808 for 0-60% centrality
- MSLE for Model-4 at different beam energies: 0.1790 at $\sqrt{s}=7.7$ GeV, 0.0875 at 39 GeV, 0.0878 at 62.4 GeV, 0.0730 at 130 GeV, and 0.0625 at 200 GeV
- When Model-4 trained on AMPT-SM was tested on AMPT default events, $R^2=0.9784$ and MSLE=0.0703

## Context

This work extends prior efforts to use machine learning for collision geometry determination, which previously focused on impact parameter estimation using neural networks by Bass (1996), Li (2020, 2021) using ANN and CNN with UrQMD, and Xiang (2022) using DNN and CNN with AMPT. The study is the first to apply deep learning specifically to predict $N_{\mathrm{part}}$ rather than impact parameter, and demonstrates robustness across different beam energies and event-generator configurations (AMPT-SM vs. AMPT default). This enables future application to experimental data from RHIC and LHC for centrality determination that is independent of the Glauber model approach used by Miller (2007).
