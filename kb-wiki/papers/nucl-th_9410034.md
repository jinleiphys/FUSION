# Mean Field Studies of Exotic Nuclei}

> {Full three dimensional static and dynamic mean field calculations using collocation basis splines with a Skyrme type Hamiltonian are described. This program is developed to address the difficult theoretical challenges offered by exotic nuclei. Ground state and deformation properties are calculated using static Hartree-Fock, Hartree-Fock+BCS and constrained Hartree-Fock models. Collective properties, such as reaction rates and resonances, are described using a new alternate method for evaluating linear response theory, which is constructed directly on top of the static calculation. This provides a consistent description of the ground state, deformation and collective nuclear properties. Sample results are presented for the giant multiple resonances of $^{16}$O. }

## Key claim

The paper presents a full three-dimensional static and dynamic mean-field framework using collocation basis splines with a Skyrme Hamiltonian, claiming that this approach provides a consistent description of ground-state, deformation, and collective nuclear properties. The hedge is that the dynamic linear response results for \({}^{16}\)O show significant sensitivity to the box size and boundary conditions, indicating that the representation of the continuum by discrete box states is not yet fully under control.

## Method

Static Hartree-Fock (HF), HF+BCS, and constrained HF calculations were performed on a three-dimensional collocation lattice using 7th-order B-splines with a Skyrme-type effective interaction. The dynamic linear response was computed by adding a Gaussian-shaped time-dependent perturbation to the static Hamiltonian, time-evolving the system via the time-dependent Hartree-Fock (TDHF) equations, and Fourier-transforming the resulting density fluctuation to obtain the response spectrum. Pairing correlations were included via constant-gap BCS or Lipkin-Nogami prescriptions.

## Key numbers

* Static HF solutions for \({}^{16}\)O were converged to an energy fluctuation \(\Delta E^2 < 10^{-5}\), corresponding to an energy difference between iterations of about \(10^{-10}\).
* The time step for TDHF evolution was \(\Delta t = 0.4\) fm/c; typical runs used 16384 time steps, giving a maximum time of \(\approx 6550\) fm/c.
* The Gaussian perturbation amplitude \(\varepsilon\) was chosen in the range \(2.0\times 10^{-4} \leq \varepsilon \leq 2\times 10^{-7}\) to remain in the linear regime.
* For the isoscalar quadrupole response of \({}^{16}\)O using the skm* Skyrme parametrization in a \((-10,+10\ \mathrm{fm})^3\) box with \(20^3\) grid points, a sharp peak was found at about 19.8 MeV, recovering about 92% of the exact energy-weighted sum rule.
* Imposing time-reversal symmetry caused small shifts in collective mode positions of about 0.3 MeV.
* For the isoscalar octupole response, three sharp resonance structures were observed at about 6–7 MeV, 13–14 MeV, and 18 MeV for both skm* and sgII interactions.

## Context

This work extends earlier static and dynamic mean-field calculations by introducing collocation basis splines on a three-dimensional lattice, which the authors argue is more accurate than finite-difference grids and more suitable for exotic nuclei than harmonic-oscillator basis expansions (Li 1991). The alternate linear-response method, built directly on the static HF solution, is a refinement of the technique used by Bertsch and Flocard (1975). The paper identifies that the box-size dependence of the response spectrum must be resolved before the method can be reliably applied to \(\beta\)-decay calculations of exotic nuclei, where consistent treatment of deformation and reaction rates is critical.
