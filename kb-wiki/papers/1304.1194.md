# Strangeness and charmness content of nucleon from overlap fermions on 2+1-flavor domain-wall fermion configurations

> We present a calculation of the strangeness and charmness contents <N|\bar{s}s|N> and <N|\bar{c}c|N> of the nucleon from dynamical lattice QCD with 2+1 flavors. The calculation is performed with overlap valence quarks on 2+1-flavor domain-wall fermion gauge configurations. The configurations are generated by the RBC collaboration on a 24^3*64 lattice with sea quark mass am_l=0.005, am_s=0.04, and inverse lattice spacing a^{-1}=1.73GeV. Both actions have chiral symmetry which is essential in avoiding contamination due to the operator mixing with other flavors. Nucleon propagator and the quark loops are both computed with stochastic grid sources, while low-mode substitution and low-mode averaging methods are used respectively which substantially improve the signal to noise ratio. We obtain the strangeness matrix element f_{T_{s}} = m_s <N|\bar{s}s|N> / M_N = 0.0334(62), and the charmness content f_{T_{c}} = m_c <N|\bar{c}c|N> / M_N = 0.094(31) which is resolved from zero by 3σprecision for the first time.

## Key claim

The authors report a lattice QCD calculation of the nucleon's strangeness and charmness scalar matrix elements using overlap valence quarks on 2+1-flavor domain-wall fermion configurations, obtaining the charmness content resolved from zero at 3σ precision for the first time. The results depend on the use of chirally symmetric fermions to avoid operator mixing with other flavors, which the authors argue is essential for avoiding the large systematic errors seen in Wilson-type fermion calculations.

## Method

The calculation was performed on a 24³ × 64 lattice with sea-quark masses am_l = 0.005 and am_s = 0.04, and inverse lattice spacing a⁻¹ = 1.73 GeV, using 2+1-flavor domain-wall fermion gauge configurations generated by the RBC collaboration. Overlap valence quarks were used for both the nucleon propagator and the quark loops, with stochastic grid sources for both; low-mode substitution was applied to the nucleon propagator and low-mode averaging to the quark loops to improve the signal-to-noise ratio. The nucleon two-point functions were computed with smeared Z₃ noise-grid sources on two time slices, and the quark loops were estimated using Z₄ noise with dilution in color and Dirac indices.

## Key numbers

- Strangeness matrix element: f_{T_s} = m_s ⟨N|s̄s|N⟩ / M_N = 0.0334(62), with more than 5σ precision
- Charmness content: f_{T_c} = m_c ⟨N|c̄c|N⟩ / M_N = 0.094(31), resolved from zero at 3σ precision
- Lattice parameters: 24³ × 64 lattice, a⁻¹ = 1.73 GeV, sea-quark masses am_l = 0.005, am_s = 0.04
- Nucleon mass from Z₃-grid + LMS + smearing + folding method: 1.14(2) GeV (at valence quark mass am ≈ 0.016, corresponding to m_π ≈ 330 MeV)

## Context

This work extends previous lattice calculations of the strangeness content that used Wilson-type fermions, which gave large values (f_{T_s} ~ 0.1–0.46) due to additive renormalization from chiral symmetry breaking, and instead obtains a small value consistent with calculations using chirally symmetric fermions (Takeda 2010, Toussaint 2009, Engelhardt 2012, Freeman 2012, Dinter 2012, Junnarkar 2013) and the Feynman-Hellman theorem approach (Young 2009). The charmness result improves upon the previous 2σ signal from Freeman (2012) using MILC HISQ configurations, and the combined results enable more precise predictions for spin-independent neutralino-nucleon scattering cross sections in dark matter searches.
