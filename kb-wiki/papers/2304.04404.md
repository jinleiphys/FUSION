# Rotation and vibration in tetraquarks

> A novel approach is introduced for obtaining precise solutions of the pairing Hamiltonian for tetraquarks, which utilizes an algebraic technique in infinite dimensions. The parameters involved in the transition phase are calibrated based on potential tetraquark candidates derived from phenomenology. Our investigation shows that the rotation and vibration transitional theory delivers a more accurate explanation for heavy tetraquarks compared to other methods utilizing the same formalism. To illustrate the concept, we compute the spectra of several tetraquarks, namely charm, bottom, bottom-charm and open charm and bottom systems, and contrast them with those of other particles.

## Key claim

The paper claims that a rotation-vibration transitional theory, based on an infinite-dimensional algebraic technique for the pairing Hamiltonian, provides a more accurate description of heavy tetraquark masses than other methods using the same formalism, but only when the control parameters (pairing strengths) are calibrated to specific values between the vibrational and rotational limits for each tetraquark system.

## Method

The authors extend an SU(1,1) algebraic technique, originally developed for bosonic pairing systems, to describe tetraquarks as diquark-antidiquark systems. They construct a Hamiltonian using Casimir operators of the dynamical symmetry group and solve it via a Bethe ansatz approach, where the control parameters \(c_{Q_i}\) and \(c_{\bar{Q}_i}\) are varied continuously between 0 (vibrational limit) and 1 (rotational limit) to model the quantum phase transition. The masses of fully-heavy tetraquarks \(T_{4c}\), \(T_{4b}\), and \(T_{2bc}\), as well as open-charm and open-bottom tetraquarks, are computed by calibrating these control parameters against meson-meson thresholds.

## Key numbers

- \(T_{4c}\) tetraquark: \(0^{++\prime}\) state mass = 5.978 GeV, \(1^{+-\prime}\) state mass = 6.155 GeV, \(2^{++}\) state mass = 6.263 GeV.
- \(T_{4b}\) tetraquark: \(0^{++\prime}\) state mass = 18.752 GeV, \(1^{+-\prime}\) state mass = 18.805 GeV, \(2^{++}\) state mass = 18.920 GeV.
- \(T_{2bc}\) tetraquark: \(0^{++}\) (scalar diquark) state mass = 12.359 GeV; \(0^{++\prime}\) (axial-vector diquark) state mass = 12.503 GeV; \(1^{+-}\) state mass = 12.016 GeV; \(1^{++}\) state mass = 12.155 GeV; \(2^{++}\) state mass = 12.897 GeV.
- Open-charm tetraquark \(cq\bar{q}\bar{q}\): \(0^{+}\) (\(A\bar{A}\)) state mass = 2.575 GeV; \(0^{+}\) (\(S\bar{S}\)) state mass = 2.320 GeV.
- Open-bottom tetraquark \(bq\bar{q}\bar{q}\): \(0^{+}\) (\(S\bar{S}\)) state mass = 5.047 GeV; \(1^{+}\) (\(S\bar{A}\)) state mass = 5.433 GeV.

## Context

This work extends the algebraic pairing model previously applied to nuclear physics (Arima and Iachello, 1975; Pan, 2002) to the study of exotic hadrons, specifically tetraquarks. It builds on the diquark-antidiquark model and the chromomagnetic interaction model (Karliner, 2016) by introducing a transitional Hamiltonian that interpolates between vibrational and rotational symmetries. The results enable comparison with experimental candidates such as the X(2900) observed by LHCb and provide predictions for future searches of fully-heavy tetraquarks, particularly the \(T_{4b}\) state around 18-19 GeV reported by CMS.
