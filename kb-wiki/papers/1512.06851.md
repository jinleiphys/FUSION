# Jet fragmentation functions in proton-proton collisions using soft-collinear effective theory

> The jet fragmentation function describes the longitudinal momentum distribution of hadrons inside a reconstructed jet. We study the jet fragmentation function in proton-proton collisions in the framework of soft-collinear effective theory (SCET). We find that, up to power corrections, the jet fragmentation function can be expressed as the ratio of the fragmenting jet function and the unmeasured jet function. Using renormalization group techniques, we are able to resum large logarithms of jet radii R in the perturbative expansion of the cross section. We use our theoretical formalism to describe the jet fragmentation functions for light hadron and heavy meson production measured at the Large Hadron Collider (LHC). Our calculations agree very well with the experimental data for the light hadron production. On the other hand, although our calculations for the heavy meson production inside jets are consistent with the PYTHIA simulation, they fail to describe the LHC data. We find that the jet fragmentation function for heavy meson production is very sensitive to the gluon-to-heavy-meson fragmentation function.

## Key claim

The jet fragmentation function can be expressed as the ratio of the fragmenting jet function and the unmeasured jet function up to power corrections, and large logarithms of jet radii R can be resummed using renormalization group techniques; however, the formalism fails to describe LHC data for heavy meson production inside jets, even though it agrees with PYTHIA simulations, and the heavy meson result is very sensitive to the gluon-to-heavy-meson fragmentation function.

## Method

The authors derived a factorized expression for the jet fragmentation function in proton-proton collisions using soft-collinear effective theory (SCET), expressing it as the ratio of the fragmenting jet function to the unmeasured jet function. They computed the matching coefficients for the fragmenting jet function at next-to-leading order for both cone and anti-kT jet algorithms, and applied renormalization group evolution to resum logarithms of the jet radius R. Numerical calculations were performed using CT14 NLO parton distribution functions and DSS07 NLO fragmentation functions for light hadrons, with comparisons to ATLAS and CMS data at center-of-mass energies of 2.76 TeV and 7 TeV.

## Key numbers

- For light hadrons at sqrt(s) = 7 TeV with anti-kT jets of R=0.6 and |y|<1.2, the theoretical calculations agree with ATLAS data across jet transverse momentum bins from 25 GeV to 500 GeV
- For light hadrons at sqrt(s) = 2.76 TeV, calculations agree with ATLAS data (R=0.4, |y|<1.6) and CMS data (R=0.3, 0.3<|y|<2)
- The characteristic scale for the unmeasured jet function is mu_J ~ p_TR = 2 p_T tan(R/2), which eliminates the logarithm L = ln(omega tan(R/2)/mu)
- For large z, choosing mu_G = p_TRZ = 2 p_T tan(R/2) (1-z) reduces theoretical uncertainty and gives an enhancement of about a factor of 2 for z greater than approximately 0.8

## Context

This work extends the fixed next-to-leading order calculation of Kaufmann (2015) by providing resummation of logarithms of the jet radius R that were not addressed previously. The formalism builds on the fragmenting jet function introduced by Procura (2009) and Jain (2011) and the unmeasured jet function from Ellis (2010). The results enable future studies of jet fragmentation in heavy ion collisions using SCET, with in-medium splitting functions already available from Ovanesyan (2011) and medium-modified fragmentation functions from Kang (2014) and Chien (2015).
