# Hadron Spectroscopy with Dynamical Chirally Improved Fermions

> We simulate two dynamical, mass degenerate light quarks on 16^3x32 lattices with a spatial extent of 2.4 fm using the Chirally Improved Dirac operator. The simulation method, the implementation of the action and signals of equilibration are discussed in detail. Based on the eigenvalues of the Dirac operator we discuss some qualitative features of our approach. Results for ground state masses of pseudoscalar and vector mesons as well as for the nucleon and delta baryons are presented.

## Key claim

The paper presents first results for hadron ground state masses from dynamical simulations using the Chirally Improved (CI) Dirac operator, which obeys the Ginsparg-Wilson relation only approximately; the results are obtained on a single lattice volume (2.4 fm) at three pion masses, without extrapolation to the physical pion mass or continuum limit.

## Method

The authors performed Hybrid Monte Carlo (HMC) simulations of two mass-degenerate dynamical light quarks on 16^3x32 lattices using the CI Dirac operator with stout smearing and the Luscher-Weisz gauge action. They employed Hasenbusch mass preconditioning, a mixed-precision conjugate gradient inverter, and chronological inversion by minimal residue extrapolation. Hadron masses were extracted using the variational method with Jacobi-smeared quark sources, solving a generalized eigenvalue problem from a matrix of cross-correlators built from multiple interpolators.

## Key numbers

- Lattice spacing: a = 0.1507(17) fm (Run A), 0.1500(12) fm (Run B), 0.1440(12) fm (Run C), set via the Sommer parameter r0 = 0.48 fm.
- AWI (PCAC) quark mass: m_AWI = 42.8(4) MeV (Run A), 34.1(2) MeV (Run B), 15.3(3) MeV (Run C).
- Pion mass range: approximately 530 MeV down to 320 MeV across the three runs.
- Integrated autocorrelation times for the plaquette: tau_int = 3.5 (Run A), 2.4 (Run B), 3.7 (Run C); for the number of CG iterations: tau_int = 4.2 (Run A), 2.7 (Run B), 3.6 (Run C).
- Acceptance rate: P_acc = 0.904 (Run A), 0.911 (Run B), 0.858 (Run C).
- Average change in Hamiltonian: <Delta H> = 0.038(11) (Run A), 2.01(1.95) (Run B), 0.089(59) (Run C); excluding a spike in Run B gives <Delta H> = 0.055(10).

## Context

This work extends earlier quenched studies with the CI Dirac operator by the BGR collaboration (Gattringer et al., 2003; Gattringer et al., 2005) to dynamical fermions, and follows smaller-scale dynamical simulations on smaller lattices (Lang et al., 2005). The approach contrasts with simulations using the exact Ginsparg-Wilson overlap operator, which is computationally more expensive and has been implemented by only a few groups (e.g., Karsch et al., 2006; DeGrand and Schaefer, 2005). The results enable future studies of excited hadron spectroscopy and chiral extrapolations with approximate Ginsparg-Wilson fermions at lighter quark masses.
