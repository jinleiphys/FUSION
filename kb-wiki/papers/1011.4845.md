In theoretical calculation, all of these observables can be classified into four Legendre classes and represented by associated Legendre polynomial function itself \cite{fasano92}. # Bayesian Analysis of $C_{x'}$ and $C_{z'}$ Double Polarizations in Kaon Photoproduction

## Key claim
The $C_{x'}$ and $C_{z'}$ double polarization observables in the $\gamma + p \to K^{+} + \Lambda$ reaction require two and three orders of associated Legendre polynomials respectively to describe the experimental data well, based on Bayesian model comparison using Nested sampling integration. The extracted coefficients from the best-fit models may reveal the structure of baryon resonances, but only qualitatively.

## Method
The experimental data for $C_{x}$ and $C_{z}$ double polarizations from the Bradford (2007) dataset were transformed to the prime coordinate system (outgoing kaon or z'-axis) using a standard rotation matrix involving the kaon scattering angle $\theta$. For each photon energy, the Bayesian evidence was computed via Nested sampling integration over the parameters of associated Legendre polynomial expansions, using uniform prior distributions. The best data model for each observable was selected by comparing the posterior probabilities among models with different polynomial orders.

## Key numbers
- $C_{x'}$ double polarization requires **two** orders of associated Legendre polynomials to describe the data well.
- $C_{z'}$ double polarization requires **three** orders of associated Legendre polynomials to describe the data well.
- The analysis used experimental data from Bradford (2007) for the $\gamma + p \to K^{+} + \Lambda$ reaction.
- The transformation from $C_{x}, C_{z}$ to $C_{x'}, C_{z'}$ uses the kaon scattering angle $\theta$ via: $C_{x'} = C_{x} \cos{\theta} - C_{z}\sin{\theta}$ and $C_{z'} = C_{x} \sin{\theta} + C_{z}\cos{\theta}$.

## Context
This work extends the Legendre class classification of Fasano (1992), which showed that 16 observables in $K^{+}\Lambda$ photoproduction can be represented by associated Legendre polynomials. The analysis uses the Nested sampling integration technique developed by Skilling (2006), previously applied in astrophysics and cosmology. The results enable future extraction of baryon resonance information from the $K\Lambda$ channel, a longstanding problem in the field, and complement upcoming experimental data for other observables such as G asymmetry from Jefferson Lab.
