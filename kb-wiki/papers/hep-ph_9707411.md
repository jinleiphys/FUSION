# Magnetic Moments of $Δ^{++}$ and $Ω^-$ from QCD Sum Rules

> QCD sum rules for the magnetic moments of $Δ^{++}$ and $Ω^-$ are derived using the external field method. They are analyzed by a Monte-Carlo based procedure, using realistic estimates of the QCD input parameters. The results are consistent with the measured values, despite relatively large errors that can be attributed mostly to the poorly-known vacuum susceptibility $χ$. It is shown that a 30% level accuracy can be achieved in the derived sum rules, provided the QCD input parameters are improved to the 10% level.

## Key claim

The QCD sum rule approach can reproduce the measured magnetic moments of the Δ⁺⁺ and Ω⁻ baryons, but only with approximately 100% uncertainties when using realistic error estimates for the QCD input parameters; a 30% level accuracy is achievable only if the QCD input parameters are improved to the 10% level.

## Method

The author derived QCD sum rules for the magnetic moments of the Δ⁺⁺ and Ω⁻ baryons using the external field method, where a constant background electromagnetic field is introduced and the linear response of the two-point correlation function is computed via operator product expansion (OPE). The sum rules were analyzed using a Monte-Carlo based procedure that generates Gaussianly-distributed sets of QCD input parameters, applies χ² minimization to extract phenomenological fit parameters, and constructs uncertainty distributions from 1000 parameter sets. A two-stage fit was performed: first the mass sum rule was fitted to obtain the mass, coupling, and continuum threshold, then the magnetic moment sum rule was fitted with the continuum threshold fixed to the mass sum rule value.

## Key numbers

- Δ⁺⁺ magnetic moment from QCD sum rules: μ = 4.13⁺¹·⁴⁰₋₁·₁₈ μ_N (with 10% input errors), compared to experimental value 4.5 ± 1.0 μ_N
- Ω⁻ magnetic moment from QCD sum rules: μ = -1.49⁺⁰·⁴⁰₋₀·₄₉ μ_N (with 10% input errors), compared to experimental value -2.024 ± 0.056 μ_N
- With realistic (50%) uncertainties on the vacuum susceptibility χ = -6.0 ± 3.0 GeV⁻², the extracted magnetic moments have approximately 100% errors: Δ⁺⁺ μ = 3.60⁺³·⁶⁸₋₃·₅₅ μ_N, Ω⁻ μ = -1.25⁺¹·¹²₋₁·¹⁷ μ_N
- The Borel window for Δ⁺⁺ spans 0.765 to 1.47 GeV; for Ω⁻ it spans 0.747 to 1.66 GeV

## Context

This work extends the QCD sum rule method, previously applied successfully to nucleon and octet baryon magnetic moments by Ioffe and Smilga (1983, 1984) and Chiu, Pasupathy, Wilson (1986), to the decuplet baryons Δ⁺⁺ and Ω⁻. The results are compared with lattice QCD calculations by Leinweber, Draper, and Woloshyn (1992), chiral perturbation theory by Butler, Savage, and Springer (1994), and constituent quark models. The large uncertainties are attributed primarily to the poorly-known vacuum susceptibility χ, and the author identifies improved determination of this parameter as the key step toward more precise predictions.
