# SHARE: Statistical Hadronization with Resonances

> SHARE is a collection of programs designed for the statistical analysis of particle production in relativistic heavy-ion collisions. With the physical input of intensive statistical parameters, it generates the ratios of particle abundances. The program includes cascade decays of all confirmed resonances from the Particle Data Tables. The complete treatment of these resonances has been known to be a crucial factor behind the success of the statistical approach. An optional feature implemented is a Breit--Wigner type distribution for strong resonances. An interface for fitting the parameters of the model to the experimental data is provided.

## Key claim

SHARE provides a standardized, modular software framework for statistical hadronization analysis of particle production in relativistic heavy-ion collisions, incorporating the complete treatment of confirmed resonances from the Particle Data Tables including cascade decays, with an optional Breit-Wigner treatment for finite resonance widths; the paper presents the program structure and physics basis but does not itself report new experimental results or model fits.

## Method

The program computes particle abundances at chemical freeze-out using Fermi-Dirac or Bose-Einstein distributions expressed as series of modified Bessel functions, with fugacities parameterized by quark chemical potentials and phase-space occupancy factors. All confirmed hadronic resonances are included, and their cascade decays are implemented sequentially from heaviest to lightest particles, with branching ratios taken from the Particle Data Tables and Clebsch-Gordan coefficients applied for isospin symmetry in strong decays. An optional feature integrates over a Breit-Wigner mass distribution with energy-dependent width for resonances, and a fitting interface using MINUIT allows optimization of thermal parameters to experimental particle ratios via a chi-squared function.

## Key numbers

* Computation time for evaluation of particle yields: up to seconds (Fortran version); rises by up to a factor of 300 during minimization, and a further factor of a few when chi-squared profiles and contours with chemical non-equilibrium are requested.
* Package size: 645 KB directory including example programs (82 KB compressed distribution archive).
* Number of lines in distributed program, including test data: 14893.
* Hagedorn temperature cited as context: T_H approximately 160 MeV.
* Example thermal input parameters shown: temperature 0.165 GeV, light quark chemical potential 0.028 GeV, strange quark chemical potential 0.006 GeV, light quark phase space occupancy 1.0, strange quark phase space occupancy 1.0, I3 fugacity -0.001 GeV, absolute normalization 1.0, charm quark fugacity 1.0, charm quark phase space occupancy 0.001, calculations accuracy 0.001.

## Context

This work extends the statistical hadronization approach originating from Fermi (1950), Pomeranchuk (1951), and Hagedorn (1965), and builds upon prior implementations by Koch et al., Broniowski and Florkowski, and others. SHARE provides a standardized, publicly available code that enables systematic comparison of different statistical models (including chemical equilibrium and non-equilibrium variants) and facilitates analysis of data from RHIC and other heavy-ion facilities; it is designed to allow the community to create an analysis standard and to compare various similar models and programs currently in use.
