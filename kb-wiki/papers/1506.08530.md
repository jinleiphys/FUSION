# $0νββ$ and $2νββ$ nuclear matrix elements in the interacting boson model with isospin restoration

> We introduce a method for isospin restoration in the calculation of nuclear matrix elements (NME) for $0νββ$ and $2νββ$ decay within the framework of interacting boson model (IBM-2). With this method, we calculate NME for all processes of interest in $0νβ^-β^-$, $2νβ^-β^-$, and in $0νβ^+β^+$, $0νβ^+ EC^+$, $R0νECEC$, $2νβ^+β^+$, $2νβ^+EC$, and $2νECEC$. With this method, the Fermi (F) matrix elements for $2νββ$ vanish, and those for $0νββ$ are considerably reduced.

## Key claim

The authors introduce a method for isospin restoration in the interacting boson model (IBM-2) that modifies the mapped transition operator rather than the wave functions, ensuring that Fermi matrix elements for two-neutrino double-beta decay vanish and reducing those for neutrinoless double-beta decay by approximately 15%, bringing them into better agreement with shell-model results.

## Method

The authors modify the radial integrals in the IBM-2 mapping of the double-beta decay transition operator by subtracting the monopole term (delta functions in all quantum numbers) from the original integrals, which enforces the condition that Fermi matrix elements for 2νββ decay vanish and reduces the Fermi contribution for 0νββ decay. They then calculate nuclear matrix elements for all double-beta decay modes (β⁻β⁻, β⁺β⁺, β⁺EC, and ECEC) for both light and heavy neutrino exchange mechanisms using the Argonne parametrization of short-range correlations, with the ratio g_V/g_A = 1/1.269.

## Key numbers

- For 0νβ⁻β⁻ decay to ground state with light neutrino exchange, the total matrix element M^(0ν) ranges from 1.75 (⁴⁸Ca) to 4.81 (²³⁸U), with an estimated error of 16% for all nuclei
- The Fermi matrix element ratio χ_F = (g_V/g_A)² M_F^(0ν)/M_GT^(0ν) is reduced from values as large as -0.40 (⁸²Se, old calculation) to -0.10 (⁸²Se, new calculation) after isospin restoration
- For 0νβ⁻β⁻ decay with heavy neutrino exchange, M_h^(0ν) ranges from 46.6 (⁴⁸Ca) to 189 (²³⁸U), with an estimated error of 28%
- For 2νββ decay, the Fermi matrix elements M_F^(2ν) are set to zero by construction after isospin restoration
- The overall reduction in M^(0ν) for light neutrino exchange is approximately 15% compared to calculations without isospin restoration

## Context

This work extends the authors' previous IBM-2 calculations of nuclear matrix elements for double-beta decay (Barea et al., 2013) by addressing the isospin problem that caused non-vanishing Fermi matrix elements for 2νββ decay and overly large Fermi contributions for 0νββ decay in nuclei where protons and neutrons occupy the same major shell. The method is similar in spirit to the isospin restoration approach introduced by Simkovic et al. (2013) for the quasiparticle random phase approximation (QRPA), but differs in practical implementation. The results enable more reliable extraction of neutrino masses from future 0νββ decay experiments and provide consistent matrix elements for comparison with the interacting shell model (ISM) and other theoretical approaches.
