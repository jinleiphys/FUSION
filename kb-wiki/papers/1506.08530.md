# $0ﾎｽﾎｲﾎｲ$ and $2ﾎｽﾎｲﾎｲ$ nuclear matrix elements in the interacting boson model with isospin restoration

> We introduce a method for isospin restoration in the calculation of nuclear matrix elements (NME) for $0ﾎｽﾎｲﾎｲ$ and $2ﾎｽﾎｲﾎｲ$ decay within the framework of interacting boson model (IBM-2). With this method, we calculate NME for all processes of interest in $0ﾎｽﾎｲ^-ﾎｲ^-$, $2ﾎｽﾎｲ^-ﾎｲ^-$, and in $0ﾎｽﾎｲ^+ﾎｲ^+$, $0ﾎｽﾎｲ^+ EC^+$, $R0ﾎｽECEC$, $2ﾎｽﾎｲ^+ﾎｲ^+$, $2ﾎｽﾎｲ^+EC$, and $2ﾎｽECEC$. With this method, the Fermi (F) matrix elements for $2ﾎｽﾎｲﾎｲ$ vanish, and those for $0ﾎｽﾎｲﾎｲ$ are considerably reduced.

## Key claim

The authors introduce a method for isospin restoration in the interacting boson model (IBM-2) that modifies the mapped transition operator rather than the wave functions, ensuring that Fermi matrix elements for two-neutrino double-beta decay vanish and reducing those for neutrinoless double-beta decay by approximately 15%, bringing them into better agreement with shell-model results.

## Method

The authors modify the radial integrals in the IBM-2 mapping of the double-beta decay transition operator by subtracting the monopole term (delta functions in all quantum numbers) from the original integrals, which enforces the condition that Fermi matrix elements for 2ﾎｽﾎｲﾎｲ decay vanish and reduces the Fermi contribution for 0ﾎｽﾎｲﾎｲ decay. They then calculate nuclear matrix elements for all double-beta decay modes (ﾎｲ竅ｻﾎｲ竅ｻ, ﾎｲ竅ｺﾎｲ竅ｺ, ﾎｲ竅ｺEC, and ECEC) for both light and heavy neutrino exchange mechanisms using the Argonne parametrization of short-range correlations, with the ratio g_V/g_A = 1/1.269.

## Key numbers

- For 0ﾎｽﾎｲ竅ｻﾎｲ竅ｻ decay to ground state with light neutrino exchange, the total matrix element M^(0ﾎｽ) ranges from 1.75 (竅ｴ竅ｸCa) to 4.81 (ﾂｲﾂｳ竅ｸU), with an estimated error of 16% for all nuclei
- The Fermi matrix element ratio ﾏ㍉F = (g_V/g_A)ﾂｲ M_F^(0ﾎｽ)/M_GT^(0ﾎｽ) is reduced from values as large as -0.40 (竅ｸﾂｲSe, old calculation) to -0.10 (竅ｸﾂｲSe, new calculation) after isospin restoration
- For 0ﾎｽﾎｲ竅ｻﾎｲ竅ｻ decay with heavy neutrino exchange, M_h^(0ﾎｽ) ranges from 46.6 (竅ｴ竅ｸCa) to 189 (ﾂｲﾂｳ竅ｸU), with an estimated error of 28%
- For 2ﾎｽﾎｲﾎｲ decay, the Fermi matrix elements M_F^(2ﾎｽ) are set to zero by construction after isospin restoration
- The overall reduction in M^(0ﾎｽ) for light neutrino exchange is approximately 15% compared to calculations without isospin restoration

## Context

This work extends the authors' previous IBM-2 calculations of nuclear matrix elements for double-beta decay (Barea et al., 2013) by addressing the isospin problem that caused non-vanishing Fermi matrix elements for 2ﾎｽﾎｲﾎｲ decay and overly large Fermi contributions for 0ﾎｽﾎｲﾎｲ decay in nuclei where protons and neutrons occupy the same major shell. The method is similar in spirit to the isospin restoration approach introduced by Simkovic et al. (2013) for the quasiparticle random phase approximation (QRPA), but differs in practical implementation. The results enable more reliable extraction of neutrino masses from future 0ﾎｽﾎｲﾎｲ decay experiments and provide consistent matrix elements for comparison with the interacting shell model (ISM) and other theoretical approaches.
