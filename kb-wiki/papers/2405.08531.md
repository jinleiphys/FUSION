# The TDHF code Sky3D version 1.2

> The Sky3D code has been widely used to describe nuclear ground states, collective vibrational excitations, and heavy-ion collisions. The approach is based on Skyrme forces or related energy density functionals. The static and dynamic equations are solved on a three-dimensional grid, and pairing is been implemented in the BCS approximation. This updated version of the code aims to facilitate the calculation of nuclear strength functions in the regime of linear response theory, while retaining all existing functionality and use cases. The strength functions are benchmarked against available RPA codes, and the user has the freedom of choice when selecting the nature of external excitation (from monopole to hexadecapole and more). Some utility programs are also provided that calculate the strength function from the time-dependent output of the dynamic calculations of the Sky3D code.

## Key claim
This updated version of the Sky3D code adds the capability to calculate nuclear multipole strength functions via time-dependent Hartree-Fock (TDHF) in the linear response regime, benchmarked against RPA codes; the abstract does not specify that the benchmark is performed for the nucleus 208Pb using the SLy5 force and that the TDHF results satisfy energy-weighted sum rules within 100±3%.

## Method
The code solves static and dynamic Skyrme-Hartree-Fock equations on a three-dimensional Cartesian grid with BCS pairing. For strength function calculations, the system is boosted with a user-defined multipole external field (monopole through hexadecapole and beyond, isoscalar or isovector), and the time-dependent expectation value of the multipole operator is Fourier-transformed to obtain the strength function. A new Python utility (Fourier.py) performs the Fourier transform with optional Cos or Exponential filtering of the time signal.

## Key numbers
- Energy-weighted sum rules (EWSR) are satisfied with a percentage score of 100±3% for all multipole calculations in 208Pb.
- The Exponential filter default parameter Γ₀ is 1 MeV.
- The Cos filter default exponent n is 6.
- The grid benchmark uses nx=24, ny=24, nz=24 with dx=dy=dz=1 fm.
- The dynamic run uses nt=50000 time steps with dt=0.2 fm/c.
- The external boost amplitude used in benchmarks is ampl_ext=5×10⁻⁵.
- The mixed pairing parameter x defaults to 1 (surface pairing) and a value of 0.5 defines a mix of volume and surface pairing, with ρ₀ fixed to 0.16 fm⁻³.

## Context
This work extends the Sky3D code series (Schuetrumpf et al., 2018) by adding strength function capabilities that were previously limited to symmetry-constrained approaches like QRPA (Colò, 2021). The code enables calculation of nuclear multipole response without assuming any symmetry, and the benchmark against the hfbcs-qrpa (v1) code for 208Pb demonstrates agreement while revealing differences due to boundary conditions (finite box in TDHF versus truncated oscillator basis in RPA). This enables future studies of mode coupling in nuclear vibrations and electromagnetic response (B(Eλ) values) via the new only_P option.
