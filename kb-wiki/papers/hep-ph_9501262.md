# Solving the Bethe-Salpeter Equation for Scalar Theories in Minkowski Space

> The Bethe-Salpeter (BS) equation for scalar-scalar bound states in scalar theories without derivative coupling is formulated and solved in Minkowski space. This is achieved using the perturbation theory integral representation (PTIR), which allows these amplitudes to be expressed as integrals over weight functions and known singularity structures and hence allows us to convert the BS equation into an integral equation involving weight functions. We obtain numerical solutions using this formalism for a number of scattering kernels to illustrate the generality of the approach. It applies even when the na\"ıve Wick rotation is invalid. As a check we verify, for example, that this method applied to the special case of the massive ladder exchange kernel reproduces the same results as are obtained by Wick rotation.

## Key claim

The Bethe-Salpeter equation for scalar-scalar bound states in scalar theories without derivative coupling can be solved directly in Minkowski space using the perturbation theory integral representation (PTIR), even when the naive Wick rotation is invalid; the method is demonstrated for several scattering kernels, and for the special case of massive ladder exchange it reproduces the same results as Wick rotation.

## Method

The authors formulate the Bethe-Salpeter equation for scalar-scalar bound states using the perturbation theory integral representation (PTIR), which expresses the BS amplitude as an integral over weight functions and known singularity structures. They convert the BS equation into a real integral equation for these weight functions in two variables (α and z), which is frame-independent and avoids the singular distributions of the original Minkowski-space equation. Numerical solutions are obtained for several scattering kernels, including the massive ladder exchange kernel and a generalized ladder kernel that becomes complex under Wick rotation.

## Key numbers

- Bound state mass M satisfies 0 < M² ≤ (2m)², where m is the renormalized constituent scalar mass.
- The threshold parameter μ is defined such that m+μ is the threshold for particle production in the single φ channel; for a φ²σ model, μ = m_σ if m_σ < 2m, or μ = 2m if m_σ > 2m.
- The support property for the vertex weight function requires β ≥ (m+μ − ½(1−|ζ|)√P²)².
- The BS amplitude weight function φ_n(α,z) = 0 when α < min[0, (m+μ−√P²/2)² − m² + P²/4].

## Context

This work extends the PTIR approach first used by Wick and Cutkosky for the massless scalar exchange ladder model (the Wick-Cutkosky model) and by Wanders for massive scalar exchange in ladder approximation. It builds on Nakanishi's systematic development of PTIR for n-point functions and provides a method to solve the BS equation without Wick rotation, which is essential when dressed propagators or complicated kernels make analytic continuation invalid. The approach enables future calculations of bound state properties in Minkowski space for applications such as high-energy scattering processes.
