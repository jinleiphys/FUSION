# A practical solution to the sign problem at finite theta-vacuum angle

> We propose a practical way of circumventing the sign problem in lattice QCD simulations with a theta-vacuum term. This method is the reweighting method for the QCD Lagrangian after the U_A(1) transformation. In the Lagrangian, the P-odd mass term as a cause of the sign problem is minimized. In order to find out a good reference system in the reweighting method, we estimate the average reweighting factor by using the two-flavor NJL model and eventually find a good reference system.

## Key claim

The authors propose a reweighting method for lattice QCD simulations with a theta-vacuum term, where the sign problem is minimized by applying a U_A(1) transformation to the Lagrangian and then using a reference system (reference C) that keeps the average reweighting factor close to 1 for all theta values, not just small ones.

## Method

The QCD Lagrangian is transformed via a U_A(1) rotation to make the P-odd mass term proportional to the light quark mass m_0, which is much smaller than the dynamical quark mass. The average reweighting factor ⟨R(θ)⟩ is then estimated using the two-flavor Nambu-Jona-Lasinio (NJL) model in the mean-field approximation, with the thermodynamic potential and Hessian matrix computed for both the original and reference theories. Three candidate reference systems (A, B, C) are tested, where the P-odd mass term is either neglected entirely (A) or replaced by a theta-even correction term of order m_0^2 (B and C), with reference C motivated by the chiral Lagrangian and the 1/N_c expansion.

## Key numbers

- Reference A is reliable for θ ≲ 1.2 at T = 100 MeV, where ⟨R(θ)⟩ exceeds 2.
- The chiral transition temperature in the original theory is 206 MeV at θ = 0 and 194 MeV at θ = π.
- Reference C satisfies the condition 0.5 ≲ ⟨R(θ)⟩ ≲ 2 for all θ at T = 100 MeV and for any temperature larger than 100 MeV.
- The correction term in reference B has coefficient m_0^2 / M_π = 0.129 MeV; in reference C the coefficient is m_0 M_π^2 / M_η'^2 = 0.114 MeV.
- The pion mass M_π = 138 MeV and pion decay constant f_π = 93 MeV are used as input at vacuum.

## Context

This work extends the authors' earlier proposal (Sasaki, 2012) for minimizing the sign problem in lattice QCD with a theta term, which previously found reference A adequate only for small and intermediate theta. The new reference C is constructed using the chiral Lagrangian result for the theta-dependent pion mass from Creutz (2004) and the 1/N_c expansion for the coefficient l_7, enabling reliable lattice simulations for all theta values including near π, where the Dashen mechanism of spontaneous P-symmetry breaking occurs. This enables first-principle studies of CP-violating phenomena such as the chiral magnetic effect in heavy-ion collisions and baryogenesis in the early universe.
