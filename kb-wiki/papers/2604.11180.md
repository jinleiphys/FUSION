# Extended Variable Phase Method for Spin-1/2 Correlation Functions

> We have developed a systematic approach to calculate the correlation function for spin-1/2 particles, incorporating both central and noncentral components of the interparticle interaction. This is achieved by extending the variable phase method to accommodate noncentral potentials and numerically solving the Schrödinger equation. Within this framework, the partial-wave contributions to the nucleon-nucleon correlation functions adopting the Reid soft-core potential are evaluated. The resulting correlation functions are then compared for Gaussian sources of different sizes.

# Extended Variable Phase Method for Spin-1/2 Correlation Functions

## Key claim

The authors extend the variable phase method to handle noncentral (tensor) potentials for spin-1/2 particles, enabling calculation of nucleon-nucleon correlation functions that include contributions from higher partial waves and coupled channels such as the ³S₁–³D₁ system. The method is demonstrated using the Reid soft-core potential, but the formalism is general and applicable to any spin-1/2 pair interaction with central and tensor components.

## Method

The authors generalize the variable phase method by reformulating the coupled radial Schrödinger equations for noncentral potentials into first-order differential equations for the bar phase shifts (ηα, ηβ) and the bar admixture parameter (ε) as functions of radial distance. These equations are derived from the integral form of the coupled Schrödinger equations using auxiliary functions analogous to the central-potential case. The exact wave function is then constructed from these distance-dependent phase parameters and the scattering matrix, with the correlation function obtained by integrating the squared wave function over the source emission function via Eq. (1). The Reid soft-core potential is used for numerical evaluation, with partial-wave contributions summed up to a cutoff angular momentum and the remaining terms treated analytically as plane waves.

## Key numbers

- Relative momentum bin width: 5 MeV/c
- Gaussian source sizes investigated: 1 fm, 3 fm, 4 fm, and 5 fm
- For a 3 fm Gaussian source, higher partial wave contributions to correlation functions are around or below 0.01 in magnitude
- Reducing source size from 3 fm to 1 fm amplifies higher angular momentum channel contributions by almost an order of magnitude
- Scattering length for the ³S₁ channel (deuteron bound state): 5.39 fm
- Scattering length for the ¹S₀ channel: -7.78 fm
- Correlation function for the T=0, S=1 channel at q=1 MeV shows a minimum as a function of Gaussian source size σ, with rebound at larger sizes

## Context

This work extends the variable phase method originally developed by Morse and Allis (1930s) for central potentials to include noncentral (tensor) forces for spin-1/2 particles. It addresses a gap in previous correlation function studies, where the tensor force has rarely been included despite its importance in nuclear physics, as demonstrated by the deuteron electric quadrupole moment arising from ³S₁–³D₁ mixing. The method enables systematic investigation of higher partial wave contributions to correlation functions, which have become experimentally accessible with advances in detector technology, and provides a computationally efficient alternative to shooting algorithms used in prior work. The formalism can be applied to future studies of resonance states, source imaging, and neutron skins, and can incorporate Coulomb effects by replacing Riccati-Bessel functions with Coulomb wave functions.
