# Mesonic Width Effects on the Momentum Dependence of the Rho-Omega Mixing Matrix Element

> It is shown, in a model independent way, that the large difference in rho and omega widths gives rise to a new source of momentum dependence for the rho-omega mixing matrix element. The q^2 dependence arising due to the meson widths leads to a significant alteration of the result obtained in the zero-width approximation usually discussed in the literature. The origin of this strong momentum dependence lies in the difference between the rho and omega meson widths.

# Mesonic Width Effects on the Momentum Dependence of the Rho-Omega Mixing Matrix Element

## Key claim

The large difference between the rho meson width (151.5 MeV) and the omega meson width (8.4 MeV) introduces a new, model-independent source of momentum dependence for the rho-omega mixing matrix element, significantly altering the result obtained in the zero-width approximation used in prior literature. Any discussion of the q^2 dependence of the mixing matrix element that neglects finite width effects is incomplete.

## Method

The authors start from a dispersion relation for the mixed rho-omega propagator, valid to leading order in quark masses. They saturate the imaginary part of the mixed propagator with rho and omega meson contributions, first using zero-width delta functions and then replacing them with normalized Breit-Wigner forms to incorporate finite widths. The mixing matrix element is extracted from the real part of the mixed propagator using three different definitions: the traditional zero-width definition, an intermediate definition using zero-width poles with width-modified propagators, and a fully consistent definition incorporating complex masses for the mesons.

## Key numbers

- The generally accepted on-shell value of the rho-omega mixing matrix element is -0.00452 ± 0.00006 GeV^2, extracted at q^2 approximately equal to m_omega^2
- The rho meson width is 151.5 MeV; the omega meson width is 8.4 MeV
- The parameter lambda, extracted from a QCD sum rule analysis by Hatsuda et al. (1994), lies in the range 1.43 ≤ lambda ≤ 1.85
- The integrals I_rho and I_omega are cut off at s = 3 GeV^2, where the integral values are stable

## Context

This work extends prior model calculations of the off-shell behavior of rho-omega mixing, which treated the rho and omega as stable particles with zero widths. Previous approaches included quark-antiquark loop calculations (Goldman et al., 1992; Krein et al., 1993), nucleon-antinucleon loop calculations (Piekarewicz and Williams, 1993), a QCD sum rule calculation (Hatsuda et al., 1994; Maltman, 1995), and a hybrid quark-meson coupling model (Mitchell et al., 1994; Mitra and Yang, 1994). The authors argue that all these calculations are incomplete because they neglect the finite widths of the rho and omega mesons. The results enable a more accurate determination of the contribution of rho-omega mixing to the charge-symmetry-breaking nucleon-nucleon potential and its effects in neutron-proton scattering.
