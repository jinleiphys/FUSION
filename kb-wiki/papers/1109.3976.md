# Theoretical description of three- and four-nucleon scattering states using bound-state-like wave functions

> Bound-state-like wave functions are used to determine the scattering matrix corresponding to low energy $N-d$ and $p-^3$He collisions. To this end, the coupled channel form of the integral relations derived from the Kohn variational principle is used. The construction of degenerate bound-state-like wave functions belonging to the continuum spectrum of the Hamiltonian is discussed. Examples are shown using realistic nucleon-nucleon forces.

# Theoretical description of three- and four-nucleon scattering states using bound-state-like wave functions

## Key claim

Bound-state-like wave functions obtained from diagonalization of the Hamiltonian can be used to compute the scattering matrix for low-energy N-d and p-³He collisions, provided the control parameter β is tuned so that sequential eigenvalues match the desired center-of-mass energy and the wave functions are good solutions of (H-E)Ψ=0 in the internal region.

## Method

The authors construct A=3 and A=4 bound-state-like wave functions using the hyperspherical harmonic (HH) expansion with Laguerre polynomials for the hyperradial functions, solving the generalized eigenvalue problem for the Hamiltonian. They use the nonlinear parameter β as a control parameter to select sequential eigenvectors whose eigenvalues match a chosen scattering energy; these square-integrable wave functions are then inserted into two integral relations derived from the Kohn variational principle (KVP) to obtain the K-matrix as K = A⁻¹B, where the matrix elements A_ij = ⟨Ψⁱ_n|H-E_n|Ω¹_j⟩ and B_ij = -⟨Ψⁱ_n|H-E_n|Ω⁰_j⟩ are computed from the bound-state-like functions and the asymptotic regular (Ω⁰) and irregular (Ω¹) channel functions.

## Key numbers

- For the A=3 Jπ=1/2⁺ state using the AV14 potential, the PHH basis dimension is D ≈ 2200, with N_c=18 channels, M_α=8 for the first 8 channels, M_α=6 for the next six, M_α=4 for the last ones, and N_L≈20 Laguerre polynomials; an accuracy of 1 keV is reached for the ground state.
- For the A=4 Jπ=0⁺, T=T_z=1 state using the N3LO-Idaho potential, the Hamiltonian matrix dimension is D ≈ 84000, obtained with K_max=44 (≈3500 HH states) and N_L=24; the ³He threshold is at -7.128 MeV.
- Phase-shifts and mixing parameters for n-d scattering (Jπ=1/2⁺, 1/2⁻, 3/2⁺) are computed at E_lab = 1, 2, and 3 MeV using the AV14 potential, with the regularization parameter γ varied from 0.25 fm⁻¹ to 1.25 fm⁻¹.
- For p-³He scattering (Jπ=0⁺), phase-shifts are computed at three lab energies (3.13, 4.05, and 5.54 MeV) where experimental data exist.

## Context

This work extends the integral-relations approach derived from the KVP (Kievsky et al., 2010) to treat coupled-channel scattering in A=3 and A=4 systems using realistic nucleon-nucleon potentials, building on prior applications with semirealistic interactions (Kievsky et al., 2010). It provides a method to compute scattering observables using bound-state techniques, which is relevant because accurate bound-state methods exist for systems beyond A=4 (e.g., Green's Function Monte Carlo and No Core Shell Model) while scattering calculations have been limited to A≤4. The results are validated against established benchmarks for n-d scattering (Benchmark 1) and p-d scattering (Kievsky et al., 2001), and against a recent p-³He benchmark (Benchmark 4). This approach enables future extension of bound-state methods to treat scattering states in systems with A>4.
