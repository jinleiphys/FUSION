# SEARCHING FOR COLD DARK MATTER

> The differential cross-section for the elastic scattering of the lightest supersymmetric particle (LSP) with nuclear targets is calculated in the context of currently fashionable supersymmetric theories (SUSY). An effective four fermion interaction is constructed by considering i) $Z^0$ exchange ii)s-quark exchange and iii) Higgs exchange. It is expressed in terms of the form factors $f^0_V,f^0_A, f^0_S$ (isoscalar) and $f^1_V,f^1_A$ and $f^1_S$ (isovector) which contain all the information of the underlining theory. Numerical values were obtained using representative input parameters in the constrained parameter space of SUSY phenomenology. Both the coherent and for odd-A nuclei the incoherent (spin) nuclear matrix elements were evaluated for nuclei of experimental interest. The spin matrix elements tend to dominate for odd nuclei but the coherent matrix elements become more important in all other cases. For the coherent part the Higgs contribution competes with the Z- and s-quark contributions. Cross-sections as high as $10^{-37} cm^2$ have been obtained.

## Key claim

The cross-section for elastic scattering of the lightest supersymmetric particle (LSP) with nuclear targets can reach values as high as \(10^{-37} \text{ cm}^2\), but this result is contingent on specific representative input parameters within the constrained parameter space of supersymmetric phenomenology, and the event rate for detection is expected to be small (approximately 2.5 events per year for a 1 kg lead target).

## Method

The differential cross-section for elastic LSP-nucleus scattering was calculated by constructing an effective four-fermion interaction at the quark level, considering three exchange mechanisms: Z\(^0\) exchange, s-quark exchange, and Higgs exchange. This interaction was expressed in terms of isoscalar and isovector form factors (\(f_V^0, f_A^0, f_S^0\) and \(f_V^1, f_A^1, f_S^1\)) which encode the underlying supersymmetric theory. Both coherent nuclear matrix elements (for all nuclei) and incoherent spin-dependent matrix elements (for odd-A nuclei) were evaluated for nuclei of experimental interest, using representative input parameters from the constrained SUSY parameter space.

## Key numbers

- Cross-section for \(^{207}\text{Pb}\) target: \(8.1 \times 10^{-38} \text{ cm}^2\) (Solution 1), \(1.9 \times 10^{-37} \text{ cm}^2\) (Solution 2), \(2.6 \times 10^{-38} \text{ cm}^2\) (Solution 3).
- Event rate for \(^{207}\text{Pb}\) target with LSP mass \(m_1 = 27 \text{ GeV}\) and cross-section \(1.9 \times 10^{-37} \text{ cm}^2\): \(6.6 \times 10^{-3}\) events/day, approximately 2.5 events/year for a 1 kg detector.
- Normalization cross-section \(\sigma_0 = 0.77 \times 10^{-38} \text{ cm}^2\).
- LSP mass range considered: 10–100 GeV, with kinetic energy 10–100 keV.
- Cosmological density parameters used: \(\Omega_{CDM} = 0.6\), \(\Omega_{HDM} = 0.3\), \(\Omega_B = 0.1\).

## Context

This work extends prior calculations of LSP-nucleus scattering cross-sections (references [11-16] in the paper) by incorporating recent developments in supersymmetric theories that have yielded a substantially constrained parameter space, and by including the Higgs exchange contribution which was found to be important for coherent scattering. The paper provides nuclear matrix element estimates for specific odd-A nuclei (\(^3\text{He}\), \(^{19}\text{F}\), \(^{23}\text{Na}\), and \(^{207}\text{Pb}\)) using nuclear structure techniques, contrasting with attempts to extract spin matrix elements from EMC data (reference [34]). The results enable experimentalists to estimate event rates for direct dark matter detection experiments using targets of about 1 kg, and suggest that detection will be extremely difficult with event rates of only a few per year.
