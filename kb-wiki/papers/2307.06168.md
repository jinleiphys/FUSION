# A comparative study of different approaches for heavy quark energy loss, based on the latest experimental data

> This paper presents a comparative analysis of three distinct methods used to calculate the collisional energy loss of heavy quarks in Quark-Gluon Plasma. The study focuses on the calculation of the nuclear suppression factor of charm quarks in Pb-Pb collisions at $\sqrt{S_{NN}} = 5.02$ TeV. All three models are examined using the same numerical evolution based on the well-known Fokker-Planck equation by considering critical phenomena like a non-equilibrium state at the onset of heavy ion collision. The outcomes of each approach are compared with the latest data from ALICE and ATLAS experiments spanning from 2018 to 2022. This study aims to compare the degree of agreement between each approach and recently obtained experimental data, in the intermediate and high $P_T$ regions.

## Key claim

The paper compares three methods for calculating collisional energy loss of charm quarks in Quark-Gluon Plasma (Bjorken, Thoma-Gyulassy, and Braaten-Thoma) using the Fokker-Planck equation, finding that while all three agree reasonably with experimental data from ALICE and ATLAS, the Braaten-Thoma model (using hard-thermal loop resummation) performs best in the intermediate transverse momentum region (2-12 GeV), though no single model is clearly superior across all momentum ranges.

## Method

The authors numerically solve the Fokker-Planck equation to evolve charm quark transverse momentum distribution functions from initial proper time tau_0 = 0.33 fm/c to freeze-out temperature T_c = 155 MeV in Pb-Pb collisions at sqrt(S_NN) = 5.02 TeV. Three distinct approaches for collisional energy loss are implemented: Model A (Bjorken, 1982) which regulates infrared singularities using a Debye mass from QED; Model B (Thoma and Gyulassy, 1990) which uses induced chromoelectric fields and dielectric functions; and Model C (Braaten and Thoma, 1991) which separates soft and hard contributions using hard-thermal loop (HTL) framework. Radiative energy loss is included using the DGLV reaction operator formalism with a generalized dead cone approach, and the drag and diffusion coefficients are computed from the total energy loss with a free parameter k controlling the radiative contribution. The resulting nuclear modification factor R_AA is compared to ALICE (2018, 2021, 2022) and ATLAS (2022) data, with parameters optimized by minimizing chi-squared using the Minuit package.

## Key numbers

- Collision energy: sqrt(S_NN) = 5.02 TeV for Pb-Pb collisions
- Initial proper time: tau_0 = 0.33 fm/c
- Initial temperature: T_0 = 403 MeV
- Freeze-out temperature: T_c = 155 MeV
- Viscosity to entropy ratio: eta/s = 1/(4 pi) (constant)
- QCD cut-off parameter: Lambda_MS = 80 MeV
- Number of active flavors: N_f = 3
- For ALICE 2021 data (intermediate P_T 2-12 GeV): Model C achieves chi-squared = 0.76, compared to Model A chi-squared = 2.5 and Model B chi-squared = 1.9
- For ALICE 2022 data (all P_T): Model B achieves chi-squared = 0.88, Model C chi-squared = 0.97, Model A chi-squared = 1.06
- For ALICE 2022 data (intermediate P_T 2-12 GeV): Model C achieves chi-squared = 0.24, Model B chi-squared = 0.52, Model A chi-squared = 0.73
- For ALICE 2018 data (all P_T): Model B chi-squared = 0.81, Model C chi-squared = 0.82, Model A chi-squared = 0.86
- For ATLAS 2022 data (all P_T): Model C chi-squared = 0.19, Model A chi-squared = 0.25, Model B chi-squared = 0.24

## Context

This work extends previous studies of heavy quark energy loss in QGP by comparing three established theoretical approaches (Bjorken (1982), Thoma and Gyulassy (1990), and Braaten and Thoma (1991)) against the most recent experimental data from ALICE (2018-2022) and ATLAS (2022) at LHC energies. The study uses the Fokker-Planck equation framework similar to earlier work by Tripathy et al. (2016, 2017) and Sheibani et al. (2021), but with the novel addition of systematically comparing multiple collisional energy loss formalisms using the same numerical evolution and optimization procedure. The results enable future work to focus on improving descriptions at high and low transverse momentum, where all models show deviations from data, and suggest that the Braaten-Thoma HTL approach is most suitable for intermediate momentum regions while no single model is globally superior.
