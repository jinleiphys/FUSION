# Longitudinal boost-invariance of charge balance function in hadron-hadron and nucleus-nucleus collisions

> Using Monte Carlo generators of the PYTHIA model for hadron-hadron collisions and a multi-phase transport (AMPT) model for nucleus-nucleus collisions, the longitudinal boost-invariance of charge balance function and its transverse momentum dependence are carefully studied. It shows that the charge balance function is boost-invariant in both {\it p}+{\it p} and Au+Au collisions in these two models, consistent with experimental data. The balance function properly scaled by the width of the pseudorapidity window is independent of the position or the size of the window and is corresponding to the balance function of the whole pseudorapidity range. This longitudinal property of balance function also holds for particles in small transverse momentum ranges in the PYTHIA and the AMPT default models, but is violated in the AMPT with string melting. The physical origin of the results are discussed.

## Key claim

The charge balance function is boost-invariant in both p+p and Au+Au collisions as simulated by the PYTHIA and AMPT models, consistent with experimental data; however, this longitudinal boost-invariance in small transverse momentum ranges is violated in the AMPT model with string melting, due to the different hadronization scheme.

## Method

The authors used the PYTHIA 5.720 Monte Carlo generator for p+p collisions at 22 and 200 GeV, and two versions of the multi-phase transport (AMPT) model (default v1.11 and string melting v2.11) for Au+Au collisions at 200 GeV. They computed the charge balance function in pseudorapidity windows of varying size and position, and examined its scaling property and transverse momentum dependence by comparing the balance functions from different windows.

## Key numbers

- For p+p collisions at sqrt(s)=22 GeV and 200 GeV in PYTHIA, the balance function measured in four pseudorapidity windows of equal size |eta_w|=3 at different positions (-3,0), (-2,1), (-1,2), and (0,3) are approximately identical.
- For Au+Au collisions at sqrt(s)=200 GeV in both AMPT default and AMPT with string melting, the balance function in five pseudorapidity windows of equal size |eta_w|=2 at positions (-3,-1), (-2,0), (-1,1), (0,2), and (1,3) shows boost-invariance.
- The scaled balance function B_s(delta_eta) deduced from six different pseudorapidity windows (e.g., (-0.8,0.8), (1,3), (-3,1), (-2.4,2.4), (0,3), (-2,-1)) coincides with the balance function of the whole pseudorapidity range (-4,4) for Au+Au collisions.
- For p+p collisions, the boost-invariance holds in three transverse momentum bins: (0 < pT < 0.2), (0.2 < pT < 0.4), and (pT > 0.2) GeV/c.
- For Au+Au collisions, the boost-invariance holds in four pT bins (0.15-0.4, 0.4-0.7, 0.7-1, and 1-2 GeV/c) in the AMPT default model, but is violated in the AMPT with string melting.

## Context

This work extends the experimental findings of the NA22 collaboration (1990s) on boost-invariance of the balance function in hadron-hadron collisions and the STAR collaboration (2000s) on the scaling property in Au+Au collisions at 200 GeV. The study uses PYTHIA and AMPT models to test whether these properties arise from the string fragmentation mechanism versus the quark-coalescence mechanism in hadronization. The results enable using the transverse momentum dependence of the balance function as a sensitive probe for charge balance in hadronization mechanisms, particularly to distinguish between string fragmentation and quark coalescence.
