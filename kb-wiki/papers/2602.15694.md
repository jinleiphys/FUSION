# Charm and strange meson fragmentation functions

> Quark fragmentation functions describe the hadronization process of a quark where any of the final-state hadrons carries a fraction of its initial momentum. We compute these fragmentation functions for a cascade that includes pions, kaons, and the charmed $D$ and $D_s$ mesons, starting from the elementary quark-to-meson fragmentation process. The latter is obtained from the relevant cut diagram, and employs Poincaré covariant Bethe-Salpeter wave functions and quark propagators. We derive a set of twenty-five coupled jet equations that describe the cascade of emitted mesons in the fragmentation process. Their solutions yield full fragmentation functions that offer a consistent picture of the quark fragmentations across the light and heavy sectors.

## Key claim

The authors compute full fragmentation functions for pions, kaons, and charmed D and D_s mesons within a Poincaré covariant framework based on Bethe-Salpeter equations, claiming a consistent description across light and heavy sectors; however, the results rely on a rainbow-ladder truncation of the Dyson-Schwinger equations and neglect contributions from vector mesons and baryons, which the authors acknowledge as future work.

## Method

The authors first compute elementary quark-to-meson fragmentation functions using cut diagrams with Poincaré covariant Bethe-Salpeter wave functions and dressed quark propagators obtained from the Dyson-Schwinger equation in rainbow-ladder truncation. They then derive a set of twenty-five coupled integral equations (jet equations) that describe the cascade of sequential meson emissions, where an initial quark produces an intermediate quark that subsequently fragments into the final meson. The full fragmentation functions are obtained by solving these coupled equations iteratively, with the elementary splitting functions related to the elementary fragmentation functions via G-parity relations.

## Key numbers

- The momentum sum rule \(\sum_m \langle z D_q^m(z) \rangle = 1\) is satisfied to within less than 1% numerical precision for all quark flavors.
- The elementary fragmentation function \(\hat{d}(z)_{u}^{\bar{D}^{0}}\) has fit parameters \(N = 67.23 \times 10^{3}\), \(\alpha = 12.92\), \(\beta = 7.04\) for the form \(N z^{\alpha}(1-z)^{\beta}\).
- The elementary fragmentation function \(\hat{d}(z)_{c}^{D^{0}}\) has fit parameters \(N = 58.12 \times 10\), \(\alpha = 10.91\), \(\beta = 2.24\).
- The elementary fragmentation function \(\hat{d}(z)_{c}^{D_{s}^{+}}\) has fit parameters \(N = 22.10 \times 10^{4}\), \(\alpha = 18.26\), \(\beta = 4.96\).
- The elementary fragmentation function \(\hat{d}(z)_{s}^{D_{s}^{-}}\) has fit parameters \(N = 26.48 \times 10^{4}\), \(\alpha = 13.89\), \(\beta = 7.53\).
- The elementary fragmentation function \(\hat{d}(z)_{u}^{K^{+}}\) has fit parameters \(N = 53.60\), \(\alpha = 5.35\), \(\beta = 2.40\).
- The elementary fragmentation function \(\hat{d}(z)_{u}^{\pi^{+}}\) has fit parameters \(N = 11.83\), \(\alpha = 2.41\), \(\beta = 1.38\).

## Context

This work extends a previous study by da Silveira et al. (2024) that was limited to pion isospin states, generalizing the formalism to include kaons and charmed D and D_s mesons. It also extends earlier work using the Nambu-Jona-Lasinio model by Matevosyan et al. (2010) to include charm quarks. The results provide a basis for future applications to include vector mesons and nucleons, and the analytic parametrizations of the elementary fragmentation functions enable implementation in DGLAP evolution software packages.
