# Neutron skin in $^{48}$Ca determined from p+$^{48}$Ca and $^{48}$Ca+$^{12}$C scattering

> In our previous paper, we determined $r_{\rm skin}^{208}({\rm exp})=0.278 \pm 0.035$~fm from $σ_{\rm R}$ for p+$^{208}$Pb scattering, using the Kyushu (chiral) $g$-matrix folding model with the densities calculated with D1S-GHFB with the angular momentum projection (AMP). The value agrees with that of PREX2. Reaction cross sections $σ_{\rm R}$ are available for p+$^{48}$Ca scattering, whereas interaction cross sections $σ_{\rm I}$ are available for $^{48}$Ca + $^{12}$C scattering. As for $^{48}$Ca, the high-resolution $E1$ polarizability experiment ($E1$pE) yields $r_{\rm skin}^{48}(E1{\rm pE}) =0.14 \sim 0.20~{\rm fm}$. We determine $r_{\rm skin}^{48}({\rm exp})$ from the data on $σ_{\rm R}$ for p+$^{48}$Ca scattering and from the data on $σ_{\rm I}$ for $^{48}$Ca+$^{12}$C scattering. We use the Kyushu $g$-matrix folding model with the densities calculated with the D1M-GHFB+AMP densities. The D1M-GHFB+AMP proton and neutron densities are scaled so as to reproduce the data under the condition that the radius $r_{\rm p}$ of the scaled proton density equals the data $r_{\rm p}({\rm exp})$ determined from the electron scattering. We deduce skin values $r_{\rm skin}=r_{\rm n}({\rm exp})-r_{\rm p}({\rm exp})$ from the resulting $r_{\rm n}({\rm exp})$ and the $r_{\rm p}({\rm exp})$ determined from electron scattering. The same procedure is taken for D1S-GHFB+AMP. We regard $r_{\rm skin}^{48}(E1{\rm pE})$ as a reference skin value. Using the reference skin value and taking D1M-GHFB+AMP, we determine $r_{\rm skin}^{48}({\rm exp})=0.158 \pm 0.025$~fm for p+$^{48}$Ca scattering and $0.160 \pm 0.058$~fm for $^{48}$Ca + $^{12}$C scattering. We take the weighted mean and its error for the two skin values. The result is $r_{\rm skin}^{48}({\rm exp})=0.158 \pm (0.023)_{\rm exp} \pm (0.012)_{\rm th}~{\rm fm}$.

## Key claim

The neutron skin thickness of $^{48}$Ca is determined from reaction and interaction cross section data using a folding model, yielding a value consistent with the range from the high-resolution $E1$ polarizability experiment ($E1$pE), but the result depends on which Gogny interaction (D1M or D1S) is used for the nuclear densities, with D1M giving better agreement with total energy and cross section data.

## Method

The authors used the Kyushu $g$-matrix folding model to analyze reaction cross sections $\sigma_{\rm R}$ for p+$^{48}$Ca scattering at incident energies of 30-48 MeV and interaction cross sections $\sigma_{\rm I}$ for $^{48}$Ca+$^{12}$C scattering at 280 MeV per nucleon. Proton and neutron densities from D1M-GHFB+AMP (Gogny-D1M Hartree-Fock-Bogoliubov with angular momentum projection) were scaled to reproduce the experimental cross sections under the condition that the proton radius $r_{\rm p}$ of the scaled density equals the experimental value from electron scattering. The neutron skin thickness $r_{\rm skin}=r_{\rm n}-r_{\rm p}$ was then extracted from the resulting neutron radius and the electron-scattering proton radius.

## Key numbers

* $r_{\rm skin}^{48}({\rm exp}) = 0.158 \pm 0.025$ fm for p+$^{48}$Ca scattering in $E_{\rm in}=30 \sim 48$ MeV, using D1M-GHFB+AMP densities
* $r_{\rm skin}^{48}({\rm exp}) = 0.160 \pm 0.058$ fm for $^{48}$Ca+$^{12}$C scattering at 280 MeV per nucleon, using D1M-GHFB+AMP densities
* Final weighted mean: $r_{\rm skin}^{48}({\rm exp}) = 0.158 \pm (0.023)_{\rm exp} \pm (0.012)_{\rm th}$ fm, where the theoretical error comes from the difference between D1M and D1S results
* For D1S-GHFB+AMP, the weighted mean is $r_{\rm skin}^{48}({\rm exp}) = 0.125 \pm 0.02$ fm
* The $r_{\rm skin}^{48}$-$L$ relation from 206 equations of state yields $r_{\rm skin}^{48} = 0.0009 L + 0.125$ fm, implying $L = 0 \sim 76$ MeV from the final skin value

## Context

This work extends the authors' previous determination of $r_{\rm skin}^{208}({\rm exp}) = 0.278 \pm 0.035$ fm for $^{208}$Pb from p+$^{208}$Pb scattering (Tagami, 2020), which agreed with the PREX2 result. For $^{48}$Ca, the result is consistent with the $E1$ polarizability experiment range of $r_{\rm skin}^{48}(E1{\rm pE}) = 0.14 \sim 0.20$ fm (Birkhan, 2016) and with coupled-cluster calculations (Hagen, 2015) giving $r_{\rm skin}^{48}({\rm CC}) = 0.12 \sim 0.15$ fm. The deduced slope parameter $L = 0 \sim 76$ MeV is smaller than the range $L = 76 \sim 165$ MeV from PREX2, an issue that the ongoing CREX experiment at Jefferson Lab is expected to resolve.
