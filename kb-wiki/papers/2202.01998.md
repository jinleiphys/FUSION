# Bayesian inference of the fluctuating proton shape

> Using Bayesian inference, we determine probabilistic constraints on the parameters describing the fluctuating structure of protons at high energy. We employ the color glass condensate framework supplemented with a model for the spatial structure of the proton, along with experimental data from the ZEUS and H1 Collaborations on coherent and incoherent diffractive $\mathrm{J}/ψ$ production in e+p collisions at HERA. This data is found to constrain most model parameters well. This work sets the stage for future global analyses, including experimental data from e+p, p+p, and p+A collisions, to constrain the fluctuating structure of nucleons along with properties of the final state.

## Key claim

Using Bayesian inference, the authors determine probabilistic constraints on parameters describing the fluctuating spatial structure of protons at high energy, finding that most model parameters are well constrained by HERA data on exclusive J/ψ production, but the number of hot spots and the minimum distance between them cannot be determined from this data alone.

## Method

The authors employ the color glass condensate (CGC) framework with the IP-Glasma model for the proton's spatial structure, representing the proton as a collection of hot spots with fluctuating densities and positions. They perform a Bayesian analysis using Gaussian process emulators trained on 1,000 simulations and Monte Carlo Markov Chain sampling, comparing model calculations of coherent and incoherent diffractive J/ψ production cross sections to H1 data at W=75 GeV.

## Key numbers

- Proton RMS radius extracted: 0.591^{+0.130}_{-0.071} fm (variable N_q) and 0.610^{+0.065}_{-0.068} fm (N_q=3 fixed), with 90% credible intervals
- Infrared regulator m: MAP value 0.506^{+1.12}_{-0.356} GeV (variable N_q) and 0.246^{+0.162}_{-0.103} GeV (N_q=3 fixed)
- Hot spot size B_q: MAP value 0.474^{+0.434}_{-0.286} GeV^{-2} (variable N_q) and 0.346^{+0.282}_{-0.202} GeV^{-2} (N_q=3 fixed)
- Saturation scale fluctuation parameter σ: MAP value 0.833^{+0.194}_{-0.441} (variable N_q) and 0.563^{+0.143}_{-0.141} (N_q=3 fixed)
- Number of hot spots N_q: MAP value 6.79^{+2.93}_{-4.83} (variable case), with prior range [1,10]
- Minimum distance d_q,Min: MAP value 0.257^{+0.221}_{-0.231} fm (variable N_q) and 0.254^{+0.222}_{-0.229} fm (N_q=3 fixed), with prior range [0, 0.5] fm

## Context

This work extends previous studies by Mantysaari et al. (2016) where model parameters were constrained "by eye" rather than statistically, and provides a rigorous Bayesian framework for extracting proton geometry from exclusive vector meson production data. The results enable future global analyses that can combine HERA DIS data with LHC data from proton-proton and proton-lead collisions to simultaneously constrain nucleon substructure fluctuations and final state properties, and set the stage for analyses with future EIC data.
