# Correct Definition of the Gluon Fragmentation Function at High Energy Colliders

> Since the definition of the gluon to hadrons fragmentation amplitude $<H,X|g>$ involves gluon field $Q^{μa}(x)$ corresponding to single gluon incoming state $|g>$, the present definition of the gluon fragmentation function at high energy colliders [which involves non-abelian field tensor $F^{μνa}(x)=\partial^μQ^{νa}(x) -\partial^νQ^{μa}(x)+gf^{abc}Q^{μb}(x)Q^{νc}(x)$ instead of $Q^{μa}(x)$ in the initial state] is not consistent with the single gluon incoming state $|g>$. In this paper we derive the correct definition of the gluon fragmentation function at high energy colliders from first principles which is obtained from the single gluon incoming state $|g>$ and is gauge invariant and is consistent with the factorization theorem in QCD.

## Key claim

The paper argues that the existing definition of the gluon-to-hadron fragmentation function at high energy colliders is incorrect because it uses the non-abelian field tensor \(F^{\mu\nu a}(x)\) (which contains quadratic, cubic, and quartic powers of the quantum gluon field) instead of the gluon field \(Q^{\mu a}(x)\) itself, making it inconsistent with the single-gluon incoming state \(|g\rangle\). The authors derive a corrected definition that uses \(Q^{\mu a}(x)\) directly, is gauge invariant, and is consistent with the factorization theorem in QCD.

## Method

The author starts from the field-theoretical definition of a scalar gluon fragmentation function, then generalizes to QCD to obtain a gauge non-invariant gluon fragmentation function involving \(Q^{\mu a}(x) Q_\mu^a(0)\). Using the background field method of QCD and the eikonal approximation for infrared divergences, the author proves factorization of infrared divergences for the non-perturbative gluon correlation function in the presence of an SU(3) pure gauge background field. This factorization is used to incorporate light-like Wilson lines in the adjoint representation, yielding a gauge invariant definition of the gluon fragmentation function that is consistent with the single-gluon initial state.

## Key numbers

* The correct definition of the gluon fragmentation function is given by Eq. (gpdffpiv): \(D_{H/g}(z) = \frac{zk^+}{32\pi} \int dx^- e^{iP^+ x^-/z} [\langle 0| Q^{\mu a}(x^-) \times \mathcal{P}\exp[-igT^{(A)b}\int_0^\infty dz^- A^{+b}(x^-+z^-)] a^\dagger_H(P^+,0_T) a_H(P^+,0_T) \times \mathcal{P}\exp[-igT^{(A)c}\int_0^\infty dz^- A^{+c}(z^-)] \times Q_\mu^a(0) |0\rangle]\).
* The existing (incorrect) definition uses the non-abelian field tensor \(F^{\mu\nu a}(x) = \partial^\mu Q^{\nu a}(x) - \partial^\nu Q^{\mu a}(x) + g f^{abc} Q^{\mu b}(x) Q^{\nu c}(x)\) instead of \(Q^{\mu a}(x)\) in the initial state.
* The factorization theorem for the non-perturbative gluon correlation function is proven at all orders in the coupling constant: \(\langle 0| Q_\mu^a(x_1) Q_\nu^b(x_2) |0\rangle = \langle 0| \Phi_{ac}(x_1) Q_\mu^c(x_1) \Phi_{bd}(x_2) Q_\nu^d(x_2) |0\rangle_A\), where \(\Phi_{ab}(x)\) is the non-abelian phase factor in the adjoint representation of SU(3).

## Context

This work extends the formalism of Collins, Soper, and Sterman (1985) for parton fragmentation functions, specifically correcting the gluon fragmentation function definition which previously used the field tensor \(F^{\mu\nu a}\) instead of the gluon field \(Q^{\mu a}\). The paper builds on the background field method of 't Hooft (1971) and Abbott (1981), and on factorization proofs for infrared divergences in QCD. The corrected definition enables consistent application of the factorization theorem for hadron production at high energy colliders, ensuring that uncanceled infrared divergences in partonic cross sections can be properly absorbed into the non-perturbative fragmentation function.
