# ADG: Automated generation and evaluation of many-body diagrams I. Bogoliubov many-body perturbation theory

> We describe the first version (v1.0.0) of the code ADG that automatically (1) generates all valid Bogoliubov many-body perturbation theory (BMBPT) diagrams and (2) evaluates their algebraic expression to be implemented for numerical applications. This is achieved at any perturbative order $p$ for a Hamiltonian containing both two-body (four-legs) and three-body (six-legs) interactions (vertices). The automated generation of BMBPT diagrams of order $p$ relies on elements of graph theory, i.e., it is achieved by producing all oriented adjacency matrices of size $(p+1) \times (p+1)$ satisfying topological Feynman's rules. The automated evaluation of BMBPT diagrams of order $p$ relies both on the application of algebraic Feynman's rules and on the identification of a powerful diagrammatic rule providing the result of the remaining $p$-tuple time integral. The diagrammatic rule in question constitutes a novel finding allowing for the straight summation of large classes of time-ordered diagrams at play in the time-independent formulation of BMBPT. Correspondingly, the traditional resolvent rule employed to compute time-ordered diagrams happens to be a particular case of the general rule presently identified. The code ADG is written in Python2.7 and uses the graph manipulation package NetworkX. The code is also able to generate and evaluate Hartree-Fock-MBPT (HF-MBPT) diagrams and is made flexible enough to be expanded throughout the years to tackle the diagrammatics at play in various many-body formalisms that already exist or are yet to be formulated.

## Key claim

The paper presents the first version (v1.0.0) of the ADG code that automatically generates and evaluates all valid Bogoliubov many-body perturbation theory (BMBPT) diagrams at any perturbative order, including a novel diagrammatic rule for evaluating the p-tuple time integral that generalizes the traditional resolvent rule; the code is demonstrated for Hamiltonians containing both two-body and three-body interactions, but the authors note that the reference state and unperturbed Hamiltonian can be defined more generally than the canonical Hartree-Fock-Bogoliubov choice.

## Method

The automated generation of BMBPT diagrams relies on graph theory: all oriented adjacency matrices of size (p+1) x (p+1) satisfying topological Feynman rules are produced, then converted into graph objects using the NetworkX package. The automated evaluation applies algebraic Feynman rules combined with a newly identified diagrammatic rule that directly provides the result of the remaining p-tuple time integral, allowing summation of large classes of time-ordered diagrams without the traditional resolvent rule. The code is written in Python2.7 and can also generate and evaluate Hartree-Fock-MBPT (HF-MBPT) diagrams as a subcase.

## Key numbers

- The code generates diagrams at any perturbative order p using oriented adjacency matrices of size (p+1) x (p+1)
- The Hamiltonian includes two-body (four-legs) and three-body (six-legs) interaction vertices
- The code handles operators containing one-body, two-body, and three-body terms (o^{[2]}, o^{[4]}, and o^{[6]})
- The unperturbed quasi-particle energies are defined as E_k > 0 for all k, with strict positivity of unperturbed excitations E_{k1 k2 ...} = E_{k1} + E_{k2} + ...
- The code is written in Python2.7 and uses the graph manipulation package NetworkX

## Context

This work extends the recent formulation of Bogoliubov many-body perturbation theory (BMBPT) by Duguet (2015) and Arthuis (2018) for treating superfluid open-shell nuclei, and builds on earlier automated diagram generation attempts by Paldus (1973), Wong (1973), Kaldor (1976), Csepes (1988), Lyons (1994), and Stevenson (2003). The code is designed to be flexible enough for future expansion to other many-body formalisms such as Gorkov-SCGF theory and particle-number projected BMBPT (PNP-BMBPT), and has already enabled second- and third-order BMBPT calculations in mid- and heavy-mass nuclei by Tichai (2018).
