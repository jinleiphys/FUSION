# Nucleon-Nucleon Potential from Holography

> In the holographic model of QCD, baryons are chiral solitons sourced by D4 flavor instantons in bulk of size 1/\sqrtλ with λ=g^2*N_c. Using the ADHM construction we explicit the exact two-instanton solution in bulk. We use it to construct the core NN potential to order N_c/λ. The core sources meson fields to order \sqrt{N_c/λ} which are shown to contribute to the NN interaction to order N_c/λ. In holographic QCD, the NN interaction splits into a small core and a large cloud contribution in line with meson exchange models. The core part of the interaction is repulsive in the central, spin and tensor channels for instantons in the regular gauge. The cloud part of the interaction is dominated by omega exchange in the central channel, by pion exchange in the tensor channel and by axial-vector exchange in the spin and tensor channels. Vector meson exchanges are subdominant in all channels.

## Key claim

In holographic QCD, the nucleon-nucleon (NN) interaction splits into a small core contribution and a large cloud contribution, with the core part being repulsive in the central, spin, and tensor channels for instantons in the regular gauge, and the cloud part dominated by omega exchange in the central channel, pion exchange in the tensor channel, and axial-vector exchange in the spin and tensor channels, all to order N_c/lambda.

## Method

The authors used the ADHM construction to obtain the exact two-instanton solution in flat R^3 x R_Z space, which in holographic QCD corresponds to a two-baryon configuration sourced by D4 flavor instantons in bulk. They constructed the core NN potential to order N_c/lambda from the two-instanton solution, then added meson cloud contributions (pions and vector mesons) to order sqrt(N_c/lambda) using semiclassical methods in the Born-Oppenheimer approximation. The core interaction was analyzed for both regular and singular gauge instantons, and the cloud contributions were projected onto nucleon-nucleon channels at large separation.

## Key numbers

* The instanton size in the holographic model is of order 1/sqrt(lambda) with lambda = g^2 N_c.
* The one-instanton stabilizes at a rescaled size wt_rho_0 ~ 9.64, with a mass correction Delta M(wt_rho_0 ~ 9.64) ~ 0.365.
* The core NN potential is constructed to order N_c/lambda, and the meson cloud contributes to order N_c/lambda.
* The instanton mass is of order N_c lambda in units of M_KK, the Kaluza-Klein scale.

## Context

This work extends the holographic QCD approach to baryons developed by Sakai and Sugimoto (2005), where baryons are chiral solitons sourced by D4 instantons in bulk. It contrasts with earlier Skyrmion-Skyrmion interaction studies using the product ansatz (ANSATZ) or variational techniques (OKA), which lacked intermediate-range attraction in the scalar channel and produced mostly repulsive scalar potentials to order N_c. The paper also builds on work by YAMA (2005) showing that ansatz dependence could be eliminated by adding the pion cloud to core Skyrmions. This holographic approach fixes core interactions in a way the Skyrme model does not, and enables quantitative analysis of NN interactions at intermediate distances where strong coupling applies.
