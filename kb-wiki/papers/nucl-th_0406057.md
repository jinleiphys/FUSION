# Nucleon resonances and processes involving strange particles

> An existing single resonance model with S11, P11 and P13 Breit-Wiegner resonances in the s-channel has been re-applied to the old pi N --> K Lambda data. It has been shown that the standard set of resonant parameters fails to reproduce the shape of the differential cross section. The resonance parameter determination has been repeated retaining the most recent knowledge about the nucleon resonances. The extracted set of parameters has confirmed the need for the strong contribution of a P11(1710) resonance. The need for any significant contribution of the P13 resonance has been eliminated. Assuming that the Baker. et al data set\cite{Bak78} is a most reliable one, the P11 resonance can not but be quite narrow. It emerges as a good candidate for the non-strange counter partner of the established pentaquark anti-decuplet.

# Nucleon resonances and processes involving strange particles

## Key claim
The P11(1710) resonance emerges as a good candidate for the non-strange counterpart of the established pentaquark anti-decuplet, but only if the Baker et al. data set is taken as the most reliable one; under that assumption the P11 resonance must be quite narrow, contrary to standard parameter sets.

## Method
An existing single resonance model with S11, P11, and P13 Breit-Wigner resonances in the s-channel was re-applied to old pi N -> K Lambda data. The resonance parameter determination was repeated by varying only the K Lambda branching fractions while keeping masses, widths, and branching fractions to other channels fixed at values from a coupled channel analysis of pi-nucleon scattering. Three solution sets were extracted: two fitting the lower and upper bands of the total cross section while imposing correct angular dependence, and one fitting only the Baker et al. data.

## Key numbers
- PDG standard parameters: S11(1650) mass 1650 MeV, width 150 MeV, x_K Lambda 7%; P11(1710) mass 1710 MeV, width 100 MeV, x_K Lambda 15%; P13(1720) mass 1720 MeV, width 150 MeV, x_K Lambda 6.5%
- Solution 3 (fitting only Baker et al. data): P11 mass 1700 MeV, width 60 MeV, x_K Lambda 30%; S11 x_K Lambda 4%; P13 x_K Lambda 0.16%
- Solution 1: P11 x_K Lambda 23%; S11 x_K Lambda 2.4%; P13 x_K Lambda 0.16%
- Solution 2: P11 x_K Lambda 35%; S11 x_K Lambda 2.4%; P13 x_K Lambda 0.16%
- The standard PDG parameter set reproduces the total cross section absolute value but fails to reproduce the differential cross section shape except at w = 1683 MeV

## Context
This work extends single resonance model studies of pi N -> K Lambda (Tsu00) and responds to the reported observation of exotic pentaquark Theta(1539) and Xi(1862) states (Pen04), which motivated the search for a non-strange pentaquark anti-decuplet member. The results contradict the standard PDG parameter set and coupled channel analyses (PWA) that accept a broad P11(1710), instead finding negligible P13 contribution and a narrow P11. The authors call for remeasuring the differential cross section in the 1600-1800 MeV energy range to enable decisive conclusions when incorporated into existing coupled channel partial wave analyses.
