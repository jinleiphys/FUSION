# Longitudinal dynamics and particle production in relativistic nuclear collisions

> This work presents a three-dimensional dynamical initialization model for relativistic heavy-ion collisions, implementing local energy-momentum conservation and baryon charge fluctuations at string junctions. Constraining parameters using experimental data from p+p collisions at various collision energies, the model provides a very good description of the charged hadron and net proton rapidity distributions in Au+Au collisions from 7.7 to 200 GeV and Pb+Pb collisions at 8.77 and 17.3 GeV. We demonstrate the importance of fluctuations of baryon densities to string junctions for describing net-proton distributions at collision energies of 62.4 and 200 GeV. Including this improved baryon stopping description along with the requirement of strangeness neutrality also yields a good description of identified particle yields as functions of the collision energy above 7.7 GeV. We further study asymmetric p+Al and (p, d, $^3$He)+Au collisions at the top RHIC energy and p+Pb, Xe+Xe, and Pb+Pb collisions at LHC energies. We identify the produced particle rapidity distributions in asymmetric collision systems as particularly useful for constraining models of the early-time longitudinal dynamics.

## Key claim

The paper presents a three-dimensional dynamical initialization model for relativistic heavy-ion collisions that, when constrained by p+p data, provides a good description of charged hadron and net proton rapidity distributions across a wide energy range, but the model's success depends on including fluctuations of baryon densities at string junctions and the requirement of strangeness neutrality, with the description being valid only for collision energies above 7.7 GeV.

## Method

The authors develop a 3D Monte-Carlo Glauber initial state model with subnucleonic structure, where nucleons are treated as collections of three valence quarks plus a soft partonic cloud, and implement a classical string deceleration model for energy loss in nucleon-nucleon collisions with a logit-normal distribution for rapidity loss fluctuations. The initial state is dynamically coupled to a (3+1)D viscous hydrodynamic framework (MUSIC) with a lattice-based equation of state at finite density (NEOS-BQS), followed by hadronic transport using UrQMD, with the complete framework integrated in the iEBE-MUSIC framework.

## Key numbers

- Valence quark mass set to m_parton = 0.312 GeV
- Transverse smearing width sigma_perp = 0.5 fm
- Specific shear viscosity eta T/(e + P) = 0.12
- Switching energy density e_sw = 0.25 GeV/fm^3 for p+p collisions (corresponding to temperature 150 MeV at zero net baryon density)
- Rapidity loss parameters: A = 1.32, alpha_1 = 1.8, alpha_2 = 0.35
- Variance of rapidity loss fluctuation sigma = 0.6
- String junction parameter lambda_B = 0.2
- Average number of strings per NN collision <N_string> ~ 1.507
- Longitudinal smearing parameter sigma_eta_s = 0.2 for sqrt(s_NN) < 25 GeV, 0.5 for sqrt(s_NN) in [25, 100] GeV, 0.6 for sqrt(s_NN) > 100 GeV
- Cold corona cutoff energy density e_low = 0.05 GeV/fm^3
- Nucleon remnant rapidity loss reduction factor alpha_rem = 0.5

## Context

This work extends the 3D Monte-Carlo Glauber model first introduced by Shen (2017) by imposing energy-momentum conservation constraints on string production and introducing baryon charge fluctuations at string junctions motivated by Kharzeev (1996). The model enables systematic studies of longitudinal dynamics across collision energies from 7.7 GeV to TeV scales, and the authors identify particle rapidity distributions in asymmetric collision systems (p+Al, p+Au, d+Au, He-3+Au, p+Pb, Xe+Xe, Pb+Pb) as particularly useful for constraining early-time longitudinal dynamics models.
