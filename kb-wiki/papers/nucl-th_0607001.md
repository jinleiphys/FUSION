# Isoscalar short-range current in the deuteron induced by an intermediate dibaryon

> A new model for short-range isoscalar currents in the deuteron and in the NN system is developed; it is based on the generation of an intermediate dibaryon which is the basic ingredient for the medium- and short-range NN interaction which was proposed recently by the present authors.This new current model can very well describe the experimental data for the three basic deuteron observables of isoscalar magnetic type, viz. the magnetic moment, the circular polarization of the photon in the $np\to dγ$ process at thermal neutron energies and the structure function B up to Q$^2$=60 fm$^{-2}$.

## Key claim

The paper proposes that a new isoscalar short-range current, generated by an intermediate dressed dibaryon state in the deuteron, can resolve long-standing discrepancies in three deuteron observables of isoscalar magnetic type, provided the dibaryon is treated as a real physical degree of freedom rather than a formal EFT device.

## Method

The authors construct a coupled-channel model where the deuteron wavefunction has both a nucleon-nucleon (NN) component and a dressed dibaryon (DB) component, with the mixing determined by a transition operator based on scalar exchange between quarks. The short-range NN interaction is represented by a separable, energy-dependent potential derived from a six-quark shell model with sigma-meson dressing, and the electromagnetic current includes both diagonal dibaryon currents and transitional currents between the NN and DB channels.

## Key numbers

- Weight of the dibaryon component in the deuteron: beta^2 = 0.025 - 0.035 (2.5% - 3.5%) for different model versions
- The dibaryon model fits NN phase shifts from 0 to 1000 MeV using only four parameters in the singlet ^1S_0 channel and a few more for the coupled ^3S_1-^3D_1 channel
- The structure function B(Q^2) is described up to Q^2 = 60 fm^{-2}
- The experimental circular polarization P_gamma = (-1.5 +/- 0.3) x 10^{-3} for the np -> d gamma process at thermal neutron energies

## Context

This work extends the authors' previously developed Moscow-Tuebingen (MT) dibaryon model for NN interactions (Kukulin et al., 2002-2005) to electromagnetic observables, addressing failures of conventional meson-exchange current models to describe the deuteron magnetic moment, circular polarization in np -> d gamma, and the magnetic form factor B(Q^2) around the diffraction minimum at Q^2 ~ 45-55 fm^{-2}. The approach contrasts with EFT treatments (Beane, 2002; Fleming, 2000) that introduce dibaryons formally, and with earlier six-quark bag models (Yamauchi, 1986; Kisslinger, 1988) that did not achieve quantitative agreement with data. The model enables consistent treatment of short-range correlations and electromagnetic currents without the need for different cutoff parameters in different processes.
