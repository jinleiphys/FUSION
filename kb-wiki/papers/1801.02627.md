# Simplicity from Recoil: The Three-Loop Soft Function and Factorization for the Energy-Energy Correlation

> We derive an operator based factorization theorem for the energy-energy correlation (EEC) observable in the back-to-back region, allowing the cross section to be written as a convolution of hard, jet and soft functions. We prove the equivalence of the soft functions for the EEC and color singlet transverse-momentum resummation to all-loop order, and give their analytic result to three-loops. Large logarithms appearing in the perturbative expansion of the EEC can be resummed to all orders using renormalization group evolution. We give analytic results for all required anomalous dimensions to three-loop order, providing the first example of a transverse-momentum (recoil) sensitive $e^+e^-$ event shape whose anomalous dimensions are known at this order. The EEC can now be computed to next-to-next-to-next-to-leading logarithm matched to next-to-next-to-leading order, making it a prime candidate for precision QCD studies and extractions of the strong coupling constant. We anticipate that our factorization theorem will also be crucial for understanding non-perturbative power corrections for the EEC, and their relationship to those appearing in other observables.

## Key claim
The paper derives an all-orders factorization theorem for the energy-energy correlation (EEC) in the back-to-back region, proving that the soft function is identical to that for color singlet transverse-momentum resummation. This equivalence holds to all loop orders, enabling the first three-loop anomalous dimensions for a transverse-momentum (recoil) sensitive \(e^+e^-\) event shape.

## Method
The authors start from the established factorization theorem for identified hadron production in the back-to-back limit and marginalize over it to obtain a factorization for the EEC. They use soft collinear effective theory (SCET\(_{\text{II}}\)) with a rapidity regulator implemented via spacetime shifts of Wilson lines, proving that the soft function is invariant under crossing of the Wilson lines. This allows them to map the EEC soft function onto the known three-loop soft function for color singlet \(q_T\) resummation, using bootstrap techniques from \(\mathcal{N}=4\) super Yang-Mills theory and a supersymmetric decomposition in transcendental weight.

## Key numbers
* The soft function for the EEC is computed analytically to three-loop order, matching the precision of the color singlet \(q_T\) soft function from Li (2016).
* The EEC can now be computed to next-to-next-to-next-to-leading logarithm (N\(^3\)LL) matched to next-to-next-to-leading order (NNLO), a first for a recoil-sensitive \(e^+e^-\) event shape.
* The back-to-back limit is characterized by \(z = (1-\cos\theta_{ij})/2 \to 1\), with the expansion parameter \(\lambda \sim \sqrt{1-z}\).

## Context
This work extends the resummation of the EEC beyond the NNLL+NNLO achieved by Tulipant (2017) using the Collins-Soper-Sterman formalism, matching the state-of-the-art precision previously available only for recoil-free observables like thrust (Becher, 2008; Abbate, 2010) and the \(C\)-parameter (Hoang, 2014). The factorization theorem enables N\(^3\)LL'+NNLO accuracy for the EEC, making it a prime candidate for precision QCD studies and extractions of the strong coupling constant, and also provides a framework for understanding non-perturbative power corrections and their relation to those in other observables.
