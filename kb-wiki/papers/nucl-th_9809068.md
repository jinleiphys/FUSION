# Full $0\hbarω$ shell model calculation of the binding energies of the $1f_{7/2}$ nuclei

> Binding energies and other global properties of nuclei in the middle of the $pf$ shell, such as M1, E2 and Gamow-Teller sum rules, have been obtained using a new Shell Model code (NATHAN) written in quasi-spin formalism and using a $j-j$-coupled basis. An extensive comparison is made with the recently available Shell Model Monte Carlo results using the effective interaction KB3. The binding energies for -nearly- all the $1f_{7/2}$ nuclei are compared with the measured (and extrapolated) results.

## Key claim

The authors demonstrate that full $0\hbar\omega$ shell model calculations using the new code NATHAN can reproduce experimental binding energies for nuclei in the $1f_{7/2}$ region with an rms deviation of about 200 keV, but only after adding a global monopole correction to the KB3 effective interaction; the agreement is limited to nuclei with $N,Z \leq 28$ because the KB3 interaction's quasiparticle gap around $^{56}$Ni is too strong by about 1 MeV, causing relative underbinding for nuclei beyond that point.

## Method

The authors performed full $pf$-shell diagonalizations using the new code NATHAN, which works in a $j$-$j$-coupled basis using quasi-spin formalism, enabling calculations that would involve more than one billion $M=0$ Slater determinants in the $m$-scheme (e.g., for $^{56}$Ni). They used the KB3 effective interaction throughout, with effective charges of 1.35 for protons and 0.35 for neutrons, bare $g$-factors, and an unquenched Gamow-Teller operator. To compare with experimental binding energies relative to $^{40}$Ca, they added Coulomb energies (from a global formula) and fitted a three-parameter monopole correction ($e_v$, $a$, $b$) to the 51 measured binding energies in the $1f_{7/2}$ region.

## Key numbers

- The rms deviation between the shell model binding energies and experiment is 227 keV (without mass-dependent scaling of matrix elements) and 215 keV (with mass-dependent scaling).
- The fitted monopole parameters without mass scaling are: $e_v = -8.67 \pm 0.01$ MeV, $a = 0.092 \pm 0.003$ MeV, $b = 0.063 \pm 0.006$ MeV.
- The fitted monopole parameters with mass scaling (at $A_0=42$) are: $e_v = -8.61 \pm 0.01$ MeV, $a = 0.041 \pm 0.003$ MeV, $b = 0.119 \pm 0.006$ MeV.
- SMMC ground state energies are above exact values by about 0.5 MeV in most cases, but discrepancies grow to 2 MeV for the heaviest nuclei studied.
- SMMC sum rules agree with exact results within 10%–15% error bars in most cases, with exceptions for Gamow-Teller strength in $^{60}$Fe, $^{62}$Ni, $^{64}$Ni, and $^{64}$Zn.

## Context

This work extends earlier full $pf$-shell calculations using the $m$-scheme code ANTOINE (Caurier et al., 1995) up to $A=52$, now covering nuclei from $A=40$ to $A=56$ with $20 \leq N,Z \leq 28$. The exact results serve as benchmarks for approximate methods such as the Shell Model Monte Carlo (SMMC) approach of Langanke et al. (1995) and the Quantum Monte Carlo diagonalization method. The binding energy systematics enable predictions for still-unmeasured masses in the $1f_{7/2}$ region and show that mass-dependent scaling of matrix elements is not critical for describing binding energies in this region, though it may become necessary for nuclei beyond $^{56}$Ni.
