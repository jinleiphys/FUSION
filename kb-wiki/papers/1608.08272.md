# The Expectation Value of S(1).S(2)-Wave Functions Don't Matter

> We consider the expectation value of the quantity [3+ σ(1).σ(2)]/4 . This has a value +1 for 2 nucleons with spin S=1and zero for S=0. We show that for the jj coupling 2 particle configuration [j(1) j(2)]^{J} the expectation value has the structure A+B J(J+1) where A and B are constants. We then show that for a 2proton-2neutron configuration with total angular momentum I the expectation value per pair is independent of the details of the wave function and has a similar structure A' +B' I(I+1) with B'=B/6.

## Key claim

For a system of two protons and two neutrons in a single-j shell (specifically the f7/2 shell), the expectation value per pair of the spin-one projection operator [3+σ(1)·σ(2)]/4 is independent of the detailed wave-function components and depends only on the total angular momentum I, taking the form A' + B' I(I+1) with B' = B/6 where B is the corresponding coefficient for the two-particle case.

## Method

The author evaluates the expectation value of the spin-one projection operator P1(J) = ⟨(jj)^J | [3+σ(1)·σ(2)]/4 | (jj)^J⟩ for two nucleons in jj coupling using unitary 9j coefficients, obtaining explicit numerical values for j = f7/2. For the four-particle system of 2 protons and 2 neutrons (⁴⁴Ti), the expectation value per pair is computed by summing over the proton-proton and neutron-neutron pair contributions weighted by the wave-function amplitudes D^I(J_P J_N) from a shell-model diagonalization with the MBZE interaction. The calculation is then simplified by noting that the two-body J·J interaction between basis states (J_P, J_N) and (J_P', J_N') vanishes unless J_P = J_P' and J_N = J_N', leading to degeneracy of all states of a given total angular momentum I.

## Key numbers

- For two nucleons in the f7/2 shell, P1(J) values range from 0.42857 (J=0) to 1.00000 (J=7), fitted as P1(J) = 0.42857 + 0.010204 J(J+1).
- For the four-particle system (⁴⁴Ti), the pair spin probability for J=0⁺ states is 0.643 for all four states (three with T=0, one with T=2), independent of wave-function details.
- The pair spin probability for 2 protons and 2 neutrons in the f7/2 shell follows 0.6428 + 0.0102 I(I+1), ranging from 0.6428 (I=0) to 0.9081 (I=12).
- The constants are A = 0.4286, B = 0.0102, A' = 0.6428, B' = 0.0034.

## Context

This work extends the author's earlier studies of spin content in nuclear wave functions within a single-j shell model space, using the MBZE interaction from a prior calculation (Zamick, 2016). The key finding that the spin expectation per pair is independent of wave-function details for a given total angular momentum I provides a baseline for distinguishing genuine S=1 correlations from trivial geometric effects in a j = L+1/2 shell. The result enables future studies to compare realistic nuclear wave functions against this correlation-independent benchmark when discussing S=0 J=0 pairing versus S=1 T=0 pairing.
