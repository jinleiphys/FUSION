# Constraining the nuclear matter equation of state around twice saturation density

> Using FOPI data on elliptic flow in Au+Au collisions between 0.4 and 1.5A GeV we extract constraints for the equation of state (EOS) of compressed symmetric nuclear matter using the transport code IQMD by introducing an observable describing the evolution of the size of the elliptic flow as a function of rapidity. This observable is sensitive to the nuclear EOS and a robust tool to constrain the compressibility of nuclear matter up to 2 $ρ_0$.

## Key claim

Using FOPI data on elliptic flow in Au+Au collisions between 0.4 and 1.5A GeV, the authors extract constraints for the nuclear matter equation of state (EOS) by introducing an observable that combines flow information across the full rapidity range, finding that a soft EOS is preferred over a stiff one, but the extracted incompressibility carries a significant uncertainty of ±30 MeV.

## Method

The authors analyzed elliptic flow data from the FOPI Collaboration for Au+Au collisions at beam energies from 0.4A to 1.5A GeV, using protons, deuterons, tritons, and ³He particles. They introduced a new observable, v₂ₙ = |v₂₀| + |v₂₂|, obtained by fitting the rapidity dependence of elliptic flow with a quadratic function v₂(y₀) = v₂₀ + v₂₂·y₀² over the scaled rapidity range |y₀| < 0.8, and compared the experimental results to simulations using the Isospin Quantum Molecular Dynamics (IQMD) transport code with four Skyrme-type parameterizations characterized by incompressibilities K₀ = 200 MeV (soft) and K₀ = 380 MeV (stiff).

## Key numbers

- Weighted average incompressibility extracted: K₀ = 190 ± 30 MeV from combined analysis of four particle species (protons, deuterons, tritons, ³He)
- Ratio between stiff (HM) and soft (SM) predictions for v₂ₙ: 1.63 ± 0.06, averaged across particle types
- For protons: v₂ₙ(SM) = 0.1658 ± 0.0030, v₂ₙ(HM) = 0.2609 ± 0.0027, v₂ₙ(FOPI) = 0.1774 ± 0.0028, yielding ΔK = +22 ± 8 MeV relative to SM
- For deuterons: v₂ₙ(SM) = 0.3676 ± 0.0080, v₂ₙ(HM) = 0.6274 ± 0.0087, v₂ₙ(FOPI) = 0.3237 ± 0.0029, yielding ΔK = -30 ± 8 MeV relative to SM
- For tritons: v₂ₙ(SM) = 0.5740 ± 0.0214, v₂ₙ(HM) = 0.9161 ± 0.0252, v₂ₙ(FOPI) = 0.5223 ± 0.0048, yielding ΔK = -27 ± 17 MeV relative to SM
- For ³He: v₂ₙ(SM) = 0.5540 ± 0.0217, v₂ₙ(HM) = 0.9048 ± 0.0265, v₂ₙ(FOPI) = 0.4537 ± 0.0050, yielding ΔK = -52 ± 18 MeV relative to SM
- Beam energy range investigated: 0.4A to 1.5A GeV
- Passing times for Au+Au range from 29.6 fm/c at 0.4A GeV to 15.3 fm/c at 1.5A GeV

## Context

This work extends earlier attempts to constrain the nuclear EOS from heavy-ion flow data, particularly the study by Danielewicz (2002) which found that elliptic flow data seemed to require K₀ = 300 MeV while directed flow favored lower values. The authors improve upon previous FOPI analyses (Andronic, 2005) by using not only protons but also light clusters (deuterons, tritons, ³He) which show larger flow signals, and by incorporating information from 80% of the target-projectile rapidity gap into a single observable. The results are consistent with the kaon production method (Fuchs, Hartnack, 2001) which favored a soft EOS, and the extracted K₀ = 190 ± 30 MeV agrees within uncertainties with the giant monopole resonance value of K₀ = 230 ± 40 MeV from Khan (2013), though the authors note the heavy-ion data probe densities around twice saturation density rather than the sub-saturation densities relevant for GMR.
