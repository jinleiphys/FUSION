# State-Specific Configuration Interaction for Excited States

> We introduce and benchmark a systematically improvable route for excited-state calculations, state-specific configuration interaction ($Δ$CI), \alert{which is a particular realization of multiconfigurational self-consistent field and multireference configuration interaction.} Starting with a reference built from optimized configuration state functions, separate CI calculations are performed for each targeted state (hence state-specific orbitals and determinants). Accounting for single and double excitations produces the $Δ$CISD model, which can be improved with second-order Epstein-Nesbet perturbation theory ($Δ$CISD+EN2) or a posteriori Davidson corrections ($Δ$CISD+Q). These models were gauged against a vast and diverse set of 294 reference excitation energies. We have found that $Δ$CI is significantly more accurate than standard ground-state-based CI, whereas close performances were found between $Δ$CISD and EOM-CC2, and between $Δ$CISD+EN2 and EOM-CCSD. For larger systems, $Δ$CISD+Q delivers more accurate results than EOM-CC2 and EOM-CCSD. The $Δ$CI route can handle challenging multireference problems, singly- and doubly-excited states, from closed- and open-shell species, with overall comparable accuracy, and thus represents a promising alternative to more established methodologies. In its current form, however, it is only reliable for relatively low-lying excited states.

## Key claim

The paper introduces state-specific configuration interaction (ΔCI) as a systematically improvable route for excited-state calculations that is a particular realization of multiconfigurational self-consistent field and multireference configuration interaction; however, in its current form it is only reliable for relatively low-lying excited states.

## Method

The authors first optimize orbitals at the MCSCF level using a minimal set of configuration state functions (CSFs) for each targeted state, producing state-specific reference wave functions (ΔCSF). Separate CI calculations are then performed for each state using single and double excitations (ΔCISD), which can be improved with second-order Epstein-Nesbet perturbation theory (ΔCISD+EN2) or a posteriori Davidson corrections (ΔCISD+Q). The methods were benchmarked against 294 reference excitation energies from the QUEST database, using aug-cc-pVDZ and 6-31+G(d) basis sets.

## Key numbers

- ΔCISD achieves a mean absolute error (MAE) of 0.18 eV for 237 singly-excited states of closed-shell systems, comparable to CC2 (0.17 eV)
- ΔCISD+EN2 achieves an MAE of 0.08 eV for the same set of 237 singly-excited states, matching CCSD (0.08 eV)
- ΔCISD+PC (Pople-corrected Davidson) achieves an MAE of 0.10 eV for singly-excited states of closed-shell systems
- For 10 doubly-excited states, ΔCISD achieves an MAE of 0.16 eV, significantly more accurate than CCSDT
- For 47 excited states of open-shell doublets, ΔCISD+EN2 achieves an MAE of 0.07 eV
- For the full set of 294 excitation energies, ΔCISD+EN2 achieves an MAE of 0.08 eV and RMSE of 0.12 eV
- ΔCSF (mean-field level) achieves an MAE of 0.62 eV for singly-excited states, comparable to CIS (0.65 eV)

## Context

This work extends the state-specific approach of ΔSCF (Ziegler 1977) and excited-state mean field theory (Shea 2018) by using a compact CSF reference followed by CISD rather than MP2 correlation treatment. The method offers a balanced description of singly- and doubly-excited states, which most standard excited-state methodologies fail to achieve without higher-order excitations (Loos 2019). The approach enables calculations on larger systems than typical MRCI because of the minimal reference space, and the authors plan to extend it to higher-lying excited states by including closed-shell determinants in the reference.
