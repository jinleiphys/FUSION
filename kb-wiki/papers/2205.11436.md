# Pion damping width and pion spectral function in hot pion gas

> The temperature behaviour of the pion width in the hadronic phase is investigated in the framework of the NJL model. The contribution to the width from the pion-pion collision is considered with a scalar sigma-meson as an intermediate state. It is shown that the pion width significantly broadens at $T>0.1$ GeV. Using the two-step iteration method, suggested by Kadanoff and Baym, the pion spectral function in a hot pion gas is calculated at different temperatures.

## Key claim
The pion damping width in hot pion gas broadens significantly above T ≈ 0.1 GeV, but only when the σ-meson exchange diagram is included in the ππ-scattering amplitude; without this contribution, the width does not broaden in that temperature region. The width reaches a maximum near T ≈ 0.145 GeV and then decreases before rising again near the critical temperature.

## Method
The authors use the SU(2) Nambu–Jona-Lasinio (NJL) model to compute the ππ-scattering amplitude, truncating the full amplitude by replacing box diagrams with a constant 4-pion interaction and using a pole approximation for the σ-meson propagator. The pion damping width is then calculated via the Kadanoff–Baym approach, integrating the scattering amplitude over phase space with thermal occupation numbers. The pion spectral function is obtained using a two-step iteration method starting from a Breit–Wigner form.

## Key numbers
- The pion width reaches a maximal value Γ ∼ 0.075 GeV at T ∼ 0.145 GeV.
- The critical temperature for chiral phase transition in the model is T = 0.192 GeV.
- The Mott temperature for the pion is T^π_Mott = 0.22 GeV.
- The σ-meson decay σ → ππ is possible up to T^σ_diss = 0.189 GeV.
- Model parameters: cutoff Λ = 0.639 GeV, current quark mass m₀ = 5.5 MeV, coupling constant G = 5.227 GeV⁻².
- The ρππ coupling constant used is g_{ρππ} = 5.14.

## Context
This work extends earlier predictions of finite pion damping width in dense matter by Blaschke et al. (1995) and studies of in-medium ρ-meson behavior by van Hees and Rapp (2000). It uses the NJL model framework previously applied to in-medium meson properties by Blaschke et al. (2003) and Yudichev and Kalinovsky (2005). The approach is limited to temperatures below the critical temperature T_c ∼ 0.19 GeV; for higher temperatures the authors note that the quark-antiquark phase shift formalism developed by Xia et al. (2014) and Blaschke et al. (2013, 2019) would be more appropriate.
