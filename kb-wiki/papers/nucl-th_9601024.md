# Neural Networks for Impact Parameter Determination

> An accurate impact parameter determination in a heavy ion collision is crucial for almost all further analysis. The capabilities of an artificial neural network are investigated to that respect. A novel input generation for the network is proposed, namely the transverse and longitudinal momentum distribution of all outgoing (or actually detectable) particles. The neural network approach yields an improvement in performance of a factor of two as compared to classical techniques. To achieve this improvement simple network architectures and a 5 by 5 input grid in (p_t,p_z) space are sufficient.

## Key claim

A neural network using a discretized grid of transverse and longitudinal momentum distributions as input can improve impact parameter determination by a factor of two compared to classical techniques, but only when the input grid dimension is at least 5 by 5; the improvement is achieved without hidden units, meaning the task is essentially linear.

## Method

Events from the Isospin Quantum Molecular Dynamics (IQMD) model for Au+Au collisions at 1 GeV/nucleon were used to generate training, validation, and test sets. The final-state momentum distributions of all baryons were discretized into two-dimensional grids of varying size (3x3 to 20x20) in (p_t, p_z) space, and these binned hit counts served as input to a feed-forward two-layer perceptron trained by error backpropagation. The network's performance was evaluated both for full 4π acceptance and for events filtered through the FOPI detector acceptance.

## Key numbers

- The neural network achieved an average absolute error Δb = 0.25 fm for unfiltered events and Δb = 0.35 fm for events processed with the FOPI detector filter, using a 10x10 input grid with 10 hidden units.
- Increasing the input grid dimension from 2 to 5 improved performance by a factor of up to two.
- The detector filter diminished network performance by approximately 30% compared to the unfiltered case.
- For input grid dimensions larger than 5, the number of hidden units was irrelevant; a simple perceptron without hidden units sufficed.
- Classical techniques achieve an accuracy estimated at best ±1 to ±1.5 fm.

## Context

This work extends earlier neural network approaches by Bass (1994) and David and Aichelin (1995) that used preprocessed observables (multiplicity, transverse energy ratio, directivity) as input, which limited the network to correlations already known. By using raw momentum distributions with minimal preprocessing, the authors demonstrate that all useful correlations for impact parameter determination are contained in a 5x5 grid of (p_t, p_z) space. The method enables application to experimental data from the FOPI detector at GSI, though the authors note that training on transport model data introduces a model bias whose magnitude can be estimated by comparing results from different transport models.
