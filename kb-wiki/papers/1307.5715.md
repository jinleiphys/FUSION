# CompOSE - CompStar Online Supernovae Equations of State

> This paper is the manual of the CompOSE service, see http://compose.obspm.fr, an equation of state (EoS) database for use in nuclear physics and in astrophysical applications, e.g. the simulation of core-collapse supernovae and compact stars. The main aims of the service are the following: to offer a robust, easily extendable data format to store and read EoS data tables and related physical quantities; to make these data available to the public -- respecting the original work by the contributors -- in a standard format designed for easy access and flexibility; and to provide routines to interpolate these data in the parameter space of density, temperature and hadronic charge fraction. The manual contains a detailed description of the service. It is divided into three parts, containing a general introduction as well as instructions for potential contributors and for users.

## Key claim

CompOSE is a repository and interpolation service for equation of state (EoS) tables in a common format, designed for use in astrophysical simulations and nuclear physics, but it cannot offer an online service for all features (e.g., running all codes online) due to limitations in storage and computation times and to avoid unphysical input parameters.

## Method

The service provides a standardized data format for storing EoS tables with thermodynamic properties, chemical composition, and microphysical quantities, along with interpolation routines to generate user-specified grids in the parameter space of temperature, baryon number density, and hadronic charge fraction. The manual defines conventions for natural units (hbar = c = k_B = 1, energies in MeV, lengths in fm), particle indexing schemes, and thermodynamic potentials, and it specifies that the state of the system is uniquely characterized by temperature T, baryon number density n_b, and charge fraction Y_q, with the assumption that strangeness chemical potential vanishes and that neutrinos are not included.

## Key numbers

- Recommended values for physical constants from the 2010 CODATA evaluation are provided, including: Planck's constant hbar = 1.054571726 x 10^{-34} J s (or 6.58211928 x 10^{-22} MeV s, or 197.3269718 MeV fm c^{-1}); Boltzmann's constant k_B = 1.3806488 x 10^{-23} J K^{-1} (or 8.6173324 x 10^{-11} MeV K^{-1}); neutron mass m_n = 939.565379 MeV c^{-2}; proton mass m_p = 938.272046 MeV c^{-2}; electron mass m_e = 0.510998928 MeV c^{-2}; muon mass m_mu = 105.6583715 MeV c^{-2}.
- Particle indices are assigned numerically, e.g., neutron index 10, proton index 11, electron index 0, muon index 1, and nuclei indices are given by 1000*A + Z.
- The fine structure constant alpha = 1/137.035999074.

## Context

This manual describes the CompOSE service (http://compose.obspm.fr), which extends prior work on EoS databases by providing a common format and interpolation tools for use in core-collapse supernova and neutron-star merger simulations. It enables users to test multiple EoS models (e.g., hadronic, quark, or hybrid models) and allows contributors to make their EoS available in a standardized way, with future extensions planned for additional models and features. The service builds on the need for robust, easily extendable data formats as identified by the CompStar research networking program.
