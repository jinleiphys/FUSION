# A chiral lagrangian with broken scale: a Fortran code with the thermal contributions of the dilaton field

> The Chiral Dilaton Model is a chiral lagrangian in which the breaking of scale invariance is regulated by the expectation value of a scalar field, called dilaton. Here we provide a Fortran code \cite{epaps,ind}, as a tool to make calculations within the Chiral Dilaton Model at finite density and temperature. The calculations are improved respect to previous works [arXiv:0704.3707,arXiv:0805.4188] by including the thermal contributions of the dilaton field.

# A chiral lagrangian with broken scale: a Fortran code with the thermal contributions of the dilaton field

## Key claim

The thermal fluctuations of the dilaton field are included in calculations within the Chiral Dilaton Model at finite density and temperature, improving upon previous work (Bonanno 2007, Bonanno 2008); however, the results do not appear very different from those earlier calculations because the dilaton mass remains large (about 1 GeV) even in the density-temperature region where scale invariance restores.

## Method

The author provides a Fortran code that solves a system of 15 coupled equations at finite baryon density and temperature within the Chiral Dilaton Model, a chiral hadronic Lagrangian containing chiral fields (sigma and pions), a dilaton field, and vector mesons (omega and rho). The thermal averages of fluctuating scalar fields (dilaton, sigma, pions) are computed using the Gaussian weighting technique of Mocsy and Mishustin (2004), which treats each isospin component independently and avoids the truncation approximations used in earlier work (Carter 1996, Carter 1997). The code uses cubic spline interpolation on precomputed data to ensure fast execution.

## Key numbers

- Baryon density range: 0.01 rho_0 to 9.01 rho_0, where rho_0 = 0.15 fm^{-3}
- Temperature range: 0 to 245 MeV
- Isospin asymmetry range: 0.3 <= Z/A <= 0.5
- Dilaton mass in vacuum: about 1.6 GeV
- Dilaton mass in the density-temperature region of interest: about 1 GeV
- The code computes neutron star matter at T=0 with electrons and muons in beta-equilibrium, and T=0 pure neutron matter

## Context

This work extends the Chiral Dilaton Model calculations of Bonanno (2007, 2008) by including thermal fluctuations of the dilaton field, which were previously omitted due to the large vacuum glueball mass. The code enables systematic study of chiral symmetry restoration and scale invariance restoration across a wide region of the density-temperature plane, including the interplay between these two phase transitions. The technique for thermal averaging follows Mocsy and Mishustin (2004), which is more reliable than the earlier approach of Carter (1996, 1997) because it does not truncate expansions. Future extensions could include hyperons via an SU(3) extension (as explored by the Frankfurt group, Dexheimer 2008) or quark degrees of freedom.
