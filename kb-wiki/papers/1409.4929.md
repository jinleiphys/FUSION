# Deformed shell model results for neutrinoless double beta decay of nuclei in A=60-90 region

> Nuclear transition matrix elements (NTME) for the neutrinoless double beta decay of $^{70}$Zn, $^{80}$Se and $^{82}$Se nuclei are calculated within the framework of the deformed shell model based on Hartree-Fock states. For $^{70}$Zn, jj44b interaction in $^{2}p_{3/2}$, $^{1}f_{5/2}$, $^{2}p_{1/2}$ and $^{1}g_{9/2}$ space with $^{56}$Ni as the core is employed. However, for $^{80}$Se and $^{82}$Se nuclei, a modified Kuo interaction with the above core and model space are employed. Most of our calculations in this region were performed with this effective interaction. However, jj44b interaction has been found to be better for $^{70}$Zn. The above model space was used in many recent shell model and interacting boson model calculations for nuclei in this region. After ensuring that DSM gives good description of the spectroscopic properties of low-lying levels in these three nuclei considered, the NTME are calculated. The deduced half-lives with these NTME, assuming neutrino mass is 1 eV, are $1.1 \times 10^{26}$ yr, $2.3 \times 10^{27}$ yr and $2.2 \times 10^{24}$ yr for $^{70}$Zn, $^{80}$Se and $^{82}$Se, respectively.

## Key claim
The deformed shell model (DSM) yields nuclear transition matrix elements for neutrinoless double beta decay of 70Zn, 80Se, and 82Se that are renormalized by a factor of 3 to match spherical shell model results, with the authors noting that the model "under estimates the spherical shell model calculated nuclear matrix elements by a factor of ~3" due to limited configuration space and different effective interactions.

## Method
The authors performed axially symmetric Hartree-Fock calculations for each parent and daughter nucleus using either the jj44b effective interaction (for 70Zn) or a modified Kuo interaction (for 80Se and 82Se) within a model space of the 2p3/2, 1f5/2, 2p1/2, and 1g9/2 orbitals with 56Ni as the core. They generated excited intrinsic states via particle-hole excitations, projected good angular momentum states from each intrinsic configuration, and performed band mixing calculations to obtain orthonormalized wave functions. The neutrinoless double beta decay nuclear transition matrix elements were then calculated using the closure approximation with a neutrino potential, incorporating short-range correlations via a Jastrow function and a renormalization constant C0 = 3.

## Key numbers
- For 70Zn: nuclear transition matrix element M0nu = 1.99 (GT contribution 1.748, Fermi contribution -0.379); deduced half-life assuming neutrino mass of 1 eV is 1.1 x 10^26 yr.
- For 80Se: nuclear transition matrix element M0nu = 3.19 (GT contribution 2.684, Fermi contribution -0.801); deduced half-life assuming neutrino mass of 1 eV is 2.3 x 10^27 yr.
- For 82Se: nuclear transition matrix element M0nu = 2.04 (GT contribution 1.853, Fermi contribution -0.296); deduced half-life assuming neutrino mass of 1 eV is 2.2 x 10^24 yr.
- For 70Zn, the 2nu double beta decay half-life calculated with DSM is 3.39 x 10^23 yr, with nuclear matrix element M2nu = 0.19 MeV^-1.
- For 80Se, the 2nu double beta decay half-life is 1.97 x 10^29 yr, with M2nu = 0.40 MeV^-1.
- For 82Se, the 2nu double beta decay half-life is 1.58 x 10^19 yr (compared to experimental value (0.92 +/- 0.07) x 10^20 yr), with M2nu = 0.24 MeV^-1.

## Context
This work extends the deformed shell model (DSM) calculations previously applied to 2nu and 0nu positron double beta decay in 64Zn, 74Se, 78Kr, and 84Sr (Sahu and Kota, 2014), and to 0nu double beta decay of 76Ge (Sahu and Kota, 2014). The results for 82Se are relevant for the upcoming SuperNEMO experiment, which follows the NEMO-3 lower limit of 3.2 x 10^23 yr at 90% C.L. The DSM results for 82Se are compared with shell model calculations by Caurier et al. (2012) and other models including QRPA (Suhonen, various years) and IBM-2 (Barea and Iachello, 2009). The work enables comparison of nuclear matrix elements across different theoretical frameworks for these candidate nuclei in the A=60-90 region.
