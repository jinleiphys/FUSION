# HFODD (v2.08k): User's Guide

> We describe the input data and installation procedures of the code HFODD (v2.08k). The present write-up contains complete and comprehensive information that has originally been given in five independent publications. It is enhanced by the subject index and indexes of variables, input-data keywords, subroutines, and files that are used in this user guide.

## Key claim

This paper provides a comprehensive user guide for the HFODD code (version 2.08k), consolidating input data specifications and installation procedures that were previously distributed across five separate publications. The guide serves as a reference manual rather than presenting new physical results, with the code itself being the primary deliverable.

## Method

The authors compiled and unified the documentation from five earlier publications into a single coherent user guide, adding a subject index, variable index, keyword index, subroutine index, and file index. The document describes the input parameters, keyword lines, data sets, and execution procedures for performing Hartree-Fock (HF) and Hartree-Fock-Bogoliubov (HFB) calculations with the HFODD code, which solves the nuclear mean-field problem using a deformed harmonic oscillator basis.

## Key numbers

- The code version is v2.08k, as stated in the title.
- The guide references example input files for specific nuclei: dy152-a.dat through dy152-e.dat (dysprosium-152), la132-a.dat and la132-b.dat (lanthanum-132), pb128-a.dat and pb208-a.dat (lead-128 and lead-208), sn120-a.dat (tin-120), and tb151-a.dat through tb151-d.dat (terbium-151).
- The index entries reference page numbers 7 through 56 for various topics, with specific page ranges such as 50-53 for example data files and 47-49 for memory and CPU time considerations.
- The keyword EPSITE (iteration convergence criterion) is referenced on pages 9 and 51, with a default value implied but not explicitly stated in the provided text.
- The keyword NOITER (number of iterations) is referenced on pages 9, 10, and 51.

## Context

This user guide consolidates documentation for the HFODD code, which is used for nuclear structure calculations involving mean-field methods with Skyrme forces and pairing correlations. The code extends earlier work on nuclear mean-field calculations by Dobaczewski and Dudek, and the guide enables users to perform calculations for superdeformed states, tilted-axis cranking, and odd and odd-odd nuclei. The comprehensive indexing system allows researchers to navigate the extensive parameter space of the code, which includes options for various symmetries (parity, signature, simplex, time-reversal), pairing treatments (BCS and HFB), and constraint types (multipole moments, spin, angular frequency).
