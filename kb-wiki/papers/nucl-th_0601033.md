# Relating Meson and Baryon Fragmentation Functions by Shower-Parton Recombination

> We relate the fragmentation functions of partons into mesons and baryons in the framework of recombination of shower partons. The results are in reasonable agreement with the data. The implication is that the meson and baryon fragmentation functions are not independent when hadronization of the shower partons are taken into account. The conclusion therefore closes a conceptual gap in the system of fragmentation functions whose $Q^2$ evolution has been more extensively studied than their interrelationship.

## Key claim

The paper shows that meson and baryon fragmentation functions are not independent but can be related through the recombination of shower partons, using shower parton distributions (SPDs) previously determined from fitting meson fragmentation functions to calculate baryon fragmentation functions without additional free parameters.

## Method

The authors first determine SPDs for light quarks in a gluon jet by fitting the known gluon-to-pion fragmentation function (FF) from BKK (2000) using a recombination formula. They then apply the same recombination formalism to compute the gluon-to-proton FF by generalizing the two-parton recombination to three-parton recombination, using the same SPDs. The resulting proton FF is corrected for baryon-antibaryon pair production by imposing momentum conservation constraints, and the same procedure is applied to compute the gluon-to-Lambda and u-quark-to-proton FFs.

## Key numbers

- For the gluon-to-pion SPD parameterization at Q² = 100 GeV²: A = 0.811, a = -0.056, b = 2.547, c = -0.176, d = 1.2
- For the proton recombination function: α = 1.75, β = 1.05
- For the Lambda recombination function: α = 1, β = 2, with g_st = 1/4
- For the strange quark SPD in a gluon jet at Q² = 100 GeV²: A = 0.069, a = -0.425, b = 2.489, c = -0.5, d = 1.1
- The calculated gluon-to-proton FF agrees with the KKP (2001) parametrization over 5 orders of magnitude in x

## Context

This work extends the recombination model previously applied to hadron production at intermediate p_T in heavy-ion collisions (Hwa and Yang, 2004) by using SPDs determined from meson FFs to calculate baryon FFs. It contradicts the implicit assumption in prior parametrizations by KKP (2001) and others that meson and baryon FFs are independent and must be fitted separately. The results enable future studies of baryon correlations at intermediate p_T in heavy-ion collisions at RHIC, where the recombination model can address hadronization from both hard and soft parton sources.
