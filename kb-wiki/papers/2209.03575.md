# Deuteron Structure and Form Factors: Using Inverse Potentials for S-waves

> In this paper, we determine deuteron's static properties, low energy scattering parameters, total cross-section and form factors from inverse S-wave potentials constructed using Morse function. The scattering phase shifts (SPS) at different lab energies are determined using phase function method. The model parameters are optimised using both machine learning algorithm and traditional data analysis by choosing mean squared error as cost function. The mean absolute error between experimental and obtained SPS for states 3S1 and 1S0 are found to be 0.35 and 0.70 respectively. The low energy scattering parameters are matching well with expected values. The contribution due to S-waves SPS towards total cross-section at various energies have been obtained and are matching well with experimental values. The analytical ground state deuteron wave-function (DWF) is obtained by utilizing the experimental value for Quadrupole moment. Other static properties and form factors determined from obtained DWF are found to be in close agreement with experimental ones.

## Key claim

The authors construct inverse S-wave potentials for the neutron-proton system using a Morse function and demonstrate that these potentials, optimized via both machine learning and traditional data analysis, can reproduce experimental scattering phase shifts, low-energy scattering parameters, total cross-sections, and deuteron static properties and form factors, with the caveat that the deuteron has only one bound state, making it impossible to uniquely fix all three Morse parameters without fitting to experimental phase shift data.

## Method

The authors use the Morse function as a reference potential and employ the phase function method (PFM) to compute scattering phase shifts (SPS) for the triplet (3S1) and singlet (1S0) S-wave channels. Model parameters (V0, rm, am) are optimized by minimizing the mean squared error between calculated and experimental SPS from the Granada MEPWAD database, using both a machine learning algorithm (fitting all data points simultaneously) and a traditional data analysis (TDA) approach that considers all possible combinations of experimental points. For the triplet state, the binding energy constraint fixes V0 in terms of am, leaving two free parameters; for the singlet state, all three parameters are varied. The analytical ground state deuteron wavefunction is obtained from the Morse potential solution, with the D-state component determined by requiring normalization and reproducing the experimental quadrupole moment.

## Key numbers

- Mean absolute error between experimental and obtained SPS: 0.35 for 3S1 state, 0.70 for 1S0 state
- Optimized MLA parameters for 3S1: [V0 = 114.153 MeV, rm = 0.841 fm, am = 0.350 fm]
- Optimized MLA parameters for 1S0: [V0 = 70.439 MeV, rm = 0.901 fm, am = 0.372 fm]
- Scattering length for 3S1: 5.35(1) fm (experimental: 5.424(3) fm)
- Effective range for 3S1: 1.75(2) fm (experimental: 1.760(5) fm)
- Scattering length for 1S0: -23.37(8) fm (experimental: -23.749(8) fm)
- Effective range for 1S0: 2.42(3) fm (experimental: 2.81(5) fm)
- Deuteron binding energy used: -2.224589(22) MeV
- Calculated total cross-section at E = 0.132 KeV: 19.171 +/- 1.280 barn (experimental: 20.491 +/- 0.014 barn)
- S-state probability: 98%, D-state probability: 2%
- Quadrupole moment used: 0.2589 fm^2
- Form factor values at Q -> 0: FC = 1.0205 (experimental: 1), FM = 1.7714 (experimental: 1.7148), FQ = 24.9724 - 27.5613 (experimental: 25.83)

## Context

This work extends previous efforts by the same authors (Khachi et al., 2021) to construct inverse S-wave potentials using the Morse function, and builds upon the reference potential approach detailed by Selg (2002, 2003). The results are compared against precision NN potentials such as Argonne V18 and Nijmegen potentials, as well as more recent N3LO calculations and the supersymmetric quantum mechanics approach of an effective deuteron potential. The analytical wavefunctions obtained enable calculation of electromagnetic form factors comparable to experimental data from Abbott, Nikolenko, Garcon, and others, though the authors note deviations at higher momentum transfer (above 3-4 fm^-1), suggesting the need for inclusion of higher partial waves or relativistic corrections.
