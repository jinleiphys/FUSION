# Generalized Effective String Rope Model for the initial stages of Ultra-Relativistic Heavy Ion Collisions

> We present a Generalized Effective String Rope Model (GESRM) for the description of the initial state of relativistic heavy ion collisions. We start from the Effective String Rope Model (ESRM) and take into account fluctuations in the initial state following the Glauber Monte Carlo approach. Results from symmetric Au+Au collisions at different impact parameters and asymmetric A+Au head on collisions are presented at RHIC energies. The produced initial state is used as an initial condition for further hydrodynamical calculations.

## Key claim

The Generalized Effective String Rope Model (GESRM) extends the Effective String Rope Model (ESRM) by incorporating event-by-event fluctuations in the initial nucleon positions via a Glauber Monte Carlo approach, while retaining the ESRM's treatment of streak-streak collisions and chromo-electric field dynamics; the authors state that the model aims to provide an initial state for further hydrodynamical calculations, but the results presented are limited to the initial state itself without coupling to a hydrodynamic evolution.

## Method

The authors start with the ESRM, which treats the collision as independent streak-streak collisions in the transverse plane, where each streak evolves under a uniform chromo-electric field (string rope) with string tension proportional to the product of baryon charges in colliding streaks. They implement fluctuations by randomly distributing nucleons within each nucleus according to a Woods-Saxon distribution (for Au) or a 2-parameter Fermi distribution (for Pb), using a Monte Carlo approach with 1000 fictitious particles per nucleon to compute baryon charge in each transverse cell. Effective streak lengths are then defined from the baryon charge in each cell using Eq. ## Key numbers

- String tension values in the ESRM are typically 6-15 GeV/fm for initial energy per nucleon ε₀ = 100 GeV (i.e., √s_NN = 200 GeV), and an order of magnitude higher for ε₀ = 1.38 TeV (LHC energies).
- The dimensionless parameter A in the string tension formula (Eq. 2) takes typical values around 0.06-0.08 (for σ measured in GeV/fm).
- For Au+Au collisions at √s_NN = 200 GeV, the nuclear radius parameter R_Au = 6.38 fm and skin depth a_Au = 0.535 fm are used for the Woods-Saxon distribution.
- The nucleon radius is taken as R_N = 0.842 fm.
- The transverse grid cell size is Δx = Δy = R_Au/10 = 0.638 fm.
- The final time for presenting the initial state is t_fin = 5 fm.
- Results are shown for impact parameter b_0 = 0.5, meaning b = 0.5(R_Au + R_Au) = R_Au = 6.38 fm.

## Context

This work extends the Effective String Rope Model (ESRM) developed by Magas, Csernai, and Strottman (2002-2003) by adding event-by-event fluctuations following the Glauber Monte Carlo approach, which is widely used in other modern initial-state models such as those by Pang et al. (2016) and Karpenko et al. (2015). The GESRM enables event-by-event analysis of initial-state fluctuations and their effect on flow coefficients, vorticity, and other observables, and is intended to provide initial conditions for (3+1)D relativistic hydrodynamic calculations, similar to how the original ESRM was used in studies of Λ hyperon polarization by Csernai et al. (2013) and Becattini et al. (2013).
