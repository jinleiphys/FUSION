# nCTEQ15HIX -- Extending nPDF Analyses into the High-$x$, Low $Q^2$ Region

> We use the nCTEQ analysis framework to investigate nuclear Parton Distribution Functions (nPDFs) in the region of large x and intermediate-to-low $Q$, with special attention to recent JLab Deep Inelastic Scattering data on nuclear targets. This data lies in a region which is often excluded by $W$ and $Q$ cuts in global nPDF analyses. As we relax these cuts, we enter a new kinematic region, which requires new phenomenology. In particular, we study the impact of i) target mass corrections, ii) higher twist corrections, iii) deuteron corrections, and iv) the shape of the nuclear PDF parametrization at large-$x$ close to one. Using the above tools, we produce a new nPDF set (named nCTEQ15HIX) which yields a good description of the new JLab data in this challenging kinematic region, and displays reduced uncertainties at large $x$, in particular for up and down quark flavors.

## Key claim
The nCTEQ15HIX analysis demonstrates that by relaxing kinematic cuts from Q>2 GeV and W>3.5 GeV to Q>1.3 GeV and W>1.7 GeV, and incorporating target mass corrections, higher twist corrections, deuteron corrections, and large-x nuclear PDF parametrization, one can successfully describe recent JLab deep inelastic scattering data on nuclear targets in the high-x, low-Q² region, provided that the higher twist corrections show no significant nuclear A-dependence (τ∼0±0.005).

## Method
The authors used the nCTEQ global analysis framework to perform nuclear PDF fits including DIS and Drell-Yan data from the original nCTEQ15 analysis plus new JLab CLAS and Hall C data on ³He, ⁴He, ⁹Be, ¹²C, ²⁷Al, ⁵⁶Fe, ⁶⁴Cu, ¹⁹⁷Au, and ²⁰⁸Pb targets spanning x from 0.2 to 0.95. They systematically incorporated target mass corrections via the Nachtmann variable formalism, phenomenological higher twist corrections following the CJ15 parameterization with an additional A^τ scaling term, deuteron structure function corrections using the CJ15 nuclear smearing model with AV18 wave function, and a 19-parameter nuclear PDF parametrization with A-dependent coefficients.

## Key numbers
- Kinematic cuts relaxed from Q=2 GeV and W=3.5 GeV (nCTEQ15) to Q=1.3 GeV and W=1.7 GeV (nCTEQ15HIX), increasing the number of data points from 708 to 1564
- The higher twist coefficient C_HT(x,A) = h_0 x^{h_1} (1+ h_2 x) A^τ with {h_0, h_1, h_2} = {-3.3 GeV², 1.9, -2.1}; the nuclear A-exponent τ was found to be indistinguishable from zero (τ∼0±0.005)
- At x=0.7, the higher twist correction factor is ∼1.5 for Q=1.3 GeV and ∼1.2 for Q=2 GeV
- The leading target mass correction ratio F₂^TMC/F₂^(leading TMC) shows up to ∼25% modification at intermediate-to-large x values, with nuclear A-dependence spread of up to ∼5% for Q=1.0 GeV decreasing to ∼1% for Q=1.3 GeV
- The deuteron-to-proton structure function ratio F₂^D/F₂^p dips to ∼70% at x∼0.7

## Context
This work extends the nCTEQ15 analysis (Kovarik et al., 2015) into the high-x, low-Q² kinematic region that was previously excluded by stringent W and Q cuts, building on the CJ15 deuteron correction model (Accardi et al., 2016) and the phenomenological higher twist parameterization from that same analysis. The new nCTEQ15HIX PDF set enables improved description of JLab CLAS data (Schmookler et al., 2019) and Hall C data (Seely et al., 2009) in the transition region between resonance production and deep inelastic scattering, and provides reduced uncertainties at large x for up and down quark flavors, which is relevant for neutrino scattering experiments and future EIC/LHeC analyses.
