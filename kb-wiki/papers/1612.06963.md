# On the Feynman-Hellmann Theorem in Quantum Field Theory and the Calculation of Matrix Elements

> The Feynman-Hellmann theorem can be derived from the long Euclidean-time limit of correlation functions determined with functional derivatives of the partition function. Using this insight, we fully develop an improved method for computing matrix elements of external currents utilizing only two-point correlation functions. Our method applies to matrix elements of any external bilinear current, including nonzero momentum transfer, flavor-changing, and two or more current insertion matrix elements. The ability to identify and control all the systematic uncertainties in the analysis of the correlation functions stems from the unique time dependence of the ground-state matrix elements and the fact that all excited states and contact terms are Euclidean-time dependent. We demonstrate the utility of our method with a calculation of the nucleon axial charge using gradient-flowed domain-wall valence quarks on the $N_f=2+1+1$ MILC highly improved staggered quark ensemble with lattice spacing and pion mass of approximately 0.15 fm and 310 MeV respectively. We show full control over excited-state systematics with the new method and obtain a value of $g_A = 1.213(26)$ with a quark-mass-dependent renormalization coefficient.

## Key claim

The Feynman-Hellmann theorem can be derived from the long Euclidean-time limit of correlation functions determined with functional derivatives of the partition function, and this insight enables an improved method for computing matrix elements of external currents using only two-point correlation functions, with the caveat that the method requires full control over excited-state systematics and contact terms which are Euclidean-time dependent.

## Method

The authors develop a method where the linear response of the effective mass to an external current is computed from the difference of two-point correlation functions evaluated at different times, using a Feynman-Hellmann propagator that sums over all current insertion times. They apply this method to a lattice QCD calculation of the nucleon axial charge using gradient-flowed domain-wall valence quarks on the $N_f=2+1+1$ MILC highly improved staggered quark ensemble, constructing both regular and Feynman-Hellmann propagators for each source. The analysis employs both frequentist and Bayesian constrained fit strategies with Gaussian priors on the two- and three-point correlation functions to extract the ground-state matrix element.

## Key numbers

- Lattice spacing: approximately 0.15 fm
- Pion mass: approximately 310 MeV
- Lattice volume: $16^3 \times 48$
- Number of configurations: 1960 with six sources per configuration
- Nucleon axial charge: $g_A = 1.213(26)$ with a quark-mass-dependent renormalization coefficient
- Ground-state energy from two-point analysis: approximately $E_0 \approx 0.82$ (in lattice units)
- Ground-state smeared overlap factor prior: $\tilde{z}_0^S = 0.0004(2)$
- Ground-state point overlap factor prior: $\tilde{z}_0^P = 0.010(5)$

## Context

This work extends the Feynman-Hellmann theorem approach previously used for scalar quark matrix elements in the nucleon (Procura 2003, 2006; Walker-Loud 2008) and for nucleon spin content (Chambers 2014, 2015), as well as the hybrid background field method introduced for few-nucleon electroweak matrix elements (Savage 2016). The method improves upon traditional three-point function calculations by using only two-point correlation functions, which have less severe stochastic noise and more tractable excited-state systematics. This enables future calculations of nucleon matrix elements relevant to dark matter detection and neutrinoless double beta decay, including disconnected diagram contributions that can be computed with selective time summation.
