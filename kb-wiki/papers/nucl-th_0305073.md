# Threshold model and the latest NA50 data on $J/ψ$ suppression in Pb+Pb collisions

> Using the QGP motivated threshold model, where all the $J/ψ$'s are suppressed above a threshold density, we have analyzed the latest version of the NA50 data on the centrality dependence of the $J/ψ$ over Drell-Yan ratio. The data are not well explain in the model, unless the threshold density is largely smeared. Large smeared threshold density effectively excludes creation of any deconfined medium in the collision.

## Key claim

The QGP-motivated threshold model, where all J/ψ mesons are suppressed above a critical transverse density, cannot adequately describe the latest NA50 data on J/ψ suppression in Pb+Pb collisions unless the threshold density is smeared considerably; this large smearing effectively excludes the formation of a deconfined medium in the collision.

## Method

The author applied the threshold model of Blaizot et al. (2000), which adds an anomalous suppression factor to conventional nuclear absorption, to the latest NA50 data on the centrality dependence of the J/ψ over Drell-Yan ratio in 158 GeV/c Pb+Pb collisions. The J/ψ-nucleon absorption cross section was fixed at 4.4 mb, consistent with new high-statistics pA data from NA50. The model parameters (threshold density n_c and smearing parameter λ) were obtained by fitting the NA50 data, using a normalizing factor derived from NA38 S+U data scaled by 1.051.

## Key numbers

- J/ψ-nucleon absorption cross section: σ^{J/ψ N}_{abs} = 4.4 ± 1.0 mb from 450 GeV pA data; common fit to pA and NA38 S+U data gives 4.4 ± 0.5 mb.
- Threshold density without smearing: n_c = 3.61 ± 0.06 fm^{-2} (best fit fails to describe intermediate E_T range).
- Best fit with smeared threshold: n_c = 3.82 ± 0.09 fm^{-2} and λ = 0.77 ± 0.14 fm^{2}.
- Over a density range of 2.4–5.2 fm^{-2}, the anomalous suppression factor changes from 0.9 to 0.1.
- Normalizing factor for S+U collisions: B_{μμ}σ^{J/ψ}_{NN}/σ^{DY}_{NN} = 39.65; scaled by 1.051 for Pb+Pb.

## Context

This work extends the threshold model analysis of Blaizot et al. (2000), which successfully described the earlier 1998 NA50 data with n_c = 3.7–3.75 fm^{-2} and σ^{J/ψ N}_{abs} = 6.4 mb, to the latest 2000 NA50 data with a revised suppression pattern and a smaller absorption cross section. The author had previously analyzed the same data in a QCD-based nuclear absorption model (Chaudhuri, 2003) without free parameters, while Capella and Sousa (2003) explained the data in a comover model with σ_{co} = 0.65 mb. The large smearing required in the threshold model suggests the suppression mechanism is gradual rather than abrupt, more consistent with a nuclear or comover environment than with sudden melting in a deconfined medium.
