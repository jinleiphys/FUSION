# First Extraction of the $ϕ$-$^{4}\mathrm{He}$ scattering length from near-threshold $ϕ$ photoproduction on helium-4

> We present a model-independent extraction of the phi-4He scattering length from near-threshold phi photoproduction on helium-4, based on LEPS Collaboration data for the coherent process gamma + 4He -> phi + 4He and the Vector Meson Dominance framework. Assuming an energy-independent differential cross section, we extract the absolute value of the phi-4He scattering length |alpha_{phi4He}| = (3.33 +- 0.06) x 10^{-4} fm from a fit at threshold t_thr. This value is orders of magnitude smaller than those for phi-N and phi-d scattering lengths, indicating an inverse dependence of |alpha_VA| on the target nucleus mass. Our results provide new insight into the phi-nucleus interaction, supporting the notion of weak phi-nucleus coupling. We further explore the dependence of |alpha_VA| on the vector meson mass, the target atom mass, and the threshold energy. An approximate exponential suppression of |alpha_VA| is observed with increasing vector meson mass or target atom mass, indicating that heavier vector mesons or heavier target nuclei exhibit weaker couplings in vector meson-nucleus interactions.

## Key claim

The paper extracts the first model-independent value of the phi-4He scattering length from near-threshold coherent phi photoproduction data, but the result relies on the assumption of an energy-independent differential cross section near threshold, which the authors acknowledge may be restrictive.

## Method

The authors fitted LEPS Collaboration differential cross-section data for the coherent process gamma + 4He -> phi + 4He at six incident photon energies (1.785, 1.935, 2.035, 2.135, 2.235, and 2.335 GeV) using an exponential function dsigma/dt = A exp(-bt). Using the Vector Meson Dominance (VMD) framework, they extrapolated the forward differential cross section to the production threshold t_thr and extracted the scattering length from the relation connecting the differential cross section at threshold to the scattering length squared.

## Key numbers

* Absolute value of the phi-4He scattering length: |alpha_{phi4He}| = (3.33 +/- 0.06) x 10^{-4} fm, obtained as the weighted average from six photon energies at threshold t_thr
* Individual scattering length values at E_gamma = 1.785 GeV: (2.28 x 10^{-3} +/- 7.88 x 10^{-5}) fm
* Individual scattering length values at E_gamma = 1.935 GeV: (1.39 x 10^{-4} +/- 6.39 x 10^{-6}) fm
* Individual scattering length values at E_gamma = 2.035 GeV: (3.76 x 10^{-3} +/- 7.99 x 10^{-5}) fm
* Individual scattering length values at E_gamma = 2.135 GeV: (1.05 x 10^{-2} +/- 1.16 x 10^{-4}) fm
* Individual scattering length values at E_gamma = 2.235 GeV: (3.42 x 10^{-3} +/- 5.21 x 10^{-5}) fm
* Individual scattering length values at E_gamma = 2.335 GeV: (5.54 x 10^{-3} +/- 5.11 x 10^{-5}) fm
* Photon-phi coupling constant used: gamma_phi = 6.71

## Context

This work extends previous extractions of vector meson-nucleon scattering lengths for omega, rho0, phi, J/psi, psi(2S), and Upsilon on proton targets by Strakovsky (2014, 2020) and Wang (2022), as well as vector meson-deuteron scattering lengths by Wang (2022, 2023). The extracted phi-4He scattering length is orders of magnitude smaller than phi-N and phi-d values, establishing a clear hierarchy |alpha_{phi N}| > |alpha_{phi d}| > |alpha_{phi 4He}|. The results enable future tests of universal scaling behavior in vector meson-nucleus interactions and motivate higher-statistics measurements at facilities such as Jefferson Lab, the Electron-Ion Collider (EIC), and the Electron-Ion Collider in China (EicC).
