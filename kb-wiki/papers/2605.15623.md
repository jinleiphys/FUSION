# Microscopic derivation of the interacting boson model parameters with machine learning

> Machine learning is applied to derive microscopically parameters of the interacting boson model for nuclear spectroscopy. A physics-guided neural network is proposed, which is trained to map the potential energy landscapes that are calculated within the nuclear density functional theory onto the bosonic parameter space. To incorporate the underlying nuclear structure information and mitigate parameter degeneracy, the network integrates a global quadrupole collectivity indicator and valence nucleon numbers as key input features. In its applications to rare-earth nuclei, by reproducing the microscopic energy landscapes without any manual parameter tuning, the trained network is shown to provide a set of the model parameters and energy spectra that reflect the nuclear structural evolution, offering a robust alternative microscopic description of nuclear collectivity.

# Microscopic derivation of the interacting boson model parameters with machine learning

## Key claim
The physics-guided neural network (PGNN) can map density-functional-theory potential energy curves onto interacting boson model (IBM-2) parameters for rare-earth nuclei without manual tuning, but the derived neutron shape parameter χ_ν does not exhibit the strong neutron-number dependence seen in earlier microscopic IBM-2 mappings, and the approach currently works only for axially symmetric configurations with a fixed deformation scale factor C_β = 4.0.

## Method
The authors train a physics-guided neural network with a branching architecture to map Gogny-D1S Hartree-Fock-Bogoliubov potential energy curves (PECs) for Nd, Sm, and Gd isotopes (neutron numbers N=86 to 100) onto the four parameters {ε, κ, χ_π, χ_ν} of the proton-neutron interacting boson model (IBM-2) Hamiltonian. The network takes as input the valence proton boson number n_π, valence neutron boson number n_ν, and the Casten P-factor; it uses separate proton, neutron, and interaction branches with sign constraints (ε>0, κ<0, χ_π<0, χ_ν<0) and a non-trainable physics layer that computes the bosonic PEC from the predicted parameters for loss calculation. Hyperparameters are optimized via Bayesian optimization (50 trials) using the mean absolute error of PECs for the validation set (N=90 nuclei).

## Key numbers
- Training data: 24 PECs for even-even ¹⁴⁶–¹⁶⁰Nd, ¹⁴⁸–¹⁶²Sm, and ¹⁵⁰–¹⁶⁴Gd isotopes
- Deformation scale factor: C_β = 4.0 (fixed for all nuclei)
- Validation set: N=90 isotopes from each chain (3 nuclei)
- Predicted energy levels in deformed nuclei (N≥90) overestimate experiment by a factor of ≈1.5–1.8 before rotational correction
- Rotational correction strength κ' ≈ −0.01 MeV for Sm isotopes
- R₄/₂ ratio: U(5) limit = 2.0, SU(3) limit = 10/3 (reference values from IBM literature)

## Context
This work extends the density-functional-theory to interacting-boson-model mapping developed by Nomura et al. (2008, 2010, 2011, 2012) by replacing the iterative per-nucleus fitting procedure with a neural network that learns the mapping across isotopic chains. It differs from purely empirical machine-learning approaches such as Ahmad (2026), which used Light Gradient Boosting Machine with IBM to study shape phase transitions from experimental data, by incorporating a differentiable physics layer that enforces consistency with microscopic DFT calculations. The method enables systematic parameter extraction without manual tuning and can be extended to triaxial degrees of freedom and broader mass regions where direct DFT spectroscopic predictions are challenging.
