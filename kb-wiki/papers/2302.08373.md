# Eigenvector continuation for the pairing Hamiltonian

> The development of emulators for the evaluation of many-body observables has gained increasing attention over the last years. In particular the framework of eigenvector continuation (EC) has been identified as a powerful tool when the Hamiltonian admits for a parametric dependence. By training the emulator on a set of training data the many-body solution for arbitrary parameter values can be robustly predicted in many cases. Furthermore, it can be used to resum perturbative expansions that otherwise diverge. In this work, we apply EC to the pairing Hamiltonian and show that EC-resummed perturbation theory is in qualitative agreement with the exact solution and that EC-based emulators robustly predict the ground-state energy once the training data are chosen appropriately. In particular the phase transition from the normal to the superfluid regime is quantitatively predicted using a very low number of training points.

## Key claim

Eigenvector continuation (EC) can robustly predict the ground-state energy of the pairing Hamiltonian across the normal-to-superfluid phase transition, but only if the training set includes points from both regimes; a training set confined to one regime fails to capture the dynamics of the other.

## Method

The authors applied eigenvector continuation to the exactly solvable pairing Hamiltonian with equidistant single-particle levels at half filling. They constructed EC emulators from exact Richardson solutions (or from truncated configuration-interaction wave functions) at selected training couplings, then solved a generalized eigenvalue problem to predict ground-state energies at arbitrary coupling values. For the resummation variant (EC-PT), they used first-order many-body perturbation theory state corrections from a Hartree-Fock reference to build the EC subspace.

## Key numbers

* The critical coupling for the normal-to-superfluid phase transition is \(g_\text{crit}=0.34\) for \(\Omega=10\) at half-filling (\(N_\text{occ}=5\)).
* With three training points in a mixed set (e.g., \(g_i \in \{-0.9, 0.3, 0.9\}\)), the EC emulator quantitatively predicts the ground-state energy across the entire coupling range, including the phase transition.
* The EC-PT(1) resummation, using only first-order state corrections, gives results similar to a 2-particle-2-hole configuration interaction (pCI-2p2h) calculation for all coupling values.
* The overlap between exact ground states from the normal regime (\(g < g_\text{crit}\)) and the superfluid regime (\(g > g_\text{crit}\)) is nearly zero, as shown by norm matrix elements sampled from 200 equidistant couplings in \(g \in [-2, 2]\).

## Context

This work extends the eigenvector continuation framework (Frame 2018) to the pairing Hamiltonian, a standard testbed for many-body methods (Hjorth-Jensen 2017). It complements recent reduced-basis studies of the same system using DMRG wave functions (Baran 2022). The results demonstrate that EC can serve as both a resummation tool for divergent perturbation theory and an emulator for parametric studies, enabling future large-scale sensitivity analyses of chiral Hamiltonians for medium-mass nuclei (Ekström 2019, Jiang 2022).
