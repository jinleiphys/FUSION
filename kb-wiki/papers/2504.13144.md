# Bayesian model-data comparison incorporating theoretical uncertainties

> Accurate comparisons between theoretical models and experimental data are critical for scientific progress. However, inferred physical model parameters can vary significantly with the chosen physics model, highlighting the importance of properly accounting for theoretical uncertainties. In this Letter, we present a Bayesian framework that explicitly quantifies these uncertainties by statistically modeling theory errors, guided by qualitative knowledge of a theory's varying reliability across the input domain. We demonstrate the effectiveness of this approach using two systems: a simple ball drop experiment and multi-stage heavy-ion simulations. In both cases incorporating model discrepancy leads to improved parameter estimates, with systematic improvements observed as additional experimental observables are integrated.

## Key claim
The paper presents a Bayesian framework that explicitly quantifies theoretical uncertainties by statistically modeling theory errors using Gaussian processes, guided by qualitative knowledge of a theory's varying reliability across the input domain. The hedge is that the framework requires only qualitative prior knowledge (e.g., "the theory is more reliable in this regime than in that one") rather than precise quantitative constraints on the discrepancy.

## Method
The authors construct a Gaussian process (GP) model for the discrepancy between experimental data and theoretical predictions, using covariance kernels that encode qualitative prior knowledge about the theory's domain of validity. They perform Bayesian parameter inference to simultaneously estimate both the physical model parameters and the GP hyperparameters, updating uniform priors to posterior distributions conditioned on observations. The framework is tested on a ball drop experiment (where the model neglects drag force) and on multi-stage heavy-ion simulations using the iEBE-MUSIC framework (where the model ignores viscous corrections during particlization).

## Key numbers
- Ball drop experiment: mock data generated with true gravitational acceleration g = 9.8 m/s^2 and initial velocity v0 = 0 m/s; observational errors for height, velocity, and acceleration have standard deviations of sigma = 0.1, 0.2, and 0.3, respectively.
- Heavy-ion simulations: Au on Au collisions at sqrt(s_NN) = 200 GeV at 20-30% collision centrality; true model uses constant eta/s = 0.1 and switching energy density epsilon_sw = 0.2 GeV/fm^3.
- Heavy-ion observables: ten p_T bins spanning the range 0-2 GeV for charged hadron multiplicity, elliptic flow v2{2}_ch, triangular flow v3{2}_ch, and identified pion and proton multiplicities and elliptic flows.
- Without model discrepancy in the two-parameter heavy-ion case, eta/s is inferred to be about 40% larger than its true value of 0.1 when all observables are included.

## Context
This work extends the Gaussian process model discrepancy framework introduced by Kennedy and O'Hagan (2002) and explored in subsequent studies such as Brynjarsdottir (2014), but differs by constructing the GP covariance kernel using only qualitative prior knowledge of the theory's domain of validity rather than requiring precise constraints. The framework enables robust parameter extraction from data that are known a priori to differ from the theoretical model, data that are typically discarded in conventional analyses, and allows models of varying complexity to be directly compared against data. The approach is demonstrated to systematically improve parameter estimates as additional experimental observables are integrated, outperforming conventional Bayesian inference without model discrepancy.
