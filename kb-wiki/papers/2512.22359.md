# Orbital angular momentum in the pion and kaon: rest-frame and light-front

> Orbital angular momentum (OAM) is not a Poincaré invariant quantity; so, its value is observer dependent. Notwithstanding that, in quantum chromodynamics, a Poincaré-invariant theory, OAM is part of every hadron wave function. Using continuum Schwinger function methods, we elucidate both the subjective character of in-hadron OAM and expose some of its impacts on pion and kaon structure and observables. For instance, working with light-front projections of their Bethe-Salpeter wave functions, it is found that the pion is a roughly 50/50 mix of light-front OAM zero and one components and the kaon is a 60/40 system. The overall picture is that (near) Nambu-Goldstone modes are complex bound states, each with significant intrinsic OAM, independent of the observer's reference frame. This feature must be accounted for in the calculation of observables. Inductively, the same is true for all hadrons.

## Key claim

Orbital angular momentum (OAM) in the pion and kaon is frame-dependent, but using continuum Schwinger function methods with a nonperturbatively improved truncation (bRL), the authors show that both mesons contain significant intrinsic OAM regardless of the observer's reference frame; the pion is a roughly 50/50 mix of light-front OAM zero and one components, and the kaon is a 60/40 system.

## Method

The authors solved the coupled gap and Bethe-Salpeter equations using continuum Schwinger function methods (CSMs) in two truncation schemes: rainbow-ladder (RL) and an EHM-improved beyond-rainbow-ladder (bRL) truncation that includes a dressed-quark anomalous chromomagnetic moment. They obtained Poincaré-covariant Bethe-Salpeter wave functions for the pion, kaon, and a fictitious heavy pion ($\pi_{s\bar s}$), then projected these onto rest-frame partial-wave components and light-front wave functions to extract OAM decompositions.

## Key numbers

- Pion light-front OAM decomposition: roughly 50/50 mix of $\ell_z=0$ and $\ell_z=\pm 1$ components (from abstract).
- Kaon light-front OAM decomposition: 60/40 system (from abstract).
- RL truncation rest-frame OAM contributions to pion charge: $S \otimes S = 16.2\%$, $S \otimes P = 84.5\%$, $P \otimes P = -0.7\%$ (Table I, Panel A).
- RL truncation rest-frame OAM contributions to kaon charge: $S \otimes S = 23.7\%$, $S \otimes P = 78.3\%$, $P \otimes P = -2.0\%$ (Table I, Panel A).
- bRL truncation rest-frame OAM contributions to pion charge: $S \otimes S = 101.0\%$, $S \otimes P = -1.6\%$, $P \otimes P = 0.6\%$ (Table I, Panel C).
- bRL truncation rest-frame OAM contributions to kaon charge: $S \otimes S = 110.3\%$, $S \otimes P = -17.4\%$, $P \otimes P = 7.1\%$ (Table I, Panel C).
- Pion charge radius (RL): $r_\pi = 0.67$ fm; kaon charge radius: $r_K = 0.60$ fm; $\pi_{s\bar s}$ charge radius: $r_{\pi_{s\bar s}} = 0.49$ fm (from text).
- Pion leptonic decay constant (bRL): $f_\pi = 0.098$ GeV; kaon: $f_K = 0.11$ GeV (Table I, Panel B).

## Context

This work extends earlier CSM studies of OAM in mesons by Bhagwat (2006) and Hilger (2015), which were restricted to rainbow-ladder truncation, by using a nonperturbatively improved bRL truncation that better accounts for emergent hadron mass (EHM) effects. It also provides the first internally consistent comparison between rest-frame and light-front OAM pictures for pseudoscalar mesons, enabled by methods from Yao (2025). The results establish that Nambu-Goldstone bosons possess significant intrinsic OAM independent of frame, which must be accounted for in calculations of observables such as form factors and parton distribution functions, and inductively suggests the same is true for all hadrons.
