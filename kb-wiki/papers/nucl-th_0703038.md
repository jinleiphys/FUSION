# Microscopic formula for transport coefficients of causal hydrodynamics

> The Green-Kubo-Nakano formula should be modified in relativistic hydrodynamics because of the problem of acausality and the breaking of sum rules. In this work, we propose a formula to calculate the transport coefficients of causal hydrodynamics based on the projection operator method. As concrete examples, we derive the expressions for the diffusion coefficient, the shear viscosity coefficient, and corresponding relaxation times.

# Microscopic formula for transport coefficients of causal hydrodynamics

## Key claim
The Green-Kubo-Nakano (GKN) formula for transport coefficients must be modified in relativistic causal hydrodynamics because it yields the inverse of the relaxation time rather than the true causal diffusion coefficient when the coarse-grained dynamics is described by the telegraph equation instead of the diffusion equation. The proposed formula reproduces the GKN result only when the effect of coarse-graining on the memory function is ignored.

## Method
The author uses the Mori projection operator method to derive a time-convolution equation for a gross (hydrodynamic) variable, then obtains an exact expression for the memory function in Fourier space in terms of correlation functions $\dot{X}^F(\mathbf{k},\omega)$ and $\ddot{X}^F(\mathbf{k},\omega)$. By separating the memory function into a frequency function and a renormalized memory function, and applying a local approximation where the frequency function converges to a finite value and the renormalized memory function relaxes rapidly to zero, the author derives expressions for the causal diffusion coefficient $D_\mathbf{k}$ and relaxation time $\tau_\mathbf{k}$.

## Key numbers
- The causal diffusion coefficient is defined as $D_{\bf k} = \frac{1}{{\bf k}^2}\lim_{t\rightarrow \infty}\Omega^2({\bf k},t)$ (Eq. 13).
- The relaxation time is defined as $\frac{1}{\tau_{\bf k}} = \int^{\infty}_{0}dt \Phi({\bf k},t)$ (Eq. 14).
- When coarse-graining is ignored for the diffusion process, the result gives $D_{\bf k} = 0$ and $\frac{1}{\tau_{\bf k}} = D_{ac}k^2$, where $D_{ac}$ is the standard GKN diffusion constant.
- For the shear viscosity of relativistic dissipative hydrodynamics, the relaxation time is $\tau = \lim_{k_y \rightarrow 0} \tau_{k_y}$ and the shear viscosity is $\eta = \lim_{k_y \rightarrow 0} 2 w D_{k_y} \tau_{k_y}$, where $w$ is the enthalpy density.

## Context
This work extends the projection operator method previously applied to the chiral order parameter by Koide (2006) and to nonrelativistic number density by Koide (2006), providing an exact (except for the local approximation) formula for transport coefficients of causal hydrodynamics. The result corrects the standard GKN formula (Kubo, 1957) which, when applied to relativistic hydrodynamics, violates causality and sum rules as shown by Kadanoff and Martin (1963) and Koide (2006). The formalism enables calculation of transport coefficients for causal hydrodynamics near phase transitions, where critical slowing down of shear viscosity is expected (McLerran, 1986), and generalizes the shear viscosity result of Hosoya (1984).
