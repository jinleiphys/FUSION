# Iterative Bayesian Monte Carlo for nuclear data evaluation

> In this work, we explore the use of an iterative Bayesian Monte Carlo (IBM) procedure for nuclear data evaluation within a Talys Evaluated Nuclear data Library (TENDL) framework. In order to identify the model and parameter combinations that reproduce selected experimental data, different physical models implemented within the TALYS code, were sampled and varied simultaneously to produce random input files with unique model combinations. All the models considered were assumed to be equal a priori. Parameters to these models were then varied simultaneously using the TALYS code system to produce a set of random ENDF files which were processed into x-y tables for comparison with selected experimental data from the EXFOR database within a Bayesian framework. To improve our fit to experimental data, we iteratively update our 'best' file - the file that maximises the likelihood function - by re-sampling model parameters around this file. The method proposed has been applied for the evaluation of p+Cd-111 and Co-59 between 1 - 100 MeV incident energy region. Finally, the adjusted files were compared with experimental data from the EXFOR database as well as with evaluations from the TENDL-2017 and JENDL-4.0/HE nuclear data libraries.

## Key claim

The authors propose an iterative Bayesian Monte Carlo (IBM) procedure that simultaneously varies both nuclear reaction models and their parameters within the TALYS code, then iteratively re-samples around the file that maximizes the likelihood function, to produce a single evaluated nuclear data file that performs globally well across multiple experimental data types. The hedge is that the method assumes all models are equally important a priori and that model defects are not explicitly treated, with convergence assessed by the relative change in the maximum likelihood estimate over the last 2-5 iterations (though only the last two iterations were used due to computational constraints).

## Method

The authors generated random TALYS input files by simultaneously sampling 52 different physical models (including 4 pre-equilibrium models, 6 level density models, 8 gamma-strength function models, and 4 JLM optical model variants) and their parameters from uniform distributions, producing a parent generation (Gen. 0) of random ENDF files. These files were converted to x-y tables and compared against selected experimental data from the EXFOR database for three data types: reaction cross sections, residual production cross sections, and elastic angular distributions. A global reduced chi-square was computed as the average of the reduced chi-squares for each experimental type, and the file maximizing the likelihood function (equivalent to minimizing chi-square) was selected as the "best" file; model parameters were then re-sampled around this file iteratively to produce subsequent generations. The method was applied to p+Co-59 and p+Cd-111 between 1-100 MeV incident energy.

## Key numbers

- 52 different physical models were varied simultaneously within the TALYS code to create random input files
- For p+Co-59: 185 experimental data points for elastic angular distributions (angles 1-180 degrees, incident energies 5-40 MeV), 169 data points for reaction cross sections across 8 channels, and 141 data points for residual production cross sections across 9 channels were considered
- For p+Cd-111: only reaction cross sections were considered, with 6 reaction channels: (p,n), (p,2n), (p,2n)g, (p,2n)m, (p,3n), and (p,4n) between 1-100 MeV
- The energy grid bin size was 20 for the parent generation (Gen. 0) and increased to 60 for subsequent generations
- Convergence criterion: relative change in the maximum likelihood estimate between the last 2-5 iterations (10-20% relative change threshold), but only the last two iterations were used in practice

## Context

This work extends the Bayesian Monte Carlo (BMC) approach (Ref. bib:2) and the Total Monte Carlo (TMC) method (Ref. bib:1) by adding iteration and simultaneous variation of both models and parameters, addressing the problem that increasing parameter space alone can lead to low-likelihood regions as observed in Refs. bib:8 and bib:7. The method is applied within the TENDL framework (Ref. bib:13) and aims to produce a single TALYS input file that performs globally across multiple experimental types, unlike previous approaches that evaluated individual channels separately. The work enables future application of Bayesian Model Averaging (BMA) over model combinations (as proposed in Ref. bib:13a) and could be extended to include integral benchmark experiments using methods like MOCABA (Ref. bib:9a).
