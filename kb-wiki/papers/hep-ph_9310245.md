# Squeezed Quantum State of Disoriented Chiral Condensate

> We consider the quantum state describing the Disoriented Chiral Condensate (DCC) which may be produced in high energy collisions. Using the approach suggested by Rajagopal and Wilczek to describe the amplification of the long wavelength classical pion modes, we consider the quantum-mechanical evolution of the initial vacuum state into the final squeezed state describing the DCC. The obtained wave function has some interesting properties which are discussed.

## Key claim

The quantum state describing the Disoriented Chiral Condensate (DCC) produced in high-energy collisions is a squeezed state, not a coherent state, and within the mean-field (Hartree-Fock) approximation its wave function is a product of squeezed states for each pion mode, with the squeezing parameters completely determined by the time evolution of the spatially averaged field \(<\phi^{2}>(t)\).

## Method

The author considers the linear sigma model with an O(4) field and applies the Rajagopal and Wilczek (1993) mechanism for amplification of long-wavelength pion modes after quenching. Using the mean-field approximation, the equation of motion for each pion mode \(\pi(\vec{k},t)\) becomes that of a parametrically excited oscillator with time-dependent frequency \(\Omega^{2}(\vec{k},t) = \vec{k}^{2} + \lambda(<\phi^{2}>(t) - v^{2})\). The Schrödinger equation for each mode is solved in the coordinate (\(\pi\)) representation, yielding a Gaussian wave function \(\psi(\pi, t) = C(t) \exp(-B(t) \pi^{2})\); the parameter \(B(t)\) is related to the solution of the classical oscillator equation and is matched to the form of a squeezed state at \(t \to \infty\), where the squeezing parameter \(r\) and phase \(\phi\) are extracted from the tunneling probability through the effective potential barrier \(V(t) = \lambda(v^{2} - <\phi^{2}>(t))\).

## Key numbers

- The squeezing parameter for a simple model where \(<\phi^{2}>(t) = 0\) for \(t \in (0, m_{\pi}^{-1})\) is estimated as \(r(k) = \sqrt{2} \frac{m_{\sigma}}{m_{\pi}} \sqrt{1 - 2\frac{\omega^{2}(k)}{m_{\sigma}^{2}}} \approx 6 \sqrt{1 - 2\frac{\omega^{2}(k)}{m_{\sigma}^{2}}}\).
- The average number of particles in each mode is \(\langle N_{k} \rangle = \sinh^{2} r_{k} = \exp\left(2 \text{Re} \int dt \sqrt{V(t) - k^{2}}\right) - 1\).
- For a squeezed state with large \(r\), the second-order correlation function at zero time delay is \(g^{2}(0) = 2 + \coth^{2} r > 3\).
- The tunneling coefficient for a mode is given by \(\frac{|D|^{2}}{|B|^{2}} = \frac{1}{\cosh^{2} r}\).

## Context

This work extends the classical description of DCC amplification by Rajagopal and Wilczek (1993) to a full quantum-mechanical treatment, showing that the resulting state is a squeezed state rather than a coherent state as previously considered. It addresses the criticism by Kluger and collaborators (1993) that a coherent state leads to unphysically large charge fluctuations, and it provides a dynamical mechanism for generating the quadratic-in-creation-operator wave functions that yield the \(1/\sqrt{f}\) neutral-pion fraction distribution. The formalism is analogous to the production of relic gravitons from zero-point fluctuations during cosmological expansion, as discussed by Grishchuk and Sidorov (1990). The results enable predictions for the momentum dependence of observables and for bunching effects (super-Poissonian statistics) in pion correlations, and suggest a similar squeezed-state description for soft gluon modes in gauge theories.
