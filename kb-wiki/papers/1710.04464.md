# Rope Hadronization and Strange Particle Production

> Rope Hadronization is a model extending the Lund string hadronization model to describe environments with many overlapping strings, such as high multiplicity pp collisions or $AA$ collisions. Including effects of Rope Hadronization drastically improves description of strange/non-strange hadron ratios as function of event multiplicity in all systems from $e^+ e^-$ to $AA$. Implementation of Rope Hadronization in the MC event generators DIPSY and Pythia8 is discussed, as well as future prospects for jet studies and studies of small systems.

## Key claim

Rope Hadronization, a model extending the Lund string hadronization to environments with many overlapping strings, can describe strangeness enhancement in collision systems from e+e- to AA without assuming a deconfined or thermalized plasma; the key condition is that the effective string tension is calculated as the difference of available energy in the full multiplet and the multiplet reached by breaking one string in the rope, rather than using the full multiplet directly.

## Method

The model calculates the transverse-space overlap of string pieces, counting the number of parallel (p) and anti-parallel (q) string pieces to determine an effective string tension via the relation κ̃ = (2p+q+2)/4 κ. This effective tension replaces the standard string tension κ in the Schwinger formula for string breaking, modifying the suppression factor for strange quarks from ρ = exp(-π(m_s² - m_u²)/κ) to ρ̃ = ρ^(κ/κ̃), which approaches unity as κ̃ grows large. The implementation was done in the DIPSY parton shower event generator and as a direct plug-in to the Pythia8 Monte Carlo event generator.

## Key numbers

- The standard string tension is κ ≈ 1 GeV/fm.
- Using constituent quark masses m_u ≈ 0.3 GeV and m_s ≈ 0.5 GeV, the default strange quark suppression factor is ρ ≈ 0.08.
- The effective string tension is given by κ̃ = (2p+q+2)/4 κ.
- In a pp collision at LHC, the modified suppression factor ρ̃ reaches values up to approximately 3ρ.

## Context

This work extends the original Lund string model (Andersson, 1983; Andersson, 1979) and the Monash tune for Pythia8 (Skands, 2014) by addressing the failure of unmodified string hadronization to describe strange particle ratios in pp collisions. The rope hadronization model builds on earlier ideas of coherent string interactions in nuclear collisions (Biro, 1984; Bialas, 1984) and provides a microscopic alternative to thermodynamic explanations of strangeness enhancement; it enables future investigations into whether such a model can also describe flow and jet quenching effects typically attributed to quark-gluon plasma formation.
