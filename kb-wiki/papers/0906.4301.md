# QRAP: a numerical code for projected (Q)uasi-particle (RA)ndom (P)hase approximation

> A computer code for quasiparticle random phase approximation-QRPA and projected quasiparticle random phase approximation-PQRPA models of nuclear structure is explained in details. An important application of the code consists in evaluating nuclear matrix elements involved in neutrino-nucleus reactions. As an example, cross section for 56Fe and 12C are calculated and the code output is explained. The application to other nuclei and the description of other nuclear and weak decay processes is also discussed.

## Key claim

The paper presents the QRAP computer code, which implements both the quasiparticle random phase approximation (QRPA) and the number-projected quasiparticle random phase approximation (PQRPA) for nuclear structure calculations, with the residual interaction approximated by a simple delta-force. The key hedge is that while the code can calculate neutrino-nucleus cross sections and muon capture rates, the PQRPA is specifically necessary for a physically sound description of weak processes in light nuclei such as the A=12 iso-triplet, where the standard QRPA fails because of degeneracy among quasiparticle energies arising from non-conservation of particle number.

## Method

The code solves the QRPA or PQRPA equations self-consistently for even-even nuclei, using a delta-force residual interaction and a BCS or projected BCS (PBCS) treatment of pairing correlations. The nuclear matrix elements for neutrino-nucleus interactions are treated as the beta inverse reaction of odd-odd nuclei as a function of the transfer momentum, with the weak Hamiltonian expressed in the Foldy-Wouthuysen transformed form and multipole-expanded into natural and unnatural parity operators. The code calculates charged-current and neutral-current neutrino-nucleus cross sections and muon capture rates for specified target nuclei.

## Key numbers

- The Fermi coupling constant used is G = 3.04545 × 10⁻¹² (in natural units).
- The effective axial-vector coupling constant used in numerical calculations is g_A^eff = 1.
- The weak-magnetism coupling constant is g_M = κ_p − κ_n = 3.70.
- The dipole form factor cutoff for finite nuclear size effects is Λ = 850 MeV.
- Typical running time is approximately 5 minutes on a 3 GHz processor for Data set 1.
- The code requires 16 Mbytes of RAM memory and 2 MB of hard disk space, with approximately 8,000 lines of Fortran-77 code.

## Context

This work extends the formalism developed in Krmpotic (2002, 2005) and Samana (2006) for neutrino-nucleus scattering, applying the PQRPA to address the known failure of standard QRPA in describing the isospin triplet T=1 states in A=12 nuclei, where earlier continuum RPA calculations by Kolbe, Langanke, and Krewald (1994) required a reduction factor of about 4 to match data. The code enables systematic comparison of different nuclear models (QRPA, PQRPA, hybrid model, relativistic QRPA) for neutrino-nucleus cross sections on nuclei such as ¹²C and ⁵⁶Fe, and can be adapted for evaluation of double-beta decay rates.
