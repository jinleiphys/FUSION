# Reexamination of the $β^+$ decay process of $^{13}\rm{N}$ nucleus

> It is known from the scattering theory that the phase-shift of elastic collision does not provide a unique potential to describe the bound state of the two-particle system. The bound state wave function is the most crucial input for various nuclear processes bearing astrophysical significance. In this paper, we emphasize on the important role of the asymptotic normalization coefficients for description of the $^{13}\rm{N}\longrightarrow {^{13}\rm{C}}+β^++ν_e$ reaction. Using experimental data of the elastic scattering phase-shift of proton and neutron on $^{12}\rm{C}$, we find the asymptotic normalization coefficients which determine the tail of the bound state wave functions. This allows us to modify the wave functions of $^{13}\rm{N}$ and $^{13}\rm{C}$ within the single-particle approach using the inverse scattering theory. These wave functions are used to determine the nuclear matrix element for calculation of the half-life and log{\textit ft} values of $β^+$ decay of $^{13}\rm{N}$. The calculated values of the log{\textit ft} and half-life are smaller than the measured values when the single-particle wave functions are employed. Using overlap functions, instead of single-particle functions, we obtain a better comparison. The overlap function is represented as the product of single-particle function and the corresponding spectroscopic factor.

## Key claim

The authors show that using single-particle wave functions alone yields calculated half-life and log *ft* values for the $^{13}\rm{N}$ $\beta^+$ decay that are smaller than the measured values; only by replacing single-particle functions with overlap functions (the product of a single-particle function and the corresponding spectroscopic factor) does the calculation achieve better agreement with experiment.

## Method

The authors treat $^{13}\rm{N}$ and $^{13}\rm{C}$ as two-body cluster systems ($^{12}\rm{C}$ plus a proton or neutron, respectively) and construct bound-state wave functions using Woods-Saxon potentials fitted to elastic scattering phase-shift data for $p$-$^{12}\rm{C}$ and $n$-$^{12}\rm{C}$ collisions. They extract asymptotic normalization coefficients (ANCs) from the phase-shift data via the effective-range method and the $\Delta$-method, then use these ANCs to normalize the single-particle wave functions. The nuclear matrix element for the $\beta^+$ decay is computed both with the single-particle wave functions and with overlap functions that incorporate spectroscopic factors, and the resulting half-life and log *ft* values are compared to experimental data.

## Key numbers

* Binding energy of the $^{12}\rm{C}$-$p$ system ($^{13}\rm{N}$): $\varepsilon_b = 1.994$ MeV.
* Binding energy of the $^{12}\rm{C}$-$n$ system ($^{13}\rm{C}$): $\varepsilon_b = 4.946$ MeV.
* Overlap integral of initial and final radial wave functions: 0.997 for the first pair of potentials, 0.995 for the second pair.
* Ratio of squared proton ANC to squared neutron ANC: $\mathcal{R}_0 = 1.169$ for the first pair of potentials, $\mathcal{R}_0 = 1.254$ for the second pair.
* For neutron scattering on $^{12}\rm{C}$, the fitted resonance energy is $E_r = 3.959$ MeV and the width is $\Gamma = 174$ keV (compared to experimental values $E_x = 8.860 \pm 0.02$ MeV and $\Gamma = 150 \pm 30$ keV for the $J^\pi = 1/2^-$ state in $^{13}\rm{C}$).

## Context

This work extends the approach of Irgaziev et al. (2015, 2018) and Orlov et al. (2016, 2017) for extracting ANCs from elastic scattering phase-shifts and applying them to nuclear reactions, previously demonstrated for the $\alpha$-$d$ system by Blokhintsev et al. (1993) and for radiative capture by Abishev et al. It addresses the known shortcoming of the single-particle shell model, which gives *ft*-values for allowed transitions that are generally smaller than observed (as noted by Gross and by Suhonen). The method enables more accurate calculation of $\beta^+$ decay rates for light nuclei with pronounced single-particle configurations, which is relevant for the CNO cycle in stars and for solar neutrino flux predictions.
