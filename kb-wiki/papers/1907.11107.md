# Collinear Drop

> We introduce collinear drop jet substructure observables, which are unaffected by contributions from collinear radiation, and systematically probe soft radiation within jets. These observables can be designed to be either sensitive or insensitive to process-dependent soft radiation originating from outside the jet. Such collinear drop observables can be exploited as variables to distinguish quark, gluon, and color neutral initiated jets, for testing predictions for perturbative soft radiation in Monte Carlo simulations, for assessing models and universality for hadronization corrections, for examining the efficiency of pileup subtraction methods, and for any other application that leaves an imprint on soft radiation. We discuss examples of collinear drop observables that are based both on clustering and on jet shapes. Using the soft-collinear effective theory we derive factorization expressions for collinear drop observables from QCD jets, and carry out a resummation of logarithmically enhanced contributions at next-to-leading-logarithmic order. We also identify an infinite class of collinear drop observables for which the leading double logarithms are absent.

## Key claim

Collinear drop observables are designed to be unaffected by collinear radiation within jets while systematically probing soft radiation, and can be tuned to be either sensitive or insensitive to process-dependent soft radiation originating from outside the jet, depending on the choice of grooming parameters.

## Method

The authors construct collinear drop observables using two approaches: (1) taking the difference between two soft-drop groomed jet masses, \(m_{{\rm SD}_1}^2 - m_{{\rm SD}_2}^2\), where SD\(_2\) grooms more aggressively than SD\(_1\) so that the remaining particles in the SD\(_2\) jet are a subset of those in the SD\(_1\) jet; (2) defining "flattened angularities" \(\tau_\omega = \sum_i z_i\,\omega(\theta_i,\theta_0)\) with an angular weight function \(\omega\) that vanishes or is exponentially suppressed for angles \(\theta \le \theta_0\). Using soft-collinear effective theory (SCET), they derive factorization expressions for these observables and perform resummation of logarithmically enhanced contributions at next-to-leading-logarithmic (NLL) order.

## Key numbers

* The soft drop condition for \(pp\) collisions: \(\min(p_{T_i},p_{T_j})/(p_{T_i}+p_{T_j}) > z_{\rm cut} (\Delta R_{ij}/R_0)^\beta\), where \(z_{\rm cut}\) is a dimensionless parameter and \(\beta\) provides an angular weight.
* The soft drop condition for \(e^+e^-\) collisions: \(\min(E_i,E_j)/(E_i+E_j) > z_{\rm cut} (\sqrt{2}\sin(\theta_{ij}/2)/\sin(R_0^{ee}/2))^\beta\).
* The collinear drop jet mass is defined as \(\Delta m^2 = m_{{\rm SD}_1}^2 - m_{{\rm SD}_2}^2\), with the requirement that \(\{{\rm jet}_{{\rm SD}_2}\} \subset \{{\rm jet}_{{\rm SD}_1}\}\).
* For the annulus energy fraction, the angular weight function \(\omega_a(\theta,R_1) = 1\) for \(R_1 < \theta < R_2\) and 0 otherwise, with \(R_1 > 0\) chosen to contain the majority of collinear radiation.

## Context

This work introduces a new class of jet substructure observables that complement existing jet grooming techniques such as soft drop (Larkoski, 2014) and conventional angularities (Berger, 2003; Almeida, 2008), which suppress soft radiation rather than collinear radiation. The collinear drop approach enables systematic study of soft radiation within jets, with applications including distinguishing quark, gluon, and color-neutral initiated jets, testing perturbative soft radiation predictions in Monte Carlo simulations (such as Pythia and Vincia), assessing hadronization correction models, and examining pileup subtraction methods. The authors note that these observables can also be used to study jet quenching in heavy ion collisions.
