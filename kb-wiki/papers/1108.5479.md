# Symmetry restoration in Hartree-Fock-Bogoliubov based theories

> We present a pfaffian formula for projection and symmetry restoration for wave functions of the general Bogoliubov form, including quasiparticle excited states and linear combinations of them. This solves a long-standing problem in calculating states of good symmetry, arising from the sign ambiguity of the commonly used determinant formula. A simple example is given of projecting good particle number and angular momentum from a Bogoliubov wave function in the Fock space of a single j-shell.

## Key claim

The paper presents a pfaffian formula for symmetry restoration that works for general Bogoliubov wave functions, including those with odd particle number and multi-quasiparticle excitations, solving the sign ambiguity of the Onishi determinant formula. The method avoids the need to invert the Bogoliubov transformation matrix U, making it applicable to wave functions that have zero overlap with the vacuum.

## Method

The authors derive a pfaffian expression for the overlap ⟨w|R|w⟩ where R is a symmetry operator, using the Bloch-Messiah decomposition of the Bogoliubov U,V matrices. They extend this to overlaps involving quasiparticle excitations by writing the wave function as a product of Bogoliubov quasiparticle annihilation operators acting on the vacuum, then evaluating contractions via the pfaffian of all possible pairings. The generalization to multi-quasiparticle overlaps with r annihilation operators on one side and s creation operators on the other is given by augmenting the pfaffian matrix with 2k additional rows and columns for k quasiparticles.

## Key numbers

- Test wave function uses a j=5/2 shell with 6-dimensional Fock space, parameters (u,v) = (0.8,0.6)
- 4-point uniform mesh for gauge angle integration, 5-point Gauss-Legendre quadrature for cos(β) integration
- For N=1, J=5/2: projected probability u² = 0.64 (exact and numerical)
- For N=3, J=3/2: projected probability v²/7 ≈ 0.05143 (exact and numerical)
- For N=3, J=5/2: projected probability v²/2 = 0.18 (exact and numerical)
- For N=3, J=9/2: projected probability 5v²/14 ≈ 0.12857 (exact and numerical)

## Context

This work extends the pfaffian approach of Robledo (2009) which required the inverse of the U matrix and thus could not handle wave functions orthogonal to the vacuum. It provides a practical method for projecting Hartree-Fock-Bogoliubov wave functions onto good particle number and angular momentum, enabling configuration mixing calculations in the Generator Coordinate Method. The authors note that Avez and Bender (2011) and Oi and Mizusaki (2011) independently obtained similar general formulas.
