# Saturation of the width of the strength function

> The strength function of a single state $|d>$ is studied using the deformed Gaussian orthogonal ensemble. In particular we study the dependence of the spreading width of $|d>$ on the degree of mixing.

## Key claim

The spreading width of a single state coupled to a chaotic background saturates as the mixing strength increases, transitioning from a quadratic dependence on the mixing parameter at weak coupling to a linear dependence and then to a very weak dependence as the mixing parameter approaches unity; the golden rule approximation fails to predict the full width at half maximum for strong mixing.

## Method

The authors modeled the Hamiltonian as a deformed Gaussian orthogonal ensemble (DGOE) where the diagonal matrix elements have variance a²/2N and the off-diagonal matrix elements have variance λ²a²/4N, with λ controlling the degree of mixing between 0 and 1. They computed the strength function F_d(E) for a single state |d⟩ by numerically diagonalizing the DGOE Hamiltonian for N=50 background states, unfolding the spectrum to unit mean level density, and averaging over 100 to 1000 realizations depending on λ. The spreading width was extracted both from the peak value of the width function Γ_d^↓(E_d) and from the full width at half maximum of the strength function.

## Key numbers

- The condition for the single state |d⟩ to be significantly mixed with more than one background state is λ > 2√π/N, which for N=50 gives λ > 0.07.
- At λ=1, the golden rule prediction for the spreading width is 2N/π, while the full width at half maximum of the strength function is essentially equal to N (=50); the golden rule is a factor 2/π smaller than the FWHM.
- For very weak coupling (λ < 0.04), the spreading width Γ_d^↓(E_d) shows a quadratic dependence on λ, accurately following the golden rule calculation using the Poisson density: 2π(λ²a²/4N)(N^(3/2)/(a√π))² = λ²N²/2.
- Between λ=0.05 and λ=0.15, the dependence of Γ_d^↓(E_d) on λ becomes linear.
- At λ=1, the DGOE calculation of Γ_d^↓(E_d) is approximately a factor (4/π)² = 1.6 greater than the calculation using the Persson and Åberg average level density formula.
- The averaging interval I used to represent the delta function was I=3, and the parameter a was set to 2.

## Context

This work extends previous studies of strength functions using random matrix theory, specifically the DGOE approach of Hussein (1993), and complements investigations by Frazier (1996) who studied the spreading width of a shell model basis state as a function of residual interaction strength and identified similar quadratic-to-linear transitions. The results enable a better understanding of the decay out of superdeformed rotational bands, which the authors state is in progress, and provide a random-matrix framework for interpreting phenomena such as isobaric analogue resonances and giant dipole resonances where a single state mixes with a complex background.
