# TPEcalc: a program for calculation of two-photon exchange amplitudes

> TPEcalc is a C++ program for calculation of two-photon exchange (TPE) amplitudes in elastic electron-hadron scattering, based on the dispersion method. It is a command-line tool which accepts kinematical parameters (Q^2 and epsilon) as input and returns TPE amplitudes. It can do calculations for proton, neutron and pion targets. Any spin 1/2+ or spin 0- target is supported, if the user supplies all necessary form factor parameterizations. This paper describes how to use TPEcalc and outlines underlying theory. The program source code can be downloaded from http://tpe.bitp.kiev.ua/

## Key claim

TPEcalc provides the first publicly available computer program for calculating two-photon exchange amplitudes in elastic electron-hadron scattering using the dispersion method, but the accuracy drops seriously as the virtual photon polarization parameter epsilon approaches zero, and calculations at epsilon=0 are impossible due to the subtraction method employed.

## Method

The program calculates TPE amplitudes by first computing the imaginary part of the scattering amplitude via a sum over intermediate hadronic states using on-shell transition form factors, then reconstructing the full amplitude through analyticity. The loop integrals are evaluated semianalytically with numerical integration as the final step, where the precision is controlled by the -n option (default 1000). Transition form factors are parameterized as sums of poles, and the elastic contribution has the standard Mo and Tsai correction subtracted to cancel infrared divergence.

## Key numbers

- Target masses used in predefined parfiles: proton M=0.938270 GeV, Delta(1232) M=1.232000 GeV, pion M=0.13957 GeV, rho M=0.7758 GeV, b1(1235) M=0.78259 GeV
- Input parameters must satisfy Q^2 > 0 and 0 < epsilon < 1
- Default calculation precision: -n=1000
- Form factor cut-off option -c=NUMBER where Lambda (in GeV) damps FFs as F(Q^2) -> F(Q^2) * Lambda^2 / (Q^2 + Lambda^2)
- For spin 1/2+ targets, the program outputs six quantities: real and imaginary parts of delta G_E/G_M, delta G_M/G_M, and delta G_3/G_M
- For spin 0- targets, the program outputs real and imaginary parts of delta F/F

## Context

This work extends the dispersion method calculations of Borisyuk and Kobushkin (2011) for TPE amplitudes in elastic electron-proton scattering, and provides a practical tool for the community that previously had no publicly available code for such calculations. The program enables experimental searches for TPE effects (such as those reviewed in TPEexp and already completed in TPEfin) and allows theorists to compute TPE corrections that remove the discrepancy between Rosenbluth and polarization methods in proton form factor measurements, as shown by Blunden and Sick (2005) for low-Q^2 and Blunden, Melnitchouk, and Tjon (2003) for high-Q^2 elastic ep scattering.
