# RADCAP: a potential model tool for direct capture reactions

> A computer program is presented aiming at the calculation of bound and continuum states, reduced transition probabilities, phase-shifts, photo-disintegration cross sections, radiative capture cross sections, and astrophysical S-factors, for a two-body nuclear system. The code is based on a potential model of a Woods-Saxon, a Gaussian, or a M3Y, type. It can be used to calculate nuclear reaction rates in numerous astrophysical scenarios.

## Key claim
RADCAP is a Fortran-77 computer program that calculates direct capture cross sections and astrophysical S-factors for two-body nuclear systems using a potential model (Woods-Saxon, Gaussian, or M3Y). The abstract omits that the code treats only the relative motion of the two bodies without internal structure, and that spectroscopic factors from experiment or shell-model calculations must be supplied externally to obtain absolute cross sections.

## Method
The program solves the radial Schrodinger equation for bound states using a WKB estimate followed by Numerov integration with matching at the nuclear surface, and for continuum states using Runge-Kutta integration matched to Coulomb wavefunctions at large distances. It computes bound and continuum wavefunctions, phase shifts, resonance widths, electromagnetic multipole matrix elements (E1, E2, M1), photodisintegration cross sections, radiative capture cross sections, and astrophysical S-factors for both charged-particle and neutron-induced reactions. The code is organized into five modules: M3Y potential calculation, bound state eigenenergies, reduced transition probabilities between bound states, continuum states and phase shifts, and astrophysical S-factors with capture cross sections.

## Key numbers
- Typical running time: about 1 minute on a 1 GHz Intel P4-processor machine for a Woods-Saxon potential.
- Memory required: 8 Mbytes of RAM and 2 MB of hard disk space; 3054 lines of Fortran-77 code.
- M3Y interaction parameters: A = 7999 MeV, B = -2134 MeV, C = -276 MeV fm^3, beta_1 = 4 fm^{-1}, beta_2 = 2.5 fm^{-1}.
- Spin-orbit potential uses pion Compton wavelength = 1.414 fm.
- For M1 transitions: gamma width formula Gamma_M1 [eV] = 1.16e-2 * E_gamma^3 [MeV] * B(M1) [mu_N^2].
- For E2 transitions: Gamma_E2 [eV] = 8.13e-7 * E_gamma^5 [MeV] * B(E2) [e^2 fm^4].
- Example system: p + ^7Be with Gaussian density radius parameters R = 0.7 fm for proton and R = 1.96 fm for ^7Be.
- Effective mean energy for thermonuclear fusion: E_0 = 1.22 (Z_a^2 Z_b^2 m_ab T_6^2)^(1/2) keV, where T_6 is temperature in 10^6 K.
- Output cross sections in millibarns, S-factors in eV.b.

## Context
This work provides a computational tool for direct capture reactions relevant to nuclear astrophysics, extending the potential model formalism developed in Christy and Duck (1961), Tombrello and Parker (1963), and Rolfs (1973). The code enables calculation of reaction rates for both charged-particle and neutron-induced processes, including the special case of halo nuclei where the simple 1/v law breaks down for low binding energies, as discussed in Bertulani and Saito (1992) and Otsuka et al. (1994). It allows users to compute astrophysical S-factors and reaction rates for numerous stellar scenarios, and can be combined with shell-model codes such as OXBASH (Brown, 1984) to supply spectroscopic factors.
