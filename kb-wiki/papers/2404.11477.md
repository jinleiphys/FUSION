# Discovering Nuclear Models from Symbolic Machine Learning

> Numerous phenomenological nuclear models have been proposed to describe specific observables within different regions of the nuclear chart. However, developing a unified model that describes the complex behavior of all nuclei remains an open challenge. Here, we explore whether novel symbolic Machine Learning (ML) can rediscover traditional nuclear physics models or identify alternatives with improved simplicity, fidelity, and predictive power. To address this challenge, we developed a Multi-objective Iterated Symbolic Regression approach that handles symbolic regressions over multiple target observables, accounts for experimental uncertainties and is robust against high-dimensional problems. As a proof of principle, we applied this method to describe the nuclear binding energies and charge radii of light and medium mass nuclei. Our approach identified simple analytical relationships based on the number of protons and neutrons, providing interpretable models with precision comparable to state-of-the-art nuclear models. Additionally, we integrated this ML-discovered model with an existing complementary model to estimate the limits of nuclear stability. These results highlight the potential of symbolic ML to develop accurate nuclear models and guide our description of complex many-body problems.

## Key claim

Symbolic machine learning, with minimal human bias, can rediscover traditional nuclear models or propose alternative analytical expressions for binding energies and charge radii that achieve precision comparable to state-of-the-art nuclear models, but only for light and medium mass nuclei (12 ≤ Z ≤ 50) as a proof of principle.

## Method

The authors developed a Multi-objective Iterated Symbolic Regression (MISR) framework that iteratively searches for analytical expressions describing multiple target observables simultaneously, accounting for experimental uncertainties and using k-fold cross-validation. The algorithm was trained on experimental binding energies from AME2020 and charge radii data for nuclei with 12 ≤ Z ≤ 50, using input variables including neutron number N, proton number Z, mass number A, isospin asymmetry I, Casten factor P, and valence nucleon numbers N_n and N_p. The MISR model was then combined with the Duflo-Zuker model using Bayesian Automatic Relevance Determination (ARD) regression to create an ensemble model for predicting nuclear stability limits.

## Key numbers

- MISR10 binding energy model achieves MAE = 0.78 MeV and RMSE = 0.99 MeV for nuclei with 12 ≤ Z ≤ 50
- MISR10 charge radius model achieves MAE = 0.009 fm on both training and test sets for nuclei with 12 ≤ Z ≤ 50
- The first-order MISR expression for binding energy contains parameters a = -1.10, b = 32.43, c = 16.70, with η₀ = 1.0 MeV
- The first-order MISR expression for charge radius contains parameters a = 0.95 fm, b = 1.48 fm, c = 0.017 fm
- The combined ARD ensemble model achieves MAE = 0.39 MeV for binding energy, 0.28 MeV for one-neutron separation energy, and 0.77 MeV for one-proton separation energy
- The ARD model yields approximately 20% relative improvement over the best-performing HFB24 model (MAE = 0.55 MeV)

## Context

This work extends the application of symbolic regression methods, previously successful in cosmology, astronomy, and particle physics, to nuclear physics observables. The MISR approach addresses limitations of conventional machine learning methods that produce black-box models with obscure physical interpretability, as noted by Munoz (2022) and Nolte (2023). The discovered analytical expressions are compared against traditional models including the Liquid Drop Model (Benzaid, 2020), Finite Range Droplet Model (Moller, 2012), Duflo-Zuker model (Mendoza-Temis, 2009), and various Density Functional Theory calculations (DD-ME2, NL3*, HFB24, UNEDF1). The combined ARD model enables predictions of nuclear stability limits for neutron-deficient and neutron-rich nuclei, addressing open questions about the boundaries of the nuclear landscape.
