# Robust extraction of proton charge radius from electron-proton scattering data

> Extracting the proton charge radius from electron scattering data requires determining the slope of the charge form factor at $Q^2$ of zero. But as experimental data never reach that limit, numerous methods for making the extraction have been proposed, though often the functions are determined after seeing the data which can lead to confirmation bias. To find functional forms that will allow for a robust extraction of the input radius for a wide variety of functional forms in order to have confidence in the extraction from upcoming low $Q^2$ experimental data such as the Jefferson Lab PRad experiment, we create a general framework for inputting form-factor functions as well as various fitting functions. The input form factors are used to generate pseudo-data with fluctuations intended to mimic the binning and random uncertainty of a given set of real data. All combinations of input functions and fit functions can then be tested repeatedly against regenerated pseudo-data. Since the input radius is known, this allows us to find fit functions that are robust for radius extractions in an objective fashion. For the range and uncertainty of the PRad data, we find that a two-parameter rational function, a two-parameter continued fraction and the second order polynomial expansion of $z$ can extract the input radius regardless of the input charge form factor function that is used. We have created an easily expandable framework to search for functional forms that allow for a robust extraction of the radius from a given binning and uncertainty of pseudo-data generated from a wide variety of trial functions. This method has enabled a successful search for the best functional forms to extract the radius from the upcoming PRad data and can be used for other experiments.

## Key claim

For the specific binning and uncertainty of the upcoming Jefferson Lab PRad experiment, a two-parameter rational function, a two-parameter continued fraction, and a second-order polynomial expansion in the transformed variable \(z\) can all robustly extract the input proton charge radius regardless of which underlying charge form factor model generated the pseudo-data, provided the full \(Q^2\) range is used.

## Method

The authors built a C++ library (using Minuit and ROOT) that generates pseudo-data for the proton electric form factor \(G_E\) from nine different input models (dipole, monopole, Gaussian, Kelly-2004, Arrington-2004, Arrington-2007, Venkat-2011, Bernauer-2014, Alarcon-2017, and Ye-2018). They added bin-by-bin Gaussian fluctuations matching the expected PRad uncertainties (0.02% to 1.1%) plus a 5% overall scaling uncertainty, then fit each pseudo-data set with multiple functional forms (polynomials in \(Q^2\), rational functions, continued fractions, and polynomials in the \(z\)-transformation variable). For each combination of generator and fit function, they repeated the generation, fluctuation, and fitting cycle 150,000 times to obtain distributions of extracted radii and quantify bias and variance.

## Key numbers

* The PRad experiment's expected \(Q^2\) range is \(3\times 10^{-4} < Q^2 < 0.072\) GeV\(^2\).
* Bin-by-bin random uncertainties range from 0.02% to 1.1%.
* An overall scaling uncertainty of 5% was added in the tests.
* For the full PRad range, the rational function with \((N,M)=(1,1)\) extracted the radius with \(\delta R < 0.42\sigma\) for all nine input models.
* The second-order continued fraction and second-order polynomial expansion in \(z\) also showed \(|\delta R| < \sigma\) for all nine input models.
* The dipole fitter showed bias larger than the fitting uncertainty (\(\sigma\)) for most input models; for example, with the dipole generator at \(R(\text{input}) = 0.85\) fm, the monopole and Gaussian fitters gave \(R(\text{mean})\) significantly deviating from the input.
* The first-order polynomial in \(Q^2\) showed \(|\delta R| > \sigma\) for all nine input models.
* The third- and fourth-order polynomials in \(Q^2\) were robust (\(|\delta R| < \sigma\)) but with significantly larger \(\sigma\) than the lower-order robust functions.

## Context

This work addresses the proton radius puzzle, where muonic hydrogen measurements (Pohl 2010, Antognini 2013) give \(R = 0.8409 \pm 0.0004\) fm while CODATA (Mohr 2015) gives \(R = 0.8751 \pm 0.0061\) fm. The authors extend prior efforts by Kraus (2014) showing that polynomial extrapolation can be unreliable, and by Sick (2003, 2017) on continued fraction expansions and higher-order moments. The framework enables an objective, data-driven selection of fitting functions for the PRad experiment and can be applied to other electron scattering experiments with different kinematics and uncertainties.
