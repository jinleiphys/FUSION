# Estimating Inverse Scattering Potentials for n-p System Using Variational Monte Carlo & Neural Networks

> The Riccati-type nonlinear differential equation, also known as the Variable Phase Approach or Phase Function Method, is used to construct local inverse potentials for the \( ^3S_1 \) and \( ^1S_0 \) states of the deuteron. The Morse potential has been optimized by adjusting parameters using the Variational Monte Carlo (VMC) and Multilayer Perceptron (MLP) type Neural Networks (NN). The inverse potentials obtained from VMC and NN show almost identical parameters. In VMC, all three parameters of the Morse potential are varied to obtain the phase shifts, while in NN, the 3D-parameter optimization problem is converted to a 1D-parameter optimization problem, thus reducing optimization parameters, time, and computational cost. Recently, the GRANADA group published a comprehensive partial wave analysis of scattering data, which includes 6713 \( np \) phase shift data points from 1950 to 2013. Using the final experimental data points from GRANADA, we obtained the parameters for the Morse potential by minimizing the mean square error (MSE) as the cost function. The MSE using VMC (NN) is found to be 0.65 (2.5) for the \( ^1S_0 \) state and 0.16 (0.22) for the \( ^3S_1 \) state. Various quantum functions, such as phase \( δ(r) \), amplitude \( A(r) \), and wave function \( u(r) \), are described up to 5 fm with energies \( E_{\ell ab} = [1-350 \text{ MeV}] \).

## Key claim

The paper demonstrates that inverse scattering potentials for the neutron-proton system can be constructed using the Morse potential optimized by either Variational Monte Carlo (VMC) or Multilayer Perceptron (MLP) neural networks, with the neural network approach reducing the original three-parameter optimization to a one-parameter problem. The authors note that the true nuclear potentials are non-local, which has not been accounted for in this work and will be included later for higher partial waves.

## Method

The authors use the Variable Phase Approach (Phase Function Method), a Riccati-type nonlinear differential equation, to compute scattering phase shifts from the Morse potential for the \(^3S_1\) and \(^1S_0\) states of the deuteron. For VMC optimization, all three Morse parameters (\(V_0\), \(r_m\), \(a_m\)) are varied randomly to minimize the mean square error between simulated and experimental phase shifts from the GRANADA database. For the neural network approach, two MLP networks with 4 hidden layers of 200 neurons each are trained on 10,000 generated Morse potentials to predict \(a_m\) and \(r_m\) from \(V_0\) and measured phase shifts, reducing the problem to a one-parameter sweep over \(V_0\).

## Key numbers

- MSE using VMC (NN) for the \(^1S_0\) state: 0.65 (2.5)
- MSE using VMC (NN) for the \(^3S_1\) state: 0.16 (0.22)
- For \(^1S_0\) state: VMC parameters \(V_0 = 70.438\) MeV, \(r_m = 0.901\) fm, \(a_m = 0.372\) fm; NN parameters \(V_0 = 73.5\) MeV, \(r_m = 0.881\) fm, \(a_m = 0.363\) fm
- For \(^3S_1\) state: VMC parameters \(V_0 = 114.153\) MeV, \(r_m = 0.841\) fm, \(a_m = 0.350\) fm; NN parameters \(V_0 = 115.25\) MeV, \(r_m = 0.832\) fm, \(a_m = 0.347\) fm
- Energies covered: \(E_{lab} = [1-350\) MeV]
- Quantum functions described up to 5 fm
- Neural networks trained on 10,000 generated Morse potentials with \(V_0 \in [-250,250]\) MeV, \(a_m \in [0,1]\) fm, \(r_m \in [0,5]\) fm

## Context

This work extends the authors' previous applications of the Variable Phase Approach to n-p scattering (Khachi et al.) and builds on Balassa's earlier work using neural networks for the deuteron \(^3S_1\) ground state inverse problem. The results are compared against the GRANADA group's comprehensive partial wave analysis of 6713 np phase shift data points from 1950 to 2013. The methodology enables future extension to higher partial waves (P, D, F, G, H, I, J) and other scattering systems such as n-alpha, p-alpha, and n-C, with the next objective being determination of non-local potentials for deuteron's higher partial waves.
