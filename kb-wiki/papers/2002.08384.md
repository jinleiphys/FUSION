# Origin of single transverse-spin asymmetries in high-energy collisions

> In this paper we perform the first simultaneous QCD global analysis of data from semi-inclusive deep inelastic scattering, Drell-Yan, $e^+e^-$ annihilation into hadron pairs, and proton-proton collisions. Consequently, we are able to extract a universal set of non-perturbative functions that describes the observed asymmetries in these reactions. The outcome of our analysis indicates single transverse-spin asymmetries in high-energy collisions have a common origin. Furthermore, we achieve the first phenomenological agreement with lattice QCD on the up and down quark tensor charges.

## Key claim
Single transverse-spin asymmetries (SSAs) in semi-inclusive deep inelastic scattering, Drell-Yan, e+e- annihilation, and proton-proton collisions share a common dynamical origin in multi-parton correlations, as demonstrated by the first simultaneous QCD global analysis of all four reaction types. The analysis achieves phenomenological agreement with lattice QCD on the up and down quark tensor charges, but this agreement depends on including the proton-proton collision data in the global fit.

## Method
The authors performed a simultaneous QCD global analysis of SSA data from semi-inclusive deep inelastic scattering (SIDIS), Drell-Yan (DY) processes, e+e- annihilation into hadron pairs (SIA), and proton-proton collisions, using a Monte Carlo framework to sample the Bayesian posterior distribution for the parameters. They parametrized transverse momentum dependent (TMD) parton distribution functions and fragmentation functions with a Gaussian ansatz for the transverse momentum dependence, and expressed the collinear twist-3 functions (transversity h1, Qiu-Sterman function FFT, and Collins first moment H1⊥(1)) using a common functional form. The analysis used leading-order DGLAP evolution for collinear functions and a double-logarithmic Q2-dependent term for the collinear twist-3 functions.

## Key numbers
- Total chi-squared per number of data points for SSA data alone: χ²/Npts = 520/517 = 1.01
- Total chi-squared per number of data points for all data including HERMES multiplicities: χ²/Npts = 1373/1324 = 1.04
- Extracted isovector tensor charge gT = δu - δd = 0.87(11) at Q² = 4 GeV², the most precise phenomenological determination to date
- Extracted up quark tensor charge δu = 0.72(19) at Q² = 4 GeV²
- Extracted down quark tensor charge δd = -0.15(16) at Q² = 4 GeV²
- For SIDIS Sivers asymmetry: χ²/Npts = 150.0/126 = 1.19
- For SIDIS Collins asymmetry: χ²/Npts = 111.3/126 = 0.88
- For SIA Collins asymmetry: χ²/Npts = 154.5/176 = 0.88
- For DY Sivers from pion-proton collisions: χ²/Npts = 5.96/12 = 0.50
- For DY Sivers from proton-proton W/Z production: χ²/Npts = 31.8/17 = 1.87
- For proton-proton single-spin asymmetry ANπ: χ²/Npts = 66.5/60 = 1.11
- The isovector tensor charge evolved from gT = 1.4(6) using only SIDIS data to gT = 0.87(25) adding SIA data, to gT = 0.87(11) in the full global fit

## Context
This work extends previous phenomenological extractions of TMDs and collinear functions by groups such as Anselmino (2013), Echevarria (2014), Kang (2015), and Radici (2018), but is the first to simultaneously fit all four SSA reaction types (SIDIS, DY, SIA, and proton-proton collisions) in a single global analysis. The results resolve the "sign-mismatch" puzzle between the Sivers function and Qiu-Sterman function found in earlier work by Kang (2011). The extracted tensor charges agree with lattice QCD computations at the physical point by Gupta (2018), Hasan (2019), and Alexandrou (2019), and with Dyson-Schwinger calculations by Pitschmann (2014). Future data from JLab-12 GeV, COMPASS, an upgraded RHIC, Belle II, and the Electron-Ion Collider will test and refine these results.
