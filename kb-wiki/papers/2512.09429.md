# Determination of nuclear deformations with an emulator for sub-barrier fusion reactions

> Based on the eigenvector continuation, which is mathematically an instance of the reduced basis method (RBM), we construct an emulator for coupled-channels calculations for heavy-ion fusion reactions at energies around the Coulomb barrier. We apply this to the $^{16}$O+$^{144,154}$Sm, $^{186}$W reactions and examine whether the emulator can be used to extract the deformation parameters of the target nuclei. We show that the emulator not only accelerates the calculations but also has an ability to accurately extract the nuclear shapes. This indicates that the emulator provides a powerful tool to systematically explore intrinsic shapes of atomic nuclei, enhancing our understanding of the fundamental properties of nuclear systems.

## Key claim

The authors demonstrate that an eigenvector continuation emulator can accurately reproduce full coupled-channels calculations for heavy-ion fusion reactions at sub-barrier energies, and can be used to extract nuclear deformation parameters from experimental fusion cross sections. The hedge is that the extracted deformation parameters should be treated as "effective values" because the coupled-channels Hamiltonian is inherently incomplete, and the optimization process may compensate for model defects such as omitted transfer channels or higher-order deformation effects.

## Method

The authors construct an emulator using eigenvector continuation (a reduced basis method) applied to the discrete basis formalism of coupled-channels calculations for heavy-ion fusion. They solve the coupled-channels equations using the CCFULL model for the 16O+144,154Sm, 186W reactions, first obtaining high-fidelity scattering wave functions at selected training points (snapshots), then projecting the full Hamiltonian onto the subspace formed by these solutions to build the emulator. The emulator rapidly computes fusion cross sections across the parameter space, and the results are compared to experimental data via a chi-squared analysis to extract the optimum deformation parameters.

## Key numbers

* For the 16O+144Sm reaction, the chi-squared minimum appears at beta_3 = 0.21, in excellent agreement with the value estimated from the measured B(E3) value.
* The emulator used N_EC = 5 training points at beta_3 = 0.05, 0.10, 0.15, 0.20, 0.25 for the 16O+144Sm system.
* The Woods-Saxon potential parameters used were V_0 = 105.1 MeV, r_0 = 1.10 fm, and a = 0.75 fm.
* The coordinate was discretized from r_min to r_max = 30 fm with mesh spacing Delta r = 0.05 fm.
* The first 3- state in 144Sm is at 1.81 MeV, and the dynamical deformation parameter beta_3 is estimated to be 0.21 from the measured B(E lambda) value.
* The barrier distribution was extracted using the point difference formula with Delta E_cm = 1.8 MeV.

## Context

This work extends the authors' previous emulator development for a simple one-dimensional coupled-channels model (Liao et al., 2024) to realistic three-dimensional scattering problems using the CCFULL code (Hagino, 1999). The approach is similar to prior applications of eigenvector continuation to nuclear structure problems (e.g., Konig, 2020; Drischler, 2021) and nuclear reaction problems (e.g., Furnstahl, 2020; Melendez, 2022). The emulator enables systematic exploration of nuclear shapes from fusion data, which previously required computationally expensive iterative coupled-channels calculations, as was done for 24Mg and 28Si using quasi-elastic scattering (Gupta, 2020, 2023).
