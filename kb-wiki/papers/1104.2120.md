# Optimal design of measurement network for neutronic activity field reconstruction by data assimilation

> Using data assimilation framework, to merge information from model and measurement, an optimal reconstruction of the neutronic activity field can be determined for a nuclear reactor core. In this paper, we focus on solving the inverse problem of determining an optimal repartition of the measuring instruments within the core, to get the best possible results from the data assimilation reconstruction procedure. The position optimisation is realised using Simulated Annealing algorithm, based on the Metropolis-Hastings one. Moreover, in order to address the optimisation computing challenge, algebraic improvements of data assimilation have been developed and are presented here.

## Key claim

The paper demonstrates that simulated annealing can optimize the placement of 50 neutron detectors in a 157-assembly PWR900 core to improve neutronic activity field reconstruction via data assimilation, but only under the idealized assumption of no operational constraints (such as incompatibility with control rods). The authors note that the standard PWR900 configuration, while designed without data assimilation, already achieves excellent reconstruction quality.

## Method

The authors used a twin-experiment methodology with synthetic data generated from the COCAGNE neutronic code, representing the core with 157 assemblies and 29 vertical levels (control vector size 4553). They applied the Metropolis-Hastings simulated annealing algorithm with 1800 iterations and an inverse temperature schedule (T_i = T_0/(i+1), T_0 = 0.05) to optimize instrument positions, using algebraic fast-calculation methods to treat instrument swaps as perturbations rather than full recalculations.

## Key numbers

- Core configuration: 157 nuclear fuel assemblies in a 900 MWe PWR, with 50 instrumented positions (Mobile Fission Chambers)
- Control vector size: 4553 (157 assemblies x 29 vertical levels)
- Observation vector size: 1450 (50 instruments x 29 vertical levels)
- Quality factor q for standard PWR900 configuration: -0.0476
- Mean quality q for random 50-instrument distributions: -0.0832 (standard deviation 0.0078, from 1000 random distributions)
- Mean quality q for random 10-instrument distributions: -0.0868 (standard deviation 0.076)
- Final quality q after optimization from random start with 50 instruments: -0.0305
- Final quality q after optimization from random start with 10 instruments: -0.0504
- Final quality q after optimization from PWR900 start: approximately -0.0325 (dispersion around 0.002, or about 5%)
- Algorithm parameters: 1800 iterations maximum, initial temperature T_0 = 0.05
- Computing time: approximately 10 hours per search per processor

## Context

This work extends prior demonstrations of data assimilation for nuclear core activity reconstruction by Bouriquet et al. (2010, 2011) and Massart (2007), moving from fixed measurement networks to optimizing instrument placement. The simulated annealing approach, originally developed by Metropolis (1953) and Hastings (1970) for statistical physics, had previously been applied to nuclear incident network design by Abida et al. (2008, 2009). The results enable future work on constrained optimization (e.g., symmetry requirements, control rod incompatibility) and optimization with heterogeneous instrument types.
