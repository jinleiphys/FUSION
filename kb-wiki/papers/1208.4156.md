# Coulomb-Sturmian basis for the nuclear many-body problem

> Calculations in ab initio no-core configuration interaction (NCCI) approaches, such as the no-core shell model or no-core full configuration methods, have conventionally been carried out using the harmonic-oscillator many-body basis. However, the rapid falloff (Gaussian asymptotics) of the oscillator functions at large radius makes them poorly suited for the description of the asymptotic properties of the nuclear wave function. We establish the foundations for carrying out NCCI calculations with an alternative many-body basis built from Coulomb-Sturmian functions. These provide a complete, discrete set of functions with a realistic exponential falloff. We present illustrative NCCI calculations for 6Li with a Coulomb-Sturmian basis and investigate the center-of-mass separation and spurious excitations.

## Key claim

The paper establishes the foundations for using Coulomb-Sturmian functions as an alternative many-body basis in no-core configuration interaction (NCCI) calculations, claiming these functions provide a complete, discrete set with realistic exponential falloff (unlike the Gaussian asymptotics of harmonic oscillator functions), but the actual implementation requires overcoming technical challenges related to transformation of matrix elements and center-of-mass factorization.

## Method

The authors define Coulomb-Sturmian single-particle basis functions with a constant length scale parameter \(b\), replacing the quantum-number-dependent scale of standard Coulomb functions. They then construct the many-body basis from antisymmetrized products of these single-particle states and perform NCCI calculations for \(^{6}\text{Li}\) using a realistic nucleon-nucleon interaction, comparing convergence properties with conventional harmonic-oscillator basis calculations of the same dimensionality. The transformation of two-body matrix elements from the oscillator basis to the Coulomb-Sturmian basis is addressed, and the evaluation of relative kinetic energy matrix elements via separability is demonstrated.

## Key numbers

- The Coulomb-Sturmian functions are defined with normalization factor \(N_{nl} = (2/b)^{3/2} [n!/(2(n+l+1)(n+2l+1)!)]^{1/2}\), where \(b\) is the length scale parameter.
- The orthogonality relation is \(\int d^3\vec{r}\, \Phi_{n'l'm'}^*(\vec{r}) (1/r) \Phi_{nlm}(\vec{r}) = [1/(b(n+l+1))] \delta_{(n'l'm')(nlm)}\).
- Illustrative NCCI calculations are performed for the nucleus \(^{6}\text{Li}\) (six nucleons).

## Context

This work extends the no-core configuration interaction (NCCI) approach, previously based almost exclusively on harmonic oscillator bases (Navratil 2000), by introducing Coulomb-Sturmian functions that have been applied in atomic physics (Hylleraas 1928, Rotenberg 1962) and hadronic physics (Jacobs 1986, Fulcher 1993). The paper addresses the long-recognized limitation (Davies 1966) that oscillator functions have Gaussian falloff while nuclear wave functions require exponential asymptotics, and it enables future NCCI calculations with improved convergence for observables sensitive to large-radius properties, such as root-mean-square radii and E2 strengths.
