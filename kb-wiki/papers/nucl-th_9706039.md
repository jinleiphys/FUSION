# Antisymmetrization in the Multicluster Dynamic Model of Nuclei and the Nucleon Exchange Effects

> A modified version of the Multicluster Dynamic Model of nuclei is proposed to construct completely antisymmetrized wave functions of multicluster systems. An overlap kernel operator is introduced to renormalize the total wave function after antisymmetrization between nucleons in different clusters. A group-theoretical method is developed to analyze the role of the exchange effects arising in the calculation of the various observables of multicluster systems due to this antisymmetrization. The Antisymmetrized version of the Multicluster Dynamic Model is applied to the six-nucleon systems treating them as alpha-2N ones.The static and dynamic characteristics of the six-nucleon systems manifested in electron and pi-meson scattering, muon capture, beta-decay, pion photoproduction, etc., are calculated.

## Key claim

The paper proposes a modified version of the Multicluster Dynamic Model (MDM) that constructs completely antisymmetrized wave functions for multicluster systems by introducing an overlap kernel operator for renormalization, rather than the simpler constant renormalization used in earlier work. The authors claim that this kernel renormalization (AMDM_K) is more correct than constant renormalization (AMDM_C) because it restores orthogonality between different antisymmetrized wave functions and does not alter the asymptotic part of the relative motion wave function where antisymmetrization should have no influence.

## Method

The authors start from the MDM wave function for a k-cluster system, which uses cluster-cluster potentials derived from phase-shift analysis and treats cluster internal wave functions as known from experiment. They then apply the full antisymmetrizer to this wave function and introduce an overlap kernel operator K that acts in the Hilbert space of the (3k-3)-dimensional orbital functions, with its eigenfunctions being many-body oscillator functions characterized by SU(3) symmetry. ## Context

This work extends the Multicluster Dynamic Model (MDM) developed in Refs. [fun92, be9, zhukov] and the earlier Antisymmetrized version (AMDM_C) from Ref. [amdm, physlet] by introducing a more rigorous kernel renormalization procedure. The approach contrasts with the Microscopic Cluster Model (MCM) and Resonating Group Method (RGM) from Refs. [lang, varga1, varga2, svm], which use realistic NN potentials but produce complicated nonlocal cluster-cluster potentials. The AMDM_K enables simultaneous description of static and dynamic observables for six-nucleon systems across electromagnetic and weak interaction processes (electron scattering, pion scattering, muon capture, beta-decay, pion photoproduction), treating the six-nucleon system as a "theoretical laboratory" for testing new approaches.
