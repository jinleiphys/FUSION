# Extracting Source Parameters from Gaussian Fits to Two-Particle Correlations

> Using a quadratic saddle-point approximation, we show how information about a particle-emitting source can be extracted from gaussian fits to two-particle correlation data. Although the formalism is completely general, extraction of the relevant parameters is much simpler for sources within an interesting class of azimuthally symmetric models. After discussing the standard fitting procedure, we introduce a new gaussian fitting procedure which is an azimuthally symmetric generalization of the Yano-Koonin formalism for spherically symmetric sources. This new fitting procedure has the advantage that in addition to being able to measure source parameters in a fixed frame or the longitudinally co-moving system, it can also measure these parameters in the local rest frame of the source.

## Key claim

Using a quadratic saddle-point approximation, the authors show that for a broad class of azimuthally symmetric sources, a generalized Yano-Koonin (GYK) fitting procedure can extract source parameters in the local rest frame of the source, whereas standard fitting procedures require a priori knowledge of the correct longitudinal reference frame to yield interpretable results.

## Method

The authors derive a Lorentz-covariant expression for the two-particle correlation function using a quadratic saddle-point approximation to a general source function S(x,K), which yields a gaussian correlation function in terms of the inverse curvature tensor (B^{-1})_{μν}. They then compare two fitting procedures for azimuthally symmetric sources: a standard three-dimensional gaussian fit in the spatial components of the momentum difference q, and a generalized Yano-Koonin (GYK) fit that includes q_0 and is boost-invariant along the longitudinal direction. The formalism is applied to three example source models: a static gaussian source, a boost-invariant longitudinally expanding source, and a finite expanding source with a gaussian cutoff in spacetime rapidity.

## Key numbers

- For a static gaussian source in its center-of-mass frame, the standard fit yields R_1^2 = R^2 + β_1^2(Δt)^2, R_2^2 = R^2, R_3^2 = L^2 + β_3^2(Δt)^2, and R_{13}^2 = β_1β_3(Δt)^2, where R, L, and Δt are the transverse, longitudinal, and temporal source extensions.
- For a boost-invariant source in the LCMS frame, the standard fit gives R_1'^2 = R^2 + β_1^2(Δτ)^2, R_2'^2 = R^2, R_3'^2 = (T/m_t)τ_0^2, and R_{13}'^2 = 0, where τ_0 is the freezeout proper time, Δτ is the duration of emission, and T is the freezeout temperature.
- For the boost-invariant source, the GYK fit yields R_t = R, R_4 = τ_0√(T/m_t), and R_0 = Δτ regardless of the longitudinal measurement frame, while v measures the velocity difference between the measurement frame and the local rest frame.
- For the finite expanding source, the saddle point in spacetime rapidity η̄ is given by the solution of th(η̄-Y) - (m_t/T) sh(η̄-Y) - η̄/(Δη)^2 = 0, where Δη is the width of the rapidity distribution.

## Context

This work extends the Yano-Koonin formalism (Yano and Kochel, 1978; Zajc, 1987) from spherically symmetric sources to azimuthally symmetric sources, and generalizes earlier saddle-point approximation work by Sinyukov (1981), Akkelin and Sinyukov, and Chapman et al. (1994). The paper enables experimentalists to extract source parameters such as freezeout proper time τ_0, transverse source size R, and emission duration Δτ from HBT correlation data without needing to guess the correct longitudinal reference frame beforehand, which is particularly relevant for heavy-ion collision experiments at the AGS and SPS. The formalism also provides a consistency test: if the left and right sides of the relations connecting standard and GYK fit parameters disagree, the source does not belong to the class of models with λ_1=λ_2 and vanishing x-t and x-z correlations.
