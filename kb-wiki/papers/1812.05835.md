# Schwinger pair production and string breaking in non-Abelian gauge theory from real-time lattice improved Hamiltonians

> Far-from-equilibrium dynamics of SU(2) gauge theory with Wilson fermions is studied in 1+1 space-time dimensions using a real-time lattice approach. Lattice improved Hamiltonians are shown to be very efficient in simulating Schwinger pair creation and emergent phenomena such as plasma oscillations. As a consequence, significantly smaller lattices can be employed to approach continuum physics in the infinite-volume limit as compared to unimproved implementations. This allows us to compute also higher-order correlation functions including four fermion fields, which give unprecedented insights into the real-time dynamics of the fragmentation process of strings between fermions and antifermions.

## Key claim

Lattice improved Hamiltonians up to next-to-next-to-leading order (NNLO) in the fermion sector can dramatically reduce the lattice sizes required to approach continuum and infinite-volume physics for real-time simulations of non-Abelian Schwinger pair production and string breaking, though the improvement is most dramatic at NLO with only minor additional gains at NNLO.

## Method

The authors simulate SU(2) gauge theory with Wilson fermions in 1+1 dimensions using classical-statistical reweighting techniques on a real-time lattice, solving the equations of motion for gauge fields and fermion mode functions via a four-step Runge-Kutta algorithm. They implement lattice improved Hamiltonians at leading order (LO, O(a) accuracy), next-to-leading order (NLO, O(a^3) accuracy), and next-to-next-to-leading order (NNLO, O(a^5) accuracy) by choosing appropriate coefficients C_k in the fermion Hamiltonian. Fermion numbers are computed using both an Abelianized definition (for homogeneous fields) and a gauge-invariant Wigner function approach with Wilson lines.

## Key numbers

- Fermion production rate per unit length per diagonalized color direction: \(\dot{n}^\pm/(L m^2) = (\epsilon/4\pi) \exp(-2\pi/\epsilon)\), verified against simulations for background field strengths \(\epsilon^1\) from 2 to 10
- Lattice parameters used for benchmark: \(N_1 = 512\), \(L = 25.6/m\), \(a = 0.05/m\), \(a_t = 0.05 a\) at NLO
- For convergence studies: \(L = 25.6/m\), \(a = L/N_1\), \(a_t = 0.02 a\), background field \(\epsilon = (2,0,0)\)
- NLO and NNLO improvements converge to continuum results for \(N_1 \gtrsim 250\); around \(N_1 \simeq 100\) NLO slightly overestimates and NNLO slightly underestimates the continuum result
- Coupling constant fixed at \(g/m = 0.1\) for homogeneous field benchmarks

## Context

This work extends prior studies of Abelian Schwinger pair production and string breaking in 1+1 dimensions using classical-statistical reweighting (Hebenstreit 2013) and staggered fermions (Kasper 2014) to the non-Abelian SU(2) case with Wilson fermions. It builds on lattice improvement techniques previously applied to quantum anomalies in QCD (Mueller 2016, Mace 2016) and QED (Mueller 2016). The results enable more efficient real-time simulations of non-Abelian gauge theory dynamics on smaller lattices, which is crucial for near-term quantum simulator implementations following the proof-of-principle trapped-ion experiment (Martinez 2016).
