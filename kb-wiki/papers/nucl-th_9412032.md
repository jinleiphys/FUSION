# Determination of bound state wave functions by a genetic algorithm

> We apply a stochastic method of minimizing the ground state energy in variational calculations of light nuclei using the Refined Resonating Group Model (RRGM). The method utilizes a bit representation of the width parameters to be varied. To find the best possible set of width parameters we use strategies familiar from biological evolution. Very complicated problems can be solved in this way because the method is intrinsically parallel. The algorithm can be used on parallel computers with any number of processors without any change.

## Key claim

A genetic algorithm can find the ground state binding energy of light nuclei in the Refined Resonating Group Model (RRGM) using only the range of physically sensible width parameters as input, whereas traditional deterministic gradient search methods require good starting values to yield reasonable performance.

## Method

The authors represent each of the width parameters in the RRGM variational wave function as a 16-bit binary number (gene), and a full set of parameters for a given nucleus forms an individual. An initial population of 50 individuals with random genes is created, and the binding energy is evaluated for each. Successive generations are produced by selecting two individuals with low binding energies, performing uniform crossover on their bit representations to create two offspring, mutating each bit with probability 0.001, evaluating the offspring binding energies, and inserting them into the population while discarding the worst individuals to maintain constant population size.

## Key numbers

- Population size of 50 individuals used for all calculations
- Mutation rate of 0.001 applied per bit
- 16-bit binary representation used for each width parameter
- 5 different width parameters were varied for the ⁷Li test case
- Total of 500 generations were computed per run
- The genetic algorithm needed about twice as many steps (twice the CPU time) as a deterministic sequential gradient method for the ⁷Li test case
- Running on 50 processors, the genetic algorithm converged dramatically faster in real time than the deterministic method

## Context

This work extends the Refined Resonating Group Model calculations of Hofmann (1984, 1987) for bound states and scattering observables of light nuclei. The genetic algorithm approach addresses the difficulty of finding minima in the highly nonlinear space of width parameters, which becomes problematic for complex nuclei with many parameters. The method contrasts with the approach of Varga, Suzuki and Lovas (1994) of simply extending the number of linear combinations, which leads to unphysical models where scattering calculations become almost impossible. The algorithm enables calculation of ground states of complicated nuclei in much shorter real time through its intrinsic parallelism and scalability.
