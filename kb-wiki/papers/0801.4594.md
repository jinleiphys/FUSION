# Lightlike Wilson loops from AdS/CFT

> We investigate the lightlike limit of stationary spacelike string configurations on a large class of five-dimensional asymptotically AdS backgrounds. Specific examples include gravity duals which incorporate finite 't Hooft coupling, curvature-squared corrections, and chemical potentials. A universal feature of these AdS/CFT models is that the string solution with minimum action yields a lightlike Wilson loop whose leading behavior is exponentially linear, rather than quadratic, in the width of the loop. Unless there is a compelling reason for discarding the leading saddlepoint contribution to the Wilson loop, following the proposal of Liu et. al. [2] leads to zero jet-quenching parameter for all of these models.

## Key claim

The minimum-action string configuration for lightlike Wilson loops in a large class of asymptotically AdS backgrounds yields a leading behavior that is exponentially linear in the loop width L, rather than quadratic; following the Liu et al. (2006) proposal for extracting the jet-quenching parameter from the L^2 coefficient would therefore give zero jet-quenching parameter for all these models, unless the leading saddlepoint contribution is discarded.

## Method

The authors compute the Nambu-Goto action for stationary spacelike string configurations on five-dimensional asymptotically AdS backgrounds with an event horizon, using the embedding t = τ, x1 = vτ, x2 = σ, x3 = 0, r = r(σ) for a quark-antiquark pair moving perpendicular to their separation. They analyze the lightlike limit (v → 1, probe brane radius r7 → ∞) for two classes of solutions: "long strings" with turning point at the horizon and "short strings" with turning point approaching the probe brane, and evaluate the regularized action by subtracting the action of two straight strings extending from the horizon to the probe brane.

## Key numbers

- For the short string on a neutral AdS black hole, the regularized action is S_r = (T√λ/β)(-1.31 + (π/2)(L/β)), where β is the inverse temperature, λ the 't Hooft coupling, and T the time interval; this is exact with no L^2 term.
- For the long string on a neutral AdS black hole, the regularized action is S_r = (T√λ/β)(0.941 L^2/β^2 + O(L^4)).
- For general asymptotically AdS backgrounds, the long string action scales as S_r ∝ L^2 with coefficient (T√λ)/(8πR^2) I^{-1}, where I = ∫_{r_h}^{∞} dr (c_R)/(c_X^2 √(c_X^2 - c_T^2)).
- For general asymptotically AdS backgrounds, the short string action in the lightlike limit is S_r = (T√λ)/(πR^2)[ -∫_{r_h}^{∞} dr c_R√(c_X^2 - c_T^2) + (1/2)(c_X√(c_X^2 - c_T^2))|_{r=r7} L ].

## Context

This work extends the analysis of Argyres (2007) on the minimum-action string configurations for lightlike Wilson loops in N=4 SYM to a broad class of asymptotically AdS backgrounds, including those with finite 't Hooft coupling corrections (α' corrections), curvature-squared corrections, and chemical potentials (three-charge AdS black holes). It directly challenges the prescription of Liu et al. (2006) for defining the jet-quenching parameter from the L^2 term in the Wilson loop, showing that the dominant saddlepoint gives a linear rather than quadratic dependence on L. The results enable further investigation into whether the leading saddlepoint should be discarded on physical grounds, and whether the linear dependence indicates a different energy-loss mechanism at strong coupling.
