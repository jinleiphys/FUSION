# A Dynamical Principle For 3D-4D Interlinkage In Salpeter-like Equations

> The half-century old Markov-Yukawa Transversality Principle ($MYTP$) which provides a theoretical rationale for the covariant instantaneous approximation ($CIA$) that underlies all Salpeter-like equations, is generalized to a covariant null-plane ansatz ($CNPA$). A common characteristic of both formulations is an exact 3D-4D interlinkage of BS amplitudes which facilitates a two-tier description: the 3D form for spectroscopy, and the 4D form for transition amplitudes as 4D loop integrals. Some basic applications of $MYTP$ on the covariant null plane (quark mass function, vacuum condensates, and decay constants) are given on the lines of earlier applications to these processes under $CIA$. ## Key claim

The Markov-Yukawa Transversality Principle (MYTP), which provides a theoretical rationale for the covariant instantaneous approximation (CIA) underlying all Salpeter-like equations, is generalized to a covariant null-plane ansatz (CNPA) that cures the "Lorentz mismatch" disease causing ill-defined 4D loop integrals in triangle and higher loops, while preserving the exact 3D-4D interlinkage of Bethe-Salpeter amplitudes.

## Method

The authors impose the MYTP transversality condition on the Bethe-Salpeter kernel by demanding it depends only on the relative momentum component transverse to the total hadron 4-momentum, but now define the longitudinal component on a covariant null-plane using the orientation vector n_μ rather than in the instant form. For fermion quarks, they employ a Gordon-reduced modification of the Bethe-Salpeter equation to simplify the Dirac structure, then reconstruct the 4D vertex function from the 3D wave function using the same D_n function formalism as in the CIA case. The CNPA formulation is applied to calculate the quark mass function via dynamical chiral symmetry breaking, vacuum condensates, and the π⁰ → 2γ decay amplitude.

## Context

This work extends the MYTP framework previously developed for the covariant instantaneous approximation (CIA) by Mitra and collaborators (1990s) to a covariant null-plane formulation, addressing the Lorentz mismatch problem identified in triangle loop calculations such as the pion form factor. The approach builds on the Dirac-Weinberg light-front dynamics developed by Kadychevsky (1968), Karmanov (1976), and the Grenoble group, and provides a systematic way to compute transition amplitudes as 4D loop integrals while maintaining the spectroscopic advantages of 3D reduction. The formalism enables consistent calculation of quark mass functions, vacuum condensates, and decay constants within a unified framework that preserves the 3D-4D interlinkage of Bethe-Salpeter amplitudes.
