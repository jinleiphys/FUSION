# Exploiting symmetries in nuclear Hamiltonians for ground state preparation

> The Lipkin and Agassi models are simplified nuclear models that provide natural test beds for quantum simulation methods. Prior work has investigated the suitability of the Variational Quantum Eigensolver (VQE) to find the ground state of these models. There is a growing awareness that if VQE is to prove viable, we will need problem inspired ansätze that take into account the symmetry properties of the problem and use clever initialization strategies. Here, by focusing on the Lipkin and Agassi models, we investigate how to do this in the context of nuclear physics ground state problems. We further use our observations to discus the potential of new classical, but quantum-inspired, approaches to learning ground states in nuclear problems.

## Key claim
By encoding the full permutation symmetry of the Lipkin model and the combined permutation and inversion symmetries of the Agassi model into the Hamiltonian Variational Ansatz (HVA), the cost function variances no longer decay exponentially with system size, but instead increase or decay at worst polynomially, thereby avoiding the barren plateau problem that plagues unstructured ansatze.

## Method
The authors apply the Hamiltonian Variational Ansatz (HVA) to the Lipkin and Agassi models, correlating the gate parameters within each layer to enforce the exact symmetries of each Hamiltonian (permutation symmetry for Lipkin; permutation and magnetisation-inversion symmetries for Agassi). They compute the variance of the VQE cost function (energy expectation value) over random parameter initialisations for increasing system sizes, comparing the symmetrised ansatz to an unsymmetrised version with independent parameters. For the Agassi model, they also perform gradient-based optimisation using the ADAM optimiser, both with random initialisations and with a warm-start strategy that transfers optimised parameters from smaller system sizes.

## Key numbers
- For the Lipkin model with the unsymmetrised ansatz, cost function variances decay exponentially with system size; for the fully permutationally symmetric ansatz, variances increase with system size (raw energy) or do not significantly decrease (normalised cost bounded between 0 and 1).
- For the Agassi model with the symmetrised HVA, the normalised cost function variance decays polynomially as \(n^{-1.6}\) with system size (where \(n = 4j\) is the number of qubits).
- The HVA for the Agassi model requires only \(3j\) parameters (linearly scaling with maximum spin \(j\)) and a linear circuit depth to prepare the ground state to within 1% error.
- Warm-starting for the Agassi model reduces the mean energy at optimisation termination by 85.12% (j=2), 30.08% (j=3), and 34.29% (j=4), and reduces standard deviations by 94.05%, 42.05%, and 63.85% respectively.

## Context
This work extends prior VQE studies of the Lipkin model using Bethe-inspired ansatze (PhysRevA.104.022412), Unitary Coupled Cluster (Chikaoka 2022), and Hamiltonian-Learning VQE (PhysRevC.108.024313), as well as digital implementations of the Agassi model (Perez 2022). It directly builds on theoretical results linking cost function variances to Dynamical Lie Algebra scaling (Ragone 2023, Fontana 2023) and on geometric quantum machine learning for encoding symmetries (Larocca 2022). The finding that the symmetrised Agassi model likely has a polynomially scaling DLA implies it may be efficiently simulable by classical Lie-algebraic methods (Goh 2023, Cerezo 2023), opening quantum-inspired classical approaches for nuclear Hamiltonians. The authors suggest that symmetrised solutions from these simplified models could provide warm-start initialisations for more realistic shell model calculations, for example along isotopic chains such as oxygen isotopes (Brown 2017).
