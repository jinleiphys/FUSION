# A first estimation of chiral four-nucleon force effects in 4He

> We estimate four-nucleon force effects between different 4He wave functions by calculating the expectation values of four-nucleon potentials which were recently derived within the framework of chiral effective field theory. We find that the four-nucleon force is attractive for the wave functions with a totally symmetric momentum part. The additional binding energy provided by the long-ranged part of the four-nucleon force is of the order of a few hundred keV.

## Key claim

The four-nucleon force in 4He is attractive for wave functions with a totally symmetric momentum part, and the additional binding energy from its long-ranged part is on the order of a few hundred keV, but the actual effect may vary from a few tens of keV to 1–2 MeV depending on the low-energy constant \(C_T\) and the wave function used.

## Method

The authors computed expectation values of the leading chiral four-nucleon (4N) potential, derived in chiral effective field theory, using several 4He wave functions. They assumed a totally symmetric momentum part for the wave function and used a Gaussian model wave function as well as two wave functions obtained from rigorous solutions of the 4N Faddeev-Yakubovsky equations with NLO chiral potentials, with the spin-isospin part taken as the totally antisymmetric state. The 18-dimensional momentum-space integrals were evaluated via Monte Carlo integration with \(10^9\) sampling points, and a regulator function \(g_{\Lambda_4}(x) = \exp[-(m x / 2 \Lambda_4^2)^3]\) was applied.

## Key numbers

- For the Gaussian wave function with regulator \(\Lambda_4 = 500\) MeV/c, the expectation values of the individual 4N force parts (in MeV, before multiplying by factor 24) are: \(V^a = -0.002906\), \(V^c = -0.005557\), \(V^e = -0.008462\), \(V^f = 0.005692\), \(V^k = 0.0005925\, C_T\), \(V^l = 0.000622657\, C_T\), \(V^n = -0.000046044\, C_T^2\).
- For the chiral NLO wave function with \((\Lambda, \tilde{\Lambda}) = (400\ \text{MeV/c}, 500\ \text{MeV/c})\) and \(\Lambda_4 = 500\) MeV/c: \(V^a = -0.00434503\) MeV, \(V^c = -0.0084033\) MeV, \(V^e = -0.0133568\) MeV, \(V^f = 0.00914028\) MeV, \(V^k = 0.000926931\, C_T\) MeV, \(V^l = 0.000964454\, C_T\) MeV, \(V^n = -0.00007243\, C_T^2\) MeV (all before factor 24; wave function norm = 1.093).
- For the chiral NLO wave function with \((\Lambda, \tilde{\Lambda}) = (550\ \text{MeV/c}, 500\ \text{MeV/c})\) and \(\Lambda_4 = 500\) MeV/c: \(V^a = -0.00222788\) MeV, \(V^c = -0.00445691\) MeV, \(V^e = -0.00683624\) MeV, \(V^f = 0.00460722\) MeV, \(V^k = 0.000489232\, C_T\) MeV, \(V^l = 0.000512526\, C_T\) MeV, \(V^n = -0.00003812\, C_T^2\) MeV (all before factor 24; wave function norm = 1.011).
- For \(C_T \approx 13\ \text{GeV}^{-2}\), the total expectation values reach a minimum of approximately -0.077 MeV (Gaussian), -0.107 MeV (400,500 wave function), and -0.061 MeV (550,500 wave function).
- For \(C_T = 0\), the total expectation values are -0.270 MeV (Gaussian), -0.386 MeV (400,500 wave function), and -0.219 MeV (550,500 wave function).
- The totally symmetric spin-isospin component constitutes 94.3% of the (400,500) wave function and 88.7% of the (550,500) wave function; components with non-zero angular momenta represent only 0.3% and 2.9% of the respective full wave functions.

## Context

This work provides the first explicit estimate of four-nucleon force effects in 4He, extending the derivation of the leading chiral 4N potential by Epelbaum et al. (2006) to a bound-state calculation. The results agree qualitatively with modern nuclear force predictions for the alpha particle, which leave little room for 4N force contributions. The study enables future fully consistent calculations at NNNLO, which are expected to reduce the strong dependence on the low-energy constant \(C_T\).
