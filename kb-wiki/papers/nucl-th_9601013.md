# Dimensional Regularization and Nuclear Potentials

> It is shown how nucleon-nucleon potentials can be defined in N dimensions, using dimensional regularization to continue amplitudes. This provides an easy way to separate out contact ($δ$-function) terms arising from renormalization. An example is worked out several ways for the case of two scalar particles exchanged between nucleons, which involves a very simple loop calculation. This leads to a Feynman-parameterized representation for the nucleon-nucleon potential. Alternately, a dispersion representation can be developed leading to a different, though equivalent, form.

## Key claim

The paper demonstrates that nucleon-nucleon potentials can be defined in N dimensions using dimensional regularization, providing a clean separation of contact (delta-function) terms that arise from renormalization; the key hedge is that the divergent contact terms are "regularized away" only when the internucleon separation r is kept finite, and they would reappear if r were taken to zero.

## Method

The author computes the two-boson-exchange potential (TBEP) for two scalar particles exchanged between nucleons, using both old-fashioned second-order perturbation theory and Feynman diagram techniques. The loop integral is evaluated in N dimensions using dimensional regularization, and the potential is defined via Fourier transform in n = N-1 spatial dimensions. Two equivalent representations are derived: a Feynman-parameterized form and a dispersion representation exploiting the analytic properties of the amplitude.

## Key numbers

- The potential for equal masses m_a = m_b = m in 4 dimensions is V_12(r) = -4m λ² / [(4π)³ r²] K_1(2mr), where K_1 is a modified Bessel function.
- For N = 4, the Feynman-parameterized potential is V_4 = -2λ² / [(4π)³ r³] ∫₀¹ dz e^{-βr} (1 + βr), with β²(z) = m² / [z(1-z)] for equal masses.
- The small-r behavior of the N-dimensional potential scales as V_N(r) ~ 1/r^(2N-5), becoming progressively more singular for increasing N.
- The dispersion representation (eq. 16) reduces to eq. (4c) for N = 4, which involves an integral over x from m_a + m_b to infinity.

## Context

This work extends the standard techniques for constructing nuclear potentials by introducing dimensional regularization, a method previously developed for field-theoretic loop diagrams (t'Hooft and Veltman, 1972). The approach maintains a strong connection between the potential and the underlying field theory, in contrast to older methods that rely on cutoff regularization. The results enable a cleaner separation of contact terms in nucleon-nucleon forces and provide a framework for systematically handling divergences in potential construction, building on earlier work by Friar and collaborators on two-boson-exchange potentials.
