# Neutrino Induced 4He Break-up Reaction -- Application of the Maximum Entropy Method in Calculating Nuclear Strength Function

> The maximum entropy method is examined as a new tool for solving the ill-posed inversion problem involved in the Lorentz integral transformation (LIT) method. As an example, we apply the method to the spin-dipole strength function of 4He. We show that the method can be successfully used for inversion of LIT, provided the LIT function is available with a sufficient accuracy.

## Key claim

The maximum entropy method (MEM) can successfully invert the Lorentz integral transformation (LIT) to reconstruct the nuclear strength function without assuming any functional form, provided the LIT function is available with sufficient accuracy, specifically, using LIT data with a Lorentz width parameter sigma_I sufficiently smaller than 10 MeV.

## Method

The authors generate pseudo LIT data L(sigma_R, sigma_I) from a known spin-dipole strength function of 4He calculated by Horiuchi (2013) using correlated Gaussians and the complex scaling method. They then apply the MEM inversion to reconstruct the strength function from these pseudo data, using a constant default model and the Shannon-Jaynes entropy, without assuming any functional form for the reconstructed strength function. The reconstructed strength function is compared to the original known strength function to assess the method's accuracy.

## Key numbers

- For sigma_I = 3 and 5 MeV, the deviation of the ratio R^MEM/R^orig from unity is within 1% in the region 25 MeV <= omega <= 120 MeV.
- For sigma_I = 10 MeV, the peak of R^MEM shifts by 1 MeV, and the deviation from R^orig is more than 5%, with nonnegligible deviation for omega < 30 MeV.
- The width of the peak structure of the original strength function R^orig is about 10 MeV.

## Context

This work extends the LIT method, previously applied to break-up reactions of few-nucleon systems using chi-squared fitting with assumed functional forms such as sums of exponentials (Efros 2007, Leidemann 2008), by introducing MEM as an alternative inversion tool that requires no a priori functional form. The MEM has been widely used in condensed-matter physics, lattice QCD (Asakawa 2001), and Green's function Monte Carlo methods (Lovato 2015). This study enables more reliable extraction of nuclear strength functions for neutrino-4He reactions relevant to supernova dynamics, though further work is needed to test the method with realistic discrete-spectrum LIT data from eigenvalue methods or the Lanczos algorithm.
