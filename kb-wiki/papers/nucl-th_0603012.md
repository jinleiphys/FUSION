# MAID Analysis Techniques

> MAID is a unitary isobar model for a partial wave analysis of pion photo- and electroproduction in the resonance region. It is fitted to the world data and can give predictions for multipoles, amplitudes, cross sections and polarization observables in the energy range from pion threshold up to W=2 GeV and photon virtualities Q^2 < 5 GeV^2. Using more recent experimental results from Mainz, Bates, Bonn and JLab for Q^2 up to 4.0 GeV^2, the Q^2 dependence of the helicity couplings A_{1/2}, A_{3/2}, S_{1/2} has been extracted for a series of four star resonances. We compare single-Q^2 analyses with a superglobal fit in a new parametrization of Maid2005. Besides the (pion) MAID, at Mainz we maintain a collection of online programs for partial wave analysis of eta, eta' and kaon photo- and electroproduction which are all based on similar footings with field theoretical background and baryon excitations in Breit-Wigner form.

## Key claim
The MAID unitary isobar model, fitted to the world data base of pion photo- and electroproduction, can extract the Q^2 dependence of helicity couplings for four-star nucleon resonances, but the model uncertainty for these amplitudes can be as large as 50% for the longitudinal amplitudes of the D13 and F15 resonances, and the fit errors from chi-squared minimization appear unrealistically small because they only reflect statistical uncertainty.

## Method
The authors extended the MAID unitary isobar model to include all four-star nucleon resonances below W=2 GeV, parameterizing the Q^2 dependence of transition amplitudes with a polynomial-exponential ansatz. They performed single-Q^2 analyses of differential cross section data from Mainz, Bates, Bonn, and JLab at fixed Q^2 values, then combined these with the world data base in a superglobal fit where all parameters were determined simultaneously. The background contributions were described by Born terms with energy-dependent mixing of pseudovector-pseudoscalar pi-NN coupling and t-channel vector meson exchanges, unitarized via the K-matrix approximation.

## Key numbers
- Energy range: from pion threshold up to W=2 GeV, with photon virtualities Q^2 < 5 GeV^2
- Recent experimental data used cover Q^2 from 0.1 to 4.0 GeV^2 and energy range 1.1 GeV < W < 2.0 GeV
- Proton helicity amplitude A_{1/2} for P33(1232): -137 x 10^{-3} GeV^{-1/2} (MD05), compared to PDG value of -135 +/- 6 x 10^{-3} GeV^{-1/2}
- Proton helicity amplitude A_{3/2} for P33(1232): -260 x 10^{-3} GeV^{-1/2} (MD05), compared to PDG value of -255 +/- 8 x 10^{-3} GeV^{-1/2}
- Proton helicity amplitude A_{1/2} for D13(1520): -27 x 10^{-3} GeV^{-1/2} (MD05), compared to PDG value of -24 +/- 9 x 10^{-3} GeV^{-1/2}
- Proton helicity amplitude A_{3/2} for D13(1520): 161 x 10^{-3} GeV^{-1/2} (MD05), compared to PDG value of 166 +/- 5 x 10^{-3} GeV^{-1/2}
- Proton helicity amplitude A_{1/2} for S11(1535): 66 x 10^{-3} GeV^{-1/2} (MD05), compared to PDG value of 90 +/- 30 x 10^{-3} GeV^{-1/2}
- Neutron helicity amplitude A_{1/2} for D13(1520): -77 x 10^{-3} GeV^{-1/2} (MD05), compared to PDG value of -59 +/- 9 x 10^{-3} GeV^{-1/2}
- Neutron helicity amplitude A_{3/2} for D13(1520): -154 x 10^{-3} GeV^{-1/2} (MD05), compared to PDG value of -139 +/- 11 x 10^{-3} GeV^{-1/2}
- About 15000 data points fitted to a small set of 10-20 parameters

## Context
This work extends the MAID unitary isobar model (first version MAID98, then MAID2000) from a model with seven resonances and limited fitting to a full partial wave analysis program fitted to the world data base of pion photo- and electroproduction. It provides Q^2-dependent helicity couplings for four-star resonances that can be compared to the GW/SAID analysis (GW02) and the Particle Data Group (PDG 2004) values. The results enable future double polarization experiments and charged pion electroproduction measurements to resolve ambiguities between partial waves differing only in isospin, and the analysis framework is also applied to neutron targets where much less data exist.
