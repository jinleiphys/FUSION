# Event-by-event mean $p_{\rm T}$ fluctuations and transverse size of color flux tube generated in $p$-$p$ collisions at $\sqrt{s}$=0.90TeV

> We propose a novel phenomenological model of mean transverse momentum fluctuations based on the Geometrical Scaling hypothesis. Bose-Einstein correlations between two gluons generated from an identical color flux tube are taken into account as a source of the fluctuation. We calculate an event-by-event fluctuation measure $\sqrt{C_m}/\langle p_{\rm T}\rangle$ and show that ALICE data observed at $\sqrt s=$0.90 TeV for $p$+$p$ collisions are reproduced. By fitting our model to the experimental data, we evaluate the transverse size of the color flux tube as a function of the multiplicity.

## Key claim

The authors propose that event-by-event mean transverse momentum fluctuations in proton-proton collisions can be explained by Bose-Einstein correlations between gluons emitted from the same color flux tube, with the transverse size of the tube determined by the saturation momentum from the Geometrical Scaling hypothesis. The model reproduces ALICE data at 0.90 TeV only when excluding the lowest-multiplicity data points (dn_ch/dy less than 3.0), and the extracted flux tube size depends sensitively on which data points are included in the fit.

## Method

The authors construct a phenomenological model where the single inclusive gluon spectrum follows a Tsallis-type universal function of the scaling variable tau = (p_T/Q_sat)^(2+lambda), with the saturation momentum Q_sat and effective interaction area S_T determined by fitting to ALICE data on mean p_T as a function of charged particle multiplicity. They introduce a Gaussian Bose-Einstein correlation function between two gluons from the same color flux tube, with correlation strength lambda_g parametrized as C times (sigma S_T Q_sat^2)^n, and fit the resulting fluctuation measure sqrt(C_m)/<p_T> to ALICE data to extract the exponent n and the parameter sigma related to the transverse tube size.

## Key numbers

- Saturation momentum Q_sat = 1.19 GeV/c for a typical event with multiplicity n_ch = 7.5 at sqrt(s) = 0.90 TeV, using parameters Q_0 = 1.0 GeV, lambda = 0.22, x_0 = 6.4 x 10^(-3)
- Effective interaction area S_T/gamma = 1.05 fm^2 for the same typical event
- Tsallis parameters from fit to p_T spectra: q = 1.124, kappa = 0.1388 (type I); q = 1.106, kappa = 0.1354 (type IIa); q = 1.106, kappa = 0.1373 (type IIb)
- Best-fit exponent n from fluctuation data: n = -0.525 (fit-A, type I); n = -0.872 (fit-B, type I)
- Best-fit sigma parameter: sigma = 0.755 (fit-A, type I); sigma = 0.096 (fit-B, type I)
- Extracted color flux tube radius r_tube = 0.2-0.3 fm (fit-A) or r_tube = 0.4-0.7 fm (fit-B)
- Parton-hadron duality factor gamma = 2.20 (type I), 2.42 (type IIa), 2.50 (type IIb)

## Context

This work extends the Geometrical Scaling hypothesis of McLerran (2014) and Praszalowicz (2013, 2015) from describing mean p_T to describing event-by-event p_T fluctuations in proton-proton collisions. The model provides an alternative to the dilution scenario of Stefan (2011) by attributing fluctuations to Bose-Einstein correlations among gluons from color flux tubes, as opposed to purely statistical fluctuations from independent particle production. The extracted flux tube sizes can be compared with HBT radii measured by ALICE (2010), and the results suggest that the correlation strength exponent n is greater than -1, deviating from the simple dilution model prediction.
