# Family of Hermitian Low-Momentum Nucleon Interactions with Phase Shift Equivalence

> Using a Schmidt orthogonalization transformation, a family of Hermitian low-momentum NN interactions is derived from the non-Hermitian Lee-Suzuki (LS) low-momentum NN interaction. As special cases, our transformation reproduces the Hermitian interactions for Okubo and Andreozzi. Aside from their common preservation of the deuteron binding energy, these Hermitian interactions are shown to be phase shift equivalent, all preserving the empirical phase shifts up to decimation scale Lambda. Employing a solvable matrix model, the Hermitian interactions given by different orthogonalization transformations are studied; the interactions can be very different from each other particularly when there is a strong intruder state influence. However, because the parent LS low-momentum NN interaction is only slightly non-Hermitian, the Hermitian low-momentum nucleon interactions given by our transformations, including the Okubo and Andreozzi ones, are all rather similar to each other. Shell model matrix elements given by the LS and several Hermitian low-momentum interactions are compared.

## Key claim

Using a Schmidt orthogonalization transformation, a family of Hermitian low-momentum nucleon-nucleon interactions can be derived from the non-Hermitian Lee-Suzuki low-momentum interaction, and these Hermitian interactions are all phase shift equivalent up to the decimation scale Lambda; however, the Hermitian interactions can be very different from each other when there is a strong intruder state influence, but because the parent Lee-Suzuki interaction is only slightly non-Hermitian for realistic nuclear potentials, the resulting Hermitian interactions are all rather similar.

## Method

The authors start from the non-Hermitian Lee-Suzuki low-momentum nucleon-nucleon interaction derived via T-matrix equivalence and the folded-diagram method. They apply a Schmidt orthogonalization transformation Z to the non-orthogonal model-space eigenvectors of the Lee-Suzuki interaction, producing orthonormal eigenvectors and a Hermitian effective interaction V_herm = Z(H_0 + V_LS)Z^{-1} - P H_0 P. They show that the Okubo transformation (using Z = P(1+omega^dagger omega)^{1/2}P) and the Andreozzi Cholesky decomposition method are special cases of this general Schmidt procedure. The method is tested on a solvable 5x5 matrix model with intruder states and then applied to realistic nucleon-nucleon potentials (CD-Bonn) to generate Hermitian low-momentum interactions.

## Key numbers

- The deuteron binding energy is preserved at 2.225 MeV by all Hermitian interactions.
- The momentum cutoff Lambda is set to 2.0 fm^{-1} for the realistic calculations.
- In the 5x5 matrix model at strength x=0.55, the non-Hermitian Lee-Suzuki interaction shows symmetric matrix elements differing by more than a factor of 4 (e.g., V_LS(1,2) = -3.4921 versus V_LS(2,1) = 0.8528).
- Shell model matrix elements for the n=0, 1S0 channel at oscillator length b=1.4 fm give -9.96 MeV for all interactions (V_LS, V_okb, V_cho, V_schm).
- Shell model matrix elements for the n=0, 3S1 channel at b=1.4 fm give -12.29 MeV for all interactions.

## Context

This work extends the low-momentum nucleon-nucleon interaction V_low-k developed by Bogner (2001, 2002), Kuo (2002), and Schwenk (2002), which was non-Hermitian and derived from realistic potentials such as CD-Bonn, Nijmegen, Argonne, and Paris. The paper unifies and generalizes prior Hermitization methods by Okubo (1954), Suzuki and Okamoto (1983), and Andreozzi (1996) into a single Schmidt-based framework. The result enables shell model calculations using Hermitian low-momentum interactions that preserve phase shifts and deuteron binding energy, with the additional finding that the Schmidt-orthogonalized interaction V_schm also preserves the deuteron wave function exactly.
