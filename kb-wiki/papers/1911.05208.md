# Uncertainty quantification of an empirical shell-model interaction using principal component analysis

> Recent investigations have emphasized the importance of uncertainty quantification (UQ) to describe errors in nuclear theory. We carry out UQ for configuration-interaction shell model calculations in the $1s$-$0d$ valence space, investigating the sensitivity of observables to perturbations in the 66 parameters (matrix elements) of a high-quality empirical interaction. The large parameter space makes computing the corresponding Hessian numerically costly, so we compare a cost-effective approximation, using the Feynman-Hellmann theorem, to the full Hessian and find it works well. Diagonalizing the Hessian yields the principal components of the interaction: linear combinations of parameters ordered by sensitivity. This approximately decoupled distribution of parameters facilitates theoretical error propagation onto structure observables: electromagnetic transitions, Gamow-Teller decays, and dark matter-nucleus scattering matrix elements.

## Key claim

The authors demonstrate that the Hessian matrix for the 66-parameter USDB shell-model interaction can be accurately approximated using only first derivatives (via the Feynman-Hellmann theorem), enabling principal component analysis (PCA) for uncertainty quantification; however, the tail-sensitive test for normality of residuals returns a p-value of 0.02, indicating that the Gaussian approximation underlying the analysis has limitations in the tails of the residual distribution.

## Method

The authors performed configuration-interaction shell-model calculations in the 1s-0d valence space using the USDB interaction, computing the linear response of 608 experimental energy levels in 77 nuclei (A=21-40) to perturbations in the 66 interaction parameters via the Feynman-Hellmann theorem. They constructed an approximate Hessian matrix from these first derivatives, diagonalized it to obtain principal components (linear combinations of parameters ordered by sensitivity), and propagated uncertainties onto observables by generating Gaussian perturbations in the PCA parameter space and sampling at least 1,000 parameter sets.

## Key numbers

- The approximate Hessian eigenvalues show the first PCA parameter carries 95% of the total sensitivity, and the first 5 PCA parameters carry 99.6% of the sensitivity
- The most sensitive PCA parameter is constrained to within 290 eV (standard deviation)
- The theoretical uncertainty added in quadrature to experimental errors was fixed at approximately 150 keV by requiring reduced chi-squared near unity
- The Kolmogorov-Smirnov test for normality of residuals returned a p-value of 0.15, while the more sensitive tail-sensitive test returned a p-value of 0.02
- 75% of shell-model energies are within 1 sigma of experiment using the estimated uncertainties

## Context

This work extends previous uncertainty quantification studies on the 0p-shell model, which examined a 7-parameter interaction using singular value decomposition (PhysRevC.96.054316) and a 17-parameter interaction without considering parameter correlations (PhysRevC.98.061301). The authors apply PCA to the full 66-parameter USDB interaction of Brown and Richter (PhysRevC.74.034315), enabling theoretical error propagation onto electromagnetic transitions, Gamow-Teller decays, and dark matter-nucleus scattering matrix elements, and demonstrating a computationally efficient method applicable to larger valence spaces where full Hessian calculations are too costly.
