# General Behavior of Double Beta Decay Amplitudes in the Quasiparticle Random Phase Approximation

> Simple formulae for the $0^+\to 0^+$ double beta decay matrix elements, as a function of the particle-particle strength $g^{pp}$, have been designed within the quasiparticle random phase approximation. The $2ν$ amplitude is a bilinear function of $g^{pp}$, and all $0ν$ moments behave as ratios of a linear function and the square root of another linear function of $g^{pp}$. It is suggested that these results are of general validity and that any modifications of the nuclear hamiltonian or the configuration space cannot lead to a different functional dependence.

## Key claim

The paper claims that within the quasiparticle random phase approximation (QRPA), the functional dependence of double beta decay matrix elements on the particle-particle strength \(g^{pp}\) (or the ratio \(t = v_t/v_s\)) is universal and independent of the nucleus, residual interaction, or configuration space; the \(2\nu\) amplitude is always a bilinear function of \(g^{pp}\), and all \(0\nu\) moments behave as ratios of a linear function and the square root of another linear function of \(g^{pp}\). The hedge is that this claim is suggested to be of "general validity" and that "any modifications of the nuclear hamiltonian or the configuration space cannot lead to a different functional dependence," but the paper only verifies it for zero-range forces and a single-mode model (SMM) plus full calculations for six nuclei.

## Method

The author derived analytic expressions for the \(0^+ \to 0^+\) double beta decay matrix elements within a single-mode model (SMM) of the QRPA, which includes only one intermediate state for each spin-parity \(J^\pi\). These expressions were then fitted to full QRPA calculations for six nuclei (\(^{48}\text{Ca}\), \(^{76}\text{Ge}\), \(^{82}\text{Se}\), \(^{90}\text{Mo}\), \(^{128}\text{Te}\), \(^{130}\text{Te}\)) using a zero-range \(\delta\) force, with the particle-particle strength parameterized by the ratio \(t = v_t/v_s\) of triplet to singlet coupling strengths. The fits used the proposed functional forms: \({\cal M}_{2\nu} \propto (1 - t/t_0)/(1 - t/t_1)\) and \({\cal M}_{0\nu} \propto (1 - t/t_0)/\sqrt{1 - t/t_1}\) for the \(J^\pi = 1^+\) component plus a linear term for higher multipoles.

## Key numbers

- For \(^{48}\text{Ca}\), the unperturbed \(2\nu\) matrix element is \({\cal M}_{2\nu} = -0.173\ \text{MeV}^{-1}\) at \(t=0\); the fitted zero parameter is \(t_0 = 1.394\) and the pole parameter is \(t_1 = 1.754\).
- For \(^{76}\text{Ge}\), the unperturbed \(2\nu\) matrix element is \({\cal M}_{2\nu} = -0.308\ \text{MeV}^{-1}\) at \(t=0\); the fitted zero parameter is \(t_0 = 1.161\) and the pole parameter is \(t_1 = 1.680\).
- For \(^{100}\text{Mo}\), the unperturbed \(2\nu\) matrix element is \({\cal M}_{2\nu} = -0.451\ \text{MeV}^{-1}\) at \(t=0\); the fitted zero parameter is \(t_0 = 1.469\) and the pole parameter is \(t_1 = 1.649\).
- For \(^{48}\text{Ca}\), the unperturbed \(0\nu\) matrix element for \(J^\pi = 1^+\) is \({\cal M}_{0\nu}(1^+) = -1.506\) at \(t=0\); the fitted zero parameter is \(t_0 = 1.244\) and the pole parameter is \(t_1 = 1.765\).
- For \(^{48}\text{Ca}\), the unperturbed \(0\nu\) matrix element for \(J^\pi \neq 1^+\) is \({\cal M}_{0\nu}(\neq 1^+) = -1.501\) at \(t=0\); the fitted zero parameter is \(t_0 = 1.227\), the pole parameter is \(t_1 = 1.768\), and the linear-decay parameter is \(t_2 = 12.82\).
- The goodness-of-fit norm \({\cal N}\) for the \(2\nu\) moments ranges from \(7.44 \times 10^{-5}\) (for \(^{82}\text{Se}\)) to \(3.26 \times 10^{-2}\) (for \(^{48}\text{Ca}\)).
- The "physical" SU(4) symmetry restoration values \(t_{\text{sym}}\) are approximately 1.50 for \(^{48}\text{Ca}\), 1.25 for \(^{76}\text{Ge}\), 1.30 for \(^{82}\text{Se}\), 1.50 for \(^{90}\text{Mo}\), and 1.40 for both \(^{128}\text{Te}\) and \(^{130}\text{Te}\).

## Context

This work extends the earlier QRPA studies of double beta decay by Vogel and Zirnbauer (1986), Engel, Vogel, and Zirnbauer (1988), and the author's own prior work (Krmpotic, Hirsch, and Dias, 1992) by providing a universal analytic parametrization of the matrix elements' dependence on the particle-particle strength. The paper enables future studies to fit full QRPA calculations with simple formulas, giving a global understanding of the transition mechanism and a way to control the calculations; it also highlights the QRPA's weakness that the pole at \(t=t_1\) occurs close to the "natural" value of \(t\), causing abrupt variations in the physically relevant interval, a problem noted by Haxton (1992) that remains unresolved.
