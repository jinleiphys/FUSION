# Event Shape Sorting: prospects and femtoscopy applications

> We demonstrate the use of Event Shape Sorting in femtoscopy. The method allows to select events with similar distributions if hadrons in azimuthal angle. We show that also their correlation radii exhibit interesting dependence on azimuthal angle with anisotropies of different orders visible at the same time. We further demonstrate such features of the hadron distribution which can be hardly recognised by Event Shape Engineering, but shows up in Event Shape Sorting. Finally, the influence of statistical fluctuations on sorting is investigated.

# Event Shape Sorting: prospects and femtoscopy applications

## Key claim
Event Shape Sorting (ESS) can reveal simultaneous second- and third-order azimuthal anisotropies in femtoscopic correlation radii that would be averaged out or missed by Event Shape Engineering (ESE), but the method's robustness against statistical fluctuations remains an open issue that requires further investigation.

## Method
The authors applied the ESS algorithm to simulated heavy-ion collision events from DRAGON, AMPT (at RHIC and LHC energies), and uRQMD generators. Events were aligned to a common second-order event plane, then iteratively sorted into ten classes based on the similarity of their azimuthal hadron distributions using a Bayesian probability approach. Correlation functions were calculated using CRAB and fitted with Gaussian forms to extract correlation radii, whose azimuthal dependence was decomposed into Fourier series including both second and third order terms.

## Key numbers
- DRAGON sample: 150,000 events with second-order anisotropy parameters a2, ρ2 ∈ (-0.1; 0.1) and third-order parameters a3, ρ3 ∈ (-0.03; 0.03)
- AMPT-RHIC sample: 10,000 events for Au+Au collisions at √sNN = 200 GeV, impact parameter 7-10 fm
- AMPT-LHC sample: 10,000 events for Pb+Pb collisions at √sNN = 2760 GeV, impact parameter 7-10 fm
- uRQMD sample: 100,000 events of Au+Au collisions at √sNN = 200 GeV, impact parameter b ∈ (7;10) fm
- For the correlated event plane test: 100,000 events generated with DRAGON, with correlated event plane admixtures of 10%, 30%, 60%, and 100%
- The uncorrelated event plane correlator value is ⟨e^{i{θ2-θ3}min}⟩ = 3/π ≈ 0.955, giving δ = 0.301374 for the correlated case

## Context
This work extends the ESS method proposed by the same group (Cimerman and Tomasik, 2019) into femtoscopy applications, where it offers advantages over the established Event Shape Engineering (ESE) approach (Schukraft et al., 2013). The authors demonstrate that ESS can resolve correlations between second- and third-order event planes that ESE would miss, as shown with a constructed counter-example. The method enables simultaneous observation of multiple harmonic orders in correlation radii, which could improve tuning of simulations to reproduce experimental data. However, the authors note that statistical fluctuations present a significant challenge, particularly when using transport model generators like uRQMD and AMPT, where the sorting signal does not clearly project onto independent test particles.
