# The quadrupole collective model from a Cartan-Weyl perspective

## Key claim

The matrix elements of quadrupole phonon creation and annihilation operators can be calculated in a purely algebraic way within an SU(1,1) x O(5) Cartan-Weyl basis, provided one uses an intermediate state method and a harmonic oscillator implementation of the SU(1,1) degree of freedom. The authors demonstrate that the radial degree of freedom (the 5-dimensional beta variable) can be included in the same algebraic framework, eliminating the earlier restriction where beta was treated as a constant of motion.

## Method

The authors construct a basis using the SU(1,1) x O(5) algebraic structure inherent in the collective quadrupole model, where SU(1,1) generators are built from quadrupole phonon pair operators and O(5) is treated in its Cartan-Weyl reduction to SU(2)_X x SU(2)_Y. They derive selection rules for phonon creation and annihilation operators from commutation relations with SU(1,1) generators, then use Racah's theorem to classify the operators as biscalar and bispinor components under the SU(2)_X x SU(2)_Y subalgebra. The double reduced matrix elements are obtained by evaluating commutation relations in the product basis and applying an intermediate state method to solve the resulting overdetermined set of algebraic equations.

## Key numbers

* The SU(1,1) Casimir quantum number lambda is related to the O(5) seniority v by lambda = v + 5/2, derived from the relation lambda(lambda-2) = v(v+3) + 5/4.
* The seniority quantum number v is an integer, and the SU(2)_X x SU(2)_Y quantum number X ranges from 0 to v/2.
* The double reduced matrix element for the biscalar (lambda=0) phonon creation operator with v' = v+1 is: <n, v+1, X || b-dagger^0 || n v X> = sqrt[ (v+2X+3)(v-2X+1) / ((2v+5)(2v+3)) ] * sqrt(2) * (2X+1) * sqrt(lambda+n).
* The double reduced matrix element for the biscalar phonon creation operator with v' = v-1 is: <n+1, v-1, X || b-dagger^0 || n v X> = sqrt[ (v+2X+2)(v-2X) / ((2v+1)(2v+3)) ] * sqrt(2) * (2X+1) * sqrt(n+1).
* The double reduced matrix element for the biscalar phonon annihilation operator with v' = v+1 is: <n-1, v+1, X || b-tilde^0 || n v X> = sqrt[ (v+2X+3)(v-2X+1) / ((2v+5)(2v+3)) ] * sqrt(2) * (2X+1) * sqrt(n).
* The double reduced matrix element for the biscalar phonon annihilation operator with v' = v-1 is: <n, v-1, X || b-tilde^0 || n v X> = sqrt[ (v+2X+2)(v-2X) / ((2v+1)(2v+3)) ] * sqrt(2) * (2X+1) * sqrt(lambda+n-1).

## Context

This work extends the algebraic treatment of the Bohr-Mottelson collective model by incorporating the radial (beta) degree of freedom into the SU(1,1) x O(5) framework, building on earlier work by De Baerdemacker (2007) that calculated matrix elements only within the O(5) basis with beta treated as constant. The approach provides a systematic method for constructing matrix representations of general collective Hamiltonians, enabling numerical studies of quantum shape phase transitions as pioneered by Iachello (2000, 2001, 2003). The Cartan-Weyl basis is not naturally compatible with the physical angular momentum quantum number L, so a numerical rotation to the physical basis is required, similar to the procedure used in earlier work by De Baerdemacker (2007).
