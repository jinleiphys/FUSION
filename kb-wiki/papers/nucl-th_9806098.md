# Parameter Symmetry of the Interacting Boson Model

> We discuss the symmetry of the parameter space of the interacting boson model (IBM). It is shown that for any set of the IBM Hamiltonian parameters (with the only exception of the U(5) dynamical symmetry limit) one can always find another set that generates the equivalent spectrum. We discuss the origin of the symmetry and its relevance for physical applications.

# Parameter Symmetry of the Interacting Boson Model

## Key claim

For any set of IBM Hamiltonian parameters (except the U(5) dynamical symmetry limit), there exists another distinct set that generates an identical eigenvalue spectrum. This parameter symmetry arises from ambiguities in the definition of boson operators and from the possibility of using different embeddings of SU(3) and SO(6) subalgebras within U(6), though the two parameter sets can sometimes be distinguished by their N-dependence across neighboring nuclei.

## Method

The authors studied the matrix of the IBM-1 Hamiltonian in the basis associated with the U(5) reduction chain, where the Hamiltonian is expressed as a superposition of Casimir operators of the groups U(5), SO(5), SO(3), SO(6), and SU(3). They applied unitary transformations that change the sign of off-diagonal matrix elements (specifically those connecting states differing by one or two d-bosons) and then compensated these sign changes by linear transformations of the Hamiltonian parameters, thereby deriving explicit parameter symmetry relations.

## Key numbers

- The parameter symmetry transformation for the case k6 ≠ 0 is: H0' = H0, k1' = k1 + 2k6, k2' = k2 + 2k6, k3' = k3 − 6k6, k4' = k4 + 2k6, k5' = k5 + 4k6, k6' = −k6.
- The parameter symmetry transformation for the case k6 = 0 is: H0' = H0 + 10k5N, k1' = k1 + 4k5(N+2), k2' = k2 − 4k5, k3' = k3 + 2k5, k4' = k4, k5' = −k5, k6' = 0.
- For 196Pt (N=6), the SO(6) DS parameter set from Arima and Iachello (1979) was k1=k2=k6=0, k3=50 keV, k4=10 keV, k5=−42.75 keV; the PS-transformed set gave k1'=−1368 keV, k2'=171 keV, k3'=−35.5 keV, k4'=10 keV, k5'=42.75 keV, k6'=0.
- The alternative SO(6) embedding Casimir operator C2(overline{SO(6)}) was expressed as 10N + 4(N+2)C1(U(5)) − 4C2(U(5)) + 2C2(SO(5)) − C2(SO(6)).

## Context

This work extends earlier discussions of boson operator ambiguities by Van Isacker, Frank, and Dukelsky (1985) and by Frank and Van Isacker (1994), which were mostly restricted to the SU(3) and SO(6) dynamical symmetry limits. The authors also draw on the concept of "form invariance" from the review by Kemmer, Polkinghorne, and Pursey (1959) in weak interaction theory. The parameter symmetry results are important for IBM applications because Hamiltonian parameters are obtained by fitting experimental spectra, and the symmetry introduces an essential ambiguity in that fitting procedure. The N-dependent parameter symmetry (for k6=0) enables discrimination between physically acceptable Hamiltonians with attractive pairing and unphysical ones with repulsive pairing by comparing spectra of neighboring isotopes, as demonstrated for platinum isotopes.
