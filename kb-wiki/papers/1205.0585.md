# Unified contraction algorithm for multi-baryon correlators on the lattice

> We propose a novel algorithm for calculating multi-baryon correlation functions on the lattice. By considering the permutation of quarks (Wick contractions) and color/spinor contractions simultaneously, we construct a unified index list for the contraction where the redundancies in the original contraction are eliminated. We find that a significant reduction in the computational cost of correlators is achieved, e.g., by a factor of 192 for $^3$H and $^3$He nuclei, and a factor of 20736 for the $^4$He nucleus, without assuming isospin symmetry. A further reduction is possible by exploiting isospin symmetry, and/or interchange symmetries associated with sink baryons, if such symmetries exist. Extensions for systems with hyperons are presented as well.

## Key claim

The paper proposes a novel algorithm for computing multi-baryon correlation functions that achieves significant computational speedup by constructing a unified index list that eliminates redundancies in Wick contractions and color/spinor contractions simultaneously, without assuming isospin symmetry.

## Method

The authors construct a coefficient matrix that combines quark permutations (Wick contractions) and color/spinor contractions into a single unified index list, where only non-zero components are tabulated. This coefficient matrix is evaluated once independently of the gauge field configuration, and then used to compute correlation functions by contracting with pre-computed baryon blocks (three-quark propagator combinations). The algorithm is applied to multi-nucleon systems with mass numbers A=2, 3, and 4, using both standard and non-relativistic baryon operators at the source.

## Key numbers

- For $^3$H and $^3$He nuclei with non-relativistic operators: a factor of 192 improvement over the block algorithm
- For $^4$He nucleus with non-relativistic operators: a factor of 20736 improvement over the block algorithm
- For $^4$He with non-relativistic operators, the number of non-zero elements in the coefficient matrix under full permutation ($N_{\rm list}$) is 518400, exactly equal to the number of quark permutations ($N_{\rm perm}$) in the block algorithm with full permutation
- For $^2$H (A=2): the number of quark permutations is 36, and the number of color/spinor loop contractions ($N_{\rm loop}$) is of order $10^5$
- For $^3$H/$^3$He (A=3): $N_{\rm perm} = 2880$, $N_{\rm loop}$ is of order $10^8$
- For $^4$He (A=4): $N_{\rm perm} = 518400$, $N_{\rm loop}$ is of order $10^{11}$

## Context

This work extends the block algorithm approach developed by Ishii (2006), Aoki (2009), and Yamazaki (2009) for computing multi-baryon correlators, which had already reduced computational cost by exploiting antisymmetry of baryon blocks under quark exchange. The unified contraction algorithm provides further significant reductions beyond the improvements achieved by Yamazaki (2009) using isospin symmetry, which reduced permutations to 93 for $^3$He and 1107 for $^4$He. The method enables more efficient lattice QCD studies of three- and four-nucleon systems, hyperon-nucleon interactions as studied by Sasaki (2011), and can be extended to multi-hadron correlators including mesons.
