# Breaking SU(3) spectral degeneracies in heavy deformed nuclei

> Symmetries are manifested in nature through degeneracies in the spectra of physical systems. In the case of heavy deformed nuclei, when described in the framework of the Interacting Boson Model, within which correlated proton (neutron) pairs are approximated as bosons, the ground state band has no symmetry partner, while the degeneracy between the first excited beta and gamma bands is broken through the use of three-body and/or four-body terms. In the framework of the proxy-SU(3) model, in which an approximate SU(3) symmetry of fermions is present, the same three-body and/or four-body operators are used for breaking the degeneracy between the ground state band and the first excited gamma band. Experimentally accessible quantities being independent of any free parameters are pointed out in the latter case.

## Key claim

Within the proxy-SU(3) model, three-body and/or four-body operators are necessary to break the degeneracy between the ground state band and the first excited gamma band, and preliminary calculations show that parameter-independent predictions for the energy differences between these bands and for odd-even staggering within the gamma bands can be obtained before any detailed fitting is performed.

## Method

The authors used the proxy-SU(3) model with Hamiltonians containing a K-band splitting operator plus either the three-body operator Omega or the four-body operator Lambda, along with L^2 and L^4 terms. They computed energy differences between the gamma_1 band and ground state band for four deformed nuclei (162Er, 160Dy, 166Yb, 178Hf) and compared them to experimental data from ENSDF. The odd-even staggering within the gamma_1 bands was also calculated and compared to experiment.

## Key numbers

- For 162Er, the Hamiltonian parameters (in keV) were: a = 1443 (x10^-2), b = 408, c = 440 (x10^-7), d = 1258 (x10^-5), with fits up to L=20 for the ground state band and L=12 for the gamma band; the rms deviation was 34 keV.
- For 178Hf, the Hamiltonian parameters (in keV) were: a = 1225 (x10^-2), b = 588, c = 748 (x10^-7), d = 890 (x10^-5), with fits up to L=20 for the ground state band and L=15 for the gamma band; the rms deviation was 58 keV.
- For 160Dy, the Hamiltonian parameters (in keV) were: a = 1025 (x10^-2), b = 445, c = 578 (x10^-7), d = 412 (x10^-5), with fits up to L=28 for the ground state band and L=23 for the gamma band.
- For 166Yb, the Hamiltonian parameters (in keV) were: a = 540 (x10^-2), b = 483, c = 2992 (x10^-7), d = 533 (x10^-5), with fits up to L=24 for the ground state band and L=13 for the gamma band.

## Context

This work extends the proxy-SU(3) model, previously applied to deformation parameters and B(E2) ratios (Bonatsos 2017, Martinou 2017), to spectral calculations using three- and four-body operators analogous to those used in the Interacting Boson Model (IBM) by DeMeyer (1985) and Vanthournout (1987). The approach parallels earlier pseudo-SU(3) calculations by Draayer (1984) and Naqvi (2011) that used similar K-band splitting operators. The results enable future inclusion of B(E2) transition rates and comparison with the work of Jolos and von Brentano (2014, 2015) on different mass coefficients for ground and gamma bands in the Bohr Hamiltonian.
