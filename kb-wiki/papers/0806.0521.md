# Theory of Scalars

## Key claim
The light scalar mesons (σ(600), κ(700–900), a₀(980), f₀(980)) are not conventional qq̄ states but have dominant four-quark (q²q̄²) components, as evidenced by their inverted mass spectrum, their production mechanisms in φ radiative decays, and their suppressed two-photon widths, though the authors acknowledge that their own tree-level linear σ model (LSM) approximation provides only a semiquantitative description below 0.4 GeV and does not include K̄K channel or f₀(980) effects.

## Method
The authors construct an SU_L(2)×SU_R(2) linear σ model (LSM) with tree-level amplitudes for ππ → ππ, unitarized via the K-matrix method to include a negative background phase that “shields” the σ resonance. They extend this to γγ → ππ via charged-pion loop diagrams, and analyze φ → γa₀, γf₀ decays using a K⁺K⁻ loop mechanism. Phenomenological chiral shielding fits are performed to experimental ππ phase shifts, inelasticities, and mass spectra from SND, CMD-2, KLOE, and Belle.

## Key numbers
- σ resonance pole position: √s_R = (0.52 − i0.25) GeV (M_R = 0.52 GeV, Γ_R/2 = 0.25 GeV)
- Breit-Wigner mass and width of σ: M_res = 0.43 GeV, Γ_res(M²_res) = 0.67 GeV; renormalized width Γ^{renorm}_res = 0.53 GeV
- σ propagator mass shift: m_σ − M_res = 0.50 GeV (half the BW mass from four-quark contribution)
- ππ S-wave scattering lengths: a⁰₀ = 0.18 m_π⁻¹, a²₀ = −0.04 m_π⁻¹
- σ → γγ width from pole residue: Γ(σ → γγ) ≈ 2 keV
- Direct σ → γγ coupling width: Γ^{(0)}_{σγγ} ≈ 0.0034 keV
- a₀(980)/f₀(980) → γγ widths predicted in q²q̄² model: ~0.27 keV; experimental values: Γ(a₀ → γγ) = (0.19 ± 0.07 +0.1/−0.07)/B(a₀ → πη) keV (Crystal Ball), (0.28 ± 0.04 ± 0.1)/B(a₀ → πη) keV (JADE); Γ(f₀ → γγ) = (0.31 ± 0.14 ± 0.09) keV (Crystal Ball), (0.24 ± 0.06 ± 0.15) keV (MARK II)
- f₀ → K⁺K⁻ → γγ loop width: ⟨Γ_{f₀ → K⁺K⁻ → γγ}⟩ ≈ 0.2 keV
- Phenomenological chiral shielding couplings: g²_{σπ⁺π⁻}/4π = 0.99 GeV², g²_{σK⁺K⁻}/4π = 2×10⁻⁴ GeV², g²_{f₀π⁺π⁻}/4π = 0.12 GeV², g²_{f₀K⁺K⁻}/4π = 1.04 GeV²; BW masses: m_{f₀} = 989 MeV, m_σ = 679 MeV, Γ_σ = 498 MeV; scattering length a⁰₀ = 0.223 m_π⁺⁻¹

## Context
This work extends the chiral shielding mechanism first proposed by the authors (Achasov, 1994; Achasov and Shestakov, 2007) to explain why the σ(600) resonance is hidden in ππ scattering phase shifts, contradicting earlier PDG conclusions that such states did not exist. It builds on Jaffe’s (1977) MIT bag model prediction of light four-quark scalar states and the authors’ own 1982 prediction of suppressed a₀/f₀ → γγ widths. The analysis of φ radiative decays (SND, CMD-2, KLOE) and Belle γγ → ππ data provides experimental support for the four-quark interpretation. The results enable future non-perturbative calculations within the SU_L(3)×SU_R(3) LSM framework and motivate further study of production mechanisms in scalar channels.
