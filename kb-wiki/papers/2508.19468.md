# Weighted Levenberg-Marquardt methods for fitting multichannel nuclear cross section data

> We present an extension of the Levenberg-Marquardt algorithm for fitting multichannel nuclear cross section data. Our approach offers a practical and robust alternative to conventional trust-region methods for analyzing experimental data. The CoH$_3$ code, based on the Hauser-Feshbach statistical model, involves a large number of interdependent parameters, making optimization challenging due to the presence of "sloppy" directions in parameter space. To address the uneven distribution of experimental data across reaction channels, we construct a weighted Fisher Information Metric by integrating prior distributions over dataset weights. This framework enables a more balanced treatment of heterogeneous data, improving both parameter estimation and convergence robustness. We show that the resulting weighted Levenberg-Marquardt method yields more physically consistent fits for both raw and smoothed datasets, using experimental data for ${}^{148}$Sm as a representative example. Additionally, we introduce a geometric scaling strategy to accelerate convergence -- a method based on the local geometry of the manifold.

## Key claim

The weighted Levenberg-Marquardt (wLM) algorithm, which incorporates dataset-specific weighting priors into the Fisher Information Metric, yields more physically consistent fits for multichannel nuclear cross section data than the standard unweighted LM method, particularly when experimental data are unevenly distributed across reaction channels; however, the most effective weighting scheme is inherently data-dependent, with the exponential prior consistently yielding lower overall chi-squared values but looser parameter uncertainty constraints compared to uniform or cropped-exponential priors.

## Method

The authors extended the standard Levenberg-Marquardt algorithm by constructing a weighted Fisher Information Metric (wFIM) through marginalization over dataset-specific weight parameters with maximum-entropy priors (exponential, uniform, or cropped-exponential distributions). They applied this wLM method to fit the CoH$_3$ Hauser-Feshbach statistical model parameters to experimental total and capture cross section data for $^{148}$Sm, using both raw experimental data and Full Bayesian Evaluation Technique (FBET)-smoothed datasets. Additionally, they introduced a geometric scaling strategy that adapts the parameter update step based on the angle between successive updates evaluated using the wFIM, acting as a deterministic counterpart to momentum-based machine learning optimizers.

## Key numbers

- The damping parameter hyperparameters were set to $(\lambda_u, \lambda_d) = (10, 2)$, with initial damping parameter $\lambda_0 = 10^3$ and initial parameter vector $\theta^\mu_0 = 1$.
- For the cropped-exponential and uniform priors, the width parameter $h$ was set to $h = 0.4$ for practical convergence across a wider set of nuclei; larger values gave unstable fits for some nuclei.
- The exponential weighting scheme consistently yielded lower overall $\chi^2$ values compared to the unweighted case and to both uniform and cropped-exponential weighting strategies for the $^{148}$Sm nucleus examined.
- The unweighted (classic) fit to smoothed capture data yielded parameter values for the imaginary surface potential depth ($t_{OW}$) and imaginary surface potential diffuseness ($t_{OAW}$) that lie outside the expected physical range.

## Context

This work extends the Levenberg-Marquardt algorithm (Marquardt, 1963) by generalizing a weighting technique originally developed for cosmological data analysis (Hobson, 2002) to nuclear cross section fitting. It addresses the "sloppy" model problem identified by Transtrum (2010, 2012), where Fisher Information Metric eigenvalues span many orders of magnitude, and provides an alternative to classical trust-region methods such as reparametrization (James, 1975, 2004). The approach enables simultaneous fitting of heterogeneous datasets from the EXFOR database (ENDF, 2014) using the CoH$_3$ code (Kawano, 2010, 2019), complementing the Full Bayesian Evaluation Technique (Leeb, 2008, 2011) and experimental uncertainty templates (Neudecker, 2018, 2023). Future work includes applying the method to a broader set of nuclei to determine the most effective data-dependent weighting choice.
