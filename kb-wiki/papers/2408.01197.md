# Pole analysis for the $D^{*}\bar K$-$D\bar{K^*}$ coupled-channel system

> By solving the Lippmann-Schwinger equation, possible hadronic molecules in the $D^*\bar K$-$D\bar K^*$ coupled-channel system are investigated with the one-meson exchange potentials, where both vector and pseudoscalar mesons are considered as exchange particles. We find an S-wave virtual state with mass $M=2487$ MeV, and a resonance with $M=2759$ and width $Γ=18$ MeV. In the $D^* \bar K$ invariant mass distribution, the virtual state appears as a cusp at the $D^*\bar K$ threshold, while the resonance potentially manifests as a dip. In particular, we take into account the $D\bar K π$ three-body dynamics due to the on-shell pion exchange and the finite decay width for $\bar K^*$.

## Key claim

By solving the Lippmann-Schwinger equation with one-meson exchange potentials that include both vector and pseudoscalar mesons, the authors predict an S-wave virtual state and a resonance in the $D^*\bar K$-$D\bar K^*$ coupled-channel system; however, the precise pole positions and widths depend on the SU(4) symmetry breaking parameter $\gamma'$ and the inclusion of momentum-dependent $\bar K^*$ decay widths, which significantly reduce the resonance width compared to prior work.

## Method

The authors derived effective potentials for the $D^*\bar K$-$D\bar K^*$ coupled-channel system using the local hidden gauge formalism with SU(4) symmetry, incorporating vector meson exchange ($\rho$, $\omega$, $D_s^*$) and pseudoscalar meson exchange ($\pi$, $\eta$, $\eta'$). They solved the integral Lippmann-Schwinger equation with a hard cutoff regularization, including the finite decay width of $\bar K^*$ via a momentum-dependent width and accounting for the $D\bar K\pi$ three-body dynamics from on-shell pion exchange. Partial wave projection was performed to separate S-wave and D-wave contributions in the four-channel basis.

## Key numbers

- Virtual state mass: $M = 2487$ MeV (below $D^*\bar K$ threshold)
- Resonance mass: $M = 2759$ MeV with width $\Gamma = 18$ MeV
- For parameter set (a) ($\gamma=1$, $\gamma'=1.065$, $\Lambda=1.1$ GeV): lower pole at $2486.5$ MeV (virtual state), upper pole at $2759.2 - 8.9i$ MeV (resonance)
- For parameter set (b) ($\gamma=1$, $\gamma'=2.82$, $\Lambda=1.1$ GeV): lower pole at $2498.5$ MeV, upper pole at $2755.6 - 38.9i$ MeV
- Single-channel $D^*\bar K$ virtual state binding energy: $21.8$ MeV (set a) and $29.8$ MeV (set b)
- Single-channel $D\bar K^*$ bound state binding energy: $0.6$ MeV for both parameter sets
- Coupling constant $g = m_\rho / (2 F_\pi)$ with $F_\pi = 93$ MeV

## Context

This work extends the earlier study by Gamermann et al. (2007), who predicted a narrow pole at $2756.08 - 2.15i$ MeV and a broad cusp at the $D^*\bar K$ threshold using only vector meson exchange and on-shell approximation of the Bethe-Salpeter equation. The present calculation incorporates pseudoscalar meson exchange and momentum-dependent $\bar K^*$ decay widths, finding that the resonance width is significantly smaller than previously predicted once the $\bar K^*$ width is included. The results also connect to the $T_{cs}(2900)$ and $T_{c\bar s}(2900)$ states observed by LHCb, which were reproduced as $D^*\bar K^*$ molecules in Molina et al. (2010, 2020, 2022). The predicted line shapes (cusp at $D^*\bar K$ threshold and dip for the resonance) provide experimental signatures for future searches in $B$ meson decays.
