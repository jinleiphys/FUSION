# Status and Scope of MONC Transport Code

> $\underline{\textbf{MO}}$nte-carlo $\underline{\textbf{N}}$ucleon transport $\underline{\textbf{C}}$ode (MONC) for nucleon transport is being developed for several years. Constructive Solid Geometry concept is applied with the help of solid bodies. Union, subtraction and intersection Boolean operations are used to construct heterogeneous zones. Scaling, rotation, and translation operation of the basic bodies are allowed to construct more complex zones. Module of repetitive structure for lattice, core calculations in reactor and detector simulation is developed. Graphical User Interface along with visualization tools is developed to make input, construction and display of geometry, and analysis of output data. Low energy neutron transport module is developed using continuous linearly interpolable point neutron cross section data below 20MeV neutron energy. The code is benchmarked for simulation of accelerator driven sub-critical system, neutron shielding, heat and neutron flux distribution and keff of the critical assemblies. It is observed that results of keff are in agreement within $\sim$ 3mk with experimental results of critical assemblies as well as the values obtained from MCNP.

# Status and Scope of MONC Transport Code

## Key claim

The MONC transport code achieves agreement in keff within approximately 3 mk of experimental critical assembly results and MCNP calculations, but this benchmark agreement is conditional on using appropriate cross-section libraries (ENDF-B/VIII.1 or ENDF-B/VII.1) and sufficient inactive cycles to reach the fundamental fission source mode, with the recommendation that keff values from all four estimators should not vary by more than 0.1%.

## Method

The MONC code implements Constructive Solid Geometry (CSG) using Boolean operations (union, subtraction, intersection) on basic bodies (sphere, cylinder, parallelepiped, cone, ellipse, hexagon) with scaling, rotation, and translation to construct heterogeneous zones, including repetitive structures for reactor lattice and core calculations. High-energy nucleon transport (above 20 MeV) uses intra-nuclear cascade, pre-equilibrium, evaporation, and fission models, while low-energy neutron transport (below 20 MeV) uses continuous linearly interpolable point neutron cross-section data in ACE format processed from ENDF libraries, with free gas thermal treatment below 4 eV and probability table methods in the unresolved resonance region. Criticality calculations employ four estimators (neutron population, collision, absorption, and track length) combined via maximum likelihood method.

## Context

This work extends the CASCADE.04 code (Kumawat 2004, 2005) by adding low-energy neutron transport with pointwise ACE-format cross sections, CSG geometry with Boolean operations, repetitive structure capability for reactor core calculations, and a graphical user interface, making MONC a unified Monte Carlo transport code covering both spallation and thermal neutron regimes. The code is benchmarked against the IAEA coordinated research project ADS benchmark (involving participants from Russia, Switzerland, Italy, France, Germany, Netherlands, Japan, Belarus, and Sweden) and against standard critical assembly problems from the Godiva and related benchmarks (Briggs 2004), with results comparable to MCNP and KENO. This enables full-core reactor calculations including burnup and decay chain analysis for accelerator-driven sub-critical systems and critical reactor design studies.
