# Effects of cluster-shell competition and BCS-like pairing in $^{12}$C

> The antisymmetrized quasi-cluster model (AQCM) was proposed to describe α-cluster and $jj$-coupling shell models on the same footing. In this model, the cluster-shell transition is characterized by two parameters; $R$ representing the distance between α clusters and α describing the breaking of α clusters, and the contribution of the spin-orbit interaction, very important in the $jj$-coupling shell model, can be taken into account starting with the α cluster model wave function. Not only the closure configurations of the major shells, but also the subclosure configurations of the $jj$-coupling shell model can be described starting with the α-cluster model wave functions; however, the particle hole excitations of single particles have not been fully established yet. In this study we show that the framework of AQCM can be extended even to the states with the character of single particle excitations. For $^{12}$C, two particle two hole (2p2h) excitations from the subclosure configuration of $0p_{3/2}$ corresponding to BCS-like pairing are described, and these shell model states are coupled with the three α cluster model wave functions. The correlation energy from the optimal configuration can be estimated not only in the cluster part but also in the shell model part. We try to pave the way to establish a generalized description of the nuclear structure.

## Key claim

The antisymmetrized quasi-cluster model (AQCM) can be extended to describe single-particle excitations (specifically two-particle two-hole, or 2p2h, excitations) from the subclosure configuration of $^{12}$C, enabling the coupling of BCS-like pairing states with three-alpha cluster wave functions, though the description of particle-hole excitations of single particles had not been fully established prior to this work.

## Method

The authors extended the AQCM framework by introducing two $\Lambda$ parameters ($\Lambda_a$ and $\Lambda_b$) for different quasi-alpha clusters to generate 2p2h configurations from the $0p_{3/2}$ subclosure, including excitations to $0p_{1/2}$ and $0d_{5/2}$ for protons, neutrons, and proton-neutron pairs. They constructed a total wave function as a linear combination of 36 basis states (6 shell-model configurations and 30 three-alpha cluster configurations with isosceles triangular shapes) using the generator coordinate method (GCM) with angular momentum and parity projection, then diagonalized the Hamiltonian using the Volkov No.2 central force and the G3RS spin-orbit force.

## Key numbers

* Width parameter: $b = 1.4$ fm for the Gaussian single-particle wave functions ($\nu = 1/2b^2$)
* For the shell model 0p0h state, $R = 0.1$ fm was used in the Gaussian center parameters
* Cluster model basis states used $d = 1, 2, \ldots, 5$ fm and $h = 1, 2, \ldots, 6$ fm for isosceles triangular configurations (30 basis states total)
* Volkov No.2 central force parameters: $V_a = -60.65$ MeV, $V_r = 61.14$ MeV, $\alpha = 1.80$ fm, $\rho = 1.01$ fm, with $M = 1-W = 0.6$ and $B = H = 0.125$
* G3RS spin-orbit force parameters: $\eta_1 = 0.447$ fm, $\eta_2 = 0.6$ fm, $V_{LS1} = -V_{LS2} = 1600$ MeV

## Context

This work extends the AQCM approach previously applied to subclosure configurations of $^{12}$C (Suhara, 2013) and to nuclei with magic numbers 28 and 50 (Itagaki, 2016). It builds on the hybrid model idea of Takigawa and collaborators, who mixed alpha cluster model and p-shell SU(3) basis states, and contrasts with antisymmetrized molecular dynamics (AMD) and Fermionic molecular dynamics (FMD) approaches by using fewer and more controllable parameters. The framework enables a generalized description of nuclear structure that can simultaneously treat cluster and shell-model aspects, including BCS-like pairing effects.
