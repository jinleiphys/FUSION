# Moments of Axial-Vector GPD from Lattice QCD: Quark Helicity, Orbital Angular Momentum, and Spin-Orbit Correlation

> In this work, we present a lattice QCD calculation of the Mellin moments of the twist-2 axial-vector generalized parton distribution (GPD), $\widetilde{H}(x,ξ,t)$, at zero skewness, $ξ$, with multiple values of the momentum transfer, $t$. Our analysis employs the short-distance factorization framework on ratio-scheme renormalized quasi-GPD matrix elements. The calculations are based on an $N_f=2+1+1$ twisted mass fermions ensemble with clover improvement, a lattice spacing of $a = 0.093$ fm, and a pion mass of $m_π= 260$ MeV. We consider both the iso-vector and iso-scalar cases, utilizing next-to-leading-order perturbative matching while omitting the disconnected contributions and gluon mixing in the iso-scalar case. For the first time, we determine the Mellin moments of $\widetilde{H}$ up to the fifth order. From these moments, we discuss the quark helicity and orbital angular momentum contributions to the nucleon spin, as well as the spin-orbit correlations of the quarks. Additionally, we perform a Fourier transform over the momentum transfer, which allows us to explore the spin structure in the impact-parameter space.

## Key claim

This work presents the first lattice QCD determination of the Mellin moments of the twist-2 axial-vector generalized parton distribution (GPD) \(\widetilde{H}(x,\xi,t)\) up to fifth order at zero skewness, using a short-distance factorization framework on ratio-scheme renormalized quasi-GPD matrix elements, but omits disconnected contributions and gluon mixing in the iso-scalar case, which limits the completeness of the iso-scalar results.

## Method

The authors computed quasi-GPD matrix elements for the axial-vector operator \(\gamma^3\gamma_5\) on an \(N_f=2+1+1\) twisted mass fermion ensemble with clover improvement, lattice spacing \(a = 0.093\) fm, and pion mass \(m_\pi = 260\) MeV, using both symmetric and asymmetric frames to access multiple momentum transfers \(t\) without additional inversions. Bare matrix elements were renormalized via a ratio scheme dividing by zero-momentum, zero-transfer iso-vector matrix elements, and the Mellin moments \(\tilde{A}_{n+1,0}(t;\mu)\) were extracted by fitting the \(P_3\) dependence of the renormalized matrix elements at fixed \(z_3\) using next-to-leading order (NLO) perturbative matching. Both iso-vector (\(u-d\)) and iso-scalar (\(u+d\)) flavor combinations were analyzed, with the iso-scalar case excluding disconnected diagrams.

## Key numbers

- Lattice spacing: \(a = 0.093\) fm; pion mass: \(m_\pi = 260\) MeV.
- Nucleon momentum \(P_3\) values used: 0.83 GeV, 1.25 GeV, and 1.67 GeV for the symmetric frame at \(-t = 0.69\) GeV\(^2\).
- Momentum transfers \(-t\) accessed: 0.17, 0.34, 0.65, 0.69, 0.81, 1.24, 1.38, 1.52, 2.29, and 2.77 GeV\(^2\).
- Renormalized iso-vector axial charge: \(g_A = 1.164(13)\), derived from the bare matrix element using \(Z_A = 0.7442\).
- Source-sink separation: \(t_s = 10a = 0.93\) fm.
- Moments extracted up to fifth order (\(\tilde{A}_{5,0}\)) for the first time; the first moment \(\tilde{A}_{1,0}(t;\mu)\) is the nucleon axial form factor.

## Context

This work extends the lattice QCD approach to axial-vector GPD moments beyond previous studies that were limited to lower-order moments (e.g., Hagler (2003), Alexandrou (2011, 2013), Constantinou (2014), Green (2014), Hasan (2017), Gupta (2017), Bali (2018), Alexandrou (2019, 2022), Jang (2023)). It builds on the Lorentz-covariant amplitude method proposed by Bhattacharya (2022) and the earlier \(x\)-dependent unpolarized GPD results from Cichy (2023) and Bhattacharya (2023). The extracted moments enable first-principles access to quark helicity and orbital angular momentum contributions to the nucleon spin via Ji's spin decomposition, as well as spin-orbit correlations, and allow Fourier transform to impact-parameter space for spatial spin structure.
