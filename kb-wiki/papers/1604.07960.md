# Delta(1232) contribution to real photon radiative corrections for elastic electron-proton scattering

> Here we consider a contribution of Delta(1232) resonance to real photon radiative corrections for elastic $ep$-scattering. The effect is found to be small for past experiments to study unpolarized cross section as well as for the recent VEPP-3 experiment to investigate two-photon exchange effects by precision measurement of $e^\pm p$-scattering cross sections ratio.

## Key claim

The contribution of the Delta(1232) resonance to real photon radiative corrections for elastic electron-proton scattering is found to be negligible (below 0.01%) for both past unpolarized cross section experiments and the recent VEPP-3 experiment, despite a naive estimate suggesting a potentially significant effect of about 0.5%.

## Method

The authors calculated the Delta(1232) contribution to proton bremsstrahlung using a hadronic model with the Zhou-Yang parametrization of the gamma-p-Delta transition vertex, employing form factors with parameters {g1, g2, g3} = {6.59, 9.08, 7.12} and cutoff parameters Lambda_1 = 0.84 GeV, Lambda_2 = 2 GeV, Lambda_3 = sqrt(2) GeV, Lambda_4 = 0.2 GeV, a = -0.3. They computed the squared matrix element for the resonant diagram (|M_Delta^(1)|^2), the non-resonant diagram (|M_Delta^(2)|^2), their interference, and the C-odd interference with electron bremsstrahlung, then integrated over phase space with experimental cuts specific to the VEPP-3 experiment using FeynCalc and Monte Carlo integration.

## Key numbers

- The rough estimate for the Delta(1232) contribution to real radiative corrections gave delta_Delta approximately 0.5%, using the branching ratio Br(Delta to N gamma) = 0.55-0.65% from PDG
- For the VEPP-3 experiment Run I, No. 1 (E_beam = 1.594 GeV, Q^2 = 1.51 GeV^2, Delta E/epsilon_3,el = 0.25, Delta theta_p = Delta phi_p = 3.0 degrees), the total interference contribution delta_Delta^(int) = (0.32 +/- 0.02) x 10^(-5)
- For the VEPP-3 experiment Run I, No. 2 (E_beam = 1.594 GeV, Q^2 = 0.298 GeV^2, Delta E/epsilon_3,el = 0.45, Delta theta_p = Delta phi_p = 5.0 degrees), delta_Delta^(int) = (3.2 +/- 0.1) x 10^(-5)
- For the VEPP-3 experiment Run II, No. 1 (E_beam = 0.998 GeV, Q^2 = 0.976 GeV^2, Delta E/epsilon_3,el = 0.29, Delta theta_p = Delta phi_p = 3.0 degrees), delta_Delta^(int) = (2.86 +/- 0.01) x 10^(-5)
- For the VEPP-3 experiment Run II, No. 2 (E_beam = 0.998 GeV, Q^2 = 0.830 GeV^2, Delta E/epsilon_3,el = 0.29, Delta theta_p = Delta phi_p = 3.0 degrees), delta_Delta^(int) = (1.95 +/- 0.01) x 10^(-5)
- The typical value W_max^2 = 1.12 GeV^2 from the SLAC experiment (PhysRevD.49.5671) leads to strong suppression of the Delta(1232) contribution

## Context

This work extends the hadronic model approach to two-photon exchange effects, where previous studies by Zhou and Yang (2014) and others considered virtual proton and Delta(1232) intermediate states in the TPE amplitude. The calculation addresses the need to accurately subtract real radiative corrections for the VEPP-3 experiment (PhysRevLett.114.062005), which measures the ratio R = sigma(e+p)/sigma(e-p) to study two-photon exchange effects. The ESEPP event generator (2014) previously accounted only for the virtual proton intermediate state in proton bremsstrahlung; this work shows that the Delta(1232) contribution is small enough to be neglected in the analysis of the VEPP-3 experiment where the TPE effect is about 1%.
