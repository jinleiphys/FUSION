# Nucleon-nucleon collision profile and cross section fluctuations

> The nucleon-nucleon collision profile, being the basic entity of the wounded nucleon model, is usually adopted in the form of hard sphere or the Gaussian shape. We suggest that the cross section fluctuations given by the gamma distribution leads to the profile function which smoothly ranges between the both limiting forms. Examples demonstrating sensitivity of profile function on cross section fluctuations are discussed.

# Nucleon-nucleon collision profile and cross section fluctuations

## Key claim
The nucleon-nucleon collision profile, which smoothly interpolates between hard-sphere and Gaussian limiting forms, can be derived from cross section fluctuations described by a gamma distribution; the profile function is given by the regularized incomplete gamma function, and the fluctuation parameter omega = 0.4 obtained from TOTEM elastic pp data at sqrt(s) = 7 TeV provides a quantitative description of the observed differential cross section.

## Method
The authors model cross section fluctuations using a gamma distribution g(sigma) with relative variance omega = Var(sigma)/<sigma>^2. They derive the nucleon-nucleon collision profile p(b) as the integral of g(sigma) over sigma from pi*b^2 to infinity, yielding the regularized incomplete gamma function. The profile is then used to compute elastic scattering amplitudes via the unitarity condition, and the resulting differential cross section is fitted to TOTEM elastic pp data at sqrt(s) = 7 TeV. The same profile is implemented in the Glauber Monte Carlo code GLISSANDO to study fluctuations in nucleus-nucleus collisions.

## Key numbers
- omega = 0.4: the relative variance of cross section fluctuations obtained from fitting TOTEM elastic differential pp cross section data at sqrt(s) = 7 TeV.
- sigma_inel = 73.2 mb: the inelastic nucleon-nucleon cross section used in the fit to TOTEM data.
- sigma_tot = 98.3 mb: the total proton-proton cross section recovered with parameter k = 0.97 in the relation R_0^2 = k * sigma_inel / pi.
- omega in (0,1): the range of the fluctuation parameter for physically meaningful profiles; omega = 0 gives hard-sphere profile, omega = 1 gives Gaussian profile.
- Approximately 50% difference in scaled variance of target participants between hard-sphere (omega = 0.01) and Gaussian (omega = 0.99) approximations for Pb+Pb minimum bias collisions at sigma_inel = 73.2 mb.

## Context
This work extends the wounded nucleon model (Bialas 1976) and its Glauber Monte Carlo implementations (Broniowski 2007) by replacing the usual hard-sphere or Gaussian collision profiles with a profile derived from cross section fluctuations. The approach builds on earlier studies of cross section fluctuations (Blaettel 1993, Guzey 2005) and the observation that such fluctuations lead to nonexponential attenuation of hadrons (Wilk 1994). The results enable more accurate modeling of nucleus-nucleus collisions, particularly for observables sensitive to fluctuations in the number of participants, and provide a connection between elastic scattering data and the partonic structure of hadrons.
