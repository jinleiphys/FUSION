# Helicity Formalism for NN Scattering without Partial Wave Decomposition

> At intermediate energies it appears advantageous to work without partial wave decomposition and use directly the momentum vectors. We derived a finite set of coupled Lippmann-Schwinger equations using helicities. They can be used for any type of realistic NN forces, which are given in operator form. As an example we have chosen the Bonn OBEPR NN potential.

# Helicity Formalism for NN Scattering without Partial Wave Decomposition

## Key claim

The authors derive a finite set of coupled Lippmann-Schwinger equations in helicity representation that can describe nucleon-nucleon scattering at intermediate energies (a few hundred MeV) without partial wave decomposition, using the Bonn OBEPR potential as an example. The method reduces algebraic and numerical work compared to solving many partial wave equations, but the formalism is demonstrated only for the two-nucleon system and requires potentials given in operator form.

## Method

The authors construct antisymmetric basis states labeled by relative momentum, total spin, total helicity, parity, and isospin. They derive a set of coupled Lippmann-Schwinger equations for the helicity T-matrix elements, which for spin S=0 reduces to a single equation and for S=1 to two coupled equations (for helicities 0 and 1). The NN potential is expressed using six Omega operators (redefined from prior work) that can be applied directly to the helicity basis states, with the relation between standard O operators and Omega operators given explicitly. The helicity T-matrix is then used to compute partial wave T-matrices and phase shifts via a transformation involving rotation matrices and Clebsch-Gordan coefficients.

## Key numbers

- Phase shifts calculated at E_lab = 100 MeV and E_lab = 325 MeV for eight partial wave states (¹S₀, ¹P₁, ¹D₂, ¹F₃, ³P₀, ³P₁, ³D₂, ³F₃) using the helicity formalism and compared with partial wave decomposition
- At E_lab = 100 MeV: ¹S₀ phase shift = 27.38 degrees (helicity method) vs 27.37 degrees (partial wave method); ³P₁ phase shift = -13.72 vs -13.72 degrees
- At E_lab = 325 MeV: ¹S₀ phase shift = -7.25 vs -7.26 degrees; ³D₂ phase shift = 36.25 vs 36.20 degrees
- All phase shifts agree to within 0.05 degrees or better between the two methods

## Context

This work extends earlier formulations for NN scattering without partial waves, particularly the helicity-based approach of prior authors (referred to as [3] in the text), by proposing a modified form of the equations. The method enables direct momentum-space calculations at intermediate energies where many partial waves would be needed, and where relativistic effects become important. The approach can be applied to any realistic NN potential given in operator form, with the Bonn OBEPR potential used as a demonstration, and the authors note that the formalism reduces the algebraic and numerical work compared to solving many partial wave equations separately.
