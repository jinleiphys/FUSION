# Fission Fragment Decay Simulations with the CGMF Code

> The CGMF code implements the Hauser-Feshbach statistical nuclear reaction model to follow the de-excitation of fission fragments by successive emissions of prompt neutrons and $γ$ rays. The Monte Carlo technique is used to facilitate the analysis of complex distributions and correlations among the prompt fission observables. Starting from initial configurations for the fission fragments in mass, charge, kinetic energy, excitation energy, spin, and parity, $Y(A,Z,KE,U,J,π)$, CGMF samples neutron and $γ$-ray probability distributions at each stage of the decay process, conserving energy, spin and parity. Nuclear structure and reaction input data from the RIPL library are used to describe fission fragment properties and decay probabilities. Characteristics of prompt fission neutrons, prompt fission gamma rays, and independent fission yields can be studied consistently. Correlations in energy, angle and multiplicity among the emitted neutrons and $γ$ rays can be easily analyzed as a function of the emitting fragments.

# Fission Fragment Decay Simulations with the CGMF Code

## Key claim

The CGMF code provides a Monte Carlo implementation of the Hauser-Feshbach statistical model to simulate the de-excitation of fission fragments through prompt neutron and gamma-ray emission, but its predictive power is limited by the quality of nuclear structure and reaction input data taken from the RIPL library, and the code is currently restricted to binary fission only for a specific set of isotopes and neutron energies.

## Method

CGMF starts from initial fission fragment configurations in mass, charge, kinetic energy, excitation energy, spin, and parity, then uses the Hauser-Feshbach statistical theory to sample neutron and gamma-ray emission probabilities at each decay step while conserving energy, spin, and parity. The code computes energy-dependent neutron transmission coefficients from optical model calculations and gamma-ray transmission coefficients from a giant dipole resonance representation, then selects decay channels based on their emission probabilities until a ground state or isomeric state is reached. The Monte Carlo technique enables event-by-event reconstruction of complete fission events, storing all fragment and emitted particle information in ASCII history files for subsequent analysis.

## Key numbers

- The code handles spontaneous fission for 238,240,242,244Pu and 252,254Cf, and neutron-induced fission from thermal up to 20 MeV for n+233,234,235,238U, n+237Np, and n+239,241Pu
- Accurately calculating the average number of prompt fission neutrons requires about 50,000 fission events, while computing the full multiplicity distribution P(nu) requires many more events depending on tail precision
- The average prompt fission gamma multiplicity shows a dependence on the time coincidence window of 5 to 13 percent as reported in Talou (2016)
- Isomeric ratios computed for 235U(n_th,f) include: 83Se (0.79), 99Nb (0.91), 133Te (0.33), and 135Xe (0.55), using threshold times ranging from 0.1 to 1 second
- The default time coincidence window in CGMF is set at 10 ns, with the option for an infinite window using -1.0

## Context

This work extends the Hauser-Feshbach statistical model approach to fission fragment decay that was previously applied in earlier versions of CGMF and in the work of Stetcu (2013) on isomeric ratios. The code is designed to interface with the MCNP6 transport code and provides a consistent framework for studying prompt fission neutrons, prompt fission gamma rays, and independent fission yields simultaneously. The CGMFtk Python analysis toolkit enables comparison with experimental data for fission observables including spectra, multiplicities, and correlations, and the time-dependent gamma emission capabilities allow investigation of isomeric state populations that affect experimental measurements depending on the coincidence window used.
