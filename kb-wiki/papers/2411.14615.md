# CompactObject: An open-source Python package for full-scope neutron star equation of state inference

> The CompactObject package is an open-source software framework developed to constrain the neutron star equation of state (EOS) through Bayesian statistical inference. It integrates astrophysical observational constraints from X-ray timing, gravitational wave events, and radio measurements, as well as nuclear experimental constraints derived from perturbative Quantum Chromodynamics (pQCD) and Chiral Effective Field Theory ($χ$EFT). The package supports a diverse range of EOS models, including meta-model like and several physics-motivated EOS models. It comprises three independent components: an EOS generator module that currently provides seven EOS choices, a Tolman-Oppenheimer-Volkoff (TOV) equation solver, that allows the determination of the Mass Radius and Tidal deformability as observables, and a comprehensive Bayesian inference workflow module, including a complete pipeline for implementing EOS Bayesian inference. Each component can be used independently in different scientific research contexts, such as nuclear physics and astrophysics. In addition, CompactObject is designed to work in synergy with existing software such as CompOSE, allowing the use of the CompOSE EOS database to extend the EOS options available.

## Key claim

CompactObject provides an open-source, modular Python framework for Bayesian inference of the neutron star equation of state (EOS) that integrates both astrophysical observations and nuclear experimental constraints, but its accuracy and scope depend on the specific EOS models chosen by the user (including meta-models and physics-motivated models) and the particular observational datasets incorporated.

## Method

The package implements three independent components: an EOS generator module offering seven EOS choices (including polytropic and speed-of-sound meta-models, Relativistic Mean Field theory and its density-dependent variant, the MIT bag model, a strangeon matter EOS, and the Nambu-Jona-Lasinio model), a Tolman-Oppenheimer-Volkoff (TOV) equation solver that computes mass-radius relations and tidal deformability, and a Bayesian inference workflow module using either UltraNest (nested sampling) or emcee (Markov Chain Monte Carlo) sampling. The inference pipeline incorporates likelihood constraints from X-ray timing mass-radius measurements, radio timing observations, gravitational wave tidal deformability data, and nuclear physics constraints from saturation properties, perturbative Quantum Chromodynamics (pQCD), and Chiral Effective Field Theory (chiEFT). The package also interfaces with the CompOSE EOS database to extend available EOS options.

## Key numbers

- Seven EOS choices are currently available in the EOS generator module.
- Two sampling algorithm options are provided: UltraNest (nested sampling) and emcee (Markov Chain Monte Carlo sampling).
- The package has been utilized to derive constraints on nucleonic RMF models (Huang 2023) and hyperonic RMF models (Huang 2024), with ongoing projects constraining strangeon star EOS (Yuan 2024) and exploring phase transitions and twin stars.

## Context

This work extends prior Bayesian EOS inference efforts such as Raaijmakers (2023) that focused on meta-models spanning mass-radius parameter space, by instead emphasizing physics-motivated EOSs driven by microphysics. The package enables researchers to perform Bayesian model comparison across different EOS families (nucleonic, hyperonic, quark matter, strangeon matter) using a unified inference framework, and future plans include expanding the EOS options and conducting cross-comparisons of Bayesian evidence using current observational and experimental constraints.
