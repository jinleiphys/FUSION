# pi N --> eta N and eta N --> eta N partial wave T-matrices in a coupled, three channel model

> The pi N --> eta N and eta N --> eta N partial wave T-matrices for the eight lowest partial waves have been obtained in a three coupled channel model with unitarity manifestly imposed. The two physical channels are pi_N and eta_N, and the third channel, pi_pi_N is an effective, but unphysical two body channel which represents all remaining processes. The pi_N elastic phase shifts and the weighted data base of the pi- p --> eta n total and differential cross sections are chosen as the input for the fitting procedure. A model containing a single resonance in each of the three partial waves that dominates the eta production at lower energies is compared with previous analyses, based on similar assumptions. A multiresonance coupled channel model is introduced which significantly improves the agreement with all input data. Our results are compared with a complementary multiresonance coupled channel analysis that is constrained with elastic and continuum production channels. The inclusion of the fourth P11 resonance in the 1440 - 2200 MeV region further improves the agreement between the analysis and the data.

## Key claim
The paper provides a set of partial wave T-matrices for pi N -> eta N and eta N -> eta N for the eight lowest partial waves, obtained from a three-channel coupled model that imposes unitarity. The authors hedge that while a single-resonance model suffices for the near-threshold region (pion momenta below 850 MeV/c), a multiresonance model with background terms is necessary to describe data up to 2.5 GeV total center-of-mass energy, and that the inclusion of a fourth P11 resonance in the 1440-2200 MeV region further improves agreement.

## Method
The authors constructed a three-channel coupled model with pi N and eta N as physical channels and an effective pi pi N channel representing all remaining processes. They fitted the model to the Karlsruhe-Helsinki pi N elastic partial wave analysis and a weighted database of pi- p -> eta n total and differential cross sections, using a unitary multiresonance formalism with up to 132 parameters. Resonance parameters were extracted numerically from the full T-matrix by solving for poles and defining mass, width, and branching ratios via a generalized Breit-Wigner denominator.

## Key numbers
- The maximum total cross section for pi- p -> eta n is 2.8 mb, which is 6% of sigma_tot(pi- p -> all).
- The S-wave eta N scattering length from earlier work by Bhalerao and Liu (1985) is a_eta N = (0.27 + i 0.22) fm; from Arima et al. (1993) it is (0.98 + i 0.37) fm; from Wilkin (1993) it is (0.55 +/- 0.20 + i 0.30) fm.
- The single-resonance model covers the energy range corresponding to pion momenta less than 850 MeV/c.
- The multiresonance fit used 90 energies from threshold to 2.5 GeV total center-of-mass energy for pi N elastic data, and 81 energies for differential cross sections and 67 energies for total cross sections of pi N -> eta N.
- The mass parameters m_pi2 for the effective pi pi N channel range from 370 MeV to 650 MeV depending on the partial wave (e.g., 450 MeV for S11, 380 MeV for P11, 370 MeV for P13, 380 MeV for D13, 400 MeV for D15, 370 MeV for F15, 650 MeV for F17, 450 MeV for G17).

## Context
This work extends earlier single-resonance coupled-channel analyses by Bhalerao and Liu (1985), Bennhold and Tanabe (1991), and others, which failed to describe eta production data above 800 MeV pion kinetic energy. It provides a more complete set of partial wave T-matrices for use in higher-order processes such as pp -> pp eta, where previous work by Vetter (1991), Laget (1991), and Carr (1993) used only the dominant S11 partial wave. The results are complementary to the multiresonance K-matrix approach of Manley and Salesky (1992, 1993), which included eta production only in the S11 and F17 partial waves and for unitarity purposes rather than as a direct constraint. The obtained T-matrices enable calculation of eta N elastic scattering and improve the basis for studying quasi-bound eta-mesic nuclei.
