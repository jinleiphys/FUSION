# Symmetry-projected variational calculations with the numerical suite TAURUS II. Configuration mixing of symmetry-projected reference states

> We present the numerical codes TAURUS_pav and TAURUS_mix that, combined, perform the configuration mixing of symmetry-projected real general Bogoliubov quasiparticle states represented in a spherical harmonic oscillator basis. The model space considered is invariant under spatial and isospin rotations but no specific set of orbits is assumed such that the codes can carry out both valence-space and no-core calculations. In addition, no number parity is assumed for the Bogoliubov quasiparticle states such that the codes can be used to describe even-even, odd-even and odd-odd nuclei. To demonstrate the potential of the codes, we perform an example no-core calculation of $^{24}$Mg using a modern microscopic interaction.

## Key claim

The paper presents the numerical codes TAURUS_pav and TAURUS_mix that perform configuration mixing of symmetry-projected real general Bogoliubov quasiparticle states, but the actual demonstration is limited to a single no-core calculation of 24Mg using the EM1.8/2.0 chiral interaction, with the caveat that the ground state is underbound by approximately 57 MeV compared to experiment due to missing weak correlations.

## Method

The codes implement the Projected Generator Coordinate Method (PGCM) by constructing linear superpositions of parity, particle-number, and total-angular-momentum projected Bogoliubov quasiparticle reference states expanded in a spherical harmonic oscillator basis. The reference states are obtained through constrained Hartree-Fock-Bogoliubov minimizations using the code TAURUS_vap, with triaxial deformations in the (beta, gamma) plane as generator coordinates. The variational parameters are determined by solving a generalized eigenvalue problem (Hill-Wheeler-Griffin equation) after applying cutoffs to discard numerically unstable states, with matrix elements computed via the Generalized Wick Theorem and Pfaffian algebra for norm overlaps.

## Key numbers

* The example calculation for 24Mg uses a model space with e_max = 6, e_3max = 18, and hbar_omega = 12 MeV.
* The angular-momentum projection integral over Euler angles is discretized using 32 x 20 x 32 points.
* The particle-number projection integral uses 5 x 5 points in [0, pi] x [0, pi] for states without proton-neutron mixing, and 7 x 7 points in [0, 2pi] x [0, 2pi] for states with proton-neutron mixing.
* The projected energy minimum for 24Mg J^pi = 0^+_1 is located at deformation beta approximately 0.54 and gamma approximately 15 degrees.
* The ground state is underbound by approximately 57 MeV compared to the experimental energy.
* Cutoff values used in configuration mixing: epsilon_1 = 10^-5, epsilon_H = -10^+2, epsilon_A = 10^-3, epsilon_J = 10^-3, epsilon_n = 10^-6.

## Context

This work extends the TAURUS numerical suite, following the previously published TAURUS_vap code (Bally 2021), by providing publicly available tools for PGCM calculations that were previously only performed with proprietary codes such as HFODD (Dobaczewski 2021). The codes have already been used in several realistic calculations (Bally 2019, Duguet 2020, Sanchez 2021, Romero 2021, Yao 2020, Frosini 2022) and enable future studies combining symmetry projection with perturbation theory as proposed by Frosini (2022) and Duguet (2023).
