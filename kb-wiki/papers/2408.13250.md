# Variational inference of effective range parameters for ${}^3$He-${}^4$He scattering

> We use two different methods, Monte Carlo sampling and variational inference (VI), to perform a Bayesian calibration of the effective-range parameters in ${}^3$He-${}^4$He elastic scattering. The parameters are calibrated to data from a recent set of $^{3}$He-${}^4$He elastic scattering differential cross section measurements. Analysis of these data for $E_{\rm lab} \leq 4.3$ MeV yields a unimodal posterior for which both methods obtain the same structure. However, the effective-range expansion amplitude does not account for the $7/2^-$ state of ${}^7$Be so, even after calibration, the description of data at the upper end of this energy range is poor. The data up to $E_{\rm lab}=2.6$ MeV can be well described, but calibration to this lower-energy subset of the data yields a bimodal posterior. After adapting VI to treat such a multi-modal posterior we find good agreement between the VI results and those obtained with parallel-tempered Monte Carlo sampling.

## Key claim

Variational inference (VI) can reliably reproduce the posterior distribution of effective-range parameters for ${}^3$He-${}^4$He scattering obtained by Monte Carlo sampling, but only when the posterior is unimodal; for the bimodal posterior that arises from fitting data up to $E_{\rm lab}=2.6$ MeV, VI must be adapted by combining multiple optima using the ELBO estimate of Bayesian evidence.

## Method

The authors performed Bayesian calibration of effective-range parameters (ERPs) for ${}^3$He-${}^4$He elastic scattering using both Monte Carlo sampling (with parallel-tempered MCMC via `ptemcee` for multi-modal cases) and variational inference with a multivariate Gaussian family that includes correlations. The likelihood function incorporated experimental uncertainties and a theory covariance matrix based on next-order EFT contributions, with hyperparameters $\Lambda_B = 1.014\ \mathrm{fm}^{-1}$ and $\bar{c} = 0.7$ fixed from the convergence analysis of Poudel (2022). The posterior was conditioned on differential cross section data from Paneru covering $0.7 \leq E_{\rm lab} \leq 4.3$ MeV, with separate analyses for the full range and a subset restricted to $E_{\rm lab} \leq 2.6$ MeV.

## Key numbers

- The full dataset contained 398 data points; the restricted dataset ($E_{\rm lab} \leq 2.6$ MeV) contained 293 data points.
- The breakdown scale used for the EFT expansion parameter was $\Lambda_B = 1.014\ \mathrm{fm}^{-1}$.
- The rms value of EFT expansion coefficients was fixed to $\bar{c} = 0.7$.
- Normalization uncertainty priors $\sigma_{f_i}$ ranged from 0.041 to 0.098 across the 10 energy bins (0.706 to 5.484 MeV).
- Prior means for effective-range parameters included $\mu = 0.025\ \mathrm{fm}^{-1}$ for $1/a_0$, $\mu = 0.8\ \mathrm{fm}$ for $r_0$, and $\mu = 13.84$ for $(C_1^+)^2$.

## Context

This work extends the Bayesian calibration of halo EFT parameters for ${}^3$He-${}^4$He scattering performed by Poudel (2022), using the same EFT framework and data from Paneru. The authors demonstrate that variational inference can match MCMC results for unimodal posteriors while being markedly faster, addressing computational demands that have motivated emulator-based approaches (e.g., Melendez 2022, Drischler 2022). The extracted low-energy parameters are key inputs to calculations of the solar capture reaction ${}^3$He(${}^4$He,$\gamma$)${}^7$Be, which affects neutrino production in the pp-II and pp-III chains (Adelberger 2011).
