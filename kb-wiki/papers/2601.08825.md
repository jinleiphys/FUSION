# The Quantum Complexity of String Breaking in the Schwinger Model

> String breaking, the process by which flux tubes fragment into hadronic states, is a hallmark of confinement in strongly-interacting quantum field theories. A suite of quantum complexity measures is examined using Matrix Product States to characterize the string breaking process in the 1+1D Schwinger model. We demonstrate the presence of nonlocal quantum correlations along the string that may affect fragmentation dynamics, and show that entanglement and magic offer complementary perspectives on string formation and breaking beyond conventional observables.

## Key claim

The paper demonstrates that quantum complexity measures (entanglement entropy, magic, and mutual information) exhibit sharp, complementary signatures during string breaking in the Schwinger model, but the nonlocal magic is only present in the string state itself and vanishes once isolated mesons form, suggesting that longer-distance correlations are predominantly classical.

## Method

The authors use Matrix Product State simulations to compute the ground state of the 1+1D Schwinger model (Kogut-Susskind Hamiltonian with staggered fermions, mapped to spins via Jordan-Wigner transformation) as a function of static external charge separation d. They compute gauge-invariant bipartite entanglement entropy, Stabilizer Renyi Entropies (SREs), antiflatness (a lower bound on nonlocal magic), Robustness of Magic (RoM), and mutual information for various bipartitions of the lattice, using system sizes up to N=880 staggered sites with lattice spacings a=1, 1/2, and 1/4.

## Key numbers

- Vector meson mass M_v = 0.13647 (in lattice units) for parameters g=0.09, m_lat=0.045, a=1, giving M_v * a_phys ~ 0.2729 and M_v * L ~ 15.0117 for L=110
- Finite volume corrections suppressed by O(e^{-M_v L/2}) ~ 2 x 10^{-4} for d = L/2
- String breaking transition occurs at d ~ 46-50 physical sites (for N=880, a=1/4)
- Bipartite complexity measures (entanglement entropy, antiflatness, upper bound on nonlocal M_2) all peak at d = 46.5 physical sites
- The 4-tangle factorizes as tau^{(4)} ~ c * tau^{(2)} * tau^{(2)} with c ~ 1.78

## Context

This work extends previous studies of string breaking in the Schwinger model using tensor network methods (Buyens 2015, Grieninger 2025) and entanglement measures (Florio 2025) by introducing multiple quantum complexity measures simultaneously, including magic and nonlocal magic. It connects to broader efforts in quantum information applied to gauge theories (Haferkamp 2021, Cao 2024) and enables future dynamical simulations of string breaking via wavepacket evolution (Farrell 2024, Zemlevskiy 2024) and experimental detection of quantum correlations in high-energy collisions (White 2024, Yazgan 2025).
