# Continuum Linear Response in Coordinate Space Hartree-Fock-Bogoliubov Formalism for Collective Excitations in Drip-line Nuclei

> We formulate a continuum linear response theory on the basis of the Hartree-Fock-Bogoliubov formalism in the coordinate space representation in order to describe low-lying and high-lying collective excitations which couple to one-particle and two-particle continuum states. Numerical analysis is done for the neutron drip-line nucleus $^{24}$O. A low-lying collective mode that emerges above the continuum threshold with large neutron strength is analyzed. The collective state is sensitive to the density-dependence of the pairing. The present theory satisfies accurately the energy weighted sum rule. This is guaranteed by treating the pairing selfconsistently both in the static HFB and in the dynamical linear response equation.

## Key claim

The paper formulates a continuum linear response theory fully based on the coordinate space Hartree-Fock-Bogoliubov (HFB) formalism that includes, for the first time in a quasiparticle linear response framework, configurations where excited two quasiparticles both occupy continuum orbits above the threshold; the theory accurately satisfies the energy-weighted sum rule only when pairing is treated self-consistently in both the static HFB ground state and the dynamical linear response equation.

## Method

The authors derive the linear response equations from the time-dependent Hartree-Fock-Bogoliubov (TDHFB) theory in coordinate space representation, using a 2x2 matrix formalism for the generalized density and the HFB single-quasiparticle Green function. They construct the unperturbed response function via a contour integral representation in the complex energy plane, imposing outgoing-wave boundary conditions on the HFB Green function to treat continuum states properly. Numerical analysis is performed for the neutron drip-line nucleus 24O using a zero-range effective interaction, with the response computed for quadrupole excitations.

## Key numbers

- The analysis is performed for the neutron drip-line nucleus 24O.
- The one-particle separation energy is given by S1 = |λ|, where λ is the chemical potential; bound single-quasiparticle states exist only for energies E < |λ|.
- The energy-weighted sum rule is accurately satisfied when pairing self-consistency is maintained between the static HFB and the dynamical linear response.

## Context

This work extends the continuum linear response theory of Shlomo and Bertsch (1975) to include pairing correlations via the coordinate space HFB formalism, going beyond earlier continuum quasiparticle approaches that relied on the BCS approximation (Pl88FFSth, Bo96FFSGT, Kam98FFSe1, Hagino). It also improves upon QRPA models using discretized canonical bases (Engel) or discrete BCS quasiparticle bases (Khan-Giai, Khan-etal, Colo-2) by properly accounting for continuum coupling. The formalism enables systematic study of low-lying collective excitations in drip-line nuclei with open shell configurations, where pairing in continuum states plays an essential role.
