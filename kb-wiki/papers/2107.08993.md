# Connecting the proxy-SU(3) symmetry to the shell model

> Proxy-SU(3) symmetry is an approximation scheme extending the Elliott SU(3) algebra of the sd shell to heavier shells. When introduced in 2017, the approximation had been justified by calculations carried out within the Nilsson model. Recently our group managed to map the cartesian basis of the Elliott SU(3) model onto the spherical shell model basis, proving that the proxy-SU(3) approximation corresponds to the replacement of the intruder orbitals by their de Shalit-Goldhaber partners, paving the way for using the proxy-SU(3) approximation in shell model calculations. The connection between the proxy-SU(3) scheme and the spherical shell model has also been worked out in the original framework of the Nilsson model, with identical results.

# Connecting the proxy-SU(3) symmetry to the shell model

## Key claim
The proxy-SU(3) approximation, which replaces intruder orbitals in each harmonic oscillator shell with their de Shalit-Goldhaber partners, is justified by a unitary transformation mapping the cartesian Elliott SU(3) basis onto the spherical shell model basis, but this correspondence works strictly only for Nilsson orbitals possessing the highest total angular momentum j within their shell, the very orbitals that are replaced in the proxy-SU(3) scheme.

## Method
The authors constructed the unitary transformation matrix R·C that connects the cartesian Elliott basis [n_z n_x n_y m_s] to the spherical shell model basis in j-j coupling [n l j m_j], using the transformation from cartesian to spherical coordinates (R) followed by Clebsch-Gordan coupling (C). They then applied this transformation to show that the Nilsson 0[110] replacements used in proxy-SU(3) correspond to |0 1 1 0⟩ replacements in the spherical shell model basis, and verified the correspondence by examining expansions of Nilsson orbitals in the shell model basis at multiple deformation values.

## Key numbers
- The transformation matrix R·C for N=2 (sd shell) is explicitly tabulated, connecting 12 shell model orbitals |n,l,j,m_j⟩ to 12 Elliott cartesian basis states |n_z,n_x,n_y,m_s⟩.
- For the Nilsson orbital 3/2[541] at deformation ε=0.05, the leading shell model eigenvector |5 5 11/2 3/2⟩ has amplitude 0.9979; at ε=0.30 this amplitude drops to 0.9366.
- For the Nilsson orbital 3/2[651] at ε=0.05, the leading shell model eigenvector |6 6 13/2 3/2⟩ has amplitude 0.9966; at ε=0.30 this amplitude drops to 0.9010.
- For the Nilsson orbital 1/2[431] (not possessing highest j), at ε=0.05 the leading eigenvector |4 4 7/2 1/2⟩ has amplitude 0.9893, but at ε=0.30 the leading amplitude is only 0.7447 with several comparable contributions.
- The proxy-SU(3) shells span the following magic number ranges: U(3) 6-14, U(6) 14-28, U(10) 28-50, U(15) 50-82, U(21) 82-126, U(28) 126-184.

## Context
This work extends the proxy-SU(3) symmetry introduced by Bonatsos et al. (2017), which was initially justified through Nilsson model calculations, by providing a formal connection to the spherical shell model basis. The mapping shows that the proxy-SU(3) replacements are identical to the de Shalit-Goldhaber pairs (1953) and the Federman-Pittel pairs (1977), expressed as |0 1 1 0⟩ in shell model notation. This connection enables the use of proxy-SU(3) symmetry in shell model calculations for heavy nuclei, analogous to the symmetry-adapted no-core shell model approach used in light nuclei by Launey et al., and complements the pseudo-SU(3) framework of Arima, Harvey, and others.
