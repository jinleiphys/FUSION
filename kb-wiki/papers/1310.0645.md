# A new Bayesian approach to the reconstruction of spectral functions

> We present a novel approach for the reconstruction of spectra from Euclidean correlator data that makes close contact to modern Bayesian concepts. It is based upon an axiomatically justified dimensionless prior distribution, which in the case of constant prior function $m(ω)$ only imprints smoothness on the reconstructed spectrum. In addition we are able to analytically integrate out the only relevant overall hyper-parameter $α$ in the prior, removing the necessity for Gaussian approximations found e.g. in the Maximum Entropy Method. Using a quasi-Newton minimizer and high-precision arithmetic, we are then able to find the unique global extremum of $P[ρ|D]$ in the full $N_ω\gg N_τ$ dimensional search space. The method actually yields gradually improving reconstruction results if the quality of the supplied input data increases, without introducing artificial peak structures, often encountered in the MEM. To support these statements we present mock data analyses for the case of zero width delta peaks and more realistic scenarios, based on the perturbative Euclidean Wilson Loop as well as the Wilson Line correlator in Coulomb gauge.

# A new Bayesian approach to the reconstruction of spectral functions

## Key claim

The authors present a Bayesian spectral function reconstruction method that, by using an axiomatically derived prior distribution and analytically integrating out the hyperparameter α, avoids artificial peak structures common in the Maximum Entropy Method (MEM) and yields gradually improving results as input data quality increases, provided the search space dimensionality is decoupled from the number of Euclidean time points.

## Method

The method modifies the likelihood to enforce the condition L = Nτ for a neutral reconstruction, then constructs a prior probability P[ρ|I] = exp[S] from axioms of subset independence, scale invariance, smoothness, and maximum at the prior, yielding S = α ∫ dω (1 − ρ/m + ln(ρ/m)). The hyperparameter α is integrated out analytically from the joint probability distribution, and the unique global extremum of P[ρ|D] is found using a quasi-Newton LBFGS minimizer in the full Nω ≫ Nτ dimensional search space with high-precision arithmetic.

## Key numbers

- Three delta peak mock spectrum: peaks at ω = −0.5 (ρ = 10⁻¹⁴), ω = 1.5 (ρ = 1/26), ω = 3 (ρ = 25/26); reconstructed from Nτ = 32 datapoints with Gaussian noise at ΔD/D = 10⁻³, 10⁻⁴, 10⁻⁵; Nω = 1200 points on ω ∈ [−2, 6]; 512-bit arithmetic used.
- For the Wilson loop reconstruction at r = 0.264 fm and T = 2.33 × 270 MeV: Nτ = 32, 128, 256 with ΔD/D = 10⁻⁴, 10⁻⁵, 10⁻⁶; Nω = 1000 on ω ∈ [−126, 189] GeV; peak position reproduced within 10% of correct value but width poorly reproduced.
- For the Wilson line reconstruction at r = 0.264 fm and T = 2.33 × 270 MeV: Nτ = 16, 32, 128 with ΔD/D = 10⁻², 10⁻⁴, 10⁻⁵; Nω = 1200 on ω ∈ [−15.7, 15.7] GeV; with Nτ = 128 and ΔD/D = 10⁻⁵, width reproduced with sub 20% deviation.
- For ΔD/D = 10⁻⁵ in the delta peak test, the optimizer step size diminishes to Δ = 10⁻⁶⁰ within several hours, while MEM values of Δ < 10⁻⁹ are not encountered even after one week.

## Context

This work extends the Bayesian inference approach to spectral function reconstruction used in lattice QCD, which was introduced to the community as the Maximum Entropy Method by Asakawa (2000) and Jarrell (1996). It addresses technical shortcomings of the MEM identified in prior studies by Rothkopf (2011), including the limited dimensionality of the search space in Bryan's standard implementation, flat directions in the Shannon-Jaynes entropy functional, and the Gaussian approximation treatment of the hyperparameter α. The method is applied to the heavy quark potential problem at finite temperature, building on hard-thermal loop calculations of the Wilson loop and Wilson line correlator by Burnier (2013), and enables improved reconstructions of spectral features from lattice simulations.
