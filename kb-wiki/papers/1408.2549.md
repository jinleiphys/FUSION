# Improved version of the PHOBOS Glauber Monte Carlo

> Glauber models are used to calculate geometric quantities in the initial state of heavy ion collisions, such as impact parameter, number of participating nucleons and initial eccentricity. Experimental heavy-ion collaboration, in particular at RHIC and LHC, use Glauber Model calculations for various geometric observables. In this document, we describe the assumptions inherent to the approach, and provide an updated implementation (v2) of the Monte Carlo based Glauber Model calculation, which originally was used by the PHOBOS collaboration. The main improvement w.r.t. the earlier version (arXiv:0805.4411) are the inclusion of Tritium, Helium-3, and Uranium, as well as the treatment of deformed nuclei and Glauber-Gribov fluctuations of the proton in p+A collisions. A users' guide (updated to reflect changes in v2) is provided for running various calculations.

## Key claim

The paper presents an updated version (v2) of the PHOBOS Monte Carlo Glauber model, which improves upon the earlier version (v1, arXiv:0805.4411) by adding support for Tritium, Helium-3, Uranium, deformed nuclei, and Glauber-Gribov fluctuations of the proton in p+A collisions, while noting that the code continues to be revised with additional elements and features in subsequent versions up to v2.7.

## Method

The Monte Carlo Glauber calculation proceeds in two steps: first, nucleon positions within each nucleus are stochastically sampled from probability density functions (typically Fermi distributions for spherical nuclei, with parameters from electron scattering data, or from wavefunction-based calculations for light nuclei); second, the two nuclei are "collided" under the eikonal approximation, where nucleons travel along straight-line trajectories and two nucleons from different nuclei are considered to have collided if their relative transverse distance is less than the ball diameter D = sqrt(sigma_NN/pi), where sigma_NN is the inelastic nucleon-nucleon cross section. The code is implemented within the ROOT framework and provides classes for nucleons, nuclei, and the main steering class TGlauberMC, along with several user functions for generating and saving event-by-event quantities.

## Key numbers

- Inelastic nucleon-nucleon cross section at top RHIC energy (sqrt(s_NN)=200 GeV): sigma_NN = 42 mb
- Inelastic nucleon-nucleon cross section at LHC for sqrt(s_NN)=2.76 TeV: sigma_NN = 64 mb
- Inelastic nucleon-nucleon cross section at LHC for sqrt(s_NN)=5.02 TeV: sigma_NN = 70 mb
- Systematic uncertainty variations for sigma_NN: typically +/- 3 mb at RHIC and +/- 5 mb at LHC
- Default minimum inter-nucleon separation within a nucleus: d_min = 0.4 fm
- Maximum impact parameter for sampling: b_max approximately 20 fm, greater than 2R_A
- Hulthén form parameters for deuteron: a = 0.228 fm^{-1}, b = 1.177 fm^{-1}
- Proton radius parameter for exponential profile: R = 0.234 fm (based on rms charge radius)
- Nuclear radius for Au-197: R = 6.38 fm, skin depth a = 0.535 fm
- Nuclear radius for Pb-207: R = 6.62 fm, skin depth a = 0.546 fm
- Nuclear radius for U-238 (deformed): R = 6.67 fm, skin depth a = 0.44 fm, deformation parameters beta_2 = 0.280, beta_4 = 0.093

## Context

This work extends the original PHOBOS Glauber Monte Carlo code (Alver 2008) by incorporating additional nuclear species and physics features, and it provides the standard tool used by RHIC and LHC experiments for centrality determination in heavy-ion collisions. The code enables calculations of geometric quantities such as number of participants, number of binary collisions, impact parameter, and eccentricities, which are essential for interpreting experimental data from Pb+Pb collisions at the LHC (Abelev 2013, Chatrchyan 2011, ATLAS 2011) and p+Pb collisions (Aad 2015, Adam 2014), as well as for providing initial conditions for hydrodynamic simulations (Bozek 2014, Koop 2015, Yan 2015).
