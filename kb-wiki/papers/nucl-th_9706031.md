# Relativistic Hartree-Bogoliubov theory in coordinate space: finite element solution for a nuclear system with spherical symmetry

> A C++ code for the solution of the relativistic Hartree-Bogoliubov theory in coordinate space is presented. The theory describes a nucleus as a relativistic system of baryons and mesons. The RHB model is applied in the self-consistent mean-field approximation to the description of ground state properties of spherical nuclei. Finite range interactions are included to describe pairing correlations and the coupling to particle continuum states. Finite element methods are used in the coordinate space discretization of the coupled system of Dirac-Hartree-Bogoliubov integro-differential eigenvalue equations, and Klein-Gordon equations for the meson fields. The bisection method is used in the solution of the resulting generalized algebraic eigenvalue problem, and the biconjugate gradient method for the systems of linear and nonlinear algebraic equations, respectively.

## Key claim

The paper presents a C++ code that solves the relativistic Hartree-Bogoliubov (RHB) equations in coordinate space using finite element methods for spherical nuclei, but the solution is restricted to single closed-shell nuclei with spherical symmetry and the code only considers contributions from J=0 pairs to the pairing matrix elements.

## Method

The coupled system of Dirac-Hartree-Bogoliubov integro-differential eigenvalue equations and Klein-Gordon equations for the meson fields is discretized on a radial finite element mesh using the method of weighted residuals (Galerkin method) with Lagrange shape functions of arbitrary order on an adaptive non-uniform radial mesh. The resulting generalized algebraic eigenvalue problem is solved using the bisection method for band matrices, while the biconjugate gradient method is used for the systems of linear and nonlinear algebraic equations arising from the Klein-Gordon equations. Pairing correlations are described by finite range Gogny forces, and the self-consistent iteration transforms the quasi-particle eigenspectrum into the canonical basis of single-particle states by diagonalizing the matrix R_{kk'} = <V_k|V_{k'}>.

## Key numbers

- The Dirac gap in vacuum equals two times the nucleon mass (2m); in a nucleus the gap extends between the sum and difference of the scalar sigma-meson potential and vector omega-meson potential
- The angular quantum number kappa takes values kappa = +/- (j+1/2) for j = l -/+ 1/2, with kappa = +/- 1, +/- 2, +/- 3, ...
- The pairing interaction uses the finite range Gogny force with two Gaussian terms: V^{pp}(1,2) = sum_{i=1,2} exp(-(r_1 - r_2)^2 / mu_i^2) (W_i + B_i P^sigma - H_i P^tau - M_i P^sigma P^tau)
- The eigenvalue problem dimension is four times the number of nodes on the finite element mesh

## Context

This work extends the authors' previous finite element method applications to relativistic mean-field equations for one-dimensional nuclear matter slabs and spherical doubly closed-shell nuclei (Poeschl, Vretenar, Ring 1996, 1997) by including pairing correlations via the RHB framework. It contrasts with earlier RHB calculations for Sn and Pb isotopes (Lalazissis, et al. 1996) that used oscillator basis expansions, which provide poor approximations to continuum states for drip-line nuclei; the coordinate space approach enables correct description of coupling between bound and continuum states, as previously demonstrated for the two-neutron halo in 11Li (Meng and Ring 1996) but only with zero-range pairing forces. The code enables future calculations of ground state properties of spherical nuclei with finite range pairing interactions, particularly for exotic nuclei near the drip lines.
