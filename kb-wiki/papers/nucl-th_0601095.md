# The new computer program for three dimensional relativistic hydrodynamical model

> An effective computer program for three dimensional relativistic hydrodynamical model has been developed. It implements a new approach to the early hot phase of relativistic heavy-ion collisions. The computer program simulates time-space evolution of nuclear matter in terms of ideal-fluid dynamics. Equations of motions of hydrodynamics are solved making use of finite difference methods. Commonly-used algorithms of numerical relativistic hydrodynamics RHLLE and MUSTA-FORCE have been applied in simulations. To speed-up calculations, parallel processing has been made available for solving hydrodynamical equations. The test results of simulations for 3D, 2D and Bjorken expansion are reported in this paper. As a next step we plan to implement the hadronization algorithm by implementing the continuous particle emission for freeze-out and comparing it with Cooper-Frye formula.

# The new computer program for three dimensional relativistic hydrodynamical model

## Key claim
The paper presents a new computer program for three-dimensional relativistic hydrodynamics that implements a novel approach to the early hot phase of relativistic heavy-ion collisions, but the program is still under development and has not yet implemented the hadronization algorithm for freeze-out, which is planned as a next step.

## Method
The program solves the relativistic ideal-fluid hydrodynamic equations using finite difference methods on a three-dimensional lattice, employing operator splitting to reduce the problem to a sequence of one-dimensional equations. Two numerical flux algorithms are implemented: the HLLE (Harten-Lax-van Leer-Einfeldt) Godunov-type method and the MUSTA-FORCE multi-stage approach. Parallel processing is enabled through two independent frameworks: the PVMFlower project using the Parallel Virtual Machine framework and the Hydrogrid project using TCP/IP network protocols.

## Key numbers
- Shock tube test: solved with MUSTA-FORCE flux using Δx = 0.02, Δt = Δx/4, grid of 500 zones, 800 time steps, ideal gas equation of state with adiabatic index Γ = 1.4 (arbitrary units)
- Ellipsoidal flow test: comparison of exact and numerical solutions at time t = 4 fm/c, with non-zero velocity for |r| > 4.5 fm attributed to numerical dissipation (the maximal range of matter defined by velocity of light is 4.5 fm)
- Adiabatic index range for the ideal gas equation of state: 1 ≤ Γ ≤ 2

## Context
This work extends the analytic solutions for three-dimensional relativistic ellipsoidal flow proposed by the authors' group (Kikola et al., 2005) and provides a numerical framework to test such solutions. The program is designed to address problems with the Cooper-Frye freeze-out prescription, which produces issues when the hypersurface is not space-like and contradicts cascade model results showing continuous particle emission. The next planned step is to implement continuous particle emission for freeze-out and compare it with the Cooper-Frye formula, which would enable more realistic simulations of heavy-ion collisions at ultra-relativistic energies.
