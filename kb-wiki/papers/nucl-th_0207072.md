# Near threshold neutral pion electroproduction on deuterium in chiral perturbation theory

> Near threshold neutral pion electroproduction on the deuteron is studied in the framework of baryon chiral perturbation theory at next-to-leading order in the chiral expansion. We develop the multipole decomposition for pion production off spin-1 particles appropriate for the threshold region. The existing data at photon virtuality k^2 = -0.1 GeV^2 can be described satisfactorily. Furthermore, the prediction for the S-wave multipole E_d at the photon point is in good agreement with the data.

## Key claim

Near threshold neutral pion electroproduction on the deuteron can be described satisfactorily at next-to-leading order in baryon chiral perturbation theory, but only if the single scattering contribution includes the full fourth-order result for the transverse and longitudinal S-waves, with parameters fixed from recent data; the description works for the existing data at photon virtuality \(k^2 = -0.1\,\text{GeV}^2\) but does not resolve the puzzle of newer proton data at \(k^2 = -0.05\,\text{GeV}^2\).

## Method

The authors develop a multipole decomposition for pion production off a spin-1 target (the deuteron) appropriate for the threshold region, expanding the invariant matrix element in terms of 13 invariant functions and projecting out electric, magnetic, and longitudinal multipoles. They compute the single scattering contribution using the full fourth-order chiral perturbation theory result for the proton and neutron S-wave multipoles, with low-energy constants fixed from fits to MAMI and NIKHEF data, and evaluate three-body (meson-exchange) corrections to third order in the chiral expansion. The calculation uses chiral effective field theory deuteron wave functions consistent with the power counting of the kernel, and includes the pion mass difference as the dominant isospin breaking effect.

## Key numbers

- Photon virtuality \(k^2 = -0.1\,\text{GeV}^2\): the existing MAMI data at this value can be described satisfactorily by the calculation.
- Photon virtuality \(k^2 = -0.05\,\text{GeV}^2\): newer MAMI measurements at this lower virtuality show results that cannot be explained in chiral perturbation theory or any sophisticated model.
- The S-wave multipole \(E_d\) at the photon point (\(k^2 = 0\)) is in good agreement with the data.
- The longitudinal deuteron multipole \(L_d\) is extracted from the MAMI data at \(k^2 = -0.1\,\text{GeV}^2\) to pin down one parameter related to a dimension-five operator.
- The cut-off range for the chiral EFT wave functions is \(\Lambda = 500\ldots 600\,\text{MeV}\).

## Context

This work extends the authors' previous threshold calculation (Krebs, Bernard, Meißner, 2001) by including above-threshold kinematics and the full fourth-order single scattering contribution, which was previously only shifted to its photon-point value. It addresses a significant discrepancy between the earlier prediction and the MAMI data (Ewald, 2001) for coherent neutral pion production on deuterium at \(k^2 = -0.1\,\text{GeV}^2\), and shows that the improved calculation resolves that discrepancy. The results deepen the mystery surrounding the MAMI proton data (Merkel, 2001) at \(k^2 = -0.05\,\text{GeV}^2\), which remain unexplained by chiral perturbation theory or any model, and call for separate investigation of that puzzle.
