# Two-photon exchange correction to muon-proton elastic scattering at low momentum transfer

> We evaluate the two-photon exchange (TPE) correction to the muon-proton elastic scattering at small momentum transfer. Besides the elastic (nucleon) intermediate state contribution, which is calculated exactly, we account for the inelastic intermediate states by expressing the TPE process approximately through the forward doubly virtual Compton scattering. The input in our evaluation is given by the unpolarized proton structure functions and by one subtraction function. For the latter, we provide an explicit evaluation based on a Regge fit of high-energy proton structure function data. It is found that, for the kinematics of the forthcoming muon-proton elastic scattering data of the MUSE experiment, the elastic TPE contribution dominates, and the size of the inelastic TPE contributions is within the anticipated error of the forthcoming data.

## Key claim

The two-photon exchange (TPE) correction to muon-proton elastic scattering at low momentum transfer is dominated by the elastic (nucleon) intermediate state contribution, and the inelastic contributions from all other intermediate states are estimated to be within the anticipated experimental error of the forthcoming MUSE experiment, though the empirical determination of the subtraction function needed for the inelastic part is only accurate to "the correct order of magnitude" due to mismatches between structure function fits.

## Method

The elastic TPE contribution was calculated exactly using the full proton electromagnetic vertex with form factors. The inelastic TPE contribution was approximated by expressing the hadronic tensor through the forward doubly virtual Compton scattering (VVCS) tensor in the low-momentum-transfer limit, using the unpolarized proton structure functions \(F_1\) and \(F_2\) and a subtraction function. The subtraction function was determined empirically via a Regge fit to high-energy proton structure function data (Donnachie-Landshoff fit) combined with the Christy-Bosted fit for the resonance region, using an unsubtracted dispersion relation for the amplitude \(\mathrm{T}_1 - \mathrm{T}_1^\mathrm{R}\).

## Key numbers

- The elastic TPE contribution provides a reduction of about 40% at \(Q^2 \approx 0.025~\mathrm{GeV}^2\) when including proton form factors compared to the point-like Dirac particle case.
- The magnetic polarizability at zero virtuality is \(\beta_M = (2.5 \pm 0.4) \times 10^{-4}~\mathrm{fm}^3\) (PDG value).
- The heavy-baryon chiral perturbation theory fit for the subtraction function uses a dipole form with \(\Lambda = 530 - 842~\mathrm{MeV}\).
- The baryon chiral perturbation theory decomposition gives \(\beta_{\Delta}(0) \simeq 7 \times 10^{-4}~\mathrm{fm}^3\), \(\beta_{\pi N}(0) = -2 \times 10^{-4}~\mathrm{fm}^3\), and \(\beta_{\pi \Delta}(0) = -1.2 \times 10^{-4}~\mathrm{fm}^3\).
- The subtraction function \(\mathrm{T}^{\mathrm{subt}}_1(0, \tilde{Q}^2)\) is compatible with zero at \(\tilde{Q}^2 = 0\) within 1–1.5\(\sigma\).
- The subtraction function changes sign in the range between 0.1 and 0.4 \(\mathrm{GeV}^2\).

## Context

This work extends the authors' previous estimate of the elastic TPE contribution (Tomalak 2014) by including inelastic intermediate states for muon-proton scattering at low momentum transfer, relevant for the MUSE experiment (Gilman 2013). It provides an empirical determination of the subtraction function using a Regge fit (Donnachie-Landshoff 2004) and the Christy-Bosted fit (2007) for the proton structure function \(F_1\), and compares with chiral perturbation theory results from Birse (2012) and Alarcon (2013). The results enable a more complete theoretical correction for extracting the proton charge radius from muon-proton scattering, helping to address the proton radius puzzle.
