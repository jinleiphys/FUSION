# Generalized parton distributions for $q\bar{q}$ pions

> We use a two-body, light-cone pion wave function (which vanishes when one of the constituents carries all of the plus-momentum) to construct a double distribution. We show that the resulting generalized parton distribution has incorrect behavior at the crossover between kinematic regions. Furthermore the positivity constraints are not respected. On the other hand, the field-theoretic, two-body double distribution gives generalized parton distributions that have physically correct crossover behavior and respect the positivity constraints independent of the model wave function used. However, the sum rule and polynomiality conditions cannot be satisfied with a two-body wave function alone since Lorentz symmetry is not maintained.

## Key claim

The paper demonstrates that a two-body light-cone pion wave function used to construct generalized parton distributions (GPDs) via double distributions yields incorrect behavior at the crossover kinematic point \(X = \zeta\) and violates positivity constraints, while a field-theoretic two-body double distribution corrects these issues but cannot satisfy sum rules and polynomiality conditions because Lorentz symmetry is not maintained.

## Method

The authors constructed a double distribution from a two-body light-cone pion wave function that vanishes when one constituent carries all plus-momentum, then derived the corresponding GPD using the standard integral relation between double distributions and GPDs. They compared this model GPD to one obtained from a field-theoretic double distribution derived directly from the Fock space representation of the matrix element defining the double distribution, using the same wave function but respecting the underlying field operator definition. The crossover behavior and positivity constraints were evaluated for both approaches, and the factorized "realistic" modification of the quark distribution function was also tested.

## Key numbers

- The model GPD at the crossover \(\mathcal{F}_{\zeta}(\zeta,t)\) is non-zero for all values of \(\zeta\) and \(t\) shown (e.g., \(\zeta = 0.2, 0.4, 0.6\) with \(-t/m^2\) ranging from 0 to 4), despite the quark distribution function vanishing at \(x=0\).
- The ratio \(R(X,\zeta) = (1 - \zeta/2) |\mathcal{F}_{\zeta}(X,0)| / \sqrt{q(X) q((X-\zeta)/(1-\zeta))}\) exceeds 1 for the model GPD for all \(X > \zeta\) at \(\zeta = 0.4\), violating the positivity constraint; the factorized realistic model satisfies the constraint.
- The field-theoretic double distribution yields a GPD that vanishes at the crossover \(X = \zeta\) and satisfies positivity, but retains explicit \(\zeta\)-dependence in the double distribution itself, breaking Lorentz symmetry.

## Context

This work extends the analysis of Mukherjee (2002) who proposed using two-body light-cone wave functions to construct pion GPDs via double distributions, following the double distribution formalism of Radyushkin (1997, 1998). The paper shows that the Mukherjee (2002) model violates both crossover behavior constraints derived from Fock space arguments (Brodsky 2001, Diehl 2000) and positivity constraints (Radyushkin 1998, Pire 1998), while the field-theoretic construction preserves these properties at the cost of Lorentz symmetry. This cautions against using such valence-only models for interpreting experimental data from deeply virtual Compton scattering, where the crossover region is accessed through beam-spin asymmetries (Saull 1999) and charge asymmetries (Belitsky 2000).
