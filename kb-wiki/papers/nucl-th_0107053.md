# A method of implementing Hartree-Fock calculations with zero- and finite-range interactions

> We develop a new method of implementing the Hartree-Fock calculations. A class of Gaussian bases is assumed, which includes the Kamimura-Gauss basis-set as well as the set equivalent to the harmonic-oscillator basis-set. By using the Fourier transformation to calculate the interaction matrix elements, we can treat various interactions in a unified manner, including finite-range ones. The present method is numerically applied to the spherically-symmetric Hartree-Fock calculations for the oxygen isotopes with the Skyrme and the Gogny interactions, by adopting the harmonic-oscillator, the Kamimura-Gauss and a hybrid basis-sets. The characters of the basis-sets are discussed. Adaptable to slowly decreasing density distribution, the Kamimura-Gauss set is suitable to describe unstable nuclei. A hybrid basis-set of the harmonic-oscillator and the Kamimura-Gauss ones is useful to accelerate the convergence, both for stable and unstable nuclei.

# A method of implementing Hartree-Fock calculations with zero- and finite-range interactions

Nakada, H.; Sato, M. (2001)

## Key claim

The authors develop a new Hartree-Fock method using a class of Gaussian bases that includes both the harmonic-oscillator and Kamimura-Gauss basis-sets, unified through Fourier transformation for computing interaction matrix elements. The method is claimed to be suitable for describing unstable (drip-line) nuclei because the Kamimura-Gauss basis-set can reproduce the exponential asymptotic density distribution, but the practical convergence depends on careful parameter selection for the basis-set (range parameters, common ratio, and number of bases).

## Method

The authors implement spherically-symmetric Hartree-Fock calculations using single-particle basis functions of the form $R_{\alpha\ell j}(r) = {\cal N}_{\alpha\ell j} r^{\ell+2p_\alpha}\exp[-(r/\nu_\alpha)^2]$, which includes both the harmonic-oscillator basis-set (fixed $\nu_\alpha$, varying $p_\alpha$) and the Kamimura-Gauss basis-set ($p_\alpha=0$, $\nu_\alpha$ in geometric progression). Two-body interaction matrix elements for central, LS, and tensor forces are computed via Fourier transformation of the interaction, reducing the spatial integrals to one-dimensional $k$-space integrals that are analytic for delta, Gaussian, and Yukawa forms. The method is applied to oxygen isotopes using the Skyrme SLy4 parameter-set and the Gogny D1S interaction, with exact treatment of the Coulomb interaction.

## Key numbers

- For the harmonic-oscillator-equivalent basis-set, numerical instability occurs for $K \geq 10$ bases in double precision; the authors restrict to $K=7$.
- For the Kamimura-Gauss basis-set, the common ratio is fixed at $b=1.33$; density distributions show bumpy structure for $b \geq 1.35$.
- The harmonic-oscillator range parameter is set by $\hbar\omega = 41.2 \times 24^{-1/3}$ MeV for oxygen isotopes.
- The Kamimura-Gauss basis-set uses $\nu_4 = \nu_\omega$ (the harmonic-oscillator range) as one of its range parameters, with $\nu_1 = \nu_\omega b^{-3}$.

## Context

This work extends the Hartree-Fock methodology to handle both zero-range (Skyrme) and finite-range (Gogny) interactions within a unified basis framework, addressing limitations of the harmonic-oscillator basis-set for describing loosely bound nuclei near the drip line. The method contrasts with the transformed harmonic-oscillator (THO) approach of Stoitsov et al. (1998), which was developed for the same purpose but depends on the characteristics of zero-range interactions and may not easily handle finite-range forces. The Kamimura-Gauss basis-set was previously shown to work efficiently in few-body systems by Kamimura (1988) and collaborators. This work enables systematic Hartree-Fock studies of unstable nuclei with various effective interactions, including finite-range forces beyond the standard Gogny D1S parameter-set.
