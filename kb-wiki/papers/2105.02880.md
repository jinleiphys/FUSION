# On the numerical evaluation of real-time path integrals: Double exponential integration and the Maslov correction

> Ooura's double exponential integration formula for Fourier transforms is applied to the oscillatory integrals occuring in the path-integral description of real-time Quantum Mechanics. Due to an inherent, implicit regularization multi-dimensional Gauss-Fresnel integrals are obtained numerically with high precision but modest number of function calls. In addition, the Maslov correction for the harmonic oscillator is evaluated numerically with an increasing number of time slices in the path integral thereby clearly demonstrating that the real-time propagator acquires an additional phase $ - π/2 $ each time the particle passes through a focal point. However, in the vicinity of these singularities an overall small damping factor is required. Prospects of evaluating scattering amplitudes of finite-range potentials by direct numerical evaluation of a real-time path integral are discussed.

# On the numerical evaluation of real-time path integrals: Double exponential integration and the Maslov correction

## Key claim

Ooura's double exponential integration formula for Fourier transforms can numerically evaluate multi-dimensional Gauss-Fresnel integrals arising in real-time path integrals with high precision and modest function calls, but only if an overall small damping factor is introduced in the vicinity of focal points where the Maslov correction (an additional phase of -pi/2 each time the particle passes through a focal point) occurs.

## Method

The author applies Ooura's double exponential integration formula for oscillatory integrals to the N-dimensional Gauss-Fresnel integral expressed in hyperspherical coordinates, performing the radial integration via Ooura's method and angular integrations via standard deterministic or Monte-Carlo routines. For the harmonic oscillator propagator, the time-sliced path integral is evaluated with N intermediate integrations using the symmetric Trotter product formula, and the prefactor is normalized to the free case to isolate the Maslov phase.

## Key numbers

- Euler's constant obtained from integrating -ln(x) sin(x) from 0 to infinity to absolute accuracy of 10^{-12} with only 80 function calls (Table 2 of Ooura and Mori)
- Gauss-Fresnel integral evaluated numerically for omega = +/- 1 across a wide range of dimensions N with relative complex deviation typically below 10^{-6} using k_max = 50-80 and epsilon_Ooura = 10^{-8}
- For N=1 (2 time slices), the Maslov phase of -pi/2 at the focal point tau = sqrt(8) = 2.828 is accurately reproduced
- For N=2 (3 time slices), damping factor eta = 0.001 is required to achieve agreement near the two focal points
- For N=3, damping eta = 0.01 is used with adaptive Gauss-Chebyshev (max 511 function calls) and VEGAS Monte-Carlo (2.5 x 10^4 function calls)

## Context

This work extends Ooura and Mori's double exponential integration method for oscillatory integrals (Ooura 2005) to the direct numerical evaluation of real-time path integrals, which are typically treated via imaginary-time Monte-Carlo methods or semiclassical approximations (Makri). The Maslov correction for the harmonic oscillator, a genuine quantum-mechanical phenomenon absent in Euclidean versions, is demonstrated numerically, confirming the analytical result of Hormander (1971) and the treatment in Schulman (1981) and Kleinert (2004). The method enables prospects for evaluating scattering amplitudes of finite-range potentials by direct numerical evaluation of real-time path integrals.
