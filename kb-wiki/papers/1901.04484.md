# GLISSANDO 3: GLauber Initial-State Simulation AND mOre..., ver. 3

> We present ver. 3 of GLISSANDO, a versatile Monte-Carlo generator for Glauber-like models of the initial stages of ultra-relativistic heavy-ion collisions. The present version incorporates the wounded quark model, which is phenomenologically successful in reproducing multiplicities of particle production at RHIC and the LHC. Within this model, one can study the nucleon substructure fluctuation effects, recently explored in p-A collisions. In addition, the code includes the possibility of investigating collisions of light nuclei, such as $^{3}$He and $^{3}$H, or the $α$-clustered $^{7,9}$Be, $^{12}$C, and $^{16}$O, where the deformation of the intrinsic wave function influences the transverse shape of the initial state. The current version, being down-compatible, retains the functionality of the previous releases, such as incorporation of various variants of Glauber-like models, a smooth $NN$ inelasticity profile in the impact parameter obtained from a parametrization of experimental data, fluctuating strength of the entropy deposition, or realistic nuclear distributions of heavy nuclei with deformation. The code can provide output in the format containing the event-by-event source location, which may be further used in modeling the intermediate evolution phase, e.g., with hydrodynamics or transport models. The interface is simplified, such that in the control input file the user may supply only the very basic information, such as the collision energy, the mass numbers of the colliding nuclei, and the model type. GLISSANDO 3 is integrated with the CERN ROOT platform. The package includes numerous illustrative and useful ROOT scripts to compute and plot various results.

## Key claim

GLISSANDO 3 extends the previous Glauber Monte Carlo framework by incorporating the wounded quark (parton) model and collisions of light clustered nuclei, claiming that these additions are phenomenologically successful in reproducing multiplicities at RHIC and the LHC, but the actual implementation requires the user to choose between the wounded nucleon and wounded parton models via preprocessor options, and the agreement with data depends on the specific parameterization of the parton-parton cross section and nucleon substructure.

## Method

The code implements a Monte Carlo simulation of the initial stage of ultra-relativistic heavy-ion collisions using Glauber-like models, where the user selects either the wounded nucleon or wounded parton model and specifies the collision energy and mass numbers of the colliding nuclei. For the wounded parton model, nucleons are composed of p partons (default p=3 for quarks) distributed according to a radial density with a scale parameter r0, and parton-parton inelasticity is taken as Gaussian; the parameters are optimized so that the resulting NN inelasticity profile matches the COMPETE model parametrization of experimental pp scattering data. For light clustered nuclei (7Be, 9Be, 12C, 16O), the code generates nucleon positions based on alpha-cluster geometries with fixed inter-cluster distances and cluster widths, using external nuclear configurations for 3He and 3H from Wiringa (2013) and Lonardoni (2017).

## Key numbers

- Inelastic NN cross section parametrization: sigma_in(s) = [40.32 (sqrt(s)/GeV + 53.08)^0.104 - 30.15 - 8.75/(sqrt(s)/GeV)] mb, valid for sqrt(s) >= 5 GeV.
- Inter-cluster distances for light nuclei: 7Be l=3.2 fm, 9Be l=3.6 fm, 12C l=2.8 fm, 16O l=3.2 fm.
- Cluster widths: r_alpha = 1.2 fm for 7Be, 1.1 fm for 9Be, 12C, and 16O; r_3He = 1.4 fm for 7Be; r_n = 1.9 fm for 9Be.
- Parton-parton cross section parametrization coefficients: A1 = -1.66 + 8.73/p + 0.11 p, A2 = -1.58 + 9.03/p + 0.08 p, A3 = 0.22 - 0.07/p + 0.01 p, A4 = -1.02 + 23.10/p - 0.05 p (for p between 3 and 10).
- Smearing reduction factor for ellipticity (n=2): <r^2>/(<r^2> + sigma^2), where sigma is the Gaussian smearing width.

## Context

This work extends the GLISSANDO series (Broniowski 2007, Rybczynski 2013) by adding the wounded quark model, which has been explored at RHIC by the PHENIX Collaboration (Adler 2013, Adare 2015) and at SPS (Kumar Netrakanti 2004), and by incorporating alpha-clustered light nuclei as proposed by Broniowski (2013) and Bozek (2014). The code enables future studies of nucleon substructure fluctuations in p-A collisions and tests of clusterization signatures in harmonic flow patterns, using the output as initial conditions for hydrodynamic or transport models.
