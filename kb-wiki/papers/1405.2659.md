# An Advanced Computational Scheme for the Optimization of 2D Radial Reflectors in Pressurized Water Reactors

> This paper presents a computational scheme for the determination of equivalent 2D multi-group heterogeneous reflectors in a Pressurized Water Reactor (PWR). The proposed strategy is to define a full-core calculation consistent with a reference lattice code calculation such as the Method Of Characteristics (MOC) as implemented in APOLLO2 lattice code. The computational scheme presented here relies on the data assimilation module known as "Assimilation de données et Aide à l'Optimisation (ADAO)" of the SALOME platform developed at Électricité De France (EDF), coupled with the full-core code COCAGNE and with the lattice code APOLLO2. A first validation of the computational scheme is made using the OPTEX reflector model developed at École Polytechnique de Montréal (EPM). As a result, we obtain 2D multi-group, spatially heterogeneous 2D reflectors, using both diffusion or $\text{SP}_{\text{N}}$ operators. We observe important improvements of the power discrepancies distribution over the core when using reflectors computed with the proposed computational scheme, and the $\text{SP}_{\text{N}}$ operator enables additional improvements.

## Key claim

The paper presents a computational scheme that can determine equivalent 2D multi-group heterogeneous reflectors for Pressurized Water Reactors, but the validation shows that for 2-group diffusion calculations the improvement over the existing Lefebvre-Lebigot method is not significant, while for 4-group calculations the scheme produces a reflector that is less performant than the Lefebvre-Lebigot reflector despite reducing power discrepancies.

## Method

The scheme couples the data assimilation module ADAO of the SALOME platform with the full-core code COCAGNE and the lattice code APOLLO2, using a 3D-VAR approach to minimize a functional that quantifies discrepancies between computed and reference power distributions. The control variables are diffusion coefficients (for diffusion calculations) or P-1 weighted macroscopic total cross-sections (for SPN calculations) in each of six reflector regions. The minimization uses the L-BFGS method with gradients computed via mesh-corner finite differences, and the reference power distribution comes from a 281-group, 142,872-region APOLLO2 calculation using the Method Of Characteristics, collapsed to 26 groups and homogenized on each assembly.

## Key numbers

- Initial 2-group Lefebvre-Lebigot reflector: maximum power discrepancy 2.5%, average absolute discrepancy 0.7%; after optimization: average absolute discrepancy 0.54%, standard deviation 0.66%
- Initial 4-group diffusion reflector: average absolute discrepancy 10.12%, standard deviation 11.92%; after optimization: average absolute discrepancy 1.01%, standard deviation 1.42%
- Pin-by-pin power discrepancy after 4-group optimization: maximum 78.9% in the last rows of pins at the core/reflector interface
- For 2-group optimization: assembly 17 discrepancy improved from -2.5% to -0.9%; assembly 29 discrepancy improved from 1.5% to 1.0%
- Initial reflectors compared: collapsing diffusion (max discrepancy 18.7%, average 8.6%), collapsing transport (max 36.5%, average 17.9%), Lefebvre-Lebigot (max 2.5%, average 0.7%)

## Context

This work extends the Lefebvre-Lebigot method (Lefebvre and Lebigot) for reflector modeling at EDF R&D, which was limited to 2-group homogeneous reflectors with diffusion operators. The scheme is validated against the OPTEX reflector model developed at Ecole Polytechnique de Montreal (Hebert), which uses Generalized Perturbation Theory. The results enable future work on multi-group heterogeneous reflectors using SPN operators, though the pin-by-pin discrepancies at the core/reflector interface (up to 78.9% for 4-group) remain unacceptable for applications such as vessel fluence calculation and require further development.
