# Further developments of a multi-phase transport model for relativistic nuclear collisions

> A multi-phase transport (AMPT) model was constructed as a self-contained kinetic theory-based description of relativistic nuclear collisions as it contains four main components: the fluctuating initial condition, a parton cascade, hadronization, and a hadron cascade. Here, we review the main developments after the first public release of the AMPT source code in 2004 and the corresponding publication that described the physics details of the model at that time. We also discuss possible directions for future developments of the AMPT model to better study the properties of the dense matter created in relativistic collisions of small or large systems.

## Key claim

The AMPT model, particularly its string melting version, can simultaneously reproduce particle yields, transverse momentum spectra, and elliptic flows of bulk matter in heavy-ion collisions at both RHIC and LHC energies, but only after adopting a very small Lund fragmentation parameter b_L = 0.15 GeV^{-2} (much lower than previously used values) together with an upper limit of 0.40 on the relative production of strange to non-strange quarks.

## Method

The authors review the main developments of the AMPT model after its first public release in 2004, including: deuteron production via reactions BB' ↔ Md added to the hadron cascade; a string melting version where excited strings are converted into partons with spatial quark coalescence for hadronization; an improved quark coalescence that allows partons freedom to form mesons or baryons based on distance criteria; incorporation of finite nuclear thickness; modern parton distribution functions; heavy quark production improvements; local nuclear scaling of input parameters; PYTHIA8 and nucleon substructure in the initial condition; and benchmark/improvement of the parton cascade algorithm.

## Key numbers

- Lund fragmentation parameter b_L = 0.15 GeV^{-2} needed for the AMPT-SM model to reproduce bulk matter observables, much lower than the HIJING value of 0.9 GeV^{-2}
- Upper limit of 0.40 on the relative production of strange to non-strange quarks required for the AMPT-SM model
- Parton cross section of ~3 mb used in the string melting AMPT model to describe large elliptic flows, compared to ~40-50 mb needed for pure minijet gluon transport without quark coalescence
- New coalescence parameter r_BM = 0.61 found to reasonably reproduce proton and antiproton yields in AA collisions with original parton distribution functions; r_BM = 0.53 for light hadrons and 1.0 for charm hadrons with modern parton distribution functions
- For central Au+Au collisions at sqrt(s_NN) = 200 GeV, the minijet gluon system has a factor of ~2.5 lower parton multiplicity at mid-rapidity but a factor of ~6 smaller mean free path than the string melting AMPT model at impact parameter b = 8 fm

## Context

This review extends the original AMPT model description (Lin 2004) and documents developments since the first public source code release in 2004. The work builds on the HIJING two-component model (Wang 1991, Gyulassy 1994) for initial conditions and the Lund string fragmentation model (Sjostrand 1993). The string melting version addresses the failure of the default AMPT model to describe elliptic flow at RHIC, which required a parton cross section of ~40-50 mb in pure parton transport (Molnar 2001, 2019). The improved quark coalescence resolves issues with antibaryon-to-baryon ratios that were severely overestimated for Xi and Omega in the old coalescence scheme. These developments enable more reliable studies of the dense matter created in relativistic collisions of both small and large systems, including predictions for Pb+Pb collisions at 5.02 TeV and studies of vorticity and polarization in heavy-ion collisions.
