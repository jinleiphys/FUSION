# Breathing-like excited state of the Hoyle state in ${^{12}{\rm C}}$

> The existence of the $0_3^+$ and $0_4^+$ states around 10 MeV excitation energy in ${^{12}{\rm C}}$ is confirmed by a fully microscopic 3$α$ cluster model. Firstly, a GCM (generator coordinate method) calculation is performed by superposing optimized 2$α$+$α$ THSR (Tohsaki-Horiuchi-Schuck-Röpke) wave functions with the radius-constraint method. The obtained two excited $0^+$ states above the Hoyle state are consistent with the recently observed states by experiment. Secondly, a variational calculation using the single 2$α$+$α$ THSR wave function orthogonalized to the ground and Hoyle states is made and it also supports the existence of the $0_3^+$ state obtained by the GCM calculation. The analysis of the obtained $0_3^+$ state is made by studying its 2$α$-$α$ reduced width amplitude, its 2$α$ correlation function, and the large monopole matrix element between this state and the Hoyle state, which shows that this $0_3^+$ state is a breathing-like excited state of the Hoyle state. This character of the $0_3^+$ state is very different from the $0_4^+$ state which seems to have a bent-arm 3$α$ structure.

# Breathing-like excited state of the Hoyle state in 12C

## Key claim

The 0₃⁺ state in ¹²C at around 9.04 MeV excitation energy is a breathing-like excited state of the Hoyle state (0₂⁺), characterized by having one more node in both the 2α-α and α-α relative wave functions than the Hoyle state, rather than being a bent-arm 3α structure like the 0₄⁺ state. This conclusion depends on the use of a radius-constraint method with a cutoff parameter R_cut = 6 fm to remove continuum contamination in the generator coordinate method calculation.

## Method

A fully microscopic 3α cluster model using extended Tohsaki-Horiuchi-Schuck-Röpke (THSR) wave functions was employed with a generator coordinate method (GCM) combined with a radius-constraint method to remove continuum contamination. An iterative one-by-one basis selection procedure was used to pick effective wave functions from a prepared set of 2592 projected 0⁺ THSR wave functions, and a separate variational calculation using a single 0₃⁺ THSR wave function orthogonalized to the ground and Hoyle states was performed. The Volkov No.2 effective nucleon-nucleon interaction with Majorana parameter M = 0.59 and oscillator parameter b = 1.35 fm was used.

## Key numbers

- The calculated 0₃⁺ and 0₄⁺ state energies are 9.38 MeV and 11.7 MeV, respectively, compared with experimental values of 9.04 MeV and 10.56 MeV
- The calculated partial α-decay widths into ⁸Be(0⁺) for the 0₂⁺, 0₃⁺, and 0₄⁺ states are 7.39 eV, 0.92 MeV, and 0.66 MeV, respectively, compared with experimental values of 8.5 eV, 1.45 MeV, and 1.42 MeV
- The monopole matrix element between the 0₂⁺ (Hoyle) and 0₃⁺ states is about 47 e²fm⁴
- The 0₃⁺ state has a root-mean-square radius larger than 5 fm, much larger than the gas-like Hoyle state
- The squared overlap between the single orthogonalized THSR wave function at the first local minimum and the GCM 0₃⁺ wave function is 0.903, and using a deformed THSR wave function this overlap reaches 0.944
- The squared overlap between the single THSR wave function and the GCM 0₃⁺ wave function at the largest component is 0.290

## Context

This work confirms and extends the experimental findings of Itoh et al. (2011), who first reported that the broad 0₃⁺ state at 10.5 MeV is divided into two 0⁺ states at 9.04 MeV and 10.56 MeV. It resolves a discrepancy where earlier microscopic calculations using Antisymmetrized Molecular Dynamics (Kanada-Enyo, 2007) and Fermionic Molecular Dynamics (Chernykh, 2007) had reported only the 0₄⁺ state, while non-microscopic Orthogonality Condition Model calculations by Kurokawa and Kato (2005) had predicted an additional 0⁺ state. The identification of the 0₃⁺ state as a breathing-like excitation of the Hoyle state, with nodes in both the 2α-α and α-α relative wave functions, enables further study of cluster excitation modes in ¹²C and provides a framework for understanding the monopole transition strength between these states.
