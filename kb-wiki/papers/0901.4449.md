# Nuclear Force from String Theory

> We compute nuclear force in a holographic model of QCD on the basis of a D4-D8 brane configuration in type IIA string theory. Repulsive core of nucleons is quite important in nuclear physics, but its origin has not been well-understood in strongly-coupled QCD. We find that string theory via gauge/string duality deduces this repulsive core at short distance between nucleons. Since baryons in the model are realized as solitons given by Yang-Mills instanton configuration on flavor D8-branes, ADHM construction of two instantons probes well the nucleon interaction at short scale, which provides the nuclear force quantitatively. We obtain, as well as a tensor force, a central force which is strongly repulsive as suggested in experiments and lattice results. In particular, the nucleon-nucleon potential V(r) (as a function of the distance) scales as 1/r^2, which is peculiar to the holographic model. We compare our results with one-boson exchange model using the nucleon-nucleon-meson coupling obtained in our previous paper (arXiv:0806.3122).

## Key claim

String theory via gauge/string duality deduces a repulsive core at short distance between nucleons, with the nucleon-nucleon potential V(r) scaling as 1/r^2, but this result is derived only for inter-baryon distances satisfying 1/M_KK < |r| < sqrt(lambda)/M_KK in rescaled coordinates, using asymptotic expansion in r rather than for nucleons on top of each other.

## Method

The authors used the D4-D8 brane configuration in type IIA string theory (holographic QCD) where baryons are realized as Yang-Mills instantons on flavor D8-branes. They constructed a two-instanton configuration using the ADHM (Atiyah-Drinfeld-Hitchin-Manin) method, computed the quantum-mechanical Hamiltonian for the two-instanton moduli parameters including corrections from the curved background and U(1) gauge fields, and evaluated the nucleon-nucleon potential by integrating over moduli parameters with tensor-product wavefunctions of single-nucleon states obtained from previous quantization of the one-instanton system.

## Key numbers

- The instanton size is fixed to be of order 1/(sqrt(lambda)*M_KK), where M_KK is the only dimensionful parameter (set to 949 MeV to fit rho meson mass m_rho ~ 776 MeV and pion decay constant f_pi ~ 92.4 MeV)
- The classical instanton size squared is rho_cl^2 = (1/(8*pi^2*a))*sqrt(6/5), where a = lambda/(216*pi^3)
- The single-instanton moduli masses are m_X = m_Z = m_y/2 = 8*pi^2*a*N_c
- The distance scale 1/M_KK ~ 0.208 fm (using M_KK = 949 MeV)
- The nucleon-nucleon potential scales as 1/|r|^2, where |r| is the three-dimensional distance between nucleons

## Context

This work extends the holographic QCD approach of Sakai and Sugimoto (2005) for mesons and baryons, and the single-baryon quantization by Hashimoto, Sakai, and Sugimoto (2008) to the two-nucleon system. It provides an analytic derivation of the short-range repulsive core that had previously only been accessible through lattice QCD calculations by Ishii et al. (2006). The result enables comparison with one-boson-exchange models using nucleon-nucleon-meson couplings from the authors' previous paper (2008), and suggests that summing the entire tower of Kaluza-Klein mesons produces the 1/r^2 behavior characteristic of the holographic description.
