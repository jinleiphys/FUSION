# Test of final state approximations using threshold $pp \to ppπ^0$

> The Watson-Migdal approximation scheme to take into account final state interactions is shown to give the actual threshold momentum dependence of the reaction $pp \to ppπ^0$. However, by an explicit plane wave replacement of the final state wave function it is stressed that not too much physical significance should be given to the proportionality coefficient extracted using this procedure. The plane wave approximation is not physically reliable even after introducing the Watson-Migdal or a more sophisticated final state interaction factor, since direct production (impulse term) is missed. Also with short range interactions there can be discrepancies of a factor two.

# Test of final state approximations using threshold $pp \to pp\pi^0$

## Key claim

The Watson-Migdal approximation correctly reproduces the threshold momentum dependence of the $pp \to pp\pi^0$ reaction, but the proportionality coefficient extracted using this procedure should not be given physical significance because the plane wave approximation is not physically reliable even after applying the Watson-Migdal or a more sophisticated final state interaction factor, since direct production (impulse term) is missed and short range interactions can produce discrepancies of a factor two.

## Method

The author performed explicit numerical calculations of the $pp \to pp\pi^0$ transition amplitude at threshold using exact distorted wave functions for both initial ($^3P_0$) and final ($^1S_0$) states from the Reid soft core potential, including Coulomb interactions. The exact results were compared with approximations where the final state wave function was replaced by a plane wave multiplied by the Watson-Migdal final state interaction factor $F_{FSI}$, and also with a more sophisticated Jost function based enhancement factor. The comparison was carried out for different production mechanisms: direct axial current (nongalilean) term, Galilean invariance term, pion $s$-wave rescattering, and a phenomenological heavy meson exchange.

## Key numbers

- The energy discussed is 290.7 MeV, corresponding to a maximum final state relative momentum $p_f = 0.35$ fm$^{-1}$, about 5 MeV above threshold in the center-of-mass system.
- The approximate impulse amplitudes (plane wave replacement) are smaller than the exact ones by two orders of magnitude (nongalilean) and three orders of magnitude (galilean).
- With the Jost function based FSI factor using scattering length $a = 7.8243$ fm and effective range $r_0 = 2.7058$ fm (Coulomb modified), the approximation overestimates short-range contributions by about a factor of three for the simple factor.
- Without Coulomb force, using $np$ singlet parameters $a_{np} = 23.715$ fm and $r_{0np} = 2.73$ fm, both approximate methods give an overestimate by factor two for matrix elements involving potential ranges.
- The initial $^3P_0$ phase shift at 290.7 MeV is $-9.5^\circ$.

## Context

This work tests the Watson-Migdal approximation (Watson, 1952) and a Jost function based enhancement factor (Goldberger and Watson, 1964) against exact calculations for threshold pion production, extending earlier studies by Engelbrecht (1995) and Bernard et al. (1996) who used such approximations in chiral perturbation theory approaches to $\pi^0$ production, and Moalem (1995) for $\eta$ production. The results show that while the energy dependence is correctly reproduced, the multiplicative FSI procedures miss the important direct production (impulse) mechanism, and that inclusion of any FSI factor risks double counting parts of the interaction. This enables more careful treatment of final state interactions in future threshold meson production calculations.
