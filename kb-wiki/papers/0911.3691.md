# The Relation Between Equal-Time and Light-Front Wave Functions

> The relation between equal-time and light-front wave functions is studied using models for which the four-dimensional solution of the Bethe-Salpeter wave function can be obtained. The popular prescription of defining the longitudinal momentum fraction using the instant-form free kinetic energy and third component of momentum is found to be incorrect except in the non-relativistic limit. The only presently known way to obtain light-front wave functions from rest-frame, instant-form wave functions is to boost the latter wave functions to the infinite momentum frame. Despite this fact, we prove a relation between certain integrals of the equal-time and light-front wave functions.

## Key claim

The popular prescription of defining the longitudinal momentum fraction using the instant-form free kinetic energy and third component of momentum is found to be incorrect except in the non-relativistic limit; the only presently known way to obtain light-front wave functions from rest-frame, instant-form wave functions is to boost the latter wave functions to the infinite momentum frame.

## Method

The authors studied the relation between equal-time (instant-form) and light-front wave functions using two classes of models: an exactly soluble toy model with point-like coupling of a hadron to two scalar constituents, and more general models defined by the Nakanishi integral representation of the Bethe-Salpeter wave function. They computed both the instant-form wave function (via integration over \(k^0\)) and the light-front wave function (via integration over \(k^-\)) from the covariant Bethe-Salpeter amplitude, then compared the two forms directly and also examined the effect of boosting the instant-form wave function to the infinite momentum frame.

## Key numbers

* For the toy model, the instant-form wave function in the rest frame is \(\psi_{IF}(\bm{k},\bm{0}) = -\frac{2 g}{M} \frac{\sqrt{\bm{k}^2 + m^2}}{M^2 - 4 (\bm{k}^2 + m^2)}\), while the light-front wave function is \(\psi_{LF}(x, \mathbf{k}_\perp) = - g \frac{ \theta[x(1-x)] }{M^2 - \frac{\bm{k}_\perp^2+m^2}{x(1-x)}}\).
* The non-relativistic limit of the two wave functions becomes identical only when the binding energy satisfies \(B/M < 0.002\).
* For the Nakanishi-type models, the non-relativistic limit of the instant-form wave function \(\psi_{IF}^{NR}(\bfk,\mathbf{0})\) and the light-front wave function \(\psi_{LF}^{NR}(\bfk)\) become identical when the ratio of bracketed terms in the instant-form expression reduces to \(1/(2m)\).

## Context

This work extends the formal understanding of the relationship between instant-form and light-front dynamics, directly contradicting the common recipe (e.g., as used in Brodsky (1997) and Tiburzi (2000)) that defines longitudinal momentum fraction via \(x = (E_k + k^3)/(2E_k)\). The authors show that this recipe fails except in the non-relativistic limit, and that the correct procedure is to boost the equal-time wave function to the infinite momentum frame, as previously suggested by Tiburzi (2004). The results enable a rigorous foundation for extracting light-front wave functions from covariant Bethe-Salpeter solutions, which is essential for interpreting high-energy hadronic processes such as electromagnetic form factors (Drell and West, 1970) and weak decay rates (Brodsky, 1998).
