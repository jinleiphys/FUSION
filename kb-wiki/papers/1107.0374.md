# How to Improve UrQMD Model to Describe NA61/SHINE Experimental Data

> The NA61/SHINE collaboration measured inclusive cross sections of pi+ and pi- meson production in the interactions of 31 GeV/c proton with carbon nuclei at small emission angles (0 - 420 mrad). The collaboration presented also predictions of Monte Carlo models - FLUKA, VENUS and UrQMD, in a comparison with the data. The worst description of the data was observed for UrQMD model results. In the present paper it is shown that the drawback of the UrQMD model is connected with an inaccurate treatment of low mass string fragmentation. The strings appeared at a diffraction of target nucleons. A simple patch is proposed to overcome the problem.

## Key claim

The UrQMD model's poor description of NA61/SHINE data for low-momentum pion production in 31 GeV/c proton-carbon collisions is caused by a specific coding error in the low-mass string fragmentation routine, not by broader physics assumptions about cascading or diffraction. The proposed patch replaces a forward/backward two-body decay (which assigns zero transverse momentum) with an isotropic two-body decay for low-mass strings.

## Method

The author systematically tested four modifications to the UrQMD 1.3 code (changing string tension, removing the leading particle effect, adjusting average transverse momentum transfer, and modifying string mass distribution) and found none resolved the discrepancy. Analysis of single-diffraction simulations revealed that most diffractive strings have masses around 1.3–1.6 GeV, causing the code's `SUBROUTINE CLUSTR` to execute a forward/backward decay that sets the transverse momentum of produced mesons to zero. The fix simply comments out the lines enforcing this forward/backward kinematics, allowing the existing isotropic two-body decay routine to run instead.

## Key numbers

- Beam momentum: 31 GeV/c protons on carbon targets.
- Angular acceptance: 0–420 mrad (with data shown for 0–20 mrad).
- String masses in single diffraction: approximately 1.3–1.6 GeV.
- Transverse momentum transfer parameter changed from 1.6 GeV/c to 0.4 GeV/c (one of the unsuccessful modifications).
- String tension parameter `XAP` changed from 1 to 0.1 fm/GeV (one of the unsuccessful modifications).

## Context

This work addresses the specific failure of the UrQMD model (version 1.3) to reproduce NA61/SHINE inclusive cross sections for charged pions, where FLUKA and VENUS gave better agreement. The bug fix enables UrQMD to correctly describe low-momentum pion production in proton-nucleus collisions at SPS energies, which is important for interpreting neutrino oscillation experiments and cosmic-ray physics that rely on such hadron production models. The paper does not modify the underlying physics of UrQMD but corrects an implementation error in the string fragmentation routine.
