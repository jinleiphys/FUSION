# Deuteron properties from muonic atom spectroscopy

> Leading order ($α^4$) finite size corrections in muonic deuterium are evaluated within a few body formalism for the $μ^- p n$ system in muonic deuterium and found to be sensitive to the input of the deuteron wave function. We show that this sensitivity, taken along with the precise deuteron charge radius determined from muonic atom spectroscopy can be used to determine the elusive deuteron D-state probability, $P_D$, for a given model of the nucleon-nucleon (NN) potential. The radius calculated with a $P_D$ of 4.3\% in the chiral NN models and about 5.7\% in the high precision NN potentials is favoured most by the $μ^-d$ data.

## Key claim

The leading-order (α⁴) finite-size correction in muonic deuterium is sensitive to the deuteron wave function, and this sensitivity, combined with the precisely measured deuteron charge radius from muonic atom spectroscopy, can be used to determine the deuteron D-state probability, P_D, for a given nucleon-nucleon potential model. The authors find that P_D ≈ 4.3% in chiral NN models and P_D ≈ 5.7% in high-precision NN potentials are favored by the μ⁻d data, but note that the results are model-dependent.

## Method

The authors constructed the muon-deuteron electromagnetic potential using a three-body formalism for the μ⁻pn system, folding the muon-nucleon potentials (including Coulomb and Darwin terms) with the deuteron wave function. They evaluated the leading-order (α⁴) finite-size corrections to the 2S energy level in muonic deuterium using first-order perturbation theory, with the deuteron charge form factor obtained from the Fourier transform of the electric potential. The deuteron charge radius was then extracted from the finite-size correction and compared with the experimental value from Pohl et al. (2016), using deuteron wave functions from various NN potentials with different D-state probabilities.

## Key numbers

- Experimental deuteron charge radius from muonic deuterium: r_d = 2.12562(78) fm (Pohl et al., 2016)
- Leading-order α⁴ finite-size correction coefficient: 6.0731 r_d² meV/fm²
- For chiral NN models, P_D = 4.3% gives r_d = 2.1268 fm (Jülich N4LO), in closest agreement with the experimental radius
- For high-precision NN potentials, P_D = 5.7–5.8% gives r_d values from 2.1221 fm (AV18) to 2.126 fm (Paris), with P_D = 5.7% favored
- Finite-size correction ΔE_FS for 2S level: 26.72 meV (5.8% D-state, Paris), 27.03 meV (1.7% D-state, Berezhnoy), 27.57 meV (0% D-state)
- Finite-size correction ΔE_FS for 1S level: 210.42 meV (5.8% D-state, Paris), 213.17 meV (1.7% D-state), 217.19 meV (0% D-state)

## Context

This work extends the analysis of finite-size corrections in muonic deuterium beyond the standard radius extraction, using the formalism of Belyaev (1990) and the authors' prior work on muonic hydrogen (Kelkar et al., 2015). It addresses the classic problem of determining the deuteron D-state probability, which Feshbach and Schwinger (1950) showed ranges between 2% and 6%, and which is not directly measurable (Amado and Friar, 1971). The method provides a complementary tool to determine P_D from muonic atom data, and the authors note that the few-body formalism can be extended to muonic helium atoms expected to be studied in future experiments.
