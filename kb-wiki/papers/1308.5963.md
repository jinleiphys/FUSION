# Program in C for studying characteristic properties of two-body interactions in the framework of spectral distribution theory

> We present a program in C that employs spectral distribution theory for studies of characteristic properties of a many-particle quantum-mechanical system and the underlying few-body interaction. In particular, the program focuses on two-body nuclear interactions given in a JT-coupled harmonic oscillator basis and calculates correlation coefficients, a measure of similarity of any two interactions, as well as Hilbert-Schmidt norms specifying interaction strengths. An important feature of the program is its ability to identify the monopole part (centroid) of a 2-body interaction, as well as its 'density-dependent' one-body and two-body part, thereby providing key information on the evolution of shell gaps and binding energies for larger nuclear systems. As additional features, we provide statistical measures for 'density-dependent' interactions, as well as a mechanism to express an interaction in terms of two other interactions. This, in turn, allows one to identify, e.g., established features of the nuclear interaction (such as pairing correlations) within a general Hamiltonian. The program handles the radial degeneracy for 'density-dependent' one-body interactions and together with an efficient linked list data structure, facilitates studies of nuclear interactions in large model spaces that go beyond valence-shell applications.

## Key claim

The paper presents a C program that uses spectral distribution theory (SDT) to compute correlation coefficients and Hilbert-Schmidt norms for two-body nuclear interactions, enabling the identification of monopole (centroid) and density-dependent one-body and two-body parts of an interaction without performing large-scale shell-model calculations. The program handles radial degeneracy across multiple harmonic oscillator shells, going beyond valence-shell or two-shell applications of earlier Fortran programs.

## Method

The program takes two-body nuclear interaction matrix elements given in a JT-coupled harmonic oscillator basis and decomposes each Hamiltonian into irreducible tensors of particle rank 0, 1, and 2 (centroid, traceless induced single-particle energies, and traceless pure two-body interaction). It then computes the inner product between Hamiltonians using propagation formulae that carry two-nucleon information into many-nucleon systems, yielding correlation coefficients (cosine of the angle between Hamiltonian vectors) and variances (Hilbert-Schmidt norms) for both scalar (fixed particle number n) and isospin-scalar (fixed n and isospin T) distributions. The program also provides a Gram-Schmidt orthogonalization procedure to project any Hamiltonian onto a plane defined by two reference Hamiltonians.

## Key numbers

- The program can handle up to 45 single-j orbits (9 major harmonic oscillator shells, eta = 0 through 8) and up to 20 Hamiltonians simultaneously in a given calculation
- Running time is less than 20 minutes (typically several seconds) using a 1.80 GHz processor
- RAM requirement is less than 10 MB
- The distributed program contains 10,888 lines of code (88,778 bytes)
- Correlation coefficient significance scale: zeta = 0.00-0.09 (trivial), 0.10-0.29 (small), 0.30-0.49 (medium), 0.50-0.69 (large), 0.70-0.89 (very large), 0.90-0.99 (nearly perfect), 1.00 (perfect)

## Context

This work extends the spectral distribution theory framework developed by French and Ratcliff (1971), Chang, French, and Thio (1971), and Hecht and Draayer (1974) by providing a modern C implementation that handles radial degeneracy across multiple harmonic oscillator shells, unlike earlier Fortran programs by Kota (1979), Chang, Dworzecka, and Weber (1982), and Grimes, Bloom, Von Behrens, and Hausman (1983) that were limited to valence-shell or two-shell applications. The code enables studies of realistic interactions (N3LO, JISP16, CD-Bonn, AV18) and schematic interactions, allowing identification of features like pairing correlations within general Hamiltonians without performing shell-model calculations, and can be extended to three-body interactions and higher-order energy moments.
