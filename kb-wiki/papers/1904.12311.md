# Testing the tetraquark mixing framework from QCD sum rules for $a_0(980)$

> According to a recent proposal of the tetraquark mixing framework, the two light-meson nonets in the $J^{P}=0^{+}$ channel, namely the light nonet composed of $a_0 (980)$, $K_0^* (800)$, $f_0 (500)$, $f_0(980)$, and the heavy nonet of $a_0 (1450)$, $K_0^* (1430)$, $f_0 (1370)$, $f_0 (1500)$, can be expressed by linear combinations of the two tetraquark types, one type containing the spin-0 diquark and the other with the spin-1 diquark. Among various consequences of this mixing model, one surprising result is that the second tetraquark with the spin-1 diquark configuration is more important for the light nonet. In this work, we report that this result can be supported by the QCD sum rule calculation. In particular, we construct a QCD sum rule for the isovector resonance $a_0(980)$ using an interpolating field composed of both tetraquark types and then perform the operator product expansion up to dimension 10 operators. Our sum rule analysis shows that the spin-1 diquark configuration is crucial in generating the $a_0(980)$ mass. Also, the mixed correlation function constructed from the two tetraquark types is found to have large strength which seems consistent with what the tetraquark mixing framework is advocating. On the other hand, the correlation function from the interpolating field with the spin-0 diquark configuration alone fails to predict the $a_0(980)$ mass mostly by the huge negative contribution from dimension 8 operators.

## Key claim

The QCD sum rule analysis supports the tetraquark mixing framework's surprising claim that the spin-1 diquark configuration is more important than the spin-0 diquark configuration for the light nonet members, specifically for the $a_0(980)$ resonance. The interpolating field containing only the spin-0 diquark configuration fails to predict the $a_0(980)$ mass, while the mixed interpolating field incorporating both tetraquark types succeeds.

## Method

The authors constructed a QCD sum rule for the isovector resonance $a_0(980)$ using an interpolating field that is a linear combination of two tetraquark types: one containing a spin-0 diquark ($J_0$) and the other containing a spin-1 diquark ($J_1$), with mixing parameters $\beta=0.5770$ and $\alpha=0.8167$ respectively. They performed the operator product expansion (OPE) up to dimension 10 operators for the three correlation functions ($\hat{\Pi}^{\rm OPE}_{0,0}$, $\hat{\Pi}^{\rm OPE}_{0,1}$, $\hat{\Pi}^{\rm OPE}_{1,1}$) and matched the OPE side to the phenomenological side through a Borel-transformed dispersion relation with continuum threshold $s_0=(1.45~{\rm GeV})^2$.

## Key numbers

- Mixing parameters for the $I=1$ channel: $\alpha=0.8167$, $\beta=0.5770$, giving approximately 67% probability for the spin-1 diquark configuration and 33% for the spin-0 diquark configuration in the light nonet
- The OPE for the spin-0 diquark-only interpolating field ($J_0$) becomes totally negative in the Borel region $0.5~\text{GeV} \le M\le 1.0~\text{GeV}$, primarily due to a huge negative contribution from dimension 8 operators containing $\left[\langle\bar{q}q\rangle \langle\bar{s}g_c\sigma\cdot Gs\rangle +\langle\bar{s}s\rangle \langle\bar{q}g_c\sigma\cdot Gq\rangle\right]$
- Standard QCD parameters used: $\langle\bar{q}q\rangle=(-0.25)^3~{\rm GeV}^3$, $m_0^2=0.8~{\rm GeV}^2$, $\langle\bar{s}s\rangle/\langle\bar{q}q\rangle=0.8$, $m_s=0.15~{\rm GeV}$, $\langle g_c^2 G^2\rangle=0.47~{\rm GeV}^4$, $\Lambda=0.5~{\rm GeV}$

## Context

This work extends the tetraquark mixing framework proposed by Kim et al. (2016-2018), which suggests that the two $J^P=0^+$ meson nonets (light: $a_0(980)$, $K_0^*(800)$, $f_0(500)$, $f_0(980)$; heavy: $a_0(1450)$, $K_0^*(1430)$, $f_0(1370)$, $f_0(1500)$) are linear combinations of two tetraquark types. It directly addresses the failure of earlier QCD sum rule calculations by Lee (2005) and Wang (2015) that used only the spin-0 diquark configuration and could not extract the light nonet mass due to large negative dimension-8 contributions. The results enable further dynamical tests of the tetraquark mixing framework using QCD sum rules for other light nonet members and potentially for the heavy nonet.
