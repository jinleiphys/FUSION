# Monopole Shift in Odd Neutron-Rich F Isotopes: A Shell Model Description

> Monopole shift in the proton states in odd neutron rich F isotopes has been studied with neutron number changing from N = 10 to 20 for three different interactions. The variation of monopole shift point out towards appearance of shell closure at N = 14 and N = 16 and its disappearance at N = 20. Low lying states of odd F isotopes have also been calculated in the limited configuration space of {\it sd} shells for these interactions.

# Monopole Shift in Odd Neutron-Rich F Isotopes: A Shell Model Description

## Key claim
The variation of effective single-particle energies (ESPEs) for proton orbitals in odd neutron-rich F isotopes, calculated using three different interactions (MSDI, USDA, USDB), indicates the appearance of shell closures at N = 14 and N = 16 and their disappearance at N = 20, but this conclusion is based on calculations in a limited sd-shell configuration space and the authors note that large-scale shell model calculations including the fp shell are needed to definitively decide about the disappearance of the N = 20 shell closure.

## Method
The monopole shift for proton orbitals (0d5/2, 1s1/2, 0d3/2) in odd F isotopes with neutron number N = 10 to 20 was calculated using the monopole Hamiltonian, where the angular-momentum-averaged proton-neutron interaction energies were weighted by the occupation number of neutrons in each valence orbital. Three different proton-neutron interactions were used: modified surface delta interaction (MSDI) and the USDA and USDB effective interactions for the sd shell. Low-lying energy levels of 19-29F isotopes were also calculated using the shell model code Nushell within the same sd-shell model space (0d5/2, 1s1/2, 0d3/2 for both protons and neutrons).

## Key numbers
- For USDA and USDB interactions, the 1s1/2 level rises relative to 0d5/2 reaching a maximum at N = 14 with ε(1s1/2) - ε(0d5/2) = 6 MeV.
- At N = 16, the ε(0d3/2) - ε(1s1/2) gap is approximately 5 MeV for USDA and USDB interactions.
- The 1s1/2 level lies midway between 0d5/2 and 0d3/2 at N = 20 for all three interactions.
- The ground state of odd F isotopes from N = 12 to 20 is 5/2+.
- MSDI interaction predicts ground state spin 5/2+ for 19F, in contrast to the experimental value of 1/2+.
- The proton single-particle energies used for MSDI are: d5/2 = -7.560 MeV, s1/2 = -6.030 MeV, d3/2 = -3.960 MeV.
- The proton single-particle energies used for USDA are: d5/2 = -3.9436 MeV, s1/2 = -3.0612 MeV, d3/2 = 1.9798 MeV.
- The proton single-particle energies used for USDB are: d5/2 = 1.9798 MeV, s1/2 = -3.2079 MeV, d3/2 = 2.1117 MeV.

## Context
This work extends shell model studies of neutron-rich nuclei beyond the line of beta stability, following earlier demonstrations that shell structure in nuclei with large neutron excess differs from that in stable nuclei. The study uses the monopole Hamiltonian approach developed by Otsuka (2001) and others (Shalom 1963, Heyde 1994, Lawson 1980, Brussaard 1977, Duflo 1996, Smirnova 2004) to examine the evolution of the proton mean field. The results enable future large-scale shell model calculations that include the fp shell to definitively test the disappearance of the N = 20 shell closure, which the authors state are underway.
