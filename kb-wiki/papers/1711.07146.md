# New trial wave function for nuclear cluster structure of nuclei

> A new trial wave function is proposed for nuclear physics, in which an exact solution to the long-standing center-of-mass problem is given. In the new approach, the widths of the single-nucleon Gaussian wave packets and the widths of the relative Gaussian wave functions describing correlations of nucleons or clusters are treated as variables in the explicit intrinsic wave function of the nuclear system. As an example, this new wave function was applied to study the typical ${^{20}{\rm Ne}}$ ($α$+${^{16}{\rm O}}$) cluster system. By removing exactly the spurious center-of-mass effect in a very simple way, the energy curve of ${^{20}{\rm Ne}}$ was obtained by the variational calculations with the width of $α$ cluster, the width of ${^{16}{\rm O}}$ cluster, and the size parameter of the nucleus. They are considered as the three crucial variational variables in describing the ${^{20}{\rm Ne}}$ ($α$+${^{16}{\rm O}}$) cluster system. This shows that the new wave function can be a very interesting new tool for studying many-body and cluster effects in nuclear physics.

## Key claim

The paper proposes a new many-body trial wave function that exactly separates the spurious center-of-mass motion from the intrinsic wave function even when the single-nucleon Gaussian wave packets have different widths, which has been a long-standing problem in cluster physics. The key hedge is that the exact separation works for the specific operator construction presented, but the practical calculations for heavier or multi-cluster systems may require Monte Carlo techniques due to computational complexity.

## Method

The author constructs a many-body integral operator \(\bm{\widehat{G}}_n (\beta_0)\) that performs an integral transformation over generator coordinates to factor out the center-of-mass wave function exactly, even when clusters have different Gaussian width parameters \(b_i\). A second correlation operator \(\bm{\widehat{L}}_{n-1} (\bm{\beta})\) introduces the widths of relative Gaussian wave functions between clusters as variational variables. The new wave function is then applied to the \({}^{20}{\rm Ne}\) (\(\alpha\)+\({}^{16}{\rm O}\)) cluster system, performing variational calculations in the three-parameter space of the \(\alpha\) cluster width, the \({}^{16}{\rm O}\) cluster width, and the size parameter \(\beta_0\).

## Key numbers

- The minimum variational energy for the ground state of \({}^{20}{\rm Ne}\) is \(-159.74\) MeV, obtained with size parameter \(\beta_0 = 4.2\) fm, \(\alpha\) cluster width \(b_{\alpha} = 1.45\) fm, and \({}^{16}{\rm O}\) cluster width \(b_{{}^{16}{\rm O}} = 1.46\) fm.
- For comparison, the traditional angular-momentum projected Brink wave function with common width and inter-cluster distance \(R\) gives a minimum energy \(E(b_{\alpha}=b_{{}^{16}{\rm O}}=1.47\) fm, \(R=3.0\) fm) = \(-158.43\) MeV, which is more than 1 MeV higher than the new wave function result.
- In the large \(\beta_0\) limit (above 20 fm), the optimum widths approach those of free clusters: \(b_{\alpha}=1.38\) fm and \(b_{{}^{16}{\rm O}}=1.49\) fm.
- In the small \(\beta_0\) limit (below 3 fm), the widths become equal, corresponding to the SU(3) shell model limit when \(\beta_0 \to \sqrt{2}b_{\alpha}=\sqrt{2}b_{{}^{16}{\rm O}}\).

## Context

This work extends the THSR (Tohsaki-Horiuchi-Schuck-Ropke) wave function and container picture developed by Zhou (2013, 2014) and Funaki (2015), which introduced the size variable \(B_i\) as a dynamical degree of freedom for cluster correlations. It directly addresses the center-of-mass problem that has limited earlier approaches such as Fermionic Molecular Dynamics (FMD) by Feldmeier (1990) and Neff (2004), and the traditional Brink wave function (1966) where different cluster widths could not be treated without spurious center-of-mass contamination. The new wave function enables more realistic studies of cluster structure, halo nuclei, and cluster distortion effects in light nuclei, and provides a simpler alternative to the double Fourier transformation methods used in the Generator Coordinate Method (GCM) by Tohsaki (1977) and others.
