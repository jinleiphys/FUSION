# Mass and width of strange baryon resonances in QCD sum rules

> The mass spectra of strange baryons in the octet family are investigated in a finite-energy QCD sum rule approach based on the Gauss-Weierstrass transform. The phenomenological form of the spectral function is saturated by the ground state and two of the lowest excited states, considered as having opposite parities. Treating the ground state parameters as known, the masses, widths and couplings to the interpolating fields are determined and compared to experiment.

## Key claim

The masses and widths of the first two excited states of strange baryon resonances in the octet family can be extracted from QCD sum rules using a Gauss-Weierstrass transform combined with finite-energy sum rules, but only by treating the ground-state pole parameters as known input and by allowing the excited states to have finite widths, which introduces significant scatter in the extracted parameters for the second excited state.

## Method

The authors compute the time-ordered two-point correlation function for baryon interpolating fields using the operator product expansion up to dimension eight, including radiative corrections, strange quark mass corrections, flavor symmetry breaking, anomalous dimension corrections, and factorization violation of the four-quark condensate. They apply the Gauss-Weierstrass transform to obtain the first three Hermite moments of the spectral functions, which are matched to a phenomenological model containing a ground-state pole and two excited states with finite widths parameterized by masses, widths, and couplings. The resulting six simultaneous nonlinear equations are solved numerically using Broyden's method and Newton-Raphson methods, with the ground-state parameters taken from prior Borel sum rule studies.

## Key numbers

- For the nucleon: first excited state mass m1 = 1.46-1.60 GeV (experiment: 1.44 GeV) and width Gamma1 = 0.01-0.13 GeV (experiment: 0.35 GeV), with cutoff s0 varied from 2.4 to 2.6 GeV
- For Sigma: first excited state mass m1 = 1.72-1.78 GeV (experiment: 1.66 GeV) and width Gamma1 = 0.01-0.07 GeV (experiment: 0.10 GeV), with cutoff s0 varied from 3.0 to 3.2 GeV
- For Lambda_octet: first excited state mass m1 = 1.78-1.86 GeV (experiment: 1.60 GeV) and width Gamma1 = 0.01-0.06 GeV (experiment: 0.15 GeV), with cutoff s0 varied from 3.30 to 3.50 GeV
- For Xi: first excited state mass m1 = 1.97-2.06 GeV (experiment: 1.95 GeV) and width Gamma1 = 0.01-0.07 GeV (experiment: 0.06 GeV), with cutoff s0 varied from 4.00 to 4.30 GeV
- The second excited state masses and widths show larger scatter: for example, nucleon m2 = 1.32-1.85 GeV and Gamma2 = 0.01-0.35 GeV
- QCD parameters used: a = 0.52 GeV^3, b = 1.2 GeV^4, m0^2 = 0.8 GeV^2, kappa_v = 2, m_s = 0.15 GeV, f_s = 0.8

## Context

This work extends the method developed by Singh (1994) for nucleon excited states to the strange members of the baryon octet (Sigma, Lambda, Xi), using an updated version of the Borel sum rules for octet baryons from Lee (2002). The approach contrasts with conventional Borel-based QCD sum rules (Shifman, Vainshtein, Zakharov 1979) which exponentially suppress excited states, and with parity-projected Borel sum rules (Jido 1997, Lee 2006) which access negative-parity states. The results enable a complementary view of baryon spectroscopy from QCD, with the first excited state well-constrained and the second excited state better described by parity-projected methods, together giving access to the lowest three states in a given channel.
