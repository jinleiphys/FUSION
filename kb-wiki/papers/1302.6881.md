# B(E2) Evaluation for 0+ to 2+ Transitions in Even-Even Nuclei

> A collaborative study by Brookhaven-McMaster-Central Michigan is underway to evaluate B(E2)$\uparrow$ for 0$^{+}_{1}$ $\rightarrow$ 2$^{+}_{1}$ transitions. This work is a continuation of a previous USNDP evaluation and has been motivated by a large number of recent measurements and nuclear theory developments. It includes an extended compilation, data evaluation procedures and shell model calculations. The subset of B(E2)$\uparrow$ recommended values for nuclei of relevance to the double-beta decay problem is presented, and evaluation policies of experimental data and systematics are discussed. Future plans for completion of the B(E2;0$^{+}_{1}$ $\rightarrow$ 2$^{+}_{1}$) evaluation project are also described.

## Key claim

This paper presents an ongoing evaluation of B(E2)↑ values for 0⁺₁ → 2⁺₁ transitions in even-even nuclei, extending previous USNDP evaluations by Raman (2001) to include new experimental data and shell model calculations; the authors explicitly state the work is a continuation and not yet complete, with the Z>56 region still under evaluation.

## Method

The compilation was based on the previous evaluation by Raman (2001), XUNDL and NSR database content, and original research papers. Measurements were separated into three classes: model independent, low model dependent, and model dependent. Recommended B(E2)↑ values were deduced using model-independent or traditional combined datasets with the AveTools software package, and final results were analyzed using shell model predictions.

## Key numbers

- For ⁴⁶Ca: B(E2)↑ = 3.61(+32-15) W.u. with E(2⁺₁) = 1346.0(3) keV
- For ⁴⁸Ca: B(E2)↑ = 1.768(+234-96) W.u. with E(2⁺₁) = 3831.72(6) keV
- For ⁷⁶Ge: B(E2)↑ = 28.16(+79-76) W.u. with E(2⁺₁) = 562.93(3) keV
- For ¹⁰⁰Mo: B(E2)↑ = 38.4(16) W.u. with E(2⁺₁) = 535.561(22) keV
- For ¹³⁶Xe: B(E2)↑ = 13.8(39) W.u. with E(2⁺₁) = 1313.027(10) keV
- The evaluation extended the list of evaluated nuclei from 53 to 68 in the Z=2-22 region and from 20 to 38 in the Z=24-30 region

## Context

This work continues the evaluation tradition established by Stelson (1965), Grodzins (1962), and Raman (1987, 2001) for quadrupole collectivity data. The subset of recommended B(E2)↑ values for nuclei relevant to double-beta decay is intended to provide guidance for theoretical calculations of double-beta decay rates, particularly where information on Gamow-Teller transitions is lacking, and supports the SciDAC and FRIB collaborations. The evaluation for the Z=2-56 region is completed, while the Z>56 region is currently underway and will become available in the next few years.
