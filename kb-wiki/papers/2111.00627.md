# H2ZIXY: Pauli spin matrix decomposition of real symmetric matrices

> We present a code in Python3 which takes a square real symmetric matrix, of arbitrary size, and decomposes it as a tensor product of Pauli spin matrices. The application to the decomposition of a Hamiltonian of relevance to nuclear physics for implementation on quantum computer is given.

## Key claim

The paper presents a Python3 code that can decompose any real symmetric square matrix into a tensor product of Pauli spin matrices, but the decomposition is only unique for matrices whose order is a power of two; matrices of other sizes must be padded with zeros to reach the next power of two, which may affect the representation.

## Method

The algorithm takes a square real symmetric matrix, pads it with zeros to the next power-of-two dimension if necessary, then generates all Kronecker products of Pauli matrices (I, X, Y, Z) of the appropriate length, excluding those with an odd number of Y matrices which would introduce imaginary components. It constructs a system of linear equations by equating each matrix element of the input Hamiltonian to the corresponding element of the sum of Pauli Kronecker products with unknown coefficients, and solves this system using NumPy's linear algebra solver.

## Key numbers

* The deuteron Hamiltonian example uses a 3x3 matrix with diagonal elements -0.43658111, 12.15, and 19.25 (in arbitrary energy units), padded to 4x4 with a zero diagonal element
* The decomposition of the 4x4 padded Hamiltonian yields coefficients: 7.766 for II, -2.143 for IX, 1.641 for IZ, -3.913 for XX, -3.913 for YY, -1.859 for ZI, -2.143 for ZX, and -7.984 for ZZ (all to 4 significant figures)
* The code uses a threshold epsilon of 1.e-5 below which coefficients are treated as zero

## Context

This work extends the approach of representing Hamiltonians in Pauli matrix form for quantum computing applications, following the Jordan-Wigner transformation used by Dumitrescu (2018) for deuteron calculations on quantum computers. The code provides a systematic alternative to the variational quantum eigensolver method of McClean (2016) and complements other encoding approaches such as those by Siwach (2021) and Cervia (2021). The method enables direct conversion of nuclear physics Hamiltonians into a form suitable for implementation on quantum computers, and has potential applications in tensor network methods as discussed by Silvi (2019).
