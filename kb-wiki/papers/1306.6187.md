# Two-neutrino double-beta decay Fermi transition and two-nucleon interaction

> An exactly solvable model for a description of the two-neutrino double beta decay transition of the Fermi type is considered. By using perturbation theory an explicit dependence of the two-neutrino double beta decay matrix element on the like-nucleon pairing, particle-particle and particle-hole proton-neutron interactions by assuming a weak violation of isospin symmetry of Hamiltonian expressed with generators of the SO(5) group. It is found that there is a dominance of double beta decay transition through a single state of the intermediate nucleus. Then, an energy weighted sum rule connecting Delta Z=2 nuclei is presented and discussed. It is suggested that this sum rule can be exploited to study the residual interactions of the nuclear Hamiltonian.

## Key claim

The two-neutrino double-beta decay Fermi matrix element \(M^{2\nu}_F\) depends explicitly on the like-nucleon pairing, particle-particle, and particle-hole proton-neutron interactions, but only when isospin symmetry is weakly violated; near the point of isospin restoration (\(4\kappa/G = 1\)) the matrix element is dominated by a single intermediate state of the odd-odd nucleus, and the mean-field part of the Hamiltonian does not enter explicitly.

## Method

The authors constructed an exactly solvable schematic Hamiltonian within the SO(5) group for protons and neutrons occupying a single j-shell, including pairing, particle-hole, and particle-particle interactions. They applied perturbation theory up to second order in the isotensor term (proportional to \(4\kappa - G\)) to derive analytic expressions for the double Fermi matrix element and for an energy-weighted sum rule connecting \(\Delta Z = 2\) nuclei. The model was solved for a single-j shell with \(\Omega = 10\) and total particle number \(N = 20\), varying the ratio \(4\kappa/G\) from 0.7 to 1.3.

## Key numbers

- The model uses \(\Omega = 10\), \(N = 20\), \(e_p = 0.3\ \text{MeV}\), \(e_n = 0.1\ \text{MeV}\), \(G = 0.165\ \text{MeV}\), \(\chi = 0.044\ \text{MeV}\), and \(0.7 \le 4\kappa/G \le 1.3\).
- Isospin symmetry is restored at \(4\kappa/G = 1\); at this point the first-order perturbation theory is sufficient for \(M^{2\nu}_F \le 0.3\).
- The energy denominator for the dominant intermediate state \(|4'3\rangle\) in the double Fermi matrix element is \(E'_{43} - (E'_{44} + E'_{22})/2 \simeq 16\chi + \frac{7}{3}(G + 2\kappa)\).
- The energy-weighted sum rule \(S^{ew}_F(4'4,2'2)\) is proportional to \(2\Omega(G - 4\kappa)\langle 4'4|[A^\dagger\tilde{A}]_2^2|2'2\rangle + 2\chi\langle 2'2|T^-T^-|4'4\rangle\).

## Context

This work extends the SO(5) model approach previously used by Kuzmin (1992), Muto (1992), Civitarese (1994), Hirsch (1997), and Krmpotic (1998) to study beta-decay and double-beta decay symmetries. It directly addresses the suppression mechanism of the double Fermi matrix element near isospin symmetry restoration, complementing the finding by Simkovic (2013) that \(M^{2\nu}_F\) depends strongly on the isovector part of the particle-particle neutron-proton interaction. The proposed energy-weighted sum rule for \(\Delta Z = 2\) nuclei provides a new tool to adjust residual interaction strengths in nuclear models, extending the earlier QRPA-based sum rule work by Rodin (2005, 2011).
