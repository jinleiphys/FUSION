# Gamow shell model description of proton scattering on $^{18}$Ne

> We formulate the GSM in coupled-channel (GSM-CC) representation to describe low-energy elastic and inelastic scattering of protons on $^{18}$Ne. The GSM-CC formalism is applied to a translationally-invariant Hamiltonian with an effective finite-range two-body interaction. We discuss in details the GSM-CC formalism in coordinate space and give the description of the novel equivalent potential method for solving the GSM-CC system of integro-differential equations. We present the first application of the GSM-CC formalism for the calculation of excited states of $^{18}$Ne and $^{19}$Na, excitation function and the elastic/inelastic differential cross-sections in the $^{18}$Ne$(p,p')$ reaction at different energies.

## Key claim

The Gamow shell model in coupled-channel representation (GSM-CC) provides a unified description of low-energy nuclear structure and reactions using the same Hamiltonian, but the accuracy of calculated cross-sections depends on the completeness of the channel basis, which can be tested by comparing GSM and GSM-CC energies of resonant states in the compound nucleus.

## Method

The authors formulated the GSM in a coupled-channel representation to describe elastic and inelastic proton scattering on \(^{18}\)Ne. They applied a translationally-invariant Hamiltonian with an effective finite-range two-body interaction (Modified Surface Gaussian interaction) and a Woods-Saxon core potential, solving the resulting system of integro-differential equations using a novel equivalent potential method with iterative determination of local and non-local optical potentials. Reaction channels were built from GSM wave functions for the ground state \(0^+\) and first excited \(2^+\) state of \(^{18}\)Ne coupled to a proton wave function expanded in different partial waves.

## Key numbers

- The harmonic oscillator expansion used 9 shells with oscillator length \(b = 2\) fm for the short-range parts of the Hamiltonian and overlap kernels.
- The matching radius for solving coupled-channel equations was chosen as \(R \sim 2-5\) fm, close to the nuclear surface.
- The asymptotic integration starting point for backward basis functions was selected at \(R_{max} = 15\) fm, after which nuclear interaction vanishes.
- The smoothing function parameter \(\alpha\) in the equivalent potential method was typically chosen between 10 and 100.

## Context

This work extends the Gamow shell model, previously used mainly for nuclear structure problems (reviewed in Michel (2009)), to reaction calculations by adding a coupled-channel formulation. It builds on the continuum shell model approach originating from Feshbach's projection formalism and the resonating-group method combined with no-core shell model (NCSM/RGM) developed by Navratil and collaborators (2008, 2011, 2013). The GSM-CC approach enables the first unified description of spectra and reaction cross-sections within the GSM framework, and can be generalized to ab initio calculations using the No-Core Gamow Shell Model and to reactions with heavier projectiles like deuterons or alpha particles.
