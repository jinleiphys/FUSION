# Relaxation Time and Relaxation Function of Quark-Gluon Plasma with Lattice QCD

> We propose a method which enables a QCD-based calculation of a relaxation time for a dissipative current in the causal and dissipative hydrodynamic equation derived by Israel and Stewart. We point out that the Israel-Stewart equation is not unique as a causal and dissipative hydrodynamic equation, and the form of the causal and dissipative hydrodynamic equation is determined by the shape of a spectral function reflecting the properties of elementary excitations in the system we consider. Our method utilizes a relaxation function, which can be calculated from QCD using the linear response theory. We show that the relaxation function can be derived from a spectral function for a microscopic representation of the dissipative current. We also show that the Israel-Stewart equation is acceptable only as long as the calculated relaxation function is approximated well by a exponentially damping function, and the relaxation time can be obtained as its damping time constant. Taking a baryon-number dissipative current of a plasma consisting of charm quarks and gluons as a simple example, we present the first calculation of the relaxation function with use of the spectral function derived employing the quenched lattice QCD together with the maximum entropy method. The calculated relaxation function shows a strongly-oscillation damping behaviour due to the charmed vector hadron $J/Ψ$ surviving above the deconfinement phase transition temperature in QCD. This result suggests that the applicability of the Israel-Stewart equation to the baryon-number dissipative current of the charm quark-gluon plasma is quite doubtful. We present an idea for the improvement of the Israel-Stewart equation by deriving the hydrodynamic equation consistent with the strongly-oscillation damping relaxation function.

## Key claim

The Israel-Stewart causal hydrodynamic equation is acceptable only when the relaxation function calculated from QCD via linear response theory is well approximated by an exponentially damping function; for the baryon-number dissipative current of a charm quark-gluon plasma, the calculated relaxation function instead shows a strongly-oscillation damping behavior, making the applicability of the Israel-Stewart equation "quite doubtful."

## Method

The author derives a microscopic representation of the relaxation function in terms of the spectral function for a conserved current using linear response theory. For a charm quark-gluon plasma at T = 1.62 T_C, the spectral function is reconstructed from lattice QCD temporal correlation functions using the maximum entropy method (MEM) with quenched lattice QCD configurations. The relaxation function is then obtained via Fourier transformation of the inferred spectral function, and its shape is compared to the exponentially damping form required by the Israel-Stewart equation.

## Key numbers

- Temperature: T = 1.62 T_C, with T_C the deconfinement phase transition temperature
- Lattice spacing: temporal a_tau = 9.75 x 10^{-3} fm, spatial a_sigma = 4 x a_tau (anisotropic lattice with a_sigma / a_tau = 4)
- Lattice size: 20^3 x 46, spatial size L_sigma = 0.78 fm
- Gauge coupling constant: beta = 7.0, bare anisotropy zeta_0 = 3.5, fermion anisotropy gamma_F = 3.476
- Number of gauge configurations: 100
- Peak structure in the spectral function appears around omega ~ 4.5 GeV, attributed to the J/Psi meson surviving above T_C
- Cut-off frequency for Fourier transformation: omega_cut = 100 GeV
- Estimated oscillation damping time constant tau_osc ~ 0.1 fm, from the width of the low-lying peak structure

## Context

This work extends the program of calculating transport coefficients of quark-gluon plasma from lattice QCD, previously done by several groups using MEM (e.g., MEM references mem003-mem006), to the calculation of relaxation times for causal hydrodynamics. It directly challenges the universal applicability of the Israel-Stewart equation (Israel and Stewart, 1979) by showing that the spectral function for the charm quark-gluon plasma contains finite-frequency peak structures from surviving hadrons like J/Psi, rather than the Lorentzian form assumed by Israel-Stewart. The method enables future construction of improved causal hydrodynamic equations consistent with the actual spectral functions of QCD, as demonstrated by the proposed relaxation equation with oscillatory damping.
