# Recursive algorithm for constructing antisymmetric fermionic states in first quantization mapping

> We devise a deterministic quantum algorithm to produce antisymmetric states of single-particle orbitals in the first quantization mapping. Unlike sorting-based antisymmetrization algorithms, which require ordered input states and high Clifford-gate overhead, our approach initializes the state of each particle independently. For a system of $η$ particles and $N$ single-particle states, our algorithm prepares antisymmetrized states of non-trivial localized (e.g., Hartree-Fock) orbitals using $O(η^2\sqrt{N})$ $T$-gates, outperforming alternative algorithms when $η\lesssim \sqrt{N}$. To achieve such scaling, we require $O(\sqrt{N})$ dirty ancilla qubits for intermediate calculations. Knowledge of the single-particle states to be antisymmetrized can be leveraged to further improve the efficiency of the circuit, and a measurement-based variant reduces gate cost by roughly a factor of two. We show example circuits for two- and three-particle systems and discuss the generalization to an arbitrary number of particles. For a specific three-particle example, we decompose the circuit into Clifford$+T$ gates and study the impact of noise on the prepared state.

# Recursive algorithm for constructing antisymmetric fermionic states in first quantization mapping

## Key claim

The paper presents a deterministic quantum algorithm for antisymmetrizing fermionic states in first quantization that does not require ordered input states, unlike sorting-based approaches. The algorithm achieves O(η²√N) T-gate scaling for non-trivial localized orbitals when the number of particles η is less than or approximately equal to √N, but this scaling requires O(√N) dirty ancilla qubits for intermediate calculations.

## Method

The algorithm recursively constructs antisymmetric states for 2, 3, up to η particles by building on the previously computed antisymmetric state for η-1 particles. At each iterative step, controlled-swap operations are performed between the new particle and each of the previous η-1 particles, controlled on ancilla qubits prepared in a specific superposition state Y_{η-1}. The ancillae are then uncomputed by applying U_η† to the first η-1 particle registers and detecting whether each particle is in the |0⟩⊗k state, using multi-controlled X-gates.

## Key numbers

- T-gate complexity for antisymmetrizing a single ordered input product state: O(η² log N) for the present work, compared to O(η log² η log N) for Berry et al. (2018) and O(η² log² N) for Abrams and Lloyd (1997)
- Each multi-controlled C^kX gate requires 8(k-1) T-gates with k-1 additional ancilla qubits, or 4(k-1) T-gates using mid-circuit measurements and feed-forward
- Each particle swap and uncomputation of the associated ancilla qubit requires 8k-4 T-gates
- The measurement-based variant reduces the number of state-preparation unitary applications at each iterative step from η-1 to ⌊η/2⌋ in the worst case
- Average number of phase corrections ⟨N_corr⟩/(η(η-1)/2) ≈ 0.5 for large η

## Context

This work extends the line of quantum algorithms for antisymmetrizing fermionic states in first quantization, following earlier approaches by Abrams and Lloyd (1997) and Berry et al. (2018) which required ordered input states. Unlike those sorting-based methods, this algorithm can simultaneously perform antisymmetrization and state preparation for Slater determinants of complicated single-particle orbitals. The approach enables more efficient preparation of antisymmetric initial states for quantum simulations of nuclear and condensed-matter systems, particularly when the number of single-particle states N is much larger than the number of particles η.
