# Chromofields of Strings and Baryons

> We calculate color electric fields of quark/antiquark ($\bar{q}q$) and 3 quark ($qqq$) systems within the chromodielectric model (CDM). We explicitly evaluate the string tension of flux tubes in the $\bar{q}q$--system and analyze their profile. To reproduce results of lattice calculations we use a bag pressure $B = (320 MeV)^4$ from which an effective strong coupling constant $α_s \approx 0.3$ follows. With these parameters we get a $Y$ shaped configuration for large $qqq$--systems.

# Chromofields of Strings and Baryons

## Key claim
The chromodielectric model (CDM) reproduces lattice QCD results for the quark-antiquark string profile and potential, and predicts a Y-shaped flux tube configuration for three-quark systems at large separations, but only when using a bag pressure of B = (320 MeV)^4 and an effective strong coupling constant α_s ≈ 0.3; the confinement field inside the string drops only to σ ≈ 0.5 σ_vac rather than being fully expelled.

## Method
The authors solved the static equations of motion for the color electric potentials Φ^a and the confinement field σ in the chromodielectric model, where quarks are treated as classical point charges with finite extension and gluons are reduced to two abelian fields (a = 3,8). The scalar field σ is driven by a quartic potential U(σ) with a dielectric function κ(σ) = exp(-σ^3/σ_0^3) that vanishes in the non-perturbative vacuum, and the total energy of the system is computed as the sum of the σ-field energy and the color electric field energy.

## Key numbers
- Bag pressure: B^(1/4) = 320 MeV, corresponding to B = (320 MeV)^4
- Effective strong coupling constant: α_s ≈ 0.3
- String tension: τ = 988 MeV/fm for the q̄q system
- Effective Coulomb coefficient from Cornell potential fit: α_eff = 0.291, compared to lattice QCD value α_eff = 0.295 (Bali 2000)
- Constant term in Cornell potential: E_0 = 560 MeV
- Vacuum value of confinement field: σ_vac = 1.5 fm^(-1)
- Dielectric constant in vacuum: κ_vac = 0.01
- Quark charge rms radius: √⟨r²⟩ ≈ 0.1 fm
- Quark-antiquark distances studied: up to 1.2 fm for string profile; linear regime for energy starts above r > 0.5 fm
- Three-quark separations: quark distance from Steiner point ℓ = 0.7 fm and 1.3 fm, corresponding to qq distances L = √3 ℓ ≈ 1.2 fm and 1.7 fm

## Context
This work extends the chromodielectric model (Friedberg 1977, Friedberg 1977, Traxler 1998) to static three-quark configurations, reproducing lattice QCD results for the q̄q string profile (Bali 2000, Bali 1995) and the Cornell potential parameters. The Y-shaped baryon configuration found for large qqq systems supports results from the Gaussian Stochastic Vacuum model (Kuzmenko 2000) and some lattice studies (Takahashi 2002), while the Δ configuration seen in other lattice work (Bali 2000, Alexandrou 2001) is not observed at large separations. The results enable further dynamical studies of confinement in baryonic systems within the CDM framework.
