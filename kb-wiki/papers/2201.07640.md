# Planck Formula for the Gluon Parton Distribution in the Proton

> We describe the gluon parton distribution function (PDF) in the proton, deduced by data from the ATLAS and HERA experiments, in the framework of the parton statistical model. The best fit parameters involved in the Planck formula that describes the gluon distribution are consistent with the results obtained from analysis of deep inelastic scattering processes. Remarkably, the agreement between the statistical model and the experimental gluon distributions is found with the same value of the "temperature" parameter $\bar{x}$ found by fitting the valence parton distributions from deep inelastic scattering. This result corroborates the validity of the statistical approach in the gluon sector.

## Key claim

The gluon parton distribution function (PDF) in the proton can be described by a Planck formula (Bose-Einstein distribution with zero chemical potential) using the same temperature parameter \(\bar{x}=0.099\) previously determined from valence quark distributions, with the parameters \(A_G\) and \(b_G\) obtained from fitting ATLAS data agreeing with those found twenty years earlier from deep inelastic scattering constraints.

## Method

The authors fit the central values of the gluon momentum distribution \(xg(x)\) from ATLAS experiment data at \(Q^2_0 = 4\,\mathrm{GeV}^2/c^4\) over the range \(x \in [0.05, 0.66]\) using the Planck distribution form \(x g(x) = A_G x^{b_G} / (\exp(x/\bar{x}) - 1)\). The temperature parameter \(\bar{x}\) was fixed to 0.099 from prior work (BBS), while \(A_G\) and \(b_G\) were treated as free parameters in a \(\chi^2\) minimization against 74 experimental points.

## Key numbers

- Best fit parameters: \(A_G = 15.853 \pm 0.225\), \(b_G = 0.792 \pm 0.006\) from fitting 74 ATLAS data points in \(x \in [0.05, 0.66]\)
- \(\chi^2 = 49.03\) with \(\nu = 72\) degrees of freedom, corresponding to a significance level near 0.975
- Integrated gluon momentum for \(x<0.2\): \(M_{g,\mathrm{low}} = 0.358 \pm 0.012\); for \(x>0.2\): \(M_{g,\mathrm{high}} = 0.084 \pm 0.002\); total: \(M_g = 0.442 \pm 0.014\)
- Temperature parameter fixed at \(\bar{x} = 0.099\) from prior work (BBS)
- Prior parameters from BBS: \(A_G = 14.28\), \(b_G = 0.75\)

## Context

This work extends the parton statistical model of BBS (2000) by directly fitting the gluon distribution to ATLAS proton-proton collision data, rather than relying on theoretical constraints from QCD sum rules or comparisons with HERA parametrizations. The Planck formula result differs significantly from the standard \((1-x)^C\) parametrization used by HERA (2007), particularly at large \(x\), and shows better agreement with NNPDF (2015) as noted in prior work by BST (2016). The confirmation that the same temperature parameter describes both valence quarks and gluons supports the statistical approach as a basis for boundary conditions to DGLAP evolution equations.
