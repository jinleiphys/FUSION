# Comparative tests of isospin-symmetry-breaking corrections to superallowed 0+-to-0+ nuclear beta decay

> We present a test with which to evaluate the calculated isospin-symmetry-breaking corrections to superallowed 0+-to-0+ nuclear beta decay. The test is based on the corrected experimental Ft values being required to satisfy conservation of the vector current (CVC). When applied to six sets of published calculations, the test demonstrates quantitatively that only one set -- the one based on the shell model with Saxon-Woods radial wave functions -- provides satisfactory agreement with CVC. This test can easily be applied to any sets of calculated correction terms that are produced in future.

## Key claim

The paper presents a quantitative test to evaluate different sets of calculated isospin-symmetry-breaking corrections (δ_C) for superallowed 0+→0+ nuclear beta decay, based on whether the corrected Ft values satisfy the Conserved Vector Current (CVC) hypothesis. The test demonstrates that only one set of calculations, the shell model with Saxon-Woods radial wave functions (SM-SW), provides satisfactory agreement with CVC, while the other five sets fail to produce statistically consistent Ft values.

## Method

The authors apply a least-squares minimization procedure where the average corrected Ft value (overline{Ft}) is treated as a single adjustable parameter, and the normalized chi-squared (χ²/n_d) quantifies the agreement between each model's calculated δ_C values and the "experimental" δ_C values derived from Eq. (finaltest). The test uses experimental ft values from the most recent survey of world data (Hardy and Towner, 2009) with re-determined normally distributed uncertainties, along with calculated radiative corrections δ'_R and δ_NS from Towner and Hardy (2008). Six sets of published δ_C calculations are compared: the Damgaard model, shell model with Saxon-Woods radial wave functions (SM-SW), shell model with Hartree-Fock radial wave functions (SM-HF), relativistic Hartree-Fock with random phase approximation (RHF-RPA), relativistic Hartree with random phase approximation (RH-RPA), and the isovector monopole resonance (IVMR) model.

## Key numbers

* For the SM-SW model, χ²/n_d = 1.2 with statistical experimental uncertainties only, corresponding to a confidence level of 26%
* For the Damgaard model, χ²/n_d = 8.3 with statistical experimental uncertainties only, corresponding to a confidence level of 0%
* For the SM-HF model, χ²/n_d = 8.3 with statistical experimental uncertainties only, corresponding to a confidence level of 0%
* For the RHF-RPA model, χ²/n_d = 7.2 with statistical experimental uncertainties only, corresponding to a confidence level of 0%
* For the RH-RPA model, χ²/n_d = 6.0 with statistical experimental uncertainties only, corresponding to a confidence level of 0%
* For the IVMR model, χ²/n_d = 48 with statistical experimental uncertainties only, corresponding to a confidence level of 0%
* The SM-SW model yields δ_C values ranging from 0.175% for 10C to 1.63% for 74Rb
* The Damgaard model yields δ_C values ranging from 0.046% for 10C to 1.422% for 74Rb
* The IVMR model yields δ_C values ranging from 0.008% for 10C to 0.235% for 74Rb

## Context

This work extends the long-standing program of superallowed beta-decay studies by Towner, Hardy, and collaborators, who have provided the most widely used isospin-symmetry-breaking corrections (Towner and Hardy, 2008). The test directly addresses recent alternative calculations from Ormand and Brown (1985, 1995), Sagawa, Van Giai and Suzuki (1996), Liang, van Giai and Meng (2009), and Auerbach (2009), as well as a claim by Miller and Schwenk (2009) that the SM-SW calculations neglect an important radial excitation term. The results enable future researchers to quantitatively evaluate any new sets of δ_C calculations against the CVC hypothesis, and the authors note that two recent papers, one by Grinyer (2010) using semi-empirical analysis and one by Satula (2010) using Skyrme-density-functional-theory, confirm the SM-SW results.
