# Energy expressions for n=3 and 4 systems in a single-j shell

> For systems with three and four fermions within a single-j shell, analytical expressions for the state energies are presented from a decomposition of the angular momentum. In some important cases the expressions acquire a very simple form. The expression may help us in understanding the structure of isomeric states. The decomposition also makes it possible to construct the algebraic condition for conservation of seniority.

# Energy expressions for n=3 and 4 systems in a single-j shell

## Key claim
Analytical expressions for state energies of three- and four-fermion systems in a single-j shell can be derived from angular momentum decomposition of the pair number coefficients, and in certain cases (such as unique states with maximum spin) these expressions acquire simple, interaction-independent forms. The decomposition also enables construction of algebraic conditions for seniority conservation, but the simplified expressions are only valid when the system is restricted to a single-j shell and the two-body interaction is treated in the multistep shell model framework.

## Method
The author decomposes the wave functions of n=3 and n=4 systems into correlated two-particle bases (proton and neutron blocks for n=4; a nucleon pair plus an odd nucleon for n=3) within the multistep shell model. The Hamiltonian matrix elements are expressed as linear combinations of two-body interaction matrix elements V_J, and the expansion coefficients C^I_J (giving the number of pairs with angular momentum J) are derived from angular momentum coupling coefficients and 6j symbols. For unique states with dimension D=1, the coefficients become interaction-independent and yield simple energy expressions.

## Key numbers
- For the unique 12+ state in ﻗ·٥ﺁ٢Fe (four-hole system in 0f_{7/2} shell): E_{12} = (6/13)ﺑ٢_5 + 3ﺑ٢_6 + (33/13)ﺑ٢_7
- For the first 10+ state in ﻗ·٥ﺁ٢Fe: E_{10^+_1} = 0.310ﺑ٢_3 + 1.429ﺑ٢_4 + 0.497ﺑ٢_5 + 1.571ﺑ٢_6 + 2.193ﺑ٢_7
- For the first 10+ state in ﻗ·٤ﻗ·٤Ti: E_{10^+_1} = 0.245V_3 + 1.473V_4 + 0.570V_5 + 1.526V_6 + 2.186V_7
- For the unique 16+ state in ﻗ·٩ﻗ·٦Cd (four-hole system in 0g_{9/2} shell): E_{16} = (8/17)ﺑ٢_7 + 3ﺑ٢_8 + (43/17)ﺑ٢_9
- For the first 14+ state in ﻗ·٩ﻗ·٦Cd: E_{14^+_1} = 0.307ﺑ٢_5 + 1.428ﺑ٢_6 + 0.493ﺑ٢_7 + 1.572ﺑ٢_8 + 2.200ﺑ٢_9
- For the unique I=17/2 state in j=7/2 shell (n=3): E_{17/2} = (19/26)V_5 + (3/2)V_6 + (10/13)V_7
- For the I=1/2 state in j shell (n=3): E_{1/2} = (3/2)V_{j-1/2} + (3/2)V_{j+1/2}
- For the unique state with maximum spin I=4j-2 (n=4): C^I_J = 0 for J < 2j-2, C^I_J = 3 for J = 2j-1
- For the unique state with maximum spin I=3j-1 (n=3): C^I_J = 0 for J < 2j-1, C^I_J = 3/2 for J = 2j-1 and J = 2j
- Dimension of I=0 states for n=4: D_0(j) = (2j+1)/2; for T=2: D_0^2(j) = [(2j+3)/6] = [(j+2)/3]

## Context
This work extends earlier studies of pair number coefficients in single-j shells by Zamick (2005), Moya de Guerra (2003), and Zamick (2007), who used coefficients of fractional parentage calculations. It provides analytic formulas that complement the numerical diagonalization approach of the multistep shell model developed by Liotta (1981, 1982). The results enable systematic analysis of spin-trap isomers such as the 12+ state in ﻗ·٥ﺁ٢Fe observed by Ur (1998) and the predicted 16+ state in ﻗ·٩ﻗ·٦Cd from Ogawa, and can be extended to many-particle systems to study Nﻗ┬┤Z nuclei and isomeric states in single-j shells.
