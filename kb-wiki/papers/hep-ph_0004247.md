# New QCD estimate of the kaon penguin matrix elements and epsilon'/epsilon

> Firstly, we use the recent ALEPH/OPAL data on the V-A spectral functions for fixing the continuum threshold with which the first and second Weinberg sum rules should be satisfied in the chiral limit. Then, we predict the values of the low-energy constants m_{π^+}-m_{π^0}, {L}_{10}, and test the values of the electroweak kaon penguin matrix elements <Q^{3/2}_{8,7}>_{2π} obtained from DMO--like sum rules. Secondly, we use the data on the τ-total hadronic width R_{τ,V/A} for extracting <Q_8^{3/2}_{2π}>, in the {MS}\bar scheme, and propose some new sum rules for < Q_7^{3/2}>_{2π} in the chiral limit, where the latter require more accurate data for the spectral functions near the $τ$-mass. Thirdly, we analyze the effects to the matrix element <Q_6^{1/2}>_{2π}, of the S_2\equiv(\bar uu+\bar dd) component of the I=0 scalar meson, with its parameters fixed from QCD spectral sum rules. Our results should stimulate a further attention on the rôle of the (expected large) gluonium component of the I=0 scalar meson and of the associated operator in the K--->ππamplitude. Finally, using our previous determinations, we deduce, in the Standard Model (SM), the conservative upper bound for the CP-violating ratio: ε'/ε< (22\pm 9) 10^{-4}, which is in agreement with the present measurements.

# New QCD estimate of the kaon penguin matrix elements and epsilon'/epsilon

## Key claim

The paper derives a conservative upper bound for the direct CP-violating ratio epsilon'/epsilon in the Standard Model, finding it less than (22 +/- 9) x 10^{-4}, which agrees with experimental measurements; however, this bound depends on the specific values of the strange quark mass and the penguin operator bag parameters, and the analysis shows that the leading 1/N_c vacuum saturation prediction (B_6^{1/2} approx B_8^{3/2} approx 1) may be violated by more than 2 sigma.

## Method

The author uses ALEPH/OPAL data on V-A spectral functions from tau decays to fix the continuum threshold t_c by requiring the first and second Weinberg sum rules to be satisfied in the chiral limit, then predicts low-energy constants and tests DMO-like sum rules for the electroweak penguin matrix elements <Q^{3/2}_{8,7}>_{2pi}. The tau total hadronic width R_{tau,V/A} is used to extract <Q_8^{3/2}>_{2pi} in the MS-bar scheme, while the effect of the S_2 = (bar u u + bar d d) component of the I=0 scalar meson on <Q_6^{1/2}>_{2pi} is analyzed using QCD spectral sum rules.

## Key numbers

- Continuum threshold t_c = (1.475 +/- 0.015) GeV^2, fixed from the first and second Weinberg sum rules
- Low-energy constant L_10 = -(6.42 +/- 0.04) x 10^{-3} (best prediction)
- Pion mass difference Delta m_pi = (4.96 +/- 0.22) MeV (best prediction)
- B^{3/2}_7 = (0.7 +/- 0.2) x (m_s(2)/119 MeV)^2 x (92.4/f_pi)^4 at mu = 2 GeV
- B^{3/2}_8 = (2.5 +/- 1.3) x (m_s(2)/119 MeV)^2 x (92.4/f_pi)^4 at mu = 2 GeV
- B^{1/2}_6 = 1.0 +/- 0.4 (from S_2 component using QCD spectral sum rules), with upper bound <= 1.5 +/- 0.4 for m_s(2) >= 90 MeV
- Conservative upper bound: epsilon'/epsilon < (22 +/- 9) x 10^{-4}
- Experimental value used for comparison: Re(epsilon'/epsilon)_exp = (21.4 +/- 4.0) x 10^{-4}
- Strange quark mass: m_s(2) = (119 +/- 12) MeV
- Im lambda_t = (1.33 +/- 0.14) x 10^{-4}

## Context

This work extends the DMO-like sum rule approach of Donoghue and collaborators (1990s) for estimating electroweak penguin matrix elements, while also addressing the QCD penguin operator <Q_6^{1/2}> via the scalar meson component. The author compares results with lattice QCD predictions (B_6^{1/2} = 0.6-0.8, B_8^{3/2} = 0.7-1.1), large N_c approaches by Hambye (1990s), and the ENJL model by Prades (1990s), noting that first-principles QCD methods fail to explain the experimental epsilon'/epsilon data while various QCD-like models can accommodate it. The analysis enables improved constraints on the bag parameters B_6^{1/2} and B_8^{3/2} and suggests that the gluonium component of the I=0 scalar meson deserves further attention for understanding the K -> pi pi amplitude.
