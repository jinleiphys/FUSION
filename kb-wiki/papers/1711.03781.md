# The rank-one separable interaction kernel for nucleon with scalar propagators

> In the paper the covariant kernel of the nucleon-nucleon interaction of particles with scalar propagators is analyzed. The Bethe-Salpeter equation for the T matrix is considered in the rank- one separable kernel. The parameters of the kernel for the specific partial-wave channels explicitly connect with the observables { low energy scattering parameters and phase shifts, deuteron binding energy. Covariant separable kernels forthe partial-wave channels with total angular momentum J = 0 (1S0, 3P0) and J = 1 (3S1 -3D1, 1P1, 3P1) are constructed.

## Key claim

The paper constructs rank-one separable covariant kernels for nucleon-nucleon interaction using scalar propagators (rather than spinor propagators) for partial-wave channels with total angular momentum J=0 and J=1, with the explicit condition that the kinetic energy is restricted to T_lab < 4 beta due to the integration method used.

## Method

The Bethe-Salpeter equation for the T matrix is solved in the rest frame of the two-nucleon system using a rank-one separable kernel ansatz, where the form factors are covariant generalizations of Yamaguchi functions for S, P, and D waves. The parameters (lambda_L, beta_L, C_L) for each partial-wave channel are obtained by fitting to experimental phase shifts from the SAID database and to the deuteron binding energy, using a chi-squared minimization procedure with the kinetic energy maximum set to 100 MeV for the 3P0 state and 200 MeV for the 1P1 and 3P1 states. For the coupled 3S1-3D1 channel, the C2 parameter is refitted to achieve specific D-state pseudoprobabilities (p_D = 4%, 5%, 6%) while the lambda parameter is refitted to reproduce the deuteron binding energy.

## Context

This work extends the authors' previous rank-one separable NN interaction kernel for nucleons with spinor propagators (Bondarenko et al., 2017) to the case of scalar propagators, and takes the 1S0 parameters from Tjon (1988) without changes. The constructed covariant separable kernels are intended for use in future calculations of three-nucleon systems via Bethe-Salpeter-Faddeev equations, where the scalar propagator approximation simplifies the 6-fold integral equations to 2-fold integral equations with generalized potentials.
