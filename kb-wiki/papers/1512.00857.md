# Calculating Soft Radiation at One Loop

> We present an efficient way to calculate the effect of soft QCD radiation at one loop, which is needed for predictions at next-to-next-to-leading logarithmic accuracy. We use rapidity coordinates and isolate the divergences in the integrand. By performing manipulations with cumulative variables, we avoid complications from plus distributions. We address rapidity divergences, divergences with an azimuthal dependence, complicated jet boundaries and multi-differential measurements. The process and measurements can be easily adjusted, as we demonstrate by reproducing many existing soft functions. The results for a general LHC process with multiple Wilson lines are obtained by treating Wilson lines that are not back-to-back using a boost. We also obtain, for the first time, the N-jettiness soft function for generic jet angularities, and the collinear-soft function for the measurement of two angularities.

## Key claim

The paper presents an efficient, general method for calculating one-loop soft QCD radiation needed for next-to-next-to-leading logarithmic (NNLL) resummation, claiming that by using rapidity coordinates, isolating divergences at the integrand level, and working with cumulative variables, the calculation can be simplified and applied to arbitrary processes and measurements, including, for the first time, the N-jettiness soft function for generic jet angularities and the collinear-soft function for the measurement of two angularities.

## Method

The authors use rapidity coordinates (transverse momentum \(k_T\), rapidity \(y\), azimuthal angle \(\phi\)) to make the symmetries of the soft matrix element manifest, and isolate divergences directly in the integrand. They work with the cumulative soft function (using step functions instead of plus distributions) to simplify manipulations, then differentiate to obtain the distribution. For \(N\)-jet processes with multiple Wilson lines, they boost to frames where pairs of Wilson lines are back-to-back, recycling dijet results. For multi-differential measurements, they extend the measurement to a vector and identify the dominant measurement at each phase-space point.

## Key numbers

* Thrust soft function (e+e-): \(S^\one(m=Q\tau,\mu) = -\frac{\alpha_s C_F}{\pi} \frac{1}{\epsilon} \left[\frac{1}{\epsilon} + 2\ln\frac{\mu}{m} + 2\epsilon\left(\ln^2\frac{\mu}{m} - \frac{\pi^2}{24}\right)\right]\), where \(m = Q\tau\) and \(f(y,\phi) = e^{-|y|}\).
* Angularities soft function (e+e-): \(S^\one(m=Q\tau_a,\mu) = \frac{\alpha_s C_F}{\pi} \frac{1}{a-1} \left[\frac{1}{\epsilon^2} + \frac{1}{\epsilon}\left(\ln\frac{\mu^2}{Q^2} - 2\ln\tau_a\right) + \frac{1}{2}\ln^2\frac{\mu^2}{Q^2} - 2\ln\frac{\mu^2}{Q^2}\ln\tau_a + 2\ln^2\tau_a - \frac{\pi^2}{12}\right]\), for \(a<2\), \(a\neq1\).
* Transverse momentum soft function (pp): \(S^\one(m=p_T,\mu) = -\frac{\alpha_s C_F}{\pi} \left[\frac{1}{\epsilon} + 2\ln\frac{\mu}{m} + 2\epsilon\left(\ln^2\frac{\mu}{m} - \frac{\pi^2}{24}\right)\right] \left(\frac{2}{\eta} - \frac{1}{\epsilon} + 2\ln\frac{\nu}{m}\right)\), with \(f(y,\phi)=1\).
* Transverse thrust soft function (e+e-): \(S^\one(m=\tau_\perp,\mu) = -\frac{\alpha_s C_F}{\pi} \left[ \frac{1}{\epsilon^2} + \frac{2}{\epsilon} \ln\frac{\mu}{4mQ\sin^2\theta} + 2\ln^2\frac{\mu}{4mQ\sin^2\theta} + \frac{7\pi^2}{12} + A(\theta) \right]\), where \(A(\theta) = \frac{1}{\pi} \int_{-\infty}^\infty dy \int_0^{2\pi} d\phi \ln\frac{f(y,\phi)}{f_\infty(y,\phi)}\).

## Context

This work extends the SCET (Soft-Collinear Effective Theory) framework for soft function calculations, reproducing existing results for thrust (Schwartz 2007, Fleming 2007), angularities (Hornig 2009, Larkoski 2014), transverse momentum (Chiu 2012), and transverse thrust (Becher 2015), as well as double-differential measurements (Larkoski 2014, Procura 2014). The method enables the first calculation of the N-jettiness soft function for generic jet angularities and the collinear-soft function for double angularity measurements, which are needed for NNLL resummation in LHC processes with complicated jet boundaries and multi-differential measurements.
