# The Drell-Yan process with pions and polarized nucleons

> The Drell-Yan process provides important information on the internal structure of hadrons including transverse momentum dependent parton distribution functions (TMDs). In this work we present calculations for all leading twist structure functions describing the pion induced Drell-Yan process. The non-perturbative input for the TMDs is taken from the light-front constituent quark model, the spectator model, and available parametrizations of TMDs extracted from the experimental data. TMD evolution is implemented at Next-to-Leading Logarithmic precision for the first time for all asymmetries. Our results are compatible with the first experimental information, help to interpret the data from ongoing experiments, and will allow one to quantitatively assess the models in future when more precise data will become available.

## Key claim

The paper presents calculations for all leading-twist structure functions in pion-induced Drell-Yan with polarized nucleons, implementing TMD evolution at Next-to-Leading Logarithmic (NLL) precision for the first time for all asymmetries, but the results are compatible with existing COMPASS data only within model uncertainties and will require more precise data for quantitative assessment of the models.

## Method

The authors compute the six leading-twist Drell-Yan structure functions using TMD factorization in impact-parameter space, evolving TMDs from an initial scale Q₀² = 2.4 GeV² to the COMPASS scale ⟨Q²⟩ = 28 GeV² using Collins-Soper-Sterman (CSS) evolution at NLL precision. Non-perturbative input for TMDs is taken from three sources: the light-front constituent quark model (LFCQM), the spectator model (SPM), and phenomenological parametrizations extracted from experimental data, with the pion Boer-Mulders function h₁,π^⟂a taken from models due to the absence of phenomenological extractions. The authors also perform "hybrid calculations" where all TMDs except the pion Boer-Mulders function are taken from phenomenological analyses.

## Key numbers

- Initial scale for TMD evolution: Q₀² = 2.4 GeV², corresponding to the average Q² in HERMES experiments
- COMPASS Drell-Yan measurement scale: ⟨Q²⟩ = 28 GeV²
- Low hadronic model scale: μ₀ ∼ 0.5 GeV, determined by requiring valence quarks carry the entire nucleon momentum (M₂^val(μ₀) = 1)
- Non-perturbative parameter for Collins-Soper kernel: g₀(b_max) = 0.84 with b_max = 1 GeV⁻¹
- Strong coupling constant at Z boson mass: α_s(M_Z) = 0.118 (world average, used as in CTEQ analysis)
- Hard factor includes terms up to O(α_s) with C_F = 4/3

## Context

This work extends previous studies of TMDs in semi-inclusive deep-inelastic scattering (SIDIS) and Drell-Yan processes by providing the first comprehensive NLL-evolved calculations for all leading-twist asymmetries in pion-induced polarized Drell-Yan. It builds on the TMD factorization theorems of Collins (2011) and the CSS evolution framework, and directly addresses the COMPASS Drell-Yan data from Aghasyan et al. (2017). The results enable quantitative tests of constituent quark models (LFCQM and spectator model) against experimental data and will allow future assessment of these models when more precise data become available.
