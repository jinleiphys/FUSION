# Collectivity without plasma in hadronic collisions

> We present a microscopic model for collective effects in high multiplicity proton--proton collisions, where multiple partonic subcollisions give rise to a dense system of strings. From lattice calculations we know that QCD strings are transversely extended, and we argue that this should result in a transverse pressure and expansion, similar to the flow in a deconfined plasma. The model is implemented in the PYTHIA8 Monte Carlo event generator, and we find that it can qualitatively reproduce the long range azimuthal correlations forming a near-side ridge in high multiplicity proton--proton events at LHC energies.

## Key claim

The authors present a microscopic model where collective effects in high-multiplicity proton-proton collisions arise from transverse repulsion between overlapping QCD flux tubes (strings), rather than from formation of a deconfined quark-gluon plasma. The model qualitatively reproduces the long-range azimuthal correlations (the "ridge") using a single tunable parameter \(g\) for the interaction strength, with the caveat that the comparison to CMS data was performed by the authors themselves and may not include all experimental corrections.

## Method

The model is implemented in the PYTHIA8 Monte Carlo event generator. Multiple partonic subcollisions produce a dense system of strings whose transverse positions are sampled from a 2D Gaussian mass distribution of the protons at the given impact parameter. The repulsive force between overlapping flux tubes is modeled using a Gaussian form \(f(d_\perp) = (g\kappa d_\perp / R^2) \exp(-d_\perp^2 / 4R^2)\), where \(d_\perp\) is the transverse separation, \(R\) the time-dependent flux tube width, and \(\kappa\) the string tension. The shoving is applied in rapidity segments of width 0.1 units over several time steps until hadronisation time (\(\tau \sim 2\) fm), with partons moving freely for the first \(\sim 1\) fm before interactions begin.

## Key numbers

- String breaking proper time: \(\langle \tau^2 \rangle = (1+a)/(b\kappa^2)\), with default PYTHIA parameters \(a=0.68\), \(b=0.98\) GeV\(^{-2}\), and \(\kappa = 0.9-1\) GeV/fm, giving a typical breaking time around 2 fm.
- The shoving parameter \(g\) is tested at values \(g=4\), \(g=10\), and \(g=40\) (the latter considered unreasonably large), with \(g=4\) producing a ridge qualitatively similar to CMS data at 7 TeV.
- Extra transverse momentum per unit rapidity \(dp_\perp/dy\) is shown for four final-state particle multiplicity bins in the region \(|y| < 2.5\), reaching large values even for \(g=4\).
- A soft gluon invariant mass cut of \(\sim 0.2\) GeV is applied; string segments with \(p_\perp > 2\) GeV before shoving are excluded from additional excitations.

## Context

This work extends the authors' earlier proof-of-principle study (Bierlich, 2016) on flow-like transverse expansion from overlapping strings in pp collisions, and builds on the rope hadronisation model (Bierlich, 2014). It contrasts with the standard heavy-ion interpretation where collective flow arises from a deconfined, thermalised quark-gluon plasma (Ollitrault, 1992; Ackermann, 2000), and with the EPOS model (Werner, 2007; Pierog, 2013) which assumes a dense thermalised core with an outer corona. The model enables future combination with rope hadronisation and extension to nucleus collisions (Bierlich, 2016), potentially challenging the current paradigm if it can describe collective effects in larger systems without invoking a thermalised plasma.
