# Spin-Dependent Neutralino-Nucleus Scattering for $A \sim 127$ Nuclei

> We perform nuclear shell model calculations of the neutralino-nucleus cross section for several nuclei in the A = 127 region. Each of the four nuclei considered is a primary target in a direct dark matter detection experiment. The calculations are valid for all relevant values of the momentum transfer. Our calculations are performed in the $3s 2d 1g_{7/2} 1h_{11/2}$ model space using extremely large bases, allowing us to include all relevant correlations. We also study the dependence of the nuclear response upon the assumed nuclear Hamiltonian and find it to be small. We find good agreement with the observed magnetic moment as well as other obervables for the four nuclei considered: ^{127}I, ^{129,131}Xe, and ^{125}Te.

# Spin-Dependent Neutralino-Nucleus Scattering for A ~ 127 Nuclei

## Key claim
Nuclear shell model calculations in a large model space (3s2d1g7/21h11/2) show that the spin-dependent neutralino-nucleus cross section for 127I, 129Xe, 131Xe, and 125Te is significantly larger than previously assumed from simpler nuclear models, with the iodine spin expectation value ⟨Sp⟩ being a factor of ~20 larger than the Odd Group Model value of 0.07 used in prior experimental analyses. The authors find that the dependence of the nuclear response on the assumed nuclear Hamiltonian (Bonn A vs. Nijmegen II) is small, but caution that uncertainties from meson exchange currents and Gamow-Teller quenching effects remain.

## Method
The authors performed large-basis nuclear shell model calculations in the 3s2d1g7/21h11/2 model space using the ANTOINE code, with model spaces of approximately 3 million Slater determinants for 127I. They employed two realistic nucleon-nucleon potentials (Bonn A and Nijmegen II) converted to shell model interactions via the G-matrix/Folded Diagram technique, with single-particle energies adjusted to reproduce experimental magnetic moments, quadrupole moments, and low-lying excited state spectra. The spin structure functions S(q) for finite momentum transfer were evaluated using Wood-Saxon wave functions and fitted as 8th-order polynomials in y = (qb/2)² times exp(-2y) out to y = 10.

## Key numbers
- For 127I using the Bonn A Hamiltonian: ⟨Sp⟩ = 0.441 (compared to OGM value of 0.07, a factor of ~20 increase)
- For 127I using the Bonn A Hamiltonian: ⟨Sn⟩ = 0.045, ⟨Lp⟩ = 0.000, ⟨Ln⟩ = 0.779, with total J = 5/2
- For 127I using the Nijmegen II Hamiltonian: ⟨Sp⟩ = 0.525, ⟨Sn⟩ = 0.029, ⟨Lp⟩ = 0.000, ⟨Ln⟩ = 0.746
- For 129Xe (Bonn A): ⟨Sp⟩ = 0.009, ⟨Sn⟩ = 0.326, magnetic moment μ = -0.777 μN (experimental: -0.778 μN)
- For 131Xe (Bonn A): ⟨Sp⟩ = 0.000, ⟨Sn⟩ = 0.199, magnetic moment μ = 0.690 μN (experimental: 0.691 μN)
- For 125Te (Bonn A): ⟨Sp⟩ = 0.000, ⟨Sn⟩ = 0.310, magnetic moment μ = -0.887 μN (experimental: -0.888 μN)
- Maximum momentum transfer for A ~ 127 nuclei: q_max ≈ 550 MeV, corresponding to y_max = (q_max b/2)² ≈ 10
- Oscillator parameter for A = 127: b = 2.282 fm = 1/86.47 MeV
- For a 100 GeV neutralino moving at mean halo velocity: y_max ≈ 0.4

## Context
This work extends previous nuclear structure calculations for dark matter detection targets, specifically improving upon the Odd Group Model (OGM) values used in experimental analyses by the DAMA (Bernabei, 1996) and UK Dark Matter (Boulby) collaborations. The authors compare their results to calculations using the Interacting Boson Fermion Model (IBFM), Independent Single Particle Shell Model (Ellis, 1988), Theory of Finite Fermi Systems (TFFS), and Quasi Tamm-Dancoff Approximation (QTDA) for 131Xe (Engel, 1992; Pacheco, 1992). The structure functions enable more accurate calculation of event rates in direct detection experiments using sodium iodide (Bottino, 1996), xenon, and tellurium targets, and can be incorporated into the Neutdriver code of Jungman, Kamionkowski, and Griest (1996).
