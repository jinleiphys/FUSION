# Proton-Proton Fusion in Effective Field Theory

> The rate for the fusion process $p + p \ra d + e^+ + ν_e$ is calculated using non-relativistic effective field theory. Including the four-nucleon derivative interaction, results are obtained in next-to-leading order in the momentum expansion. This reproduces the effects of the effective range parameter. Coulomb interactions between the incoming protons are included non-perturbatively in a systematic way. The resulting fusion rate is independent of specific models and wavefunctions for the interacting nucleons. At this order in the effective Lagrangian there is an unknown counterterm which limits the numerical accuracy of the calculated rate given by the squared reduced matrix element $Λ^2(0) = 7.37$. Assuming the counterterm to have a natural magnitude, we estimate the accuracy of this result to be 6\% - 8\%. This is consistent with previous nuclear physics calculations based on effective range theory and inclusion of axial two-body weak currents. The true magnitude of the counterterm can be determined from a precise measurement of the cross-section for low-energy neutrino scattering on deuterons.

## Key claim

The proton-proton fusion rate is calculated using non-relativistic effective field theory to next-to-leading order in the momentum expansion, yielding a squared reduced matrix element of Λ²(0) = 7.37; however, an unknown counterterm at this order limits the numerical accuracy to an estimated 6%–8%, assuming the counterterm has a natural magnitude.

## Method

The authors used the Kaplan-Savage-Wise (KSW) effective field theory for non-relativistic nucleons, integrating out pions and representing all interactions via local contact terms with coupling constants C₀ and C₂. Coulomb interactions between the incoming protons were included non-perturbatively using the PDS regularization scheme, and the weak axial current transition matrix element was calculated from Feynman diagrams including one-body and two-body operators, with the deuteron treated as a bound state via its wavefunction renormalization constant.

## Key numbers

- Squared reduced matrix element for the fusion rate: Λ²(0) = 7.37
- Estimated accuracy of this result: 6%–8%
- Leading-order result for the reduced matrix element: Λ₀(0) = 2.51, giving Λ₀²(0) = 6.30
- Proton-proton scattering length: a_p = −7.82 fm
- Proton-proton effective range parameter: ρ_p = 2.79 fm
- Spin-triplet pn effective range parameter: ρ_d = 1.76 fm
- Deuteron binding energy: B = 2.225 MeV
- Neutron-proton mass difference: ΔM = 1.294 MeV
- Energy release carried by leptons: 0.931 MeV
- Parameter χ = αM/γ = 0.15, with integral I(0.15) = 4.96
- Expansion parameter γρ_d = 0.41
- Momentum scale γ = √(BM) = 45.7 MeV

## Context

This work extends the effective field theory approach to proton-proton fusion previously developed by the same authors at leading order (Kong and Ravndal, 1998, 1999), reproducing the canonical nuclear physics result from effective range theory originally derived by Salpeter (1952) and later refined by Bahcall and May (1969). The result is consistent with calculations by Park, Kubodera, Min and Rho (1998) using chiral perturbation theory, but provides a more model-independent, purely analytical derivation. The unknown counterterm can be determined from precise measurements of low-energy neutrino scattering on deuterons, as investigated in the same effective theory by Butler and Chen (2000).
