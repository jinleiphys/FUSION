# Pion in the Bethe-Salpeter approach with separable kernel

> In the paper, the static and dynamic properties of the pion in the Bethe-Salpeter approach are considered. The rank-one separable kernel of the quark-antiquark interaction is used to solve the equation analytically. Multidimensional integrals describing pion properties are calculated by several numerical methods and compared with previous papers. An error in the calculation of the interaction part of the elastic pion form factor is found. Using the corrected results, the kernel parameters are refitted. The calculated static properties as well as transition and elastic form factors are presented in comparison with the recent experimental data.

## Key claim

The Bethe-Salpeter approach with a rank-one separable kernel can describe pion static and dynamic properties, but only after correcting an error in the interaction part of the elastic form factor found in prior work by Ito and collaborators (1991). The model achieves agreement with experimental data within 10% for static constants when the quark mass and interaction range parameter are refitted.

## Method

The authors solve the Bethe-Salpeter equation analytically using a rank-one separable kernel for the quark-antiquark interaction, with the pion vertex function taken in a simple monopole form proportional to the trial function of the kernel. They calculate the weak decay constant, two-photon decay width, transition radius, charge radius, and both the relativistic impulse approximation and interaction current contributions to the elastic form factor. Three independent numerical methods are employed for the multidimensional integrals: Cauchy theorem with residue calculation, Feynman parametrization, and Wick rotation with additional three-dimensional integrals for pole contributions.

## Key numbers

- Pion mass fixed at \(m_{\pi}=140\) MeV as an implicit model parameter
- Set V parameters: quark mass \(m = 260\) MeV, interaction range \(\Lambda = 550\) MeV
- Set V results: weak decay constant \(f_{\pi} = 143.37\) MeV (experimental: \(130.41 \pm 0.0002\) MeV)
- Set V results: transition radius \(r_{\pi\gamma} = 0.639\) fm (experimental: \(0.659 \pm 0.004\) fm)
- Set V results: charge radius squared \(<r^2_{\pi}> = 0.459\) fm\(^2\) (experimental: \(0.430\) fm\(^2\))
- Set V results: two-photon decay width \(\Gamma_{\pi^0\rightarrow\gamma\gamma} = 7.249\) eV (experimental: \(7.57 \pm 0.03\) eV)
- Set I parameters (from Ito 1991): \(m = 300.0\) MeV, \(\Lambda = 500.0\) MeV, giving \(f_{\pi} = 152.74\) MeV, \(r_{\pi\gamma} = 0.575\) fm, \(<r^2_{\pi}> = 0.400\) fm\(^2\), \(\Gamma_{\pi^0\rightarrow\gamma\gamma} = 5.633\) eV

## Context

This work extends the Bethe-Salpeter approach with separable kernel originally developed by Ito and collaborators (1991, 1992) by correcting an error in the interaction current contribution to the elastic pion form factor. The model is relevant to ongoing experiments measuring the pion charge and transition form factors (E12-22-003, E12-06-101, E12-19-006, PR12-16-003), which contribute to the anomalous magnetic moment of the muon. The results enable improved parameter sets for describing pion structure within this covariant framework, though the authors note the need to investigate the influence of the full Dirac structure of the vertex function.
