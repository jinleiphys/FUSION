# Two-Loop DGLAP Splitting Functions from Light Cone Perturbation Theory

> We perform a two-loop calculation in Light Cone Perturbation Theory (LCPT) to evaluate the next-to-leading order nonsinglet splitting function. Our calculation demonstrates the methodology and feasibility of performing higher order calculations in LCPT. Since in Hamiltonian perturbation theory the longitudinal $k^+$ momentum is always positive, poles in $1/k^+$ can be regularized by a simple cutoff which cancels in physical results, without any associated ambiguities. For transverse momentum integrals we use dimensional regularization. Developing methods for loop calculations in LCPT paves the way for a systematical, automatizable procedure for precision calculations in this framework with a transparent physical partonic interpretation. This can provide a standard framework in higher order calculations in the gluon saturation regime of QCD.

# Two-Loop DGLAP Splitting Functions from Light Cone Perturbation Theory

## Key claim

The authors demonstrate that two-loop calculations in Light Cone Perturbation Theory (LCPT) are possible and free of the ambiguities sometimes attributed to axial gauge, provided the longitudinal $k^+$ momentum is treated as strictly positive and regularized by a simple cutoff that cancels in physical results. The calculation is performed for the nonsinglet splitting function as a proof of principle, but the authors hedge that this is only a first step toward a systematized, automatizable procedure.

## Method

The authors compute the next-to-leading order (NLO) nonsinglet DGLAP splitting function by evaluating the expectation value of the bare parton number operator in a dressed quark state, expanded in Light Cone Perturbation Theory. Transverse momentum integrals are regularized using dimensional regularization in $D = 4 - 2\varepsilon$ dimensions, while longitudinal $k^+$ integrals are regularized by a cutoff exploiting the positivity of $k^+$ in Hamiltonian perturbation theory. The splitting function is extracted from the coefficient of the $1/\varepsilon$ pole in the bare parton distribution function after subtracting the convolution of leading-order splitting functions with finite parts, following the operator-based renormalization approach of Curci, Furmanski and Petronzio (1980).

## Key numbers

- The leading order quark-quark splitting function obtained is $P^{(0)}_{qq} = C_F \left[ \frac{1+z^2}{(1-z)_+} + \frac{3}{2} \delta(1-z) \right]$, where $C_F = (N_c^2 - 1)/(2N_c)$.
- The calculation uses dimensional regularization with $D = 4 - 2\varepsilon$ dimensions for transverse momentum integrals.
- The incoming quark is placed off-shell with invariant mass $m^2 = -\mu_0^2$, implemented via $p^- = -\mu_0^2/(2p^+)$.
- The leading order wave function squared yields a factor $\mathbf{k}^2 / [\mathbf{k}^2 + x(1-x)\mu_0^2]^2$ in the transverse momentum integral.
- The renormalization scale is set via $\bar{\mu} = 4\pi e^{-\gamma_E} \mu$, with the bare coupling $g$ made dimensionless by the factor $\bar{\mu}^\varepsilon$.

## Context

This work extends the LCPT-based approach used in recent next-to-leading order Color Glass Condensate calculations (e.g., Chirilli 2011, Boussarie 2016, Beuf 2016, Hanninen 2017, Taels 2022) to the two-loop level, demonstrating feasibility for higher-order calculations. It directly builds on the operator definition of parton distribution functions advocated by Collins (2011) and the axial gauge splitting function calculation of Curci, Furmanski and Petronzio (1980), while addressing the long-standing concern about $1/k^+$ pole ambiguities in axial gauge raised by Belitsky (2002). The methodology enables future systematic, automatizable higher-order calculations in LCPT, particularly relevant for precision calculations in the gluon saturation regime of QCD.
