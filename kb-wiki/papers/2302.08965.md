# Multi-body wave function of ground and low-lying excited states using unornamented deep neural networks

> We propose a method to calculate wave functions and energies not only of the ground state but also of low-lying excited states using a deep neural network and the unsupervised machine learning technique. For systems composed of identical particles, a simple method to perform symmetrization for bosonic systems and antisymmetrization for fermionic systems is also proposed.

## Key claim
The authors propose a method to calculate wave functions and energies of both ground and low-lying excited states using a deep neural network with unsupervised machine learning, where the key innovation is a simple approach to enforce symmetrization for bosons and antisymmetrization for fermions at the level of the loss function rather than through network architecture. The method is demonstrated only for one-dimensional one-particle and few-body systems, and the authors note that the final accuracy for the square-well potential is only about 1.0 × 10⁻², significantly worse than the loss function convergence.

## Method
The wave function is represented by a deep neural network with Nd input units (spatial coordinates of all particles) and one output unit, using the softplus activation function and the Adam optimizer. The Hamiltonian is discretized on a spatial mesh, and the expectation value ⟨H⟩ serves as the loss function minimized during training. For identical particles, (anti)symmetrization is implemented by modifying the loss function rather than the network architecture, allowing the wave function to perfectly satisfy (anti)symmetry. Excited states are calculated sequentially by imposing orthogonality conditions to previously obtained states.

## Key numbers
- For the one-dimensional harmonic oscillator (ω = 1.0), the DNN with 4 units in one layer achieved total energy +0.500075, compared to the exact value ω/2 = 0.5, with 22448 epochs at 474.909 μs per epoch.
- For the harmonic oscillator (ω = 1.0), the best result was +0.499997 (16 units, 2 layers, 15275 epochs, 869.706 μs per epoch).
- For the square-well potential (V₀ = 0.5), the DNN with 4 units in one layer achieved total energy -0.228012, with 26600 epochs at 579.787 μs per epoch.
- The final accuracy for the harmonic oscillator was about 1.0 × 10⁻⁴, while for the square-well potential it was about 1.0 × 10⁻².
- The spatial coordinate was discretized with 1024 meshes (M = 1024) for one-dimensional calculations.

## Context
This work extends the unsupervised machine learning approach for quantum many-body problems pioneered by Saito (2018) for few-body bosonic systems and Keeble (2020) for the deuteron, which did not address fermion antisymmetrization or excited states. The method also builds on the variational principle and the universal approximation theorem of Cybenko (1989) and Hornik (1991). It enables calculation of low-lying excited states using DNNs without requiring specialized network architectures for (anti)symmetrization, though the authors note that systematic improvement remains difficult once a trial wave function ansatz is introduced.
