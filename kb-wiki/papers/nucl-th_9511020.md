# Parity-violating alpha-decay of the 3.56-MeV J,T=0+,1 state of Li-6

> We study the parity-violating alpha+d decay of the lowest 0+ state of Li-6 in a microscopic three-cluster model. The initial bound and the final scattering states are described consistently within the same model. The parity-violating decay width is calculated in perturbation theory using the parity-nonconserving nucleon-nucleon interaction of Desplanques, Donoghue, and Holstein (DDH). We find that the decay width is sensitive to dynamical degrees of freedom which are beyond the alpha+p+n model, for example, alpha excitation and breakup. In our full model, which contains breathing excitations of the alpha particle and H-3 + He-3 rearrangement, the parity-nonconserving decay width is Gamma=3.97x10^{-9} eV, using the DDH coupling constants.

# Parity-violating alpha-decay of the 3.56-MeV J,T=0+,1 state of Li-6

## Key claim

The parity-violating alpha+d decay width of the 3.56-MeV 0+ state in Li-6 is calculated to be 3.97x10^{-9} eV using the DDH weak coupling constants, but the result is highly sensitive to dynamical degrees of freedom beyond the alpha+p+n three-cluster model, such as alpha-particle breathing excitations and t+h rearrangement, and the model does not constrain the width sufficiently well despite successfully reproducing other low-energy properties of the six-nucleon system.

## Method

The authors used a microscopic three-cluster model (alpha+p+n) with consistent descriptions of both the initial 0+ bound state and the final 0- scattering state, solving the six-nucleon Schrodinger equation with the Minnesota effective interaction for the parity-conserving strong force. The parity-violating decay width was calculated in perturbation theory using the Desplanques-Donoghue-Holstein (DDH) parity-nonconserving nucleon-nucleon potential, with the bound and scattering wave functions expanded in tempered Gaussian basis functions and determined variationally. Four increasingly sophisticated model spaces were considered, sequentially adding monopole breathing modes of the alpha particle and the t+h rearrangement channel.

## Key numbers

- Parity-violating alpha+d decay width in the full model (iv) with DDH coupling constants: Gamma = 3.97x10^{-9} eV
- Experimental upper limit for the decay width: Gamma <= 6.5x10^{-7} eV (Robertson, 1984)
- Final state energy for the decay: E_f = 2.088 MeV (from 3.563 - 1.475 MeV)
- Binding energy of the 0+ state relative to alpha+p+n threshold: 0.137 MeV
- Pion-only contribution to the decay width in the simplest model (i): 1.06x10^{-9} eV
- Pion-only contribution in the full model (iv): 3.55x10^{-9} eV
- Individual (L,S)=(0,0) component width in model (i): 2.13x10^{-10} eV
- Individual (L,S)=(1,1) component width in model (i): 2.23x10^{-8} eV
- Decay width using DZ coupling constants in full model: 4.28x10^{-10} eV
- Decay width using FCDH coupling constants in full model: 1.73x10^{-9} eV
- Deuteron binding energy reproduced: -2.20 MeV
- Deuteron point-nucleon rms radius reproduced: 1.95 fm

## Context

This work extends previous theoretical studies of parity violation in the six-nucleon system, specifically addressing the discrepancy between the shell-model calculation of Robertson and Brown (1983), which gave 1.3x10^{-8} eV for the pion term, and the harmonic oscillator cluster model of Burov et al. (1986), which gave 1.1x10^{-11} eV. The authors note that both earlier studies used inconsistent descriptions of the bound and scattering states and did not properly account for the spatially extended halo-like nature of the 0+ state. The present results show that the decay width is sensitive to model-space components not well constrained by other low-energy properties, similar to findings in cluster-model studies of beta-delayed deuteron emission in He-6 (Csoto and Baye, 1994). Improved calculations in larger six-body model spaces, including deuteron and alpha-particle D-state components, would be needed to clarify the role of additional degrees of freedom.
