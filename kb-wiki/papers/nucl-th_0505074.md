# Tensor Glueball Photoproduction and Decay

> Using vector meson dominance (VMD), tensor glueball photoproduction cross sections, asymmetries and widths are calculated. The predicted hadronic VV' decays are comparable for different vector meson (V = rho, omega and phi) channels with the omega phi width the largest but the radiative omega gamma and phi gamma decays are suppressed relative to rho gamma by over a factor of 2. This decay profile is distinct from typical meson branching rates and may be a useful glueball detection signature. Results are compared to a previous VMD scalar glueball study.

## Key claim

Using vector meson dominance (VMD) and flavor independence of quark-gluonic couplings, the authors predict that tensor glueball photoproduction cross sections are measurable and that a distinct decay signature, comparable hadronic VV' widths with the largest branch to ωφ, together with suppressed ωγ and φγ radiative decays relative to ργ, may identify states with a significant gluonic component; however, the predicted suppression factor Rω/ρ = 0.44 is highly sensitive to the relative phase of the ω and φ couplings and would drop to 0.0064 if that phase were negative.

## Method

The authors construct gauge-invariant f2Vγ vertices with two coupling constants, set g1 = 0 and g2 = e g_{f2Vγ} via VMD, and express radiative couplings as sums over vector meson contributions weighted by leptonic decay constants fV. They assume flavor independence for glueball-vector meson couplings (g_{G2VV} = g_{G2V'V''}) and extract fρ, fω, fφ from measured V → e+e− decays. Hadronic G2 → VV' widths are computed using the same tensor vertex with the photon replaced by a second vector meson, and the tensor glueball coupling g_{G2VV} is estimated by saturating the total width of candidate states f2(2010) and f2(2300) with VV' decays. Photoproduction cross sections and asymmetries are calculated from t-channel vector meson exchange amplitudes in the helicity formalism.

## Key numbers

- Leptonic decay constants from V → e+e−: |fρ| = 4.965, |fω| = 17.06, |fφ| = 13.38 (dimensionless, extracted from measured widths)
- Radiative decay width ratio Rω/ρ = Γ_{G2→ωγ} / Γ_{G2→ργ} = 0.44 for same-sign phase; would be 0.0064 for opposite-sign phase
- Tensor glueball coupling g_{G2VV} = 1.60 for f2(2010) (mass 2011 MeV, total width 202 MeV); g_{G2VV} = 1.14 for f2(2300) (mass 2297 MeV, total width 149 MeV); adopted average 1.37
- Hadronic decay widths for G2(2010): Γ(ρ0ρ0) = 26.2 MeV, Γ(ωω) = 25.8 MeV, Γ(φφ) = 10.3 MeV, Γ(ωφ) = 33.0 MeV
- Hadronic decay widths for G2(2300): Γ(ρ0ρ0) = 37.2 MeV, Γ(ωω) = 36.8 MeV, Γ(φφ) = 20.3 MeV, Γ(ωφ) = 44.7 MeV
- Radiative decay widths for G2(2010): Γ(ργ) = 298 keV, Γ(ωγ) = 129 keV, Γ(φγ) = 91.6 keV, Γ(γγ) = 1.72 keV
- Radiative decay widths for G2(2300): Γ(ργ) = 377 keV, Γ(ωγ) = 164 keV, Γ(φγ) = 128 keV, Γ(γγ) = 1.96 keV
- Scalar glueball G(1700) radiative widths: Γ(ργ) = 1950 keV, Γ(ωγ) = 844 keV, Γ(φγ) = 453 keV, Γ(γγ) = 15.1 keV

## Context

This work extends a previous VMD scalar glueball study by Cotanch and Williams (2005) to the tensor glueball case, using the same VMD framework and flavor independence assumption. The predicted decay signature, comparable VV' hadronic widths with ωφ largest, and suppressed ωγ/φγ relative to ργ, is contrasted with meson decay calculations by Deandrea and Barnes (2003) which predict an order-of-magnitude enhancement for Rω/ρ in scalar meson radiative decays, and with tensor meson decay studies by Klempt and collaborators showing flavor-sensitive branching ratios varying by over an order of magnitude. The results enable experimental searches for tensor glueballs in photoproduction at Jefferson Lab's Hall D, and the authors note that an improved mixing analysis including both quark and glue sectors is in progress.
