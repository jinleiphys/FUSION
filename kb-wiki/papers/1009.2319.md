# The six-nucleon Yakubovsky equations for 6He

## Key claim

The paper formulates the six-nucleon bound state problem for 6He within the Yakubovsky scheme, reducing the full set of equations to 5 coupled equations for 5 independent amplitudes, but only provides hints for a numerical implementation rather than actual numerical results. The formulation stops at the level of 3-fragment subclusterings, exploiting the expected dominant alpha-core-plus-two-neutron structure, though the authors note that the additional step to 2-fragment subclusterings could be performed.

## Method

The authors start from the homogeneous six-body bound state equation and apply the Yakubovsky decomposition, first summing each pair force to infinite order via two-body t-operators, then introducing components labeled by sequential subclusterings (a5, a4, a3). By implementing the identity of nucleons through permutation operators, the original set of equations is reduced to 5 coupled Yakubovsky equations for 5 independent amplitudes: ψ_{12,123}^{1234}, ψ_{12;12,34}^{1234}, ψ_{12,123}^{123,45}, ψ_{12;12,34}^{125,34} + ψ_{12;12,34}^{345,12}, and ψ_{12;12,34}^{12,34,56}. The total wave function is expressed in terms of these amplitudes with explicit permutation operators, and Jacobi momenta and partial-wave basis states are defined for numerical implementation.

## Key numbers

* The T-matrix T^{a4}_{a5 b5} for a4 of type "123,4,5,6" is a 3x3 matrix; for a4 of type "12,34;5,6" it is a 2x2 matrix.
* For a3 = "1234,5,6" there are 18 pairs of (a5,a4); for a3 = "123,45,6" there are 9 pairs; for a3 = "12,34,56" there are 6 pairs.
* The kinetic energy in the chosen Jacobi coordinates for ψ_{12,34}^{123} is (1/m)(a1^2 + (3/4)a2^2 + (2/3)a3^2 + a4^2 + (3/8)a5^2).
* The total state Ψ contains a piece Ψ4 that has exactly the form of a 4-nucleon bound state based on two Yakubovsky components ψ_{12,123}^{1234} and ψ_{12;12,34}^{1234}.

## Context

This work extends the Yakubovsky approach previously applied to the 4-nucleon (alpha-particle) problem by Kamada et al. (circa 1990) to the 6-nucleon system. It contrasts with effective alpha-n-n 3-body models that use ad hoc 3-body forces and approximate Pauli projection, and with existing direct 6-nucleon approaches such as the no-core shell model (NCSM) using chiral forces and the Green's function Monte Carlo method (GFMC) using AV18 plus Urbana three-nucleon forces, both of which showed underbinding for 6He and 7Li. The formulation enables future rigorous treatment of modern chiral two- and three-nucleon forces and even four-nucleon forces in the 6-nucleon system, going beyond the effective 3-body model by allowing additional subclusterings.
