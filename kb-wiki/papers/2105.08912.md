# Application of machine learning in the determination of impact parameter in the $^{132}$Sn+$^{124}$Sn system

> Background: $^{132}$Sn+$^{124}$Sn collisions at the beam energy of 270 MeV$/$nucleon have been performed at the Radioactive Isotope Beam Factory (RIBF) in RIKEN to investigate the nuclear equation of state. Reconstructing impact parameter is one of the important tasks in the experiment as it relates to many observables. Purpose: In this work, we employ three commonly used algorithms in machine learning, the artificial neural network (ANN), the convolutional neural network (CNN) and the light gradient boosting machine (LightGBM), to determine impact parameter by analyzing either the charged particles spectra or several features simulated with events from the ultra-relativistic quantum molecular dynamics (UrQMD) model. Method: To closely imitate experimental data and investigate the generalizability of the trained machine learning algorithms, incompressibility of nuclear equation of state and the in-medium nucleon-nucleon cross sections are varied in the UrQMD model to generate the training data. Results: The mean absolute error $Δb$ between the true and the predicted impact parameter is smaller than 0.45 fm if training and testing sets are sampled from the UrQMD model with the same parameter set. However, if training and testing sets are sampled with different parameter sets, $Δb$ would increase to 0.8 fm. Conclusion: The generalizability of the trained machine learning algorithms suggests that these machine learning algorithms can be used reliably to reconstruct impact parameter in experiment.

## Key claim

The mean absolute error between true and predicted impact parameter is smaller than 0.45 fm when training and testing data are generated with the same UrQMD model parameter set, but increases to about 0.8 fm when training and testing data come from different parameter sets, indicating that the generalizability of the trained machine learning algorithms is limited by variations in the nuclear equation of state and in-medium nucleon-nucleon cross sections.

## Method

The authors simulated 60,000 events of 132Sn+124Sn collisions at 270 MeV per nucleon using the ultra-relativistic quantum molecular dynamics (UrQMD) model with four different parameter sets that varied the nuclear incompressibility (K0=200 MeV for soft EoS and K0=380 MeV for hard EoS) and the nucleon-nucleon cross section (free versus in-medium). Three machine learning algorithms were applied: artificial neural network (ANN), convolutional neural network (CNN), and light gradient boosting machine (LightGBM), using either two-dimensional transverse momentum and rapidity spectra of charged particles (DATASET1) or seven engineered features including charged particle multiplicity, transverse kinetic energy, and particle yields (DATASET2). The mean absolute error between true and predicted impact parameter was used as the performance metric.

## Key numbers

- Mean absolute error Δb smaller than 0.45 fm when training and testing sets are sampled from the same UrQMD parameter set
- Mean absolute error Δb increases to 0.8 fm when training and testing sets are sampled from different parameter sets
- For central collisions with b_true=0 fm, the predicted impact parameter averaged about 1.0 fm, showing a systematic bias in the most central events
- The Pearson correlation coefficient between impact parameter and total charged particle multiplicity M_ch was 0.93, and between impact parameter and transverse kinetic energy E_perp was 0.94
- For testing data from SM-I(MST) mode (different cluster recognition parameters), Δb values from LightGBMb ranged from 0.50 to 0.72 fm depending on training data source

## Context

This work extends previous applications of machine learning to impact parameter reconstruction in heavy-ion collisions, such as the use of ANN or support vector machine in Refs. [CDPRC, SBASS, Haddad, JD] and the authors' prior work using CNN and LightGBM (Li et al., 2020). The key advance is testing generalizability by varying the nuclear equation of state and in-medium cross sections in the UrQMD model, which previous studies did not address. The results enable application of these trained algorithms to experimental data from the 132Sn+124Sn collisions at RIBF in RIKEN, as the authors note that Δb remains below 0.8 fm even under model parameter variations, giving confidence for experimental use.
