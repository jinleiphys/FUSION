# Dynamically generated hadron resonances

> Recent developments in hadron spectroscopy triggers the discussion on various exotic configurations beyond the simple three-quark state for baryons and quark-anti-quark pair for mesons. In particular, the states observed near a two-hadron threshold are often regarded as candidates of hadronic molecules, which are made of constituent hadrons and dynamically generated by the hadronic interactions. However, identification of the hadronic molecules involves several subtle difficulties which are often overlooked or underestimated. Here we summarize these subtleties and present a promising approach to distinguish the dynamically generated states from others using observable quantities.

## Key claim

The paper argues that identifying hadronic molecules (dynamically generated states) is subtle because the decomposition of the full Hamiltonian into a free part and an interaction is not unique, but that for near-threshold s-wave states the weak-binding relation can determine the compositeness from observable quantities, even for unstable resonances.

## Method

The author reviews Weinberg's weak-binding relation for stable bound states, which connects the scattering length, binding energy, and compositeness (the weight of the two-body component in the wave function). This relation is then generalized to unstable resonances using an effective field theory that includes a two-body decay channel, yielding a modified weak-binding relation that depends on the complex eigenenergy and scattering length. The generalized relation is applied to the high-mass pole of the Λ(1405) resonance, using four sets of input parameters from the literature.

## Key numbers

*   For the deuteron, the weak-binding relation uses the scattering length \(a_0\) and binding energy \(B\) with the condition \(R \gg R_{\rm typ}\), where \(R = 1/\sqrt{2\mu B}\) and \(R_{\rm typ} \sim 1/m_\pi\) (pion mass).
*   For the high-mass pole of Λ(1405), four data sets give eigenenergies \(E_h\) (measured from the \(\bar{K}N\) threshold) and \(\bar{K}N\) scattering lengths \(a_0\): Set 1: \(E_h = -10 - i26\) MeV, \(a_0 = 1.39 - i0.85\) fm; Set 2: \(E_h = -13 - i20\) MeV, \(a_0 = 1.30 - i0.85\) fm; Set 3: \(E_h = 2 - i10\) MeV, \(a_0 = 1.21 - i1.47\) fm; Set 4: \(E_h = -3 - i12\) MeV, \(a_0 = 1.52 - i1.85\) fm.
*   The resulting real-valued \(\bar{K}N\) compositeness \(\tilde{X}_{\bar{K}N}\) for the high-mass pole of Λ(1405) is close to unity in all sets: Set 1: \(1.0^{+0.0}_{-0.4}\); Set 2: \(0.9^{+0.1}_{-0.4}\); Set 3: \(0.6^{+0.3}_{-0.1}\); Set 4: \(0.8^{+0.2}_{-0.2}\).

## Context

This work extends Weinberg's (1965) weak-binding relation for stable bound states, originally applied to the deuteron, to unstable hadron resonances. It builds on earlier generalizations by Baru et al. (2003) and Kamiya et al. (2015, 2016) using effective field theory. The approach enables the characterization of near-threshold states like the Λ(1405) as hadronic molecules, addressing the ambiguity in defining dynamically generated states, and it opens the way to establishing a standard prescription for the structure of unstable resonances in hadron spectroscopy.
