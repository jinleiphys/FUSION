# Relativistic separable interaction kernel of the neutron-proton system with inelasticities

> Within a covariant Bethe-Salpeter approach, the relativistic complex separable neutron-proton interaction kernel is proposed. The uncoupled partial-wave states with the total angular momentum $J$=0,1 are considered. The multirank separable potentials elaborated earlier are real-valued and, therefore, enable to describe the elastic part (phase shifts, low-energy parameters, etc.) of the scattering only. The description of the inelasticity parameter comes out of the imaginary part introduced into them. To obtain parameters of the complex potentials the elastic neutron-proton scattering experimental data up to 3 GeV are used. A signal of dybaryon resonances in the $^3P_0^+$ partial-wave state is discussed.

## Key claim

The paper proposes a relativistic complex separable neutron-proton interaction kernel within the Bethe-Salpeter approach, where the imaginary part is introduced as a minimal extension to previously published real-valued potentials to describe inelasticity in elastic scattering. The authors note that the observed deviation in the inelasticity parameter for the $^3P_0^+$ partial-wave state "can be interpreted as a presence of the dybaryon resonances," but caution that "this conclusion needs further investigation."

## Method

The authors start from real-valued multirank separable interaction kernels previously obtained by fitting phase shifts and low-energy parameters (Bondarenko et al., 2002; earlier papers npa1, npa2). They fix all parameters of the real part and introduce an imaginary part $\lambda^i_{mn}(s) = \theta(s-s_{th}) (1-s_{th}/s) \bar\lambda^i_{mn}$ to the separable kernel, where $s_{th}$ is the inelasticity threshold. The new imaginary parameters $\bar\lambda^i_{mn}$ are determined by minimizing a $\chi^2$ function that includes both phase shifts and inelasticity parameters from the SAID program for uncoupled partial-wave states with $J=0,1$, using elastic neutron-proton scattering data up to laboratory kinetic energy $T_{\rm Lab} \sim 3$ GeV.

## Key numbers

- Inelasticity thresholds $T_{\rm Lab, th}$: 0.25 GeV for $^3P_0^+$, 0.3 GeV for $^1S_0^+$, and 0.35 GeV for both $^1P_1^+$ and $^3P_1^+$ partial-wave states.
- Imaginary kernel parameters for the $^3P_0^+$ state (rank-two): $\bar\lambda^i_{11} = 97.12885$ GeV$^4$, $\bar\lambda^i_{12} = -114.857$ GeV$^4$, $\bar\lambda^i_{22} = -35.16663$ GeV$^4$.
- Imaginary kernel parameters for the $^1S_0^+$ state (rank-three): $\bar\lambda^i_{11} = -0.01332595$ GeV$^2$, $\bar\lambda^i_{22} = -58097.6$ GeV$^2$, $\bar\lambda^i_{33} = -217.6001$ GeV$^2$.
- For the $^3P_0^+$ state, a Breit-Wigner fit to the difference $\Delta\eta$ yields two possible dybaryon resonances with masses $M^*_1 = 2.27$ GeV and $M^*_2 = 2.55$ GeV, and widths $\Gamma_1 = 0.199$ GeV and $\Gamma_2 = 1.335$ GeV.

## Context

This work extends the real-valued relativistic separable potentials developed by Bondarenko, Burov, and collaborators (2002; earlier papers npa1, npa2) which described only elastic scattering (phase shifts, low-energy parameters, deuteron properties) up to $T_{\rm Lab} \sim 3$ GeV. By adding an imaginary part, the authors enable the description of the inelasticity parameter, following the approach of using complex NN potentials as done by Eyser and Knyr (no year given). The resulting complex kernels (labeled MYI2, MYI3) are compared with the SP07 solution from Arndt et al. (2007) and reproduce experimental data up to 3 GeV, while the observed deviation in the $^3P_0^+$ state opens the possibility of studying dibaryon resonances in future work.
