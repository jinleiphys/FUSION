# Study on N Omega-bar systems in a chiral quark model

> The N Omega-bar systems with spin S=1 and S=2 are dynamically investigated within the framework of the chiral SU(3) quark model and the extended chiral SU(3) quark model by solving the resonating group method (RGM) equation. The model parameters are taken from our previous work, which gave a good description of the energies of the baryon ground states, the binding energy of deuteron, and the experimental data of the nucleon-nucleon (NN) and nucleon-hyperon (NY) scattering processes. The results show that N Omega-bar states with spin S=1 and S=2 can be bound both in the chiral SU(3) and extended chiral SU(3) quark models, and the binding energies are about 28-59 MeV. When the annihilation effect is considered, the binding energies increase to about 37-130 MeV, which indicates the annihilation effect plays a relatively important role in forming an N Omega-bar bound state. At the same time, the N Omega-bar elastic scattering processes are also studied. The S, P, D partial wave phase shifts and the total cross sections of S=1 and S=2 channels have been calculated by solving the RGM equation for scattering problems.

## Key claim

The paper claims that N Omega-bar systems with spin S=1 and S=2 can form bound states in both the chiral SU(3) quark model and the extended chiral SU(3) quark model, with binding energies of about 28-59 MeV without annihilation and 37-130 MeV when annihilation is included. The hedge is that these results depend on model parameters taken from previous work that described nucleon-nucleon and nucleon-hyperon scattering, and that the annihilation effect is treated only through a specific rearrangement mechanism into K and K* mesons.

## Method

The authors solved the resonating group method (RGM) equation for the N Omega-bar six-quark system using two variants of the chiral SU(3) quark model: the standard chiral SU(3) quark model (with one-gluon exchange and scalar/pseudoscalar meson exchanges) and the extended chiral SU(3) quark model (which additionally includes vector meson exchanges). The model parameters were taken from previous work by the same group that reproduced baryon ground state energies, the deuteron binding energy, and NN and NY scattering data. The annihilation effect was incorporated via a rearrangement mechanism where u(d) and anti-s quarks annihilate into K and K* mesons, with coupling constants fitted to the K and K* meson masses.

## Key numbers

- Binding energy of N Omega-bar (S=1) without annihilation: 28.3 MeV in chiral SU(3) quark model (set I), 58.8 MeV in extended chiral SU(3) quark model (set II), 53.7 MeV in extended chiral SU(3) quark model (set III)
- Binding energy of N Omega-bar (S=2) without annihilation: 28.8 MeV in set I, 59.5 MeV in set II, 54.4 MeV in set III
- Binding energy of N Omega-bar (S=1) with annihilation: 46.2 MeV in set I, 129.5 MeV in set II, 132.2 MeV in set III
- Binding energy of N Omega-bar (S=2) with annihilation: 37.2 MeV in set I, 102.0 MeV in set II, 107.3 MeV in set III
- RMS radii for bound states without annihilation: 0.8 fm for set I, 0.7 fm for sets II and III (both S=1 and S=2)
- RMS radii for bound states with annihilation: 0.8 fm for set I, 0.6 fm for sets II and III (both S=1 and S=2)
- Energy shift from annihilation for S=1: about 18 MeV in set I, 70 MeV in set II, 78 MeV in set III
- Energy shift from annihilation for S=2: about 8 MeV in set I, 42 MeV in set II, 54 MeV in set III
- Greatest difference in total cross sections between set I and set II: about 170 mb in the very low energy region
- Total cross sections in medium energy region (E_cm approximately 10-30 MeV): around 200-250 mb for all three parameter sets

## Context

This work extends the authors' previous studies of baryon-antibaryon systems using chiral quark models, following their earlier investigations of N Omega systems (Zhang et al., 2006) where binding energies were found to be much smaller (3.0-20.4 MeV). The N Omega-bar system is identified as a particularly clean system for studying the rearrangement annihilation mechanism because, unlike N anti-N systems, it has only one annihilation channel (into K and K* mesons) rather than three competing mechanisms. The results enable future experimental searches for N Omega-bar bound states in heavy ion collision experiments, where the abundance of both nucleons and Omega (anti-Omega) baryons is expected to be significant.
