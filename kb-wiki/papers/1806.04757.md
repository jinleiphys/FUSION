# Number of spin-J states and odd-even staggering for identical particles in a single-j shell

> In this work, new recursion relations for the number of spin-J states for identical particles in a single-j shell are presented. Such relations are obtained using the generating-function technique, which enables one to exhibit an odd-even staggering in the spin distribution of an even number of fermions in a single-j shell: the number of states with an even value of J is larger than the number of states with an odd value of J. An analytical expression of the excess of states with an even value of J is provided, and its asymptotic behavior for large values of j is discussed.

## Key claim

The paper presents new recursion relations for the number of spin-\(J\) states of identical fermions in a single-\(j\) shell, derived using generating functions, and demonstrates that for an even number of fermions the number of states with even \(J\) always exceeds the number with odd \(J\), with the excess given by a simple binomial coefficient.

## Method

The generating function \(f_j(x,z) = \prod_{i=1}^{g} (1 + z x^{m_i})\) (with \(g = 2j+1\)) is expanded to obtain the distribution \(D(M,j,n)\) of the angular-momentum projection \(M\). Two new recursion relations for \(D(M,j,n)\) are derived: one by removing one single-particle state (Eq. (recint)), and another by summing over all single-particle states (Eq. (mainresu)). The number of spin-\(J\) states is then obtained via \(N(J,j,n) = D(J,j,n) - D(J+1,j,n)\). The excess of even-\(J\) states for \(n = 2k\) is evaluated by setting \(x = -1\) in the generating function, yielding \(E(j^{2k}) = \binom{j+1/2}{k}\).

## Key numbers

* For the shell \((j=7/2)^4\): 7 even-\(J\) states and 1 odd-\(J\) state (the odd state has \(J=5\)); excess = 6.
* For the shell \((j=11/2)^4\): 24 even-\(J\) states and 9 odd-\(J\) states; excess = 15.
* For the shell \((j=11/2)^6\): 39 even-\(J\) states and 19 odd-\(J\) states; excess = 20.
* For the shell \((j=15/2)^6\): excess = 56.
* Asymptotic minimum of the ratio \(r(j^{2k}) = E / N_{\text{tot}}\) for large \(j\): \(r_{\text{min}} \approx \sqrt{2\pi/3} \, j^{3/2} / 2^j\).
* Variance of the \(M\) distribution: \(v(j^n) = n(2j+1-n)(j+1)/6\).

## Context

This work extends the long line of research on enumerating spin-\(J\) states in the single-\(j\) shell, beginning with Bethe (1936). It provides new recursion relations distinct from the one derived by Talmi (2005), which is recovered in the appendix. The odd-even staggering result complements earlier empirical formulas by Zhao and Arima (2003) and the Ginocchio-Haxton formula (1993) for \(N(j,j,3)\). The generating-function approach enables efficient computation (polynomial cost in \(j\) and \(n\)) and opens the way to studying similar staggering effects in systems with isospin or other constraints.
