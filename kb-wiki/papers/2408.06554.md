# Application of the shift-invert Lanczos algorithm to a non-equilibrium Green function for transport problems

> Non-equilibrium Green's function theory and related methods are widely used to describe transport phenomena in many-body systems, but they often require a costly inversion of a large matrix. We show here that the shift-invert Lanczos method can dramatically reduce the computational effort. We apply the method to two test problems, namely a simple model Hamiltonian and to a more realistic Hamiltonian for nuclear fission. For a Hamiltonian of dimension 66103 we find that the computation time is reduced by a factor of 33 compared to the direct calculation of the Green's function.

## Key claim

The shift-invert Lanczos method can dramatically reduce the computational effort required for non-equilibrium Green's function calculations in transport problems, but only when the number of eigenstates needed to represent the Green function is small (e.g., k=10 for a matrix of dimension 66103) and when ensemble averaging over many samples is performed to achieve convergence.

## Method

The authors apply the shift-invert Lanczos algorithm to compute selected eigenstates of the effective Hamiltonian matrix around a target energy E, then use these eigenstates to construct the Green function via spectral decomposition. For the GOE+barrier+GOE model, they treat the decay matrix perturbatively for small widths and use the shift-invert Arnoldi method for overlapping resonances. For the realistic fission Hamiltonian of 236U, they construct a block-tridiagonal Hamiltonian matrix from density functional theory configurations along a fission path, with GOE matrices replacing the leftmost and rightmost blocks.

## Key numbers

- For a Hamiltonian of dimension 66103, the shift-invert Lanczos method reduces computation time by a factor of 33.4 (real time) compared to direct matrix inversion.
- For the GOE+barrier+GOE model with 100 samples, the error in transition probabilities becomes less than 1% when using k=20 eigenstates.
- For the 236U fission problem at E=6.5 MeV with 100 samples, the error is smaller than 3% for both capture and fission probabilities, and results are almost converged with k=10 eigenstates.
- The GOE model uses N_GOE=100 and v=0.1 (in units of the typical energy scale), with gamma_in=1.25x10^-3, gamma_L=3.25x10^-3, and gamma_R=1x10^-3.
- For the 236U calculation, gamma_n=0.01 MeV, gamma_cap=0.00125 MeV, and gamma_fis=0.015 MeV.
- The fission path has 14 reference points from Q=14 barn to Q=83 barn, with individual Q-block dimensions ranging from 2196 to 15088.

## Context

This work extends the application of the shift-invert Lanczos method, previously used in electronic structure calculations (Grosso 1995), to non-equilibrium Green's function calculations for nuclear transport problems. It directly addresses the computational bottleneck identified by Bertsch (2023) for neutron-induced fission calculations, where matrix dimensions can reach 10^6. The method enables efficient calculation of transmission probabilities for nuclear fission reactions, and the authors note that future work should include finite seniority configurations beyond the seniority-zero approximation used here.
