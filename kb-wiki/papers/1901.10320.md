# On the elliptic flow of heavy quarkonia in $pA$ collisions

> Using the dilute-dense factorization in the Color Glass Condensate framework, we investigate the azimuthal angular correlation between a heavy quarkonium and a charged light hadron in proton-nucleus collisions. We extract the second harmonic $v_2$, commonly known as the elliptic flow, with the light hadron as the reference. This particular azimuthal angular correlation between a heavy meson and a light hadron has first been measured at the LHC recently. The experimental results indicate that the elliptic flows for heavy-flavor mesons ($J/ψ$ and $D^0$) are almost as large as those for light hadrons. Our calculation demonstrates that this result can be naturally interpreted as an initial state effect due to the interaction between the incoming partons from the proton and the dense gluons inside the target nucleus. Since the heavy quarkonium $v_2$ exhibits a weak mass dependence according to our calculation, we predict that the heavy quarkonium $Υ$ should have a similar elliptic flow as compared to that of the $J/ψ$, which can be tested in future measurements.

## Key claim

The observed large elliptic flow ($v_2$) of heavy quarkonia in proton-nucleus collisions can be naturally interpreted as an initial state effect arising from color interactions between incoming partons and dense gluons in the target nucleus, rather than requiring final-state hydrodynamic evolution. The heavy quarkonium $v_2$ exhibits a weak mass dependence, leading to the prediction that the $\Upsilon$ meson should have a similar elliptic flow to the $J/\psi$, though the calculation uses a simplified model based on the Color Evaporation Model and the dilute-dense factorization in the Color Glass Condensate framework.

## Method

The authors computed the azimuthal angular correlation between a heavy quarkonium (produced via $g \to c\bar c$ splitting) and a reference light quark in proton-nucleus collisions using the dilute-dense factorization of the Color Glass Condensate framework. They employed the Color Evaporation Model to convert the heavy quark pair into a quarkonium, and evaluated the three-dipole correlators in the McLerran-Venugopalan model up to $1/N_c^2$ order, with the incoming parton distributions modeled as Gaussian Wigner functions. The elliptic flow $v_2$ was extracted as the second Fourier harmonic of the azimuthal angle difference between the heavy quarkonium and the reference light hadron, normalized by the integrated $v_2$ of the reference quark.

## Key numbers

* The integrated $v_2$ for $J/\psi$ and $\Upsilon$ mesons is roughly between $5\%$ and $10\%$, which is about $2/3$ of that for light reference quarks, as a function of the saturation momentum $Q_s^2$.
* The calculation uses $Q_s^2 = 5\ \text{GeV}^2$ for LHC energies (compared to $Q_s^2 = 4\ \text{GeV}^2$ for RHIC).
* The threshold for the $J/\psi$ production in the Color Evaporation Model is taken as $1.4\ \text{GeV}$ for the integration over the relative momentum of the heavy quark pair.
* The $k_\perp$-dependent $v_2$ of $J/\psi$ shows excellent agreement with CMS data (CMS 2018) across the transverse momentum range, with both systematic and statistical error bars shown.

## Context

This work extends the Color Glass Condensate framework for two-particle azimuthal correlations developed in earlier works (Lappi 2015, Dusling 2017, Davy 2018) to include heavy quarkonium production via gluon splitting. It directly addresses the experimental measurements by ALICE (2017) and CMS (2018) showing large $v_2$ for $J/\psi$ and $D^0$ mesons in proton-lead collisions, which had been difficult to explain with hydrodynamic approaches alone (Du 2018). The calculation enables future predictions for $\Upsilon$ meson elliptic flow and provides a consistent framework to describe both the elliptic flow and the nuclear modification factor $R_{pPb}$ for heavy quarkonia in the low transverse momentum region.
