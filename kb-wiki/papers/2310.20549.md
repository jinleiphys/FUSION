# Taweret: a Python package for Bayesian model mixing

> Uncertainty quantification using Bayesian methods is a growing area of research. Bayesian model mixing (BMM) is a recent development which combines the predictions from multiple models such that each model's best qualities are preserved in the final result. Practical tools and analysis suites that facilitate such methods are therefore needed. Taweret introduces BMM to existing Bayesian uncertainty quantification efforts. Currently Taweret contains three individual Bayesian model mixing techniques, each pertaining to a different type of problem structure; we encourage the future inclusion of user-developed mixing methods. Taweret's first use case is in nuclear physics, but the package has been structured such that it should be adaptable to any research engaged in model comparison or model mixing.

## Key claim

Taweret introduces Bayesian model mixing (BMM) to existing Bayesian uncertainty quantification efforts in nuclear physics, but the package is structured to be adaptable to any research engaged in model comparison or mixing; the authors encourage future inclusion of user-developed mixing methods beyond the three currently implemented.

## Method

The paper describes the development of a Python package that implements three distinct Bayesian model mixing techniques: bivariate linear mixing (for two models using either density-mixing or mean-mixing with step, sigmoid, or asymmetric 2-step weight functions), multivariate mixing (combining K models by weighting each by the inverse of their variance, producing a Gaussian posterior predictive distribution), and BART-based mixing (using Bayesian Additive Regression Trees to learn weight functions adaptively). The package uses abstract base classes (BaseModel and BaseMixer) to ensure compatibility and uniformity, requiring specific functions such as evaluate, train, predict, and predict_weights for interoperability.

## Key numbers

- Bivariate linear mixing can mix exactly 2 models (K=2) with 1-dimensional input parameter x
- Multivariate mixing can combine K models (K >= 2) with 1-dimensional input space and 1 output
- BART mixing can handle K models (K >= 2) with >=1 input dimensions and 1 output
- The multivariate mixing method produces a mixed model posterior predictive distribution that is Gaussian with precision Z_P = sum_{k=1}^{K} 1/sigma^2_k

## Context

This work extends the BAND (Bayesian Analysis of Nuclear Dynamics) collaboration's cyber-infrastructure framework, which includes emulation, calibration, and experimental design as its other central pillars (Phillips 2020, bandframework). The package improves upon SAMBA (Semposki 2022), a prior BAND effort that was developed for testing BMM methods on a toy model, by offering increased functionality and a diverse selection of mixing methods. Outside physics, the techniques relate to model stacking and Bayesian model averaging used in statistics (Yao 2018, 2022), meteorology (Sloughter 2007), and neuroscience (FitzGerald 2014).
