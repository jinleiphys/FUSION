# Topics Concerning the Quadrupole-Quadrupole Interaction

> We address some properties of the quadrupole-quadrupole ($Q \cdot Q$) interaction in nuclear studies. We first consider how to restore $SU(3)$ symmetry even though we use only coordinate and not momentum terms. Using the Hamiltonian $H=\sum_i (p^2/2m + m/2 ω^2 r_i^2) -χ\sum_{i < j}Q(i) \cdot Q(j) - χ/2 \sum_i Q(i) \cdot Q(i)$ with $Q_μ=r^2 Y_{2,μ}$, we find that only 2/3 of the single-particle splitting ($ε_{0d}-ε_{1s}$) comes from the diagonal term of $Q \cdot Q$ -the remaining 1/3 comes from the interaction of the valence nucleus with the core. On another topic, a previously derived relation, using $Q \cdot Q$, between isovector orbital $B(M1)$ (scissors mode) and the ``difference'' ($B(E2, isoscalar)-B(E2, isovector)$) is discussed. It is shown that one needs the isovector $B(E2)$ in order that one get the correct limit as one goes to nuclei sufficiently far from stability so that one subshell (neutron or proton) is closed.

# Topics Concerning the Quadrupole-Quadrupole Interaction

## Key claim
Using only the coordinate part of the quadrupole-quadrupole interaction (without momentum terms) in a shell model calculation, only 2/3 of the single-particle splitting needed to reproduce Elliott's SU(3) results comes from the diagonal Q·Q term; the remaining 1/3 comes from the interaction of valence nucleons with the core. The previously derived sum rule relating isovector orbital B(M1) (scissors mode) to the difference between isoscalar and isovector B(E2) requires the isovector B(E2) to be non-negligible in order to give the correct vanishing scissors mode rate in nuclei with one closed subshell.

## Method
The authors performed shell model calculations using the Hamiltonian H = Σᵢ(p²/2m + m/2 ω² rᵢ²) − χ Σᵢ<ⱼ Q(i)·Q(j) − χ/2 Σᵢ Q(i)·Q(i) with Q_μ = r² Y₂,μ, evaluating the single-particle potential U(r) = −χ/2 Q(i)·Q(i) = 4χ̄(r/b)⁴ and its expectation values for various single-particle states. They then compared the diagonal Q·Q contribution to the single-particle splitting with the full SU(3) result, and extended the analysis to the 0f-1p shell. For the scissors mode relation, they calculated B(M1) and B(E2) values for the even-even Be isotopes ⁶Be, ⁸Be, ¹⁰Be, and ¹²Be using the Q·Q interaction, examining the sum rule of Zheng and Zamick (1992).

## Key numbers
- Only 2/3 of the single-particle splitting (ε₀d−ε₁s) comes from the diagonal Q·Q term; the diagonal contribution is 12χ̄ out of the required 18χ̄ for the 1s-0d shell.
- In the 0f-1p shell, the required SU(3) splitting is ε₀f−ε₁p = 30χ̄, with only 20χ̄ (2/3) coming from the diagonal Q·Q term.
- For ⁸Be, the calculated B(M1,0₁→1₁) is 2/π μ_N² = 0.637 μ_N².
- For ¹⁰Be, the calculated B(M1) values are 9/32π μ_N² = 0.0895 μ_N² (T=1→T=1) and 15/32π μ_N² = 0.149 μ_N² (T=1→T=2), giving a ratio ¹⁰Be/⁸Be = 3/8.
- For ⁸Be, B(E2,isoscalar) = 82.66 e²fm⁴ and B(E2,isovector) = 7.371 e²fm⁴ (using b=1.650 fm).
- For ⁶Be and ¹²Be, B(E2,isoscalar) = B(E2,isovector) = 19.92 e²fm⁴.

## Context
This work extends the SU(3) framework of Elliott (1958) by showing how to obtain his results using only the coordinate Q·Q interaction without momentum terms, explicitly quantifying the role of particle-core interactions. The scissors mode analysis builds on the sum rule of Zamick and Zheng (1992) and the empirical observation by Ziegler, Rangacharyulu, Richter, and Spieler (1990) that B(M1) in Sm isotopes is proportional to B(E2). The calculations for Be isotopes extend earlier work by Fayache, Sharma, and Zamick (1995) and demonstrate that the isovector B(E2) is essential for the sum rule to correctly predict vanishing scissors mode strength in singly-closed-shell nuclei, which is relevant for studies of nuclei far from stability with radioactive beams.
