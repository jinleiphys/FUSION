# Exact three-dimensional wave function and the on-shell t-matrix for the sharply cut off Coulomb potential: failure of the standard renormalization factor

> The 3-dimensional wave function for a sharply cut-off Coulomb potential is analytically derived. The asymptotic form of the related scattering amplitude reveals a failure of the standard renormalization factor which is believed to be generally valid for any type of screening.

# Exact three-dimensional wave function and the on-shell t-matrix for the sharply cut off Coulomb potential: failure of the standard renormalization factor

## Key claim

The standard renormalization factor, believed to be generally valid for any type of screening, fails for the sharply cut-off Coulomb potential: the asymptotic scattering amplitude contains an additional term beyond the expected infinitely oscillating phase factor, meaning the literature derivations based on partial wave decomposition must be incomplete.

## Method

The authors analytically solved the 3-dimensional Schrodinger equation for a sharply cut-off Coulomb potential (V(r) = Θ(R-r) e²/r) by using the Lippmann-Schwinger equation directly in 3 dimensions, avoiding partial wave decomposition. They derived the exact wave function inside the screening radius (r < R) using parabolic coordinates and confluent hypergeometric functions, then obtained the exact scattering amplitude via an integral representation of the hypergeometric function. The asymptotic limit R → ∞ was evaluated using the method of steepest descent around integration boundaries.

## Key numbers

- The exact wave function for r < R is Ψ_R^(+)(r) = (1/(2π)^(3/2)) * (1/₁F₁(-iη,1,iT)) * e^(ip·r) * ₁F₁(-iη, 1, i(pr - p·r)), where T = 2pR and η = me²/(2p) is the Sommerfeld parameter.
- The asymptotic scattering amplitude for R → ∞ is f_R = (2pR)^(-2iη) A_c(θ) + (η/(4p sin²(θ/2))) [e^(2ipR sin(θ/2)) ((1-sin(θ/2))/2)^(iη) + e^(-2ipR sin(θ/2)) ((1+sin(θ/2))/2)^(iη)], where A_c(θ) is the physical Coulomb amplitude.
- Numerical tests for pp scattering at E_p^lab = 13 MeV showed that the exact renormalization factor from the derived expression brings numerical solutions into agreement with the pure Coulomb amplitude, while the standard renormalization (using only the first term) fails, leaving strong oscillations in both real and imaginary parts.
- The exact factor A = 1/₁F₁(-iη,1,iT) was verified numerically to four significant digits for various cut-off radii R.

## Context

This work extends the rigorous 3-dimensional approach of Gorshkov (1961, 1962) for Yukawa potentials to the sharply cut-off Coulomb potential, and critically examines the partial-wave-based derivations of Ford (1964, 1966) and Taylor (1974, 1983) which claimed the standard renormalization factor was generally valid. The paper shows that the standard renormalization factor (the first term in the asymptotic amplitude) is incomplete for sharp cut-off, and that the additional oscillatory terms must be included. This raises questions about whether similar failures occur for finite screening exponents n in the screened potential V_R(r) = (e²/r) exp(-(r/R)^n), which remains an open question. The results enable more rigorous treatment of screened Coulomb potentials in few-body scattering calculations, particularly in the context of the 3-body Faddeev equations where screened Coulomb t-matrices are commonly used (Alt 1978, 2002; Deltuva 2005).
