# New symmetry-adapted ab initio approach to nuclear reactions for intermediate-mass nuclei

> With a view toward describing reactions of intermediate-mass nuclei from first principles, we present first results for the norm and Hamiltonian overlaps (kernels) for the p-α, p-16O and p-20Ne cluster systems using realistic nucleon-nucleon interactions. This is achieved in the framework of a new ab initio approach that combines the symmetry-adapted no-core shell model (SA-NCSM) with the resonating group method (RGM). In this model, a physically relevant basis based on the SU(3) symmetry is used. The structure of the clusters is provided by the ab initio SA-NCSM, which enables the description of spatially enhanced nuclear configurations and heavier nuclei, by exploiting symmetries known to dominate in nuclei. Here, we discuss the applicability and efficacy of this approach.

## Key claim

The paper presents first results for norm and Hamiltonian kernels for p-alpha, p-16O, and p-20Ne cluster systems using realistic nucleon-nucleon interactions, but these results are reported with the center-of-mass motion included for the projectile-target system (removal is work in progress), and the authors note this CM effect is expected to be negligible only for targets with A greater than or equal to 16.

## Method

The authors combine the symmetry-adapted no-core shell model (SA-NCSM) with the resonating group method (RGM) to compute norm and Hamiltonian kernels for nucleon-nucleus systems. The SA-NCSM provides cluster wave functions using an SU(3)-symmetry basis that allows drastic truncation of the configuration space, while the RGM framework organizes nucleons into clusters and enforces antisymmetrization through an exchange matrix that becomes diagonal within the SU(3) basis. The kernels are computed in laboratory coordinates and then transformed to a partial-wave expansion for input to an R-matrix approach.

## Key numbers

* Benchmark calculation for p-4He: the SA-RGM exchange norm kernel (ell = 0) using a 4He wave function truncated to only several SU(3) basis states shows excellent agreement with the full NCSM/RGM calculation, performed in 4 shells with hbar-Omega = 15 MeV.
* For p-16O: the norm kernel is calculated using the chiral NNLO_sat interaction in 10 shells with hbar-Omega = 16 MeV, selecting SU(3) configurations with contribution greater than 2%.
* For p-20Ne: the norm kernel is calculated using the chiral NNLO_opt interaction in 13 shells with hbar-Omega = 15 MeV, selecting SU(3) configurations with contribution greater than 2%.
* Convergence study for p-16O: including up to 4 shells for the projectile yields a converged norm kernel using the NNLO_sat interaction.
* The direct part of the Hamiltonian kernel for p+20Ne shows positive peaks around r = 3 fm, which the authors relate to the deformed and clustered structure of 20Ne.

## Context

This work extends the NCSM/RGM approach developed by Quaglioni (2009) and collaborators, which successfully described reactions of light nuclei but becomes numerically challenging for heavier systems due to configuration space size. The authors address this limitation by incorporating the SA-NCSM framework, previously demonstrated by Dytrych (2013) and Launey (2016) for medium-mass nuclei, which uses SU(3) symmetry to drastically reduce the required model spaces. This enables first-principles reaction calculations for intermediate-mass nuclei relevant to astrophysical simulations, such as X-ray burst nucleosynthesis studied by Cyburt (2016), and opens the path toward obtaining first-principle optical potentials for these systems.
