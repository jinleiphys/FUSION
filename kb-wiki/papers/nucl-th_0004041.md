# The decay $蟻^{0}\to 蟺^{+}+蟺^{-}+纬$ and the coupling constant g$_{蟻蟽纬}$

> The experimental branching ratio for the radiative decay $蟻^{0}\to蟺^{+}+蟺^{-}+纬$ is used to estimate the coupling constant $g_{蟻蟽纬}$ for a set of values of $蟽$-meson parameters M$_蟽$ and $螕_蟽$. Our results are quite different than the values of this constant used in the literature.

# The decay $\rho^{0}\to \pi^{+}+\pi^{-}+\gamma$ and the coupling constant $g_{\rho\sigma\gamma}$

## Key claim

The coupling constant $g_{\rho\sigma\gamma}$ is estimated from the experimental branching ratio of the radiative decay $\rho^{0}\to \pi^{+}+\pi^{-}+\gamma$, yielding values that are "quite different" from the value $g_{\rho\sigma\gamma}\simeq 2.71$ used in one-meson exchange photoproduction calculations, but the result depends strongly on the assumed mass $M_{\sigma}$ and width $\Gamma_{\sigma}$ of the $\sigma$-meson.

## Method

The authors calculate the branching ratio for $\rho^{0}\to \pi^{+}+\pi^{-}+\gamma$ using Feynman diagrams that include both internal bremsstrahlung (where a charged pion emits the photon) and structural radiation dominated by a $\sigma$-meson intermediate state ($\rho^{0}\to \sigma+\gamma$, $\sigma\to\pi^{+}\pi^{-}$). The $\rho\sigma\gamma$ vertex is described by an effective Lagrangian with coupling constant $g_{\rho\sigma\gamma}$, and the $\sigma$-meson propagator includes a complex mass $M_{\sigma}-i\Gamma_{\sigma}/2$. By equating the calculated decay width to the experimental branching ratio, they solve a quadratic equation for $g_{\rho\sigma\gamma}$ for a set of $\sigma$-meson parameter values.

## Key numbers

- Experimental branching ratio used: $0.0099 \pm 0.0016$ for $\rho^{0}\to \pi^{+}+\pi^{-}+\gamma$ (photon energies $E_{\gamma} \ge 50$ MeV)
- Maximum photon energy: $E_{\gamma,\text{max}} = 334$ MeV
- $\rho$-meson width: $\Gamma = 151$ MeV, yielding $g_{\rho\pi\pi}^{2}/4\pi = 2.91$
- For $M_{\sigma}=500$ MeV, $\Gamma_{\sigma}=800$ MeV: $g_{\rho\sigma\gamma} = 8.45 \pm 1.77$ (positive solution) and $-6.96 \pm 1.78$ (negative solution)
- For $M_{\sigma}=600$ MeV, $\Gamma_{\sigma}=600$ MeV: $g_{\rho\sigma\gamma} = 9.83 \pm 1.85$ (positive) and $-6.68 \pm 1.85$ (negative)
- For $M_{\sigma}=800$ MeV, $\Gamma_{\sigma}=600$ MeV: $g_{\rho\sigma\gamma} = 15.05 \pm 2.64$ (positive) and $-9.11 \pm 2.64$ (negative)
- For $M_{\sigma}=900$ MeV, $\Gamma_{\sigma}=900$ MeV: $g_{\rho\sigma\gamma} = 17.78 \pm 3.23$ (positive) and $-11.35 \pm 3.23$ (negative)
- Using Serot and Walecka parameters ($M_{\sigma}=520$ MeV, $g_{\sigma\pi\pi}=12$, implying $\Gamma_{\sigma}\simeq 1700$ MeV): $g_{\rho\sigma\gamma}=11.91$
- Using Soyeur parameters ($M_{\sigma}=500$ MeV, $\Gamma_{\sigma}=250$ MeV): $g_{\rho\sigma\gamma}=6.08$

## Context

This work extends the approach of Renard (1969), who first identified the $\sigma$-meson intermediate state as the dominant contribution to structural radiation in $\rho^{0}\to \pi^{+}\pi^{-}\gamma$ and estimated $g_{\rho\sigma\gamma}\simeq 1$ using quark-model spin independence. The authors contrast their results with the value $g_{\rho\sigma\gamma}\simeq 2.71$ obtained by Friman and Soyeur (1996) from Vector Meson Dominance in $\rho^{0}$ photoproduction, and with the value used by Titov et al. (1999) in $\phi$-meson photoproduction studies. The analysis enables future studies of the dependence of vector meson photoproduction observables on $\sigma$-meson parameters when using one-meson exchange mechanisms.
