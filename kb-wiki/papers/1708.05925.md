# Neutron-proton pairing and double-beta decay in the interacting boson model

> Background: The interacting boson model (IBM) has been used extensively to calculate the matrix elements governing neutrinoless double-beta decay. Studies within other models indicate that a good description of neutron-proton pairing is essential for accurate calculations of those matrix elements. The usual interacting boson model is based only on like-particle pairs, however, and the extent to which it captures neutron-proton pairing is not clear. Purpose: To determine whether neutron-proton pairing should be explicitly included as neutron-proton bosons in IBM calculations of neutrinoless double-beta decay matrix elements. Method: An isospin-invariant version of the nucleon-pair shell model is applied to carry out shell-model calculations in a large space and in a collective subspace, and to define effective operators in the latter. A democratic mapping is then used to define corresponding boson operators for the IBM, with and without an isoscalar neutron-proton pair boson. Results: IBM calculations with and without the isoscalar boson are carried out for nuclei near the beginning of the $pf$ shell, with a realistic shell-model Hamiltonian and neutrinoless double-beta-decay operator as the starting point. Energy spectra and double-beta matrix elements are compared to those obtained in the underlying shell model. Conclusions: The isoscalar boson does not improve energy spectra but does improve double-beta matrix elements. To be useful at the level of precision we need, the mapping procedure must be further developed to better determine the dependence of the boson Hamiltonian and decay operator on particle number and isospin. But the benefits provided by the isoscalar boson suggest that through an appropriate combination of mappings and fitting, it would make IBM matrix elements more accurate for the heavier nuclei used in experiments.

## Key claim

The isoscalar (neutron-proton) boson does not improve energy spectra but does improve the calculated neutrinoless double-beta decay matrix elements; however, the mapping procedure must be further developed to determine the dependence of the boson Hamiltonian and decay operator on particle number and isospin before the improvement becomes useful at the required precision level.

## Method

An isospin-invariant version of the nucleon-pair shell model (NPSM) was applied to carry out shell-model calculations in the full $pf$-shell space and in a collective subspace, defining effective operators in the latter via the Suzuki-Lee method. A democratic mapping was then used to define corresponding boson operators for the interacting boson model (IBM), both without and with an isoscalar neutron-proton pair boson (the $p$-boson). The boson Hamiltonians and $0\nu\beta\beta$-decay operators were mapped order-by-order from the KB3G fermion interaction and compared to exact shell-model results for nuclei near the beginning of the $pf$ shell.

- Two-boson interaction matrix elements (in MeV) for the $J=0$, $T=0$ channel: $\braket{s^2|\hat V|s^2} = -4.440$ MeV (IBMb, mapped from KB3G) and $-6.379$ MeV (IBMe, mapped from effective fermion interaction)
- Two-boson interaction matrix element for $J=0$, $T=0$: $\braket{d^2|\hat V|d^2} = -4.258$ MeV (IBMb) and $-5.013$ MeV (IBMe)
- Two-boson interaction matrix element for $J=0$, $T=0$: $\braket{s^2|\hat V|d^2} = -3.402$ MeV (IBMb) and $-3.489$ MeV (IBMe)
- Two-boson interaction matrix element for $J=2$, $T=0$: $\braket{sd|\hat V|sd} = -6.350$ MeV (IBMb) and $-8.263$ MeV (IBMe)
- Two-boson interaction matrix element for $J=1$, $T=1$: $\braket{d^2|\hat V|d^2} = -3.029$ MeV (IBMb) and $-4.861$ MeV (IBMe)
- Two-boson interaction matrix element for $J=2$, $T=2$: $\braket{sd|\hat V|sd} = +0.896$ MeV (IBMb) and $+0.473$ MeV (IBMe)

## Context

This work extends the IBM calculations of $0\nu\beta\beta$ decay matrix elements by Baranger et al. (2009, 2013, 2015) by investigating whether an isoscalar neutron-proton boson should be explicitly included. It builds on shell-model results from Menendez et al. (2016) showing that isoscalar pairing in the Hamiltonian reduces $0\nu\beta\beta$ matrix elements, and on generator-coordinate-method calculations by Hinohara et al. (2014) showing that including isoscalar-pairing amplitude as a generator coordinate reduces matrix elements significantly. The results suggest that adding an isoscalar boson could make IBM matrix elements more accurate for the heavier nuclei used in experiments, once the mapping procedure is further developed.
