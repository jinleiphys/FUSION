# Degeneracies when only T=1 two-body interactions are present

> In the nuclear f_7/2 shell, the nucleon-nucleon interaction can be represented by the eight values E(J)=<(f^2_{7/2})^J |V| (f^2_{7/2})^J>, J=0,1,...,7, where for even J the isospin is 1, and for odd J it is 0. If we set the T=0 (odd J) two-body matrix elements to 0 (or to a constant), we find several degeneracies which we attempt to explain in this work. We also give more detailed expressions than previously for the energies of the states in question. New methods are used to explain degeneracies that are found in {45}Ti (I=25/2- and 27/2-), {46}V (I=12^+_1 and 13^+_1, as well as I=13^+_2 and 15+), and {47}V (I=29/2- and 31/2-).

# Degeneracies when only T=1 two-body interactions are present

## Key claim

When the T=0 (odd J) two-body matrix elements in the nuclear f7/2 shell are set to zero or to a constant, several degeneracies appear in the spectra of 45Ti, 46V, and 47V that require a different explanation than the partial dynamical symmetry previously invoked for lighter nuclei. These degeneracies arise because, in the absence of T=0 interactions, the proton-neutron interaction expectation value becomes exactly half the sum of the proton-proton and neutron-neutron expectation values for certain specific configurations.

## Method

The authors performed shell-model calculations in a single-j (f7/2) configuration space using two interactions: V(42Sc), which uses experimental excitation energies of the lowest state of each angular momentum J in 42Sc, and T0V(42Sc), which retains the same even-J (T=1) matrix elements but sets all odd-J (T=0) matrix elements to zero. They derived analytic expressions for the energies of selected states in 43Sc, 44Ti, 45Ti, 46V, and 47V in terms of the eight two-body matrix elements E(J), and used m-scheme wave functions and recoupling techniques to demonstrate the degeneracies when T=0 matrix elements vanish.

## Key numbers

- In 43Sc, the energy difference E(17/2-) - E(19/2-) = 0.73077 [E(5) - E(7)]; using V(42Sc) this equals 0.65316 MeV, and vanishes when T=0 matrix elements are set equal to a constant.
- In 45Ti, the energy difference E(25/2-) - E(27/2-) = 0.545454 E(3) + 0.115384 E(5) - 0.660838 E(7); using V(42Sc) this equals 0.57992 MeV.
- In 47V, the energy difference E(29/2-) - E(31/2-) = 0.369048 E(1) + 0.234848 E(3) + 0.028388 E(5) - 0.632285 E(7); using V(42Sc) this equals 0.22873 MeV.
- The experimental E(J) values for 42Sc used in V(42Sc) are (in MeV): J=0: 0.0, J=1: 0.6111, J=2: 1.5863, J=3: 1.4904, J=4: 2.8153, J=5: 1.5101, J=6: 3.2420, J=7: 0.6163.
- In 46V with T0V(42Sc), the lowest I=12 (T=0) state is degenerate with the lowest I=13 (T=0) state at 7.26601 MeV; the second I=13 state is degenerate with the unique I=15 state at 9.27745 MeV.

## Context

This work extends previous studies by Robinson and Zamick (2001, 2002) that found degeneracies in 43Sc and 44Ti when T=0 matrix elements were set to zero, where the degenerate states had angular momenta that could not occur for systems of identical particles. The present paper addresses additional degeneracies in 45Ti, 46V, and 47V that require a different explanation, using m-scheme techniques and properties of 6j symbols. The results enable further study of T=0 two-body matrix elements, which are more elusive than T=1 matrix elements because identical-particle systems (like tin isotopes with only valence neutrons) only involve T=1 interactions. The authors note that similar degeneracies should appear in other shells, such as the g9/2 shell, where for a system of 3 protons and 4 neutrons the states with J=43/2- and 45/2- become degenerate with T0V(42Sc).
