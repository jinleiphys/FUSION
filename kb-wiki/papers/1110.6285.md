# Improved Maximum Entropy Analysis with an Extended Search Space

> The standard implementation of the Maximum Entropy Method (MEM) follows Bryan and deploys a Singular Value Decomposition (SVD) to limit the dimensionality of the underlying solution space apriori. Here we present arguments based on the shape of the SVD basis functions and numerical evidence from a mock data analysis, which show that the correct Bayesian solution is not in general recovered with this approach. As a remedy we propose to extend the search basis systematically, which will eventually recover the full solution space and the correct solution. In order to adequately approach problems where an exponentially damped kernel is used, we provide an open-source implementation, using the C/C++ language that utilizes high precision arithmetic adjustable at run-time. The LBFGS algorithm is included in the code in order to attack problems without the need to resort to a particular search space restriction.

## Key claim

The standard Maximum Entropy Method (MEM) implementation following Bryan, which uses a Singular Value Decomposition (SVD) to restrict the search space to \(N_\tau\) dimensions, does not in general recover the correct Bayesian solution; the correct global extremum of the functional lies outside this restricted subspace, and the reconstruction can always be made to fail by choosing a sufficiently negative lower frequency cutoff \(\omega_{\rm min}\).

## Method

The author demonstrates the inadequacy of Bryan's SVD-based search space by analyzing the functional form of the SVD basis functions for a Laplace transform kernel \(K(\tau,\omega)=e^{-\omega\tau}\) and by performing a mock data analysis with a known four-Gaussian-peak spectrum. The proposed remedy is to systematically extend the search space from \(N_\tau\) to \(N_{\rm ext}\) basis vectors (with \(N_\tau < N_{\rm ext} < N_\omega\)), using the columns of the SVD matrix \(U\), and to increase \(N_{\rm ext}\) until the minimal value of the functional \(\mathcal{Q}(\rho,D,m)\) stops decreasing. The implementation uses high-precision arithmetic (adjustable at run-time) and the LBFGS algorithm to avoid reliance on a particular search space restriction.

## Key numbers

- The mock spectrum consists of four Gaussian peaks with amplitudes \(3\times10^{-8}\), 0.6, 0.25, and 0.2; positions at -2.3, 0.52, 2.6, and 7.5; and widths 0.1, 0.1, 0.4, and 1.4 (dimensionless units).
- The mock data uses \(N_\tau=12\) data points in \(\tau\in[0,6.1]\), with noise parameter \(\eta=0.0001\).
- The MEM reconstruction uses \(N_\omega=1500\) frequency points over \(\omega\in[-10,20]\) with \(\Delta\omega=0.02\), and internal arithmetic set to 384 bits of precision.
- With Bryan's prescription (\(N_{\rm ext}=N_\tau=12\)), the residual \(\mathcal{Q}\simeq 10000\), dominated by a likelihood term \(\mathcal{L}\) of the same order; the correct solution requires \(\mathcal{L}\sim\mathcal{O}(1-10)\).
- Extending the search space to \(N_{\rm ext}=28\) and \(50\) reduces the discrepancy in reproducing the data at \(\tau\simeq5\) and brings \(\mathcal{Q}\) into the \(\mathcal{O}(1-10)\) range.
- The evaluation time of \(\mathcal{Q}\) grows linearly with \(N_{\rm ext}\); the overall running time increases monotonically for \(N_{\rm ext}>40\).

## Context

This work extends the standard MEM implementation of Bryan (1990) by showing that its SVD-based search space restriction is insufficient for problems involving an exponentially damped kernel, such as those encountered in lattice QCD spectral reconstruction (Rothkopf 2011). The paper provides an open-source C/C++ implementation with adjustable high-precision arithmetic and the LBFGS algorithm, enabling future applications to deconvolution problems where the correct Bayesian solution lies outside the SVD subspace. The approach enables systematic identification of which spectral features are constrained by data versus prior information through variation of the prior function \(m(\omega)\).
