# Parton distribution functions on the lattice and in the continuum

> Ioffe-time distributions, which are functions of the Ioffe-time $ν$, are the Fourier transforms of parton distribution functions with respect to the momentum fraction variable $x$. These distributions can be obtained from suitable equal time, quark bilinear hadronic matrix elements which can be calculated from first principles in lattice QCD, as it has been recently argued. In this talk I present the first numerical calculation of the Ioffe-time distributions of the nucleon in the quenched approximation.

## Key claim

The paper presents the first numerical calculation of Ioffe-time distributions of the nucleon from lattice QCD, using a ratio of matrix elements that cancels ultraviolet divergences and reduces finite-spacing corrections, but the results are obtained only in the quenched approximation with unphysically heavy pion and nucleon masses.

## Method

The authors compute equal-time, quark-bilinear hadronic matrix elements with a straight-line gauge link in the quenched approximation on a \(32^3 \times 64\) lattice at \(\beta=6.0\) using the non-perturbatively tuned clover fermion action. They extract the reduced Ioffe-time distribution \(\mathfrak{M}(\nu, z_3^2)\) as a double ratio of effective matrix elements at large Euclidean time separation, which removes the need for renormalization and has a well-defined continuum limit. The data are then evolved using leading-order DGLAP evolution to a common scale to test consistency with perturbative evolution.

## Key numbers

* Pion mass: \(601(1)\) MeV; nucleon mass: \(1411(4)\) MeV, from the quenched lattice calculation at \(\beta=6.0\) with lattice spacing \(a=0.093\) fm.
* Maximum nucleon momentum used: \(2.5\) GeV.
* Valence PDF fit parameters (statistical errors only) from the evolved Ioffe-time data: \(a = 0.36(6)\) and \(b = 3.95(22)\) for the form \(q_v(x) = N(a,b) x^a (1-x)^b\).
* Evolution applied using \(\alpha_s/\pi = 0.1\) to evolve data from various \(z_3\) values to a reference scale \(\mu_0 = 1\) GeV (corresponding to \(z_0 = 2a\)).

## Context

This work extends the quasi-PDF approach introduced by Ji (2013) and the Ioffe-time distribution formalism advocated by Radyushkin (2017) by providing the first lattice test of the ratio method that reduces sensitivity to higher-twist corrections. The results enable future realistic dynamical fermion calculations with physical quark masses and higher-order perturbative matching, as the authors note they are currently applying the method to such calculations. The approach contrasts with earlier quasi-PDF lattice calculations by Lin (2014), Alexandrou (2015), and others, which required large hadron momenta and explicit renormalization.
