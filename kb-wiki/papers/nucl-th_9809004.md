# QCD sum rule calculation of the $πNN$ coupling-revisited

> In the conventional QCD sum rule calculation for the $πNN$ coupling beyond the soft-pion limit, there are three distinct Dirac structures, each of which can be used to construct separate sum rule. We point out subtleties in the previous sum rule results, based on one of the Dirac structure, associated with using either the PV or the PS coupling schemes in modeling the phenomenological side. We propose a sum rule coming from a different Dirac structure, which is independent of the coupling schemes used and has less uncertainty in the OPE. The obtained value is $g_{πN}=9.76 \pm 2.04$ where the uncertainty mainly comes from the quark-gluon mixed condensate.

## Key claim

The authors propose a new QCD sum rule for the pion-nucleon coupling constant \(g_{\pi N}\) using the \(\gamma_5 \sigma^{\mu\nu}\) Dirac structure, which is independent of the pseudoscalar versus pseudovector coupling scheme ambiguity and has less uncertainty from the operator product expansion than previous approaches, yielding \(g_{\pi N}=9.76 \pm 2.04\) where the uncertainty mainly comes from the quark-gluon mixed condensate.

## Method

The authors compute the two-point correlation function \(\Pi(q,p) = i \int d^4x e^{iq\cdot x} \langle 0 | T[J_p(x) \bar{J}_n(0)] | \pi^+(p) \rangle\) using the Ioffe nucleon interpolating field, evaluating the operator product expansion up to dimension 7. They match this to the phenomenological side constructed from the \(\gamma_5 \sigma^{\mu\nu} p_\mu q_\nu\) Dirac structure, which contains only a double pole and is free of the coupling-scheme dependence that affects the \(\gamma_5\) and \(\gamma_5 \not{p}\) structures used in prior work. The sum rule is obtained by taking the ratio with the chiral-odd nucleon sum rule to eliminate the nucleon coupling \(\lambda_N\), and the unknown single-pole parameter \(B\) is determined by fitting the Borel-transformed expression to a linear function \(a + b M^2\) within a chosen Borel window.

## Key numbers

* \(g_{\pi N} = 9.76 \pm 2.04\) from the new sum rule, with the uncertainty dominated by the quark-gluon mixed condensate parameter \(m_0^2\).
* For \(m_0^2 = 0.6\) GeV\(^2\): \(g_{\pi N} = 11.8\); for \(m_0^2 = 1.0\) GeV\(^2\): \(g_{\pi N} = 9.14\); for \(m_0^2 = 1.4\) GeV\(^2\): \(g_{\pi N} = 7.72\) (Table 1).
* The empirical value of \(g_{\pi N}\) is around 13.4.
* The Borel window used: \(M^2_{max} \sim 1.24\) GeV\(^2\) (from requiring continuum contribution below 50% of the first term in the nucleon sum rule); \(M^2_{min}\) ranges from 0.55 to 1.05 GeV\(^2\) depending on \(m_0^2\) (from requiring highest power correction below 10% of total OPE).
* The nucleon mass \(m = 0.94\) GeV and pion decay constant \(f_\pi = 0.093\) GeV are used to extract \(g_{\pi N} = m a / f_\pi\).
* The gluon condensate is taken as \(\langle \frac{\alpha_s}{\pi} \mathcal{G}^2 \rangle = (0.33\ \text{GeV})^4\).

## Context

This work extends the QCD sum rule calculation of the \(\pi NN\) coupling beyond the soft-pion limit, following the approach of Birse and Krippa (1996) who first considered the sum rule beyond that limit but obtained \(g_{\pi N} = 12 \pm 5\) with large uncertainties from poorly known condensates and the twist-4 term \(\delta^2\). The authors point out that the earlier work by Shiomi and Hatsuda (1995) using the \(\gamma_5\) structure in the soft-pion limit is scheme-dependent because the pseudovector coupling gives zero in that limit. The new sum rule using the \(\gamma_5 \sigma^{\mu\nu}\) structure avoids these issues and enables a determination with reduced uncertainty, though the result is smaller than the empirical value of 13.4.
