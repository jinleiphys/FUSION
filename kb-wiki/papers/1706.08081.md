# Predicting the optical observables for nucleon scattering on even-even actinides

> Previously derived Lane consistent dispersive coupled-channel optical model for nucleon scattering on $^{232}$Th and $^{238}$U nuclei is extended to describe scattering on even-even actinides with $Z=$90--98. A soft-rotator-model (SRM) description of the low-lying nuclear structure is used, where SRM Hamiltonian parameters are adjusted to the observed collective levels of the target nucleus. SRM nuclear wave functions (mixed in $K$ quantum number) have been used to calculate coupling matrix elements of the generalized optical model. The "effective" deformations that define inter-band couplings are derived from SRM Hamiltonian parameters. Conservation of nuclear volume is enforced by introducing a dynamic monopolar term to the deformed potential leading to additional couplings between rotational bands. Fitted static deformation parameters are in very good agreement with those derived by Wang and collaborators using the Weizsäcker-Skyrme global mass model (WS4), allowing to use the latter to predict cross section for nuclei without experimental data. A good description of scarce "optical" experimental database is achieved. SRM couplings and volume conservation allow a precise calculation of the compound-nucleus formation cross sections, which is significantly different from the one calculated with rigid-rotor potentials coupling the ground-state rotational band. Derived parameters can be used to describe both neutron and proton induced reactions.

## Key claim

The previously developed Lane-consistent dispersive coupled-channel optical model, which was validated for major actinides like ²³²Th and ²³⁸U, can be extended to predict nucleon scattering observables for even-even actinides with Z=90–98 without additional optical model parameter fitting, provided that the soft-rotator-model Hamiltonian parameters are adjusted to the observed collective levels of each target nucleus and that static deformation parameters are taken either from fits to experimental data or from global mass models (preferably WS4 rather than FRDM2012).

## Method

The authors used a dispersive coupled-channel optical model implemented in the OPTMAN code, with coupling matrix elements calculated from soft-rotator-model (SRM) nuclear wave functions that are mixed in the K quantum number. SRM Hamiltonian parameters were fitted to reproduce the low-lying collective level schemes of even-even actinides (Z=90–98) that have sufficient level data. Static deformation parameters β₂, β₄, and β₆ were either fitted to available experimental cross-section data (for ²³²Th, ²³⁸U, ²⁴⁰Pu, ²⁴²Pu) or taken from the FRDM2012 and WS4 global mass models. Nuclear volume conservation was enforced by introducing a dynamic monopolar term to the deformed potential, leading to additional couplings between rotational bands.

## Key numbers

- SRM Hamiltonian parameter ℏω₀ ranges from 0.702 MeV (²³²Th) to 1.270 MeV (²⁴⁶Cm) across the fitted even-even actinides.
- Best-fit static deformation parameters for ²³²Th: β₂=0.201, β₄=0.067, β₆=-0.007; for ²³⁸U: β₂=0.221, β₄=0.056, β₆=-0.001; for ²⁴⁰Pu: β₂=0.212, β₄=0.085, β₆=-0.027; for ²⁴²Pu: β₂=0.213, β₄=0.040, β₆=-0.016.
- Calculated χ² values for ²³²Th: best fit = 2.65, FRDM2012 = 11.8, WS4 = 3.45; for ²³⁸U: best fit = 1.87, FRDM2012 = 14.6, WS4 = 2.30; for ²⁴⁰Pu: best fit = 0.56, FRDM2012 = 1.26, WS4 = 0.81; for ²⁴²Pu: best fit = 0.60, FRDM2012 = 1.26, WS4 = 0.69.
- Evaluated s-wave neutron strength function S₀ for ²³²Th: 0.94±0.07 (Minsk evaluation); best-fit calculation gives S₀=0.93, FRDM2012 gives 1.14, WS4 gives 0.80.
- For ²³⁸U, the compound-nucleus formation cross section calculated with WS4 deformations agrees perfectly with the best-fit reference calculation above 100 keV, while FRDM2012 deformations give higher values below 100 keV.

## Context

This work extends the dispersive coupled-channel optical model previously developed for major actinides (²³²Th, ²³⁸U, ²³³U, ²³⁵U, ²³⁹Pu) by Soukhovitskii, Capote, and collaborators (2016) to a broader set of even-even actinides. It improves upon rigid-rotor models (e.g., Soukhovitskii 2004, Capote 2008) by using soft-rotator-model couplings that account for vibrational bands and volume conservation, which were shown to be important for accurate compound-nucleus formation cross sections. The approach enables prediction of optical observables for minor actinides with scarce or absent experimental data, using either FRDM2012 (Moller 2012) or WS4 (Wang 2014) global mass model deformations, with WS4 deformations recommended for energies below 100 keV. A similar extension to odd actinides is identified as the next step.
