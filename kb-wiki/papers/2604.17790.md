# A microscopic analysis of sub-barrier photo-induced fission in $^{236}$U$(γ,f)$ based on the non-equilibrium Green function method

> Sub-barrier photo-induced fission in $^{236}$U$(γ,f)$ is investigated within the non-equilibrium Green function (NEGF) method. A model space for the fission process is constructed by superposing Skyrme-Hartree-Fock wave functions along the fission path allowing the particle-hole excitation. Then, the transition from the photo-absorption channel to the fission channel is described by the non-equilibrium Green-function formalism. The calculated fission cross section in the incident gamma-ray energy range $5 ~ {\rm MeV} \leq E_γ\leq 6 ~ {\rm MeV}$ reproduces the overall behavior of the experimental data, including the suppression below the fission barrier. An eigenchannel analysis of the wave propagation in the present fission model space is also performed, and the first eigenchannel is found to dominate the fission probability. This result supports the Bohr-Wheeler transition-state picture from a microscopic viewpoint.

# A microscopic analysis of sub-barrier photo-induced fission in $^{236}$U$(γ,f)$ based on the non-equilibrium Green function method

## Key claim
The non-equilibrium Green function (NEGF) method reproduces the overall behavior of the sub-barrier photo-induced fission cross section in $^{236}$U$(γ,f)$, including the suppression below the fission barrier, but the agreement is qualified: the cross section is somewhat overestimated in the range 5.3 MeV ≤ Eγ ≤ 5.7 MeV, while agreement within a factor of 2 is achieved only in the deep sub-barrier region 5.0 MeV ≤ Eγ ≤ 5.2 MeV.

## Method
The author constructs a model space by superposing Skyrme-Hartree-Fock wave functions along the fission path in the Q20-Q30 deformation space, including particle-hole excited configurations with K=0 up to a cutoff Emax=7.0 MeV, yielding 145124 basis states. The transition from the photo-absorption channel to the fission channel is described using the non-equilibrium Green function formalism, with the E1 photon transmission coefficients determined from empirical gamma strength function parameters (RIPL-3) and the Gilbert-Cameron level density formula. The fission cross section is computed via the trace formula T_{ab} = Tr[Γ_a G(E) Γ_b G†(E)] using the shift-invert Arnoldi method for the Green function, with energy averaging over ΔE=1 keV and ensemble averaging over 96 random seeds for the residual random interaction.

## Key numbers
- Fission barrier height: 5.7 MeV (experimental value adopted, consistent with Uzawa (2025))
- Energy range studied: 5 MeV ≤ Eγ ≤ 6 MeV
- Number of basis states: Ndim = 145124
- Cutoff parameter: Emax = 7.0 MeV
- Overlap between neighboring deformation points: ⟨Qi|Qi+1⟩ = 0.52
- Number of reference states along fission path: 19
- Deformation range: Qmin = (Q20=14 b, Q30=0) to Qmax = (Q20=84 b, Q30=19 b^{3/2})
- Pairing strength: G = 0.1465 MeV
- Diabatic interaction parameter: h2 = 1.5 MeV
- Random residual interaction strengths: v = 0.025 MeV (proton-neutron), v = 0.02 MeV (proton-proton/neutron-neutron)
- Fission width: ⟨Γfis⟩ = 100 keV
- Energy averaging width: ΔE = 1 keV
- Energy discretization mesh: dE' = 10^{-8} MeV
- Number of photo-absorption/emission channels: 100 configurations at Qmin
- GDR parameters for 236U: E1=10.93 MeV, σ1=278.4 mb, Γ1=2.58 MeV; E2=13.80 MeV, σ2=410.3 mb, Γ2=4.78 MeV
- Above barrier (Eγ ≥ 5.7 MeV): NEGF results agree with experimental values within a factor of 5
- Deep sub-barrier (5.0 MeV ≤ Eγ ≤ 5.2 MeV): calculated cross section agrees with experimental data within a factor of 2
- Photo-absorption cross section from NEGF agrees with empirical value within a factor of 1.2

## Context
This work extends the NEGF-based theoretical framework developed by Uzawa (2025) for the 235U(n,f) reaction to the 236U(γ,f) reaction, where sub-barrier energies can be accessed. The approach contrasts with time-dependent density functional theory (TDDFT), which cannot describe quantum tunneling through the fission barrier, and with the time-dependent generator coordinate method (TDGCM), which has arbitrariness in preparing initial states for induced fission. The eigenchannel analysis shows that the first eigenchannel dominates the fission probability, supporting the Bohr-Wheeler transition-state picture (Bohr, 1939) from a microscopic viewpoint. This work enables future applications of the NEGF method to other photo-induced fission reactions and provides a framework for analyzing entrance-channel effects in sub-barrier fission.
