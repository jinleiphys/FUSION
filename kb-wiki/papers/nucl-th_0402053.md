# Anisotropic flow from Lee-Yang zeroes: a practical guide

## Key claim

The paper presents a practical implementation guide for a new method to extract anisotropic flow from correlations among a large number of particles, rather than the few-particle correlations used in prior methods. The method is based on finding the first minimum of the modulus of an event-averaged generating function, which corresponds to a Lee-Yang zero, and the authors claim this approach yields smaller systematic errors than any other flow analysis method.

## Method

For each event, the complex-valued generating function g^θ(ir) = ∏ [1 + ir w_j cos(n(ϕ_j − θ))] is computed for several values of r and θ, along with the sums Q_x and Q_y. These are averaged over events to obtain G^θ(ir), and the position r_0^θ of the first positive minimum of |G^θ(ir)| is found; the integrated flow V_n is then estimated as V_n^θ{∞} = j_01 / r_0^θ, where j_01 ≈ 2.40483 is the first zero of the Bessel function J_0. For differential flow of a specific particle type, the estimate v'_mn^θ{∞} is obtained from a ratio involving the generating function at r_0^θ and the proton azimuthal angles, with the resolution parameter χ ≡ V_n{∞}/σ computed from the Q-vector averages.

## Key numbers

- j_01 ≈ 2.40483: the first zero of the Bessel function J_0, used to convert the minimum position r_0^θ into the integrated flow estimate V_n^θ{∞}
- The relative statistical error on V_n^θ{∞} is given by (1/(2 N_ev j_01^2 J_1(j_01)^2)) [exp(j_01^2/(2χ^2)) + exp(−j_01^2/(2χ^2)) J_0(2j_01)], where N_ev is the number of events
- The relative systematic error on v'_mn{∞} is of order O(1/M) + O(1/(M v_n)^2) + O(v_{2n}/(M v_n^2)), where M is the multiplicity per event
- For higher harmonics (m > 1), an additional absolute systematic error of order O(v'_{(m-1)n} / (M v_n)) arises

## Context

This work extends the authors' earlier theoretical proposals (Bhalerao 2003) by providing practical recipes for implementation, and contrasts with prior methods based on two-particle correlations (Danielewicz, Ollitrault 1997, Poskanzer 1998) or cumulants of up to 8 particles (Borghini 2001). The method enables flow analysis using correlations among all detected particles simultaneously, with the conceptual foundation that the zero of the generating function near the origin scaling inversely with system size signals collective behavior, analogous to Lee-Yang theory of phase transitions (Lee and Yang 1952). The authors note that for an anisotropic detector, V_n^θ{∞} shows an oscillatory pattern computable from the detector acceptance Fourier coefficients, with an extra proportionality factor close to unity.
