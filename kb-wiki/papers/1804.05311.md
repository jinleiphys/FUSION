# In depth analysis of the combined HERA data in the dipole models with and without saturation

> We present an updated impact parameter dependent saturation model (IPsat) determined trough a fit to the combined HERA I and I+II reduced cross section data. The same HERA data are used to fit the linearized (IPnonsat) version of the applied dipole amplitude, which makes it possible to estimate the magnitude of the saturation effects in various experiments. We find that both parametrizations provide comparable descriptions of the considered data when an effective confinement scale dynamics is incorporated with quark masses. Moreover, it is possible to consistently determine the light and charm quark masses. The role of potentially non-perturbatively large dipoles is examined in detail, with the result that, especially in case of the structure function $F_2$, their contribution is numerically significant. Potential to discriminate between the two models in future $e+p$ and $e+A$ experiments is also illustrated.

## Key claim

Both the saturation (IPsat) and linearized (IPnonsat) dipole models provide comparably good descriptions of the combined HERA reduced cross section data when an effective confinement scale is incorporated via nonzero light quark masses, meaning that the inclusive HERA data alone cannot clearly distinguish saturation effects from confinement-scale dynamics.

## Method

The authors performed simultaneous fits of the impact parameter dependent saturation model (IPsat) and its linearized version (IPnonsat) to the combined HERA I and HERA I+II reduced cross section data, including both inclusive and charm contributions. The dipole cross section was expressed in terms of a DGLAP-evolved gluon distribution with a variable flavor number scheme, and the light and charm quark masses were treated as free parameters constrained by the fit, with data restricted to x < 0.01 and Q² between 1.5 and 50 GeV² for the main result.

## Key numbers

- IPsat fit to HERA I data (Q²_max = 50 GeV²): χ²/N = 1.0978 for 156+33 data points; charm quark mass m_c = 1.3528 GeV; light quark mass fixed to m_l = 0.03 GeV; gluon parameters A_g = 2.1953, λ_g = 0.08289; scale parameter C = 2.2894 GeV²
- IPnonsat fit to HERA I data (Q²_max = 50 GeV²): χ²/N = 1.122 for 156+33 data points; charm quark mass m_c = 1.3504 GeV; light quark mass m_l = 0.1516 GeV; gluon parameters A_g = 3.0391, λ_g = -0.006657; scale parameter C = 4.2974 GeV²
- IPsat fit to HERA I+II data (Q²_max = 50 GeV²): χ²/N = 1.2781 for 410+33 data points; charm quark mass m_c = 1.3210 GeV; light quark mass fixed to 0.03 GeV; gluon parameters A_g = 2.0670, λ_g = 0.09575; scale parameter C = 1.8178 GeV²
- IPnonsat fit to HERA I+II data (Q²_max = 50 GeV²): χ²/N = 1.3023 for 410+33 data points; charm quark mass m_c = 1.3180 GeV; light quark mass m_l = 0.1497 GeV; gluon parameters A_g = 2.8460, λ_g = 0.008336; scale parameter C = 3.5445 GeV²
- For F₂ at Q² = 500 GeV², 10% of the total structure function originates from dipoles larger than 1 fm
- Saturation scale at central impact parameter reaches perturbative values (Q_s² > 1 GeV²) only below x ≈ 10⁻⁴ to 10⁻⁵

## Context

This work extends previous IPsat fits by Kowalski (2003, 2006) and Rezaeian (2012) by simultaneously fitting the linearized IPnonsat model to the same combined HERA data, enabling a direct estimate of saturation effects. The authors find that both models describe the data equally well, consistent with the observation by Ball (2017) that low-x HERA data can be explained by resummed logarithmic corrections rather than saturation. The results enable predictions for future facilities such as the EIC and LHeC, where measurements of F_L and charm structure functions could discriminate between the saturation and linearized models.
