# Weakly bound systems, continuum effects, and reactions

> Structure of weakly bound/unbound nuclei close to particle drip lines is different from that around the valley of beta stability. A comprehensive description of these systems goes beyond standard Shell Model and demands an open quantum system description of the nuclear many-body system. We approach this problem using the Gamow Shell Model which provides a fully microscopic description of bound and unbound nuclear states, nuclear decays, and reactions. We present in this paper the first application of the GSM for a description of the elastic and inelastic scattering of protons on 6He.

# Weakly bound systems, continuum effects, and reactions

## Key claim

The Gamow Shell Model coupled-channel (GSM-CC) formalism provides a unified description of nuclear structure and reactions for weakly bound/unbound nuclei, but the approximation of including only a limited number of target states (here, only the ground state and first excited state of ⁶He) may be insufficient for accurate reaction cross sections, requiring strong renormalization of the effective interaction to compensate for missing continuum configurations.

## Method

The authors formulate coupled-channel equations in coordinate space using the Gamow Shell Model, where the many-body basis is built from the Berggren ensemble of single-particle states (bound, resonant, and non-resonant continuum states). The Hamiltonian is decomposed into a basis part and a residual finite-range part; the resulting non-local differential equations are solved numerically using a modified equivalent potential method with iterative procedures. The method is applied to proton scattering on a ⁶He target, using a Woods-Saxon core potential and modified surface Gaussian interaction (MSGI) for the residual two-body interaction, with the target described by its 0₁⁺ ground state and 2₁⁺ resonance.

## Key numbers

- The core potential parameters: radius R = 1.993 fm (protons) and 1.954 fm (neutrons); central depth V₀ = 50.188 MeV (protons) and 49.268 MeV (neutrons); diffuseness d = 0.497 fm (protons) and 0.657 fm (neutrons); spin-orbit strength V_so = 7.488 MeV (protons) and 8.23 MeV (neutrons); Coulomb radius R_C = 1.952 fm.
- The MSGI two-body interaction coupling strengths used in numerical tests: V(J=0,T=1) = -20 MeV·fm³ and V(J=2,T=1) = -15 MeV·fm³.
- The GSM-CC calculation for ⁷Li low-lying states yields energies: 3/2⁻ at -10.946 MeV, 1/2⁻ at -10.469 MeV, 7/2⁻ at -6.307 MeV, and 5/2⁻ at -4.509 MeV; the corresponding full GSM diagonalization gives -17.83, -21.18, -21.01, and -31.26 MeV respectively, showing dramatic differences due to missing target continuum states.
- Angular cross sections for ⁶He+p reaction are calculated at center-of-mass energy E_CM = 35.1 MeV and compared qualitatively with experimental data from Lag01.
- The scattering wave function converges with 17 continuum states on the discretized p₃/₂ contour.

## Context

This work extends the Gamow Shell Model (GSM), previously used primarily for nuclear structure (reviewed in Mic09), to reaction problems via a coupled-channel formulation. It builds on the real-energy continuum shell model (CSM) approaches of Mah69, Bar77, Phi77, Ben99, Vol06, and Rot06, which treat up to two nucleons in the scattering continuum, and on the Berggren completeness relation (Berggren). The GSM-CC formalism enables future application in the No-Core Gamow Shell Model (papa12) with bare nucleon-nucleon interactions. The results show that neglecting high-lying continuum states of the target, as done in standard coupled-channel approaches, leads to large discrepancies, consistent with findings in the Shell Model Embedded in the Continuum (Oko08). The main open question is how to include a sufficient number of target continuum states to achieve convergent microscopic descriptions of reactions with weakly bound exotic nuclei.
