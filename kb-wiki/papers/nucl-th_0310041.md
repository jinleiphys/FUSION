# Electroproduction of nucleon resonances

> The unitary isobar model MAID has been extended and used for a partial wave analysis of pion photo- and electroproduction in the resonance region W < 2 GeV. Older data from the world data base and more recent experimental results from Mainz, Bates, Bonn and JLab for Q^2 up to 4.0 (GeV/c)^2 have been analyzed and the Q^2 dependence of the helicity amplitudes have been extracted for a series of four star resonances. We compare single-Q^2 analyses with a superglobal fit in a new parametrization of Maid2003 together with predictions of the hypercentral constituent quark model. As a result we find that the helicity amplitudes and transition form factors of constituent quark models should be compared with the analysis of bare resonances, where the pion cloud contributions have been subtracted.

# Electroproduction of nucleon resonances

## Key claim
The helicity amplitudes and transition form factors extracted from constituent quark models should be compared with the analysis of *bare* resonances, where pion cloud contributions have been subtracted, rather than with the fully dressed resonance amplitudes obtained from empirical partial wave analyses.

## Method
The unitary isobar model MAID was extended to include all four-star nucleon resonances below W = 2 GeV and used for a partial wave analysis of pion photo- and electroproduction data. Single-Q² fits were performed on separate data sets at fixed Q² values, and a superglobal fit was carried out using a parametrization of the Q² dependence of the helicity amplitudes. The results were compared with predictions from the hypercentral constituent quark model (hCQM) and with pion cloud contributions extracted using the dynamical model DMT.

## Key numbers
- Data analyzed cover Q² up to 4.0 (GeV/c)² and an energy range 1.1 < W < 2.0 GeV.
- For the Δ(1232) resonance, the hCQM transition moment μ_{NΔ} is lower than the empirical value by about 30% in simple SU(2) calculations and can be as low as only half the empirical value in more refined calculations.
- For the P₁₁(1440) and S₁₁(1535) resonances, pion cloud contributions are most important near the photon point and become negligible around Q² = 0.5 GeV².
- Model uncertainty for the longitudinal amplitudes of the D₁₃(1520) and F₁₅(1680) resonances can be as large as 50%.
- The phase factor e^{iφ} in the unitarization procedure can reach values of about 60° for resonances other than the Δ(1232).

## Context
This work extends the unitary isobar model MAID (Maid, 1999) and the dynamical model DMT (DMT, 1999) to include all four-star resonances below 2 GeV, building on earlier partial wave analyses of πN scattering by Höhler (1979) and the GWU group (Arndt, 1997). The hCQM predictions (hqcm95) are compared with the empirical amplitudes, and the pion cloud contributions are estimated using the DMT model. The results enable future comparisons with lattice QCD calculations such as those by Frank (2003) and provide a framework for analyzing neutron resonance data, which will be implemented in the next version of MAID (MAID2003).
