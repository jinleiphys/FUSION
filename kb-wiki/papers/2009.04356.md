# Minimal complete sets for two pseudoscalar meson photoproduction

> For photoproduction reactions with final states consisting of two pseudoscalar mesons and a spin-1/2 baryon, 8 complex amplitudes need to be determined uniquely. A modified version of Moravcsik's theorem is employed for these reactions, resulting in slightly over-complete sets of polarization observables that are able to determine the amplitudes uniquely. Further steps were taken to reduce the found sets to minimal complete sets. As a final result, multiple minimal complete sets without any remaining ambiguities are presented for the first time. These sets consist of $2N=16$ observables, containing only one triple polarization observable.

## Key claim
The paper presents the first derivation of minimal complete sets of polarization observables for two pseudoscalar meson photoproduction that are free of any remaining discrete ambiguities, using a modified version of Moravcsik's theorem. The authors note that these complete experiments are an idealization for data with no uncertainty.

## Method
The authors applied a modified version of Moravcsik's theorem, which treats each of the 8 complex amplitudes as a node in a graph and each bilinear product as an edge, requiring a connected graph with an odd number of edges corresponding to imaginary parts of bilinear products. They analyzed cycle graphs (where each node has degree two) as the most economical type for eliminating discrete ambiguities, and used the 64 polarization observables organized into eight groups according to their mathematical structure. The algorithm was automated to generate 5964 unique but slightly over-complete sets, which were then reduced to minimal complete sets containing exactly 16 observables.

## Key numbers
- The process is described by N = 8 complex amplitudes, allowing for 64 measurable polarization observables.
- Minimal complete sets consist of 2N = 16 observables, containing only one triple polarization observable.
- The analysis resulted in 5964 unique, slightly over-complete sets of observables.
- For the reaction gamma p -> p pi0 pi0, the most common observable is the unpolarized cross section I0, followed by beam asymmetries I^s, I^c, and I^odot.

## Context
This work extends the complete experiment analysis performed analytically by Chiang and Tabakin (1997) for single pseudoscalar meson photoproduction, and addresses limitations of prior work by Arenhövel and Fix (2014) who used the inverse function theorem (yielding sets that might only be locally ambiguity-free) and a graph theoretical approach (yielding sets with 25 observables that still had quadratic ambiguities). The results enable unambiguous determination of the 8 complex amplitudes for two pseudoscalar meson photoproduction, which is essential for studying resonant states such as Delta(1232), N(1440)1/2+, and N(1520)3/2- via partial wave analysis.
