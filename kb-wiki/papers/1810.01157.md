# Event by event fluctuations of the source shape: implications for the Levy shape, and Event Shape Sorting

> In the first part of this contribution we show that the Levy stable shape of the correlation function can be caused by averaging of the measured correlation functions over a large number of events. In the second part it is demonstrated how a sample of events sorted by Event Shape Sorting technique exhibits different azimuthal dependence of correlation radii in each event class.

## Key claim

The Levy-stable shape of the measured correlation function can arise from averaging correlation functions over many events with different source sizes and orientations, rather than reflecting the shape of any single-event source; the Event Shape Sorting technique can then reduce this averaging by grouping events with similar azimuthal hadron distributions.

## Method

The authors first constructed a toy-model source with a two-dimensional Gaussian profile whose sizes were distributed according to the optical Glauber model and whose orientation angle was uniformly distributed, then averaged the resulting correlation functions. They also used a blast-wave model extended to include transverse anisotropies up to second order, with temperature T=120 MeV, radius R0=7 fm, flow rapidity ρ0=0.8, and anisotropy parameters a2=ρ2=0.2. For the Event Shape Sorting study, they generated 150,000 events using the DRAGON Monte Carlo blast-wave generator with resonances and azimuthal anisotropy, then sorted them and computed the azimuthal dependence of correlation radii for each event class.

## Key numbers

- The toy-model averaging produced a correlation function best fit by a Levy-stable distribution with index parameter α = 1.70.
- The blast-wave model used parameters: T = 120 MeV, R0 = 7 fm, ρ0 = 0.8, a2 = 0.2, ρ2 = 0.2.
- 150,000 events were generated by the DRAGON Monte Carlo generator for the Event Shape Sorting analysis.
- Event classes 6, 9, and 10 showed the clearest third-order component in the azimuthal dependence of the correlation radii.

## Context

This work extends the investigation by Plumberg (2015) on how event-by-event fluctuations impact the event-averaged correlation function, and recalls the earlier suggestion by Bialas (1990) that Levy-stable shapes could result from summing over sources with a distribution of sizes. The Event Shape Sorting technique, introduced by Kopecna (2015), is contrasted with Event Shape Engineering (Schukraft, 2012) and offers the unique opportunity to observe all orders of azimuthal oscillation in correlation radii simultaneously, whereas previous experimental analyses could only look at each order separately after rotating events to the n-th order event plane.
