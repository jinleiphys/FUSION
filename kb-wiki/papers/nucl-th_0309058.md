# New Glance at the Experimental Data for Low Lying Collective Excited States

> Recently the classification of low-lying excited 0+ states in even even deformed nuclei has been done. The available experimental data were represented as the energies parabolic distributed by number of monopole excitations. With other words each 0+ state now is determined as the collective state with the corresponding number of monopole type phonons n . In this paper we discuss whether the experimental data for low-lying excited states possessing not equal to zero spins can also be described with parabolic distribution function depending on integer classification parameter and find any vindication of the connection between this integer parameter and the number of collective excitations building the corresponding state.

## Key claim
The paper proposes that the energies of low-lying collective excited states with non-zero spin (2+, 4+, 6+, 8+) in even-even deformed nuclei can be described by the same parabolic distribution function previously applied to 0+ states, with the integer classification parameter interpreted as a measure of collectivity (number of monopole-type phonons or the SU(3) label lambda).

## Method
The author extends a previous classification of 0+ excited states in even-even deformed nuclei, where experimental energies were fitted to a parabolic function E_n = An - Bn^2 + C derived from a monopole boson Hamiltonian. For states with non-zero spin, the Interacting Vector Boson Model (IVBM) of Georgieva, Raychev, and Roussev is used, where the SU(3) label lambda (related to the number of bosons N and pseudospin T via T = lambda/2, N = 2*mu + lambda) serves as the classification parameter. The energy spectrum for a given (lambda,0) multiplet is written as E(lambda) = A*lambda - B*lambda^2 + C, and experimental data for 2+, 4+, 6+, and 8+ states in selected nuclei are plotted against this parabolic form.

## Key numbers
- For 162Dy: the energies of 2+, 4+, 6+, and 8+ excited states are shown to follow the parabolic distribution (Figure 3 and Figure 4 in the paper; specific numerical values not given in the text).
- For 240Pu and 250Cf: 2+ excited state energies are shown to follow the parabolic distribution (Figure 4; specific numerical values not given in the text).
- The parabolic distribution function E(lambda) = A*lambda - B*lambda^2 + C is used, with A, B, C as combinations of free model parameters a, b, alpha_3, beta_3, alpha_1 from the IVBM Hamiltonian.

## Context
This work extends the author's previous classification of 0+ excited states in even-even deformed nuclei (Garistov, 2003) to states with non-zero spin, using the same parabolic energy distribution. It builds on the Interacting Vector Boson Model (IVBM) developed by Georgieva, Raychev, and Roussev, which uses two kinds of vector bosons and the group Sp(12,R) to describe collective nuclear excitations. The proposed classification may help experimentalists resolve ambiguous spin assignments in low-energy nuclear spectra and provides a simple phenomenological tool for organizing collective state energies across a wide range of even-even nuclei.
