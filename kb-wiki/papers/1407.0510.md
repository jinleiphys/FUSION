# Bound clusters on top of doubly magic nuclei

> An effective $α$ particle equation is derived for cases where an $α$ particle is formed on top of a doubly magic nucleus. As an example, we consider $^{212}$Po with the $α$ on top of the $^{208}$ Pb core. We will consider the core nucleus infinitely heavy, so that the $α$ particle moves with respect to a fixed center, i.e., recoil effects are neglected. The fully quantal solution of the problem is discussed. The approach is inspired by the THSR (Tohsaki-Horiuchi-Schuck-Röpke) wave function concept that has been successfully applied to light nuclei. Shell model calculations are improved by including four-particle ($α$-like) correlations that are of relevance when the matter density becomes low. In the region where the $α$-like cluster penetrates the core nucleus, the intrinsic bound state wave function transforms at a critical density into an unbound four-nucleon shell model state. Exploratory calculations for $^{212}$Po are presented. Such preformed cluster states are only hardly described by shell model calculations. Reasons for different physics behavior of an $α$-like cluster with respect to a deuteron-like cluster are discussed.

## Key claim

An effective alpha-particle equation is derived for an alpha cluster on top of a doubly magic nucleus, but the authors explicitly neglect recoil effects by treating the core nucleus as infinitely heavy, and they note that the intrinsic bound state wave function transforms at a critical density into an unbound four-nucleon shell model state when the cluster penetrates the core.

## Method

The authors separate the center-of-mass motion from the intrinsic motion of the four-nucleon cluster using Jacobi-Moshinsky coordinates, then derive coupled Schrodinger equations for the center-of-mass and intrinsic wave functions. For the specific case of 212Po with a 208Pb core, they apply a Tamm-Dancoff approximation with a separable nucleon-nucleon interaction having Gaussian form factor, using a variational approach with Gaussian trial wave functions and treating the Pauli blocking via a Local Density Approximation.

## Key numbers

* The empirical binding energy of the free alpha particle is E_alpha^(0) = -B_alpha = -28.3 MeV.
* The saturation density of nuclear matter is n_0 approximately 0.15 fm^{-3}.
* The alpha particle as a well-formed cluster disappears around a baryonic density n_B,cluster approximately n_0/5.
* The preformation amplitude of core + alpha clustering in the parent state is about 30 percent, much higher than former microscopic estimates.

## Context

This work extends the THSR (Tohsaki-Horiuchi-Schuck-Ropke) wave function concept, previously applied successfully to light nuclei such as 12C for the Hoyle state, to heavy doubly magic nuclei. It addresses the long-standing problem of alpha clustering on top of 208Pb to describe 212Po, which has been treated previously by Varga, Lovas, Ren, Wyss, and Xu, and by Buck for alpha decay widths. The approach aims to unify cluster formation at low densities outside the nucleus with shell model approaches at high densities, enabling improved treatment of alpha decay widths that shell model calculations (Mang) tend to substantially underestimate.
