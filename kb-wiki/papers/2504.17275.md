# Physics-Embedded Bayesian Neural Network (PE-BNN) to predict Energy Dependence of Fission Product Yields with Fine Structures

> We present a physics-embedded Bayesian neural network (PE-BNN) framework that integrates fission product yields (FPYs) with prior nuclear physics knowledge to predict energy-dependent FPY data with fine structure. By incorporating an energy-independent phenomenological shell factor as a single input feature, the PE-BNN captures both fine structures and global energy trends. The combination of this physics-informed input with hyperparameter optimization via the Watanabe-Akaike Information Criterion (WAIC) significantly enhances predictive performance. Our results demonstrate that the PE-BNN framework is well-suited for target observables with systematic features that can be embedded as model inputs, achieving close agreement with known shell effects and prompt neutron multiplicities.

## Key claim

The paper claims that incorporating a phenomenological shell factor as an input feature into a Bayesian neural network enables simultaneous prediction of both fine structures and global energy trends in fission product yields, with the shell factor being crucial for reproducing fine structure variations while theoretical calculations primarily improve global structure.

## Method

The authors developed a Bayesian neural network with a two-hidden-layer architecture trained on a hybrid dataset combining 80% JENDL-5 evaluated data, experimental FPY data from EXFOR, and theoretical model calculations, with prioritized weighting. A novel shell factor input was designed using Gaussian distributions centered at mass numbers A=134, 140, and 144 (with mirror terms for the light fragment side) multiplied by a Boltzmann damping factor exp(-E/kT) with kT=1.5 MeV, where the weights W1=4, W2=2, and W3=1 reflect double shell closure and deformed shell structures. Model selection was performed using the Watanabe-Akaike Information Criterion (WAIC) to optimize the network configuration, selecting an 11-11 neuron double hidden-layer architecture, with inference conducted via the No-U-Turn Sampler in NumPyro.

## Key numbers

* Shell factor inclusion improved log-likelihood by 21.3% on training data and 34.6% on testing data for the optimal 11-11 neuron configuration
* Across architectures from 5-5 to 20-20 neurons, the shell factor improved training performance by 19.6% and testing performance by 29.4% on average
* Training dataset comprised 6554 data points from JENDL-5, 989 experimental data points from EXFOR, and 2062 theoretical model calculations
* The effective damping scale was set to kT = 1.5 MeV, guided by the Fermi-gas relation and supported by multidimensional Langevin simulations predicting characteristic scission temperature of roughly 1.5 MeV for actinide low-energy fission
* The shell factor weights were assigned as W1=4, W2=2, and W3=1, with Gaussian standard deviation sigma=1 for all six Gaussian components centered at A=134, 140, and 144 (and their light-fragment mirrors)

## Context

This work extends previous Bayesian neural network applications to fission product yields by Chen et al. (2023) that used data augmentation through weighted training, and builds on earlier BNN applications in nuclear physics including nuclear mass predictions by Niu and Liang (2018). The approach contrasts with semi-empirical models such as GEF, which incorporate fine structures through a large set of adjustable parameters optimized to reproduce experimental data, whereas the PE-BNN framework encodes phenomenological shell information explicitly as an input feature. The model enables predictions of energy-dependent fission yields across a broader range of incident neutron energies than the three specific energies (thermal, 0.5 MeV, and 14 MeV) currently provided in standard nuclear data libraries, addressing the need for more accurate predictive methods for applications in reactor physics, nuclear forensics, and astrophysical nucleosynthesis.
