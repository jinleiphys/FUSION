# Multi-Phonon $Оі$-Vibrational Bands and the Triaxial Projected Shell Model

> We present a fully quantum-mechanical, microscopic, unified treatment of ground-state band and multi-phonon $Оі$-vibrational bands using shell model diagonalization with the triaxial projected shell model. The results agree very well with data on the g- and $Оі$-band spectra in $^{156-170}$Er, as well as with recently measured $4^+$ 2-phonon $Оі$-bandhead energies in $^{166}$Er and $^{168}$Er. Multi-phonon $Оі$-excitation energies are predicted.

## Key claim

The Triaxial Projected Shell Model (TPSM) provides a fully quantum-mechanical, microscopic, unified description of ground-state bands and multi-phonon Оі-vibrational bands in a single shell-model diagonalization, but the Оі-bands arise from the introduction of triaxiality in the deformed basis (a single adjustable parameter ОµвЂІ) rather than from multi-quasiparticle admixtures, which had previously failed to produce low-lying Оі-states in the axial Projected Shell Model.

## Method

The authors performed shell-model diagonalization within a basis of angular-momentum-projected triaxial quasiparticle vacuum states (the simplest possible configuration space for even-even nuclei), using a Hamiltonian with monopole pairing, quadrupole pairing, and a quadrupole-quadrupole interaction whose coupling constants are fixed by self-consistency and standard systematics, not adjusted to Оі-band energies. The triaxial deformation parameter ОµвЂІ was the single adjustable parameter, set to 0.13 (corresponding to Оі в‰€ 25.5В°) for all Er isotopes studied, and calculations were carried out for spins up to I=10.
