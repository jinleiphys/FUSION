# Differential cross section analysis in kaon photoproduction using associated legendre polynomials

> Angular distributions of differential cross sections from the latest CLAS data sets \cite{bradford}, for the reaction $γ+p {\to} K^{+} + Λ$ have been analyzed using associated Legendre polynomials. This analysis is based upon theoretical calculations in Ref. \cite{fasano} where all sixteen observables in kaon photoproduction can be classified into four Legendre classes. Each observable can be described by an expansion of associated Legendre polynomial functions. One of the questions to be addressed is how many associated Legendre polynomials are required to describe the data. In this preliminary analysis, we used data models with different numbers of associated Legendre polynomials. We then compared these models by calculating posterior probabilities of the models. We found that the CLAS data set needs no more than four associated Legendre polynomials to describe the differential cross section data. In addition, we also show the extracted coefficients of the best model.

## Key claim
The CLAS differential cross section data for the reaction γ + p → K⁺ + Λ require no more than four associated Legendre polynomials to describe the angular distributions, with the maximum posterior probability generally occurring at fourth order across photon energies from threshold into the resonance region.

## Method
The authors constructed data models as expansions of associated Legendre polynomials of varying order (maximum number of polynomials) for the Legendre class ℒ₀, which describes differential cross section observables. They fitted each model to the CLAS angular distribution data using the MINUIT minimization package, then compared models by calculating posterior probabilities via Bayes' theorem, integrating over the parameter space with Gaussian error assumptions and a Taylor expansion of χ² about its minimum.

## Key numbers
- The maximum posterior probability for the example photon energy bin at Eγ = 1.824 GeV is given by the data model containing four associated Legendre polynomials.
- The order of the data model with greatest probability generally increases from threshold into the resonance region, but the maximum is mostly at the fourth order across all photon energy bins.
- The analysis used CLAS data sets for the reaction γ + p → K⁺ + Λ, with differential cross sections measured as angular distributions.

## Context
This work extends the theoretical framework of Fasano (1992), who classified all sixteen observables in kaon photoproduction into four Legendre classes (ℒ₀, ℒ₁ₐ, ℒ₁ᵦ, ℒ₂), each expandable in different associated Legendre polynomials. The analysis addresses the open question of how many terms are required in each expansion, focusing here on the differential cross section (ℒ₀ class). This enables model-independent extraction of polynomial coefficients for comparison with isobar models (Mart 1996, Mart 1997; Ireland 2004; Janssen 2001, Janssen 2002) and coupled channel models (Shyklar 2004; Usov 2004; Penner 2002), which often produce very different predictions.
