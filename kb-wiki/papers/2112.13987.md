# Machine learning phase transitions of the three-dimensional Ising universality class

> Exploration of the QCD phase diagram and critical point is one of the main goals in current relativistic heavy-ion collisions. The QCD critical point is expected to belong to a three-dimensional (3D) Ising universality class. Machine learning techniques are found to be powerful in distinguishing different phases of matter and provide a new way to study the phase diagram. We investigate phase transitions in the 3D cubic Ising model using supervised learning methods. It is found that a 3D convolutional neural network can be trained to effectivelly predict physical quantities in different spin configurations. With a uniform neural network architecture, it can encode phases of matter and identify both second- and first-order phase transitions. The important features that discriminate different phases in the classification processes are investigated. These findings can help study and understand QCD phase transitions in relativistic heavy-ion collisions.

# Machine learning phase transitions of the three-dimensional Ising universality class

## Key claim
A single 3D convolutional neural network architecture can be trained on raw spin configurations to identify both second-order and first-order phase transitions in the 3D Ising model, and the network learns different physical quantities (absolute magnetization and energy for second-order; magnetization for first-order) as the discriminating features. The authors hedge that this approach can help study QCD phase transitions, but only if the QCD critical point indeed belongs to the same 3D Ising universality class.

## Method
The authors performed Monte Carlo simulations of the 3D cubic Ising model using the Metropolis algorithm on lattices of size L x L x L with periodic boundary conditions, generating 20000 independent spin configurations at each selected temperature and external magnetic field. They trained a 3D convolutional neural network (CNN) with two 3D convolutional layers (four filters each of size 2x2x2), a pooling layer, a fully connected hidden layer with 16 neurons, and a softmax output layer, using 90% of configurations for training and 10% for testing. The network was applied to classify phases for second-order transitions (varying temperature at zero magnetic field) and first-order transitions (varying magnetic field at fixed temperatures below T_c), and finite-size scaling was performed on the network outputs to extract critical parameters.

## Key numbers
- Critical temperature extracted from finite-size scaling of CNN outputs: T_c = 4.517 ± 0.003, compared to Monte Carlo result T_c = 4.512 ± 0.001.
- Critical exponent ν extracted from finite-size scaling: ν = 0.63 ± 0.09, compared to Monte Carlo result ν = 0.63 ± 0.01.
- Test accuracy for second-order phase transition classification: 86.3% for L = 8, increasing to 96.7% for L = 28.
- Pearson correlation coefficients between network outputs and physical quantities for second-order transition: 0.93 with absolute magnetization, 0.85 with energy, 0.17 with magnetization.
- Pearson correlation coefficients for first-order transition: 1.0 with magnetization, 0.1 with absolute magnetization, 0.08 with energy.
- Monte Carlo simulations used 50 sweeps between independent configurations to reduce correlations.

## Context
This work extends prior machine learning studies of phase transitions from two-dimensional Ising models (CNNIsing1, MLIsing1, MLIsing2, MLIsing3, MLIsing4) to the three-dimensional case, and from only second-order transitions to include first-order transitions as well. The authors note that previous investigations focused on 2D models and were restricted to second-order phase transitions, leaving the 3D Ising phase diagram and first-order transition unexplored by ML methods. The results enable future application of similar supervised learning techniques to Monte Carlo models of QCD phase transitions and to experimental data from relativistic heavy-ion collisions at SPS and RHIC, where measured high-order cumulants and intermittency suggest large density fluctuations near the QCD critical point.
