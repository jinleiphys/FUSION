# ADG: Automated generation and evaluation of many-body diagrams II. Particle-number projected Bogoliubov many-body perturbation theory

> We describe the second version (v2.0.0) of the code ADG that automatically (1) generates all valid off-diagonal Bogoliubov many-body perturbation theory diagrams at play in particle-number projected Bogoliubov many-body perturbation theory (PNP-BMBPT) and (2) evaluates their algebraic expression to be implemented for numerical applications. This is achieved at any perturbative order $p$ for a Hamiltonian containing both two-body (four-legs) and three-body (six-legs) interactions (vertices). All valid off-diagonal BMBPT diagrams of order $p$ are systematically generated from the set of diagonal, i.e., unprojected, BMBPT diagrams. The production of the latter were described at length in https://doi.org/10.1016/j.cpc.2018.11.023 dealing with the first version of ADG. The automated evaluation of off-diagonal BMBPT diagrams relies both on the application of algebraic Feynman's rules and on the identification of a powerful diagrammatic rule providing the result of the remaining $p$-tuple time integral. The new diagrammatic rule generalizes the one already identified in https://doi.org/10.1016/j.cpc.2018.11.023 to evaluate diagonal BMBPT diagrams independently of their perturbative order and topology. The code ADG is written in Python3 and uses the graph manipulation package NetworkX. The code is kept flexible enough to be further expanded throughout the years to tackle the diagrammatics at play in various many-body formalisms that already exist or are yet to be formulated.

## Key claim

The paper extends the automated diagram generation and evaluation code ADG from diagonal (unprojected) Bogoliubov many-body perturbation theory (BMBPT) to off-diagonal BMBPT diagrams required for particle-number projected Bogoliubov many-body perturbation theory (PNP-BMBPT), where the broken U(1) gauge symmetry is exactly restored at any truncation order rather than only in the all-orders limit.

## Method

The code generates all valid off-diagonal BMBPT diagrams of arbitrary perturbative order p by starting from the set of diagonal BMBPT diagrams (produced by the first version of ADG) and applying systematic transformations. Each diagram is encoded as an oriented adjacency matrix, converted to a graph object using the NetworkX package, and then evaluated algebraically by applying Feynman rules together with a newly identified diagrammatic rule that provides the result of the remaining p-tuple time integral, generalizing the rule previously used for diagonal diagrams. The method works for Hamiltonians containing both two-body (four-leg) and three-body (six-leg) interactions.

## Key numbers

- The code handles perturbative orders p up to any value, with the number of diagrams growing combinatorially with p (no specific maximum p given in the text).
- The volume of the U(1) group is given as v_U(1) = 2π.
- The code supports operators up to rank deg_max = 6 (three-body interactions), with matrix elements fully antisymmetrized.

## Context

This work extends the first version of ADG (Arthuis et al., 2019) which handled only diagonal BMBPT diagrams, and builds on the PNP-BMBPT formalism introduced by Duguet et al. (2015). The code enables numerical implementations of particle-number projected perturbation theory for open-shell superfluid nuclei, where straight BMBPT suffers from symmetry contamination at finite order. The development responds to the need for automated diagram handling as many-body perturbation theory calculations reach higher orders, where manual diagram generation becomes intractable.
