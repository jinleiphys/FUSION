# Pionic Decay of a Possible d'-Dibaryon

> The pionic decay of a possible d'-dibaryon in the process d' --> pi + N + N is studied in the microscopic quark shell model and with a single-quark transition operator describing the transition q --> pi + q'. For the d' with quantum numbers J^P=0^-, T=0, we employ a six-quark shell-model wave function with a spatial s^5p [51]_X-configuration with N=1 harmonic oscillator quanta. It is shown that the pionic decay width depends strongly on the mass and size of the d'. In the case that the calculated d' mass is close to the experimental one a small pionic decay width of 0.04 MeV is obtained. This is an order of magnitude smaller than the experimentally suggested value of 0.5 MeV. Two possibilities to improve the calculated width are suggested. The effect of the nonstatic correction term in the transition operator and the influence of the form factor at the decay vertex on the decay width are also discussed.

## Key claim

The pionic decay width of the d'-dibaryon depends strongly on its mass and size; when the calculated d' mass is close to the experimental value of 2065 MeV, a pionic decay width of only 0.04 MeV is obtained, which is an order of magnitude smaller than the experimentally suggested value of 0.5 MeV.

## Method

The authors employed a microscopic quark shell model with a six-quark wave function in the s^5p [51]_X configuration with N=1 harmonic oscillator quanta for the d'-dibaryon (J^P=0^-, T=0). They used a single-quark transition operator q → q + π to calculate the pionic decay width for the process d' → π + N + N, evaluating the transition matrix element by inserting intermediate six-quark states (s^6 and s^4p^2 configurations) and integrating over the three-body phase space. The effects of a nonstatic correction term in the transition operator and a form factor at the qqπ-vertex were also investigated.

## Key numbers

- The experimental d'-dibaryon mass is M_d' = 2065 MeV, with spin-parity J^P = 0^- and isospin T = 0
- The experimentally suggested pionic decay width is Γ_π^exp ≈ 0.5 MeV
- For parameter set 3 (linear confinement), the calculated d' mass is M_d' = 2162 MeV with size parameter b_6 = 1.24 fm, yielding a total pionic decay width Γ_π = 0.04 MeV
- For parameter set 5 (quadratic confinement), the calculated d' mass is M_d' = 2112 MeV with size parameter b_6 = 0.95 fm, yielding a total pionic decay width Γ_π = 0.04 MeV
- For parameter set 1, the calculated d' mass is M_d' = 2705 MeV with b_6 = 0.59 fm, yielding Γ_π = 16.0 MeV
- For parameter set 2, the calculated d' mass is M_d' = 2680 MeV with b_6 = 0.65 fm, yielding Γ_π = 16.6 MeV
- For parameter set 4, the calculated d' mass is M_d' = 2484 MeV with b_6 = 0.78 fm, yielding Γ_π = 8.69 MeV
- The overlap-squared between the final two-nucleon state and the six-quark configuration is approximately 0.6 for sets 1, 2, and 4, but only 0.05 for set 3 and 0.28 for set 5
- The inclusion of the nonstatic correction term reduces the decay width by roughly 10% for sets 1, 2, and 4, and by half for set 3

## Context

This work extends the Tubingen group's previous nonrelativistic quark model studies of the d'-dibaryon structure using both the quark cluster model and the quark shell model (Buchmann 1995, Wagner 1995). The d'-dibaryon was proposed to explain a resonance-like peak observed in pionic double charge exchange (π^+, π^-) reactions on nuclear targets from ^4He to ^56Fe at incident pion energy T_π ≈ 50 MeV (Bilger 1992, Clement 1994). The paper contrasts with the approach of Valcarce (1995), who suggested the resonance might be an isospin-2 resonance in the nucleon-Δ channel. The results indicate that reproducing both the d' mass and decay width simultaneously requires either additional quark-quark interactions not considered here or a more complete treatment of the final NNπ scattering state, as suggested by Schepkin (1993) who estimated an enhancement factor of about 5 from final state interactions.
