# Transverse Momentum Distributions in p-Pb collisions and Tsallis Thermodynamics

> The transverse momentum distributions of charged particles in p-Pb collisions as sqrt{s_{NN}} = 5.02 TeV measured by the ALICE collaboration are fitted using Tsallis statistics. The use of a thermodynamically consistent form of this distribution leads to an excellent description of the transverse momentum distributions for all rapidity intervals. The values of the Tsallis parameter q, the temperature T and the radius R of the system do not change within the measured pseudorapidity intervall.

## Key claim

The Tsallis distribution, using a thermodynamically consistent form, provides an excellent description of transverse momentum distributions in p-Pb collisions at √sNN = 5.02 TeV across all measured pseudorapidity intervals, with the caveat that the temperature T is substantially higher than in p-p collisions while the q parameter and radius R are compatible.

## Method

The authors fitted the ALICE collaboration's transverse momentum distributions of charged particles in p-Pb collisions at √sNN = 5.02 TeV using a sum of three Tsallis distributions for π+, K+, and protons, with antiparticles included via a factor of two. The fit was performed at mid-rapidity with zero chemical potential using the thermodynamically consistent form of the Tsallis distribution, and the parameters q, T, and R were extracted for three pseudorapidity intervals.

## Key numbers

- Tsallis parameter q = 1.140 ± 0.001 for pseudorapidity interval -0.3 < η < 0.3
- Temperature T = 112.81 ± 2.35 MeV for pseudorapidity interval -0.3 < η < 0.3
- Radius R = 4.03 ± 0.12 fm for pseudorapidity interval -0.3 < η < 0.3
- Tsallis parameter q = 1.139 ± 0.001 for pseudorapidity interval 0.3 < η < 0.8
- Temperature T = 113.35 ± 2.47 MeV for pseudorapidity interval 0.3 < η < 0.8
- Radius R = 4.07 ± 0.13 fm for pseudorapidity interval 0.3 < η < 0.8
- Tsallis parameter q = 1.139 ± 0.001 for pseudorapidity interval 0.8 < η < 1.3
- Temperature T = 111.92 ± 2.63 MeV for pseudorapidity interval 0.8 < η < 1.3
- Radius R = 4.19 ± 0.14 fm for pseudorapidity interval 0.8 < η < 1.3
- The temperature in p-Pb collisions is around 112 MeV, clearly higher than in p-p collisions

## Context

This work extends the application of Tsallis statistics from p-p collisions, where it had been shown to give excellent fits with q values around 1.1-1.2 (Cleymans 2011, Sena 2012, Wong 2013), to p-Pb collisions measured by the ALICE collaboration (ALICE 2012). The results show that q and R are consistent with p-p values while T is substantially higher, and the authors note that extending measurements to lower transverse momentum could clarify the sensitivity of T and R in that region.
