# Numerical Simulation of Shell Model Single Particle Energy States using Matrix Numerov Method in Gnumeric Worksheet

> Single particle energy states as described by nuclear shell model are obtained for doubly magic nuclei using Gnumeric worksheet environment. Numerov method rephrased in matrix form is utilised to solve time-independent Schrodinger equation (TISE) within mean-field approximation, described by Woods-Saxon (WS) potential along with spin-orbit term, to obtain the single particle energies for both neutron and proton states. The WS model parameters are chosen from previous simulation results performed using matrix methods technique involving sine basis, where optimization was done w.r.t available experimental single particle energies for 208Pb and 48Ca. In this paper, only the algorithm parameters, step size h and matrix size N are optimized to obtain the expected energy level sequence obtained using matrix methods. An attempt is made, by incorporating this tool within the framework of guided inquiry strategy (a constructivist approach to learning), to actively engage the students in assigning appropriate spin-parity configurations for ground states of nuclei neighbouring the doubly magic ones. It has been observed that the ground state configurations could be better predicted when energy level sequences are known for all nuclei as compared to what is usually obtained from that of 208Pb alone.

# Numerical Simulation of Shell Model Single Particle Energy States using Matrix Numerov Method in Gnumeric Worksheet

## Key claim

The Matrix Numerov method implemented in a Gnumeric worksheet environment can reproduce single-particle energy levels for doubly magic nuclei that match experimental values to two decimal places, but only after optimizing algorithm parameters (step size h and matrix size N) rather than the physical Woods-Saxon model parameters, which were fixed from prior work. The energy level sequence is shown to be mass-dependent, contradicting the common textbook practice of using a single universal level ordering for all nuclei.

## Method

The time-independent Schrodinger equation with a Woods-Saxon potential including spin-orbit and Coulomb terms was solved using the Numerov method recast into a matrix eigenvalue equation. The algorithm was implemented entirely within the Gnumeric spreadsheet environment using its built-in matrix inversion and eigenvalue solver functions. Physical model parameters (potential depths, radius R0=1.28 fm, diffuseness a=0.66 fm, spin-orbit strength V1=-0.44V0, and r0=0.90) were taken from prior optimization against experimental data for 208Pb and 48Ca using a sine-basis matrix method. Only the step size h=0.1 fm and matrix size N (ranging from 96 for 16O to 229 for 208Pb) were optimized in this work.

## Key numbers

- Step size h = 0.1 fm used for all calculations
- Matrix sizes N: 96 for 16O, 131 for 40Ca, 138 for 48Ca, 145 for 56Ni, 177 for 100Sn, 194 for 132Sn, and 229 for 208Pb
- For 40Ca proton states: 1d5/2 at -12.19 MeV (experimental -15.07 MeV), 2s1/2 at -8.14 MeV (experimental -10.92 MeV), 1d3/2 at -6.85 MeV (experimental -8.33 MeV)
- For 40Ca neutron states: 1d5/2 at -19.54 MeV (experimental -22.39 MeV), 2s1/2 at -15.54 MeV (experimental -18.19 MeV), 1d3/2 at -14.28 MeV (experimental -15.64 MeV)
- Chi-squared values for neutron states: 0.24 (16O), 1.18 (48Ca), 0.09 (56Ni), 0.06 (100Sn), 0.03 (132Sn), 0.06 (208Pb)
- Chi-squared values for proton states: 1.53 (16O), 0.31 (48Ca), 0.48 (56Ni), 0.25 (100Sn), 0.03 (132Sn), 0.11 (208Pb)
- Woods-Saxon potential depth parameters: V0 = -51 +/- 33((N-Z)/A) MeV (sign depending on neutron/proton)
- Spin-orbit strength V1 = -0.44 * V0
- Radius parameter R0 = 1.28 fm, surface diffuseness a = 0.66 fm
- Reduced mass uses mp = 938.272 MeV/c^2 and mn = 939.565 MeV/c^2
- hbar*c = 197.329 MeV-fm

## Context

This work extends the authors' prior matrix method using sine-wave basis in Scilab (Sastri et al., 2021) by replacing it with the Matrix Numerov method, which avoids numerical integration of matrix elements and can be implemented in a simple spreadsheet. The results confirm that energy level sequences differ between light and heavy nuclei, as noted in Bohr and Mottelson, contradicting the single universal sequence presented in standard textbooks such as Krane (1988). The approach enables a guided inquiry teaching strategy where students can determine ground state spin-parity assignments for nuclei near doubly magic shells and test the limits of the shell model.
