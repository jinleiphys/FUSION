# Fission Fragment Yields Of $^{235}$U$(n_{th},f)$ Evaluated By The CCONE Code System

> Fission fragment yield evaluations are one of the important nuclear data studies. Fission accompanies various physical observables such as prompt fission neutron, prompt fission gamma, and delayed-neutrons. When evaluating fission fragment yields, a study including correlations among those observables is essentially required. However, fission fragment yield data in the past JENDL libraries have been made by focusing only on experimental fragment yields, decay heats, and delayed neutron yields, and they have not been expanded into a wider range of fission observables. This is because the evaluation method adopted in the JENDL libraries could not study fission fragment yields and particle emissions from fragments simultaneously. To solve this problem, a calculation system with CCONE code is newly developed to estimate not only independent and cumulative fission fragment yields but also prompt fission neutron, prompt fission gamma, decay heats, and delayed-neutrons simultaneously. This system enables us to study a correlation between various fission observables. To determine lots of parameters in this system efficiently, a Gaussian process and a least square fitting are adopted. We tested the calculation system through a thermal neutron-induced fission on $^{235}$U. In this paper, we demonstrate the performance of the parameter search method and show that experimental fission fragment yield data and other observables resulting from fission are reproduced well by the new calculation system.

## Key claim

The authors claim that their newly developed calculation system, combining the CCONE code with a Gaussian process and generalized least squares parameter search, can simultaneously reproduce experimental fission fragment yields and other fission observables (prompt neutrons, prompt gamma rays, decay heats, delayed neutrons) for thermal neutron-induced fission on $^{235}$U, but the parameter search with Gaussian process alone does not converge even after 200 iterations for the full 113-parameter space, requiring a hybrid GP+GLS approach.

## Method

The system uses phenomenologically determined pre-neutron mass yields (sum of five Gaussians) and total kinetic energy distributions, then calculates independent and cumulative fission yields via the Zp model for charge distributions and the Hauser-Feshbach statistical model for particle evaporation from fission fragments. Parameters (113 total: 54 for delta Zp, 54 for delta sigma_p, plus R_T, f_Z, f_N, f_a, f_s) are determined by minimizing an objective function against 1055 experimental data points using a hybrid approach: first a Bayesian optimization with Gaussian process on a discrete grid, then a generalized least squares fit starting from the GP result. The system was tested on thermal neutron-induced fission of $^{235}$U at incident neutron energy 0.0253 eV.

## Key numbers

- Total number of experimental data points used for parameter search: N_exp = 1055
- Average total kinetic energy set to match experimental data: overline{TKE} = 171.1 MeV
- Incident neutron energy for thermal fission: E_in = E_thermal = 0.0253 eV
- Number of free parameters for delta Z_p(A) and delta sigma_p(A): 55 each (from A=64 to 118), with delta Z_p(A=118) = delta sigma_p(A=118) = 0, leaving 54 each
- Total parameters in full search: 113 (54 delta Z_p + 54 delta sigma_p + R_T + f_Z + f_N + f_a + f_s)
- Reduced parameter test: 15 parameters (delta Z_p and delta sigma_p divided into 5 mass regions)
- Objective function value for GP+GLS with reduced parameters: O(x)/N_exp approximately 72
- Objective function value for GP alone with reduced parameters: O(x)/N_exp approximately 77
- Correlation coefficient between sigma_p(3) and sigma_p(4): COR(X,Y) = -0.54

## Context

This work extends the JENDL evaluation approach by enabling simultaneous calculation of fission fragment yields and particle emissions, which previous JENDL libraries (JENDL-4.0, JENDL-5) could not do, leading to discrepancies such as overestimation of delayed-neutron yields in JENDL-4.0 (Shibata, 2012). The system is similar in concept to the BeoH code (Okumura, 2018; Lovell, 2021; Okumura, 2021), FIFRELIN code (Litaize, 2015; Piau, 2023), and FREYA (Verbeke, 2015; Verbeke, 2018), but introduces a novel GP+GLS parameter search method. The results enable future evaluations that respect correlations among all fission observables, potentially improving nuclear data libraries for reactor applications and neutrino monitoring.
