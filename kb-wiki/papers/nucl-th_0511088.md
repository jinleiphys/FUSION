# Nuclear mass systematics by complementing the Finite Range Droplet Model with neural networks

> A neural-network model is developed to reproduce the differences between experimental nuclear mass-excess values and the theoretical values given by the Finite Range Droplet Model. The results point to the existence of subtle regularities of nuclear structure not yet contained in the best microscopic/phenomenological models of atomic masses. Combining the FRDM and the neural-network model, we create a hybrid model with improved predictive performance on nuclear-mass systematics and related quantities.

# Nuclear mass systematics by complementing the Finite Range Droplet Model with neural networks

## Key claim

A neural network trained on the differences between experimental mass-excess values and the Finite Range Droplet Model (FRDM) predictions reveals that a significant portion (perhaps 30-40%) of the residual error in the FRDM can be treated systematically, indicating the existence of subtle nuclear-structure regularities not yet captured by the best microscopic/phenomenological mass models, rather than being purely random or chaotic fluctuations.

## Method

A multilayer feedforward neural network with architecture (4-6-6-6-1) and 169 weight parameters was constructed, taking as input the atomic number Z, neutron number N, and their respective parities, and outputting the mass-excess difference between experiment and the FRDM. The network was trained on 1276 nuclei (learning set M1) and validated on 344 nuclei (validation set M2), drawn from the 1620 nuclei in the FRDM database after screening against the 2003 Atomic Mass Evaluation (AME03). A separate test set M3 of 529 nuclei, predominantly far from stability, was reserved for evaluating predictive performance. The trained neural network was then combined with the FRDM to create a hybrid model for mass-excess prediction.

## Key numbers

* Hybrid model rms error on mass excess: 0.40 MeV (learning set M1), 0.49 MeV (validation set M2), 0.41 MeV (prediction set M3)
* FRDM rms error on mass excess: 0.68 MeV (M1), 0.71 MeV (M2), 0.58 MeV (M3)
* HFB2 rms error on mass excess: 0.67 MeV (M1), 0.68 MeV (M2), 0.67 MeV (M3)
* Hybrid model rms error for one-proton separation energy S(p): 0.36 MeV over all AME03 nuclides (1968 cases)
* Hybrid model rms error for one-neutron separation energy S(n): 0.35 MeV over all AME03 nuclides (1988 cases)
* Hybrid model rms error for alpha-decay Q-value Q(alpha): 0.48 MeV over all AME03 nuclides (2039 cases)
* Hybrid model rms error for beta-minus decay Q-value Q(beta-): 0.42 MeV over all AME03 nuclides (1868 cases)
* Hybrid model rms error on prediction set M3 for S(p): 0.41 MeV (453 cases); for S(2p): 0.39 MeV (434 cases); for S(n): 0.37 MeV (435 cases); for S(2n): 0.41 MeV (418 cases); for Q(alpha): 0.48 MeV (465 cases); for Q(beta-): 0.50 MeV (387 cases)

## Context

This work extends the Finite Range Droplet Model of Moller, Nix, and coworkers (1995) and the Hartree-Fock-Bogoliubov model HFB2 of Pearson, Tondeur, and coworkers, which represent the state of the art in global mass models with high theoretical input. It also builds on earlier neural-network mass models developed by the same group (Athanassopoulos et al., 2004; Gernoth et al., 1993) that directly predicted mass-excess values rather than residuals. The hybrid approach enables improved predictive performance for nuclei far from stability and for mass-related quantities such as separation energies and decay Q-values, which are critical input for theories of nucleosynthesis and supernova explosions. The results suggest that further neural-network studies could extract additional regularities from the residual discrepancies between experiment and theoretical mass models.
