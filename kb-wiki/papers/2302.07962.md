# \texttt{NuHamil}: A numerical code to generate nuclear two- and three-body matrix elements from chiral effective field theory

> The applicability of nuclear {\it ab initio} calculations has rapidly extended over the past decades. However, starting research projects is still challenging due to the required numerical expertise in the generation of underlying nuclear interaction matrix elements and many-body calculations. To ease the first issue, in this paper we introduce the numerical code \texttt{NuHamil} to generate the nucleon-nucleon (NN) and three-nucleon (3N) matrix elements expressed in a spherical harmonic-oscillator basis, inputs of many-body calculations. The ground-state energies for the selected doubly closed shell nuclei are calculated with the no-core shell-model (NCSM) and in-medium similarity renormalization group (IMSRG). The code is written in modern Fortran, and OpenMP+MPI hybrid parallelization is available for the 3N matrix-element calculations.

## Key claim

The `NuHamil` code provides a user-friendly tool to generate nucleon-nucleon (NN) and three-nucleon (3N) matrix elements in a spherical harmonic-oscillator basis for use in *ab initio* many-body calculations, but the code does not fully support non-local 3N matrix elements due to numerical instability in higher angular momentum partial waves, requiring external input files for those cases.

## Method

The code computes NN matrix elements via the Talmi-Moshinsky transformation from relative-momentum-space interactions, and 3N matrix elements via a three-body Talmi-Moshinsky transformation using antisymmetrized Jacobi-coordinate basis states. For 3N matrix elements, the code reads momentum-space matrix elements from HDF files and performs cubic b-spline interpolation for the integral over harmonic-oscillator wave functions. The code supports OpenMP+MPI hybrid parallelization for 3N matrix-element calculations and outputs matrix elements in formats compatible with the `BIGSTICK` and `imsrg++` codes.

## Key numbers

- Ground-state energy of $^{4}$He with EM500 interaction at $\lambda = 1.8$ fm$^{-1}$: $-28.45$ MeV (NCSM, $N_{\rm max}=16$, no 3N) and $-28.43$ MeV (IMSRG(2), $e_{\rm max}=14$, no 3N)
- Ground-state energy of $^{4}$He with EM500 interaction at $\lambda = 1.8$ fm$^{-1}$ including 3N ($E_{\rm 3max}=16$): $-25.61$ MeV (IMSRG(2), $e_{\rm max}=14$)
- Ground-state energy of $^{16}$O with EM500 interaction at $\lambda = 1.8$ fm$^{-1}$: $-173.3$ MeV (IMSRG(2), $e_{\rm max}=14$, no 3N) and $-121.6$ MeV (IMSRG(2), $e_{\rm max}=14$, $E_{\rm 3max}=16$)
- Ground-state energy of $^{40}$Ca with EM500 interaction at $\lambda = 1.8$ fm$^{-1}$: $-639.4$ MeV (IMSRG(2), $e_{\rm max}=14$, no 3N) and $-352.3$ MeV (IMSRG(2), $e_{\rm max}=14$, $E_{\rm 3max}=16$)
- Ground-state energy of $^{132}$Sn with N$^{3}$LO$_{\rm lnl}$ interaction at $\lambda = 2.0$ fm$^{-1}$: $-1064$ MeV (IMSRG(2), $e_{\rm max}=14$, $E_{\rm 3max}=24$)
- Typical limit of the three-body Talmi-Moshinsky transformation is $E_{\rm 3max}=16$, recently extended to $E_{\rm 3max}=28$ for normal-ordered two-body approximation

## Context

This work extends the ecosystem of open-source nuclear-structure codes, complementing the computational environment for nuclear structure (CENS) and the NuclearToolkit code (Yoshida, 2022). The code enables researchers to generate matrix elements for chiral EFT interactions including the Entem-Machleidt-Nosyk N$^{4}$LO (Entem, 2017), N$^{2}$LO$_{\rm opt}$ (Ekstrom, 2013), N$^{2}$LO$_{\rm sat}$ (Ekstrom, 2015), and Delta-full EFT series from the Gothenburg-Oak Ridge collaboration (Jiang, 2020), facilitating benchmark calculations with NCSM and IMSRG methods for doubly closed shell nuclei up to $^{132}$Sn.
