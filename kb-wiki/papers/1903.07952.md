# Proton scalar dipole polarizabilities from real Compton scattering data, using fixed-t subtracted dispersion relations and the bootstrap method

> We perform a fit of the real Compton scattering (RCS) data below pion-production threshold to extract the electric ($α_{E1}$) and magnetic ($β_{M1}$) static scalar dipole polarizabilities of the proton, using fixed-$t$ subtracted dispersion relations and a bootstrap-based fitting technique. The bootstrap method provides a convenient tool to include the effects of the systematic errors on the best values of $α_{E1}$ and $β_{M1}$ and to propagate the statistical errors of the model parameters fixed by other measurements. We also implement various statistical tests to investigate the consistency of the available RCS data sets below pion-production threshold and we conclude that there are not strong motivations to exclude any data point from the global set. Our analysis yields $α_{E1} = (12.03^{+0.48}_{-0.54})\times 10^{-4} \text{fm}^3$ and $β_{M1} = (1.77^{+0.52}_{-0.54})\times 10^{-4} \text{fm}^3$, with p-value $= 12\%$.

## Key claim

Using fixed-t subtracted dispersion relations and a bootstrap-based fitting technique, the authors extract the proton's electric and magnetic static scalar dipole polarizabilities from real Compton scattering data below pion-production threshold, concluding that there are not strong motivations to exclude any data point from the global set, with the analysis yielding a p-value of 12%.

## Method

The authors performed fits of unpolarized real Compton scattering cross section data below pion-production threshold using fixed-t subtracted dispersion relations as the theoretical framework. They employed a parametric bootstrap technique with 10000 Monte Carlo replicas of experimental data to include systematic errors and propagate statistical errors from model parameters fixed by other measurements, such as spin polarizabilities from double polarization RCS and GDH experiments. Various fit configurations were tested, including fits with and without the Baldin sum rule constraint, with and without systematic errors, and with the backward spin polarizability as an additional free parameter.

## Key numbers

- Electric polarizability: alpha_E1 = (12.03^{+0.48}_{-0.54}) x 10^{-4} fm^3 (from Fit 1' with FULL data set, including systematic errors and Baldin sum rule)
- Magnetic polarizability: beta_M1 = (1.77^{+0.52}_{-0.54}) x 10^{-4} fm^3 (from Fit 1' with FULL data set, including systematic errors and Baldin sum rule)
- Baldin sum rule value used as constraint: alpha_E1 + beta_M1 = 13.8 +/- 0.4 (weighted average from Hagelstein (2015))
- Backward spin polarizability from Fit 3' (FULL data set): gamma_pi = 8.59^{+2.85}_{-2.94} x 10^{-4} fm^4 (with pi0-pole contribution excluded)
- Total backward spin polarizability from Fit 3' (FULL): gamma_pi_tot = -38.11^{+2.85}_{-2.94} x 10^{-4} fm^4 (including pi0-pole contribution of -46.7)
- p-value for Fit 1' (FULL data set): 12%
- chi-squared per degree of freedom for Fit 1' (FULL data set): 1.25

## Context

This work extends previous extractions of proton scalar dipole polarizabilities using fixed-t dispersion relations by Drechsel (1999), Pasquini (2007), and others, as well as chiral perturbation theory analyses by Bernard (1995), McGovern (2012), and Lensky (2015). The bootstrap method was previously applied to extract energy-dependent dynamical polarizabilities by Pasquini (2017), and this work applies it to static polarizabilities while systematically addressing data set consistency questions raised by Krupina (2017) and Griesshammer (2012). The results enable improved comparisons with chiral perturbation theory predictions and provide input for calculations of the proton's structure-dependent contributions to precision measurements.
