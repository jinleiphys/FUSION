# Solution of the Skyrme HF+BCS equation on a 3D mesh. II. A new version of the Ev8 code

> We describe a new version of the EV8 code that solves the nuclear Skyrme-Hartree-Fock+BCS problem using a 3-dimensional cartesian mesh. Several new features have been implemented with respect to the earlier version published in 2005. In particular, the numerical accuracy has been improved for a given mesh size by (i) implementing a new solver to determine the Coulomb potential for protons (ii) implementing a more precise method to calculate the derivatives on a mesh that had already been implemented earlier in our beyond-mean-field codes. The code has been made very flexible to enable the use of a large variety of Skyrme energy density functionals that have been introduced in the last years. Finally, the treatment of the constraints that can be introduced in the mean-field equations has been improved. The code Ev8 is today the tool of choice to study the variation of the energy of a nucleus from its ground state to very elongated or triaxial deformations with a well-controlled accuracy.

## Key claim

The paper presents a significantly updated version of the Ev8 code that solves the Skyrme-Hartree-Fock+BCS problem on a 3D Cartesian mesh, claiming improved numerical accuracy for a given mesh size through a new Coulomb solver and more precise derivative calculation method, while also enabling the use of a wider variety of modern Skyrme energy density functionals.

## Method

The code discretizes single-particle wave functions on a 3D Cartesian mesh and solves the nonlinear mean-field equations using the imaginary time step method, with time-reversal invariance and spatial symmetries that allow restriction to one octant of the box. The total energy is separated into kinetic, particle-hole (Skyrme and Coulomb), and pairing contributions, with the single-particle Hamiltonian derived from functional derivatives of the energy density with respect to local densities. The BCS equations determine occupation numbers from single-particle energies and pairing gaps, with the two systems coupled by self-consistency.

## Key numbers

- RAM requirement: on the order of 64 megabytes for the examples provided
- Running time: a few minutes for the examples provided, which concern rather heavy nuclei in modest boxes with an initial guess of Nilsson wavefunctions
- The code can handle two density-dependent terms with different exponents, as used in parametrizations of Cochet (2004) and Lesinski (2006)

## Context

This work extends the Ev8 code first published by Bonche, Flocard, and Heenen (2005) and the original version from 1985, incorporating improvements already implemented in the group's beyond-mean-field codes for angular-momentum projection and configuration mixing. The code enables studies of nuclear binding energy evolution from ground states to very elongated or triaxial deformations with controlled accuracy, and serves as a starting point for beyond-mean-field calculations using the Generator Coordinate Method as described by Bender and Heenen (2008).
