# Bound state equation for the Nakanishi weight function

> The bound state Bethe-Salpeter amplitude was expressed by Nakanishi using a two-dimensional integral representation, in terms of a smooth weight function $g$, which carries the detailed dynamical information. A similar, but one-dimensional, integral representation can be obtained for the Light-Front wave function in terms of the same weight function $g$. By using the generalized Stieltjes transform, we first obtain $g$ in terms of the Light-Front wave function in the complex plane of its arguments. Next, a new integral equation for the Nakanishi weight function $g$ is derived for a bound state case. It has the standard form $g= N g$, where $N$ is a two-dimensional integral operator. We give the prescription for obtaining the kernel $ N$ starting with the kernel $K$ of the Bethe-Salpeter equation. The derivation is valid for any kernel given by an irreducible Feynman amplitude.

## Key claim

The paper derives a new integral equation for the Nakanishi weight function \(g\) in the canonical form \(g = N g\) that is valid for any kernel given by an irreducible Feynman amplitude, not only for the ladder kernel as in prior work by Frederico et al. (2014). The derivation uses an analytic inversion of the Stieltjes transform rather than relying on the uniqueness of the Nakanishi representation.

## Method

The authors first analytically invert the Stieltjes transform of order \(\rho=2\) to obtain an explicit expression for the inverse operator \(\hat{L}^{-1}\) acting on functions of the form \(f(\gamma) = \int_0^\infty g(\gamma')/(\gamma'+\gamma+b)^2 d\gamma'\). They apply this inverse to the previously established equation \(\hat{L} g = \hat{V} g\) (Eq. 3 in the paper) to obtain \(g = \hat{L}^{-1}\hat{V} g \equiv \hat{N} g\), where the kernel \(N\) is given by a one-dimensional complex contour integral of the kernel \(V\). The kernel \(V\) itself is derived from the original Bethe-Salpeter kernel \(K\) via the prescription of Carbonell et al. (2009).

## Key numbers

* For the one-boson-exchange ladder kernel with parameters \(\alpha=1\), \(m=1\), \(\mu=0.15\), \(M=1.9\), and variables \(\gamma=0.1\), \(\gamma'=1\), \(z=0.2\), \(z'=0.35\) (where \(\eta=-1.24<0\)), the kernel \(N\) computed via the new formula equals \(0.1153980591510\), matching the kernel \(\mathcal{V}\) from Frederico et al. (2014) to within \(0.1153980591509\) (12-digit agreement).
* For the same parameters but \(\gamma=0.5\), \(\gamma'=0.1\), \(z=0.2\), \(z'=0.35\) (where \(\eta=0.27>0\)), the kernel \(N\) equals \(-0.0531890858160\), matching \(\mathcal{V}\) to within \(-0.0531890858158\) (12-digit agreement).
* The binding energy \(B = 2m - M\) is implicitly determined by the equation; the example uses \(M=1.9\) with \(m=1\), giving \(B=0.1\) (in the units where \(m=1\)).

## Context

This work extends the method of Frederico et al. (2014), which derived a canonical equation for the Nakanishi weight function only for the ladder kernel using the uniqueness ansatz. The new derivation applies to any kernel given by irreducible Feynman amplitudes, including cross-ladder contributions previously treated by Carbonell et al. (2009, 2013) and Frederico et al. (2014). The result enables solving the Bethe-Salpeter equation for more general interactions in Minkowski space and provides a formal link between the Light-Front wave function and the Bethe-Salpeter amplitude via the Nakanishi weight function. The analytic inversion technique may also be useful in other contexts where integral transforms are used, as pioneered by Efros (1985, 2007).
