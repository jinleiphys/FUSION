# Hybrids with heavy quarks: from potential to string

> We consider hybrid states with heavy quarks in the frame of recently proposed constituent gluon model. Limiting regimes for model Hamiltonian are discussed. It is shown that these regimes match smoothly, hence predictions of the model are definite enough. We predict lowest $c \bar c g$ hybrids at $4.1 \pm 0.1$ GeV and $b \bar b g$ hybrids at $10.5 \pm 0.1$ GeV. Regge-trajectories for hybrids are derived from the formalism. Experimental aspects of our predictions are discussed.

# Hybrids with heavy quarks: from potential to string

## Key claim
The constituent gluon model predicts that potential and string regimes for the QCD string in heavy hybrids match smoothly within about 50 MeV, making the model's predictions definite enough despite the approximations involved. The lowest hybrid masses are predicted at 4.1 ± 0.1 GeV for c-cbar-g and 10.5 ± 0.1 GeV for b-bbar-g, but the string regime results are strictly valid only for b-bbar-g hybrids where the condition 1 << L_gl << m/sqrt(sigma) is satisfied.

## Method
The author constructs a Hamiltonian for the gluonic subsystem of a heavy quark-antiquark-gluon hybrid using the constituent gluon model based on Vacuum Background Correlators and the QCD string model, with assumptions including quenched approximation, area law for Wilson loops, and S-wave for quarks. The Hamiltonian is analyzed in two limiting regimes: a potential regime (zero gluon angular momentum) where the gluon behaves as a massless particle confined by a scalar potential, and a string regime (large gluon angular momentum) where the string energy follows Regge trajectories. Variational calculations and extremal value methods are used to compute gluon subsystem energies, and the hybrid mass is obtained by combining these with heavy-light meson masses via an additive constant procedure.

## Key numbers
- Lowest c-cbar-g hybrid mass: 4.1 ± 0.1 GeV (from Table I, with quark mass 1.2–1.7 GeV, sigma = 0.18 GeV^2, alpha_s = 0.3)
- Lowest b-bbar-g hybrid mass: 10.5 ± 0.1 GeV
- Gluon subsystem energy from potential regime: 4.36 sqrt(sigma) ≈ 1.7 GeV
- Gluon subsystem energy from string regime (including first correction): 4.51 sqrt(sigma) ≈ 1.8 GeV
- String regime leading-order energy for L_gl = 1: 2.98 sqrt(sigma) ≈ 1.2 GeV
- First correction to string regime: 1.53 sqrt(sigma)
- Slope of Regge trajectories for hybrids: 2 pi sigma
- Quark correction to string regime for L_gl = 1: about 100 MeV
- Corrections from rho^2/r^2 terms: about 400 MeV for charmonium hybrids, about 200 MeV for bottomonium hybrids

## Context
This work extends the constituent gluon model developed in prior studies (Simonov 1995, Yufryakov 1995) and the QCD string formalism (Simonov 1988, 1991) to heavy quark hybrids, providing a unified treatment of potential and string regimes. The predictions for heavy hybrid masses coincide with the flux-tube model (Isgur and Paton 1985) and lattice calculations, though the author argues the flux-tube model's small-oscillation approximation is doubtful. The results enable experimental searches for hybrids in e+e- collisions, p-pbar annihilation, and highly excited quarkonium decays, with the expectation that these states should be narrow due to phase space suppression near open charm and bottom thresholds.
