# On the resonance spectra of particle-unstable light nuclei with a Sturmian approach that preserves the Pauli principle

> The fundamental ingredients of the MCAS (multi-channel algebraic scattering) method are discussed. The main feature, namely the application of the sturmian theory for nucleon-nucleus scattering, allows solution of the scattering problem given the phenomenological ingredients necessary for the description of weakly-bound (or particle-unstable) light nuclear systems. Currently, to describe these systems, we use a macroscopic, collective model. Analyses show that the couplings to low-energy collective-core excitations are fundamental but they are physically meaningful only if the constraints introduced by the Pauli principle are taken into account. For this we introduce in the nucleon-nucleus system the Orthogonalizing Pseudo-Potential formalism, extended to collective excitations of the core. The formalism leads one to discuss a new concept, Pauli hindrance, which appears to be important especially to understand the structure of weakly-bound and unbound systems.

## Key claim

The MCAS (multi-channel algebraic scattering) method, using Sturmian theory and the Orthogonalizing Pseudo-Potential (OPP) formalism extended to collective core excitations, can describe the resonance spectra of weakly-bound and particle-unstable light nuclei, but only if the Pauli principle constraints are properly enforced; the paper introduces the new concept of "Pauli hindrance" for states that are neither fully forbidden nor fully allowed, which appears important for understanding weakly-bound and unbound systems.

## Method

The authors solve the nucleon-nucleus scattering problem using Sturmian (Weinberg) functions, which allow algebraic determination of S-matrices, bound states, and resonances via matrix manipulation. They employ a macroscopic collective model with a coupled-channel potential expanded over central, spin-orbit, orbit-orbit, and spin-spin operators, including couplings to low-lying collective core excitations (quadrupole, octupole). The Pauli principle is enforced by adding an Orthogonalizing Pseudo-Potential (OPP) term to the Hamiltonian, which eliminates deeply-bound spurious states by making them forbidden in the limit of large lambda parameter; for weakly-bound or unbound systems, intermediate lambda values define "Pauli hindered" states.

## Key numbers

- For n-12C scattering, the OPP lambda parameter was initially set to 100 MeV, but values up to 1000 MeV were needed to stabilize resonance positions; a narrow 5/2- resonance converged to a position around 0.1 MeV in agreement with evaluated data only at higher lambda values.
- For the n-14C/p-14O system (15C/15F), the coupled-channel potential parameters were: V0 = -45.0 MeV, Vll = 0.42 MeV, Vls = 7.0 MeV, R0 = 3.1 fm, a0 = 0.65 fm, beta2 = -0.50.
- For the mass-7 systems (7Li, 7He, 7Be, 7B), the nucleon-nucleus(mass-6) potential parameters were: V0 = -36.817 MeV, Vll = -1.2346 MeV, Vls = 14.9618 MeV, VIs = 0.8511 MeV, R0 = 2.8 fm, a = 0.88917 fm, beta2 = 0.7298.
- For 7He, the ground state 3/2- resonance is at 0.445 MeV (experimental) vs 0.43 MeV (theory), both with width 0.15 and 0.1 MeV respectively.
- For 7B, the ground state 3/2- resonance is at 2.21 MeV (experimental) vs 2.10 MeV (theory), with widths 1.4 and 0.19 MeV respectively.
- For the Pauli hindrance in mass-7 systems, the lambda parameters were: lambda(0p3/2[0+1;2+1;2+2]) = 17.6 MeV, lambda(0p1/2[0+1]) = 36.0 MeV, lambda(0p1/2[2+1;2+2]) = 5.6 MeV.

## Context

This work extends the MCAS method previously applied to n-12C scattering (Amos, Canton, Pisent) by introducing the OPP formalism for collective excitations and the concept of Pauli hindrance. It builds on the Orthogonality Condition Model (Saito) and earlier OPP approaches (Kukulin), as well as supersymmetric quantum mechanics transformations (Baye) used to remove spurious states. The approach enables predictions for unbound nuclei like 15F and 7B, and provides an alternative to cluster models (e.g., 7Li as 3H+alpha dicluster) and microscopic models (no-core shell model, Green's function Monte Carlo) for mass-7 systems, with comparisons given in Canton (2006a).
