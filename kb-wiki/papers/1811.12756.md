# Determination of hexadecapole ($β_{4}$) deformation of the light-mass nucleus $^{24}$Mg using quasi-elastic measurement

> Quasi-elastic scattering measurements have been performed using $^{16}$O and $^{24}$Mg projectiles off $^{90}$Zr at energies around the Coulomb barrier. Experimental data have been analyzed in the framework of coupled channels (CC) calculations using the code CCFULL. The quasi-elastic scattering excitation function and derived barrier distribution for $^{16}$O + $^{90}$Zr reaction are well reproduced by the CC calculations using the vibrational coupling strengths for $^{90}$Zr reported in the literature. Using these vibrational coupling strengths, a Bayesian analysis is carried out for $^{24}$Mg + $^{90}$Zr reaction. The $β_{2}$ and $β_{4}$ values for $^{24}$Mg are determined to be $+0.43 \pm 0.02$ and $ - 0.11 \pm 0.02$, respectively. The $β_{2}$ parameter determined in the present work is in good agreement with results obtained using inelastic scattering probes. The hexadecapole deformation of $^{24}$Mg has been measured very precisely for the first time. Present results establish that quasi-elastic scattering could provide a useful probe to determine the ground state deformation of atomic nuclei.

## Key claim

The hexadecapole deformation parameter β₄ of ²⁴Mg has been determined with a 95% confidence limit for the first time using quasi-elastic scattering, establishing that this technique can serve as a probe for ground-state deformation of exotic nuclei with low-intensity radioactive beams, though the method relies on the assumption that transfer channel coupling strengths are weak.

## Method

Quasi-elastic scattering excitation functions and derived barrier distributions were measured for ¹⁶O + ⁹⁰Zr and ²⁴Mg + ⁹⁰Zr reactions at backward angles (136.9°, 147.3°, and 158.0°) using silicon-surface barrier telescopes at beam energies of 36–62 MeV (¹⁶O) and 61–93 MeV (²⁴Mg). Coupled channels calculations with the code CCFULL were performed, first using vibrational coupling strengths for ⁹⁰Zr (2⁺ and 3⁻ states) determined from the ¹⁶O + ⁹⁰Zr data, then including rotational couplings of ²⁴Mg as a rigid rotor. A Bayesian analysis with Markov-Chain Monte Carlo was carried out in the two-dimensional parameter space of β₂ and β₄ to extract the deformation parameters and their uncertainties.

## Key numbers

- β₂ for ²⁴Mg = +0.43 ± 0.02 (95% confidence interval)
- β₄ for ²⁴Mg = −0.11 ± 0.02 (95% confidence interval)
- β₂ and β₄ are moderately anticorrelated with a correlation of approximately −0.298
- Beam energies used: 36 to 62 MeV for ¹⁶O (1 MeV steps) and 61 to 93 MeV for ²⁴Mg (2 MeV steps)
- Vibrational coupling strengths for ⁹⁰Zr: β₂ = 0.089 (2⁺ state at 2.19 MeV) and β₃ = 0.211 (3⁻ state at 2.75 MeV)
- χ² minimization performed over effective energy range 50 to 73 MeV

## Context

This work extends the application of quasi-elastic barrier distributions, previously demonstrated only in the heavy-mass region of rare earths by Jia (2014), to a light-mass nucleus in the 2s-1d shell. Prior determinations of β₄ for ²⁴Mg using electron scattering (Horikawa 1971, Akira 1972), proton scattering (Swiniarski 1969, Leo 1979), neutron scattering (Haouat 1984), ³He scattering (Peterson 1978), and α scattering (Rebel 1972, Harakeh 1979) showed large variations and uncertainties, with many values lacking quoted errors. The present result provides the first precise β₄ measurement with a 95% confidence limit and enables future determination of ground-state deformations of exotic nuclei using low-intensity radioactive ion beams, where fusion cross-section measurements would be difficult.
