# An analysis of the impact of LHC Run I proton-lead data on nuclear parton densities

> We report on an analysis of the impact of available experimental data on hard processes in proton-lead collisions during Run I at the Large Hadron Collider on nuclear modifications of parton distribution functions. Our analysis is restricted to the EPS09 and DSSZ global fits. The measurements that we consider comprise production of massive gauge bosons, jets, charged hadrons and pions. This is the first time a study of nuclear PDFs includes this number of different observables. The goal of the paper is twofold: i) checking the description of the data by nPDFs, as well as the relevance of these nuclear effects, in a quantitative manner; ii) testing the constraining power of these data in eventual global fits, for which we use the Bayesian reweighting technique. We find an overall good, even too good, description of the data, indicating that more constraining power would require a better control over the systematic uncertainties and/or the proper proton-proton reference from LHC Run II. Some of the observables, however, show sizable tension with specific choices of proton and nuclear PDFs. We also comment on the corresponding improvements on the theoretical treatment.

# An analysis of the impact of LHC Run I proton-lead data on nuclear parton densities

## Key claim

The LHC Run I proton-lead data on hard processes (gauge bosons, jets, charged hadrons, pions) are described overall well, even too well, by existing nuclear parton distribution functions (EPS09 and DSSZ), meaning that more constraining power would require better control over systematic uncertainties and/or a proper proton-proton reference from LHC Run II; however, some observables show sizable tension with specific choices of proton and nuclear PDFs.

## Method

The authors used Bayesian reweighting technique to quantitatively estimate the constraining power of 165 data points from LHC Run I p-Pb collisions at sqrt(s)=5.02 TeV per nucleon, restricting the analysis to the EPS09 and DSSZ nuclear PDF global fits combined with two sets of free proton PDFs (MSTW2008 and CT10). They computed theoretical predictions at next-to-leading order using MCFM for electroweak bosons and other appropriate generators for jets, charged hadrons, and pions, constructing forward-to-backward ratios of cross sections to minimize dependence on free proton PDFs and avoid normalization issues from constructed p-p references.

## Key numbers

- 165 total data points from LHC Run I p-Pb collisions at sqrt(s)=5.02 TeV per nucleon were analyzed
- For CMS W- forward-to-backward ratios (5 data points), chi-squared values were 4.055 (CT10+EPS09) and 4.140 (CT10+DSSZ) versus 19.227 (CT10 only, no nuclear effects), showing clear preference for nuclear modifications
- For CMS Z boson forward-to-backward ratios (5 data points), chi-squared values were 4.182 (CT10+EPS09) and 3.385 (CT10+DSSZ) versus 7.336 (CT10 only)
- For ATLAS Z boson forward-to-backward ratios (7 data points), chi-squared values were 9.815 (CT10+EPS09) and 11.465 (CT10+DSSZ) versus 25.177 (CT10 only)
- 10,000 replicas (N_rep) were generated for the Bayesian reweighting procedure
- EPS09 has 31 eigenvector members (2N_eig+1), DSSZ has 51 eigenvector members

## Context

This work extends the pseudodata study of Armesto et al. (2013) by using actual LHC p-Pb measurements instead of pseudodata, and applies the improved Bayesian reweighting method developed by Paukkunen (2014). The analysis checks the universality conjecture of nuclear PDFs against the first LHC p-Pb data, following the deuteron-gold program at RHIC. The results indicate that while current nPDFs describe the data well, the lack of correlated systematic uncertainty information in the experimental publications undermines the constraining power; the authors call for publishing correlation information as is standard in p-p and p-antiproton collisions. The upcoming LHC Run II p-p reference at 5.02 TeV is expected to improve the situation.
