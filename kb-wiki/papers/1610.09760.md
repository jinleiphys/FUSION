# Proton radius from electron-proton scattering and chiral perturbation theory

> We determine the root-mean-square proton charge radius, $R_{\rm p}$, from a fit to low-$Q^2$ electron-proton elastic scattering cross section data with the higher moments fixed (within uncertainties) to the values predicted by chiral perturbation theory. We obtain $R_{\rm p}=0.857(11)$ fm. This number falls between the value obtained from muonic hydrogen analyses and the CODATA value (based upon atomic hydrogen spectroscopy and electron-proton scattering determinations).

# Proton radius from electron-proton scattering and chiral perturbation theory

## Key claim
The root-mean-square proton charge radius is determined to be 0.855(11) fm by fitting low-Q² electron-proton elastic scattering data while fixing the higher-order electric and magnetic moments to values predicted by chiral perturbation theory, rather than allowing them to float freely. This result falls between the muonic hydrogen value (approximately 0.841 fm) and the CODATA value (approximately 0.875 fm), but with an uncertainty an order of magnitude larger than the muonic hydrogen determination.

## Method
The authors fit the low-Q² MAMI electron-proton elastic scattering cross section data (up to Q²_max = 0.023 GeV²) using Taylor expansions of the Sachs form factors G_E(Q²) and G_M(Q²). The higher-order moments (⟨r⁴⟩_E, ⟨r⁶⟩_E, ⟨r⁸⟩_E, ⟨r²⟩_M, ⟨r⁴⟩_M, ⟨r⁶⟩_M) are fixed to the values predicted by chiral perturbation theory (including Δ-particle effects) within their uncertainties, while only R_p = √⟨r²⟩_E is allowed to float in the least-squares fit. The Feshbach two-photon-exchange correction is replaced with a more complete calculation, and separate normalization constants for each data group are floated.

## Key numbers
- R_p = 0.855(11) fm: final determination from averaging fits at Q²_max = 0.012 GeV² and Q²_max = 0.023 GeV²
- At Q²_max = 0.012 GeV² (118 cross sections): R_p = 0.8538(117) fm, with fit uncertainty 0.0104 fm and moment uncertainties: M2 = 0.0042 fm, E4 = 0.0035 fm, M4 = 0.0001 fm, E6 = 0.0002 fm
- At Q²_max = 0.023 GeV² (270 cross sections): R_p = 0.8566(112) fm, with fit uncertainty 0.0053 fm and moment uncertainties: M2 = 0.0078 fm, E4 = 0.0059 fm, M4 = 0.0004 fm, E6 = 0.0006 fm
- Chiral perturbation theory predictions for moments (π&Δ row): ⟨r⁴⟩_E = 0.60(29) fm⁴, ⟨r⁶⟩_E = 5.0(2.0) fm⁶, ⟨r⁸⟩_E = 99(37) fm⁸, ⟨r²⟩_M = 0.44(16) fm², ⟨r⁴⟩_M = 0.79(28) fm⁴, ⟨r⁶⟩_M = 6.9(2.4) fm⁶
- The charged-pion production threshold at Q² = -0.078 GeV² sets the upper limit for the Taylor expansion validity

## Context
This work extends the approach of Horbatsch (2015), which showed that R_p values from 0.84 to 0.89 fm were possible from MAMI data depending on the functional form used for extrapolation. By using chiral perturbation theory predictions for higher moments (following the framework of Bernard (1998) and Peset (2014)), the authors provide external constraints that reduce the model dependence in the electron-proton scattering extraction. The result is compared to the muonic hydrogen determinations of Pohl (2010) and Antognini (2013) (0.8409(4) fm) and the chiral-EFT-motivated muonic hydrogen value of Peset (2015) (0.8413(15) fm), as well as CODATA 2010 (0.8775(51) fm) and CODATA 2014 (0.8751(61) fm). The method is directly applicable to ongoing and planned very-low-Q² elastic scattering measurements.
