# Dynamic critical phenomena from spectral functions on the lattice

> We investigate spectral functions in the vicinity of the critical temperature of a second-order phase transition. Since critical phenomena in quantum field theories are governed by classical dynamics, universal properties can be computed using real-time lattice simulations. For the example of a relativistic single-component scalar field theory in 2+1 dimensions, we compute the spectral function described by universal scaling functions and extract the dynamic critical exponent z. Together with exactly known static properties of this theory, we obtain a verification from first principles that the relativistic theory is well described by the dynamic universality class of relaxational models with conserved density (Model C).

## Key claim

The authors verify from first principles that a relativistic single-component scalar field theory in 2+1 dimensions belongs to the dynamic universality class of relaxational models with conserved density (Model C), but note that the present statistics is chosen to be just able to distinguish between Model C (z=2) and Model A (z≈2.16) for two spatial dimensions.

## Method

The authors use classical-statistical real-time lattice simulations of a relativistic φ⁴ theory in 2+1 dimensions, exploiting the fact that critical phenomena at non-zero temperature are governed by classical dynamics. They generate thermal initial conditions via a Metropolis algorithm, evolve the fields using Hamiltonian dynamics with a leap-frog discretization, and compute the spectral function from the statistical two-point function using the classical fluctuation-dissipation relation ρ_cl(t,p,T) = -(1/T) ∂_t F_cl(t,p,T). The dynamic critical exponent z is extracted from power-law behavior of the spectral function at critical temperature, from scaling collapse of data at different reduced temperatures, and from the temperature dependence of the temporal correlation length.

## Key numbers

- Critical temperature: T_c/|m| = 4.444 ± 0.005 (on N=1024 lattice)
- Dynamic critical exponent: z = 2.0 ± 0.1 (from power-law fit at T_c with η=1/4)
- Dynamic critical exponent from scaling function fit: z = 2.05 ± 0.15
- At T = 2 T_c: quasi-particle peak position ω₀/T_c ≃ 0.12, width γ₀/T_c = 0.036
- Power-law exponent at T_c: (2-η)/z ≃ 0.87, leading to z = 2.0 ± 0.1
- Temporal power-law exponent from time-domain data: 1-(2-η)/z ≃ 0.10–0.11
- Static exponents used: ν = 1, η = 1/4, β = 1/8, δ = 15 (exactly known for 2D Ising universality class)
- Lattice sizes: N=1024, 256, 128; lattice spacing a with m²a² = -1, λa = 1, a_t/a = 0.1
- Number of thermalized configurations per temperature: 30–160

## Context

This work extends earlier suggestions to compute spectral functions from classical field averages using the fluctuation-dissipation relation (Aarts 2001) and non-equilibrium studies of systems near second-order phase transitions (Borsanyi 2001, Rajantie 1999). The results confirm that the relativistic scalar theory falls into Model C in the Halperin-Hohenberg classification scheme (1977), where the dynamic exponent z = 2 + α/ν with α=0 in d=2. This enables first-principles computation of transport properties near the QCD critical point and provides a benchmark for effective descriptions of spectral functions in the temperature range where a twin-peak structure appears from the interplay between propagating relativistic modes and diffusive processes.
