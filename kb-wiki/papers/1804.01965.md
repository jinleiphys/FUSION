# First Monte Carlo global QCD analysis of pion parton distributions

> We perform the first global QCD analysis of parton distribution functions (PDFs) in the pion, combining $πA$ Drell-Yan data with leading neutron electroproduction from HERA within a Monte Carlo approach based on nested sampling. Inclusion of the HERA data allows the pion PDFs to be determined down to much lower values of $x$, with relatively weak model dependence from uncertainties in the chiral splitting function. The combined analysis reveals that gluons carry a significantly higher pion momentum fraction, $\sim 30\%$, than that inferred from Drell-Yan data alone, with sea quarks carrying a somewhat smaller fraction, $\sim 15\%$, at the input scale. Within the same effective theory framework, the chiral splitting function and pion PDFs can be used to describe the $\bar d-\bar u$ asymmetry in the proton.

## Key claim
The first global QCD analysis of pion parton distribution functions (PDFs) that simultaneously fits pion-nucleus Drell-Yan data and leading neutron electroproduction data from HERA, using a Monte Carlo approach based on nested sampling, finds that gluons carry approximately 30% of the pion momentum at the input scale, about three times larger than inferred from Drell-Yan data alone, but this result depends on the model chosen for the chiral splitting function, with the quoted value coming from the best-fit model (an s-dependent exponential regulator).

## Method
The analysis performs a next-to-leading order (NLO) global QCD fit combining 142 Drell-Yan data points from CERN NA10 and Fermilab E615 experiments on tungsten targets with 108 leading neutron electroproduction data points from H1 and ZEUS at HERA. Pion valence, sea quark, and gluon PDFs are parameterized at the input scale Q0² = (1.3 GeV)² using a three-parameter form, and the fit uses a Bayesian Monte Carlo method with nested sampling to explore parameter space and quantify uncertainties. The chiral splitting function for the proton-to-neutron-pion transition is modeled with five different regulator forms, and the minimum xL cut for leading neutron data is fitted rather than assumed.

## Key numbers
- Combined χ²/Ndat = 0.98 (244.8/250) for the best fit (model i, s-dependent exponential regulator with cutoff Λ = 1.31 ± 0.04 GeV)
- Momentum fractions at input scale Q² = mc² for full DY+LN fit: valence = 54(1)%, sea = 16(2)%, gluons = 30(2)%
- Momentum fractions at Q² = 5 GeV² for full DY+LN fit: valence = 48(1)%, sea = 17(1)%, gluons = 35(2)%
- DY-only fit momentum fractions at input scale: valence = 60(1)%, sea = 30(5)%, gluons = 10(5)%
- Cutoff parameters for other regulator models: Λ = 0.58(2) GeV (model ii), 0.52(2) GeV (model iii), 0.78(5) GeV (model iv), 0.25(1) GeV (model v)
- Combined fit including E866 data: χ²/Ndat = 1.03 (272.4/265), cutoff Λ = 1.35(2) GeV for model (i)

## Context
This work extends previous pion PDF analyses that used only Drell-Yan data (Owens 1984, SMRS, GRS, Wijesooriya 2005, Aicher 2010) by incorporating leading neutron electroproduction data from HERA for the first time in a global QCD fit, following the initial steps by McKenney et al. (2015) who studied the model dependence of the chiral splitting function. The results enable more precise predictions for future experiments including COMPASS, the Tagged DIS experiment at Jefferson Lab, and the SeaQuest experiment, and provide a framework for simultaneous fits of pion, proton, and nuclear PDFs within the same Monte Carlo methodology.
