# Weak reactions with light nuclei - 6He beta-decay as a test case for the nuclear weak current

> We present a microscopic calculation of the 6He beta-decay into the ground state of 6Li. To this end we use the impulse approximation to describe the nuclear weak current. The ground state wave functions are obtained from the solution of the nuclear 6-body problem. The nucleonnucleon interaction is described via the J-matrix inverse scattering potential (JISP), and the nuclear problem is solved using the hyperspherical-harmonics approach. This approach results in numerical accuracy of about 2 per mil in the transition matrix element. Bearing in mind that the contribution of meson-exchange currents to the transition matrix element is about 5%, these results pave the way for accurate estimation of their effect.

## Key claim

The authors achieve a numerical accuracy of about 2 per mil in the Gamow-Teller transition matrix element for 6He beta-decay using the impulse approximation, which is sufficient to disentangle numerical uncertainties from the physics of meson-exchange currents, whose contribution to the matrix element is about 5%.

## Method

The authors solve the nuclear 6-body problem using the hyperspherical-harmonics expansion to obtain ground state wave functions for 6He and 6Li. The nucleon-nucleon interaction is described by the J-matrix inverse scattering potential (JISP16), a soft potential that reproduces NN phase shifts up to pion threshold and binding energies of light nuclei with A ≤ 4. The nuclear weak current is treated in the impulse approximation, meaning only one-body (Fermi and Gamow-Teller) operators are included, and the Gamow-Teller matrix element is extrapolated to the infinite basis limit using the expression GT(K_max) = GT_∞ + B/K_max^β for K_max ≥ 6.

## Key numbers

- Extrapolated Gamow-Teller matrix element in the impulse approximation: GT_∞ = 2.228(3), with numerical accuracy at the few per mil level.
- Experimental Gamow-Teller matrix element for 6He beta-decay: 2.170 (from the table, compared to 2.173 ± 0.002 stated in the text).
- Extrapolated binding energy of 6He: 29.041 MeV (experimental value: 29.269 MeV).
- Extrapolated binding energy of 6Li: 31.923 MeV (experimental value: 31.995 MeV).
- Extrapolated rms matter radius of 6He: 2.165 fm (experimental value: 2.18 fm).
- Half-life of 6He: τ_{1/2} = 806.7 ± 1.5 msec.

## Context

This work extends prior microscopic calculations of 6He beta-decay by Schiavilla and Wiringa (2002), who used the Argonne v18 potential with Urbana-IX three-nucleon force and variational Monte Carlo wave functions, finding that two-body currents increased the Gamow-Teller matrix element by only 1.7% and over-predicted the measured half-life by about 8%. The present calculation achieves higher numerical precision using the hyperspherical-harmonics method with the JISP16 potential, enabling a clean test of meson-exchange current models without the confounding factor of wave function inaccuracy. The results are consistent with other calculations using different potentials, such as those by Navratil and Ormand (2003) and Navratil and Caurier (2004).
