# Efficient computation of Hamiltonian matrix elements between non-orthogonal Slater determinants

> We present an efficient numerical method for computing Hamiltonian matrix elements between non-orthogonal Slater determinants, focusing on the most time-consuming component of the calculation that involves a sparse array. In the usual case where many matrix elements should be calculated, this computation can be transformed into a multiplication of dense matrices. It is demonstrated that the present method based on the matrix-matrix multiplication attains $\sim$80% of the theoretical peak performance measured on systems equipped with modern microprocessors, a factor of 5-10 better than the normal method using indirectly indexed arrays to treat a sparse array. The reason for such different performances is discussed from the viewpoint of memory access.

# Efficient computation of Hamiltonian matrix elements between non-orthogonal Slater determinants

## Key claim

The paper presents a numerical method that transforms the most time-consuming component of Hamiltonian matrix element calculations between non-orthogonal Slater determinants, a fourfold summation involving a sparse array, into dense matrix-matrix multiplication, achieving approximately 80% of theoretical peak performance on modern microprocessors, but only when many matrix elements are computed simultaneously; the method does not improve performance for single matrix element calculations.

## Method

The authors reformulate the two-body part of the Hamiltonian overlap ⟨V⟩ = Σ ρ₃₁ v̄₁₂₃₄ ρ₄₂ by grouping density matrix elements and two-body matrix elements according to the conserved z-component of angular momentum (Δm), transforming the sparse four-index array into block-antidiagonal dense submatrices. For repeated calculations with different density matrices, they bind multiple density vectors into a matrix and perform a single matrix-matrix multiplication (ṽθ) using optimized BLAS libraries, rather than repeated matrix-vector operations. The method is benchmarked on two processor architectures (Xeon X5570 and SPARC64 VII) using a nuclear system with 70 single-particle states and 31,250 repeated calculations for angular-momentum and parity projection.

## Key numbers

- The matrix-matrix method attains approximately 70-80% of theoretical peak performance on both Xeon X5570 (11.72 GFLOPS peak) and SPARC64 VII (10 GFLOPS peak) systems, compared to the indirect-index method which achieves much lower performance.
- The matrix-matrix method outperforms the indirect-index method by a factor of 5-10 in performance.
- The number of elementary floating-point operations is 20,992,518 for the indirect-index method and 10,365,224 for both the matrix-vector and matrix-matrix methods.
- The largest submatrix used in the benchmark system is of size 390 × 390, classified by z-component of angular momentum and parity.
- The benchmark calculation requires 25³ × 2 = 31,250 computations of the two-body part ⟨V⟩ for angular-momentum and parity projection.
- For the model space with 5 harmonic-oscillator major shells (N_shell=5), the number of single-particle states N_s is 70.
- For N_shell=10, the computational time is approximately 10³ times larger than for N_shell=5, and the largest matrix size is 12,444.
- In parallel processing with 32 MPI processes on a 4-node Xeon X5570 system, the matrix-matrix method with N_vec=100 reaches 8.5 GFLOPS/core, close to the single-process performance of 9.1 GFLOPS/core.

## Context

This work extends the applicability of methods using superpositions of non-orthogonal Slater determinants, which are widely used in nuclear many-body problems including the generator coordinate method (GCM) as described by Ring and Schuck (1980), symmetry restoration as reviewed by Bender, Heenen, and Reinhard (2003), and the Monte Carlo shell model (MCSM) as developed by the authors' group. The method enables large-scale calculations for density-functional theory and shell-model applications where thousands of Hamiltonian overlaps must be computed, particularly for angular-momentum projection requiring on the order of 10⁴ mesh points. The approach is also applicable to quasiparticle vacuum states used in Hartree-Fock-Bogoliubov theory for including pairing correlations.
