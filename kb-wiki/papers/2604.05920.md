# Reference Energies for Non-Relativistic Core Ionization Potentials

> Deep-lying core electrons carry highly localized, site-specific information that forms the basis of X-ray photoelectron spectroscopy. Accurately predicting their associated core ionization potentials (IPs) is a demanding theoretical task, requiring a balanced treatment of strong orbital relaxation, electron correlation, and relativistic effects. Over the years, a variety of approaches have been developed, ranging from state-specific wave function methods to linear-response formalisms and Green's function techniques. However, their assessment has often relied on comparisons with experiment, where multiple sources of error (basis set incompleteness, relativistic corrections, and vibrational effects) are entangled, making it difficult to isolate the performance of correlation treatments. In the present work, we establish a consistent, theory-based benchmark for core IPs by computing 84 non-relativistic values (73 second-row and 11 third-row IPs) at the full configuration interaction level within the core-valence separation approximation, using large correlation-consistent basis sets augmented with tight-core and diffuse functions (aug-cc-pCVXZ). These results define theoretical best estimates within a fixed finite basis set, providing a chemically accurate reference for method development and validation. Importantly, our dataset allows for systematic, theory-versus-theory comparisons that disentangle correlation and relaxation effects from other physical contributions. On this basis, we assess the performance of widely used approximate methods, including equation-of-motion coupled-cluster approaches up to the inclusion of quadruple excitations, the one-shot $G_0W_0$ scheme, as well as state-specific methods.

# Reference Energies for Non-Relativistic Core Ionization Potentials

## Key claim

The authors establish 84 non-relativistic core ionization potentials (IPs) computed at the full configuration interaction level within the core-valence separation approximation (CVS-FCI) as theoretical best estimates in fixed finite basis sets, explicitly noting that these values exclude relativistic, vibrational, and continuum effects and therefore are not directly comparable to experiment without additional corrections.

## Method

Core IPs were computed using selected configuration interaction (CIPSI algorithm) extrapolated to the FCI limit within the core-valence separation approximation, using aug-cc-pCVXZ basis sets (X = D, T, Q) with all electrons correlated. Ground-state calculations used neutral HF orbitals; core-ionized states were first optimized at the ROHF level using the maximum overlap method, then a CVS-SCI calculation was performed restricting determinants to those containing at least one hole in the targeted core orbital. Variational energies were extrapolated to the FCI limit using weighted linear fits over the last 3–6 CIPSI iterations.

## Key numbers

- 84 total non-relativistic core IPs: 73 second-row and 11 third-row values
- CVS-FCI reference values in ACVTZ basis: H2O oxygen 1s IP = 539.87 eV; CH4 carbon 1s IP = 290.85 eV; N2 nitrogen 1s IP = 409.88 eV; HF fluorine 1s IP = 694.10 eV
- Third-row examples in ACVTZ: HCl chlorine 1s IP = 2821.42 eV; H2S sulfur 1s IP = 2471.69 eV; PH3 phosphorus 1s IP = 2145.73 eV; SiH4 silicon 1s IP = 1843.30 eV
- CVS-IP-EOM-CCSDTQ in ACVTZ yields MAE of 0.07 eV versus CVS-FCI reference for 11 small molecules
- Composite CCSDTQ/ACVTZ scheme (using CCSDT/ACVTZ and CCSDTQ/ACVDZ) yields MAE of 0.03 eV versus CVS-FCI reference
- Orbital choice (optimized vs. neutral HF) affects CVS-FCI core IPs by only 0.02 eV average absolute deviation

## Context

This work extends the QUEST database (Loos 2018, 2020, 2025) from neutral excitation energies to charged excitations (core ionization), providing the first large-scale collection of CVS-FCI benchmark values for core IPs. It builds on the approach of Liu (2019) who computed 21 core IPs at the CVS-IP-EOM-CCSDTQ level, but expands the dataset to 84 values and uses FCI-level reference quality. The dataset enables systematic theory-versus-theory comparisons that disentangle correlation and relaxation errors from relativistic, vibrational, and continuum effects, allowing assessment of approximate methods including equation-of-motion coupled-cluster approaches (CC2 through CCSDTQ), the one-shot G0W0 scheme, and state-specific methods.
