# FELIX-2.0: New version of the finite element solver for the time dependent generator coordinate method with the Gaussian overlap approximation

> The time-dependent generator coordinate method (TDGCM) is a powerful method to study the large amplitude collective motion of quantum many-body systems such as atomic nuclei. Under the Gaussian Overlap Approximation (GOA), the TDGCM leads to a local, time-dependent Schrödinger equation in a multi-dimensional collective space. In this paper, we present the version 2.0 of the code FELIX that solves the collective Schrödinger equation in a finite element basis. This new version features: (i) the ability to solve a generalized TDGCM+GOA equation with a metric term in the collective Hamiltonian, (ii) support for new kinds of finite elements and different types of quadrature to compute the discretized Hamiltonian and overlap matrices, (iii) the possibility to leverage the spectral element scheme, (iv) an explicit Krylov approximation of the time propagator for time integration instead of the implicit Crank-Nicolson method implemented in the first version, (v) an entirely redesigned workflow. We benchmark this release on an analytic problem as well as on realistic two-dimensional calculations of the low-energy fission of Pu240 and Fm256. Low to moderate numerical precision calculations are most efficiently performed with simplex elements with a degree 2 polynomial basis. Higher precision calculations should instead use the spectral element method with a degree 4 polynomial basis. We emphasize that in a realistic calculation of fission mass distributions of Pu240, FELIX-2.0 is about 20 times faster than its previous release (within a numerical precision of a few percents).

## Key claim

FELIX-2.0 solves the time-dependent generator coordinate method with the Gaussian overlap approximation (TDGCM+GOA) equation including an arbitrary real positive metric term, which was not possible in version 1.0. The speedup factor of about 20 over the previous release is conditional on achieving only a few percent numerical precision in realistic fission calculations.

## Method

The code discretizes the N-dimensional collective space using a continuous Galerkin finite element method with either simplex or N-orthotope cells, computing matrix elements via Gauss-Legendre or Gauss-Lobatto quadrature. Time propagation of the resulting sparse matrix equation is performed using an explicit Krylov approximation of the exponential propagator (Arnoldi algorithm) instead of the implicit Crank-Nicolson scheme used in version 1.0. The spectral element method, using N-orthotope cells with Gauss-Lobatto node placement, yields a diagonal overlap matrix that trivializes the Cholesky decomposition step.

## Key numbers

* For a realistic two-dimensional calculation of Fm256, the new quadrature-based matrix element computation speeds up Hamiltonian matrix assembly by a factor of 100 compared to the previous symbolic integration approach.
* In a realistic calculation of fission mass distributions of Pu240, FELIX-2.0 is about 20 times faster than FELIX-1.0 within a numerical precision of a few percent.
* Low to moderate precision calculations are most efficiently performed with simplex elements of polynomial degree 2; higher precision calculations should use the spectral element method with polynomial degree 4.
* The Krylov space dimension for time propagation is typically around 10.

## Context

This work extends the TDGCM+GOA approach to fission fragment distributions pioneered by Goutte (2005), Younes (2012), and Regnier (2016) by adding metric terms to the collective Hamiltonian and substantially improving numerical efficiency. The new version enables calculations in higher-dimensional collective spaces, which earlier studies by Dubray (2012) and others identified as necessary for increased accuracy. The code is benchmarked against analytic problems and realistic two-dimensional calculations of Pu240 and Fm256 fission, and it provides a foundation for future three-dimensional TDGCM+GOA simulations.
