# WEB Portal for Monte Carlo Simulations in High Energy Physics - HEPWEB

> A WEB-portal HepWeb allows users to perform the most popular calculations in high energy physics - calculations of hadron-hadron, hadron-nucleus and nucleus-nucleus interaction cross sections as well as calculations of secondary particles characteristics in the interactions using Monte Carlo event generators. The List of the generators includes Dubna version of the intra-nuclear cascade model (CASCADE), FRITIOF model, ultra-relativistic quantum molecular dynamic model (UrQMD), HIJING model, and AMPT model. Setting up the colliding particles/nucleus properties (collision energy, mass numbers and charges of nuclei, impact parameters of interactions, and number of generated events) is realized by a WEB interface. A query is processed by a server, and results are presented to the user as a WEB-page. Short descriptions of the installed generators, the WEB interface implementation and the server operation are given.

## Key claim

The HepWeb portal provides a web-based interface for performing Monte Carlo simulations of hadron-hadron, hadron-nucleus, and nucleus-nucleus interactions using several event generators, but it is designed primarily for users who need episodic access to these models rather than for expert users who work with them routinely.

## Method

The authors implemented a client-server web portal using JavaServer Pages (JSP) technology, where users set collision parameters (energy, mass numbers, charges, impact parameter range, number of events) through HTML forms. The server checks parameter correctness, stores requests in a MySQL database, and runs the corresponding Monte Carlo generator (CASCADE, FRITIOF, UrQMD, HIJING, or AMPT) as an executable module. Results are presented as web pages with histograms in PNG format and downloadable ROOT files, with previously computed results retrieved from the database to avoid redundant calculations.

## Key numbers

- The radius of a nucleus with mass number A is estimated as R_A ~ 1.2 * A^(1/3) fm, used for impact parameter selection guidance.
- For nucleus-nucleus interactions, the maximum impact parameter can be estimated as R_P + R_T + 2 fm, where "2" represents double the radius of NN-interactions.
- The Wood-Saxon density parameters for nuclei with A > 10 are R_A = 1.07 * A^(1/3) fm and c = 0.545 fm.
- The nucleon core exclusion radius used in coordinate sampling is R_c = 0.4 fm.
- The strong interaction radius used for determining possible participant nucleons is R_int = 1.3 fm.
- The UrQMD model switches to calculations without potential interactions at energies larger than 4 GeV/nucleon.

## Context

This work extends the concept of the JetWeb server (2002) for Monte Carlo tuning and validation, which used PYTHIA and HERWIG with HERA, LEP, and Tevatron data but is no longer operational. It also builds on the NRV server (Nuclear Reactions Video) at JINR for low-energy nuclear dynamics, but targets high-energy physics calculations. The portal enables experimental collaborations such as CBM and PANDA at future GSI accelerators to perform background estimations and detector design studies without needing to install and configure Monte Carlo codes locally.
