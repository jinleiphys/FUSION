# Relations Between Coefficients of Fractional Parentage

> For each of the (9/2), (11/2) and (13/2) single j shells we have only one state with J=j V=3 for a five particle system. For four identical particles there can be more than one state of seniority four. We note some ``ratio'' relations for the coefficients of fractional parentage for the four and five identical particle systems.

# Relations Between Coefficients of Fractional Parentage

## Key claim
For the (9/2), (11/2), and (13/2) single-j shells, the coefficients of fractional parentage (cfp) for four identical particles of seniority four and five identical particles of seniority three obey a ratio relation, C(v_i)/C(v_j) = D(v_i)/D(v_j). This relation can be used to show that a specific linear combination of the two degenerate (g_{9/2})^4 I=4 v=4 states (designated v_a) cannot be admixed with an I=4 v=2 state, even via seniority-violating interactions.

## Method
The author extracted cfp values from the Bayman-Lande tables for the (9/2)^4 configuration with I=4 and v=4, and for the (9/2)^5 configuration with J=j and v=3. Using the ratio relation and the Redmond recursion relation as modified by Zamick and Escuderos (2006), the author constructed a linear combination v_a of the two degenerate v=4 states and demonstrated analytically that the matrix element M = <I=4 v=2 | V | I=4 v_a> vanishes for any interaction, by showing that each two-body matrix element contribution M(J_A) vanishes individually for J_A = 0, 2, 4, 6, 8.

## Key numbers
- For (9/2)^4 I=4 v=4: C(v_1) = 0.030755, C(v_2) = 0.347913; for (9/2)^5 J=j v=3: D(v_1) = -0.018452, D(v_2) = -0.208747; both ratios equal 11.312.
- The cfp for the v_a state with v_o=1 and J=j=9/2 vanishes (0.), as does the cfp for v_o=3 and J=j=9/2 (0.).
- The energy of the (9/2)^4 I=4 v_a=4 state is given by E = (68/33) E(2) + E(4) + (13/15) E(6) + (114/55) E(8), where the coefficient of E(4) is exactly 1.
- For J_A=4, the coefficient G(4) = 6[1/3 + 2/3(-1/4)] + vanishing cfps = 1.

## Context
This work extends the cfp ratio relations noted in the textbooks of de Shalit and Talmi (1963) and Talmi (1993), particularly Eq. (19.31) of the latter, which were previously used to explain the vanishing of a five-particle cfp. The author applies these relations to the specific case of the g_{9/2} shell, building on the Bayman-Lande tables (1966) and the Redmond recursion relation (1965) as modified by Zamick and Escuderos (2006). The result demonstrates that a particular seniority-four state in the g_{9/2} shell is immune to seniority mixing, analogous to the well-known property of the f_{7/2} shell where no seniority mixing occurs for any interaction. This enables further study of seniority-conserving and -violating interactions in medium-weight nuclei.
