# B-Spline Finite Elements and their Efficiency in Solving Relativistic Mean Field Equations

> A finite element method using B-splines is presented and compared with a conventional finite element method of Lagrangian type. The efficiency of both methods has been investigated at the example of a coupled non-linear system of Dirac eigenvalue equations and inhomogeneous Klein-Gordon equations which describe a nuclear system in the framework of relativistic mean field theory. Although, FEM has been applied with great success in nuclear RMF recently, a well known problem is the appearance of spurious solutions in the spectra of the Dirac equation. The question, whether B-splines lead to a reduction of spurious solutions is analyzed. Numerical expenses, precision and behavior of convergence are compared for both methods in view of their use in large scale computation on FEM grids with more dimensions. A B-spline version of the object oriented C++ code for spherical nuclei has been used for this investigation.

## Key claim

B-spline finite elements reduce the number of required mesh points compared to Lagrangian finite elements for solving relativistic mean field equations, but the paper's investigation is restricted to spherical nuclei and does not definitively prove that B-splines eliminate spurious solutions in the Dirac spectrum; the analysis only compares numerical expenses, precision, and convergence behavior between the two methods.

## Method

The author discretized the coupled system of Dirac eigenvalue equations and inhomogeneous Klein-Gordon equations for spherical nuclei using both B-spline and Lagrangian-type finite element shape functions on a uniform radial mesh. A weighted residual formulation transformed the system into a generalized algebraic eigenvalue problem, solved with a bisection method for band matrices, while the Klein-Gordon equations were solved using a biconjugate gradient method. The comparison was performed using an object-oriented C++ code for spherical nuclei, with both methods applied to the same physical problem to evaluate numerical cost, precision, and convergence.

## Key numbers

- The fine-structure constant used in the model is \( e^2 / 4\pi = 1/137.036 \)
- For a given element order \( n^{\rm ord} \), the number of occupied \( 2\times 2 \)-blocks in the stiffness matrix for Lagrange FEM is \( n^{fe}\cdot\bigl[ (n^{\rm ord})^2 + 2n\bigr]+1 \), while for B-spline FEM it increases to \( n^{fe}\cdot\bigl[ 2(n^{\rm ord})^2 + n \bigr] \) (the text truncates the second expression)
- The total number of mesh points used in the example discretization between \( r=0\,{\rm fm} \) and \( r=10\,{\rm fm} \) is 16 for both methods when using \( 3^{\rm rd} \) order elements

## Context

This work extends the author's previous finite element method solutions of spherical relativistic mean field equations (Poeschl, 1997) and relativistic Hartree-Bogoliubov equations (Poeschl, 1997), where spurious solutions appeared in the Dirac spectrum when using Lagrangian-type finite elements. The study is motivated by the need for efficient multidimensional FEM grids for deformed exotic nuclei, where eliminating spurious states a priori becomes important. The results are intended to inform large-scale computations on two- and three-dimensional finite element meshes, building on prior work using Krylov-subspace methods (Marquette, 1995) and the bisection method for the spherical case (Poeschl, 1997).
