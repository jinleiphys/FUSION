# An analysis of variational wave function for the pairing problem in strongly correlated system

> We report a theoretical analysis of variational wave functions for the BCS pairing problem. Starting with a Jastrow-Feenberg (or, in a more recent language "fixed-node") wave function for the superfluid state, we develop the full optimized Fermi-Hypernetted Chain (FHNC-EL) equations which sum a local approximation of the parquet-diagrams. Close examination of the procedure reveals that it is essential to go beyond the usual Jastrow-Feenberg approximation to guarantee the correct stability range.

# An analysis of variational wave function for the pairing problem in strongly correlated system

## Key claim

The Euler equation for local correlations in a Jastrow-Feenberg wave function for the superfluid state has no physically meaningful solution for net attractive interactions, regardless of how small the pairing gap is; this spurious instability is a consequence of the local correlation approximation itself and can only be removed by going beyond local correlations, for example by adding an infinite sum of terms in correlated basis function perturbation theory.

## Method

The authors develop the full optimized Fermi-Hypernetted Chain (FHNC-EL) equations for a correlated BCS wave function, starting from a Jastrow-Feenberg ansatz for the superfluid state. They decompose the superfluid free energy into diagonal, off-diagonal, and energy-numerator terms, then analyze the Euler equation for local correlations in the "uniform limit" approximation, which retains only chain diagrams. The stability condition is examined by comparing the collective approximation for the static structure function with the full random phase approximation using the superfluid Lindhard function.

## Key numbers

- The FHNC-EL method is accurate within better than one percent at densities less than 25% of the saturation density of liquid $^3$He.
- For neutron matter, a pairing gap of the order of half of the Fermi energy $\EF$ is found, in agreement with fixed-node Monte Carlo calculations.
- The stability condition from the collective approximation is $1 + \frac{3}{4}\frac{3 m}{\hbar^2 \KF^2} \tilde V(0+) > 0$, while the correct stability condition from the full Lindhard function is $1 + \frac{3 m}{\hbar^2 \KF^2} \tilde V(0+) > 0$ for the normal system.
- In the superfluid case, the stability condition becomes $1 + \frac{3 \tilde V(0+)}{2\mu} > 0$ as the gap $\Delta \rightarrow 0$.

## Context

This work extends the authors' previous correlated BCS calculations (Fan and Krotscheck, 2017; Krotscheck and collaborators, 2015) which encountered two problems: divergence of the in-medium scattering length indicating BCS-BEC crossover, and pairing gaps in neutron matter too large for the weak-coupling approximation. The analysis reveals that the Jastrow-Feenberg wave function for superfluids has a spurious instability for net attractive interactions, a problem that also applies to fixed-node Monte Carlo calculations. The solution requires going beyond local correlations by adding CBF perturbation theory terms, as previously carried out for normal systems by Krotscheck and collaborators (1998), or by formulating Coupled Cluster theory with correlated wave functions.
