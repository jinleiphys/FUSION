# Fully Heavy Pentaquarks with JETHAD: A High-Energy Viewpoint

> We examine the leading-power fragmentation of fully heavy pentaquarks in high-energy hadronic collisions. To this end, we complete the release of the hadron-structure-oriented PQ5Q1.0 fragmentation functions, by discussing the $P_{5c}$ set and delivering the $P_{5b}$ one. These functions incorporate an improved computation of the initial-scale input for the constituent heavy-quark fragmentation channel, making them particularly suitable for describing both the direct formation of a compact multicharm state and the hadronization from a diquark-antiquark-diquark configuration. For phenomenological applications, we employ the data-validated (sym)JETHAD framework to compute and analyze NLL/NLO$^+$ semi-inclusive production rates of pentaquark-plus-jet systems at the upcoming HL-LHC and the future FCC. This study marks a further step toward connecting hadronic structure, precision QCD, and the emerging physics of exotic matter.

## Key claim

The paper claims that leading-power fragmentation of fully heavy pentaquarks (pentacharms and pentabottoms) can be described by the new PQ5Q1.0 fragmentation functions, which incorporate an improved initial-scale input for the constituent heavy-quark channel; however, the claim is hedged by the fact that the precise nature of the transition between compact multicharm states and diquark-antiquark-diquark configurations requires solving the five-body bound-state problem with confining forces, which remains elusive.

## Method

The authors construct collinear ZM-VFNS (Zero-Mass Variable-Flavor Number Scheme) fragmentation functions for S-wave fully heavy pentaquarks, using a Suzuki-inspired framework to compute the initial-scale input for the [Q → P5Q] fragmentation channel in a direct multiquark scenario. They then evolve these inputs via NLO DGLAP equations using the (sym)JETHAD framework, and employ an NLL/NLO⁺ hybrid-factorization scheme that embeds high-energy resummation into collinear factorization for phenomenological predictions at HL-LHC and FCC.

## Key numbers

- Charm-quark mass set to m_c = 1.5 GeV and bottom-quark mass set to m_b = 4.5 GeV for the fragmentation function calculations
- The initial-scale fragmentation function for direct [Q → P5Q] takes the form D(z, μ_F,0) ∝ (1−z)⁴ z⁴, with explicit dependence on mass ratios R_P/Q = M_P5Q/m_Q and transverse-momentum ratios R_qT/Q = √⟨q_T²⟩/m_Q
- Production rate of B_c mesons from prior work (Celiberto 2024) is below 0.1% of that for singly bottomed B mesons, serving as a benchmark for the ZM-VFNS approach at high transverse momentum

## Context

This work extends the author's prior program of constructing fragmentation functions for exotic hadrons, including the TQHL1.1 set for doubly heavy tetraquarks and TQ4Q1.1 for fully heavy tetraquarks (Celiberto 2024). It builds upon the Suzuki-inspired formalism for pentaquark fragmentation developed by Farashaeian (2024) and complements the pentacharm study initiated in Celiberto (2025) by delivering the bottom-flavored P5b set for the first time. The results enable future phenomenological analyses at HL-LHC and FCC aimed at discriminating between different exotic formation dynamics through multimodal fragmentation functions.
