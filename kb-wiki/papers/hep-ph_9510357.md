# Constituent string model for hybrid mesonic excitations

> The model for hybrid excitations of the QCD string with quarks is presented starting from the perturbation theory in the nonperturbative background. The propagation of a system containing $q \bar q$-pair and gluon is considered. The simplified version of the Hamiltonian, including both long-range nonperturbative interaction and Coulomb force, is derived. The masses of the lowest $q \bar q g$ hybrids are evaluated, and numerical results for the spectra are listed.

## Key claim

The paper presents a constituent string model for hybrid mesons derived from perturbation theory in a nonperturbative QCD background, claiming that the resulting spectra are compatible with experimental data on light quark meson spectroscopy, but only under the condition that the additive constant term for the hybrid (with two strings) is taken to be twice that fitted from the ordinary \(q\bar q\) meson spectrum.

## Method

The authors split the gluonic field into a nonperturbative background field \(B_{\mu}\) and a perturbation \(a_{\mu}\), then construct the Green function for the \(q\bar q g\) system using the Feynman–Schwinger representation. They average the Wilson loop configuration over the background using the cluster expansion method, assuming a generalized area law, and reduce the four-dimensional dynamics to a three-dimensional effective Hamiltonian in the centre-of-mass frame. The Hamiltonian includes both the long-range nonperturbative string interaction (with string tension \(\sigma\)) and a short-range Coulomb interaction, and the problem is treated variationally with Gaussian-type radial wave functions, imposing transversality of the gluon state.

## Context

This work extends the Vacuum Background Correlator method (developed by Simonov and collaborators) to the \(q\bar q g\) hybrid system, building on earlier studies of the \(q\bar q\) system in the same framework. The results are compared to the flux-tube model (Isgur and Paton, 1985) and its "one-bead" version (references [10,11]), with which the spectra are found to be similar, though the quantum number assignments differ: the present model gives a constituent gluon with its own quantum numbers, while the flux-tube model describes string excitations via collective phonon modes. The paper enables future work on hybrid decay signatures and the search for \(P\)-even hybrid states, which remain unobserved experimentally.
