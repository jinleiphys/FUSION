# The Modified J-Matrix Approach for Cluster Descriptions of Light Nuclei

> We present a fully microscopic three-cluster nuclear model for light nuclei on the basis of a J-Matrix approach. We apply the Modified J-Matrix method on $^{6}He$ and $^{6}Be$ for both scattering and reaction problems, analyse the Modified J-Matrix calculation, and compare the results to experimental data.

## Key claim

The Modified J-Matrix (MJM) approach, which introduces a semiclassical approximation for matrix elements in the highly excited internal region, can be extended to fully microscopic three-cluster descriptions of light nuclei, but the validity of the decoupling approximation for charged clusters (where off-diagonal matrix elements of the effective charge are assumed sufficiently small) must be checked for any specific three-cluster system.

## Method

The authors formulate a three-cluster model using antisymmetrized wave functions built from fixed harmonic oscillator (0s)-state Slater determinants for clusters with A_i ≤ 4, with relative motion expanded in a complete oscillator basis classified by hyperspherical harmonics (Zernike-Brinkman basis). The Schrödinger equation is cast into a coupled-channels matrix form using the J-Matrix approach, where asymptotic solutions for neutral clusters are given by Hankel functions and for charged clusters by Whittaker functions, with the internal region truncated at oscillator index N and solved via standard linear algebra for residual coefficients and S-matrix elements. Matrix elements are computed using the Generating Function Method with hyperspherical coordinates.

## Context

This work extends the Modified J-Matrix approach previously developed for two-cluster descriptions (Arickx et al., 2002; Vasilevsky et al., 2002) to three-cluster systems, building on earlier cluster model studies of light p-shell nuclei (Vasilevsky, 1992; ITP+RUCA1, ITP+RUCA2). The method is applied to ⁶He and ⁶Be for both scattering and reaction problems, and the results are compared to experimental data. The approach enables fully microscopic three-cluster calculations that account for the long-range 1/ρ^3 potential tail, which requires special care for convergence in low-energy scattering calculations.
