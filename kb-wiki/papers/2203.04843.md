# Measures of complexity and entanglement in fermionic many-body systems

> There is no unique and widely accepted definition of the complexity measure (CM) of a many-fermion wave function in the presence of interactions. The simplest many-fermion wave function is a Slater determinant. In shell-model or configuration interaction (CI) and other related methods, the state is represented as a superposition of a large number of Slater determinants, which in case of CI calculations reaches about 20 billion terms. Although in practice this number has been used as a CM for decades, it is ill defined: it is not unique, and it depends on the particular type and the number of single-particle wave functions used to construct the Slater determinants. The canonical wave functions/natural orbitals and their corresponding occupation probabilities are intrinsic properties of any many-body wave function, irrespective of the representation, and they provide a unique solution to characterize the CM. The non-negative orbital entanglement entropy, which vanishes for a Slater determinant, provides the simplest CM, while a more complete measure of complexity is the entanglement spectrum. We illustrate these aspects in the case of a complex non-equilibrium time-dependent process, induced nuclear fission described within a real-time Density Functional Theory framework extended to superfluid systems, which can describe simultaneously the long-range and the short range correlations between fermions.

## Key claim

The paper argues that the number of Slater determinants used in configuration-interaction calculations is an ill-defined complexity measure because it depends on the choice of single-particle basis, and proposes instead that the canonical (natural) orbital occupation probabilities and the orbital entanglement entropy provide unique, representation-independent measures of many-fermion wave-function complexity.

## Method

The authors solve the static self-consistent Hartree-Fock-Bogoliubov (HFB) equations in one dimension using the Discrete Variable Representation method for a Woods-Saxon-like potential with a pairing field, using four different lattice constants (dx = 1, 0.5, 0.25, 0.125 fm) in a box of size L = 80 fm. They diagonalize the one-body density matrix to obtain canonical occupation probabilities and canonical wave functions, and compute the orbital entanglement entropy (Boltzmann entropy) from these occupation probabilities. They then apply this analysis to a time-dependent Density Functional Theory description of induced nuclear fission.

## Key numbers

* The number of Slater determinants in CI calculations reaches about 20 billion terms (Johnson, 2018).
* The minimal number of Slater determinants required to represent the fissioning nucleus wave function is about 10^500.
* The realistic fission case is equivalent to a system of 23.328 × 10^9 interacting quantum spin-1/2 particles.
* The high-momentum occupation probabilities follow n_k ≈ C/k^4, with approximately 20% of spectral strength found for momenta k > k_F.
* The canonical occupation probabilities were computed down to values n_k ≈ 10^(-6) for accurate entropy evaluation.
* The phase-space estimate for the number of relevant canonical states is (8π²/9)(r_0/dx)³ A, where r_0 = 1.2 fm.
* The maximum momentum cutoff is Λ = πℏ/dx, where dx is the spatial resolution.

## Context

This work extends the concept of natural orbitals introduced by Löwdin (1956) and the canonical basis used in superfluid systems by Bogoliubov (1958) and Valatin (1958), applying them as complexity measures for many-fermion wave functions. It contradicts the common practice in shell-model and CI calculations (Johnson, 2018) of using the number of Slater determinants as a complexity measure, showing this number is basis-dependent and ill-defined. The paper enables the study of quantum entanglement in the largest systems studied so far (23 billion effective spin-1/2 particles) and provides tools for analyzing non-equilibrium processes like induced nuclear fission, where the entanglement entropy shows non-monotonic time-dependence similar to many-body localization phenomena studied in 1D systems.
