# Linear Langevin Equation of Critical Fluctuation in Chiral Phase Transition

> We derive the linear Langevin equation that describes the behavior of critical fluctuation above the critical temperature of the chiral phase transition in the Nambu-Jona-Lasinio model. The Langevin equation relaxes exhibiting oscillation and shows thermalization. The relaxation becomes slower as the system approaches the critical point(the critical slowing down). The time correlation function also calculated using the Langevin equation shows that there exists the same soft mode as the previous calculation in the linear response theory.

# Linear Langevin Equation of Critical Fluctuation in Chiral Phase Transition

## Key claim

The linear Langevin equation derived for the chiral order parameter fluctuation above the critical temperature in the Nambu-Jona-Lasinio model exhibits critical slowing down, thermalization, and a soft mode, but only after artificially removing the oscillation effect from the memory function to obtain a renormalized memory function without long-time correlation.

## Method

The authors start from the Nambu-Jona-Lasinio (NJL) model with two flavors and three colors, using the Mori projection operator to project dynamics onto the order parameter (chiral condensate) as the sole gross variable. They derive a time-convolution (TC) equation, expand the memory term to lowest order in the interaction Hamiltonian using a ring diagram approximation, and then separate the memory function into an oscillatory part (frequency shift) and a damping part to obtain a linear Langevin equation with a renormalized memory function that has no long-time tail.

## Key numbers

- The NJL model parameters are fixed to reproduce the pion decay constant $f_{\pi} = 93\ \mathrm{MeV}$ and the chiral condensate $\langle \bar{q}q \rangle = (-250\ \mathrm{MeV})^3$, giving $g = 5.01\ \mathrm{GeV^{-2}}$ and $\Lambda = 650\ \mathrm{MeV}$.
- The critical temperature of the chiral phase transition in the mean field approximation is $T_{c} = 185\ \mathrm{MeV}$.
- At $\epsilon = (T-T_{c})/T_{c} = 0.2$, the relaxation time $\tau_{rt}$ is about $10\ \mathrm{fm}$, which is the same order as the expected lifetime of the quark-gluon plasma.
- The memory function $\Gamma^{L}(\mathbf{0},s)$ vanishes at $T_{c}$ due to the self-consistency condition $1-2g\beta\chi_{0}(\mathbf{0})|_{T=T_{c}} = 0$.

## Context

This work extends the linear response theory calculation of Hatsuda and Kunihiro (1994) by deriving a Langevin equation that explicitly describes the time evolution of critical fluctuations in the chiral phase transition, rather than only the spectral function. The authors note that the resulting dynamics show oscillation and cannot be described by a simple diffusion-type equation like the time-dependent Ginzburg-Landau (TDGL) equation. The approach enables future study of nonlinear effects via mode coupling theory and application to finite density and color superconducting phase transitions.
