# Kernel methods for evolution of generalized parton distributions

> Generalized parton distributions (GPDs) characterize the 3-dimensional structure of hadrons, combining information about their internal quark and gluon longitudinal momentum distributions and transverse position within the hadron. The dependence of GPDs on the factorization scale $Q^2$ allows one to connect hard exclusive processes involving GPDs at disparate energy and momentum scales, which is needed in global analyses of experimental data. In this work we explore how finite element methods can be used to construct fast and differentiable $Q^2$ evolution codes for GPDs in momentum space, which can be used in a machine learning framework. We show numerical benchmarks of the methods' accuracy, including a comparison to an existing evolution code from PARTONS/APFEL++, and provide a repository where the code can be accessed.

## Key claim

The paper demonstrates that finite element methods (specifically, a "Simple Method" using modified cubic Hermite splines with fixed Gauss-Legendre quadrature and a "Refined Method" using piecewise Lagrange interpolation) can construct fast and differentiable Q² evolution codes for GPDs in momentum space suitable for machine learning frameworks, but the first code release only implements leading-order (LO) evolution, with applicability to higher orders stated but not yet demonstrated.

## Method

The authors discretize the GPD's x-dependence onto a grid and convert the integro-differential evolution equations into matrix equations by representing convolution integrals as tensor contractions. They implement two finite element approaches: the Simple Method uses modified cubic Hermite spline interpolation with fixed Gauss-Legendre quadrature to construct kernel matrices, while the Refined Method uses piecewise Lagrange interpolation (interpixels) to build basis functions for more accurate integration. The evolution equations for nonsinglet and coupled singlet-gluon combinations are then solved using Runge-Kutta integration in Q², with the zero-mass variable flavor number scheme for heavy quarks.

## Key numbers

- The Simple Method uses Gauss-Legendre quadrature with n_g quadrature points, where the quadrature rule allows exact integration of polynomials of degree 2n_g - 1
- The Refined Method uses fifth-order (N_L = 5) piecewise Lagrange interpolation, constructing interpixels by pasting together basis functions from subsets of six x_i values
- The x grid is linearly spaced with n_x points: x_i = -1 + 2(i-1)/(n_x-1) for i in {1,2,...,n_x}
- The paper provides numerical benchmarks comparing accuracy against the existing PARTONS/APFEL++ evolution code, though specific numerical accuracy values are not given in the truncated text

## Context

This work extends the finite element approach previously used for forward parton distribution function evolution in codes such as HOPPET (Salam, 2008) and APFEL (Bertone, 2013) to the more complex case of GPD evolution, which involves an additional skewness parameter ξ. The method also builds on the recent finite element evolution code used by PARTONS (Bertone, 2023). The resulting differentiable evolution code enables integration into machine learning frameworks for global analyses of hard exclusive processes, which will be needed for interpreting data from Jefferson Lab's 12 GeV beam and the future Electron-Ion Collider.
