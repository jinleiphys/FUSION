# Variational Optimization for Constructing Inverse Potentials of Proton-Proton Scattering: A Phase Function Method Study

> Background: The phase-shift analysis for proton-proton scattering has been studied by various research groups using the realistic potentials to be comprised of various internal interactions based on an exchange of pions and mesons, involving a large number of parameters. Purpose: The goal of the research is to construct inverse potentials for various l-channels of proton-proton (pp) elastic scattering using the 3-parameter Morse function in combination with atomic Hulthen by utilizing the phase function method and variational optimization technique. Methodology: The implementation of variational optimization begins with randomly assigning initial values to the Morse model parameters. Utilizing the Morse + Hulthen potential as input, the phase equations for various l-channels are numerically solved using the RK-5 method for obtaining the simulated Scattering Phase Shift (SPS). Mean Squared error between simulated and expected SPS has been chosen as the cost function. Variational optimization proceeds iteratively by adjusting potential parameters and re-evaluating the cost function until convergence is achieved. Results: All the obtained scattering phase shifts for various l-channels have been found to converge to a mean squared error <= 0.3. The computed cross-sections matched the experimental ones to less than 1% for energies up to 25 MeV. The scattering parameters are also found to closely match the experimental data. Conclusion: The inverse potentials constructed for various l-channels using Morse + atomic Hulthen are on par with the currently available high-precision realistic potentials.

## Key claim

The authors construct inverse potentials for proton-proton elastic scattering across multiple partial waves using a 3-parameter Morse function combined with an atomic Hulthen screened Coulomb potential, achieving scattering phase shifts with mean squared error ≤ 0.3 against the Granada group experimental data, but note that the screening parameter in the atomic Hulthen potential must decrease with increasing angular momentum to maintain agreement.

## Method

The phase function method (PFM) is used to convert the radial Schrodinger equation into first-order Riccati equations for each partial wave, which are solved numerically using the 5th-order Runge-Kutta method with the Morse + atomic Hulthen potential as input. Variational optimization proceeds by randomly initializing the three Morse parameters (V₀, rₘ, aₘ) and the Hulthen screening radius a, then iteratively adjusting each parameter by adding a random number from a shrinking interval, accepting changes that reduce the mean squared error between simulated and experimental scattering phase shifts from the Granada group's MEPWAD database. The process continues until the cost function converges, yielding optimized potential parameters for each ℓ-channel from S-wave through H-wave.

## Key numbers

- Mean squared error (MSE) between simulated and experimental scattering phase shifts ≤ 0.3 for all ℓ-channels; specific values include 0.1 for ¹S₀, 0.3 for ³P₀, 0.04 for ¹D₂, and 0.0002 for ³F₄ and ³H₄ states
- Computed total cross-sections match experimental values to less than 1% for energies up to 25 MeV
- For the ¹S₀ state: scattering length a₀ = -7.8544 fm (experimental: -7.8063 fm) and effective range r₀ = 2.215 fm (experimental: 2.794 fm)
- Optimized Morse parameters for ¹S₀: V₀ = 91.007 MeV, rₘ = 0.872 fm, aₘ = 0.331 fm, with Hulthen screening radius a = 9 fm
- Screening radius a decreases with increasing ℓ: from 9 fm (¹S₀) to 0.2 fm (¹G₄ and ³H₄)
- Phase shifts calculated for energies from 1 MeV to 350 MeV across all partial waves

## Context

This work extends the authors' previous inverse potential constructions for neutron-proton scattering (Khachi 2023) using the Morse function alone, now incorporating the atomic Hulthen screened Coulomb potential to handle charged proton-proton scattering. The approach provides an alternative to high-precision realistic potentials such as Argonne v18 (Wiringa 1995), which uses 18 parameters, and the Nijmegen potentials (Stoks 1993), by achieving comparable results with only 3-4 parameters per channel. The resulting inverse potentials are intended for use in ab-initio nuclear structure calculations and can be ported more easily than the multi-parameter realistic potentials.
