# Charged-particle multiplicity and transverse-energy distribution using the Weibull-Glauber approach in heavy-ion collisions

> The charged-particle multiplicity distribution and the transverse-energy distribution measured in heavy-ion collisions at top RHIC and LHC energies are described using the two-component model approach based on a convolution of the Monte Carlo Glauber model with the Weibull model for particle production. The model successfully describes the multiplicity and transverse-energy distribution of minimum-bias collision data for a wide range of energies. The Weibull-Glauber model can be used to determine the centrality classes in heavy-ion collisions as an alternative to the conventional negative binomial distribution (NBD)-Glauber approach.

## Key claim

The Weibull-Glauber model, a two-component approach combining the Monte Carlo Glauber model with the Weibull distribution for particle production, successfully describes the charged-particle multiplicity and transverse-energy distributions in heavy-ion collisions at top RHIC and LHC energies, and can serve as an alternative to the conventional negative binomial distribution (NBD)-Glauber approach for centrality determination, though the model shows some deviation for the most peripheral and most central events due to electromagnetic contamination, trigger inefficiency, and fluctuations of participant number.

## Method

The authors used a Monte Carlo Glauber model to simulate nucleus-nucleus collisions event-by-event, obtaining the number of participant nucleons (Npart) and binary collisions (Ncoll) for each impact parameter. The number of particle-emitting ancestors was parametrized as a two-component function of Npart and Ncoll, and for each ancestor the charged-particle multiplicity (or transverse energy) was sampled from a two-parameter Weibull distribution. The free parameters (lambda, k, and the two-component mixing parameter x) were determined by chi-square minimization against experimental data from STAR, PHENIX, and ALICE. The model was also validated by comparing centrality classes derived from HIJING event generator with those from the Weibull-Glauber approach.

## Key numbers

- For Au-Au collisions at sqrt(s_NN) = 200 GeV, the charged-particle multiplicity distribution was described with lambda = 0.685, k = 1.17, and x = 0.805.
- For Pb-Pb collisions at sqrt(s_NN) = 2.76 TeV, the charged-particle multiplicity distribution was described with lambda = 33.20, k = 1.17, and x = 0.8015.
- For Au-Au collisions at sqrt(s_NN) = 200 GeV, the transverse-energy distribution was described with lambda = 4.58, k = 1.17, and x = 0.802.
- For Pb-Pb collisions at sqrt(s_NN) = 2.76 TeV, the transverse-energy distribution was described with lambda = 1.81, k = 1.17, and x = 0.799.
- For the 0-5% centrality class in Pb-Pb collisions at 2.76 TeV, the model gave <Npart> = 354.9 compared to HIJING value of 364.7; for 20-30% centrality, model gave 144.0 versus HIJING 144.6; for 50-60% centrality, model gave 34.24 versus HIJING 34.01; for 70-80% centrality, model gave 6.713 versus HIJING 6.052.

## Context

This work extends the two-component Glauber approach previously used by the ALICE collaboration (2011, 2013) with the negative binomial distribution (NBD) to describe multiplicity distributions in Pb-Pb collisions at 2.76 and 5.02 TeV. The authors replace the NBD with the Weibull distribution, which had been shown by earlier studies (2010, 2012) to describe multiparticle production in hadronic and leptonic collisions. The model enables an alternative method for determining centrality classes in heavy-ion collisions, providing a way to estimate average Npart and Ncoll values for a given centrality interval without relying on the NBD formalism.
