# Instruction of my personal computing library

> This document is prepared to introduce and explain how to use the computing library composed by T. Oishi. The library-01 TOSPEM solves, for the spherical nucleus, (i) the Schroedinger equation for the single-nucleon states within the Woods-Saxon potential, (ii-a) the electric or magnetic transition strength, B(EJ) or B(MJ), between the arbitrary set of initial and final states of the nucleus of interest, and (ii-b) Weisskopf estimate for comparison with results in (ii-a). The library-02 RESONA is composed to solve the resonant eigenstates of spherical Schroedinger equations. The final version is expected to be published for educational and commercial purposes. Before the official publication, under the agreement with publishers, I make the current, preliminary version open for public. Two applications, GFORTRAN and GNUPLOT, are necessary for full usage. Feedbacks and comments on products will be appreciated. The source codes etc. are available in the GitHub repository [1].

## Key claim

This document describes two personal computing libraries (TOSPEM and RESONA) for solving nuclear structure problems, but the full text only provides details on TOSPEM and its underlying numerical methods; the paper is a preliminary user manual rather than a research paper presenting new physics results, and the author notes the libraries are not yet officially published.

## Method

The library TOSPEM solves the spherical Schrödinger equation for single-nucleon states in a Woods-Saxon potential using the Numerov method for bound states and a Runge-Kutta method (including a complex-scaled version) for resonant states. For a given spherical nucleus (Z,N), the code computes single-particle energies and wave functions, then calculates electric (EJ) or magnetic (MJ) transition strengths between arbitrary initial and final states using the reduced matrix element formalism from Ring and Schuck (1980) and Suhonen (2007), and compares these to Weisskopf estimates. The code uses a node-counting procedure to determine eigenenergies and requires matching forward and backward solutions for bound states.

## Context

This work extends the author's previous calculations (Oishi 2011, Oishi and Potel 2019) by providing a documented, publicly available computing library for spherical nuclear structure calculations. The library implements standard nuclear physics methods including the Woods-Saxon potential, Numerov method (as described in Hairer 1993), and transition strength formulas from Ring and Schuck (1980) and Suhonen (2007). The code enables users to compute single-particle states and electromagnetic transitions for any spherical nucleus, with the Weisskopf estimate (Weisskopf 1951) provided for comparison; future work could extend the library to deformed nuclei or include more sophisticated interactions.
