# Hessian PDF reweighting meets the Bayesian methods

> We discuss the Hessian PDF reweighting - a technique intended to estimate the effects that new measurements have on a set of PDFs. The method stems straightforwardly from considering new data in a usual $χ^2$-fit and it naturally incorporates also non-zero values for the tolerance, $Δχ^2>1$. In comparison to the contemporary Bayesian reweighting techniques, there is no need to generate large ensembles of PDF Monte-Carlo replicas, and the observables need to be evaluated only with the central and the error sets of the original PDFs. In spite of the apparently rather different methodologies, we find that the Hessian and the Bayesian techniques are actually equivalent if the $Δχ^2$ criterion is properly included to the Bayesian likelihood function that is a simple exponential.

# Hessian PDF reweighting meets the Bayesian methods

## Key claim

Hessian PDF reweighting and Bayesian reweighting techniques are equivalent if the tolerance criterion Δχ² is properly included in the Bayesian likelihood function as a simple exponential, but only when the Giele-Keller weights are rescaled by Δχ²; the chi-squared weights used in NNPDF-style fits yield different results.

## Method

The authors derive a Hessian reweighting method by adding new data contributions to the χ² function expanded around the minimum of a global PDF fit, using linear approximations for observables computed from the central and error PDF sets. They compare this to Bayesian reweighting methods that construct ensembles of PDF replicas from Hessian error sets using Gaussian random numbers, then assign weights based on the new data. A simplified pseudodata example with a known functional form is used to test equivalence, followed by a practical application to CMS inclusive jet data at 7 TeV using CTEQ6.6 PDFs.

## Key numbers

- For CTEQ6.6 PDFs, the global tolerance is Δχ² = 100.
- The CMS inclusive jet data at 7 TeV contains N = 133 data points.
- Before reweighting, the initial χ² per data point for CTEQ6.6 against CMS jet data is χ²/N ≈ 2.1.
- After Hessian reweighting with CMS jet data, the penalty is approximately 20 units, and the new global χ² changes by about -30 units.
- After Bayesian reweighting with chi-squared weights, the new global χ² increases by approximately 330 units (480 - (2.1-1.0)×133).

## Context

This work extends the Hessian PDF reweighting method developed by Paukkunen and Zurita (2013, 2014) and compares it to Bayesian reweighting techniques originally proposed by Giele and Keller (1998) and later revived by the NNPDF collaboration (Ball 2010, 2011). The paper resolves apparent discrepancies between the two approaches by showing that the Giele-Keller weights must be rescaled by the tolerance Δχ² to match the Hessian method. This enables efficient reweighting of existing Hessian PDF sets (such as CTEQ6.6 from Nadolsky 2008) using new LHC data without performing a full global fit, and clarifies which Bayesian weighting scheme is appropriate for Hessian PDFs.
