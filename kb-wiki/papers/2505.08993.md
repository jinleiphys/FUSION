# A modern Fortran library for SU(3) coupling and recoupling coefficients

> The group $\mathrm{SU}(3)$ has applications in several branches of physics. Many of these applications depend on availability of $\mathrm{SU}(3)$ coupling and recoupling coefficients. We have developed a modern Fortran library for calculation of the coupling coefficients, for both the $\mathrm{SU}(3)\supset\mathrm{U}(1)\times\mathrm{SU}(2)$ and $\mathrm{SU}(3)\supset\mathrm{SO}(3)$ group chains, and the recoupling coefficients. The library implements the algorithms of Draayer, Akiyama, and Millener, which are laid out in the paper. Performance of the library has been tested and compared to the Akiyama-Draayer (AD) library implementing the same algorithms as well as to a more recent implementation. Our library works for a larger range of $\mathrm{SU}(3)$ quantum numbers and provides more accurate coupling coefficients with large quantum numbers than the AD library.

## Key claim

The authors present a modern Fortran library (ndsu3lib) that implements the Draayer-Akiyama and Millener algorithms for SU(3) coupling and recoupling coefficients, claiming it works for a larger range of SU(3) quantum numbers and provides more accurate coupling coefficients with large quantum numbers than the existing Akiyama-Draayer (AD) library, while achieving comparable computational speed.

## Method

The library implements the Draayer-Akiyama (DA) algorithm for SU(3)⊃U(1)×SU(2) reduced coupling coefficients using recurrence relations derived from the method of infinitesimal generators, with the Biedenharn-Louck-Hecht (BLH) prescription for resolving outer multiplicities. The SU(3)⊃SO(3) coupling coefficients are obtained by transforming the canonical chain coefficients using Elliott basis states and Gram-Schmidt orthonormalization. Recoupling coefficients (U, Z, and 9-(λ,μ) coefficients) are computed following Millener's extensions of the DA algorithm, using multiprecision arithmetic via MPFUN2020 for large quantum numbers.

## Key numbers

* The library supports SU(3) irreps labeled by quantum numbers (λ,μ) with no explicit upper bound stated; testing was performed for quantum numbers where the AD library loses precision and produces incorrect results
* The highest-weight quantum numbers for an irrep (λ,μ) are given by ε^H = -λ-2μ, Λ^H = λ/2, M_Λ^H = -λ/2
* The lowest-weight quantum numbers for an irrep (λ,μ) are given by ε^L = 2λ+μ, Λ^L = μ/2, M_Λ^L = μ/2
* The orthonormalization matrix O^(λ,μ)L has size κ_max × κ_max, where κ_max is the number of distinct K values for a given L in the irrep (λ,μ)
* The library is written in Fortran 2003 with C/C++ headers, licensed under MIT, and available at DOI 10.5281/zenodo.16655521

## Context

This work extends the original Akiyama-Draayer (AD) Fortran library (Akiyama and Draayer, 1970s) and the more recent C++ implementation SU3lib by Dytrych et al., all based on the same underlying DA algorithm with the same phase conventions and multiplicity prescriptions. The library is intended for use in symmetry-guided ab initio nuclear structure calculations, particularly the symplectic no-core configuration interaction framework, and enables calculations in larger model spaces and for heavier nuclei than previously possible with the AD library.
