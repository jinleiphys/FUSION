# Wounded quarks and diquarks in heavy ion collisions

> A model in which the soft collisions of the nucleon are described in terms of interactions of its two constituents (a quark and a diquark) is proposed. When adjusted to describe precisely the elastic proton-proton scattering data and supplemented with the idea of wounded constituents, the model accounts rather well for the centrality dependence of particle production in the central rapidity region at RHIC energies.

## Key claim

The paper proposes that a model where the nucleon is composed of two constituents (a quark and a diquark), with the assumption that each wounded constituent emits the same number of secondary partons independently of the number of collisions it undergoes, can account for the centrality dependence of particle production in the central rapidity region at RHIC energies, provided the model parameters are first adjusted to describe elastic proton-proton scattering data precisely.

## Method

The authors model the nucleon as a quark and a diquark with Gaussian spatial distributions and Gaussian impact-parameter-dependent cross-sections, with the ratio of cross-sections fixed as σ_qq : σ_qd : σ_dd = 1 : 2 : 4. The five free parameters of the model are determined by fitting the total inelastic cross-section, the slope of the elastic cross-section, the position of the first diffractive minimum, and the height of the second maximum in elastic proton-proton scattering at ISR energies. Using these parameters, the average number of wounded constituents in nucleon-nucleon collisions is computed, and the model's predictions for particle production in Au-Au collisions are evaluated via a Glauber-type counting of wounded quarks and diquarks as a function of impact parameter.

## Key numbers

- The ratio σ_qq / σ_NN (quark-quark to nucleon-nucleon inelastic cross-section) is 0.147–0.148, determined from fits to elastic pp data at √s = 30.5 GeV.
- The average number of wounded constituents in a single nucleon-nucleon collision, w_N^(q+d) = w_q + w_d, is 1.182–1.186.
- For Au-Au collisions at √s = 200 GeV, the inelastic pp cross-section used is σ_NN = 42 mb, giving σ_qq / σ_NN = 0.149 and w_N^(q+d) = 1.187.
- The model parameters include constituent radii: R_q ≈ 0.3 fm, R_d ≈ 0.75 fm, and R ≈ 0.3 fm for the nucleon thickness parameter.
- The model applies only for soft collisions where transverse mass of emitted partons does not exceed ~200 MeV.

## Context

This work extends the wounded nucleon model (Bialas, Bleszynski, Czyz, 1976) which fails to describe the centrality dependence of particle production at RHIC, and improves upon the wounded quark model (Bialas, Czyz, Lesniak, 1968) which required an unrealistically large number of quark-quark collisions. The model successfully reproduces PHOBOS data on Au-Au collisions at √s = 19.6, 62.4, 130, and 200 GeV, and its predictions for d-Au collisions deviate less than 10% from the wounded nucleon model, preserving agreement with existing d-Au data.
