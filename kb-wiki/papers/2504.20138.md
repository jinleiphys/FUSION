# Shear and bulk viscosity for a pure glue theory using an effective matrix model

> At nonzero temperatures, the deconfining phase transition can be analyzed using an effective matrix model to characterize the change in holonomy. The model includes gluons and two-dimensional ghost fields in the adjoint representation, or ``teens''. As ghosts, the teen fields are responsible for the decrease of the pressure as $T \rightarrow T_d$, with $T_d$ the transition temperature for deconfinement. Using the solution of this matrix model for a large number of colors, the parameters of the teen fields are adjusted so that the expectation value of the Polyakov loop is close to the values from the lattice. The shear, $η$, and bulk, $ζ$, viscosities are computed in weak coupling but nonzero holonomy. In the pure glue theory, the value of the Polyakov loop is relatively large in the deconfined phase, $\approx 1/2$ at $T_d$. Consequently, if $s$ is the entropy density, while $η/s$ decreases as $T\rightarrow T_d$, it is still well above the conformal bound. In contrast, $ζ/s$ is largest at $T_d$, comparable to $η/s$, then falls off rapidly with increasing temperature and is negligible by $\sim 2 T_d$.

## Key claim

The shear viscosity to entropy density ratio (η/s) in pure glue theory decreases as temperature approaches the deconfining transition temperature T_d but remains well above the conformal bound, while the bulk viscosity to entropy density ratio (ζ/s) is largest at T_d and becomes negligible by approximately 2 T_d, computed using an effective matrix model with nonzero holonomy and two-dimensional ghost fields ("teens") at leading logarithmic order in weak coupling.

## Method

The authors use an effective matrix model for SU(N_c) gauge theory at large N_c that includes gluons and two-dimensional adjoint ghost fields ("teens") to describe nonzero holonomy. They compute shear and bulk viscosities from 2→2 scattering processes involving gluons and teens to leading logarithmic order in weak coupling, adjusting the model parameters so that the Polyakov loop matches lattice values for N_c=5. The temperature dependence of the matrix model solution is modified through two ansatzes (d_A(T) and d_B(T)) to improve agreement with lattice Polyakov loop data.

## Key numbers

- Polyakov loop value at T_d: ℓ₁(T_d) = 1/2 in the matrix model solution; lattice value for N_c=3 is ℓ₁(T_d) ≈ 0.4, for N_c=5 is ℓ₁(T_d) ≈ 0.5
- The transition occurs when d(T_d) = 2π in the matrix model
- ζ/s is comparable to η/s at T_d and becomes negligible by approximately 2 T_d
- The interaction measure from the matrix model shows a peak near T_d, though the model does not reproduce the sharp peak at approximately 1.1 T_d seen in lattice data

## Context

This work extends previous matrix model analyses of the deconfining phase transition (Pisarski 2000, Dumitru 2002-2005) and computations of shear and bulk viscosities in hot gauge theories (Arnold 2000, 2003, 2006) to include nonzero holonomy effects. The teen fields, introduced as two-dimensional ghosts in the adjoint representation, follow earlier work on effective models for the semi-QGP (Hidaka 2008-2020, Pisarski 2012, Lin 2013). The results enable comparison with lattice QCD transport coefficients and provide predictions for heavy-ion collision phenomenology near the deconfinement temperature.
