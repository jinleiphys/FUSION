# Light Front Cloudy Bag Model: Nucleon Electromagnetic Form Factors

> The nucleon is modeled, using light front dynamics, as a relativistic system of three bound constituent quarks emersed in a cloud of pions. The pionic cloud is important for understanding low-momentum transfer physics, especially the neutron charge radius, but the quarks are dominant at large momentum transfer. The model achieves a very good description of existing data for the four electromagnetic elastic form factors.

# Light Front Cloudy Bag Model: Nucleon Electromagnetic Form Factors

## Key claim

The combination of Poincaré invariance (via light-front dynamics) and chiral symmetry (via a pionic cloud) is sufficient to describe existing nucleon electromagnetic form factor data up to about Q² = 20 GeV², but the model keeps only two necessary effects and omits quark-configuration mixing, Q²-dependent quark mass, exchange currents, and intermediate Δ contributions.

## Method

The nucleon is modeled as a relativistic system of three bound constituent quarks, using light-front dynamics with a spatial wave function of the form Φ(M₀) = N/(M₀² + β²)^γ, where M₀ is the mass-squared operator for a non-interacting system. Pionic cloud effects are included by evaluating Feynman diagrams for photon interactions with a bare nucleon, a nucleon with a pion present, and a charged pion in flight, using a relativistic πN form factor and performing the integrals over virtual pion four-momentum analytically in the light-front framework. The resulting form factors are expressed as F_{iα}(Q²) = Z[F_{iα}^{(0)}(Q²) + F_{ibα}(Q²) + F_{icα}(Q²)], where the wave function renormalization constant Z is fixed by requiring the proton charge to be unity.

## Key numbers

- Neutron charge radius squared: three-quark model gives R_n² = -0.025 fm²; experimental value is R_n² = -0.113 ± 0.005 fm²; the Foldy contribution alone is R_F² = -0.111 fm²
- Three-quark model parameters (Schlumpf): β = 0.607 GeV, γ = 3.5, m = 0.267 GeV
- Parameter Set 1 (solid): m = 1.8 fm⁻¹, β = 3.65 fm⁻¹, Λ = 3.1 fm⁻¹, γ = 4.1, yielding -R_n² = 0.111 fm², μ_n = -1.73, μ_p = 2.88
- Parameter Set 2 (dot-dash): m = 1.7 fm⁻¹, β = 3.4 fm⁻¹, Λ = 3.1 fm⁻¹, γ = 3.9, yielding -R_n² = 0.110 fm², μ_n = -1.79, μ_p = 2.95
- Parameter Set 3 (dash): m = 1.7 fm⁻¹, β = 2.65 fm⁻¹, Λ = 3.1 fm⁻¹, γ = 3.7, yielding -R_n² = 0.109 fm², μ_n = -1.79, μ_p = 2.95
- πN coupling constant: g²/4π = 14
- The model describes data up to approximately Q² = 20 GeV²
- The ratio QF₂/F₁ is predicted to be constant for Q² up to about 20 GeV²

## Context

This work extends the earlier three-quark light-front model of Schlumpf and the work of Frank, Jennings, and Miller (1995) that predicted the strong decrease of G_E/G_M, which was subsequently confirmed by experiments at Jefferson Lab (Jones 1999, Gayou 2001). The model also builds on the cloudy bag model of static nucleons used to compute pion cloud contributions to the neutron charge radius. The light-front cloudy bag model enables testable predictions for upcoming high-precision measurements of neutron electric and magnetic form factors (Madey, Will) and for exclusive reactions such as pp elastic scattering and γd → np at high Q².
