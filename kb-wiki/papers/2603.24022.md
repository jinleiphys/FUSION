# Precision Tests of Isospin Symmetry through Coulomb excitation of A = 62 Nuclei

> Isospin symmetry in the $A=62$ mass system was investigated through Coulomb excitation reactions at the RIKEN Radioactive Isotope Beam Factory. Beams of $^{62}$Zn, $^{62}$Ga, and $^{62}$Ge were studied using the BigRIPS-ZeroDegree-DALI2$^+$ setup under identical experimental conditions, allowing for cancellation of systematic uncertainties. Inelastic scattering cross sections measured with two different targets were used to extract nuclear deformation lengths and $E2$ matrix elements. The isospin symmetry of the $A=62$ system was rigorously tested by examining the linearity of the proton matrix elements within the triplet with high precision. The observed linear relationship between the reduced proton matrix elements for the three nuclei holds within experimental uncertainties, providing a stringent test of isospin symmetry. This experiment provides the most accurate test, to date, of isospin symmetry rules using transition matrix elements. These results were interpreted using large-scale shell-model calculations, offering valuable insights into isospin symmetry behavior in this region of the nuclear chart.

## Key claim

The linear relationship between reduced proton matrix elements \(M_p\) for the \(A=62\), \(T=1\) triplet nuclei \(^{62}\)Zn, \(^{62}\)Ga, and \(^{62}\)Ge holds within experimental uncertainties, providing the most accurate test of isospin symmetry using transition matrix elements to date, with the caveat that this precision is achieved by using nearly identical experimental conditions for all three nuclei to cancel systematic uncertainties.

## Method

Beams of \(^{62}\)Zn, \(^{62}\)Ga, and \(^{62}\)Ge produced via fragmentation of a 345 \(A\)MeV \(^{78}\)Kr primary beam were directed onto \(^{197}\)Au and \(^{12}\)C targets at the RIKEN Radioactive Isotope Beam Factory. Inelastic scattering cross sections were measured using the BigRIPS-ZeroDegree-DALI\(^{2+}\) setup, and nuclear deformation lengths \(\delta_N\) and \(E2\) matrix elements \(M_p\) were extracted by iteratively varying both parameters to reproduce the experimental cross sections for the two targets simultaneously using a modified version of the distorted wave coupled channels code Fresco.

## Key numbers

*   The \(B(E2; 0^+_1 \rightarrow 2^+_1)\) values extracted are 1456(184) \(e^2\)fm\(^4\) for \(^{62}\)Ge, 1406(233) \(e^2\)fm\(^4\) for \(^{62}\)Ga, and 1405(174) \(e^2\)fm\(^4\) for \(^{62}\)Zn.
*   The proton matrix elements \(M_p\) are 38.1(24) \(e\)fm\(^2\) for \(^{62}\)Ge, 37.5(31) \(e\)fm\(^2\) for \(^{62}\)Ga, and 37.4(23) \(e\)fm\(^2\) for \(^{62}\)Zn.
*   The average uncertainty across the triplet for the linearity test is 1.2%, representing at least a fourfold improvement in precision over lower mass \(T=1\) triplets.
*   The isoscalar and isovector matrix elements derived from the full triplet are \(M_0 = 75.4(4)\) \(e\)fm\(^2\) and \(M_1 = +0.6(4)\) \(e\)fm\(^2\).

## Context

This work extends the test of the linearity of proton matrix elements \(M_p(T_z)\) from lighter systems such as \(A=42-46\) (Boso 2019) and the \(A=70\) triplet (Wimmer 2021) to the \(A=62\) mass region. Unlike previous studies where different experimental techniques were used for each member of a triplet (Prados 2007), this experiment uses identical conditions for all three nuclei, enabling the highest precision test of isospin symmetry using transition matrix elements. The results confirm isospin symmetry in weakly collective nuclei and suggest that the \(3\sigma\) deviation from linearity observed for \(A=70\) (Wimmer 2021) is driven by deformation effects, motivating future measurements of the \(A=66\) triplet and heavier, more collective systems.
