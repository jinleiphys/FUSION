# Semirelativistic approximation to the $γ^\ast N \to N(1520)$ and $γ^\ast N \to N(1535)$ transition form factors

> The representation of the wave functions of the nucleon resonances within a relativistic framework is a complex task. In a nonrelativistic framework the orthogonality between states can be imposed naturally. In a relativistic generalization, however, the derivation of the orthogonality condition between states can be problematic, particularly when the states have different masses. In this work we study the $N(1520)$ and $N(1535)$ states using a relativistic framework. We considered wave functions derived in previous works, but impose the orthogonality between the nucleon and resonance states using the properties of the nucleon, ignoring the difference of masses between the states (semirelativistic approximation). The $N(1520)$ and $N(1535)$ wave functions are then defined without any adjustable parameters and are used to make predictions for the valence quark contributions to the transition form factors. The predictions compare well with the data particularly for high momentum transfer, where the dominance of the quark degrees of freedom is expected.

## Key claim

The paper shows that by ignoring the mass difference between the nucleon and the resonances N(1520) and N(1535) (the semirelativistic approximation), one can define the resonance wave functions using the nucleon's radial wave function without any adjustable parameters, and the resulting predictions for the valence quark contributions to the transition form factors agree well with experimental data at high momentum transfer, though the agreement is less good at low Q² where meson cloud contributions are expected to dominate.

## Method

The author uses the covariant spectator quark model, where baryons are described as quark-diquark systems, and imposes orthogonality between the nucleon and resonance states by setting both masses equal to the average M = (M_N + M_R)/2, which allows the resonance radial wave function to be taken directly from the nucleon without additional parameters. The transition form factors for γ*N → N(1520) (J^P = 3/2⁻) and γ*N → N(1535) (J^P = 1/2⁻) are then calculated from the overlap integral of the radial wave functions, and the helicity amplitudes are obtained using the physical masses of the nucleon and resonances.

## Key numbers

- For the γ*N → N(1520) transition, the experimental magnetic form factor at Q²=0 is G_M(0) = -0.393 ± 0.044 (from PDG).
- The model uses nucleon radial wave function parameters β₁ = 0.049 and β₂ = 0.717, determined from a fit to nucleon electromagnetic form factor data.
- The semirelativistic approximation describes CLAS data for the A_{1/2} and S_{1/2} helicity amplitudes for Q² > 1 GeV².
- For the γ*N → N(1520) transition, the model approaches the G_E data for Q² > 3 GeV².
- The mass difference between N(1520) and the nucleon gives K = |q| = (M_R² - M_N²)/(2M_R) at Q²=0, approximately 0.2 GeV².

## Context

This work extends the covariant spectator quark model previously applied to the nucleon (Ramalho, 2014) and to the N(1520) and N(1535) resonances (Ramalho, 2015), where the orthogonality condition required introducing a new momentum scale parameter in the radial wave functions. The semirelativistic approximation avoids this by using the nucleon's radial wave function directly, enabling parameter-free predictions for the valence quark contributions. The results enable future work to separate valence quark contributions from meson cloud effects, which are expected to dominate at low Q² based on coupled-channel reaction models (EBAC, 2010).
