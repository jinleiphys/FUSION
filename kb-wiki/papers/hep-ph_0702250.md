# Determination of fragmentation functions and their uncertainties

> Fragmentation functions and their uncertainties are determined for pion, kaon, and proton by a global $χ^2$ analysis of charged-hadron production data in electron-positron annihilation and by the Hessian method for error estimation. It is especially important that the uncertainties of the fragmentation functions are estimated in this analysis. The results indicate that the fragmentation functions, especially gluon and light-quark fragmentation functions, have large uncertainties at small $Q^2$. There are large differences between widely-used functions by KKP (Kniehl, Kramer, and Pötter) and Kretzer; however, they are compatible with each other and also with our functions if the uncertainties are taken into account. We find that determination of the fragmentation functions is improved in next-to-leading-order (NLO) analyses for the pion and kaon in comparison with leading-order ones. Such a NLO improvement is not obvious in the proton. Since the uncertainties are large at small $Q^2$, the uncertainty estimation is very important for analyzing hadron-production data at small $Q^2$ or $p_T$ ($Q^2, p_T^2 << M_Z^2$) in lepton scattering and hadron-hadron collisions. A code is available for general users for calculating obtained fragmentation functions.

## Key claim

The authors determine fragmentation functions for pions, kaons, and protons with quantified uncertainties for the first time, finding that the widely-used KKP and Kretzer functions are compatible with each other and with the new functions when the estimated uncertainties are taken into account, especially noting that gluon and light-quark fragmentation functions have large uncertainties at small Q².

## Method

The authors performed a global chi-squared analysis of charged-hadron production data from electron-positron annihilation experiments (TASSO, TPC, HRS, TOPAZ, SLD, ALEPH, OPAL, DELPHI) using the Hessian method for error estimation. Fragmentation functions were parametrized at an initial scale Q₀² = 1 GeV² using a polynomial form D_i^h(z,Q₀²) = N_i^h z^{α_i^h} (1-z)^{β_i^h}, with parameters determined by minimizing χ² using the CERN MINUIT subroutine, and Q² evolution was calculated using timelike DGLAP equations in both leading order and next-to-leading order.

## Key numbers

- Initial scale for light-quark fragmentation functions: Q₀² = 1 GeV²
- Heavy-quark masses used: m_c = 1.43 GeV, m_b = 4.3 GeV
- Lambda parameters for four flavors: Λ^{(4)}_{LO} = 0.220 GeV, Λ^{(4)}_{NLO} = 0.323 GeV
- Minimum z value for data inclusion: z_min = 0.1 for data at √s < M_Z, z_min = 0.05 for data at √s = M_Z
- Total data points used: 264 for pions, 245 for kaons, 249 for protons
- Example Δχ² value for N=14 parameters: Δχ² = 15.94 for one-sigma error range

## Context

This work extends the methodology of uncertainty estimation previously applied to unpolarized parton distribution functions (PDFs), polarized PDFs, and nuclear PDFs to the fragmentation functions for the first time. The analysis uses the same Hessian method developed in those PDF uncertainty studies (Refs. on unpolarized, polarized, and nuclear PDF errors). The results enable reliable use of fragmentation functions with quantified uncertainties in hadron-production analyses at small Q² or p_T (Q², p_T² << M_Z²) in lepton scattering and hadron-hadron collisions, and a code is made available for general users.
