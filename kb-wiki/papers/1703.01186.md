# The frozen nucleon approximation in two-particle two-hole response functions

> We present a fast and efficient method to compute the inclusive two-particle two-hole (2p-2h) electroweak responses in the neutrino and electron quasielastic inclusive cross sections. The method is based on two approximations. The first neglects the motion of the two initial nucleons below the Fermi momentum, which are considered to be at rest. This approximation, which is reasonable for high values of the momentum transfer, turns out also to be quite good for moderate values of the momentum transfer $q\gtrsim k_F$. The second approximation involves using in the "frozen" meson-exchange currents (MEC) an effective $Δ$-propagator averaged over the Fermi sea. Within the resulting "frozen nucleon approximation", the inclusive 2p-2h responses are accurately calculated with only a one-dimensional integral over the emission angle of one of the final nucleons, thus drastically simplifying the calculation and reducing the computational time. The latter makes this method especially well-suited for implementation in Monte Carlo neutrino event generators.

# The frozen nucleon approximation in two-particle two-hole response functions

## Key claim
The frozen nucleon approximation, which neglects the initial momenta of the two struck nucleons and uses a Fermi-averaged Delta propagator, can compute inclusive 2p-2h electroweak response functions with only a one-dimensional integral, but only works well when the momentum transfer is above the Fermi momentum (q > k_F) and requires tunable parameters for the Delta propagator that depend on the momentum transfer.

## Method
The authors set the initial hole momenta to zero (h1 = h2 = 0) and the initial nucleon energies to the rest mass (E1 = E2 = m_N) in the hadronic tensor integral for 2p-2h responses in the relativistic Fermi gas model. For the forward Delta current, they replace the bare Delta propagator with an effective propagator obtained by averaging over the Fermi sphere, using a non-relativistic approximation for the hole energies. The resulting response functions are computed with a single integral over the polar emission angle of one final nucleon, and the effective Delta propagator parameters (a shift Sigma_frozen and width Gamma_frozen) are fitted for each momentum transfer value.

## Key numbers
- Fermi momentum for 12C: k_F = 225 MeV/c
- Frozen Delta propagator parameters for q = 300 MeV/c: Sigma_frozen = 20 MeV, Gamma_frozen = 130 MeV
- Frozen Delta propagator parameters for q = 1000 MeV/c: Sigma_frozen = 100 MeV, Gamma_frozen = 100 MeV
- Frozen Delta propagator parameters for q = 2000 MeV/c: Sigma_frozen = 150 MeV, Gamma_frozen = 0 MeV
- The frozen approximation reduces the number of nested integrals from 7 (full calculation) to 1
- The Delta peak position for q = 1000 MeV/c occurs at omega approximately 650 MeV

## Context
This work extends the frozen nucleon approximation previously explored for the 2p-2h phase-space function (Simo 2014) to the full hadronic tensor including electroweak meson-exchange currents from the model of Simo (2016). It provides a computationally efficient alternative to the full 7D integration that is suitable for implementation in Monte Carlo neutrino event generators such as GENIE, which currently uses the IFIC Valencia model (Nieves 2011) and plans to incorporate the Lyon model (Martini 2009). The approach enables rapid calculation of 2p-2h responses without interpolating pre-calculated tables, allowing model parameters to be modified inside the codes.
