# Partial Wave Analyses of the pp data alone and of the np data alone

> We present results of the Nijmegen partial-wave analyses of all NN scattering data below Tlab = 500 MeV. We have been able to extract for the first time the important np phase shifts for both I = 0 and I = 1 from the np scattering data alone. This allows us to study the charge independence breaking between the pp and np I = 1 phases. In our analyses we obtain for the pp data chi^2_{min}/Ndf = 1.13 and for the np data chi^2_{min}/Ndf = 1.12.

# Partial Wave Analyses of the pp data alone and of the np data alone

## Key claim
The Nijmegen group succeeded for the first time in extracting the important np phase shifts for both isospin I=0 and I=1 from np scattering data alone, without borrowing I=1 phases from pp analyses. This extraction became possible only when the np partial-wave analysis was extended to energies up to Tlab = 500 MeV, whereas earlier attempts below Tlab = 350 MeV had failed.

## Method
The authors performed separate partial-wave analyses of all pp and all np scattering data below Tlab = 500 MeV by solving the relativistic Schrodinger equation with a long-range potential (including electromagnetic interaction, one-pion-exchange tail, and the Nijmegen potential) for radii r >= 1.4 fm, and an energy-dependent square-well potential for the short-range region r < 1.4 fm. The pp analysis used 36 parameters and the np analysis used 38 parameters, with up to four parameters per partial wave for the real part of the potential and an imaginary part included only in selected partial waves above threshold.

## Key numbers
- chi^2_min/Ndf = 1.13 for the pp data analysis (3555.4 chi^2_min for 3455 pp scattering data points)
- chi^2_min/Ndf = 1.12 for the np data analysis (4142.0 chi^2_min for 3968 np scattering data points)
- Energy range: all NN scattering data below Tlab = 500 MeV
- Short-range boundary radius: b = 1.4 fm

## Context
This work extends the Nijmegen group's previous partial-wave analyses of pp data below Tlab = 350 MeV (Bergervoet et al., 1990) and a preliminary pp analysis up to 500 MeV (de Kok et al., 1993). Earlier attempts to extract np phase shifts from np data alone below 350 MeV had failed, forcing analysts to borrow I=1 phases from pp analyses. The extension to 500 MeV enabled the first direct determination of all important np partial waves, allowing the study of charge independence breaking between pp and np I=1 phases beyond the well-known 1S0 channel. The results can be compared with single-energy analyses and with the work of Arndt et al. (1992), Dubois et al. (1982), and Bugg et al. (1990).
