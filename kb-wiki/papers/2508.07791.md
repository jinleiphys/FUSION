# Vector meson dominance in photon structure functions at small $x$ from holography

> We investigate the photon structure functions via the photon-photon and photon-vector meson scattering within the framework of holographic QCD, focusing on the small Bjorken $x$ region and assuming that the Pomeron exchange dominates. The quasi-real photon structure functions are formulated as the convolution of the known U(1) vector field wave function with the Brower-Polchinski-Strassler-Tan (BPST) Pomeron exchange kernel in the five-dimensional AdS space. Assuming the vector meson dominance, the photon structure functions can be calculated in a different way with the BPST kernel and vector meson gravitational form factor, which can be obtained in a bottom-up AdS/QCD model, for the Pomeron-vector meson coupling. It is shown that the obtained $F_2$ structure functions in the both ways agree with the experimental data, which implies the realization of the vector meson dominance within the present model setup. Calculations for the longitudinal structure function and the longitudinal-to-transverse ratio are also presented.

# Vector meson dominance in photon structure functions at small $x$ from holography

## Key claim

The photon structure functions calculated via photon-photon scattering and via photon-rho meson scattering (assuming vector meson dominance) both agree with OPAL experimental data and with the GRS parton distribution set, suggesting that vector meson dominance emerges naturally from the holographic framework rather than being imposed as an input; however, deviations between the two approaches are visible at very small $x$, and more data are needed to further constrain the descriptions.

## Method

The authors compute photon structure functions $F_2^\gamma$ and $F_L^\gamma$ at small Bjorken $x$ using the Brower-Polchinski-Strassler-Tan (BPST) Pomeron exchange kernel in five-dimensional AdS space, with a modified kernel incorporating confinement via parameter $z_0 = 6$ GeV$^{-1}$. For the photon-photon scattering case, the overlap functions use the five-dimensional U(1) vector field wave functions with a weight $w = 0.6198$ on the longitudinal component; for the photon-rho meson scattering case, the target overlap function is replaced by the rho meson gravitational form factor from a bottom-up AdS/QCD model. The single free parameter $g_0^2$ is determined by fitting to nine OPAL data points at $x \leq 0.025$ using the MINUIT package.

## Key numbers

- $g_0^2 = 23.82 \pm 1.49$ for the photon-photon scattering case, determined by fitting to OPAL data
- $g_0^2 = 72.58 \pm 4.51$ for the photon-rho meson scattering case, determined by fitting to OPAL data
- $z_0 = 6$ GeV$^{-1}$ for the modified BPST kernel, adopted from prior nucleon structure function studies
- $P^2 = 0.01$ GeV$^2$ assumed for the target photon virtuality
- $w = 0.6198$ applied as weight on the longitudinal component of the photon wave function
- $z_0 = 1/\Lambda_{\mathrm{QCD}} = 1/322$ MeV$^{-1}$ used in the rho meson wave function
- $m_\rho = 775.36$ MeV used for the rho meson mass in the gravitational form factor
- Structure functions computed at $Q^2 = 1.9, 3.7, 8.9, 10.7, 17.8$ GeV$^2$ for comparison with OPAL data

## Context

This work extends the holographic QCD approach to photon structure functions, building on the BPST Pomeron kernel framework previously applied to nucleon deep inelastic scattering by Brower et al. (2007, 2010) and Watanabe et al. (2012, 2015, 2019). The results are consistent with the GRS photon parton distribution functions of Gluck, Reya, and Schienbein (1999), which include hadronic components dominant at small $x$. The study demonstrates that vector meson dominance emerges naturally from the holographic setup, contrasting with conventional phenomenological approaches where the vector meson component must be introduced explicitly. Future linear collider experiments such as ILC and CEPC are identified as platforms to further test these predictions.
