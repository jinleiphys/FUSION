# A convenient implementation of the overlap between arbitrary Hartree-Fock-Bogoliubov vacua for projection

> Overlap between Hartree-Fock-Bogoliubov(HFB) vacua is very important in the beyond mean-field calculations. However, in the HFB transformation, the $U,V$ matrices are sometimes singular due to the exact emptiness ($v_i=0$) or full occupation ($u_i=0$) of some single-particle orbits. This singularity may cause some problem in evaluating the overlap between HFB vacua through Pfaffian. We found that this problem can be well avoided by setting those zero occupation numbers to some tiny values (e.g., $u_i,v_i=10^{-8}$). This treatment does not change the HFB vacuum state because $u_i^2,v_i^2=10^{-16}$ are numerically zero relative to 1. Therefore, for arbitrary HFB transformation, we say that the $U,V$ matrices can always be nonsingular. From this standpoint, we present a new convenient Pfaffian formula for the overlap between arbitrary HFB vacua, which is especially suitable for symmetry restoration. Testing calculations have been performed for this new formula. It turns out that our method is reliable and accurate in evaluating the overlap between arbitrary HFB vacua.

## Key claim

The authors claim that the singularity of the $U,V$ matrices in Hartree-Fock-Bogoliubov (HFB) transformations, which causes problems in evaluating the overlap between HFB vacua via Pfaffian, can be avoided by setting zero occupation numbers ($u_i=0$ or $v_i=0$) to a tiny value $\varepsilon=10^{-8}$, because this treatment does not change the HFB vacuum state numerically.

## Method

The authors derive a new Pfaffian formula for the overlap between arbitrary HFB vacua, starting from the work of Bertsch and Robledo (2012), by assuming that the inverse of the $V$ matrix exists after replacing zero occupation numbers with a small parameter $\varepsilon$. They transform the overlap expression into a form involving the matrix $\mathcal{W}(\Xi)$, which avoids evaluating $V^T\mathbb{D}(\Xi)V'^*$ at each mesh point in symmetry restoration calculations. Numerical tests compare the new formula with the Onishi formula for the deformed heavy nucleus $^{226}$Th using the Nilsson+BCS method.

## Key numbers

- For empty orbits with $v_i=0$, the authors set $v_i=\varepsilon=10^{-8}$ when using double precision data type, satisfying $1+\varepsilon^2=1$ numerically.
- The norm of the neutron BCS vacuum for $^{226}$Th with 96 active neutrons out of 145 levels is $\langle\phi_n|\phi_n\rangle = (10^{-16})^{(290-96)/2} = 10^{-1552}$; for protons with 70 active out of 110 levels, $\langle\phi_p|\phi_p\rangle = (10^{-16})^{(220-70)/2} = 10^{-1200}$.
- The relative difference $R = |\mathbb{N}_{\mathrm{Onishi}}(\phi_n)/\mathbb{N}_{\mathrm{pf}}(\phi_n) - 1|$ between the new formula and the Onishi formula is less than $10^{-12}$ with double precision.
- Particle number projection using 16 mesh points yields $\langle\psi_n|\hat P^N|\psi_n\rangle=1$ for $N=96$ and 0 for $N\neq96$ with numerical errors less than $10^{-13}$.
- Angular momentum projection with 100 mesh points for spins $0\leq I\leq70$ reproduces the sum rule $\sum_{N,Z,I}\langle\psi|\hat P^N\hat P^Z \hat P^I_{00}|\psi\rangle=1$ with numerical error around $10^{-12}$.

## Context

This work extends the Pfaffian-based overlap formulas for HFB vacua developed by Bertsch and Robledo (2012), Robledo (2009), and others (Oi 2012, Avez 2012), which required either inversion of the $U$ matrix or omission of empty orbits. The new formula resolves the singularity problem for empty or fully occupied orbits by a simple numerical regularization, making it applicable to arbitrary HFB vacua including pure Slater determinants. This enables more stable and efficient symmetry restoration calculations (particle number and angular momentum projection) for heavy nuclear systems such as $^{226}$Th, and can be applied in beyond-mean-field methods like the Generator Coordinate Method.
