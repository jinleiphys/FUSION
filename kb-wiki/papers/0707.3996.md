# Stability of multiquarks in a simple string model

> A simple string model inspired by the strong-coupling regime of Quantum ChromoDynamics is used as a potential for studying the spectrum of multiquark systems with two quarks and two antiquarks, with a careful treatment of the four-body problem. It is found that the ground state is stable, lying below the threshold for dissociation into two isolated mesons.

## Key claim
Using a string model for confinement that includes four-body forces (flip-flop and butterfly configurations), the ground state of a system of two quarks and two antiquarks of equal mass is found to be stable below the threshold for dissociation into two mesons, but this stability depends crucially on including angular correlations between Jacobi coordinates that were neglected in earlier work.

## Method
The authors solve the four-body Schrödinger equation using variational trial wave functions (correlated Gaussians and exponentials) for a confinement potential that is the minimum of two string configurations: a "flip-flop" term where each quark links to one antiquark, and a "butterfly" term where the two quarks form a diquark connected to an antidiquark. The calculation is performed in the non-relativistic limit, neglecting Coulomb and spin-dependent interactions, with the string tension set to unity by scaling.

## Key numbers
- For equal masses (M/m=1), the four-quark variational energy with the full string potential is E₄ = 4.639, below the two-meson threshold T₄ = 4.676.
- For the flip-flop model alone with equal masses, E₄ = 4.644 (Gaussian) or 4.872 (exponential), both below threshold.
- For the flavor-exotic system (QQ\bar{q}\bar{q}) with mass ratio M/m=5, E₄ = 3.873 with the full string potential, below threshold T₄ = 3.942.
- For the hidden-flavor system (Q\bar{Q}q\bar{q}) with M/m=2, E₄' = 4.313, above its threshold T₄' = 4.194, indicating no binding.
- The meson ground-state energy for equal masses is E₂(1,1) = 2.338 (exact) or 2.476 (variational with single exponential).

## Context
This work extends the string model of confinement from mesons and baryons (Y-shape potential) to tetraquark systems, contrasting with earlier calculations using additive pairwise potentials (the "1/2-rule" or color-dependent pairwise forces) that generally do not bind tetraquarks except at high mass ratios. The authors explicitly show that neglecting angular correlations between Jacobi coordinates, as was done in Carlson (1991), leads to the incorrect conclusion that no bound exotic states exist. The results enable future studies incorporating relativistic effects, spin-dependent terms, and Fermi statistics, and suggest that the string model provides a qualitatively different starting point for multiquark spectroscopy than additive potentials.
