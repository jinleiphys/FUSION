# A Computer Program for Relativistic Multiple Coulomb and Nuclear Excitation

> A computer program is presented by which one may calculate the multiple electric dipole, electric quadrupole and magnetic dipole Coulomb excitation with relativistic heavy ions. The program applies to an arbitrary nucleus, specified by the spins and energies of the levels and by all E1, E2 and M1 matrix elements. Nuclear excitation is calculated optionally for monopole, dipole and quadrupole excitations and needs inputs of optical potentials. For given bombarding conditions, the differential cross sections and statistical tensors (useful to calculate gamma-ray angular distribution functions) are computed.

# A Computer Program for Relativistic Multiple Coulomb and Nuclear Excitation

## Key claim
The paper presents a computer program that calculates multiple Coulomb excitation (E1, E2, M1) and optional nuclear excitation (monopole, dipole, quadrupole) for relativistic heavy-ion collisions, but the accuracy of the calculation depends on the user supplying all required matrix elements and optical potentials, and the semiclassical straight-line trajectory approximation is only valid when the projectile wavelength is much smaller than the characteristic lengths of the system.

## Method
The program solves the coupled-channels (CC) equations for time-dependent amplitudes of nuclear eigenstates using a semiclassical approach with a straight-line trajectory and dimensionless time variable τ = γvt/b. The electromagnetic interaction is expanded into multipoles (E1, E2, M1) with relativistic corrections, while nuclear excitation uses transition potentials derived from the optical potential via the Bohr-Mottelson particle-vibrator coupling model. Absorption at small impact parameters is treated either through the eikonal approximation with an optical potential or via the Glauber theory optical limit with nucleon-nucleon cross section σ_NN = 40 mb. The time integration uses an adaptive Runge-Kutta method with user-specified accuracy, and the program outputs differential cross sections and statistical tensors for gamma-ray angular distributions.

## Key numbers
- σ_NN = 40 mb: the nucleon-nucleon cross section used in the Glauber absorption formula for high-energy collisions
- ACCUR = 0.001 (0.1%): the recommended accuracy parameter for the time integration of the coupled-channels equations
- b_0 = 1.2 (A_P^(1/3) + A_T^(1/3)) fm: the grazing impact parameter around which half the impact parameter points are concentrated in a finer mesh
- Upper impact parameter limit: b = 200 fm, beyond which excitation probabilities are negligible except for very low excitation energies (E_j << 1 MeV) combined with very large bombarding energies (γ >> 1)
- The finer mesh covers the interval b_0/2 fm ≤ b ≤ 3b_0/2 fm, and the second mesh extends from 3b_0/2 fm to 200 fm

## Context
This work extends the semiclassical coupled-channels formalism of Alder and Winther (1965) for non-relativistic Coulomb excitation to the relativistic regime, incorporating the relativistic multipole interaction derived by Bertulani and collaborators (1996). The program enables calculations for arbitrary nuclei with user-specified level schemes and matrix elements, and it can be used to analyze experiments at relativistic heavy-ion facilities. The code is made publicly available and can be applied to study giant resonances, multiple excitation processes, and gamma-ray angular distributions following Coulomb excitation.
