# Solution of the Skyrme-Hartree-Fock-Bogolyubov equations in the Cartesian deformed harmonic-oscillator basis. (VIII) hfodd (v2.73y): a new version of the program

> We describe the new version (v2.73y) of the code HFODD which solves the nuclear Skyrme Hartree-Fock or Skyrme Hartree-Fock-Bogolyubov problem by using the Cartesian deformed harmonic-oscillator basis. In the new version, we have implemented the following new features: (i) full proton-neutron mixing in the particle-hole channel for Skyrme functionals, (ii) the Gogny force in both particle-hole and particle-particle channels, (iii) linear multi-constraint method at finite temperature, (iv) fission toolkit including the constraint on the number of particles in the neck between two fragments, calculation of the interaction energy between fragments, and calculation of the nuclear and Coulomb energy of each fragment, (v) the new version 200d of the code HFBTHO, together with an enhanced interface between HFBTHO and HFODD, (vi) parallel capabilities, significantly extended by adding several restart options for large-scale jobs, (vii) the Lipkin translational energy correction method with pairing, (viii) higher-order Lipkin particle-number corrections, (ix) interface to a program plotting single-particle energies or Routhians, (x) strong-force isospin-symmetry-breaking terms, and (xi) the Augmented Lagrangian Method for calculations with 3D constraints on angular momentum and isospin. Finally, an important bug related to the calculation of the entropy at finite temperature and several other little significant errors of the previous published version were corrected.

## Key claim

Version v2.73y of the HFODD code extends the previous version by implementing eleven new features, including full proton-neutron mixing in the particle-hole channel for Skyrme functionals and the Gogny force in both channels, but the code remains fully compatible with all previous versions in serial mode and the tensor-kinetic density F is not yet implemented.

## Method

The program solves the nuclear Skyrme Hartree-Fock or Skyrme Hartree-Fock-Bogolyubov problem by expanding single-particle or single-quasiparticle wave functions in a Cartesian deformed harmonic-oscillator basis, with expansion coefficients determined by iterative diagonalization of mean-field Hamiltonians or Routhians that depend nonlinearly on local or nonlocal densities. For the Gogny force implementation, matrix elements of the Gaussian potential are computed directly in configuration space using spin-isospin expansions of the density matrix and pairing tensor, with the 12-dimensional summations over harmonic-oscillator quantum numbers performed effectively by factorizing into products of one-dimensional integrals. The proton-neutron mixing implementation employs three-dimensional isocranking in isospace to enforce total isospin, adding an isocranking term to the mean-field Hamiltonian with adjustable frequencies.

## Context

This work extends the series of HFODD code publications beginning with Dobaczewski and Dudek (1997) and continuing through versions v1.60r, v1.75r, v2.08i, v2.08k, v2.40h, and v2.49t (Schunck et al., 2012). The new version enables self-consistent calculations for different members of an isobaric multiplet with full isospin mixing, as demonstrated in Sato et al. (2013), and provides a fission toolkit for studying nuclear fission including neck constraints and fragment energies. The implementation of the Gogny force follows the formalism of Dechargé and Gogny (1980), while the generalized Skyrme energy density functionals with proton-neutron mixing follow the rules defined by Perlińska et al. (2004).
