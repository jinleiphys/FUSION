# Calculation of kinetic parameters $β_{\mathit{eff}}$ and $Λ$ with modified open source Monte Carlo code OpenMC(TD)

> This work presents the methodology used to expand the capabilities of the Monte Carlo code OpenMC for the calculation of reactor kinetic parameters: effective delayed neutron fraction $β_{\mathit{eff}}$ and neutron generation time $Λ$. The modified code, OpenMC(Time-Dependent) or OpenMC(TD), was then used to calculate the effective delayed neutron fraction by using the prompt method, while the neutron generation time was estimated using the pulsed method, fitting $Λ$ to the decay of the neutron population. OpenMC(TD) is intended to serve as an alternative for the estimation of kinetic parameters when licensed codes are not available. The results obtained are compared to experimental data and MCNP calculated values for $18$ benchmark configurations.

## Key claim

The authors modified the open-source Monte Carlo code OpenMC to calculate reactor kinetic parameters (effective delayed neutron fraction and neutron generation time) as an alternative to licensed codes like MCNP, but the results show that for the effective delayed neutron fraction, OpenMC(TD) values fall within benchmark errors for only 4 out of 12 cases where experimental data exist, indicating the prompt method approximation has limited accuracy for some configurations.

## Method

The authors added a time label to neutron particles in OpenMC version 0.10 (Fortran) and implemented a time filter for tallies to monitor neutron population evolution. The effective delayed neutron fraction was calculated using the prompt method, which estimates beta_eff as 1 minus the ratio of the multiplication factor without delayed neutrons (k_p) to the effective multiplication factor (k_eff). The neutron generation time was obtained using the pulsed method, where a neutron pulse was inserted into a subcritical system and Lambda was extracted by fitting an exponential decay to the simulated neutron population decay curve.

## Key numbers

- For Godiva (bare highly enriched uranium sphere), OpenMC(TD) calculated beta_eff = 633(7) pcm compared to the experimental value of 659(10) pcm, a difference of 26(12) pcm
- For Jezebel-Pu (bare plutonium sphere), OpenMC(TD) calculated beta_eff = 181(7) pcm compared to the experimental value of 194(10) pcm, a difference of 13(12) pcm
- For TCA (light water moderated low-enriched UO2 core), OpenMC(TD) calculated beta_eff = 766(10) pcm compared to the experimental value of 771(7) pcm, a difference of 5(12) pcm
- For Godiva, OpenMC(TD) calculated Lambda = 5.699(95) x 10^-9 seconds, while MCNP gave 5.689(8) x 10^-9 seconds, a difference of -0.010(100) x 10^-9 seconds
- For TCA, OpenMC(TD) calculated Lambda = 3.742(32) x 10^-5 seconds, while MCNP gave 3.522(3) x 10^-5 seconds, a difference of -0.220(32) x 10^-5 seconds
- For Stacy-029 (water-reflected uranyl nitrate solution), OpenMC(TD) calculated Lambda = 6.147(38) x 10^-5 seconds, while MCNP gave 5.962(5) x 10^-5 seconds, a difference of -0.185(39) x 10^-5 seconds
- For the RA-6 research reactor case 2 (graphite reflected), OpenMC(TD) calculated Lambda = 3.757(27) x 10^-5 seconds, while MCNP gave 3.637(7) x 10^-5 seconds, a difference of -0.120(28) x 10^-5 seconds
- Differences between MCNP and OpenMC(TD) for Lambda were less than 7 percent for all cases except Topsy, where the difference was 8.1 percent

## Context

This work extends the capabilities of the open-source Monte Carlo code OpenMC, originally developed at MIT since 2013, to calculate reactor kinetic parameters that previously required licensed codes such as MCNP. The benchmarks were taken from the ICSBEP International Handbook of Evaluated Criticality Safety Benchmark Experiments, and results were compared to experimental data and MCNP calculated values from Leppanen and Meulekamp (2006). The modified code, OpenMC(TD), enables researchers without access to licensed codes to estimate kinetic parameters, though the prompt method approximation shows systematic differences for some configurations, particularly the SNEAK-7A and SNEAK-7B assemblies where beta_eff differences reached 76 pcm and 55 pcm respectively.
