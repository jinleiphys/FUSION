# LevelScheme: A level scheme drawing and scientific figure preparation system for Mathematica

> LevelScheme is a scientific figure preparation system for Mathematica. The main emphasis is upon the construction of level schemes, or level energy diagrams, as used in nuclear, atomic, molecular, and hadronic physics. LevelScheme also provides a general infrastructure for the preparation of publication-quality figures, including support for multipanel and inset plotting, customizable tick mark generation, and various drawing and labeling tasks. Coupled with Mathematica's plotting functions and powerful programming language, LevelScheme provides a flexible system for the creation of figures combining diagrams, mathematical plots, and data plots.

## Key claim

LevelScheme provides a flexible system for constructing level scheme diagrams and publication-quality figures within Mathematica, automating tedious aspects of diagram preparation while allowing extensive manual fine-tuning through an option-based system with dynamic scoping and inheritance.

## Method

The system implements a library of drawing objects (levels, transition arrows, labels, and general shapes) whose properties are controlled through Mathematica's option system with extensions for dynamic scoping and inheritance from a parent SchemeObject. A layered drawing system assigns elements to numbered layers (outlines/fills in layer 1, white-out boxes in layer 2, text in layer 3) to prevent overlapping elements from obscuring each other. Multiple coordinate systems (canvas, region, user) are maintained with affine transformations to support multipanel figures and collective repositioning of diagram components.

## Key numbers

- The distributed program occupies 3,051,807 bytes in tar.gz format
- The system was tested under Microsoft Windows XP, Macintosh OS X, and Linux
- Default layering assigns outlines and fills to layer 1, white-out boxes to layer 2, and text to layer 3
- The example figure code uses ImageSize -> 72*{8, 4} (576 by 288 points) with PlotRange -> {{-0.8, 6.3}, {-100, 900}}

## Context

This work extends the capabilities of Mathematica's built-in graphics display functions (Wolfram 1999) for scientific figure preparation, specifically addressing the construction of level energy diagrams used in nuclear, atomic, molecular, and hadronic physics. It complements existing specialized programs for band structure diagrams and decay schemes such as Radware (Radford 1995) and ENSDAT (Dunford 2003), while providing a more general infrastructure for multipanel figures, customizable tick marks, and integration with Mathematica's plotting functions. The system enables creation of publication-quality figures combining diagrams, mathematical plots, and data plots within a single Mathematica environment.
