# Improved superscaling description of electron and charged-current neutrino quasielastic scattering using effective mass dynamics

> We present an improved version of the Superscaling Analysis with Relativistic Effective Mass, denoted as SuSAM-v2. In the original SuSAM model, a universal scaling function was fitted to a selected set of quasielastic electron scattering (e,e') cross section data, using a phenomenological ansatz inspired by the Relativistic Mean Field model of nuclear matter. In this work, we refine the procedure by first fitting a longitudinal scaling function directly to experimental longitudinal response data. Subsequently, a separate transverse scaling function is extracted from purely transverse data, after subtracting the longitudinal contribution already determined. We find that the resulting transverse scaling function must exhibit an explicit dependence on the momentum transfer q in order to reproduce all kinematics consistently. The resulting SuSAM-v2 model simultaneously describes inclusive quasielastic cross sections and both longitudinal and transverse response functions in electron scattering. The model is then applied to neutrino-nucleus scattering, showing an improved prediction compared to the previous SuSAM-v1 version, due to a more accurate treatment of the relative contributions of the longitudinal and transverse weak nuclear responses.

## Key claim

The SuSAM-v2 model improves upon SuSAM-v1 by separately fitting a longitudinal scaling function directly to experimental longitudinal response data and then extracting a transverse scaling function from transverse data, but the transverse scaling function must exhibit an explicit dependence on the momentum transfer \(q\) to reproduce all kinematics consistently.

## Method

The authors first extracted a longitudinal scaling function \(f_L^*(\psi^*)\) by dividing experimental longitudinal response data for \(^{12}\mathrm{C}\) by the appropriate single-nucleon prefactors, fitting the result with a sum of two Gaussian functions. They then subtracted the longitudinal contribution and a two-particle-two-hole (2p2h) meson-exchange current contribution from inclusive quasielastic electron scattering cross section data to isolate the transverse response, from which they extracted a transverse scaling function \(f_T^*(\psi^*, q)\) by fitting a single Gaussian in each of 18 momentum transfer intervals. The \(q\)-dependence of the Gaussian parameters was parametrized using Fermi functions to provide a smooth analytical form.

## Key numbers

- The effective mass parameter for \(^{12}\mathrm{C}\) is \(M^* = 0.8\) (dimensionless, relative to the free nucleon mass).
- The vector potential energy for \(^{12}\mathrm{C}\) is \(E_v = 141\) MeV.
- The longitudinal scaling function fit parameters are: \(a_1 = 0.67732\), \(b_1 = 0.0\), \(c_1 = 0.497381\), \(a_2 = 0.103387\), \(b_2 = 1.05065\), \(c_2 = 0.302541\).
- The transverse scaling function peak position \(b(q)\) shifts from approximately \(b \sim 0.5\) for \(q < 240\) MeV/c to \(b = -0.25\) at \(q = 1000\) MeV/c.
- The transverse scaling function height \(a(q)\) stabilizes around \(a \approx 0.83\) for \(q \gtrsim 400\) MeV/c.
- The Fermi function parameters for \(a(q)\) include \(\alpha_1 = -0.653271\), \(\alpha_2 = 276.616\) MeV/c, \(\alpha_3 = 44.0707\) MeV/c, \(\alpha_4 = 0.823775\).

## Context

This work extends the SuSAM-v1 model (Martinez-Consentino et al., 2017; Amaro et al., 2018), which fitted a single scaling function to inclusive cross section data but failed to reproduce separated longitudinal and transverse responses. The approach parallels the SuSA-v2 framework (Gonzalez-Jimenez et al., 2014; Amaro et al., 2021), which also introduced a \(q\)-dependent transverse scaling function using relativistic mean field results, but SuSAM-v2 instead derives the transverse function phenomenologically from data after subtracting the longitudinal contribution. The model enables improved predictions for neutrino-nucleus scattering cross sections needed for neutrino oscillation experiments, particularly by providing separate control over the longitudinal and transverse weak nuclear responses.
