# Consistent description of magnetic dipole properties in transitional nuclei

> It is shown that a consistent description of magnetic dipole properties in transitional nuclei can be obtained in the interacting boson model-2 by F-spin breaking mechanism, which is associated with differences between the proton and neutron deformations. In particular, the long standing anomalies observed in the $g$-factors of the Os-Pt isotopes are resolved by a proper inclusion of F-spin breaking.

# Consistent description of magnetic dipole properties in transitional nuclei

## Key claim

The long-standing anomalies in the g-factors of Os-Pt isotopes can be resolved within the interacting boson model-2 (IBM-2) by properly including F-spin breaking, which is associated with differences between proton and neutron deformations, without resorting to anomalous boson g-factors or effective boson numbers.

## Method

The authors performed numerical diagonalization of the IBM-2 Hamiltonian using the NPBOS code for the isotopes 186-192Os and 190-198Pt, employing a Hamiltonian with proton and neutron d-boson energies, a quadrupole-quadrupole interaction, and a Majorana operator. Scalar parameters (epsilon_s, chi_s) were fitted to energy levels and E2 transitions, while vector parameters (epsilon_v, chi_v) were determined from M1 properties, using bare boson g-factors (g_pi=1, g_nu=0) throughout. The calculations were guided by analytic results from the intrinsic state formalism and 1/N expansion methods.

## Key numbers

- The amount of F-spin admixture in the low-lying levels varies between 2-4%
- For 188Os, the newly measured mixing ratio delta(E2/M1) = +7.2 +/- 1.1, which has the opposite sign to the previously published value
- Effective charges used for E2 calculations: e_pi = e_nu = 0.15 eb
- Fixed Hamiltonian parameters: kappa = -0.15 MeV (Os), -0.18 MeV (Pt), and xi = 0.17 MeV
- Scalar and vector parameters for each isotope are tabulated, with epsilon_s ranging from 0.32 to 0.58 MeV and epsilon_v ranging from -0.28 to +0.10 MeV across the Os-Pt chains

## Context

This work extends earlier IBM-2 calculations by Sambataro (1984) and others that used F-spin symmetric formulas (Eq. 2) and found large deviations from measured g-factors in transitional nuclei, particularly the Os-Pt isotopes where g(2_2+) values decrease while g(2_1+) values increase with neutron number. Previous attempts by Stuchbery (1985) and Zimmerman (1985) to include F-spin mixing via numerical diagonalization used parameters from Bijker (1980) that were insensitive to F-spin mixing. The present work resolves these discrepancies and enables future experimental tests of the predicted proton-neutron deformation ratios in the Os-Pt region, similar to measurements performed in 165Ho by Knudson (1991).
