# Investigation of ${}^9$Be from nonlocalized clustering concept

> The nonlocalized aspect of clustering, which is a new concept for self-conjugate nuclei, is extended for the investigation of the N{\not=}Z nucleus ${}^9$Be. A modified version of the THSR (Tohsaki-Horiuchi-Schuck-Röpke) wave function is introduced based on the container picture. It is found that the constructed negative-parity THSR wave function is very suitable for describing the cluster states of ${}^9$Be. Namely the nonlocalized clustering is shown to prevail in ${}^9$Be. The calculated binding energy and radius of ${}^9$Be are consistent with calculations in other models and with experimental values. The squared overlaps between the THSR wave function and the Brink+GCM wave function for the $3/2^-$ rotational band of ${}^9$Be are found to be near 96%. Furthermore, by showing the density distribution of the ground state of ${}^9$Be, the π-orbit structure is naturally reproduced by using this THSR wave function.

## Key claim

The nonlocalized clustering concept, previously applied only to self-conjugate (N=Z) nuclei, is extended to the N≠Z nucleus ⁹Be by introducing a modified THSR wave function with a phase factor e^(imφ) that gives intrinsic negative parity for m=±1; the single THSR wave function describes the 3/2⁻ rotational band with squared overlaps near 96% relative to the full Brink+GCM calculation, though the authors note they "essentially have not found anything for the ⁹Be structure which was not known before."

## Method

A modified THSR wave function was constructed for ⁹Be using creation operators for two α-clusters and one extra neutron, where the neutron creation operator includes a phase factor e^(imφ_Rn) to produce negative parity. The wave function was variationally optimized with respect to five parameters (b, β_α,xy, β_α,z, β_n,xy, β_n,z) using Volkov No. 2 central force and G3RS spin-orbit interaction, with angular-momentum projection applied to obtain states of the 3/2⁻ rotational band.

## Key numbers

- Binding energy of the 3/2⁻ ground state: -55.4 MeV (THSR), compared to -56.4 MeV (Brink+GCM) and -58.2 MeV (experimental)
- Spin-orbit contribution to ground state: -2.2 MeV for K=3/2 coupling, versus +2.2 MeV for K=1/2 coupling
- Squared overlap between THSR and Brink+GCM wave functions: 0.96 for 3/2⁻ ground state, 0.95 for 5/2⁻ excited state, 0.93 for 7/2⁻ excited state
- Excitation energies: 2.4 MeV for 5/2⁻ state (experimental 2.4 MeV), 6.8 MeV for 7/2⁻ state (experimental 6.4 MeV)
- Point-RMS radius of ground state: 2.55 fm (THSR), compared to experimental 2.45 fm
- Optimum size parameter b = 1.35 fm for α-clusters
- Optimum deformation parameters: β_α,xy = 0.1 fm, β_α,z = 4.2 fm, β_n,xy = 2.5 fm, β_n,z = 2.8 fm
- Inter-cluster distance in ⁹Be: about 3.4 fm, compared to about 4.6 fm in ⁸Be
- With b = 1.46 fm (same as Molecular Orbit model): THSR gives -54.7 MeV, MO gives -54.8 MeV, MO+GCM gives -56.1 MeV

## Context

This work extends the THSR wave function approach, previously successful for self-conjugate nuclei like ⁸Be, ¹²C (Hoyle state), and ¹⁶O (Tohsaki 2001, Funaki 2002, Yamada 2005), to the N≠Z nucleus ⁹Be. It builds on earlier studies of ⁹Be using antisymmetrized molecular dynamics (Enyo 1995) and nuclear molecular orbit models (Okabe 1977, Oertzen 1996, Itagaki 2000), but provides a new perspective where clusters and the extra neutron undergo nonlocalized motion without assuming molecular orbits a priori. The work enables future application of the THSR framework to other N≠Z nuclei and to states such as the 1/2⁺ state of ⁹Be, which the authors plan to study in a subsequent paper.
