# Compatibility of Neutrino DIS Data and Its Impact on Nuclear Parton Distribution Functions

> In global analyses of nuclear parton distribution functions (nPDFs), neutrino deep-inelastic scattering (DIS) data have been argued to exhibit tensions with the data from charged-lepton DIS. Using the nCTEQ framework, we investigate these possible tensions both internally and with the data sets used in our recent nPDF analysis nCTEQ15WZSIH. We take into account nuclear effects in the calculation of the deuteron structure function $F_2^D$ using the CJ15 analysis. The resulting nPDF fit, nCTEQ15WZSIHdeut, serves as the basis for our comparison with inclusive neutrino DIS and charm dimuon production data. Using $χ^2$ hypothesis testing, we confirm evidence of tensions with these data and study the impact of the proton PDF baseline as well as the treatment of data correlation and normalization uncertainties. We identify the experimental data and kinematic regions that generate the tensions and present several possible approaches how a consistent global analysis with neutrino data can be performed. We show that the tension can be relieved using a kinematic cut at low $x$ ($x>0.1$) and also investigate a possibility of managing the tensions by using uncorrelated systematic errors. Finally, we present a different approach identifying a subset of neutrino data which leads to a consistent global analysis without any additional cuts. Understanding these tensions between the neutrino and charged-lepton DIS data is important not only for a better flavor separation in global analyses of nuclear and proton PDFs, but also for neutrino physics and for searches for physics beyond the Standard Model.

# Compatibility of Neutrino DIS Data and Its Impact on Nuclear Parton Distribution Functions

## Key claim

The authors confirm that tensions exist between neutrino deep-inelastic scattering (DIS) data and charged-lepton DIS data in global nuclear parton distribution function (nPDF) analyses, but demonstrate that these tensions can be relieved using a kinematic cut at low x (x > 0.1) or by identifying a specific subset of neutrino data that yields a consistent global analysis without additional cuts, with the caveat that compatibility depends on the specific nPDF fitting framework including parameterization, data selection, and proton PDF baseline.

## Method

Using the nCTEQ framework, the authors performed global nPDF analyses that simultaneously fit charged-lepton DIS data, Drell-Yan data, LHC W and Z boson production data, single inclusive hadron production data from RHIC and LHC, and neutrino DIS data from CDHSW, CCFR, NuTeV, and Chorus experiments, including charm dimuon production data from NuTeV and CCFR. They applied deuteron structure function corrections using the CJ15 analysis and employed chi-squared hypothesis testing with correlated systematic uncertainties to evaluate tensions between data sets, fitting approximately 40 nuclear parameters with 27 free parameters plus 10 normalization parameters to 940 data points in the baseline fit.

## Context

This work extends previous nCTEQ analyses (Kovarik 2010) that first identified incompatibility between neutrino DIS and charged-lepton data, and contrasts with EPS nPDF framework studies (Paukkunen 2010, 2013) that found only mild tension and suggested data normalization as the cause. The analysis also builds on the nCTEQ15WZSIH analysis (Duwentaster 2021) by adding neutrino data sets not previously included, improving deuteron corrections, and allowing strange quark PDF parameters to vary freely rather than fixing them relative to light sea quarks. The results enable future global nPDF releases that can consistently incorporate neutrino DIS data for improved flavor separation, particularly for strange quark determination, and have implications for neutrino physics and beyond-Standard-Model searches.
