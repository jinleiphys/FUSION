# Quantum Mechanical Localization Effects for Bose-Einstein Correlations

> For a set of N identical massive boson wavepackets with optimal initial quantum mechanical localization, we calculate the Hanbury-Brown/Twiss (HBT) two-particle correlation function. Our result provides an algorithm for calculating one-particle spectra and two-particle correlations from an arbitrary phase space occupation (q_i,p_i,t_i)_{i=1,N} as e.g. returned by event generators. It is a microscopic derivation of the result of the coherent state formalism, providing explicit finite multiplicity corrections. Both the one- and two-particle spectra depend explicitly on the initial spatial wavepacket width which parametrizes the quantum mechanical wavepacket localization. They provide upper and lower bounds which suggest that a realistic value for this width has the order of the Compton wavelength.

## Key claim

The paper derives an algorithm for calculating one-particle spectra and two-particle Bose-Einstein correlations from an arbitrary discrete phase-space distribution of identical massive boson wavepackets, where the initial spatial wavepacket width σ is a free parameter that must be fixed by comparison to data; the abstract's suggestion that σ has the order of the Compton wavelength is qualified as a bound inferred from measurements on very small systems like p-pbar annihilation and Z0 decays.

## Method

The authors associate to each phase-space emission point (q_i, p_i, t_i) a free Gaussian wavepacket of initial spatial width σ that saturates the Heisenberg uncertainty relation, then construct symmetrized two-boson wavefunctions for each pair. From the two-particle Wigner phase-space density, they compute the detection probability for two bosons with measured momenta P1 and P2, sum over all pairs, and normalize by the corresponding probability for unsymmetrized (mixed) pairs to obtain the correlation function C(P1,P2). The large-N limit of this result reproduces the coherent-state formalism with an explicit emission function, while the full expression provides finite-multiplicity corrections.

## Key numbers

- For a pointlike source with no momentum dependence, the HBT radius parameter is R_point^HBT = σ/√2 (Eq. 20b).
- For a Boltzmann distribution of temperature T, the effective temperature of the one-particle spectrum is T_eff = T + 1/(2mσ²) (Eq. 19).
- For pions, the additional quantum contribution to the temperature from the localization width is of order 1/(2mσ²) ≈ 100 MeV.
- From pion interferometric measurements of p-pbar annihilation or Z0-decays, the inferred pion wavepacket width is σ ≈ 1 fm.
- The correlator is always smaller than 2 and equals 2 for P1 - P2 = 0.

## Context

This work extends the coherent-state formalism of Sinyukov (1973) and Chapman and Heinz (1994) by providing a microscopic derivation that includes finite-multiplicity corrections, and it addresses inconsistencies in the ad-hoc cos-prescription used by event generators such as RQMD, VENUS, and ARC, which can produce correlators below unity. The algorithm enables event generators to calculate both one-particle spectra and two-particle correlations from their phase-space output without binning, and it shows that the wavepacket width σ must be treated as a physical parameter rather than taken to infinity as is current practice; the tight bounds on σ from one-particle spectra (lower bound) and two-particle correlators (upper bound) suggest a value near the pion Compton wavelength, which can be tested against data from small systems like p-pbar annihilation (Goldhaber et al.) and Z0-decays (OPAL).
