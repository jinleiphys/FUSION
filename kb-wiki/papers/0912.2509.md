# Relationships between nonmesonic-weak-decays in different hypernuclei

> Using as a tool the s-wave approximation (sWA), this work demonstrates that the nonmesonic weak decay transition rates $Γ_{n}$ and $Γ_{p}$ can be expressed in all hypernuclei up to $^{29}_Λ$Si (and very likely in heavier ones too) in the same way as in the s-shell hypernuclei, i.e. as a linear combination of only three elementary transition rates. This finding leads to the analytic prediction that, independently of the transition mechanism, all hypernuclei that are on the stability line (N = Z), i.e. $^5_Λ$He, $^7_Λ$Li, $^9_Λ$Be, $^{11}_Λ$B, $^{13}_Λ$C, $^{17}_Λ$O, $^{29}_Λ$Si, etc should roughly have the same ratio $Γ_{n}/Γ_{p}$, the magnitude of which rapidly increases when one approaches the neutron drip-line (N >> Z), and opposite happens when one goes toward the proton drip-line (N << Z).

## Key claim

Using the s-wave approximation (sWA), the nonmesonic weak decay transition rates Γ_n and Γ_p can be expressed in all hypernuclei up to ²⁹_ΛSi (and likely heavier ones) as a linear combination of only three elementary transition rates, leading to the analytic prediction that all hypernuclei on the stability line (N = Z) should roughly have the same ratio Γ_n/Γ_p, while this ratio rapidly increases near the neutron drip-line and decreases near the proton drip-line, independently of the transition mechanism.

## Method

The author employs the independent particle shell model (IPSM) together with the s-wave approximation (sWA), which sets the relative orbital angular momentum l = 0 in the two-nucleon matrix elements. The nonmesonic weak decay transition rates Γ_n and Γ_p are expressed as linear combinations of the elementary s-shell transition rates R_N0 and R_N1, with generalized spectroscopic factors F_NJ derived for each hypernucleus. The one-meson-exchange (OME) model with π + K exchange potentials is used for numerical comparisons between exact and sWA results, including short-range correlations and finite nucleon size effects.

## Key numbers

- For ⁵_ΛHe: exact Γ_n = 0.149, Γ_p = 0.358, Γ_nm = 0.507, Γ_n/p = 0.417 (units: Γ in units of the free Λ decay rate Γ_Λ)
- For ¹²_ΛC: exact Γ_n = 0.200, Γ_p = 0.628, Γ_nm = 0.828, Γ_n/p = 0.319
- For ⁹_ΛHe: exact Γ_n = 0.265, Γ_p = 0.317, Γ_nm = 0.583, Γ_n/p = 0.836
- For ⁹_ΛC: exact Γ_n = 0.131, Γ_p = 0.676, Γ_nm = 0.807, Γ_n/p = 0.194
- For ²⁹_ΛSi: exact Γ_n = 0.311, Γ_p = 0.806, Γ_nm = 1.117, Γ_n/p = 0.385
- The sWA reproduces exact results with differences of the same order or smaller than kinematical and nonlocality effects discussed in Ba03
- The ratio Γ_n/p for ⁹_ΛHe is 2 times that of ⁹_ΛBe and 4 times that of ⁹_ΛC

## Context

This work extends the formalism of Block and Dalitz (1963) for s-shell hypernuclei to p-shell and d-shell hypernuclei, building on prior IPSM calculations by Alberico and Garbarino (2000) and Cohen (1990). The author previously showed that the asymmetry parameter a_Λ has the same value in all hypernuclei with full proton subshells (Ba07). The results enable analytic predictions for Γ_n/Γ_p ratios across the nuclear chart without detailed dynamical calculations, and establish simple relationships between decay rates of hypernuclei with the same mass number A but different neutron/proton content, which could guide future experiments at facilities like J-PARC.
