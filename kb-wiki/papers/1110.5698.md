# Effects of initial state fluctuations in the final state elliptic flow measurements using the NeXSPheRIO model

> We present a systematic study of the effects due to initial condition fluctuations in systems formed by heavy-ion collisions using the hydrodynamical simulation code NeXSPheRIO. The study was based on a sample of events generated simulating Au+Au collisions at center of mass energy of 200 GeV per nucleon pair with impact parameter ranging from most central to peripheral collisions. The capability of the NeXSPheRIO code to control and save the initial condition (IC) as well as the final state particles after the 3D hydrodynamical evolution allows for the investigation of the sensitivity of the experimental observables to the characteristics of the early IC. Comparisons of results from simulated events generated using fluctuating initial conditions and smooth initial condition are presented for the experimental observable elliptic flow parameter ($v_2$) as a function of the transverse momentum, $p_t$, and centrality. We compare $v_2$ values estimated using different methods, and how each method responds to effects of fluctuations in the initial condition. Finally, we quantify the flow fluctuations and compare to the fluctuations of the initial eccentricity of the energy density distribution in the transverse plane.

# Effects of initial state fluctuations in the final state elliptic flow measurements using the NeXSPheRIO model

## Key claim

The NeXSPheRIO hydrodynamic model shows that initial state fluctuations significantly affect elliptic flow ($v_2$) estimates, with different experimental methods responding differently to these fluctuations; the participant plane eccentricity, rather than the reaction plane eccentricity, drives the final state elliptic flow in events with fluctuating initial conditions, and the ordering $v_2\{4\} < v_2\{\mathrm{EP}\} < v_2\{2\}$ observed experimentally is reproduced only when fluctuations are included.

## Method

The authors used the NeXSPheRIO 3D hydrodynamic simulation code to generate Au+Au collisions at 200 GeV per nucleon pair, producing events with both fluctuating initial conditions (from the NeXuS microscopic model) and smooth initial conditions (averaged over many events). They calculated elliptic flow $v_2$ using five different methods: true reaction plane, event plane method, participant plane method, and two- and four-particle cumulant methods, then compared results across centrality classes from 0-10% to 40-60% and as a function of transverse momentum.

## Key numbers

- Center of mass energy: 200 GeV per nucleon pair for Au+Au collisions
- Centrality classes: 0-10% (impact parameter 0.00-4.78 fm, $\langle dN_{ch}/d\eta \rangle = 576.5 \pm 0.4$), 10-20% (4.78-6.77 fm, $393.8 \pm 0.3$), 20-30% (6.77-8.29 fm, $271.5 \pm 0.2$), 30-40% (8.29-9.57 fm, $182.3 \pm 0.1$), 40-60% (9.57-11.72 fm, $89.9 \pm 0.1$)
- Minimum of 30,000 events per centrality class, total sample exceeding half a million events
- Charged particles selected within $|\eta| < 1.0$ and $0.15 < p_t < 2.0$ GeV/c for $v_2$ calculations
- Event plane determination used particles from $2.5 < |\eta| < 4.0$ with minimum 15 charged particles per event
- The quantity $v_2\{2\}^2 - v_2\{4\}^2$ (sensitive to flow fluctuations plus non-flow) was non-zero for fluctuating IC but consistent with zero for smooth IC, with higher values for peripheral 40-60% events compared to central 0-10% events

## Context

This work extends previous studies showing that initial state fluctuations improve agreement between hydrodynamic models and experimental data (PhysRevLett.93.182301, PhysRevLett.97.202302, BrazJPhys.37.717). The NeXSPheRIO code had already been tested against experimental data (BrazJPhys.37.717, JPhysG.31.S1041). The results confirm that the event plane angle correlates more strongly with the participant plane angle than with the reaction plane angle, consistent with PhysRevC.83.034901. The study enables quantitative comparison between event-by-event hydrodynamic calculations and experimental flow fluctuation measurements, and provides a baseline for future viscous hydrodynamic studies as noted by Song et al. (PhysRevLett.106.192301).
