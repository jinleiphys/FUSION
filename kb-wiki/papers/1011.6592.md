# Axial and Vector Structure Functions for Electron- and Neutrino- Nucleon Scattering Cross Sections at all $Q^2$ using Effective Leading order Parton Distribution Functions

> We construct a model for inelastic neutrino- and electron-nucleon scattering cross sections using effective leading order parton distribution functions with a new scaling variable $ξ_w$. Non-perturbative effects are well described using the $ξ_w$ scaling variable, in combination with multiplicative $K$ factors at low $Q^2$.Our model describes all inelastic charged lepton-nucleon scattering (including resonance) data (HERA/NMC/BCDMS/SLAC/JLab) ranging from very high $Q^2$ to very low $Q^2$ and down to the photo-production region. The model describes existing inelastic neutrino-nucleon scattering measurements, and has been developed to be used in analysis of neutrino oscillation experiments in the few GeV region.

# Axial and Vector Structure Functions for Electron- and Neutrino-Nucleon Scattering Cross Sections at all $Q^2$ using Effective Leading order Parton Distribution Functions

## Key claim

The authors construct a model for inelastic neutrino- and electron-nucleon scattering cross sections using effective leading order parton distribution functions with a new scaling variable $\xi_w$, claiming that non-perturbative effects are well described using $\xi_w$ in combination with multiplicative $K$ factors at low $Q^2$, but the axial structure functions at very low $Q^2$ are not constrained by muon and electron scattering data and may differ from vector structure functions.

## Method

The authors modify GRV98 leading order parton distribution functions by replacing the standard Bjorken scaling variable $x$ with a new scaling variable $\xi_w$ that accounts for target mass corrections, dynamic higher twist effects, and higher order QCD terms. They apply separate multiplicative vector $K$ factors for valence and sea quarks to describe data down to the photoproduction limit ($Q^2=0$), and freeze PDF evolution below $Q^2=0.8$ (GeV/c)$^2$. For neutrino scattering, they add a correction factor $H(x,Q^2)$ to account for differences in higher order QCD corrections between ${\cal F}_2$ and $x{\cal F}_3$, and apply the same vector $K$ factors to the vector part of neutrino structure functions.

## Key numbers

- First iteration fit parameters: $A=0.419$, $B=0.223$, $C_{v1}=0.544$, $C_{v2}=0.431$, $C_{sea}=0.380$ (all in units of (GeV/c)$^2$), with $\chi^2/DOF = 1235/1200$ and normalization factor $N=1.011$
- Second iteration fit parameters: $A=0.621 \pm 0.009$, $B=0.380 \pm 0.004$, $C_{v1d}=0.341 \pm 0.007$, $C_{v1u}=0.417 \pm 0.024$, $C_{v2d}=0.323 \pm 0.051$, $C_{v2u}=0.264 \pm 0.015$, $C^{low-\nu}=0.218 \pm 0.015$, with $\chi^2/DOF = 2357/1717$ and $N=1.026 \pm 0.003$
- The minimum $Q^2$ value for GRV98 PDFs is 0.8 (GeV/c)$^2$; PDF evolution is frozen below this value
- For charm production in neutrino scattering, the final state quark mass $M_f=1.32$ (GeV/c)$^2$ is used
- The $H(x,Q^2)$ factor is fitted as $H(x,Q^2) = 0.914 + 0.296x - 0.374x^2 + 0.165x^3$, calculated at $Q^2=8$ (GeV/c)$^2$ using the TR-VFS scheme with MRST99 NLO PDFs

## Context

This work extends earlier duality-based models of neutrino interactions (Bodek and Yang, 2003; Bodek and Yang, 2004) that were incorporated into Monte Carlo generators including NEUT, GENIE, NEUGEN, and NUANCE. The model describes all inelastic charged lepton-nucleon scattering data from HERA, NMC, BCDMS, SLAC, and JLab, and is developed for use in neutrino oscillation experiments such as MINOS, NOvA, T2K, and MiniBooNE in the few GeV region. The approach enables a unified description of resonance production and deep inelastic scattering using quark-hadron duality with the modified scaling variable $\xi_w$ and low $Q^2$ $K$ factors, whereas other studies using unmodified GRV PDFs with the standard $\xi$ scaling variable find large deviations from duality in the resonance region.
