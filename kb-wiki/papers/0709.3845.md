# Multi-boson correlations using wave-packets II

> We investigate the analytically solvable pion-laser model, and its generalization to arbitrary multiplicity distributions. Although this kind of extension of the model is possible, the pion laser model in its original form is unique: it is the only model in its class that has an analytic solution.

## Key claim

The pion-laser model can be generalized to arbitrary multiplicity distributions, but the original form is unique: it is the only model in its class that retains both analytic solvability and physically correct intercept behavior (C₂(k,k)=2) of the correlation functions.

## Method

The authors start from the analytically solvable pion-laser model defined by Gaussian wave-packet states and a density matrix that cancels the normalization factor of multi-particle states. They introduce a general multiplicity distribution pₙ via an integral transform (Poisson transformation) of an arbitrary function H(y), and compute inclusive n-particle spectra using ring-algebra recurrence relations for the auxiliary functions Gₙ(p,q) and normalization constants ωₙ. The condition for the inclusive spectra to have a simple analytic form and physical intercept behavior is shown to be f_{n+m}=fₙfₘ, which holds only when H(y) is a Dirac delta.

## Key numbers

- In the rare gas limit (R_eff² σ_T² → ∞), the normalization constants become ωₙ = 1/n!, giving a Poisson multiplicity distribution.
- In the dense gas limit (R_eff² σ_T² → 0), the normalization constants become ωₙ = 1, yielding a geometric (negative binomial) distribution pₙ = ⟨n⟩ⁿ/(⟨n⟩+1)^{n+1}.
- The two-particle correlation function intercept is C₂(p,p) = 2 for the original model, and higher-order correlation functions satisfy Cₙ(p,...,p) = n!.
- The initial values for the recurrence relations are h₁ = 1/(π σ_T²)^{3/2}, a₁ = 1/(2σ_T²) + R_eff²/2, and g₁ = R_eff², with σ_T² = σ² + 2mT and R_eff² = R² + mT/(σ²σ_T²).

## Context

This work extends the analytically solvable pion-laser model of Pratt (1993), Zimanyi and Csorgo (1997), and Csorgo (1997), which used ring algebra to solve multi-boson symmetrization for wave-packet states. The authors show that generalizing the multiplicity distribution (e.g., to achieve a negative binomial in the rare gas limit, as proposed by Heinz, Scotto, and Zhang (2000)) destroys both the analytic simplicity and the physical intercept behavior. This establishes the original model as unique among this class of analytically solvable multi-boson correlation models, and suggests that further progress requires different factorizations of the density function.
