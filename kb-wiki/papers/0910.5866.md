# Nuclear matrix elements for double beta decay

> The present status of calculations of the nuclear matrix elements for neutrinoless double beta decay is reviewed. A proposal which allows in principle to measure the neutrinoless double beta decay Fermi matrix element is briefly described.

# Nuclear matrix elements for double beta decay

## Key claim
The paper reviews the status of nuclear matrix element calculations for neutrinoless double beta decay, arguing that while QRPA results from different groups appear to converge, shell model matrix elements are systematically smaller by up to a factor of 2, and that this discrepancy likely arises from the too-small single-particle model space used in shell model calculations rather than from deficiencies in QRPA.

## Method
The author compares nuclear matrix elements calculated using three approaches: the quasiparticle random phase approximation (QRPA) and its renormalized version (RQRPA), the nuclear shell model (NSM), and the interacting boson model (IBM). For the QRPA calculations, experimental two-neutrino double beta decay lifetimes are used to calibrate the particle-particle interaction strength, and calculations are performed with single-particle model spaces comprising 2, 3, and 5 major oscillator shells, using different treatments of short-range correlations including Jastrow functions and the unitary correlation operator method.

## Key numbers
- Shell model matrix elements for neutrinoless double beta decay are systematically smaller than QRPA results by up to a factor of 2 for lighter nuclei such as 76Ge
- The estimated matrix element for the double Fermi transition via the isobaric analog state is approximately 0.005, about a thousand times smaller than the first-leg matrix element which is approximately sqrt(N-Z)
- The ratio of Gamow-Teller to Fermi matrix elements is estimated as M_GT^0nu / M_F^0nu approximately -2.5, with an uncertainty of a few percent
- QRPA results obtained with unquenched axial coupling g_A = 1.25 and quenched g_A = 1.0 are presented, with variances substantially less than the average values

## Context
This review extends the author's previous work (Rodin 2003) that demonstrated how constraining QRPA calculations with experimental two-neutrino double beta decay lifetimes reduces the spread of results for neutrinoless double beta decay matrix elements. The paper contrasts the QRPA approach with shell model calculations (Caurier 2008) and very recent IBM results (Barea 2009). The proposal to measure the Fermi matrix element via charge-exchange reactions, exploiting the similarity between the neutrino potential and the Coulomb interaction, opens a possible experimental path to discriminate between nuclear models that differ by as much as a factor of 5 in their Fermi matrix element predictions.
