# Nucleon-nucleus scattering as a test of shell structure of some light mass exotic nuclei

> Shell model wave functions have been used to form microscopic g-folding optical potentials with which elastic scattering data from 8He, 10,11C, and 18,20,22O scattering on hydrogen has been analyzed. Those potentials, the effective two-nucleon interaction used in their formation, and the shell model details, then have been used in distorted wave approximation calculations of differential cross sections from inelastic scattering to the first excited states of five of those radioactive ions.

## Key claim

The microscopic g-folding optical potential, built from shell model wave functions and the Melbourne effective nucleon-nucleon interaction, can predict elastic scattering cross sections for exotic light ions on hydrogen without parameter adjustment, but the distorted wave approximation for inelastic scattering systematically underpredicts the magnitude of the data, requiring scaling factors that are attributed to core polarization corrections missing from the limited shell model spaces used.

## Method

The authors constructed microscopic g-folding optical potentials by folding the Melbourne energy- and density-dependent effective nucleon-nucleon g-matrices with one-body density matrix elements (OBDME) from shell model calculations of the ground states of 8He, 10,11C, and 18,20,22O. Elastic scattering cross sections were calculated using these potentials, and inelastic scattering to the first excited states was treated in the distorted wave approximation (DWA) using the same effective interaction as the transition operator and the same single-nucleon wave functions. The shell model spaces used were a complete (0+2+4)h-bar-omega space for 8He, a (0+2)h-bar-omega space for 10,11C, and a 0h-bar-omega space with the USD interaction for the oxygen isotopes.

## Key numbers

* 8He: root mean square radius of 2.63 fm from the (0+2+4)h-bar-omega shell model, in agreement with the value of 2.6 fm extracted from high energy data using a cluster model (Al1998).
* 8He: elastic scattering at 15.7A MeV predicted well by the g-folding model using Woods-Saxon single-nucleon wave functions, with no parameter adjustment.
* 10C: inelastic scattering to the 2+ state at 3.354 MeV at 45.3A MeV required a scaling factor of 1.5 on the Woods-Saxon DWA result, corresponding to a polarization charge of approximately 0.11 e.
* 11C: inelastic scattering to the 5/2- state at 4.32 MeV at 40.6A MeV required a scaling factor of 1.4 on the Woods-Saxon DWA result.
* 18O: inelastic scattering to the 2+ state at 43A MeV required a scaling factor of 2.
* 20O: inelastic scattering to the 2+ state at 43A MeV required a scaling factor of 5.
* 22O: inelastic scattering to the 2+ state at 46.6A MeV required a scaling factor of 1.6.

## Context

This work extends the Melbourne g-folding approach (Amos 2000), previously applied to stable nuclei and to 6He (Lapoux 2001, Stiliaris 2002), to a set of exotic light ions with recent scattering data. It directly reanalyzes data that had been treated with the JLM phenomenological model by Skaza (2005) for 8He, Jouanne (2005) for the carbon isotopes, and Beaumel (2006) and Khan (2000) for the oxygen isotopes, where parameter adjustments masked structural tests. The results demonstrate that the g-folding method provides a more stringent test of shell model wave functions, and the systematic scaling factors needed for inelastic transitions quantify the core polarization missing from the limited model spaces, pointing toward the need for larger shell model calculations or explicit treatment of core polarization.
