# Blast-wave model description of the HBT radii measured in $pp$ collisions at the LHC energies

> Talk presented at the XI Workshop on Particle Correlations and Femtoscopy, dedicated to Professor Jan Pluta on the occasion of his 70th birthday

## Key claim

The blast-wave model, assuming a "shell-like" transverse density profile, can describe the ALICE HBT radii data for pionic systems in high-multiplicity $pp$ collisions at LHC energies, provided the freeze-out temperature is fixed to 100 MeV and the shell width to 0.75 fm, leaving only the source radius and freeze-out time as free parameters per multiplicity class.

## Method

The author applies a hydro-inspired blast-wave model with boost-invariant longitudinal expansion and Hubble-like transverse flow, where pions freeze out at a fixed proper time. The source function uses a Gaussian shell density profile $f(r) \sim e^{-(r-R)^2/\delta^2}$ with radius $R$ and width $\delta$. The average transverse momentum from CMS data fixes the relation between $\omega R$ and $R/\delta$, leaving $R$ and $\tau_f$ as free parameters fitted to ALICE HBT radii via $\chi^2$ minimization across seven multiplicity classes.

## Key numbers

- Freeze-out temperature fixed at $T = 100$ MeV; shell width fixed at $\delta = 0.75$ fm.
- For multiplicity class 1–11 ($\langle N_c \rangle = 6.3$): source radius $R = 1.15$ fm, freeze-out time $\tau_f = 1.90$ fm.
- For multiplicity class 42–51 ($\langle N_c \rangle = 45.6$): $R = 2.49$ fm, $\tau_f = 2.80$ fm.
- For multiplicity class 23–28 at $P_\perp = 0.163$ GeV: model $R_{\rm side} = 1.26$ fm (experiment $1.30 \pm 0.12$ fm), model $R_{\rm out} = 1.32$ fm (experiment $1.18 \pm 0.17$ fm).
- For multiplicity class 23–28 at $P_\perp = 0.648$ GeV: model $R_{\rm side} = 0.89$ fm (experiment $0.91 \pm 0.12$ fm), model $R_{\rm out} = 0.65$ fm (experiment $0.48 \pm 0.13$ fm).
- $\chi^2$ values range from 0.96 (class 1–11) to 11.63 (class 42–51) with 13 degrees of freedom.

## Context

This work extends the application of hydro-inspired blast-wave models, previously used for heavy-ion collisions, to high-multiplicity $pp$ collisions at the LHC, following earlier studies by Ferroni (2011), Bozek (2013), and Ghosh (2014). The shell-like density profile resolves the "HBT puzzle" where $R_{\rm out}$ becomes smaller than $R_{\rm side}$ at high transverse momentum, similar to observations by Csorgo (1999) in NA22 data. The results enable further thermal and statistical model interpretations of hadron production in small systems, building on the Therminator code developed by Kisiel (2005) and the Krakow-Warsaw collaboration.
