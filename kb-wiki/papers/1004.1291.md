# Quark model description of the tetraquark state X(3872) in a relativistic constituent quark model with infrared confinement

> We explore the consequences of treating the X(3872) meson as a tetraquark bound state. As dynamical framework we employ a relativistic constituent quark model which includes infrared confinement in an effective way. We calculate the decay widths of the observed channels X-> Jpsi+2π(3π) and X-> \bar D0+D0+\pi0 via the intermediate off--shell states X-> Jpsi+ρ(ω) and X-> \bar D + D*. For reasonable values of the size parameter of the X(3872) we find consistency with the available experimental data. We also discuss the possible impact of the X(3872) in a s-channel dominance description of the Jpsi-dissociation cross section.

## Key claim

The X(3872) meson can be described as a tetraquark bound state within a relativistic constituent quark model that includes infrared confinement, and for reasonable values of the size parameter Lambda_X between 3 and 4 GeV, the calculated decay widths are consistent with the experimental bound of less than 2.3 MeV, though the decay width is quite sensitive to the choice of this size parameter.

## Method

The authors employed a relativistic constituent quark model with infrared confinement, using a nonlocal four-quark interpolating current for the X(3872) as a J^PC=1^{++} tetraquark state with symmetric spin distribution [cq]_{S=0}[cbar qbar]_{S=1} + [cq]_{S=1}[cbar qbar]_{S=0}. The coupling strength of the X(3872) to its constituent quarks was determined from the compositeness condition Z_H=0, and the decay widths for X -> J/psi + 2pi(3pi) and X -> D0bar + D0 + pi0 were calculated via intermediate off-shell states X -> J/psi + rho(omega) and X -> D + D* using the narrow width approximation.

## Key numbers

- Mass of X(3872): M_X = 3871.51 +/- 0.22 MeV (new average from CDF)
- Binding energy relative to D0 + D*0 threshold: M_X - (M_D*0 + M_D0) = -0.30 +/- 0.34 MeV
- Binding energy relative to J/psi + rho threshold: m_X - (m_J/psi + m_rho) = -0.90 +/- 0.41 MeV
- Ratio of branching fractions: B(X -> J/psi pi+ pi- pi0) / B(X -> J/psi pi+ pi-) = 1.0 +/- 0.4 (stat) +/- 0.3 (syst)
- Mixing angle determined from ratio: theta = +/- 18.4 degrees
- Decay width Gamma(X -> J/psi + n pi) decreases from 0.30 to 0.07 MeV as Lambda_X varies from 3 to 4 GeV
- Decay width Gamma(X_l -> D0bar D0 pi0) decreases from 1.88 to 0.41 MeV as Lambda_X varies from 3 to 4 GeV
- Theoretical ratio Gamma(X -> D0bar D0 pi0) / Gamma(X -> J/psi pi+ pi-) = 6.0 +/- 0.2
- Experimental ratio from PDG: 10.5 +/- 4.7
- Maximum cross section for J/psi + rho -> X -> D + D* (charged) is about 0.32 mb at E = 3.88 GeV

## Context

This work provides an independent analysis of the X(3872) as a tetraquark state, extending the approach of Maiani (2004) who first proposed the tetraquark interpretation and found a width of 1.6 MeV using an effective three-meson Lagrangian. The authors use their relativistic constituent quark model with infrared confinement, previously developed by Branz (2009), which treats multiquark systems on the same footing as quark-antiquark states. The results contradict the QCD sum rule calculations by Nielsen (2009) which found widths around 50 MeV, and the work enables further analysis of the J/psi dissociation cross section via the s-channel contribution of the X(3872) tetraquark state.
