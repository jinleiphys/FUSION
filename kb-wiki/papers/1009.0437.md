# A numerical algorithm for the explicit calculation of SU(N) and SL(N,C) Clebsch-Gordan coefficients

> We present an algorithm for the explicit numerical calculation of SU(N) and SL(N,C) Clebsch-Gordan coefficients, based on the Gelfand-Tsetlin pattern calculus. Our algorithm is well-suited for numerical implementation; we include a computer code in an appendix. Our exposition presumes only familiarity with the representation theory of SU(2).

## Key claim

The paper presents an algorithm for the explicit numerical calculation of SU(N) and SL(N,C) Clebsch-Gordan coefficients that is well-suited for numerical implementation, but it is limited to cases where the user can specify any N and any two irreducible representations; the algorithm does not provide a closed-form analytic generalization of Racah's formula for SU(2) to arbitrary N, S, and S'.

## Method

The algorithm uses the Gelfand-Tsetlin pattern calculus to label irreducible representations and basis states of SU(N), then generalizes the SU(2) strategy of solving a homogeneous linear system for the highest-weight state coefficients using the raising operator condition, followed by applying lowering operators to obtain coefficients for all lower-weight states. The matrix elements of the raising and lowering operators are taken from the closed-form expressions of Gelfand and Tsetlin (1950). The outer multiplicities for the tensor product decomposition are determined using the Littlewood-Richardson rule.

## Key numbers

- The dimension formula for an SU(N) irrep S = (m_{k,N}) is given by dim(S) = product over 1 ≤ k < k' ≤ N of (1 + (m_{k,N} - m_{k',N})/(k' - k)).
- For SU(2), the outer multiplicity N^{S''}_{S,S'} equals 1 for |S - S'| ≤ S'' ≤ S + S' and 0 otherwise.
- The SU(2) dimension formula reduces to dim(j) = 2j + 1.
- The Gelfand-Tsetlin matrix element for J^{(l)}_- (Eq. 4.1) involves products over k' from 1 to l+1 and l-1, with factors of the form (m_{k',l+1} - m_{k,l} + k - k' + 1), and the matrix element for J^{(l)}_+ (Eq. 4.2) involves similar products with shifted indices.

## Context

This work extends the Gelfand-Tsetlin (1950) pattern calculus for SU(N) representations to provide a practical numerical algorithm for Clebsch-Gordan coefficients, which previously had explicit formulas only for special cases such as those by Vilenkin (1991, 1992) for the defining representation. The algorithm enables efficient numerical treatment of SU(N)-symmetric quantum impurity models using the numerical renormalization group (Toth, 2008) and could enable SU(N)-symmetric implementations of the density matrix renormalization group (McCulloch, 2002, 2007) or tensor network models (Singh, 2009). The authors note that previous algorithmic attempts by Williams (1994), Rowe (1997), and Gliske (2007) were either insufficiently general or required more advanced mathematical methods.
