# Two-fermion relativistic bound states in Light-Front Dynamics

> In the Light-Front Dynamics, the wave function equations and their numerical solutions, for two fermion bound systems, are presented. Analytical expressions for the ladder one-boson exchange interaction kernels corresponding to scalar, pseudoscalar, pseudovector and vector exchanges are given. Different couplings are analyzed separately and each of them is found to exhibit special features. The results are compared with the non relativistic solutions.

## Key claim

The paper presents the formalism and numerical solutions for two-fermion bound states in the explicitly covariant version of Light-Front Dynamics, but the results are limited to J=0 and J=1 states and the only physical system considered is positronium, with each coupling (scalar, pseudoscalar, pseudovector, vector) analyzed separately to study their special features rather than to produce a complete physical model.

## Method

The authors solve the Light-Front Dynamics two-body wave function equation in the ladder approximation for two fermions interacting via one-boson exchange kernels corresponding to scalar, pseudoscalar, pseudovector, and vector couplings. Analytical expressions for the interaction kernels are derived, and the angular momentum problem is addressed by constructing states as superpositions of A^2 eigenstates, with coefficients determined by requiring the wave function to become independent of the light-front direction in the k→0 limit. The coupled equations for the wave function components are solved numerically for J=0 and J=1 states.

## Key numbers

- The normalization condition for the two-fermion wave function is given by (m/(2π)^3) ∫ (f_1^2 + f_2^2) d^3k/ε_k = 1, where f_1 and f_2 are the two independent wave function components for J=0 states
- For J=0 states, the number of independent amplitudes N = 2, following from N = (1/2)(2J+1)(2σ_1+1)(2σ_2+1) with σ_1 = σ_2 = 1/2 and parity conservation
- For J=1 states, the number of independent amplitudes N = 6, following from the same formula with J=1
- The kernel for vector coupling without tensor term (f_t=0) simplifies to K = (g_v^2)/(4m^2) * 1/(μ^2+Q^2) * { [ū(k_1)γ^α u(k'_1)][ū(k_2)γ_α u(k'_2)] - (ττ'/μ^2)[ū(k_1)ω̂ u(k'_1)][ū(k_2)ω̂ u(k'_2)] }

## Context

This work extends the authors' previous studies of two-scalar bound states in Light-Front Dynamics (Mangin-Brinet et al., 2000; Karmanov et al., 2001) to the two-fermion case, and builds upon earlier perturbative light-front solutions for the NN system (Carbonell and Karmanov, 1995) that were applied to deuteron electromagnetic form factors measured at Jefferson Laboratory. The formalism enables future extensions to two-body scattering solutions and three-particle systems, with the three-boson case already considered in a separate work.
