# Schwinger functions and light-quark bound states, and sigma terms

> We explore the viability of using solely spacelike information about a Schwinger function to extract properties of bound states. In a concrete example it is not possible to determine properties of states with masses \gsim 1.2 GeV. Modern Dyson-Schwinger equation methods supply a well-constrained tool that provide access to hadron masses and σ-terms. We report values of the latter for a range of hadrons. Of interest is analysis relating to a u,d scalar meson, which is compatible with a picture of the lightest 0^{++} as a bound state of a dressed-quark and -antiquark supplemented by a material pion cloud. A constituent-quark σ-term is defined, which affords a means for assessing the flavour-dependence of dynamical chiral symmetry breaking.

# Schwinger functions and light-quark bound states, and sigma terms

## Key claim
Using only spacelike information about a Schwinger function, it is possible to extract reliable bound-state properties only for states with masses below approximately 1.2 GeV; for heavier states, Padé approximant fits to spacelike data produce misleading results even when the data are perfect and noiseless.

## Method
The authors solve the inhomogeneous Bethe-Salpeter equation for the pseudoscalar vertex in QCD using a renormalisation-group-improved rainbow-ladder truncation of the Dyson-Schwinger equations. They then apply diagonal Padé approximants of varying order to the spacelike vertex function to extract pole positions and residues, testing the method first on a model vertex with known analytic structure and known bound-state masses and residues. Sigma terms are computed via the derivative of each hadron mass with respect to the current-quark mass, using solutions of the quark gap equation and homogeneous Bethe-Salpeter equation.

## Key numbers
- Bound states with masses above approximately 1.2 GeV cannot be reliably extracted from spacelike data alone (concrete example with model parameters).
- Ground-state pion mass: 0.14 GeV (model input); first radial excitation mass: 1.06 GeV (from homogeneous BSE).
- Sigma term for the ground-state pion: sigma_pi = 0.069 GeV; for the first radial excitation: sigma_{pi1}^{RL} = 0.018 GeV.
- Rainbow-ladder scalar meson mass: m_sigma^{RL} = 0.675 GeV; after including pi-pi loop correction: m_sigma^{RL+pi pi} = 0.624 GeV.
- Scalar meson sigma term from rainbow-ladder: sigma_sigma^{RL} = 0.025 GeV; after one-loop pi-pi correction: sigma_sigma^{Re(1-loop)} = 0.0073 GeV.
- Scalar meson pole position after self-consistent loop treatment: sqrt(s_sigma) = 0.578 - i 0.311 GeV.
- Constituent-quark masses: M^E_{u,d} = 0.42 GeV, M^E_s = 0.56 GeV, M^E_c = 1.57 GeV, M^E_b = 4.68 GeV.
- Dimensionless sigma-term ratios: sigma_pi/m_pi = 0.498, sigma_{pi1}/m_{pi1} = 0.017, sigma_sigma/m_sigma = 0.013, sigma_rho/m_rho = 0.021, sigma_omega/m_omega = 0.034, sigma_N/m_N = 0.064, sigma_Delta/m_Delta = 0.041.
- Constituent-quark sigma-term ratios: sigma_{u,d}/M^E_{u,d} = 0.023, sigma_s/M^E_s = 0.230, sigma_c/M^E_c = 0.637, sigma_b/M^E_b = 0.851.

## Context
This work extends the Dyson-Schwinger equation studies of Maris and Roberts (1997) and the systematic truncation scheme of Munczek (1995) and Bender (1996) by testing the limits of extracting bound-state information from spacelike data alone, a question relevant to lattice-QCD simulations that are restricted to Euclidean momenta. The sigma-term calculations improve upon the earlier estimates of Flambaum (2005) for the rho and omega mesons and provide new results for the pion radial excitation and scalar meson. The analysis of the scalar meson supports the picture that the lightest 0++ state is a dressed-quark-antiquark bound state with a substantial two-pion cloud, consistent with the phenomenological difficulties discussed by the Particle Data Group. The constituent-quark sigma terms provide a new tool for quantifying the flavour dependence of dynamical chiral symmetry breaking across the quark mass spectrum.
