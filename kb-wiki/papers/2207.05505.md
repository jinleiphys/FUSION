# Systematics of fully heavy dibaryons

> We systematically study the mass spectra of the fully heavy dibaryons in an extended chromomagnetic model, which includes both the colorelectric and chromomagnetic interactions. We find no stable state below the corresponding baryon-baryon thresholds. Besides the masses, we also estimate the relative width ratios of the two-body decay channels. We hope our study will be of help for future experiments.

## Key claim

The extended chromomagnetic model predicts no stable fully heavy dibaryon state below the corresponding baryon-baryon thresholds, contrary to some lattice QCD results that found bound states for the $\Omega_{ccc}\Omega_{ccc}$ and $\Omega_{bbb}\Omega_{bbb}$ systems.

## Method

The authors employed an extended chromomagnetic model Hamiltonian containing both colorelectric and chromomagnetic interactions, with parameters previously fitted from conventional meson and baryon masses. They constructed wave functions for all possible fully heavy dibaryon configurations ($c^6$, $b^6$, $c^5b$, $b^5c$, $c^4b^2$, $b^4c^2$, and $c^3b^3$), diagonalized the Hamiltonian to obtain mass spectra, and estimated relative decay widths by projecting dibaryon wave functions onto two-baryon color-singlet components.

## Key numbers

- $cccccc$ $0^+$ state mass: 9684.8 MeV, which lies above the $\Omega_{ccc}\Omega_{ccc}$ threshold by approximately 113.5 MeV
- $bbbbbb$ $0^+$ state mass: 28680.7 MeV, which lies above the $\Omega_{bbb}\Omega_{bbb}$ threshold by approximately 61.3 MeV
- $cccccb$ $1^+$ state mass: 12862.1 MeV (lightest in this system)
- $bbbbbc$ $1^+$ state mass: 25526.0 MeV (lightest in this system)
- $ccccbb$ $2^+$ state mass: 15999.0 MeV (lowest in this system)
- $bbbbcc$ $2^+$ state mass: 22330.9 MeV (lowest in this system)
- $cccbbb$ $0^+$ scattering state mass: 19090.4 MeV, with 98.8% overlap with the $\Omega_{ccc}\Omega_{bbb}$ channel
- $cccbbb$ $0^+$ genuine dibaryon mass: 19297.9 MeV
- $cccbbb$ $1^+$ genuine dibaryon mass: 19244.3 MeV
- Relative width ratio for $D(cccccb,12862.1,1^+)$: $\Gamma(\Omega_{ccc}\Omega_{ccb}^*)/\Gamma(\Omega_{ccc}\Omega_{ccb}) \sim 1.004$
- Relative width ratio for $D(bbbbbc,25526.0,1^+)$: $\Gamma(\Omega_{bbb}\Omega_{bbc}^*)/\Gamma(\Omega_{bbb}\Omega_{bbc}) \sim 1.3$
- Relative width ratio for $D(ccccbb,16129.9,0^+)$: $\Gamma_{\Omega_{ccb}^*\Omega_{ccb}^*}:\Gamma_{\Omega_{ccb}\Omega_{ccb}} \sim 4.4$

## Context

This work extends the chromomagnetic model previously applied to tetraquarks (Weng 2020, Guo 2021), pentaquarks (Weng 2019, An 2020), and baryonia (Liu 2021) to the fully heavy dibaryon sector. The results contradict lattice QCD findings by Lyu (2021) that predicted a loosely bound $\Omega_{ccc}\Omega_{ccc}$ state (binding energy about 5.68 MeV) and by Mathur (2022) that found a deeply bound $\Omega_{bbb}\Omega_{bbb}$ state (binding energy about 89 MeV), while they agree with the Diffusion Monte Carlo study by Alcaraz-Pelegrina (2022) and the work by Richard (2020) that found no bound states. The predicted masses and decay patterns provide guidance for future experimental searches for fully heavy dibaryons at facilities such as LHCb.
