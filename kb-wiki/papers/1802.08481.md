# A femtoscopic Correlation Analysis Tool using the Schrödinger equation (CATS)

> We present a new analysis framework called "Correlation Analysis Tool using the Schrödinger equation" (CATS) which computes the two-particle femtoscopy correlation function $C(k)$, with $k$ being the relative momentum for the particle pair. Any local interaction potential and emission source function can be used as an input and the wave function is evaluated exactly. In this paper we present a study on the sensitivity of $C(k)$ to the interaction potential for different particle pairs: p-p, p-$\mathrmΛ$, $\mathrm{K^-}$-p, $\mathrm{K^+}$-p, p-$\mathrmΞ^-$ and $\mathrmΛ$-$\mathrmΛ$. For the p-p Argonne $v_{18}$ and Reid Soft-Core potentials have been tested. For the other pair systems we present results based on strong potentials obtained from effective Lagrangians such as $χ$EFT for p-$\mathrmΛ$, Jülich models for $\mathrm{K(\bar{K})}$-N and Nijmegen models for $\mathrmΛ$-$\mathrmΛ$. For the p-$\mathrmΞ^-$ pairs we employ the latest lattice results from the HAL QCD collaboration. Our detailed study of different interacting particle pairs as a function of the source size and different potentials shows that femtoscopic measurements can be exploited in order to constrain the final state interactions among hadrons. In particular, small collision systems of the order of 1~fm, as produced in pp collisions at the LHC, seem to provide a suitable environment for quantitative studies of this kind.

## Key claim

The paper presents CATS as a tool that can compute the two-particle femtoscopy correlation function exactly for any local interaction potential and emission source function, but the key hedge is that the tool's utility for constraining final state interactions depends on the source size: small collision systems of the order of 1 fm, as produced in pp collisions at the LHC, provide a suitable environment for quantitative studies, while larger systems typical of heavy-ion collisions may not be as sensitive.

## Method

CATS numerically solves the Schrödinger equation for a two-particle system by performing a partial wave decomposition, using an Euler method with an adaptive grid to compute the radial wave function, and matching the numerical solution to asymptotic free or Coulomb wave functions to obtain correct normalization and phase shifts. The correlation function is then computed by convolving the squared wave function with an emission source function, which can be either an analytical parametrization (e.g., Gaussian) or extracted from transport model output in OSCAR 1997 A format. The tool accounts for different spin, isospin, and total angular momentum states by defining interaction channels with user-provided weights based on degeneracy formulas, and sums over all contributing partial waves up to a convergence limit.

## Key numbers

* For p-p pairs, the Argonne v18 and Reid Soft-Core potentials were tested as input interaction potentials.
* For pp collisions at LHC energies, the extracted source can be smaller than 1.5 fm, and the EPOS transport model predicts a non-Gaussian source peaking at distances of around 1 fm.
* For heavy-ion collisions, the source radius is typically between 2 and 6 fm.
* The HADES collaboration extracted a source radius for p-p pairs in p-Nb reactions at sqrt(s_NN) = 3.18 GeV, and CATS was used to fit the same data.
* The paper presents results for correlation functions of p-p, p-Lambda, K--p, K+-p, p-Xi-, and Lambda-Lambda pairs, using potentials from chiEFT for p-Lambda, Jülich models for K(bar-K)-N, Nijmegen models for Lambda-Lambda, and HAL QCD lattice results for p-Xi-.

## Context

This work extends the femtoscopy formalism beyond the limitations of existing tools like the Correlation Afterburner (CRAB), which only delivers asymptotic solutions at large relative distances leading to incorrect correlation functions at small distances, and the Lednický-Lyuboshits analytical model, which assumes a Gaussian source and uses the effective range approximation sensitive only to the asymptotic region of the interaction. The Koonin model (1977) used a Gaussian source with Reid-Soft Core and Coulomb potentials but was limited to p-p correlations. CATS enables exact numerical solutions valid at short distances, which is critical for small collision systems like pp at the LHC, and can be linked to external fitting frameworks for data analysis, potentially allowing global analyses of femtoscopic data across different collision systems and particle pairs.
