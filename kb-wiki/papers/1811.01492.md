# Neutron width statistics using a realistic description of the neutron channel

> A basic prediction of the statistical model of compound nucleus reactions is that the partial widths for decay into any open channel channel fluctuate according to the Porter-Thomas distribution (PTD). A recent experiment on $s$- and $p$-wave neutron scattering from platinum isotopes found that the experimental $s$-wave partial neutron width distributions deviated substantially from the PTD. Several explanations for this finding have been proposed within the statistical model, but none has resolved this issue. Here, we review the application of a realistic resonance-reaction model to $s$-wave neutron scattering from $^{194}$Pt. Our main conclusion is that the PTD provides an excellent description of the partial neutron width distribution, provided that the secular energy dependence of the average neutron width is correctly described. Within a realistic range of model parameters, there can be a near-threshold bound or virtual state of the neutron channel that changes this secular dependence from the usual $\sqrt{E}$ dependence, as proposed by Weidenmüller [1]. In this case, the use of the $\sqrt{E}$ dependence to analyze the data will lead to apparent deviations from the PTD. We discuss the limited parameter range where such a near threshold state can have a significant effect.

## Key claim

The Porter-Thomas distribution (PTD) provides an excellent description of the partial neutron width distribution for s-wave neutron scattering from 194Pt, provided that the secular energy dependence of the average neutron width is correctly described; a near-threshold bound or virtual state of the neutron channel can change this secular dependence from the usual sqrt(E) form, and using the sqrt(E) dependence to analyze data in such a case will lead to apparent deviations from the PTD.

## Method

The authors constructed a resonance-reaction model combining a realistic description of the s-wave neutron channel (discretized radial Schrodinger equation with a Woods-Saxon potential) with a GOE description of the internal compound-nucleus states, coupled at a single spatial site. They calculated resonance energies and partial neutron widths by solving for S-matrix poles iteratively, using 100 GOE realizations with 160 resonances each, for six parameter sets varying the neutron channel potential depth and the channel-CN coupling strength.

## Key numbers

- Average resonance spacing D = 82 eV and total gamma decay width Gamma_gamma = 72 meV for n + 194Pt, taken from the RIPL-3 database
- Baseline model used Woods-Saxon parameters (r0, a) = (1.27, 0.67) fm from Bohr (1969) and V0 = -44.54 MeV, with coupling v0 = 11.0 keV-fm^(1/2) tuned to reproduce the RIPL-3 neutron strength function S0 = 2 x 10^(-4) eV^(-1/2) at 8 keV
- For parameter set M4 (V0 = -41.15 MeV, v0 = 1.6 keV-fm^(1/2)), a near-threshold bound state exists with energy E0 approximately -2 keV
- Reduced chi-squared values for reduction B (using sqrt(E) dependence) were chi^2_r = 1.0 for baseline, 5.8 for M4, 6.0 for M5, and 6.1 for M6, where chi^2_r approximately 1 indicates a good fit to the PTD
- For set M4 in reduction B, the best-fit chi-squared distribution had nu_fit = 0.92 degrees of freedom, compared to nu = 1 for the PTD
- Average elastic scattering cross section at 8 keV was 30 b for baseline and 279 b for M4; average capture cross section over 5-7.5 keV was 0.44 b for baseline and 0.47 b for M4 (experimental value 0.6 b from Koehler (2013))

## Context

This work extends the statistical model of compound nucleus reactions, which predicts Porter-Thomas fluctuations for partial widths (Porter 1956, Mitchell 2010), and addresses a discrepancy found in an experiment on neutron scattering from platinum isotopes (Koehler 2010). The authors test two proposed explanations: a near-threshold bound or virtual state modifying the energy dependence of the average width (Weidenmuller 2010), and non-statistical interactions among resonances (Celardo 2011, Fyodorov 2015, Volya 2015). The model rules out explanation (ii) (non-statistical interactions) and shows that explanation (i) can produce apparent PTD violation only within a limited parameter range, suggesting that a measurement of the elastic scattering cross section could help confirm or rule out the existence of such a near-threshold state.
