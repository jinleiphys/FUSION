# Volume extrapolation via eigenvector continuation

> We develop an extension of eigenvector continuation (EC) that makes it possible to extrapolate simulations of quantum systems in finite periodic boxes across large ranges of box sizes. The formal justification for this approach, which we call finite-volume eigenvector continuation (FVEC), is provided by matching periodic functions at different box sizes. As concrete FVEC implementation we use a discrete variable representation based on plane-wave states and present several applications calculated within this framework.

## Key claim

The paper develops an extension of eigenvector continuation (EC) called finite-volume eigenvector continuation (FVEC) that enables extrapolation of quantum system simulations across large ranges of periodic box sizes, with the formal justification provided by matching periodic functions at different box sizes through dilatation operations, though the method's accuracy depends on the choice and number of training data points.

## Method

The authors construct a vector space that accommodates states with arbitrary periodicities by defining dilatation operators that map functions between different box sizes, then use these to define inner products and matrix elements between states from different volumes. They implement FVEC using a discrete variable representation (DVR) based on plane-wave states, constructing Hamiltonian and norm matrices from training data at selected box sizes and solving the generalized eigenvalue problem at the target volume.

## Key numbers

- For a two-body system with Gaussian potential (R=2, V0=-4.0 in natural units), FVEC using training data at L=6,7,8,9 with four states each (16 total) achieved about 4% deviation between FVEC and exact calculation for the ground state at L=20
- For the three-boson resonance system, training data at five box sizes L=21,22,23,24,25 fm with N=28 and 8 states per volume (40 total) reproduced the resonance at E_R=-5.32(1) MeV, consistent with the reference value E_R=-5.31 MeV with half width 0.12 MeV
- For the three-neutron system with momentum cutoff Lambda=250 MeV and nn scattering length a_nn=-18.9 fm, training data at L=19,20,21 fm with N=22 DVR calculations were used, with the DVR basis dimension dim(B)=28,344,960
- The FVEC calculation for the three-boson system required roughly 40 matrix-vector products compared to approximately 1100 for exact calculations, providing a speedup factor of roughly 28 for a single box size

## Context

This work extends eigenvector continuation, first introduced by Frame (2017), beyond simple parametric dependencies of the Hamiltonian to handle volume extrapolation in finite-box simulations. It builds on Luscher's finite-volume relations for extracting infinite-volume properties from finite-volume calculations and connects to recent work on few-body resonances in finite volume by Klos (2018) and Dietz (2021). The method enables more efficient studies of few-body resonances and can benefit lattice QCD extrapolations to infinite volume via matching to effective field theory descriptions as discussed by Detmold (2021).
