# The link between two-body model space and many-body model space

> An exact relation which links the ideal model space to be used in A-body calculations when the two-body interaction is given in a truncated model space is derived. Its implications on the effective field theory (EFT) approach to no-core-shell-model (NCSM) is analyzed. Some insights regarding whether details of two-body interaction becomes less important in the calculations of many-body system are given. The result suggests that there might be a way to establish an EFT expansion for heavy nuclei and nuclear matter with an effective intereaction which has a much simpler form than the nucleon-nucleon (NN) interaction in the vacuum.

## Key claim

The paper derives an exact relation linking the two-body model space cutoff to the required many-body model space, showing that for an A-body system the maximum Jacobi momentum grows as (A-1) times the two-body cutoff, which implies that to describe physics up to a fixed energy scale, the required two-body cutoff decreases drastically with increasing particle number.

## Method

The author uses Jacobi coordinates to analyze the maximum possible relative momenta in an A-body system when the two-body interaction is restricted to a finite momentum cutoff Lambda. By considering an extreme alignment of all particles where each successive particle has a relative momentum increment of 2Lambda with respect to particle 1, the maximum value of the (N-1)th Jacobi momentum is derived analytically. This result is then translated into the maximum number of harmonic oscillator excitations for the A-body system using the relation between Jacobi momentum and oscillator quantum numbers.

## Key numbers

- Maximum (N-1)th Jacobi momentum: |pi_{N-1}^{max}| = (N-1)Lambda, where Lambda is the two-body momentum cutoff
- Total maximum HO model space for an A-particle system: N_A^{max} = [A(A-1)(2A-1)/6] * (Lambda^2 / (M hbar omega)), where N_2^{max} = Lambda^2/(M hbar omega)
- Maximum total accumulated momentum for A-body system: P_A^{max} = A(A-1)Lambda/2, showing A^2 growth
- For a three-body system, only 1/3 of the two-body cutoff Lambda is needed to describe a three-body observable at the same energy scale as a two-body observable

## Context

This work extends the program of the Arizona group (2007-2012) advocating for interactions defined directly in finite model spaces using effective field theory, as carried out recently in Refs. [oak, hobet, hobet2, yang16]. The derived relation provides a theoretical foundation for why pion-exchange effects can be integrated out in many-body systems, as demonstrated for pure neutron matter where the equation of state up to twice saturation density follows a simple parametrization governed by the Bertsch parameter (2000). The result enables establishing an EFT expansion for heavy nuclei and nuclear matter with simpler effective interactions than the vacuum nucleon-nucleon interaction, and supports recent attempts to apply unitarity-based pionless EFT to light nuclei calculations.
