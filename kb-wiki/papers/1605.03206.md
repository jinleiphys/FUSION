# Radiative nucleon capture with quasi-separable potentials

> We study radiative capture reactions using quasi-separable potentials. This procedure allows an easier treatment of non-local effects that can be extended to three-body problems. Using this technique, we calculate the neutron and proton radiative capture cross sections on $^{12}$C. The results obtained are shown to be in good agreement with the available experimental data.

# Radiative nucleon capture with quasi-separable potentials

## Key claim
Quasi-separable potentials can reproduce experimental neutron and proton radiative capture cross sections on 12C with good agreement, provided the continuum wavefunctions are derived from phase shifts that match experimental scattering data. The method is presented as an alternative to standard Woods-Saxon potential models, with the advantage of easier treatment of non-local effects and extension to three-body problems, but the bound state wavefunctions are still calculated using Woods-Saxon potentials for convenience.

## Method
The authors calculate continuum wavefunctions for nucleon-12C scattering using quasi-separable potentials with parameters (beta and lambda) fitted to reproduce experimental phase shifts for s, p, and d partial waves. Bound state wavefunctions for 13C and 13N are obtained separately using Woods-Saxon potentials with depths adjusted to match the binding energies of each state. The radiative capture cross sections for electric dipole (E1) transitions are then computed using the reduced matrix element formalism, with spectroscopic factors taken from the literature to scale the cross sections.

## Key numbers
- Neutron separation energies for 13C states: 4.95 MeV (ground state 1/2-), 1.86 MeV (1st excited state 1/2+), 1.27 MeV (2nd excited state 3/2-), 1.09 MeV (3rd excited state 5/2+)
- Proton separation energy for 13N ground state (1/2-): 1.94 MeV
- Woods-Saxon parameters for 13C: r0 = 1.236 fm, a = 0.62 fm, Vso = -7 MeV; potential depths: 41.35 MeV (1/2-), 56.90 MeV (1/2+), 28.81 MeV (3/2-), 56.85 MeV (5/2+)
- Woods-Saxon parameters for 13N: r0 = 1.25 fm, a = 0.65 fm, Vso = -10 MeV; potential depth: 41.65 MeV
- Spectroscopic factors (C2S) for 13C: 0.77 (ground state), 1.0 (1st excited state), 0.14 (2nd excited state), 0.58 (3rd excited state)
- Spectroscopic factor for 13N ground state: 0.21
- Proton resonance in s-wave at 0.422 MeV center-of-mass energy for p-12C scattering
- Fitting parameter beta for s1/2: 0.57 fm-1 (neutron), 0.5723 fm-1 (proton, adjusted to place resonance at 0.422 MeV)
- Fitting parameter beta for p1/2 in 13C: 1.76 fm-1 (modified from 1.72 fm-1 for best fit)

## Context
This work extends the quasi-separable potential technique developed by Cattapan et al. (1980s) for nucleon-nucleus scattering to radiative capture reactions of astrophysical interest. The reactions 12C(n,gamma)13C and 12C(p,gamma)13N are important in the CNO cycle in stars, including cold and hot CNO cycles in massive stars, novae, and x-ray bursts. Previous calculations using Woods-Saxon potential models (e.g., Mengoni et al., Huang et al.) and other direct capture models have been used for these reactions; the present method offers a path toward consistent treatment of non-local effects and extension to three-body systems via Faddeev-AGS equations, with applications to deuteron-induced reactions in progress.
