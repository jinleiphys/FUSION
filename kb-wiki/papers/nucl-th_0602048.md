# MOMDIS: a Glauber model computer code for knockout reactions

> A computer program is described to calculate momentum distributions in stripping and diffraction dissociation reactions. A Glauber model is used with the scattering wavefunctions calculated in the eikonal approximation. The program is appropriate for knockout reactions at intermediate energy collisions (30 MeV $\leq$ E$_{lab}/$nucleon $\leq 2000$ MeV). It is particularly useful for reactions involving unstable nuclear beams, or exotic nuclei (e.g. neutron-rich nuclei), and studies of single-particle occupancy probabilities (spectroscopic factors) and other related physical observables. Such studies are an essential part of the scientific program of radioactive beam facilities, as in for instance the proposed RIA (Rare Isotope Accelerator) facility in the US.

## Key claim

MOMDIS is a Fortran-77 computer code that calculates momentum distributions for stripping and diffraction dissociation reactions using a Glauber model with eikonal approximation, specifically designed for intermediate-energy knockout reactions (30 MeV ≤ E_lab/nucleon ≤ 2000 MeV) involving unstable nuclear beams, but the code's accuracy depends on the validity of the eikonal approximation and the t-ρρ method for obtaining eikonal phase-shifts, with medium modifications such as Pauli blocking optionally included.

## Method

The code solves the radial Schrödinger equation for bound states using Numerov integration with matching at the nuclear surface, then calculates eikonal S-matrices using the t-ρρ approximation with nucleon-nucleon scattering amplitudes parameterized from high-energy data. Momentum distributions for stripping and diffraction dissociation are obtained by expanding the S-matrices as a sum of Gaussian functions (typically N=20 terms) and performing the resulting integrals analytically where possible, with Simpson's method used for the remaining numerical integrations.

## Key numbers

- Energy range: 30 MeV ≤ E_lab/nucleon ≤ 2000 MeV for which the code is appropriate
- Memory required: 16 Mbytes of RAM and 2 MB of hard disk space
- Code size: 4320 lines of Fortran-77
- Typical running time: up to 30 minutes on a 2 GHz Intel P4-processor machine for transverse momentum distributions
- Gaussian expansion uses N=20 complex coefficients α_j with R_L = 10-20 fm for realistic S-matrices
- Spin-orbit potential uses pion Compton wavelength ℏ/m_πc = 1.414 fm

## Context

This code extends the theoretical framework developed by Hussein and McVoy (1985) and Bertulani and McVoy (1992) for knockout reactions, incorporating later refinements by Alkhalili and Tostevin (1996) and Bonaccorso and collaborators (1988-2004). The code enables extraction of spectroscopic factors and orbital angular momentum information from momentum distributions measured at radioactive beam facilities, and is particularly relevant for the proposed Rare Isotope Accelerator (RIA) facility in the US.
