# Two-cluster approach to the properties of one- and two-neutron-halo nuclei

> In this work, we present a new approximate method for obtaining simple wave functions for the ground state of exotic nuclei with a neutron halo. We model the system as a two-cluster structure, treating the core and halo as inert objects. The relative wave function is expressed as a combination of simple harmonic oscillator states, with the oscillator parameter determined from the separation energy. Since these wave functions lack the expected exponential decay, we introduce a simple multiplicative renormalization factor based on the nuclear root mean square radius. This approach, combining oscillator wave functions and the renormalization factor, is then applied to calculate dipole strength distributions $dB(E1,\varepsilon)/d\varepsilon$ and Coulomb dissociation cross section $dσ(E1,\varepsilon)/d\varepsilon$ for several $1n$- and $2n$-halo nuclei. The results show excellent agreement with the available experimental data.

## Key claim

The authors present a simplified two-cluster model using harmonic oscillator wave functions with a multiplicative renormalization factor, claiming that despite the Gaussian (rather than exponential) decay of the wave functions, the approach yields dipole strength distributions in "excellent agreement" with experimental data for both one- and two-neutron halo nuclei, with only a single free parameter (the mixing between 0s and 0d orbitals).

## Method

The projectile is modeled as an inert core plus a cluster of valence neutron(s), with the relative wave function expressed as a combination of harmonic oscillator states. The oscillator parameter is determined from the separation energy via the relation R0 = sqrt(3 hbar^2 / (2 mu epsilon0)). Since the harmonic oscillator wave functions lack the expected exponential tail, a multiplicative renormalization factor C0 is applied to the radial part of the electric dipole operator, extracted from the root-mean-square distance between core and valence cluster. The dipole strength distribution dB(E1)/d epsilon is then calculated using Eq. (13) with a combination of 0s and 0d orbitals, fitting the spectroscopic weights to experimental data.

- For 15C (14C+n): R0 = 7.43 fm, C0^2 = 0.34, calculated B(E1) = 0.73 e^2 fm^2 (experimental: 0.53(5), 0.77(7) e^2 fm^2), calculated r_cv^rms = 4.36 fm (experimental: 4.5(2) fm), calculated S1 = 2.46 e^2 fm^2 (cluster sum rule: 2.55 e^2 fm^2)
- For 6He (4He+2n): R0 = 5.42 fm, C0^2 = 0.55, calculated B(E1) = 1.53 e^2 fm^2 (experimental: 1.2(2), 1.6(2) e^2 fm^2), calculated r_cv^rms = 3.79 fm (experimental: 3.36(39), 3.9(2) fm), calculated S1 = 8.81 e^2 fm^2 (cluster sum rule: 4.95 e^2 fm^2)

- The peak in dB(E1)/d epsilon for 6He is reproduced at 1.4 MeV

## Context

This work extends the cluster orbital shell model approximation (COSM/COSMA) developed by Zhukov (1993), Suzuki (1988, 1990, 1991), and collaborators, providing a simpler alternative to full three-body calculations for core+n+n systems such as those by Matsumoto (2004), Casal (2013), and Fortunato (2014). The approach is motivated by the observation of Moro (2007) that a simplified two-body (core-(2n)) description effectively captures key reaction dynamics for Coulomb dissociation of two-neutron halo nuclei on heavy targets. The method enables rapid calculation of dipole strength distributions for halo nuclei without full continuum discretization, and the authors suggest future application to heavier nuclei and proton-drip-line systems.
