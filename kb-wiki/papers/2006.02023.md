# Extended parity doublet model with a new transport code

> A new transport code "DaeJeon Boltzmann-Uehling-Uhlenbeck (DJBUU)" had been developed and enables to describe the dynamics of heavy-ion collisions in low-energy region. To confirm the validity of the new code, we first calculate Au + Au collisions at Ebeam = 100 and 400A MeV and also perform the box calculation to check the detail of collisions and Pauli blocking without mean-field potential as suggested by the Transport Code Comparison Project. After confirming the validity of new transport code, we study low-energy heavy-ion collisions with an extended parity doublet model. Since the distinctive feature of the parity doublet model is the existence of the chiral invariant mass that contributes to the nucleon mass, we investigate how physical quantities depend on the chiral invariant mass in heavy ion collisions at low energies. For this, we calculate physical quantities such as the effective nucleon mass in central collisions and transverse flow in semi-central collisions of Au + Au at Ebeam = 400A MeV with different values of the chiral invariant masses.

## Key claim
The paper demonstrates that the newly developed DJBUU transport code produces results consistent with the Transport Code Comparison Project (TCCP) within model uncertainties, and then uses this validated code to investigate how heavy-ion collision observables depend on the chiral invariant mass in the extended parity doublet model, where the chiral invariant mass is uncertain and its value is not yet well understood.

## Method
The authors developed the DaeJeon Boltzmann-Uehling-Uhlenbeck (DJBUU) transport code, which solves the relativistic BUU equation using a test particle method with 100 test particles per nucleon and 10 independent simulations. They validated the code by comparing Au+Au collisions at Ebeam = 100 and 400A MeV and box calculations against TCCP results, then implemented the extended parity doublet model to study the dependence of effective nucleon mass and transverse flow on the chiral invariant mass.

## Key numbers
- Au+Au collisions at Ebeam = 100A MeV (B-mode) and 400A MeV (D-mode) with impact parameter b = 7 fm
- For B-Full mode: slope parameter = 46.5 ± 5.3 MeV/c (compared to BUU average 51 ± 11 MeV/c)
- For D-Full mode: slope parameter = 126.1 ± 8.7 MeV/c (compared to BUU average 143 ± 19 MeV/c)
- Pauli blocking factor at √s = 1.9 GeV: B-Cascade = 0.677, B-Full = 0.700, D-Full = 0.630
- Box calculation collision rates: CT0 initial = 112.8 c/fm, CT5 initial = 116.8 c/fm; saturated rates: 110.2 c/fm (T=0) and 113.8 c/fm (T=5)
- Fitted Boltzmann temperatures from box calculation: TB = 14.355 MeV (T0) and 15.399 MeV (T5)
- Chiral invariant mass range considered: 600 to 900 MeV, with incompressibility K = 215 MeV and K = 240 MeV

## Context
This work extends the Transport Code Comparison Project (TCCP) studies by Xu (2016) and Zhang (2017) by introducing a new BUU code and applying it to the extended parity doublet model. The parity doublet model was formulated by Detar (1988) and Jido (2001), and the chiral invariant mass was previously estimated to be around 800 MeV by Zschiesche (2006) using nuclear matter properties, while Motohiro (2015) found a range of 500 to 900 MeV. This work enables future constraints on the chiral invariant mass through heavy-ion collision observables.
