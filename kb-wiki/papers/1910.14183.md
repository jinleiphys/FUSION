# Prospects of Event Shape Sorting

> Event Shape Sorting is a novel method which is devised to organise a sample of collision events in such a way, that events with similar final state distribution of hadrons end up sorted close to each other. Such events are likely to have evolved similarly. Thus the method allows to focus at finer features of the collision evolution because it would allow for averages over similar events that do not wash away these features. The algorithm is shortly explained. We also point out the distinction of Event Shape Sorting from the well established technique of Event Shape Engineering.

# Prospects of Event Shape Sorting

## Key claim

Event Shape Sorting (ESS) can identify correlations between second-order and third-order event planes that would remain hidden in standard Event Shape Engineering (ESE), even when the correlator ⟨e^{i{θ₂−θ₃}min}⟩ takes the same value as for uncorrelated planes; however, the method's sensitivity to statistical fluctuations remains unresolved and may obscure genuine physical signals.

## Method

The ESS algorithm iteratively sorts collision events by comparing their azimuthal hadron distributions without requiring a pre-defined sorting observable. Events are first rotated to align second-order event planes, then initially sorted by an observable such as q₂, divided into deciles, and each event is reassigned to the decile whose summed histogram most probably produced it, using Bayes' theorem; the process repeats until convergence. The method was tested on simulated events from the DRAGON blast-wave Monte Carlo generator and from AMPT, using reference particles for sorting and test particles for measuring anisotropic flow.

## Key numbers

- DRAGON simulations used temperature T = 120 MeV, transverse radius R = 7 fm, Bjorken proper time τ = 10 fm/c, and transverse flow gradient ρ₀ = 0.8.
- Second-order anisotropies in shape (a₂) and flow (ρ₂) fluctuated within intervals (0, 0.1); third-order anisotropies (a₃, ρ₃) fluctuated within (0, 0.03).
- Four event sets were generated with 100%, 60%, 30%, and 10% of events having correlated second- and third-order event planes, each containing 100,000 events.
- For the artificial correlated case where only two values ±δ of (θ₂−θ₃) occur, the correlator ⟨e^{i{θ₂−θ₃}min}⟩ = 3/π gives δ ≈ 0.301.
- AMPT events: 10,000 events of Au+Au collisions at √s_NN = 200 GeV with impact parameters between 7 and 10 fm.
- Reference particles: charged hadrons with |y| < 0.4; test particles: charged hadrons with 0.5 < |y| < 0.8.

## Context

This work extends the concept of Event Shape Engineering (ESE), introduced in prior literature, by proposing an unsupervised sorting method that does not require a pre-defined sorting observable. The authors demonstrate that ESS can discover correlations between second- and third-order event planes that ESE with q₂ as the sorting variable would miss, as shown in simulations with DRAGON. The method enables future searches for unexpected event features in heavy-ion collisions, though the influence of statistical fluctuations on ESS requires further investigation before it can be reliably applied to experimental data.
