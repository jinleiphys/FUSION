# Relation between the phenomenological interactions of the algebraic cluster model and the effective two--nucleon forces

> We determine the phenomenological cluster--cluster interactions of the algebraic model corresponding to the most often used effective two--nucleon forces for the $^{16}$O + $α$ system.

## Key claim

The phenomenological cluster–cluster interaction parameters of the vibron model (with U(3) dynamical symmetry) can be related to the effective two–nucleon forces used in microscopic cluster models by equating the corresponding SU(3) matrix elements, but the analytical expressions are too complicated even for the simplest closed–shell case; instead, the numerical values of the phenomenological parameters can be obtained easily from the microscopic forces, and for the four forces tested the average root–mean–square deviation between the microscopic and phenomenological matrix elements is modest (1.15–1.79 MeV) over an energy range spanning more than 100 MeV.

## Method

The authors equate the matrix elements of the vibron model Hamiltonian (expressed in terms of Casimir operators of the U(4) ⊃ U(3) ⊃ O(3) chain) with the SU(3) matrix elements of the microscopic cluster model for the ¹⁶O + α system, calculated using the generator coordinate method (GCM) with Slater determinants of harmonic oscillator single–particle orbits. They compute the microscopic matrix elements for four effective two–nucleon interactions (Volkov V2, Hasegawa–Nagata HN1 and HN2, and a modified Hasegawa–Nagata force MHN) and then extract the phenomenological parameters β, γ, δ, ε by fitting to 101 matrix elements with nπ from 8 to 20.

## Key numbers

- For the Volkov V2 force (ν = 0.16 fm⁻², Majorana parameter M_k = 0.62), the extracted phenomenological parameters are β = 0.0493 MeV, γ = 12.289 MeV, δ = –0.119 MeV, ε = –91.329 MeV, with an average rms deviation d = 1.15 MeV between microscopic and phenomenological matrix elements.
- For the HN1 force: β = 0.0720 MeV, γ = 13.776 MeV, δ = –0.149 MeV, ε = –123.28 MeV, d = 1.79 MeV.
- For the HN2 force: β = 0.0687 MeV, γ = 13.042 MeV, δ = –0.136 MeV, ε = –106.01 MeV, d = 1.77 MeV.
- For the MHN force: β = 0.0562 MeV, γ = 12.415 MeV, δ = –0.121 MeV, ε = –93.145 MeV, d = 1.46 MeV.
- The best fit to the experimental spectrum of ²⁰Ne (Kπ = 0₁⁺, 0⁻, 0₄⁺, 0₅⁺ bands) gave β = 0.161 MeV, γ = 13.601 MeV, δ = –0.571 MeV, ε = –71.040 MeV.
- The Wildermuth condition for the ¹⁶O + α system gives nπ ≥ 8.

## Context

This work extends the semimicroscopic algebraic cluster model approach of Cseh and Levai (1988, 1989, 1991) by providing a direct numerical mapping between the phenomenological vibron model parameters and the effective two–nucleon forces used in microscopic GCM calculations (e.g., Matsuse, Kamimura and Fukushima, 1975). It enables future studies to derive algebraic cluster–cluster interactions from microscopic forces for other systems, and to test whether the simple one– and two–body vibron Hamiltonian can approximate more realistic interactions. The paper notes that the experimental spectrum requires larger β and δ values than those obtained from any of the four microscopic forces tested, indicating that the phenomenological parameters fitted to data produce stronger splitting in L and nπ than the microscopic forces predict.
