# Softness of Sn isotopes in relativistic semi-classical approximation

> Within the frame-work of relativistic Thomas-Fermi and relativistic extended Thomas-Fermi approximations, we calculate the giant monopole resonance (GMR) excitation energies for Sn and related nuclei. A large number of non-linear relativistic force parameters are used in this calculations. We find that a parameter set is capable to reproduce the experimental monopole energy of Sn isotopes, when its nuclear matter compressibility lies within $210-230$ MeV, however fails to reproduce the GMR energy of other related nuclei. That means, simultaneously a parameter set can not reproduce the GMR values of Sn and other nuclei.

# Softness of Sn isotopes in relativistic semi-classical approximation

## Key claim
A relativistic mean-field parameter set can reproduce the experimental giant monopole resonance (GMR) excitation energies of Sn isotopes only when its nuclear matter compressibility lies within 210–230 MeV, but the same parameter set simultaneously fails to reproduce the GMR energies of other related nuclei (such as Pb isotopes) with the same accuracy.

## Method
The authors used the relativistic Thomas-Fermi (RTF) and relativistic extended Thomas-Fermi (RETF) approximations, which are semi-classical approaches to the non-linear sigma-omega model, to calculate GMR excitation energies via both scaling and constraint methods. They applied these formalisms to Sn isotopes (112–124), Cd isotopes, and Pb isotopes using multiple non-linear relativistic force parameter sets (NL-SH, NL1, NL2, NL3, FSUGold) and compared the resulting momentum ratios (m3/m1)^(1/2) and (m1/m-1)^(1/2) with experimental data from RCNP and with QRPA calculations.

## Key numbers
- Nuclear matter compressibility range for Sn isotope GMR reproduction: 210–230 MeV (from the abstract and summary)
- FSUGold compressibility: 230.28 MeV; NL1 compressibility: 210 MeV; T5 parameter set compressibility: 202 MeV; T6 parameter set compressibility: 236 MeV
- For 208Pb, experimental GMR energy: 14.17 ± 0.1 MeV; RETF(FSUG) result: 14.04 MeV
- For 112Sn, experimental (m3/m1)^(1/2): 16.7 ± 0.2 MeV; QRPA(T5): 16.6 MeV; RETF(FSUG): 17.42 MeV; RETF(NL1): 15.86 MeV
- For 124Sn, experimental (m3/m1)^(1/2): 15.8 MeV; RETF(FSUG): 16.67 MeV; RETF(NL1): 15.1 MeV
- For 204Pb, experimental (m1/m-1)^(1/2): 13.7 ± 0.1 MeV; RETF result: 13.6 MeV; pairing+MEM result: 13.4 MeV
- For 208Pb, experimental (m1/m-1)^(1/2): 13.5 ± 0.1 MeV; RETF result: 13.44 MeV; pairing+MEM result: 13.4 MeV
- Difference between RETF and RTF results for Sn isotopes: approximately 0.2 MeV for NL3 parameter set
- Rho-meson contribution to GMR in RETF for 112Sn with FSUG: 21.85 MeV; in RTF: 0.00467 MeV

## Context
This work extends the ongoing debate about the "softness" of Sn isotopes, which began with experiments by U. Garg and collaborators (2007) showing that Sn isotopes require a lower nuclear matter compressibility than other nuclei. The authors compare their semi-classical results with quantal QRPA calculations by V. Tselyaev et al. (2009) using T5 and T6 parameter sets, and with pairing plus mutually enhancement model (MEM) calculations by Khan et al. (2009). The study shows that the RETF approximation reproduces Pb isotope data better than pairing+MEM approaches, but confirms that no single parameter set can simultaneously describe GMR energies across Sn, Cd, and Pb isotopes, leaving the softening of Sn as an open problem requiring further theoretical work.
