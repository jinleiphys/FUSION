# Solution of Faddeev integral equations in configuration space using the hyperspherical harmonics expansion method

> A method has been developed to solve three-particle Faddeev equations in the configuration space making use of a series expansion in hyperspherical harmonics. The following parameters of the bound state of triton and helium-3 nuclei have been calculated: the binding energies, the weights of symmetric and mixed-symmetry components of the wave function, the magnetic moments, and the charge radii.

# Solution of Faddeev integral equations in configuration space using the hyperspherical harmonics expansion method

## Key claim

The method combines the Faddeev approach with a hyperspherical harmonic expansion in configuration space to reduce the three-particle problem to a system of one-dimensional integral equations, yielding bound-state properties of triton and helium-3 that agree reasonably with experiment for the semirealistic potentials tested, though the calculated magnetic moments and charge radii systematically deviate from experimental values.

## Method

The author expands the difference between the total three-particle wave function and the asymptotic wave function into a series of hyperspherical harmonics (eigenfunctions of the angular part of the six-dimensional Laplace operator) and substitutes this expansion into the Faddeev equations in configuration space. Using the orthonormality and completeness relations for the hyperspherical harmonics and for the radial eigenfunctions of the kinetic energy operator, the system is reduced to a set of one-dimensional integral equations for the radial expansion coefficients. For the triton and helium-3 bound states, the expansion is truncated to hypermoment values K = 0, 2, 4, and the resulting coupled integral equations are solved iteratively using semirealistic nucleon-nucleon potentials (Malfliet–Tjon, Volkov, and Eikemeier–Hackenbroich).

## Key numbers

- Triton binding energy: 7.981 MeV (Malfliet–Tjon), 7.665 MeV (Volkov), 8.942 MeV (Eikemeier–Hackenbroich); experimental value 8.482 MeV
- Helium-3 binding energy: 7.241 MeV (Malfliet–Tjon), 6.951 MeV (Volkov), 8.165 MeV (Eikemeier–Hackenbroich); experimental value 7.719 MeV
- Triton symmetric component weight (K=0): 95.91% (Malfliet–Tjon), 95.75% (Volkov), 97.03% (Eikemeier–Hackenbroich)
- Triton mixed-symmetry component weight: 1.42% (Malfliet–Tjon), 1.71% (Volkov), 1.21% (Eikemeier–Hackenbroich)
- Triton magnetic moment: 2.748 nuclear magnetons (Malfliet–Tjon), 2.739 (Volkov), 2.755 (Eikemeier–Hackenbroich); experimental value 2.979
- Triton charge radius: 1.667 fm (Malfliet–Tjon), 1.692 fm (Volkov), 1.644 fm (Eikemeier–Hackenbroich); experimental value 1.755 fm
- Helium-3 magnetic moment: -1.865 nuclear magnetons (Malfliet–Tjon), -1.855 (Volkov), -1.871 (Eikemeier–Hackenbroich); experimental value -2.127
- Helium-3 charge radius: 1.814 fm (Malfliet–Tjon), 1.777 fm (Volkov), 1.737 fm (Eikemeier–Hackenbroich); experimental value 1.959 fm
- Contribution of partial wave with K=6 to the triton wave function is about 1% of that from K=0

## Context

This work extends the earlier "hybrid" approach of Dzhibuti (1990s), who combined Faddeev equations with hyperspherical harmonics in momentum space using the Raynal–Revai transformation, by instead working directly in configuration space and expanding the total wave function rather than individual Faddeev components. The method avoids the Raynal–Revai transformation entirely. The results are compared with benchmark calculations using the Argonne v18 plus Urbana IX potential (reported in Ref. 24 of the paper) and with experimental data. The approach is intended as a step toward treating systems with four or more nucleons, where the standard Faddeev method becomes computationally demanding.
