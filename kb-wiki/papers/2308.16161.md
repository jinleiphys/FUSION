# Lepton-induced reactions on nuclei in a wide kinematical regime

> Inclusive differential cross sections for various $e A$ and $νA$ reactions are analyzed within the GiBUU theoretical framework and code. The treatment of electron-nuclus reactions has been significantly improved by implementing a parametrized description of electron-nucleon interactions for a nucleon. Using the momentum of a nucleon inside the Fermi sea the electron-nucleon cross sections are then Lorentz-boosted to obtain the electron structure functions for nuclei. The neutrino structure functions are obtained from the ones for electrons by a transformation that involves the axial formfactors and kinematical factors that account for the difference of vector and axial currents. Special emphasis is put on analyzing data from various different experiments in different neutrino energy regimes with one and the same theoretical input.

## Key claim

The paper demonstrates that a single theoretical framework (GiBUU) with one consistent parameter set can simultaneously describe inclusive lepton-nucleus cross section data across a wide kinematical range, from low-energy experiments like MicroBooNE to medium-energy experiments like MINERvA, provided that the axial formfactor of the Delta resonance is hardened and the isospin parameter T is set to 1, both within theoretical uncertainties.

## Method

The authors implemented a parametrized description of electron-nucleon interactions from the Bosted-Christy analysis (2007) for the resonance and background regions, then Lorentz-boosted these cross sections from the nucleon rest frame to the nuclear lab frame using the momentum distribution of nucleons in a local Thomas-Fermi ground state with a momentum-dependent mean-field potential. Neutrino structure functions were obtained from electron structure functions using Walecka-type relations that involve the axial formfactor and kinematic factors accounting for the difference between vector and axial currents, with the isospin parameter T treated as an adjustable integer. The quasielastic scattering used an axial dipole formfactor with axial mass 1.0 GeV, the meson exchange current component came from a fit to electron data using a two-Gaussian parametrization, and the deep inelastic scattering region above W=3 GeV relied on PYTHIA.

## Context

This work extends the GiBUU framework (Leitner 2009, Buss 2011, Gallmeister 2016) by implementing improved electron-nucleon cross sections from Bosted and Christy (2007) and a new meson exchange current parametrization from Bodek and Christy (2022), replacing the earlier MAID 2007 analysis for the resonance region. The paper contrasts with the common approach of tuning generators separately for each experiment (as done by GENIE for MINERvA, NOvA, and MicroBooNE), and addresses the earlier finding by MiniBooNE of a significantly increased axial mass that later turned out to be due to missing 2p2h reaction components (Martini 2009, Nieves 2011). The results enable a unified description of neutrino-nucleus cross sections across different energy regimes, which is essential for reliable neutrino energy reconstruction in long-baseline experiments.
