# Presence of Extra P11 Resonances in Zagreb Analysis Since 1995

> The partial wave T-matrices for the pi N, eta N and pi2 N channels have been obtained within the framework of the coupled channel model using the pi N elastic and pi N --> eta N data base as input. It has been shown that for the P11 partial wave an equally good representation of the experimental data (namely the T(pi N,pi N) and T(pi N,eta N) T-matrices) can be obtained using either three, or four poles for the Green function propagator. However, the three Green function pole solution is not acceptable due to the structure of the extracted resonances. The two out of four P11 resonances, those lying in the energy range 1700 MeV < MR < 1800 MeV, are poorly determined, but they seem to be strongly inelastic. The inclusion of other inelastic channels is needed to determine masses and widths of missing resonances with greater precision.

## Key claim

The paper shows that the same quality of fit to the existing pi N elastic and pi N to eta N data can be achieved with either three or four poles in the Green function propagator for the P11 partial wave, but the three-pole solution is rejected because its extracted resonances couple strongly only to the eta N channel and show no branching to other inelastic channels that are experimentally established.

## Method

The authors used a three-body coupled channel model (CMU-LBL) to obtain partial wave T-matrices for the pi N, eta N, and pi^2 N channels, fitting to pi N elastic and pi N to eta N experimental data. The number of Green function poles (interpreted as resonant states) was chosen in advance, and the pole positions and channel-resonance mixing parameters were adjusted to reproduce the input data. Resonance parameters were extracted a posteriori from the final T-matrices for both the three-pole and four-pole solutions.

## Key numbers

- For the three-pole solution: P11(1440/135) mass 1426(25) MeV, width 287(53) MeV, pi N branching 61(9)%, eta N branching 0(0)%, pi^2 N branching 39(9)%.
- For the three-pole solution: P11(1710/120) mass 1724(35) MeV, width 116(47) MeV, eta N branching 89(7)%.
- For the three-pole solution: P11(2100/200) mass 2175(89) MeV, width 659(207) MeV, eta N branching 89(3)%.
- For the four-pole solution: P11(1440/135) mass 1439(19) MeV, width 437(141) MeV, pi N branching 62(4)%, eta N branching 0(0)%, pi^2 N branching 38(4)%.
- For the four-pole solution: P11(1710/120) mass 1729(16) MeV, width 180(17) MeV, pi^2 N branching 72(23)%.
- For the four-pole solution: additional P11 resonance at mass 1740(11) MeV, width 140(25) MeV, pi^2 N branching 60(35)%.
- For the four-pole solution: P11(2100/200) mass 2157(42) MeV, width 355(88) MeV, eta N branching 83(5)%.

## Context

This work extends the Zagreb group's ongoing analysis of P11 resonances since 1995, using the coupled channel formalism originally developed by Cutkosky (1979) and Batinić (1998). The finding that two extra resonances in the 1700-1800 MeV range are poorly determined but strongly inelastic indicates that the current data base (pi N elastic and pi N to eta N) is insufficient even to determine the eta N elastic channel. The authors propose that including resonance parameters as quasi-input in the fitting procedure, or adding other inelastic channels such as pi N to pi pi N and pi N to K Lambda, is needed to determine masses and widths of missing resonances with greater precision.
