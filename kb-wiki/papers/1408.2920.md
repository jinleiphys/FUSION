# The container picture with two-alpha correlation for the ground state of $^{12}$C

> It is shown that the single $0^+$ THSR (Tohsaki-Horiuchi-Schuck-Röpke) wave function which is extended to include 2$α$ correlation is almost completely equivalent to the 3$α$ wave function obtained as the full solution of 3$α$ cluster model. Their squared overlap is as high as 98\% while it is 93\% if the 2$α$ correlation is not included. This result implies that, by incorporating the 2$α$ correlation in the 3$α$ model, the ground state of $^{12}$C is describable in the container picture which is valid for the Hoyle state for which the 2$α$ correlation is weak and a single $0^+$ THSR wave function without 2$α$ correlation is almost completely equivalent to the full solution of 3$α$ cluster model.

## Key claim

By extending the single $0^+$ THSR wave function to include $2\alpha$ correlation, the squared overlap with the full $3\alpha$ cluster-model wave function for the ground state of $^{12}$C reaches 98%, compared to 93% without the $2\alpha$ correlation, showing that the compact ground state is describable in the container picture when this correlation is incorporated.

## Method

The authors constructed a $2\alpha+\alpha$ THSR wave function with two deformed size parameters ($\boldsymbol{\beta}_1$ and $\boldsymbol{\beta}_2$) to incorporate the $2\alpha$ correlation, then performed variational calculations in a four-parameter space ($\beta_{1x}=\beta_{1y}$, $\beta_{1z}$, $\beta_{2x}=\beta_{2y}$, $\beta_{2z}$) after angular momentum projection to $0^+$. They compared the single optimum THSR wave function to the full THSR-GCM solution (obtained by solving the Hill-Wheeler equation with superposed $2\alpha+\alpha$ THSR wave functions) using two sets of effective interactions (Volkov No.1 with $M=0.575$, $b=1.41$ fm and Volkov No.2 modified with $M=0.59$, $b=1.35$ fm).

## Key numbers

- Squared overlap between the single optimum $2\alpha+\alpha$ THSR wave function and the THSR-GCM wave function: 97.5% (Force 1) and 97.8% (Force 2).
- Minimum energy from single optimum $2\alpha+\alpha$ THSR wave function: $-87.28$ MeV (Force 1) and $-89.05$ MeV (Force 2), which are about 1.2 MeV and 1.4 MeV deeper than the corresponding $3\alpha$ THSR results ($-86.09$ MeV and $-87.68$ MeV).
- THSR-GCM converged ground-state energies: $-87.98$ MeV (Force 1) and $-89.65$ MeV (Force 2).
- Full $3\alpha$ cluster-model ground-state energies from literature: $-87.92$ MeV (Uegaki et al., Force 1) and $-89.4$ MeV (Kamimura et al., Force 2).

## Context

This work extends the container picture, previously successful for the Hoyle state (Funaki et al., 2003) and for the compact ground state of $^{20}$Ne (Zhou et al., 2012, 2013), to the compact ground state of $^{12}$C. The 93% overlap found earlier for the $^{12}$C ground state using a single $3\alpha$ THSR wave function (Funaki et al., 2003) had suggested the container picture might be less applicable for compact states; the present 98% overlap resolves this discrepancy by showing that $2\alpha$ correlation is essential. This enables future studies of excited $0^+$ states and the $K^\pi=3^-$ band using the $2\alpha+\alpha$ THSR wave function, and clarifies that the ground state contains non-negligible $2\alpha$ correlation beyond the dominant $D_{3h}$ equilateral-triangle symmetry.
