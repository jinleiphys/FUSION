# Memory effect on the heavy quark dynamics in hot QCD matter

> We study the heavy quark dynamics in the presence of memory within the framework of a generalized Langevin equation. Time correlated thermal noise with power-law decay is generated by a fractional differential equation, formulated using the Caputo fractional derivative with order parameter $ν$. The effect of memory is calculated through the momentum correlation, the time evolution of the average squared momentum, the average squared displacement, and the average kinetic energy. The effect of memory is further studied for the higher normalised central moments of the heavy quark transverse-momentum distribution. The results indicate that time correlated thermal noise substantially influences heavy quark dynamics in the quark gluon plasma.

## Key claim

The paper shows that time-correlated thermal noise with power-law decay, generated via a Caputo fractional derivative with order parameter ν, substantially alters heavy quark dynamics in the quark-gluon plasma; however, the memory effects only reshape the path to thermal equilibrium rather than preventing it, because the fluctuation-dissipation theorem is implemented consistently so that the heavy quarks still approach thermal equilibrium at late times for all ν values considered.

## Method

The authors formulate a generalized Langevin equation for heavy quark dynamics in a thermalized QGP medium at fixed temperature, where the stochastic force is constructed from a time-correlated noise process h(t) generated by a fractional stochastic equation using the Caputo fractional derivative of order ν (0 < ν ≤ 1). The Caputo derivative is discretized numerically using the L1 scheme, and the resulting colored noise is embedded into the discretized generalized Langevin equation, which is solved simultaneously for the momentum and position evolution of charm and bottom quarks in three dimensions, with the dissipation kernel constrained by the fluctuation-dissipation theorem in the relativistic regime.

## Key numbers

- The memory parameter ν is varied over ν = 0.001, 0.3, 0.5, 0.7, and 0.9 for numerical validation, and ν = 0.3, 0.4, 0.6, and 0.8 for the main results.
- The characteristic memory time τ is fixed at 1 fm/c throughout the study.
- The diffusion coefficient D is set to 0.2 GeV²/fm for the main results, with D = 0.1 GeV²/fm used in the validation.
- Charm quark mass is Mc = 1.3 GeV and bottom quark mass is Mb = 4.5 GeV.
- Medium temperatures considered are T = 250 MeV and T = 500 MeV.
- For the purely diffusive 1D validation case, the analytical equal-time momentum variance is ⟨p(t)²⟩ = (2D/τ^{2ν}) × t^{2ν+1} / [(2ν+1)Γ(ν+1)²] for 0 < ν ≤ 1.
- The realistic initial transverse-momentum distribution uses the FONLL form with parameters x₀ = 6.36548 × 10⁸, x₁ = 9.0, and x₂ = 10.27890.

## Context

This work extends studies of non-Markovian dynamics in heavy quark physics by using the Caputo fractional derivative to generate power-law correlated noise, distinguishing it from the approach of Ruggieri (2022) which used exponentially decaying noise correlators, and from Pooja (2023) which employed Riemann-Liouville fractional integrals. The Caputo formulation is favored over the Riemann-Liouville derivative because it avoids fractional-order initial conditions that lack direct physical interpretation, as argued by Sandev (2015) and Fa (2007). The results enable future phenomenological studies of how memory effects modify heavy quark observables such as the nuclear modification factor R_AA and elliptic flow in heavy-ion collisions at RHIC and LHC.
