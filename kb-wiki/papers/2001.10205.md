# Comment on "Heavy Quarkonium in Extreme Conditions"

> In a recent paper (arXiv:1912.02253), Rothkopf claims that the Bryan method, which is widely used to obtain the solution in the maximum entropy method and makes use of the singular value decomposition of a matrix, limits the search space for the solution. He even presents a counterexample to the Bryan method. In this comment, we first recapitulate the mathematical basis of the Bryan method, and reconfirm that it makes use of no approximations and that it is therefore mathematically rigorous. In the second part, we explicitly show that Rothkopf's ``counterexample'' actually does not constitute a counterexample on the basis of the definition of singular value decomposition itself.

## Key claim

The Bryan method, which uses singular value decomposition to solve the maximum entropy method extremum problem, is mathematically rigorous and does not limit the search space for the solution, contrary to Rothkopf's claim; Rothkopf's presented counterexample is shown to be invalid based on the definition of singular value decomposition itself.

## Method

The author recapitulates the mathematical basis of the Bryan method for solving the extremum problem in maximum entropy method (MEM), which involves maximizing αS − L with respect to the spectral function A(ω) using the Shannon-Jaynes entropy S and a likelihood function L. The method uses singular value decomposition (SVD) of the kernel matrix K^t to show that the solution vector a (defined by A_l = m_l exp a_l) lies in the singular space spanned by the first N_s columns of U, where N_s = rank K^t ≤ N (N being the number of imaginary-time data points). The author then explicitly computes the Euclidean data corresponding to Rothkopf's proposed counterexample vector u_{N+1} and shows it yields the zero vector, which is trivially reproducible within the singular space.

## Key numbers

- N_ω (number of discretized energy points) is of order O(10^3)
- N (number of imaginary-time data points) is of order O(10)
- N_s = rank K^t ≤ N, where N is the number of imaginary-time data points
- The singular values ξ_i are ordered such that ξ_1 ≥ ξ_2 ≥ ... ≥ ξ_{N_s} > ξ_{N_s+1} = ... = 0
- The null space V_null has dimension N_ω − N_s

## Context

This comment directly addresses claims made by Rothkopf (2019) in arXiv:1912.02253 regarding limitations of the Bryan method for MEM, which is widely used in nuclear and particle physics to infer spectral functions from imaginary-time lattice data with noise, building on the foundational MEM work by Bryan and by Asakawa (2000). The author shows that Rothkopf's numerical differences between SVD and extended search space methods indicate inaccuracy in Rothkopf's calculations, and notes that MEM and pNRQCD-based analyses (Burnier, 2017) are complementary when utilized properly.
