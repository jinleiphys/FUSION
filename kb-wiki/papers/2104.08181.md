# Calculation of generating function in many-body systems with quantum computers: technical challenges and use in hybrid quantum-classical methods

> The generating function of a Hamiltonian $H$ is defined as $F(t)=\langle e^{-itH}\rangle$, where $t$ is the time and where the expectation value is taken on a given initial quantum state. This function gives access to the different moments of the Hamiltonian $\langle H^{K}\rangle$ at various orders $K$. The real and imaginary parts of $F(t)$ can be respectively evaluated on quantum computers using one extra ancillary qubit with a set of measurement for each value of the time $t$. The low cost in terms of qubits renders it very attractive in the near term period where the number of qubits is limited. Assuming that the generating function can be precisely computed using quantum devices, we show how the information content of this function can be used a posteriori on classical computers to solve quantum many-body problems. Several methods of classical post-processing are illustrated with the aim to predict approximate ground or excited state energies and/or approximate long-time evolutions. This post-processing can be achieved using methods based on the Krylov space and/or on the $t$-expansion approach that is closely related to the imaginary time evolution. Hybrid quantum-classical calculations are illustrated in many-body interacting systems using the pairing and Fermi-Hubbard models.

## Key claim

The paper proposes a hybrid quantum-classical method where a quantum computer computes the generating function \(F(t)=\langle e^{-itH}\rangle\) over a short time interval using only one ancillary qubit, and a classical computer then uses the extracted Hamiltonian moments \(\langle H^K \rangle\) for post-processing; however, the authors acknowledge that extracting high-order moments (\(K>7\) in their model cases) with sufficient precision via finite difference or polynomial methods is impractical, and only Fourier-transform-based approaches achieve global convergence at the cost of requiring long-time generating function data.

## Method

The generating function \(F(t)=\langle e^{-itH}\rangle\) is computed on a quantum computer using the Hadamard test (for the real part) and a modified Hadamard test (for the imaginary part), each requiring one ancillary qubit and a set of measurements per time point. The time-evolution operator is implemented via the Trotter-Suzuki decomposition with small time steps \(\Delta t\). From the generating function, the moments \(\langle H^K \rangle\) are extracted, and classical post-processing methods, specifically the \(t\)-expansion approach with Padé approximants, Krylov-space methods, or Fourier-transform-based moment reconstruction, are used to predict ground-state energies, excited-state energies, or long-time dynamics.

## Key numbers

- For the pairing model with \(N=4\) pairs on \(M=8\) doubly degenerate levels (\(\varepsilon_p = p\Delta e\), \(g_{pq}=g/\Delta e=1\)), the generating function was computed with \(\Delta t \cdot \Delta e = 0.002\) and \(10^4\) measurements per time point.
- For the Fermi-Hubbard model with \(N=4\) particles on \(M=4\) sites (\(U/J=1\)), the generating function was computed with \(\Delta t \cdot J = 0.02\) and \(10^4\) measurements per time point.
- The finite difference method can achieve "relatively good accuracy" for moments up to \(K=10\) to \(15\) in the model Hamiltonians, but precision degrades for higher \(K\).
- In the \(t\)-expansion method for the pairing model, using \(M=10\) (i.e., 12 cumulants as inputs) with a Padé\([3,7]\) approximation gave results close to the exact ground-state energy; using only \(M=4\) (Padé\([1,3]\)) led to a "very bad asymptotic value" due to a pole.

## Context

This work extends the use of generating functions in quantum computing, previously applied in quantum phase estimation (Nielsen and Chuang, 2000) and symmetry restoration (Lacroix, 2020), by minimizing ancillary qubit requirements for NISQ-era devices. The \(t\)-expansion method follows the approach of Horn and Weinstein (1984) and was recently considered in quantum computing by Seki and Yunoki (2020). The paper identifies a key limitation: extracting high-order moments with sufficient precision for post-processing requires either Fourier-transform-based methods (which demand long-time generating function data, reducing the NISQ advantage) or alternative approaches explored in a companion work (Ruiz Guzman and Lacroix, 2021).
