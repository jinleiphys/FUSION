# Container structure of alpha alpha Lambda clusters in $_Λ^9$Be

> New concept of clustering is discussed in $Λ$ hypernuclei using a new-type microscopic cluster model wave function, which has a structure that constituent clusters are confined in a container, whose size is a variational parameter and which we refer to as Hyper-Tohsaki-Horiuchi-Schuck-Röpke (Hyper-THSR) wave function. By using the Hyper-THSR wave function, $2α+ Λ$ cluster structure in ${^{9}_Λ{\rm Be}}$ is investigated. We show that full microscopic solutions in the $2α+ Λ$ cluster system, which are given as $2α+ Λ$ Brink-GCM wave functions, are almost perfectly reproduced by the single configurations of the Hyper-THSR wave function. The squared overlaps between the both wave functions are calculated to be $99.5$%, $99.4$%, and $97.7$% for $J^π=0^+$, $2^+$, and $4^+$ states, respectively. We also simulate the structural change by adding the $Λ$ particle, by varying the $ΛN$ interaction artificially. As the increase of the $ΛN$ interaction, the $Λ$ particle gets to move more deeply inside the core and invokes strongly the spatial core shrinkage, and accordingly distinct localized $2α$ clusters appear in the nucleonic intrinsic density, though in ${^{8}{\rm Be}}$ rather gaslike $2α$-cluster structure is shown. The origin of the localization is associated with the strong effect of Pauli principle. We conclude that the container picture of the $2α$ and $Λ$ clusters is essential in understanding the cluster structure in ${^{9}_Λ{\rm Be}}$, in which the very compact spatial localization of clusters is shown in the density distribution.

# Container structure of ααΛ clusters in \(^{9}_{\Lambda}\mathrm{Be}\)

## Key claim

The paper claims that the full microscopic solutions for the \(2\alpha + \Lambda\) cluster system in \(^{9}_{\Lambda}\mathrm{Be}\), obtained via Brink-GCM wave functions, are almost perfectly reproduced by single configurations of the newly introduced Hyper-THSR wave function, with the caveat that this equivalence holds on the subspace where the \(\Lambda\) particle is restricted to an \(S\)-wave channel \((L,\lambda) = (J,0)\).

## Method

The authors introduced a Hyper-THSR wave function for the \(2\alpha + \Lambda\) system, which confines the \(\alpha\) clusters and the \(\Lambda\) particle in a container whose size is characterized by variational parameters \(\beta_\perp\) and \(\beta_z\). They performed full microscopic \(2\alpha + \Lambda\) Brink-GCM calculations for \(J^\pi = 0^+, 2^+, 4^+\) states using two \(\Lambda N\) interactions (YNG-ND and YNG-JA), then computed squared overlaps between the single-configuration Hyper-THSR wave functions and the Brink-GCM wave functions projected onto the \((L,\lambda) = (J,0)\) channel. They also varied the \(\Lambda N\) interaction strength artificially to simulate the structural change induced by the \(\Lambda\) particle.

## Key numbers

* Squared overlaps between Hyper-THSR and Brink-GCM wave functions: 99.5% for \(J^\pi = 0^+\), 99.4% for \(J^\pi = 2^+\), and 97.7% for \(J^\pi = 4^+\) states, using the YNG-ND interaction.
* Binding energy of \(^{9}_{\Lambda}\mathrm{Be}\) ground state from Brink-GCM: \(-61.78\) MeV with YNG-ND and \(-60.98\) MeV with YNG-JA, corresponding to \(\Lambda\) binding energies \(B_\Lambda = 7.33\) MeV (ND) and \(6.53\) MeV (JA).
* Core rms radius of \(^{9}_{\Lambda}\mathrm{Be}\) ground state: \(R_{\mathrm{rms}}^{(c)} = 2.31\) fm, compared to \(R_{\mathrm{rms}} = 2.87\) fm for \(^{8}\mathrm{Be}\).
* \(\Lambda\) binding energy of \(^{5}_{\Lambda}\mathrm{He}\) used to calibrate the YNG interactions: 3.12 MeV (empirical value), with adopted Fermi-momentum parameters \(k_F = 0.962\) fm\(^{-1}\) for ND and \(k_F = 0.757\) fm\(^{-1}\) for JA.

## Context

This work extends the THSR wave function concept, previously applied to ordinary nuclei such as \(^{8}\mathrm{Be}\) and the Hoyle state in \(^{12}\mathrm{C}\) by Tohsaki, Horiuchi, Schuck, and Röpke (2001), to hypernuclei by introducing the Hyper-THSR wave function. It builds on earlier cluster model studies of \(^{9}_{\Lambda}\mathrm{Be}\) using GCM and OCM by various authors (e.g., Hiyama et al., 1997; Yamada et al., 1985) that demonstrated the rotational spectrum and core-shrinkage effect due to the \(\Lambda\) particle. The results enable a unified container description of both gaslike and localized cluster structures in hypernuclei, and suggest that the apparent localization of \(2\alpha\) clusters in \(^{9}_{\Lambda}\mathrm{Be}\) arises from the Pauli principle rather than from genuine localized clustering.
