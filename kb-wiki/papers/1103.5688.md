# Nucleus from String Theory

> In generic holographic QCD, we find that baryons are bound to form a nucleus, and that its radius obeys the empirically-known mass number (A) dependence r A^{1/3} for large A. Our result is robust, since we use only a generic property of D-brane actions in string theory. We also show that nucleons are bound completely in a finite volume. Furthermore, employing a concrete holographic model (derived by Hashimoto, Iizuka, and Yi, describing a multi-baryon system in the Sakai-Sugimoto model), the nuclear radius is evaluated as O(1) x A^{1/3} [fm], which is consistent with experiments.

## Key claim

The paper claims that, in generic holographic QCD (large \(N_c\), strong coupling), baryons inevitably bind into a nucleus whose radius scales as \(A^{1/3}\) for large mass number \(A\), and that this result is robust because it follows only from the generic D-brane action structure, not from a specific model. The hedge is that the derivation holds at leading order in the large \(A\) and large \(N_c\) expansions, and the explicit numerical radius is obtained only after fixing parameters using a concrete model (Hashimoto, Iizuka, and Yi in the Sakai-Sugimoto model).

## Method

The authors model a system of \(A\) baryons in the gravity dual of a confining gauge theory as a collection of \(A\) D-brane vertices (baryon vertices). The effective action for the adjoint scalar fields \(X^I\) (encoding baryon positions and holographic coordinates) is a dimensionally reduced Yang-Mills action, i.e., a bosonic matrix quantum mechanics. They then apply a large-\(D\) expansion (where \(D\) is the number of matrix directions) and a 'tHooft expansion in the matrix model coupling to compute the ground-state eigenvalue distribution, which gives the baryon density via a RR-charge density formula.

## Key numbers

* Nuclear mean square radius from the explicit matrix model (Hashimoto, Iizuka, Yi) with inputs \(g_{\pi NN}\sim13.2\) and \(m_\rho\sim 776\) MeV: \(\sqrt{r^2_{\rm mean}}\sim 0.7\, A^{1/3}\) [fm] (compared to the observed \(\sqrt{r^2_{\rm mean}} \simeq 1.0\, A^{1/3}\) [fm]).
* Alternative nuclear radius \(r_0\) from the density distribution, with the same inputs: \(r_0 \sim 0.8 \times A^{1/3}\) [fm].
* The matrix model parameters used: \(\lambda = 5.31\) and \(M_{\rm KK} = 949\) MeV, derived from fitting the \(\rho\) meson mass and the NN\(\pi\) coupling in the Sakai-Sugimoto model (Hashimoto 2008).
* The density distribution \(\rho(r)\) is exactly zero for \(r > r_0\), where \(r_0 = (2A/\tilde{\lambda}_A^{1/3})^{1/2}\) in the large-\(D\) limit.

## Context

This work extends the holographic QCD approach to nuclear physics by showing that the multi-baryon bound state (nucleus) emerges generically from the D-brane description of baryons, without relying on a specific model. It builds on the earlier matrix model for multi-baryons in the Sakai-Sugimoto model (Hashimoto, Iizuka, Yi 2010) and on the theorem by Luscher (1982) about eigenvalue binding in dimensionally reduced Yang-Mills theories. The result enables future calculations of nuclear properties (e.g., density distributions, radii) directly from string theory parameters, and opens the way to study finite-temperature effects and isospin structure in holographic nuclei.
