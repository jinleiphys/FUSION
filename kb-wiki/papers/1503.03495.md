# Realistic estimate of valence transversity distributions from inclusive dihadron production

> We present an updated extraction of the transversity parton distribution based on the analysis of pion-pair production in deep-inelastic scattering off transversely polarized targets in collinear factorization. Data for proton and deuteron targets make it possible to perform a flavor separation of the valence components of the transversity distribution, using di-hadron fragmentation functions taken from the semi-inclusive production of two pion pairs in back-to-back jets in e+e- annihilation. The e+e- data from Belle have been reanalyzed using the replica method and a more realistic estimate of the uncertainties on the chiral-odd interference fragmentation function has been obtained. Then, the transversity distribution has been extracted by using the most recent and more precise COMPASS data for deep-inelastic scattering off proton targets. Our results represent the most accurate estimate of the uncertainties on the valence components of the transversity distribution currently available.

## Key claim

The paper presents an updated extraction of the valence components of the transversity parton distribution using dihadron production in collinear factorization, claiming the most accurate estimate of uncertainties currently available, but the result is conditional on the use of the replica method for error propagation and on the specific choice of the strong coupling constant value used in QCD evolution.

## Method

The authors extract the chiral-odd interference dihadron fragmentation function (DiFF) from Belle e+e- annihilation data for back-to-back pion pairs using the replica method with 100 replicas, fitting a 9-parameter function at the starting scale Q0^2 = 1 GeV^2 and evolving it to the Belle scale using leading-order QCD evolution. They then extract the valence transversity distributions for up and down quarks by combining these DiFFs with COMPASS deep-inelastic scattering data for identified pi+ pi- pairs off transversely polarized proton targets and with previous COMPASS data on deuteron targets, using the same replica method to propagate uncertainties from both the DiFF extraction and the SIDIS asymmetry measurements.

## Key numbers

- The fit of the polarized DiFF uses 46 bins of experimental data for the Artru-Collins asymmetry from Belle, organized in a (z, M_h) grid of 64 bins with some empty or scarcely populated bins excluded
- The unpolarized DiFF fit used approximately 32000 bins of PYTHIA-generated data with an average chi^2 per degree of freedom of 1.62 using 79 parameters
- The polarized DiFF fit uses 9 parameters (N_H, gamma_1, gamma_2, delta_1, delta_2, delta_3, delta_4, delta_5, eta) with the Breit-Wigner parameters fixed at m_rho = 0.776 GeV and Gamma_rho = 0.150 GeV
- Two values of alpha_s(M_Z^2) are used for evolution: 0.125 and 0.139, to account for theoretical uncertainties on the Lambda_QCD parameter
- The number of replicas used is N = 100 for both the DiFF extraction and the transversity extraction
- The integrals n_q and n_u^uparrow are evaluated with experimental cuts: 0.2 < z < 1 and 0.5 GeV < M_h < 1 GeV for HERMES, 0.2 < z < 1 and 0.29 GeV < M_h < 1.29 GeV for COMPASS

## Context

This work extends the previous point-by-point extraction of transversity by Bacchetta et al. (2012) and the DiFF extraction by Courtoy et al. (2012), updating both with the replica method for more realistic uncertainty estimates and using the most recent COMPASS data for identified pi+ pi- pairs from the 2010 run (Braun, 2015). The analysis enables future comparisons with transversity extractions from single-hadron SIDIS using transverse-momentum-dependent factorization, as previously explored by Martin et al. (2014), and can be extended when real data for the unpolarized e+e- cross section become available instead of the PYTHIA-based parametrization used here.
