# Broken Pairs and High-Spin States in Transitional Nuclei

> The Interacting Boson plus broken-pairs Model has been used to describe high-spin bands in spherical and transitional nuclei. In the spherical nucleus $^{104}$Cd the model reproduces the structure of high-spin bands built on a vibrational structure. Model calculations performed with a single set of parameters reproduce ten bands in $^{136}$Nd, including the two dipole high-spin bands based on the $(πh_{11/2})^2$ $(νh_{11/2})^2$ configuration.

# Broken Pairs and High-Spin States in Transitional Nuclei

## Key claim
The Interacting Boson plus broken-pairs Model, using a single set of parameters, reproduces ten bands in ¹³⁶Nd including the two dipole high-spin bands based on the (πh₁₁/₂)² (νh₁₁/₂)² configuration, but only after including an additional higher-order term in the boson quadrupole operator and a proton-neutron delta-interaction; the authors argue that the alternative projected shell model interpretation assigning a νf₇/₂ orbital to these bands is incorrect.

## Method
The authors extend the IBM-1 by allowing one or two bosons to be destroyed and replaced by non-collective fermion pairs (one or two broken pairs), creating a model space of (N bosons) ⊕ ((N-1) bosons ⊗ 1 broken pair) ⊕ ((N-2) bosons ⊗ 2 broken pairs). The Hamiltonian includes boson, fermion, boson-fermion interaction, and pair-breaking mixing terms, with a modified quadrupole operator containing a higher-order χ′ term. For ¹⁰⁴Cd only neutron broken pairs are included; for ¹³⁶Nd both proton and neutron h₁₁/₂ orbitals are used, with the boson Hamiltonian prediagonalized and fermion states coupled only to the collective ground state band to reduce the two-broken-pair space.

## Key numbers
- ¹⁰⁴Cd: boson Hamiltonian parameters ε = 0.658 MeV, C₄ = 0.117 MeV, boson number N = 4
- ¹⁰⁴Cd: neutron quasiparticle energies E(νd₅/₂) = 1.113 MeV, E(νs₁/₂) = 2.287 MeV, E(νh₁₁/₂) = 2.691 MeV, E(νg₇/₂) = 1.316 MeV
- ¹⁰⁴Cd: occupation probabilities v²(νd₅/₂) = 0.57, v²(νs₁/₂) = 0.06, v²(νh₁₁/₂) = 0.04, v²(νg₇/₂) = 0.23
- ¹⁰⁴Cd: boson-fermion parameters Γ₀ = 0.2 MeV, χ = -0.9, Λ₀ = 0.2 MeV; pair-breaking strength U₂ = 0.1 MeV
- ¹⁰⁴Cd: experimental B(E2, 2₁ → 0₁) = 0.068 e²b² and Q(2₁) = -0.25 eb for ¹⁰⁶Cd used to set χ
- ¹⁰⁴Cd: calculated (h₁₁/₂)² I=10⁺ band-head at 5310 keV (not observed experimentally)
- ¹³⁶Nd: boson number N = 8; boson Hamiltonian parameters ε = 0.36 MeV, C₀ = 0.16 MeV, C₂ = -0.12 MeV, C₄ = 0.19 MeV, V₂ = 0.11 MeV, V₀ = -0.3 MeV
- ¹³⁶Nd: quasiparticle energies E_ν(h₁₁/₂) = 1.75 MeV, E_π(h₁₁/₂) = 1.60 MeV; occupation probabilities v²_ν(h₁₁/₂) = 0.83, v²_π(h₁₁/₂) = 0.07
- ¹³⁶Nd: neutron dynamical interaction Γ₀ = 0.3 MeV, χ = -1, χ′ = -0.2; proton Γ₀ = 0.22 MeV, χ = +1, χ′ = +0.2
- ¹³⁶Nd: exchange interaction Λ₀^ν = 1.0, Λ₀^π = 1.5; pair-breaking U₂ = 0.2 MeV for both protons and neutrons
- ¹³⁶Nd: surface delta-force strengths V_νν = -0.1, V_ππ = -0.1, V_νπ = -0.9
- ¹³⁶Nd: calculated states up to angular momentum 29 ħ in band 11 at more than 13 MeV excitation energy

## Context
This work extends the Interacting Boson plus broken-pairs Model previously applied to Hg (Iudice and Vretenar, 1991; Vretenar et al., 1993, 1995), Sr-Zr (Choi et al., 1991; Lister et al., 1993; Chiari et al., 1993; Cacciamani et al., unpublished), and Nd-Sm regions (Dean et al., 1994; Ragnarsson et al., 1996; Petrache et al., 1997). For ¹³⁶Nd, the authors directly challenge the projected shell model interpretation of Sun et al. (1996), who assigned a νf₇/₂ orbital to the dipole bands; the present model instead attributes the dipole bands to (πh₁₁/₂)² (νh₁₁/₂)² configurations coupled to the ground state band. The approach enables description of high-spin bands in spherical and transitional nuclei without assuming a permanent deformation, and produces wave functions suitable for calculating electromagnetic properties in the laboratory frame.
