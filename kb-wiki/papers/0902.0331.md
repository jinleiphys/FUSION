# The 0nbb-decay nuclear matrix elements with self-consistent short-range correlations

> A self-consistent calculation of nuclear matrix elements of the neutrinoless double beta decays (0nbb) of 76Ge, 82Se, 96Zr, 100Mo, 116Cd, 128Te, 130Te and 130Xe is presented in the framework of the renormalized quasiparticle random phase approximation (RQRPA) and the standard QRPA. The pairing and residual interactions as well as the two-nucleon short-range correlations are for the first time derived from the same modern realistic nucleon-nucleon potentials, namely from charge-dependent Bonn potential (CD-Bonn) and the Argonne V18 potential. In a comparison with the traditional approach of using the Miller-Spencer Jastrow correlations matrix elements for the 0nbb-decay are obtained, which are larger in magnitude. We analyze the differences among various two-nucleon correlations including those of the unitary correlation operator method (UCOM) and quantify the uncertainties in the calculated 0nbb-decay matrix elements.

## Key claim

The paper claims that using self-consistent short-range correlations (SRC) derived from the same realistic nucleon-nucleon potentials (CD-Bonn and Argonne V18) that define the pairing and residual interactions yields 0nbb-decay nuclear matrix elements that are larger in magnitude than those obtained with the traditional Miller-Spencer Jastrow approach, but the authors note that the uncertainty in the calculated matrix elements remains on the order of 25%.

## Method

The authors performed calculations of 0nbb-decay nuclear matrix elements for eight isotopes using both the renormalized quasiparticle random phase approximation (RQRPA) and the standard QRPA. The two-nucleon short-range correlations were determined within the coupled cluster method (exponential S approach) using the CD-Bonn and Argonne V18 realistic nucleon-nucleon potentials, and these same potentials were used consistently for the pairing and residual interactions in the nuclear structure calculation. The resulting correlated two-body wave functions were used to construct analytic Jastrow-like correlation functions, which were then applied to the two-body transition matrix elements in the QRPA/RQRPA framework.

## Key numbers

- For 76Ge, the bare nuclear matrix element M0nu (without FNS or SRC) is reduced by approximately 20% by finite nucleon size effects alone, by about 20-30% with CCM Argonne SRC, and by 40-50% with Miller-Spencer Jastrow SRC.
- The CCM CD-Bonn correlation function parameters are: a = 1.52 fm^-2, b = 1.88 fm^-2, c = 0.46; the CCM Argonne correlation function parameters are: a = 1.59 fm^-2, b = 1.45 fm^-2, c = 0.92.
- The Miller-Spencer Jastrow correlation function uses parameters a = 1.1 fm^-2 and b = 0.68 fm^-2.
- The accuracy of the calculated 0nbb-decay nuclear matrix elements is of the order of 25% with respect to the central value.
- The differences between results calculated with Jastrow and CCM SRC are about 20-30%.

## Context

This work extends previous QRPA and RQRPA calculations of 0nbb-decay nuclear matrix elements (Rodin 2006, Rodin 2003) by introducing for the first time a consistent treatment where short-range correlations, pairing, and ground state correlations are all derived from the same realistic nucleon-nucleon potentials. The paper improves upon the traditional Miller-Spencer Jastrow approach (Spencer) and the unitary correlation operator method (UCOM) (Kortelainen 2007) by using coupled cluster method correlations, and finds that the self-consistent treatment yields matrix elements that are 20-30% larger than those obtained with Jastrow SRC. This work enables more reliable comparisons between calculated 0nbb-decay half-lives and experimental limits, and the authors note that the Large-Scale Shell Model results (LSSM) would likely increase by 20-30% if CCM SRC were adopted.
