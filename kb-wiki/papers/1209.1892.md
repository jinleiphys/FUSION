# Strong contribution to octet baryon mass splittings

> We calculate the $m_d-m_u$ contribution to the mass splittings in baryonic isospin multiplets using SU(3) chiral perturbation theory and lattice QCD. Fitting isospin-averaged perturbation theory functions to PACS-CS and QCDSF-UKQCD Collaboration lattice simulations of octet baryon masses, and using the physical light quark mass ratio $m_u/m_d$ as input, allows $M_n-M_p$, $M_{Σ^-}-M_{Σ^+}$ and $M_{Ξ^-}-M_{Ξ^0}$ to be evaluated from the full SU(3) theory. The resulting values for each mass splitting are consistent with the experimental values after allowing for electromagnetic corrections. In the case of the nucleon, we find $M_n-M_p= 2.9 \pm 0.4 \textrm{MeV}$, with the dominant uncertainty arising from the error in $m_u/m_d$.

## Key claim

The strong (QCD) contribution to the neutron-proton mass difference is calculated as \(M_n-M_p = 2.9 \pm 0.4\) MeV, but this result depends on using the physical light quark mass ratio \(m_u/m_d = 0.553 \pm 0.043\) as input; using a different estimate for that ratio (\(0.47 \pm 0.04\)) yields \(3.7 \pm 0.4\) MeV instead.

## Method

The authors fit isospin-averaged SU(3) heavy-baryon chiral perturbation theory functions to octet baryon masses from PACS-CS and QCDSF-UKQCD lattice QCD simulations. They then extend the chiral expansion to allow \(m_d - m_u \ne 0\), including \(\pi^0\)-\(\eta\) mixing, and evaluate the mass splittings at the physical point using the fitted parameters and the experimental light quark mass ratio as input.

## Key numbers

- \(M_n-M_p = 2.9 \pm 0.4\) MeV (using \(m_u/m_d = 0.553 \pm 0.043\)) from PACS-CS data
- \(M_{\Sigma^-}-M_{\Sigma^+} = 7.5 \pm 1.0\) MeV (using \(m_u/m_d = 0.553 \pm 0.043\)) from PACS-CS data
- \(M_{\Xi^-}-M_{\Xi^0} = 4.6 \pm 0.6\) MeV (using \(m_u/m_d = 0.553 \pm 0.043\)) from PACS-CS data
- \(M_n-M_p = 2.4 \pm 0.3\) MeV (using \(m_u/m_d = 0.553 \pm 0.043\)) from QCDSF-UKQCD data
- \(M_{\Sigma^-}-M_{\Sigma^+} = 7.0 \pm 0.9\) MeV (using \(m_u/m_d = 0.553 \pm 0.043\)) from QCDSF-UKQCD data
- \(M_{\Xi^-}-M_{\Xi^0} = 4.6 \pm 0.6\) MeV (using \(m_u/m_d = 0.553 \pm 0.043\)) from QCDSF-UKQCD data
- Experimental total splittings: \(M_n-M_p = 1.2933322(4)\) MeV, \(M_{\Sigma^-}-M_{\Sigma^+} = 8.079(76)\) MeV, \(M_{\Xi^-}-M_{\Xi^0} = 6.85(21)\) MeV

## Context

This work extends earlier lattice determinations of the QCD contribution to baryon mass splittings (Beane 2006, Blum 2010, Walker-Loud 2010, de Divitiis 2011, Horsley 2012) by using SU(3) chiral perturbation theory to extract the strong component from isospin-averaged lattice data, rather than requiring fully SU(2)-broken simulations. The results are consistent with the QCDSF-UKQCD Collaboration's own analysis (Horsley 2012) and with both the traditional electromagnetic correction estimate of Gasser and Leutwyler (1982) and the larger value proposed by Walker-Loud et al. (2012). The method enables improved determination of the light quark mass ratio \(m_u/m_d\) if combined with precise electromagnetic contributions, or conversely, more precise electromagnetic contributions if the quark mass ratio is better known.
