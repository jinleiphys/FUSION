# The near-threshold peak of the phi-photoproduction cross section and its interpretation

> We study, within phenomenological Lagrangian approach, the possibility that the near-threshold peak found in the phi-photoproduction cross section is caused by a resonance. We show that, by employing a J^P=3/2^- resonance with a mass of 2.08 +/- 0.05 GeV and a width of 0.570 +/- 0.169 GeV, the LEPS data which include new data on nine spin-density matrix elements can indeed be described reasonably well. We also find that the ratio of helicity amplitudes A_1/2/A_3/2 calculated from the resulting coupling constants differs in sign from that of D13(2080). The resonance is further found to be able to improve the theoretical description of omega photoproduction against a new set of data if a large value for OZI evading parameter is assumed.

## Key claim

The near-threshold peak in the phi-photoproduction cross section can be described by a J^P = 3/2^- resonance, but the authors caution against identifying this resonance with the known D13(2080) state listed in the PDG, because the ratio of helicity amplitudes A_1/2/A_3/2 differs in sign between the two.

## Method

The authors employed a phenomenological Lagrangian approach at tree level, adding a resonance to a model consisting of Pomeron and (pi, eta) exchange amplitudes. They fitted the mass, width, and products of coupling constants of the resonance using MINUIT, fitting to LEPS differential cross section data and spin-density matrix element data as well as previous experimental data from Durham.

## Key numbers

- Resonance mass: 2.08 +/- 0.05 GeV
- Resonance width: 0.570 +/- 0.169 GeV
- Product coupling constant eg_gamma N N*^(1) g_phi N N*^(1) = -0.205 +/- 0.095
- Product coupling constant eg_gamma N N*^(2) g_phi N N*^(1) = -0.266 +/- 0.136
- Ratio of helicity amplitudes A_1/2/A_3/2 = 1.05 for the fitted resonance, compared to -1.18 for D13(2080) from the PDG

## Context

This work extends the authors' previous study (Kiswandhi 2010) by fitting to new LEPS spin-density matrix element data in addition to earlier differential cross section data. It addresses the inability of t-channel exchange models (Titov 1997-2007) to describe the near-threshold peak. The analysis corrects a previous confusion in the phase of the Pomeron-exchange amplitude found in a coupled-channel analysis (Ozaki 2009, Hosaka 2010). The results enable further investigation of whether this resonance can also improve theoretical descriptions of omega photoproduction.
