# Rotational covariance and light-front current matrix elements

> Light-front current matrix elements for elastic scattering from hadrons with spin~1 or greater must satisfy a nontrivial constraint associated with the requirement of rotational covariance for the current operator. Using a model $ρ$ meson as a prototype for hadronic quark models, this constraint and its implications are studied at both low and high momentum transfers. In the kinematic region appropriate for asymptotic QCD, helicity rules, together with the rotational covariance condition, yield an additional relation between the light-front current matrix elements.

# Rotational covariance and light-front current matrix elements

## Key claim

For elastic scattering from hadrons with spin 1 or greater, light-front current matrix elements computed using only one-body current operators cannot satisfy the rotational covariance constraint; the breakdown becomes significant when the dimensionless parameter eta = Q^2 / 4M^2 reaches order unity, and at high Q^2 the rotational covariance condition yields an additional relation among helicity matrix elements that is consistent with perturbative QCD power-counting rules but cannot be reproduced by constituent quark models with one-body currents alone.

## Method

The author constructs a model rho meson as a quark-antiquark bound state with a harmonic-oscillator potential, using a mass operator of the form M^2 = M_0^2 + 2mU + U_0, where M_0^2 = 4(m^2 + k^2) is the non-interacting mass operator. The S- and D-state wave functions are Gaussians with oscillator parameter alpha = 0.45 GeV/c, and a D-state admixture amplitude of 0.04 is included. The light-front current matrix elements I_{mu',mu} of I^+(0) are computed in a frame with q^+ = 0, and the three elastic form factors G_0, G_1, G_2 are extracted from these matrix elements. The rotational covariance condition Delta(Q^2) = (1+2eta)I_{1,1} + I_{1,-1} - sqrt(8eta)I_{1,0} - I_{0,0} = 0 is then evaluated to test whether the one-body current calculation satisfies the constraint.

## Key numbers

- Oscillator parameter alpha = 0.45 GeV/c, extracted from the Godfrey-Isgur results (Kokoski, 1987)
- D-state admixture amplitude coefficient of 0.04 used for the rho meson
- For M_rho = 750 MeV and m_q = 10 MeV with Melosh rotations ON: charge radius sqrt(<r^2>) = 1.17 fm, rotational covariance parameter delta = 0.03
- For M_rho = 750 MeV and m_q = 300 MeV with Melosh rotations ON: charge radius sqrt(<r^2>) = 0.61 fm, delta = 0.17
- For M_rho = 750 MeV and m_q = 1000 MeV with Melosh rotations ON: charge radius sqrt(<r^2>) = 0.51 fm, delta = 0.36
- For M_rho = 2000 MeV and m_q = 10 MeV with Melosh rotations ON: charge radius sqrt(<r^2>) = 1.11 fm, delta = 0.11
- For M_rho = 2000 MeV and m_q = 1000 MeV with Melosh rotations ON: charge radius sqrt(<r^2>) = 0.33 fm, delta = 0.04
- The rotational covariance uncertainty function Delta(Q^2) becomes comparable to the form factor G_0 in the region 1-2 (GeV/c)^2 for the physical rho meson mass of 750 MeV

## Context

This work extends earlier studies of light-front form factors for the pion (Chung, Coester, and Polyzou, 1988) and the nucleon (Chung and Coester, 1991), which could not test rotational covariance because spin 0 and spin 1/2 have no such constraint. It directly applies the angular condition formalism developed for the deuteron by Grach and Kondratyuk (1984) and by Chung, Coester, Keister, and Polyzou (1988) to a quark-model rho meson, where the characteristic constituent momentum is not small compared to the quark mass. The results enable future work to incorporate two-body current operators in constituent quark models to restore rotational covariance, and they clarify the connection between the light-front rotational covariance condition and the helicity power-counting rules of perturbative QCD derived by Carlson and Gross (1984).
