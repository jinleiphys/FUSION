# The mixed quark-gluon condensate from an effective quark-quark interaction

> We exhibit the method for obtaining non perturbative quark and gluonic vacuum condensates from a model truncation of QCD. The truncation allows for a phenomenological description of the quark-quark interaction in a framework which maintains all global symmetries of QCD and allows an 1/N_c expansion. Within this approach the functional integration over the gluon fields can be performed and therefore any gluonic vacuum observable can be expressed in terms of a quark operator and the gluon propagator. As a special case we calculate the mixed quark gluon condensate. We investigate how the value depends on the form of the model quark-quark interaction. A comparison with the results of quenched lattice QCD, the instanton liquid model and QCD sum rules is drawn.

# The mixed quark-gluon condensate from an effective quark-quark interaction

## Key claim
The mixed quark-gluon condensate calculated within the global color model (GCM) truncation of QCD yields values compatible with quenched lattice QCD, the instanton liquid model, and QCD sum rules, but only for model gluon 2-point functions that are tuned to reproduce the pion decay constant and the chiral low-energy coefficients. The quark condensate itself is systematically smaller than the standard value of -(220 MeV)^3.

## Method
The author truncates the QCD partition function to include only the gluon 2-point function, defining the global color model (GCM) with a bilocal quark-quark interaction kernel. The functional integration over gluon fields is performed analytically, reducing any gluonic vacuum observable to an expression involving the quark propagator and the gluon propagator. The mixed condensate is then evaluated at the mean-field (saddle-point) level, using the rainbow Dyson-Schwinger equation to determine the quark self-energy functions A(p^2) and B(p^2), with the gluon 2-point function chosen to reproduce f_pi = 87 MeV and the chiral coefficients L_i.

## Key numbers
- The mixed quark-gluon condensate g_s <qbar G_{mu nu} sigma^{mu nu} q> at renormalization scale mu = 1 GeV^2 ranges from (395 MeV)^5 to (459 MeV)^5 across three different model gluon 2-point functions.
- The quark condensate <qbar q> at mu = 1 GeV^2 ranges from (151 MeV)^3 to (183)^3 across the same models.
- The pion decay constant in the chiral limit is fixed to f_pi = 87 MeV in all cases.
- The chiral low-energy coefficients obtained are L_1 = 0.79-0.85 x 10^{-3}, L_3 = -3.76 to -4.46 x 10^{-3}, L_5 = 0.82-2.38 x 10^{-3}, L_8 = 0.80-1.03 x 10^{-3}, compared to phenomenological values L_1 = 0.7 +/- 0.5, L_3 = -3.6 +/- 1.3, L_5 = 1.4 +/- 0.5, L_8 = 0.9 +/- 0.3 (all x 10^{-3}).
- Comparison values from other methods: QCD sum rules give <qbar q>^{1/3} = 210-230 MeV and mixed condensate^{1/5} = 375-395 MeV; quenched lattice gives 225 MeV and 402-429 MeV; instanton liquid model gives 272 MeV and 490 MeV.

## Context
This work extends the global color model (GCM) approach developed by Cahill, Roberts, Williams, Tandy, and others, which had previously been used to study mesonic properties and chiral low-energy coefficients. The author applies the GCM framework specifically to calculate the mixed quark-gluon condensate, which was among the least well-determined vacuum condensates. The results are compared directly with QCD sum rules (Narison, 1989), quenched lattice QCD (Schierholz, 1995), and the instanton liquid model (Weiss, 1996). The method enables systematic calculation of any gluonic vacuum observable within the GCM truncation, though higher-order operators (such as the gluon condensate involving A^4) become increasingly involved.
