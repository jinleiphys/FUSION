# Pion fragmentation functions from a quark-jet model in a functional approach

> The elementary fragmentation function that describes the process $q\to π$ is predicted applying crossing and charge symmetry to the cut diagram of the pion valence quark distribution function. This elementary probability distribution defines the ladder-kernel of a quark jet fragmentation equation, which is solved self-consistently to obtain the full pion fragmentation function. The hadronization into a pion employs the complete Poincaré invariant Bethe-Salpeter wave function, though the overwhelming contribution to the fragmentation function is due the leading Bethe-Salpeter amplitude. Compared to a Nambu--Jona-Lasinio model prediction, the fragmentation function we obtain is enhanced in the range $z \lesssim 0.8$ but otherwise in good qualitative agreement. The full pion fragmentation function is overall greater than the elementary fragmentation function below $z\lesssim 0.6$.

## Key claim

The elementary \(q \to \pi\) fragmentation function is predicted from a cut-diagram representation using crossing and charge symmetry, and the full pion fragmentation function is obtained by solving a quark-jet fragmentation equation; the result is enhanced by about 10–12% compared to a Nambu–Jona-Lasinio model prediction for momentum fractions \(z \lesssim 0.8\), but the qualitative behavior is very similar, chiefly because the fragmentation function is dominated by the leading Bethe-Salpeter amplitude.

## Method

The elementary fragmentation function \(d_q^\pi(z)\) was derived from a cut-diagram representation using the Drell-Levy-Yan relation, employing the fully dressed quark propagator from the Dyson-Schwinger equation and the Poincaré-invariant Bethe-Salpeter wave function of the pion from the Bethe-Salpeter equation in the rainbow-ladder truncation. The quark propagator was represented with a complex-conjugate pole parametrization and the Bethe-Salpeter amplitudes with a generalized Nakanishi integral representation, allowing analytic integration over light-front variables \(k^+\) and \(k^-\) and numerical integration over transverse momentum. The full pion fragmentation function \(D_q^\pi(z)\) was obtained by solving the Volterra integral equations of the quark-jet model, which resum the cascade of pions produced by the fragmenting quark.

## Key numbers

* The elementary \(u \to \pi^+\) fragmentation function is enhanced by about 10–12% compared to the NJL model prediction for \(z \lesssim 0.8\).
* The active constituent quark mass used in the mass-shell condition is \(M_E = 0.41\) GeV.
* The model scale is \(Q_0 = 0.63\) GeV, determined by matching the first moment of the valence quark PDF from a \(\pi N\) Drell-Yan analysis, \(2 \langle x \rangle_v = 0.47(2)\).
* Moments of the fragmentation functions at the model scale \(Q_0 = 0.63\) GeV: \(\langle z \rangle_{D^\pi_0}^{Q_0} = 0.91\), \(\langle z \rangle_{D^\pi_1}^{Q_0} = 0.51\), \(\langle z \rangle_{D_u^{\pi}}^{Q_0} = 0.47\).
* Moments after leading-order DGLAP evolution to \(\mu = 2\) GeV: \(\langle z \rangle_{D_u^{\pi}}^{\mu} = 0.31\), \(\langle z \rangle_{D_{u,d\bar d}^{\pi}}^{\mu} = 0.009\), \(\langle z \rangle_{D_{u,s\bar s}^{\pi}}^{\mu} = 0.009\), \(\langle z \rangle_{D_{u,c\bar c}^{\pi}}^{\mu} = 0.003\).

## Context

This work extends the NJL model calculation of Ito et al. (2009) by using a functional Dyson-Schwinger and Bethe-Salpeter approach that incorporates momentum-dependent quark masses and a spatially extended pion Bethe-Salpeter amplitude, rather than a point-like one. The resulting fragmentation functions satisfy momentum and isospin sum rules and can be evolved via DGLAP equations to experimental scales. The approach is readily applicable to other light and heavy meson fragmentation functions, and future work should include gluon contributions to the isospin-singlet fragmentation function and coupled jet equations for cascades involving mesons other than pions.
