# Intriguing feature of multiplicity distributions

> Multiplicity distributions, P(N), provide valuable information on the mechanism of the production process. We argue that the observed P(N) contain more information (located in the small N region) than expected and used so far. We demonstrate that it can be retrieved by analysing specific combinations of the experimentally measured values of P(N) which we call {it modified combinants, Cj, and which show distinct oscillatory behavior, not observed in the usual phenomenological forms of the P(N) used to fit data. We discuss the possible sources of these oscillations and their impact on our understanding of the multiparticle production mechanism.

# Intriguing feature of multiplicity distributions

## Key claim

The observed multiplicity distributions P(N) contain additional information in the small N region that is not captured by standard phenomenological fits; this information can be retrieved by analyzing "modified combinants" C_j, which exhibit distinct oscillatory behavior that standard distributions like the Negative Binomial Distribution cannot reproduce, but the physical origin of these oscillations remains an open question.

## Method

The authors define modified combinants C_j from experimentally measured multiplicity distributions P(N) using the recurrence relation (N+1)P(N+1) = ⟨N⟩ Σ_{j=0}^{N} C_j P(N-j), which can be inverted to compute C_j directly from data. They then compare the C_j obtained from CMS and ALICE data at √s = 7 TeV with those predicted by various phenomenological forms: single Negative Binomial Distributions (NBD), multi-component NBDs, Binomial Distributions (BD), and compound distributions combining BD with Poisson or NBD components. The oscillatory pattern of C_j is used as a diagnostic to discriminate between models.

## Key numbers

- CMS charged hadron multiplicity distributions at √s = 7 TeV for |η| < 2: NBD fit parameters ⟨N⟩ = 25.5 and k = 1.45; the ratio R = data/fit deviates dramatically from unity at small N for both single NBD and 2-component NBD fits.
- For the modified NBD (MNB) with m(N) = c exp[a₁|N-b| + a₂(N-b)⁴], the parameters that yield flat R(N)=1 are c = 20.252, a₁ = 0.044, a₂ = 1.04×10⁻⁹, and b = 11.
- The oscillations in C_j from CMS data at |η| < 2 have a period roughly equal to 16, gradually disappearing with N.
- For the Compound Binomial Distribution (CBD) with K=3 and λ=10, the oscillation period equals 2λ = 20.
- For the 3-component CBD fit to CMS data: weights ω₁=0.34, ω₂=0.4, ω₃=0.26; p₁=0.22, p₂=0.22, p₃=0.12; K₁=10, K₂=12, K₃=30; λ₁=4, λ₂=9, λ₃=14.
- For the 2-component CBD (BD+NBD) fit to CMS data: K₁=K₂=3, p₁=0.7, p₂=0.67, k₁=4, k₂=2.3, m₁=6, m₂=19.0, w₁=w₂=0.5.
- For ALICE data at three rapidity windows: the period and amplitude of C_j oscillations increase with the width of the rapidity window; for |η| < 3 the amplitude is nearly constant, while for -3.4 < η < 5 it shows a dramatic increase.

## Context

This work extends the authors' previous analysis (Rybczyński, Wilk, and Włodarczyk, 2016) where they first proposed using modified combinants to extract information from the small N region of multiplicity distributions. It contradicts the common practice of using single or multi-component NBD fits, which fail to reproduce the oscillatory behavior of C_j observed in high-statistics LHC data from CMS (2010) and ALICE (2018). The paper demonstrates that compound distributions based on Binomial Distributions combined with NBD components can reproduce both P(N) and the oscillating C_j, suggesting that the production mechanism involves a limited number of sources (as in BD) rather than infinite divisibility (as in NBD). This enables future theoretical work to identify the specific physical mechanism responsible for the oscillations, which remains an open question. The authors note that a recent review by Alkin (2017) attributed the oscillations to experimental unfolding procedures, but this claim has not been substantiated.
