# Parameter Optimization and Uncertainty Analysis of FREYA for Spontaneous Fission

> Background: The complete event fission simulation code FREYA is used to study correlations in fission. To make the best possible simulations, FREYA one must find the optimized values of the five physics-based parameters which characterize the events produced by FREYA. So far this has been done only empirically or by brute force computational techniques. Purpose: We seek to develop a method of parameter optimization for FREYA that would include uncertainty quantification and an analysis of the correlations between the parameters. Methods: We focus on spontaneous fission as a simpler test case of the method. We first check the results of previous optimizations for $^{252}$Cf(sf) and then go on to develop a simulated annealing approach to optimize the parameters. Although $^{252}$Cf(sf) has the most measured observables, we are able to apply the technique to all spontaneously fissioning isotopes in the current version of FREYA. Results: We find optimal values of the five FREYA parameters, with uncertainties, for all isotopes studied. Our results are compared with available data. The parameter values are physically reasonable and in accord with intuition, even in cases where the data for optimization are sparse to non-existent. Conclusions: The simulated annealing method provides a way to determine the best parameters for the phenomenologically-successful fission model FREYA. We have developed an algorithm that can be updated to include additional isotopes or more data sets for the current isotopes as they become available.

## Key claim

The simulated annealing optimization method successfully determines optimal values for the five FREYA parameters with quantified uncertainties for all spontaneously fissioning isotopes in the code, but the quality of the constraints varies dramatically depending on how many experimental data sets are available for each isotope, with only 252Cf(sf) having data for all eight observables used in the optimization.

## Method

The authors generated 1,000,000 FREYA events for each parameter set and calculated eight observables: average neutron multiplicity, second and third moments of the neutron multiplicity, neutron multiplicity distribution, neutron multiplicity as a function of total kinetic energy and fragment mass, neutron energy spectrum, average photon multiplicity, photon multiplicity distribution, and average energy per photon. They employed a simulated annealing algorithm with an acceptance probability function based on chi-squared differences and a temperature scaling factor alpha between 0.85 and 1, followed by a fine grid search around the candidate solution. The total chi-squared was summed over all available observables for each isotope, with the reduced chi-squared per observable calculated using n-5 degrees of freedom for the five physics-based parameters.

## Context

This work extends earlier brute-force grid search optimizations of FREYA by Andrew (2015) for 252Cf(sf) and provides the first complete parameter optimization with uncertainty quantification for all spontaneously fissioning isotopes in the code, including 238U(sf), 238Pu(sf), 240Pu(sf), 242Pu(sf), and 244Cm(sf). The simulated annealing method enables optimization even when data are sparse, as for 238U(sf) and 238Pu(sf) which only have the neutron multiplicity evaluation from Smith (2014). The algorithm can be updated as new experimental data become available, and the correlation matrices provide insight into the physical relationships between parameters, such as the strong correlation between e0 and cS reflecting their joint influence on the scission temperature.
