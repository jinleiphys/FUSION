# Patches to UrQMD Model Code

> Patches to the code of the Ultra-relativisic Quantum Molecular Dynamics model (UrQMD version 1.3) are described. They improve the code operation. The most important one is done for the file ANNDEC.F.

## Key claim

The paper describes a set of software patches to the UrQMD (Ultra-relativistic Quantum Molecular Dynamics) model code version 1.3 that fix type mismatches, missing SAVE statements, and a critical bug where the Pauli-blocking was permanently disabled after final-state decays, but the authors note that the most essential improvement came from combining all subroutines into one file and eliminating compiler warnings.

## Method

The authors modified the UrQMD 1.3 Fortran source code across multiple files: in URQMD.F they added logic to restore the Pauli-blocking option after final-state decays; in PROPPOT.F they corrected REAL*8/REAL*4 type mismatches for the ERF function by inserting SNGL() wrappers; in STRING.F they changed a literal constant from -1. to -1.d0 to match the subroutine GETMAS argument type; in ANNDEC.F they added branching-ratio checks before calling the spline interpolation function fbrancx; in BLOCKRES.F they adjusted the decay branching ratios for the Delta(1950) resonance; and in INIT.F they reduced the parameter nnuc from 11 to 1 to sample nuclear coordinates and momenta for every event rather than reusing sampled values.

## Key numbers

- The branching ratio threshold for decay channels in tabinit.f is set at 1d-9 (bran.gt.1d-9) to determine whether spline coefficients are computed
- The Delta(1950) decay branching ratio to the channel with probability 0.04 was removed (changed from 0.04 to 0.00) in the BLOCKRES.F file
- The parameter nnuc in INIT.F was changed from 11 to 1, so that nuclear nucleon coordinates and momenta are sampled for each event rather than for every 11th event

## Context

This work extends the UrQMD model (Bass et al., 1998; Bleicher et al., 1999) which is used as the main event generator in the CBM and PANDA computing frameworks for planning experiments at the future FAIR facility at GSI, Darmstadt. The patches correct code inconsistencies that could affect simulation results for antiproton interactions with protons and nuclei, and enable more reliable event generation for the PANDA collaboration's simulation framework.
