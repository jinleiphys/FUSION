# Using hadron-in-jet data in a global analysis of $D^{*}$ fragmentation functions

> We present a novel global QCD analysis of charged $D^{*}$-meson fragmentation functions at next-to-leading order accuracy. This is achieved by making use of the available data for single-inclusive $D^{*}$-meson production in electron-positron annihilation, hadron-hadron collisions, and, for the first time, in-jet fragmentation in proton-proton scattering. It is shown how to include all relevant processes efficiently and without approximations within the Mellin moment technique, specifically for the in-jet fragmentation cross section. The presented technical framework is generic and can be straightforwardly applied to future analyses of fragmentation functions for other hadron species, as soon as more in-jet fragmentation data become available. We choose to work within the Zero Mass Variable Flavor Number Scheme which is applicable for sufficiently high energies and transverse momenta. The obtained optimum set of parton-to-$D^{*}$ fragmentation functions is accompanied by Hessian uncertainty sets which allow one to propagate hadronization uncertainties to other processes of interest.

## Key claim

The paper presents the first global QCD analysis of charged $D^{*}$-meson fragmentation functions that includes hadron-in-jet data from proton-proton collisions, alongside traditional electron-positron annihilation and hadron-hadron collision data, working within the Zero Mass Variable Flavor Number Scheme which is only applicable for sufficiently high energies and transverse momenta (p_T ≫ m_c).

## Method

The authors perform a next-to-leading order global QCD fit of parton-to-D* fragmentation functions using the Mellin moment technique, parametrizing the non-perturbative input at an initial scale μ₀ = m_c = 1.3 GeV with a functional form D_i(z,μ₀²) = N_i z^{α_i} (1-z)^{β_i} / B[2+α_i, β_i+1] for gluon, charm, and bottom quark fragmentation functions. They include single-inclusive e⁺e⁻ annihilation data from ALEPH and OPAL at Q = M_Z, inclusive D* production in pp and p-pbar collisions from CDF, ALICE, ATLAS, and LHCb, and for the first time, D*-in-jet fragmentation data from ATLAS using the anti-k_T algorithm with jet size R=0.6. The analysis uses the CT14 NLO parton distribution functions and the Hessian method to estimate uncertainties.

## Key numbers

- Initial scale for fragmentation functions: μ₀ = m_c = 1.3 GeV (charm quark mass from CT14)
- Bottom quark mass: m_b = 4.75 GeV (from CT14)
- Cut applied to pp data: p_T ≥ 10 GeV (conservative cut); results can be reliably extrapolated down to p_T ≈ 5 GeV
- Cut applied to SIA data: 0.1 < z < 0.95
- Jet size parameter for ATLAS in-jet data: R = 0.6
- Branching ratios used for D* decay corrections: B₁(D*⁺ → D⁰ π⁺) = (67.7 ± 0.5)%, B₂(D⁰ → K⁻ π⁺) = (3.93 ± 0.04)%
- 8 free parameters in the fragmentation function parametrization (9 parameters minus β_g which is fixed as unconstrained by data)

## Context

This work extends the framework of the DSS group (de Florian, Sassot, Stratmann, 2007) for light hadron fragmentation functions to charmed mesons, and builds upon the KKKS analysis (Kneesch, Kniehl, Kramer, Schienbein, 2007) which used only SIA data. The inclusion of in-jet fragmentation data addresses the finding by Chien et al. (2015) that existing D* fragmentation functions from Kneesch et al. (2007) did not describe ATLAS D*-in-jet data well, despite describing SIA and inclusive pp data. The technical framework is generic and can be applied to other hadron species as more in-jet fragmentation data become available.
