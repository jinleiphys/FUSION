# The TDHF Code Sky3D

> The nuclear mean-field model based on Skyrme forces or related density functionals has found wide-spread application to the description of nuclear ground states, collective vibrational excitations, and heavy-ion collisions. The code Sky3D solves the static or dynamic equations in a three-dimensional Cartesian mesh with isolated or periodic boundary conditions and no further symmetry assumptions. Pairing can be included in the BCS approximation. The code is implemented with a view to allow easy modifications for including additional physics or special analysis of the results.

## Key claim

The Sky3D code solves the static and time-dependent Hartree-Fock equations using a fully three-dimensional Cartesian mesh with no symmetry assumptions, but the authors note that the reliability of the mean-field approximation limits applications to collision energies of about a few MeV per nucleon above the Coulomb barrier and to relatively short interaction times, and that missing time-odd terms may restrict applications to even-even nuclei.

## Method

The code represents nucleonic wave function spinors on a three-dimensional Cartesian grid with periodic boundary conditions, evaluating all spatial derivatives using the finite Fourier transform method. Static Hartree-Fock equations are solved with a damped gradient iteration method accelerated by kinetic-energy damping, while time-dependent Hartree-Fock equations are solved using an expansion of the time-development operator. Pairing is included in the BCS approximation, and the code allows any number of initial nuclei to be placed in the mesh with arbitrary positions and initial velocities.

## Key numbers

- RAM requirement: 1 GB
- Typical step size for static iteration: δ = 0.1 to 0.8, with damping regulator E₀ = 100 MeV recommended as a safe choice
- Coulomb constant: e² = 1.43989 MeV·fm
- Typical pairing density parameter: ρ₀,ₚₐᵢᵣ = 0.16 fm⁻³ for surface pairing
- Recommended number of single-nucleon states for BCS pairing: N_q + (5/3)N_q^(2/3)
- Running time on a single-processor PC-type computer: varies between a few minutes and weeks depending on grid size, nucleon number, and collision duration

## Context

This work provides a publicly accessible implementation of three-dimensional TDHF calculations, extending earlier restricted-representation studies that used axial symmetry or reflection symmetries (Bonche 1976, Negele 1982, Davies 1985). The code enables simulations of nuclear vibrations, heavy-ion collisions, and ground-state structure calculations, and is designed for easy modification to include additional physics, building on the revival of unrestricted 3D TDHF studies by groups such as Simenel (2003), Maruhn (2005), Umar (2005), and Nakatsukasa (2005).
