# General transformation of alpha cluster model wave function to jj-coupling shell model in various 4N nuclei

> The antisymmetrized quasi-cluster model (AQCM) is a method to describe a transition from the alpha-cluster wave function to the jj-coupling shell model wave function. In this model, the cluster-shell transition is characterized by only two parameters; R representing the distance between alpha clusters and Lambda describing the breaking of alpha clusters, and the contribution of the spin-orbit interaction, very important in the jj-coupling shell model, can be taken into account starting with the alpha cluster model wave function. In this article we show the generality of AQCM by extending the application to heavier region; various 4N nuclei from 4He to 52Fe. We show and compare the energy curves for the alpha+40Ca cluster configuration calculated with and without alpha breaking effect in 44Ti.

# General transformation of alpha cluster model wave function to jj-coupling shell model in various 4N nuclei

## Key claim

The antisymmetrized quasi-cluster model (AQCM) can generate jj-coupling shell model wave functions with spin-orbit contributions starting from alpha cluster wave functions for any 4N nucleus from 4He to 52Fe, using only two parameters (R for inter-cluster distance and Lambda for cluster breaking), but the transformation to the exact jj-coupling subshell closure configuration is only achieved in the simultaneous limits of R approaching zero and Lambda approaching 1.

## Method

The authors start with Brink-Bloch alpha cluster wave functions where each nucleon is described by a Gaussian wave packet centered at a real parameter R_i. They transform these into AQCM wave functions by making the Gaussian center parameters complex: zeta_i = R_i + i Lambda e^{spin}_i x R_i, where Lambda is a real control parameter and e^{spin}_i is a unit vector for intrinsic spin orientation. For each nucleus from 4He to 52Fe, they place alpha clusters and quasi-clusters in specific geometric configurations (tetrahedron, pentagon, etc.) and vary R and Lambda to demonstrate the transition from cluster structure to jj-coupling shell model states, evaluating expectation values of the one-body spin-orbit operator and principal quantum number of the harmonic oscillator.

## Key numbers

- For 8Be: at R = 0.01 fm and Lambda = 1, the one-body spin-orbit expectation value reaches 2.00 (from 0.00 at Lambda = 0), corresponding to four nucleons in p3/2 orbits each contributing 1/2
- For 8Be: the principal quantum number n decreases from 5.39 at R = 3 fm to 4.00 at R = 0.01 fm
- For 12C: at R = 0.01 fm and Lambda = 1, the one-body spin-orbit expectation value reaches 4.00 (from 0.00 at Lambda = 0), corresponding to eight nucleons in p3/2 orbits
- For 12C: the principal quantum number n decreases from 11.22 at R = 3 fm to 8.00 at R = 0.01 fm
- For 28Si: at R = 0.01 fm and Lambda = 1, the one-body spin-orbit expectation value reaches 12, corresponding to twelve nucleons in d5/2 orbits (each contributing 1)
- For 28Si: the principal quantum number n is 36, constant and independent of Lambda
- For 40Ca: at R = 0 fm, the principal quantum number n converges to 60 (twelve nucleons in p shell, twenty-four in sd shell)
- For 44Ti: at Lambda = 1 with one quasi-cluster at R = 0.1 fm from the 40Ca core, the one-body spin-orbit expectation value reaches 6, corresponding to four nucleons in f7/2 orbits (each contributing 3/2)
- For 52Fe: at Lambda = 1 with three quasi-clusters at R = 0.25 fm from the 40Ca core, the one-body spin-orbit expectation value reaches 18, corresponding to twelve nucleons in f7/2 orbits
- For 44Ti: the energy difference between the Lambda = 0 (pure alpha cluster) and optimal Lambda curves is about 10 MeV around R = 1 fm

## Context

This work extends the AQCM framework, previously applied to lighter nuclei such as 12C (Suhara, 2013) and to 20Ne and 24Mg (Itagaki, Cseh, and Ploszajczak, 2011), to the full range of 4N nuclei from 4He through 52Fe. The method provides a unified description bridging the alpha cluster model and the jj-coupling shell model, which is important because cluster models cannot describe spin-orbit effects with simple (0s)^4 alpha configurations, while shell model calculations require large model spaces to describe spatially correlated nucleons. The approach enables future studies of particle-hole excitations and microscopic descriptions of alpha decay, where mixing shell and cluster components is essential to reproduce experimental decay widths (Varga, 1992).
