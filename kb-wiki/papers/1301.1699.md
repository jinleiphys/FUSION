# Report on Thermal Neutron Diffusion Length Measurement in Reactor Grade Graphite Using MCNP and COMSOL Multiphysics

> Neutron diffusion length in reactor grade graphite is measured both experimentally and theoretically. The experimental work includes Monte Carlo (MC) coding using 'MCNP' and Finite Element Analysis (FEA) coding suing 'COMSOL Multiphysics' and Matlab. The MCNP code is adopted to simulate the thermal neutron diffusion length in a reactor moderator of 2m x 2m with slightly enriched uranium ($^{235}U$), accompanied with a model designed for thermal hydraulic analysis using point kinetic equations, based on partial and ordinary differential equation. The theoretical work includes numerical approximation methods including transcendental technique to illustrate the iteration process with the FEA method. Finally collision density of thermal neutron in graphite is measured, also specific heat relation dependability of collision density is also calculated theoretically, the thermal neutron diffusion length in graphite is evaluated at $50.85 \pm 0.3cm$ using COMSOL Multiphysics and $50.95 \pm 0.5cm$ using MCNP. Finally the total neutron cross-section is derived using FEA in an inverse iteration form.

## Key claim

The thermal neutron diffusion length in reactor-grade graphite is measured as 50.85 ± 0.3 cm using COMSOL Multiphysics and 50.95 ± 0.5 cm using MCNP, with the experimental validation performed using an Am-Be source and a BF3 detector in a 2 m x 2 m moderator assembly with slightly enriched uranium fuel rods.

## Method

The study combines Monte Carlo simulations using MCNP and finite element analysis using COMSOL Multiphysics to model thermal neutron transport in a 2 m x 2 m reactor-grade graphite moderator containing slightly enriched 235U fuel rods. The Boltzmann transport equation is solved using the diffusion approximation, with the MCNP model employing continuous-energy neutron transport and the COMSOL model using the general form of partial differential equations with an Arbitrary Lagrangian Eulerian (ALE) mesh. Experimental validation was performed using an Am-Be neutron source with a cadmium cover (0.1 cm thickness, 0.55 eV cutoff) and a BF3 detector moved at 25 cm intervals in horizontal and vertical directions.

## Key numbers

- Thermal neutron diffusion length in reactor-grade graphite: 50.85 ± 0.3 cm (COMSOL Multiphysics)
- Thermal neutron diffusion length in reactor-grade graphite: 50.95 ± 0.5 cm (MCNP)
- Moderator assembly dimensions: 2 m x 2 m
- Cadmium cover thickness: 0.1 cm with cutoff energy of 0.55 eV
- Detector position intervals: 25 cm in horizontal and vertical directions
- Number of meshes used in COMSOL iteration: 5,000,000
- Maximum logarithmic energy loss (q_m) for graphite (M=12): 0.72

## Context

This work extends prior measurements of neutron diffusion length in graphite by comparing two computational approaches (MCNP and COMSOL) with experimental data from an Am-Be source. The results are compared with absorption cross-section measurements from multiple sources including IKE, ORNL, NCSU, Neill, Steyerl, and Palevsky, as well as the compilation by Wolfgang. The study enables improved modeling of thermal neutron behavior in graphite-moderated reactors and provides a benchmark for validating computational neutron transport codes against experimental measurements.
