# Anchor-based optimization of energy density functionals

> A new anchor-based optimization method of defining the energy density functionals (EDFs) is proposed. In this approach, the optimization of the parameters of EDF is carried out for the selected set of spherical anchor nuclei the physical observables of which are modified by the correction function which takes into account the global performance of EDF. It is shown that the use of this approach leads to a substantial improvement in global description of binding energies for several classes of covariant EDFs. The computational cost of defining a new functional within this approach is drastically lower as compared with the one for the optimization which includes the global experimental data on spherical, transitional and deformed nuclei into the fitting protocol.

## Key claim

The anchor-based optimization method improves the global description of binding energies for covariant energy density functionals (CEDFs) by correcting the binding energies of spherical anchor nuclei with a linear correction function that accounts for the functional's global performance, while requiring drastically lower computational cost compared to global fits that include deformed nuclei.

## Method

The method iteratively optimizes CEDF parameters using a set of spherical anchor nuclei, whose experimental binding energies are modified by a correction function \(E_{corr}(Z,N) = \alpha_i(N-Z) + \beta_i(N+Z) + \gamma_i\) determined from global calculations of all even-even nuclei with known masses. At each iteration, global RHB calculations are performed, the correction function parameters are optimized to minimize the rms deviation between calculated and experimental binding energies, and the anchor nuclei energies are redefined using this correction before re-optimizing the functional parameters. The procedure converges when the correction function parameters approach zero, typically requiring only a few iterations (one step for DD-MEY, NL5(Y), and PC-Y functionals; three steps for DD-MEX1).

## Key numbers

- DD-MEX1 functional: \(\Delta E_{rms} = 1.637\) MeV for binding energies of 855 even-even nuclei (AME2016), improved from 2.436 MeV for DD-ME2
- DD-MEY functional: \(\Delta E_{rms} = 1.734\) MeV overall, and \(\Delta E_{rms} = 1.414\) MeV for nuclei with \(A > 70\) (best among considered functionals for heavy nuclei)
- DD-MEX functional (fitted only to spherical nuclei): \(\Delta E_{rms} = 2.849\) MeV, showing the bias from spherical-only fitting
- Charge radii rms deviations for most new functionals: \(\Delta (r_{ch})_{rms} \approx 0.026\) fm (approximately 0.5% precision)
- DD-MEX1 incompressibility: \(K_0 = 291.8\) MeV (exceeds SET2b upper limit of 270 MeV)
- DD-MEY nuclear matter properties: \(K_0 = 265.8\) MeV, \(J = 32.8\) MeV, \(L_0 = 51.8\) MeV (all within SET2b limits)
- UNEDF0 Skyrme functional: \(\Delta E_{rms} = 1.428\) MeV for comparison

## Context

This work extends the development of covariant energy density functionals by addressing the bias toward spherical nuclei that plagues most existing CEDFs (over 300 relativistic functionals exist, but only one CEDF has been fitted globally to deformed nuclei). The method produces functionals with global mass description comparable to the UNEDF0 (2010), UNEDF1 (2013), and UNEDF2 Skyrme functionals from the UNEDF collaboration, but at drastically lower computational cost. The approach can be generalized to beyond-mean-field methods such as the five-dimensional collective Hamiltonian, potentially enabling the generation of functionals that include rotational and vibrational correlations without extreme computational expense.
