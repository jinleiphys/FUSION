# BiFold: A Python code for the calculation of double folded (bifold) potentials with density-in/dependent nucleon-nucleon interactions

> BiFold calculates the density-dependent (DDM3Y$n$, BDM3Y$n$, CDM3Y$n$) or independent double folded potentials between two colliding spherical nuclei. It is written in a Python package form to give the ability to use the potentials directly in a nuclear reaction/structure code. In addition to using Woods-Saxon/Fermi or Gaussian functions, the code also allows for the definition of nuclear matter densities using pre-calculated densities in a data file. The manuscript provides an overview of the double folding model and the use of the code.

## Key claim

BiFold provides a Python-based implementation of the double folding model for calculating the real part of the nuclear potential between two spherical nuclei, supporting both density-independent and density-dependent nucleon-nucleon interactions (DDM3Yn, BDM3Yn, CDM3Yn) with the ability to use pre-calculated nuclear matter densities from data files, though the code is limited to spherical nuclei and the frozen density approximation.

## Method

The code computes double folded potentials by performing a six-dimensional integration over the nuclear matter densities of two colliding spherical nuclei and an effective nucleon-nucleon interaction, using the Fourier transform method in spherical coordinates to simplify the calculation. The integration is performed by default with Simpson's method, with Filon's method available as an alternative, and the exchange part of the potential is treated either through a zero-range approximation or a finite-range approach requiring an iterative self-consistency solution for the local momentum of relative motion.

## Key numbers

- M3Y-Reid direct interaction parameters: 7999 exp(-4s)/(4s) - 2134 exp(-2.5s)/(2.5s) MeV
- M3Y-Paris direct interaction parameters: 11062 exp(-4s)/(4s) - 2538 exp(-2.5s)/(2.5s) MeV
- M3Y-Reid zero-range exchange strength: J_ex(E) approximately -276[1-0.005 epsilon] MeV fm^3
- M3Y-Paris zero-range exchange strength: J_ex(E) approximately -590[1-0.002 epsilon] MeV fm^3
- Density-dependent interaction parameters for DDM3Y1 (Reid): C=0.2845, alpha=3.6391, beta=2.9605 fm^3, nuclear incompressibility K=171 MeV
- Density-dependent interaction parameters for CDM3Y1 (Paris): C=0.3429, alpha=3.0232, beta=3.5512 fm^3, gamma=0.5, n=1, K=188 MeV
- Energy dependence factor g(E) approximately 1 - kappa epsilon, with kappa=0.002 for M3Y-Reid and kappa=0.003 for M3Y-Paris types

## Context

This work extends the double folding model formalism developed by Satchler and Love (1979), Kobos et al. (1982), and Khoa et al. (2007) by providing an open-source Python implementation that addresses limitations of existing codes such as DFPOT (Cook, 1982), which cannot calculate DDM3Y, BDM3Yn, or CDM3Yn type interactions, and DFMSPH (Gontchar, 2010), which does not support BDM3Y2, BDM3Y3, or the first version of DDM3Y. The code enables reproducible calculations of double folded potentials for nuclear reaction and structure studies, with the ability to define custom nucleon-nucleon interactions and nuclear matter densities.
