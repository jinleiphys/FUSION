# Modelling overlap functions for one-nucleon removal: role of the effective three-nucleon force

> One-nucleon overlap functions, needed for nucleon-removal reaction calculations, are solutions of an inhomogeneous equation with the source term defined by the wave functions of the initial and final nuclear states and interaction between the removed nucleon with the rest. The source term approach (STA) allows the overlaps with correct asymptotic decrease to be modelled while using nuclear many-body functions calculated in minimal model spaces. By properly choosing the removed nucleon interaction the minimum-model-space STA can reproduce reduced values of spectroscopic factors extracted from nucleon-removal reactions and predicts isospin asymmetry in the spectroscopic factor reduction. It is well-known that model space truncation leads to the appearance of higher-order induced forces, with three-nucleon force being the most important. In this paper the role of such a force on the source term calculation is studied. Applications to one-nucleon removal from double-magic nuclei show that three-nucleon force improves the description of available phenomenological overlap functions and reduces isospin asymmetry in spectroscopic factors.

## Key claim

The paper shows that including an effective three-nucleon (3N) force in the source term approach (STA) for modelling one-nucleon overlap functions improves agreement with phenomenological overlaps extracted from (e,e'p) reactions for double-magic nuclei, and reduces the isospin asymmetry in spectroscopic factor reduction that was previously predicted using only two-nucleon (NN) forces.

## Method

The author extends the source term approach (STA) by adding a contribution from a 3N force to the inhomogeneous equation that defines the overlap function, using 0ℏω harmonic oscillator wave functions for the initial and final nuclear states. The 3N force is modelled with a Gaussian form in the hyperradius of the three-nucleon system, and its matrix elements are evaluated analytically for double-magic nuclei. The STA calculations are performed for one-proton and one-neutron removal from ¹⁶O, ⁴⁰Ca, ⁴⁸Ca, ⁵⁶Ni, ¹³²Sn, and ²⁰⁸Pb, using the M3YE NN interaction and various oscillator radii, with the 3N force parameters (depth W₀ = 1 MeV, range ρ₀ = 2 fm) chosen to improve agreement with phenomenological overlaps from (e,e'p) data.

## Key numbers

- For proton removal from ¹⁶O (p₁/₂, ground state, ε = 12.13 MeV): STA spectroscopic factor with NN only = 1.42, with NN+3N = 1.23, compared to (e,e'p) value 1.27(13)
- For proton removal from ⁴⁰Ca (d₃/₂, ground state, ε = 8.33 MeV): STA SF with NN only = 3.12, with NN+3N = 2.63, compared to (e,e'p) value 2.58(19)
- For proton removal from ⁴⁰Ca (s₁/₂, E_x = 2.52 MeV, ε = 10.85 MeV): STA SF with NN only = 1.53, with NN+3N = 1.12, compared to (e,e'p) value 1.03(7)
- For proton removal from ²⁰⁸Pb (s₁/₂, ground state, ε = 8.00 MeV): STA SF with NN only = 1.64, with NN+3N = 1.04, compared to (e,e'p) value 0.98(9)
- For proton removal from ²⁰⁸Pb (d₃/₂, E_x = 0.35 MeV, ε = 8.35 MeV): STA SF with NN only = 3.28, with NN+3N = 2.52, compared to (e,e'p) value 2.31(22)
- For proton removal from ²⁰⁸Pb (h₁₁/₂, E_x = 1.35 MeV, ε = 9.35 MeV): STA SF with NN only = 7.24, with NN+3N = 6.70, compared to (e,e'p) value 6.85(68)
- For proton removal from ²⁰⁸Pb (d₅/₂, E_x = 1.67 MeV, ε = 9.67 MeV): STA SF with NN only = 5.00, with NN+3N = 4.22, compared to (e,e'p) value 2.93(28)
- For proton removal from ²⁰⁸Pb (g₇/₂, E_x = 3.47 MeV, ε = 11.47 MeV): STA SF with NN only = 6.36, with NN+3N = 5.55, compared to (e,e'p) value 2.06(20)
- Spectroscopic factors change within 30% for ⁴⁰Ca and 24% for ²⁰⁸Pb when varying oscillator radius, while squared ANCs change within 87% for ⁴⁰Ca and a factor of 4 for ²⁰⁸Pb
- Coulomb correction renormalisation reduces spectroscopic factors by 5% and affects squared ANCs by about 0.8% for both ⁴⁰Ca and ²⁰⁸Pb

## Context

This work extends the source term approach (STA) developed by Timofeyuk (2009, 2010, 2011, 2013) for modelling one-nucleon overlap functions using minimal model spaces, which previously used only NN interactions. The paper addresses the known issue that model space truncation induces many-body forces, particularly 3N forces (Feldmeier 1998, Furnstahl 2012), and shows that including an effective 3N force improves agreement with phenomenological overlaps extracted from (e,e'p) reactions by Kramer et al. (2001). The reduction of isospin asymmetry in spectroscopic factors when adding the 3N force is relevant to ongoing debates about spectroscopic factor systematics (Gomez-Ramos 2018, Vaquero 2020). The results enable future searches for a universal effective 3N force applicable across all nuclei, though the author notes this remains an open task.
