# Partition Functions in Statistical Mechanics, Symmetric Functions, and Group Representations

> Partition functions for non-interacting particles are known to be symmetric functions. It is shown that powerful group-theoretical techniques can be used not only to derive these relationships, but also to significantly simplify calculation of the partition functions for particles that carry internal quantum numbers. The partition function is shown to be a sum of one or more group characters. The utility of character expansions in calculating the partition functions is explored. Several examples are given to illustrate these techniques.

## Key claim

The paper demonstrates that partition functions for non-interacting particles (and some interacting systems) can be expressed as sums of group characters, which are symmetric functions; this identification is shown to simplify calculations for particles carrying internal quantum numbers, though the techniques are combinatorial in nature and apply even when temperature is not well-defined.

## Method

The author first provides a combinatorial proof that the N-particle partition function for bosons is a complete homogeneous symmetric function and for fermions an elementary symmetric function of the single-particle Boltzmann factors \(x_i = \exp(-\beta \epsilon_i)\). This result is then rederived using group representation theory, where the partition function is identified as the character of U(m) in the completely symmetric (bosons) or antisymmetric (fermions) irreducible representation. Character expansion techniques from U(N) group theory are then applied to project grand canonical partition functions onto specific representations of internal symmetry groups, with examples worked for SU(2) isospin symmetry for nucleons and pions, and for a two-level interacting fermion system.

## Key numbers

- For a one-dimensional harmonic oscillator with zero-point energy set to zero (\(\hat H = \hbar \omega \hat N\)), the elementary symmetric function is \(a_N = x^{N(N-1)/2} \prod_{n=1}^N \frac{1}{(1-x^n)}\) where \(x = \exp(-\beta \hbar \omega)\).
- For the bilayer quantum Hall system with K-fold degeneracy, the 1-particle partition function is \(Z_1 = K (x_1 + x_2)\) where \(x_1 = \exp(-\beta \epsilon/2)\) and \(x_2 = \exp(+\beta \epsilon/2)\).
- For the same system, the 2-particle partition function is \(Z_2 = \frac{K(K-1)}{2} (x_1^2 + x_2^2 + x_1 x_2) + \frac{K(K+1)}{2} (x_1 x_2)\).
- For the bilayer system, the 3-particle partition function is \(Z_3 = \frac{K(K-1)(K-2)}{3!} (x_1^3 + x_1^2 x_2 + x_1 x_2^2 + x_2^3) + \frac{K(K+1)(K-1)}{3} (x_1 x_2)(x_1 + x_2)\).
- For the bilayer system, the 4-particle partition function is \(Z_4 = \frac{K(K-1)(K-2)(K-3)}{4!} (x_1^4 + x_1^3 x_2 + x_1^2 x_2^2 + x_1 x_2^3 + x_2^4) + \frac{K(K+1)(K-1)(K-2)}{8} (x_1 x_2)(x_1^2 + x_2^2 + x_1 x_2) + \frac{K^2(K^2-1)}{12} (x_1^2 x_2^2)\).

## Context

This work extends the known relationship between partition functions of non-interacting quantum systems and symmetric functions, previously highlighted by Schmidt and Schnack (2001) and noted by Balantekin (1991). It provides a unified group-theoretical framework that connects earlier results including recursion relations for partition functions (Borrmann and Franke, 1993; Brosens, Devreese, and Lemmens, 1997), spectral equivalence of bosons and fermions in one-dimensional harmonic potentials (Schmidt and Schnack, 1999; Crescimanno and Landsberg, 2001), and character expansion methods for internal symmetries used in contexts from p\(\bar p\) annihilation (Muller, 1982) to color neutrality in quark-gluon plasma (Elze, 1986). The techniques enable simplified calculation of partition functions for particles with internal quantum numbers and extend to mixed boson-fermion (supersymmetric) systems and certain interacting systems where the Hamiltonian can be written as an element of an algebra.
