# SWANLOP : Scattering waves off nonlocal optical potentials in the presence of Coulomb interactions

> We introduce the package SWANLOP to calculate scattering waves and corresponding observables for nucleon elastic collisions off spin-zero nuclei. The code is capable of handling local and nonlocal optical potentials superposed to long-range Coulomb interaction. Solutions to the implied Schrödinger integro-differential equation are obtained by solving an integral equation of Lippmann-Schwinger type for the scattering wavefunctions, $ψ=ϕ_{C} + {G}_{C} {U}_{S}ψ$, providing and exact treatment to the Coulomb force [Phys. Lett. B 789, 256 (2019)]. The package has been developed to handle potentials either in momentum or coordinate representations, providing flexible options under each of them. The code is fully self-contained, being dimensioned to handle any $A\!\geq\!4$ target for nucleon beam energies of up to 1.1 GeV. Accuracy and benchmark applications are presented and discussed.

## Key claim

SWANLOP provides an exact treatment of the long-range Coulomb interaction superposed to nonlocal optical potentials by solving a Lippmann-Schwinger type integral equation for the scattering wavefunction, avoiding the approximations (iterative procedures, mean-value approximations, or Taylor expansions) used in prior methods.

## Method

The code solves the Schrödinger integro-differential equation for nucleon elastic scattering off spin-zero nuclei by discretizing the radial coordinate over a uniform mesh and inverting the matrix (1 − 𝕂) to obtain the scattering wavefunction u = (1 − 𝕂)⁻¹ u₀, where the kernel 𝕂 is constructed from Coulomb propagators and the nonlocal potential multipoles Uⱼₗ(r′,r). Potentials can be supplied either in coordinate or momentum representation; for momentum-space potentials, the code performs a double Fourier transform using Riccati-Bessel functions to obtain the coordinate-space multipoles needed for the kernel. Phase shifts and elastic scattering observables (differential cross section, analyzing power, spin rotation, and integrated cross sections) are extracted from the asymptotic behavior of the solution.

## Key numbers

- The code handles nucleon beam energies up to **1.1 GeV** for targets with mass number A ≥ 4.
- Running time ranges from **1 second** (low-energy nucleons off light targets) to **100 seconds** (1-GeV nucleons off heavy targets) on a conventional 2.6 GHz laptop.
- The maximum integration radius is set by Rₘₐₓ = r₀A¹/³ + R̅ with r₀ = **1.2 fm** and R̅ = **8 fm**.
- The radial step h is constrained by h ≲ π/(6k), requiring at least **6 sampling points per half-cycle** of the free wave.
- For 1-GeV protons off ²²⁶Ra, Rₘₐₓ ≈ **16 fm** and Lₘₐₓ = **130**.

## Context

SWANLOP extends the exact Coulomb treatment introduced in Arellano (2019) to a fully self-contained computational package, contrasting with the Perey-Buck method (Perey 1962) which uses a separable approximation and with iterative approaches (Titus 2014, 2016) that lack guaranteed convergence. The code complements the recently released SIDES package (Blanchon 2020), which solves the integro-differential equation via finite-difference techniques rather than the integral-equation approach used here. SWANLOP enables accurate calculation of scattering waves for use in distorted-wave Born approximations and provides a benchmark tool for testing nonlocal optical potentials derived from ab-initio approaches (Rotureau 2017, Idini 2019), energy-density functionals (Mizuyama 2012, Blanchon 2015), and g-matrix or t-matrix based models (Arellano 1990, 1995; Elster 1990; Vorabbi 2016).
