# Role of initial transverse momentum in a hybrid approach

> The purpose of this work is to study the effect of exchanging initial condition models in a modular hybrid approach. The focus lies on the event-by-event correlations of elliptic and triangular flow. This study is performed in the hybrid approach SMASH-vHLLE, composed of the hadronic transport approach SMASH and the (3+1)d viscous hydrodynamic code vHLLE. The initial condition models investigated are SMASH IC, Trento and IP-Glasma. Correlations are calculated on an event-by-event basis between the eccentricities and momentum anisotropies of the initial state as well as the momentum anisotropies in the final state, both for ultra-central and off-central collisions for AuAu collisions at $\sqrt{s_{NN}} = 200$ GeV. This work demonstrates that, although averaged values for the eccentricities of these models are very similar, substantial differences exist both in the distributions of eccentricities, the correlations amongst the initial state properties as well as in the correlations between initial state and final state properties. Notably, whereas initial state momentum anisotropy is shown to not affect the final state flow, the presence of radial flow affects the emergence of final state momentum anisotropies. Inclusion of radial flow in the linear fit improves the prediction of final state flow from initial state properties. The presence of momentum in the initial state has an effect on the emergence of flow and is therefore a relevant part of initial state models, challenging the common understanding of final state momentum anisotropies being a linear response to initial state eccentricity only.

## Key claim

Although averaged eccentricities across the SMASH IC, Trento, and IP-Glasma initial condition models are very similar for large nuclei, substantial differences exist in event-by-event distributions and correlations between initial and final state properties; notably, initial state radial flow (not momentum anisotropy) significantly affects the emergence of final state flow, challenging the common assumption that final state momentum anisotropies are solely a linear response to initial state eccentricity.

## Method

The study was performed using the SMASH-vHLLE hybrid approach, combining the hadronic transport code SMASH with the (3+1)d viscous hydrodynamic code vHLLE for Au-Au collisions at sqrt(s_NN) = 200 GeV. Three initial condition models were compared: SMASH IC (which provides full 3D initial conditions including momentum information), Trento (a parametric entropy deposition model), and IP-Glasma (a Color-Glass Condensate based model). Event-by-event correlations were calculated between initial state eccentricities (epsilon_2, epsilon_3), initial state momentum anisotropies, and final state flow coefficients (v_2, v_3) for both ultra-central (0-5%) and off-central (20-30%) collisions, using 750 hydrodynamic events per setup with 1000 sampled afterburner events for IP-Glasma.

## Key numbers

- Au-Au collisions at sqrt(s_NN) = 200 GeV were the primary system studied, with additional comparisons for Pb-Pb at sqrt(s_NN) = 5020 GeV and O-O at sqrt(s_NN) = 7 TeV
- The switching energy density for the transition from hydrodynamics to hadronic transport was set to epsilon_switch = 0.5 GeV/fm^3
- IP-Glasma events were evolved until tau = 0.5 fm before hydrodynamic initialization
- 750 event-by-event viscous hydrodynamic events were generated for each setup, with 1000 sampled afterburner events for IP-Glasma
- The SMASH initial condition proper time was defined as tau_0 = (R_P + R_T) / sqrt((sqrt(s_NN)/(2 m_N))^2 - 1), where m_N is the nucleon mass
- For Au-Au at 200 GeV, IP-Glasma produced the highest eccentricity values overall, except for epsilon_3 in very peripheral collisions at high energies where SMASH gave slightly higher values
- The Pearson correlation between epsilon_2 and v_2 was strongest for Trento, followed by SMASH, and weakest for IP-Glasma (especially in central collisions)
- Inclusion of initial state radial flow (langle p_T^{IC} rangle) in linear regression models improved the r^2 value considerably more than inclusion of any other independent variable for predicting final state flow

## Context

This work extends comparisons of initial condition models in hybrid approaches, specifically building on the SMASH-vHLLE framework developed by Schafer (2021) and the shear viscosity parameterization of Gotz (2022). It directly compares three commonly used initial condition models, SMASH IC, Trento (Moreland, 2014), and IP-Glasma (Schenke, 2012), that are frequently employed in Bayesian inference studies for extracting transport coefficients such as shear viscosity over entropy ratio (eta/s). The finding that initial state radial flow significantly affects final state flow challenges the linear response assumption commonly used in analyses by Noronha-Hostler (2015) and others, and suggests that Bayesian inference studies relying on a single initial state model may have limited predictive strength. The results enable improved modeling of the initial state by demonstrating that momentum information, particularly radial flow, is a relevant component that should be included in initial condition models for heavy-ion collisions.
