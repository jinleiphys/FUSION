# Extensions of Superscaling from Relativistic Mean Field Theory: the SuSAv2 Model

> We present a systematic analysis of the quasielastic scaling functions computed within the Relativistic Mean Field (RMF) Theory and we propose an extension of the SuperScaling Approach (SuSA) model based on these results. The main aim of this work is to develop a realistic and accurate phenomenological model (SuSAv2), which incorporates the different RMF effects in the longitudinal and transverse nuclear responses, as well as in the isovector and isoscalar channels. This provides a complete set of reference scaling functions to describe in a consistent way both $(e, e')$ processes and the neutrino/antineutrino-nucleus reactions in the quasielastic region. A comparison of the model predictions with electron and neutrino scattering data is presented.

## Key claim

The SuSAv2 model extends the original SuSA approach by incorporating RMF-based differences between longitudinal and transverse nuclear responses and between isoscalar and isovector channels, while correcting the RMF model's overly strong momentum-transfer dependence by fixing the reference scaling functions at q = 650 MeV/c.

## Method

The authors computed quasielastic scaling functions for electron, neutrino, and antineutrino scattering on 12C using the Relativistic Mean Field (RMF) and Relativistic Plane Wave Impulse Approximation (RPWIA) models. They identified three distinct families of scaling functions (transverse, longitudinal isovector, and longitudinal isoscalar) and parametrized each using a skewed-Gumbel function fitted to the RMF results at q = 650 MeV/c, then used these as reference functions to construct all nuclear responses via q-dependent ratios.

## Key numbers

- The RMF longitudinal scaling function at q = 650 MeV/c provides the best fit to experimental longitudinal data, with the minimum reduced chi-squared found around this momentum transfer.
- RMF peak heights are approximately 25% lower than RPWIA values for the transverse set, and 30-70% lower for the longitudinal isovector set depending on q.
- The integrals of RMF transverse scaling functions exceed unity and increase almost linearly with q, while the longitudinal isovector integrals fall below unity and stabilize around 0.7 for q between 900 and 1200 MeV/c.
- RPWIA integrals are nearly q-independent: approximately 1 for the longitudinal set and approximately 1.05 for the transverse set.

## Context

This work extends the SuperScaling Approach (SuSA) of Amaro et al. (2005) by incorporating the RMF model's successful description of the experimental longitudinal scaling function from Jourdan (1996) and its prediction of f_T > f_L, a violation of zeroth-kind scaling first analyzed by Caballero et al. (2007). The SuSAv2 model enables fast numerical predictions for neutrino-nucleus cross sections that retain the essential physics of the computationally expensive RMF model, and it provides a consistent framework for describing both electron and neutrino scattering in the quasielastic region.
