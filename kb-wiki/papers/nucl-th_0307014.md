# Proton-Induced Fission Cross Section Calculation with the LANL Codes CEM2K+GEM2 and LAQGSM+GEM2

> The improved Cascade-Exciton Model code CEM2k and the Los Alamos version of the Quark-Gluon String Model code LAQGSM, previously merged with the Generalized Evaporation Model code of Furihata (GEM2) were further modified to provide reliable proton-induced fission cross sections for applications. By adjusting two parameters in GEM2 for each measured reaction, we were able to describe very well with CEM2k+GEM2 and LAQGSM+GEM2 all available experimental fission cross sections induced by protons with energies from 20 MeV to 10 GeV both for subactinide and actinide targets. We also successfully tested our approach on several reactions induced by neutrons, pions, and photons.

## Key claim

By adjusting only two parameters in the GEM2 fission model (the level-density parameter ratio for preactinides and the C(Z) constant for actinides), the authors achieved good agreement with Prokofiev's systematics for proton-induced fission cross sections across energies from 20 MeV to 10 GeV, but the fitted parameters must be interpolated or extrapolated for nuclei and energies not covered by those systematics.

## Method

The authors merged the Cascade-Exciton Model code CEM2k and the Quark-Gluon String Model code LAQGSM with the Generalized Evaporation Model GEM2, then introduced multiplicative scaling factors C_a (for the level-density parameter a_f) and C_c (for the fission-width constant C(Z)) into Atchison's fission model within GEM2. They fitted these two parameters for each target nucleus and incident proton energy to reproduce Prokofiev's systematics of proton-induced fission cross sections, then stored the fitted values in data blocks with interpolation/extrapolation routines for use outside the fitted region.

## Key numbers

* Prokofiev's systematics for subactinide nuclei (Ho to Bi) at proton energies above 70 MeV uses four fitting parameters P1-P4, with P4 = 0 for Z^2/A ≤ 32.32 and P4 = -1.140 + 0.0352*(Z^2/A) for Z^2/A > 32.32.
* For actinide nuclei (Th to Pu) at proton energies above 20 MeV, Prokofiev's systematics gives P2 = 12.1, P3 = 0.111, P4 = 0.067, and P1 = 2572 * {1 - exp[-2.069*(Z^2/A - 34.99)]}.
* For preactinides between Au and Bi at proton energies 35-70 MeV, the systematics uses a Gaussian with E0 = 76.3 MeV and parameters w and sigma0 fitted with coefficients a = -33.667, b = 1.5699, c = 0.30069 for w.
* The interpolation constants for nuclei between Bi and Th at energies above 70 MeV include C11 = -27.74, C12 = 0.9906, C21 = 25.83, C22 = -0.6567, C31 = -45.80, C32 = 1.227, C41 = -10.95, C42 = 0.2320.

## Context

This work extends the earlier merging of CEM2k and LAQGSM with GEM2 (Mashnik et al., 2002) by addressing the failure of the unmodified GEM2 to reproduce fission cross sections when coupled with the authors' intra-nuclear cascade codes, which differ from the Bertini INC used by Atchison (1980) and Furihata (2000). The approach enables reliable fission cross section calculations for applications such as accelerator-driven systems and space radiation shielding, and was successfully tested on neutron-, pion-, and photon-induced reactions without further parameter adjustment.
