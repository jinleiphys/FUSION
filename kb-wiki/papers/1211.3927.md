# Complete Experiments for Pion Photoproduction

> The possibilities of a model-independent partial wave analysis for pion, eta or kaon photoproduction are discussed in the context of complete experiments. It is shown that the helicity amplitudes obtained from at least 8 polarization observables including beam, target and recoil polarization can not be used to analyze nucleon resonances. However, a truncated partial wave analysis, which requires only 5 observables will be possible with minimal model assumptions.

## Key claim

A model-independent amplitude analysis for pion photoproduction requires at least 8 polarization observables but cannot be used to analyze nucleon resonances due to an unknown energy- and angle-dependent phase; however, a truncated partial wave analysis (TPWA) requires only 5 observables and is feasible with minimal model assumptions.

## Method

The author discusses the mathematical framework for pseudoscalar meson photoproduction using helicity amplitudes and 16 polarization observables grouped into single-spin, beam-target, beam-recoil, and target-recoil categories. A truncated partial wave analysis (TPWA) is performed by expanding observables in Legendre or cosine series up to a maximal angular momentum, then fitting multipoles directly from the angular distributions of a minimal set of observables. Pseudo-data were generated from the MAID2007 model for gamma p -> pi0 p and gamma p -> pi+ p over an energy range of 200-1200 MeV, with single-energy fits using 4, 6, and 8 observables.

## Key numbers

- Pseudo-data were generated over an energy range from E_lab = 200-1200 MeV and a full angular range of theta = 0-180 degrees.
- Beam energy bins of Delta E_gamma = 10 MeV and angular bins of Delta theta = 10 degrees were used.
- For each observable, typically 5 x 10^6 events were generated over the full energy range.
- For an S,P wave truncation (l_max = 1), there are 4 complex multipoles (E0+, E1+, M1+, M1-) leading to 7 free real parameters and an arbitrary phase.
- For any PWA with truncation at l_max, one can measure (8*l_max, 8*l_max, 8*l_max+4, 8*l_max+4) coefficients for types (S, BT, BR, TR), respectively.

## Context

This work extends the classical "complete experiments" framework of Barker, Donnachie and Storrow (1975) and addresses discrete ambiguity studies from the 1990s. It contradicts the assumption that a full set of 8 polarization observables is sufficient for resonance analysis, showing instead that a truncated partial wave analysis with only 5 observables (e.g., sigma0, Sigma, T, P, F or sigma0, Sigma, T, F, G) is more practical. This enables model-independent extraction of nucleon resonance parameters using data from facilities like Mainz, Bonn, and JLab, building on prior work by Grushin (1980s) for the Delta(1232) region and recent efforts by Workman (2010, 2011), Dey (2010), and Sandorfi (2010).
