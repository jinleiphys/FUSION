# A covariant model for the negative parity resonances of the nucleon

> We present a model for the $γ^\ast N \to N^\ast$ helicity amplitudes, where $N$ is the nucleon and $N^\ast$ is a negative parity nucleon excitation, member of the $SU(6)$-multiplet $[70,1^-]$. The model combines the results from the single quark transition model for the helicity amplitudes with the results of the covariant spectator quark model for the $γ^\ast N \to N^\ast(1535)$ and $γ^\ast N \to N^\ast(1520)$ transitions. With the knowledge of the amplitudes $A_{1/2}$ and $A_{3/2}$ for those transitions we calculate three independent coefficients defined by the single quark transition model and make predictions for the helicity amplitudes associated with the $γ^\ast N \to N^\ast(1650)$, $γ^\ast N \to N^\ast(1700)$, $γ^\ast N \to Δ(1620)$, and $γ^\ast N \to Δ(1700)$ transitions. In order to facilitate the comparison with future experimental data at high $Q^2$, we provide also simple parametrizations for the amplitudes, compatible with the expected falloff at high $Q^2$.

# A covariant model for the negative parity resonances of the nucleon

## Key claim

The model combines the covariant spectator quark model with the single quark transition model (SQTM) to predict helicity amplitudes for negative parity nucleon resonances in the $[70,1^-]$ multiplet, but the predictions are expected to be accurate only for $Q^2 \gtrsim 2$ GeV$^2$ where valence quark effects dominate and meson cloud contributions are suppressed.

## Method

The covariant spectator quark model, treating baryons as quark-diquark systems with on-mass-shell spectator diquarks, was used to calculate the $\gamma^\ast N \to N(1535)$ and $\gamma^\ast N \to N(1520)$ helicity amplitudes. These amplitudes were then used to determine three independent SQTM coefficients ($A$, $B$, $C$) via the relations $A = 2 A_{1/2}^{S11}/\cos\theta_S + \sqrt{2} A_{1/2}^{D13} + \sqrt{6} A_{3/2}^{D13}$, $B = 2 A_{1/2}^{S11}/\cos\theta_S - 2\sqrt{2} A_{1/2}^{D13}$, and $C = -2 A_{1/2}^{S11}/\cos\theta_S - \sqrt{2} A_{1/2}^{D13} + \sqrt{6} A_{3/2}^{D13}$, from which predictions for the $\gamma^\ast N \to N(1650)$, $\gamma^\ast N \to N(1700)$, $\gamma^\ast N \to \Delta(1620)$, and $\gamma^\ast N \to \Delta(1700)$ transitions were derived. Two model variants were considered: model 1 with no meson cloud ($A_{3/2}^{D13}=0$, giving $C=-A$) and model 2 including meson cloud effects via a parametrized $G_4^\pi$ function fitted to data.

## Key numbers

- Mixing angle for $N\frac{1}{2}^-$ states: $\theta_S = 31^\circ$
- Mixing angle for $N\frac{3}{2}^-$ states: $\theta_D = 6^\circ$
- For $N(1650)$ $A_{1/2}$ parametrization at high $Q^2$: $D = 68.90 \times 10^{-3}$ GeV$^{-1/2}$, $\Lambda^2 = 3.35$ GeV$^2$
- For $N(1700)$ $A_{1/2}$ parametrization: $D = -8.51 \times 10^{-3}$ GeV$^{-1/2}$, $\Lambda^2 = 2.82$ GeV$^2$
- For $N(1700)$ $A_{3/2}$ parametrization: $D = 4.36 \times 10^{-3}$ GeV$^{-1/2}$, $\Lambda^2 = 3.61$ GeV$^2$
- For $\Delta(1700)$ $A_{1/2}$ parametrization: $D = 39.22 \times 10^{-3}$ GeV$^{-1/2}$, $\Lambda^2 = 2.69$ GeV$^2$
- For $\Delta(1700)$ $A_{3/2}$ parametrization: $D = 42.15 \times 10^{-3}$ GeV$^{-1/2}$, $\Lambda^2 = 8.42$ GeV$^2$
- For $\Delta(1620)$ $A_{1/2}$ parametrization: $A_{1/2} = 77.21 (\Lambda^2/(\Lambda^2+Q^2))^{5/2}$ in units $10^{-3}$ GeV$^{-1/2}$, with $\Lambda^2 = 1$ GeV$^2$ (note the $5/2$ power instead of the expected $3/2$)
- The Pauli transition form factor $F_2^\ast$ for $N(1535)$ vanishes for $Q^2 \gtrsim 1.5$ GeV$^2$
- Expected falloff behavior: $A_{1/2} \propto 1/Q^3$ and $A_{3/2} \propto 1/Q^5$ for $Q^2 \approx 5$ GeV$^2$

## Context

This work extends the covariant spectator quark model previously applied to the $N(1535)$ and $N(1520)$ transitions (Ramalho, 2011; Ramalho, 2013) by combining it with the SQTM framework of Burkert (2003) to predict amplitudes for the remaining $[70,1^-]$ multiplet members. The predictions are intended to guide experiments at the Jefferson Lab 12 GeV upgrade, particularly for resonances in the second and third resonance regions ($W = 1.4$--$1.8$ GeV). The model reveals that for some transitions, such as $\gamma^\ast N \to \Delta(1620)$, valence quark effects are strongly suppressed at $Q^2 = 1$--$2$ GeV$^2$, contrary to usual expectations, and that meson cloud contributions (as calculated by EBAC at Jefferson Lab) are essential for describing the $N(1520)$ $A_{3/2}$ amplitude.
