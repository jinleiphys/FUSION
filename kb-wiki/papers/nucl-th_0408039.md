# Kaon electromagnetic production: constraints set by new data

> The CLAS data on the photo-production of K+ off the proton are utilised to study reaction mechanism of the process in frame of the isobaric approach. The missing D13 resonance is shown to be important for successful description of the data in the whole kinematical region. Constructed models provide satisfactory predictions for the process.

# Kaon electromagnetic production: constraints set by new data

## Key claim
The CLAS data on K⁺ photoproduction off the proton require inclusion of the "missing" D₁₃(1895) resonance in isobaric models to correctly describe the resonant structure of cross sections, particularly the second maximum in the energy region 1.3–1.8 GeV. The N*(1710) resonance facilitates description of the first maximum but is not strictly necessary.

## Method
The authors used an isobaric Feynman-diagram approach with one-particle exchange, including Born terms, kaon resonances K*(891) and K₁(1273), nucleon resonances (S₁₁(1650), P₁₁(1710), P₁₃(1720)), hyperon resonances, and optionally the D₁₃(1895) resonance. Hadron structure was incorporated via dipole-type form factors introduced in a gauge-invariant way following Davidson and Workman (2001). Model parameters (coupling constants and form-factor cut-offs) were fitted to 918 differential cross-section data points and 220 polarization data points from CLAS, with coupling constants constrained within 20% broken SU(3) symmetry limits.

## Key numbers
- CLAS data set used: 918 differential cross-section points and 220 polarization points
- SU(3) symmetry constraints: −4.4 ≤ g_{KNΛ} ≤ −3.0 and 0.8 ≤ g_{KNΣ} ≤ 1.3
- Cut-off mass constraints: 0.6 ≤ Λ ≤ 2.0 GeV
- χ² values for models: 2.85 (M1, without D₁₃), 2.70 (M2, with D₁₃), 3.40 (M3, without N6), 2.55 (H1, with hyperons Y2 and L5), 2.74 (H2, with hyperons Y2 and Y3)
- Re-fitted Saclay-Lyon models: χ² improved from 398 to 5.7 (SL) and from 412 to 7.2 (SLA)
- Cut-off parameters for best hyperon models: 0.73 GeV (H1) and 0.88 GeV (H2)
- First maximum in cross section occurs at E_γ^{lab} ≈ 1.15 GeV

## Context
This work extends earlier isobaric models, Saclay-Lyon (SL96, SLA98), Kaon-MAID (Ben99), and Janssen et al., which had difficulty describing new CLAS and SAPHIR data. The analysis shows that the D₁₃(1895) resonance, predicted by quark models and already assumed in Kaon-MAID but not reliably observed, is essential for correct description of the second maximum in the cross section across all angles. The results enable more reliable elementary amplitudes for hypernucleus production calculations, particularly at very small kaon lab angles (θ_K^{lab} < 10 deg) where the process contributes dominantly to hypernucleus production rates.
