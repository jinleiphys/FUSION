# Fresh extraction of the proton charge radius from electron scattering

> We present a novel method for extracting the proton radius from elastic electron-proton ($ep$) scattering data. The approach is based on interpolation via continued fractions augmented by statistical sampling and avoids any assumptions on the form of function used for the representation of data and subsequent extrapolation onto $Q^2\simeq 0$. Applying the method to extant modern $e p$ data sets, we find that all results are mutually consistent and, combining them, arrive at $r_p=0.847(8)\,$fm. This result compares favourably with values obtained from contemporary measurements of the Lamb shift in muonic hydrogen, transitions in electronic hydrogen, and muonic deuterium spectroscopy.

## Key claim

The authors claim that, using a model-independent interpolation method (the Schlessinger Point Method, SPM) on existing electron-proton scattering data, all modern data sets are mutually consistent and yield a combined proton charge radius of 0.847(8) fm, which resolves the "proton radius puzzle" by suggesting that the earlier discrepancy arose from systematic errors introduced by specific functional forms used to extrapolate scattering data to zero momentum transfer.

## Method

The authors applied the Schlessinger Point Method (SPM), a continued-fraction interpolation technique, to elastic electron-proton scattering data from the PRad experiment at Jefferson Lab and the A1 experiment at Mainz. Before interpolation, each data set was smoothed using cubic splines with a roughness penalty, with the optimal smoothing parameter determined by cross-validation. The SPM was then applied to 1000 bootstrap replicas of each smoothed data set, generating 5000 interpolating functions per replica from randomly selected subsets of data points, from which the proton charge radius was extracted via the derivative of the electric form factor at Q²=0.

## Key numbers

- Combined result from all data sets: r_p = 0.847 ± 0.008_stat fm
- PRad 1.1 GeV data alone: r_p = 0.842 ± 0.008_stat fm
- PRad 2.2 GeV data alone: r_p = 0.824 ± 0.003_stat fm
- Combined PRad data: r_p = 0.838 ± 0.005_stat fm
- A1 low-Q² data (3.8×10⁻³ ≤ Q²/GeV² ≤ 1.4×10⁻²): r_p = 0.856 ± 0.014_stat fm
- A1 full data set: r_p = 0.857 ± 0.021_stat fm
- PRad experiment reached Q² = 2.1×10⁻⁴ GeV², the lowest momentum transfer achieved in electron-proton scattering
- For comparison: muonic hydrogen Lamb shift gives r_p = 0.84136(39) fm (Pohl 2010, Antognini 2013)

## Context

This work extends the ongoing effort to resolve the "proton radius puzzle" that emerged when muonic hydrogen measurements (Pohl 2010) gave a significantly smaller proton radius than earlier electron scattering analyses. The authors' SPM approach avoids the functional-form assumptions that previous analyses (including the PRad collaboration's own bootstrap procedure described by Yan 2018) relied upon. The result enables a consistent interpretation of all modern proton radius measurements from electron scattering, muonic hydrogen, and electronic hydrogen spectroscopy, and suggests that future analyses of electron scattering data should employ form-unbiased interpolation methods to avoid systematic errors.
