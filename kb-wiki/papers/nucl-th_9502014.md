# Inverted Pairing and Excitation Induced by Quadrupole-Quadrupole Interaction

> In this paper, eit is shown for a single-$j$ shell that the intrinsic excitations induced by the quadrupole-quadrupole interaction can be viewed as that due to the inverted surface-delta interaction whose multipole-pairing component with the largest spin is the dominant term and is attractive. This allow us to trace the origin of the recently introduced optimum pair description of rotation and leads naturally to analytical expressions for the rotational energies for the various bands in a single $j$-shell with the quadrupole-quadrupole interaction.

## Key claim

For a single-\(j\) shell, the intrinsic excitations induced by the quadrupole-quadrupole (QQ) interaction can be viewed as arising from an inverted surface-delta interaction (SDI), whose multipole-pairing component with the largest spin is the dominant term and is attractive; this equivalence holds only after subtracting a constant plus an \(L^2\) term from the QQ interaction via an auxiliary Hamiltonian.

## Method

The authors decompose the QQ interaction in a single-\(j\) shell into a collective part (constant plus \(L^2\) term) and an intrinsic part. They introduce an auxiliary Hamiltonian \(H' = H_{QQ} - \alpha \sum_{i<j} I_{ij} - \gamma \mathbf{L}^2\) and choose parameters \(\alpha\) and \(\gamma\) so that the remaining multipole-pairing strengths \(C'_\Lambda\) follow an inverted SDI pattern (strengths increasing with \(\Lambda\)). They then apply the Tamm-Dancoff approximation (TDA) to this inverted SDI to obtain analytical expressions for rotational energies, and for the four-particle case they derive a more accurate \(\omega\) parameter via a generalized pair field method.

## Key numbers

- For \(j = 21/2\), the multipole-pairing strengths \(C'_\Lambda\) (inverted SDI) are: \([-0.001, 0, 0, -0.0093, -0.04, -0.11, -0.24, -0.46, -0.79, -1.27, -1.95]\) for \(\Lambda = 0, 2, 4, \dots, 20\).
- The TDA ground state energy for four particles in \(j = 21/2\) is only a third (for identity interaction) and a fifth (for QQ interaction) of the exact value.
- The analytical expression for the moment of inertia for a two-pair system is \(\frac{\hbar^2}{2\Im} = \chi [2 \sqrt{L_1(L_1+1)} + 2\sqrt{L_2(L_2+1)} - 1]\), with \(\chi = \frac{15 [3L_1(L_1+1)-4j(j+1)-3][3L_2(L_2+1)-4j(j+1)-3]}{j(j+1)(2j-1)(2j+1)(2j+3) (2L_1-1)(2L_2-1)(2L_1+3)(2L_2+3)}\).
- The total energy for a two-pair system is given by \(E = E_o + \frac{\hbar^2}{2\Im} L(L+1) - \chi L^2(L+1)^2\), where the \(\chi L^2(L+1)^2\) term is negligible for the QQ interaction.

## Context

This work extends the optimum pair description of rotation introduced by Chen, Wu, Feng, and Arima (1994) by tracing its origin to the inverted SDI nature of the QQ interaction in a single-\(j\) shell. It provides analytical expressions for rotational energies that agree well with exact shell model calculations for four particles in \(j=21/2\), as shown in Table I. The approach enables a simple Tamm-Dancoff approximation treatment of excited bands and suggests a path toward generalizing the results to systems with an arbitrary number of pairs, though the number dependence is not yet understood.
