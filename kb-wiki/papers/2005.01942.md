# Graphic Method for Arbitrary $n$-body Phase Space

> In quantum field theory, the phase space integration is an essential part in all theoretical calculations of cross sections and decay widths. It is also needed for computing the imaginary part of a physical amplitude. A key problem is to get the phase space formula expressed in terms of any chosen invariant masses in an $n$-body system. We propose a graphic method to quickly get the phase space formula of any given invariant masses intuitively for an arbitrary $n$-body system in general $D$-dimensional spacetime, with the involved momenta in any reference frame. The method also greatly simplifies the phase space calculation just as what Feynman diagrams do in calculating scattering amplitudes.

# Graphic Method for Arbitrary \(n\)-body Phase Space

## Key claim
The authors propose a graphic method that can intuitively and efficiently produce the phase space formula for any chosen invariant masses in an arbitrary \(n\)-body system in general \(D\)-dimensional spacetime, with the involved momenta in any reference frame. The method is presented as a simplification tool analogous to Feynman diagrams for scattering amplitudes, but the authors note that the diagrams "should not be understood as Feynman diagrams."

## Method
The method is based on a recursive relation for the \(n\)-body phase space element, which is represented graphically using single lines (single particles), double lines (multi-particle systems with invariant mass squared factors \((2\pi)^3 \mathrm{d} m_{j_1j_2\cdots j_l}^2\)), and vertices (phase space elements \(\mathrm{d}\Phi_l\)). Five drawing rules govern the construction of diagrams, including that there must be exactly one route of double lines between any two vertices, and that duplicate single lines for the same particle are represented by dashed lines with a factor that cancels double-counting. The \(n\)-body phase space element is then decomposed into products of 2-body phase space elements, which are given explicitly in both general frames and the center-of-mass frame.

## Key numbers
- For an \(n\)-body system with \(n \geq 3\), at most \(3n-7\) variables can be replaced by invariant masses, with the remaining three variables replaced by Euler angles \(\alpha, \beta, \gamma\).
- For a complete-expansion diagram, the relation \(d - l = n - 2\) holds, where \(d\) is the number of double lines and \(l\) is the number of internal single lines.
- For a tree diagram (\(l = 0\)), one finds \(d = n - 2\), meaning an \(n\)-body phase space element with \(n-2\) invariant masses as integral variables.
- If \(d\) takes its maximal value \(3n-7\), one gets \(l = 2n-5\), corresponding to a \((2n-5)\)-loop diagram for an \(n\)-body phase space element with \(3n-7\) invariant masses.
- The chain tree diagram for \(n\)-body phase space yields the expression \(\mathrm{d}\Phi_n = \frac{|\mathbf{p}_1| \mathrm{d}\Omega_1}{2^n (2\pi)^{3n} m} \prod_{i=2}^{n-1} |\mathbf{p}_i| \mathrm{d}\Omega_i \mathrm{d} m_{(i)}\), with integration region for each invariant mass \(m_{(i)}\) given by \(\left[\sum_{k=i}^n m_k, m_{(i-1)} - m_{i-1}\right]\) for \(i = 2,3,\ldots,n-1\) and \(m_{(1)} = m\).
- For the 4-body phase space expressed in terms of three invariant masses, the result is \(\mathrm{d}\Phi_4 = \frac{|\mathbf{p}'_{12}| |\mathbf{p}''_1|}{(2\pi)^{12} 2^7 m m_{12} m_{234} |\mathbf{p}^*_2|} \mathrm{d} m_{12}^2 \mathrm{d} m_{34}^2 \mathrm{d} m_{23}^2 \mathrm{d}\Omega_{12}' \mathrm{d}\Omega_1'' \mathrm{d}\phi_4^* \times \theta_{[-1,1]}\left(\frac{m_{23}^2 + m_2^2 - m_3^2 - 2p_{23}^{*0} p_2^{*0}}{2|\mathbf{p}_2^*| |\mathbf{p}_4^*|}\right)\), with integration regions for \(m_{12}\) and \(m_{34}\) being \([m_1+m_2, m-m_3-m_4]\) and \([m_3+m_4, m-m_{12}]\), respectively.

## Context
This work extends the standard phase space formulas given in the Review of Particle Physics (Zyla et al., 2020) for 2- and 3-body systems to arbitrary \(n\)-body systems, addressing the difficulty that phase space integration becomes "much more involved" for more than three final-state particles. The method is applicable to dimensional regularization contexts such as soft-gluon resummation (Forte, 2002) and general \(D\)-dimensional spacetime quantum field theory. It enables straightforward computation of invariant mass distributions for any \(n\)-body final state, which is important for identifying resonances (excited hadrons, Higgs particle) as peaks in invariant mass distributions, and for calculating the imaginary part of physical amplitudes through unitarity.
