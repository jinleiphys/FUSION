# $ΔπN$ coupling constant in light cone QCD sum rules

> We employ the light cone QCD sum rules to calculate $ΔπN$ coupling constant by studying the two point correlation function between the vacuum and the pion state. Our result is consistent with the traditional QCD sum rules calculations and it is in agreement with the experimental value.

## Key claim

The authors use light cone QCD sum rules to calculate the ΔπN coupling constant, obtaining a value consistent with both traditional QCD sum rules and experiment, but the result depends on the choice of continuum threshold and Borel parameter and carries a ±1.5 GeV⁻¹ uncertainty.

## Method

The authors study a two-point correlation function between the vacuum and the pion state, using interpolating currents for the Δ and nucleon. They perform a double Borel transformation to separate out excited states and continuum contributions, then match the phenomenological and theoretical sides of the sum rule for the Lorentz structure qμ. The theoretical side is computed using light cone operator product expansion with two-particle pion wave functions up to twist four.

## Key numbers

- gΔπN = (14.5 ± 1.5) GeV⁻¹ from the light cone QCD sum rule calculation.
- The experimental value derived from the Δ⁺⁺ decay width is gΔπN = (15.2 ± 0.1) GeV⁻¹.
- Input parameters include: fπ = 0.132 GeV, μπ = 1.65 GeV, ⟨q̄q⟩ = -(0.225 GeV)³, m₀² = 0.8 GeV², ⟨gs²G²⟩ = 0.474 GeV⁴, and continuum threshold s₀ = 3 GeV².
- The residues λN and λΔ are obtained from sum rules using a = 0.5 GeV³ and b = 0.12 GeV⁴.

## Context

This work extends the traditional QCD sum rules calculation of the ΔπN coupling constant in the soft pion limit (Ref. R11) by applying the light cone QCD sum rules method. The result is consistent with that earlier calculation and with the experimental value from the Δ⁺⁺ decay width. The approach enables future studies of hadronic couplings using light cone wave functions, which are better known at specific points than the full wave functions.
