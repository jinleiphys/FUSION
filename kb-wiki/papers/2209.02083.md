# Trimmed Sampling Algorithm for the Noisy Generalized Eigenvalue Problem

> Solving the generalized eigenvalue problem is a useful method for finding energy eigenstates of large quantum systems. It uses projection onto a set of basis states which are typically not orthogonal. One needs to invert a matrix whose entries are inner products of the basis states, and the process is unfortunately susceptible to even small errors. The problem is especially bad when matrix elements are evaluated using stochastic methods and have significant error bars. In this work, we introduce the trimmed sampling algorithm in order to solve this problem. Using the framework of Bayesian inference, we sample prior probability distributions determined by uncertainty estimates of the various matrix elements and likelihood functions composed of physics-informed constraints. The result is a probability distribution for the eigenvectors and observables which automatically comes with a reliable estimate of the error and performs far better than standard regularization methods. The method should have immediate use for a wide range of applications involving classical and quantum computing calculations of large quantum systems.

## Key claim
The trimmed sampling algorithm uses Bayesian inference with physics-informed constraints to produce posterior distributions for eigenvectors and observables that automatically yield reliable error estimates and perform "far better" than standard regularization methods like ridge regression, though the method shows some systematic underestimation of error near avoided level crossings.

## Method
The algorithm starts with noisy estimates of Hamiltonian and norm matrices (H̃, Ñ) and their one-standard-deviation uncertainties (ΔH̃, ΔÑ). A prior distribution is taken as a product of uncorrelated Gaussians centered on these estimates, and a likelihood function enforces two physics-informed constraints: the norm matrix must be positive definite, and the convergence ratio of ground-state energies with increasing subspace size must not exceed a chosen upper bound C (set to 2.5 in the benchmarks). Random samples are drawn from the prior using a heat bath algorithm, then reweighted by the likelihood to produce the posterior distribution, from which weighted median values and 16th/84th percentile error bars are computed.

## Key numbers
- For the Bose-Hubbard model (4 bosons on a 4×4×4 lattice, μ = -6t, five training values U/t = -2.5, -1.9, -1.8, -1.7, -1.6): noise was introduced by rounding matrix entries at the sixth decimal place, with error estimates ΔH̃_ij = ΔÑ_ij = (1/√12) × 10⁻⁶.
- For the Heisenberg model (N=10 sites, subspace with Σ σ_z = 0): random Gaussian noise with standard deviation σ = 0.01 was applied to each matrix element, with uncertainty estimates ΔH̃_ij = ΔÑ_ij = 0.01.
- For both benchmarks, 500 samples with nonzero posterior probability were produced, using C = 2.5 for the convergence likelihood factor.
- Error bars are reported as weighted median with lower/upper limits at the 16th and 84th percentiles.

## Context
This work extends eigenvector continuation (Frame 2017) and the broader class of reduced basis methods (Quarteroni 2015) to handle stochastic noise from Monte Carlo simulations and quantum computing measurements. It directly addresses the ill-posed inverse problem that arises in the Hill-Wheeler equation (Hill 1952, Griffin 1957) and generator coordinate method, where Tikhonov/ridge regression (Tikhonov 1943) has been the standard regularization but requires hand-tuning of the regularization parameter ε and provides no systematic error estimation. The method enables reliable extraction of energy eigenvalues and observables from noisy generalized eigenvalue problems arising in classical and quantum computing calculations of large quantum systems.
