# A model for multi-quark systems

> As a step towards understanding multi-quark systems abundant in nature we construct a model that reproduces the binding energies of static four-quark systems. These energies have been calculated using SU(2) lattice gauge theory for a set of six different geometries representative of the general case. The model is based on ground and excited state two-body potentials and multi-quark interaction terms.

## Key claim

The authors construct a model that reproduces the binding energies of static four-quark systems calculated using SU(2) lattice gauge theory, but only by including excited gluonic states (a 6×6 basis) in addition to the usual two-body potentials and a multi-quark interaction term; a simpler model using only two-body potentials (2×2 basis) fails qualitatively.

## Method

The model uses a basis of three ground-state quark pairings (A, B, C) plus three excited-state pairings (A*, B*, C*) where the gluonic field is in the E_u representation. The potential matrix elements are constructed from two-body potentials fitted to lattice data (Eqs. Vfit and V*fit), with off-diagonal elements modulated by overlap factors f, f^a, f^c that depend on the minimal area S between quarks. The eigenvalues of the 6×6 generalized eigenvalue problem [V - E N] = 0 are fitted to 100 lattice data points from six geometries (tetrahedra, squares, rectangles, quadrilaterals, non-planar, linear) using seven free parameters.

## Key numbers

- Lattice spacing: a = 0.119(1) fm at β = 2.4 on a 16³ × 32 lattice
- Best fit χ²/d.o.f. = 1.08 for the 6×6 basis model (100 data points, 7 parameters)
- For comparison, the 2×2 basis model (no multi-quark interaction) gives χ²/d.o.f. = 146
- Fitted parameters: k_f = 1.25(6), k_a = 0.54(11), f^a_S = 0.046(3), k_c = 0.04(20), a_0 = 4.4(3), b_0 = 2.2(6), c_0 = 8.0(4)
- Two-body ground-state potential fit: v(r) = 0.562 + 0.0696 r - 0.255/r - 0.045/r² (in lattice units)
- E_u excitation energy: v*(r) - v(r) = π/r - 4.24/r² + 3.983/r⁴

## Context

This work extends earlier lattice calculations by the Helsinki group (Green et al., 1993-1996) of four-quark energies in various geometries, using SU(2) instead of full SU(3) to save computational resources (the authors note less than 10% quantitative differences in glueball spectrum and topological susceptibility). The model is motivated by the success of effective two-nucleon potentials in nuclear physics, but the authors emphasize that multi-quark systems differ fundamentally due to strongly self-interacting gluon fields. The results enable future application to dynamic (non-static) quark systems via the Schrödinger equation, and work on simulating such systems is in progress.
