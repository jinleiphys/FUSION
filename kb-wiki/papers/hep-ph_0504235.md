# Spontaneous current generation in the gapless 2SC phase

> It is found that, except chromomagnetic instability, the gapless 2SC phase also exhibits a paramagnetic response to the perturbation of an external color neutral baryon current. The spontaneously generated baryon current driven by the mismatch is equivalent to the one-plane wave LOFF state. We describe the 2SC phase in the nonlinear realization framework, and show that each instability indicates the spontaneous generation of the corresponding pseudo Nambu-Golstone current. We show this Nambu-Goldstone currents generation state covers the gluon phase as well as the one-plane wave LOFF state. We further point out that, when charge neutrality condition is required, there exists a narrow unstable LOFF (Us-LOFF) window, where not only off-diagonal gluons but the diagonal 8-th gluon cannot avoid the magnetic instability. We discuss that the diagonal magnetic instability in this Us-LOFF window cannot be cured by off-diagonal gluon condensate in color superconducting phase, and it will also show up in some constrained Abelian asymmetric superfluid/superconducting system.

## Key claim

The gapless 2SC (g2SC) phase exhibits a paramagnetic response to an external color-neutral baryon current, in addition to the previously known chromomagnetic instability; this instability indicates that a baryon current can be spontaneously generated in the ground state, and this spontaneously generated current is equivalent to the one-plane wave LOFF state.

## Method

The author uses a gauged SU(2) Nambu–Jona-Lasinio (NJL) model in β-equilibrium to describe the 2SC phase, computing the vector current response function Π_V^{μν}(P) via linear response theory. The static long-wavelength limit of this response function yields the baryon number susceptibility ξ_n and baryon current susceptibility ξ_c, which are shown to be proportional to the 8th gluon's Debye and Meissner masses squared, respectively. The modified quark propagator including a spatial vector condensate Σ_V is then shown to have the same form as the propagator in the one-plane wave LOFF state, and the free energy is minimized to determine the ground state under charge neutrality.

## Key numbers

- In the g2SC phase, the baryon current susceptibility ξ_c (proportional to the 8th gluon Meissner mass squared m_{8,M}^2) becomes negative, indicating paramagnetic instability.
- Without charge neutrality, the LOFF phase exists for mismatch δμ in the range 0.706 Δ_0 < δμ < 0.754 Δ_0, corresponding to diquark gap ratio 0 < Δ/Δ_0 < 0.242.
- The stable LOFF (S-LOFF) window free of magnetic instability exists for 0 < Δ/Δ_0 < 0.39.
- The diagonal-stable LOFF (Ds-LOFF) window, free of the 8th gluon's magnetic instability but not off-diagonal gluons, exists for 0.39 < Δ/Δ_0 < 0.83.
- The unstable LOFF (Us-LOFF) window, where all magnetic instabilities persist including the diagonal 8th gluon, exists for 0.83 < Δ/Δ_0 < r_c, with r_c ≡ Δ(δμ=Δ)/Δ_0 ≃ 1.

## Context

This work extends the study of chromomagnetic instability in the gapless 2SC phase, first identified by the same community (chromo-ins-g2SC-1, chromo-ins-g2SC-2, 2004-2005), and connects it to the LOFF state proposed by Giannakis and Ren (2005) as a possible cure. The author shows that the charge-neutral LOFF state cannot fully resolve all instabilities, identifying a narrow Us-LOFF window where even the diagonal 8th gluon remains unstable, a result that challenges earlier assumptions and points to the need for spin-1 condensates or other mechanisms. The nonlinear realization framework used here parallels techniques from high-T_c superconductivity (Emery-Kivelson, 1995) and unifies the gluon condensate phase (GHM-gluon, 2005) with the LOFF state under a single Nambu-Goldstone current generation picture.
