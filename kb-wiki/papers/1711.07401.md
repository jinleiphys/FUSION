# First extraction of the scalar proton dynamical polarizabilities from real Compton scattering data

> We present the first attempt to extract the scalar dipole dynamical polarizabilities from proton real Compton scattering data below pion-production threshold. The theoretical framework combines dispersion relations technique, low-energy expansion and multipole decomposition of the scattering amplitudes. The results are obtained with statistical tools that have never been applied so far to Compton scattering data and are crucial to overcome problems inherent to the analysis of the available data set.

## Key claim

The authors present the first attempt to extract the scalar dipole dynamical polarizabilities from real Compton scattering data below pion-production threshold, using a parametric bootstrap statistical technique that has never been applied to Compton scattering data before. The extraction is qualified as a first attempt because the quality of the present experimental database is too poor to allow meaningful estimates of higher-order coefficients in the low-energy expansion, and the results show strong correlations between parameters due to low sensitivity of the data.

## Method

The theoretical framework combines dispersion relations (DRs), low-energy expansion (LEX), and multipole decomposition of the scattering amplitudes. The scalar dipole dynamical polarizabilities are parametrized as a polynomial expansion up to order ω^5, with recoil contributions fixed from known static polarizabilities and the remaining dispersive coefficients fitted to unpolarized real Compton scattering cross section data. The fitting procedure uses a parametric bootstrap technique with 10000 replicas, where each bootstrap measurement is Gaussian-distributed around the experimental data point with standard deviation given by its statistical error, and all bootstrapped points of a given subset are shifted by the same quantity proportional to the published systematic error assumed to be uniformly distributed.

## Key numbers

- Static electric dipole polarizability from FULL data set: α_E1 = (13.3 ± 0.8) × 10^(-4) fm^3
- Static magnetic dipole polarizability from FULL data set: β_M1 = (0.4 ∓ 0.9) × 10^(-4) fm^3
- Static electric dipole polarizability from TAPS data set: α_E1 = (11.6 ± 1.1) × 10^(-4) fm^3
- Static magnetic dipole polarizability from TAPS data set: β_M1 = (2.2 ∓ 1.1) × 10^(-4) fm^3
- Dispersive electric polarizability from FULL data set: α_E1,ν = (-8.8 ± 2.5) × 10^(-4) fm^5
- Dispersive magnetic polarizability from FULL data set: β_M1,ν = (10.8 ± 2.8) × 10^(-4) fm^5
- Dispersive electric polarizability from TAPS data set: α_E1,ν = (-3.2 ± 3.1) × 10^(-4) fm^5
- Dispersive magnetic polarizability from TAPS data set: β_M1,ν = (5.1 ± 3.7) × 10^(-4) fm^5
- Baldin sum rule value used as constraint: α_E1 + β_M1 = (13.8 ± 0.4) × 10^(-4) fm^3 (TAPS value)
- PDG reference values: α_E1 = (11.2 ± 0.4) × 10^(-4) fm^3, β_M1 = (2.5 ∓ 0.4) × 10^(-4) fm^3

## Context

This work extends previous theoretical calculations of dynamical polarizabilities using dispersion relations and effective field theories (Hildebrandt 2003, Lensky 2015) by attempting a direct extraction from experimental data. A feasibility study for extracting spin dynamical polarizabilities from unpolarized Compton scattering data was presented previously (Hildebrandt 2005, Griesshammer 2004) but did not turn out to be conclusive. More recently, a partial-wave analysis of unpolarized Compton scattering data was discussed (Krupina 2017), pointing out the need to improve the accuracy of the experimental data set. The present results enable future extractions with improved data quality and quantity, and the bootstrap technique provides a framework for including systematic errors and computing realistic probability distributions for the fitted parameters.
