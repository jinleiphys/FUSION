# On the infrared behavior of the shear spectral function in hot Yang-Mills theory

> We revisit the determination of the two-loop spectral function in the shear channel of hot Yang-Mills theory. Correcting a technical error in an earlier computation and extending the result with a leading order Hard Thermal Loop resummation is seen to improve the infrared behavior of the quantity significantly. This makes it possible to straightforwardly use the result in the corresponding imaginary time correlator and the shear sum rule.

## Key claim

Correcting a technical error in an earlier two-loop computation of the shear spectral function in hot Yang-Mills theory, and extending the result with a leading order Hard Thermal Loop (HTL) resummation, significantly improves the infrared behavior of the quantity, making it possible to straightforwardly use the result in the corresponding imaginary time correlator and the shear sum rule.

## Method

The authors performed an independent evaluation of the unresummed next-to-leading order (NLO) shear spectral function in SU(N) Yang-Mills theory, using a projection operator formalism and dimensional regularization to compute the one- and two-loop Feynman diagrams. They then implemented an HTL resummation following the logic of Laine (2011), approximating the resummed spectral function as the sum of the naive QCD result minus the naive HTL result plus the resummed HTL result, where the HTL calculation employed resummed gluon propagators with transverse and longitudinal self-energies.

## Key numbers

* The unresummed NLO shear spectral function is written as \(\frac{\rho_\eta(\omega)}{4d_A} = \frac{\omega^4}{4\pi}(1 + 2 n_{\omega/2})\{-\frac{1}{10} + \frac{g^2 N_c}{(4\pi)^2}(\frac{2}{9} + \phi_T^\eta(\omega/T))\}\).
* The analytic large-\(\omega\) result for the dimensionless function is \(\phi_T^\eta(\omega/T) = \frac{41\pi^6 T^6}{3\omega^6} + \mathcal{O}(T^8/\omega^8)\).
* The Debye mass squared is \(\mE^2 = \frac{N_c}{3} g^2 T^2\).
* The naive HTL result is \(\frac{\rho^{\text{HTL}}_\eta(\omega)}{4d_A}|_{\text{naive}} = -\frac{1}{4\pi}(1 + 2 n_{\omega/2})\{\frac{\omega^4}{10} + \frac{\mE^2}{45}[\pi^2 \omega T + \omega^2(12 + \frac{3}{2}\ln 2 - \frac{15}{2}\ln\frac{\omega}{T})]\}\).
* The resummed HTL contribution is written as \(\frac{\rho^{\text{HTL}}_\eta(\omega)}{4d_A}|_{\text{resummed}} = \frac{\rho^{\text{HTL}}_\eta(\omega)}{4d_A}|_{\text{naive}} + \frac{\mE^4}{4\pi}(1 + 2 n_{\omega/2})\phi_{\text{HTL}}^\eta(\omega/T, \mE/T)\).

## Context

This work extends the earlier NLO shear spectral function calculation of Zhu (2012), correcting a technical error in that computation that had led to an unphysical constant behavior at small frequencies. The corrected result now shows a linear behavior at low \(\omega\), enabling its use in the imaginary time correlator and the shear sum rule, following the program of analytic continuation of lattice results proposed by Burnier (2011, 2012). The HTL resummation follows the approach of Laine (2011) for the bulk channel, and the results are relevant for extracting the shear viscosity of the quark-gluon plasma from Euclidean lattice data and for comparisons with lattice and AdS/CFT calculations by Huebner (2008), Iqbal (2009), Springer (2010), and Kajantie (2010, 2011, 2013).
