# Systematics of the Quadrupole-Quadrupole Interaction and Convergence Properties

> Our main concern in this work is to show how higher shell admixtures affect the spectrum of a Q.Q interaction. We first review how, in the valence space, the familiar SU(3) result for the energy spectrum can be obtained using a coordinate space Q.Q interaction rather than the Elliott one which is symmetric in r and p. We then reemphasize that the Elliott spectrum goes as L(L+1) where L is the orbital angular momentum. While in many cases this is compatible with the rotational formula which involves I(I+1), where I is the total angular momentum, there are cases, e.g. odd-odd nuclei, where there is disagreement. Finally, we consider higher shell admixtures and devise a scheme so as to obtain results, with the Q.Q interaction, which converge as the model spaces are increased. We consider not only ground state rotational bands but also those that involve intruder states.

## Key claim

The paper shows that a coordinate-space quadrupole-quadrupole (Q.Q) interaction, rather than the Elliott interaction symmetric in position and momentum, can reproduce Elliott's SU(3) results in a valence space when the particle-core interaction is included, and that this coordinate-space Q.Q interaction allows controlled mixing of higher shells to achieve convergent results as the model space is increased.

## Method

The authors perform shell-model calculations using a coordinate-space Q.Q interaction, first within a single major shell to reproduce Elliott's SU(3) results, then in expanded model spaces including up to 4ħω excitations. They devise a convergence scheme by adjusting the interaction strength χ_n for each model space size n so that the energy of the first 2⁺ state in an open-shell nucleus matches experiment. Calculations are performed using the OXBASH program with center-of-mass spurious states removed via the Gloeckner-Lawson method.

## Key numbers

- For ⁸Be, the Q.Q interaction strength χ required to fit the 2₁⁺ state decreases with increasing model space: 0.5216 MeV fm⁻⁴ (0ħω), 0.4119 MeV fm⁻⁴ (0+2ħω), and 0.3369 MeV fm⁻⁴ (0+2+4ħω).
- In the 0+2ħω calculation for ⁸Be, the I=0⁺ ground state contains 21.77% 2ħω admixtures.
- For ⁸Be I=0⁺ states, percentage deviations between (0+2)ħω and (0+2+4)ħω calculations for the first four states are 0.77%, 2.22%, 2.75%, and 3.93%.
- In ¹⁹F, the K=1/2 band shows degenerate pairs (3/2⁺,5/2⁺), (7/2⁺,9/2⁺), (11/2⁺,13/2⁺) consistent with decoupling parameter a=+1.
- In ⁴³Sc, the K=1/2 band shows degenerate pairs (1/2⁻,3/2⁻), (5/2⁻,7/2⁻),...,(17/2⁻,19/2⁻) consistent with decoupling parameter a=-1.
- For ²²Na, the T=0 rotational band follows E*(I)=AI(I+1) with A=E(2⁺)-E(1₁⁺)/6, while the rotational model would give A'=E(2⁺)-E(1₁⁺)/4.
- The single-particle splitting between Nℓ and Nℓ' states from the i=j part of Qʳ·Qʳ is Δ¹_{N(ℓ,ℓ')}=2χ̄[ℓ(ℓ+1)-ℓ'(ℓ'+1)], and from the particle-core interaction is Δ²_{N(ℓ,ℓ')}=χ̄[ℓ(ℓ+1)-ℓ'(ℓ'+1)].

## Context

This work extends the SU(3) model of Elliott (1958) by showing that a coordinate-space Q.Q interaction can reproduce Elliott's results while also allowing major-shell mixing, which the momentum-symmetric Elliott interaction forbids. The paper addresses the convergence problem noted in earlier symplectic group studies by Rosensteel and Rowe (1980), demonstrating a practical renormalization scheme. The results enable systematic studies of intruder states and rotational bands in odd-odd nuclei, where the SU(3) L(L+1) spectrum differs from the rotational model I(I+1) formula, as demonstrated for ²²Na.
