# A covariant model for the $γ^\ast N \to N^\ast(1520)$ reaction

> We apply the covariant spectator quark model to the study of the electromagnetic structure of the $N^\ast(1520)$ state ($J^{P}= \frac{3}{2}^-$), an important resonance from the second resonance region in both spacelike and timelike regimes. The contributions from the valence quark effects are calculated for the $γ^\ast N \to N^\ast(1520)$ helicity amplitudes. The results are used to parametrize the meson cloud dominant at low $Q^2$.

# A covariant model for the γ*N → N*(1520) reaction

## Key claim

The covariant spectator quark model (CSQM) describes the valence quark contributions to the γ*N → N*(1520) helicity amplitudes, but only for Q² > 1.5 GeV² where meson cloud effects are expected to be small; the model is then used to extract the meson cloud contributions from data at lower Q² by assuming a parametrization regulated by the pion.

## Method

The authors construct covariant wave functions for the nucleon (S-wave quark-diquark system) and the N*(1520) state (Jᴾ = 3/2⁻) as a covariant generalization of non-relativistic angular momentum couplings, with the radial part ψ_R adjusted phenomenologically to data for Q² > 1.5 GeV². The quark electromagnetic current is described using vector meson dominance (VMD), and the transition form factors are computed in impulse approximation by integrating over the on-shell diquark degrees of freedom. The meson cloud contributions are then extracted by fitting a pion-regulated parametrization with effective momentum cutoffs to the global data.

## Key numbers

- The spin-3/2 admixture in the N*(1520) wave function is small, with sin θ_D ≈ 0.1.
- The valence quark fit was performed for Q² > 1.5 GeV², where meson cloud effects are expected to be small.
- The dipole form factor G_D is defined as G_D = (1 + Q²/0.71)⁻², where Q² is in GeV².
- Form factors are extended to predictions up to 20 GeV², corresponding to the Jlab 12-GeV upgrade regime.

## Context

This work extends the covariant spectator quark model, previously applied to the nucleon (Ramalho, 2010), the Δ(1232) (Ramalho, 2011), the N*(1440) (Roper), and the N*(1535) (S11), to the N*(1520) resonance in the second resonance region. The result that A₃/₂ receives only meson cloud contributions differs from other quark model estimates but is consistent with the large meson cloud estimate from the EBAC group (Diaz, 2008). The parametrization can be extended to the timelike regime (Q² < 0) as was previously done for the Δ(1232) (Ramalho, 2012).
