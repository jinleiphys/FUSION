# Relativistic Heavy--Ion Collisions in the Dynamical String--Parton Model

> We develop and extend the dynamical string parton model. This model, which is based on the salient features of QCD, uses classical Nambu-Gotō strings with the endpoints identified as partons, an invariant string breaking model of the hadronization process, and interactions described as quark-quark interactions. In this work, the original model is extended to include a phenomenological quantization of the mass of the strings, an analytical technique for treating the incident nucleons as a distribution of string configurations determined by the experimentally measured structure function, the inclusion of the gluonic content of the nucleon through the introduction of purely gluonic strings, and the use of a hard parton-parton interaction taken from perturbative QCD combined with a phenomenological soft interaction. The limited number of parameters in the model are adjusted to $e^+e^-$ and $p $--$p$ data. Utilizing these parameters, the first calculations of the model for $p $--$A$ and $A$--$A$ collisions are presented and found to be in reasonable agreement with a broad set of data.

## Key claim

The Dynamical String-Parton (DSP) model, extended with phenomenological mass quantization, gluonic strings, and combined hard/soft parton interactions, reproduces a broad set of hadronic data using parameters fixed only from \(e^+e^-\) and \(p\)–\(p\) collisions, making its predictions for \(p\)–\(A\) and \(A\)–\(A\) collisions fully predictive rather than fitted.

## Method

The model evolves classical Nambu-Gotō strings in 3+1 dimensions, with endpoints identified as quarks, diquarks, or gluons; hadronization proceeds via an invariant area-based string breaking probability \(\delta\mathcal{P} = \Lambda\,\delta A\). The nucleon is initialized as a distribution of string configurations derived from the experimentally measured structure function (CTEQ4M parameterization), including a 58% quark / 42% gluon momentum fraction. Quark-quark, quark-gluon, and gluon-gluon interactions combine a hard perturbative QCD amplitude with a phenomenological soft interaction; final-state hadron masses are quantized by assigning mass windows around physical meson masses.

## Key numbers

- String constant \(\kappa = 0.88\) GeV/fm, taken from the experimental Regge slope.
- Decay parameter \(\Lambda = 1\) fm\(^{-2}\) (value suggested by the LUND model), determined from \(e^+e^-\) multiplicity and rapidity distributions at \(\sqrt{s}=14\) GeV.
- Transverse momentum parameter \(\alpha = 3.88\) (GeV/c)\(^{-1}\) in the exponential distribution \(f(p_T) \propto e^{-\alpha p_T}\), giving \(\langle p_T\rangle \approx 0.35\) GeV/c.
- Quark momentum fraction in the nucleon: 58% at \(Q=1.6\) GeV; gluon fraction: 42%.
- Charged particle multiplicity in \(e^+e^-\) at \(\sqrt{s}=14\) GeV: the model with \(\Lambda=1\) fm\(^{-2}\) and rescattering reproduces the TASSO data peak.
- Hadron multiplicities in \(e^+e^-\) at \(\sqrt{s}=10\) GeV: \(\pi\) 9.33 (data 9.8±0.5), \(K\) 1.83 (data 1.81±0.09), \(\rho,\omega\) 0.54 (data 0.65±0.12); at \(\sqrt{s}=30\) GeV: \(\pi\) 16.08 (data 15.9±0.7), \(K\) 2.92 (data 2.96±0.16).
- Mass windows for quantized mesons: e.g., pion window 0.14–0.42 GeV, kaon window 0.49–0.63 GeV, proton window 0.94–1.30 GeV.

## Context

This work extends the earlier DSP model of Dean et al. (1990s) by adding mass quantization (Malov 1998), analytic structure-function initialization (Malov 1999), gluonic strings, and combined hard+soft parton interactions. It contrasts with the LUND, FRITIOF, RQMD, and QGSM models by using fully dynamical (non-longitudinal) string evolution and a simpler, flavor-independent fragmentation scheme. The model enables predictive calculations of \(p\)–\(A\) and \(A\)–\(A\) collisions without additional parameter adjustment, providing a tool to study stopping, transverse momentum production, and the approach to quark-gluon plasma conditions.
