# Multistep shell model description of spin-aligned neutron-proton pair coupling

> The recently proposed spin-aligned neutron-proton pair coupling scheme is studied within a non-orthogonal basis in term of the multistep shell model. This allows us to identify simultaneously the roles played by other configurations such as the normal pairing term. The model is applied to four-, six- and eight-hole $N=Z$ nuclei below the core $^{100}$Sn.

## Key claim

The multistep shell model (MSM) method, extended to include both neutron and proton degrees of freedom, can simultaneously describe the spin-aligned neutron-proton (np) pair coupling scheme and competing configurations like normal pairing in N=Z nuclei, but only within the restricted single \(0g_{9/2}\) shell for simplicity.

## Method

The authors extended the MSM method to treat systems with both neutrons and protons by constructing non-orthogonal overcomplete bases from products of two-particle states (like-particle pairs and np pairs), then solving the shell-model equation in successive steps for four-, six-, and eight-particle systems. The physical structure of each state was extracted by computing the cosine \(x\) of the angle between each MSM basis vector and the physical eigenvector, which gives the probability amplitude of that basis component. The method was applied to four-, six-, and eight-hole N=Z nuclei below the core \(^{100}\)Sn using only the single \(0g_{9/2}\) shell with interaction matrix elements from Qi (2011).

## Key numbers

- For \(^{96}\)Cd at \(\delta=0\), the ground state \(0^+\) has \(x^2 = 0.9216\) for the configuration \(|((\nu\pi)_9)^2;0\rangle\) (spin-aligned np pair), while the normal pairing configuration \(|(\nu\nu)_0(\pi\pi)_0;0\rangle\) has \(x^2 = 0.57608\).
- For \(^{96}\)Cd at \(\delta=0\), the first \(2^+\) state has \(x^2 = 0.9722\) for the configuration \(|((\nu\pi)_9)^2;2\rangle\).
- The angle between the aligned state \(|((\nu\pi)_9)^2;0\rangle\) and the normal pairing state \(|(\nu\nu)_0(\pi\pi)_0;0\rangle\) in \(^{96}\)Cd gives \(\cos(\phi)=0.62\), i.e., \(\phi = 52^\circ\).
- The norm of the aligned state \(|((\nu\pi)_9)^2;0\rangle\) in \(^{96}\)Cd is 2.00001, indicating negligible Pauli principle effects.
- For \(^{94}\)Ag at \(\delta=0\), the ground state \(0^+\) has \(x^2 = 0.962361\) for the configuration \(|(\nu\pi)_0(\nu\pi)_9(\nu\pi)_9);0\rangle\).
- For \(^{92}\)Pd, the ground state \(0^+_1\) has \(x^2 = 0.85\) for the configuration of four \(9^+\) aligned np pairs, and \(x^2 = 0.76\) for the configuration of two \(9^+\) aligned pairs plus normal pairing.
- In \(^{94}\)Ag, the measured \(21^+\) state at 6.67 MeV is in reasonable agreement with the calculation.

## Context

This work extends the MSM method of Liotta (1981) from identical particles to systems with both neutrons and protons, and applies it to the spin-aligned np pair coupling scheme proposed by Cederwall (2011) and Qi (2011) for N=Z nuclei near \(^{100}\)Sn. The results confirm that the low-lying yrast states in \(^{96}\)Cd, \(^{94}\)Ag, and \(^{92}\)Pd are dominated by isoscalar np pairs coupled to maximum angular momentum \(J=9\), contradicting the seniority coupling scheme that would favor like-particle pairs. This opens the way for truncating shell model bases in terms of spin-aligned np pairs and for extending calculations to many shells and larger particle numbers.
