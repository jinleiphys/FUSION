# Fixed point structure of the Abelian Higgs model

> The order of the superconducting phase transition is analyzed via the functional renormalization group approach. For the first time, we derive fully analytic expressions for the $β$ functions of the charge and the self-coupling in the Abelian Higgs model with one complex scalar field in $d=3$ dimensions that support the existence of two charged fixed points: an infrared (IR) stable fixed point describing a second-order phase transition and a tritical fixed point controlling the region of the parameter space that is attracted by the former one. It is found that the region separating first- and second-order transitions can be uniquely characterized by the Ginzburg-Landau parameter $κ$, and the system undergoes a second order transition, only if $κ>κ_c \approx 0.62/\sqrt2$.

## Key claim

For the first time, fully analytic beta functions for the charge and self-coupling in the Abelian Higgs model with one complex scalar field are derived directly in three dimensions, supporting the existence of two charged fixed points. The system undergoes a second-order phase transition only if the Ginzburg-Landau parameter satisfies κ > κ_c ≈ 0.62/√2.

## Method

The authors employed the functional renormalization group (FRG) formalism using the Wetterich equation, with a derivative expansion up to O(∂²) and a Litim-type regulator. They derived flow equations for the wave function renormalizations and coupling constants in the Abelian Higgs model in d=3 dimensions, using an R_ξ gauge fixing and approximating the scalar potential near the critical point as a quartic form. The beta functions were obtained by projecting the flow equation onto the relevant operators, discarding regulator-induced photon mass terms and fixing the gauge parameter to ξ=2 for consistency in d=3.

## Key numbers

- Critical Ginzburg-Landau parameter: κ_c ≈ 0.62/√2, separating first- and second-order phase transitions.
- Infrared stable charged fixed point (IV): λ̄_IV = (9π²/20)(31 + √461), ē²_IV = 15π²/4.
- Tricritical charged fixed point (III): λ̄_III = (9π²/20)(31 - √461), ē²_III = 15π²/4.
- Wilson-Fisher fixed point (II): λ̄_II = 9π²/10, ē²_II = 0.
- Universal value of the Ginzburg-Landau parameter at the transition point: κ_+ ≈ 1.45/√2.
- Beta function for charge in d=3: β_{e²} = -ē²_k + (4/15π²) ē⁴_k.
- Beta function for self-coupling in d=3: β_λ = -λ̄_k + (72 ē⁴_k - 72 ē²_k λ̄_k + 10 λ̄²_k)/(9π²).

## Context

This work extends earlier renormalization group analyses of the Abelian Higgs model, which using epsilon expansion from d=4 failed to produce an infrared stable fixed point in d=3. The results contradict the perturbative finding that N ≥ 183 scalar components are needed for charged fixed points (Zinn-Justin, 2002). The analytic d=3 beta functions yield a critical κ_c in fair agreement with Monte Carlo simulations (Kajantie et al., 1998; Mo et al., 2002) and with duality arguments (Kleinert, 1982). The method enables future nonperturbative studies of the scalar potential and generalizations to N-component scalar fields.
