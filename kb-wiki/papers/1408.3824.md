# A computer code for calculations in the algebraic collective model of the atomic nucleus

> A Maple code is presented for algebraic collective model (ACM) calculations. The ACM is an algebraic version of the Bohr model of the atomic nucleus, in which all required matrix elements are derived by exploiting the model's SU(1,1) x SO(5) dynamical group. This paper reviews the mathematical formulation of the ACM, and serves as a manual for the code. The code enables a wide range of model Hamiltonians to be analysed. This range includes essentially all Hamiltonians that are rational functions of the model's quadrupole moments $q_M$ and are at most quadratic in the corresponding conjugate momenta $π_N$ ($-2\le M,N\le 2$). The code makes use of expressions for matrix elements derived elsewhere and newly derived matrix elements of the operators $[π\otimes q \otimesπ]_0$ and $[π\otimesπ]_{LM}$. The code is made efficient by use of an analytical expression for the needed SO(5)-reduced matrix elements, and use of SO(5)$\,\supset\,$SO(3) Clebsch-Gordan coefficients obtained from precomputed data files provided with the code.

## Key claim

The paper presents a Maple code for algebraic collective model (ACM) calculations that avoids numerical integration entirely by using precomputed SO(5)⊃SO(3) Clebsch-Gordan coefficients and analytic expressions for SO(5)-reduced matrix elements, but the code is restricted to Hamiltonians that are rational functions of the quadrupole moments and at most quadratic in the conjugate momenta.

## Method

The ACM Hilbert space is factored into a radial part (SU(1,1) irreps with modified oscillator wave functions) and an angular part (SO(5) spherical harmonics). Matrix elements are computed analytically using SU(1,1) raising/lowering operators for the radial coordinate β and precomputed SO(5)⊃SO(3) Clebsch-Gordan coefficient data files for the angular coordinates (γ,Ω). The code implements newly derived matrix elements for the operators [π⊗q⊗π]₀ and [π⊗π]_{LM}, and uses an analytical expression for SO(5)-reduced matrix elements of SO(5) spherical harmonics.

## Key numbers

- The SO(5) multiplicity d_{νL} for a given seniority ν and angular momentum L is given by d_{νL} = (⌊⅓(ν−b)⌋+1)θ_{ν−b} − ⌊⅓(ν−L+2)⌋θ_{ν−L+2}, where b = L/2 for L even and b = (L+3)/2 for L odd
- The radial basis functions are normalized such that ∫₀^∞ R^{(λ,a)}_μ(β) R^{(λ,a)}_ν(β) dβ = δ_{μν}
- The SU(1,1) generator S₀ has matrix elements ⟨λ,ν|S₀|λ,ν⟩ = ½(λ+2ν)
- The matrix element of β² between radial basis states is a²⟨λ,μ|β²|λ,ν⟩ = δ_{μ,ν+1}√((λ+ν)(ν+1)) + δ_{μ,ν−1}√((λ+ν−1)ν) + δ_{μ,ν}(λ+2ν)
- The matrix element of 1/β² for μ≥ν is (1/a²)⟨λ,μ|1/β²|λ,ν⟩ = (−1)^{μ−ν}/(λ−1) √(μ! Γ(λ+ν) / (ν! Γ(λ+μ))) for λ>1

## Context

This work extends the algebraic collective model developed by Rowe and collaborators (Rowe, Welsh, and Wood, 2009; Caprio, Rowe, and Welsh, 2009) and provides a computational implementation that supersedes earlier numerical approaches such as the Frankfurt code (Hess, Maruhn, and Greiner, 1980; Troltenier, Maruhn, and Hess, 1991) which used U(5)⊃O(5)⊃SO(3) bases with partial numerical integration. The code enables rapid exploratory studies of Bohr model Hamiltonians and can serve as a resource for extensions including vorticity degrees of freedom, particle-core coupling models, and the interacting boson model as pursued by Thiamova and Rowe (2012).
