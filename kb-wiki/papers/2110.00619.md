# Monte Carlo simulations of γ-directional correlations and their application on FIFRELIN cascades

> Angular distribution and correlation measurements are an essential part in nuclear structure experiments, especially when spectroscopic information of a specific nucleus is unknown. In most cases, the experimental determination of the spins, parities of the studied nuclear states, as well as the possible mixing between two electric/magnetic multipoles of a transition are determined using angular correlation measurements. In this work, the full effect of directional γ-correlations is simulated, by using the formal theory of angular distributions. The density matrix formalism along with its multipole expansions called statistical tensors is employed, enabling to perform a full simulation of the angular correlation effects in a cascade of an arbitrary number of γ transitions. A triple γ angular correlation simulation is demonstrated for the first time. The present approach was coupled with the Monte Carlo code FIFRELIN, which can simulate the de-excitation of fission fragments or of excited nuclei after neutron capture. It provides a complete description of the spatial distributions of all the γ rays in the cascade, that can be used for simulation purposes in various applications both in nuclear and particle physics. The potential for a novel approach in data analysis of angular correlation measurements is discussed thoroughly.

## Key claim

The paper demonstrates that the full effect of directional γ-ray angular correlations in a cascade of an arbitrary number of transitions can be simulated using the density matrix formalism and statistical tensors, coupled with the Monte Carlo code FIFRELIN; the authors claim a triple γ angular correlation simulation is performed for the first time, but this is demonstrated only for a specific cascade in ¹¹⁰Cd with a particular geometry (N1 geometry) and with a large number of events (~800 million).

## Method

The authors implemented the formal theory of angular distributions using the density matrix formalism and statistical tensors to recursively calculate the angular distribution function for each γ-ray in a cascade, starting from an initial state with known orientation. For each step in the cascade, a two-dimensional angular distribution function is evaluated from the statistical tensor of the current state, and a direction (θᵢ, φᵢ) is generated via an acceptance-rejection algorithm. This method was coupled with the FIFRELIN Monte Carlo Hauser-Feshbach code, which generates nuclear level schemes for fission fragments or neutron-capture products, to produce complete spatial distributions of all γ rays in the cascade.

## Key numbers

- For the cascade 4⁺ → 2⁺ → 0⁺ (pure E2 transitions), the simulated angular correlation coefficients yield a₂ = 0.102 and a₄ = 0.009, with a reduced χ²/NDF = 1.30 compared to theory.
- For the cascade 5⁻ → 4⁺ → 2⁺ with mixing ratio δ = -0.42, the simulated angular correlation yields a reduced χ²/NDF = 0.86 compared to theory.
- For the triple γ cascade 5⁻ → 4⁺ → 2⁺ → 0⁺ in the N1 geometry, the simulated coefficients are Γ₂ = -0.307 and Γ₄ = -0.013, with a reduced χ²/NDF = 1.09, using approximately 800 million simulated events.
- For the cascade 6⁺ → 4⁺ → 2⁺ in the ground-state rotational band of ¹⁵⁶Gd, the theoretical coefficients are a₂ = 0.102 and a₄ = 0.009.
- For the cascade 2⁺₄ → 4⁺₁ → 2⁺₁ in ¹⁵⁶Gd, the theoretical coefficients are a₂ = 0.200 and a₄ = 0.093.
- For the cascade 3⁻ → 4⁺ → 2⁺ in ¹⁵⁶Gd, the theoretical coefficients are a₂ = -0.14 and a₄ = 0.
- For the cascade 2⁻₂ → 2⁺₃ → 0⁺_g.s. in ¹⁵⁶Gd, the theoretical coefficients are a₂ = 0.25 and a₄ = 0.

## Context

This work extends the formal theory of angular correlations developed by Biedenharn, Rose, and Brink (1953, 1967) and the statistical tensor formalism described by Fano (1957) and Hamilton (1975), by implementing it as a Monte Carlo simulation that can handle an arbitrary number of γ transitions. The method is coupled with the FIFRELIN code (Litaize 2010, 2012, 2015, 2016), which is used for simulations of the STEREO experiment (Allemandou 2018, Almazan 2019), and the authors note that GEANT4 (Allison 2016) has similar capabilities but with less detailed documentation and restriction to radioactive sources (Turner 2020). The approach enables direct comparison of simulated angular correlations with experimental data without requiring solid-angle correction factors, and opens the possibility of analyzing higher-order correlations beyond triple γ cascades.
