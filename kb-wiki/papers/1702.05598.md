# Alpha decay calculations with a new formula

> A new semi-empirical formula for calculations of $α$~decay halflives is presented. It was derived from Royer relationship by introducing new parameters which are fixed by fit to a set of experimental data. We are using three sets: set A with 130 e-e (even-even), 119 e-o (even-odd), 109 o-e, and 96 o-o, set B with 188 e-e, 147 e-o, 131 o-e, and 114 o-o, and set C with 136 e-e, 84 e-o, 76 o-e, and 48 o-o alpha emitters. A comparison of results obtained with the new formula and the following well known relationships: semFIS (semiempirical based on fission theory), ASAF (analytical superAsymmetric fission) model, and UNIV (universal formula) is made in terms of rms standard deviation. We also introduced a weighted mean value of this quantity, allowing to compare the global properties of a given model. For the set B the order of the four models is the following: semFIS, UNIV, newF, and ASAF. Nevertheless for even-even alpha emitters UNIV gives the 2nd best result after semFIS, and for odd-even parents the 2nd is newF. Despite its simplicity in comparison with semFIS the new formula, presented in this article, behaves quite well, competing with the others well known relationships.

## Key claim

The new semi-empirical formula (newF), derived from the Royer relationship by adding two parameters depending on the neutron-proton asymmetry I=(N-Z)/A, achieves competitive accuracy in calculating alpha-decay half-lives compared to the more complex semFIS, UNIV, and ASAF models, but only for certain data sets and parent-nucleus parity groups; globally, for the largest data set B, semFIS remains the best model, followed by UNIV, then newF, and finally ASAF.

## Method

The authors modified the Royer formula by adding two terms linear and quadratic in the asymmetry parameter I=(N-Z)/A, giving five free parameters (a, b, c, d, e) per parity group (even-even, even-odd, odd-even, odd-odd). They optimized these parameters by fitting to three different experimental data sets (A: 454 emitters, B: 580 emitters, C: 344 emitters) and compared the resulting root-mean-square (rms) standard deviations against those from the semFIS, ASAF, and UNIV models using the same data sets. A weighted mean rms deviation was introduced to compare global performance across parity groups.

## Key numbers

- For data set B (580 emitters), the global weighted rms standard deviation for newF is 0.6299; for semFIS it is 0.4269; for UNIV it is 0.5634; for ASAF it is 0.6313.
- For even-even emitters in set B, newF gives sigma = 0.540 (with 188 nuclei), compared to semFIS sigma = 0.223 and UNIV sigma = 0.223.
- For even-even emitters in set C (136 nuclei), newF gives sigma = 0.298, compared to semFIS sigma = 0.287 and UNIV sigma = 0.287.
- For odd-even emitters in set B, newF gives sigma = 0.522 (with 131 nuclei), compared to semFIS sigma = 0.442.
- For odd-odd emitters in set B, newF gives sigma = 0.840 (with 114 nuclei), compared to ASAF sigma = 0.876.
- The Royer formula (before modification) gave sigma = 0.560 for 130 even-even, 1.050 for 119 even-odd, 0.871 for 109 odd-even, and 0.926 for 96 odd-odd nuclei (set A).

## Context

This work extends the semi-empirical approach of Royer (2010) by adding asymmetry-dependent terms to improve accuracy across a wider range of nuclei. It directly compares the new formula with three established models: the semFIS formula based on fission theory (Rurarz, 1982; Hatakeyama, 1990), the analytical superasymmetric fission (ASAF) model (Poenaru, 1984), and the universal curve (UNIV) model (Poenaru, 2011). The results enable better selection of models for predicting alpha-decay half-lives, particularly for superheavy nuclei where Wang et al. (2015) previously found semFIS2 to be the best among 18 formulae.
