# New configuration set of HAL QCD collaboration

> We give a brief report on the basic properties and cutoff scale of our new configuration set (HAL-Conf-2023). We generated 8,000 trajectories of the gauge configurations on $96^4$ lattices with the same lattice parameters as the PACS collaboration \cite{Ishikawa:2018jee, PACS:2019ofv}. The topological distribution, the PCAC masses, and decay constants for pseudo-scalar mesons are studied. As for the scale setting, we utilize the $Ω$ baryon mass as a reference scale and carefully investigate the operator dependence of the correlation function. As a result, we obtain $a^{-1}=2338.8(1.5)^{+0.2}_{-3.3} \ [{\rm MeV}]$ as a lattice cutoff. Our hadron spectra in the physical unit reproduce well the experimental results.

## Key claim

The HAL QCD collaboration generated 8,000 trajectories of gauge configurations on a \(96^4\) lattice at the physical point, obtaining a lattice cutoff of \(a^{-1}=2338.8(1.5)^{+0.2}_{-3.3}\) MeV using the \(\Omega\) baryon mass as a reference scale, with the caveat that the scale setting depends on the operator used in the correlation function analysis.

## Method

The authors employed \(N_f=2+1\) nonperturbatively \(\mathcal{O}(a)\)-improved Wilson quark action with stout smearing and the Iwasaki gauge action at \(\beta = 1.82\) on a \(96^4\) lattice, matching the lattice parameters of the PACS collaboration (Ishikawa 2018, PACS 2019). They generated 8,000 Monte Carlo trajectories across five independent run series, each starting from different random seeds and old configurations with heavier s-quark mass. The scale was set by measuring the \(\Omega\) baryon mass using wall-source correlation functions and the variational method with Gaussian smearing, then comparing to the experimental value of 1672.45 MeV.

## Key numbers

- Lattice cutoff: \(a^{-1} = 2338.8(1.5)^{+0.2}_{-3.3}\) MeV, corresponding to lattice spacing \(a = 0.084372(54)^{+118}_{-6}\) fm
- Average plaquette value: \(\langle \mathrm{plaquette} \rangle = 0.5039576(3)\) with bin-size of 100 trajectories
- Reference scale from gradient flow: \(t_0/a^2 = 2.1047(4)\)
- Topological susceptibility: \(\chi_Q = 4.90(19) \times 10^{-6}\)
- Integrated auto-correlation time for topological charge: approximately 12 molecular dynamics trajectories
- Experimental reference mass for \(\Omega\) baryon: \(m_{\Omega} = 1672.45\) MeV (from Particle Data Group)

## Context

This work extends the lattice QCD configuration generation of the PACS collaboration (Ishikawa 2018, PACS 2019), who used the same action and \(\beta\) but on \(64^4\) and \(128^4\) lattices (the latter called "PACS10" configurations) with a cutoff of \(a^{-1} = 2316.2(4.4)\) MeV set using the \(\Xi\) baryon mass. The new HAL-Conf-2023 set on a \(96^4\) lattice provides an independent scale determination using the \(\Omega\) baryon and enables future studies of hadron interactions and spectra at the physical point, with detailed analyses to appear in a forthcoming full paper.
