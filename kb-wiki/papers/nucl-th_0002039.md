# Extracting particle freeze-out phase-space densities and entropies from sources imaged in heavy-ion reactions

> The space-averaged phase-space density and entropy per particle are both fundamental observables which can be extracted from the two-particle correlation functions measured in heavy-ion collisions. Two techniques have been proposed to extract the densities from correlation data: either by using the radius parameters from Gaussian fits to meson correlations or by using source imaging, which may be applied to any like pair correlation. We show that the imaging and Gaussian fits give the same result in the case of meson interferometry. We discuss the concept of an equivalent instantaneous source on which both techniques rely. We also discuss the phase-space occupancy and entropy per particle. Finally, we propose an improved formula for the phase-space occupancy that has a more controlled dependence on the uncertainty of the experimentally measured source functions.

# Extracting particle freeze-out phase-space densities and entropies from sources imaged in heavy-ion reactions

## Key claim

The space-averaged phase-space density extracted from two-particle correlation functions via source imaging is a generalization of the Gaussian-fit method for meson interferometry, giving the same result for pions while being applicable to any like-pair correlation; however, the extraction relies on an "equivalent instantaneous source" approximation that replaces the true time-dependent freeze-out process with a single freeze-out time, which is not uniquely defined.

## Method

The authors derive the relationship between the imaged source function (extracted by inverting the Koonin-Pratt equation via chi-squared minimization) and the space-averaged phase-space density, showing that the zero-intercept of the source function at r'→0 directly yields the phase-space density when combined with the single-particle spectrum. They demonstrate that for Gaussian pion sources, the imaging result reduces to Bertsch's (1999) formula using HBT radius parameters. They then propose an improved formula for the phase-space occupancy that replaces a squared-source-intercept expression with a linear one, reducing the propagation of experimental uncertainties.

## Key numbers

- For Pb+Pb collisions at 158 GeV/A (NA49 experiment): average phase-space occupancy for π⁻ is ⟨f⟩ = 0.19 ± 0.06 using the improved formula, compared to 0.14 ± 0.08 using the older formula (25% smaller uncertainty); entropy per pion is S/Nπ⁻ = 3.9 ± 1.8.
- For ¹⁴N+²⁷Al at 75 MeV/A (NSCL): average phase-space occupancy for protons is ⟨f⟩ = 0.27 ± 0.12 using the improved formula versus 0.25 ± 0.04 using the older formula (factor of 3 improvement in uncertainty); entropy per proton is S/Np = 2.7 ± 0.7.
- Using more precise source intercepts from Brown and Danielewicz (1998) for the NSCL data: ⟨f⟩ = 0.30 ± 0.02 from the improved formula versus 0.30 ± 0.05 from the older formula; entropy per proton is S/Np = 2.45 ± 0.21.

## Context

This work extends the method of Bertsch (1999) for extracting phase-space densities from Gaussian fits to pion correlations, showing that the source imaging technique of Brown and Danielewicz (1997, 1998) generalizes this to any like-pair correlation (protons, kaons, etc.) while giving identical results for pions. The improved formula for phase-space occupancy reduces sensitivity to experimental uncertainties compared to the earlier Brown and Danielewicz (1997) expression. The approach is complementary to the deuteron-to-proton ratio method of Siemens and Kapusta (1979) for nucleon phase-space occupancy. The results enable extraction of freeze-out densities and entropies from future relativistic heavy-ion experiments, potentially revealing signatures of phase transitions or exotic phenomena such as pion lasers.
