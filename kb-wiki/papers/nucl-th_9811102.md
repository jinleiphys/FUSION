# Missing momentum distributions and nuclear transparency in semi-inclusive A(e,e'p)X processes

> A linked cluster expansion for the distorted one-body mixed density matrix is obtained within the Glauber multiple scattering theory with correlated wave functions. The nuclear transparency for 16O is calculated using realistic central and non-central correlations. The convergence of the expansion is investigated in the case of 4He for which the transparency and the distorted momentum distributions are calculated to all order in the correlations using a variational wave function obtained from realistic NN interactions. The important role played by non central correlations is illustrated.

## Key claim

The paper claims that the effect of nucleon-nucleon correlations on nuclear transparency is strongly dependent on the inclusion of non-central (tensor) correlations: with only central correlations the effect is about 12%, but with realistic central and non-central correlations the effect reduces to about 2-3% for both 16O and 4He, contradicting earlier claims of negligible correlation effects.

## Method

The authors develop a linked cluster expansion for the distorted one-body mixed density matrix within Glauber multiple scattering theory, using correlated wave functions. For 16O they apply the first-order cluster expansion with both a phenomenological Jastrow central correlation and a realistic wave function based on the Reid V8 interaction. For 4He they perform exact calculations using the ATMS variational wave function and compare with the lowest-order cluster expansion to test convergence.

## Key numbers

- For 16O with central correlations: nuclear transparency T = 0.57 (from shell model value 0.51 plus correlation contributions totaling 0.06)
- For 16O with realistic (Reid V8) correlations: nuclear transparency T = 0.52 (from shell model value 0.51 plus correlation contributions totaling 0.01)
- For 4He with Jastrow central correlations: exact transparency T_EX = 0.80 (lowest order cluster expansion gives T_LOC = 0.80)
- For 4He with realistic ATMS wave function (Reid V8): exact transparency T_EX = 0.78 (lowest order cluster expansion gives T_LOC = 0.77)
- The Glauber profile parameters used: sigma_tot = 43 mb, alpha = -0.33, b_o = 0.5 fm

## Context

This work extends previous calculations of nuclear transparency and distorted momentum distributions that used phenomenological central correlations (e.g., Nikolayev and collaborators) by incorporating realistic non-central correlations from the Reid V8 interaction. The authors show that earlier claims of negligible correlation effects on transparency (from Nikolayev) arose from using only central correlations and a non-converging cluster expansion. The paper enables more reliable predictions for semi-inclusive A(e,e'p)X experiments by demonstrating that the number-conserving linked cluster expansion converges rapidly and that realistic ground state correlations significantly reduce the apparent effect of final state interactions.
