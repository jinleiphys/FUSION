# NLOAccess: automated online computations for collider physics

> We present NLOAccess, an online platform acting as a virtual access to automated perturbative computations of physical observables related to collider physics within collinear factorisation. We discuss the design of the project and illustrate how, with few steps and without any need of pre-code or code compilation, a user can generate events and calculate cross sections related for instance to quarkonium production and/or heavy-ion collisions at leading and next-to-leading order in $α_s$ using MadGraph5_aMC@NLO or HELAC-Onia. Finally, we report about the obtained results in terms of number of users and runs, and we illustrate the future plans of the project.

## Key claim

NLOAccess provides an online platform for automated perturbative computations in collider physics that, for the first time, allows users to run MadGraph5_aMC@NLO in its full next-to-leading-order version via a web interface, without requiring any code compilation or pre-code preparation.

## Method

The platform uses a Flask Python micro-framework to serve both front-end and back-end, with a three-part architecture consisting of a server side (Flask server and internal NLOAccess servers), a cluster side (managed through HTCondor batch system), and a cloud side (OwnCloud server for secure user storage). Users submit run requests through dedicated web forms or by uploading input files; the system then submits jobs to the batch system, retrieves results, stores them in the user's personal cloud folder, and notifies the user via email upon completion. Two tools are currently available: HELAC-Onia for heavy quarkonium production at Born order within Non-Relativistic QCD, and MadGraph5_aMC@NLO for leading-order and next-to-leading-order computations in the Standard Model and beyond.

## Key numbers

- Approximately 400 users from all over the world have used the platform
- Over 4000 runs have been performed on NLOAccess
- HELAC-Onia allows users to set decay branching ratios, for example a J/psi decaying to di-muons with a branching ratio of 6% (specified as "decay cc~(3S11) > m+ m- @ 0.06")
- The platform is part of the EU Horizon 2020 integrated initiative STRONG-2020 under Grant agreement no. 824093

## Context

This work extends the capabilities of existing online tools such as APFEL Web (Carrazza, 2014), which only allowed visualization of parton distribution functions and computation of parton luminosities, by providing full access to automated event generation and cross-section computation. Previous online versions of MadGraph existed but were limited to leading-order calculations only. The platform enables students, non-experts, and experienced researchers to perform perturbative computations for quarkonium production and heavy-ion collisions without technical barriers, and the design allows any code compilable on a Unix shell to be included in the future.
