# THERMINATOR simulations and PHENIX images of a heavy tail of particle emission in 200 GeV Au+Au collisions

> Correlation between emitted particles from a thermalized medium carry important information about the space-time extent and the dynamics of the particle emitting source. Pion emission source functions S(r) have been measured in PHENIX using an imaging technique, and it has been found that it contains a heavy (power-law) tail. It leads us to the assumption that there is a halo of secondary pions, surrounding the core system that undergoes the hydrodynamical evolution. THERMINATOR, a Monte Carlo event generator designed for studies in relativistic heavy-ion collisions is used to model and investigate the influence of resonance decays on the tail of particle emission source. It lacks the implementation of rescattering, which, as we have learnt from previous Hadron Rescattering Code simulations, is a possible explanation of the power-law tail. Our studies also have shown that none of the pion sources alone are responsible for the tail: either the resonance decays or rescattering or a combination of these effects is a possible cause of the heavy tail.

## Key claim
The heavy (power-law) tail observed in PHENIX pion emission source functions can be reproduced by either resonance decays (via THERMINATOR) or rescattering (via Hadron Rescattering Code simulations), but neither mechanism alone is sufficient to fully explain the data; a combination of these effects is a possible cause.

## Method
Pion pair source functions S(r) were extracted from PHENIX data using a model-independent imaging technique applied to two-particle correlation functions of charged pions from Au+Au collisions at sqrt(s_NN)=200 GeV. The data were compared to two Monte Carlo simulations: THERMINATOR, which implements a single freezeout model with 385 resonance decays but no rescattering, and the Hadron Rescattering Code (HRC), which simulates the 8 most abundant resonances with rescattering. Source functions were decomposed into core (primordial and short-lived resonance decay products), omega (omega(782) meson decays), and halo (long-lived resonance decays) components.

## Key numbers
- PHENIX data taken at sqrt(s_NN)=200 GeV Au+Au collisions, mid-rapidity (|y|<0.5 GeV)
- Low kT range: 200 MeV < kT < 360 MeV; high kT range: 480 MeV < kT < 600 MeV
- Core group lifetime: less than 20 fm/c; omega group lifetime: approximately 23 fm/c; halo group lifetime: greater than 25 fm/c
- Short-lived resonances defined as those with Gamma >= 150 MeV
- Centrality bins: 0-20% for central, 40-90% for peripheral collisions

## Context
This work extends PHENIX source imaging results (PHENIX Collaboration, 2005) that first observed the power-law tail in pion emission source functions. It compares two theoretical approaches: the Hadron Rescattering Code of Humanic (2002, 2006), which attributes the tail to anomalous diffusion from rescattering, and the THERMINATOR model based on the Cracow Single Freezeout Model (2002). The study finds that neither resonance decays alone (as proposed by Bialas, 1992) nor rescattering alone fully explain the data, and suggests that measuring source functions for kaons and protons could distinguish between these mechanisms. Direction-dependent studies by Chung et al. (2007) showed that THERMINATOR fails in the "outward" direction of the Bertsch-Pratt parametrization.
