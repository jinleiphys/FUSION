# An Efficient Quantum Circuit for Block Encoding a Pairing Hamiltonian

> We present an efficient quantum circuit for block encoding pairing Hamiltonian often studied in nuclear physics. Our block encoding scheme does not require mapping the creation and annihilation operators to the Pauli operators and representing the Hamiltonian as a linear combination of unitaries. Instead, we show how to encode the Hamiltonian directly using controlled swap operations. We analyze the gate complexity of the block encoding circuit and show that it scales polynomially with respect to the number of qubits required to represent a quantum state associated with the pairing Hamiltonian. We also show how the block encoding circuit can be combined with the quantum singular value transformation to construct an efficient quantum circuit for approximating the density of states of a pairing Hamiltonian. The techniques presented can be extended to encode more general second-quantized Hamiltonians.

## Key claim

The authors present an efficient quantum circuit for block encoding a pairing Hamiltonian that avoids mapping creation/annihilation operators to Pauli operators or representing the Hamiltonian as a linear combination of unitaries; instead, the scheme encodes the Hamiltonian directly using controlled swap operations, with the caveat that the technique is demonstrated for pairing Hamiltonians where the phase factor from fermionic anti-commutation is always one.

## Method

The authors construct a block encoding circuit for the pairing Hamiltonian by implementing a select oracle that decomposes into controlled operations for each term \( \mathcal{H}_\ell = a^\dagger_p a_q \). For each term, a controlled swap operation is applied to the Fock state qubits when the condition \( j_q = 1 \) and \( j_p = 0 \) (for \( p \neq q \)) is satisfied, using a validation qubit to track valid operations. The \( O_\mathcal{H} \) circuit encodes the nonzero matrix elements via controlled \( R_y \) rotations on a rotation qubit, with rotation angles \( \theta_l = \arccos(v_l) \), and the full block encoding follows the sparse matrix block encoding theorem from Camps (2022) and Gilyen (2019).

## Key numbers

* The gate complexity of the block encoding circuit scales polynomially with respect to the number of qubits required to represent a quantum state associated with the pairing Hamiltonian (no explicit polynomial degree or constant given in the text).
* The number of selection qubits is \( m = \log L \), where \( L \) is the number of \( (p,q) \) pairs in the Hamiltonian (no explicit \( L \) value provided).
* The rotation angle for each coefficient is \( \theta_l = \arccos(v_l) \), where \( v_l \) is the coefficient \( h_{pq} \) from the Hamiltonian (no explicit numerical values for \( v_l \) given).

## Context

This work extends the block encoding techniques for sparse matrices developed by Camps (2022) and Gilyen (2019) to the specific case of pairing Hamiltonians common in nuclear physics. The authors note that their approach avoids the overhead of converting fermionic operators to Pauli strings via Jordan-Wigner or Bravyi-Kitaev transformations. The block encoding circuit can be combined with quantum singular value transformation (QSVT) to construct efficient quantum circuits for approximating the density of states of a pairing Hamiltonian, and the techniques are stated to be extendable to more general second-quantized Hamiltonians.
