# Two-proton radioactivity of exotic nuclei beyond proton drip-line

> To search for new candidates of the true and simultaneous two-proton ($2p$) radioactivity, the $2p$ decay energies (\textit{Q}$_{2p}$) are extracted by the Weizsäcker-Skyrme-4 (WS4) model, the finite-range droplet model (FRDM), the Kourra-Tachibaba-Uno-Yamada (KTUY) model and the Hartree-Fock-Bogoliubov mean-field model with the BSk29 Skyrme interaction (HFB29). Then, the $2p$ radioactivity half-lives are calculated within the generalized liquid drop model (GLDM) by inputting the four types of \textit{Q}$_{2p}$ values. By the energy and half-life constraints, it is found that the probable $2p$ decay candidates are the nuclei beyond the proton-drip line in the region of \textit{Z} $<$50 or \textit{Z}$\leq 50$ based on each nuclear mass model. In the region beyond \textit{Z}=50, the $2p$-decaying candidates are predicted only using the HFB29 mass model. Finally, the competition between the true $2p$ radioactivity and $α$-decay for the nuclei above the \textit{N}=\textit{Z}=50 shell closures is discussed. It is shown that $^{101}$Te, $^{111}$Ba and $^{114}$Ce prefer to $2p$ radioactivity and the dominant decay mode of $^{107}$Xe and $^{116}$Ce is $α$-decay.

# Two-proton radioactivity of exotic nuclei beyond proton drip-line

## Key claim
The authors predict that true and simultaneous two-proton (2p) radioactivity candidates exist primarily in the region Z < 50 or Z ≤ 50 across multiple mass models, but beyond Z = 50 such candidates are found only when using the HFB29 mass model. The claim is conditional on an extended half-life criterion (-12.00 < log10 T1/2 < 2.00 s) rather than the narrower range used in prior work by Olsen (2013).

## Method
The authors extracted Q2p values from four nuclear mass models (WS4, FRDM, KTUY, and HFB29) and calculated 2p radioactivity half-lives within the generalized liquid drop model (GLDM), treating the emission as a preformed 2He cluster that penetrates the Coulomb barrier. They applied energy criteria (Q2p > 0 and Qp < 0.2 Q2p) and an extended half-life criterion to identify true and simultaneous 2p emitters, and compared 2p decay with alpha-decay for nuclei above the N = Z = 50 shell closures using the same GLDM framework.

## Key numbers
- For 58Ge, predicted (Q2p, log10 T1/2) values vary across models: WS4 gives (2.68 MeV, -9.80 s), FRDM gives (2.57 MeV, -9.34 s), KTUY gives (2.45 MeV, -8.80 s), and HFB29 gives (1.77 MeV, -4.76 s).
- For 101Te using HFB29: Q2p = 5.45 MeV, Qα = -0.19 MeV, log10 T1/2(2p) = -9.54 s, indicating 2p radioactivity is the dominant decay mode.
- For 107Xe using HFB29: Q2p = 3.08 MeV, Qα = 4.80 MeV, log10 T1/2(2p) = -0.24 s, log10 T1/2(α) = -5.70 s, indicating alpha-decay is dominant.
- For 111Ba using HFB29: Q2p = 3.46 MeV, Qα = 4.10 MeV, log10 T1/2(2p) = -2.28 s, log10 T1/2(α) = -1.40 s, indicating 2p radioactivity is dominant.
- For 114Ce using HFB29: Q2p = 4.94 MeV, Qα = 4.25 MeV, log10 T1/2(2p) = -7.12 s, log10 T1/2(α) = -1.11 s, indicating 2p radioactivity is dominant.
- For 116Ce using HFB29: Q2p = 3.22 MeV, Qα = 4.31 MeV, log10 T1/2(2p) = -0.01 s, log10 T1/2(α) = -1.45 s, indicating alpha-decay is dominant.

## Context
This work extends the landscape of 2p radioactivity candidates beyond the earlier predictions by Olsen (2013), who used energy density functional theory with Skyrme interactions and identified candidates primarily in two mass regions (germanium to krypton, and just above tin). The present study uses four different mass models (WS4, FRDM, KTUY, HFB29) and an extended half-life criterion to include more candidates, finding that only HFB29 predicts 2p emitters beyond Z = 50. The competition between 2p radioactivity and alpha-decay for nuclei above N = Z = 50 shell closures (101Te, 107Xe, 111Ba, 114Ce, 116Ce) provides specific targets for future experimental searches at radioactive beam facilities.
