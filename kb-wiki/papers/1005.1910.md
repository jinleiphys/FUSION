# Impulse approximation in the n p --> d pi^0 reaction reexamined

> The impulse approximation (one-body operator) in the n p --> d pi^0 reaction is reexamined with emphasis on the issues of reducibility and recoil corrections. An inconsistency when one pion exchange is included in the production operator is demonstrated and then resolved via the introduction of "wave function corrections" which nearly vanish for static nucleon propagators. Inclusion of the recoil corrections to the nucleon propagators is found to change the magnitude and sign of the impulse production amplitude, worsening agreement with the experimental cross section by approximately 30%. A cutoff is used to account for the phenomenological nature of the external wave functions, and is found to have a significant impact up to approximately 2.5 GeV.

## Key claim

The impulse approximation (one-body operator) in the \(np\rightarrow d\pi^0\) reaction contains a formal inconsistency when one-pion exchange is included in the production operator, which can be resolved by introducing "wave function corrections" that nearly vanish only when static nucleon propagators are used; however, including recoil corrections to the nucleon propagators changes the magnitude and sign of the impulse production amplitude, worsening agreement with experimental cross sections by approximately 30%.

## Method

The authors used a hybrid methodology where two-particle-irreducible diagrams (operators) were calculated perturbatively using baryon chiral perturbation theory with modified counting scheme (MCS), then convolved with nucleon-nucleon wave functions obtained from three phenomenological potentials: Argonne v18, Nijmegen II, and Reid '93. They evaluated the impulse approximation diagram and the one-pion exchange reducible and irreducible diagrams in position space, treating the nucleon propagators both with the free recoil approximation and with exact treatment of momenta as operators, and applied a Gaussian cutoff to account for the phenomenological nature of the external wave functions.

## Key numbers

- Initial relative momentum at threshold: \(|\vec{p}| = 359\) MeV
- Reduced matrix element for impulse approximation with Av18 potential: \(A_0^{\text{imp}} = 4.9\)
- Reduced matrix element for impulse approximation with Nijm II potential: \(A_0^{\text{imp}} = 1.3\)
- Reduced matrix element for impulse approximation with Reid '93 potential: \(A_0^{\text{imp}} = 3.5\)
- Reduced matrix element for OPE reducible final state diagram with Av18: \(A_0^{\text{OPE,red,f}} = 75.2\)
- Reduced matrix element for OPE irreducible final state diagram with Av18: \(A_0^{\text{OPE,irr,f}} = 75.6\)
- Final state wave function correction (exact propagators) with Av18: \(-11.7\)
- Initial state wave function correction (exact propagators) with Av18: \(-11.1 + 16.3i\)
- Cutoff dependence found significant for \(\Lambda \lesssim 2.5\) GeV
- Experimental threshold parameter \(\alpha\) for \(np\rightarrow d\pi^0\): \(184 \pm 5\) \(\mu\)b (Hutcheon 1989)
- Experimental threshold parameter \(\alpha\) for \(pp\rightarrow d\pi^+\) (Coulomb corrected): \(208 \pm 5\) \(\mu\)b (Heimberg 1996) and \(205 \pm 9\) \(\mu\)b (Drochner 1998)
- Pionic deuterium decay gives \(\alpha = 252^{+5}_{-11}\) \(\mu\)b (Strauch 2010)

## Context

This work extends the analysis of reducibility issues in pion production first identified by Lensky (2005) for the rescattering diagram, applying similar considerations to the impulse approximation. The authors previously introduced "wave function corrections" in Bolton (2009) for charge symmetry breaking in \(np\rightarrow d\pi^0\), but that calculation used an approximation (free recoil approximation) which this paper corrects. The findings enable future investigation into a clear procedure for deciding what to include in the impulse diagram, which is important not only for the total cross section but also for \(p\)-wave pion production where the impulse diagram provides the leading contribution.
