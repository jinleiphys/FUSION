# Higher-order corrections to exclusive heavy vector meson production

> We present results for higher-order corrections to exclusive $\mathrm{J}/ψ$ production. This includes the first relativistic correction of order $v^2$ in quark velocity, and next-to-leading order corrections in $α_s$ for longitudinally polarized production. The relativistic corrections are found to be important for a good description of the HERA data, especially at small values of the photon virtuality. The next-to-leading order results for longitudinal production are evaluated numerically. We also demonstrate how the vector meson production provides complementary information to the structure functions for extracting the initial condition for the small-$x$ evolution of the dipole-proton scattering amplitude.

## Key claim

The first relativistic corrections of order \(v^2\) in quark velocity are important for describing HERA data on exclusive J/\(\psi\) production, especially at low photon virtuality, while next-to-leading order \(\alpha_s\) corrections for longitudinally polarized production affect the cross section only modestly, with more variation between different NLO dipole amplitude fits than between LO and NLO results.

## Method

The authors compute exclusive J/\(\psi\) production in the dipole picture, using the IPSat parametrization for the dipole amplitude fitted to HERA structure function data. For the meson wave function, they employ an NRQCD-based approach that includes relativistic corrections of order \(v^2\) via two nonperturbative constants determined from charmonia decay constants, with the rest frame wave function related to the light front wave function by a Melosh rotation. The NLO calculation for longitudinally polarized production combines NLO corrections to the vector meson wave function and the longitudinal photon wave function with massive quarks, demonstrating cancellation of UV and IR divergences and the appearance of the Balitsky-Kovchegov equation for the energy dependence.

## Key numbers

- Total exclusive J/\(\psi\) production cross section as a function of photon virtuality \(Q^2\): relativistic corrections have greatest impact at small \(Q^2\) and become negligible at large \(Q^2\); the fully nonrelativistic wave function overestimates the total cross section at small \(Q^2\) compared to HERA data from Aktas (2005) and Chekanov (2004).
- Diffractive cross section for longitudinal J/\(\psi\) production as a function of center-of-mass energy: NLO results with different dipole amplitude parametrizations (KCBK, TBK, ResumBK) show more variation between them than between LO and NLO results.
- Nuclear suppression ratio for longitudinal J/\(\psi\) production: both NLO and relativistic corrections of order \(v^2\) reduce the nuclear suppression slightly, with effects smaller than those found in Lappi (2020) for the total cross section.

## Context

This work extends the dipole picture description of exclusive vector meson production by incorporating the first relativistic corrections to the meson wave function via NRQCD, following the approach of Lappi (2020), and by presenting the first NLO calculation for longitudinally polarized J/\(\psi\) production, building on NLO wave function developments by Escobedo (2019) and Beuf (2021). The results show that vector meson production provides complementary information to structure function analyses for extracting the initial condition for small-\(x\) evolution of the dipole-proton scattering amplitude, as all NLO dipole amplitudes fitted to HERA structure function data (Beuf, 2020) yield different predictions for vector meson production. Future work will include NLO calculations for transversely polarized photons to enable comparisons with HERA data from Alexa (2013) and Chekanov (2002, 2004), LHC data from Aaij (2014) and Acharya (2018), and predictions for the future Electron-Ion Collider.
