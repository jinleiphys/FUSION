# An Efficient Learning Method to Connect Observables

> Constructing fast and accurate surrogate models is a key ingredient for making robust predictions in many topics. We introduce a new model, the Multiparameter Eigenvalue Problem (MEP) emulator. The new method connects emulators and can make predictions directly from observables to observables. We present that the MEP emulator can be trained with data from Eigenvector Continuation (EC) and Parametric Matrix Model (PMM) emulators. A simple simulation on a one-dimensional lattice confirms the performance of the MEP emulator. Using $^{28}$O as an example, we also demonstrate that the predictive probability distribution of the target observables can be easily obtained through the new emulator.

# Multiparameter Eigenvalue Problem (MEP) Emulator

## Key claim
The MEP emulator can construct a direct mapping from observables to observables, bypassing the need to infer unmeasurable low-energy constants (LECs) in nuclear Hamiltonians, but only for propagating constraints from input observables to target observables, systematic uncertainties from model discrepancy are not addressed.

## Method
The authors formulate the nuclear Hamiltonian eigenvalue problem as a multiparameter eigenvalue problem (MEP) where input observables (e.g., ground-state energies of light nuclei) replace Hamiltonian parameters. They then apply a reduced-basis projection using training eigenvectors from Eigenvector Continuation (EC) or Parametric Matrix Model (PMM) emulators, reducing the Kronecker-product eigenvalue problem to a tractable generalized eigenvalue problem of dimension equal to the number of training samples. The resulting emulator \(f_{\rm MEP}\) directly maps input observables to target observables without explicit parameter inference.

## Key numbers
- Training samples for the one-dimensional lattice toy model used \(V_0 = 0.5, -1, -2, -3, -4, -5\) (six training points) for both two- and three-body systems.
- The MEP emulator for the \(^{28}\)O study used \(12 \times 12\) dimensional matrices, enabling computation of \(10^5\) samples in a few minutes.
- For the \(^{28}\)O realistic calculation, VS-IMSRG calculations used 13 major-shell harmonic-oscillator space with \(\hbar\omega = 16\) MeV and \(E_{\rm 3max}=18\); basis truncation error is estimated at 0.2 MeV for \(^{24}\)O.
- The calibrated input distribution used \(E_{^{16}{\rm O}} = -127 \pm 2\) MeV and \(E_{^{24}{\rm O}} = -168 \pm 3\) MeV as conditioning constraints.

## Context
This work extends the Eigenvector Continuation (EC) method of Frame (2017) and the Parametric Matrix Model (PMM) of Cook (2024) by unifying them into a single emulator that connects observables directly. It addresses the computational bottleneck in Bayesian uncertainty quantification workflows such as those used by Hu (2021), Elhatisari (2022), and Jiang (2022), where parameter inference requires repeated expensive Hamiltonian diagonalization. The method enables efficient exploration of observable correlations and uncertainty propagation for exotic nuclei like \(^{28}\)O, recently measured by Kondo (2023), without explicit LEC determination.
