# Pole structure of $P_ψ^N(4312)^+$ via machine learning and uniformized S-matrix

> We probed the pole structure of the $P_ψ^{N}(4312)^{+}$ using a trained deep neural network. The training dataset was generated using uniformized independent S-matrix poles to ensure that the obtained interpretation is as model-independent as possible. To prevent possible ambiguity in the interpretation of the pole structure, we included the contribution from the off-diagonal element of the S-matrix. Five out of the six neural networks we trained favor $P_ψ^{N}(4312)^{+}$ as possibly having a three-pole structure, with one pole on each of the unphysical sheets - a first in its report. The two poles can be associated to a pole-shadow pair which is a characteristic of a true resonance. On the other hand, the last pole is most likely associated with the coupled-channel effect. The combined effect of these poles produced a peak below the $Σ^{+}_C\bar{D}^0$ which mimic the line shape of a hadronic molecule.

## Key claim

The paper claims that the $P_\psi^N(4312)^+$ pentaquark likely has a three-pole structure, with one pole on each of the three unphysical Riemann sheets of the $\Lambda_b^0 \to J/\psi p$ decay, a configuration reported for the first time. This conclusion is hedged by the fact that five out of six trained neural networks favored this interpretation, rather than all six.

## Method

The authors constructed a two-channel S-matrix using a uniformized variable $\omega$ to remove branch-cut singularities, allowing independent parametrization of poles on different Riemann sheets. They generated a training dataset of line shapes from eight distinct pole configurations and trained six deep neural network classifiers to identify which pole configuration best reproduces the experimental $P_\psi^N(4312)^+$ signal. The full S-matrix included both diagonal ($S_{11}$) and off-diagonal ($S_{12}$) elements to resolve ambiguities in the line shape interpretation.

## Key numbers

* The $P_\psi^N(4312)^+$ lies roughly 5 MeV below the $\Sigma_c^+\bar{D}^0$ threshold.
* The $P_c(4312)^+$ pentaquark was discovered with a statistical significance of $7.3\sigma$ by LHCb Run 2.
* Five out of six neural network models favored a three-pole configuration for the $P_\psi^N(4312)^+$ signal.

## Context

This work extends prior machine-learning studies of the $P_\psi^N(4312)^+$ by Sombillo et al. (2020) and Co et al. (2024), which classified the state as a virtual state or molecular structure using neural networks. It also builds on the bottom-up analysis of the same pentaquark by the same group (2020), which favored a virtual-state interpretation. The present study introduces a uniformized S-matrix approach with independent poles to reduce model bias, and it incorporates the off-diagonal S-matrix element to avoid ambiguous line shapes, as demonstrated in the authors' earlier work (Santos and Sombillo, 2023). The result suggests that the observed peak below the $\Sigma_c^+\bar{D}^0$ threshold may arise from a coupled-channel effect rather than a single hadronic molecule or compact pentaquark.
