# R-Matrix parametrization for γ decays to unbound states

> R-matrix theory was originally developed to describe nuclear reactions. The framework was further extended to describe β decay to unbound states. However, at the time writing, no clear description of γ decays to unbound states exist. Such a description will be presented in this note.

# R-Matrix parametrization for γ decays to unbound states

## Key claim

The paper presents a formal R-matrix description for γ decays to unbound states, which did not exist at the time of writing, under the condition that the initial reaction proceeds through a single isolated resonance in the compound system and that photon channels can be treated via perturbation theory due to their small coupling to the nucleus.

## Method

The author extends the R-matrix formalism of Lane and Thomas (1958) for sequential nuclear reactions, treating the γ decay as a transition from an initial isolated resonance λ to an unbound intermediary state λ′ which subsequently decays via particle emission. The formalism uses the Brune (2002) observable-parameter approach to work directly with observed resonance energies and reduced widths, and defines a γ-ray penetrability as P_L ≡ E^(2L+1) to maintain symmetric notation between photon and particle channels. The differential partial width of the intermediary state is expressed as an integral over the resonance peak, which for an isolated intermediary level with approximately linear shift function can be evaluated analytically.

## Key numbers

- The partial width for emission of an L-wave photon from a level λ is given by Γ_{λL} = 2 E^(2L+1) γ²_{λL}, where E is the photon energy.
- The partial width for a particle channel is given as Γ_{λc} = 2 P_c γ²_{λc}, where P_c is the penetrability.
- The observed partial decay width of λ to a specific resonance λ′ is approximated as Γ⁰_{λc′(λ′)} ≈ (2 P_c′ γ̃²_{λc′(λ′)}) / (1 + Σ_c γ̃²_{λ′c} dS_c/dE|_{Ẽ_{λ′}}), identical to the expression in Brune (2002).

## Context

This work extends the R-matrix framework originally developed by Lane and Thomas (1958) for nuclear reactions and later adapted by Barker (1969) and Warburton (1986) for β decays to unbound states. The formalism enables the description of reactions such as p + ⁷Li → ⁸Be* → γ + ⁸Be** → γ + α + α, where a γ decay populates an unbound state that subsequently breaks up. The approach allows future calculations of cross sections for γ decays through unbound intermediary resonances using observable parameters, and can incorporate contributions from different multipolarities added either coherently or incoherently.
