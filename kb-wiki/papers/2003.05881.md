# On the Interpretation of the Balance Function

> We construct a simple toy model and explicitly demonstrate that the Balance Function (BF) can become negative for some values of the rapidity separation and hence can not have any probabilistic interpretation. In particular, the BF can not be interpreted as the probability density for the balancing charges to occur separated by the given rapidity interval.

## Key claim

The Balance Function (BF) can become negative for some values of rapidity separation, contradicting the claim that it represents a probability density for balancing charges to be separated by a given rapidity interval. The paper explicitly demonstrates this using a simple toy model, showing that negative BF values arise from anticorrelations between identical charges rather than from correlations between opposite charges.

## Method

The author constructs a toy model with independent identical sources, each producing a fixed number of correlated plus-minus pairs uniformly distributed in a wide rapidity interval. The model is analyzed under conditions of translation invariance in rapidity and charge symmetry, using the relation between the BF and the correlation functions for same-sign and opposite-sign pairs. The author then introduces additional correlations between identical charges (first with a delta-function, then with a step-function smearing) and computes the resulting BF and the related variable nu_s.

## Key numbers

- For the model with one plus-minus pair per source (k=1), nu_s(deta) = deta/Y, where Y is the total rapidity interval width (Y >> 1).
- For the model with two plus-minus pairs per source and maximally strong correlation between identical charges (delta-function), nu_s(deta) = 2*deta/Y - 1, which becomes negative for deta < Y/2.
- For the model with gentle correlation between identical charges (step-function of width a), the BF becomes negative when deta < a < (1 - 1/sqrt(2))*Y, approximately a < 0.29*Y.
- In the model with two independent plus-minus pairs per source (k=2, no same-charge correlation), the BF equals 1/Y, a constant positive value.

## Context

This work extends the discussion of net-charge fluctuations and the Balance Function introduced by Bass (2000), who proposed that the BF represents a probability density for balancing charge separation. The paper contradicts that probabilistic interpretation by showing that the BF can take negative values, as already hinted by the fact that the related variable nu_dyn can be both positive and negative (ALICE Collaboration, 2016). The results enable a re-evaluation of the physical meaning of the Balance Function in heavy-ion collisions, suggesting it should be understood as a correlation measure rather than a probability density.
