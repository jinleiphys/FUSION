# Shell model predictions for $^{124}$Sn double-beta decay

> Neutrinoless double-beta ($0 νββ$) decay is a promising beyond Standard Model process. Two-neutrino double-beta ($2 νββ$) decay is an associated process that is allowed by the Standard Model, and it was observed in about 10 isotopes, including decays to the excited states of the daughter. $^{124}$Sn was the first isotope whose double-beta decay modes were investigated experimentally, and despite few other recent efforts, no signal has been seen so far. Shell model calculations were able to make reliable predictions for $2 νββ$ decay half-lives. Here we use shell model calculations to predict the $2 νββ$ decay half-life of $^{124}$Sn. Our results are quite different from the existing quasiparticle random-phase approximation (QRPA) results, and we envision that they will be useful for guiding future experiments. We also present shell model nuclear matrix elements for two potentially competing mechanisms to the $0 νββ$ decay of $^{124}$Sn.

# Shell model predictions for $^{124}$Sn double-beta decay

## Key claim

Shell model calculations predict a $2\nu\beta\beta$ decay half-life for $^{124}$Sn of $1.6 \times 10^{21}$ yr for the ground-state transition, which is significantly different from the existing QRPA prediction of $7.8 \times 10^{19}$ yr, though the authors note a theoretical uncertainty of at least $\pm 20\%$ due to the choice of the Gamow-Teller quenching factor.

## Method

The authors performed shell model calculations using the SVD effective Hamiltonian in the $jj55$ model space (consisting of the $0g_{7/2}, 1d_{5/2}, 1d_{3/2}, 2s_{1/2}$, and $0h_{11/2}$ valence orbitals), which was fine-tuned using experimental data for Sn isotopes. They validated the Hamiltonian by comparing calculated energy spectra, $B(E2)\uparrow$ transition probabilities, occupation probabilities, and Gamow-Teller strengths against available experimental data for $^{124}$Sn and $^{124}$Te. The $2\nu\beta\beta$ nuclear matrix elements were computed using a strength function approach, and $0\nu\beta\beta$ nuclear matrix elements were calculated within the closure approximation using an optimal closure energy of 3.5 MeV.

## Key numbers

- $2\nu\beta\beta$ half-life for $^{124}$Sn decay to ground state of $^{124}$Te: $T_{1/2}^{2\nu} = 1.6 \times 10^{21}$ yr (using $q_f = 0.74$)
- $2\nu\beta\beta$ half-life for decay to first excited $0^+_1$ state: $6.2 \times 10^{5}$ yr (with $Q_{\beta\beta} = 0.630$ MeV)
- $2\nu\beta\beta$ half-life for decay to first excited $2^+_1$ state: $2.2 \times 10^{5}$ yr (with $Q_{\beta\beta} = 1.686$ MeV)
- $0\nu\beta\beta$ light neutrino-exchange NME for ground-state transition: $M^{0\nu}_{\nu} = 2.17$ (CD-Bonn SRC) and $2.00$ (Argonne-V18 SRC)
- $0\nu\beta\beta$ heavy neutrino-exchange NME for ground-state transition: $M^{0\nu}_{N} = 144$ (CD-Bonn SRC) and $97.4$ (Argonne-V18 SRC)
- $0\nu\beta\beta$ half-life estimate for ground-state transition using light neutrino mechanism: $2.3 \times 10^{25}$ yr (with $\mid \eta_{\nu L} \mid = 10^{-6}$)
- $0\nu\beta\beta$ half-life estimate for ground-state transition using heavy neutrino mechanism: $3.7 \times 10^{25}$ yr (with $\mid \eta_{N R} \mid = 1.2 \times 10^{-8}$)
- Q-value for $^{124}$Sn double-beta decay to ground state: $2.289$ MeV
- Best experimental upper limit for $^{124}$Sn double-beta decay half-life: $1 \times 10^{21}$ yr (Barabash 2008)

## Context

This work extends the authors' previous shell model calculations for $2\nu\beta\beta$ decay of $^{48}$Ca (Horoi 2013), $^{76}$Ge (Senkov and Horoi 2014), $^{82}$Se (Senkov, Horoi, and Brown 2014), $^{130}$Te, and $^{136}$Xe (Neacsu and Horoi 2015) to the case of $^{124}$Sn. The results contradict earlier QRPA predictions by Aunola (1996) and Suhonen and Civitarese (1998), which gave $T_{1/2}^{2\nu} = 7.8 \times 10^{19}$ yr for the ground-state transition, and also differ from earlier shell model results by Caurier and Nowacki (1999) that predicted $2.9 \times 10^{20}$ yr. The calculations are intended to guide the TIN.TIN experiment (Vandana 2014) and other experimental efforts searching for double-beta decay in $^{124}$Sn, potentially saving time and resources by providing more reliable half-life predictions.
