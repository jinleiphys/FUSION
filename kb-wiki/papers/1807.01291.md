# PISTA: Posterior Ion STAcking

> We present a program allowing for the construction of a heavy ion Monte Carlo event using a general purpose proton--proton event generator of the users' choice to deliver sub--collisions. In this heavy ion event, one or more jets can be added, again using a jet calculation of the users' choice, allowing for more realistic simulation studies of jets in a heavy ion background.

## Key claim

PISTA provides a generator-independent framework for constructing heavy ion Monte Carlo events by stacking proton-proton sub-collisions, but the underlying event model is a simplified implementation of the Angantyr recipe that treats secondary absorptive sub-collisions as approximately single diffractive events, and the program's predictive power for minimum bias observables is traded off when parameters are retuned for better background modeling.

## Method

The program performs a Glauber calculation to determine the number of sub-collisions in a proton-nucleus or nucleus-nucleus event, using a Woods-Saxon distribution for nuclear density and treating nucleons as black disks with radius determined by the inelastic non-diffractive nucleon-nucleon cross section. Sub-events from user-supplied HepMC-compatible event generators are read through FIFO pipes, sorted by hardness (default: scalar sum of hadron transverse momentum), and merged according to the Angantyr recipe where the hardest sub-collision defines the main event and secondary sub-collisions are treated as diffractive-like contributions. A primitive momentum conservation scheme restricts the summed momentum in beam directions to not exceed incoming energies using a default rapidity cutoff of eta = 7.0.

## Key numbers

* In PbPb collisions at sqrt(s_NN) = 2.76 TeV, both Pista + Pythia 8 and Pista + Herwig 7 overshoot the most central (0-5%) charged particle multiplicity data by approximately 20%
* For pPb collisions at sqrt(s_NN) = 5.02 TeV, Glauber-Gribov fluctuation parameters are set by default to Omega = 0.25, sigma_0 = 85 mb, lambda = 0.63
* The program uses a default rapidity cutoff of eta = 7.0 for momentum conservation, which includes the acceptance of all present detector experiments
* For Z+jet embedding, the Z is required to have pT > 60 GeV and muons pT > 10 GeV, with a jet considered back-to-back if Delta phi > 7pi/8

## Context

This work extends the Angantyr model for heavy ion collisions (Bierlich 2016, 2018) by making it generator-independent, allowing users to combine any HepMC-compatible proton-proton event generator (such as Pythia 8, Herwig 7, or Sherpa) with special-purpose generators that include medium response (such as JEWEL or JETSCAPE). The program enables realistic simulation studies of jets in a heavy ion background where the centrality definition is based on event activity rather than impact parameter, matching experimental practice and enabling direct comparison to data using the Rivet framework.
