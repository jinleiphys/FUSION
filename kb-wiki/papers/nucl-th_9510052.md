# SMMC method for two-neutrino double beta decay

> Shell Model Monte Carlo (SMMC) techniques are used to calculate two-neutrino double beta decay matrix elements. We validate the approach against direct diagonalization for $^{48}$Ca in the complete $pf$-shell using the KB3 interaction. The method is then applied to the decay of $^{76}$Ge in the $(0f_{5/2},1p,0g_{9/2})$ model space using a newly calculated realistic interaction. Our result for the matrix element is $0.13\pm0.05$ MeV$^{-1}$, in agreement with the experimental value.

# SMMC method for two-neutrino double beta decay

## Key claim

The Shell Model Monte Carlo (SMMC) method can calculate two-neutrino double beta decay matrix elements in model spaces too large for traditional shell model diagonalization, but the extrapolation procedure used to circumvent the sign problem introduces systematic uncertainty that may underestimate the closure matrix element by a factor of roughly 20 in the $^{48}$Ca validation case.

## Method

The authors compute the two-neutrino double beta decay matrix element via a thermal trace function $\phi(\tau,\tau^{\prime})$ that is evaluated using SMMC techniques based on the Hubbard-Stratonovich transformation. To avoid the sign problem with realistic interactions, they define a family of Hamiltonians $H(g,\chi)$ that are sign-problem-free for $g \leq 0$, calculate matrix elements at several negative $g$ values, and linearly extrapolate to the physical value at $g=1$. The method is validated against direct Lanczos diagonalization for $^{48}$Ca in the complete $pf$-shell with the KB3 interaction, then applied to $^{76}$Ge in the $(0f_{5/2},1p,0g_{9/2})$ model space using a realistic interaction derived from the Paris potential via a $G$-matrix folded-diagram method.

## Key numbers

- For $^{48}$Ca: SMMC closure matrix element $M_c = -0.21 \pm 0.29$ (dimensionless), compared to direct diagonalization result of $0.29$; the natural sum-rule scale is approximately 21, indicating the SMMC reproduces the shell model suppression of a factor of 70.
- For $^{48}$Ca: SMMC exact $2\nu$ matrix element $M^{2\nu} = 0.15 \pm 0.07$ MeV$^{-1}$, compared to the calculation of Caurier et al. (1990) giving $0.08$ MeV$^{-1}$.
- For $^{76}$Ge: combined SMMC exact matrix element $M^{2\nu} = 0.13 \pm 0.05$ MeV$^{-1}$, compared to the experimental value of $0.14 \pm 0.01$ MeV$^{-1}$ (using $g_A = 1.26$).
- For $^{76}$Ge: the effective interaction reproduces the experimental mass splitting of $^{76}$Ge and $^{76}$Se as $20.41 \pm 1.3$ MeV versus the experimental $20.72$ MeV.
- For $^{76}$Ge: $\beta^-$ Gamow-Teller strength $B(GT_-) = 12.6 \pm 0.3$ with energy centroid $6.3 \pm 0.2$ MeV, compared to experimental values of $19.9$ and $9.1$ MeV respectively.

## Context

This work extends the interacting shell model approach to two-neutrino double beta decay beyond $^{48}$Ca, which was previously the only nucleus tractable by direct diagonalization. Prior calculations for heavier nuclei relied on the quasi-particle random phase approximation (QRPA), which is sensitive to the $J=1^+, T=0$ particle-particle interaction as noted by Vogel and Zirnbauer (1986), or on the closure approximation used by Haxton and Stephenson (1984) with severely truncated bases. The SMMC method enables calculations in model spaces containing approximately $10^8$ configurations for $^{76}$Ge, and the authors plan to apply it to several other heavier nuclei.
