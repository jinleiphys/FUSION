# Polyakov loop spectroscopy in the confined phase of gluodynamics and QCD

> By using a simple relativistic model, we compute the glueball and gluelump spectra and relate these quantities, respectively, to the trace anomaly and Polyakov loop in the adjoint representation of gluodynamics. This spectroscopic description of thermodynamics is extended with the inclusion of quarks. The relation between the hadron resonance gas and the Polyakov loop in the fundamental and higher representations is addressed.

## Key claim

The paper demonstrates that the Polyakov loop in any group representation can be interpreted as a partition function of heavy-light bound states (hadrons, hybrids, exotics) in the confined phase, and that the trace anomaly in gluodynamics can be related to the glueball spectrum, with the caveat that this spectroscopic description relies on a simple relativistic model with a linear confining potential and neglects interactions between the bound states.

## Method

The authors compute the glueball spectrum from a relativistic two-gluon model with a linear potential \(V(r) = \sigma_8 r\) and Casimir scaling \(\sigma_8 = \frac{N_c^2}{4}\sigma_3\), then obtain the trace anomaly from the quantum partition function via a transformation that introduces an auxiliary mass variable \(\mu\). For the Polyakov loop in the adjoint representation, they similarly compute the gluelump spectrum (one static gluon dressed by a dynamical gluon) using the same potential. In QCD, they extend a Polyakov-Constituent Quark Model by expanding the action in powers of the Polyakov loop operators, identifying each term with multi-quark/gluon states, and then quantizing to obtain the hadron resonance gas (HRG) representation of the Polyakov loop in any representation.

## Key numbers

* The fit of the trace anomaly to lattice data yields \(T_c/\sqrt{\sigma_3} = 0.774\) with \(\chi^2/\text{dof}=0.54\) in the range \(0.70 < T/T_c < 0.94\) for gluodynamics, using gluon spin degeneracy \(\gamma=3\). This is compared to the lattice value \(T_c/\sqrt{\sigma_3} \approx 0.629(3)\) from Borsanyi (2012).
* The classical scaling relation between the adjoint Polyakov loop and the trace anomaly is \(L_{\bf 8}(T) \approx \frac{\pi^2}{3\gamma} \Delta_{[gg]}(T)\), which holds for temperatures \(T \gtrsim 0.5 T_c\).
* For the Polyakov loop in QCD with \(N_f=2\), the model uses constituent quark mass \(M_q = 300\ \text{MeV}\) and gluon mass \(M_g = 664\ \text{MeV}\) to compute the loop in representations \({\bf 3}, {\bf 8}, {\bf 6}, {\bf 15}, {\bf 27}, {\bf 10}, {\bf 24}, {\bf 42}, {\bf 64}, {\bf 15'}, {\bf 35}\).

## Context

This work extends the hadron resonance gas description of QCD thermodynamics (Hagedorn 1984; Borsanyi 2010; Bazavov 2011) by providing a spectroscopic interpretation of the Polyakov loop, building on the authors' earlier proposal (Megias 2012) that the fundamental Polyakov loop can be expressed as a sum over heavy-light hadron states. The approach enables a physical understanding of the Polyakov loop in higher representations (adjoint, sextet, etc.) as partition functions of bound states containing a static source, which could be tested against lattice calculations of excited and exotic hadron spectra.
