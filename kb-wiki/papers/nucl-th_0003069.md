# Fine structure of alpha decay in odd nuclei

> Using an alpha decay level scheme, an explanation for the fine structure in odd nuclei is evidenced by taking into account the radial and rotational couplings between the unpaired nucleon and the core of the decaying system. It is stated that the experimental behavior of the alpha decay fine structure phenomenon is directed by the dynamical characteristics of the system.

## Key claim

Using a two-center shell model with radial and rotational couplings between the unpaired valence nucleon and the core, the fine structure in odd-nuclei alpha decay can be explained quantitatively, but the tunneling velocity must be treated as a fit parameter (here 9×10⁶ fm/fs) rather than derived from first principles.

## Method

The superasymmetric two-center shell model (STCSM) was modified to reproduce alpha-particle single-particle levels by forcing the oscillator stiffness ℏω₂ to vary from the standard STCSM value to 21.8 MeV at scission, with additional modifications to mass-asymmetry terms and spin-orbit/L² matrix elements. The coupled differential equations for occupation probabilities of daughter levels by the unpaired neutron were solved including both Landau-Zener radial coupling at avoided level crossings and Coriolis rotational coupling, using the STCSM to provide interaction energies, diabatic level energies, and wave functions. Barrier penetrabilities for each channel were computed using the Yukawa-plus-exponential nuclear potential, with angular momentum distributions weighted by Clebsch-Gordan coefficients.

## Key numbers

- ℏω_α = 21.8 MeV (oscillator stiffness for the alpha particle at scission)
- ℏω₂ = 41A₂^{1/3} (standard STCSM value before modification, at R_n=0)
- Spin-orbit and L² coefficients for ²¹¹Po and ²⁰⁷Pb: κ = 5.75×10⁻², η = 0.43, V_c = 55.80 MeV
- Alpha particle single-particle energies (MeV): 1s_{1/2} = -12, 1p_{3/2} = 1.38, 1p_{1/2} = 1.68, 2s_{1/2} = 5.34, 1d_{5/2} = 10.43, 1d_{3/2} = 11.86
- For alpha particle oscillator fit: κ = 4.51×10⁻³, η = 0, V_c = 52.91 MeV
- Experimental fine structure for ²¹¹Po alpha decay: 98.9% to 3p_{1/2}^D ground state, 0.55% to 2f_{5/2}^D first excited state, 0.54% to 3p_{3/2}^D second excited state
- Tunneling velocity used as fit parameter: 9×10⁶ fm/fs
- Calculated intensity ratio (first excited state / ground state): 0.0071
- Calculated intensity ratio (second excited state / ground state): 0.0062
- Quantum time-dependent approach gives tunneling velocity ~1×10⁷ fm/fs (from Ref. s1)

## Context

This work extends the Landau-Zener-based formalism previously applied to cluster decay fine structure (Mirea, 1998; 1999) to the case of alpha decay, providing a dynamical alternative to the traditional overlap-based approach of Mang (1960) and Mang (1964). The model successfully reproduces the fine structure hindrance factors for ²¹¹Po alpha decay, and suggests that the same promotion mechanism governing cluster decay also governs alpha decay fine structure. This enables future application of molecular-type models to alpha decay and opens the possibility of treating the reorganization of single-particle levels during disintegration as a dynamical process rather than a static overlap calculation.
