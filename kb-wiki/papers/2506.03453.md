# Permutationally-Invariant N-body gates via Tavis-Cummings Hamiltonian

> Widely used in atomic and superconducting qubit systems, the Jaynes-Cummings (JC) Hamiltonian is a simple, yet powerful model for a two-level system interacting with a quantum harmonic oscillator. In this paper, we focus on a system of n qubits, identically coupled to a single oscillator via JC interaction, also known as the Tavis-Cummings (TC) Hamiltonian. We show that all permutationally-invariant unitaries on an arbitrary number of qubits can be realized using this permutationally-invariant Hamiltonian, which couples the qubits to an oscillator initialized in its vacuum state, together with global uniform x and z fields on all qubits. This includes useful gates, such as controlled-Z gate with an arbitrary number of control qubits. As a corollary, we find that all permutationally invariant states -- including useful entangled states such as GHZ and Dicke states -- can be prepared using this interaction and global fields. We also characterize unitaries that can be realized on the joint Hilbert space of the qubits and oscillator with the TC interaction and global z field, and develop new methods for preparing the state of the oscillator in an arbitrary initial state. We present various examples of explicit circuits for the case of n=2 qubits. In particular, we develop new methods for implementing controlled-Z, SWAP, iSWAP, and $\sqrt{i\text{SWAP}}$ gates using only the TC interaction and a global z field. Our work also reveals an accidental symmetry in the TC Hamiltonian and shows that it can be explained using Schwinger's oscillator model of angular momentum.

## Key claim

All permutationally-invariant (PI) unitary transformations on an arbitrary number of qubits can be realized using the Tavis-Cummings (TC) Hamiltonian coupling the qubits to an oscillator initialized in its vacuum state, together with global uniform x and z fields on all qubits, up to a global phase. This result holds despite a previously unidentified accidental symmetry of the TC Hamiltonian and a no-go theorem (Marvian 2022) that restricts realizing PI unitaries using k-body interactions with k < n.

## Method

The authors consider a system of n qubits identically coupled to a single quantum harmonic oscillator via the Tavis-Cummings Hamiltonian H_TC = g_TC (J_+ a + J_- a^dagger), where J_+ and J_- are collective qubit operators and a is the oscillator annihilation operator. They combine this interaction with global uniform fields H_x = omega_x J_x and H_z = omega_z J_z, which can be turned on and off at will, and use the oscillator initialized in the vacuum state as an ancilla. The proof decomposes the overall unitary into sequences of three families of unitaries: R_x(theta), R_z(theta), and V_TC(r) = exp(-i r H_TC / g_TC), and exploits the U(1) symmetry of the TC Hamiltonian associated with the conserved operator J_z + a^dagger a to truncate the infinite-dimensional Hilbert space into finite-dimensional subspaces.

## Key numbers

- For n=2 qubits, the controlled-Z (CZ) gate requires total TC interaction time tau = 2.87 x 2pi / g_TC
- For n=2 qubits, the SWAP gate requires total TC interaction time tau = 1.27 x 2pi / g_TC
- For n=2 qubits, the iSWAP gate requires total TC interaction time tau = 2.55 x 2pi / g_TC
- For n=2 qubits, the sqrt(iSWAP) gate requires total TC interaction time tau = 2.68 x 2pi / g_TC
- For any two-qubit PI unitary that conserves J_z, the required total interaction time is bounded by tau <= 3.92 x 2pi / g_TC

## Context

This work extends the understanding of controllability in systems with permutation symmetry, directly addressing restrictions identified by Marvian (2022) on realizing PI unitaries using only 2-local PI gates. The results enable preparation of all PI states including GHZ and Dicke states using only collective control, without individual qubit addressing, which simplifies experimental implementation in trapped-ion systems (Haffner 2005) and circuit QED (Fink 2009). The characterization of unitaries realizable with TC and global z field alone resolves an open question from Keyl (2014) and reveals that for n >= 4 qubits, not all PI U(1)-invariant unitaries can be realized without the global x field, unlike the n=2 and n=3 cases.
