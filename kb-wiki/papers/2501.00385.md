# Closed-form formulas in number-conserved pairing theory

## Key claim
The paper presents closed-form formulas for the norm and many-body density matrices between general wave functions with exact particle numbers in pairing theory, using properties of the generalized Kronecker delta. These formulas apply to both even and odd particle-number systems and accommodate pair condensate as well as broken-pair configurations, but the practical advantage in computational speed is demonstrated only for a specific nucleus (70Ge) using a particular effective interaction (JUN45) and single-particle space.

## Method
The author derives expressions for the norm and matrix elements of many-body operators between broken-pair states by expressing overlaps as sums of minors and Pfaffians, using the generalized Kronecker delta to simplify the combinatorics. The formulas are validated through a benchmark calculation with a pairing Hamiltonian in a doubly degenerate Hartree-Fock basis, and applied to angular momentum projected number-conserved BCS (PNBCS) calculations for 70Ge using the JUN45 shell-model effective interaction within the 1p1/2 1p3/2 0f5/2 0g9/2 single-particle space, solving the Hill-Wheeler equation.

## Key numbers
- For degenerate single-particle states with constant pairing strength, the energy of an N-pair condensate is E = 2εN - gN(Ω - N + 1), where ε is the single-particle energy, g is the pairing strength, Ω is the number of pair-degenerate levels, and N is the number of pairs.
- The norm of an N-pair condensate with uniform pair coefficients reaches a maximum value of (N!)^2 * binomial(Ω, N) / Ω^N, which approaches N! in the limit Ω → ∞.
- For 70Ge, the PNBCS computation using the new formulas is approximately four times faster for the one-body density matrix and twice as fast for the two-body interactions compared to a code based on the generalized Wick theorem.
- The time complexity for computing the matrix element of a two-body operator using Eq. (8) is O[binomial((ω-ν)/2, N)^2 * binomial(n, 2)^2 * n^3], and using Eq. (7) is O[binomial(ω-2, n-2) * binomial(n, ν) * n^3], where ω = 2Ω is the dimension of the canonical basis, n = 2N + ν is the total particle number, N is the number of pairs, and ν is the seniority number.

## Context
This work extends the number-conserved pairing formalism previously developed using recursive formulas derived from the generalized Wick theorem (Talmi, 1972; various papers by the author and collaborators, 2009-2019) and the Pfaffian approach introduced by Robledo (2009) for HFB wave functions. The closed-form formulas directly facilitate applications in the generator coordinate method and symmetry restoration techniques including angular momentum projection, as demonstrated by the PNBCS calculation for 70Ge. The results suggest that the ground and γ bands in 70Ge correspond to a triaxially-deformed intrinsic configuration while the β band arises from axially symmetric deformation, though the experimental γ band indicates a γ-soft configuration whereas the calculation predicts γ-rigid behavior, pointing to the need for modified effective interactions and expanded single-particle spaces in future work.
