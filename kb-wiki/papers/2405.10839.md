# Model orthogonalization and Bayesian forecast mixing via Principal Component Analysis

> One can improve predictability in the unknown domain by combining forecasts of imperfect complex computational models using a Bayesian statistical machine learning framework. In many cases, however, the models used in the mixing process are similar. In addition to contaminating the model space, the existence of such similar, or even redundant, models during the multimodeling process can result in misinterpretation of results and deterioration of predictive performance. In this work we describe a method based on the Principal Component Analysis that eliminates model redundancy. We show that by adding model orthogonalization to the proposed Bayesian Model Combination framework, one can arrive at better prediction accuracy and reach excellent uncertainty quantification performance.

## Key claim

The paper claims that adding Principal Component Analysis (PCA) orthogonalization to a Bayesian Model Combination (BMC) framework improves prediction accuracy and uncertainty quantification by eliminating redundancy among similar or identical models, but the method's effectiveness depends on treating the number of retained principal components as a hyperparameter selected via validation set performance rather than a fixed quantity.

## Method

The authors first construct a matrix of model forecasts across all input locations, center it by subtracting the average forecast of all models, and perform Singular Value Decomposition to extract principal components. They then replace the original model forecasts with a reduced set of orthogonal principal components in a Bayesian linear mixing model, using a multivariate Gaussian prior for the component weights and a Gamma prior for the error scale, with posterior inference performed via Gibbs sampling. The number of principal components retained is selected by evaluating root mean squared error on a validation set.

## Key numbers

- The synthetic data case study used 629 binding energies of even-even nuclei with 8 ≤ Z ≤ 102, split into 300 training, 71 validation, and 258 testing datapoints.
- The realistic case study used 15 nuclear mass models from the MassExplorer database, HFB-24, and FRDM-2012 mass tables, applied to the same 629 nuclei.
- The weakly informed prior for the error scale used shape parameter ν₀ = 10 and scale parameter σ₀ = 2 MeV for the first case study, and ν₀ = 10 and σ₀ = 0.5 MeV for the realistic case.
- In scenario S3 of the liquid drop model study, the root mean squared error saturated after retaining 3 principal components, matching the 4 distinct model classes present.

## Context

This work extends Bayesian model mixing frameworks previously developed by StackingGelman (2018) and Kejzlar (2023) for combining nuclear physics models, and applies PCA-based orthogonalization following the approach of Clyde (1996) for model combination. The method enables practitioners to identify the effective number of independent models in a set, filter out noise directions, and produce more robust extrapolations into unknown regions of the nuclear chart, building on prior nuclear mass studies by Neufcourt (2018, 2020) and Wu (2024).
