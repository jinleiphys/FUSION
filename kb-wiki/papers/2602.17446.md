# Model bias and parameter optimisation with the example of INCL/ABLA

> The accuracy and precision of high-energy spallation models play a crucial role in the design and development of new applications and experiments, as well as in data analysis. We discuss the complementarity between parameter optimisation and model bias estimation approaches within a Bayesian framework. This is illustrated using the IntraNuclear Cascade model of Liège (INCL) together with the Ablation model (ABLA), for which these two approaches for model bias estimation have been applied independently in previous works.

## Key claim

The paper argues that parameter optimisation and model bias estimation are complementary Bayesian approaches that can be applied together to improve nuclear model predictions, but the synergy is not guaranteed and depends on whether the parameter optimisation corrects genuine physical deficiencies rather than compensating for model errors.

## Method

The authors apply Gaussian Process (GP) regression within a Bayesian framework to two independent approaches for the INCL/ABLA spallation model combination. Parameter optimisation iteratively adjusts model parameters using a local linearisation of the model and experimental data to find an optimal parameter set, followed by a Gibbs sampling-like phase to estimate parameter uncertainties. Model bias estimation uses a separate GP formulation with a covariance matrix constructed from experimental uncertainties, model uncertainties, and a physical correlation kernel (Matérn, constant, and diagonal kernels) whose hyperparameters are determined by Marginal Likelihood Optimisation using the L-BFGS-B algorithm.

## Key numbers

- The fission-dissipation coefficient in ABLA was intentionally deteriorated from its default value of 4.5 (arbitrary unit) to 7(±5); after optimisation it returned to 4.40 ± 0.59
- The level density curvature parameter changed from an initial value of 0 ± 0.1 to -0.016 ± 0.028 after optimisation
- The χ²/DoF for proton-induced fission cross sections of Bi decreased from 4.8 to 1.7 after parameter optimisation
- The two optimised parameters (fission-dissipation coefficient and level density curvature) exhibit a strong correlation of 0.96
- Experimental data used in the study are limited to energies below 1.2 GeV

## Context

This work extends the authors' previous independent developments of parameter optimisation (Hirtz et al., 2022) and model bias estimation (Schnabel et al., 2021) for the INCL/ABLA model combination, which are integrated into major transport codes such as Geant4. The paper demonstrates that combining both approaches reduces uncertainty on bias-corrected predictions, particularly away from experimental data points, enabling more reliable model predictions for applications in neutrino physics, meteorite studies, antimatter research, and hadron therapy.
