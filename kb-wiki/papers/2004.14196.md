# The Neutron Star Outer Crust Equation of State: A Machine Learning approach

> Constructing the outer crust of the neutron stars requires the knowledge of the Binding Energy (BE) of the atomic nuclei. Although the BE of a lot of the nuclei is experimentally determined and can be obtained from the AME data table, for the others we need to depend on theoretical models. There exist a lot of physical theories to predict the BE, each with its own strengths and weaknesses. In this paper, we apply Machine Learning (ML) algorithms on AME2016 data set to predict the Binding Energy {of atomic nuclei}. The novel feature of our work is that it is model independent. We do not assume or use any nuclear physics model but use only ML algorithms directly on the AME2016 data set. Our results are further refined by using another ML algorithm to train the errors of the first algorithm, and repeating this process iteratively. Our best algorithm gives $σ_{\rm rms} \approx 0.58$ MeV for Binding Energy on randomized testing sets. This is comparable to all physics models or ML improved physics models studied in literature till date. Using the predictions of our Machine Learning algorithm, we construct the outer crust equation of state (EoS) of a neutron star and show that our model is comparable to existing models. This work also demonstrates the use of various ML algorithms and a detailed analysis on how we arrived at our best algorithm. It will help the physics community in understanding how to choose an ML algorithm which would be suited for their data set. Our algorithms and best fit model is also made publicly available for the use of the community.

## Key claim

The authors claim that a purely machine-learning-based model, trained only on the atomic number (Z) and neutron number (N) from the AME2016 data set without any nuclear physics input, can predict nuclear binding energies with a root-mean-square error comparable to physics-based models. The hedge is that this model is not intended to compete with physics models but to explore how accurately ML can predict binding energies in the absence of physics inputs, and it cannot predict other nuclear properties.

## Method

The authors applied supervised machine learning regression algorithms (Linear Regression, Decision Tree, Random Forest, Polynomial Regression, Support Vector Machine, and Artificial Neural Network) to the AME2016 data set of 3435 nuclei, using only Z and N as features and binding energy as the target variable. They then refined predictions by training a Random Forest model on the errors of the base algorithm and iterating this process until the error saturated, a technique described as stacked generalization. The data set was split randomly into 60% training, 20% validation, and 20% testing sets, and the best final model (SVM base with 10 Random Forest iterations, named MIML) was used to construct the neutron star outer crust equation of state.

## Key numbers

*   The best algorithm (SVM base with 10 Random Forest iterations, MIML) achieved a root-mean-square error of approximately 0.58 MeV for binding energy on randomized testing sets.
*   The root-mean-square error for light nuclei (Z < 20) was 505 keV, for nuclei with 20 ≤ Z ≤ 92 it was 248 keV, and for heavy nuclei (Z > 92) it was 127 keV (these values include training, validation, and test sets).
*   The root-mean-square error for the base SVM model alone was 1.81 MeV on the test set.
*   The root-mean-square error for the base Linear Regression model was 45.78 MeV on the test set, with the equation BE = 8.3 Z + 6.7 N.

## Context

This work extends prior efforts to use machine learning for nuclear mass prediction, such as neural networks (Gazula 1992) and Bayesian Neural Networks (Utama 2016, Niu 2018), but is novel in being completely model-independent, using no underlying nuclear physics model. It enables the construction of the neutron star outer crust equation of state using purely data-driven binding energy predictions, and the authors note that their approach avoids the high errors for light and heavy nuclei that plague mean-field theories and models like the Bethe-Weizsacker formula or the finite-range droplet model.
