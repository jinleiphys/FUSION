# On the Gerasimov-Drell-Hearn sum rule for the deuteron

> The Gerasimov-Drell-Hearn sum rule is evaluated for the deuteron by explicit integration up to 550 MeV including contributions from the photodisintegration channel and from coherent and incoherent single pion production as well. The photodisintegration channel converges fast enough in this energy range and gives a large negative contribution, essentially from the $^1S_0$ resonant state near threshold. Its absolute value is about the same size as the sum of proton and neutron GDH values. It is only partially cancelled by the single pion production contribution. But the incoherent channel has not reached convergence at 550 MeV.

## Key claim
The Gerasimov-Drell-Hearn sum rule for the deuteron is evaluated by explicit integration up to 550 MeV, but the incoherent single pion production channel has not reached convergence at this energy, so the total integrated value of -183 μb is incomplete and a more realistic estimate using multipole analysis data gives -19 μb, still requiring positive contributions from higher energies to satisfy the sum rule.

## Method
The authors computed the GDH integral for the deuteron by summing contributions from three channels: photodisintegration (γd → np), coherent single pion production (γd → π⁰d), and incoherent single pion production (γd → πNN). The photodisintegration calculation used a nonrelativistic framework with the Bonn potential, including one-body currents, meson exchange currents, isobar configurations, and leading order relativistic contributions up to multipole order L=4. Coherent pion production was modeled with Δ-resonance excitation including pion rescattering via coupled NΔ, NNπ, and NN channels, while incoherent pion production used a spectator nucleon approach with the resonant M_{1+}^{3/2} multipole.

## Key numbers
- Photodisintegration contribution to GDH integral up to 550 MeV: -413 μb (with full model including relativistic corrections)
- Coherent π⁰ production contribution: +63 μb (up to 550 MeV)
- Incoherent π⁰ production contribution: +288 μb (up to 550 MeV)
- Incoherent nnπ⁺ production contribution: -35 μb (up to 550 MeV)
- Incoherent ppπ⁻ production contribution: -86 μb (up to 550 MeV)
- Total GDH integral from all channels up to 550 MeV: -183 μb
- Sum of free proton and neutron GDH values: 204.8 μb (proton) + 233.2 μb (neutron) = 438 μb
- Deuteron anomalous magnetic moment κ_d = -0.143, giving sum rule prediction I^{GDH}_d = 0.65 μb
- Using multipole analysis data (fit SM95) instead of model: I^{GDH}_n(550 MeV) + I^{GDH}_p(550 MeV) = 331 μb, giving deuteron estimate of -19 μb

## Context
This work extends earlier evaluations of the GDH sum rule for nucleons by Karliner (1973) and more recent multipole analyses (Sandorfi and Whisnant, 1994) to the deuteron, which has a very small anomalous magnetic moment (κ_d = -0.143) but large contributions from its constituent nucleons. The calculation demonstrates that the photodisintegration channel provides a large negative contribution that nearly cancels the sum of free nucleon GDH values, with the dominant effect coming from the M1 transition to the resonant ^1S_0 state near threshold as first discussed by Berman and Drell (1967). The results enable future experimental checks of the low-energy spin asymmetry and motivate improved theoretical treatment of the elementary pion photoproduction operator above the Δ-resonance region, as well as inclusion of two-pion production contributions.
