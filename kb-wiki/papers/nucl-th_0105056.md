# Spin structure of the Delta(1232) and inelastic Compton scattering

> Radiative transitions gamma + Delta(1232) -> N^* are discussed in the nonrelativistic quark model with spin-orbit corrections for the 70-plet L^P=1^- nucleon resonances N^*. The reaction gamma + N -> gamma + Delta is considered as a tool to measure some of these transitions. A particular sensitivity to photoexcitations of S_{11}(1535), D_{13}(1700), and D_{15}(1675) is predicted.

# Spin structure of the Delta(1232) and inelastic Compton scattering

## Key claim

The Delta(1232) must have strong photocouplings to nucleon resonances of spin J >= 5/2, particularly D15(1675), to satisfy the Gerasimov-Drell-Hearn sum rule for the Delta, because the dominant M1 transition N -> Delta gives a large negative contribution to the GDH integral that must be compensated by higher-spin states.

## Method

Radiative transitions gamma + Delta(1232) -> N* were calculated using the nonrelativistic quark model with spin-orbit corrections (including both additive and non-additive two-body terms) for the 70-plet L^P = 1^- nucleon resonances. The photocouplings were computed for Delta+ to seven negative-parity baryons, and these amplitudes were used to estimate differential cross sections for inelastic Compton scattering gamma N -> gamma Delta including s-channel and u-channel resonances with experimental masses and widths.

## Key numbers

- Photocoupling A_{3/2} for gamma N -> Delta in the nonrelativistic quark model without spin-orbit corrections: -175, -194, and -219 x 10^{-3} GeV^{-1/2} in the center-of-mass, Breit, and lab frames respectively; with spin-orbit corrections included: -198, -194, and -182 x 10^{-3} GeV^{-1/2}, showing restored frame independence
- Photocoupling A_{5/2} for Delta+ -> D15(1675) with spin-orbit corrections: -267 x 10^{-3} GeV^{-1/2}
- Photocoupling A_{3/2} for Delta+ -> D13(1700) with spin-orbit corrections: -175 x 10^{-3} GeV^{-1/2}
- Photocoupling A_{1/2} for Delta+ -> S11(1535) with spin-orbit corrections: -86 x 10^{-3} GeV^{-1/2}
- GDH integral for the nucleon: I = 216 microbarns from the N -> Delta transition, compared to experimental values I_p = 205 microbarns and I_n = 233 microbarns
- GDH integral contribution from N -> Delta for the Delta target: I = -323 microbarns

## Context

This work extends the analysis of the Gerasimov-Drell-Hearn sum rule for the Delta(1232) previously reported by L'vov (2000), and builds on the nonrelativistic quark model calculations of Close (1970, 1990) and Capstick (2000). The calculation confirms the earlier proposal by Carlson and Carone (1998) that inelastic Compton scattering gamma N -> gamma Delta can serve as a practical tool to measure Delta photocouplings, particularly for the D15(1675) resonance which is decoupled from the proton due to the Moorhouse selection rule and can only be studied via neutron targets through gamma d -> gamma p_s Delta^0.
