# A predictive phenomenological tool at small Bjorken-x

> We present the results from global fits of inclusive DIS experimental data using the Balitsky-Kovchegov equation with running coupling.

## Key claim

The Balitsky-Kovchegov equation with running coupling (rcBK) can accurately describe high-precision HERA inclusive DIS data and charm structure function data, provided that separate fit parameters are allowed for light and heavy quark contributions and that the running coupling is regulated in the infrared.

## Method

The authors performed global fits of the H1 and ZEUS combined reduced cross-section data (196 points) and, separately, of those data plus available $F_2^c$ data (265 points total), using the rcBK evolution equation for the dipole-target scattering amplitude. The dipole amplitude was evolved from an initial condition at $x_0 = 0.01$ using the GBW parametrization, with the running coupling evaluated at 1-loop accuracy in coordinate space and regulated in the infrared. Fits were performed both with only light quarks ($u,d,s$) contributing and with a variable-flavor scheme including charm and bottom quarks, allowing independent normalization and initial-condition parameters for light and heavy flavors.

## Key numbers

- Light-only fit to 196 H1/ZEUS data points ($Q^2 < 50$ GeV$^2$, $x < 0.01$): $\chi^2/\mathrm{d.o.f.} = 206/192 = 1.07$, with parameters $\sigma_{0,\mathrm{light}} = 33.1$ mb, $Q_{s\,0,\mathrm{light}}^2 = 0.26$ GeV$^2$, $\gamma_{\mathrm{light}} = 0.97$, $C = 2.3$, and $m_{\mathrm{light}} = 0.14$ GeV.
- Light+heavy fit to 265 data points (H1/ZEUS plus $F_2^c$ data): $\chi^2/\mathrm{d.o.f.} = 312/257 = 1.2$, with parameters $\sigma_{0,\mathrm{light}} = 32.7$ mb, $Q_{s\,0,\mathrm{light}}^2 = 0.26$ GeV$^2$, $\gamma_{\mathrm{light}} = 1.24$, $\sigma_{0,\mathrm{heavy}} = 28.4$ mb, $Q_{s\,0,\mathrm{heavy}}^2 = 0.18$ GeV$^2$, $\gamma_{\mathrm{heavy}} = 0.95$, $C = 3.3$, and $m_{\mathrm{light}} = 0.104$ GeV.
- Heavy quark masses fixed at $m_c = 1.4$ GeV and $m_b = 4.75$ GeV.
- Infrared regulation of the running coupling: $\alpha_s(r^2 > r^2_{\mathrm{fr}}) = \alpha_{\mathrm{fr}} = 0.7$.

## Context

This work extends the earlier global fit of Albacete et al. (2009) by using the new high-precision H1 and ZEUS combined data set for the reduced cross section, which eliminates the theoretical bias from extracting $F_2$ and $F_L$ separately. It also incorporates charm structure function data to constrain heavy quark contributions. The results strengthen the rcBK equation as a viable phenomenological tool for small-$x$ physics at the LHC, replacing earlier CGC-motivated dipole models that lacked rigorous evolution. A parametrization of the dipole-proton cross section for use in computing physical observables is promised in a forthcoming publication.
