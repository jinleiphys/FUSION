# A self-consistent calculation of non-spherical Bose-Einstein correlation functions with Coulomb final-state interaction

> Particle correlations and femtoscopy are a rich subfield of high-energy physics. As the experimental data become more precise, there is an increasing need for the theoretical calculations to provide better and more general descriptions of the measurements. One of the important new directions is the investigation of the precise shape of the Bose-Einstein correlation functions utilizing Lévy-stable distributions. This work is a direct follow-up to our previous study, in which we developed a novel method for calculating Bose-Einstein correlation functions including the Coulomb final-state interaction. In this paper, we present a self-consistent generalization of the previous approach to non-spherical source functions and investigate the validity of the previously applied approximations assuming spherical symmetry. We present a software package that includes the calculation of a fully three-dimensional correlation function including the Coulomb interaction.

## Key claim

The authors present a self-consistent generalization of their previous method for calculating Bose-Einstein correlation functions with Coulomb final-state interaction to non-spherical source functions, and they investigate the validity of the previously applied approximation that assumed spherical symmetry for the Coulomb correction factor.

## Method

The method expresses the two-particle source function \(D(\mathbf{r})\) as a Fourier transform of a kernel \(f(\mathbf{q})\), then interchanges the spatial and Fourier-space integrals in the Yano-Koonin formula using a regulator \(e^{-\lambda r}\) and Lebesgue-dominated convergence theorems. The resulting integrals are evaluated analytically using a modified Nordsieck method, yielding an expression for the correlation function \(C(\mathbf{Q})\) that separates into a free-particle term, a delta-function term, and two integral terms \(\mathcal{A}_1\) and \(\mathcal{A}_2\) expressed in specially tailored coordinate systems. The implementation is provided as a software package for fully three-dimensional calculations.

## Key numbers

* The Sommerfeld parameter is defined as \(\eta = mc^2\alpha / (2\hbar k c)\), with the fine structure constant \(\alpha \approx 1/137\).
* The Gamow factor (normalization squared) is \(|\mathcal{N}|^2 = 2\pi\eta / (e^{2\pi\eta} - 1)\).
* In the interaction-free limit (\(\eta = 0\)), the correlation function reduces to \(C^{(0)}(\mathbf{Q}) = 1 + f(\mathbf{Q})\), recovering the standard Fourier-transform result.
* The final expression for the correlation function is \(C(\mathbf{Q}) = |\mathcal{N}|^2 [1 + f(\mathbf{Q}) + (\eta/\pi)(\mathcal{A}_1 + \mathcal{A}_2)]\), generalizing the spherically symmetric result from Nagy (2023).

## Context

This work extends the authors' previous method (Nagy, 2023) from spherically symmetric sources to fully three-dimensional non-spherical source functions, enabling precision analysis of Bose-Einstein correlation measurements. It directly addresses the approximation used in many experimental analyses (STAR, 2003; STAR, 2004; PHENIX, 2004; STAR, 2009; STAR, 2014; PHENIX, 2014) where the Coulomb correction \(K_C(\mathbf{Q})\) was calculated assuming spherical symmetry, and tests the accuracy of that approximation against the full 3D calculation. The method enables future experimental fits using elliptically contoured Levy-stable source distributions, which recent measurements (PHENIX, 2017; PHENIX, 2024; NA61/SHINE, 2023; CMS, 2023) have shown to describe two-pion sources in heavy-ion collisions.
