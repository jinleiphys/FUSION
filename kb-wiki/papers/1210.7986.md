# Finite Size Corrected Relativistic Mean-Field Model and QCD Critical End Point

> The effect of finite size of hadrons on the QCD phase diagram is analyzed using relativistic mean field model for the hadronic phase and the Bag model for the QGP phase. The corrections to the EOS for hadronic phase are incorporated in a thermodynamic consistent manner for Van der Waals like interaction. It is found that the effect of finite size of baryons is to shift CEP to higher chemical potential values.

## Key claim

The finite size of baryons shifts the critical end point (CEP) of the quark-hadron phase transition to higher baryon chemical potential values compared to the point-like baryon case, and the first-order phase transition is not possible for the entire QCD phase diagram when finite-sized baryons are considered.

## Method

The authors constructed a relativistic mean-field (RMF) model for the hadronic phase that incorporates finite-size volume corrections in a thermodynamically consistent manner, treating the correction as a Van der Waals-like interaction. For the quark-gluon plasma (QGP) phase, they used a Bag model equation of state with perturbative corrections of order alpha_s. The phase transition was determined using the Gibbs criteria of equal pressure, temperature, and chemical potential between the two phases, solving five coupled field equations for the hadronic phase self-consistently.

## Key numbers

- The baryon radius used for finite-size corrections was taken as r_B = 0.8 fm, chosen from the range r_B = 0.5-0.8 fm.
- The bag constant B was set to 344 MeV/fm^3, following Satarov (2009).
- The interaction parameter zeta (alpha_s) was fixed at zeta = 0.2, as this value produces QGP behavior consistent with lattice QCD.
- For point-like baryons (r_B = 0 fm), the CEP coordinates were (T, mu_B) = (154, 120) MeV.
- For finite-sized baryons (r_B = 0.8 fm), the CEP coordinates were (T, mu_B) = (152, 140) MeV.
- The first-order phase transition was found to be absent for baryon chemical potential values below 420 MeV, where the transition becomes a crossover.

## Context

This work extends the authors' previous RMF model (Uddin, Bashir, and Ahmad, 2012) which did not include finite-size corrections and found a CEP consistent with the LR2 variant of lattice gauge theory (Fodor and Katz, 2004). The results are consistent with finite-size scaling (FSS) analysis of RHIC data, which negates a CEP with baryon chemical potential values less than 400 MeV. The findings enable more realistic modeling of the QCD phase diagram for heavy-ion collision phenomenology, particularly for the search of the CEP in future experiments.
