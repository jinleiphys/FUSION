# Toward extracting scattering phase shift from integrated correlation functions on quantum computers

> Based on a established relation in Refs.~\cite{Guo:2023ecc,Guo:2024zal,Guo:2024pvt} that relates the integrated correlation functions for a trapped system to the infinite volume scattering phase shifts through a weighted integral, we propose to extract the infinite volume scattering phase shifts through quantum simulation of the integrated correlation functions of trapped two-particle systems on quantum computers. The integrated correlation function can be computed by an ancilla-based algorithm proposed in Ref.~\cite{PhysRevA.64.022319}. The proposal is demonstrated with a simple contact interaction fermion model.

# Toward extracting scattering phase shift from integrated correlation functions on quantum computers

## Key claim
The paper proposes that infinite-volume scattering phase shifts can be extracted from quantum simulation of integrated correlation functions of trapped two-particle systems, provided the phase shift is first modeled and parametrized (e.g., via chiral perturbation theory or K-matrix formalism) and then fit to the simulation data through a weighted integral relation.

## Method
The Hamiltonian of a trapped two-fermion system with contact interaction is expanded in the trap basis (e.g., plane waves in a periodic box) and mapped onto qubit registers via Jordan-Wigner transformation. The integrated two-particle correlation function C(t) is computed using an ancilla-based algorithm on a quantum circuit, and the difference C(t) - C_0(t) between interacting and non-interacting systems is related to the infinite-volume scattering phase shift through a weighted integral. The approach is demonstrated analytically for a contact interaction model in a periodic box, where the finite-volume correlation functions are computed exactly and then averaged over short time intervals to smooth out oscillatory finite-volume effects.

## Key numbers
- The demonstration uses a periodic box of size L = 90 (in natural units) with momentum cutoff N = 300 for the raw correlation function and N = 1000 for the averaged version.
- The contact interaction strength is set to V_0 = 2.5, with reduced mass mu = 1.
- The time averaging procedure uses t_0 = 2 and N_t = 20 segments, with each segment width Delta t = t_0 / N_t satisfying Delta t >> L / (2 pi N).
- The analytic infinite-volume limit of the correlation function difference is given by: C(t) - C_0(t) -> (1/2) erfc(mu V_0 sqrt(i t / (2 mu))) exp((mu V_0)^2 i t / (2 mu)) - 1/2.

## Context
This work extends the integrated correlation function formalism developed by Guo (2023, 2024) that relates trapped-system correlation functions to infinite-volume phase shifts. It contrasts with existing quantum computing approaches such as the Luscher formula-like method using Variational Quantum Eigensolver (PhysRevC 2024), wave packet time delay measurements (PhysRevD 2021), and ancillary spin-1/2 reconstruction (Mussardo 2024). The proposal enables direct computation of scattering phase shifts on quantum computers without extracting discrete eigenenergies, potentially avoiding some limitations of classical Monte Carlo methods such as sign problems and signal-to-noise ratio issues in lattice QCD.
