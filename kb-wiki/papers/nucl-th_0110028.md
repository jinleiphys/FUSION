# Covariant meson-exchange model of the anti-kaon nucleon interaction

> A covariant meson-exchange model of the anti-kaon nucleon interaction within the framework of the Bethe-Salpeter equation is presented. With just one free parameter we are able to get a good description of the available experimental data from below threshold to 300 MeV laboratory momentum.

## Key claim

A covariant meson-exchange model of the anti-kaon nucleon interaction, solved via the four-dimensional Bethe-Salpeter equation with a single adjustable cutoff mass parameter, yields a good description of experimental data from below threshold up to 300 MeV laboratory momentum, though the agreement with threshold branching ratios is only "reasonable" rather than exact.

## Method

The multi-channel Bethe-Salpeter equation was solved for the anti-kaon nucleon system including eight coupled channels (K- p, anti-K0 n, Lambda pi0, Sigma- pi+, Sigma0 pi0, Sigma+ pi-, Lambda eta, and Sigma0 eta). Interaction kernels were constructed from s- and u-channel baryon pole diagrams and t-channel vector meson pole diagrams using SU(3)-symmetric interaction Lagrangians, with all basic coupling constants fixed from external sources such as decay widths and vector meson dominance. A partial wave decomposition was applied, singularities in relative-energy variables were handled via a Wick rotation, and all propagators were multiplied by form factors with a common cutoff mass Lambda = 2.42 GeV as the only free parameter.

## Key numbers

- Cutoff mass Lambda = 2.42 GeV (the single free parameter of the model)
- K- p scattering length: a = -0.54 + i1.2 fm (compared to experimental value a = -0.78 +/- 0.18 + i(0.49 +/- 0.37) fm from kaonic hydrogen experiment by Iwasaki (1997))
- Threshold branching ratio gamma = 2.14 (experimental value 2.36 +/- 0.04)
- Threshold branching ratio R_c = 0.651 (experimental value 0.664 +/- 0.011)
- Threshold branching ratio R_n = 0.132 (experimental value 0.189 +/- 0.015)
- Coupling constant g_{pi pi rho} = 6.05 (from rho0 -> pi+ pi- decay width)
- Coupling constant g_{N N rho} = 2.52 (from rho0 -> e+ e- decay width)
- Pion-nucleon coupling f^2_{N N pi} / 4 pi = 0.075 (from nucleon-nucleon data)

## Context

This work extends the long history of nonperturbative anti-kaon nucleon models dating back to Dalitz, Wong, and Rajasekaran (1967), who used a coupled-channel Schrodinger equation with static vector meson exchange potentials to dynamically generate the Lambda(1405) resonance. Unlike previous approaches that relied on non-relativistic formulations or approximate solutions of the Bethe-Salpeter equation, this model provides a fully covariant treatment with a four-dimensional Bethe-Salpeter equation solved via Wick rotation. The results enable future extension to higher energies and searches for additional dynamically-generated S- and P-wave hyperon resonances.
