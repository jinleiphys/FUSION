# Testing and improvements of gamma-ray strength functions for nuclear model calculations of nuclear data

> A closed-form thermodynamic pole approach,TPA, is developed for average description of the E1 radiative strength functions using the microcanonical ensemble for initial states. A semiclassical description of the collective excitation damping in this method is based on modern physical notion on the relaxation processes in Fermi systems.The TPA model gives rather accurate means of simultaneous description of the gamma- decay and photoabsorption strength functions in the medium and heavy nuclei. It is able to cover a relatively wide energy interval, ranging from zeroth gamma-ray energy to values above GDR peak energy, as compared with the others closed-form models for calculation of the E1 strength.

## Key claim

The thermodynamic pole approach (TPA) provides a unified closed-form description of both gamma-decay and photoabsorption E1 strength functions in medium and heavy nuclei, covering a wider energy range (from zero gamma-ray energy to above the GDR peak) than other closed-form models, but the method violates the Brink hypothesis by making the strength function depend on excitation energy.

## Method

The TPA model derives the average radiative width using a microcanonical ensemble for initial states, connecting the gamma-decay strength function to the imaginary part of the dipole response function of the heated nucleus via the fluctuation-dissipation theorem. The damping width is decomposed into a collisional (two-body) component with memory effects and a fragmentation (one-body) component, both described semiclassically using an extended Steinwedel-Jensen hydrodynamic model with the Landau-Vlasov kinetic equation. The spectral function is approximated by the first term of an expansion, yielding a Lorentzian-like form with an energy- and temperature-dependent width.

## Key numbers

- The TPA spectral function uses a prefactor of \(8.674 \times 10^{-8}\) (in MeV\(^{-3}\)) multiplied by \(\sigma_r \Gamma_r\) and the energy-dependent width \(\Gamma(\epsilon_\gamma, \mathcal{T})\).
- The collisional relaxation time is given by \(\hbar / \tau_c(\epsilon_\gamma, T) = (T^2 / \alpha) [1 + (\epsilon_\gamma / 2\pi T)^2]\), with \(\alpha = 4.7 / F\) where the free-space parameter \(\alpha_{\text{free}} = 4.7\) MeV corresponds to \(\sigma_{\text{free}}(np) = 5\) fm\(^2\).
- The one-body (fragmentation) width is \(\Gamma_s = (m^*/m) k_s(\epsilon_\gamma) \Gamma_w\), with \(\Gamma_w = \hbar \bar{v} / R\) and \(k_s \approx 0.1\) when collective phonon energy exceeds nuclear binding energy, \(k_s \approx 0.7\) when it is negligible.
- The GDR energy \(E_r = \epsilon_{n=1}\) and the first expansion term uses \(z_1 = 2.08\), with parameters \(w_1 = 1\), \(x_1 = 1\) for the first resonance.
- The zero-energy limit of both gamma-decay and photoexcitation strength functions is \(\overleftarrow{f}_{E1}(0, T) = \overrightarrow{f}_{E1}(0, T) = 8.674 \times 10^{-8} q_{K \to \infty} \sigma_r \Gamma_r T \Gamma(0, T) / E_r^4\), where \(q_{K \to \infty} = 1.008656\).
- The EGLO model uses an empirical width \(\Gamma_k(\epsilon_\gamma, \mathcal{T})\) with a parameter \(\kappa_0(A)\) and \(\epsilon_0 = 4.5\) MeV.

## Context

This work extends the thermodynamic pole approximation (TPA) as an alternative to the standard Lorentzian (SLO) model and the enhanced generalized Lorentzian (EGLO) model recommended by the IAEA (RIPL database). The TPA model improves upon the SLO model, which strongly underestimates gamma-ray spectra at low energies below about 1 MeV (Pop 1982, Grudz 1999), and provides a more physically motivated description than the empirical EGLO approach (Kuc 1993, RIPL). The method enables unified calculations of gamma-decay and photoabsorption strengths for nuclear data applications, with further work needed to refine the dependence of the collisional and fragmentation width components on gamma-ray energy and mass number.
