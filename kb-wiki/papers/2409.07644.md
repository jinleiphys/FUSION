# Computing the QRPA Level Density with the Finite Amplitude Method

> We describe a new algorithm to calculate the vibrational nuclear level density of an atomic nucleus. Fictitious perturbation operators that probe the response of the system are generated by drawing their matrix elements from some probability distribution function. We use the Finite Amplitude Method to explicitly compute the response for each such sample. With the help of the Kernel Polynomial Method, we build an estimator of the vibrational level density and provide the upper bound of the relative error in the limit of infinitely many random samples. The new algorithm can give accurate estimates of the vibrational level density. Since it is based on drawing multiple samples of perturbation operators, its computational implementation is naturally parallel and scales like the number of available processing units.

## Key claim

The paper presents a new algorithm that can estimate the vibrational nuclear level density by using random excitation operators and the Finite Amplitude Method, without requiring explicit diagonalization of the QRPA matrix. The authors show that the estimator's relative error is bounded by a quantity that depends on the deviation of the Y-matrix column norms from their average, which is expected to be small except for a few low-lying modes.

## Method

The authors generate fictitious perturbation operators by drawing their matrix elements from a multivariate Gaussian distribution with zero mean and equal variance. For each random sample operator, they use the Finite Amplitude Method (FAM) to compute the linear response function without constructing the full QRPA matrix. They then apply the Kernel Polynomial Method to build an estimator of the vibrational level density from the collection of response functions, and derive an upper bound for the relative error in the limit of infinitely many random samples.

## Key numbers

* The QRPA matrix in deformed nuclei can reach dimensions of \(10^6 \times 10^6\) for realistic harmonic oscillator bases, making direct diagonalization computationally prohibitive.
* The variance of each transition matrix element component is \(\sigma_i^2 = \sigma^2 + 2\sigma^2 ||Y\boldsymbol{e}_i||^2\), where \(\sigma^2\) is the variance of the Gaussian-distributed matrix elements of the random excitation operator.
* The expected value of the squared transition strength for each mode is \(\mathbb{E}[|\langle i | \hat{F} | 0 \rangle|^2] = 4\sigma^2 \left( \frac{1}{2} + ||Y\boldsymbol{e}_i||^2 \right)\).
* The relative error at each eigenfrequency \(\Omega_i\) is \(\varepsilon_i = \left|\frac{ ||Y\boldsymbol{e}_i||^2 - \frac{||Y||_F^2}{N_p} }{ \frac{1}{2} + \frac{||Y||_F^2}{N_p} }\right|\), and the global relative error bound is \(\max_{i=1,\dots,N_p} \varepsilon_i\).

## Context

This work extends the combinatorial method for nuclear level densities, which was previously limited by the computational cost of constructing and diagonalizing the QRPA matrix in deformed nuclei (Hilaire 2001, Hilaire 2006, Goriely 2008, Hilaire 2023). The Finite Amplitude Method (FAM) developed by Nakatsukasa (2007) and Avogadro (2011) provided a matrix-free way to compute the QRPA linear response for a given operator, but did not give direct access to the full vibrational spectrum. The new algorithm enables efficient computation of vibrational level densities for arbitrarily large nuclei by combining FAM with random sampling and the Kernel Polynomial Method, with a computational cost that scales linearly with the number of available processing units.
