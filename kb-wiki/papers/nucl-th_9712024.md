# Polarization observables in the reaction $NN \to NN Φ$

> We study the reaction $NN \to NN Φ$ slightly above the threshold within an extended one-boson exchange model which also accounts for $uud$ knock-out. It is shown that polarization observables, like the beam-target asymmetry, are sensible quantities for identifying a $s \bar s$ admixture in the nucleon wave function on the few per cent level.

# Polarization observables in the reaction \(NN \to NN \Phi\)

## Key claim

Polarization observables such as the beam-target asymmetry are sensitive to a few-percent-level \(s \bar s\) admixture in the nucleon wave function, but only at finite energies above threshold; the threshold prediction of \(C^{BT}_{zz}=1\) for \(pp\) collisions is strongly modified by interaction dynamics already at \(\Delta \sim 0.1\) GeV.

## Method

The reaction \(NN \to NN \Phi\) is studied slightly above threshold using an extended one-boson exchange (OBE) model that includes conventional meson-exchange diagrams (with \(\pi\), \(\rho\), \(\sigma\), \(\omega\) exchange) and a \(uud\) knock-out mechanism, where the exchanged meson interacts with the \(uud\) cluster of a five-quark nucleon wave function containing an \(s \bar s\) component. The OBE parameters are fixed by fitting the \(\pi^- p \to n \Phi\) reaction, and two parameter sets (set I from the Bonn OBE model, set II from Giessen) are employed. The beam-target asymmetry \(C^{BT}_{zz}\) and \(\Phi\) spin density matrix elements are computed for both \(pp\) and \(pn\) collisions as functions of the \(\Phi\) production angle at \(\Delta = 0.1\) GeV.

## Key numbers

* Beam-target asymmetry at threshold: \(C^{BT}_{zz} = 1\) for \(pp\) collisions, \(C^{BT}_{zz} = 0.8\) for \(pn\) collisions.
* Ratio of total cross sections at threshold: \(\sigma_{np}^{tot} / \sigma_{pp}^{tot} = 5\).
* Coupling constant \(g_{\Phi\rho\pi} = 1.10\) GeV\(^{-1}\) from the \(\Phi \to \rho\pi\) decay width \(\Gamma(\Phi\to\rho\pi) = 0.69\) MeV.
* Cut-off parameter \(\Lambda^\rho_{\Phi\rho\pi} = 1.9\) GeV for set I; set II uses \(\Lambda^\rho_{\Phi\rho\pi} = \infty\).
* For \(B^2 = 1\%\) strangeness probability, the total cross section difference between conventional OBE and knock-out channels is about two orders of magnitude at \(\Delta \simeq 0.1\) GeV; in forward/backward directions the difference reduces to a factor of 5–8.
* \(\Phi\) spin density matrix elements for the sum of channels at finite energy: \(\rho_{00} = 0.12\) (0.22), \(\rho_{11} \simeq 0.44\) (0.39) for \(B^2 = 2\%\) (5\%) in backward and forward directions.

## Context

This work extends the threshold analysis of Rekalo (1995) for \(pp \to pp\Phi\), which predicted \(C^{BT}_{zz}=1\) and \(\rho_{00}=0\) at threshold, by computing finite-energy modifications and including a strangeness admixture in the nucleon wave function. The knock-out mechanism is analogous to that used in \(\Phi\) photoproduction by Titov (1996) and Henley (1996). The results enable experimental searches for hidden strangeness in the nucleon using polarization measurements at facilities such as ANKE at COSY, where precision polarization observables are envisaged.
