# Probing the isoscalar excitations of 12C with inelastic alpha scattering

> The robust (spin and isospin zero) $α$-particle remains one of the best projectiles to probe the nuclear isoscalar excitations. In the present work, a microscopic folding model analysis of the \ac inelastic scattering to the 2$^+$ (4.44 MeV), 0$^+$ (7.65 MeV), 3$^-$ (9.64 MeV), 0$^+$ (10.3 MeV) and 1$^-$ (10.84 MeV) states in $^{12}$C has been performed using the 3-$α$ resonating group method wave functions. The isoscalar transition strengths of these states were carefully studied based on the coupled-channel analysis using the microscopic folded form factors. A correlation between the weak binding and/or short lifetime of the excited state and absorption in the exit channel of inelastic scattering has been established.

## Key claim

The paper demonstrates that the isoscalar transition strengths of excited states in 12C, as extracted from inelastic alpha scattering, are systematically suppressed when using conventional distorted-wave Born approximation (DWBA) or coupled-channel (CC) analysis, and that this suppression can be resolved by introducing enhanced absorption in the exit channel that correlates with the weak binding and short lifetime of each excited state.

## Method

The author performed a microscopic double-folding model analysis of inelastic α+12C scattering at 240 MeV, using complex density-dependent CDJLM nucleon-nucleon interactions and 3-α resonating group method (RGM) wave functions from Kamimura (1981) to generate nuclear densities and transition form factors. Coupled-channel calculations were carried out with the code ECIS97, including 23 diagonal and nondiagonal transition densities, treating both direct and two-step excitations (e.g., 0+1 → 2+ → 0+2) on equal footing. To reconcile the inelastic scattering data with electron scattering data, an enhanced imaginary surface term was added to the optical potential in the exit channels of weakly-bound, short-lived states.

## Key numbers

- For the 2+ state (4.44 MeV), the RGM calculation gives B(E2) = 46.5 e²fm⁴ versus experimental B(E2) = 40 ± 4 e²fm⁴, with lifetime τ = 6×10⁻¹⁴ s, RMS radius 2.38 fm, scaling factor Sρ = 1.00, and absorption ratio SW = 1.00.
- For the 0+ Hoyle state (7.65 MeV), the RGM calculation gives B(E0) = 6.62 e²fm⁴ versus experimental B(E0) = 5.4 ± 0.2 e²fm⁴, with lifetime τ = 8×10⁻¹⁷ s, RMS radius 3.47 fm, Sρ = 0.55, and SW = 3.27.
- For the 3- state (9.64 MeV), the RGM calculation gives B(E3) = 872 e²fm⁶ versus experimental B(E3) = 610 ± 90 e²fm⁶, with lifetime τ = 2×10⁻²⁰ s, RMS radius 2.77 fm, Sρ = 0.75, and SW = 1.75.
- For the 0+ state (10.3 MeV), the RGM calculation gives B(E0) = 6.33 e²fm⁴, with lifetime τ = 2×10⁻²² s, RMS radius 3.14 fm, Sρ = 0.45, and SW = 4.85.
- For the 1- state (10.84 MeV), the RGM calculation gives B(E1) = 4.16 e²fm⁶, with lifetime τ = 2×10⁻²¹ s, RMS radius 3.39 fm, Sρ = 0.20, and SW = 8.60.
- The Hoyle state's E0 transition strength exhausts 15-23% of the isoscalar monopole energy-weighted sum rule (EWSR) from electron scattering, but only about 6.9% from conventional DWBA analysis of inelastic alpha scattering.

## Context

This work extends the folding model analysis of Khoa (2008) for the Hoyle state to include other isoscalar excitations of 12C (2+, 3-, 0+ at 10.3 MeV, and 1-), using the same RGM wave functions from Kamimura (1981). It directly addresses the discrepancy between inelastic alpha scattering results (Johnston et al., 2003) and electron scattering data (Strehl, 1970; Cramell et al., 2005; Lenecek et al., 2006) for transition strengths, particularly the Hoyle state's monopole strength. The finding that enhanced absorption in the exit channel correlates with weak binding and short lifetime provides a systematic correction mechanism for future studies of alpha-cluster states excited in inelastic alpha-nucleus or nucleus-nucleus scattering.
