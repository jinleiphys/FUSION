# Estimating theoretical uncertainties of the two-nucleon observables by using backpropagation

> We present a novel approach to calculating theoretical uncertainties in few-nucleon calculations, making use of automatic differentiation via backpropagation, which is particularly efficient when there are many input variables but only a few outputs. The methods described in this paper constitute tools that can be used to investigate the properties of scalar functions used to define nuclear potentials and quantify their contribution to the uncertainty of few nucleon calculations. We demonstrate these methods in deuteron bound state and nucleon - nucleon scattering calculations. Backpropagation, implemented in the Python pytorch library, is used to calculate the gradients with respect to model parameters and propagate errors from these parameters to the deuteron binding energy and selected phase-shift parameters. The uncertainty values obtained using this approach are validated by directly sampling from the potential parameters. We find very good agreement between two ways of estimating that uncertainty.

## Key claim

The authors demonstrate that automatic differentiation via backpropagation, implemented in the PyTorch library, can efficiently propagate uncertainties from two-nucleon potential parameters to deuteron binding energy and nucleon-nucleon phase shifts, but only under the assumption of uncorrelated errors with standard deviations proportional to the absolute values of the scalar functions defining the potential.

## Method

The authors implemented deuteron bound state calculations (solving the Schrödinger equation in integral form) and nucleon-nucleon scattering calculations (solving the Lippmann-Schwinger equation) entirely within the PyTorch library, using its automatic differentiation capability to compute gradients of observables with respect to the scalar functions that define the two-nucleon potential. The partial wave decomposition of the Bonn-B potential was performed numerically, and uncertainties were propagated using the standard error formula from gradients, assuming uncorrelated errors proportional to the absolute values of the scalar function values. The results were validated by direct Monte Carlo sampling of 100 random potential parameter sets.

## Key numbers

- Deuteron binding energy for the Bonn-B potential: -2.2272 MeV (consistent across all relative error values from 0.01 to 0.1)
- For relative error α=0.05 on all scalar functions, deuteron binding energy uncertainty: 0.0052 MeV
- For relative error α=0.1 on all scalar functions, deuteron binding energy uncertainty: 0.0104 MeV
- For the ¹S₀ channel at p₀=2 fm⁻¹ with α=0.1: phase shift -11.8988 degrees, backpropagation uncertainty 3.38991 degrees, sampling uncertainty 3.46838 degrees (100 samples)
- For the ¹P₁ channel at p₀=2 fm⁻¹ with α=0.1: phase shift -30.1 degrees, backpropagation uncertainty 4.8 degrees, sampling uncertainty 4.6 degrees
- For the ¹P₁ channel at p₀=0.5 fm⁻¹ with α=0.05: backpropagation uncertainty 3225.6 degrees versus sampling uncertainty 0.4 degrees, indicating numerical sensitivity
- Kolmogorov-Smirnov test p-values for normality of sampled eigenvalues: 0.955 (α=0.2 on one scalar function), 0.382 (α=0.4 on one scalar function), 0.424 (α=0.05 on all scalar functions)

## Context

This work extends previous uncertainty quantification studies for few-nucleon systems, particularly the statistical sampling approaches used by the Bochum group for the chiral SMS potential (Reinert, 2018) and by Skibiński (2018) and Volkotrub (2020) for nucleon-deuteron scattering. The key advance is replacing the computationally expensive direct sampling method (which requires solving the quantum mechanical problem many times) with a single calculation plus gradient backpropagation, which is especially efficient when there are many input variables but few outputs. This enables future work to incorporate correlated errors between potential parameters and to extend the method to three-nucleon systems, where the computational cost of direct sampling becomes prohibitive.
