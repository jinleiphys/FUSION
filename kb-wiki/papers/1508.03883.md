# ND^(*) and NB^(*) interactions in a chiral quark model

> ND and ND^* interactions become a hot topic after the observation of new charmed hadrons Σ_c(2800) and Λ_c(2940)^+. In this letter, we have preliminary investigated S-wave ND and ND^* interactions with possible quantum numbers in the chiral SU(3) quark model and the extended chiral SU(3) quark model by solving the resonating group method equation. The numerical results show that the interactions between N and D or N and D^* are both attractive, which are mainly from σexchanges between light quarks. Further bound-state studies indicate the attractions are strong enough to form ND or ND^* molecules, except for (ND)_{J=3/2} and (ND^*)_{J=3/2} in the chiral SU(3) quark model. In consequence ND system with J=1/2 and ND^* system with J=3/2 in the extended SU(3) quark model could correspond to the observed Σ_c(2800) and Λ_c(2940)^+, respectively. Naturally, the same method can be applied to research NB and NB^* interactions, and similar conclusions obtained, i.e. NB and NB^* attractive forces may achieve bound states, except for (NB^*)_{J=3/2} in the chiral SU(3) quark model. Meanwhile, S partial wave phase shifts of ND^{(*)} and NB^{(*)} elastic scattering are illustrated, which are qualitatively consistent with results from bound state problem.

# ND^(*) and NB^(*) interactions in a chiral quark model

## Key claim
The observed Σ_c(2800) and Λ_c(2940)^+ may be explained as ND molecular state with J=1/2 and ND^* molecular state with J=3/2, respectively, but only in the extended chiral SU(3) quark model; in the simpler chiral SU(3) quark model these specific channels do not form bound states.

## Method
The authors solved the resonating group method (RGM) equation for S-wave ND, ND^*, NB, and NB^* systems in both the chiral SU(3) quark model and the extended chiral SU(3) quark model. The Hamiltonian includes confinement, one-gluon-exchange (OGE), and chiral field coupling terms, with Goldstone boson exchanges between light quarks only; heavy-light quark pairs receive only confinement and OGE contributions. Binding energies were computed from the RGM bound-state problem, and elastic scattering S-wave phase shifts were obtained from the same framework.

## Key numbers

- (ND)_{J=1/2} binding energy: 10.0 MeV in extended chiral SU(3) quark model set II, 5.9 MeV in set III; unbound in chiral SU(3) quark model set I
- (ND^*)_{J=3/2} binding energy: 6.6 MeV in extended chiral SU(3) quark model set II, 3.2 MeV in set III; unbound in set I
- (ND^*)_{J=1/2} binding energy: 2.0 MeV in set I, 18.1 MeV in set II, 12.8 MeV in set III
- (NB)_{J=1/2} binding energy: 1.1 MeV in set I, 18.2 MeV in set II, 12.8 MeV in set III
- (NB^*)_{J=1/2} binding energy: 7.7 MeV in set I, 28.6 MeV in set II, 22.0 MeV in set III
- (NB^*)_{J=3/2} binding energy: unbound in set I, 13.7 MeV in set II, 8.9 MeV in set III
- Heavy quark masses used: m_c = 1430 MeV, m_b = 4720 MeV
- σ meson mass in chiral SU(3) quark model: 595 MeV; in extended model set II: 535 MeV; set III: 547 MeV

## Context
This work extends the chiral SU(3) quark model and extended chiral SU(3) quark model, previously successful for light-flavor baryon-baryon, meson-baryon, and meson-meson systems, into the heavy-flavor sector. It follows earlier applications of these models to DK, D\bar{D}, B\bar{B}, Σ_c\bar{D}, and Λ_c\bar{D} interactions, and to the structures of X(3872), Z_b(10610), and Z_b(10650). The results provide a molecular interpretation for the charmed hadrons Σ_c(2800) and Λ_c(2940)^+ observed by Belle (2005) and Babar (2007), and predict analogous NB and NB^* bound states in the bottom sector, enabling future experimental searches for bottom analogs.
