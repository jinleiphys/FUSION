# Analysis of the 16O(p,gamma)17F capture reaction using the shell model embedded in the continuum

> We apply the realistic shell model which includes the coupling between many-particle (quasi-)bound states and the continuum of one-particle scattering states, called the Shell Model Embedded in the Continuum, to the spectroscopy of mirror nuclei: 17F and 17O, as well as to the description of low energy cross section (the astrophysical S-factor) for E1, E2 and M1 components in the capture reaction: 16O(p,gamma)17F. With the same microscopic input we calculate the phase shifts and differential cross-section for the elastic scattering of protons on 16O.

# Analysis of the 16O(p,gamma)17F capture reaction using the shell model embedded in the continuum

## Key claim

The Shell Model Embedded in the Continuum (SMEC) can describe the low-energy cross section (astrophysical S-factor) for the 16O(p,gamma)17F capture reaction and the spectroscopy of mirror nuclei 17F and 17O using the same microscopic input, but only when the ground state of 16O is approximated as a pure 0p-0h configuration and the positive parity states in 17F and 17O are treated as pure one-particle configurations, with matrix elements of the residual coupling quenched accordingly.

## Method

The authors construct a realistic shell model in the 0p1s0d model space using the CK(8-16) interaction for the 0p shell, the Brown-Wildenthal interaction for the 1s0d shell, and the Kahana-Lee-Scott G matrix for cross-shell matrix elements, with monopole adjustments to reproduce spectra of 15N, 15O, 17O, and 17F. They then couple these shell model states to the one-particle scattering continuum through the SMEC formalism, solving coupled channel equations with self-consistent average potentials for each channel, using either a Wigner-Bartlett (WB) zero-range force or a density-dependent interaction (DDSM1) for the residual coupling between bound and scattering states. The radiative capture cross sections for E1, E2, and M1 transitions are calculated from the resulting SMEC wave functions.

## Key numbers

- The difference of single-particle energies epsilon_p(1s_{1/2}) - epsilon_p(0d_{5/2}) = 0.495 MeV in 17F and epsilon_n(1s_{1/2}) - epsilon_n(0d_{5/2}) = 0.871 MeV in 17O, reflecting the Thomas-Ehrman shift
- The equivalent potential depth for both WB and DDSM1 residual interactions is V_0 = -52.46 MeV in 17F and V_0 = -52.49 MeV in 17O
- Common Woods-Saxon parameters: radius R_0 = 3.214 fm, diffuseness a = 0.58 fm, spin-orbit strength V_SO = 3.683 MeV
- The DDSM1 force parameters: v_00^{in} = 36.45, v_00^{ex} = -297.9, v_01^{in} = 109.4, v_01^{ex} = 115.5, all in units MeV*fm^3, reduced by a constant factor 0.68 compared to the Schwesinger-Wambach interaction
- The WB residual interaction overall strength V_12^{(0)} = 300 MeV*fm^3 with spin exchange parameter (1 - alpha) = 0.27
- The 1s_{1/2} state in 17F has a binding energy Q = 105 keV relative to the 16O + p threshold

## Context

This work extends the SMEC formalism previously applied by Bennaceur, Nowacki, Okolowicz, and Ploszajczak to the mirror reactions 7Be(p,gamma)8B and 7Li(n,gamma)8Li (1999) to the 16O(p,gamma)17F reaction, which is important for CNO cycle nucleosynthesis in hydrogen-burning stars. The paper addresses the strikingly different behavior of the astrophysical S-factor for capture into the 5/2+ ground state versus the weakly bound 1/2+ state (identified as a proton halo state by Morlock et al., 1997), which previous approaches including the potential model (Morlock, Rolfs), the Generator Coordinate Method (Baye, Descouvemont, Hesse, 1998), and K- and R-matrix analysis (Brune) had attempted to describe. The SMEC approach enables a unified microscopic description of both bound state spectroscopy and continuum capture reactions, though the authors note that at higher excitation energies above the alpha emission threshold, the one-particle continuum approximation becomes too restrictive and methods based on cluster expansion or three-body continuum models (Descouvemont, 2000; Grigorenko, 2000) would be needed.
