# Deuteron disintegration in three dimensions

> We compare results from the traditional partial wave treatment of deuteron electro-disintegration with a new approach that uses three dimensional formalism. The new framework for the two-nucleon (2N) system using a complete set of isospin - spin states made it possible to construct simple implementations that employ a very general operator form of the current operator and 2N states.

## Key claim

The paper demonstrates that a three-dimensional (3D) operator formalism for deuteron electro-disintegration yields results that agree with traditional partial wave calculations, but the 3D approach is simpler to implement numerically because it avoids the need for partial wave decomposition of the current operator and uses direct matrix representations in a 16-dimensional isospin-spin space.

## Method

The authors constructed a 3D formalism for the two-nucleon (2N) system using a complete set of isospin-spin states, representing all operators and states as 16-dimensional vectors and 16x16 matrices via Kronecker products. They applied this formalism to electron-induced deuteron disintegration (e + 2H → e + p + n) in the non-relativistic approximation, using a chiral NNLO potential with Λ = 550 MeV/c and Λ̃ = 600 MeV/c, and compared the resulting observables with traditional partial wave calculations.

## Key numbers

- The chiral NNLO potential used had cutoff parameters Λ = 550 MeV/c and Λ̃ = 600 MeV/c.
- The half-shell t-matrix was tested at p^f ≈ 1.9 fm^{-1}, corresponding to a 2N center-of-mass energy E_{2N} = 150 MeV.
- The δ_{p^f} used for averaging the t-matrix at the on-shell point was approximately 0.01 fm^{-1}.
- The deuteron wave function components φ_1(p) and φ_2(p) were shown for relative momenta up to about 6 fm^{-1}, with perfect agreement between the 3D operator approach and standard partial wave decomposition for both s-wave and d-wave components.

## Context

This work extends the 3D formalism previously developed for the 2N t-matrix and deuteron bound state (Topolnicki et al., 2011; Golak et al., 2010) to include electromagnetic currents, filling a gap in the 3D framework for electroweak processes. The approach enables simpler numerical implementations for processes such as muon capture or neutrino-induced deuteron disintegration, which have been performed recently using partial wave expansions (Marcucci, 2010; Shen, 2012). The method can be generalized to any operators acting on the same degrees of freedom, making it applicable to a wide class of nuclear reactions.
