# Vector interaction bounds in NJL-like models from LQCD estimated curvature of the chiral crossover line

> We obtain improved bounds on both the flavor-independent and -dependent vector interactions in a $2+1$-flavor Nambu\textendash Jona-Lasinio (NJL) model using the latest precise LQCD results of the curvature coefficients of the chiral crossover line. We find that these lattice estimated curvature coefficients allow for both attractive and repulsive types of interactions in both the cases. With this constrained ranges of vector interactions, we further predict the behavior of the second $(κ_2^B)$ and fourth $(κ_4^B)$ order curvature coefficients as a function of the strangeness chemical potential $(μ_S)$. We observe that the flavor mixing effects, arising from the flavor-independent vector interaction as well as from the 't Hooft interaction, play an important role in $k_2^B$. We propose that the mixing effects due to the vector interaction can be separated from those arising from the 't Hooft interaction by analyzing the behavior of $k_2^B$ as a function of $μ_S$. Finally, we locate the critical endpoint in the $T-μ_B$ plane using the model-estimated ranges of vector interactions and find the model's predictions to be consistent with the latest LQCD bounds.

## Key claim

The paper uses precise lattice QCD curvature coefficients to constrain both attractive and repulsive vector interaction strengths in a 2+1-flavor NJL model, finding that the allowed ranges accommodate both signs of the interaction, contrary to earlier studies that predominantly favored repulsive interactions.

## Method

The authors employ a 2+1-flavor Nambu-Jona-Lasinio (NJL) model with two types of vector interactions: flavor-independent (coupling G_V) and flavor-dependent (coupling g_V). They compute the second-order curvature coefficient κ₂ᴮ of the chiral crossover line as a function of the vector coupling strengths and compare these results with lattice QCD data from the HotQCD Collaboration (2018) for both the μ_S = 0 case and the strangeness neutrality condition (n_s = 0). The allowed ranges of G_V/G_S and g_V/G_S are determined by the intersection of the model predictions with the lattice QCD bands.

## Key numbers

- The allowed range for G_V/G_S from κ₂ᴮ at μ_S = 0 is [-0.117, 0.212] using parameter Set I.
- The allowed range for G_V/G_S from κ₂ᴮ under strangeness neutrality (n_s = 0) is [-0.038, 0.334] using parameter Set I.
- The allowed range for g_V/G_S from κ₂ᴮ at μ_S = 0 is [-0.296, 0.552] using parameter Set I.
- The allowed range for g_V/G_S from κ₂ᴮ under strangeness neutrality (n_s = 0) is [-0.076, 0.667] using parameter Set I.
- Without vector interactions, κ₂ᴮ at μ_S = 0 is 0.0163 for Set I and 0.0162 for Set II, compared to the lattice QCD value of 0.016(6) from HotQCD (2018).
- Without vector interactions, κ₂ᴮ under strangeness neutrality is 0.0148 for Set I and 0.0143 for Set II, compared to the lattice QCD value of 0.012(4) from HotQCD (2018).

## Context

This work extends earlier studies by Kashiwa (2011), Bratovic (2012), Contrera (2012), Friesen (2014), Steinheimer (2014), and Contrera (2016) that used less precise lattice QCD curvature data and predominantly favored repulsive vector interactions. The improved lattice QCD results from HotQCD (2018) allow for both attractive and negative vector couplings. The authors build on their own recent work (Ali 2024) which showed that curvature coefficients in the NJL model align well with lattice QCD and proposed using the μ_S dependence of κ₂ᴮ to distinguish flavor mixing from the 't Hooft interaction versus vector interaction effects. The constrained vector interaction ranges enable predictions for the critical endpoint location in the T-μ_B plane, which the authors find consistent with lattice QCD bounds from Borsanyi (2021) and Philipsen (2021).
