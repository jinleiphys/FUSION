# Preliminary study of the anti-K N interaction in a chiral constituent quark model

> A preliminary investigation of the anti-K N interaction is performed within a chiral constituent quark model by solving the resonating group method (RGM) equation. The model parameters are taken from our previous work, which gave a satisfactory description of the S-, P-, D-, F-wave KN scattering phase shifts. The channel-coupling between anti-K N, pi Lambda and pi Sigma is considered, and the scattering phase shifts as well as the bound-state problem of anti-K N are dynamically studied. The results show that the S-wave anti-K N interaction in the isospin I=0 channel is attractive, and in the extended chiral SU(3) quark model such an attraction can make for an anti-K N bound state, which appears as a pi Sigma resonance in the coupled-channel calculation, while the chiral SU(3) quark model cannot accommodate the existence of an anti-K N bound state. It seems that the vector meson exchanges are necessary to be introduced in the quark-quark interactions if one tries to explain the Lambda(1405) as an anti-K N bound state or a pi Sigma - anti-K N resonance state.

## Key claim

The extended chiral SU(3) quark model, which includes vector meson exchanges, can produce a bound state in the S-wave anti-K N system with isospin I=0 that appears as a pi Sigma resonance in coupled-channel calculations, while the original chiral SU(3) quark model without vector meson exchanges cannot accommodate such a bound state; this suggests vector meson exchanges are necessary if one tries to explain the Lambda(1405) as an anti-K N bound state or a pi Sigma - anti-K N resonance state.

## Method

The authors solved the resonating group method (RGM) equation for the anti-K N system within both the chiral SU(3) quark model and the extended chiral SU(3) quark model, using model parameters previously fitted to S-, P-, D-, and F-wave KN scattering phase shifts. Channel coupling between anti-K N, pi Lambda, and pi Sigma was included, and both scattering phase shifts and the bound-state problem were studied dynamically. The quark-quark interaction in the chiral SU(3) quark model includes confinement, one gluon exchange (OGE), and scalar and pseudoscalar boson exchanges, while the extended model additionally includes vector meson exchanges, with the OGE coupling constants greatly reduced.

## Key numbers

- In the extended chiral SU(3) quark model, the pi Sigma and anti-K N bound states have binding energies of about 4 MeV and 12 MeV, respectively.
- The harmonic oscillator width parameter for the chiral SU(3) quark model is b_u = 0.5 fm; for the extended chiral SU(3) quark model it is b_u = 0.45 fm.
- The constituent quark masses used are m_u(d) = 313 MeV and m_s = 470 MeV.
- The OGE coupling constant g_u^2 is 0.7704 in the chiral SU(3) quark model and 0.0748 in the extended chiral SU(3) quark model; g_s^2 is 0.5525 and 0.0001, respectively.
- The anti-K N phase shifts in the figures start at E_c.m. = 102 MeV.

## Context

This work extends the authors' previous studies of KN scattering phase shifts (Huang et al., 2005) and NN, YN interactions using the same chiral quark models. It directly contrasts the predictions of the chiral SU(3) quark model and the extended chiral SU(3) quark model for the anti-K N system, finding that only the extended model with vector meson exchanges produces bound states, similar to results from effective chiral Lagrangian approaches (Kaiser et al., 1995) where the Weinberg-Tomozawa term (equivalent to vector meson exchange) produces attraction in the anti-K N channel. The results suggest that including a genuine three-quark component coupled to the pi Sigma - anti-K N channels could produce a resonance near the Lambda(1405), as proposed by Takeuchi and Shimizu (2007), and the authors plan further work along this line.
