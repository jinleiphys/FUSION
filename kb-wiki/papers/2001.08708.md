# Strong decays of strange quarkonia in a corrected 3P0 model

> Extensively applied to both light and heavy meson decay and standing as one of the most successful strong decay models is the $^{3}P_{0}$ model, in which $q\bar{q}$ pair production is the dominant mechanism. In this paper we evaluate strong decay amplitudes and partial widths of strange $S$ and $D$ state mesons, namely $ ϕ(1020)$, $ϕ(1680) $, $ ϕ(2050) $, $ϕ_1 (1850) $, $ϕ_2 (1850) $ and $ ϕ_3 (1850) $, in the bound-state corrected $^{3}P_{0}$ decay model (C$^{3}P_{0}$). The C$^{3}P_{0}$ model is obtained in the context of the Fock-Tani formalism, which is a mapping technique.

## Key claim

The bound-state corrected 3P0 model (C3P0), derived from the Fock-Tani formalism, reproduces the standard 3P0 model results at lowest order but adds corrections from the bound-state kernel that modify the pair-creation strength; for strange quarkonia decays, these corrections reduce the overestimation of widths for higher-L states that occurs in the standard 3P0 model with parameters optimized for light 1S and 1P decays.

## Method

The authors start from a microscopic quark-antiquark pair-creation Hamiltonian and apply the Fock-Tani unitary mapping transformation to obtain an effective decay Hamiltonian in an ideal-meson Fock space. The resulting C3P0 Hamiltonian contains a leading term identical to the standard 3P0 model plus bound-state correction terms involving the kernel Delta, which sums over intermediate meson states (eta, eta-prime, phi, omega). Decay amplitudes and partial widths are then computed for selected strangeonium resonances by evaluating the matrix elements of this effective Hamiltonian between initial and final ideal-meson states, using harmonic-oscillator wave functions and integrating over phase space.

## Key numbers

* For phi(1020) decaying to KK, the C3P0 model yields a width of 4.2 MeV (standard 3P0 gives 4.4 MeV; experimental value is 4.3 MeV).
* For phi(1680) decaying to KK, the C3P0 width is 0.8 MeV (standard 3P0: 1.0 MeV); for KK* the C3P0 width is 22.5 MeV (standard 3P0: 25.0 MeV); for eta phi the C3P0 width is 1.2 MeV (standard 3P0: 1.4 MeV).
* For phi(2050) decaying to KK, the C3P0 width is 2.1 MeV (standard 3P0: 2.5 MeV); for KK* the C3P0 width is 8.3 MeV (standard 3P0: 9.5 MeV); for K*K* the C3P0 width is 12.0 MeV (standard 3P0: 14.0 MeV).
* For phi_2(1850) decaying to KK, the C3P0 width is 0.3 MeV (standard 3P0: 0.4 MeV); for KK* the C3P0 width is 5.1 MeV (standard 3P0: 6.0 MeV); for K*K* the C3P0 width is 3.8 MeV (standard 3P0: 4.5 MeV); for eta phi the C3P0 width is 0.2 MeV (standard 3P0: 0.3 MeV).
* For phi_3(1850) decaying to KK, the C3P0 width is 0.1 MeV (standard 3P0: 0.1 MeV); for KK* the C3P0 width is 2.0 MeV (standard 3P0: 2.4 MeV); for K*K* the C3P0 width is 1.5 MeV (standard 3P0: 1.8 MeV); for KK_1(1270) the C3P0 width is 0.5 MeV (standard 3P0: 0.6 MeV); for eta phi the C3P0 width is 0.1 MeV (standard 3P0: 0.1 MeV).

## Context

This work extends the systematic survey of strange meson decays performed by Barnes et al. (2003) using the standard 3P0 model, which found that parameters optimized for light 1S and 1P states overestimate widths of higher-L states. The C3P0 model provides a formal derivation of bound-state corrections that naturally reduce these overestimates, bringing predictions closer to experimental data where available. The results enable more reliable predictions for the strangeonium spectrum up to 2.2 GeV, which is relevant for interpretation of data from the GlueX experiment at Jefferson Lab.
