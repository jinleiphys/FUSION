# Computing Scattering Cross Sections For Spherically Symmetric Potentials

> This paper introduces students and instructors to the use of Scilab for calculating phase shifts from phenomenological potentials in nuclear, atomic, and molecular scattering, beginning with a Riccati-type nonlinear differential equation derived from the time-independent Schr$\ddot{\text{o}}$dinger equation. For spherically symmetric potentials, this equation can be easily solved using Xcos, a Scilab toolbox. \cite{PRC}\cite{bobby} Scilab is open source software that is used for numerical computations and can be freely downloaded for Windows, Linux and Mac OS.\cite{scilab} Xcos is a Scilab toolbox dedicated to the modeling and simulation of hybrid dynamic systems \cite{rachna}. Xcos provides users with the ability to model complex dynamical systems using a block diagram editor (GUI based), offering a step-by-step procedure for learning to solve complex differential equations interactively.

# Computing Scattering Cross Sections For Spherically Symmetric Potentials

## Key claim

The paper demonstrates that Scilab's Xcos graphical toolbox can solve the Riccati-type nonlinear differential equation for phase shifts from spherically symmetric potentials, but only for potentials that become negligible at large distances; the method becomes ill-defined for long-range potentials such as the unscreened Coulomb interaction, and requires careful numerical treatment for higher partial waves (typically ℓ > 6–8).

## Method

The authors transform the time-independent Schrödinger equation for a particle of wavenumber *k* and orbital angular momentum ℓ into a first-order nonlinear Riccati-type phase function equation (Eq. PFMeqn). They implement this equation in Scilab's Xcos block-diagram environment to compute the phase shift δℓ(k,r) as a function of radial distance *r* for several spherically symmetric potentials, obtaining the asymptotic phase shift δ(k) as r → ∞. The method is applied to Kronig-Penney type periodic potentials, attractive Yukawa, repulsive Yukawa, and a combination of attractive and repulsive Yukawa potentials.

## Key numbers

- Attractive Yukawa potential: V(r) = −(10/r)e^(−r) (strength 10, range parameter 1); phase shifts computed for ℓ = 0 and ℓ = 2 at k = 0.1, 1, 10, and 100.
- Repulsive Yukawa potential: V(r) = +(20/r)e^(−2r) (strength 20, range parameter 2); ℓ = 0 phase shifts computed at multiple k values.
- Combined attractive+repulsive Yukawa: V(r) = −(10/r)e^(−r) + (20/r)e^(−2r); phase shifts computed for ℓ = 0, 2, 4, and 6.
- For the attractive Yukawa with ℓ = 0, the phase shift reaches a constant value for r ≥ 2.5 fm.
- For the combined potential with ℓ = 0 and k = 0.02 and 0.05, the phase shift is negative for r < 0.7 and positive for r > 0.7.
- The VPA method encounters limitations for partial waves with ℓ > 6–8 due to complexity of Bessel functions.

## Context

This work extends the Variable Phase Approach (VPA), originally proposed by Morse and Allis and later developed by Calogero (Calogero, 1967), which has been applied to nuclear scattering reactions such as np and α−α scattering (Khachi, 2015; Khachi, 2017). The paper provides a pedagogical implementation using the open-source Scilab/Xcos environment, contrasting with traditional methods that require solving the Schrödinger equation in interior and exterior regions and matching boundary conditions (Griffiths, 2005; Erich, 2010), or using the Numerov method which requires prior knowledge of the wavefunction. The approach enables students to interactively observe how different potential shapes affect phase shift accumulation, and can be extended to compute scattering lengths and energy-dependent phase shifts.
