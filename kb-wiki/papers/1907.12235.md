# Structure of 16C from analyses of proton scattering data and the B(E2) problem

> The B(E2) value for the decay of the 2+_1 state in 16C to ground has been the subject of much discussion. Analyses assuming a simple model of two neutrons coupled to a 14C core of the available data, which extend over an order of magnitude,gives reasonable agreement but with an inclusion of a large effective charge for the neutrons. To assess this situation, a large scale (2+4)hw shell model calculation of 16C has been made from which the wave functions have been used to obtain the B(E2) value. As a check, comparison is made with available data on the spectrum of 16C and intermediate-energy elastic proton scattering.

## Key claim

The large-scale $(2+4)\hbar\omega$ shell model calculation of $^{16}$C yields a $B(E2)$ value of 1.35 $e^2$fm$^4$ using bare operators, which falls within the wide range of experimental values (0.63 to 4.15 $e^2$fm$^4$), and requires a much smaller effective neutron charge (0.09e to 0.12e) than the approximately 0.4e needed in simpler models assuming a $\nu(sd)^2$ configuration coupled to a $^{14}$C core.

## Method

A no-core $(2+4)\hbar\omega$ shell model calculation was performed for the positive parity states of $^{16}$C using a single-particle basis spanning six major shells from $0s_{1/2}$ to $0h1f2p$, with the interaction of Zheng et al. (1995) and the code OXBASH. The resulting ground-state wave function was used to compute the $B(E2)$ value for the $2^+_1 \rightarrow 0^+_1$ transition, and the one-body density matrix elements were folded with the Bonn B nucleon-nucleon interaction via the Melbourne $g$-folding model to analyze elastic scattering of 300 MeV protons from $^{16}$C.

## Key numbers

- Ground state of $^{16}$C: 72.82% $2\hbar\omega$ components and 27.18% $4\hbar\omega$ components; the $\nu(sd)^2$ configurations constitute approximately 60% of the total wave function (22.63% from $\nu(0d_{5/2})^2$ and 22.37% from $\nu(1s_{1/2})^2$).
- $B(E2)$ value from the shell model using bare operators: 1.35 $e^2$fm$^4$.
- $B(E2)$ value with effective charge of 0.09e: 2.79 $e^2$fm$^4$; with effective charge of 0.12e: 3.39 $e^2$fm$^4$.
- Experimental $B(E2)$ values cited: 0.63 $\pm$ 0.11 $\pm$ 0.16 $e^2$fm$^4$ (Fujii et al.), 4.15 $\pm$ 0.73 $e^2$fm$^4$ (Wiedeking et al.), and values from Ong et al. ranging from 1.4 $\pm$ 0.6 $\pm$ 0.4 to 2.7 $\pm$ 0.2 $\pm$ 0.7 $e^2$fm$^4$.
- Oscillator parameter used: $b = 1.7$ fm for mass-16 nuclei.
- First excited $2^+$ state in $^{16}$C at 1.77 MeV; second excited state at 3.03 MeV (tentatively $0^+$).

## Context

This work extends previous shell-model calculations of $^{16}$C by Fujii et al. (2007), Corragio et al. (2010), and Yuan et al. (2012), which used effective interactions derived from free nucleon-nucleon interactions but required large effective neutron charges (approximately 0.4e) to reproduce the $B(E2)$ value. The present calculation demonstrates that a larger model space including $(2+4)\hbar\omega$ components yields a more reliable description of $^{16}$C structure without such large effective charges, and the successful reproduction of 300 MeV elastic proton scattering data from Terashima et al. (2014) provides an independent validation of the wave functions. This work enables future studies of $^{17}$C and other heavy carbon isotopes using the same large-scale shell model approach, and suggests that the simple $\nu(sd)^2$ picture of $^{16}$C is insufficient.
