# Conformal Relativistic Viscous Hydrodynamics: Applications to RHIC results at sqrt(s_NN) = 200 GeV

> A new set of equations for relativistic viscous hydrodynamics that captures both weak-coupling and strong-coupling physics to second order in gradients has been developed recently. We apply this framework to bulk physics at RHIC, both for standard (Glauber-type) as well as for Color-Glass-Condensate initial conditions and show that the results do not depend strongly on the values for the second-order transport coefficients. Results for multiplicity, radial flow and elliptic flow are presented and we quote the ratio of viscosity over entropy density for which our hydrodynamic model is consistent with experimental data. For Color-Glass-Condensate initial conditions, early thermalization does not seem to be required in order for hydrodynamics to describe charged hadron elliptic flow.

## Key claim

The paper applies a new set of conformal relativistic viscous hydrodynamics equations to RHIC Au+Au collisions at 200 GeV and finds that results for multiplicity, radial flow, and elliptic flow do not depend strongly on the values of the second-order transport coefficients, and that for Color-Glass-Condensate initial conditions, early thermalization does not appear to be required for hydrodynamics to describe charged hadron elliptic flow.

## Method

The authors solved 2+1 dimensional boost-invariant relativistic viscous hydrodynamics using the second-order conformal equations from Baier et al. (2007), which include five second-order transport coefficients (tau_Pi, lambda_1, lambda_2, lambda_3, kappa) but reduce to four in flat spacetime. They implemented a numerical algorithm using matrix inversion with a naive discretization (first order in time, second order in space), and used the equation of state from Laine and Schroder (2006) that interpolates between a hadron resonance gas at low temperatures and perturbative QCD at high temperatures. Initial conditions were taken from both Glauber-type and Color-Glass-Condensate models, with freeze-out via the Cooper-Frye formula using an off-equilibrium distribution function.

## Key numbers

* For a strongly coupled N=4 plasma, eta/s = 1/(4 pi), tau_Pi = (2 - ln 2)/(2 pi T), and lambda_1 = eta/(2 pi T)
* In the 0+1 dimensional toy model, second-order corrections to the shear tensor increase its value by a factor of 1 + (1 - ln 2)/(3 pi T tau) relative to the first-order Navier-Stokes value
* The numerical algorithm requires eta/s > 10^-4 for stability, and values between eta/s = 10^-2 and eta/s = 10^-4 show no significant changes in hydrodynamic results
* Simulation parameters used for the freeze-out hypersurface plot include tau_0 = 1 fm/c, T_i = 0.36 GeV, T_f = 0.15 GeV

## Context

This work extends the first dynamic viscous hydrodynamic calculations for RHIC by Romatschke (2007) and others, using the complete second-order conformal framework of Baier et al. (2007) that captures both weak-coupling (Boltzmann equation) and strong-coupling (AdS/CFT) physics. It enables systematic extraction of the shear viscosity to entropy density ratio from experimental data and tests the sensitivity of observables to second-order transport coefficients, which were previously unconstrained. The results contradict the assumption that early thermalization is necessary for hydrodynamics to describe elliptic flow, at least for CGC initial conditions.
