# News from bottomonium spectral functions in thermal QCD

> New results on bottomonium at nonzero temperature are presented, using the FASTSUM Generation 2L ensembles. Preliminary results for spectral function reconstruction using Kernel Ridge Regression, a machine learning technique, are shown as well and compared to results from the Maximum Entropy Method.

# News from bottomonium spectral functions in thermal QCD

## Key claim
The paper presents preliminary results showing that bottomonium spectral functions reconstructed via Kernel Ridge Regression (KRR) exhibit a first peak whose position coincides with conventional fitting and Maximum Entropy Method (MEM) results, with a drop in peak size above T = 187 MeV; however, the second peak observed in KRR is not yet reliable and requires further study.

## Method
The authors used lattice NRQCD on FASTSUM Generation 2L anisotropic ensembles with N_f = 2+1 flavours of Wilson-clover quarks, a pion mass of m_pi = 236(2) MeV, and a physical strange quark. They computed bottomonium correlators in the Upsilon (S wave) and chi_b1 (P wave) channels, and applied Kernel Ridge Regression, a machine learning technique using a training set of 9000 spectral functions parametrised by N_a = 80 basis coefficients, to reconstruct spectral functions from Euclidean correlators at multiple temperatures.

## Key numbers
- Pion mass: m_pi = 236(2) MeV
- Pseudocritical temperature: T_pc = 162(1) MeV
- Temporal lattice cutoff: a_tau^{-1} = 5.997(34) GeV
- Anisotropy: xi = a_s/a_tau = 3.453(6)
- Spatial lattice spacing: a_s = 0.1136(6) fm
- Energy constant E_0 = 7464.5 MeV, fixed via experimental Upsilon(1S) mass
- Effective mass at lowest temperature: m(1S) = 9.460 GeV and m(1P) = 10.005 GeV
- Training set size: N_train = 9000 spectral functions
- Number of basis coefficients: N_a = 80

## Context
This work extends previous FASTSUM Generation 1 and Generation 2 analyses (Aarts et al., 2010-2014) by using ensembles with a lighter pion mass (236 MeV versus 384 MeV in Gen 2), finding quantitative agreement that indicates only weak pion mass dependence. The paper introduces Kernel Ridge Regression for spectral reconstruction on actual lattice QCD data for the first time, following its application to mock data in quantum many-body physics (KRR, 2018), and compares results with the Maximum Entropy Method (MEM) used in prior lattice studies (Aarts, 2007). The results enable future cross-validation of spectral reconstruction methods and further investigation of the second peak structure.
