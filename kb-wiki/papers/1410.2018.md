# Forward Hadron Productions in high energy pp collisions from a Monte-Carlo generator for Color Glass Condensate

> We develop a Monte-Carlo event generator based on combination of a parton production formula including the effects of parton saturation (called the DHJ formula) and hadronization process due to the Lund string fragmentation model. This event generator is designed for the description of hadron productions at forward rapidities and in a wide transverse momentum range in high-energy proton-proton collisions. We analyze transverse momentum spectra of charged hadrons as well as identified particles; pion, kaon, (anti-)proton at RHIC energy, and ultra-forward neutral pion spectra from LHCf experiment. We compare our results to those obtained in other models based on parton-hadron duality and fragmentation functions.

## Key claim

The authors develop a Monte-Carlo event generator that combines the DHJ (Dumitru-Hayashigaki-Jalilian-Marian) formula for parton production with the Lund string fragmentation model, claiming it provides a unified description of hadron spectra from low to high transverse momentum in forward rapidity pp collisions; however, the model overestimates STAR neutral pion data by a factor of 2 and requires an extremely large K-factor (K=22) to fit LHCf data when using the LPHD approach, indicating that the CGC-based description alone is insufficient for ultra-forward pions.

## Method

The authors implement the DHJ hybrid formalism, which treats collisions between a large-x₁ parton from the projectile and a small-x₂ gluon from the target, using unintegrated gluon distributions from numerical solutions of the running-coupling Balitsky-Kovchegov (rcBK) equation fitted to HERA data (parameter set g1.101). Partons are generated according to the DHJ formula with initial and final state DGLAP radiations, multiple parton interactions simulated via an eikonal model, and hadronization performed using the Lund string fragmentation model as implemented in PYTHIA 8.186, with 70 million events generated at RHIC energy and 1 million at LHC energy.

## Key numbers

- Saturation scale parameters: Q_s0² = 0.157 GeV², γ = 1.101, Λ = 0.241 GeV from rcBK fit to HERA data
- Average transverse area of proton: σ₀/2 = 16.5 mb obtained from DIS fits at HERA
- Minimum parton momentum cutoff: p_T,min = 1 GeV (results insensitive to values down to 0.25 GeV)
- Impact parameter Gaussian width: B = 0.25 fm² for multiple parton interaction modeling
- K-factor of 2.5 used for DHJ+LPHD and DHJ+DSS calculations at RHIC energy (√s = 200 GeV)
- For LHCf data at √s = 7 TeV, DHJ+LPHD requires K = 22 (inelastic) or K = 15 (non-diffractive) assuming σ_inel = 73.6 mb or σ_nondif = 48.45 mb
- DHJ+Lund model overestimates STAR neutral pion data at √s = 200 GeV by a factor of 2

## Context

This work extends the DHJ hybrid formalism (Dumitru, Hayashigaki, Jalilian-Marian, 2006) by implementing it as a full Monte-Carlo event generator with Lund string hadronization, building on earlier attempts such as the BBL Monte-Carlo model (Drescher, 2004) which used the KLN k_T-factorization formula. The model is compared to the LPHD approach (Levin, 2010) and fragmentation function approach (Albacete, 2010; Fujii, 2011; Lappi, 2013), as well as conventional pQCD-based generators HIJING and PYTHIA, finding that soft string excitations dominate ultra-forward pion production at LHC energies and that distinct CGC effects are not clearly visible in pp collisions at √s = 200 GeV. The authors plan to extend the model to proton-nucleus and nucleus-nucleus collisions.
