# Symplectic no-core configuration interaction framework for nuclear structure

> We present the symplectic no-core configuration interaction (SpNCCI) framework, in which the nuclear many-body problem is solved a symmetry-adapted basis that explicitly encodes approximate symmetries associated with nuclear collectivity and deformation. In this framework, calculations are carried out in a basis organized into Sp(3,R) irreducible representations (irreps), each of which can be expressed as an infinite tower of U(3) irreps. In this framework, matrices of realistic relative two-body operators, such as the nuclear Hamiltonian, are computed directly in the Sp(3,R) many-body basis, obviating the need to expand all Sp(3,R) many-body states in, e.g., a U(3)-coupled configuration basis. Instead, many-body matrix elements are obtained via a recurrence relation that expresses a given matrix element in terms of matrix elements between basis states with fewer oscillator quanta. To use this recurrence method for computing matrix elements of relative two-body operators, we must first expand each operator into components of U(3) tensors. To this end, we present a method for decomposing arbitrary operators into U(3) tensor components.

## Key claim

The SpNCCI framework enables *ab initio* nuclear structure calculations in a basis organized by Sp(3,R) irreducible representations, using a recurrence relation to compute matrix elements of realistic two-body operators without fully expanding the symplectic basis in a U(3)-coupled configuration basis. The method requires first decomposing arbitrary operators into U(3) tensor components, and the recurrence seeds are matrix elements between special "lowest grade" states that are explicitly constructed in the U(3)-coupled basis.

## Method

The authors construct a many-body basis organized into Sp(3,R) irreducible representations (irreps), each containing an infinite tower of U(3) irreps. For each Sp(3,R) irrep, a single "highest weight state" belonging to the unique U(3) irrep with fewest oscillator quanta (the lowest grade irrep, LGI) is explicitly expanded in a U(3)-coupled configuration basis; remaining states are obtained by acting with Sp(3,R) algebra operators. Matrix elements of relative two-body operators are computed via a recurrence relation that expresses a given matrix element in terms of matrix elements between basis states with fewer oscillator quanta, with seeds provided by matrix elements between the LGI states. Operators are decomposed into U(3) tensor components to enable use of the recurrence method.

## Key numbers

- The Sp(3,R) Lie algebra includes the quadrupole operator, generators of orbital angular momentum, kinetic energy, and harmonic oscillator Hamiltonian, as the "smallest algebra" containing these operators.
- The quadrupole operator is given by \( Q_{2M} = \sqrt{\frac{16\pi}{5}} \sum_{s=1}^{A} \frac{\mathbf{x}_s^2}{b^2} Y_{2M}(\hat{\mathbf{x}}_s) \), with harmonic oscillator length parameter \( b = \hbar/(m_N\Omega) \).
- The algebraic quadrupole operator is \( \mathcal{Q}_{2M} = \sqrt{\frac{4\pi}{5}} \sum_{n=1}^{A} \left[ \frac{\mathbf{x}_s^2}{b^2} Y_{2\mu}(\hat{\mathbf{x}}_s) + b^2\hbar^2 \mathbf{k}_s^2 Y_{2\mu}(\hat{\mathbf{p}}_s) \right] \).
- The reduced matrix element of the U(3) generator \( C^{(1,1)} \) within an irrep \( \omega \) is \( \sqrt{\frac{4}{3}} (\lambda_\omega^2 + \lambda_\omega\mu_\omega + \mu_\omega^2 + 3\lambda_\omega + 3\mu_\omega)^{\frac12} \delta_{\omega,\omega'} \).
- The reduced matrix element of the harmonic oscillator Hamiltonian \( H^{(0,0)} \) is \( N_\omega \delta_{\omega,\omega'} \), where \( N_\omega \) is the total number of oscillator quanta.

## Context

This work extends the symmetry-adapted no-core shell model (SA-NCSM) framework of Dytrych, Launey, and collaborators (2016-2025), which carried out *ab initio* calculations in a U(3)-coupled basis but could only identify subspaces spanned by all Sp(3,R) irreps with the same quantum numbers rather than resolving individual irreps. The SpNCCI framework builds on the recurrence approach initially proposed by Reske (1984) for evaluating matrix elements in a symplectic basis, and on earlier symplectic model calculations by Rowe and Rosensteel (1977-1985) that demonstrated E2 transition strengths could be obtained without effective charges. This method enables fine-grained truncation by individual Sp(3,R) irreps, potentially reducing basis size by orders of magnitude and extending *ab initio* descriptions to more highly deformed collective nuclei.
