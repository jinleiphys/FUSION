# The Counting of Generalized Polarizabilities

> We demonstrate a concise method to enumerate the number of generalized polarizabilities---quantities characterizing the independent observables in singly-virtual Compton scattering---for a target particle of arbitrary spin s. By using crossing symmetry and J^{PC} conservation, we show that this number is (10s+1+delta_{s,0}).

# The Counting of Generalized Polarizabilities

## Key claim
The number of independent generalized polarizabilities (GPs) for singly-virtual Compton scattering off a target of arbitrary spin s is (10s + 1 + δ_{s,0}), but only after imposing crossing symmetry and charge conjugation symmetry; without these constraints, the naive multipole counting gives (18s + 1 + 2δ_{s,0}) amplitudes, meaning that 8s + δ_{s,0} of the previously counted GPs are redundant.

## Method
The author counts independent GPs by first enumerating partial-wave amplitudes in the s-channel (γ* + t → γ + t) using both multipole expansions and helicity amplitudes, then repeats the counting in the t-channel (γ* + γ → t + t̅) where crossing symmetry becomes charge conjugation symmetry of the t + t̅ system. The t-channel counting uses J^{PC} conservation and the restriction that only E1 and M1 multipoles contribute to amplitudes linear in the soft photon energy, yielding the minimal set of independent GPs. The mapping between s-channel and t-channel soft-photon amplitudes is verified to preserve linearity in the photon energy.

## Key numbers
- Number of independent GPs for arbitrary spin s: (10s + 1 + δ_{s,0}) (Equation labeled GPnew).
- Number of multipole amplitudes without crossing constraints: (18s + 1 + 2δ_{s,0}) (Equation labeled GPold).
- For s = 0: 2 independent GPs (from the formula 10×0 + 1 + 1 = 2).
- For s = 1/2: 6 independent GPs (from the formula 10×(1/2) + 1 + 0 = 6).
- For s = 1 (deuteron): 11 independent GPs (from the formula 10×1 + 1 + 0 = 11).
- Number of real Compton scattering (RCS) helicity amplitudes: 2(s+1)(2s+1).
- The difference between RCS amplitudes and GPs is (2s−1)² − δ_{s,0}, which vanishes for s = 0 and s = 1/2 but is nonzero for s ≥ 1.

## Context
This work resolves a discrepancy in the literature: earlier counting by Guichon, Liu, and Thomas (GLT) found 3 GPs for spin-averaged (s=0) and 10 for spin-1/2 targets using multipole expansions without crossing constraints, but subsequent work by Drechsel, Kamalov, and collaborators (DKMS, DKKMS) showed that only 2 and 6 GPs are independent, respectively, by constructing tensors with crossing symmetry. Lebed provides a general formula for arbitrary spin s, confirms the s=0 and s=1/2 results, and extends the counting to higher spins such as the deuteron (s=1, giving 11 GPs). The result enables systematic measurement programs for GPs of composite targets, where the 11 deuteron GPs combined with the 6 proton GPs could yield information on neutron electromagnetic structure.
