# Color Transparency and Light Front Holographic QCD

> Color transparency, the reduction of initial- or final-state interactions in coherent nuclear processes, is a natural prediction of QCD provided that small-sized or point-like configurations (PLCs) are responsible for high-momentum transfer, high-energy, semi-exclusive processes. I use the FMS criteria for the existence of PLCs to show that the wave functions of light front holographic QCD, as currently formulated, do not contain a PLC.

## Key claim

Light front holographic QCD wave functions, as currently formulated, do not contain point-like configurations (PLCs), meaning they do not predict the appearance of color transparency no matter how large the value of \(Q^2\). The FMS criterion shows that the quantity \(b^2(Q^2)\) rises with increasing \(Q^2\) for these wave functions, rather than dropping as required for PLC existence.

## Method

The author applies the Frankfurt-Miller-Strikman (FMS) criterion, which defines a quantity \(b^2(Q^2)\) as the ratio of the matrix element of the transverse separation squared operator \(b^2\) to the form factor \(F(Q^2)\). If \(b^2(Q^2)\) drops with increasing \(Q^2\), the model wave function admits PLCs. The author evaluates \(b^2(Q^2)\) for two light front holographic QCD wave functions: an early pion wave function from Brodsky (2007) and the universal light front wave functions of de Teramond (2018) for Fock state components with \(\tau = 2, 3, 4\) constituents.

## Key numbers

- The universal scale \(\lambda\) is fixed from the \(\rho\) mass: \(\sqrt{\lambda} = \kappa = m_\rho / \sqrt{2} = 0.548\) GeV.
- The flavor-independent parameter \(a = 0.531 \pm 0.037\).
- For the \(\tau = 2\) wave function, the ratio \(q_2(x)/f(x)\) diverges as \(4\lambda/(1-x)\) as \(x \to 1\), causing \(F_{b^2}(Q^2)\) to be infinite for all \(Q^2\).
- For \(\tau = 3\) and \(\tau = 4\), \(b^2_\tau(Q^2)\) rises with increasing \(Q^2\) (shown in Figure 2 in units of GeV\(^{-2}\)), meaning these wave functions do not admit PLCs.
- The early pion wave function form factor falls asymptotically as \(\sim 1/Q^2\).

## Context

This work extends the FMS criterion (Frankfurt, Miller, Strikman, 1993) for determining whether a hadronic wave function admits PLCs, applying it to light front holographic QCD wave functions from the review by Brodsky (2014) and the universal GPD model of de Teramond (2018). The result that these wave functions favor the Feynman mechanism over the pQCD mechanism for high-momentum transfer reactions is consistent with the experimental finding of Bhetuwal (2020) that color transparency does not occur in \((e,e'p)\) reactions up to \(Q^2 = 14.2\) GeV\(^2\). The author notes that adding a perturbative QCD tail to the momentum-space wave function could change this result.
