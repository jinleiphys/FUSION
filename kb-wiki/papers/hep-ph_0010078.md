# Instantaneous Bethe-Salpeter Equation: (Semi-)Analytical Solution

> The Bethe-Salpeter equation for bound states of a fermion-antifermion pair in the instantaneous approximation for the involved interaction kernel is converted into an equivalent matrix eigenvalue problem with explicitly (algebraically) given matrices.

## Key claim

The Bethe-Salpeter equation for a fermion-antifermion bound state in the instantaneous approximation can be reduced to an eigenvalue problem for an explicitly given algebraic matrix, enabling (semi-)analytical solution; however, the method relies on truncating infinite series expansions, and the resulting accuracy depends on the matrix size and the number of basis states retained.

## Method

The authors start from the instantaneous Bethe-Salpeter equation for massless fermion-antifermion bound states with quantum numbers \(J^{PC}=0^{-+}\), using a free-propagator approximation and a Lorentz vector interaction kernel (\(\gamma^0 \otimes \gamma^0\)). They expand the radial Salpeter amplitudes in sets of basis states for \(L_2(\mathbb{R}^+)\) with a variational parameter \(\mu\), convert the coupled integral equations into a matrix eigenvalue problem for the matrix \(\mathcal{M}_{ij}\), and evaluate all integrals via truncated expansions whose coefficients are given algebraically. The matrix is then diagonalized numerically (or analytically for matrix sizes \(\le 4\)) to obtain bound-state masses.

## Key numbers

- For a linear confinement potential \(V(r) = \lambda r\) with \(\lambda = 0.2\ \text{GeV}^2\), the one-dimensional (single-basis-state) analytical result after minimization gives \(M = 1.696\ \text{GeV}\), which is 2.4% away from the numerical result \(M = 1.656\ \text{GeV}\) obtained with \(15 \times 15\) matrices and \(N = 49\) basis states.
- For nonvanishing constituent mass \(m\), the analytical expression for the bound-state mass squared is \(M^2 = 8 m^2 + \frac{8896}{315\pi} \lambda + \frac{23}{7} \left( \frac{128 \lambda}{45 \pi m} \right)^2\).
- For \(15 \times 15\) matrices and \(N = 49\), the relations among expansion coefficients are fulfilled with relative errors less than 3%.

## Context

This work extends earlier studies of the instantaneous Bethe-Salpeter equation (e.g., Lagae, Olsson) by providing a systematic algebraic method to convert the integral equations into a matrix eigenvalue problem, avoiding purely numerical integration. It enables semi-analytical computation of bound-state masses for confining potentials, with the linear potential as a simple model for quark confinement. The approach can be applied to other interaction kernels and constituent masses, and the accuracy of truncation is explicitly quantified, pointing to the need for sufficiently large basis sets (e.g., \(N=49\)) to achieve percent-level precision.
