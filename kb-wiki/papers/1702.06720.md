# Combining few-body cluster structures with many-body mean-field methods

> Nuclear cluster physics implicitly assumes a distinction between groups of degrees-of-freedom, that is the (frozen) intrinsic and (explicitly treated) relative cluster motion. We formulate a realistic and practical method to describe the coupled motion of these two sets of degrees-of-freedom. We derive a coupled set of differential equations for the system using the phenomenologically adjusted effective in-medium Skyrme type of nucleon-nucleon interaction. We select a two-nucleon plus core system where the mean-field approximation corresponding to the Skyrme interaction is used for the core. A hyperspherical adiabatic expansion of the Faddeev equations is used for the relative cluster motion. We shall specifically compare both the structure and the decay mechanism found from the traditional three-body calculations with the result using the new boundary condition provided by the full microscopic structure at small distance. The extended Hilbert space guaranties an improved wave function compared to both mean-field and three-body solutions. We shall investigate the structures and decay mechanism of $^{22}$C ($^{20}$C+n+n). In conclusion, we have developed a method combining nuclear few- and many-body techniques without losing the descriptive power of each approximation at medium-to-large distances and small distances respectively. The coupled set of equations are solved self-consistently, and both structure and dynamic evolution are studied.

## Key claim

The paper develops a method that combines a many-body mean-field description (using Skyrme interactions) with a few-body cluster description (using hyperspherical adiabatic expansion of Faddeev equations) for a two-nucleon-plus-core system, such that the effective cluster interactions are derived from the underlying many-body interaction rather than being phenomenologically adjusted. The method is applied to the specific system of \(^{22}\)C (\(^{20}\)C+n+n), and the authors find that the short-range structure of the three-body wave function differs significantly from a traditional three-body calculation, which will affect predicted decay mechanisms.

## Method

The authors start from a many-body Hamiltonian with Skyrme-type density-dependent two-body interactions and separate it into a core Hamiltonian, a three-body Hamiltonian for the two valence neutrons, and a mixing term. They choose a wave function that is an antisymmetrized product of a core Slater determinant (mean-field) and a three-body wave function expressed in hyperspherical coordinates. Varying the energy with respect to both wave functions yields a set of coupled differential equations: a Hartree-Fock-like equation for the core single-particle states (with potentials modified by the three-body wave function) and a three-body equation for the valence neutrons (with potentials modified by the core density). The three-body equation is solved using a hyperspherical adiabatic expansion of the Faddeev equations in coordinate space.

## Key numbers

* The test system is \(^{22}\)C, viewed as a \(^{20}\)C core plus two valence neutrons; the system is at the neutron dripline where \(^{21}\)C is unbound but \(^{22}\)C is bound.
* The three-body wave function squared is shown as a function of two relative coordinates: the distance between the core and one valence neutron (\(r_{n,c}\)) and the distance between the remaining valence neutron and the core-neutron system (\(r_{n,cn}\)).
* In the traditional three-body calculation, the system forms an equidistant configuration with both neutrons at roughly the same distance from the core; in the new method, one neutron is situated close to the core and the other further away.

## Context

This work extends the traditional mean-field Hartree-Fock approach (championed by Vautherin (1972)) and the traditional three-body cluster method using hyperspherical adiabatic expansion of Faddeev equations (Nie (2001)) by coupling them together. The method addresses the difficulty of determining effective cluster interactions phenomenologically, which limits cluster models when experimental information is lacking. The approach enables calculation of any observable accessible to either mean-field or three-body methods, and the authors plan to compare predicted momentum distributions following one- or two-neutron removal (Kobayashi (2012)) and invariant mass spectra with experiment.
