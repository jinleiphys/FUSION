# Balance functions from a thermal model

> A calculation of the pion balance functions in a thermal model is presented. The total result consists of resonance and non-resonance parts. A satisfactory agreement with the data on Au+Au collisions at 130 GeV is found.

## Key claim

The thermal model with resonances, using parameters fixed by fitting particle abundances and transverse-momentum spectra, yields a satisfactory description of the pion balance functions in Au+Au collisions at 130 GeV, although the calculated total width for the most central collisions (⟨δ⟩ = 0.66) is slightly larger than the experimental value (0.594 ± 0.019).

## Method

The balance function was calculated as a sum of two contributions: a resonance part from decays of neutral hadronic resonances (K_S, η, η′, ρ^0, ω, σ, and f_0) that produce π⁺π⁻ pairs, and a non-resonance part where opposite-charge pions are produced at the same space-time point with thermal velocities. The model used a temperature of T = 165 MeV and expansion parameters fitted to hadron spectra, yielding an average transverse flow of 0.5c. The normalization of the model curves was adjusted for each centrality class to account for limited detector efficiency and acceptance.

## Key numbers

- Temperature used in the calculation: T = 165 MeV
- Average transverse flow: ⟨β_⊥⟩ = 0.5
- Calculated total width of the balance function: ⟨δ⟩ = 0.66
- STAR experimental width for most central events (c = 0–10%): ⟨δ⟩ = 0.594 ± 0.019
- STAR experimental width for mid-central events (c = 10–40%): ⟨δ⟩ = 0.622 ± 0.020
- STAR experimental width for mid-peripheral events (c = 40–70%): ⟨δ⟩ = 0.633 ± 0.024
- STAR experimental width for peripheral events (c = 70–96%): ⟨δ⟩ = 0.664 ± 0.029

## Context

This work extends the successful application of thermal (statistical) models from hadron yields, spectra, elliptic-flow coefficients v₂, and HBT radii to the new observable of balance functions. It provides an alternative to the coalescence model explanation of Bialas (2003) for the small widths of balance functions observed by the STAR Collaboration, and is complementary to the canonical blast wave model of Pratt et al. (2004). The centrality dependence of the balance function width could not be reproduced by varying the transverse flow within limits consistent with single-particle spectra, leaving this as an open question for future work.
