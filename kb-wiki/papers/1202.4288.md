# The effect of core polarization on longitudinal form factors in $^{10}$B

> Electron scattering Coulomb form factors for the single-particle quadrupole transitions in $p$-shell $^{10}$B nucleus have been studied. Core polarization effects are included through a microscopic theory that includes excitations from the core orbits up to higher orbits with 2$\hbar$$ω$ excitations. The modified surface delta interaction (MSDI) is employed as a residual interaction. The effect of core polarization is found essential in both the transition strengths and momentum transfer dependence of form factors, and gives a remarkably good agreement with the measured data with no adjustable parameters.

## Key claim

The paper claims that including core polarization effects via first-order perturbation theory with the modified surface delta interaction (MSDI) and 2ħω excitations brings calculated C2 form factors for ¹⁰B into remarkably good agreement with measured data with no adjustable parameters, but the agreement is not uniform across all transitions: for the 3⁺₂ state at 4.774 MeV the core polarization correction makes the calculation worse, and the average B(C2↑) value reaches only about 90% of the average experimental value.

## Method

The authors calculate longitudinal Coulomb form factors for quadrupole (C2) transitions in ¹⁰B using a microscopic theory that combines 1p-shell model wave functions from the Cohen-Kurath interaction with first-order core polarization corrections. The core polarization includes excitations from core orbits up to the 2p1f-shell (2ħω excitations) using the modified surface delta interaction (MSDI) as the residual interaction, with single-particle harmonic-oscillator wave functions using a size parameter b = 1.71 fm. The reduced matrix elements of the electron scattering operator are computed including the one-body density matrix elements, and the form factors include corrections for center-of-mass motion and nucleon finite size.

## Key numbers

- For the 1⁺ state at Ex = 0.718 MeV: B(C2↑) = 0.889 e²fm⁴ (1p-shell only), 1.77 e²fm⁴ (with core polarization), compared to measured 1.7 ± 0.3 e²fm⁴ (Cichocki et al. 1995)
- For the 2⁺ state at Ex = 3.587 MeV: B(C2↑) = 0.568 e²fm⁴ (1p-shell only), 1.55 e²fm⁴ (with core polarization), compared to measured 0.6 ± 0.1 e²fm⁴ (Cichocki et al. 1995)
- For the 3⁺₂ state at Ex = 4.774 MeV: B(C2↑) = 0.56 e²fm⁴ (1p-shell only), 1.66 e²fm⁴ (with core polarization), compared to measured < 0.04 e²fm⁴ (Cichocki et al. 1995)
- For the 4⁺ state at Ex = 6.025 MeV: B(C2↑) = 5.79 e²fm⁴ (1p-shell only), 11.67 e²fm⁴ (with core polarization), compared to measured 17.4 ± 0.7 e²fm⁴ (Cichocki et al. 1995)
- The harmonic oscillator size parameter b = 1.71 fm (from Evseev et al. 1987)
- Momentum transfer range studied: up to q ≤ 2.58 fm⁻¹

## Context

This work extends the approach of Radhi et al. (2001, 2002, 2003) who showed core polarization effects are essential for p-shell and sd-shell form factors, and directly addresses the failure of the 1p-shell Cohen-Kurath (1965) model to describe C2 form factors in ¹⁰B. Cichocki et al. (1995) found that expanding the shell-model space to include 2ħω configurations gave only a 10% improvement, whereas the present first-order perturbation treatment with MSDI achieves substantially better agreement for most transitions. The results enable future work using modern effective interactions to potentially improve the description further, particularly for the problematic 3⁺₂ transition.
