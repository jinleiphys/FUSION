# Excited states from eigenvector continuation: the anharmonic oscillator

> Eigenvector continuation (EC) has recently attracted a lot attention in nuclear structure and reactions as a variational resummation tool for many-body expansions. While previous applications focused on ground-state energies, excited states can be accessed on equal footing. This work is dedicated to a detailed understanding of the emergence of excited states from the eigenvector continuation approach. For numerical applications the one-dimensional quartic anharmonic oscillator is investigated, which represents a strongly non-perturbative quantum system where the use of standard perturbation techniques break down. We discuss how different choices for the construction of the EC manifold affect the quality of the EC resummation and investigate in detail the results from EC for excited states compared to results from a full diagonalization as a function of the basis-space size.

# Excited states from eigenvector continuation: the anharmonic oscillator

## Key claim

Eigenvector continuation (EC) can access excited states on equal footing with ground states, but the convergence for excited states is significantly slower than for the ground state, requiring many more EC basis states; the EC manifold constructed from perturbative state corrections for a specific excited state yields faster convergence for that target state than using ground-state perturbative corrections.

## Method

The authors apply eigenvector continuation to the one-dimensional quartic anharmonic oscillator, partitioning the Hamiltonian into an unperturbed harmonic oscillator part and a quartic perturbation. They construct the EC manifold from perturbative state corrections up to order P, either using the ground state as reference (EC0) or using excited states of the unperturbed Hamiltonian as reference (ECk). The generalized eigenvalue problem formed from Hamiltonian and norm matrices in this basis is solved, with re-orthogonalization via Householder transformations to handle ill-conditioned norm matrices. Results are benchmarked against full diagonalization in the harmonic oscillator basis.

## Key numbers

- Model-space dimension set to Nmax = 50 (maximum oscillator quantum number) for all calculations
- Condition number of the norm matrix exceeds kappa > 10^16 for P > 25, indicating a numerically ill-conditioned problem
- For EC0 targeting the ground state, good accuracy is obtained within p ≈ 5, while the first excited state requires p ≈ 20 for convergence
- For excited states with k >= 3, the full model space (P = Nmax = 50) must be exhausted to reproduce exact energies
- Singular values of the norm matrix fall below 10^-10 for perturbation orders p > 25

## Context

This work extends eigenvector continuation, previously applied mainly to ground-state energies in nuclear structure (Demol 2020, Demol 2020 BMBPT), to excited states. The anharmonic oscillator benchmark follows the classic analysis of Bender and Wu (1969) showing that perturbation theory yields only an asymptotic expansion for this system. The results contrast with previous EC applications using softer Hamiltonians in many-body problems (Demol 2020), where EC efficiently selected relevant low-dimensional subspaces; here the canonical configuration interaction basis outperforms EC for low-order calculations. The work enables future investigation of size-extensivity properties and diagrammatic expansions of EC at low order.
