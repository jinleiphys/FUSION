# The Unitary Fermi Gas in a Harmonic Trap and its Static Response

> We use harmonically trapped systems to find the leading gradient corrections of the supefluid local density approximation (SLDA) - a density functional theory (DFT) describing the unitary Fermi gas (UFG). We find the leading order correction to be negative, and predict the q^2 coefficient of the long-range static response c = 1.5(3) - a factor of two smaller than predicted by mean-field theory - thereby establishing a new and experimentally measurably universal constant.

## Key claim

The leading gradient correction to the superfluid local density approximation (SLDA) for the unitary Fermi gas is negative, and the predicted \(q^2\) coefficient of the long-range static response is \(c_\chi = 1.5(3)\), a factor of two smaller than mean-field theory predicts, but the author notes that this value cannot be directly extracted from quantum Monte Carlo results without a model to extrapolate to the thermodynamic limit.

## Method

The author uses the SLDA density functional, which includes auxiliary kinetic and anomalous densities, to describe harmonically trapped unitary Fermi gases. By fitting the gradient correction parameter \(\delta\lambda\) to reproduce quantum Monte Carlo energies of trapped systems, the static response coefficient \(c_\chi\) is extracted from the relation \(c_\chi = 7\alpha/3 + 9\delta\lambda\), where \(\alpha\) is the inverse effective mass. The gradient corrections are included perturbatively, and the results are compared with the simpler extended Thomas-Fermi (ETF) functional.

## Key numbers

* The best fit to homogeneous quantum Monte Carlo results gives \(\xi = 0.3742(5)\), \(\alpha = 1.104(8)\), and \(\eta = 0.651(9)\).
* The SLDA predicts \(c_\chi \approx 1.5(3)\) for the static response coefficient, compared to the mean-field BdG value of \(c_\chi = 7/3\).
* The gradient correction required to describe both homogeneous boxes and trapped systems is \(\delta\lambda \approx -0.12(3)\).
* The ETF model with \(\xi = 0.3742\) and \(\delta\lambda = 0\) gives \(c_\chi = 9/4 = 2.25\).
* For the non-interacting system, the static response coefficient is \(c_\chi = 1\), with a fundamental uncertainty band \(2/3 \leq c_\chi \lessapprox 1.7\) from shell effects.

## Context

This work extends the SLDA functional, previously constrained by quantum Monte Carlo calculations of homogeneous matter (Forbes 2012), to describe inhomogeneous trapped systems and extract the next-to-leading order coefficient of the superfluid effective field theory developed by Son and Wingate (2006). The predicted static response coefficient \(c_\chi\) is about half the value from Bogoliubov-de Gennes mean-field theory (Manes 2009) and differs from the epsilon-expansion result (Rupak 2008). The result enables experimental measurement of this universal constant through density profile measurements in modulated traps or optical lattices, providing a benchmark for models of the unitary Fermi gas and its application to neutron star phenomenology.
