# Elliptical flow coalescence to identify the $f_{0}$(980) content

> We use a simple coalescence model to generate $f_{0}$(980) particles for three configurations: a ${s\bar{s}}$ meson, a ${u\bar{u}s\bar{s}}$ tetraquark, and a ${K^{+}K^{-}}$ molecule. The phase-space information of the coalescing constituents is taken from a multi-phase transport (AMPT) simulation of heavy-ion collisions. It is shown that the number of constituent quarks scaling of the elliptic flow anisotropy can be used to discern ${s\bar{s}}$ from ${u\bar{u}s\bar{s}}$ and ${K^{+}K^{-}}$ configurations.

## Key claim

The number of constituent quarks (NCQ) scaling of elliptic flow can distinguish an \(s\bar{s}\) meson configuration of the \(f_0(980)\) from a tetraquark (\(u\bar{u}s\bar{s}\)) or \(K^{+}K^{-}\) molecule configuration, but it cannot reliably differentiate between the tetraquark and molecule states because the molecule effectively behaves as a four-quark state.

## Method

The authors used a simple coalescence model with a Gaussian Wigner function to generate \(f_0(980)\) particles in three configurations (\(s\bar{s}\), \(u\bar{u}s\bar{s}\), \(K^{+}K^{-}\)), as well as pions, protons, kaons, and \(\phi\) mesons for validation. The phase-space information of coalescing constituents (quarks, antiquarks, and kaons) was taken from the string-melting version of the AMPT (a multi-phase transport) simulation of mid-central Au+Au collisions at \(\sqrt{s_{NN}}=200A\) GeV. The elliptic flow \(v_2\) was calculated using the second harmonic plane from the initial spatial configuration, and the NCQ scaling relation \(v_{2,h}(p_{T,h}) = n_q \cdot v_{2,q}(p_{T,h}/n_q)\) was tested for each configuration.

## Key numbers

- For pions, the Gaussian Wigner function width was set to \(\sigma_r = 0.64\) fm based on \(\langle r^2 \rangle = (0.61 \pm 0.15)\) fm\(^2\).
- For kaons, \(\sigma_r = 0.48\) fm based on \(\langle r^2 \rangle = (0.34 \pm 0.05)\) fm\(^2\).
- For protons, \(\sigma_r = 0.72\) fm based on \(\sqrt{\langle r^2 \rangle} = 0.88\) fm.
- For the \(f_0(980)\) in all three configurations, \(\sigma_r = 2.4\) fm, derived from \(\omega = 67.8\) MeV and strange quark mass \(m_s = 0.199\) GeV/\(c^2\).
- The \(v_2(p_T)\) of the \(f_0(980)\) with \(s\bar{s}\) configuration is very different from the other configurations, especially when \(p_T > 1\) GeV.
- When \(\sigma_r\) was artificially set to 10 fm for the tetraquark state, the \(v_2/n_q\) vs. \(p_T/n_q\) lay midway between those of u-quarks and s-quarks, as expected in the ideal NCQ scaling picture.

## Context

This work extends the use of NCQ scaling, previously observed experimentally for conventional hadrons (Adams 2003, Adams 2004, Adams 2005, Adare 2006, Abelev 2007, Adamczyk 2013, Abelev 2014), to the controversial \(f_0(980)\) particle, whose configuration has been debated since the 1970s (Protopopescu 1973, Hyams 1973, Grayer 1974). The study uses the AMPT model (Lin 2004) as input to a custom coalescence model, since AMPT does not natively produce tetraquark hadrons. The results enable experimental discrimination of the \(s\bar{s}\) configuration from four-quark states via \(v_2\) measurements, but further separation of tetraquark from \(K^{+}K^{-}\) molecule states would require yield studies as suggested by Cho (2010).
