# Projection of angular momentum via linear algebra

> Projection of many-body states with good angular momentum from an initial state is usually accomplished by a three-dimensional integral. We show how projection can instead be done by solving a straightforward system of linear equations. We demonstrate the method and give sample applications to $^{48}$Cr and $^{60}$Fe in the $pf$ shell. This new projection scheme, which is competitive against the standard numerical quadrature, should also be applicable to other quantum numbers such as isospin and particle number.

# Projection of angular momentum via linear algebra

## Key claim

Angular momentum projection can be accomplished by solving a system of linear equations rather than the standard three-dimensional integral over Euler angles, provided the initial state contains only a finite number of angular momenta and the rotation eigenfunctions are linearly independent. The method is numerically competitive with standard quadrature but requires careful handling of small eigenvalues and mesh selection for the inversion to remain stable.

## Method

The authors replace the standard three-dimensional integral over Euler angles for angular momentum projection with a linear algebra problem by treating the norm and Hamiltonian matrix elements as unknowns in a system of linear equations. They construct a mesh of Euler angles, evaluate the overlap and Hamiltonian matrix elements of the rotated initial state at those angles, and solve for the projected norm and Hamiltonian matrices using the linear independence of Wigner D-matrices. The method is implemented for Slater determinants in finite shell-model spaces using the code SHERPA, with a separable inversion strategy that first projects out the magnetic quantum numbers K and M via Fourier sums, then projects out total angular momentum J using a mesh on the Euler angle beta.

## Key numbers

- For $^{48}$Cr in the $pf$ shell with interaction GXPF1A, the linear algebra projection (LAP) with 12,615 evaluations and the "need-to-know" variant with 4,375 evaluations both agreed with quadrature at 40 points per angle (64,000 evaluations) to within 2 keV for states up to J=10, and within approximately 10 keV for the J=12 state (which had fractional occupation f_J = 0.007).
- The fractional occupation f_J for uncranked Hartree-Fock in $^{60}$Fe in the $pf$ shell reached a maximum J of 12 with f_J approximately 0.001; for a strongly cranked Hartree-Fock state with cranking frequency hbar omega = 1.0 MeV, most of the state had J=12 with a range between J=6 and J=16.
- A tolerance of epsilon approximately 0.001 was used for excluding small eigenvalues of the norm matrix and for determining the maximum angular momentum J_max such that the sum of f_J over J up to J_max was at least 0.999.
- For a typical case with J_max = 12, the simplest LAP mesh required 13 x 25 x 25 = 8,125 Euler angles, compared to quadrature meshes between 15,625 and 32,768 angles.

## Context

This work extends the standard angular momentum projection method described in Ring and Schuck (2004), which uses three-dimensional quadrature over Euler angles with Wigner D-matrices. The linear algebra approach is inspired by Fourier methods used in shell-model Monte Carlo for extracting traces with good particle number (PhysRevC.49.1422) and good M (z-component of angular momentum) (PhysRevLett.99.162504), but is the first to fully project out angular momentum via inversion of linear equations. The method enables faster projection for applications including projected Hartree-Fock, projected Hartree-Fock-Bogoliubov, the projected shell model, and projected generator coordinate methods, and should be applicable to other quantum numbers such as isospin and particle number.
