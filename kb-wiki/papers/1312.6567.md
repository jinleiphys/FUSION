# Relation between the dipole polarizabilities of charged and neutral pions

> Using the fact that the contribution of the states with isospin I=0 in the difference of the amplitudes of the processes γγ->π^+π^- and γγ->π^0π^0 is very small, we have analyzed the dispersion sum rules for the difference between the dipole polarizabilities of the charged and neutral pions as a function of the σ-meson parameters. Then taken into account the current perturbation value of (α-β)_π^0=-1.9, we have found (α-β)_π^(+-)=9.4--8.2 for values of the σ-meson parameter within the region: M_s=400--550 MeV, Γ_s=400--600 MeV, Γ_{\s->γγ}= 0--3 keV. It has been shown that the value of the decay width of the h_1(1170)- meson into γπ^0 can be found if the difference (α-β)_π^(+-) is reliably determined from the experiment. Estimation of the optimal value of the decay width σ->γγhas given Γ_{\s->γγ}<0.7$keV.

## Key claim

Using dispersion sum rules for the difference between charged and neutral pion dipole polarizabilities, the charged pion polarizability difference is found to be in the range 9.4 to 8.2 (in units of 10^-4 fm^3) for sigma-meson parameters within M_s = 400–550 MeV, Gamma_s = 400–600 MeV, and Gamma_{sigma->gamma gamma} = 0–3 keV, but this result depends on adopting the chiral perturbation theory value for the neutral pion polarizability difference of -1.9.

## Method

The author analyzed dispersion sum rules (DSR) for the difference between the dipole polarizabilities of charged and neutral pions, Delta(alpha-beta) = (alpha-beta)_{pi+-} - (alpha-beta)_{pi0}, using fixed-u dispersion relations without subtractions for the helicity amplitude M_{++}. The DSR was saturated by contributions from rho(770), b1(1235), a1(1260), a2(1320) mesons in the s-channel and sigma, f0(980), f0'(1370) mesons in the t-channel, plus a nonresonant S-wave contribution from two charged pions, with the sigma-meson parameters varied within the stated ranges.

## Key numbers

- (alpha-beta)_{pi0} = -1.9 (the chiral perturbation theory value used as input)
- (alpha-beta)_{pi+-} = 9.4 to 8.2 (in units of 10^-4 fm^3) for sigma-meson parameters: M_s = 400–550 MeV, Gamma_s = 400–600 MeV, Gamma_{sigma->gamma gamma} = 0–3 keV
- Gamma_{sigma->gamma gamma} less than or approximately 0.7 keV (estimated optimal value from the DSR analysis)
- If the experimental result (alpha-beta)_{pi+-} = 11.6 from Mainz is confirmed, then Gamma_{h1->gamma pi0} = 0.875 MeV
- The h1(1170) meson decay width into gamma pi0 is estimated as 1.6845 +/- 0.44 MeV using nonet symmetry

## Context

This work extends the author's previous dispersion relation analyses of gamma gamma -> pi+ pi- and gamma gamma -> pi0 pi0 (Fil'kov 2013, earlier works) by focusing on the difference between charged and neutral pion polarizabilities to reduce model dependence from isoscalar contributions. The results agree with experimental values from Mainz (radiative pion photoproduction) and Serpukhov (Coulomb scattering) but disagree with chiral perturbation theory predictions (Gasser 2006, Burgi 1996) which give lower values around 4.7–6.7. The analysis enables determination of the h1(1170) -> gamma pi0 decay width once the charged pion polarizability is reliably measured experimentally.
