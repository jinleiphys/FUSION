# Analysing supersymmetric transformed $α$-nucleus potentials with electric-multipole transitions

> Alpha($^{4}$He)-cluster models have often been used to describe light nuclei. Towards the application to multi-cluster systems involving heavy clusters, we study the relative wave functions of the $α+^{16}$O and $α+^{40}$Ca systems generated from phase-shift-equivalent potentials. In general, a potential between clusters is deep accommodating several redundant bound states which should be removed in an appropriate way. To avoid such a complicated computation, we generate a shallow-singular potential by using supersymmetric transformations from the original deep potential. Changes in the relative wave functions by the transformations are quantified with electric-multipole transitions which give a different radial sensitivity to the wave function depending on their multipolarity. Despite the fact that the original and transformed potentials give exactly the same phase shift, some observables are unfavorably modified. A possible way to obtain a desired supersymmetric potential is proposed.

## Key claim

Supersymmetric (SUSY) transformations can generate phase-shift-equivalent shallow-singular potentials from deep cluster potentials, but the resulting wave functions are modified in ways that unfavorably alter some observables such as electric-multipole transition probabilities, even though the phase shifts remain identical.

## Method

The authors start with phenomenological deep α-nucleus potentials for α+¹⁶O and α+⁴⁰Ca that contain redundant forbidden bound states arising from the Pauli principle. They apply standard SUSY transformations to remove these forbidden states, generating phase-equivalent shallow-singular potentials. They then introduce a "designed" SUSY transformation (SUSY-β) with an arbitrary parameter β that can be fixed to reproduce a specific physical observable, such as the ground-state root-mean-square radius. The relative wave functions from the original deep potentials, standard SUSY potentials, and SUSY-β potentials are compared using electric-multipole transition probabilities (E2 and E4) and rms radii.

## Key numbers

- For ²⁰Ne with Set C potentials: ground-state rms radius r(0⁺) = 2.889 fm (reproduced from experiment); B(E2;2⁺→0⁺) = 54.7 e²fm⁴ (17.0 W.u.) for the deep OCM potential, compared to 65.5±3.2 e²fm⁴ (20.3±1.0 W.u.) experimentally.
- For ²⁰Ne: standard SUSY transformation gives B(E2;2⁺→0⁺) = 23.1 W.u., while SUSY-β with β = −1.8670 gives 16.9 W.u., matching the OCM value of 17.0 W.u.
- For ²⁰Ne: B(E4;4⁺→0⁺) = 47.3 W.u. (SUSY), 25.5 W.u. (SUSY-β), and 37.2 W.u. (OCM).
- For ⁴⁴Ti with Set C potentials: ground-state rms radius r(0⁺) = 3.515 fm; B(E2;2⁺→0⁺) = 122.4 e²fm⁴ (13.3 W.u.) for OCM, compared to 120±30 e²fm⁴ (13±3 W.u.) experimentally.
- For ⁴⁴Ti: standard SUSY gives B(E2;2⁺→0⁺) = 20.0 W.u., while SUSY-β with β = −1.3791 gives 13.3 W.u., matching the OCM value.
- For ⁴⁴Ti: B(E4;4⁺→0⁺) = 53.6 W.u. (SUSY), 23.7 W.u. (SUSY-β), and 36.7 W.u. (OCM).
- The β values used are −1.8670 for ²⁰Ne and −1.3791 for ⁴⁴Ti.

## Context

This work extends the SUSY transformation method developed by Baye (1987, 2004, 2014) for generating phase-equivalent shallow potentials from deep cluster potentials, which was previously applied to systems such as ¹²C=α+α+α by Tursunov (2003) and Suzuki (2008) and to ⁶He=α+n+n by Pinilla (2011). The paper quantifies for the first time how SUSY transformations modify wave functions and observables beyond phase shifts, using electric-multipole transitions as a diagnostic tool. The proposed SUSY-β method enables the construction of forbidden-state-free potentials that preserve specific physical observables, facilitating future applications to multi-cluster systems involving heavier clusters such as ²¹Ne=¹⁶O+α+n, ²⁴Mg=¹⁶O+α+α, ⁴⁵Ti=⁴⁰Ca+α+n, and ⁴⁸Cr=⁴⁰Ca+α+α.
