# Analysing supersymmetric transformed $őĪ$-nucleus potentials with electric-multipole transitions

> Alpha($^{4}$He)-cluster models have often been used to describe light nuclei. Towards the application to multi-cluster systems involving heavy clusters, we study the relative wave functions of the $őĪ+^{16}$O and $őĪ+^{40}$Ca systems generated from phase-shift-equivalent potentials. In general, a potential between clusters is deep accommodating several redundant bound states which should be removed in an appropriate way. To avoid such a complicated computation, we generate a shallow-singular potential by using supersymmetric transformations from the original deep potential. Changes in the relative wave functions by the transformations are quantified with electric-multipole transitions which give a different radial sensitivity to the wave function depending on their multipolarity. Despite the fact that the original and transformed potentials give exactly the same phase shift, some observables are unfavorably modified. A possible way to obtain a desired supersymmetric potential is proposed.

## Key claim

Supersymmetric (SUSY) transformations can generate phase-shift-equivalent shallow-singular potentials from deep cluster potentials, but the resulting wave functions are modified in ways that unfavorably alter some observables such as electric-multipole transition probabilities, even though the phase shifts remain identical.

## Method

The authors start with phenomenological deep őĪ-nucleus potentials for őĪ+¬Ļ‚Ā∂O and őĪ+‚Āī‚ĀįCa that contain redundant forbidden bound states arising from the Pauli principle. They apply standard SUSY transformations to remove these forbidden states, generating phase-equivalent shallow-singular potentials. They then introduce a "designed" SUSY transformation (SUSY-ő≤) with an arbitrary parameter ő≤ that can be fixed to reproduce a specific physical observable, such as the ground-state root-mean-square radius. The relative wave functions from the original deep potentials, standard SUSY potentials, and SUSY-ő≤ potentials are compared using electric-multipole transition probabilities (E2 and E4) and rms radii.

## Context

This work extends the SUSY transformation method developed by Baye (1987, 2004, 2014) for generating phase-equivalent shallow potentials from deep cluster potentials, which was previously applied to systems such as ¬Ļ¬≤C=őĪ+őĪ+őĪ by Tursunov (2003) and Suzuki (2008) and to ‚Ā∂He=őĪ+n+n by Pinilla (2011). The paper quantifies for the first time how SUSY transformations modify wave functions and observables beyond phase shifts, using electric-multipole transitions as a diagnostic tool. The proposed SUSY-ő≤ method enables the construction of forbidden-state-free potentials that preserve specific physical observables, facilitating future applications to multi-cluster systems involving heavier clusters such as ¬≤¬ĻNe=¬Ļ‚Ā∂O+őĪ+n, ¬≤‚ĀīMg=¬Ļ‚Ā∂O+őĪ+őĪ, ‚Āī‚ĀĶTi=‚Āī‚ĀįCa+őĪ+n, and ‚Āī‚ĀłCr=‚Āī‚ĀįCa+őĪ+őĪ.
