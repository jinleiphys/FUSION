# Implementation of machine learning techniques to predict impact parameter and transverse spherocity in heavy-ion collisions at the LHC

> Machine learning techniques have been quite popular recently in the high-energy physics community and have led to numerous developments in this field. In heavy-ion collisions, one of the crucial observables, the impact parameter, plays an important role in the final-state particle production. This being extremely small (i.e. of the order of a few fermi), it is almost impossible to measure impact parameter in experiments. In this work, we implement the ML-based regression technique via Gradient Boosting Decision Trees (GBDT) to obtain a prediction of impact parameter in Pb-Pb collisions at $\sqrt{s_{NN}}$ = 5.02 TeV using A Multi-Phase Transport (AMPT) model. After its successful implementation in small collision systems, transverse spherocity, an event shape observable, holds an opportunity to reveal more about the particle production in heavy-ion collisions as well. In the absence of any experimental exploration in this direction at the LHC yet, we suggest an ML-based regression method to estimate centrality-wise transverse spherocity distributions in Pb-Pb collisions at $\sqrt{s_{NN}}$ = 5.02 TeV by training the model with minimum bias collision data. Throughout this work, we have used a few final state observables as the input to the ML-model, which could be easily made available from collision data. Our method seems to work quite well as we see a good agreement between the simulated true values and the predicted values from the ML-model.

## Key claim

The authors demonstrate that a Gradient Boosting Decision Trees (GBDT) regression model trained on final-state observables from the AMPT model can predict impact parameter and transverse spherocity distributions in Pb-Pb collisions at LHC energies, with the caveat that the method's accuracy depends on the Monte Carlo model describing input observables as closely as possible to experimental data.

## Method

The authors used the AMPT model to generate minimum-bias Pb-Pb collision events at sqrt(s_NN) = 5.02 TeV and 2.76 TeV. They applied a supervised regression technique via the Gradient Boosting Decision Trees (GBDT) algorithm from the scikit-learn Python package, using charged-particle multiplicity (dN_ch/deta), charged-particle multiplicity in the transverse region (N_ch^TS), and average transverse momentum (p_T) as input variables to predict impact parameter (b) and transverse spherocity (S_0). The model was trained on minimum-bias events and then used to predict centrality-wise distributions.

## Key numbers

* Impact parameter prediction: The correlation matrix shows a significant anti-correlation (rho approximately -0.8 to -0.9) between impact parameter and charged-particle multiplicity (dN_ch/deta) in Pb-Pb collisions at sqrt(s_NN) = 5.02 TeV.
* Transverse spherocity prediction: The correlation matrix shows sufficiently high correlation between input variables (dN_ch/deta, N_ch^TS, p_T) and transverse spherocity in Pb-Pb collisions at sqrt(s_NN) = 5.02 TeV, though the correlation with p_T is small but significant.
* Energy extrapolation: The model trained on Pb-Pb collisions at sqrt(s_NN) = 5.02 TeV successfully predicted spherocity distributions for Pb-Pb collisions at sqrt(s_NN) = 2.76 TeV over wide spherocity ranges.

## Context

This work extends prior ML-based impact parameter determination studies using support vector machines and neural networks (Bass 1993, David 1994, Bass 1996, Haddad 1996, De Sanctis 2009, Li 2020) by applying GBDT regression to both impact parameter and transverse spherocity in heavy-ion collisions. The method enables experimental access to transverse spherocity distributions in Pb-Pb collisions at the LHC, where no experimental measurements currently exist, and suggests that the correlation structure of spherocity with input observables is similar across LHC energies.
