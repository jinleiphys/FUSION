# Shell-model calculations of two-neutrino double-beta decay rates of $^{48}$Ca with GXPF1A interaction

> The two-neutrino double beta decay matrix elements and half-lives of $^{48}$Ca, are calculated within a shell-model approach for transitions to the ground state and to the $2^+$ first excited state of $^{48}$Ti. We use the full $pf$ model space and the GXPF1A interaction, which was recently proposed to describe the spectroscopic properties of the nuclei in the nuclear mass region A=47-66. Our results are $T_{1/2}(0^{+}\to 0^{+})$ = $3.3\times 10^{19}$ $yr$ and $T_{1/2}(0^{+}\to 2^{+})$ = $8.5\times 10^{23}$ $yr$. The result for the decay to the $^{48}$Ti 0$^+$ ground state is in good agreement with experiment. The half-life for the decay to the 2$^+$ state is two orders of magnitude larger than obtained previously.

## Key claim

The two-neutrino double-beta decay half-life of $^{48}$Ca to the ground state of $^{48}$Ti calculated with the GXPF1A interaction in the full $pf$ shell-model space is $3.3\times 10^{19}$ yr, which falls within the experimental range of $(4.3^{+3.3}_{-1.3})\times 10^{19}$ yr, while the half-life for the transition to the $2^+$ first excited state is $8.5\times 10^{23}$ yr, two orders of magnitude larger than previously reported.

## Method

The authors performed large-scale shell-model calculations in the full $pf$ model space using the GXPF1A effective interaction, which was fitted to spectroscopic data for nuclei in the mass region A=47-66. They computed the Gamow-Teller nuclear matrix elements for the $2\nu\beta\beta$ decay of $^{48}$Ca to both the $0^+$ ground state and the $2^+_1$ excited state of $^{48}$Ti, using up to 250 intermediate $1^+$ states in $^{48}$Sc and applying a standard quenching factor of 0.77 for the Gamow-Teller operator.

## Key numbers

- $T_{1/2}(0^{+}\rightarrow 0^{+})$ = $3.3\times 10^{19}$ yr for the ground-state to ground-state transition using GXPF1A interaction
- $T_{1/2}(0^{+}\rightarrow 2^{+})$ = $8.5\times 10^{23}$ yr for the transition to the $2^+_1$ excited state using GXPF1A interaction
- $\mid M(0^{+}\rightarrow 0^{+})\mid$ = 0.0539 MeV$^{-1}$ for the ground-state to ground-state nuclear matrix element with GXPF1A
- $\mid M(0^{+}\rightarrow 2^{+})\mid$ = 0.0122 MeV$^{-3}$ for the ground-state to $2^+_1$ nuclear matrix element with GXPF1A
- Phase space factors: $1.044\times10^{-17}$ yr$^{-1}$ MeV$^2$ for the $0^+\rightarrow 0^+$ transition and $1.958\times10^{-19}$ yr$^{-1}$ MeV$^6$ for the $0^+\rightarrow 2^+$ transition
- Experimental half-life range for $^{48}$Ca $2\nu\beta\beta$ decay: $(4.3^{+3.3}_{-1.3})\times 10^{19}$ yr from Balysh et al. (1996)

## Context

This work extends earlier shell-model calculations of $^{48}$Ca double-beta decay by Zhao, Brown and Richter (1990) using the OXBASH code with MH and MSOBEP interactions, and by Caurier, Poves and Zuker (1990) using the ANTOINE code with the KB3 interaction. The authors use the GXPF1A interaction for the first time in such calculations, which was recently developed to improve spectroscopic descriptions in the A=47-66 region. The results enable more reliable predictions for the neutrinoless double-beta decay mode of $^{48}$Ca and can guide future experiments such as CANDLES and CARVEL.
