# Spinor Representations for Fields with any Spin: Lorentz Tensor Basis for Operators and Covariant Multipole Decomposition

> This paper discusses a framework to parametrize and decompose operator matrix elements for particles with higher spin $(j > 1/2)$ using chiral representations of the Lorentz group, i.e. the $(j,0)$ and $(0,j)$ representations and their parity-invariant direct sum. Unlike traditional approaches that require imposing constraints to eliminate spurious degrees of freedom, these chiral representations contain exactly the $2j+1$ components needed to describe a spin-$j$ particle. The central objects in the construction are the $t$-tensors, which are generalizations of the Pauli four-vector $σ^μ$ for higher spin. For the generalized spinors of these representations, we demonstrate how the algebra of the $t$-tensors allows to formulate a generalization of the Dirac matrix basis for any spin. For on-shell bilinears, we show that a set consisting exclusively of covariant multipoles of order $0\leq m \leq 2j$ forms a complete basis. We provide explicit expressions for all bilinears of the generalized Dirac matrix basis, which are valid for any spin value. As a byproduct of our derivations we present an efficient algorithm to compute the $t$-tensor matrix elements. The formalism presented here paves the way to use a more unified approach to analyze the non-perturbative QCD structure of hadrons and nuclei across different spin values, with clear physical interpretation of the resulting distributions as covariant multipoles.

## Key claim

The paper demonstrates that chiral representations $(j,0)$ and $(0,j)$ of the Lorentz group, which contain exactly $2j+1$ components without requiring constraints to eliminate spurious degrees of freedom, provide a unified framework for decomposing operator matrix elements for any spin $j > 1/2$, with the caveat that this framework is intended for parametrizing QCD matrix elements of asymptotic states rather than constructing consistent interacting Lagrangians.

## Method

The authors construct generalized spinors using $t$-tensors, which are rank-$2j$ generalizations of the Pauli four-vector $\sigma^\mu$, and develop the algebra of these tensors including cubic and quadratic product reductions. They then formulate a generalized Dirac matrix basis for any spin $j$ using the $(j,0)\oplus(0,j)$ bispinor representation, and derive explicit expressions for all bilinears in this basis, demonstrating that on-shell bilinears reduce to a complete set of covariant $\mathfrak{sl}(2,\mathbb{C})$ multipoles of order $0 \leq m \leq 2j$.

## Key numbers

* The chiral representations $(j,0)$ and $(0,j)$ contain exactly $2j+1$ components needed to describe a spin-$j$ particle, eliminating the need for constraints that traditional approaches require for $j > 1/2$.
* The covariant multipole basis for on-shell bilinears consists of multipoles of order $0 \leq m \leq 2j$, forming a complete set.
* The cubic product of $t$-tensors reduces to an irreducible form: $t^{\mu_{1} \cdots \mu_{2j} } \bar{t}^{\rho_{1} \cdots \rho_{2j} } t^{\sigma_{1} \cdots \sigma_{2j} } = \frac{1}{[(2j)!]^2} \mathcal{S} \left( \prod_{l=1}^{2j} {\mathcal{C}}^{\mu_{l} \rho_{l} \sigma_{l} \alpha_{l}} \right) t_{\alpha_{1} \cdots \alpha_{2j}}$, where ${\mathcal{C}}^{\mu \rho \sigma \alpha } = g^{\mu \rho } g^{\sigma \alpha } - g^{\mu \sigma } g^{\rho \alpha } + g^{\mu \alpha } g^{\rho \sigma } + {\rm i} \epsilon^{\mu \rho \sigma \alpha }$.
* The paper provides explicit expressions for all bilinears of the generalized Dirac matrix basis valid for any spin value $j$.

## Context

This work extends the chiral representation formalism originally developed by Weinberg (1964) and earlier work by Joos (1962) and Barut, Muzinich, and Williams (1963) for constructing causal fields of any spin. It contrasts with traditional approaches that use Rarita-Schwinger vector-spinors (Rarita and Schwinger, 1941) for spin-3/2 or four-vectors for spin-1, which require constraints such as $p_\mu \epsilon^\mu(p,\lambda)=0$ or $\gamma^\mu \psi_\mu(p,\lambda)=0$ to eliminate spurious degrees of freedom. The framework builds on work by Cotogno et al. (2019) on covariant multipole decomposition and enables a unified approach to analyze non-perturbative QCD structure of hadrons and nuclei across different spin values, with applications to spin-1 (deuteron, vector mesons) and spin-3/2 (baryon decuplet) systems that previously required separate case-by-case analyses as in Berger (2001), Cosyn (2018), Bacchetta (2000), and Fu (2022, 2024).
