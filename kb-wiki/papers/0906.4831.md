# Fluctuation properties of strength function associated with the giant quadrupole resonance in 208Pb

> We performed fluctuation analysis by means of the local scaling dimension for the strength function of the isoscalar (IS) giant quadrupole resonance (GQR) in 208Pb where the strength function is obtained by the shell model calculation including 1p1h and 2p2h configurations. It is found that at almost all energy scales, fluctuation of the strength function obeys the Gaussian orthogonal ensemble (GOE) random matrix theory limit. This is contrasted with the results for the GQR in 40Ca, where at the intermediate energy scale about 1.7 MeV a deviation from the GOE limit was detected. It is found that the physical origin for this different behavior of the local scaling dimension is ascribed to the difference in the properties of the damping process.

## Key claim

The fluctuation of the strength function for the isoscalar giant quadrupole resonance (GQR) in 208Pb follows the Gaussian orthogonal ensemble (GOE) random matrix theory limit at almost all energy scales, in contrast to the GQR in 40Ca where a deviation from the GOE limit was detected at an intermediate energy scale of about 1.7 MeV. The authors ascribe this different behavior to the difference in the properties of the damping process between the two nuclei.

## Method

The authors performed a shell model calculation using the second Tamm-Dancoff approximation (TDA) including 1p1h and 2p2h configurations for the isoscalar GQR in 208Pb, with a Landau-Migdal-type residual interaction and single-particle states from a Woods-Saxon potential within four major shells. They applied fluctuation analysis using the local scaling dimension, a quantity derived from the partition function of binned strength distributions, to characterize the fluctuation properties at each energy scale. The authors also evaluated various spreading widths (Landau damping width, spreading width of 1p1h states, spreading width of 2p2h states) and compared them with the average level spacings to understand the damping mechanism.

## Key numbers

- Total width of the strength function for the IS GQR in 208Pb: FWHM = 0.63 MeV, obtained from the smooth strength function using the Strutinsky method with smoothing width 0.2 MeV
- Average excitation energy of the IS GQR in 208Pb: 10.5 MeV, with standard deviation 2.6 MeV
- Landau damping width for 208Pb: Gamma_L = 0.18 MeV (from level spacing between the two largest TDA peaks)
- Spreading width of the GQR TD state due to coupling with 2p2h states in 208Pb: Gamma_2 = 0.41 MeV
- Spreading width of TD (1p1h) states in 208Pb: gamma_12 = 0.38 MeV
- Spreading width of 2p2h states in 208Pb: gamma_22 = 0.75 MeV
- Average level spacing of 2p2h states in 208Pb: d_2p2h = 1.2 keV
- Average level spacing of 1p1h states in 208Pb: D_1p1h = 230 keV
- Number of 1p1h states: 39; number of 2p2h states: 8032 (with 3342 in the 9-13 MeV energy range)
- For 40Ca: total width Gamma = 4000 keV, Landau damping width Gamma_L = 4000 keV, gamma_12 = 1500 keV, D_1p1h = 500 keV, d_2p2h = 11 keV

## Context

This work extends the authors' previous fluctuation analysis of the GQR in 40Ca (Aiba et al., 2008) to the heavier nucleus 208Pb, using the same local scaling dimension method. The study contrasts with the results for 40Ca, where a clear deviation from GOE behavior was detected at the intermediate energy scale corresponding to the spreading width of 1p1h states. The findings show that for 208Pb, the Landau damping width is smaller than both the spreading width of 1p1h states and the average spacing of 1p1h states, which prevents the detection of the doorway state effect in the fluctuation properties. This work enables a deeper understanding of how the damping mechanism changes from light to heavy nuclei, and provides criteria for when the spreading width of doorway states can be detected in the fluctuation properties of strength functions.
