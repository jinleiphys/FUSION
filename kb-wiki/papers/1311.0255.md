# Quark mass functions and pion structure in Minkowski space

> We present a study of the dressed quark mass function and the pion structure in Minkowski space using the Covariant Spectator Theory. The quark propagators are dressed with the same kernel that describes the interaction between different quarks. We use an interaction kernel in momentum space that is a relativistic generalization of the linear confining q-qbar potential and a constant potential shift that defines the energy scale. The confining interaction has a Lorentz scalar part that is not chirally invariant by itself but decouples from the equations in the chiral limit and therefore allows the Nambu-Jona-Lasinio mechanism to work. We adjust the parameters of our quark mass function calculated in Minkowski-space to agree with Lattice QCD data obtained in Euclidean space. Results of a calculation of the pion electromagnetic form factor in the relativistic impulse approximation using the same mass function are presented and compared with experimental data.

## Key claim

The authors present a self-consistent model for the dressed quark mass function and pion structure in Minkowski space using the Covariant Spectator Theory, where the quark propagators are dressed with the same kernel that describes the quark-antiquark interaction. The model achieves agreement with lattice QCD data for the quark mass function and with experimental data for the pion electromagnetic form factor, but the pion form factor calculation uses an approximated vertex function and does not include explicit rho-meson contributions, which should be important according to vector meson dominance.

## Method

The authors use a momentum-space interaction kernel that is a relativistic generalization of the linear confining quark-antiquark potential plus a constant potential shift, with strong phenomenological form factors for each quark line. The quark mass function is calculated directly from the interaction kernel via a gap equation, with parameters adjusted to fit lattice QCD data (Bowman 2005) extrapolated to the chiral limit. The pion electromagnetic form factor is then computed in the relativistic impulse approximation using the same mass function in the chiral limit, with an approximated pion vertex function and a dressed quark current satisfying the Ward-Takahashi identity.

## Key numbers

- Constituent quark mass in the chiral limit: m_chi = 0.308 GeV, fixed by fit to lattice data extrapolated to m_0 = 0
- Mass parameter from quark form factors: M_g = 1.734 GeV, fixed in the chiral limit
- The mass function is fitted to the first 50 lattice data points in the low-p^2 region between 0 and -1.94 GeV^2
- The pion mass mu = 0.42 GeV gives the best fit to experimental pion form factor data (Huber 2008) over the full Q^2 range
- For large Q^2, form factors for different pion masses scale as F_pi(Q^2, lambda*mu) approximately equals lambda^2 * F_pi(Q^2, mu) when Q^2 >> mu^2

## Context

This work extends earlier CST-based models by Gross, Milana, and Savkli (various years) where the quark mass function was approximated phenomenologically, whereas here it is calculated self-consistently from the interaction kernel. The approach differs from Dyson-Schwinger approaches (e.g., Qin 2011) by working in Minkowski space rather than Euclidean space, enabling straightforward calculation of transition form factors in the timelike region. The results enable future work solving the complete four-channel CST equation for light meson properties and incorporating a more refined dressed quark current including the rho-meson pole contribution.
