# Axially-deformed solution of the Skyrme-Hartree-Fock-Bogoliubov equations using the transformed harmonic oscillator basis (IV) hfbtho (v4.0): A new version of the program

> We describe the new version 4.0 of the code hfbtho that solves the nuclear Hartree-Fock-Bogoliubov problem by using the deformed harmonic oscillator basis in cylindrical coordinates. In the new version, we have implemented the restoration of rotational, particle number, and reflection symmetry. The restoration of rotational symmetry does not require using bases closed under rotation. Furthermore, the SeaLL1 functional was added and the calculation of the Coulomb potential was improved. Finally, the code has been refactored to facilitate maintenance and future developments.

## Key claim

Version 4.0 of the hfbtho code implements restoration of rotational, particle number, and reflection symmetry for even-even nuclei using a formalism that works with deformed harmonic oscillator bases not closed under rotation, bypassing the need for rotationally-invariant bases that are computationally intractable for heavy or highly deformed nuclei.

## Method

The code solves the nuclear Hartree-Fock-Bogoliubov (HFB) problem by expanding quasiparticle wave functions in an axial harmonic oscillator (HO) or transformed harmonic oscillator (THO) single-particle basis and iteratively diagonalizing the HFB Hamiltonian. Symmetry restoration is performed through standard projection techniques: angular momentum projection uses a three-dimensional Euler angle integration reduced by axial symmetry, particle number projection uses the Fomenko expansion with discretized gauge angles, and parity projection uses the standard parity operator. For bases not closed under rotation, the code implements a reformulation of Wick's theorem following Robledo (1994) to compute rotated norm overlap kernels, density matrices, and pairing tensors.

## Key numbers

- The code version is 4.0 (v4.0)
- The projection operator for angular momentum uses the factor (2J+1)/(16 pi^2) in the Euler angle integration
- The particle number projection uses the Fomenko expansion with gauge angles phi_l = (pi/N_phi) * l, where N_phi is the number of gauge angle points
- The parity projection operator is P^p = (1/2)(1 + p*Pi) where p = +1 for positive parity and p = -1 for negative parity
- The integration interval over the gauge angle for particle number projection is reduced to [0, pi] using number parity conservation
- The integration interval over the Euler angle gamma is reduced to [0, 2*pi] using number parity conservation

## Context

This work extends the hfbtho program series (Stoitsov 2005, Stoitsov 2013, Perez 2017) from single-reference energy density functional (SR-EDF) capabilities to multi-reference energy density functional (MR-EDF) capabilities. It provides an alternative to the hfodd code (Dobaczewski 2021) for symmetry restoration, with the key advantage that angular momentum projection can be performed in deformed (stretched) harmonic oscillator bases rather than requiring rotationally-invariant closed bases, enabling large-scale calculations of potential energy surfaces for heavy and superheavy nuclei. The implementation enables future studies of nuclear fission and the structure of superheavy nuclei where deformed bases are essential for numerical convergence.
