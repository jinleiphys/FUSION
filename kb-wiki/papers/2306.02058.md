# Shell model study of first-forbidden beta decay around $^{208}$Pb

> In the present work, we report a systematic theoretical study of the $\log ft$ values for the forbidden $β^-$ decay transitions in the $^{208}$Pb region. For this, we have considered $^{206}$Hg $\rightarrow$ $^{206}$Tl, $^{208}$Hg $\rightarrow$ $^{208}$Tl, $^{206}$Tl $\rightarrow$ $^{206}$Pb and $^{208}$Tl $\rightarrow$ $^{208}$Pb transitions. We have performed shell model calculations using KHH7B interaction in valence shell 58-114 for protons and 100-164 for neutrons by considering ${\it 1p-1h}$ excitations for both protons and neutrons simultaneously for daughter nuclei. This study presents the first shell model results of $β^-$-decay corresponding to the recent experimental data.

# Shell model study of first-forbidden beta decay around $^{208}$Pb

## Key claim

The authors report the first shell-model calculations of $\log ft$ values for first-forbidden $\beta^-$ decay transitions in the $^{208}$Pb region, finding generally good agreement with recent experimental data, but with significant deviations for some transitions (notably $^{208}$Hg $\rightarrow$ $^{208}$Tl to the $1_1^-$ and $1_2^-$ states) that they attribute to the need for an extended model space or higher-order particle-hole excitations.

## Method

Large-scale shell-model calculations were performed using the KHH7B interaction in a model space spanning proton orbitals from Z=58 to 114 and neutron orbitals from N=100 to 164 around $^{208}$Pb, with $1p-1h$ excitations for both protons and neutrons simultaneously for the daughter nuclei. The one-body transition densities (OBTDs) were obtained from shell-model diagonalization using the NUSHELLX code, and the $\log ft$ values were computed using two different sets of quenching factors for the weak coupling constants: set I with ($g_A/g_A^{free}$, $g_V/g_V^{free}$) = (0.32, 1.00) and set II with (0.38, 0.51), including a mesonic enhancement factor $\epsilon_{MEC} = 2.01$ for $\Delta J=0^-$ transitions.

## Key numbers

- For $^{206}$Hg($0^+$) $\rightarrow$ $^{206}$Tl($0_1^-$): experimental $\log ft$ = 5.41(6); set I gives 5.082, set II gives 4.945
- For $^{206}$Hg($0^+$) $\rightarrow$ $^{206}$Tl($1_1^-$): experimental $\log ft$ = 5.24(10); set I gives 4.997, set II gives 5.254
- For $^{206}$Tl($0^-$) $\rightarrow$ $^{206}$Pb($0_1^+$): experimental $\log ft$ = 5.1775(13); set I gives 5.120, set II gives 4.983
- For $^{206}$Tl($0^-$) $\rightarrow$ $^{206}$Pb($2_1^+$) (first forbidden unique): experimental $\log ft$ = 8.60(3); set I gives 9.303, set II gives 9.174
- For $^{208}$Hg($0^+$) $\rightarrow$ $^{208}$Tl($0_1^-$): experimental $\log ft$ = 5.2(1); set I gives 5.352, set II gives 5.215
- For $^{208}$Hg($0^+$) $\rightarrow$ $^{208}$Tl($1_1^-$): experimental $\log ft$ = 6.0(2); set I gives 7.406, set II gives 7.862
- For $^{208}$Hg($0^+$) $\rightarrow$ $^{208}$Tl($1_2^-$): experimental $\log ft$ = 5.3(1); set I gives 7.689, set II gives 7.695
- For $^{208}$Tl($5^+$) $\rightarrow$ $^{208}$Pb($5_1^-$): experimental $\log ft$ = 5.61(1); set I gives 5.202, set II gives 5.261
- For $^{208}$Tl($5^+$) $\rightarrow$ $^{208}$Pb($4_1^-$): experimental $\log ft$ = 5.68(1); set I gives 5.351, set II gives 5.612

## Context

This work extends previous experimental studies of $\beta^-$ decay in the $^{208}$Pb region, particularly the recent CERN-ISOLDE experiment by Carroll and collaborators that reported the first $\log ft$ values for three excited states of $^{208}$Tl from $^{208}$Hg decay. The calculations use the KHH7B interaction (developed by Hosaka, Kuo, and Warburton) and the quenching factors from Sharma and from Zhi, and the authors note that results might be improved using the KHM3Y interaction with 24 orbitals as suggested by Brown. The study enables future work incorporating higher-order ($3p-3h$) excitations for beta decay to higher excited states, which is relevant for modeling the third peak of the solar abundance pattern in r-process nucleosynthesis.
