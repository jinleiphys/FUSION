# Scaling properties of multiplicity fluctuations in heavy ion collisions simulated by AMPT model

> Three-dimensional, as well as one- and two-dimensional, studies of multiplicity fluctuation are performed using AMPT model to generate central Au-Au collision events at ${\sqrt s_{NN}}= 200$ GeV. Two- and three-dimensional normalized factorial moments in rapidity, transverse momentum and azimuthal angle are found to exhibit power-low scaling when partitioning with the same number of bins in each direction, indicating that the fluctuation are isotropic, i.e. the fractality is self-similar in multiparticle production of central Au-Au collisions. Further, we measured the parameter $ν$ which it characterizes the intermittency indices derived in particular analysis. It is found that our model result ${ν_{yϕp_t}}= 1.86{\pm}0.07$ is larger than $ν = 1.304$,which is the value of Ginzburg-Landau type of phase transition. We also explored the intermittent or fluctuational dependence on the transverse momentum. The result shows that the intermittency or fluctuation increase rapidly with the increasing of transverse momentum.

## Key claim

The authors claim that the fractality in multiparticle production of central Au-Au collisions at 200 GeV simulated by the AMPT model is self-similar (isotropic), but they note that the measured parameter ν = 1.86 ± 0.07 is larger than the Ginzburg-Landau phase transition value of 1.304, which may indicate the AMPT model does not include the physics of phase transition.

## Method

The authors used the default AMPT model to generate 5000 central Au-Au collision events at √sNN = 200 GeV with impact parameter b ≤ 2 fm and parton cross-section 3 mb. They performed one-, two-, and three-dimensional normalized factorial moment (NFM) analyses in rapidity (y), transverse momentum (pt), and azimuthal angle (φ) phase space, using cumulant variables to flatten distributions. Hurst exponents were extracted from one-dimensional NFM saturation curves using the Ochs formula, and self-similar scaling was tested by partitioning phase space with equal bin numbers in each direction.

## Key numbers

- Hurst exponents from one-dimensional analysis: Hyp_t = 0.94 ± 0.05, Hφpt = 0.94 ± 0.06, Hyφ = 1.00 ± 0.04
- Three-dimensional intermittency parameter νyφpt = 1.86 ± 0.07, compared to Ginzburg-Landau value ν = 1.304
- Effective fluctuation strengths αeff for Au-Au collisions are constant at 0.024 ± 0.001 for q = 2-9, compared to 0.356 ± 0.011 (q=2) for π+(K+)p collisions and 0.635 ± 0.010 (q=2) for e+e- collisions
- Two-dimensional ν parameters: νyφ = 1.85 ± 0.13, νφpt = 1.94 ± 0.10, νypt = 1.94 ± 0.19
- Three-dimensional NFM fit parameters for q=2: c = 0.0009 ± 0.0004, φq = 0.0003 ± 0.0001, χ²/DF = 0.2/8

## Context

This work extends the study of multiplicity fluctuations and fractal scaling from hadron-hadron and e+e- collisions to heavy ion collisions using the AMPT model. Previous studies by NA22 (hadron-hadron) and L3/DELPHI (e+e-) found anisotropic (self-affine) and isotropic (self-similar) scaling respectively, while this work finds self-similar scaling in Au-Au collisions. The results enable comparison with experimental data to test whether the AMPT model requires modification to describe multiplicity fluctuations, and provide a baseline for searching for signals of the Ginzburg-Landau type phase transition in heavy ion collisions.
