# Deformed Boson Scheme Stressing Even-Odd Boson Number Difference. II

> The boson-pair coherent state developed in Part (I) is generalized to the case in which the state is a mixture of even-boson and odd-boson number states. A general framework is shown and its three concrete examples are discussed. Main idea is in the application of the deformed boson scheme presented by the present authors.

# Deformed Boson Scheme Stressing Even-Odd Boson Number Difference. II

## Key claim

The paper generalizes the boson-pair coherent state from Part (I) to a unified form that is a superposition of even-boson and odd-boson number states, parameterized by a mixing parameter that interpolates between the two previously separate cases. The unified state reduces to the conventional boson coherent state only under a specific condition on the mixing parameters, and the formalism is built on the deformed boson scheme previously developed by the same authors.

## Method

The authors construct a unified coherent state |cr⟩ = u|ch⟩ + (γ*/|γ|)v|sh⟩, where |ch⟩ and |sh⟩ are the even-boson and odd-boson number states from Part (I), and (u, v) are parameters satisfying u² + |v|² = 1. They derive the canonical variable transformation linking (γ, γ*) and (v, v*) to new canonical variables (c, c*) and (η, η*) that satisfy classical Poisson bracket relations. Three concrete examples of the deformation function f̃(k) are analyzed: one corresponding to exact su(1,1)-algebra realization, one giving an approximate solution, and one extending the conventional boson coherent state. The formalism is also expressed in terms of the MYT boson mapping method.

## Key numbers

- For the su(1,1)-algebra case (f̃(2n)=2n+1, f̃(2n-1)=1): the expectation value (τ₀)cr = |c|² + |η|²/2 + 1/4, giving the su(1,1) spin magnitude t = 1/4 + |η|²/2, which ranges from 1/4 to 3/4 as |η| varies from 0 to 1.
- For the approximate case (f̃(2n)=√(2n+1), f̃(2n-1)=1): the relation |γ|⁴ = 2|c|² and γ = √(√2 c) are obtained exactly.
- For the conventional boson extension (f̃(2n)=1, f̃(2n-1)=1): in the region |γ|² ∼ 0, Fcr ∼ (1-2|η|²/3)⁻¹ + (1/3)(1-14|η|²/15)(1-2|η|²/3)⁻²·|γ|⁴, and in the asymptotic limit |γ|² → ∞, Fcr → 2|c|² + 2|η|².
- For the su(1,1)-coherent state with t=1/4: (τ₀)₀ = |c₀|² + 1/4, where c₀ = √(1/2)·γ/√(1-γ*γ).
- For the su(1,1)-coherent state with t=3/4: (τ₀)₁ = |c₁|² + 3/4, where c₁ = √(3/2)·γ/√(1-γ*γ).

## Context

This paper extends the boson-pair coherent state formalism developed in Part (I) (Kuriyama et al., 2003) by unifying the separate even-boson and odd-boson number states into a single coherent state with a mixing parameter. The work builds on the authors' earlier deformed boson scheme (Kuriyama et al., 2003) and their investigation of boson-pair coherent states in connection with su(1,1)-spin with magnitudes t=1/4 and 3/4 (Kuriyama et al., 2003). The formalism enables the description of time-evolution of many-boson systems via the time-dependent variational method when the initial state is a mixture of even and odd boson numbers, and provides a framework for treating the expectation values of su(1,1) generators with continuously varying spin magnitude between 1/4 and 3/4.
