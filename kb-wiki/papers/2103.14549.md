# Massive quarks in NLO dipole factorization for DIS: Longitudinal photon

> In this work, we will present the first complete calculation of the one-loop longitudinal photon-to-quark-antiquark light cone wave function, with massive quarks. The quark masses are renormalized in the pole mass scheme. The result is used to calculate the next-to-leading order correction to the high energy Deep Inelastic Scattering longitudinal structure function on a dense target in the dipole factorization framework. For massless quarks the next-to-leading order correction was already known to be sizeable, and our result makes it possible to evaluate it also for massive quarks.

## Key claim

This paper presents the first complete calculation of the one-loop longitudinal photon-to-quark-antiquark light cone wave function with massive quarks, enabling the evaluation of next-to-leading order (NLO) corrections to the longitudinal structure function in deep inelastic scattering (DIS) for massive quarks, whereas previously only the massless quark NLO correction was known.

## Method

The authors use light cone perturbation theory (LCPT) to compute the one-loop corrections to the longitudinal virtual photon wave function, with quark masses renormalized in the pole mass scheme. They regularize ultraviolet divergences via dimensional regularization in (D-2) transverse dimensions and soft divergences with a longitudinal momentum cutoff, presenting results in both conventional dimensional regularization (CDR) and four-dimensional helicity (FDH) schemes. The calculation is performed in momentum space, then Fourier transformed to mixed transverse coordinate-longitudinal momentum space, and the cross section is derived from the wave functions including subtractions to cancel divergences between real and virtual contributions.

## Key numbers

- The leading order longitudinal photon-to-quark-antiquark wave function contains a vertex factor proportional to \(ee_f Q/q^+\), where \(Q\) is the photon virtuality and \(q^+\) the longitudinal momentum.
- The NLO cross section is calculated to order \(\alpha_{em}\alpha_s\) (where \(\alpha_{em}=e^2/(4\pi)\) and \(\alpha_s=g^2/(4\pi)\)), requiring the quark-antiquark wave function to one-loop order and the quark-antiquark-gluon wave function at tree level.
- The color factor for the quark-antiquark-gluon contribution is \(C_F = (N_c^2-1)/(2N_c)\), with \(N_c\) the number of colors.

## Context

This work extends the NLO dipole factorization framework for DIS, previously developed for massless quarks in a series of papers (Beuf 2011, Balitsky 2012, Beuf 2016, Beuf 2017, Ducloue 2017, Hanninen 2017), to include massive quarks. The calculation is motivated by the fact that inclusive heavy quark DIS cross sections can be more reliably perturbative probes of gluon saturation than light quark cross sections, because finite quark masses cut off the "aligned jet" configurations that introduce systematic uncertainty. This paper addresses only the longitudinal photon case, with transverse photons and a more detailed treatment of mass renormalization in LCPT to be addressed in separate future work.
