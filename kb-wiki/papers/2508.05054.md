# Analysis of (p,$γ$) capture cross-sections relevant to p-process using TALYS for A=75-110

> The proton capture (p, $γ$) cross-sections for eight different atomic nuclei in the mass region A=75-110 were calculated within the nuclear reaction model code TALYS. For all the reactions, we tested different combinations of inputs for level density (l.d) parameter and gamma strength function ($γ_{sf}$). Finally, it was observed that application of hybrid input in TALYS (macroscopic l.d and microscopic or semi-microscopic $γ_{sf}$ or in abbreviation mac-mic) resulted successful agreement of theoretical prediction with the existing experimental data. Isospin correction was also incorporated in a few cases which improved the matching if the centre of mass energy reaches the threshold energy of the opening of (p,n) channel. The corresponding thermonuclear reaction rates were calculated for all the nuclei and some discrepancies were found with the prediction of the NON-SMOKER code. Using the particular mac-mic input combination, the cross-section and reaction rate for the nuclei $^{92}$Nb and $^{92}$Mo are calculated within TALYS. These two nuclei lack experimental data but are highly important for understanding early solar system processes. This is a rare attempt to explain the p-capture cross-section of different p-nuclei (A=75-110 range) with similar set of input combinations in TALYS, which may help to remove the uncertainty generated due to the variation of input parameters within nuclear statistical model code

## Key claim

The authors claim that a "hybrid" input combination in TALYS (macroscopic level density model with microscopic or semi-microscopic gamma strength function, abbreviated mac-mic) yields successful agreement with existing experimental (p,γ) cross-section data for eight nuclei in the A=75-110 mass region, but this agreement is qualified by the need for isospin correction (via parameter fiso) in cases where the center-of-mass energy reaches the threshold for the (p,n) channel, and by the observation that for some nuclei (e.g., ⁸⁴Sr) even the best combination required reducing the gamma decay width to one-quarter of its default value to match the data.

## Method

The authors used the TALYS nuclear reaction code (version 2.0) to calculate (p,γ) cross-sections for eight nuclei (⁷⁵As, ⁸⁴Sr, ⁹¹Nb, ⁹³Nb, ⁹³Tc, ⁹⁹Tc, ¹⁰⁸Cd, ¹¹⁰Cd) by systematically varying the level density model (options 1-6) and gamma strength function (options 1-9) and comparing predictions against experimental data extracted from the literature. The best-performing combination was identified as TALYS-(2,5): the back-shifted Fermi gas (BSFG) level density model (option 2) with the microscopic Goriely table gamma strength function (option 5) and local optical model potential, sometimes with isospin correction factor fiso=0.6 or reduced gamma width. This combination was then applied to predict cross-sections and thermonuclear reaction rates for ⁹²Nb and ⁹²Mo, which lack experimental data.

## Key numbers

- For ⁷⁵As, TALYS-(2,5) with fiso=0.6 gave reduced χ² = 1.4×10⁻⁴ and maximum deviation 0.015, compared to NON-SMOKER values of 0.8420 and 0.612, respectively
- For ⁸⁴Sr, TALYS-(2,5) with Γ_γ = 1/4 gave reduced χ² = 26.9 and maximum deviation 86.9, while NON-SMOKER gave 311.4 and 428.7
- For ⁹¹Nb, TALYS-(2,5) gave reduced χ² = 3.16×10⁻⁵ and maximum deviation 3.6×10⁻²
- For ⁹³Nb, TALYS-(2,5) with fiso=0.6 gave reduced χ² = 1.1×10⁻⁴ and maximum deviation 3.6×10⁻²
- For ¹¹⁰Cd, TALYS-(2,5) with fiso=0.6 gave reduced χ² = 0.2 and maximum deviation 0.9, while NON-SMOKER gave 217.7 and 208.5
- The (p,n) threshold energies relevant for isospin correction were: 3.39 MeV for ⁷⁴Ge, 2.82 MeV for ⁹²Zr, 2.22 MeV for ¹⁰⁹Ag, 2.06 MeV for ⁹²Nb, and 5.27 MeV for ⁹²Mo
- Thermonuclear reaction rates were calculated for temperatures T = 1-5 GK

## Context

This work extends previous studies that found varying success among Hauser-Feshbach codes (TALYS, EMPIRE, NON-SMOKER) in reproducing (p,γ) cross-sections, with some cases (Spyrou et al., Khaliel et al.) showing no unique input combination that could reproduce all data even for a single nucleus. The authors address the known underproduction problem for p-nuclei in the A=90-96 mass range (⁹²,⁹⁴Mo and ⁹⁶,⁹⁸Ru) by providing calculated cross-sections and reaction rates for ⁹²Nb and ⁹²Mo, which are important for understanding early solar system processes and testing nucleosynthesis models. The work enables more reliable theoretical predictions for p-capture reactions on nuclei where experimental data are scarce or absent, and highlights discrepancies with the NON-SMOKER code predictions (Rauscher and Thielemann).
