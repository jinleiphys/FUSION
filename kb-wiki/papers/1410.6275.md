# Nuclear level density as a tool for probing the inelastic scattering of 6He

> The cross sections are calculated for the both elastic and inelastic scattering of 6He from 12C and 4He. A phenomenological optical potential is used to describe the elastic scattering. 4He is taken as spherical and inelastic couplings to the first excited states of 6He and 12C are described with collective rotational model and coupled-channels method. Deformation lengths for 6He and 12C are determined from semi-classical nuclear level density model by using Laplace-like formula for the nuclear level density parameter. The comparison of the predicted and the measured cross sections are presented to test the applicability of nuclear level density model to the light exotic nuclei reactions. Good agreement is achieved between the predicted and measured cross sections.

# Nuclear level density as a tool for probing the inelastic scattering of 6He

## Key claim
The nuclear level density model with a Laplace-like formula for the level density parameter can be used to determine deformation lengths for light exotic nuclei like 6He, enabling coupled-channels calculations of inelastic scattering cross sections that agree with measured data, although the predicted deformation parameter for 6He (2.45) has a large magnitude that requires reconsideration with a more complicated shape definition.

## Method
The authors used a phenomenological Woods-Saxon optical potential to describe elastic scattering of 6He from 12C and 4He targets. Inelastic couplings to the first excited states of 6He and 4He were treated with the collective rotational model and coupled-channels method, where deformation lengths were determined from a semi-classical nuclear level density model using a Laplace-like formula for the energy-dependent level density parameter. The deformation parameter for each nucleus was found by solving for the value that gives the cumulative number of levels equal to 1 at the experimentally known energy of the first excited state.

## Key numbers
- 6He has a two-neutron separation energy S_{2n} = 0.975 MeV
- Deformation parameter predicted for 12C: -0.6
- Deformation parameter predicted for 6He: 2.45
- Elastic and inelastic scattering calculations performed for 6He + 12C at 18.0 MeV and 30.0 MeV laboratory energies
- Elastic and inelastic scattering calculations performed for 6He + 4He at six center-of-mass energies: 3.8, 4.2, 4.7, 5.1, 5.4, and 5.8 MeV
- Optical potential parameters for 6He + 12C at 18.0 MeV: V0 = 44.789 MeV, r_v = 0.738 fm, a_v = 0.867 fm, W0 = 11.284 MeV, r_w = 1.227 fm, a_w = 0.696 fm
- Optical potential parameters for 6He + 12C at 30.0 MeV: V0 = 48.775 MeV, r_v = 0.931 fm, a_v = 0.745 fm, W0 = 6.991 MeV, r_w = 1.310 fm, a_w = 0.890 fm
- For 6He + 4He, the same geometrical parameters (r_v = 0.910 fm, a_v = 0.660 fm, r_w = 0.900 fm, a_w = 0.650 fm) were used at all six energies, with V0 varying from 64.800 MeV to 60.800 MeV and W0 varying from 0.500 MeV to 3.200 MeV

## Context
This work extends the authors' previous development of a Laplace-like formula for the nuclear level density parameter (Canbula 2014) to the problem of inelastic scattering of the halo nucleus 6He. The approach provides an alternative to the continuum-discretized coupled-channels (CDCC) method, which is considered the most appropriate but computationally intensive method for halo nuclei. The predicted deformation parameter for 12C agrees with existing literature (Nunes 1996, Vermeer 1983), while the large magnitude of the 6He deformation parameter indicates that a more sophisticated shape definition is needed for the collective amplitude calculation. This work enables future studies using nuclear level density as a tool for inelastic scattering calculations of light exotic nuclei.
