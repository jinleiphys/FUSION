# Solution of universal nonrelativistic nuclear DFT equations in the Cartesian deformed harmonic-oscillator basis. (IX) HFODD (v3.06h): a new version of the program

> We describe the new version (v3.06h) of the code HFODD that solves the universal nonrelativistic nuclear DFT Hartree-Fock or Hartree-Fock-Bogolyubov problem by using the Cartesian deformed harmonic-oscillator basis. In the new version, we implemented the following new features: (i) zero-range three- and four-body central terms, (ii) zero-range three-body gradient terms, (iii) zero-range tensor terms, (iv) zero-range isospin-breaking terms, (v) finite-range higher-order regularized terms, (vi) finite-range separable terms, (vii) zero-range two-body pairing terms, (viii) multi-quasiparticle blocking, (ix) Pfaffian overlaps, (x) particle-number and parity symmetry restoration, (xi) axialization, (xii) Wigner functions, (xiii) choice of the harmonic-oscillator basis, (xiv) fixed Omega partitions, (xv) consistency formula between energy and fields, and we corrected several errors of the previous versions.

## Key claim

The paper describes version v3.06h of the HFODD code, which implements fifteen new features for solving nuclear DFT equations, but the parallel mode, temperature-related options, and fission options were not thoroughly tested and should be considered fragile or not recommended for use.

## Method

The code solves the universal nonrelativistic nuclear DFT Hartree-Fock or Hartree-Fock-Bogolyubov problem using a Cartesian deformed harmonic-oscillator basis. The new version implements zero-range three- and four-body central terms, zero-range three-body gradient terms, zero-range tensor terms, zero-range isospin-breaking terms, finite-range higher-order regularized and separable terms, zero-range two-body pairing terms, multi-quasiparticle blocking, Pfaffian overlaps, particle-number and parity symmetry restoration, axialization, Wigner functions, choice of the harmonic-oscillator basis, fixed Omega partitions, and a consistency formula between energy and fields, along with corrections of several errors from previous versions.

## Key numbers

- The code version is v3.06h, with 311 available keywords in the input list.
- The tensor interaction introduces twelve new coupling constants $B_t^{\rm J}$, $B_t^{\rm X}$, $B_t^{\rm T}$, $B_t^{\Delta s}$, $B_t^{\rm F}$, and $B_t^{\nabla s}$ for $t=0,1$, which relate to two parameters $t_e$ and $3t_o$ via equations such as $B_0^{\rm J} = \tfrac{1}{8} (t_e + 3t_o)$ and $B_1^{\rm J} = -\tfrac{1}{8} (t_e - t_o)$.
- The three-body gradient term has five parameters: $u_1$, $y_1$, $u_2$, $y_{21}$, and $y_{22}$.
- The isospin-breaking terms include class-II and class-III interactions at leading order (LO) and next-to-leading order (NLO) with parameters $t_0^{\rm II}$, $x_0^{\rm II}$, $t_1^{\rm II}$, $x_1^{\rm II}$, $t_2^{\rm II}$, $x_2^{\rm II}$ and $t_0^{\rm III}$, $x_0^{\rm III}$, $t_1^{\rm III}$, $x_1^{\rm III}$, $t_2^{\rm III}$, $x_2^{\rm III}$.

## Context

This work extends the series of HFODD code publications, starting with Dobaczewski (1997) for the method and continuing through versions v1.60r, v1.75r, v2.08i, v2.08k, v2.40h, v2.49t, and v2.73y. The new version enables calculations with three- and four-body contact interactions motivated by Skyrme (1959) and Vautherin and Brink (1972), tensor terms as considered by Skyrme (1959) and later by Stancu (1977), PerliĹ„ska (2004), and others, and isospin-breaking terms that address mirror displacement energies and triplet displacement energies which cannot be reproduced by Coulomb interaction alone, as shown by Nolen (1969), Ormand (1989), and Baczyk (2018, 2019).
