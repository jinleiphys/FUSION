# Transport Coefficients of Non-Newtonian Fluid and Causal Dissipative Hydrodynamics

> A new formula to calculate the transport coefficients of the causal dissipative hydrodynamics is derived by using the projection operator method (Mori-Zwanzig formalism) in [T. Koide, Phys. Rev. E75, 060103(R) (2007)]. This is an extension of the Green-Kubo-Nakano (GKN) formula to the case of non-Newtonian fluids, which is the essential factor to preserve the relativistic causality in relativistic dissipative hydrodynamics. This formula is the generalization of the GKN formula in the sense that it can reproduce the GKN formula in a certain limit. In this work, we extend the previous work so as to apply to more general situations.

## Key claim

The paper derives a new formula for transport coefficients in causal dissipative hydrodynamics by extending the projection operator method (Mori-Zwanzig formalism) without the Q approximation, which generalizes the Green-Kubo-Nakano (GKN) formula to non-Newtonian fluids; the key condition is that the new formula reproduces the GKN formula only in the limit where the Q approximation is valid, and the authors assume that the renormalized memory function relaxes rapidly while the frequency function converges to a finite value at late times.

## Method

The authors used the projection operator method (Mori-Zwanzig formalism) to derive an exact expression for the memory function without applying the Q approximation, which replaces the projection operator Q with unity. They then applied this expression to the case of shear flow, choosing the energy-momentum tensor component \(T^{0x}(k_y)\) as a single gross variable, and introduced coarse-graining in time by separating the memory function into a frequency function and a renormalized memory function. The resulting equation was compared to the telegraph-type equation of causal dissipative hydrodynamics to extract the causal shear viscosity coefficient and relaxation time.

## Key numbers

- The causal shear viscosity coefficient is defined as \(\eta^{NN} = \lim_{k_y \rightarrow 0} 2 w D_{k_y} \tau\), where \(w\) is the enthalpy density, \(D_{k_y} = (1/k_y^2) \lim_{t \rightarrow \infty} \Omega^2(k_y,t)\), and \(\tau = \lim_{k_y \rightarrow 0} \tau_{k_y}\).
- The relaxation time is defined as \(1/\tau_{k_y} = \int_0^\infty dt \Phi(k_y,t)\), where \(\Phi(k_y,t)\) is the renormalized memory function.
- In the Q approximation limit, the Navier-Stokes shear viscosity coefficient is given by \(\eta^{NS} = \int_{-\infty}^\infty dt \theta(t) \frac{1}{10} \int d^3\mathbf{x}_1 (\pi^{\alpha\beta}(\mathbf{x},t), \pi_{\alpha\beta}(\mathbf{x}_1,0)) (T^{0x}(\mathbf{x}_1,0), T^{0x}(\mathbf{0},0))^{-1}\), reproducing the GKN formula.

## Context

This work extends the earlier derivation by Koide (2007) for a single gross variable to more general situations with multiple gross variables, and it provides a microscopic foundation for transport coefficients in causal dissipative hydrodynamics, which is essential for relativistic heavy-ion collisions. The new formula corrects the standard GKN formula (Green-Kubo-Nakano, 1950s) by accounting for memory effects that make the fluid non-Newtonian, and it enables the calculation of shear viscosity and relaxation time in systems where the Q approximation fails, such as in glass dynamics studied by mode-coupling theory (Kawasaki, 1970s; Gotze, 1990s). The approach also relates to the extended thermodynamics framework of Jou (1990s) and the work of Ichiyanagi on defining projection operators with currents.
