# Global analysis of hadron-production data in e^+ e^- annihilation for determining fragmentation functions

> Fragmentation functions of pion, kaon, and nucleon are determined by global analyses of hadron-production data in $e^+e^-$ annihilation. It is particularly important that uncertainties of the fragmentation functions are estimated for the first time. We found that light-quark and gluon fragmentation functions have large uncertainties, so that one should be careful in using these functions for hadron-production processes in heavy-ion collisions and lepton scattering. The analysis is extended to possible exotic hadron search by fragmentation functions. We found that internal structure of $f_0 (980)$, such as $s\bar s$ or tetraquark configuration, can be determined by noting differences between favored and disfavored fragmentation functions.

## Key claim

Fragmentation functions for pions, kaons, and nucleons are determined from global analyses of e+e- annihilation data with, for the first time, estimated uncertainties; however, the light-quark and gluon fragmentation functions retain large uncertainties even at next-to-leading order, so conclusions drawn from hadron-production data in heavy-ion collisions and lepton scattering must account for these uncertainties.

## Method

The fragmentation functions D_i^h(z,Q^2) are parametrized at an initial scale Q_0^2 as N_i^h z^{α_i^h} (1-z)^{β_i^h}, with parameters determined by a χ² fit to e+e- → hX cross-section data. Uncertainties are computed using the Hessian method. The analysis is performed at both leading order (LO) and next-to-leading order (NLO) in QCD.

## Key numbers

- Pion fragmentation functions and their uncertainties are shown at Q² = 1 GeV², m_c², or m_b² for both LO and NLO.
- Comparisons of (π⁺+π⁻)/2 fragmentation functions are shown at Q² = 2, 10, or 100 GeV² across different analysis groups (KKP, Kretzer, AKK, DSS).
- For the f₀(980) analysis, the uncertainties of the fragmentation functions are estimated to be ten times larger than the functions themselves.

## Context

This work extends prior fragmentation function parametrizations by KKP (Kniehl, Kramer, Pötter), Kretzer, AKK (Albino, Kniehl, Kramer), and DSS (de Florian, Sassot, Stratmann) by providing the first uncertainty estimates; all prior parametrizations fall within the HKNS uncertainty bands despite large differences in gluon and strange-quark functions. The analysis enables future exotic hadron searches by using differences between favored and disfavored fragmentation functions, specifically for determining whether f₀(980) is an s\bar{s} state or a tetraquark configuration, though current e+e- data are insufficient to distinguish these possibilities.
