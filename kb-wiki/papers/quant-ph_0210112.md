# Explicit Solution of the Time Evolution of the Wigner Function

> Previously, an explicit solution for the time evolution of the Wigner function was presented in terms of auxiliary phase space coordinates which obey simple equations that are analogous with, but not identical to, the classical equations of motion. They can be solved easily and their solutions can be utilized to construct the time evolution of the Wigner function. In this paper, the usefulness of this explicit solution is demonstrated by solving a numerical example in which the Wigner function has strong spatial and temporal variations as well as regions with negative values. It is found that the explicit solution gives a correct description of the time evolution of the Wigner function. We examine next the pseudoparticle approximation which uses classical trajectories to evolve the Wigner function. We find that the pseudoparticle approximation reproduces the general features of the time evolution, but there are deviations. We show how these deviations can be systematically reduced by including higher-order correction terms in powers of $\hbar^2$.

## Key claim
The explicit solution for the time evolution of the Wigner function, expressed in terms of auxiliary phase space coordinates that obey equations analogous to but not identical with classical equations of motion, correctly reproduces the time evolution even when the Wigner function contains regions of negative values; the lowest-order pseudoparticle approximation using classical trajectories reproduces general features but shows deviations that can be systematically reduced by including higher-order correction terms in powers of $\hbar^2$.

## Method
The author formulates an explicit solution for the Wigner function time evolution using an auxiliary coordinate s, where the phase space coordinates R and P obey equations of motion that depend on s. For a one-dimensional Gaussian potential well, a non-stationary state is constructed as an equal superposition of the two lowest-energy even-parity eigenstates, and the Wigner function is propagated forward in time using small time increments via cosine and sine transforms. The pseudoparticle method is then tested by following classical trajectories for the phase space coordinates, with corrections applied by expanding the propagator in powers of $\hbar^2$ to include higher-order spatial derivatives of the potential.

## Key numbers
- Eigenvalues of the two lowest even-parity states in the Gaussian potential (width parameter sigma=3): E0 = -0.844 and E1 = -0.312 (dimensionless units).
- For p=0.6 at t=3, the eigenfunction solution gives a maximum Wigner function value of 0.9313 at x=1.692, while the explicit solution gives 0.9206 at the same position (approximately 1% difference).
- For p=0.6 at t=3, the eigenfunction solution gives a minimum Wigner function value of -0.3888 at x=-1.1667, while the explicit solution gives -0.3735 at the same position.
- The time evolution from t=0 to t=3 was calculated using 30 time steps.

## Context
This work extends the author's earlier formulation (Wong 1982) of an explicit solution for propagating the Wigner function, which overcame the difficulty of handling negative-valued regions by working with amplitudes rather than probabilities. The numerical test uses a Gaussian potential well and compares the explicit solution against results obtained from eigenfunctions and eigenvalues. The pseudoparticle approximation using classical trajectories was previously applied to time-dependent Hartree-Fock calculations (Wong 1982), and the present work provides a systematic method for improving its accuracy through $\hbar^2$ corrections, building on the observation by Zachos and Curtright (1999) regarding the use of classical trajectories for approximate Wigner function evolution.
