# $\texttt{cuSkyrmion}$: A CUDA-OpenGL framework for interactive simulation and visualization of nuclei as Skyrmions

> We introduce $\texttt{cuSkyrmion}$, a 3-dimensional Skyrme model computation and visualization software, that is written in $\texttt{CUDA C}$ for rapid computation and visualization of especially the arrested Newton flow algorithm. The programme is interactive and lets the user construct Skyrmions either with configuration files, specifying coordinates, or simply in run-time using the keyboard and mouse. Rational map ansatz constituent Skyrmions can be inserted at any time and a random generator can produce a stochastic initial configuration. The software is composed into three main modules being a computational module, a rendering module and a main programme. The rendering/visualization module can readily be used by other computational modules and a $\texttt{Python}$-fork, $\texttt{skyrmion_solver}$, has been developed demonstrating the re-usability of the code.

## Key claim

The paper presents `cuSkyrmion` as the first CUDA-accelerated framework that integrates real-time interactive visualization with arrested Newton flow minimization for computing static Skyrmion solutions, but the actual claim is that this enables "direct, real-time inspection of Skyrmion formation, deformation, and symmetry breaking as the flow proceeds" rather than claiming superior accuracy or speed over all existing methods in absolute terms.

## Method

The authors implement a fourth-order finite-difference discretization of the Skyrme model in the sigma-model (vector) formulation on a uniform cubic lattice, using the arrested Newton flow method as a second-order relaxation scheme in fictitious time. The entire minimization procedure is executed on the GPU using CUDA C, with the field represented as a unit four-vector satisfying the point-wise constraint enforced by explicit projection after each update. The software is composed of three main modules: a computational module, a rendering module, and a main programme, with rational map ansatz Skyrmions and product ansatz (smorgasbord) configurations available as initial conditions.

## Key numbers

* The rational map ansatz provides symmetry groups for baryon numbers B=1 through 8, including O(3) for B=1, T_d for B=3, O_h for B=4, and Y_h for B=7, as detailed in Table 1 of the paper.
* The energy and length units are defined as \(\tilde{E}=F_\pi/4e\) (MeV) and \(\tilde{L}=2\hbar/e F_\pi\) (fm), respectively, with the rescaled pion mass given by \(m = 2 m_\pi/F_\pi e\).
* The spatial derivatives are approximated using fourth-order finite difference schemes with coefficients such as \(-1/12\), \(2/3\), \(-2/3\), and \(1/12\) for first derivatives, and \(-1/12\), \(4/3\), \(-5/2\), \(4/3\), and \(-1/12\) for second derivatives on a lattice with spacing \(h_x\).

## Context

This work extends the arrested Newton flow method previously applied to Skyrme-type models in two and three spatial dimensions, and builds on the rational map ansatz of Houghton (1998) and the smorgasbord method of Gudnason (2022) for constructing multi-Skyrmion configurations. The software enables real-time exploration of the Skyrmion energy landscape and computation of physical observables including centre of mass, root-mean-square radius, electric quadrupole tensor, inertia tensors, and stress-tensor derived quantities such as the virial constraint and D-term, which can be compared with modern descriptions of hadrons and nuclei.
