# Simplified Version of the Nucleon and Delta-Isobar Description

## Key claim
The paper formulates a simplified theoretical scheme for describing the nucleon and delta-isobar by assuming synchronous spin-isospin rotation as a collective motion, but this scheme cannot serve as an acceptable alternative to the complete dynamic description (CDD) formalism because field fluctuations in this approach lack clear physical meaning and the wave functions differ radically from those in the CDD formalism.

## Method
The author constructs a collective Hamiltonian by introducing a single set of SU(2) group parameters that describe synchronous rotation in isotopic and spin spaces, using the substitution φ(x) = Tφ(Tx) where T is a rotation matrix. The analysis imposes orthogonality conditions to ensure nondegenerate variable changes, derives the collective Hamiltonian H_c = J²/(2b) + H(φ,0) for stationary states, and shows that dynamic stresses vanish when the condition J + IT = 0 holds, which rigidly binds spin and isospin. The resulting Hamiltonian is then quantized to obtain state vectors for the nucleon and delta-isobar.

## Key numbers
- The collective Hamiltonian takes the form H_c = J²/(2b) + H(φ,0), where b(φ) is defined as b(φ) ≡ k_i k_r (D^i D^r) = -⟨k Î φ^s g_st(φ) k Î φ^t⟩.
- The condition for vanishing dynamic stresses is J + IT = 0, which yields J² = I² for axially-symmetrical configurations.
- The Poisson bracket algebra yields {σ_i, σ_k} = ε_ikl σ_l, {σ_k, T_sn} = ε_ikn T_si, and {σ_k, (Tσ)_i} = 0.

## Context
This work extends the moduli space approximation (MSA) approach used in Skyrme model studies since the early 1980s (ANW, early eighties) and the complete dynamic description (CDD) formalism (Tsukanov, prior work). The simplified scheme reproduces MSA results for nucleon and delta-isobar state vectors while overcoming some MSA limitations, such as allowing rotational deformation and not requiring an obligatory static soliton solution. However, the approach is inadequate for describing the full variety of nucleon resonances and gives an incorrect quantum description compared to the CDD formalism, where resonances are attributed to the soliton spectrum and field fluctuations represent low-amplitude pion meson fields.
