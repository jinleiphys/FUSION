# FSQP nuclear matrix elements for two neutrino double beta decays

> Nuclear matrix elements (NMEs) for two neutrino double beta decays are discussed in terms of the Fermi Surface Quasi-Particle (FSQP) Model. The evaluated NMEs agree with the observed NMEs . The new NMEs for two neutrino double beta decays from Kr78, Cd106, Ba130, and Pd110 were evaluated by FSQP. The NMEs show shell configuration dependence. Impacts of the FSQP NMEs for two neutrino double beta decays on the NMEs for neutrino less double beta decays are discussed.

# FSQP nuclear matrix elements for two neutrino double beta decays

## Key claim
The Fermi Surface Quasi-Particle (FSQP) model, which uses experimental single-beta nuclear matrix elements for low-lying quasi-particle states rather than adjustable parameters, reproduces observed two-neutrino double beta decay nuclear matrix elements and reveals a shell-configuration dependence that may also appear in neutrinoless double beta decay matrix elements, contrary to the general belief that all neutrinoless double beta decay matrix elements are large and nearly identical across nuclei.

## Method
The author evaluates two-neutrino double beta decay nuclear matrix elements using the FSQP model, where the matrix element is expressed as the sum over intermediate FSQP states of the product of the beta-minus and beta-plus Gamow-Teller matrix elements divided by the energy denominator. The single-beta Gamow-Teller matrix elements are obtained from experimental charge exchange reactions and beta/EC decay rates, with the effective axial coupling constant and pairing correlation coefficients derived from data rather than adjusted to fit the double beta decay results. The FSQP matrix elements are then compared with experimental matrix elements extracted from observed half-lives for nine beta-minus-beta-minus decay nuclei and three electron-capture/positron-emitting double beta decay nuclei.

## Key numbers
- For $^{76}$Ge: experimental NME = 0.063, FSQP NME = 0.052 (Q value = 2.039 MeV)
- For $^{100}$Mo: experimental NME = 0.126, FSQP NME = 0.096 (Q value = 3.034 MeV)
- For $^{110}$Pd: FSQP NME = 0.145 (new evaluation, no experimental NME available; predicted half-life = 1.3 × 10$^{20}$ y)
- For $^{136}$Xe: experimental NME = 0.010, FSQP NME = 0.012 (Q value = 2.467 MeV)
- For $^{130}$Ba (ECEC): experimental NME = 0.105, FSQP NME = 0.067 (Q value = 2.610 MeV)
- For $^{78}$Kr (ECEC): FSQP NME = 0.065, experimental upper limit ≤ 0.34 (Q value = 2.866 MeV)
- For $^{106}$Cd (ECEC): FSQP NME = 0.11, experimental upper limit ≤ 0.39 (Q value = 2.770 MeV)
- The FSQP reduction coefficient product $k^-P^-k^+P^+$ ≈ 0.01–0.005 relative to single-particle values
- The effective axial coupling reduction coefficient $k^{eff}$ ≈ 0.2–0.3 for both Gamow-Teller and spin-dipole transitions

## Context
This work extends the FSQP model previously applied by Ejiri (2009, 2012) to additional nuclei including $^{110}$Pd, $^{78}$Kr, $^{106}$Cd, and $^{130}$Ba, and contrasts with quasiparticle random phase approximation (QRPA) calculations by Vogel (1986) and Rodin (2006), which reproduce experimental matrix elements by adjusting the particle-particle interaction parameter $g_{pp}$ and the axial coupling $g_A$. The FSQP results show that two-neutrino and neutrinoless double beta decay matrix elements share a common dependence on shell configuration and pairing factors, contrary to the long-held view that they are independent. This finding impacts the selection of nuclei for future neutrinoless double beta decay experiments, as nuclei near shell closures (like $^{136}$Xe) may have smaller matrix elements than previously assumed. The author encourages experimental studies of two-neutrino ECEC decays of $^{78}$Kr, $^{106}$Cd, $^{110}$Pd, and $^{130}$Ba to verify the FSQP predictions.
