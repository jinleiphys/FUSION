# A new picture for the chiral symmetry properties within a particle-core framework

> The Generalized Coherent State Model, proposed previously for a unified description of magnetic and electric collective properties of nuclear systems, is extended to account for the chiral like properties of nuclear systems. To a phenomenological core described by the GCSM a set of interacting particles are coupled. Among the particle-core states one identifies a finite set which have the property that the angular momenta carried by the proton and neutron quadrupole bosons and the particles respectively, are mutually orthogonal. All terms of the model Hamiltonian satisfy the chiral symmetry except for the spin-spin interaction. The magnetic properties of the particle-core states, where the three mentioned angular momenta are orthogonal, are studied. A quantitative comparison of these features with the similar properties of states, where the three angular momenta belong to the same plane, is performed.

# A new picture for the chiral symmetry properties within a particle-core framework

## Key claim

The paper extends the Generalized Coherent State Model (GCSM) to show that chiral-like configurations, where the angular momenta carried by proton quadrupole bosons, neutron quadrupole bosons, and quasiparticles are mutually orthogonal, can exist within a particle-core framework, but the chiral symmetry is broken by the spin-spin interaction term in the Hamiltonian, meaning the symmetry is not exact but approximate.

## Method

The authors couple a set of interacting quasiparticles (restricted to a single-j shell with two particles) to a phenomenological core described by the GCSM, which uses separate proton and neutron quadrupole bosons. They construct particle-core basis states by combining two-quasiparticle states with angular-momentum-projected dipole boson states, then analytically compute the average angular momenta carried by protons, neutrons, and quasiparticles to determine the angles between these angular momentum vectors. The model Hamiltonian includes a GCSM core term, a spherical shell-model mean field with pairing, a quadrupole-quadrupole particle-core interaction, and a spin-spin interaction.

## Key numbers

- For the pure boson dipole state with equal proton and neutron deformations (d = 0.2), the angle between proton and neutron angular momenta is 90 degrees for the first three dipole states with angular momenta 1, 2, and 3.
- For the pure boson dipole state with unequal deformations (d_p = 0.2, d_n = 2.4), the angle between proton and neutron angular momenta varies between 91.5 degrees and 87 degrees over the considered angular momentum interval.
- For the two-quasiparticle dipole states with equal deformations (d = 0.2) and quasiparticle angular momentum J = 0, 2, 4, 6, 8, 10, or 12, the total angular momenta I for which the proton-neutron angle is 90 degrees are I = J+1, J+2, and J+3.
- For ground band states with J = 0, the proton and neutron angular momenta are anti-aligned (180 degrees) regardless of deformation parameters.

## Context

This work extends the Generalized Coherent State Model (GCSM) previously developed by Raduta et al. to describe collective magnetic and electric properties, including the scissors mode and dipole bands. The paper builds on prior studies of chiral symmetry in nuclear systems, where a triaxial core is coupled to proton and neutron orbitals, and addresses whether the standard picture of three orthogonal angular momenta is unique for producing large M1 transitions. The results enable future identification of chiral bands within a particle-core framework where the core is described by separate proton and neutron quadrupole bosons rather than a single triaxial rotor.
