# Muon capture on deuteron using local chiral potentials

> The muon capture reaction $μ^- + d \rightarrow n + n + ν_μ$ in the doublet hyperfine state is studied using nuclear potentials and consistent currents derived in chiral effective field theory, which are local and expressed in coordinate-space (the so-called Norfolk models). Only the largest contribution due to the $^1S_0$ $nn$ scattering state is considered. Particular attention is given to the estimate of the theoretical uncertainty, for which four sources have been identified: (i) the model dependence, (ii) the chiral order convergence for the weak nuclear current, (iii) the uncertainty in the single-nucleon axial form factor, and (iv) the numerical technique adopted to solve the bound and scattering $A=2$ systems. This last source of uncertainty has turned out essentially negligible. The $^1S_0$ doublet muon capture rate $Γ^D(^1S_0)$ has been found to be $Γ^D(^1S_0)=255.8(0.6)(4.4)(2.9)$ s$^{-1}$, where the three errors come from the first three sources of uncertainty. The value for $Γ^D(^1S_0)$ obtained within this local chiral framework is compared with previous calculations and found in very good agreement.

## Key claim

The doublet muon capture rate on deuteron, restricted to the dominant \(^1S_0\) final-state channel, is calculated using local chiral potentials and consistent currents (Norfolk models) and found to be \(\Gamma^D(^1S_0)=255.8(0.6)(4.4)(2.9)\) s\(^{-1}\); the three uncertainties arise from model dependence, chiral-order convergence of the weak current, and the single-nucleon axial form factor, while the numerical method contributes negligibly.

## Method

The authors computed the muon capture rate \(\mu^- + d \rightarrow n + n + \nu_\mu\) in the doublet hyperfine state using four versions of the local Norfolk potentials (NVIa, NVIb, NVIIa, NVIIb) with consistent weak vector and axial currents derived in chiral effective field theory. Only the \(^1S_0\) partial wave of the final \(nn\) scattering state was retained. The deuteron bound state and \(nn\) scattering wave functions were obtained with both a variational method (Laguerre-polynomial expansion) and a Numerov method to check numerical uncertainty. The single-nucleon axial form factor was parameterized via a small-momentum expansion with axial radius \(r_A^2 = 0.46(16)\) fm\(^2\).

## Key numbers

* \(\Gamma^D(^1S_0) = 255.8(0.6)(4.4)(2.9)\) s\(^{-1}\) for the doublet muon capture rate on deuteron in the \(^1S_0\) channel, where the three errors come from model dependence, chiral-order convergence of the weak current, and the single-nucleon axial form factor uncertainty, respectively.
* The axial charge radius squared used is \(r_A^2 = 0.46(16)\) fm\(^2\), a weighted average from neutrino-deuterium scattering and MuCap experiments.
* The LEC \(c_D\) values for the four Norfolk potentials are: \(-0.635\) (NVIa), \(-4.71\) (NVIb), \(-0.61\) (NVIIa), and \(-5.25\) (NVIIb).
* The \(\chi^2\)/datum for the NVIa, NVIIa, NVIb, and NVIIb potentials are 1.05, 1.37, 1.07, and 1.37, respectively.

## Context

This work extends the consistent chiral effective field theory treatment of muon capture on deuteron to local, coordinate-space potentials (Norfolk models), complementing prior non-local chiral calculations by Adam (2011), Marcucci (2012), and Acharya (2018). The result \(\Gamma^D(^1S_0)=255.8\) s\(^{-1}\) agrees well with the corrected value \(253.5(1.2)\) s\(^{-1}\) from Marcucci (2012) and the \(252.8(4.6)(3.9)\) s\(^{-1}\) from Acharya (2018). This local framework is a necessary step toward calculating the \(pp\) fusion reaction, where Coulomb and electromagnetic effects are more tractable in coordinate space; a full calculation of \(\Gamma^D\) including all partial waves, as well as muon capture on \(A=3\) and 6 nuclei, is reported as underway.
