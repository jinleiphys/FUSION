# Description of the proton and neutron radiative capture reactions in the Gamow shell model

> We formulate the Gamow shell model (GSM) in coupled-channel (CC) representation for the description of proton/neutron radiative capture reactions and present the first application of this new formalism for the calculation of cross-sections in mirror reactions 7Be(p,gamma)8B and 7Li(n,gamma)8Li. The GSM-CC formalism is applied to a translationally-invariant Hamiltonian with an effective finite-range two-body interaction. Reactions channels are built by GSM wave functions for the ground state 3/2- and the first excited state 1/2- of 7Be/7Li and the proton/neutron wave function expanded in different partial waves.

## Key claim

The Gamow shell model in coupled-channel representation (GSM-CC) is formulated for the first time to describe proton and neutron radiative capture reactions, specifically applied to the mirror reactions 7Be(p,gamma)8B and 7Li(n,gamma)8Li, using a translationally-invariant Hamiltonian with an effective finite-range two-body interaction; the abstract omits that the formalism is demonstrated only for low-energy capture and that the coupling constants of the Furutani-Horiuchi-Tamagaki interaction are adjusted in the present work.

## Method

The GSM-CC formalism expands the many-body eigenstates in a complete basis of channel states built from Gamow shell model wave functions for the target (ground state 3/2- and first excited state 1/2- of 7Be/7Li) and projectile wave functions expanded in different partial waves using the Berggren ensemble of single-particle states. The coupled-channel equations are derived from the Schrödinger equation projected onto channel basis states, with antisymmetry between projectile and target treated explicitly for low-energy states and neglected for high-energy states. The resulting non-local potentials are handled using the equivalent potential method with a smoothing function to cancel divergences near zeros of the radial wave functions.

## Key numbers

- The 7Be(p,gamma)8B reaction cross-section is calculated at center-of-mass energies down to approximately 20 keV (solar energies), where the cross-section is too small to be directly measurable.
- The 7Li(n,gamma)8Li reaction cross-section is calculated at very low energies relevant for primordial nucleosynthesis bridging the mass A=8 gap.
- The smoothing function parameter alpha in the equivalent potential method varies in the interval 10 < alpha < 100.

## Context

This work extends the GSM-CC formalism previously applied to elastic/inelastic scattering and excited states in 18Ne and 19Na (Jaganathen 2012, 2014) to radiative capture reactions. It provides a unified theoretical framework for nuclear structure and reactions that goes beyond earlier approaches such as the potential model (Kim 1987), R-matrix approach (Barker 1995), shell model embedded in the continuum (Bennaceur 1999), microscopic cluster model (Descouvemont 2004), and the no-core shell model with resonating-group method (Navratil 2011). The formalism enables future applications to ab initio no-core GSM calculations and reactions with heavier projectiles like alpha-particles.
