# $U(1)$ current from the AdS/CFT: diffusion, conductivity and causality

> For a holographically defined finite temperature theory, we derive an off-shell constitutive relation for a global $U(1)$ current driven by a weak external non-dynamical electromagnetic field. The constitutive relation involves an all order gradient expansion resummed into three momenta-dependent transport coefficient functions: diffusion, electric conductivity, and "magnetic" conductivity. These transport functions are first computed analytically in the hydrodynamic limit, up to third order in the derivative expansion, and then numerically for generic values of momenta. We also compute a diffusion memory function, which, as a result of all order gradient resummation, is found to be causal.

## Key claim

The paper derives an off-shell constitutive relation for a global $U(1)\) current in a holographically defined finite temperature theory, where the transport coefficient functions (diffusion, electric conductivity, and magnetic conductivity) are resummed to all orders in the gradient expansion, and this resummation restores causality in the diffusion memory function.

## Method

The authors solve the bulk Maxwell equations in a static Schwarzschild-AdS\(_5\) black brane geometry for a probe Maxwell field, using unspecified boundary conditions \(A_\mu^{(0)}\) imposed at the conformal boundary as sources. They decompose the bulk field corrections into a basis constructed from the boundary fields \(\rho(x_\alpha)\) and \(A_\mu^{(0)}(x_\alpha)\), reducing the problem to a set of ordinary differential equations for the decomposition coefficients in Fourier space. The transport coefficient functions \(\mathcal{D}(\omega, q^2)\), \(\sigma_e(\omega, q^2)\), and \(\sigma_m(\omega, q^2)\) are then read off from the near-boundary expansion of these coefficients, computed analytically in the hydrodynamic limit up to third order in derivatives and numerically for generic momenta.

## Key numbers

* Diffusion constant \(\mathcal{D}_0 = 1/2\) (in units where \(\pi T = 1\)), consistent with prior work by Policastro, Son, and Starinets (2002).
* DC electric conductivity \(\sigma_e^0 = 1\) (in units where \(\pi T = 1\)), consistent with results of Herzog (2007).
* Relaxation time for electric conductivity \(\tau_e = \log 2 / (2\pi T)\).
* Diffusion relaxation time \(\tau_D = \pi/8\) (in dimensionless units with \(\pi T = 1\)).
* The hydrodynamic expansion of \(\sigma_m\) starts from a term linear in \(\omega\): \(\sigma_m = (i\omega/16)(2\pi - \pi^2 + 4\log 2) + \cdots\), rather than from a constant.
* The ratio of viscosity to entropy density for theories with Einstein gravity duals is \(\eta_0 / s = 1/(4\pi)\).

## Context

This work extends the all-order gradient resummation approach previously applied to the stress-energy tensor by Bu, Lublinsky, and Sharon (2014, 2015) to the case of a conserved \(U(1)\) current, building on the fluid/gravity correspondence framework of Bhattacharyya et al. (2008). It generalizes prior holographic studies of R-charge diffusion and conductivity by Policastro, Son, and Starinets (2002), Herzog (2007), and others, providing new results for the magnetic conductivity \(\sigma_m\) and the momentum-dependent transport functions beyond the hydrodynamic limit. The causal memory function derived here enables a well-defined initial value problem for diffusion, addressing the causality violation present in the simple diffusion equation.
