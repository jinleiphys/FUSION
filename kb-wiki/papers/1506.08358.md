# Analysis of the effect of core structure upon dineutron correlation using antisymmetrized molecular dynamics

> We extend the method of antisymmetrized molecular dynamics to investigate dineutron correlation. We apply this method to $^{10}$Be as an example and investigate the motion of two neutrons around a largely deformed $^8$Be core by analyzing the two-neutron overlap function around the core. We show that the core structure plays an important role in dineutron formation and expansion from the core and that the present framework is effective for the studies of dineutron correlation.

## Key claim

The core structure (specifically the deformation and shape fluctuation of the \(^8\)Be core) significantly affects dineutron formation and expansion, with larger core deformation enhancing dineutron correlation by weakening the spin-orbit dissociation effect at the nuclear surface, but the abstract does not specify that this enhancement is measured by the mixing ratio of different-parity single-particle orbits rather than by the total spin-singlet probability.

## Method

The authors extended antisymmetrized molecular dynamics (AMD) to describe dineutron correlation by treating \(^{10}\)Be as a \(^8\)Be core plus two valence neutrons, using a \(d_{2n}\)-constraint energy variation to control the core–2n distance and varying the Gaussian width \(\nu_{2n}\) to account for dineutron size changes. They prepared core wave functions with three \(\alpha\)-\(\alpha\) distances (\(d_\alpha = 2, 3, 4\) fm) and superposed bases with different \(d_{2n}\) values (eight values with spin-orbit force, four values without spin-orbit force for \(d_{2n} < 2\) fm) and three \(\nu_{2n}\) values (0.235, 0.125, 0.08 fm\(^{-2}\)), totaling 108 basis states. They analyzed the two-neutron overlap function \(f^{S=0}_{l=L=0}\) projected onto spin-singlet, relative \(s\)-wave, and center-of-mass \(S\)-wave components to quantify dineutron correlation via the asymmetry between dineutron and cigar peaks.

## Key numbers

- Two-neutron separation energy \(S_{2n} = 6.71\) MeV in the full-\(d_\alpha\) calculation (with Majorana parameter \(m = 0.60\)) and \(S_{2n} = 6.34\) MeV in the fixed-\(d_\alpha\) calculation (with \(m = 0.64\)); experimental value is 8.48 MeV.
- Matter radius \(r_m = 2.43\) fm (full-\(d_\alpha\)) and 2.33 fm (fixed-\(d_\alpha\)); experimental value is \(2.30 \pm 0.02\) fm.
- Neutron spin expectation value \(\langle S_n^2 \rangle = 0.39\) (full-\(d_\alpha\)) and 0.50 (fixed-\(d_\alpha\)), indicating stronger spin-triplet mixing (LS dissociation) in the fixed-\(d_\alpha\) calculation.
- Probability of spin-singlet \(2n\) pair in \(l=L=0\) component: \(P^{S=0}_{l=L=0} = 0.515\) (full-\(d_\alpha\)) and 0.546 (fixed-\(d_\alpha\)).
- Mixing ratio \(P^{S=0(++)}_{l=L=0} / P^{S=0(--)}_{l=L=0} = 5.1\%\) (full-\(d_\alpha\)) and 3.0\% (fixed-\(d_\alpha\)), showing enhanced dineutron correlation with core deformation.
- Root-mean-square relative distance \(\sqrt{\langle r^2 \rangle^{S=0}_{l=L=0}} = 3.70\) fm (full-\(d_\alpha\)) and 3.75 fm (fixed-\(d_\alpha\)).
- Root-mean-square center-of-mass distance \(\sqrt{\langle r_G^2 \rangle^{S=0}_{l=L=0}} = 2.53\) fm (full-\(d_\alpha\)) and 2.37 fm (fixed-\(d_\alpha\)).

## Context

This work extends the authors' previous studies using dineutron condensate (DC) wave functions (Kobayashi et al., 2012, 2013, 2014) by incorporating core structure changes explicitly via AMD, going beyond the inert spherical core assumption of three-body models (Bertsch et al., 1991; Zhukov et al., 1993; Hagino et al., 2005, 2007). The method enables systematic investigation of how core deformation and excitation affect dineutron correlation, and the analysis using the \(2n\) overlap function provides a visualization tool applicable to other neutron-rich nuclei. The framework can be generalized to treat deformed cores via \(\beta\)-constraint variation, opening the way to study dineutron correlation in heavier systems.
