# Precise description of medium-induced emissions

> We study jet fragmentation via final-state parton splittings in the medium. These processes are usually calculated theoretically by invoking the large-$N_c$ limit. In this paper we perform the first computation of a $1\to2$ parton splitting in a thermal medium at finite numbers of colors $N_c$, for arbitrary momentum-sharing fraction $z$ and with full transverse dynamics. We show how the problem can be transformed into a system of coupled Schrödinger equations, that we solve numerically. The novel numerical results are used to estimate the accuracy of several widely used approximations. We check the error introduced while going from finite $N_c$ (i.e. $N_c =3$) to the large-$N_c$ limit, which we find to be small. For unbalanced splittings, e.g. when $z\to 0$, only one of the partons is affected by transverse momentum exchanges with the medium. The emission process then separates into a term responsible for the $1 \to 2$ splitting and the subsequent independent broadening of the daughter partons. This is also referred to as the factorizable term. For finite $z$, further contributions arise that are responsible for the coherent color dynamics of the two-parton system, and these are referred to as non-factorizable terms. These were argued to be small for soft (unbalanced) splittings and for large media. In this work we therefore determine the accuracy of keeping only the factorizable term of the large-$N_c$ solution. We find that the error is insignificant at a small splitting fraction $z \sim 0$, but can be sizable in a more balanced splitting with $z\sim 0.5$. Finally, we also examine the eikonal approximation, which amounts to approximating the partons' paths through the medium as straight lines. We find that it is associated with a substantial error for the parameter values we explored in this work.

## Key claim

The first computation of a $1\to2$ parton splitting in a thermal medium at finite number of colors $N_c$ (specifically $N_c=3$) with full transverse dynamics shows that the large-$N_c$ approximation introduces only a small error, while the factorizable approximation (keeping only the term where emission separates from independent broadening) introduces negligible error for soft splittings ($z\sim0$) but sizable error for balanced splittings ($z\sim0.5$), and the eikonal approximation (straight-line parton trajectories) is associated with substantial error for the parameter values explored.

## Method

The problem was transformed into a system of coupled Schrödinger equations describing the color dynamics of the two-parton system after splitting, which were solved numerically. The medium-induced splitting function was derived in both momentum-space and position-space formulations, incorporating the quadrupole correlation function that governs the full dynamics of the daughter parton pair and accounts for all possible intermediate color representations. The numerical solutions were used to evaluate the accuracy of three widely used approximations: the large-$N_c$ limit, the factorizable approximation (keeping only the term where the $1\to2$ splitting separates from independent broadening of daughters), and the eikonal approximation (straight-line trajectories).

## Context

This work extends previous calculations of medium-induced splitting functions that were performed only in the large-$N_c$ limit (Blaizot 2012) or for soft emissions with $z\ll1$ (Wiedemann 2000, Andres 2020, Barata 2021). It also goes beyond the eikonal approximation introduced by Dominguez (2019) for hard emissions with $z\approx1/2$, which neglected transverse momentum broadening entirely. The results enable precise quantification of approximation errors in jet quenching calculations and provide a framework for extending the description of multiple in-medium splittings beyond the probabilistic, Markovian picture.
