# Constraining quark angular momentum through semi-inclusive measurements

> The determination of quark angular momentum requires the knowledge of the generalized parton distribution E in the forward limit. We assume a connection between this function and the Sivers transverse-momentum distribution, based on model calculations and theoretical considerations. Using this assumption, we show that it is possible to fit at the same time nucleon magnetic moments and semi-inclusive single-spin asymmetries. This imposes additional constraints on the Sivers function and opens a plausible way to quantifying quark angular momentum.

## Key claim

The authors assume a simple relation between the generalized parton distribution \(E^a\) in the forward limit and the Sivers transverse-momentum distribution \(f_{1T}^{\perp a}\), based on model calculations and theoretical considerations, and show that this assumption is compatible with existing semi-inclusive deep-inelastic scattering data and nucleon magnetic moments, thereby opening a plausible way to quantify quark angular momentum.

## Method

The authors perform a combined \(\chi^2\) fit to 105 HERMES proton data points, 104 COMPASS deuteron data points, and 8 JLab neutron data points of the Sivers asymmetry in semi-inclusive deep-inelastic scattering (SIDIS), together with the known values of the proton and neutron anomalous magnetic moments. They parametrize the Sivers function using a Gaussian transverse-momentum distribution and relate it to the GPD \(E^a\) via a flavor-independent lensing function \(L(x) = K/(1-x)^\eta\). The fit determines up to 10 free parameters, including the lensing parameters, the Sivers function normalizations, and the width parameter \(M_1\), at a starting scale \(Q_0 = Q_L = 1\) GeV.

## Key numbers

* Best-fit values for the Sivers function parameters (with \(C^{s_v} = C^{\bar{s}} = 0\)): \(C^{u_v} = -0.229 \pm 0.002\), \(C^{d_v} = 1.591 \pm 0.009\), \(C^{\bar{u}} = 0.054 \pm 0.107\), \(C^{\bar{d}} = -0.083 \pm 0.122\); \(M_1 = 0.346 \pm 0.015\) GeV; lensing parameters \(K = 1.888 \pm 0.009\) GeV, \(\eta = 0.392 \pm 0.040\); node parameter \(\alpha^{u_v} = 0.783 \pm 0.001\).
* Valence quark contributions to the anomalous magnetic moment: \(\kappa^{u_v} = 1.673 \pm 0.003^{+0.011}_{-0.000}\), \(\kappa^{d_v} = -2.033 \pm 0.002^{+0.011}_{-0.000}\), \(\kappa^{s_v} = 0^{+0.011}_{-0.000}\).
* Total longitudinal angular momentum at \(Q^2 = 4\) GeV\(^2\): \(J^u = 0.229 \pm 0.002^{+0.008}_{-0.012}\), \(J^d = -0.007 \pm 0.003^{+0.020}_{-0.005}\), \(J^{\bar{u}} = 0.015 \pm 0.003^{+0.001}_{-0.000}\), \(J^{\bar{d}} = 0.022 \pm 0.005^{+0.001}_{-0.000}\), \(J^s = 0.006^{+0.002}_{-0.006}\), \(J^{\bar{s}} = 0.006^{+0.000}_{-0.005}\).
* Total contribution to the nucleon spin from quarks and antiquarks at \(Q^2 = 4\) GeV\(^2\): \(0.271 \pm 0.007^{+0.032}_{-0.028}\), of which 85% is carried by the up quark.
* The fit yields \(\chi^2/\text{dof} = 1.323\) for the best scenario.

## Context

This work extends the approach of Burkardt (2002), who proposed a connection between the GPD \(E\) and the Sivers function, and builds on model calculations from spectator models (Burkardt 2003, Lu 2006, Meissner 2007, Bacchetta 2008, Bacchetta 2010). It provides a simultaneous description of SIDIS Sivers asymmetry data from HERMES, COMPASS, and JLab, together with nucleon anomalous magnetic moments, which previous extractions of the Sivers function (Vogelsang 2005, Anselmino 2005, Anselmino 2008, Arnold 2008) did not incorporate. The results enable a flavor-decomposed estimate of quark angular momentum, comparable to analyses of the GPD \(E\) from deeply virtual Compton scattering (Diehl 2004, Guidal 2004, Ahmad 2006, Goloskokov 2008) and to lattice QCD calculations (Bratt 2010).
