# Analytic continuation of Green's functions with a neural network

> An important problem in many-body physics is to reconstruct the spectral density from the imaginary-time domain Green's function. Typically, the imaginary-time Green's function is generated by Monte Carlo methods. As the one-point fermionic kernel diverges exponentially for large frequencies, numerical noise generically causes instabilities. We use a convolutional neural network to obtain the spectral density for a given imaginary time Green's function. The network is trained by data which we generate using random Gaussians. We improve the training data set available by including collision centers for the Gaussians rather than employing uniformly distributed Gaussians. Our network is constructed in such a way that its output fulfills positive semidefiniteness. We compare the results of our network with results of the Maximum Entropy method (MaxEnt), a standard method for the same reconstruction problem for the spectral density. This comparison is performed for three different cases, namely our Gaussian based test data as well as two physical models, the 1d Hubbard model showing spin-charge separation, and the two-dimensional SSH model in the self-consistent Born approximation. We find that the network outperforms MaxEnt when presented data close to the training set. For the physical models considered, MaxEnt recognizes physical features more precisely as compared to our network prediction. While it is hard to improve MaxEnt, the quality of the network depends on the training data set which can be systematically enhanced and improved.

## Key claim

The authors claim that a convolutional neural network can outperform the Maximum Entropy method (MaxEnt) for analytic continuation of imaginary-time Green's functions when presented with data similar to its training set, but acknowledge that for physically motivated test cases (the 1D Hubbard model and the 2D SSH model), MaxEnt currently recognizes physical features more precisely than the network.

## Method

The authors generate artificial spectral density functions from random Gaussian peaks arranged around collision centers, with added Heaviside step functions and low-pass filtering to increase complexity. They compute the corresponding imaginary-time Green's functions via the forward integral kernel, add pink noise to mimic Monte Carlo correlations, and train a convolutional neural network with three convolutional layers, three dense layers, and a deconvolution section, using a combined L1+L2 loss function and stochastic gradient descent. The network output is constrained to be positive semidefinite via ReLU activation and is normalized to satisfy the sum rule.

## Key numbers

- Training set size: 35,008 samples; test set size: 3,200 samples; generated at inverse temperatures beta = 10 and beta = 20.
- Network trained for 2,000 epochs with learning rates decreasing from 10^{-2} to 10^{-5}.
- On 64 validation samples, network achieved MSE = 0.108, MAE = 0.896, Wasserstein distance = 0.511; MaxEnt achieved MSE = 0.121, MAE = 0.950, Wasserstein distance = 0.559.
- Gaussian peaks generated with variance sigma_Gauss in [0.1, 3.0]; collision centers uniformly distributed in omega_c in [-7.5, 7.5]; number of collision centers n_col = 4 to 8; number of Gaussians per center n_Gauss = 1 to 12.
- Step functions added to 20% of generated data, with n_step = 1 to 3 and step height h_step in [-5, 5].
- Pink noise exponent alpha randomly selected from [1, 5].

## Context

This work extends the machine-learning approach to analytic continuation pioneered by the kernel ridge regression method of regression_kernel, which used uniformly distributed Gaussians for training. The authors compare their neural network against the classic Maximum Entropy method of Jarrell (1996) and Linden (1995), and apply it to the 1D Hubbard model showing spin-charge separation (studied using auxiliary-field quantum Monte Carlo methods by the ALF collaboration) and the 2D SSH model within the self-consistent Born approximation (Goetz, 2021). The results suggest that systematic improvement of the training data generation scheme could enable neural networks to surpass MaxEnt, and the authors propose future training on experimental spectroscopic data and results from approximation methods.
