# HFBTHO-AD: Differentiation of a nuclear energy density functional code

> The HFBTHO code implements a nuclear energy density functional solver to model the structure of atomic nuclei. HFBTHO has previously been used to calibrate energy functionals and perform sensitivity analysis by using derivative-free methods. To enable derivative-based optimization and uncertainty quantification approaches, we must compute the derivatives of HFBTHO outputs with respect to the parameters of the energy functional, which are a subset of all input parameters of the code. We use the algorithmic/automatic differentiation (AD) tool Tapenade to differentiate HFBTHO. We compare the derivatives obtained using AD against finite-difference approximation and examine the performance of the derivative computation.

## Key claim

The paper demonstrates that algorithmic differentiation (AD) can be successfully applied to the HFBTHO nuclear energy density functional solver to compute derivatives of outputs with respect to energy functional parameters, enabling derivative-based optimization and uncertainty quantification that previously relied on derivative-free methods. The authors acknowledge that this initial effort required disabling certain physics modules (collective inertia package and finite-range Gogny package) and that the eigensolver routine DSYEVR could not be differentiated through its source code, requiring a hand-coded higher-level derivative rule.

## Method

The authors used the Tapenade source-transformation AD tool to differentiate the HFBTHO Fortran code, specifically the top-level routine HFBTHO_DFT_SOLVER, generating a new source code that computes derivatives in forward mode. They identified 14 input parameters (the Skyrme energy functional parameters) and 52 outputs (arrays eresu and ala2 containing nuclear observables), then used vector forward mode AD to compute derivatives with respect to all inputs simultaneously. The AD-generated code was integrated into the existing build system, with OpenMP parallel loops handled by Tapenade and the DSYEVR eigensolver replaced by a hand-coded forward-mode derivative formula based on the assumption of distinct eigenvalues.

## Key numbers

- The UNEDF0 dataset contains 108 experimental values associated with 72 different nucleus configurations
- The parameter space has 12 input parameters for the UNEDF0 calibration (14 for the UNEDF2 configuration used in this work)
- Strong-scaling tests used 8 OpenMP threads per MPI process for non-AD evaluations (78% parallel efficiency) and 4 threads per process for AD-based evaluations (71% parallel efficiency)
- Non-AD jobs used 32 MPI processes across two nodes; AD-based jobs used 32 MPI processes on one node
- The OpenMP stack size was set to 16 MB for non-AD configurations and 10 GB for AD configurations to accommodate the memory needs of Tapenade-generated differentiation code

## Context

This work extends prior calibration efforts by the Universal Nuclear Energy Density Functional (UNEDF) collaboration, which used derivative-free optimization methods (Kortelainen 2010, Kortelainen 2012, Kortelainen 2014, Wild 2015) because HFBTHO outputs did not include derivatives with respect to calibration parameters. The paper builds on the UNEDF0 study (Kortelainen 2010) and uses its parameter points (SLy4, UNEDFnb, UNEDFpre) for validation. The AD approach enables future derivative-based optimization and uncertainty quantification methods that could improve upon the derivative-free approaches previously required, though the authors note that the eigensolver differentiation issue and the need to disable certain physics modules represent current limitations.
