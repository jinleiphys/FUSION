# Consistent large-scale shell-model analysis of the two-neutrino $ββ$ and single $β$ branchings in $^{48}\rm Ca$ and $^{96}\rm Zr$

> Two-neutrino double-beta-decay matrix elements $M_{2ν}$ and single beta-decay branching ratios were calculated for $^{48}$Ca and $^{96}$Zr in the interacting nuclear shell model using large single-particle valence spaces with well-tested two-body Hamiltonians. For $^{48}$Ca the matrix element $M_{2ν}=0.0511$ is obtained, which is 5.5\% smaller than the previously reported value of 0.0539. For $^{96}$Zr this work reports the first large-scale shell-model calculation of the nuclear matrix element, yielding a value $M_{2ν}=0.0747$ with extreme single-state dominance. If the scenario where the first $1^+$ state in $^{96}$Nb is at 694.6 keV turns out to be correct, the matrix element is increased to 0.0854. These matrix elements, combined with the available $ββ$-decay half-life data, yield effective values of the weak axial coupling which in turn are used to produce in a consistent way the $β$-decay branching ratios of $(7.5\pm2.8)$ % for $^{48}$Ca and $(18.4\pm0.09)$ % for $^{96}$Zr. These are larger than obtained in previous studies, implying that the detection of the $β$-decay branches could be possible in dedicated experiments sometime in the (near) future.

## Key claim

The paper reports that the two-neutrino double-beta-decay matrix element for ⁴⁸Ca is 5.5% smaller than previously reported, and provides the first large-scale shell-model calculation for ⁹⁶Zr, finding a matrix element of 0.0747 with extreme single-state dominance; however, if the first 1⁺ state in ⁹⁶Nb is at 694.6 keV, the matrix element increases to 0.0854.

## Method

The authors performed interacting nuclear shell-model calculations using the code NuShellX@MSU with large single-particle valence spaces. For ⁴⁸Ca they used the full fp model space with the GXPF1A interaction, including 9470 intermediate 1⁺ states up to 60 MeV excitation energy. For ⁹⁶Zr they used a model space including proton orbitals 0f₅/₂, 1p₃/₂, 1p₁/₂, 0g₉/₂ and neutron orbitals 0g₇/₂, 1d₅/₂, 1d₃/₂, 0s₁/₂ with the glekpn interaction, including 5894 intermediate 1⁺ states up to roughly 18 MeV. The computed matrix elements were combined with experimental half-lives to extract effective values of the weak axial coupling gA, which were then used to calculate beta-decay branching ratios.

## Key numbers

* ⁴⁸Ca: |M₂ν| = 0.0511, which is 5.5% smaller than the previously reported value of 0.0539
* ⁹⁶Zr: |M₂ν| = 0.0747 (using shell-model excitation energies); increases to 0.0854 if the first 1⁺ state in ⁹⁶Nb is at 694.6 keV
* Effective axial coupling gA^eff = 0.80 ± 0.04 for ⁴⁸Ca
* Effective axial coupling gA^eff = 1.04^{+0.03}_{-0.02} for ⁹⁶Zr (0.97^{+0.03}_{-0.02} with the 694.6 keV assumption)
* Beta-decay branching ratio for ⁴⁸Ca: (7.5 ± 2.8)% to the 5⁺ state
* Beta-decay branching ratio for ⁹⁶Zr: (18.4 ± 0.09)% to the 5⁺ state
* ⁴⁸Ca Qββ = 4269.08(8) keV, Qβ = 279(5) keV
* ⁹⁶Zr Qββ = 3356.03(7) keV, Qβ = 163.97(10) keV

## Context

This work extends previous shell-model studies of ⁴⁸Ca by Horoi (2007) and Haaranen (2014), providing an updated matrix element with a more complete set of intermediate states (9470 versus 250). For ⁹⁶Zr it provides the first large-scale shell-model calculation, confirming the extreme single-state dominance observed experimentally by Thies (2012) in a high-resolution ⁹⁶Zr(³He,t) experiment. The consistent treatment of both double-beta and single-beta decays using the same effective gA values enables improved predictions for beta-decay branching ratios that are larger than those from earlier QRPA calculations by Heiskanen (2007) and the smaller-model-space shell-model study by Alanssari (2016), suggesting that experimental detection of these beta-decay branches may be possible in dedicated underground experiments.
