# Demonstration of the Rodeo Algorithm on a Quantum Computer

> The rodeo algorithm is an efficient algorithm for eigenstate preparation and eigenvalue estimation for any observable on a quantum computer. This makes it a promising tool for studying the spectrum and structure of atomic nuclei as well as other fields of quantum many-body physics. The only requirement is that the initial state has sufficient overlap probability with the desired eigenstate. While it is exponentially faster than well-known algorithms such as phase estimation and adiabatic evolution for eigenstate preparation, it has yet to be implemented on an actual quantum device. In this work, we apply the rodeo algorithm to determine the energy levels of a random one-qubit Hamiltonian, resulting in a relative error of $0.08\%$ using mid-circuit measurements on the IBM Q device Casablanca. This surpasses the accuracy of directly-prepared eigenvector expectation values using the same quantum device. We take advantage of the high-accuracy energy determination and use the Hellmann-Feynman theorem to compute eigenvector expectation values for a different random one-qubit observable. For the Hellmann-Feynman calculations, we find a relative error of $0.7\%$. We conclude by discussing possible future applications of the rodeo algorithm for multi-qubit Hamiltonians.

# Demonstration of the Rodeo Algorithm on a Quantum Computer

## Key claim
The rodeo algorithm achieves a relative error of 0.08% for energy eigenvalues of a random one-qubit Hamiltonian on the IBM Q device Casablanca, surpassing the accuracy of directly-prepared eigenvector expectation values on the same device, but only when the initial state has sufficient overlap probability with the desired eigenstate.

## Method
The authors implemented the rodeo algorithm on a two-qubit system (one object qubit and one ancilla qubit) using mid-circuit measurements on the IBM Q device Casablanca. They applied N=3 successive cycles of controlled time evolution with random time durations sampled from a Gaussian distribution, followed by phase rotations and ancilla measurements, scanning the target energy E across the energy domain to identify peaks in the success probability distribution. For Hellmann-Feynman calculations, they computed energy eigenvalues of H_obj(phi) = H^(0) + phi H^(1) for small phi values and extracted expectation values of H^(1) from numerical derivatives.

## Key numbers
- Relative error of 0.08% for energy eigenvalues of H^(0) using the rodeo algorithm, averaged over both eigenstates
- Relative error of 0.7% for Hellmann-Feynman calculations of eigenvector expectation values of H^(1)
- N = 3 successive rodeo cycle measurements used for all quantum circuits
- 6 total CNOT gates per circuit (2 per cycle)
- For the upper eigenvalue E_1: 2500 measurements for 25 random sets of t_k
- For the lower eigenvalue E_2: 5000 measurements for 50 random sets of t_k
- Rodeo result for <H^(0)> for |psi_1(0)>: 1.00681(66) versus exact 1.00690
- Rodeo result for <H^(0)> for |psi_2(0)>: -1.1750(12) versus exact -1.1768
- Rodeo result for <H^(1)> for |psi_1(0)>: -0.8338(89) versus exact -0.8254
- Rodeo result for <H^(1)> for |psi_2(0)>: -0.868(14) versus exact -0.8653
- Directly-prepared eigenvector results without error mitigation: relative error of 5% for H^(0) expectation values
- Directly-prepared eigenvector results with measurement error mitigation: relative error of 0.2% for H^(0) expectation values

## Context
This work presents the first experimental implementation of the rodeo algorithm on a quantum device, following its theoretical introduction by Choi (2020). The algorithm is exponentially faster than quantum phase estimation (Abrams, 1999; Cleve, 1998) and adiabatic evolution (Farhi, 2000; Wiebe, 2011) for eigenstate preparation. The demonstration on a one-qubit Hamiltonian establishes a benchmark for future applications to multi-qubit Hamiltonians using Suzuki-Trotter decomposition, with potential applications in nuclear physics for computing spectra and spectral response functions of atomic nuclei, as well as in condensed matter physics and quantum chemistry.
