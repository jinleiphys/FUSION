# Constraining input parameters of AMPT model with $ϕ$ meson production

> A Multi-Phase Transport Model (AMPT) has been extensively used to understand the physics behind the experimental observation. Like other models, the outcome of the AMPT model depends on the initial parameters. Therefore, one needs to choose suitable initial parameters before using the model. Lund string fragmentation function has been used to create quark-antiquarks pairs in the AMPT model. The Lund string fragmentation parameters determine the yields and transverse momentum ($p_{T}$) spectra of particles produced in nucleus-nucleus collisions. The values of Lund string fragmentation parameters were determined by fitting the charged particle yield and $p_{T}$ spectra measured in the experiment. In this paper, we have shown the yield of strange quarks carrying hadrons, e.g. $ϕ$ mesons, are more sensitive to Lund string fragmentation parameters compared to non-strange pions. Using $ϕ$ meson spectra measured at RHIC, we have obtained new sets of Lund parameters for Au+Au collisions at $\sqrt{s_{\mathrm {NN}}}$ = 11.5, 39, and 200 GeV. We have found that using the same set of parameters, we can explain $ϕ$-meson yield at $\sqrt{s_{\mathrm {NN}}}$ = 39 and 200 GeV, however, we need a different set of parameters for $\sqrt{s_{\mathrm {NN}}}$ = 11.5 GeV. This suggests that at low energy, $\sqrt{s_{\mathrm {NN}}}$ = 11.5 GeV, the underlying mechanism for particle production is different compared to top RHIC energies. We have also predicted invariant yield of $π$ and $ϕ$ mesons as a function of $p_{T}$ in U+U collisions at $\sqrt{s_{\mathrm {NN}}}$ = 196 GeV to be measured by STAR experiment.

## Key claim

The yield of strange-quark-carrying hadrons, particularly phi mesons, is more sensitive to the Lund string fragmentation parameters (a and b) in the AMPT model than the yield of non-strange pions. Using the same set of parameters (a=0.55, b=0.15) can explain both pion and phi meson spectra at sqrt(s_NN) = 39 and 200 GeV, but a different set (a=2.2, b=0.15) is needed for sqrt(s_NN) = 11.5 GeV, suggesting a different underlying particle production mechanism at low energy.

## Method

The authors used the string-melting version of the AMPT model (AMPT-SM) to simulate Au+Au collisions at sqrt(s_NN) = 11.5, 39, and 200 GeV. They calculated the transverse momentum spectra of pions and phi mesons using two sets of Lund string fragmentation parameters (set 1: a=0.55, b=0.15; set 2: a=2.2, b=0.15) and compared the model results to experimental data from the STAR experiment. They also implemented a deformed Woods-Saxon profile for the uranium nucleus in the AMPT model to predict pion and phi meson spectra in U+U collisions at sqrt(s_NN) = 196 GeV.

## Key numbers

* At sqrt(s_NN) = 11.5 GeV, changing the Lund parameter a from 0.55 to 2.2 (with b=0.15 fixed) changes the pion spectra by less than 15%, but changes the phi meson spectra by about 50% at low pT.
* At sqrt(s_NN) = 11.5 GeV, the same parameter change alters kaon spectra by about 20-25% and proton spectra by about 10-15%.
* The invariant yield of pions and phi mesons in U+U collisions at sqrt(s_NN) = 196 GeV is predicted to be about 20% higher than in Au+Au collisions at sqrt(s_NN) = 200 GeV.
* The parton-parton cross-section was set to 3 mb (using strong coupling constant alpha_S = 0.33) for both parameter sets.
* The hadronic cascade termination time was taken to be 30 fm/c.
* For the uranium nucleus, the Woods-Saxon parameters used were: radius R0 = 6.81 fm, surface diffuseness a = 0.55 fm, deformation parameters beta2 = 0.28 and beta4 = 0.093.

## Context

This work extends the use of the AMPT model by constraining its Lund string fragmentation parameters using phi meson production data from the RHIC Beam Energy Scan program, building on prior parameter determinations that used only charged particle yields (Lund parameter references). The finding that a different parameter set is needed at sqrt(s_NN) = 11.5 GeV is consistent with the STAR experiment's observation (phi_omega_star reference) of a change in the underlying mechanism for particle production at low energies. The predictions for U+U collisions at sqrt(s_NN) = 196 GeV are intended for comparison with future STAR experiment measurements, and the implementation of the deformed uranium nucleus in AMPT enables studies of how initial geometry affects particle production.
