# Fission barriers heights in A$\sim$ 200 mass region

> Statistical model analysis has been carried out for $p$ and $α$ induced fission reactions using a consistent description for fission barrier and level density in A $\sim$ 200 mass region. A continuous damping of shell correction with excitation energy have been considered. Extracted fission barriers agree well with the recent microscopic-macroscopic model. The shell corrections at the saddle point were found to be not significant.

# Fission barriers heights in A~200 mass region

## Key claim
The extracted fission barriers for nuclei in the A~200 mass region agree well with recent microscopic-macroscopic model predictions, but the shell corrections at the saddle point are found to be not significant. This conclusion depends on using an energy-dependent shell damping factor (η = 0.054 + 0.002×E*) rather than the standard energy-independent damping factor, which alone could not fit the excitation functions across the entire energy range.

## Method
Statistical model analysis using the code PACE with a modified prescription for fission barrier and level density was performed for p+²⁰⁹Bi and α+¹⁸⁴W, ²⁰⁶,²⁰⁸Pb, ²⁰⁹Bi systems. The fission barrier was expressed as B_f(J) = c_f × B_F^RFRM(J) - Δ_n + Δ_f, with continuous damping of shell correction with excitation energy described by an energy-dependent level density parameter a_x(U) = ã_x[1 + (Δ_x/U_x)(1 - e^{-ηU_x})]. The asymptotic liquid drop value of the level density parameter at equilibrium was taken as ã_n = A/9 MeV⁻¹, while the ratio ã_f/ã_n was treated as a free parameter determined by fits to experimental fission probabilities.

## Key numbers
- Fission barrier for ²¹⁰Po (from p+²⁰⁹Bi): 21.8 ± 0.2 MeV, compared to calculated value of 22.14 MeV from the macroscopic-microscopic finite-range liquid-drop model (Moller 2009)
- Fission barrier for ²¹⁰Po (from α+²⁰⁶Pb): 22.1 ± 0.3 MeV, compared to calculated value of 22.14 MeV
- Fission barrier for ²¹²Po (from α+²⁰⁸Pb): 20.3 ± 0.3 MeV, compared to calculated value of 20.27 MeV
- Fission barrier for ²¹³At (from α+²⁰⁹Bi): 17.8 ± 0.2 MeV, compared to calculated value of 18.56 MeV
- Fission barrier for ¹⁸⁸Os (from α+¹⁸⁴W): 24.3 ± 0.6 MeV (no calculated value given)
- Ratio ã_f/ã_n values range from 1.034 (²¹³At) to 1.082 (¹⁸⁸Os)
- Scaling factor c_f values range from 1.04 to 1.18
- Shell correction at equilibrium Δ_n ranges from -1.89 MeV (¹⁸⁸Os) to -10.62 MeV (²¹⁰Po)
- Energy-dependent damping factor: η = 0.054 + 0.002×E* MeV⁻¹

## Context
This work extends the statistical model analysis approach of Mahata (2006), which had previously analyzed the ¹²C+¹⁹⁸Pt system and required large shell corrections at the saddle point yielding barriers of 13.4 MeV. The present analysis of light-ion induced reactions (p and α) yields substantially higher fission barriers (17.8-24.3 MeV) and finds negligible shell corrections at the saddle point, contradicting the earlier heavy-ion results. The extracted barriers agree well with the macroscopic-microscopic finite-range liquid-drop model calculations of Moller (2009). The work enables further investigation into the statistical nature of pre-fission neutron emission and the contributions of dynamical emission and near-scission emission in heavy-ion fusion-fission reactions, as the current model without saddle-point shell corrections substantially underpredicts measured pre-fission neutron multiplicity data from heavy-ion reactions.
