# A novel emulator for fission event generators

> A wide variety of emulators have been developed for nuclear physics, particularly for use in quantifying and propagating parametric uncertainties to observables. Most of these methods have been used to emulate structure observables, such as energies, masses, or separation energies, or reaction observables, such as cross sections. Rarely, if ever, have event generators for theory models been emulated. Here, we describe one such novel emulator for the fission fragment decay code, $\texttt{CGMF}$, which calculates the emission of prompt neutrons and $γ$ rays from fission fragments. The emulator described in this work uses a combination of a noisy emission model and mixture density network to model the neutron and $γ$-ray multiplicities and energies. In this manuscript, we display the power of this type of emulator for not only modeling average prompt fission observables but also correlations between fission fragment initial conditions and these observables, using both neutron-induced and spontaneous fission reactions.

# A novel emulator for fission event generators

## Key claim

The authors develop a novel emulator for the fission fragment decay code CGMF that combines a noisy emission model and mixture density network to model prompt neutron and gamma-ray multiplicities and energies, achieving event-by-event emulation that retains correlations between fission fragment initial conditions and prompt observables. The emulator is demonstrated for both neutron-induced and spontaneous fission reactions, with the caveat that performance for gamma-ray multiplicity is significantly worse than for neutron multiplicity, though still within experimental uncertainty tolerances.

## Method

The emulator operates in two stages: first, a noisy emission model (NEM) using a neural network predicts neutron and gamma-ray multiplicities by learning a Gaussian probability density function over a continuous parameter that is discretized into integer multiplicities; second, a mixture density network (MDN) predicts neutron and gamma-ray energies conditional on the multiplicities. The training set includes 100,000 events from each of 19 fission reactions (spontaneous fission of 252Cf and neutron-induced fission of 235U, 238U, and 239Pu at thermal, 1, 2, 3, 4, and 5 MeV incident neutron energies), with validation on additional reactions including 233U, 234U, 237Np, and 241Pu.

## Key numbers

- Neutron multiplicity emulator achieves percent error in average neutron multiplicity below 0.4% for all fission reactions within the training set
- For reactions outside the training set, percent error in average neutron multiplicity is at most 0.6% (for 237Np(n,f))
- Gamma-ray multiplicity emulator reproduces average gamma-ray multiplicity for 252Cf(sf) within 2% of CGMF (emulator value 4.41 vs CGMF value 4.50)
- Percent error in average gamma-ray multiplicity peaks below 2% for trained reactions and ranges from about 1% to 2% for reactions outside the training set
- Neutron energy emulator using CGMF multiplicities agrees with CGMF to better than 0.3% for all incident neutron energies; using NEM multiplicities gives percent error below 1%
- Prompt fission neutron spectrum for 252Cf(sf) shows average difference between CGMF and either emulator better than 3%

## Context

This work extends the application of emulators in nuclear physics beyond structure observables (energies, masses, separation energies) and reaction observables (cross sections) to event generators for theory models, which had rarely if ever been done previously. The approach contrasts with Gaussian process emulators (Novak 2014, Sangaline 2016, Melendez 2019) and reduced basis methods (Frame 2018, Sarkar 2021, Koenig 2020) that have been used for other nuclear physics calculations, and with neural network approaches for uncertainty quantification (Utama 2016, Neufcourt 2018, Wang 2019). The emulator enables uncertainty quantification studies of CGMF itself, where iterative parameter adjustments would otherwise require prohibitive computational resources on high performance computing systems.
