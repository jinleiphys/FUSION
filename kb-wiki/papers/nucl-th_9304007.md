# Chiral Baryon with Quantized Pions

> We study a hybrid chiral model for the nucleon based on the linear sigma model with explicit quarks. We solve the model using a Fock-space configuration consisting of three quarks plus three quarks and a pion as the ground state ansatz in place of the ``hedgehog'' ansatz. We minimize the expectation value of the chiral hamiltonian in this ground state configuration and solve the resulting equations for nucleon quantum numbers. We calculate the canonical set of nucleon observables and compare with previous work.

# Chiral Baryon with Quantized Pions

## Key claim

The authors show that a hybrid chiral model for the nucleon, solved using a Fock-space configuration of three quarks plus three quarks and a quantized pion (rather than the hedgehog ansatz), can produce reasonable predictions for nucleon observables even with this extremely limited configuration space, provided the free parameters are re-tuned away from the values that optimize the hedgehog description.

## Method

The authors start from the linear sigma model with explicit quark degrees of freedom, fixing the pion mass at 139.6 MeV and the pion decay constant at 93 MeV. They construct a ground state ansatz as a superposition of a three-quark component and a three-quark-plus-one-pion component, both coupled to nucleon quantum numbers (j=1/2, t=1/2), with quarks in the 1s1/2 orbital and pions in the 1P orbital. They minimize the expectation value of the chiral Hamiltonian in this Fock-space configuration, treating the sigma meson as a classical coherent field and the pion as a quantum field, and solve the resulting coupled radial equations self-consistently.

## Key numbers

- With parameters m_q = 500 MeV and m_sigma = 1200 MeV (the Birse-Banerjee values): net nucleon energy before center-of-mass correction is 1196 MeV (compared to 1116 MeV from the hedgehog ansatz); pion kinetic energy is 164 MeV (compared to 565 MeV from Birse and Banerjee)
- With parameters m_q = 375 MeV and m_sigma = 600 MeV: center-of-mass corrected baryon mass is 937 MeV (essentially equal to the experimental value)
- Axial vector coupling g_A = 1.41 with the 500/1200 parameter set (hedgehog: 1.86, experiment: 1.25); g_A = 1.68 with the 375/600 parameter set
- Sigma commutator = 36.9 MeV with the 500/1200 parameter set (hedgehog: 94.0 MeV, experimental range: 25-60 MeV); sigma commutator = 86 MeV with the 375/600 parameter set
- Pion-nucleon coupling constant (g_piNN * m_pi / 2M) = 1.25 with the 500/1200 parameter set (hedgehog: 1.53, experiment: 1.0); = 1.0 with the 375/600 parameter set
- Proton magnetic moment = 1.67 nuclear magnetons with the 500/1200 parameter set (hedgehog: 2.88, experiment: 2.79); = 2.18 nuclear magnetons with the 375/600 parameter set
- RMS charge radius = 0.60 fm with the 500/1200 parameter set (experiment: 0.85 fm); = 0.82 fm with the 375/600 parameter set

## Context

This work extends the hybrid chiral soliton model of Birse and Banerjee (1985), which used the hedgehog ansatz that breaks rotational and isospin invariance and requires projection onto physical states. The authors had previously attempted a deformed shell-model approach (Price and Shepard, 1991) that also required projection. The present Fock-space approach imposes nucleon quantum numbers from the start, avoiding projection entirely. The method is analogous to relativistic positronium calculations by Darewicz, Horbatsch, and Koniuk (1992). The authors note that including multi-pion components and higher-order quark and pion wavefunctions is a straightforward generalization and is in progress.
