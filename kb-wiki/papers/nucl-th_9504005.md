# Polaron Variational Methods In The Particle Representation Of Field Theory : II. Numerical Results For The Propagator

> For the scalar Wick-Cutkosky model in the particle representation we perform a similar variational calculation for the 2-point function as was done by Feynman for the polaron problem. We employ a quadratic nonlocal trial action with a retardation function for which several ansätze are used. The variational parameters are determined by minimizing the variational function and in the most general case the nonlinear variational equations are solved numerically. We obtain the residue at the pole, study analytically and numerically the instability of the model at larger coupling constants and calculate the width of the dressed particle.

## Key claim

The variational method applied to the Wick-Cutkosky model's two-point function reveals that the model becomes unstable beyond a critical coupling constant, with no real variational solutions existing above that threshold; the instability manifests as a complex physical mass for the dressed particle, with the width growing nonanalytically as (alpha - alpha_c)^(3/2) near the critical point.

## Method

The authors extended Feynman's polaron variational method to the scalar Wick-Cutkosky model in the particle representation, using a quadratic nonlocal trial action with a retardation function. They employed three approaches: a "Feynman" parametrization with two variational parameters v and w, an "improved" parametrization with the same parameters but different functional form, and a fully variational solution where the profile function A(E) was determined by solving coupled nonlinear integral equations iteratively. The variational parameters were fixed by minimizing the variational inequality for the physical mass, and for couplings above the critical value, complex solutions were sought to determine the width of the unstable dressed particle.

## Key numbers

- Critical coupling constant alpha_c = 0.815 for the fully variational solution (with meson mass m = 140 MeV and nucleon mass M_phys = 939 MeV)
- Critical coupling alpha_c = 0.824 for the "Feynman" parametrization and alpha_c = 0.817 for the "improved" parametrization
- At m = 0 (massless mesons), the critical coupling alpha_c = 0.641
- For the "Feynman" parametrization at alpha = 0.8: sqrt(v) = 1325 MeV, sqrt(w) = 1254 MeV, lambda = 0.63714, M_1 = 490.60 MeV, A(0) = 1.2485
- For the "improved" parametrization at alpha = 0.8: sqrt(v) = 468.2 MeV, sqrt(w) = 434.5 MeV, lambda = 0.62429, M_1 = 493.44 MeV, A(0) = 1.3482
- For the variational solution at alpha = 0.8: lambda = 0.62262, M_1 = 493.55 MeV, A(0) = 1.3188
- Width Gamma = 0.13 MeV at alpha = 0.818 (phase chi = 0.05)
- Width Gamma = 134 MeV at alpha = 1.153 (phase chi = 0.50)
- Width Gamma = 301 MeV at alpha = 1.425 (phase chi = 0.65)

## Context

This work extends the polaron variational method previously developed by Feynman for nonrelativistic field theory to the relativistic scalar Wick-Cutkosky model, building on the formal framework established in the authors' earlier paper (Rosenfelder and Schreiber, 1994). The study reveals the model's instability at strong coupling, which was known from classical potential analysis (Baym, 1961) but does not appear in perturbative calculations or one-loop effective potential approaches. The results enable future calculations of physical observables in the Wick-Cutkosky model and provide a variational framework that could be applied to more realistic pion-nucleon systems, though the authors note that spin, isospin, and chiral symmetry are missing from the current model.
