# A new class of Hanbury-Brown/Twiss parameters

> In heavy ion collisions resonances can create strong non-Gaussian effects in the 2-pion correlation data. Hence, the commonly used Gaussian fit parameters do not fully characterize these correlators. We suggest a different set of HBT parameters which does not presuppose a particular shape of the correlator and allows to extract additional (non-Gaussian) information. Within a simple model for an expanding source including resonance decays it is shown that this additional information provides a clean distinction between scenarios with and without transverse flow.

## Key claim

The paper proposes that the commonly used Gaussian fit parameters for two-pion correlation functions in heavy ion collisions are insufficient when resonance decays create non-Gaussian effects, and that a new set of HBT parameters based on q-moments can extract additional non-Gaussian information that provides a clean distinction between scenarios with and without transverse flow, even when resonance decays are present.

## Method

The authors define new HBT parameters called q-variances, which are expectation values of the correlator in relative momentum space, and higher-order q-cumulants (specifically the kurtosis) that quantify deviations from a Gaussian shape. They compute these parameters for a model pion emission function that includes resonance decay channels from ρ, Δ, K*, Σ*, ω, η, η', K_S^0, Σ, and Λ, using a thermal freeze-out model with Gaussian spatial and temporal profiles and a linear transverse rapidity profile for flow.

## Key numbers

- Source parameters used in the model: T = 150 MeV, R = 5 fm, Δη = 1.2, τ₀ = 5 fm/c, Δτ = 1 fm/c, μ_B = μ_S = 0
- Transverse flow values studied: η_f = 0 (no transverse flow) and η_f = 0.3
- The transverse homogeneity length R_t scales approximately as R_t ≈ R (1 + (M_⊥/T) η_f²)^{-1/2}
- For the scenario with η_f = 0.3, the kurtosis Δ_s is an order of magnitude smaller than for the scenario without transverse flow
- The Gaussian fit was performed using n equidistant points q_i^(j) between 0 and 50 MeV

## Context

This work extends the standard Gaussian framework for HBT interferometry in heavy ion collisions, which was previously developed by Chapman et al. (1995) and Wiedemann et al. (1996), by addressing the non-Gaussian effects from resonance decays that were not captured by the conventional radius parameters R_ij²(K). The method enables future experimental analyses, particularly with Pb-beam experiments at the CERN SPS, to distinguish between transverse flow and resonance decay contributions to the K_⊥-dependence of HBT radii, which was previously ambiguous as shown by the authors' model calculations where resonance decays alone could mimic the flow signature in the Gaussian radius R_s.
