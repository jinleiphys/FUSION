# Breathing mode of rapidly rotating Bose-Einstein condensates

> We show that the breathing mode of a rapidly-rotating, harmonically-trapped Bose-Einstein condensate may be described by a generalized lowest Landau level (LLL) wave function, in which the oscillator length is treated as a variable. Using this wave function in a variational Lagrangian formalism, we show that the frequency of the breathing mode for a two-dimensional cloud is $2ω_{\perp}$, where $ω_{\perp}$ is the trap frequency. We also study large-amplitude oscillations and confirm that the above result is not limited to linear oscillations. The resulting mode frequency can be understood in terms of orbits of a single particle in a harmonic trap. The mode frequency is also calculated for a cloud in three dimensions and the result for the axial breathing mode frequency agrees with recent experimental data in the rapid rotation regime.

## Key claim

The breathing mode frequency of a rapidly rotating two-dimensional Bose-Einstein condensate is exactly \(2\omega_{\perp}\) (where \(\omega_{\perp}\) is the transverse trap frequency), and this result holds for both linear and large-amplitude oscillations, provided the cloud is in the mean-field lowest Landau level (LLL) regime and the vortex lattice is large enough that surface effects can be neglected.

## Method

The author introduced an extended LLL wave function that treats the oscillator length as a dynamical variable \(\lambda\) and includes a radial velocity field via a phase parameter \(\beta\). Using a variational Lagrangian formalism with the averaged vortex approximation, the Euler-Lagrange equations were derived for the cloud width parameter (Gaussian profile) or cloud radius (Thomas-Fermi profile). The equations of motion were linearized for small oscillations and also solved analytically for large-amplitude oscillations.

## Key numbers

- Breathing mode frequency for a two-dimensional cloud: \(\omega = 2\omega_{\perp}\), where \(\omega_{\perp}\) is the transverse trap frequency.
- Equilibrium cloud width for Gaussian profile: \(\sigma_0^4 = d_{\perp}^4 (l^2 + \kappa)\), where \(l = L_z/\hbar + 1\) and \(\kappa\) is the dimensionless interaction strength.
- Equilibrium cloud radius for Thomas-Fermi profile: \(R_{\perp,0}^4 = d_{\perp}^4 (9 l^2 + 8\kappa)\).
- Period of oscillation in the nonlinear regime: \(\tau = \pi\) in units of \(\omega_{\perp}^{-1}\), giving frequency \(2\omega_{\perp}\).
- Example parameters used in nonlinear oscillation plots: \(l = 100\) and \(\kappa = 100\), giving \(\sigma_0 \simeq 10.02 d_{\perp}\) and \(R_{\perp,0} \simeq 17.36 d_{\perp}\).

## Context

This work extends prior studies of collective modes in rotating Bose-Einstein condensates, particularly the hydrodynamic calculations of Sedrakian and Cozzini (2002) which gave the same frequency in the slow rotation regime, and the experimental measurements by Schweikhard et al. (2004) on axial breathing modes. The paper provides a unified LLL-based description that bridges the slow and rapid rotation regimes, and enables future experimental tests of the predicted transverse breathing mode frequency in rapidly rotating condensates. The result also connects to the single-particle picture of a particle in a harmonic trap, showing that interactions do not affect the mode frequency in two dimensions.
