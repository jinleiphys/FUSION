# Extension of the CEM2k and LAQGSM Codes to Describe Photo-Nuclear Reactions

> The improved Cascade-Exciton Model (CEM) code CEM2k+GEM2 and the Los Alamos version of the Quark-Gluon String Model code LAQGSM are extended to describe photonuclear reactions. First, we incorporate into CEM2k+GEM2 new evaluations of elementary cross sections based on the latest experimental data and also make several improvements in the description of the de-excitation of nuclei remaining after the cascade stage of reactions induced by arbitrary projectiles. Next, for photonuclear reactions we include in CEM2k+GEM2 a normalization to evaluated experimental absorption cross sections based on the recent systematics by Kossov. Then, we extend our high-energy code LAQGSM by adding the photonuclear mode which was ignored in all its previous versions, and add to it the photonuclear part from our improved CEM2k+GEM2. In this work we present a short description of the photonuclear mode as incorporated into our codes, show several illustrative results, and point out some unresolved problems.

## Key claim

The authors extend the CEM2k+GEM2 and LAQGSM codes to describe photonuclear reactions at intermediate energies (from about 30 MeV to about 1.5 GeV), but note that the current version still has unresolved problems and is under further development, even though it already describes intermediate-energy photonuclear reactions reasonably well.

## Method

The authors incorporated into CEM2k+GEM2 new evaluations of elementary cross sections based on latest experimental data and improved the description of nuclear de-excitation after the cascade stage. For photonuclear reactions specifically, they added a normalization to evaluated experimental absorption cross sections based on the Kossov systematics, and extended LAQGSM by adding a photonuclear mode that was absent in all previous versions. The photonuclear model is based on the Dubna Intranuclear Cascade Photonuclear Reaction Model, using the Levinger quasi-deuteron model below pion-production threshold and considering one- or two-pion production at higher energies, with new approximations for angular distributions of secondary particles from the Moscow INC event generator.

## Key numbers

- Photon energy range of applicability: from about 30 MeV to about 1.5 GeV
- The Levinger model parameter L is approximately 10 for the quasi-deuteron absorption cross section formula
- The Dubna INC considers photon energies up to about 1.5 GeV, above which three-pion production becomes relevant
- The Kossov approximation for photoabsorption cross sections covers energies from hadron production threshold to about 40 TeV
- The Moscow INC event generator provides angular distribution data at 50 different gamma energies from 117.65 to 6054 MeV
- The Delta isobar mass used is M = 1232 MeV
- The angular distribution data file tabulates values for cos Theta from 0 to 180 degrees with step Delta Theta = 10 degrees, for 60 different channels of gamma p and gamma n reactions

## Context

This work extends the Cascade-Exciton Model code lineage (CEM95, CEM97, CEM2k) and the Los Alamos Quark-Gluon String Model code to handle photonuclear reactions, which neither code could previously describe. The photonuclear model builds on the Dubna Intranuclear Cascade Photonuclear Reaction Model developed by Gudima, Iljinov, and Toneev (1969-1974), and incorporates the Kossov systematics (1999) for total photoabsorption cross sections. The improved codes have been incorporated into transport codes MARS15 and LAHET3, and are planned for inclusion in MCNPX and MCNP6, enabling simulation of intermediate-energy photonuclear reactions in thick targets of arbitrary geometry.
