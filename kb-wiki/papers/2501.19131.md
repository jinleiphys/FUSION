# Calculation of harmonic oscillator brackets in SU(3) basis

> We present a new approach for the Talmi-Moshinsky transformation representation in the harmonic oscillator basis. We utilize the SU(3) scheme for the calculation of harmonic oscillator brackets. Using this scheme we obtain the explicit relations for numeric evaluation and present a computational approach.

## Key claim

The paper presents a new approach for calculating harmonic oscillator brackets (HOBs) using the SU(3) scheme, claiming that while the matrix-element code is slower than existing methods, it achieves better accuracy for larger oscillator energies, particularly for total energy quanta E above 12.

## Method

The authors express the Talmi-Moshinsky transformation in an SU(3) basis by rewriting the transformation matrix as a representation of SU(2) (the double covering group of SO(2)), parametrized by an angle θ related to the mass-ratio parameter d. They then derive the HOB as a sum over SU(3) Clebsch-Gordan isofactors and Wigner d-matrices, implementing the computation in Fortran90 using existing SU(3) ⊃ SO(3) Clebsch-Gordan coefficient code from Bahri, Rowe, and Draayer (2004). The calculation time and numerical accuracy are compared against the existing HOB code from Kamuntavičius et al. (2001).

## Key numbers

- For total HO energy E = 20, the SU3HOB matrix element code takes 137 seconds, compared to 3.46 seconds for the HOB code and 0.491 seconds for the matrix multiplication version.
- For E = 23, the SU3HOB code achieves error δ = 0.279E-05, while the HOB code gives δ = 0.133E-04, showing better accuracy for the SU3 approach at higher energies.
- For E = 23 and L = 1, the SU3HOB error is 0.118E-08 versus the HOB error of 0.176E-04, a factor of approximately 15000 improvement.
- The Wigner d-matrix calculation takes only 0.51% of total computation time, while Clebsch-Gordan coefficient computation takes 82.46%.
- For E = 23 and L = 23, the SU3HOB error is 0.668E-02 versus the HOB error of 0.842E-11, showing the SU3 approach deteriorates for large L values.

## Context

This work extends the authors' previous research on harmonic oscillator brackets for two and three Jacobi coordinates (Mickevičius et al., 2012; Stepšys et al., 2019) and provides an alternative to the expression derived by Buck and Merchant (1996) and reformulated by Kamuntavičius et al. (2001). The new SU(3) scheme enables calculation of the Wigner d-matrix with different d parameters for various Jacobi coordinate transformations while computing the transformation between SO(3) and SU(3) basis only once, potentially allowing formulation of the ab initio no-core shell model in the unitary scheme using Jacobi coordinates.
