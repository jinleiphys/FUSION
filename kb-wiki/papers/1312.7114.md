# Nucleon structure functions at small $x$ via the Pomeron exchange in AdS space with a soft infrared wall

> We present analyses on nucleon structure functions at the small Bjorken-$x$ in the framework of holographic QCD. In this study, we improve the description of the target nucleon in the current setup of the holographic model by introducing a soft-wall AdS/QCD model, in which the AdS geometry is smoothly cut off at IR. Combining the improved Pomeron-nucleon coupling and the wave function of the 5D U(1) vector field with the BPST Pomeron exchange kernel, then we obtain the structure functions. Here we focus on the nonperturbative kinematical region, where $10^{-6} \leq x \leq 10^{-2}$ and $0.1 \leq Q^2 \leq 10$ [GeV$^2$], and show that our calculations for $F_2^p$ and $F_L^p$ are consistent with experimental data of the deep inelastic scattering at HERA. Furthermore, we find that the resulting longitudinal-to-transverse ratio of the structure functions, $F_L^p/F_T^p$, depends on both of $x$ and $Q^2$.

## Key claim

The authors show that a soft-wall AdS/QCD model, which smoothly cuts off the AdS geometry in the infrared, provides a better description of the nucleon structure functions at small Bjorken-\(x\) than the hard-wall model used in their previous work (Watanabe 2012), because the soft-wall model captures nonperturbative effects at larger bulk coordinates that are lost in the hard-wall approach.

## Method

The authors compute the nucleon structure functions \(F_2^p\) and \(F_L^p\) using the BPST Pomeron exchange kernel in AdS space, combined with overlap functions for the virtual photon (from a 5D U(1) vector field) and the nucleon (from a soft-wall AdS/QCD Dirac field). The nucleon-Pomeron coupling is derived from the three-point function of the graviton with nucleon fields in the soft-wall model, using the dilaton profile \(\Phi(z) = \kappa^2 z^2\) with \(\kappa = 0.350\) GeV and conformal mass \(M = 3/2\). The structure functions are evaluated in the single-Pomeron exchange approximation, using both a conformal kernel and a modified kernel that includes a conformal-breaking term parameterized by \(z_0\).

## Key numbers

* The soft-wall parameter is fixed at \(\kappa = 0.350\) GeV, reproducing the proton and \(\rho\) meson masses simultaneously.
* The conformal-breaking parameter is optimized to \(z_0 = 4.25\) GeV\(^{-1}\).
* For the modified kernel, the fitted parameters are \(\rho = 0.783\) and \(g_0^2 = 1.62 \times 10^2\); for the conformal kernel, \(\rho = 0.856\) and \(g_0^2 = 0.908 \times 10^2\).
* The calculations cover the kinematic region \(10^{-6} \leq x \leq 10^{-2}\) and \(0.1 \leq Q^2 \leq 10\) GeV\(^2\).
* The longitudinal-to-transverse ratio \(R = F_L^p/F_T^p\) increases by about 30% from \(Q^2 = 0.1\) GeV\(^2\) to \(Q^2 = 1\) GeV\(^2\).
* The nucleon overlap function peaks at \(z' \sim 4\) GeV\(^{-1}\) (\(\sim 0.8\) fm), consistent with the nucleon size.

## Context

This work extends the holographic QCD approach to DIS initiated by Polchinski and Strassler (2002) and the BPST Pomeron kernel developed by Brower, Polchinski, Strassler, and Tan (2006). It improves upon the authors' previous hard-wall model calculation (Watanabe 2012) by using a soft-wall AdS/QCD model that reproduces the correct Regge behavior \(m_n^2 \propto n\) for excited hadrons, following the soft-wall model of Karch, Katz, Son, and Stephanov (2006) and the nucleon description of Abidin and Carlson (2009). The results enable predictions for the longitudinal structure function \(F_L^p\) and the ratio \(R = F_L^p/F_T^p\) in the nonperturbative region, and the framework can be extended to other two-body high-energy scattering processes.
