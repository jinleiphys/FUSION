# Dissociation by acceleration

> We show that mesons, described using rotating relativistic strings in a holographic setup, undergo dissociation when their acceleration 'a' exceeds a value which scales with the angular momentum 'J' as a_max ~ \sqrt{T_s/J}, where 'T_s' is the string tension.

## Key claim
Mesons described as rotating relativistic strings in a holographic setup undergo dissociation when their four-dimensional acceleration exceeds a critical value that scales as a_max ~ sqrt(T_s/J), where T_s is the string tension and J is the angular momentum; this bound arises from the formation of a Rindler horizon in the space-like direction orthogonal to the rotation plane.

## Method
The authors model large-spin mesons as rotating Nambu-Goto strings in flat spacetime, described in Rindler coordinates adapted to constant acceleration. They impose boundary conditions that force the string endpoints to move with constant acceleration in a direction orthogonal to the rotation plane, solve the equations of motion analytically for the string profile, and compute the endpoint acceleration as a function of the integration constant C. The critical acceleration is found by minimizing the endpoint acceleration with respect to C.

## Key numbers
- Critical acceleration for meson dissociation: a_max = 0.529 sqrt(T_s/J)
- For string tension T_s = 0.3 GeV^2, naive critical accelerations for J=1, 2, and 10 are a_max = 0.290, 0.205, and 0.092 GeV respectively
- Angular momentum as function of string length L: J = (pi/8) T_s L^2
- For acceleration in the direction of endpoint separation, the critical acceleration from a simplified analysis is a_c = sqrt(pi T_s / 8J)

## Context
This work extends the holographic analysis of meson melting at finite temperature (Peeters 2006) to the case of acceleration-induced dissociation via a Rindler horizon in four-dimensional spacetime. It contrasts with Berenstein (2007), who studied acceleration in the holographic direction and related it to dissociation temperature, whereas here the acceleration is a genuine four-dimensional effect. The results enable experimental searches for acceleration-induced meson dissociation, potentially relevant for heavy-ion collisions where nucleon decelerations can reach up to 1 GeV, and suggest that higher-spin mesons dissociate at lower accelerations.
