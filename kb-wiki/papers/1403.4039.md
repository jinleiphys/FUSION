# DIRHB -- a relativistic self-consistent mean-field framework for atomic nuclei

> The DIRHB package consists of three Fortran computer codes for the calculation of the ground-state properties of even-even atomic nuclei using the framework of relativistic self-consistent mean-field models. Each code corresponds to a particular choice of spatial symmetry: the DIRHBS, DIRHBZ and DIRHBT codes are used to calculate nuclei with spherical symmetry, axially symmetric quadrupole deformation, and triaxial quadrupole shapes, respectively. Reflection symmetry is assumed in all three cases. The latest relativistic nuclear energy density functionals are implemented in the codes, thus enabling efficient and accurate calculations over the entire nuclide chart.

## Key claim

The DIRHB package provides a unified computational framework for solving the relativistic Hartree-Bogoliubov equations in even-even nuclei under three spatial symmetries (spherical, axially deformed, and triaxial), assuming both time-reversal and reflection symmetry, and implementing the latest density-dependent meson-exchange and point-coupling functionals.

## Method

The codes solve the stationary relativistic Hartree-Bogoliubov equations iteratively, expanding single-nucleon wave functions in a basis of spherical, axially symmetric, or triaxial harmonic oscillators depending on the assumed nuclear shape. At each iteration, the matrix elements are updated using either the modified Broyden method or linear mixing, and for constrained calculations of specific deformation parameters the augmented Lagrangian method is employed. The particle-particle channel is described by a separable finite-range pairing force.

## Key numbers

* The triaxial test case requires 300 Mb of RAM.
* Running time ranges from a few seconds (spherical) up to a few hours (triaxial) for the test cases.
* The free values used for the omega and rho meson masses are m_omega = 783 MeV and m_rho = 763 MeV.
* The density-dependent meson-exchange functional DD-ME2 and the point-coupling functional DD-PC1 are included; DD-PC1 parameters were adjusted to binding energies of 64 axially symmetric deformed nuclei in the mass regions A approximately 150-180 and A approximately 230-250.

## Context

This work extends the earlier relativistic mean-field code for axially deformed systems published by Ring (1997) by adding triaxial capability and implementing modern density-dependent functionals such as DD-ME2 (Lalazissis, 2005) and DD-PC1 (Niksic, 2008). The package enables systematic ground-state calculations across the entire nuclide chart for even-even nuclei, and can be used to generate input for studies of collective excitations such as giant monopole and dipole resonances.
