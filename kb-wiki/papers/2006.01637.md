# Minimal Theory of Isomerism -Q.Q and Other Interactions

> We perform shell model calculations using a quadrupole-quadrupole interaction (Q.Q) in single a j shell space. We show that this one-parameter interaction is a good predictor of where nuclear isomerism occurs and where it does not occur. The limitations of this interaction are also discussed. We then include other interactions e.g. in the f_{7/2} shell those obtained from the (local) spectrum of a 2 particle system and then from a 2 hole system. In the g_{9/2}where there is insufficient empirical data from the 2 hole system and so various empirical interactions are used.

## Key claim

The quadrupole-quadrupole (Q.Q) interaction, a one-parameter interaction in a single j shell, is a good predictor of where nuclear isomerism occurs, but it does not always distinguish between a true spin-gap isomer (where no lower-lying states of J-1 or J-2 exist) and a weak isomer (where an E2 or M1 transition is allowed but the energy difference is very small).

## Method

Shell model calculations were performed in a single j shell space using a quadrupole-quadrupole (Q.Q) interaction with a single strength parameter. The authors then compared these results with calculations using empirical two-body matrix elements: for the f7/2 shell, matrix elements were taken from the particle-particle spectrum of 42Sc and the hole-hole spectrum of 54Co; for the g9/2 shell, where insufficient empirical data exists for the two-hole system 98In, matrix elements from the CCGI interaction and from Qi et al. were used. Energy levels and half-lives were calculated for selected nuclei in both shells.

## Key numbers

- For 44Ti with the Q.Q interaction, the J=12+ state lies at 6.277 MeV, below J=11+ (7.466 MeV) and J=10+ (6.501 MeV), making it a spin-gap isomer in the calculation.
- For 52Fe with the Q.Q interaction, the J=12+ state lies at 6.277 MeV, below J=11+ (7.466 MeV) and J=10+ (6.501 MeV), making it a spin-gap isomer in the calculation.
- For 43Sc with the Q.Q interaction, the 19/2- state lies at 2.700 MeV, below 17/2- (4.088 MeV) and 15/2- (3.467 MeV), making it a spin-gap isomer in the calculation.
- For 96Cd with the Q.Q interaction, the J=16+ state lies at 10.163 MeV, below J=15+ (12.585 MeV) and J=14+ (12.075 MeV), making it a spin-gap isomer in the calculation.
- For 44Ti using the 42Sc interaction, the J=12+ state (7.708 MeV) lies above J=10+ (7.390 MeV), so no spin-gap isomer is predicted; for 52Fe using the 54Co interaction, the J=12+ state (6.413 MeV) lies below J=10+ (6.611 MeV), so a spin-gap isomer is predicted.
- For 43Sc, the calculated half-life of the 19/2- state using the Q.Q interaction with the experimental transition energy (0.136 MeV) is 2.041 x 10^3 ns, compared to the experimental value of 472 (4) ns.
- For 44Ti, the calculated half-life of the 12+ state using the Q.Q interaction with the experimental transition energy (0.369 MeV) is 3.759 ns, compared to the experimental value of 2.1 (4) ns.

## Context

This work extends the single-j shell model approach to isomerism using a simple quadrupole-quadrupole interaction, building on the foundational shell model work of Mayer and Jensen (1955) and the empirical matrix element approach of deShalit and Talmi (1963). The authors compare their Q.Q results with earlier empirical interactions from Bayman et al. (1964) and the updated MBZE interaction from Escuderos, Zamick and Bayman (2006), as well as with the CCGI interaction from Coraggio et al. and the interaction from Qi et al. The work enables systematic predictions of isomerism in the f7/2 and g9/2 shells, showing where simple interactions suffice and where more detailed local interactions are needed to reproduce experimental distinctions between weak and spin-gap isomers.
