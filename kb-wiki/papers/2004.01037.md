# Quark model relations for b-baryon decay

> Properties of b-baryon decay matrix elements (amplitudes) have been derived in a nonsymmetric quark model without any use of SU(6), SU(3), or SU(2) (isotopic spin) groups. Equalities between pairs of amplitudes are derived, and $Λ-Σ$ mixing is used to calculate the branching ratio for the transition $Λ_b\rightarrow Σ^0$.

# Quark model relations for b-baryon decay

## Key claim
The paper derives equalities between b-baryon decay matrix elements using a nonsymmetric quark model that avoids SU(6), SU(3), or SU(2) symmetry groups, and uses Lambda-Sigma mixing to predict a branching ratio for Lambda_b to Sigma^0 that would otherwise vanish; the prediction is an order of magnitude below the current experimental upper limit from LHCb.

## Method
The author constructs three-quark spin wave functions for spin-1/2 baryons using the nonsymmetric quark model, where quark order is coordinated with spin states to implement the Pauli principle without group symmetrization. Decay matrix elements for b-baryon transitions (b to d or b to s) are computed by evaluating scalar and pseudoscalar amplitudes between initial and final baryon states, with the transition operator converting a b quark to a d or s quark. Lambda-Sigma mixing is then applied to the otherwise vanishing Lambda_b to Sigma^0 transition, using a mixing angle derived from baryon mass differences and two-body interaction energies.

## Key numbers
- Mixing angle theta_m = 0.014 = 0.80 degrees +/- 0.02 degrees, derived from epsilon = -1.07 +/- 0.02 MeV and mass difference (M_Sigma - M_Lambda) = 77 MeV
- Branching ratio prediction for Lambda_b to Sigma^0 relative to Lambda_b to Lambda: R = (2.1 +/- 0.3) x 10^{-4}, including phase space correction factor Phi = 1.058
- Experimental upper limit from LHCb: R < 21 x 10^{-4} at 95% confidence level
- Matrix element relations: A(Xi_b^0 to Sigma^0 + S) = A(Xi_b^- to Sigma^- + S) = sqrt(3) A(Xi_b^0 to Lambda + S) = -A(Lambda_b to n + S) for b to d transitions
- Matrix element relations: A(Xi_b^0 to Xi^0 + S) = A(Xi_b^- to Xi^- + S) = -(sqrt(3)/2) A(Lambda_b to Lambda + S) for b to s transitions

## Context
This work extends the nonsymmetric quark model approach previously introduced by Franklin (2015) to b-baryon decays, providing an alternative to SU(3)-based analyses such as those by Geng and collaborators (various years) and Dery and collaborators (various years). The results reproduce the same branching ratios as the DGGS group theory approach, but without invoking internal symmetry groups, because the Clebsch-Gordan coefficients of SU(3) match the spin addition coefficients. The prediction for Lambda_b to Sigma^0 decay enables experimental tests at improved sensitivity levels, as current LHCb limits are an order of magnitude above the predicted value.
