# Structure function of a damped harmonic oscillator

> Following the Caldeira-Leggett approach to describe dissipative quantum systems the structure function for a harmonic oscillator with Ohmic dissipation is evaluated by an analytic continuation from euclidean to real time. The analytic properties of the Fourier transform of the structure function with respect to the energy transfer (the ``characteristic function'') are studied and utilized. In the one-parameter model of Ohmic dissipation we show explicitly that the broadening of excited states increases with the state number without violating sum rules. Analytic and numerical results suggest that this is a phenomenologically relevant, consistent model to include the coupling of a single (sub-)nuclear particle to unobserved and complex degrees of freedom.

## Key claim

The Caldeira-Leggett model of a damped harmonic oscillator with Ohmic dissipation provides a consistent quantum-mechanical framework for inclusive scattering processes, where the broadening of excited states increases with state number without violating sum rules, but only in the underdamped case (γ < 2ω₀) and with the caveat that the narrow-width approximation (Γₙ ≈ nγ) is only approximate and leads to a small non-vanishing structure function for negative energy transfers.

## Method

The structure function for a harmonic oscillator with Ohmic dissipation was evaluated by analytically continuing the euclidean-time correlation function T(q,τ) to real time via τ → it, using the path integral formalism where the environmental oscillators are integrated out exactly. The resulting "characteristic function" Φ_q(t) was expressed in terms of a Vineyard function ξ_V(τ), whose analytic properties (including a logarithmic cut) were studied and used to compute the structure function numerically via Fourier transform, with sum rule conservation verified by numerical integration.

## Key numbers

- The renormalized oscillator length b² = b₀² · (ω₀/Ω) · (2/π) arctan(2Ω/γ) ≤ b₀², where Ω = √(ω₀² − γ²/4) for the underdamped case γ < 2ω₀
- In the narrow-width approximation, the width of the n-th excited state is Γₙ ≈ nγ
- The asymptotic structure function for large energy transfer ν → ∞: S(q,ν) → (γ/(mπ)) · (q²/ν³) + (3/2) · (γ/(m²π)) · (q⁴/ν⁴) + O(ln ν/ν⁵)
- For γ/ω₀ = 0.2, the non-energy-weighted sum rule (NEWSR) is satisfied to 0.99965–0.99998 for q·b₀ = 0.5–4.0 with appropriate integration limits
- For γ/ω₀ = 0.2, the energy-weighted sum rule (EWSR) divided by q²/(2m) is satisfied to 0.9899–0.9987 for q·b₀ = 0.5 and 0.9413–0.9987 for q·b₀ = 1.0

## Context

This work extends the Caldeira-Leggett approach (Caldeira and Leggett, 1983) for dissipative quantum systems to inclusive scattering structure functions, providing a more consistent alternative to ad hoc smearing of delta-functions by Breit-Wigner distributions as done in earlier studies (smearing, 1990s). The model enables phenomenological applications in nuclear physics (e.g., quasielastic scattering with multi-nucleon emission) and hadronic physics (quark-hadron duality), with the Vineyard function form previously derived by Vineyard (1958) for neutron scattering from quantum liquids and by Rosenfelder (1980s) for relativistic deep inelastic scattering.
