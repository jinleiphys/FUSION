# Extracting baryon-antibaryon strong interaction potentials from p$\barΛ$ femtoscopic correlation function

> The STAR experiment has measured $pΛ$, $\bar{p}\barΛ$, $\bar{p}Λ$, and $p\barΛ$ femtoscopic correlation functions in central Au+Au collisions at $\sqrt{s_{NN}}=200$ GeV. The system size extracted for $pΛ$ and $\bar{p}\barΛ$ is consistent with model expectations and results for other pair types, while for $p\barΛ$ and $\bar{p}Λ$ it is not consistent with the other two and significantly lower. In addition an attempt was made to extract the unknown parameters of the strong interaction potential for this baryon-antibaryon ($B\bar{B}$) pair. In this work we reanalyze the STAR data, taking into account residual femtoscopic correlations from heavier $B\bar{B}$ pairs. We obtain new estimates for the system size, consistent with the results for $pΛ$ and $\bar{p}\barΛ$ pairs and with model expectations. We give new estimates for the strong interaction potential parameters for $p\barΛ$ and show that similar constraints can be given for parameters for other, heavier $B\bar{B}$ pairs.

## Key claim

The authors reanalyze STAR data on the \(p\bar{\Lambda}\) femtoscopic correlation function and show that, when residual correlations from heavier baryon-antibaryon pairs are properly included, the extracted source size becomes consistent with results for \(p\Lambda\) and \(\bar{p}\bar{\Lambda}\) pairs and with hydrodynamic model expectations, contradicting the earlier STAR result that had reported a 50% smaller radius.

## Method

The authors construct a full femtoscopic correlation function for the \(p\bar{\Lambda}\) system by summing contributions from the primary \(p\bar{\Lambda}\) pair and nine residual correlation components from heavier baryon-antibaryon parent pairs (e.g., \(\Lambda\bar{\Lambda}\), \(\Sigma^{+}\bar{\Sigma}^{0}\), \(p\bar{\Xi}^{0}\)) that decay into a proton and a \(\Lambda\) or are misidentified as such. For each parent pair, the correlation function is calculated using the Lednicky formalism with a Gaussian source of size \(r_0\) and a complex scattering length \(f_0\), then transformed via a decay-kinematics matrix \(W\) to the daughter \(p\bar{\Lambda}\) relative momentum space. The full formula is fitted to the STAR experimental correlation function with \(\chi^2\) minimization, treating \(r_0\), \(\Re f_0\), and \(\Im f_0\) as free parameters while fixing the effective range \(d_0 = 0\) fm and assuming the same source size and scattering length for all baryon-antibaryon pairs.

## Key numbers

* Source size extracted with full residual correlations: \(r_0 = 2.83 \pm 0.12\) fm, consistent with the \(p\Lambda\) system and hydrodynamic scaling, in contrast to the earlier STAR result that was 50% lower.
* Scattering length from the full residual-correlation fit: \(f_0 = 0.49 \pm 0.21 + i (1.00 \pm 0.21)\) fm.
* Imaginary part of the scattering length for the \(p\bar{\Lambda}\) system is significant and consistent with the known value for \(p\bar{p}\) (\(\Im f_0 = 0.88 \pm 0.09\) fm from Batty (1989)).
* When no residual correlations are included (reproducing the STAR procedure), the extracted radius is small, approximately \(1.5 \pm 0.1\) fm, and \(\Im f_0\) remains consistent with the full RC fit.
* The primary \(p\bar{\Lambda}\) pairs constitute only 15% of the sample; the remaining 85% come from residual pairs listed in Table 1 of the paper.

## Context

This work extends the femtoscopic analysis of baryon-antibaryon pairs beyond the \(p\bar{p}\) system, which has been studied theoretically by Batty (1989) and others, and addresses a discrepancy in the STAR measurement by Adams (2005) where the extracted \(p\bar{\Lambda}\) source size violated the expected \(1/\sqrt{m_T}\) scaling observed for all other pair types at RHIC. The authors show that the UrQMD model assumption (Bleicher, 1999) that all baryon-antibaryon annihilation cross-sections equal that of \(p\bar{p}\) at the same relative momentum is consistent with the data, while the alternative scaling by \(\sqrt{s}\) of the pair is not ruled out but leads to internal inconsistency. The results enable future extractions of strong interaction parameters for heavier baryon-antibaryon pairs from femtoscopic data.
