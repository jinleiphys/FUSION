# Decay: A Monte Carlo library for the decay of a particle with ROOT compatibility

> Recently, there is a need for a general-purpose event generator of decays of an elementary particle or a hadron to a state of higher multiplicity ($N > 2$) that is simple to use and universal. We present the structure of such a library to produce generators that generate kinematics of decay processes and can be used to integrate any matrix element squared over phase space of this decay. Some test examples are presented, and results are compared with results known from the literature. As one of examples we consider the Standard Model Higgs boson decay into four leptons. The generators discussed here are compatible with the ROOT interface.

## Key claim

The paper presents a new Monte Carlo library for generating decay kinematics of a particle into higher-multiplicity final states (N > 2) that is compatible with the ROOT framework, but the library's effectiveness depends on the user's choice between a simple uniform sampling mode (mode A) and an adaptive integration mode (mode B) that handles integrands with small support in phase space.

## Method

The library implements three generator classes: `TGenDecay` for uniform phase-space sampling using the Raubold-Lynch algorithm, and `TGenFoamDecay` which wraps the FOAM adaptive Monte Carlo integrator around the decay kinematics. Users define a derived class that overrides the virtual `Integrand` method to supply the squared matrix element for a specific decay process. The generators produce four-vectors of decay products and can output events to text files and ROOT histograms.

## Key numbers

- For muon decay in 4-Fermi theory, the exact analytical decay width is Gamma_exact = 3.01315 x 10^{-19} GeV; with 10^6 events using `TGenDecay`, the computed width is 3.01232 x 10^{-19} GeV, giving a difference of 8.3 x 10^{-23} GeV
- The Fermi coupling constant used in the muon decay test is G_F = 1.166379 x 10^{-5} GeV^{-2}
- The Higgs boson mass used in the four-lepton decay example is M_H = 125.1 GeV
- For the two-particle decay test, the analytical phase-space integral formula is given as R_2 = (pi/(2 m_a^2 (2pi)^2)) * sqrt((m_a^2 - (m_b+m_c)^2)(m_a^2 - (m_b-m_c)^2))

## Context

This work extends the functionality of the ROOT `TGenPhaseSpace` class (based on the GENBOD algorithm by James, 1968) and builds on the authors' previous exclusive Monte Carlo generator GenEx (Kycia, 2014; Kycia, 2017). The library is motivated by the need for simple yet versatile decay generators in theoretical studies of exotic meson decays, glueballs, and tetraquarks, as well as for central exclusive diffractive production processes studied at RHIC and LHC (Adam, 2020; Sirunyan, 2020). The adaptive FOAM-based generator enables integration of matrix elements with small support in Lorentz Invariant Phase Space, which is not possible with the standard uniform sampling approach.
