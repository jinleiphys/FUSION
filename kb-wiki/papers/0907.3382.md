# Continuum Random Phase Approximation for Relativistic Point Coupling Models

> Relativistic Continuum Random Phase Approximation (CRPA) is used to investigate collective excitation phenomena in several spherical nuclei along the periodic table. We start from relativistic mean field calculations based on a covariant density functional with density dependent zero range forces. From the same functional an effective interaction is obtained as the second derivative with respect to the density. This interaction is used in relativistic continuum-RPA calculations for the investigation of isoscalar monopole, isovector dipole and isoscalar quadrupole resonances of spherical nuclei. In particular we study the low-lying E1 strength in the vicinity of the neutron evaporation threshold. The properties of the resonances, such as centroid energies and strengths distributions are compared with results of discrete RPA calculations for the same model as well as with experimental data.

## Key claim

The paper demonstrates that relativistic continuum random phase approximation (CRPA) calculations using zero-range point coupling models (specifically the PC-F1 functional) can reproduce giant resonance properties with quality comparable to standard finite-range meson-exchange models, provided the coupling to the continuum is treated exactly via the response formalism in coordinate space rather than through discrete basis expansions.

## Method

The authors start from relativistic mean field (RMF) calculations using the density-dependent zero-range point coupling Lagrangian PC-F1, obtaining self-consistent Dirac spinors for spherical nuclei. From the same energy density functional, an effective particle-hole interaction is derived as the second functional derivative with respect to the density matrix. The linearized Bethe-Salpeter equation for the response function is then solved in coordinate space using the single-particle Green's function with proper boundary conditions, which allows exact treatment of continuum states. The strength functions for isoscalar monopole, isovector dipole, and isoscalar quadrupole resonances are computed and compared with results from discrete RPA calculations using the same functional and with experimental data.

## Key numbers

* The parameter set PC-F1 contains 11 coupling constants, with values such as α_S = -14.935894 fm^{-2}, α_V = 10.098025 fm^{-2}, α_TV = 1.350268 fm^{-2}, β_S = 22.994736 fm^{-5}, γ_S = -66.769116 fm^{-8}, and γ_V = -8.917323 fm^{-8}.
* The free response function R^0(ω) is expressed as R_{klk'l'}^0(ω) = (n_k - n_l)/(ω - ε_k + ε_l + iη) δ_{kk'} δ_{ll'}, where n_k = 1 for hole states with ε_k ≤ ε_F and 0 for particle states with ε_k > ε_F.
* The smearing parameter Δ is introduced in the strength function S(ω + iΔ/2) = -1/π Im R_{FF}(ω + iΔ/2) to produce a Lorentzian folding of width Δ.
* The ah-pairs (particles in the Dirac sea and holes in the Fermi sea) are more than 1.2 GeV away from normal ph-pairs, justifying their neglect in the calculations.

## Context

This work extends the relativistic continuum RPA method previously applied to finite-range meson-exchange models (SRM 1989, Pie 2000, Pie 2001) to zero-range point coupling models, which are closer to the philosophy of density functional theory and numerically simpler for complex calculations beyond mean field. The authors compare their continuum treatment with the spectral representation method used in earlier RRPA calculations for point coupling models (NVR 2005). The results enable future applications of point coupling functionals to more advanced approaches such as Particle Vibrational Coupling (LRT 2008) and Generator Coordinate Method calculations (NVR 2006), where the simplicity of zero-range forces is particularly advantageous.
