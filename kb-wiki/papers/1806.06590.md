# Calculating reactions with use of no-core shell-model states

> A scheme to compute reactions is described that uses the Slater determinants constructed of oscillator orbitals. Simple linear equations are suggested for this purpose and shown to be efficient in model examples. A universal method to evaluate the required matrix elements is given.

## Key claim

The paper proposes a simple and universal extension of the no-core shell-model (NCSM) to compute two-fragment reactions using linear projection equations that avoid antisymmetrization between nucleons belonging to different reaction partners, with the claim that these equations are efficient in model examples and that the required matrix elements can be evaluated using universal formulae independent of fragment size.

## Method

The author constructs trial continuum wave functions as a sum of channel functions (containing regularized Coulomb functions for relative motion of fragments) plus short-ranged basis states built from Slater determinants of oscillator orbitals. The expansion coefficients are found by solving linear equations of the form (χ_k, [H-E]Ψ_j)=0, which involve only bound-bound and bound-free matrix elements and omit inter-fragment antisymmetrization. The required bound-free matrix elements are evaluated by expressing fragment wave functions with excited center-of-mass states as linear combinations of Slater determinants, using the fact that NCSM calculations provide fragment internal wave functions multiplied by the lowest oscillator function of the fragment center of mass.

## Key numbers

* For the s-wave scattering model with V₀=E₀ and kR=0.1, the exact tanδ equals -0.9798735; using N-1=22 basis functions with d/R=0.4 reproduces this value to all listed digits.
* For the same model with d/R=1.0, N-1=48 basis functions are needed to reach tanδ = -0.9798735.
* In the three-particle model with a Gaussian well of depth V₀=75 MeV and range R₀=2.5 f, retaining only N=7 lowest oscillator functions (with oscillator radius 2 f) yields phase shifts nearly indistinguishable from the exact result for R_cut values of 2.5 f, 3 f, 4 f, and 5 f.
* The binding energy in the Gaussian well model is about 3.5 MeV.

## Context

This work extends the NCSM method for bound states (Bar) to describe reactions, building on prior resonating-group method approaches by Efros and collaborators (qn, bar, nq, qnrh) that used integro-differential equations and became increasingly complicated for heavier fragments. The proposed linear equations avoid the "free-free" matrix elements required by the Hulthen-Kohn variational method and the complicated antisymmetrization formulae of earlier work (Nav). The method enables ab initio reaction calculations using standard NCSM wave functions without requiring Jacobi coordinates or fragment-specific formulae, and the model examples suggest that convergence may be achievable with a moderate number of many-body oscillator quanta.
