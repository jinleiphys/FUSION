# Numerical modelling of Bose-Einstein correlations

> We propose extension of the algorithm for numerical modelling of Bose-Einstein correlations (BEC), which was presented some time ago in the literature. It is formulated on quantum statistical level for a single event and uses the fact that identical particles subjected to Bose statistics do bunch themselves, in a maximal possible way, in the same cells in phase-space. The bunching effect is in our case obtained in novel way allowing for broad applications and fast numerical calculations. First comparison with $e^+e^-$ annihilations data performed by using simple cascade hadronization model is very encouraging.

# Numerical modelling of Bose-Einstein correlations

## Key claim

The authors propose a generalisation of an earlier algorithm for modelling Bose-Einstein correlations (BEC) that can be applied to essentially any event generator, by reassigning charges to already-produced particles in each event to maximise same-charge bunching in phase-space cells, while leaving the original energy-momentum pattern and single-particle distributions intact. The method is claimed to produce BEC of all orders and to be fast and broadly applicable, but the authors note that it works only when one can resign from controlling the charge flow during hadronisation.

## Method

The algorithm operates on a single-event basis: for each event, a charge sign (+,-,0) is randomly chosen with probability proportional to the remaining multiplicity of that sign, and assigned to a randomly selected particle. Distances in momentum space between this particle and all other unassigned particles are computed and sorted; a probability P(i,j) (either constant or derived from the generator's output) determines whether each neighbour also receives the same charge, forming a phase-space cell. The procedure repeats until all particles have been reassigned charges, preserving the total number of particles of each charge per event. The method is tested on a one-dimensional cascade hadronisation model (CAS) that initially shows no Bose-Einstein statistics.

## Key numbers

- For constant probability P=0.5, the extracted "size" parameter R from exponential fits varies weakly between 0.66 fm (for source mass M=10 GeV) and 0.87 fm (for M=100 GeV).
- For the "minimal" weight P(ij) = exp[-0.5 * delta^2_ij(x) * delta^2_ij(p)], R varies from 0.64 fm (M=10 GeV) to 0.44 fm (M=100 GeV).
- For a single source with P=0.5, the "real" (mean) sizes of CAS sources change from 0.43 fm (M=10 GeV) to 1.82 fm (M=100 GeV).
- For a source composed of multiple subsources treated together ("Split" type) with P=0.5, R increases from 0.87 fm (1 subsource) to 3.35 fm (8 subsources); for the "minimal" weight, R increases from 0.57 fm to 5.59 fm over the same range.
- For independently treated subsources ("Indep" type), C2(Q=0)-1 = lambda falls roughly as 1/2^k with increasing number of subsources k.
- The "best fit" to DELPHI e+e- annihilation data at M=91.3 GeV is obtained with two or three subsources.

## Context

This work extends the approach of OMT (1999) and BSWW (1999), which modelled BEC by constructing a dedicated generator that groups bosons in phase-space cells according to a Bose-Einstein distribution from the start. The present algorithm instead post-processes the output of any event generator (such as the CAS cascade model) by reassigning charges to produce bunching, making it applicable where the original generator lacks Bose statistics. The method enables fast numerical modelling of BEC and can be used to fit experimental data from e+e- annihilations (DELPHI, 2000) and potentially to understand the inter-W BEC problem in W+W- final states, where the authors suggest treating each W as an independent source.
