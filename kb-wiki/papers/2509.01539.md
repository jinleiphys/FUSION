# Physics-Guided Neural Networks for Constructing Nucleon-Nucleon Inverse Potentials

> We propose a physics-guided neural network (PGNN) framework for constructing nucleon-nucleon inverse potentials based on inverse scattering theory. The framework integrates the Phase Function Method (PFM) with a two-stage supervised multi-layer perceptron (MLP) model to extract the optimal parameters of the Malfliet-Tjon (MT) potential from sparse phase-shift data. A synthetic dataset of phase shifts is generated by solving the phase equation for angular momentum $\ell$ = 0, using the fifth-order Runge-Kutta method, ensuring physically consistent training data. The first neural network predicts the attractive potential strength, $\tilde{V}_A$, while the second estimates the repulsive strength, $\tilde{V}_R$. The optimal range parameter, $μ$, is obtained through error minimization between predicted and expected phase shifts, thereby enhancing both stability and accuracy compared to conventional inversion techniques. The PGNN framework is validated for the $^1S_0$ state of neutron-proton (n-p), proton-proton (p-p), and neutron-neutron (n-n) scattering at low energies. The constructed inverse potentials accurately reproduce the phase shifts reported in the literature and exhibit the expected features of nucleon-nucleon interactions, including a short-range repulsive core and an intermediate-range attractive well, with the n-p system showing the deepest potential minimum due to stronger binding. These results demonstrate that the proposed PGNN framework provides an efficient and accurate approach for constructing nuclear potentials, effectively bridging machine learning techniques with quantum scattering theory.

## Key claim

The paper proposes a physics-guided neural network (PGNN) framework that integrates the Phase Function Method with a two-stage supervised multi-layer perceptron model to extract optimal parameters of the Malfliet-Tjon potential from sparse phase-shift data, but the validation is limited to the \(^1S_0\) state at low energies and the framework's performance depends on the quality of the synthetic dataset generated from the same potential form used for inversion.

## Method

The authors generated a synthetic dataset of 10,000 unique potential configurations by uniformly sampling the Malfliet-Tjon potential parameters \(V_A \in [-100, 100]~\mathrm{fm^{-2}}\), \(V_R \in [-300, 300]~\mathrm{fm^{-2}}\), and \(\mu \in [0.01, 4]~\mathrm{fm^{-1}}\), then solving the phase equation for \(\ell=0\) using the fifth-order Runge-Kutta method at 11 laboratory energies from 1 to 350 MeV. Two separate multi-layer perceptron networks, each with three hidden layers of 128, 64, and 64 neurons using ReLU activation, were trained sequentially: the first network \(F_1\) predicts the attractive strength \(\tilde{V}_A\) from \(\mu\) and the phase shifts, while the second network \(F_2\) predicts the repulsive strength \(\tilde{V}_R\) using \(\mu\), the predicted \(\tilde{V}_A\), and the same phase shifts. The optimal range parameter \(\mu\) was then determined by sweeping over the interval \([0.01, 4]~\mathrm{fm^{-1}}\) with step size 0.01 and minimizing the mean squared error between simulated and expected phase shifts.

## Key numbers

- Synthetic dataset: 10,000 unique potential configurations generated for each of the n-n, n-p, and p-p scattering systems.
- Laboratory energies used: 1, 5, 10, 25, 50, 100, 150, 200, 250, 300, 350 MeV (11 energies total).
- Validation losses for n-n system: \(F_1\) loss = \(7.00 \times 10^{-4}\), \(F_2\) loss = \(6.59 \times 10^{-3}\).
- Validation losses for n-p system: \(F_1\) loss = \(5.49 \times 10^{-4}\), \(F_2\) loss = \(5.22 \times 10^{-4}\).
- Validation losses for p-p system: \(F_1\) loss = \(6.20 \times 10^{-4}\), \(F_2\) loss = \(6.29 \times 10^{-4}\).
- Optimal \(\mu\) values: \(\mu_{\mathrm{opt}}^{(n-n)} = 2.210~\mathrm{fm^{-1}}\), \(\mu_{\mathrm{opt}}^{(n-p)} = 2.285~\mathrm{fm^{-1}}\), \(\mu_{\mathrm{opt}}^{(p-p)} = 2.335~\mathrm{fm^{-1}}\).
- Optimal potential parameters for n-n: \(V_A^* = 29.731~\mathrm{fm^{-2}}\), \(V_R^* = 123.411~\mathrm{fm^{-2}}\).
- Optimal potential parameters for n-p: \(V_A^* = 30.344~\mathrm{fm^{-2}}\), \(V_R^* = 123.095~\mathrm{fm^{-2}}\).
- Optimal potential parameters for p-p: \(V_A^* = 33.797~\mathrm{fm^{-2}}\), \(V_R^* = 147.367~\mathrm{fm^{-2}}\).

## Context

This work extends the authors' previous approach (Awasthi 2024) that combined the Phase Function Method with a reference potential approach using three Morse functions optimized by a Genetic Algorithm for systems including alpha-helium-3, alpha-tritium, neutron-deuteron, and proton-deuteron scattering. It contrasts with traditional phenomenological potentials such as Argonne \(V_{18}\) (Wiringa 1995), CD-Bonn (Machleidt 2001), and Nijmegen (Stoks 1994), which require fitting 40-50 free parameters, and with chiral effective field theory potentials (Machleidt 2011) that depend on cutoff choices and regularization schemes. The framework enables efficient construction of nucleon-nucleon inverse potentials from sparse phase-shift data and could be extended to higher partial waves and broader energy ranges, potentially accelerating nuclear many-body calculations by providing a machine-learning-based alternative to traditional inversion techniques.
