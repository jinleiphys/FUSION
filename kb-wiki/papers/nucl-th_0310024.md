# A new screening function for Coulomb renormalization

> We introduce a new screening function which is useful for the few-body Coulomb scattering problem in ``screening and renormalization'' scheme. The new renormalization phase factor of the screening function is analytically shown. The Yukawa type of the screening potential has been used in several decades, we modify it to make more useful. As a concrete example, we compare the proton-proton scattering phase shifts calculated from these potentials. The numerical results document that high precision calculations of the renormalization are performed by the new screening function.

# A new screening function for Coulomb renormalization

## Key claim
The authors introduce a generalized screening function of the form exp[-(r/R)^n]/r for Coulomb renormalization in few-body scattering, which for n >= 2 yields high-precision results at screening radii as small as R = 50 fm, whereas the standard Yukawa-type screening (n=1) requires R ~ 600 fm for comparable accuracy.

## Method
The authors modify the standard Yukawa-type screened Coulomb potential V^R(r) = e^2 exp(-r/R)/r to a generalized form V^R_n(r) = e^2 exp[-(r/R)^n]/r, and derive the corresponding analytic renormalization phase factor phi_R = -eta [ln(2pR) - gamma/n]. They then compute proton-proton scattering phase shifts in the ^1S_0 state using the Reid Soft Core potential plus the screened Coulomb potential, and compare the deviation |Delta delta| = |delta^C - delta^R - eta[gamma/n - ln(2pR)]| as a function of center-of-mass energy E_cm, screening radius R, and the power n.

## Key numbers
- The standard Yukawa screening (n=1) used by Alt et al. required a screening radius R ~ 600 fm for successful three-body scattering calculations.
- The new screening function with n=2 and R = 50 fm is reported as "a good enough choice" to perform high-precision calculations.
- The renormalization phase for the new screening function is phi_R = -eta [ln(2pR) - gamma/n], where gamma = 0.5772... is the Euler number.
- For n -> infinity, the renormalization phase reduces to phi_R = -eta ln(2pR), corresponding to a sharp cut-off potential.

## Context
This work extends the "screening and renormalization" approach for few-body Coulomb scattering problems, which was previously applied by Alt et al. (2002) using a Yukawa-type screened potential. The new screening function bridges the gap between the Yukawa-type (n=1) and sharp cut-off (n -> infinity) potentials, enabling calculations with significantly smaller screening radii. This reduces computational resource requirements in memory and elapsed time for few-body charged-particle scattering calculations.
