# Approximation of scattering phases for Reid93 potential

> For a single-channel nucleon-nucleon scattering, a well-known and convenient variable phase approach is considered, which is widely used for practical problems of atomic and nuclear physics. Approximation of the $pp$- and $np$- scattering phases obtained for the modern realistic phenomenological nucleon-nucleon potential Reid93 was carried out. The approximation function is used as a well-known formula for a parabolic-type quadratic function.

## Key claim

The paper demonstrates that the scattering phases for the Reid93 nucleon-nucleon potential, obtained via the variable phase approach and from the original Nijmegen analysis, can be approximated by a simple parabolic quadratic function, with the approximation quality varying by spin state.

## Method

The variable phase approach (phase-function method) was used to solve the radial Schrodinger equation for single-channel nucleon-nucleon scattering, yielding phase shifts for pp- and np-scattering for the Reid93 potential. These phase shifts, along with those from the original Reid93 paper by Stoks (1994), were then fitted using a parabolic quadratic function y(x) = a + bx + cx^2, following the approach previously applied to the Paris potential by Dolgopolov (2017). The quality of each fit was evaluated using the standard deviation, chi-squared per degree of freedom, and correlation coefficient.

## Key numbers

- For pp-scattering in the 1S0 state from Zhaba (2015-2016): standard deviation sigma = 7.10054, correlation coefficient R = 0.96600, with coefficients a = 50.54306, b = -27.22381, c = 2.71849
- For pp-scattering in the 1D2 state from Stoks (1994): sigma = 0.12204, R = 0.99963, with coefficients a = -0.24939, b = 4.41924, c = -0.39851
- For np-scattering in the 1S0 state from Stoks (1994): sigma = 2.53466, R = 0.99674, with coefficients a = 61.73167, b = -38.2194, c = 5.19865
- For np-scattering in the 3P0 state from Zhaba (2015-2016): sigma = 3.35480, R = 0.93191, with coefficients a = 3.97839, b = 5.63373, c = -3.38344
- For np-scattering in the 1D2 state from Stoks (1994): sigma = 0.13709, R = 0.99956, with coefficients a = -0.26844, b = 4.52958, c = -0.40765
- The minimum correlation coefficient across all states was for the 3P0 state (both pp- and np-scattering), and the maximum was for the 1D2 state
- The approximation is valid in the laboratory energy interval T_lab = 0-350 MeV

## Context

This work extends the approximation method previously applied to the Paris potential by Dolgopolov (2017) to the modern realistic Reid93 potential, using phase shifts calculated via the variable phase approach by Zhaba (2015-2016) and from the original Nijmegen analysis by Stoks (1994). The results show that phase shifts from the Reid93 potential agree within two percent when calculated by different methods, and within five percent when compared to other potentials (NijmI, NijmII, Argonne v18, CD-Bonn) and partial wave analysis. The quadratic approximation enables convenient recalculation of scattering amplitudes, total cross-sections, and partial scattering amplitudes without needing to compute phase shifts separately for each energy value, and the approach can be extended to other modern phenomenological potentials.
