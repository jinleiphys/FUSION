# The d'-dibaryon in a colored cluster model

> We calculate the mass and structure of a J^P=0^-, T=0 six-quark system using a colored diquark-tetraquark cluster wave function and a nonrelativistic quark model Hamiltonian. The calculated mass is some 350 MeV above the empirical value if the same confinement strength as in the nucleon is used. If the effective two-body confinement strength is weaker in a compound six-quark system than in a single baryon, as expected from a simple harmonic oscillator model, one obtains M_d' = 2092 MeV close to experiment.

# The d'-dibaryon in a colored cluster model

## Key claim

The J^P=0^-, T=0 six-quark system (d'-dibaryon) can be described by a colored diquark-tetraquark cluster model, but only if the effective two-body confinement strength is weaker in the six-quark system than in a single baryon; with the same confinement strength as in the nucleon, the calculated mass is some 350 MeV above the empirical value.

## Method

The authors used a nonrelativistic quark model Hamiltonian with one-gluon-exchange, two-body confinement, and regularized one-pion and one-sigma exchange between constituent quarks, and applied the Resonating Group Method (RGM) to a six-quark wave function expanded in a diquark-tetraquark cluster basis. The antisymmetrizer included quark permutation operators to respect the Pauli principle for the whole six-quark system, automatically incorporating q^3-q^3, q^2-q^4, and q^1-q^5 partitions as well as the q^6 compound state. Two parameter sets were compared: one with universal confinement strength for baryons and dibaryons (set I), and one with a weaker effective two-body confinement in the six-quark system derived from a harmonic oscillator model (set II).

## Key numbers

- Experimental d' mass: M_d' = 2065 MeV, with free hadronic decay width Gamma_d' approximately 0.5 MeV
- With universal confinement strength (set I): M_d' = 2440 MeV, nearly 400 MeV above experiment; diquark mass M_D = 643 MeV, tetraquark mass M_T = 1456 MeV; harmonic oscillator parameter b_6 = 0.75 fm; rms intercluster radius r_RGM_d' = 1.10 fm
- With weaker confinement in six-quark system (set II): M_d' = 2092 MeV, close to experiment; diquark mass M_D = 621 MeV, tetraquark mass M_T = 1309 MeV; b_6 = 0.95 fm; rms intercluster radius r_RGM_d' = 1.39 fm
- Quark exchange interactions (Pauli principle) contribute an additional 80-120 MeV to the d' mass (M_no QEX = 2316 MeV for set I, 2013 MeV for set II)
- Constituent quark mass: m_q = 313 MeV = m_N/3

## Context

This work extends the bag-string model of Mulders (1980), which predicted M_d' approximately 2100 MeV using a single non-antisymmetrized q^2-q^4 dumbbell configuration, by including exact center-of-mass removal, full antisymmetrization, and a dynamical determination of the orbital configuration via RGM. The results contradict the bag-string assumption of inert colored clusters at the ends of a stretched bag, as the RGM wave function shows considerable cluster overlap and resembles a pure six-quark harmonic oscillator state. The model accurately reproduces the deuteron mass and, with the weaker confinement hypothesis, yields a d' mass compatible with experiment; subsequent decay width calculations by Itonaga (1996) showed that for M_d' approximately 2100 MeV the pionic decay width is also compatible with experiment.
