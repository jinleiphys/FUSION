# 3D Imaging in Heavy-Ion Reactions

> We report an extension of the source imaging method for imaging full three-dimensional sources from three-dimensional like-pair correlations. Our technique consists of expanding the correlation data and the underlying source function in spherical harmonics and inverting the resulting system of one-dimensional integral equations. With this method of attack, we can image the source function quickly, even with the extremely large data sets common in three-dimensional analyses. We apply our method to the recently measured E859 un-Coulomb corrected data.

## Key claim

The paper extends the one-dimensional source imaging method to three dimensions by expanding correlation data and the source function in spherical harmonics, enabling fast inversion even with large datasets. The authors apply this technique to AGS-E859 data and find a two-component structure in the outward direction, but note that this result is tentative because the data may contain an unphysical anti-correlation artifact that Gaussian fits are insensitive to.

## Method

The authors convert the three-dimensional Koonin-Pratt equation into a series of one-dimensional integral equations by expanding the correlation function and source function in spherical harmonics and the kernel in Legendre polynomials. They represent the radial dependence of the source terms using Basis splines and solve the resulting linear inverse problem by minimizing chi-squared to obtain source coefficients. The method is tested on a simulated emission function with finite lifetime (20 fm/c), temperature (50 MeV), and Gaussian spatial profile (R_x, R_y, R_z = 4 fm), then applied to 14.6 AGeV Si+Au data from the AGS-E859 experiment.

## Key numbers

- Test problem source radii from imaging: Rs = 4.15 ± 0.08 fm, Ro = 5.17 ± 0.12 fm, Rl = 4.53 ± 0.09 fm
- Test problem source height at origin: S(r=0) = 13 ± 3 × 10^(-5) fm^(-3)
- Test problem resolved pair fraction: lambda = 0.566 ± 0.025
- Test problem emission function lifetime: tau = 20 fm/c
- Test problem temperature: T = 50 MeV
- Test problem Gaussian spatial profile radii: R_x, R_y, R_z = 4 fm

## Context

This work extends the one-dimensional source imaging technique introduced in earlier work (Imag, year not specified) to full three-dimensional analysis, allowing direct extraction of source functions without assuming a Gaussian form. The method is applied to data from the AGS-E859 experiment (14.6 AGeV Si+Au), and the authors note that the imaged source shows differences from the Gaussian fits previously reported by E859, particularly a two-component structure in the outward direction that may indicate a finite emission lifetime in the longitudinally co-moving frame. The authors plan to merge their imaging code with the CorAL code and Scott Pratt's kernel code, and to apply the method to PHENIX Run2 data.
