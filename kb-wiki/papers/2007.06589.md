# Satisfying the compressibility sum rule in neutron matter

> The static-response function of strongly interacting neutron matter contains crucial information on this interacting many-particle system, going beyond ground-state properties. In the present work, we tackle this problem with quantum Monte Carlo (QMC) approaches at several different densities, using both phenomenological forces and (for the first time) chiral effective field theory interactions. We handle finite-size effects via self-consistent energy-density functional (EDF) calculations for 8250 particles in a periodic volume. We combine these QMC and EDF computations in an attempt to produce a model-independent extraction of the static response function. Our results are consistent with the compressibility sum rule, which encapsulates the limiting behavior of the response function starting from the homogeneous equation of state, without using the sum rule as an input constraint. Our predictions on inhomogeneous neutron matter can function as benchmarks for other many-body approaches, thereby shedding light on the physics of neutron-star crusts and neutron-rich nuclei.

## Key claim
The authors demonstrate that the static-response function of neutron matter extracted from quantum Monte Carlo (QMC) calculations, when combined with a novel energy-density functional (EDF) based finite-size correction, satisfies the compressibility sum rule (CSR) without using the sum rule as an input constraint. The key hedge is that this consistency is achieved only after replacing the standard non-interacting-gas finite-size prescription with a new EDF-guided prescription that accounts for interaction effects on finite-size errors.

## Method
The authors performed Auxiliary Field Diffusion Monte Carlo (AFDMC) calculations for 66 neutrons in a periodic box at four densities (0.04, 0.06, 0.08, and 0.1 fm^-3), using both the phenomenological AV8'+UIX potential and local N^2LO chiral interactions. They applied an external cosine potential with seven different periodicities and five strength parameters to extract the static-response function. To correct for finite-size effects, they developed a self-consistent Skyrme-Hartree-Fock code to compute the response for 8250 particles using five different Skyrme parametrizations (SLy4, SkM*, KDE0v1, NRAPR, SKRA), then used the average EDF-based correction to extrapolate the 66-particle QMC results to the thermodynamic limit.

## Key numbers
- Four densities studied: 0.04, 0.06, 0.08, and 0.1 fm^-3
- QMC calculations used 66 particles in a periodic box
- EDF finite-size calculations used 8250 particles (66 x 5^3)
- Seven response points obtained per density: 1, 2, 3, 4, 6, 8, and 10 periods of the external potential inside the box
- Five strength parameters of the external potential used for each periodicity
- Five Skyrme parametrizations used for the EDF-based finite-size prescription: SLy4, SkM*, KDE0v1, NRAPR, and SKRA
- Standard isovector gradient coefficient values for these functionals: C^Δn_1 = -15.7, -17.1, -11.4, -16.5, and -17.1 MeV fm^5 respectively

## Context
This work extends previous ab initio studies of the static response in neutron matter by Buraczynski (2016, 2017) by using chiral effective field theory interactions for the first time and by introducing a more sophisticated finite-size correction scheme. The results provide microscopic benchmarks for other many-body approaches and have direct application to neutron-star crust physics, where the modified isovector gradient coefficient impacts the transition density for the core-crust boundary as discussed by Lim (2017). The density-dependent isovector gradient coefficient extracted from combining QMC with EDF is consistent with results from the density-matrix expansion by Kaiser (2012).
