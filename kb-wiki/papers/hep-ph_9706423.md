# Structure of Vacuum Condensates

> It is essential to know the space-time structure of the nonlocal vacuum condensates for application to medium energy processes. Using the Dyson-Schwinger formalism in the rainbow approximation for the quark propagator, we study the nonlocal quark condensate and model forms for the nonperturbative gluon propagator constrained by fits to local condensates and deep inelastic scattering with nucleon targets.

## Key claim

Using the rainbow Dyson-Schwinger formalism with a model gluon propagator, the authors show that the nonlocal quark condensate can be described by a dipole form in space-time, with a parameter close to that previously found from fits to sea-quark distribution data, provided the gluon propagator has a small range parameter in the infrared.

## Method

The authors solve the rainbow Dyson-Schwinger equations for the quark propagator self-consistently, using a Gaussian model for the nonperturbative gluon propagator in either Landau gauge or a Feynman-like gauge. From the resulting dressed quark propagator functions A(p^2) and B(p^2), they compute the nonlocal quark condensate via a Fourier transform and extract the space-time function g(x^2) by normalizing to the local condensate.

## Key numbers

* The dipole fit parameter for the nonlocal condensate from prior work is κ² = 0.15 to 0.20 GeV² (Ref. jk1).
* Four parameter sets for the gluon propagator are used: Set 1 (Δ = 2.0 × 10⁻³ GeV², χ = 1.40 GeV); Set 2 (Δ = 1.0 × 10⁻² GeV², χ = 1.56 GeV); Set 3 (Δ = 2.0 × 10⁻² GeV², χ = 1.58 GeV); Set 4 (Δ = 2.0 × 10⁻³ GeV², χ = 1.23 GeV, Feynman-like gauge).
* All parameter sets reproduce the pion decay constant in the chiral limit, f_π = 88 MeV.
* Chiral low-energy coefficients (×10⁻³) for Set 1: L₁ = 0.86, L₃ = -4.53, L₅ = 0.78, L₈ = 0.84; for Set 4: L₁ = 0.85, L₃ = -4.46, L₅ = 0.82, L₈ = 0.93.

## Context

This work extends prior Dyson-Schwinger studies of local condensates (Cahill, 1991; Frank and Meissner, 1996) to the nonlocal quark condensate, which is needed for form factors and transition matrix elements at low to medium momentum transfer where the operator product expansion fails. The dipole form of the nonlocal condensate was previously used by Jaffe and Kharzeev (1996) to fit parton distributions and the pion susceptibility; the present work provides a dynamical derivation of that form from the Dyson-Schwinger formalism, and shows that the result is robust against changing the gauge choice for the gluon propagator. The authors plan to extend the approach to nonlocal four-quark condensates using the Bethe-Salpeter formalism.
