# Stringification of Chiral Dynamics: Wess-Zumino interaction

> The QCD hadronic string is supplemented with the reparameterization-invariant boundary interaction to background chiral fields associated with pions in a way compatible with the conformal symmetry. It allows the full reconstruction of the P-even part of the Chiral Lagrangian in a good agreement with the phenomenology of P-even meson interactions. The modification of boundary interaction necessary to induce the parity-odd Chiral Dynamics (WZW action) is outlined.

# Stringification of Chiral Dynamics: Wess-Zumino interaction

## Key claim
The QCD hadronic string can be supplemented with a reparameterization-invariant boundary interaction to background chiral fields that reconstructs the P-even part of the Chiral Lagrangian, and a modification of this boundary interaction can induce the parity-odd Wess-Zumino-Witten action, provided the conformal symmetry (vanishing beta functions) is enforced at two-loop level in four dimensions.

## Method
The authors start from a minimal boundary Lagrangian for quark fields interacting with chiral fields \(U(x) = \exp(i\pi(x)/f_\pi)\) on the string boundary, previously shown to yield the P-even Chiral Lagrangian when conformal symmetry is imposed. They then derive a more general boundary Lagrangian from a projection of the Chiral Quark Model onto the one-dimensional string boundary, introducing two free parameters \(z\) and \(g_\sigma\). The consistency of the string with chiral fields is enforced by requiring vanishing beta functions (conformal symmetry) at one-loop in two-dimensional QCD and at two-loop in four-dimensional QCD, which fixes the parameters and generates the Wess-Zumino-Witten action.

## Key numbers
- The weak pion decay constant is given as \(f_\pi \simeq 90\ \text{MeV}\).
- The Regge trajectory slope is \(\alpha' \simeq 0.9\ \text{GeV}^{-2}\), and the string tension is \(T = 1/(2\pi\alpha')\).
- For the P-even sector, the dim-4 chiral structural constants are predicted as \(L_1 = \frac12 L_2 = -\frac14 L_3 = f_\pi^2 \alpha' / 16 = f_\pi^2 / (32\pi T)\).
- In two-dimensional QCD, the unitarity condition from the one-loop beta function gives \(g_A^2 - z^2 = 1\), with the QCD bosonization choice \(z = 0,\ g_A = 1\).
- In four-dimensional QCD, the one-loop unitarity condition becomes \(3g_A^2 - z^2 = 1\).
- The anomaly-derived relation between scales is \(f_\pi^2 \simeq 1/(16\pi^2 \alpha') = T/(8\pi)\).

## Context
This work extends the hadronic string approach to chiral dynamics developed in earlier work by the same group (Andrianov, Espriu, and collaborators, cited as "string" in the text), which had only reconstructed the parity-even sector of the Chiral Lagrangian. It also builds on the Chiral Quark Model and the known Wess-Zumino-Witten action (Witten, cited as "witt" in the text). The result enables a unified string description of both parity-even and parity-odd meson interactions, and provides a relation between the pion decay constant and the string tension that ties Goldstone boson physics to the fundamental scale of the QCD string.
