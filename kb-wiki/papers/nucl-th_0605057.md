# Quark-gluon vertex dressing and meson masses beyond ladder-rainbow truncation

> We include a generalized infinite class of quark-gluon vertex dressing diagrams in a study of how dynamics beyond the ladder-rainbow truncation influences the Bethe-Salpeter description of light quark pseudoscalar and vector mesons. The diagrammatic specification of the vertex is mapped into a corresponding specification of the Bethe-Salpeter kernel, which preserves chiral symmetry. This study adopts the algebraic format afforded by the simple interaction kernel used in previous work on this topic. The new feature of the present work is that in every diagram summed for the vertex and the corresponding Bethe-Salpeter kernel, each quark-gluon vertex is required to be the self-consistent vertex solution. We also adopt from previous work the effective accounting for the role of the explicitly non-Abelian three gluon coupling in a global manner through one parameter determined from recent lattice-QCD data for the vertex. With the more consistent vertex used here, the error in ladder-rainbow truncation for vector mesons is never more than 10% as the current quark mass is varied from the u/d region to the b region.

## Key claim

The paper shows that when a more self-consistent infinite class of quark-gluon vertex dressing diagrams is included (where every internal vertex is itself the fully dressed vertex), the error in the ladder-rainbow truncation for vector meson masses is never more than 10% as the current quark mass is varied from the u/d region to the b region, provided the effective non-Abelian three-gluon coupling is accounted for through a parameter fitted to lattice-QCD data.

## Method

The authors use the Munczek-Nemirovsky (MN) delta-function interaction kernel to reduce the quark Dyson-Schwinger equation and meson Bethe-Salpeter equation to algebraic form. They implement an iterative scheme for the quark-gluon vertex where each diagram sums over all possible ways to attach one additional gluon line with dressed vertices at every internal location, generating a wider class of planar diagrams than the simple ladder sum. The corresponding chiral symmetry-preserving Bethe-Salpeter kernel is generated from the quark self-energy via the functional relation of Munczek (1995). An effective color factor C, fitted to lattice-QCD data for the vertex from Skullerud (2003, 2004), accounts for the non-Abelian three-gluon coupling contribution.

## Key numbers

* The effective color factor fitted to lattice-QCD data for the vertex is C = 0.51.
* With the more consistent vertex, the error in ladder-rainbow truncation for vector mesons is never more than 10% as the current quark mass is varied from the u/d region to the b region.
* The ladder-summed vertex (without the three-gluon coupling) gives alpha_1(0) approximately 0.94, while lattice data indicates alpha_1(0) greater than or approximately 2.2.
* The color factor of the omitted three-gluon term at one-loop is -N_c^2 times that of the retained ladder term; for N_c=3, the effective color factor for the pure ladder sum (omitting three-gluon coupling) is C = -1/8.

## Context

This work extends the analysis of Bhagwat (2004) by incorporating a wider class of vertex dressing diagrams where internal vertices are self-consistently dressed, rather than using bare vertices at internal gluon-quark couplings. It builds on the algebraic framework of the MN model used by Bender (1996, 2002) and Bhagwat (2004) to study corrections to ladder-rainbow truncation. The results enable a more reliable assessment of the importance of higher-order diagrams for the Bethe-Salpeter kernel in light and heavy meson systems, and provide guidance for developing more realistic truncation schemes beyond the simple ladder-rainbow approximation.
