# Universal Relations for Range Corrections to Efimov Features

> In a three-body system of identical bosons interacting through a large S-wave scattering length $a$, there are several sets of Efimov features related by discrete scale invariance. Effective field theory was recently used to derive universal relations between these Efimov features that include the first-order correction due to a nonzero effective range $r_s$. We reveal a simple pattern in these range corrections that had not been previously identified. The pattern is explained by the renormalization group for the effective field theory, which implies that the Efimov three-body parameter runs logarithmically with the momentum scale at a rate proportional to $r_s/a$. The running Efimov parameter also explains the empirical observation that range corrections can be largely taken into account by shifting the Efimov parameter by an adjustable parameter divided by a. The accuracy of universal relations that include first-order range corrections is verified by comparing with various theoretical calculations using models with nonzero range.

## Key claim
The paper reveals a simple pattern in first-order effective-range corrections to Efimov features that had not been previously identified: the corrections can be expressed as a term linear in the branch index \(n\) multiplied by a universal number \(\sigma = 1.095\). This pattern is explained by a renormalization-group running of the Efimov three-body parameter \(\kappa_*\) at a rate proportional to \(r_s/a\), which also provides a theoretical justification for the empirical shift \(\kappa_* \to \kappa_* + \Gamma/a\) observed by Kievsky and Gattobigio (2012).

## Method
The authors use an effective field theory (EFT) with atom and molecule fields, where the two-body parameters (scattering length \(a\) and effective range \(r_s\)) are tuned via a cutoff \(\Lambda\), and the three-body coupling \(h\) is expressed as a log-periodic function \(H(\Lambda)\). They expand \(H(\Lambda)\) to first order in \(r_s\) and identify a term proportional to \(\log(\Lambda/\mu_0)\) that breaks discrete scale invariance. This term is absorbed into a running Efimov parameter \(\bar{\kappa}_*(\mu_0,a) = (\mu_0/\kappa_*)^{-\gamma r_s/a} \kappa_*\), which is then used to derive next-to-leading-order (NLO) and renormalization-group-improved NLO universal relations for Efimov features such as \(a_{-,n}\), \(a_{+,n}\), and \(a_{*,n}\). The predictions are tested against numerical results from separable Gaussian potential models (Deltuva 2012) and a three-parameter Gaussian form-factor model (Schmidt et al. 2012).

## Key numbers
- Discrete scaling factor for identical bosons: \(\lambda = e^{\pi/s_0} = 22.6944\) with \(s_0 = 1.00624\).
- Universal numbers for zero-range limit: \(\theta_- = -1.50763\), \(\theta_+ = 0.316473\), \(\theta_* = 0.0707645\).
- Universal number \(\sigma = 1.095\) for the \(n\)-linear term in the NLO range expansion.
- Universal number \(\gamma = 0.351\) (within 5% of the analytic value from Ji et al.) for the running rate.
- Differences between coefficients \(J_i\): \(J_+ - J_- = 0.548\), \(J_* - J_- = 1.250\); after RG improvement: \(\tilde{J}_+ - \tilde{J}_- = 0.000\), \(\tilde{J}_* - \tilde{J}_- = 0.177\).
- For Deltuva's model: predicted ratio \((a_{-,3}/a_{-,2})/\lambda = 0.9998\) (input \(n=0,1\) ratios 0.7822 and 0.9665); agreement to four digits.
- For Schmidt et al. model with \(s_{\rm res}=100\): NLO prediction for \((a_{*,1}/a_{*,0})/\lambda\) is \(-8.814\) (unphysical), while RG-NLO gives 0.0002; for \(s_{\rm res}=1\): NLO gives \(-0.431\), RG-NLO gives 0.240.
- For the empirical shift prescription: predicted universal value \(J_* - J_+ = 0.702\); two models give 0.671 and 0.753.

## Context
This work extends the effective-field-theory framework of Ji, Phillips, and Platter (2010, 2011) for first-order range corrections to Efimov features, revealing a pattern not previously identified. It provides a renormalization-group interpretation that explains the empirical shift prescription of Kievsky and Gattobigio (2012) and Garrido et al. (2013), where range corrections are absorbed into a shifted Efimov parameter \(\kappa_* + \Gamma/a\). The results enable more accurate predictions of Efimov features for ultracold atoms (e.g., \(^7\)Li and \(^{133}\)Cs) and connect to van der Waals universality, which predicts \(r_s = 2.79 r_{\rm vdW}\) and a narrow range for the coefficient \(J_-\).
