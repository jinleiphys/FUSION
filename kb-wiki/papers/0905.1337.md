# Gauge invariance of a critical number of flavours in QED3

> The fermion propagator in an arbitrary covariant gauge can be obtained from the Landau gauge result via a Landau-Khalatnikov-Fradkin transformation. This transformation can be written in a practically useful form in both configuration and momentum space. It is therefore possible to anticipate effects of a gauge transformation on the propagator's analytic properties. These facts enable one to establish that if a critical number of flavours for chiral symmetry restoration and deconfinement exists in noncompact QED3, then its value is independent of the gauge parameter. This is explicated using simple forms for the fermion-photon vertex and the photon vacuum polarisation. The illustration highlights pitfalls that must be avoided in order to arrive at valid conclusions. Landau gauge is seen to be the covariant gauge in which the propagator avoids modification by a non-dynamical gauge-dependent exponential factor, whose presence can obscure truly observable features of the theory.

## Key claim

If a critical number of flavours for chiral symmetry restoration and deconfinement exists in noncompact QED3, then its value is independent of the covariant gauge parameter, as established through the Landau-Khalatnikov-Fradkin (LKF) transformation. This gauge invariance holds provided one uses a sound vertex ansatz in Landau gauge and transforms it appropriately, avoiding pitfalls that can obscure the result.

## Method

The authors used the LKF transformation to relate the fermion propagator in an arbitrary covariant gauge to the Landau gauge result, working in both configuration and momentum space. They employed simple models for the fermion-photon vertex (the central Ball-Chiu vertex) and the photon vacuum polarisation (a parameterised form incorporating dynamical chiral symmetry breaking feedback) to solve the Dyson-Schwinger equations for the fermion propagator. Order parameters for chiral symmetry restoration (the ratio of the mass function at zero momentum) and deconfinement (the location of the first zero of the configuration-space Schwinger function) were computed as functions of the number of flavours for several gauge parameters.

## Key numbers

- The critical number of flavours for chiral symmetry restoration in the model is \(N_f^c = \frac{32}{\pi^2} = 3.24\), independent of the gauge parameter \(\xi\).
- The Landau gauge one-loop contribution to the fermion wave function renormalisation \(A(p)\) vanishes; the perturbative series is reliable for \(p/e^2 \gg 1/40\) in Landau gauge, but only for \(p/e^2 \gg 1/4\) in Feynman gauge (\(\xi=1\)).
- The LKF mass parameter is \(\varsigma = \xi \frac{e^2}{8\pi} = \frac{1}{2} \xi \alpha\), which has unit mass dimension and sets the scale at which the gauge-dependent exponential factor dominates the infrared behaviour of the propagator for \(p < \varsigma \approx \xi/25\).
- The vacuum fermion condensate is manifestly independent of the gauge parameter.

## Context

This work extends the analysis of Bashir et al. (2008), which showed that noncompact QED3 can possess a critical number of flavours if the fermion wave function renormalisation and photon vacuum polarisation are homogeneous functions at infrared momenta. It contradicts the notion that the critical number might depend on gauge, and supports the view that Landau gauge is special because it avoids a non-dynamical gauge-dependent exponential factor that can obscure observable features. The results enable future studies to focus on Landau gauge analyses, with confidence that conclusions about the critical number of flavours will persist in any covariant gauge, and highlight the need for quantitatively reliable Landau-gauge Dyson-Schwinger equation truncations.
