# AMD + HF model and its application to Be isotopes

> In order to study light unstable nuclei systematically, we propose a new method ''AMD + Hartree-Fock''. This method introduces the concept of the single particle orbits into the usual AMD. Applying AMD + HF to Be isotopes, it is found that the calculated lowest intrinsic states with plus and minus parities have rather good correspondence with the explanation by the two-center shell model. In addition, by active use of the single particle orbits extracted from AMD wave function, we construct the first excited 0^+ state of $^{10}$Be. The obtained state appears in the vicinity of the lowest 1^- state. This result is consistent with the experimental data.

## Key claim
The AMD+HF method introduces single-particle orbits into the usual AMD framework, and when applied to Be isotopes, the calculated lowest intrinsic states with plus and minus parities show "rather good correspondence" with the two-center shell model, though the correspondence fails for some states (e.g., the minus-parity state of $^{10}$Be and $^{11}$Be) due to parity mixing or insufficient single-particle description.

## Method
The authors propose the AMD+HF method, which extracts single-particle orbits from an AMD wave function by constructing an orthonormal basis from the non-orthogonal AMD single-particle wave functions, then diagonalizing a Hartree-Fock-like single-particle Hamiltonian matrix within that basis. The wave function is a Slater determinant of single Gaussians (one Gaussian per nucleon), and energy variation is performed after parity projection using a cooling equation with constraints on deformation parameter $\beta$. The method is applied to $^{6-11}$Be isotopes, and for $^{10}$Be the first excited $0^+$ state is constructed by projecting the plus-parity component from the intrinsic state of the lowest minus-parity state.

## Key numbers
- $^{9}$Be($-$): deformation parameter $\beta = 0.521$, binding energy B.E. = $-$46.96 MeV; highest neutron level has $\Omega^2 = 1.260$ and $L^2 = 2.996$
- $^{9}$Be($+$): $\beta = 0.801$, B.E. = $-$42.97 MeV; last neutron level has $\Omega^2 = 0.250$ and $L^2 = 4.885$
- $^{10}$Be($+$): $\beta = 0.381$, B.E. = $-$53.11 MeV; highest neutron level has $\Omega^2 = 1.241$ and $L^2 = 2.487$
- $^{10}$Be($-$): $\beta = 0.637$, B.E. = $-$44.95 MeV; highest neutron level has $\Omega^2 = 0.667$, $L^2 = 3.387$, and parity-mixing ratio $\epsilon_1^2 = 0.60$ (plus), $\epsilon_2^2 = 0.40$ (minus)
- $^{11}$Be($+$): $\beta = 0.505$, B.E. = $-$47.25 MeV; last neutron (halo level) has $\Omega^2 = 0.272$ and $L^2 = 4.463$
- $^{11}$Be($-$): $\beta = 0.271$, B.E. = $-$54.70 MeV; last neutron level has $\Omega^2 = 1.314$ and $L^2 = 2.350$
- $^{10}$Be first excited $0^+$ state appears in the vicinity of the lowest $1^-$ state, consistent with experimental data (6.179 MeV for $0^+$, 5.960 MeV for $1^-$)

## Context
This work extends the AMD approach, previously successful for light nuclei (as cited in the paper's references), by explicitly incorporating the concept of single-particle motion in the mean field, which was absent in earlier AMD formulations. The method enables systematic study of Be isotopes and comparison with the two-center shell model (cited as Ref. [18-20] and [24] in the paper). The construction of the first excited $0^+$ state of $^{10}$Be via particle-hole excitation demonstrates that AMD-HF orbits can be actively used to describe excited states, opening the way for studying neutron halo phenomena and excited-state configurations in unstable nuclei.
