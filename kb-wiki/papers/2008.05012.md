# Improving Hamiltonian encodings with the Gray code

> Due to the limitations of present-day quantum hardware, it is especially critical to design algorithms that make the best possible use of available resources. When simulating quantum many-body systems on a quantum computer, straightforward encodings that transform many-body Hamiltonians into qubit Hamiltonians use $N$ of the available basis states of an $N$-qubit system, whereas $2^N$ are in theory available. We explore an efficient encoding that uses the entire set of basis states, where terms in the Hamiltonian are mapped to qubit operators with a Hamiltonian that acts on the basis states in Gray code order. This encoding is applied to the commonly-studied problem of finding the ground state energy of a deuteron with a simulated variational quantum eigensolver (VQE). It is compared to a standard "one-hot" encoding, and various trade-offs that arise are analyzed. The energy distribution of VQE solutions has smaller variance than the one obtained by the one-hot encoding even in the presence of simulated hardware noise, despite an increase in the number of measurements. The reduced number of qubits and a shorter-depth variational ansatz enables the encoding of larger problems on current-generation machines. This encoding also demonstrates improvements for simulating time evolution of the same system, producing circuits for the evolution operators with reduced depth and roughly half the number of gates compared to a one-hot encoding.

## Key claim

The Gray code encoding uses exponentially fewer qubits than the standard one-hot encoding for the deuteron problem, but this comes at the cost of more Pauli terms and higher-weight operators; the trade-off is beneficial for noisy intermediate-scale quantum (NISQ) devices because the reduction in qubit count and circuit depth outweighs the increase in measurement requirements.

## Method

The authors map the deuteron Hamiltonian, expressed in a harmonic oscillator basis with matrix elements from pionless effective field theory, onto qubit Hamiltonians using both a one-hot encoding (N qubits for N basis states) and a Gray code encoding (⌈log₂ N⌉ qubits). They construct variational quantum eigensolver (VQE) circuits for each encoding, using a hardware-efficient ansatz for the Gray code and a cascade of controlled rotations for the one-hot encoding. The performance is compared through simulated VQE runs including shot noise and hardware noise models, and through analysis of Hamiltonian simulation circuits for time evolution.

## Key numbers

- For N=4 basis states, the one-hot Hamiltonian uses 4 qubits with 11 Pauli terms (Eq. 4), while the Gray code Hamiltonian uses 2 qubits with 8 Pauli terms (Eq. 5)
- For N=8, the Gray code Hamiltonian uses 3 qubits with 31 Pauli terms, compared to 8 qubits for one-hot
- The Gray code VQE ansatz for N=4 uses 2 two-qubit gates and 3 single-qubit gates, versus 7 two-qubit gates and 6 single-qubit gates for one-hot (before basis rotations)
- Total gates for VQE evaluation: one-hot uses 18N - 21 gates; Gray code uses 2(η+1)(2^η-1) - η^2 gates, where η = ⌈log₂ N⌉
- The deuteron ground-state energy converges to -2.224 MeV in the limit N → ∞ with ℏω = 7 MeV and V₀ = -5.68658111 MeV

## Context

This work extends the deuteron quantum simulation approach of Dumitrescu (2018) and Shehab (2019), who used one-hot encodings, by applying a Gray code encoding inspired by Lewis (2019) and Sawaya (2020). The Gray code approach is particularly efficient for tridiagonal Hamiltonians like the deuteron. The reduced qubit count and shallower circuits enable larger problems on current NISQ devices, and the encoding also produces time-evolution circuits with roughly half the number of gates compared to one-hot encoding.
