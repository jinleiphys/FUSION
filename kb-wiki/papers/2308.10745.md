# A novel method for calculating Bose-Einstein correlation functions with Coulomb final-state interaction

> Measurements of Bose-Einstein correlations played a crucial role in the discovery and the subsequent detailed exploration of the Quark-Gluon-Plasma (QGP) created in high-energy collisions of heavy nuclei. Such measurements gave rise to femtoscopy, a flourishing sub-field of high-energy physics, and there are important new directions to explore and discoveries to be made in the near future. One of these important current topics is the precise investigation of the shape of the correlation functions utilizing Lévy-stable sources. In this paper, we present a novel method of calculating the shape of the two-particle correlation functions, including the Coulomb final-state interaction. This method relies on an exact calculation of a large part of the necessary integrals of the Coulomb wave function and can be utilized to calculate the correlation function for any source function with an easily accessible Fourier transform. In this way, it is eminently applicable to Lévy-stable source functions. In this particular case, we find that the new method is more robust and allows the investigation of a wider parameter range than the previously utilized techniques. We present an easily applicable software package that is ready to use in experimental studies.

## Key claim

The paper presents a novel method for calculating two-particle Bose-Einstein correlation functions that include the Coulomb final-state interaction, which is more robust and computationally cheaper than existing techniques, and is applicable to any source function with an easily accessible Fourier transform, particularly Lévy-stable source functions. The method relies on an exact calculation of a large part of the necessary integrals of the Coulomb wave function, but the authors note that the generalization to non-spherical source functions is still being worked out.

## Method

The authors start from the standard expression for the two-particle correlation function with Coulomb interaction, which involves integrating the modulus squared of the Coulomb wave function weighted by the two-particle source function D(r). They rewrite this integral by expressing D(r) as the inverse Fourier transform of its Fourier transform f(q), then interchange the order of integration using an exponential regularization factor e^{-λr} and carefully taking the λ→0 limit. For spherically symmetric source functions, they perform the angular integrals analytically, leaving a one-dimensional integral over the magnitude of the auxiliary momentum variable q, which can be evaluated numerically.

## Key numbers

* The Gamow factor (normalization of the Coulomb wave function) is given by |N|² = 2πη/(e^{2πη} - 1), where η = (mc²α)/(2ħkc) is the Sommerfeld parameter and α ≈ 1/137 is the fine structure constant.
* For a Lévy-stable source with exponent α=2, the source function reduces to a Gaussian: L(α=2,R,r) = 1/(2πR²)^{3/2} e^{-r²/(2R²)}.
* For a Lévy-stable source with exponent α=1, the source function reduces to a Cauchy distribution: L(α=1,R,r) = 8/(π²R³) × 1/(1 + 4r²/R²)².
* For large r, the spherically symmetric Lévy distribution behaves as r²L(α,R,r) ∝ r^{-1-α} for α ≠ 2.
* The two-particle source function D(r) for a Lévy single-particle source S(r) with parameters α and R is given by D(r) = L(α, 2^{1/α}R, r).

## Context

This work extends previous femtoscopy studies that used Gaussian source functions (Lisa 2005, Csanád 2008, Pratt 2008) and later Lévy-stable source functions (PHENIX 2017, NA61/SHINE 2023, CMS 2023) to describe Bose-Einstein correlation measurements in heavy-ion collisions. The new method improves upon earlier numerical approaches used by Csanád (2019) and Kincses (2019) for calculating the Coulomb integral, which required computing the slowly decreasing D(r) function and then integrating it against the oscillating Coulomb wave function. The method enables self-consistent fitting procedures where the Coulomb effect is calculated at the actual fitted parameter values rather than at pre-defined arbitrary values, and the authors provide a ready-to-use software package for experimental studies.
