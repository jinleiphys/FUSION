# PRyMordial: The First Three Minutes, Within and Beyond the Standard Model

> In this work we present PRyMordial: A package dedicated to efficient computations of observables in the Early Universe with the focus on the cosmological era of Big Bang Nucleosynthesis (BBN). The code offers fast and precise evaluation of BBN light-element abundances together with the effective number of relativistic degrees of freedom, including non-instantaneous decoupling effects. PRyMordial is suitable for state-of-the-art analyses in the Standard Model as well as for general investigations into New Physics active during BBN. After reviewing the physics implemented in PRyMordial, we provide a short guide on how to use the code for applications in the Standard Model and beyond. The package is written in Python, but more advanced users can optionally take advantage of the open-source community for Julia. PRyMordial is publicly available on GitHub.

## Key claim

PRyMordial is a new public Python package that enables fast and precise computation of Big Bang Nucleosynthesis observables (light-element abundances and the effective number of relativistic degrees of freedom) within the Standard Model and for New Physics scenarios, by interconnecting a first-principles computation of the thermal background (including non-instantaneous neutrino decoupling) with an ab-initio calculation of neutron-to-proton conversion rates beyond the Born approximation.

## Method

The code solves coupled Boltzmann equations for the photon-electron-positron and neutrino energy densities to track the thermal bath temperature evolution, including finite-temperature QED corrections and non-instantaneous decoupling effects. It computes weak rates for neutron-proton interconversion using the full matrix element from the weak effective theory for beta decay, incorporating finite nucleon-mass effects, weak magnetism, and QED radiative corrections following the treatment in PRIMAT (Pitrou 2018). The BBN nuclear reaction network is evolved using tabulated thermonuclear rates, with the stiff differential equations solved in Python 3 with an optional Julia backend via the SciML kit.

## Key numbers

- Standard Model prediction for the effective number of relativistic degrees of freedom: \(N_{\rm eff} = 3.044\) at temperatures \(T_{\gamma} \ll\) MeV
- Neutrino temperature today: \(T_{\nu,0} = 0.1682\) meV, obtained from solving the Boltzmann system with photon temperature \(T_{\gamma,0} = 0.2348\) meV
- Relativistic neutrino relic abundance: \(\Omega^{\rm(rel)}_{\nu} h^2 = 5.70 \times 10^{-6}\)
- Non-relativistic neutrino relic abundance: \(\Omega^{\rm(nr)}_{\nu} h^2 = \sum_i m_{\nu_i} / (93.03\ {\rm eV})\)
- Neutron freeze-out temperature in the Standard Model: slightly below 1 MeV
- Deuterium binding energy: \(B_{\rm D} = 2.2\) MeV, with nucleosynthesis onset at temperatures around \(B_{\rm D}/\log(1/\eta_B) \sim 0.1\) MeV

## Context

This work extends the capabilities of existing BBN codes such as PArthENoPE (Pisanti 2007, Consiglio 2017, Gariazzo 2021), PRIMAT (Pitrou 2018), and AlterBBN (Arbey 2011, 2018) by providing a modern Python framework that combines precise thermal background computation (following the approach of NUDEC_BSM, Escudero 2019, 2020) with ab-initio weak rates and flexible BSM exploration. The code enables state-of-the-art analyses for next-generation CMB experiments and observational astronomy campaigns targeting percent-level determinations of primordial deuterium and helium-4 abundances.
