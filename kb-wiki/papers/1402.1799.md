# Practical Calculation Scheme for Generalized Seniority

> We propose a scheme or procedure for doing practical calculations with generalized seniority. It reduces the total computing time by calculating and storing in advance a set of intermediate quantities, taking advantage of the memory capability of modern computers. The requirements and performance of the algorithm are analyzed in detail.

# Practical Calculation Scheme for Generalized Seniority

## Key claim
The paper proposes a practical algorithm for generalized seniority calculations that reduces total computing time by precomputing and storing a set of intermediate "density matrix" quantities (pair-transfer amplitudes) in computer memory, but the method's limiting factor remains the many-body space dimension, which may still become intractably huge at relatively high seniority even with the seniority truncation.

## Method
The algorithm computes and stores in memory a set of pair-transfer amplitudes \( t_{\alpha_1...\alpha_p;\beta_1...\beta_q}^{M} \) defined on the pair condensate, using recursive relations derived from the closed algebra of pair creation, pair annihilation, and number operators. Matrix elements of arbitrary operators between seniority states are then expressed in terms of these stored amplitudes, which are calculated once and reused. For spherical shell model spaces with rotational symmetry, the amplitudes are stored in an "occupation representation" that counts how many times each \( j \)-level appears, independent of magnetic quantum numbers.

## Key numbers
- The number of different \( t \) amplitudes needed for a calculation truncated at seniority \( S = 2s \) is counted by solving \( \sum_i n_i^\alpha = p-\mu \), \( \sum_i n_i^\beta = p \), \( \sum_i n_i^\gamma = r \) with constraints \( 0 \le n_i^\alpha + n_i^\beta + n_i^\gamma \le \Omega_i \) for each \( j \)-level of degeneracy \( 2\Omega_i \).
- The number of non-negative integer solutions before applying the \( \Omega_i \) constraint is \( C_{r+D-1}^{D-1} C_{p+D-1}^{D-1} C_{p-\mu+D-1}^{D-1} \), where \( D \) is the number of \( j \)-levels.
- The allowed values satisfy \( 0 \le \nu \le s \), \( 0 \le \mu \le \nu \), \( \mu \le p \le \nu+1 \), and \( \nu+1-p \le r \le 2(\nu+1-p) \).
- Modern computers have several gigabytes of memory in a laptop and several terabytes at a workstation, making storage of the intermediate quantities feasible.

## Context
This work extends the generalized seniority formalism introduced by Talmi (1971) and others, which has been applied mainly for seniorities \( S \le 2 \) (one broken pair) using explicit expressions by Gambhir (1969, 1971), Bonsignori (1978), Pittel (1982), and Scholten (1983), or using recursive relations by Luo (2011) that become very time-consuming for higher seniorities. The proposed scheme enables calculations at higher seniorities by exploiting modern computer memory, and is designed to be compatible with existing "m-scheme" shell model codes, allowing additional truncations such as limiting the number of unpaired particles excited to upper major shells or using collective pairs with certain multiplicity as in the interacting boson model.
