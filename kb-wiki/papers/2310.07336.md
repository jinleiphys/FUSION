# Numerical Simulation Study of Neutron-Proton Scattering using Phase Function Method

> In this article, we propose a numerical approach to solve quantum mechanical scattering problems, using phase function method, by considering neutron-proton interaction as an example. The nonlinear phase equation, obtained from the time-independent Schrodinger equation, is solved using the Runge-Kutta method for obtaining S-wave scattering phase shifts for neutron-proton interaction modeled using Yukawa and Malfliet-Tjon potentials. While scattering phase shifts of S-states using Yukawa match with experimental data for only lower energies of 50 MeV, Malfliet-Tjon potential with repulsive term gives very good accuracy for all available energies up to 350 MeV. Utilizing these S-wave scattering phase shifts, low energy scattering parameters, and total S-wave cross section have been calculated and found to be consistent with experimental results. This simulation methodology can be easily extended to study scattering phenomenon using phase wave analysis approach in the realms of atomic, molecular, and nuclear physics.

# Numerical Simulation Study of Neutron-Proton Scattering using Phase Function Method

## Key claim

The phase function method solved with a 5th-order Runge-Kutta algorithm can reproduce experimental neutron-proton S-wave scattering phase shifts up to 350 MeV laboratory energy when using the Malfliet-Tjon potential, but only up to about 50 MeV when using the simple Yukawa potential. The authors claim this approach is accessible to undergraduate students using spreadsheet software.

## Method

The authors transformed the radial time-independent Schrodinger equation into a first-order nonlinear Ricatti-type phase equation for S-wave (l=0) scattering, then solved it numerically using 5th-order Runge-Kutta (RK-5) integration. They modeled the neutron-proton interaction using both the Yukawa potential (single attractive term) and the Malfliet-Tjon potential (attractive plus repulsive Yukawa-type terms), with model parameters optimized via a variational Monte Carlo procedure against experimental phase shift data from the Granada group and Nijmegen database. Scattering lengths and effective ranges were extracted from the low-energy phase shifts using the effective range expansion, and total S-wave cross sections were computed from the weighted average of triplet and singlet partial cross sections.

## Key numbers

- Yukawa potential matches experimental S-wave phase shifts only for laboratory energies up to about 50 MeV; Malfliet-Tjon potential matches up to 350 MeV
- Malfliet-Tjon potential parameters for triplet state (3S1): V_R = 9435.57 MeV, V_A = 2134.88 MeV, mu_A = 2.54 fm^-1, with MAPE of 0.61% up to 350 MeV
- Malfliet-Tjon potential parameters for singlet state (1S0): V_R = 6806.60 MeV, V_A = 1522.42 MeV, mu_A = 2.42 fm^-1, with MAPE of 1.91% up to 350 MeV
- Calculated triplet scattering length a = 5.534 +/- 0.032 fm (experimental: 5.397 +/- 0.011 fm); triplet effective range r0 = 1.705 +/- 0.012 fm (experimental: 1.727 +/- 0.013 fm)
- Calculated singlet scattering length a = -24.038 +/- 0.045 fm (experimental: -23.678 +/- 0.028 fm); singlet effective range r0 = 2.307 +/- 0.025 fm (experimental: 2.44 +/- 0.11 fm)
- Total S-wave cross section extrapolated to E = 0.000132 MeV gives 20.641 barns, compared to experimental deuteron cross section of 20.491 barns (Krane)
- Deuteron binding energy calculated from the 3S1 potential: -2.026 MeV (experimental: -2.225 MeV); unbound singlet state energy: 76 keV (expected: 77 keV)

## Context

This work extends the phase function method originally developed by Morse and Allis, Babikov, and Calogero to neutron-proton scattering using modern numerical techniques. It contrasts with traditional approaches like the R-matrix method, S-matrix method, or Jost function method that require full wavefunction matching. The authors demonstrate that the Malfliet-Tjon potential (Malfliet and Tjon, 1969) significantly outperforms the simple Yukawa potential (Yukawa, 1935) for reproducing experimental data across the full energy range. This methodology can be extended to other two-body scattering systems including neutron-deuteron, neutron-alpha, proton-proton, and alpha-alpha scattering, and is designed to be implementable by undergraduate students using spreadsheet environments such as Gnumeric, Excel, or LibreOffice-Calc.
