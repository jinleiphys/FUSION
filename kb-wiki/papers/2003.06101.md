# Study of nuclear matter properties for Hybrid EoS

> We study the nuclear matter properties like symmetry energy, slope parameter, curvature, skewness and incompressibility for Hybrid EoS. The hybrid EoS is constructed using the recently proposed Effective-Field-Theory motivated Relativistic Mean-Field model (E-RMF) for the hadron phase with different parameter sets and MIT Bag model for the quark phase with different Bag constants to construct hybrid EoS. The mixed-phase formed by the hadron-quark phase transition is studied using the Gibbs construction. The nuclear matter properties for hybrid EoS are calculated and their variation with the bag constant is determined.

## Key claim

The paper calculates nuclear matter properties (symmetry energy, slope parameter, curvature, skewness, and incompressibility) for a hybrid equation of state (EoS) that combines a hadronic phase described by Effective-Field-Theory motivated Relativistic Mean-Field (E-RMF) models with a quark phase described by the MIT Bag model, using the Gibbs construction for the mixed phase. The authors find that these properties vary with the bag constant, but the abstract does not specify that the results are only shown for a limited set of hadronic parameter sets (NL3, FSUGarnet, G3, IOPB-I) and that the values for the hybrid EoS are not tabulated in the provided text.

## Method

The hadronic phase was modeled using the E-RMF Lagrangian with four parameter sets (NL3, FSUGarnet, G3, IOPB-I), and the quark phase was modeled using the MIT Bag model for unpaired quark matter (u, d, s quarks) with bag constants ranging from B^{1/4} = 100 to 200 MeV. The mixed hadron-quark phase was constructed using the Gibbs construction, imposing global charge neutrality and beta-equilibrium conditions. The symmetry energy S(rho) and its derivatives (L, K_sym, Q_sym) were then calculated from the resulting hybrid EoS.

## Key numbers

*   For the hadronic phase with the NL3 parameter set: symmetry energy J = 37.43 MeV, slope parameter L = 118.65 MeV, curvature K_sym = 101.34 MeV, skewness Q_sym = 177.90 MeV, and incompressibility K = 271.38 MeV.
*   For the hadronic phase with the G3 parameter set: J = 31.84 MeV, L = 49.31 MeV, K_sym = -106.07 MeV, Q_sym = 915.47 MeV, and K = 243.96 MeV.
*   For the hadronic phase with the IOPB-I parameter set: J = 33.30 MeV, L = 63.58 MeV, K_sym = -37.09 MeV, Q_sym = 862.70 MeV, and K = 222.65 MeV.
*   For the hadronic phase with the FSUGarnet parameter set: J = 30.95 MeV, L = 51.04 MeV, K_sym = 59.36 MeV, Q_sym = 130.93 MeV, and K = 229.5 MeV.
*   The saturation density rho_0 for all parameter sets is approximately 0.148-0.153 fm^{-3}, with binding energy per nucleon epsilon_0 around -16.02 to -16.29 MeV.
*   The bag constant range considered for the quark phase is B^{1/4} = 100-200 MeV, with the authors noting that values 130 MeV < B^{1/4} < 160 MeV are suitable for explaining quark matter in neutron stars.

## Context

This work extends previous studies of nuclear matter properties from pure hadronic EoS to a hybrid EoS that includes a quark phase, using the Gibbs construction as pioneered by Glendenning (1992). It builds on the E-RMF formalism of Furnstahl, Serot, and Tang (1996) and uses parameter sets (NL3, FSUGarnet, G3, IOPB-I) that have been previously applied to finite nuclei and neutron stars. The results enable future calculations of neutron star mass-radius relations for hybrid stars, which can be compared to astrophysical observations to constrain the bag constant and the properties of the mixed phase.
