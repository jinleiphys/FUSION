# Predicting Beta Decay Energy with Machine Learning

> $Q_β$ represents one of the most important factors characterizing unstable nuclei, as it can lead to a better understanding of nuclei behavior and the origin of heavy atoms. Recently, machine learning methods have been shown to be a powerful tool to increase accuracy in the prediction of diverse atomic properties such as energies, atomic charges, volumes, among others. Nonetheless, these methods are often used as a black box not allowing unraveling insights into the phenomena under analysis. Here, the state-of-the-art precision of the $β$-decay energy on experimental data is outperformed by means of an ensemble of machine-learning models. The explainability tools implemented to eliminate the black box concern allowed to identify uncertainty and atomic number as the most relevant characteristics to predict $Q_β$ energies. Furthermore, physics-informed feature addition improved models' robustness and raised vital characteristics of theoretical models of the nuclear structure.

## Key claim

The authors claim that an ensemble of machine-learning models outperforms the state-of-the-art precision for predicting beta-decay energy from experimental data, but this result depends on augmenting the training data via Monte Carlo sampling of experimental uncertainties and injecting physics-informed features such as magic-number distances, rather than relying on raw nuclear features alone.

## Method

The authors trained an ensemble of models (ordinary least squares, multilayer perceptron, gradient-boosted decision trees via XGBoost, and TabNet attention network) on the AME2020 atomic mass evaluation dataset containing 2813 unstable isotopes with features including neutron number, proton number, mass number, mass excess, and binding energy. They augmented the training set from 1912 to 573600 samples by Monte Carlo sampling each variable's uncertainty assuming normally distributed errors, and injected physics-informed features including distances to magic numbers and the Z/N ratio. The ensemble combined predictions via weighted voting based on validation-set performance.

## Key numbers

* The ensemble model trained on the augmented dataset achieved RMSE = 0.828 MeV and R² = 0.987 on the test set.
* The ensemble model trained on the augmented dataset with injected physics features achieved RMSE = 0.714 MeV and R² = 0.991.
* The baseline model (previous approach using only N and Z with a single hidden layer of 100 neurons) achieved RMSE = 2.033 MeV and R² = 0.926.
* The ensemble model on the original AME2020 dataset (no augmentation) achieved RMSE = 1.518 MeV and R² = 0.957.
* The gradient-boosted decision tree (BDT) on the augmented dataset achieved RMSE = 0.865 MeV and R² = 0.986.
* The TabNet model on the augmented dataset achieved RMSE = 1.149 MeV and R² = 0.958.
* The ordinary least squares model achieved RMSE = 2.818 MeV and R² = 0.868.

## Context

This work extends prior machine-learning approaches to beta-decay energy prediction, such as Niu (2019) and Bayram (various years), which used only neutron and proton numbers as inputs and achieved higher RMSE values. The authors improve upon these by incorporating experimental uncertainties from the AME2020 database and adding physics-motivated features based on nuclear shell structure (magic numbers). The use of explainability tools (SHAP) to identify uncertainty and atomic number as the most relevant features enables future work to combine machine learning with nuclear structure theory for more robust predictions.
