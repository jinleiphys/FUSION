# Relativistic two-phonon model for low-energy nuclear response

> A two-phonon version of the relativistic quasiparticle time blocking approximation introduces as a new class of many-body models for nuclear structure calculations based on the covariant energy density functional. As a fully consistent extension of the relativistic quasiparticle random phase approximation, the relativistic two-phonon model implies fragmentation of nuclear states over two-quasiparticle and two-phonon configurations coupled to each other. In particular, we show how the lowest two-phonon $1^-$ state, identified as a member of the $[2^+\otimes 3^-]$ quintuplet, emerges from the coherent two-quasiparticle pygmy dipole mode in vibrational nuclei. The inclusion of the two-phonon configurations into the model space allows a quantitative description of the positions and the reduced transition probabilities of the lowest 1$^-$ states in tin isotopes $^{112,116,120,124}$Sn as well as the low-energy fraction of the dipole strength below the giant dipole resonance without any adjustment procedures. The model is applied to the low-lying dipole strength in neutron-rich $^{68,70,72}$Ni isotopes. Recent experimental data for $^{68}$Ni are reproduced fairly well.

## Key claim

The relativistic two-phonon model (RQTBA-2) extends the conventional relativistic quasiparticle time blocking approximation by including two-phonon configurations, which allows a quantitative description of the positions and reduced transition probabilities of the lowest 1⁻ states in tin isotopes and the low-energy dipole strength in neutron-rich nickel isotopes without any adjustment procedures, though the model is limited to vibrational nuclei and relies on the covariant energy density functional with the NL3 parameter set.

## Method

The method solves the Bethe-Salpeter equation for the nuclear response function within the relativistic quasiparticle time blocking approximation, using a covariant energy density functional with the NL3 parameter set and monopole pairing. The key innovation replaces the uncorrelated two-quasiparticle propagator in the phonon-coupling amplitude with the correlated RQRPA propagator, which introduces two-phonon configurations (e.g., [2⁺ ⊗ 3⁻]) into the model space. The calculations proceed in four steps: (i) solving the Relativistic Hartree-Bogoliubov equation for single-quasiparticle energies and wave functions, (ii) computing phonon frequencies, coupling vertices, and transition densities within self-consistent RQRPA, (iii) solving the Bethe-Salpeter equation for the correlated propagator with the two-phonon amplitude, and (iv) solving for the full response function in the momentum-channel representation.

## Context

This work extends the relativistic quasiparticle time blocking approximation (RQTBA) developed by Litvinova, Ring, and Tselyaev (2008) by incorporating two-phonon configurations, following the theoretical framework proposed by Tselyaev (2007) for the quasiparticle time blocking approximation. The approach builds on the Quasiparticle-Phonon Model (QPM) of Soloviev (1992) and the "Q-phonon" approach, but provides a fully self-consistent description based on covariant density functional theory without empirical input for phonon constituents. The model enables the description of fine structure in the low-energy dipole response below the giant dipole resonance, which is important for understanding the pygmy dipole resonance and its correlation with neutron skin thickness and symmetry energy, as well as for improving predictions of radiative neutron capture rates in r-process nucleosynthesis.
