# On Exotic Hybrid Pseudo-Potentials and Gauge/String Duality

> We use gauge/string duality to model some hybrid pseudo-potentials in a pure SU(N) gauge theory. The pseudo-potentials under consideration can't be described by a single Nambu-Goto string. This is why we call them "exotic". A comparison with the calculation of the pseudo-potential shows the universality of the spatial string tension.

## Key claim

The paper uses gauge/string duality to model hybrid pseudo-potentials (excited-state spatial Wilson loop energies) in pure SU(N) gauge theory at finite temperature, claiming that the spatial string tension extracted from these hybrid pseudo-potentials is universal, i.e., identical to that of the ground-state pseudo-potential, but only within the regime where the effective five(ten)-dimensional string description is reliable (T ≲ 3 T_c).

## Method

The author models hybrid pseudo-potentials by inserting a point-like "defect" (representing gluonic excitations) on a Nambu-Goto string stretched between a quark-antiquark pair in a deformed AdS₅ black-hole background (the "soft-wall" model). Two defect models are considered: Model A uses a five-dimensional point-particle action, while Model B uses a ten-dimensional brane-antibrane action. The pseudo-potential Σ_u^- is computed in parametric form by extremizing the total action (two string segments plus defect) and then evaluated numerically for a pure SU(3) gauge theory with parameters fixed from zero-temperature lattice data of Kuti (2006).

## Key numbers

* Spatial string tension σ_s: σ_s = σ for T ≤ T_c, and σ_s = σ (T/T_c)² exp{(T_c/T)² − 1} for T > T_c, where σ = 0.44 GeV² and T_c = √σ/π ≈ 211 MeV (from σ = 0.44 GeV²).
* Model parameters fixed from zero-temperature lattice data: γ = 0.176, σ = 0.44 GeV², C = 0.71 GeV, κ = 2.3 for Model A, κ = 2000 for Model B.
* Short-distance coefficient A (scheme-independent): at zero temperature, A = γ/4 (σ/λ_*)³/² exp(λ_*) / [∫₀¹ dv v² exp(λ_*(1−v²)) (1−(λ_*/τ)² v⁴)^{−½}], with λ_* the solution of ρ(λ)=0.
* Gap Δ_s between Σ_g⁺ and Σ_u⁻ at large R: for Model A, Δ_s = 2κ √(γ σ_s); for Model B at T ≥ T_c, Δ_s = 2κ (T/T_c) exp{−2(T_c/T)² − 1} √(e γ σ).

## Context

This work extends the author's previous gauge/string duality studies of spatial Wilson loops (Andreev 2012) and multiquark pseudo-potentials (Andreev 2011) to excited (hybrid) states, and parallels the zero-temperature hybrid potential construction of Andreev (2012) for the Σ_u⁻ state. The results enable lattice tests of the predicted temperature dependence of hybrid pseudo-potentials, which have not yet been computed on the lattice. The paper contrasts with the dimensionally reduced gauge theory approach (Laine 2006) and with lattice data for SU(2) and SU(3) spatial string tensions (Bali 1993).
