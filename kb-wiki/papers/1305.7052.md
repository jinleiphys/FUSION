# Relativistic viscous hydrodynamics for heavy-ion collisions with ECHO-QGP

> We present ECHO-QGP, a numerical code for $(3+1)$-dimensional relativistic viscous hydrodynamics designed for the modeling of the space-time evolution of the matter created in high energy nuclear collisions. The code has been built on top of the \emph{Eulerian Conservative High-Order} astrophysical code for general relativistic magneto-hydrodynamics [\emph{Del Zanna et al., Astron. Astrophys. 473, 11, 2007}] and here it has been upgraded to handle the physics of the Quark-Gluon Plasma. ECHO-QGP features second-order treatment of causal relativistic viscosity effects in both Minkowskian or Bjorken coordinates; partial or complete chemical equilibrium of hadronic species before kinetic freeze-out; initial conditions based on the optical Glauber model, including a Monte-Carlo routine for event-by-event fluctuating initial conditions; a freeze-out procedure based on the Cooper-Frye prescription. The code is extensively validated against several test problems and results always appear accurate, as guaranteed by the combination of the conservative (shock-capturing) approach and the high-order methods employed. ECHO-QGP can be extended to include evolution of the electromagnetic fields coupled to the plasma.

## Key claim

ECHO-QGP is a new (3+1)-dimensional relativistic viscous hydrodynamics code for heavy-ion collisions that preserves the conservative, shock-capturing, and high-order numerical methods of the original astrophysical ECHO code, while adding second-order causal viscosity, multiple equations of state, and Glauber-based initial conditions. The authors claim the code is accurate based on validation tests, but note that the inclusion of finite baryon density and electromagnetic field evolution are left for future work.

## Method

The code solves the conservation equations for energy-momentum and a tracer number density, coupled to Israel-Stewart-type second-order evolution equations for the bulk viscous pressure Π and shear stress tensor π^{μν}, written as a system of 12 balance laws in conservative form. The primitive variables (fluid velocity, energy density) are recovered via an iterative procedure that treats viscous corrections as small perturbations. The code operates in both Minkowski and Bjorken coordinates, uses either tabulated lattice-QCD-matched equations of state (EOS-I, EOS-LS, EOS-CE, EOS-PCE) or an ultrarelativistic ideal gas EOS, and initializes conditions via the optical Glauber model with optional Monte-Carlo event-by-event fluctuations.

## Key numbers

- Shear viscosity to entropy density ratio η/s varied in the range 0.08 ≤ η/s ≤ 0.16 for Glauber initial conditions, with a maximum suggested value of η/s ∼ 0.16 from RHIC elliptic flow measurements.
- For Color Glass Condensate type initial conditions, the maximum η/s is ∼ 0.24.
- Bulk viscosity to entropy density ratio set as ζ/s = 2(η/s)(1/3 − c_s^2), following strongly coupled gauge theory results.
- Shear and bulk relaxation times set as τ_π = τ_Π = 3η/(sT).
- Initial conditions for viscous terms in Bjorken coordinates at proper time τ_0: Π = −ζ/τ, 2π^{xx} = 2π^{yy} = −τ^2π^{ηη} = (4/3)η/τ.
- Matching between HRG and lattice-QCD EOS performed at temperature T = 150 MeV.
- The HRG EOS sums all hadrons and resonances up to a mass of 2 GeV.
- Transverse momentum spectra reproduced up to about 1.5–2.0 GeV/c, covering more than 99% of emitted particles.

## Context

This work extends the astrophysical ECHO code (Del Zanna et al., Astronomy and Astrophysics, 2007) to heavy-ion collision physics, building on prior second-order viscous hydrodynamics efforts by Israel and Stewart (1970s) and Müller, as well as more recent (3+1)-dimensional simulations by Hirano, Schenke, Bozek, and Pratt. The code enables future inclusion of electromagnetic field evolution coupled to the plasma, motivated by the possible role of large magnetic fields (up to ∼10^15 Tesla) in heavy-ion collisions, particularly in connection with non-trivial topological configurations of the color field (Kharzeev, 2008).
