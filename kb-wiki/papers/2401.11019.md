# Quark propagator with complex-valued momentum from Schwinger-Dyson equation in the Euclidean space

> In the Euclidean-space formulation of integral equations for the structure of quantum chromodynamics (QCD) bound states, the quark propagators with complex-valued momentum are densely sampled. We therefore propose an accurate and efficient algorithm to compute these propagators. The quark propagator both on the spacelike real axis and at complex-valued momenta is determined from its Schwinger-Dyson equation (SDE). We first apply an iterative solver to determine the quark propagator on the spacelike real axis. The propagator at complex-valued momenta is then computed from its self-energy based on this solution, where demanding integrals are encountered. In order to compute of these integrals, we apply customized variable transformations for the radial integral after subtracting the asymptotics. We subsequently apply an optional compound of quadrature rules for the angular integral. The contribution from the asymptotics is added at the last step. The accuracy and the performance of this algorithm for the quark propagator at complex-valued momentum are tested in comparison with an adaptive quadrature.

## Key claim

The paper proposes an accurate and efficient algorithm to compute quark propagators at complex-valued momenta from their Schwinger-Dyson equation (SDE) in Euclidean space, using fixed-grid quadrature rules with customized variable transformations and asymptotic subtraction, rather than relying on Cauchy integration methods that are limited by singularities nearest to the positive real axis.

## Method

The quark propagator on the spacelike real axis is first determined using an iterative solver applied to the SDE with the Maris-Tandy model for the gluon propagator. For complex-valued momenta, the propagator is computed from its self-energy using the real-axis solution, where the radial integrals are handled by subtracting the asymptotic contribution and applying a variable transformation to a Gauss-Chebyshev quadrature of the second kind with 201 points, while the angular integrals use a compound of two quadrature rules. When the external momentum satisfies Re{x} >= 7.5 GeV^2 and z*Re{sqrt(x)}/omega >= 2.0, the infrared contribution to the radial integral is computed separately using Gauss-Hermite quadrature.

## Key numbers

- The Gauss-Chebyshev quadrature of the second kind uses N_GC = 201 points for the mapped radial variable theta
- The threshold for separate treatment of the infrared term is Re{x} >= r_threshold = 7.5 GeV^2
- The threshold for the Gauss-Hermite quadrature condition is z*Re{sqrt(x)}/omega >= h_threshold = 2.0
- The scale parameter for the radial variable transformation is c = 1.0 GeV^2 for the Maris-Tandy model
- The Maris-Tandy model parameters used are: omega = 0.4 GeV, D = 0.859 GeV^2, N_f = 4, Lambda_QCD = 0.234 GeV, m_t = 0.5 GeV

## Context

This work extends the approach of Windisch (2016) for computing quark propagators at complex momenta from their SDE, and provides an alternative to the Cauchy integration method of Dorkin (2013) which cannot go beyond singularities nearest to the positive real axis. The algorithm enables efficient computation of quark propagators for use in Bethe-Salpeter equations for mesons and Faddeev equations for baryons within the rainbow-ladder truncation of QCD, where the Maris-Tandy model (Maris, 1999) provides the gluon interaction.
