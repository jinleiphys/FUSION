# Effective operators within the ab initio no-core shell model

> We implement an effective operator formalism for general one- and two-body operators, obtaining results consistent with the no-core shell model (NCSM) wave functions. The Argonne V8' nucleon-nucleon potential was used in order to obtain realistic wave functions for 4He, 6Li and 12C. In the NCSM formalism, we compute electromagnetic properties using the two-body cluster approximation for the effective operators and obtain results which are sensitive to the range of the bare operator. To illuminate the dependence on the range, we employ a Gaussian two-body operator of variable range, finding weak renormalization of long range operators (e.g., quadrupole) in a fixed model space. This is understood in terms of the two-body cluster approximation which accounts mainly for short-range correlations. Consequently, short range operators, such as the relative kinetic energy, will be well renormalized in the two-body cluster approximation.

## Key claim

The two-body cluster approximation for effective operators in the no-core shell model produces weak renormalization for long-range operators like the quadrupole, but strong renormalization for short-range operators like the relative kinetic energy; this range dependence means that the two-body cluster level is insufficient to correct the underestimated E2 transition strengths in small model spaces.

## Method

The authors implemented an effective operator formalism within the ab initio no-core shell model (NCSM) using the Argonne V8' nucleon-nucleon potential to generate wave functions for 4He, 6Li, and 12C. They computed effective operators in the two-body cluster approximation by applying a unitary transformation that decouples the model space from excluded configurations, solving the two-body problem in a large harmonic oscillator basis and then using the resulting transformation operator to renormalize general one- and two-body operators. For operators that cannot be treated identically to the Hamiltonian, they developed a convergence procedure that adds correlations from the excluded space a few shells at a time.

## Key numbers

- For 6Li in a 2-hbar-Omega model space (hbar-Omega = 13 MeV), the B(E2; 1+0 -> 3+0) transition strength was 2.647 e^2 fm^4 with the bare operator and 2.784 e^2 fm^4 with the effective operator, compared to the experimental value of 21.8(4.8) e^2 fm^4
- For 6Li in a 2-hbar-Omega model space, the B(E2; 2+0 -> 1+0) transition strength was 2.183 e^2 fm^4 with the bare operator and 2.269 e^2 fm^4 with the effective operator, compared to the experimental value of 4.41(2.27) e^2 fm^4
- For 6Li in a 10-hbar-Omega model space, the bare B(E2; 1+0 -> 3+0) was 10.221 e^2 fm^4, and the bare B(E2; 2+0 -> 1+0) was 4.502 e^2 fm^4
- For the deuteron in a 4-hbar-Omega model space, the bare quadrupole moment was 0.179 e fm^2, while the effective operator gave 0.270 e fm^2 (the AV8' potential value)
- For the Gaussian two-body operator with range 0.2 fm in 4He, the effective operator expectation value was flat across model spaces, while for range 1.0 fm, the bare and effective values showed similar model-space dependence

## Context

This work extends the NCSM effective interaction formalism to general one- and two-body operators, building on prior NCSM calculations that had only computed effective operators for the point-proton radius, relative kinetic energy, and NN pair density (Benchmark, 2002). The weak renormalization of the quadrupole operator at the two-body cluster level contrasts with earlier restricted-model calculations by Navratil (1997) that obtained significant effective quadrupole charges by including higher-body correlations through explicit truncation of large-basis NCSM results. The findings enable future investigations of electromagnetic processes at large momentum transfer, where two-body cluster renormalization is expected to play an important role, and motivate development of techniques to include higher-body correlations for long-range operators.
