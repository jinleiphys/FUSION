# Nucleon-Nucleon Spin Dependent Scattering Amplitudes to Describe Final State Interactions in Electromagnetic and Electroweak Nuclear Processes

> There are currently no models readily available that provide nucleon-nucleon spin dependent scattering amplitudes at high energies ($s \geq 6$ GeV$^2$). This work aims to provide a model for calculating these high-energy scattering amplitudes. The foundation of the model is Regge theory since it allows for a relativistic description and full spin dependence. A parameterization of the amplitudes is presented, and comparisons of the solution to the assembled data set are shown. In addition, an application of the model to describe final state interactions in deuteron electrodisintegration is presented. Overall the model works as intended, and provides an adequate description of the nucleon-nucleon interaction at these energies.

## Key claim

This work provides a model for calculating elastic, spin-dependent nucleon-nucleon scattering amplitudes at high energies (Mandelstam \(s \geq 6\) GeV\(^2\)), where no such model was previously available, by using Regge theory to parameterize the amplitudes and then extrapolating from fits to lower-energy data.

## Method

The author assumes isospin symmetry and describes the five independent helicity amplitudes for nucleon-nucleon scattering by expressing them in terms of Fermi invariants (scalar, pseudoscalar, vector, axial-vector, tensor). The Fermi invariants are then parameterized using Regge exchanges with definite parity, G-parity, and isospin, which are analytically continued from the \(t\)-channel nucleon-antinucleon system back to the \(s\)-channel. Fits are performed to low-energy nucleon-nucleon data, and the resulting amplitudes are extrapolated to higher energies and applied to describe final state interactions in deuteron electrodisintegration.

## Key numbers

- The model targets Mandelstam \(s \geq 6\) GeV\(^2\), where no spin-dependent amplitudes were previously available; the SAID analysis provides proton-neutron amplitudes up to \(s \approx 6\) GeV\(^2\) and proton-proton amplitudes up to \(s \approx 9.8\) GeV\(^2\).
- The Pomeron trajectory used in the fit has an intercept of 1.08 and a slope of 0.25 GeV\(^{-2}\).
- The model is applied to deuteron electrodisintegration (\(e+d\rightarrow e+p+n\)) at kinematics overlapping with the SAID analysis, and comparisons between the two final-state interaction models are presented.

## Context

This work extends the JVO model (Jeschonnek and Van Orden, 2008–2009), which requires the five spin-dependent nucleon-nucleon amplitudes as input to describe final state interactions in deuteron electrodisintegration but was limited by the kinematic range of the SAID amplitudes. The new Regge-based model enables calculations at higher energies, relevant for experiments at Jefferson Lab after the 12 GeV upgrade, and provides a framework for extrapolating to regimes where data are scarce, particularly for proton-neutron scattering.
