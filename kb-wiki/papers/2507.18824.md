# Deep Neural Network Driven Simulation Based Inference Method for Pole Position Estimation under Model Misspecification

> Simulation Based Inference (SBI) is shown to yield more accurate resonance parameter estimates than traditional chi-squared minimization in certain cases of model misspecification, demonstrated through a case study of pi-pi scattering and the rho(770) resonance. Models fit to some data sets using chi-squared minimization can predict inaccurate pole positions for the rho(770), while SBI provides more robust predictions across the same models and data. This result is significant both as a proof of concept that SBI can handle model misspecification, and because accurate modeling of pi-pi scattering is essential in the study of many contemporary physical systems (e.g., a1(1260), omega(782)).

## Key claim

Simulation Based Inference (SBI) yields more accurate resonance parameter estimates than traditional chi-squared minimization under model misspecification, specifically when no values of the model parameters exist for which the data are normally distributed about the function; the paper demonstrates this through a case study of pi-pi scattering and the rho(770) resonance, but notes that in ideal cases where data are normally distributed and the global minimum can be found, chi-squared minimization is guaranteed to give more probable fit parameters than SBI.

## Method

The authors generate pseudodata by randomly sampling model parameters, using the model to produce synthetic data points with normally distributed noise matching the experimental uncertainties, and retaining only pseudodata sets yielding physically reasonable pole positions. A deep neural network with four hidden layers (40, 30, 20, and 10 nodes) using relu activation functions is trained on the pseudodata to predict model parameters from the data values, then applied to actual experimental data to obtain parameter estimates; this process is repeated 100 times to compute average predictions and 95% confidence intervals. The method is applied to a K-matrix model of pi-pi scattering with two or three subtractions, fit to phase-shift data from Protopopescu (1973) and Estabrooks (1974).

## Key numbers

* Toy model: SBI predicted width Gamma = 136.9 MeV versus the true unmodified value of 119.0 MeV, while chi-squared minimization gave Gamma = 662.6 MeV (more than an order of magnitude farther from the true value)
* Toy model: SBI predicted mass M = 799.3 MeV versus true value 857.0 MeV; chi-squared gave M = 887.3 MeV
* For Estabrooks data with two subtractions: SBI pole position E* = 766.3(1.8) - i 79.0(0.7) MeV; chi-squared gave E* = 750.3(0.6) - i 69.8(0.5) MeV; PDG e+e- reference is 775.3(0.3) - i 73.9(0.5) MeV
* For Protopopescu data with two subtractions: SBI pole position E* = 763.7(1.0) - i 75.5(0.3) MeV; chi-squared gave E* = 755.6(0.8) - i 70.4(0.6) MeV
* For combined data with three subtractions: SBI pole position E* = 772.4(2.3) - i 75.4(0.4) MeV; chi-squared gave E* = 763.7(0.9) - i 76.9(0.6) MeV
* Classifier network assigned probability P(3) = 0.86657(00122) for three subtractions being the correct model

## Context

This work extends prior efforts to understand how SBI performs under model misspecification, following studies by Cannon (2022) showing challenges with SBI, Anau Montel (2024) using SBI to test for misspecification, and Huang (2023) and Wehenkel (2024) developing more robust SBI versions. The paper addresses an open question from Ananthanarayan (2000), Colangelo (2001), Garcia-Martin (2011), and Pelaez (2019) about whether inconsistencies in rho(770) pole positions extracted from different pi-pi scattering data sets could be mitigated by a method other than chi-squared minimization. The results enable application of SBI to other hadronic systems with data ambiguities, such as the Lambda(1405) resonance from K-p scattering and baryon resonance parameters from meson-electroproduction data.
