# Intrinsic Pair Operators and the Number-conserved Treatment of Pairing Correlations in Nuclei

> By introducing the intrinsic pair operators which commute with number fluctuation operator, a new formalism is given for the number-conserving description of the pairing correlations. The difficulty in the conventional RPA treatment for pairing rotation is removed.

## Key claim

By introducing intrinsic pair operators that commute with the number fluctuation operator, the authors propose a number-conserving description of pairing correlations that removes the difficulty of divergent RPA ground-state norms, while avoiding the need for explicit number projection integration.

## Method

The authors start from a general effective Hamiltonian and apply the Bogoliubov-Valatin transformation to obtain quasiparticle operators. They define a Tamm-Dancoff collective pair operator \(X_c^\dagger\) from the number fluctuation operator and construct non-collective pair operators \(X_n^\dagger\) orthogonal to it. Intrinsic pair operators \(\bar{X}_n^\dagger\) and \(\bar{B}_q\) are then defined by requiring they commute with the number fluctuation operator \(\Delta N\), and are constructed via an expansion in the small parameter \(\varepsilon = 1/L\) where \(L^2 = \langle \Delta N^2 \rangle\). The Hamiltonian is separated into rotational, intrinsic, and coupling parts, and the eigenvalue problem reduces to diagonalizing the intrinsic Hamiltonian \(H_{intr}\) in the intrinsic subspace, with the number projection factor \(\langle 0|P_N|0\rangle\) factoring out.

## Key numbers

- Ground state energies for Sn isotopes (in MeV) are tabulated for mass numbers 104, 108, 112, 116, 120, 124, 128; for example, for A=104: BCS = -2.313, RPA = -2.742, PW0 = -2.742, PW2 = -2.744, exact = -2.745
- For A=112: BCS = 0.930, RPA = 0.140, PW0 = 0.140, PW2 = 0.131, exact = 0.134
- For A=128: BCS = 33.837, RPA = 33.499, PW0 = 33.499, PW2 = 33.501, exact = 33.500
- The quantity \(L^2 = \langle \Delta N^2 \rangle = 2 \sum_\alpha u_\alpha^2 v_\alpha^2\) defines the small parameter \(\varepsilon = 1/L\)

## Context

This work extends the conventional RPA treatment of pairing correlations in nuclei, which suffers from a divergent ground-state norm due to the zero-energy pairing rotation mode. The authors cite the standard BCS/HFB theory (R-S), number projection methods (N-PRJ), and RPA approaches (RPA, M-RPA) as prior work. The method combines advantages of both RPA and projection methods: it avoids the zero-energy mode difficulty while eliminating the need for explicit phase-angle integration in number projection. The numerical test on Sn isotopes with constant pairing interaction uses single-particle energies and pairing force strength from K-S (2001) and shows that the lowest-order results (PW0) agree with conventional RPA, while second-order results (PW2) generally improve upon RPA.
