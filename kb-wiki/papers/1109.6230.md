# Ab-initio self-consistent Gorkov-Green's function calculations of semi-magic nuclei - I. Formalism at second order with a two-nucleon interaction

> An ab-initio calculation scheme for finite nuclei based on self-consistent Green's functions in the Gorkov formalism is developed. It aims at describing properties of doubly-magic and semi-magic nuclei employing state-of-the-art microscopic nuclear interactions and explicitly treating pairing correlations through the breaking of U(1) symmetry associated with particle number conservation. The present paper introduces the formalism, necessary to undertake applications at (self-consistent) second-order using two-nucleon interactions, in a detailed and self-contained fashion. First applications of such a scheme will be reported soon in a forthcoming publication. Future works will extend the present scheme to include three-nucleon interactions and implement more advanced truncation schemes.

## Key claim

The paper develops an ab-initio calculation scheme for finite nuclei based on self-consistent Green's functions in the Gorkov formalism, which explicitly treats pairing correlations through the breaking of U(1) symmetry associated with particle number conservation, but the present work is limited to introducing the formalism at (self-consistent) second-order using only two-nucleon interactions, with applications to be reported in a forthcoming publication.

## Method

The authors formulate the Gorkov-Green's function approach for finite nuclei by extending the normal self-consistent Green's function method to include anomalous propagators that account for pairing correlations via the breaking of particle-number conservation. The self-energy is expanded diagrammatically at second order in the two-nucleon interaction, and the resulting set of coupled Dyson equations for the normal and anomalous components of the one-body Green's function are presented in a self-consistent framework. The formalism is developed in a detailed and self-contained manner, including the explicit diagrammatic representation of the self-energy components at first and second order.

## Key numbers

- The paper is a formalism paper with no numerical results; applications are promised in a forthcoming publication.

## Context

This work extends the self-consistent Green's function method, previously applied to doubly-magic nuclei, to the description of semi-magic nuclei by incorporating pairing correlations through the Gorkov formalism. It builds upon the normal self-consistent Green's function approach and aims to enable ab-initio calculations of open-shell nuclei using state-of-the-art microscopic nuclear interactions. The scheme is designed to be extended in future work to include three-nucleon interactions and more advanced truncation schemes beyond second order.
