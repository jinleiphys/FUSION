# Equation-of-state-informed pulse profile modeling

> NICER has enabled mass-radius inferences for pulsars using pulse profile modeling (PPM), providing constraints on the equation of state (EOS) of cold, dense matter. To date, PPM and EOS inference have been carried out as two separate steps, with the former using EOS-agnostic priors. This approach has several drawbacks. Ideally, one would perform a fully hierarchical Bayesian inference where the pulse profile and EOS model parameters are jointly fit, but implementing such a framework is complex and computationally demanding. Here, we present an intermediate solution introducing an EOS-informed prior on mass-radius into the existing PPM pipeline using normalizing flows. By focusing on the parameter space consistent with certain EOSs, this approach both tightens constraints on neutron star parameters while reducing computational costs and requiring minimal additional implementation effort. We test this approach on two pulsars, PSR J0740+6620 and PSR J0437-4715, and with two EOS model families: a model based on the speed of sound inside the neutron star interior (CS) and a piecewise-polytropic (PP) model. Both EOS models implement constraints from chiral effective field theory calculations of dense matter. For both pulsar datasets, the inferred radius credible intervals are narrower than in the EOS-agnostic case, with CS favoring smaller radii and PP favoring larger radii. For PSR J0437-4715, the EOS-informed priors reveal a new, more extreme geometric mode that is statistically favored but physically questionable. Including the PPM posteriors in the subsequent EOS inference further tightens the mass-radius posteriors through the chiral effective field theory constraints. However, there is also a sensitivity to the high-density extensions, where the PP (CS) model produces a shift towards larger (smaller) radii and corresponding stiffening (softening) of the pressure-energy density relation.

## Key claim

The authors present an intermediate solution between EOS-agnostic pulse profile modeling and a fully hierarchical Bayesian joint fit, introducing an EOS-informed prior on mass-radius into the existing PPM pipeline using normalizing flows. This approach tightens constraints on neutron star parameters and reduces computational costs, but the results show sensitivity to the choice of high-density EOS extension, with the piecewise-polytropic (PP) and speed-of-sound (CS) models producing shifts toward larger and smaller radii respectively.

## Method

The authors used the X-PSI pulse profile modeling code with MultiNest nested sampling to analyze NICER and XMM-Newton data for two pulsars, PSR J0740+6620 and PSR J0437-4715. They replaced the standard EOS-agnostic flat priors on mass and radius with EOS-informed priors derived from two model families (CS and PP), both incorporating chiral effective field theory constraints up to 1.5 times nuclear saturation density, with the prior distribution in mass-radius space approximated using normalizing flows trained on 100,000 samples from each EOS family. The resulting PPM posteriors were then used in a subsequent EOS inference step using the NEoST framework.

## Key numbers

* For PSR J0437-4715, the EOS-agnostic inference required approximately 783,000 core hours to achieve convergence, while the EOS-informed approach reduced computational costs by restricting the parameter space.
* The chiral effective field theory calculations used to constrain the EOS prior were applied up to 1.5 times the nuclear saturation density, where the saturation density is n0 = 0.16 fm^-3.
* The normalizing flow was trained on 100,000 mass-radius samples from each EOS prior distribution, with 70% used for training, 15% for validation, and 15% for testing.
* For PSR J0437-4715, the EOS-informed priors revealed a new, more extreme geometric mode that is statistically favored but physically questionable.

## Context

This work extends the NICER pulse profile modeling program, which has enabled mass-radius inferences for five pulsars including PSR J0030+0451 (Riley 2019, Miller 2019), PSR J0437-4715 (Choudhury 2024), PSR J0740+6620 (Miller 2021, Riley 2021, Salmi 2022), PSR J1231-1411 (Salmi 2024), and PSR J0614-3329 (Mauviard 2025). Previous analyses by Raaijmakers (2018) and Riley (2018) used EOS-agnostic flat priors on mass and radius, with EOS inference performed as a separate step using the NEoST framework. The approach enables tighter constraints on neutron star parameters through more physically informed priors while leaving the two-step PPM and EOS inference framework intact, but the authors caution that Bayesian model comparison via posterior odds should not be performed with this method due to double counting of the EOS prior.
