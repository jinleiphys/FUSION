# Reconstructing jet anisotropies with cumulants

> In relativistic heavy-ion collisions, where quark-gluon plasma forms, hadron production is anisotropic at both low and high transverse momentum, driven by flow dynamics and spatial anisotropies. To better understand these mechanisms, we use multi-particle correlations to reconstruct jet anisotropies. We simulate data using \textsc{TennGen}\xspace as a hydro-like background and combine it with \textsc{Pythia-8}\xspace generated jets, clustering them with the anti-$k_{\mathrm{t}}$\xspace algorithm. Jet anisotropies are unfolded using a Bayesian technique, ensuring the robustness of the reconstructed signals. Our results demonstrate that multi-particle cumulant methods can accurately capture the differential jet azimuthal anisotropies, providing crucial insights into high-$p_{T}\xspace$ behavior and the dynamics within heavy-ion collisions.

# Reconstructing jet anisotropies with cumulants

## Key claim

Multi-particle cumulant methods can accurately reconstruct differential jet azimuthal anisotropies in simulated heavy-ion collisions, provided that Bayesian unfolding is applied to correct for bin migration effects when the jet anisotropy has a nonzero dependence on jet transverse momentum. The method is demonstrated to be robust even for physically unrealistic functional forms of the jet anisotropy, but the authors note that in real collisions, effects such as uncorrelated jets, resonance decays, and event-by-event flow fluctuations would need to be addressed for proper interpretation.

## Method

The authors simulate Au+Au collisions at 200 GeV by merging PYTHIA-8 generated jets with a TennGen hydro-like background tuned to 20-30% central events. Jets are reconstructed using the anti-kt algorithm with resolution parameter R=0.4 and corrected via jet-multiplicity background subtraction. The differential jet azimuthal anisotropies are calculated using 2- and 4-particle cumulant methods, with reference flow particles selected from pseudorapidity region 3.1 < |eta| < 5.1 and jets from |eta| < 1.1 - R. A Bayesian unfolding procedure is applied to the single-event correlations using a four-dimensional response matrix to correct for bin migration effects.

## Key numbers

- Reference multiplicity M fixed at 363, corresponding to average 20-30% central Au+Au collisions at sqrt(s_NN) = 200 GeV
- Jet resolution parameter R = 0.4 used for anti-kt clustering
- Minimum hard scattering cutoff for PYTHIA: hat{p}_T^{min} > 5 GeV
- Merged jet candidates required to satisfy p_T,jet^{corr} > 5 GeV and A_jet > 0.6 pi R^2
- Ghost particles with p_T^{ghost} = 1 MeV used for jet area estimation
- Nominal input values: alpha_2 = 4%, alpha_3 = 1%, alpha_4 = 0.5%, beta = 0.2
- 10 million TennGen+PYTHIA events generated for each sample
- Bayesian unfolding typically converges in 3-5 iterations
- Matching criterion for jet matching: Delta R < 0.75R (where R = 0.4)
- Jet momentum bin width of 5 GeV used for analysis

## Context

This work extends previous measurements of jet differential azimuthal anisotropies using event plane methods by ATLAS (2013, 2021) and ALICE (2015), which found nonzero dependence on jet p_T. The authors note that discrepancies between event plane and cumulant methods can reach up to 10% as shown by Andres (2019), Zigic (2022, 2021), and STAR (2021). The cumulant approach is particularly important because it cannot reconstruct an event plane in small collision systems, and the method enables future studies of jet anisotropies in p+A and smaller systems where jet quenching signatures remain an unresolved puzzle. The simulation package developed is publicly available as JetVnCumulant.
