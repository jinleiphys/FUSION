# Relativistic resistive magneto-hydrodynamics code for high-energy heavy-ion collisions

> We construct a relativistic resistive magneto-hydrodynamic (RRMHD) numerical simulation code for high-energy heavy-ion collisions. We split the system of differential equations into two parts, a non-stiff and a stiff part. For the non-stiff part, we evaluate the numerical flux using HLL approximated Riemann solver and execute the time integration by the second-order of Runge-Kutta algorithm. For the stiff part, which appears in Ampere's law, we integrate the equations using semi-analytic solutions of the electric field. We employ the generalized Lagrange multiplier method to ensure the divergence-free constraint for the magnetic field and Gauss's law. We confirm that our code reproduces well the results of standard RRMHD tests in the Cartesian coordinates. In the Milne coordinates, the code with high conductivity is validated against relativistic ideal MHD tests. We also verify the semi-analytic solutions of the accelerating longitudinal expansion of relativistic resistive magneto-hydrodynamics in high-energy heavy-ion collisions in a comparison with our numerical result. Our numerical code reproduces these solutions.

## Key claim

The authors construct a relativistic resistive magneto-hydrodynamics (RRMHD) numerical simulation code specifically for high-energy heavy-ion collisions, but the code's correctness is only confirmed for standard RRMHD test problems in Cartesian coordinates and for relativistic ideal MHD tests in Milne coordinates under high conductivity, with validation against semi-analytic solutions of accelerating longitudinal expansion.

## Method

The system of differential equations is split into non-stiff and stiff parts using operator splitting; the non-stiff part is solved with an HLL approximated Riemann solver and second-order Runge-Kutta time integration, while the stiff part in Ampere's law is integrated using semi-analytic solutions of the electric field. The generalized Lagrange multiplier (GLM) method is employed to enforce the divergence-free constraint for the magnetic field and Gauss's law. Primitive variables are recovered from conservative variables using the Newton-Raphson algorithm with an ideal gas equation of state.

## Key numbers

- Electrical conductivity values used in test problems are not explicitly stated in the provided text; the code is validated for "high conductivity" regimes where the stiff source term becomes problematic.
- The metric in Milne coordinates has $g_{33} = \tau^2$, with $\sqrt{-g} = \tau$, producing a non-zero source term proportional to $\frac{1}{2}\partial_0 g_{33} = \tau$ in the energy conservation equation.
- The semi-analytic solutions for the stiff part of the electric field use exponential decay factors: $E_{\parallel}^{'i} = E_{0\parallel}^{'i}\exp{-\sigma\gamma t}$ and $E_{\perp}^{'i} = E_{\perp}^{*i} + (E^{'i}_{0\perp} - E_{\perp}^{*i}) \exp(-\sigma t/\gamma)$.
- The GLM method introduces decay constants $\kappa$ for the auxiliary fields $\psi$ and $\phi$, with solutions $\psi' = \psi'_0\exp(-\kappa t)$ and $\phi' = \phi'_0\exp(-\kappa t)$.

## Context

This work extends previous relativistic ideal MHD simulations for heavy-ion collisions (Inghirami 2016, Inghirami 2019) by incorporating finite electrical conductivity, which is necessary to study charge-dependent observables. The code builds on astrophysical RRMHD techniques including the semi-analytic method of Komissarov (2007) and the GLM divergence cleaning method of Munz (2000) and Dedner (2002). The new test problem of accelerating longitudinal expansion with semi-analytic solutions from PhysRevD.102.014017 enables verification of the code in Milne coordinates relevant to heavy-ion collision dynamics.
