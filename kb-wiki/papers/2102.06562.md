# User guide for the hfbcs-qrpa(v1) code

> We briefly explain the structure of the Hartree-Fock-Bardeen-Cooper-Schrieffer (HFBCS) and Quasi-particle Random Phase Approximation (QRPA) code hfbcs-qrpa, highlighting only the differences with the code skyrme_rpa that has been previously published [G. Colò, L. Cao, N. Van Giai, and L. Capelli, Comp. Phys. Comm. 184, 142 (2013)]. The code deals with open shell spherical systems and non-spin flip, non-charge-exchange and natural parity $(-1)^l$ excitations.

## Key claim

The hfbcs-qrpa(v1) code extends the previously published skyrme_rpa code (Colò et al., 2013) to handle open-shell spherical nuclei by incorporating Hartree-Fock-Bardeen-Cooper-Schrieffer (HF-BCS) pairing and Quasiparticle Random Phase Approximation (QRPA), but only for non-spin-flip, non-charge-exchange, natural parity (-1)^l excitations, and with the caveat that tensor terms are implemented at the HF-BCS level but not in the residual QRPA interaction.

## Method

The code solves HF-BCS equations in coordinate space assuming spherical symmetry, using a standard Skyrme interaction with a zero-range, density-dependent pairing force. The QRPA equations are then solved on a two-quasiparticle basis built from the HF-BCS states, with the particle-hole effective interaction taken from the earlier skyrme_rpa code and the particle-particle interaction derived from the pairing force. The code provides automatic and manual pairing modes, with the automatic mode using a volume pairing force fitted to reproduce the experimental neutron pairing gap in 120Sn.

## Key numbers

- Experimental neutron pairing gap in 120Sn used for automatic mode calibration: Delta = 1.245 MeV
- Default pairing window in automatic mode: bound states above the neutron Fermi energy up to a maximum of 6 neutron single-particle levels
- Pairing force density parameter: rho_0 = 0.16 fm^-3
- Proton and neutron electric sizes used for charge radius calculation: <r_p^2> = 0.64 fm^2, <r_n^2> = -0.11 fm^2
- Constants used in the code: hbar*c = 197.327053 MeV fm, m*c^2 = 938.91869 MeV (average of proton and neutron masses)
- Example calculation for 120Sn (Z=50, A=120) with SLy5 interaction: total binding energy = -1019.3 MeV, neutron pairing gap = 1.4505 MeV, charge radius = 4.6418 fm, isospin mixing e^2 = 0.31506%
- Example calculation CPU time: 2017.41 seconds

## Context

This code extends the HF+RPA code skyrme_rpa (Colò et al., 2013) to include pairing correlations via HF-BCS and QRPA, enabling calculations for open-shell spherical nuclei that were previously inaccessible with the earlier code. The work enables systematic studies of collective excitations in open-shell nuclei using Skyrme functionals, with the specific Skyrme interactions available being SkM*, SkP, SLy4, SLy5, SkX, KDE0-J33 (KDE33), and SAMi. The code also provides isospin mixing evaluation following the approach of Roca-Maza (2020) and charge radius calculations including spin-orbit contributions as described by Horowitz (2012).
