# Drawing Feynman Diagrams with LaTeX and Metafont

> Feynmf is a LaTeX package for easy drawing of professional quality Feynman diagrams with Metafont (or Metapost). Feynmf lays out most diagrams satisfactorily from the structure of the graph without any need for manual intervention. Nevertheless all the power of Metafont (or Metapost) is available for the most complicated cases.

## Key claim

Feynmf is a LaTeX package that enables automatic layout of Feynman diagrams from a mathematical description of the graph structure, using a tension-based linear minimization algorithm, while also providing an immediate mode for manual control when the automatic layout is insufficient.

## Method

The package implements two modes: graph mode, where the user specifies vertices and arcs and the layout is determined by minimizing a weighted sum of squared arc lengths (a linear system solved by Metafont's built-in linear algebra), and immediate mode, where arbitrary Metafont paths can be drawn. The LaTeX macros write Metafont code to an external file, which is processed separately; label positions are calculated in Metafont and communicated back to LaTeX via the log file.

## Key numbers

- A simple scattering diagram can be specified completely in five lines of LaTeX code (the example shows a 40x30 unit diagram with two incoming and two outgoing fermion lines connected by a photon).
- The tension matrix elements \(t_{ij}\) default to 1 and can be tuned; examples show tensions of 4, 1, and 1/4 producing visibly different layouts.
- The linear system for a two-vertex scattering diagram with four fixed external vertices and unit tensions yields the solution \(v_5 = \frac{1}{8}(3v_1 + 3v_2 + v_3 + v_4)\) and \(v_6 = \frac{1}{8}(3v_3 + 3v_4 + v_1 + v_2)\).

## Context

This work extends the capabilities of existing LaTeX-based Feynman diagram packages such as Levine's feynman package (1990), which uses the standard LaTeX picture environment and produces less than perfect graphics, and Vermaseren's axodraw package (1994), which uses PostScript primitives and is inherently not portable. Feynmf is unique in combining complete portability (requiring only a standard TeX installation) with automatic algorithmic layout from the graph structure, freeing users from manual coordinate calculations. The package enables physicists to include high-quality Feynman diagrams in LaTeX documents without external graphics tools, and its extension mechanism allows users to define custom line styles for specialized applications.
