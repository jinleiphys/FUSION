# Anisotropic fluid dynamical simulations of heavy-ion collisions

> We present VAH, a (3+1)-dimensional simulation that evolves the far-from-equilibrium quark-gluon plasma produced in ultrarelativistic heavy-ion collisions with anisotropic fluid dynamics. We solve the hydrodynamic equations on an Eulerian grid using the Kurganov-Tadmor algorithm in combination with a new adaptive Runge-Kutta method. Our numerical scheme allows us to start the simulation soon after the nuclear collision, largely avoiding the need to integrate it with a separate pre-equilibrium dynamics module. We test the code's performance by simulating on the Eulerian grid conformal and non-conformal Bjorken flow as well as conformal Gubser flow, whose (0+1)-dimensional solutions are precisely known. Finally, we compare non-conformal anisotropic hydrodynamics to second-order viscous hydrodynamics in central Pb+Pb collisions and find that the former's longitudinal flow profile responds more consistently to the fluid's gradients along the spacetime rapidity direction.

## Key claim

The paper presents VAH, a (3+1)-dimensional anisotropic fluid dynamics simulation that can start as early as τ₀ = 0.05 fm/c after a heavy-ion collision, largely eliminating the need for a separate pre-equilibrium dynamics module, while maintaining numerical stability where second-order viscous hydrodynamics would fail due to negative pressures.

## Method

The authors implement anisotropic hydrodynamics using the (P_L, P_⊥)-matching scheme on an Eulerian grid, solving the conservation and relaxation equations with the Kurganov-Tadmor algorithm combined with a new adaptive Runge-Kutta method that adjusts the time step after each iteration. The code evolves 20 dynamical variables (T^{τμ}, P_L, P_⊥, W_{⊥z}, π_⊥) and four inferred variables (ε, u^x, u^y, u^η) in Milne coordinates, with the option to run either anisotropic or second-order viscous hydrodynamics within the same framework. The simulation is tested against conformal and non-conformal Bjorken flow and conformal Gubser flow, whose (0+1)-dimensional solutions are precisely known.

## Key numbers

- Starting time τ₀ = 0.05 fm/c for anisotropic hydrodynamics, allowing simulation soon after nuclear collision
- Global memory usage of 1.2 GB for a 129 × 129 × 63 grid
- Running time of approximately 530 seconds for a (3+1)-dimensional non-conformal anisotropic fluid dynamical simulation of central Pb+Pb on a 129 × 129 × 63 grid using an Intel Xeon E5-2680 v4 multi-core processor with OpenMP acceleration
- Shear viscosity to entropy density ratio η/s ≈ 0.1 at temperatures around the deconfinement temperature T_c = 154 MeV
- Hydrodynamization time τ_hydro ∼ 1 fm/c

## Context

This work extends the GPU-accelerated viscous hydrodynamic code GPU-VH (Bazow et al., 2016, 2017) by implementing anisotropic hydrodynamics, which treats pressure anisotropy and bulk viscous pressure non-perturbatively. The code addresses limitations of second-order viscous hydrodynamics codes such as MUSIC (Schenke et al., 2010) and iEBE-VISHNU (Shen et al., 2014), which require pre-equilibrium dynamics modules and can suffer from negative pressures at early times. The VAH code enables future integration into the JETSCAPE Bayesian framework (Everett et al., 2020) to provide an additional discrete model for the fluid dynamical stage, potentially constraining QCD transport properties more accurately.
