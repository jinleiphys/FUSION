# Coulomb wave function correction including momentum resolution for charged hadron pairs: Analysis of data for $ŽĆ^+ŽĆ^-$ pair production in $p\;+$ Ta reaction at 70 GeV/c

> We propose a new method for the Coulomb wave function correction including the momentum resolution for charged hadron pairs and apply it to the precise data on $ŽĆ^+ŽĆ^-$ correlations obtained in $p + \mbox{Ta}$ reaction at 70 GeV/c. It is found that interaction regions of this reaction (assuming Gaussian source function) are $5.6 \pm 3.0$ and $4.4 \pm 2.6$ fm for the thicknesses of the target $8$ and $1.4$ microns, respectively. The physical picture of the source size obtained in this way is discussed.

## Key claim

The extracted interaction region sizes depend on the target thickness and are significantly larger than the Ta nuclear radius, which the authors attribute to pions originating from long-lived resonance decays rather than directly from the interaction.

## Method
The authors first reconstruct the original analysis using the Gamow factor with momentum resolution, where Gaussian random numbers are generated for longitudinal and transverse momentum components using setup resolutions Žā_L = 1.3 MeV/c and Žā_T = 0.6 MeV/c (8 micron target) or Žā_T = 0.4 MeV/c (1.4 micron target). They then replace the Gamow factor with a full Coulomb wave function correction expressed in terms of confluent hypergeometric functions, assuming a Gaussian source distribution Žü(r) = (╬▓┬│/ŌłÜŽĆ┬│) exp(-╬▓┬▓r┬▓), and apply the same random number method for momentum resolution. The correlation function is fitted as R(q) = C╠ā_C(-╬Ę)b + (1-b) where b is a free parameter, using data with q > 3 MeV/c.

## Context
This work extends the authors' previous Coulomb wave function correction formulae (Mizoguchi, Biyajima 1995, 1996) to include finite momentum resolution, addressing a limitation pointed out by Afanas'ev (1990) regarding the earlier analysis. The method is applied to data from Afanas'ev (1990, 1996) on ŽĆŌü║ŽĆŌü╗ correlations in p + Ta reactions at 70 GeV/c, which had previously been corrected using only the Gamow factor. The results enable estimation of the interaction region from unlike-sign pion correlations, with the finding that the extracted source sizes are larger than the nuclear radius, suggesting that most pions originate from decays of long-lived resonances such as ╬Ę, KŌü░_S, and ╬ø rather than directly from the interaction region.
