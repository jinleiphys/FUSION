# Spin density matrices for nuclear density functionals with parity violations

> The spin density matrix (SDM) used in atomic and molecular physics is revisited for nuclear physics, in the context of the radial density functional theory. The vector part of the SDM defines a "hedgehog" situation, which exists only if nuclear states contain some amount of parity violation.

## Key claim

The vector part of the spin density matrix (SDM) in nuclear density functional theory defines a "hedgehog" situation, but this vector profile exists only if nuclear states contain some amount of parity violation; for an isolated nucleus with exact rotational invariance and parity conservation, the vector part of the SDM essentially vanishes.

## Method

The authors adapt the spin density matrix formalism from atomic and molecular physics to nuclear physics within the framework of radial density functional (RDF) theory. They expand local creation and annihilation operators in an ℓs shell-model basis, recouple orbital and spin angular momenta using Clebsch-Gordan coefficients and Wigner 3j- and 6j-symbols, and derive the scalar (S=0) and vector (S=1) components of the SDM. They then construct a toy model of ¹⁶O with a harmonic oscillator Hamiltonian, mixing 0p₁/₂ and 1s₁/₂ orbitals via a small parameter ε to introduce parity violation, and compute the resulting scalar and vector radial profiles.

## Key numbers

- The scalar profile D₀₀₀₀(r) for the toy model is proportional to φ₀₀²(r) + 3 φ₀₁²(r), where φ₀₀(r) = 2 π⁻¹/⁴ e^{-r²/2} and φ₀₁(r) = 2^(3/2) 3^(-1/2) π⁻¹/⁴ r e^{-r²/2}.
- The vector (hedgehog) profile D_hh(r) for the toy model is proportional to φ₀₁(r) φ₁₀(r), where φ₁₀(r) = 6^(1/2) (1 - 2/3 r²) e^{-r²/2}.
- The mixing parameter ε is taken to first order, with the wave function |φ⟩ = |0⟩ + ε Σ_τ |τ⟩.

## Context

This work extends the spin density matrix concept from atomic and molecular physics (Gunnarsson and Lundqvist 1976; von Barth and Hedin 1972) to nuclear density functional theory, building on the proof by Giraud (2009) that the nuclear density functional is a scalar under rotations. The paper shows that a standard radial density functional using only the scalar (S=0) profile cannot accommodate spin-dependent forces or parity violations, and that the vector (S=1) hedgehog profile serves as a signature of parity violation. This enables future studies of spin density functionals for nuclei under external perturbations that break rotational symmetry and parity, such as those considered by Kohn and Sham (1965) for Kohn-Sham potentials.
