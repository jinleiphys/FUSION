# Bounce Configuration from Gradient Flow

> Based on the gradient flow, we propose a new method to determine the bounce configuration for false vacuum decay. Our method is applicable to a large class of models with multiple fields. Since the bounce is a saddle point of an action, a naive gradient flow method which minimizes the action does not work. We point out that a simple modification of the flow equation can make the bounce its stable fixed point while the false vacuum configuration an unstable one. Consequently, the bounce configuration can be obtained simply by following the flow without a careful choice of an initial configuration. With numerical analysis, we confirm the validity of our claim, checking that the flow equation we propose indeed has solutions that flow into the bounce configuration.

## Key claim

The authors propose a modified gradient flow equation that makes the bounce configuration a stable fixed point (and the false vacuum an unstable one), enabling determination of the bounce without careful choice of initial configuration, provided the dimensionless parameter β > 1 and the auxiliary function g_A is chosen appropriately.

## Method

The method modifies the naive gradient flow equation ∂_s Φ_A = F_A by adding a term −β⟨F|g⟩ g_A, where F_A = −δS/δΦ_A and g_A is a normalized function satisfying the same boundary conditions as the bounce fluctuations. For β > 1, the flow equation makes the bounce a stable fixed point while destabilizing the false vacuum and other classical solutions. The authors discretize the radial coordinate using a tanh mapping and solve the resulting ordinary differential equations in flow time σ = s^(1/2)/R, taking g_A ∝ r ∂_r Φ_A.

## Key numbers

- For the single-scalar potential with k₁ = 0.47 (thin-walled bounce), the bounce action obtained is S = 1086.6, compared to 1092.8 from CosmoTransitions.
- For the single-scalar potential with k₁ = 0.2 (thick-walled bounce), the bounce action obtained is S = 6.6360, compared to 6.6490 from CosmoTransitions.
- For the double-scalar potential with k₂ = 2, the bounce action obtained is S = 1763.7, compared to 1767.7 from CosmoTransitions.
- For the double-scalar potential with k₂ = 80, the bounce action obtained is S = 4.4585, compared to 4.4661 from CosmoTransitions.

## Context

This work extends prior efforts to find bounce configurations for false vacuum decay, which is important for calculating decay rates in particle physics and cosmology (e.g., the electroweak vacuum metastability problem). Previous methods include the back-stepping gradient flow approach of Cline (1998, 1999) and the improved action method of Kusenko (1995), as well as the CosmoTransitions code of Wainwright (2011). The new method enables straightforward numerical determination of bounces in multi-field models without special initial choices, and the authors note it is applicable to generic saddle-point problems beyond vacuum decay.
