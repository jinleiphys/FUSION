# Solving Schrodinger equations using physically constrained neural network

> Deep neural network (DNN) and auto differentiation have been widely used in computational physics to solve variational problems. When DNN is used to represent the wave function to solve quantum many-body problems using variational optimization, various physical constraints have to be injected into the neural network by construction, to increase the data and learning efficiency. We build the unitary constraint to the variational wave function using a monotonic neural network to represent the Cumulative Distribution Function (CDF) $F(x) = \int_{-\infty}^{x} ψ^*ψdx'$. Using this constrained neural network to represent the variational wave function, we solve Schrodinger equations using auto-differentiation and stochastic gradient descent (SGD), by minimizing the violation of the trial wave function $ψ(x)$ to the Schrodinger equation. For several classical problems in quantum mechanics, we obtain their ground state wave function and energy with very low errors. The method developed in the present paper may pave a new way in solving nuclear many body problems in the future.

## Key claim

The authors claim that representing the cumulative distribution function (CDF) with a monotonic neural network, rather than representing the wave function directly, automatically enforces normalization and eliminates numerical integration, yielding high-accuracy ground state energies and wave functions for several quantum mechanical potentials. The hedge is that the method performs poorly on potentials with discontinuities, such as the infinite potential well, where the relative error in ground state energy is about 1%.

## Method

The authors construct a feedforward neural network with non-negative weights and sigmoid activation functions to represent the monotonic CDF F(x) = ∫ ψ*ψ dx'. The wave function ψ(x) is obtained as the square root of the first derivative dF/dx via automatic differentiation. The training objective minimizes the violation of the trial wave function and a trainable energy parameter E₀ to the time-independent Schrödinger equation, plus boundary terms for the CDF and an exponential term e^(0.001 E₀) to drive E₀ toward the ground state energy, using stochastic gradient descent with the Adam optimizer.

## Key numbers

- Harmonic oscillator ground state energy after 1500 iterations: E₀ = 0.50038 ℏω, relative error within 0.06%; after 10000 iterations, error below 0.002%.
- Harmonic oscillator partial-wave fidelity K = 0.9999967 with 4 hidden layers and 16 neurons per layer.
- Woods-Saxon potential ground state energy from DNN: E₀ = -0.97382, relative error 0.002% compared to exact result -0.97385; fidelity K = 0.999964.
- Infinite potential well ground state energy from DNN: E₀ = 0.07787, relative error about 1% compared to accurate result 0.07710; fidelity K = 0.9977475.
- Error range of ground state wave function for harmonic oscillator and Woods-Saxon potential: within 0.0002; for infinite potential well: error range expanded to 0.02.

## Context

This work extends the variational artificial neural network (VANN) approach of Carleo and Troyer (2017) and the physics-informed neural network framework of Raissi et al. (2017) by replacing direct wave function representation with a monotonic CDF representation, eliminating numerical integration. The method differs from supervised learning approaches like Keeble and Rios (2020) and Saito (2018) that require knowledge of the exact solution form for pre-training. The authors identify that the current method struggles with discontinuous potentials and note that future work must incorporate anti-symmetric constraints for fermionic systems, with the next step being application to the deuteron ground state.
