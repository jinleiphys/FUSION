# Stable Bose-Einstein correlations

> The shape of Bose-Einstein (or HBT) correlation functions is determined for the case when particles are emitted from a stable source, obtained after convolutions of large number of elementary random processes. The two-particle correlation function is shown to have a {\it stretched exponential} shape, characterized by the Lévy index of stability $ 0 < α\le 2$ and the scale parameter $R$. The normal, Gaussian shape corresponds to a particular case, when $α= 2$ is selected. The asymmetry parameter of the stable source, $β$ is shown to be proportional to the angle, measured by the normalized three-particle cumulant correlations.

# Stable Bose-Einstein correlations

## Key claim

The two-particle Bose-Einstein (HBT) correlation function takes a stretched exponential form, characterized by the Lévy index of stability α (0 < α ≤ 2) and scale parameter R, when particle emission coordinates result from convolutions of a large number of elementary random processes; the Gaussian shape is recovered only for the special case α = 2.

## Method

The authors derive the two-particle Bose-Einstein correlation function from a source distribution that is a stable (Lévy) distribution, using the fact that the Fourier transform of a convolution is a product of Fourier transforms. They show that the characteristic function of symmetric stable distributions yields a correlation function of the form C(q) = 1 + exp(-|qR|^α). They then generalize this result to multidimensional expanding systems, including non-relativistic and relativistic (Buda-Lund) parameterizations, and derive the relationship between the three-particle cumulant correlation and the asymmetry parameter β of the stable source.

## Key numbers

- The Lévy index of stability α satisfies 0 < α ≤ 2, with α = 2 corresponding to the Gaussian case.
- The two-particle correlation function for symmetric stable distributions is C(q) = 1 + exp(-|qR|^α).
- The normalized three-particle cumulant correlation w(1,2,3) equals cos φ, where φ = (β/2) R^α tan(απ/2) [Σ |q_ij|^α sign(q_ij)] for α ≠ 1.
- The asymmetry parameter β is limited to the range -1 ≤ β ≤ 1.
- The scale parameter R is related to the Lévy scale parameter γ by R = 2^(1/α) γ.
- For large |x|, Lévy stable source densities decay as |x|^(-1-α).

## Context

This work extends the standard Gaussian treatment of Bose-Einstein correlations by introducing Lévy stable distributions as the natural limit distributions arising from multiple convolutions of elementary random processes, motivated by Bialas's work on fluctuating Gaussian sources and numerical studies by Utyuzh, Wilk and Włodarczyk on fractal source structures. The results enable experimental extraction of the Lévy index α and asymmetry parameter β from two- and three-particle correlation data, providing a more general framework for analyzing HBT correlations in high-energy nuclear and particle physics.
