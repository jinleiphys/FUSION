# Threshold Production of the Hypertriton

> The cross section for threshold production of the hypertriton in the reaction pd->^3_ΛH K^+ is calculated in a two-step model and compared to the break-up process pd -> d\LambdaK^+. The latter process is shown to be dominant already at 2 MeV above threshold. The amplitude squared at threshold for the pd -> ^3_ΛH K^+ reaction is |f|^2 = 1.0 nb/sr.

## Key claim

The cross section for threshold production of the hypertriton in the reaction pd → ³_ΛH K⁺ is calculated using a two-step model and found to be small; the break-up process pd → dΛK⁺ is shown to dominate already at 2 MeV above threshold, with the amplitude squared at threshold for the bound-state reaction being |f|² = 1.0 nb/sr.

## Method

The calculation employs a two-step model where the exchanged momentum is shared between the two nucleons of the deuteron, with the intermediate deuteron and neutron placed on their mass shells to treat the subprocesses as nearly real. The cross sections for the bound final state (hypertriton) and the break-up final state (deuteron + Λ) are computed in the forward direction using empirical parametrizations of the pp → π⁺d and π⁺n → ΛK⁺ vertex cross sections, with S-wave approximations for the deuteron and hypertriton wave functions. The transition matrices are evaluated in configuration space using a modified Hulthén wave function for the hypertriton and the Paris wave function for the deuteron, with scattering wave functions for the Λd system derived from the bound-state wave function via the relation of Fäldt and Wilkin.

## Key numbers

- Amplitude squared at threshold for pd → ³_ΛH K⁺: |f|² = 1.0 nb/sr (abstract) and 1.03 nb/sr (results section).
- Hypertriton binding energy: E_B = 0.13 MeV.
- Threshold laboratory kinetic energy for the proton: T_p^lab = 1126.517 MeV.
- Hypertriton wave function parameters: α_τ = 0.068 fm⁻¹, β_τ = 1.616 fm⁻¹; spin-3/2 scattering state parameter α'_τ = −0.1334 fm⁻¹.
- At threshold, the pp → π⁺d cross section (forward) is 53.3 μb/sr, and the π⁺n → ΛK⁺ cross section is 50.3 μb/sr.
- The ratio of the cross sections for the unbound and bound Λd spin-1/2 channels follows σ_{1/2}(dΛK)/σ(³_ΛH K) ≈ (1/4)(Q/E_B)^{3/2}(1+√(1+Q/E_B))⁻².
- The cross section for the unbound Λd spin-3/2 channel is larger than the spin-1/2 channel by a factor of 0.7 near threshold, increasing to 1.9 at T_p^lab = 1137 MeV.

## Context

This work extends the two-step model previously applied by Laget to the reaction pd → ³He η, adapting it to hypertriton production near threshold. It contrasts with the calculations of Komarov et al. and Kondratyuk et al., which focused on energies far from threshold and angular distributions, by concentrating on the near-threshold behavior and the competition with the break-up channel. The results enable future improvements using more accurate wave functions (e.g., Miyagawa et al., 1995), higher energies, angular dependence, and better experimental data for the π⁺n → ΛK⁺ vertex.
