# A program for 3D nuclear static and time-dependent density-functional theory with full Skyrme energy density functional: HIT3D

> This work presents a computer program that performs symmetry-unrestricted 3D nuclear time-dependent density function theory (DFT) calculations. The program features the augmented Lagrangian constraint in the static calculation. This allows for the calculation of the potential energy surface. In addition, the code includes the full energy density functionals derived from the Skyrme interaction, meaning that the time-even and time-odd tensor parts are included for the time-dependent calculations. The results of the hit3d code are carefully compared with the Sky3D and Ev8 programs. The testing cases include unconstrained DFT calculations for doubly magic nuclei, the constrained DFT + BCS calculations for medium-heavy nucleus 110Mo, and the dynamic applications for harmonic vibration and nuclear reactions.

## Key claim

The HIT3D code performs symmetry-unrestricted 3D nuclear static and time-dependent density-functional theory calculations that include the full Skyrme energy density functional, including time-even and time-odd tensor parts, but the authors note that terms with coefficients \(b_{18,19,20,21}\) are always ignored because their inclusion causes instabilities in static calculations.

## Method

The code represents single-particle wave functions on a three-dimensional Cartesian coordinate grid and solves the Hartree-Fock (Kohn-Sham) equations using the imaginary time step method for static problems, with the augmented Lagrangian method applied for deformation-constrained calculations. For time-dependent calculations, the time propagator is approximated by Taylor expansion, and the code includes BCS pairing with either delta or mixed pairing forces. The code is written in C++ and can be parallelized using C++ Standard Parallelism or C++-based CUDA.

* The pairing force uses a saturation density of \(\rho_0 = 0.16\) fm\(^{-3}\) for the mixed force option.
* The kinetic energy includes a one-body center-of-mass correction factor of \((1 - 1/A)\), where \(A\) is the mass number.

## Context

This work extends the capabilities of existing 3D nuclear DFT codes such as Sky3D (Maruhn et al., 2014; Schuetrumpf et al., 2018) and Ev8 (Ryssens et al., 2015) by adding the augmented Lagrangian constraint method for static calculations and including the full Skyrme energy density functional with tensor and time-odd contributions for time-dependent calculations. The code enables consistent static and dynamic simulations without spatial symmetry restrictions, allowing future studies of odd-\(A\) nuclei at the Hartree-Fock level and calculations of potential energy surfaces for fission or fusion reactions.
