# Event-plane correlators

> Correlators between event planes of different harmonics in relativistic heavy-ion collisions have the potential to provide crucial information on the initial state of the matter formed in these collisions. We present a new procedure for analyzing such correlators, which is less demanding in terms of detector acceptance than the one used recently by the ATLAS collaboration to measure various two-plane and three-plane correlators in Pb-Pb collisions at LHC. It can also be used unambiguously for quantitative comparison between theory and data. We use this procedure to carry out realistic simulations within the transport model AMPT. Our theoretical results are in excellent agreement with the ATLAS data, in contrast with previous hydrodynamic calculations which only achieved qualitative agreement. We present predictions for new correlators, in particular four-plane correlators, which can easily be analyzed with our new method.

# Event-plane correlators

## Key claim

Event-plane correlators between different Fourier harmonics in relativistic heavy-ion collisions can be reliably analyzed using only two symmetric pseudorapidity windows (rather than the three or more windows required by the ATLAS method), provided one uses the scalar-product method instead of the event-plane method; the scalar-product method removes the ambiguity that previously prevented quantitative comparison between theory and data.

## Method

The authors define event-plane correlators using products of flow vectors \(Q_n\) (complex sums over particles in a given pseudorapidity window) and take ratios that cancel dependence on the magnitudes of anisotropic flow coefficients \(v_n\). All positive harmonics are placed in one pseudorapidity subevent \(A\) and all negative harmonics in the other subevent \(B\), with a pseudorapidity gap between them. The analysis is applied to events generated by the AMPT transport model (with string melting and parton coalescence) for Pb-Pb collisions at \(\sqrt{s_{NN}} = 2.76\) TeV, using two symmetric pseudorapidity intervals \(-4.8 < \eta < -0.4\) and \(0.4 < \eta < 4.8\).

## Key numbers

- The difference between scalar-product (SP) and event-plane (EP) results for two-plane correlators \(\cos(4(\Phi_2 - \Phi_4))\) and \(\cos(6(\Phi_2 - \Phi_3))\) is in the range 10–15%.
- For four-plane correlators, the difference between SP and EP methods is roughly 50%.
- The four-plane correlator \(c\{2,2,2,2,-3,-5\}\) is compatible with zero in AMPT calculations.
- One four-plane correlator (the last in Fig. 4p(f)) is predicted to exceed unity when analyzed using the scalar-product method.
- Self-correlation contributions to the three-particle correlator \(c\{2,2,-4\}\) are of relative order \(1/N_A\), where \(N_A\) is the number of particles in subevent \(A\).
- Self-correlation contributions to the three-plane correlator \(c\{2,3,-5\}\) are of relative order \(1/N_A\).

## Context

This work extends the event-plane correlator analysis introduced by the ATLAS collaboration (Jia 2012) for Pb-Pb collisions at the LHC, which required separate pseudorapidity windows for each harmonic and suffered from an ambiguity in the resolution correction when flow fluctuations are present (Luzum 2012). Previous hydrodynamic calculations (Qiu 2012, Teaney 2012) achieved only qualitative agreement with ATLAS data using the event-plane method. The scalar-product method presented here enables unambiguous quantitative comparison between theory and experiment, and the AMPT model results show excellent agreement with ATLAS data for the first time. The approach enables measurement of new four-plane correlators and can be applied to test initial-state models such as Monte-Carlo Glauber and color-glass-condensate (Gelis 2010).
