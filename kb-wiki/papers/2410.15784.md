# Shell quenching in nuclear charge radii based on Monte Carlo dropout Bayesian neural network

> Charge radii can be generally used to encode information about various fine structures of finite nuclei. In this work, a constructed Bayesian neural network based on the Monte Carlo dropout approach is proposed to accurately describe the charge radii of nuclei with proton number $Z\geq20$ and mass number $A\geq40$. More motivated underlying mechanisms are incorporated into this combined model in addition to the basic building blocks with the specific number of protons and neutrons, which naturally contain the pairing effect, the isospin effect, the shell closure effect associated with the Casten factor $P$, the valence neutrons, the valence protons, the quadrupole deformation $β_{20}$, the high order hexadecapole deformation $β_{40}$, and the local shape staggering effect of $^{181,183,185}$Hg. To avoid the distorted cases of the traditional Casten factor at the fully filled shells, the modified Casten factor $P^{*}$ is introduced into the input structure parameter sets. The standard root-mean-square deviation is reduced to $0.0084$ fm for the training data set and $0.0124$ fm for the validation data set with the modified Casten factor $P^{*}$. Meanwhile, the shell closure effect of nuclear charge radii can be reproduced remarkably well. We have successfully demonstrated the ability of this constructed model to significantly increase the accuracy in predicting the nuclear charge radii.

## Key claim

The authors claim that a Monte Carlo dropout Bayesian neural network (MC-dropout BNN) incorporating a modified Casten factor \(P^{*}\) along with quadrupole and hexadecapole deformation parameters can describe nuclear charge radii with a root-mean-square deviation of 0.0084 fm for the training set and 0.0124 fm for the validation set, but only for nuclei with proton number \(Z \geq 20\) and mass number \(A \geq 40\); the model still shows larger deviations in regions with inverse odd-even staggering (e.g., Eu and At isotopes) and along manganese isotopes.

## Method

The authors constructed a Monte Carlo dropout Bayesian neural network with three hidden layers (28, 300, and 71 neurons) and a dropout probability of 0.5. The input features included proton number \(Z\), mass number \(A\), isospin asymmetry \(I^{2}\), pairing effect \(\delta\), valence neutrons \(\nu_{n}\), valence protons \(\nu_{p}\), the modified Casten factor \(P^{*}\), quadrupole deformation \(\beta_{20}\), hexadecapole deformation \(\beta_{40}\), and a local shape-staggering indicator for \(^{181,183,185}\)Hg. The model was trained on 815 experimental charge radii from Angeli (2013) and validated on 118 more recent data points from Li (2021) and PhysRevLett.126.032502.

## Key numbers

*   Training set RMSD: 0.0084 fm for the \(P^{*}\) model.
*   Validation set RMSD: 0.0124 fm for the \(P^{*}\) model.
*   Total set RMSD: 0.0090 fm for the \(P^{*}\) model.
*   For comparison, the \(P\) model (using the traditional Casten factor) gave training RMSD of 0.0124 fm, validation RMSD of 0.0147 fm, and total RMSD of 0.0128 fm.
*   The hyperparameter \(\lambda_{\mathrm{decay}}\) for \(L_{2}\) regularization was set to \(3 \times 10^{-9}\).
*   The np formula parameters used were \(r_{0}=0.966\) fm, \(b=0.182\), and \(c=1.652\) from Bayram (2013).

## Context

This work extends prior Bayesian neural network approaches to charge radii by Utama (2016) (RMSD ~0.02 fm) and Dong (2023) (RMSD 0.014 fm) by incorporating quadrupole and hexadecapole deformation parameters and a modified Casten factor \(P^{*}\) to avoid distortions at fully filled shells. The model reproduces shell quenching at magic numbers (e.g., \(N=28, 50, 82, 126\)) where the traditional Casten factor \(P\) model fails, particularly for nickel and lead isotopes. The results enable more reliable extrapolations of charge radii for neutron-rich and neutron-deficient nuclei, though the authors note that inverse odd-even staggering in Eu and At isotopes remains poorly captured.
