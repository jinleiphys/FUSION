# Thick-Restart Block Lanczos Method for Large-Scale Shell-Model Calculations

> We propose a thick-restart block Lanczos method, which is an extension of the thick-restart Lanczos method with the block algorithm, as an eigensolver of the large-scale shell-model calculations. This method has two advantages over the conventional Lanczos method: the precise computations of the near-degenerate eigenvalues, and the efficient computations for obtaining a large number of eigenvalues. These features are quite advantageous to compute highly excited states where the eigenvalue density is rather high. A shell-model code, named KSHELL, equipped with this method was developed for massively parallel computations, and it enables us to reveal nuclear statistical properties which are intensively investigated by recent experimental facilities. We describe the algorithm and performance of the KSHELL code and demonstrate that the present method outperforms the conventional Lanczos method.

## Key claim

The thick-restart block Lanczos method, implemented in the KSHELL code, outperforms the conventional Lanczos method for large-scale shell-model calculations, particularly when a large number of eigenstates (O(10²)–O(10³)) are required for studying highly excited states and nuclear statistical properties, though the performance gain depends on the balance between block size, restart parameters, and the trade-off between reduced iteration count and increased cost of matrix-block products.

## Method

The authors implemented a thick-restart block Lanczos algorithm that combines the block Lanczos method (using a block of p vectors instead of a single vector) with the thick-restart procedure to limit the number of Lanczos vectors and reduce reorthogonalization costs. The KSHELL code generates Hamiltonian matrix elements on the fly to avoid storing them, and the block method reduces the frequency of this generation by performing matrix-block products at once. The algorithm was tested on the ⁴⁸Cr nucleus in the pf-shell model space with the GXPF1A interaction, comparing convergence properties and performance across four variants: simple Lanczos, thick-restart Lanczos, block Lanczos, and thick-restart block Lanczos.

## Key numbers

- The M-scheme dimension for ⁴⁸Cr with the pf-shell model space and GXPF1A interaction is 1,963,461.
- The lowest eigenvalue of ⁴⁸Cr converges at the 44th iteration using the simple Lanczos method, while the 32nd eigenvalue converges at the 466th iteration.
- The convergence criterion used is that the change of Ritz values as a function of iteration number is smaller than 10⁻⁶ MeV.
- The maximum M-scheme dimension achieved by the KSHELL code reached 10¹¹ for ³²Mg with sd- and pf-shells model space allowing up to 8ħω configurations.
- For the thick-restart Lanczos method with l_s=40 and l_m=100, the 32nd eigenvalue shows reasonably fast convergence requiring modest storage of 100 Lanczos vectors.
- The block Lanczos method with block sizes p=2, 4, and 8 reaches convergence in a number of iterations roughly equal to or slightly larger than 1/p of the simple Lanczos method.

## Context

This work extends the thick-restart Lanczos method (Wu and Simon, 2000) and the block Lanczos method (Cullum and Willoughby, 1985; Jia, 1995) to create a combined approach for large-scale shell-model calculations. The KSHELL code enables calculations of nuclear statistical properties such as γ-ray strength functions and level densities (Oslo method), which require O(10²)–O(10³) eigenstates, going beyond traditional shell-model studies that focused on a small number of low-lying states. The method is compared with the Sakurai-Sugiura method (Mizusaki, 2008) and the Locally Optimal Block Preconditioned Conjugate Gradient method used in no-core shell-model calculations.
