# A covariant constituent-quark formalism for mesons

> Using the framework of the Covariant Spectator Theory (CST) [1] we are developing a covariant model formulated in Minkowski space to study mesonic structure and spectra. Treating mesons as effective $q\bar{q}$ states, we focused in [2] on the nonrelativistic bound-state problem in momentum space with a linear confining potential. Although integrable, this kernel has singularities which are difficult to handle numerically. In [2] we reformulate it into a form in which all singularities are explicitely removed. The resulting equations are then easier to solve and yield accurate and stable solutions. In the present work, the same method is applied to the relativistic case, improving upon the results of the one-channel spectator equation (1CSE) given in [3].

# A covariant constituent-quark formalism for mesons

## Key claim
The singularity-removal method previously developed for the nonrelativistic Schrödinger equation with a linear confining potential is successfully extended to the relativistic one-channel spectator equation (1CSE), yielding faster convergence and enabling calculations for partial waves beyond S-wave that were not accessible with the unsubtracted version.

## Method
The authors solve the covariant spectator theory (CST) bound-state equation for mesons as effective \(q\bar{q}\) states, using a linear confining kernel that is regularized by a subtraction technique that removes singularities analytically. The 1CSE is transformed into an eigenvalue problem for bound-state masses, with wave functions expanded in a basis of cubic B-splines, and solved in momentum space using a helicity representation for arbitrary partial waves \(\ell\).

## Key numbers
- Mass ratio \(m_1/m_2 = 5\) (heavy-light meson system) and string tension \(\sigma = 0.2\) GeV\(^2\) used for numerical tests.
- Computing time reduced to 1/6 of the original to obtain the same accuracy when using the singularity-free version compared to the unsubtracted version.
- For \(\ell > 2\), no converged results were obtained with the unsubtracted version of the 1CSE, even after increasing the number of splines; the subtraction technique resolves this issue.
- Results computed with 64 B-splines for the 7 lowest positive energies \(E_D \equiv \mu - m_1\) in S-wave (\(\ell=0\)) with pure scalar interaction (\(y=0\)).

## Context
This work extends the singularity-removal method developed by the same authors for the nonrelativistic limit (Leitão et al., 2014) to the relativistic 1CSE, improving upon earlier results by Uzzo (2009) which only treated the S-wave case. The formalism builds on the Covariant Spectator Theory framework of Gross, Milana, and Savkli (1990s) and is part of a broader program to develop a unified covariant model for both light and heavy mesons. Future work will generalize the method to the four-channel spectator equation (4CSE), which is necessary for realistic light meson systems.
