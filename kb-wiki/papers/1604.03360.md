# Relativistic hydrodynamics on graphics processing units

> Hydrodynamics calculations have been successfully used in studies of the bulk properties of the Quark-Gluon Plasma, particularly of elliptic flow and shear viscosity. However, there are areas (for instance event-by-event simulations for flow fluctuations and higher-order flow harmonics studies) where further advancement is hampered by lack of efficient and precise 3+1D~program. This problem can be solved by using Graphics Processing Unit (GPU) computing, which offers unprecedented increase of the computing power compared to standard CPU simulations. In this work, we present an implementation of 3+1D ideal hydrodynamics simulations on the Graphics Processing Unit using Nvidia CUDA framework. MUSTA-FORCE (MUlti STAge, First ORder CEntral, with a~slope limiter and MUSCL reconstruction) and WENO (Weighted Essentially Non-Oscillating) schemes are employed in the simulations, delivering second (MUSTA-FORCE), fifth and seventh (WENO) order of accuracy. Third order Runge-Kutta scheme was used for integration in the time domain. Our implementation improves the performance by about 2~orders of magnitude compared to a~single threaded program. The algorithm tests of 1+1D~shock tube and 3+1D~simulations with ellipsoidal and Hubble-like expansion are presented.

## Key claim

The paper presents a GPU-based implementation of 3+1D ideal relativistic hydrodynamics that achieves approximately two orders of magnitude speedup over a single-threaded CPU program, but only the MUSTA-FORCE algorithm was benchmarked for performance; the WENO schemes were found to be more accurate and less computationally expensive, yet no direct GPU-vs-CPU speed comparison is reported for WENO.

## Method

The authors discretize the relativistic hydrodynamic equations on a Cartesian grid using either a MUSTA-FORCE scheme (second-order accurate, with MUSCL reconstruction and slope limiters) or finite-difference WENO schemes (fifth- and seventh-order accurate). Time integration is performed with a third-order Runge-Kutta method. The code is implemented in NVIDIA CUDA and tested on 1+1D shock-tube problems and 3+1D simulations with ellipsoidal and Hubble-like expansion.

## Key numbers

* GPU implementation of the MUSTA-FORCE algorithm ran approximately 200 times faster than an equivalent single-threaded CPU implementation on contemporary hardware.
* MUSTA-FORCE scheme delivers second order of accuracy; WENO schemes deliver fifth (WENO5) and seventh (WENO7) order of accuracy.
* Third order Runge-Kutta scheme was used for time integration.

## Context

This work extends prior GPU-accelerated hydrodynamics efforts by implementing both MUSTA-FORCE and high-order WENO schemes for 3+1D ideal relativistic hydrodynamics, targeting event-by-event simulations of quark-gluon plasma flow fluctuations and higher-order flow harmonics. The authors note that existing CPU-based 3+1D programs are insufficiently efficient for such studies, and their GPU approach aims to fill that gap. The paper tests against known analytic solutions including the Sod shock tube and the ellipsoidal flow solution of Sinyukov (2005), and finds WENO schemes superior to MUSTA-FORCE in both accuracy and computational cost.
