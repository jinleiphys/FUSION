# Self-interacting QCD strings and string balls

> Strings at T ~ T_c are known to be subject to the so-called Hagedorn phenomenon, in which a string's entropy (times T) and energy cancel each other and result in the evolution of the string into highly excited states, or "string balls". Intrinsic attractive interaction of strings -- gravitational for fundamental strings or in the context of holographic models of the AdS/QCD type, or sigma exchanges for QCD strings -- can significantly modify properties of the string balls. If heavy enough, those start approaching properties of the black holes. We generate self-interacting string balls numerically, in a thermal string lattice model. We found that in a certain range of the interaction coupling constants they morph into a new phase, the "entropy-rich" string balls. These objects can appear in the so-called mixed phase of hadronic matter, produced in heavy ion collisions, as well as possibly in the high multiplicity proton-proton or proton-nucleus collisions. Among discussed applications are jet quenching in the mixed phase and also the study of angular deformations of the string balls.

## Key claim

The paper claims that self-interacting QCD strings near the Hagedorn temperature can form a new phase called "entropy-rich" string balls, but this occurs only in a certain range of the interaction coupling constants and depends on the specific value of the string self-coupling, which is treated as an unknown parameter.

## Method

The authors developed a thermal string lattice model in three spatial dimensions where strings are placed on lattice links and interact via sigma meson exchange, with the temperature being space-dependent rather than using periodic boundary conditions. They performed numerical simulations of ensembles of string balls of various sizes, studying the dependence of string ball properties on the self-interaction coupling constant, with the lattice spacing fixed by requiring the Hagedorn temperature to match lattice QCD results.

## Key numbers

- String tension: sigma_T = (0.42 GeV)^2
- Critical temperature of gluodynamics: T_c ≈ 270 MeV
- Hagedorn temperature ratio: T_H / T_c = 1.11
- Three-dimensional lattice spacing: a_3 = 2.73 GeV^{-1} ≈ 0.54 fm
- Sigma meson mass in vacuum: m_sigma = 0.6 GeV
- Maximal scalar Newton's constant for QCD strings: g_N^{max} ≈ 13 GeV^{-2}
- Critical mass density for maximal coupling: rho_c ≈ 0.28 GeV/fm^3
- Plaquette energy (lowest glueball mass): E_plaquette = 4 sigma_T a ≈ 1.9 GeV

## Context

This work extends the study of self-interacting string balls from fundamental string theory (Horowitz and Polchinski (1997), Damour and Veneziano (1999)) to QCD strings at temperatures near T_c, where the Hagedorn phenomenon occurs. The authors use lattice data from the Vienna group (Zach et al., 1997) on flux tube attraction and from Iritani et al. (2013) on sigma meson clouds around strings to motivate their model. The results enable improved estimates for jet quenching in the mixed phase of heavy ion collisions and provide a framework for studying angular deformations of string balls that could appear in high-multiplicity proton-proton collisions.
