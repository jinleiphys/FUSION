# Tribaryons in a constituent quark model

> We calculate the matrix elements of the color-spin interaction for all possible multi-quark states of tribaryons in flavor SU(3) broken case. For that purpose, we construct the flavor$\otimes$color$\otimes$spin wave functions of the tribaryons, which are taken to be antisymmetric to satisfy the Pauli exclusion principle. Furthermore, we analyze the diquark structure of the tribaryon configurations using the symmetric and antisymmetric basis set of flavor, color and spin states.

## Key claim
The paper calculates the color-spin interaction matrix elements for all possible tribaryon (nine-quark) states in a constituent quark model, explicitly including flavor SU(3) symmetry breaking via a strange quark mass parameter δ = 1 − m_u/m_s. The authors show that while intrinsic three-body forces vanish in the flavor SU(3) symmetric limit, the situation changes when SU(3) breaking is taken into account, potentially reviving solutions to the hyperon puzzle in neutron stars.

## Method
The authors construct fully antisymmetric flavor⊗color⊗spin wave functions for all possible tribaryon configurations using Young-Yamanouchi basis states and the symmetric properties of Young diagrams. They then compute the expectation values of the color-spin interaction operator V_CS = Σ (1/m_i m_j) λ^c_i λ^c_j σ_i·σ_j for each configuration, treating the strange quark mass difference through the parameter δ. The diquark structure of each tribaryon is analyzed by projecting onto symmetric and antisymmetric basis sets for each quark pair and calculating the probability of each diquark configuration within the total wave function.

## Key numbers
- For the q^8 s configuration (one strange quark, eight light quarks) with I=2, S=3/2 (flavor 64 multiplet): H_CS = 56 − (16/3)δ
- For the q^6 s^3 configuration (three strange quarks, six light quarks) with I=3, S=3/2 (flavor 64 multiplet): H_CS = 56 − 16δ + 8δ^2
- For the q^6 s^3 configuration with I=0, S=9/2 (flavor singlet): H_CS = 24 − 16δ + 8δ^2
- For the q^3 s^6 configuration (six strange quarks, three light quarks) with I=3/2, S=3/2 (flavor 64 multiplet): H_CS = 56 − 96δ + 48δ^2
- For the q^3 s^6 configuration with I=1/2, S=1/2 (flavor 35 multiplet): H_CS = 40 − 96δ + 48δ^2

## Context
This work extends the authors' previous study (Park 2019) which showed that intrinsic three-body forces vanish in the flavor SU(3) symmetric case, undermining the solution for the hyperon puzzle in neutron stars proposed by Lattimer (2006) and Bombaci (2016). The tribaryon configuration is related to the three-nucleon interaction at short distance, whose strength is important for the stability of nuclear matter at high density. The results enable future calculations of three-baryon interactions with explicit SU(3) breaking, which could provide a solution to the hyperon puzzle when the symmetry breaking parameter δ is taken into account.
