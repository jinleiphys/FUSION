# The hyperfine anomaly in mercury and test of the Moskowitz-Lombardi rule

> The Moskowitz-Lombardi rule gives a simple relation between the magnetic moment of an atomic nucleus and the effect of its radial distribution on the hyperfine structure - the magnetic hyperfine anomaly or "Bohr-Weisskopf" effect. It was originally formulated for mercury, for which experimental data for nuclear magnetic moments and hyperfine constants were available for a number of isotopes. While the relation for the differential effect between isotopes may be completely determined experimentally, the value for the additive constant that is needed to give the Bohr-Weisskopf (BW) effect for a single isotope has remained untested. In this work, we determine the BW effect in singly-ionized and neutral mercury from experimental muonic Hg-199 data together with our atomic calculations. We check this result by directly extracting the BW effect from the hyperfine constant for singly-ionized Hg-199 using state-of-the-art atomic many-body calculations. From this we deduce an empirical value for the additive constant in the Moskowitz-Lombardi rule, which differs significantly from the values advocated previously.

## Key claim

The Moskowitz-Lombardi rule, which relates the Bohr-Weisskopf (BW) effect to the inverse of the nuclear magnetic moment, has an additive constant that had remained untested for mercury; this work determines an empirical value for that constant that differs significantly from previously advocated values.

## Method

The authors extracted the BW effect in the 1s state of muonic Hg-199 from experimental hyperfine data combined with Dirac equation calculations for the point-nucleus hyperfine constant and QED corrections. They then translated this effect to H-like, singly-ionized, and neutral Hg-199 using electronic screening factors from relativistic atomic many-body calculations at the random phase approximation with exchange level, and independently verified the result by directly extracting the BW effect from the measured hyperfine constant of Hg-199+ using all-orders correlation potential atomic calculations.

## Key numbers

- Extracted BW effect in muonic Hg-199 1s state: A_BW_exp = -0.99(12) keV, corresponding to a relative correction epsilon_exp = -68(8)%
- Relative BW effect in H-like Hg-199 (Hg-199 79+): -2.39(45)%
- Relative BW effect in singly-ionized Hg-199 (Hg-199+): -2.59(49)% from muonic data, and -3.4(10)% from direct atomic extraction
- Relative BW effect in neutral Hg-199: -2.34(44)%
- Empirical Moskowitz-Lombardi parameters for neutral Hg: c_exp = -3.6(8) x 10^-3, alpha_exp = 0.98(5) x 10^-2 mu_N
- Empirical Moskowitz-Lombardi parameters for singly-ionized Hg: c_exp = -4.0(9) x 10^-3, alpha_exp = 1.09(6) x 10^-2 mu_N

## Context

This work extends the Moskowitz-Lombardi rule originally formulated for mercury by Moskowitz and Lombardi (1973) and the theoretical analysis by Fujita and Arima (1975) that added an additive constant. The results show significant deviation from the Fujita-Arima values, indicating omitted many-body nuclear effects in that earlier theory, while being consistent with the original Moskowitz-Lombardi data. The empirical BW effects enable improved precision in calculations of hyperfine structure for mercury, which is used in atomic clocks and electric dipole moment searches, and provide tests of nuclear structure theory.
