# iQMC: Iterative Quasi-Monte Carlo for k-Eigenvalue Neutron Transport Simulations

> The Iterative Quasi-Monte Carlo method, or iQMC, replaces standard quadrature techniques used in deterministic linear solvers with Quasi-Monte Carlo simulation for more accurate and efficient solutions to the neutron transport equation. This work explores employing iQMC in the Monte-Carlo Dynamic Code (MCDC) to solve k-eigenvalue problems for neutron transport with both the standard power iteration and the generalized Davidson method, a Krylov Subspace method. Results are verified with the 3-D, 2-group, Takeda-1 Benchmark problem.

## Key claim

The Iterative Quasi-Monte Carlo (iQMC) method achieves the theoretical O(N^{-1}) convergence rate for k-eigenvalue neutron transport problems by replacing pseudo-random sampling with low-discrepancy sequences in a fixed-seed approach, but only when the number of particle histories per iteration exceeds the number of spatial cells in the mesh.

## Method

The authors implement iQMC in the Monte Carlo Dynamic Code (MCDC) by treating scattering and fission terms as internal fixed sources, reducing the QMC simulation to a particle ray-trace with continuous weight reduction and path-length tally estimation. They solve the k-eigenvalue problem using both standard power iteration and the generalized Davidson method, where the QMC sweep acts as a linear operator for matrix-vector products without explicitly forming matrices. The method uses a fixed-seed approach where the low-discrepancy sequence is reset to the beginning at each iteration, and verification is performed against the 3-D, 2-group Takeda-1 Benchmark problem with a 25x25x25 uniform Cartesian mesh.

## Key numbers

- Reference Monte Carlo solution obtained k_eff = 0.97779 ± 3×10^{-5}, agreeing with the Takeda-1 reference k_eff = 0.9778 ± 5×10^{-4}
- Convergence criterion set as Δk_eff/k_eff ≤ 1×10^{-4} for both methods, with internal source iteration converging to Δφ/φ ≤ 1×10^{-4}
- Davidson method requires nearly 1/2 the number of transport sweeps as power iteration for eigenvalue convergence
- Theoretical QMC convergence rate of O(N^{-1}) achieved for both iterative methods, where N is the number of particles per iteration
- Experiments run on 64 cores from Intel Xeon E5-2680 processors at 2.5 GHz

## Context

This work extends the iQMC method previously applied to 1D fixed-source problems (Pasmann 2023) into k-eigenvalue criticality problems for neutron transport. The generalized Davidson method has been shown to outperform power iteration in similar k-eigenvalue problems (Subramanian 2011), and the authors demonstrate this advantage in the QMC context. The results enable more efficient eigenvalue solvers for neutron transport by replacing standard source iteration with Krylov subspace methods, though the scalar flux accuracy of the Davidson method is limited by its rough preconditioner solve compared to the precise inner solve of power iteration.
