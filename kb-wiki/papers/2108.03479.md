# Forming Molecular States with Hadronic Rescattering

> A method for modelling the prompt production of molecular states using the hadronic rescattering framework of the general-purpose Pythia event generator is introduced. Production cross sections of possible exotic hadronic molecules via hadronic rescattering at the LHC are calculated for the $χ_{c1}(3872)$ resonance, a possible tetraquark state, as well as three possible pentaquark states, $P_c^+(4312)$, $P_c^+(4440)$, and $P_c^+(4457)$. For the $P_c^+$ states, the expected cross section from $Λ_b$ decays is compared to the hadronic-rescattering production. The $χ_{c1}(3872)$ cross section is compared to the fiducial $χ_{c1}(3872)$ cross-section measurement by LHCb and found to contribute at a level of O(1%). Finally, the expected yields of $P_c^+$ production from hadronic rescattering during Run 3 of LHCb are estimated. The prompt background is found to be significantly larger than the prompt $P_c^+$ signal from hadronic rescattering.

## Key claim

The paper introduces a method for modeling prompt production of exotic hadronic molecular states using the hadronic rescattering framework of the Pythia event generator, but finds that hadronic rescattering contributes at only the O(1%) level to chi_c1(3872) production and that the prompt background is significantly larger than the prompt P_c^+ signal from hadronic rescattering for LHCb Run 3.

## Method

The authors extended the hadronic rescattering framework in Pythia 8.306 to allow arbitrary hadron resonance production from 2-to-1 scattering, using nonrelativistic Breit-Wigner cross sections that depend on the mass, total width, and partial widths of each resonance. They defined specific configurations for the chi_c1(3872) tetraquark and three P_c^+ pentaquark candidates (4312, 4440, 4457) using experimentally measured branching ratios for the tetraquark and theoretical partial widths from Lin (2019) for the pentaquarks, then simulated pp collisions at sqrt(s) = 13 TeV to compute differential cross sections in transverse momentum and rapidity.

## Key numbers

- chi_c1(3872) total width set to 1.19 +/- 0.21 MeV (world average from ParticleDataGroup 2020)
- Predicted hadronic-rescattering cross section for chi_c1(3872) at sqrt(s) = 7 TeV in LHCb fiducial region: 0.04 nb, compared to measured LHCb cross section of 5.4 +/- 1.3 +/- 0.8 nb
- Inelastic pp cross section at sqrt(s) = 13 TeV predicted as 78 mb, in agreement with LHC measurements
- Average multiplicities at sqrt(s) = 13 TeV: pi^0 and pi^+/- each O(50), D^0/Dbar^0 and D*^0/Dbar*^0 each O(10^{-1}), Lambda_c^+/- O(10^{-2}), J/psi O(10^{-3}), Sigma_c^+/- and Sigma_c^*+/- O(10^{-4}) and O(10^{-3}) respectively
- For model 1 P_c^+(4312), BR(Lambda_b^0 -> P_c^+ K^-) = 3.7 x 10^{-3}; for model 2, BR = 1.3 x 10^{-4}
- For model 1 P_c^+(4440), BR(Lambda_b^0 -> P_c^+ K^-) = 2.8 x 10^{-3}; for model 2, BR = 1.3 x 10^{-4}
- For model 1 P_c^+(4457), BR(Lambda_b^0 -> P_c^+ K^-) = 2.5 x 10^{-3}; for model 2, BR = 5.1 x 10^{-5}

## Context

This work extends the hadronic rescattering framework developed by Sjostrand (2020) and Bierlich (2021) in Pythia to predict prompt exotic hadron production, complementing coalescence-type models used by Karliner (2004) and Ling (2021) for deuteron and molecular state production at the LHC. The results enable future LHCb measurements to separate prompt from feed-down production of the chi_c1(3872) and P_c^+ states, potentially distinguishing between molecular models for the pentaquarks where the expected differences in Lambda_b decay branching ratios can be as large as O(100).
