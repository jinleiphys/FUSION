# Isgur-Wise Function and $V_{cb}$ from Bethe-Salpeter Equations

> We calculate the Isgur-Wise function from the solutions of the Bethe-Salpeter equations. The shape of the Isgur-Wise function thus calculated is a prediction of the Bethe-Salpeter equations and does not depend on undetermined parameters. We develop an analytical approximation to our Isgur-Wise function in the form $ξ( ω) = η[ 1 - \frac{ρ^2}η (ω- 1) + a (ω- 1)^{3/2}]$ where $ρ^2 = 1.279$, $a = .91$, $η= .9942$ and $ω$ is the recoil velocity. The Isgur-Wise function is then used to obtain $V_{cb}$ from the recent experimental data of ${\bar B} \to D^* \ell \barν$ decay. Our best estimate of $V_{cb}$ is $(34.7 \pm 2.5) \times 10^{-3}$, which is comparable to some of the latest estimates in the literature.

## Key claim

The Isgur-Wise function is predicted from Bethe-Salpeter equations without any additional parameter fitting, but the function does not normalize to unity at zero recoil (ξ(1) = 0.9942) due to finite heavy quark mass corrections that are incorporated in the BSE model.

## Method

The authors solved a covariant reduction of the Bethe-Salpeter equation using a quasipotential framework with a ladder-diagram kernel containing a one-gluon exchange term and a long-range confining potential. The seven parameters (quark masses, string tension, and running coupling parameters) were fixed by fitting the calculated meson spectrum to over 40 observed meson masses. The resulting wavefunctions for the B and D* mesons were then used in the overlap integral formula for the Isgur-Wise function, which was subsequently employed to extract Vcb from experimental differential decay rate data for B → D* ℓ ν decay.

## Key numbers

- Isgur-Wise function slope at zero recoil: ρ² = 1.279
- Isgur-Wise function normalization at zero recoil: η = 0.9942
- Analytical approximation parameter: a = 0.91
- Best estimate of Vcb from CLEO 94 data: (34.7 ± 2.5) × 10⁻³
- Vcb from CLEO 93 data: (34.8 ± 6.1) × 10⁻³
- Vcb from ARGUS 93 data: (35.8 ± 8.3) × 10⁻³
- B meson wavefunction parameters: λ = 0.59 GeV, n = 1.16
- D* meson wavefunction parameters: λ = 0.42 GeV, n = 1.52

## Context

This work extends the heavy quark effective theory (HQET) framework developed by Isgur and Wise (1989) and Neubert (1994) by providing a parameter-free prediction of the Isgur-Wise function from first-principles Bethe-Salpeter dynamics, rather than relying on phenomenological parameterizations. The calculation enables a direct extraction of the CKM matrix element Vcb from experimental data (ARGUS 1993, CLEO 1993, CLEO 1994) without the need to fit additional shape parameters, and the authors note that future work could calculate the four form factors directly without using the heavy quark limit to test the accuracy of the heavy quark approximation.
