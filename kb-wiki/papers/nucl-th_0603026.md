# SHAREv2: fluctuations and a comprehensive treatment of decay feed-down

> This the user's manual for SHARE version 2. SHARE (Statistical Hadronization with Resonances) is a collection of programs designed for the statistical analysis of particle production in relativistic heavy-ion collisions. While the structure of the program remains similar to v1.X, v2 provides several new features such as evaluation of statistical fluctuations of particle yields, and a greater versatility, in particular regarding decay feed-down and input/output structure. This article describes all the new features, with emphasis on statistical fluctuations.

## Key claim

SHAREv2 provides a standardized framework for evaluating both particle yields and event-by-event fluctuations within the grand-canonical ensemble statistical hadronization model, with the explicit caveat that the choice of ensemble (micro-canonical, canonical, or grand-canonical) must be made based on physical conditions, and that fluctuations are considerably less robust than yields against systematic effects from limited experimental acceptance.

## Method

The program calculates particle yields and fluctuations using Bessel function expansions of the grand-canonical ensemble partition function, with particle fugacities derived from quark flavor content including chemical non-equilibrium parameters. Resonance decay feed-down is treated with flexible experimental acceptance coefficients that can be entered for any decay channel, and dynamical fluctuations are computed by subtracting mixed-event statistical fluctuations from raw fluctuations. The MINUIT fitting algorithm from the CERN library is used to perform chi-squared minimization to experimental data.

## Key numbers

- Computation times up to seconds for evaluation of particle yields, rising by up to a factor of 300 during minimization
- Package size: 167 KB directory (without libraries), 26017 lines of FORTRAN77 code
- For particle ratios in mixed events, the statistical fluctuation formula reduces to (sigma^2_{N_i/N_j})_{stat} = 1/<N_i> + 1/<N_j>
- The scaled variance for a single particle in the thermodynamic limit varies by as much as an order of magnitude between different statistical ensembles

## Context

This work extends the SHAREv1.x statistical hadronization code (Torrieri, Jeon, Letessier, Rafelski, 2004) by adding fluctuation calculations and more flexible decay feed-down treatment. It addresses the need to distinguish between competing freeze-out models, such as chemical equilibrium at T approximately 170 MeV versus chemical non-equilibrium at lower temperature, which cannot be resolved using yields alone. The code enables falsification of statistical hadronization models by comparing both yields and fluctuations to experimental data from SPS, RHIC, and future LHC experiments, and can help determine whether canonical or grand-canonical ensembles are physically appropriate for a given system.
