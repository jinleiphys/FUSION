# New limit on fundamental weak-interaction parameters from superallowed beta decay

> A new critical survey of all world data on superallowed beta decays provides demanding tests of, and tight constraints on, the weak interaction. In confirmation of the conserved vector current (CVC) hypothesis, the vector coupling coupling constant is demonstrated to be constant to better than 3 parts in 10^4, and any induced scalar current is limited to fs \leq 0.0013 in electron rest-mass units. The possible existence of a fundamental scalar current is similarly limited to |Cs/Cv| \leq 0.0013. The superallowed data also determine the CKM matrix element to be Vud = 0.9738(4). With PDG values for Vus and Vub, the top-row test of CKM unitarity yields Vud^2 + Vus^2 + Vub^2 = 0.9966(14); although, if a recent result on Ke3 decay is used exclusively to obtain Vus, this sum becomes 0.9999(16). Limits on possible right-hand currents are given.

# New limit on fundamental weak-interaction parameters from superallowed beta decay

## Key claim
The conserved vector current (CVC) hypothesis is confirmed to better than 3 parts in 10^4, and limits on scalar currents are set at |C_S/C_V| ≤ 0.0013, but the CKM unitarity test yields a sum of 0.9966(14) that fails unitarity by 2.4 standard deviations unless a single recent K_e3 decay result for V_us is used exclusively, in which case the sum becomes 0.9999(16) and unitarity is satisfied.

## Method
The authors performed a critical survey of all world data on superallowed 0+ → 0+ beta decays published before November 2004, adjusting Q_EC values to modern calibration standards and applying rigorous statistical averaging following Particle Data Group procedures. They applied transition-dependent radiative corrections (δ_R′) and nuclear-structure-dependent corrections (δ_NS and δ_C) based on shell-model calculations tuned to measured binding energies, charge radii, and isobaric multiplet mass equation coefficients, then extracted corrected ℱt values for twelve transitions from A=10 to A=74.

## Key numbers
- Average corrected ℱt value: 3072.7(8) s, with χ²/ν = 0.42 for twelve superallowed transitions
- Vector coupling constant constancy demonstrated to better than 3 × 10⁻⁴ fractional uncertainty
- Induced scalar current limit: |f_S| ≤ 0.0013 in electron rest-mass units
- Fundamental scalar current limit: |C_S/C_V| ≤ 0.0013, corresponding to Fierz interference constant b_F = +0.0001(26)
- CKM matrix element V_ud = 0.9738(4)
- CKM unitarity sum using PDG values: |V_ud|² + |V_us|² + |V_ub|² = 0.9966(14)
- CKM unitarity sum using E865 K_e3 result for V_us: 0.9999(16)
- First-column unitarity sum: 0.9985(54)
- Right-hand current mixing angle: ζ = 0.00176(74) from the unitarity-failing sum; ζ = -0.00007(84) using E865 V_us

## Context
This work updates the previous survey by Hardy and collaborators (1990) with a large amount of new experimental data from the intervening 14 years, improving the CVC test precision by 30% and reducing the scalar current central values by a factor of 30 compared to the authors' own earlier result (2003). The V_ud value is more precise than the neutron decay determination (0.9745(16)) and is the most precisely known CKM matrix element by more than an order of magnitude. The results enable tighter constraints on right-hand currents and motivate improved calculations of the transition-independent radiative correction Δ_R^V, which contributes ~90% of the V_ud uncertainty and limits further progress from nuclear, neutron, and pion decays alike.
