# {\it Ab initio} nuclear structure - the large sparse matrix eigenvalue problem

> The structure and reactions of light nuclei represent fundamental and formidable challenges for microscopic theory based on realistic strong interaction potentials. Several {\it ab initio} methods have now emerged that provide nearly exact solutions for some nuclear properties. The {\it ab initio} no core shell model (NCSM) and the no core full configuration (NCFC) method, frame this quantum many-particle problem as a large sparse matrix eigenvalue problem where one evaluates the Hamiltonian matrix in a basis space consisting of many-fermion Slater determinants and then solves for a set of the lowest eigenvalues and their associated eigenvectors. The resulting eigenvectors are employed to evaluate a set of experimental quantities to test the underlying potential. For fundamental problems of interest, the matrix dimension often exceeds $10^{10}$ and the number of nonzero matrix elements may saturate available storage on present-day leadership class facilities. We survey recent results and advances in solving this large sparse matrix eigenvalue problem. W also outline the challenges that lie ahead for achieving further breakthroughs in fundamental nuclear theory using these {\it ab initio} approaches.

## Key claim

The no core shell model (NCSM) and no core full configuration (NCFC) methods frame the nuclear quantum many-body problem as a large sparse matrix eigenvalue problem that can provide nearly exact solutions for light nuclei, but the matrix dimension often exceeds 10^10 and the number of nonzero matrix elements may saturate available storage on leadership-class facilities, limiting convergence.

## Method

The Hamiltonian matrix is evaluated in a harmonic oscillator single-particle basis using many-fermion Slater determinants, with the basis truncated by the parameter N_max (total oscillator quanta above the minimum configuration). The lowest eigenvalues and eigenvectors are obtained using the Lanczos algorithm with distributed re-orthonormalization, implemented in the code MFDn (Many Fermion Dynamics - nuclear). The converged eigenvectors are then used to evaluate experimental observables such as ground state energies, quadrupole moments, and spectra.

## Key numbers

- Quadrupole moment of ^6Li: experimentally Q = -0.083 e fm^2; NCFC results with JISP16 NN potential for 15 MeV < hbar-Omega < 25 MeV strongly suggest agreement with experiment.
- Ground state energy of ^6Li: smooth uniform convergence shown with JISP16 NN potential as function of hbar-Omega for several N_max values.
- Matrix dimension for ^12C at N_max=10: 7.8 x 10^9 basis states.
- Matrix dimension for ^16O at N_max=10: 2.4 x 10^10 basis states.
- Storage requirement for ^12C at N_max=10 with 2-body potential: 80 TB; with 3-body potential: 5 PB; with 4-body potential: 140 PB.
- Storage requirement for ^16O at N_max=10 with 2-body potential: 230 TB; with 3-body potential: 12 PB; with 4-body potential: 350 PB.
- Storage requirement for ^28O at N_max=8 with 2-body potential: 1 PB; with 3-body potential: 56 PB; with 4-body potential: 2 EB.
- MFDn performance: overall speed improved by about a factor of 3 over a 2 year span on a test case in ^13C using a NNN potential on 4950 Franklin processors.
- Lanczos iterations: 5600 iterations performed on a matrix with dimension 595 million, converging about 450 eigenvalues and eigenvectors.

## Context

This work extends the NCSM approach of Navratil (2001) and the NCFC method of Maris (2009) to larger basis spaces and heavier nuclei, using realistic NN potentials such as JISP16 from Shirokov (2007) and chiral NN+NNN interactions from Navratil (2007). The paper surveys computational challenges for reaching N_max >= 10 needed for convergence, and outlines future directions including symplectic NCSM (Dytrych), importance-truncated basis spaces (Roth 2007), and realistic single-particle bases (Negoita 2008). The results enable systematic ab initio calculations for oxygen isotopes out to the neutron drip line and for the Hoyle state in ^12C.
