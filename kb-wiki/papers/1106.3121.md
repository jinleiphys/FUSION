# Quantum Monte Carlo calculations of spectroscopic overlaps in $A \leq 7$ nuclei

> We present Green's function Monte Carlo calculations of spectroscopic overlaps for $A \leq 7$ nuclei. The realistic Argonne v18 two-nucleon and Illinois-7 three-nucleon interactions are used to generate the nuclear states. The overlap matrix elements are extrapolated from mixed estimates between variational Monte Carlo and Green's function Monte Carlo wave functions. The overlap functions are used to obtain spectroscopic factors and asymptotic normalization coefficients, and they can serve as an input for low-energy reaction calculations.

## Key claim

Green's function Monte Carlo (GFMC) calculations of one-nucleon spectroscopic overlaps for nuclei with A ≤ 7, using the realistic Argonne v18 two-nucleon and Illinois-7 three-nucleon interactions, produce overlap functions that are extrapolated from mixed estimates between variational Monte Carlo and GFMC wave functions, with the caveat that the GFMC propagator actually uses a simplified Hamiltonian (AV8' plus a three-nucleon part) rather than the full Hamiltonian, so the wave functions are eigenstates of the simplified Hamiltonian.

## Method

The authors constructed variational Monte Carlo (VMC) trial wave functions for A ≤ 7 nuclei using the Argonne v18 two-nucleon and Illinois-7 three-nucleon interactions, then improved them by propagating in imaginary time via the Green's function Monte Carlo method using a simplified Hamiltonian (AV8' plus a three-nucleon part). Spectroscopic overlaps were computed as matrix elements between parent and core-valence states, with the overlap functions obtained from mixed estimates between VMC and GFMC wave functions using a linear extrapolation formula. The overlap functions were then used to compute spectroscopic factors (norms of the overlaps) and asymptotic normalization coefficients.

## Key numbers

- Spectroscopic factors and asymptotic normalization coefficients were calculated for all A ≤ 7 nuclei (specifically 3H, 3He, 4He, 6Li, 6He, 7Li, and 7Be) using the AV18+IL7 Hamiltonian
- For 6Li ground state, the α+d cluster asymptotic boundary condition was imposed with the Whittaker function W−η,l+1/2(2kr)/r, where the Sommerfeld parameter η and wave number k are related to the experimental α-d separation energy
- The GFMC propagation used a simplified Hamiltonian H' = AV8' + H'3N where the three-nucleon part (UIX or IL7) had its central repulsive strength adjusted so that ⟨H'⟩ ≈ ⟨H⟩
- The overlap matrix elements were extrapolated using the formula ⟨O(τ)⟩ ≈ ⟨O(τ)⟩M + [⟨O(τ)⟩M − ⟨O⟩V], neglecting terms of order [δΨ(τ)]²

## Context

This work extends previous variational Monte Carlo overlap calculations for nuclei up to A=10 (unpublished, available from the authors' website) and a more recent VMC-based calculation of asymptotic normalization coefficients in light nuclei by Nollett and Wiringa (2011). It also builds on the VMC/GFMC method that has been used to describe structure and reaction aspects of light nuclei by Pieper, Wiringa, and collaborators (2001-2008). The results can serve as input for low-energy reaction calculations, improving upon earlier approximations that used shell-model or Woods-Saxon single-particle functions, and complement calculations from the no-core shell model (Navratil et al., 2004, 2007) and coupled-cluster method (Jansen et al., 2010).
