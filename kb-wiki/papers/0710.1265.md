# Quantum Monte Carlo calculations of electroweak transition matrix elements in A = 6,7 nuclei

> Green's function Monte Carlo calculations of magnetic dipole, electric quadrupole, Fermi, and Gamow-Teller transition matrix elements are reported for A=6,7 nuclei. The matrix elements are extrapolated from mixed estimates that bracket the relevant electroweak operator between variational Monte Carlo and GFMC propagated wave functions. Because they are off-diagonal terms, two mixed estimates are required for each transition, with a VMC initial (final) state paired with a GFMC final (initial) state. The realistic Argonne v18 two-nucleon and Illinois-2 three-nucleon interactions are used to generate the nuclear states. In most cases we find good agreement with experimental data.

## Key claim

Green's Function Monte Carlo calculations of electroweak transition matrix elements in A=6,7 nuclei, using the Argonne v18 two-nucleon and Illinois-2 three-nucleon interactions, generally show good agreement with experimental data, but the results are obtained using only one-body transition operators and neglect two-body meson-exchange currents, which are known to be significant for magnetic transitions.

## Method

The authors used Green's Function Monte Carlo (GFMC) to project out the exact lowest-energy states for given quantum numbers, starting from Variational Monte Carlo (VMC) trial wave functions constructed with two- and three-body correlation operators. For each off-diagonal transition, two mixed estimates were computed by pairing a VMC initial (final) state with a GFMC final (initial) state, then extrapolated to obtain the full matrix element. The realistic Argonne v18 two-nucleon and Illinois-2 three-nucleon interactions were used to generate the nuclear states, with the GFMC propagation carried out using a simplified Hamiltonian (AV8' + IL2') for computational efficiency.

## Key numbers

- For the ^6Li (3+;0) to ^6Li (1+;0) E2 transition: GFMC reduced matrix element = 9.03(5) e fm^2, giving a width of 4.68(5) x 10^-4 eV (experimental: 4.40(34) x 10^-4 eV)
- For the ^6Li (0+;1) to ^6Li (1+;0) M1 transition: GFMC reduced matrix element = 3.619(4) mu_N, giving a width of 6.86(2) eV (experimental: 8.19(17) eV)
- For the ^6He (2+;1) to ^6He (0+;1) E2 transition: GFMC reduced matrix element = 1.81(3) e fm^2, giving a width of 1.03(3) x 10^-5 eV (experimental: 1.63(10) x 10^-5 eV)
- For the ^7Li (1/2-) to ^7Li (3/2-) E2 transition (Type I): GFMC reduced matrix element = 5.69(6) e fm^2, giving a width of 3.24(7) x 10^-7 eV (experimental: 3.30(20) x 10^-7 eV)
- For the ^7Li (1/2-) to ^7Li (3/2-) M1 transition (Type I): GFMC reduced matrix element = 2.695(4) mu_N, giving a width of 4.58(3) x 10^-3 eV (experimental: 6.30(31) x 10^-3 eV)
- For the ^6He (0+;1) to ^6Li (1+;0) Gamow-Teller transition (Type I): GFMC reduced matrix element = 2.157(1)
- For the ^7Be (3/2-) to ^7Li (3/2-) Gamow-Teller transition (Type I): GFMC reduced matrix element = 2.274(2)

## Context

This work extends previous VMC calculations of electroweak transitions in A=6,7 nuclei using the AV18+UIX Hamiltonian (Wiringa and Schiavilla, 1998; Schiavilla and Wiringa, 2002) by employing the more accurate GFMC method with the AV18+IL2 Hamiltonian, which better reproduces p-shell binding energies. This is the first off-diagonal matrix element calculation using the nuclear GFMC method, and the technique should be applicable to other nuclear calculations such as isospin-mixing and low-energy nuclear reactions. The results enable future inclusion of two-body meson-exchange current corrections and provide benchmarks for shell-model calculations such as Cohen-Kurath (1965) and no-core shell model approaches.
