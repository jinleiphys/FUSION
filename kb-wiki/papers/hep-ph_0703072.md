# Axial current matrix elements and pentaquark decay widths in chiral soliton models

> Here I explain why in chiral soliton models the hadronic transition operator of the pentaquark decay cannot be identified from the axial current.

## Key claim
In chiral soliton models, the hadronic transition operator for the pentaquark decay Θ⁺ → KN cannot be identified from the axial current via the generalized Goldberger-Treimann relation, contrary to what many prior computations assumed; the correct width function contains only a single collective coordinate operator and does not exhibit the cancellations that would yield a small width.

## Method
The paper analyzes the rigid rotator approach to the Skyrme model, where baryons emerge as topological soliton configurations and collective coordinates A(t) ∈ SU(3) are introduced to generate states with good spin and flavor quantum numbers. The author compares the axial current matrix element approach (which uses the GTR to map soliton models onto a Yukawa interaction) with a consistent treatment of kaon-nucleon scattering that solves the equations of motion for small-amplitude fluctuations about the rotating hedgehog, including rotation-vibration coupling. The phase shifts from the adiabatic approximation (exact to leading order in the large-N_C expansion) are compared with the resonance phase shift extracted from the difference between total and background phase shifts, and the width function from the collective coordinate operator is derived.

## Key numbers
- The axial current approach yields coupling constants G₁₀ = G₀ + ½G₁ for the decuplet (Δ) and Ḡ₁₀ = G₀ − G₁ − ½G₂ for the anti-decuplet (Θ⁺), with model calculations indicating G₀ and G₁ are comparable, causing Ḡ₁₀ to be "rather small" (no explicit numerical values given for these integrals).
- The correct width function from the rotation-vibration coupling analysis contains the matrix element X_Θ = √(32/N_C) ⟨Θ⁺↑|X₄₃ + iX₅₃|n↑⟩, normalized such that lim_{N_C→∞} X_Θ = 1.
- The resonance phase shift from the correct treatment exactly matches the Yukawa-exchange phase shift computed from the width function Γ(ω_k) = 2k ω₀ X_Θ² |⟨η̄_{ω_k}|(2λ)|z⟩|² in the limit N_C → ∞.
- The |p_K|³ behavior from the axial current approach is seen only in the energy regime slightly above threshold; at larger energies the correct width function levels off.

## Context
This work extends the detailed analysis of Walliser (2005) which showed that the transition matrix element for Θ⁺ → NK does not equal the axial current matrix element suggested by the generalized GTR, contradicting earlier computations by Diakonov (1997) and Ellis (2004) that relied on this identification. The paper also addresses criticisms from Itzhaki (2003) regarding inconsistencies with the large N_C limit, where Skyrme model KN phase shifts from Karliner (1986) and Scoccola (1990) do not exhibit narrow resonances. The correct treatment enables reliable extraction of pentaquark properties from kaon-nucleon scattering within chiral soliton models without mapping onto the Yukawa model, and the author argues that the non-observation of a narrow Θ⁺ resonance should not be used against the chiral soliton picture for baryons.
