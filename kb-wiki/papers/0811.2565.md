# Hypertriton Electric Polarizability

> A rigorous formalism for determining the electric dipole polarizability of a three-hadron bound complex in the case that the system has only one bound (ground) state has been elaborated. On its basis, by applying a model wave function that takes into account specific features of the structure of the lightest hypernucleus and using the known low-energy experimental data for the p-n and lambda-d systems as input data, we have calculated the value of the electric dipole polarizability of the lambda hypertriton. It follows from our study that the polarizability of the lambda hypertriton is close to 3 fm^3 exceeding the polarizabilities of the ordinary three-nucleon nuclei by an order of magnitude.

## Key claim

The electric dipole polarizability of the lambda hypertriton is close to 3 fm^3, exceeding the polarizabilities of ordinary three-nucleon nuclei by an order of magnitude, based on a model wave function that assumes the hypertriton is a weakly bound two-body system of a deuteron core and a lambda hyperon.

## Method

The authors derived a rigorous expression for the electric dipole polarizability of a three-hadron bound complex with only one bound state, using the formalism of effective interaction with an external Coulomb field and the analytical properties of the three-body transition matrix. They applied this formalism to the hypertriton using a factorized wave function that treats the system as a deuteron core plus a loosely bound lambda hyperon, with the deuteron wave function and the lambda-deuteron wave function obtained from separable S-wave interaction potentials fitted to experimental low-energy data for the p-n and lambda-d systems.

## Key numbers

- Hypertriton electric polarizability with zero-range interactions for both p-n and lambda-d: alpha_E(ZR,ZR) = 1.701 fm^3
- Hypertriton electric polarizability with Yukawa formfactor for p-n and zero-range for lambda-d: alpha_E(Yu,ZR) = 2.479 fm^3
- Hypertriton electric polarizability with Yukawa formfactors for both interactions: alpha_E(Yu,Yu) = 2.914 fm^3
- Hypertriton electric polarizability with exponential formfactors: alpha_E(Exp,Exp) = 2.933 fm^3
- Hypertriton electric polarizability with delta-shell formfactors: alpha_E(DS,DS) = 2.964 fm^3
- Deuteron binding energy used as input: B_d = 2.224575(9) MeV (Ref. 22)
- Triplet p-n scattering length used as input: ^3a_pn = 5.424(3) fm (Ref. 23)
- Lambda hyperon binding energy in hypertriton used as input: B_Lambda = 0.13(5) MeV (Ref. 24)
- Doublet lambda-d scattering length used as additional parameter: ^2a_Lambda d = 15.9 fm (Ref. 25)
- Ratio of lambda binding energy to deuteron binding energy: B_Lambda/B_d = 0.058
- Ratio of momentum scales: kappa_Lambda/kappa_d = 0.295
- Overlap probability of deuteron with hypertriton wave function: P_d(^3_Lambda H) = 0.987 (Ref. 21)

## Context

This work extends the formalism previously developed by Kharchenko and Shadchin (1993, 1997) for two-hadron bound complexes to the three-hadron case, and provides the first calculation of the electric dipole polarizability of the hypertriton. The result that the hypertriton polarizability is an order of magnitude larger than that of ordinary three-nucleon nuclei (triton and helion-3) reflects its halo structure, where the lambda hyperon is loosely bound to a deuteron core. The authors note that further refinement could be achieved using three-body calculational techniques for the bound state wave function, and that the existence of the double-lambda hypernucleus ^4_Lambda Lambda H remains unproven (Randenia and Hungeford, 2007).
