# Fuzzy Bag Models

> We show how hadronic bag models can be generalized to implement effects of a smooth and extended boundary. Our approach is based on fuzzy set theory and can be straightforwardly applied to any type of bag model. We illustrate the underlying ideas by calculating static nucleon properties in a fuzzy chiral bag model.

## Key claim

The paper shows that hadronic bag models can be generalized to include a smooth, extended boundary by using fuzzy set theory, but the approach is a phenomenological prescription rather than a dynamical derivation: the fuzzy boundary shape (membership function) must be chosen based on general physical considerations, not derived from first principles.

## Method

The bag radius is promoted from a real number \(R\) to a fuzzy set \(\rho(R)\), a membership function that assigns a degree of membership (between 0 and 1) to each possible bag radius. Observables \(A(R)\) from a conventional crisp bag model are then extended to fuzzy sets \(\nu(A)\) via the extension principle of fuzzy set theory, and numerical predictions are obtained by taking the centroid of \(\nu(A)\). The method is illustrated by applying a Gaussian membership function to the nonlinear chiral bag model and calculating static nucleon properties.

## Key numbers

- The bag constant is \(B \simeq (150 \, \text{MeV})^4\), the pion decay constant is \(f_\pi = 93 \, \text{MeV}\), and the Skyrme parameter is \(e = 4.5\).
- For a fuzzy boundary with \(R_0 = 0.7 \, \text{fm}\) and \(\Delta = 0.3 \, \text{fm}\), the fuzzy set for the hedgehog energy is computed over the reference set \(\mathcal{R}_\epsilon = [0, 1.5] \, \text{fm}\).
- For the same \(R_0\) and \(\Delta\), the fuzzy set for the axial coupling \(g_A\) is computed over \(\mathcal{R}_\gamma = [0, 1] \, \text{fm}\).
- With \(\Delta = 0.4 \, \text{fm}\) and \(R_0\) in the range \(0 \le R_0 \le 1 \, \text{fm}\), the fuzzy-bag \(g_A\) deviates less than 10% from its experimental value of 1.26.
- The minimum of the fuzzy-bag energy shifts from \(R = 0.85 \, \text{fm}\) (crisp bag) to \(R_0 = 0.5 \, \text{fm}\) (fuzzy bag with \(\Delta = 0.4 \, \text{fm}\)).
- The fuzzy equality measure between Gaussian and triangular boundaries yields \(\parallel \rho^{(g)} = \rho^{(t)} \parallel \; \simeq 0.9\), \(\parallel \epsilon^{(g)} = \epsilon^{(t)} \parallel \; \simeq 0.85\), and \(\parallel \gamma^{(g)} = \gamma^{(t)} \parallel \;\simeq 0.9\), almost independent of \(\Delta\).

## Context

This work extends the standard bag model approach of Chodos et al. (1974) and the chiral bag model reviewed by Hosaka and Toki (1996) by replacing the sharp bag boundary with a smooth transition region. It contrasts with earlier attempts to implement extended boundaries, such as those by Nogami and collaborators (1984), which were technically involved and model-specific. The fuzzy bag model enables a simpler and more systematic study of surface-sensitive observables (e.g., properties of excited and deformed hadrons, low-energy diffractive scattering) and may help reveal aspects of the transition between QCD vacuum phases inside hadrons.
