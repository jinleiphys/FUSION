# New boson realization of the Lipkin model obeying the su(2)-algebra

> New boson representation of the su(2)-algebra proposed by the present authors for describing the damped and amplified oscillator is examined in the Lipkin model as one of simple many-fermion models. This boson representation is expressed in terms of two kinds of bosons with a certain positive parameter. In order to describe the case of any fermion number, third boson is introduced. Through this examination, it is concluded that this representation is well workable for the boson realization of the Lipkin model in any fermion number.

# New boson realization of the Lipkin model obeying the su(2)-algebra

## Key claim

The new boson representation of the su(2)-algebra, previously proposed by the authors for the damped and amplified oscillator, is shown to be workable for the boson realization of the Lipkin model for any fermion number, not only the conventional closed-shell case where the total fermion number N equals 2Ω. The representation requires a third boson to handle cases where N differs from 2Ω, and the conventional case (N=2Ω, s=Ω) serves as a basic building block from which results for other cases can be obtained.

## Method

The authors construct a boson representation of the su(2)-algebra using two kinds of bosons (â, â*) and (b̂, b̂*) with a positive parameter Cm, together with a third boson (ĉ, ĉ*) introduced to describe any fermion number. They map the fermion operators of the Lipkin model, which consists of two single-particle levels (p and h) each with degeneracy 2Ω, onto boson operators by identifying the fermion vacuum with the boson vacuum and setting Cm = 2Ω. The orthogonal basis states are constructed using the raising operators of both the primary su(2)-algebra (Ŝ±) and an auxiliary su(2)-algebra (Λ̂±) that commutes with the primary one, yielding states labeled by Ω, total fermion number N, and the su(2) quantum numbers s and s₀.

## Key numbers

- The degeneracy of each single-particle level is 2Ω = 2j+1, where j is half-integer.
- The parameter Cm in the boson representation is set to Cm = 2Ω.
- For the conventional closed-shell case N = 2Ω, the allowed values of s range from 0 to Ω (s = Ω, Ω-1, ..., 1, 0).
- For N = 0, only s = 0 is allowed; for N = 4Ω, only s = 0 is allowed.
- For N = 2, 4, ..., 2Ω-2, the allowed s values are s = N/2, N/2-1, ..., 1, 0.
- For N = 2Ω+2, 2Ω+4, ..., 4Ω, the allowed s values are s = 2Ω-N/2, 2Ω-N/2-1, ..., 1, 0.
- For odd N = 1, 3, ..., 2Ω-1, the allowed s values are s = N/2, N/2-1, ..., 3/2, 1/2.
- For odd N = 2Ω+1, 2Ω+3, ..., 4Ω-1, the allowed s values are s = 2Ω-N/2, 2Ω-N/2-1, ..., 3/2, 1/2.
- The Hamiltonian considered is Ĥ = εŜ₀ - χ(Ŝ₊² + Ŝ₋²), where ε is the energy difference between levels and χ is the interaction strength.

## Context

This work extends the authors' previous boson representation of the su(2)-algebra (Tsue et al., 2015, referred to as Paper I), which was originally developed for the damped and amplified oscillator in the thermo field dynamics formalism. The representation is applied to the Lipkin model (Lipkin, 1965), a standard many-fermion model for particle-hole pair correlations. The approach generalizes the conventional treatment of the Lipkin model, which typically considers only the closed-shell case N = 2Ω, by introducing an auxiliary su(2)-algebra that enables treatment of non-closed-shell systems (N ≠ 2Ω) and cases where the particle-hole seniority quantum number s differs from Ω even when N = 2Ω. The construction parallels the authors' earlier treatment of the pairing model (Tsue et al., 2015) and uses ideas from the Holstein-Primakoff representation (Holstein and Primakoff, 1940) and the Schwinger boson representation (Schwinger, 1965).
