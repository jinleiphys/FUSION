# Event-plane flow analysis without non-flow effects

> The event-plane method, which is widely used to analyze anisotropic flow in nucleus-nucleus collisions, is known to be biased by nonflow effects,especially at high $p_t$. Various methods (cumulants, Lee-Yang zeroes) have been proposed to eliminate nonflow effects, but their implementation is tedious, which has limited their application so far. In this paper, we show that the Lee-Yang-zeroes method can be recast in a form similar to the standard event-plane analysis. Nonflow correlations are strongly suppressed by using the information from the length of the flow vector, in addition to the event-plane angle. This opens the way to improved analyses of elliptic flow and azimuthally-sensitive observables at RHIC and LHC.

## Key claim

The Lee-Yang-zeroes method for anisotropic flow analysis can be rewritten in a form mathematically equivalent to the standard event-plane method, using an event weight that depends on the length of the flow vector in addition to the event-plane angle, thereby strongly suppressing nonflow effects and autocorrelations without requiring the tedious implementation of earlier formulations.

## Method

The authors recast the Lee-Yang-zeroes method by first computing the flow vector Q for each event and its projection Q_theta onto a fixed direction. They locate the first zero r_theta of the Fourier transform G_theta(r) of the Q_theta distribution, which gives the integrated flow V_n = j_01 / r_theta. In a second pass, each event receives a weight W_R derived from the Bessel function J_1(r_theta Q), and differential flow v_n is computed as a weighted average of cos(n(phi - Phi_EP)) over selected particles, where Phi_EP is the event-plane angle. The method is validated with Monte-Carlo simulations using the GeVSim program, comparing results to 2- and 4-particle cumulant methods.

## Key numbers

- Input average elliptic flow in simulations: ⟨v_2⟩ = 0.0625.
- For flow-only simulations with χ=1, v_2{LYZ} = 0.0626 ± 0.0005, v_2{4} = 0.0624 ± 0.0005, v_2{2} = 0.0626 ± 0.0003.
- For simulations with nonflow effects (each track used twice), v_2{LYZ} = 0.0629 ± 0.0007, v_2{4} = 0.0627 ± 0.0007, v_2{2} = 0.0764 ± 0.0004 (overestimating by >20%).
- The first zero of the Bessel function J_0 is j_01 ≈ 2.40483.
- Recommended number of equally-spaced θ values for the analysis: 5 between 0 and π.
- The method is recommended for resolution parameter χ > 0.8; for χ < 0.6 statistical errors blow up exponentially.

## Context

This work extends the Lee-Yang-zeroes method developed by Bhalerao (2003) and the cumulant method by Borghini (2001) for eliminating nonflow effects in anisotropic flow analyses, which had seen limited use due to implementation complexity. The reformulation makes the method as intuitive as the standard event-plane method by Poskanzer (1998), enabling improved analyses of elliptic flow and azimuthally-sensitive observables at RHIC and LHC. The approach also addresses autocorrelations that require particle subtraction in the standard method, and opens the way to measuring jet production with respect to the reaction plane at LHC without the bias from nonflow correlations within jets.
