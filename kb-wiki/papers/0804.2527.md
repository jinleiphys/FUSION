# Spin-dependent empirical formula for the lowest excitation energies of the natural parity states in even-even nuclei

> We present an empirical expression which holds for the lowest excitation energy of the natural parity states in even-even nuclei throughout the entire periodic table. This formula contains spin-dependent factors so that it is applied to different multipole states with the same model parameters in contrast to the recently proposed empirical expression where the model parameters had to be fitted for each multipole separately.

# Spin-dependent empirical formula for the lowest excitation energies of the natural parity states in even-even nuclei

## Key claim

The authors propose a single empirical formula with spin-dependent factors that can describe the lowest excitation energies of natural parity states in even-even nuclei across the entire periodic table using only eight parameters for all even multipoles and eight for all odd multipoles, rather than fitting six parameters separately for each multipole. The formula reproduces the previous multipole-by-multipole fits with only a ~5% increase in the chi-squared value.

## Method

The authors first fitted the six-parameter formula E_x = alpha A^{-gamma} + beta_p exp(-lambda_p N_p) + beta_n exp(-lambda_n N_n) separately for each multipole (2+, 4+, 6+, 8+, 10+ for even; 3-, 5-, 7-, 9- for odd) using a least chi-squared procedure with logarithmic errors on 1936 even-multipole and 1031 odd-multipole data points. They then examined the J-dependence of the fitted parameters and introduced a spin-dependent parametrization: alpha = alpha_0 J^a, gamma = gamma_0 J^c, lambda_p = lambda_p^0 / sqrt(J), lambda_n = lambda_n^0 / sqrt(J), with beta_p and beta_n taken as constants. The eight parameters for even and eight for odd multipoles were then fitted simultaneously using all data for each parity class.

## Key numbers

* For even multipoles (2+ through 10+): alpha_0 = 11.94 MeV, a = 2.26, gamma_0 = 1.25, c = 0.06, beta_p = 1.04 MeV, beta_n = 1.39 MeV, lambda_p^0 = 0.66, lambda_n^0 = 0.43; overall chi-squared = 0.082
* For odd multipoles (3- through 9-): alpha_0 = 28.15 MeV, a = 1.05, gamma_0 = 0.75, c = 0.12, beta_p = 0.84 MeV, beta_n = 1.02 MeV, lambda_p^0 = 0.77, lambda_n^0 = 0.97; overall chi-squared = 0.048
* The increase in chi-squared when using the spin-dependent formula instead of separate multipole fits is only ~5% for both parity classes
* The previous separate fits used 30 free parameters (6 per multipole for 5 even multipoles) while the new formula uses 8 parameters for all even multipoles combined

## Context

This work extends the empirical formula of Kim, Ha, and Cha (2000) and Jin, Yoon, and Cha (2008), which fitted the lowest excitation energies of natural parity states in even-even nuclei using six parameters per multipole separately. The new spin-dependent formula reduces the number of free parameters from 30 to 8 for even multipoles and from 24 to 8 for odd multipoles, while maintaining nearly identical predictive power. The authors note that the alpha parameter depends on J in an almost quadratic manner for even multipoles (consistent with rotational band energies proportional to J(J+1) in deformed nuclei, as discussed in Bohr and Mottelson (1975)) and an almost linear manner for odd multipoles, for which no theoretical origin has yet been identified. The formula enables systematic predictions of excitation energies across the nuclear chart without requiring separate fits for each multipole.
