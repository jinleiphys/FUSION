# Quark-Hadron Duality for Hybrid Mesons at Large-Nc

> We investigate implications of quark-hadron duality for hybrid mesons in the large-Nc limit. A simple formalism is developed which implements duality for QCD two-point functions of currents of quark bilinears, with any number of gluons. We argue that the large-Nc meson masses share a common parameter, which is related to the QCD string tension. This parameter is fixed from correlators of conserved vector and axial-vector currents, and using lattice QCD determinations of the string tension. Our results predict towers of hybrid mesons which, within expected 1/Nc corrections, naturally accommodate the 1^(-+) experimental hybrid candidates.

## Key claim

The paper argues that in the large-Nc limit, hybrid meson masses share a universal parameter related to the QCD string tension, and that the predicted hybrid towers can accommodate the experimental 1^(-+) candidates only within expected 1/Nc corrections, not as exact predictions.

## Method

The author develops a formalism implementing quark-hadron duality for QCD two-point functions of currents of quark bilinears with any number of gluons, parameterizing meson masses as M^2(n) = M^2 + Λ^2 n and decay constants as F^2(n) = F^2 + E^(2+2m)/M^(2m)(n). The universal parameter Λ is fixed by matching the hadronic spectral function to the operator product expansion for conserved vector and axial-vector currents, using the experimental ρ(770) and ρ'(1450) masses and Fπ = 93 MeV as input, together with lattice QCD determinations of the string tension.

## Key numbers

- Λ = 1172 ± 43 MeV (from analysis of vector and axial-vector correlators)
- √σ = 468 ± 12 MeV (inferred string tension from Λ = √(2πσ))
- αs(Λ) = 0.11 (using two-loop beta function with Λ_MSbar = 380 MeV)
- Fπ = 93 MeV (input)
- Mρ = 770 MeV (input)
- Mρ' = 1465 ± 25 MeV (input)
- Predicted lowest-lying 1^(-+) hybrid mass under universality: 1190 ± 20 MeV
- Predicted first excited 1^(-+) hybrid mass under universality: 1670 ± 40 MeV
- Predicted lowest-lying 0^(--) hybrid mass under universality: 4180 ± 5 MeV
- When using the ~1.4 GeV candidate as input ground state, predicted first excited state: ~2.0 GeV
- When using the ~1.6 GeV candidate as input ground state, predicted first excited state: ~2.4 GeV

## Context

This work extends the large-Nc duality framework previously applied to ordinary mesons by Golterman and Peris (2001) and Beane (2000) to the case of hybrid mesons, which couple to currents containing one gluon. The analysis uses the operator product expansion results for hybrid correlators from Govaerts et al. (1984) and compares predictions to experimental 1^(-+) candidates from the E852 Collaboration (1997, 1998) and Crystal Barrel Collaboration (1998). The results enable future tests of whether the observed hybrid candidates are predominantly gluonic excitations or four-quark states, since four-quark couplings are subleading in 1/Nc.
