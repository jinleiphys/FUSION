# Fault-tolerant fermionic quantum computing

> Simulating the dynamics of electrons and other fermionic particles in quantum chemistry, materials science, and high-energy physics is one of the most promising applications of fault-tolerant quantum computers. However, the overhead in mapping time evolution under fermionic Hamiltonians to qubit gates renders this endeavor challenging. We introduce fermionic fault-tolerant quantum computing, a framework which removes this overhead altogether. Using native fermionic operations we first construct a repetition code which corrects phase errors only. Within a fermionic color code, which corrects for both phase and loss errors, we then realize a universal fermionic gate set, including transversal fermionic Clifford gates. Interfacing with qubit color codes we introduce qubit-fermion fault-tolerant computation, which allows for qubit-controlled fermionic time evolution, a crucial subroutine in state-of-the-art quantum algorithms. As an application, we consider simulating crystalline materials, finding an exponential improvement in circuit depth for a single time step from $\mathcal{O}(N)$ to $\mathcal{O}(\log(N))$ with respect to lattice site number $N$ while retaining a site count of $\tilde{\mathcal{O}}(N)$, implying a linear-in-$N$ end-to-end gate depth for simulating materials, as opposed to quadratic in previous approaches. We also introduce a fermion-inspired qubit algorithm with $O(\mathrm{log}(N)$ depth, but a prohibitive number of additional ancilla qubits. We show how our framework can be implemented in neutral atoms, overcoming the apparent inability of neutral atoms to implement non-number-conserving gates. Our work opens the door to fermion-qubit fault-tolerant quantum computation in platforms with native fermions such as neutral atoms, quantum dots and donors in silicon, with applications in quantum chemistry, material science, and high-energy physics.

## Key claim

The paper claims that by encoding logical fermions directly into physical fermions (rather than mapping fermions to qubits), the overhead of simulating fermionic time evolution under fault-tolerant quantum computation can be removed entirely, achieving exponential improvement in circuit depth for a single Trotter step from O(N) to O(log(N)) with respect to lattice site number N. However, this exponential depth improvement requires the use of O(N log(N)) ancilla fermionic sites for the interaction terms, and the end-to-end gate depth remains linear in N when accounting for the number of Trotter steps.

## Method

The authors construct fermionic error-correcting codes using physical fermionic degrees of freedom (such as fermionic atoms in optical tweezers or electrons in quantum dots) rather than qubits. They first prove that fermionic error correction is impossible using only number-conserving gates, then circumvent this theorem by using fermion parity eigenstates as codestates and introducing a non-number-conserving braiding gate. They build a repetition code for phase errors only, then extend to a fermionic color code (the fermionic version of weakly self-dual CSS codes) that corrects both phase and loss errors, with transversal fermionic Clifford gates. They interface this fermionic code with qubit color codes using a qubit-fermion CZ gate to enable qubit-controlled fermionic time evolution.

## Key numbers

- Circuit depth for a single Trotter step of crystalline material simulation improves from O(N) to O(log(N)) with respect to lattice site number N
- Fermion-site count for the log-depth Trotter step is O(N log(N)) for the fermionic approach
- Circuit volume (depth times number of ancillas) for fermions is a factor of N/log(N) lower than for qubits
- The fermionic color code with odd distance d encodes a single logical fermion into N = (3d^2 + 1)/4 physical fermions; for d=3 this is the fermionic version of the Steane code with 7 physical fermions
- The number of Trotter steps required for time evolution up to time t is O(Nt) for fixed simulation accuracy and fixed Trotter order

## Context

This work extends the concept of Majorana error correction codes originally noted by Bravyi, Terhal, and Leemhuis (2010) for Majorana nanowire platforms, but applies it to complex physical fermions in platforms such as neutral atoms, quantum dots, and donors in silicon. The paper directly addresses the overhead problem identified in prior qubit-based fermion simulation approaches by Abrams and Lloyd (1997), Bravyi and Kitaev (2002), and the plane-wave-dual basis method of Babbush et al. (2018). The framework enables fault-tolerant fermionic quantum computation with applications in quantum chemistry, materials science, and high-energy physics, and provides a path to implement these codes in neutral atoms by overcoming the previously assumed restriction to number-conserving gates in that platform.
