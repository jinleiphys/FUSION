# Equivalence between local Fermi gas and shell models in inclusive muon capture from nuclei

> Motivated by recent studies of inclusive neutrino nucleus processes and muon capture within a correlated local Fermi gas model (LFG), we discuss the relevance of nuclear finite size effects in these reactions at low energy, in particular for muon capture. To disentangle these effects from others coming from the reaction dynamics we employ here a simple uncorrelated shell model that embodies the typical finite size content of the problem. The integrated decay widths of muon atoms calculated with this shell model are then compared for several nuclei with those obtained within the uncorrelated LFG, using in both models exactly the same theoretical ingredients and parameters. We find that the two predictions are in quite good agreement, within 1--7%, when the shell model density and the correct energy balance is used as input in the LFG calculation. The present study indicates that, despite the low excitation energies involved in the reaction, integrated inclusive observables, like the total muon capture width, are quite independent of the fine details of the nuclear wave functions.

## Key claim

The paper demonstrates that for inclusive muon capture, the total integrated decay width calculated with a simple uncorrelated shell model (using Woods-Saxon potentials) agrees with the local Fermi gas model to within 1–7%, provided the shell model density and correct energy balance are used as input in the LFG calculation. This agreement holds despite the low excitation energies involved, which are the worst conditions for the LFG.

## Method

The authors computed inclusive muon capture widths for closed-shell nuclei (¹²C, ¹⁶O, ⁴⁰Ca, ²⁰⁸Pb) using an uncorrelated shell model with Woods-Saxon single-particle potentials fitted to experimental Q-values, and compared these to results from an uncorrelated local Fermi gas model using the same static charged-current operator and identical theoretical ingredients. The shell model calculation included both discrete transitions to bound neutron states and continuum transitions to unbound neutron states, with the muon wave function obtained from the Schrödinger equation in the nuclear Coulomb potential. The LFG width was obtained by averaging the Fermi gas width over the nuclear density distribution weighted by the muon density.

## Key numbers

* For ¹²C with WS1 potential: shell model total width = 0.4406 × 10⁵ s⁻¹, LFG width = 0.4548 × 10⁵ s⁻¹, difference = 3.2%
* For ¹²C with WS2 potential: shell model total width = 0.4289 × 10⁵ s⁻¹, LFG width = 0.4360 × 10⁵ s⁻¹, difference = 1.7%
* For ¹⁶O with WS1 potential: shell model total width = 1.282 × 10⁵ s⁻¹, LFG width = 1.360 × 10⁵ s⁻¹, difference = 6.1%
* For ⁴⁰Ca with WS1 potential: shell model total width = 37.12 × 10⁵ s⁻¹, LFG width = 36.73 × 10⁵ s⁻¹, difference = -1.1%
* For ²⁰⁸Pb with WS1 potential: shell model total width = 390.3 × 10⁵ s⁻¹, LFG width = 399.4 × 10⁵ s⁻¹, difference = 2.3%
* For ¹²C with WS3 potential (unrealistic parameters): difference = -14.1%, showing the importance of using realistic densities

## Context

This work extends the authors' previous correlated LFG study of inclusive neutrino-nucleus processes and muon capture (Nie04) by isolating the role of nuclear finite size effects. It provides a controlled comparison between shell model and Fermi gas approaches, unlike earlier comparisons where different residual interactions, wave functions, and current operators made it impossible to disentangle finite-size effects. The results enable the use of the simpler LFG for integrated inclusive observables in muon capture and low-energy neutrino reactions, with a quantified uncertainty of 1–7% from finite size effects, and complement microscopic calculations by Mea01, Ko94, Ko00, Kol99, Hay00, Vol00, Krm04, Mai03, and Meu03 that treat finite size correctly but use different dynamics.
