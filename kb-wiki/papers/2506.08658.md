# Crust (Unified) Tool for Equation-of-state Reconstruction (CUTER) v2

> The equation of state (EoS) is a needed input to determine the neutron-star global properties and to relate them. It is thus important to provide consistent and unified EoSs to avoid possible biases in the analyses coming from the use of inconsistent EoSs. We propose a numerical tool, CUTER, allowing the user to consistently match a nuclear-physics informed crust to an arbitrary higher density EoS. We present here the second version of this tool, CUTER v2. Two functionalities are available with the CUTER v2 tool, allowing the user to reconstruct either the whole (outer and inner) crust, or the outer crust only. We show that the code, that has been tested and validated for use by the astrophysical community, is able to efficiently perform both tasks, allowing the computation of neutron-star global properties in a consistent way.

## Key claim

CUTER v2 allows users to consistently match a nuclear-physics informed crust to an arbitrary higher-density equation of state, with the caveat that the reconstruction relies on a meta-model approach and a set of isoscalar nuclear empirical parameters that must be either provided or assumed (defaulting to BSk24 values) to perform the inversion procedure.

## Method

The tool reconstructs the crust equation of state by extracting the isovector nuclear empirical parameters from a given beta-equilibrated high-density EoS and a set of isoscalar parameters through a matrix inversion procedure, using a meta-model expansion of the nucleonic energy functional truncated at order N. The crust EoS and crust-core transition point are then calculated within a compressible liquid-drop model in a one-component-plasma approach, where matter is modeled as a periodic lattice of Wigner-Seitz cells containing one type of cluster immersed in a uniform gas of electrons and, in the inner crust, neutrons. For the outer-crust functionality, analytical representations of the Brussels-Montreal Skyrme (BSk) EoSs are used to replace missing or thermodynamically inconsistent outer crust tables.

## Key numbers

- The saturation density used as default is n_sat = 0.1578 fm^-3, with E_sat = -16.048 MeV and K_sat = 245.5 MeV (from the BSk24 functional)
- The whole-crust functionality operates when the original EoS starts at a baryon number density above 10^-4 fm^-3
- The outer-crust functionality runs when the EoS starts below 10^-4 fm^-3, with a default lower limiting density of 10^-11 fm^-3
- For the RG(SLY4) EoS, the reconstructed crust-core transition occurs at n_cc = 0.075 fm^-3 and P_cc = 0.32 MeV fm^-3, compared to the original EoS values of n_cc = 0.052 fm^-3 and P_cc = 0.16 MeV fm^-3
- For the GPPVA(DDME2) EoS, the reconstructed crust-core transition occurs at n_cc = 0.048 fm^-3 and P_cc = 0.22 MeV fm^-3

## Context

This work extends the CUTER v1 tool presented by Davis (2024) by adding an outer-crust reconstruction functionality, addressing issues where outer crust tables are missing or exhibit thermodynamic inconsistencies such as non-monotonic pressure or enthalpy. The tool enables consistent computation of neutron-star global properties from the Tolman-Oppenheimer-Volkoff equations, avoiding biases that can arise from non-unified EoSs as shown by Fortin (2016), Ferreira (2020), Suleiman (2021), and Davis (2024). The code is validated against EoSs from the CompOSE database, including RG(SLY4) by Gulminelli (2015) and GPPVA(DDME2) by Grill (2012, 2014), and is publicly available on Zenodo for use by the astrophysical community.
