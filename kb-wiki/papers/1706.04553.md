# Calculating the norm matrix to solve the A-body Schrödinger equation within a set of non-orthogonal many-body states

> There are efficient many-body methods, such as the (symmetry-restored) generator coordinate method in nuclear physics, that formulate the A-body Schrödinger equation within a set of nonorthogonal many-body states. Solving the corresponding secular equation requires the evaluation of the norm matrix and thus the capacity to compute its entries consistently and without any phase ambiguity. This is not always a trivial task, e.g. it remained a long-standing problem for methods based on general Bogoliubov product states. While a solution to this problem was found recently in Ref. [L. M. Robledo, Phys. Rev. C79, 021302 (2009)], the present work introduces an alternative method that can be generically applied to other classes of states of interest in many-body physics. The method is presently exemplified in the case of Bogoliubov states and numerically illustrated on the basis of a toy model.

# Calculating the norm matrix to solve the A-body Schrödinger equation within a set of non-orthogonal many-body states

## Key claim
The paper introduces an alternative method to compute the norm matrix entries (overlaps) for non-orthogonal many-body states that can be generically applied to other classes of states beyond Bogoliubov product states, whereas the existing solution from Robledo (2009) was specific to Bogoliubov states. The method expresses each overlap as the exponential of an integral of a phase-ambiguity-free linked-connected kernel along an auxiliary manifold connecting the two states.

## Method
The authors derive a master formula expressing the overlap between any two states as the exponential of the integral of an off-diagonal linked-connected kernel of a hermitian operator S[k,l] that unitarily connects them. For a set of N Bogoliubov states, they compute the norm matrix in three steps: (1) normalize all diagonal elements to unity; (2) compute the N-1 elements of the first row using the pivot state as reference, imposing a phase convention that makes these overlaps real; (3) compute the remaining (N-1)(N-2)/2 off-diagonal elements using the same operator S[1,l] but evaluated with different bra states. The off-diagonal kernels are evaluated using the off-diagonal Wick theorem, and the operator S[1,l] is extracted from the Bogoliubov transformations via a matrix logarithm.

## Key numbers
- The single-particle Hilbert space dimension used in the toy model is n=10, consisting of five doubly-degenerated single-particle levels.
- The set M contains N=3 different Bogoliubov states, yielding a 3x3 norm matrix with N(N+1)/2 = 6 independent entries.
- The BCS occupations v_p^2[k] for p=1,...,5 are chosen decreasingly in the interval ]0,1[ to mimic a realistic fully paired system.

## Context
This work extends the solution to the long-standing problem of computing overlaps between general Bogoliubov product states, which was first solved by Robledo (2009) using a Pfaffian of a skew-symmetric matrix based on the Thouless representation. The present method provides an alternative that can be applied to other classes of many-body states beyond Bogoliubov states, such as those used in symmetry-restored generator coordinate method (GCM) calculations in nuclear physics and in quantum chemistry. The method enables consistent computation of the norm matrix for generalized eigenvalue problems arising in GCM, symmetry restoration, and particle-number-restored Bogoliubov coupled-cluster theories, and could be extended to quantum Monte Carlo approaches using more elaborate trial states than Slater determinants.
