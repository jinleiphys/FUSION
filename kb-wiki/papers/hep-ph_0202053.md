# Mesons in a Poincare Covariant Bethe-Salpeter Approach

> We develop a covariant approach to describe the low-lying scalar, pseudoscalar, vector and axialvector mesons as quark-antiquark bound states. This approach is based on an effective interaction modeling of the non--perturbative structure of the gluon propagator that enters the quark Schwinger-Dyson and meson Bethe-Salpeter equations. We consistently treat these integral equations by precisely implementing the quark propagator functions that solve the Schwinger-Dyson equations into the Bethe-Salpeter equations in the relevant kinematical region. We extract the meson masses and compute the pion and kaon decay constants. We obtain a quantitatively correct description for pions, kaons and vector mesons while the calculated spectra of scalar and axialvector mesons suggest that their structure is more complex than being quark-antiquark bound states.

## Key claim

The authors develop a Poincaré-covariant Bethe-Salpeter approach using a Gaussian effective gluon propagator (without the perturbative logarithmic tail) and find that while pions, kaons, and vector mesons are well described as quark-antiquark bound states, the calculated spectra of scalar and axialvector mesons suggest their structure is more complex than simple quark-antiquark pairs.

## Method

The authors model the non-perturbative gluon propagator with a Gaussian dressing function (parameters ω and D) and use the rainbow-ladder approximation for the quark-gluon vertex. They solve the quark Schwinger-Dyson equations for the propagator functions A(p²) and B(p²) on the real positive (spacelike) axis, then analytically continue these solutions to complex momenta for use in the homogeneous Bethe-Salpeter equations. The Bethe-Salpeter equations are solved as eigenvalue problems by tuning the meson momentum P² = -M² to find bound state masses, with the vertex functions expanded in Chebyshev polynomials.

## Key numbers

- For ω = 0.5 GeV, D = 16.0 GeV⁻², m_u = 5×10⁻³ GeV, m_s = 0.115 GeV: M_π = 0.137 GeV, f_π = 0.133 GeV, M_K = 0.492 GeV, f_K = 0.164 GeV (experiment: M_π = 0.135 GeV, f_π = 0.131 GeV, M_K = 0.498 GeV, f_K = 0.160 GeV)
- For the same parameter set: M_ρ = 0.758 GeV, M_K* = 0.946 GeV, M_φ = 1.078 GeV (experiment: M_ρ = 0.770 GeV, M_K* = 0.892 GeV, M_φ = 1.020 GeV)
- Scalar meson masses for ω = 0.5 GeV: M_{u\bar{u}} = 0.645 GeV, M_{u\bar{s}} = 0.903 GeV, M_{s\bar{s}} = 1.113 GeV
- Axialvector (J^PC = 1⁺⁺) meson masses for ω = 0.5 GeV: M_{u\bar{u}} = 0.927 GeV, M_{u\bar{s}} = 1.140 GeV, M_{s\bar{s}} = 1.292 GeV (experiment: 1.230 GeV, 1.270 GeV, 1.282 GeV)
- Quark mass function at zero momentum for ω = 0.5 GeV, D = 16.0 GeV⁻²: M(p²=0) = 0.397 GeV (m_0 = 0) and 0.622 GeV (m_0 = 0.115 GeV)

## Context

This work extends the approach of Maris and Roberts (1999) by omitting the perturbative logarithmic tail of the gluon propagator, which the authors argue is an unnecessary complication for low-energy meson phenomenology. The study provides the first comprehensive investigation of scalar, pseudoscalar, vector, and axialvector mesons within a consistent Schwinger-Dyson/Bethe-Salpeter framework with a Gaussian interaction. The results enable future work on whether scalar mesons require a two-quark–two-antiquark description as suggested by Jaffe (1977) and the Syracuse group (2001), and whether going beyond ladder approximation is necessary for axialvector channels.
