# Mean transverse momenta correlations in hadron-hadron collisions in MC toy model with repulsing strings

> In the present work, Monte-Carlo toy model with repulsing quark-gluon strings in hadron-hadron collisions is described. String repulsion creates transverse boosts for the string decay products, giving modifications of observables. As an example, long-range correlations between mean transverse momenta of particles in two observation windows are studied in MC toy simulation of the heavy-ion collisions.

## Key claim

The paper presents a Monte-Carlo toy model where repulsion between quark-gluon strings generates transverse boosts for decay products, and shows that this mechanism can produce long-range correlations between mean transverse momenta in two separated pseudorapidity windows, with the correlation strength depending on the assumed string radius.

## Method

The model simulates hadron-hadron collisions by distributing partons inside nucleons (Woods-Saxon for nuclei, 2D-Gaussian for partons with σ_xy = 0.4 fm) and forming strings when partons from different hadrons are within distance d_p = 0.4 fm. Strings are then repelled from each other based on their overlap area and an excess energy density λ_1, giving each string a transverse Lorentz boost β_string. The strings hadronize into ρ-mesons (mass 775 MeV, Gaussian smearing σ = 80 MeV) which decay into pions, and all particles from one string receive the same transverse boost.

## Key numbers

- String radius R_string values tested: 0.25 fm, 1 fm, 2 fm, and 3 fm
- Event-averaged string boost ⟨β_string⟩ ≈ 0.65 kept constant in most central collisions by adjusting λ_1
- In most central events, charged particle multiplicity in |η| < 0.5 is ~1600, similar to central Pb-Pb at √s_NN = 2.76 TeV
- Correlation coefficient b_corr^(pT-pT) grows with centrality to ≈ 0.4 units for R_string = 1, 2, and 3 fm, while for R_string = 0.25 fm it is significantly lower
- For R_string = 1 fm, the increase in b_corr starts at ≈ 80% centrality; for R_string = 3 fm, it starts at ≈ 50% centrality
- Pion pT spectra shown for impact parameters b = 0 and 14 fm (no numerical values given for the spectra themselves)
- Woods-Saxon parameters for Pb^208: radius 6.62 fm, parameter a = 0.546 fm
- Mean number of partons per nucleon: 15
- String energy density λ = 1.0 GeV
- 3% probability for a string to be "hard scattered"
- ρ-meson charge probabilities: 25% positive, 25% negative, 50% neutral

## Context

This work extends the string interaction ideas of Abramovsky (no year given in text) on attraction and repulsion of chromoelectric tubes, and contrasts with the string fusion model (references stringFusion1, stringFusion2, stringFusion3) which predicts different modifications to multiplicity and pT spectra. The toy model enables comparison between string repulsion and string fusion scenarios using observables like mean pT correlations and dihadron correlations, with the latter also presented in a companion paper in the same proceedings. The model reproduces some experimentally observed features of azimuthal asymmetry (near-side ridge structure) but has difficulties with strangeness enhancement, which is possible in the string fusion scenario.
