# TOUCANS: a versatile Monte Carlo neutron transport code based on Geant4

> This paper reports the development of TOUCANS, a new Monte Carlo neutron transport code fully written using the Geant4 toolkit. It aims at modeling complex systems easily and rapidly, thanks to a simple key-value input file. While its main focus is the transport of low energy neutrons -- below 20 MeV -- using evaluated cross-section libraries, it also takes advantage of the state-of-the-art capabilities offered by Geant4 regarding the modeled cross-sections for practically all particles at all energies. Hence it gives access to non-specialists to a wide variety of applications involving different particles, such as for instance the simulation of compact accelerator based neutron sources (CANS) which initially motivated its development. It leverages on recent implementations performed within Geant4 to provide both shielding calculation capabilities (using Adaptive Multilevel Splitting, which is a fully automated variance reduction technique) and criticality calculation capabilities (to estimate the keff of a multiplicative system). It also benefits from a tight coupling with CADMesh to readily import CAD models, with NCrystal to extend the support of thermal neutron transport to crystals, with FIFRELIN to increase compound nucleus de-excitation accuracy, and with FUNZ to provide to the user parametric simulation capabilities (such as multi-objective algorithms, calculation of Pareto sets, etc). Thanks to recent improvements of the neutron physics within the Geant4 Neutron-HP package, the TOUCANS code can be considered on-par with industrial neutron transport codes such as Tripoli-4 or MCNP in term of neutron physics accuracy. Its sources should be released soon amongst Geant4 advanced examples.

## Key claim

TOUCANS is a Monte Carlo neutron transport code built on Geant4 that achieves neutron physics accuracy on-par with industrial codes such as Tripoli-4 or MCNP for low-energy neutrons below 20 MeV, but the paper qualifies this by noting the code's primary validation has been against compact accelerator based neutron source (CANS) experimental configurations rather than a comprehensive suite of benchmarks.

## Method

The code uses Geant4's Neutron-HP package with evaluated cross-section libraries (ENDF/B-VIII.0, JEFF-3.3) for neutron transport below 20 MeV, and implements thermal scattering laws with the SVT algorithm, Doppler Broadening Rejection Correction (DBRC), and probability tables for unresolved resonance regions. It provides a key-value input file interface that masks Geant4's C++ complexity, and integrates external tools including CADMesh for CAD geometry import, NCrystal for crystalline thermal neutron scattering, FIFRELIN for compound nucleus de-excitation, and FUNZ for multi-objective optimization coupling. The code implements Adaptive Multilevel Splitting (AMS) for variance reduction in shielding calculations and a power iteration method with combing algorithm for criticality (keff) estimation.

## Key numbers

- Neutron transport validated for energies below 20 MeV (the primary focus range)
- AMS benchmark achieved agreement with analytical formula within 2% difference for a neutron attenuation of 10^+25 in a 1D nested sphere benchmark (radii 1 cm to 90 cm, 1 cm steps)
- AMS benchmark used parameters: N=10000 particles per batch, K=100 particles killed per iteration, B=1000 batches, with Sigma_t=1 cm^-1, sigma_el=5 barns, sigma_c=5 barns
- AMS benchmark completed in 1 hour on a mono-processor simulation
- NCrystal library version 3.5.1 contains more than 130 materials
- Validation performed against CANS experimental configurations at the 3 MeV proton IPHI accelerator at CEA-Saclay in 2016, 2019, and 2022

## Context

This work extends the capabilities of Geant4 for low-energy neutron transport, which was historically limited compared to dedicated codes like Tripoli-4 (developed by CEA) and MCNP (developed by Los Alamos). The code was initially motivated by the SONATE CANS facility development at CEA-Saclay (France), described by Ott (2020). It builds on recent improvements to Geant4's Neutron-HP package by Mendoza (2014, 2018), Tran (2018), Thulliez (2022), and Marek (2023a, 2023b). The AMS implementation follows the work of Cerou (2007) and its application to neutron transport in Tripoli-4 by Louvin (2017). The code enables non-specialist access to complex multiparticle simulations including CANS design, shielding studies, criticality-safety studies, and multi-objective optimization of experimental setups.
