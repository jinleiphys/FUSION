# Implementation of residual nucleus de-excitations associated with proton decays in $^{12} {\rm C}$ based on the GENIE generator and TALYS code

> We implement the de-excitation processes of residual nuclei associated with proton decays in $^{12} {\rm C}$ based on the GENIE generator and TALYS code. To derive the reasonable excitation energy spectra of residual nuclei $^{11} {\rm B}$, $^{10} {\rm B}$ and $^{10} {\rm Be}$, the default GENIE nucleon decay generator is modified in terms of the Spectral Function nuclear model. Then we use the TALYS code to estimate the de-excitation processes of residual nuclei. The TALYS calculation can partly account for the experimental data.

# Implementation of residual nucleus de-excitations associated with proton decays in $^{12} {\rm C}$ based on the GENIE generator and TALYS code

## Key claim
The authors implement de-excitation processes of residual nuclei from proton decays in $^{12} {\rm C}$ by modifying the default GENIE nucleon decay generator with the Spectral Function nuclear model and then using the TALYS code to estimate de-excitation branching ratios. The TALYS calculation can *partly* account for the experimental data, with the predicted excitation energy spectrum matching measurements in the 15-40 MeV range but showing discrepancies below 15 MeV.

## Method
The default GENIE (version 3.0.2 with G18_02a_00_000 model set) Bodek-Ritchie relativistic Fermi gas model was modified to use the Spectral Function nuclear model for bound protons in $^{12} {\rm C}$, replacing the narrow energy range of 910.0-922.4 MeV with a shell-structured distribution. The TALYS code (version 1.95) was then used to calculate de-excitation branching ratios for residual nuclei $^{11} {\rm B}$, $^{10} {\rm B}$, and $^{10} {\rm Be}$ based on their excitation energy spectra, with the initial population for $^{11} {\rm B}$ following spin and parity $J^\pi=1/2^+$ and a 2 MeV bin width for $E_x \geq 15.9$ MeV.

## Key numbers
- Proton separation energy $S^p = 15.96$ MeV for $^{12} {\rm C}$
- Default GENIE bound proton energy range: $910.0 \, {\rm MeV} \leq E_p \leq 922.4$ MeV
- Forbidden ratio for $p \rightarrow \mu^{+}\rho^0$: 5.4% (default GENIE) vs 28.3% (modified GENIE)
- Forbidden ratio for $p \rightarrow \mu^{+}\omega$: 6.5% (default GENIE) vs 37.8% (modified GENIE)
- Forbidden ratio for $p \rightarrow e^{+}K^{*}(892)^0$: 21.5% (default GENIE) vs 51.2% (modified GENIE)
- Excitation energy peaks of $^{11} {\rm B}$: left peak at 0.0 MeV ($p$-shell), right peak at 20.2 MeV ($s$-shell)
- $^{11} {\rm B}$ state fractions: 53.33% ground state ($E_x \leq 6.2$ MeV), 13.33% medium excitation ($6.2 < E_x < 15.9$ MeV), 33.33% highly excited ($E_x \geq 15.9$ MeV)
- Branching ratio for $n + {^{10}{\rm B}}$ from $^{11} {\rm B}$ de-excitation: 8.8%
- Branching ratio for $n + \alpha + {^{6}{\rm Li}}$ from $^{11} {\rm B}$ de-excitation: 11.3%
- Average kinetic energies from $^{11} {\rm B}$ de-excitation: 6.6 MeV (neutrons), 7.7 MeV (protons), 7.0 MeV (deuterons), 8.0 MeV (tritons), 8.5 MeV ($^{3}{\rm He}$), 6.1 MeV (alphas)
- Neutron multiplicity from $^{11} {\rm B}$ de-excitation: 1.35
- $^{10} {\rm B}$ states: 2.7% ground state ($E_x < 0$)
- Branching ratio for $\gamma + {^{10}{\rm Be}}$ from $^{10} {\rm Be}$ de-excitation: 16.3%
- Branching ratio for $n + {^{9}{\rm Be}}$ from $^{10} {\rm Be}$ de-excitation: 19.6%

## Context
This work extends the GENIE neutrino Monte Carlo generator (version 3.0.2) to include nuclear de-excitation effects for proton decay searches in liquid scintillator detectors such as JUNO and LENA. The modification addresses limitations of the default Bodek-Ritchie nuclear model (Bodek, 1981) which does not reproduce the shell structure observed in $^{12} {\rm C}(e,e'p)^{11} {\rm B}$ experiments (Nakamura, 1976) and leaves residual nuclei in the ground state. The Spectral Function model (Benhar, 2005) provides a more realistic 2-dimensional distribution of momentum and removal energy. The TALYS code enables calculation of de-excitation branching ratios that were previously unavailable for the $s$-hole state of $^{11} {\rm B}$ at excitation energies above 16 MeV. The work enables improved simulations for proton decay searches in JUNO, which has adopted this work for preliminary sensitivity studies of $p \rightarrow \bar{\nu} K^+$. Comparison with quasifree $^{12} {\rm C}(p,2p)^{11} {\rm B}$ data from Yosoi (2003) and Panin (2016) shows reasonable agreement for excitation energies between 15 and 40 MeV.
