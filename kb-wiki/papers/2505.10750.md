# Probing the refined performance of the Categorical-Boosting algorithm to the Hartree-Fock-Bogoliubov mass model with different Skyrme forces

> Nuclear mass can offer profound insights into many physical branches, e.g., nuclear physics and astrophysics, while the predicted accuracy by nuclear mass models is usually far from satisfactory until now, especially within the fully microscopic self-consistent mean-field theory. In this project, we present the predictive power for the binding energy within the the Hartree-Fock-Bogoliubov (HFB) methods with six widely used Skyrme forces (SkM*, SkP, SLy4, SV-min, UNEDF0 and UNEDF1) and evaluate the refined performance of the machine learning based on a novel Categorical Boosting (CatBoost) algorithm to the Skyrme HFB mass models. The root-mean-square (rms) deviations between the bare HFB calculations with different Skyrme forces and the available experimental data range from the minimum, about 1.43 MeV, for the UNEDF0 parameter set to the maximum, about 7.03 MeV, for the SkM* paraterer set. For the CatBoost-refined HFB predictions, the predictive power can be significantly improved. All the prediction accurancies on the testing set can reach the level around 0.2 MeV and, meanwhile, the large model bias can be reduced. The model-repair coefficients for the adopted Skyrme parameter sets are uniformly more than 80\%. Moreover, for 21 newly measured nuclei outside AME2020, the predicted masses by the CatBoost-refined HFB models are also in good agreement with the experimental data, illustrating their good generalization abilities. Intrestingly, it is found that the optimal Skyrme parameter set that possesses the highest predictive power for the bare HFB mass calculations may be not the best candidate for the CatBoost-refined HFB model, indicating the different abilities of picking up the missing ``physics'' for different Skyrme forces by the CatBoost algorithm.

## Key claim

The CatBoost machine learning algorithm can reduce the root-mean-square (rms) deviations of Hartree-Fock-Bogoliubov (HFB) mass calculations with six different Skyrme forces to approximately 0.2 MeV on the testing set, but the Skyrme parameter set that performs best for the bare HFB calculations (UNEDF0) is not necessarily the best candidate after CatBoost refinement, indicating that the algorithm has different abilities to pick up missing physics for different Skyrme forces.

## Method

The authors performed HFB calculations using six Skyrme parameter sets (SkM*, SkP, SLy4, SV-min, UNEDF0, and UNEDF1) to generate theoretical binding energies for 2457 nuclei with Z, N >= 8 from the AME2020 dataset. They then applied the CatBoost gradient boosting algorithm, using seven input features (Z, N, N/Z, even-odd indicators for Z and N, and distances to nearest magic numbers), to learn the residuals between the HFB predictions and experimental binding energies. The model was trained with 10-fold cross-validation and a 4:1 training-to-testing split, using the residual as the target variable to refine the HFB mass predictions.

## Key numbers

* The rms deviations between bare HFB calculations and experimental data range from a minimum of about 1.43 MeV (UNEDF0 parameter set) to a maximum of about 7.03 MeV (SkM* parameter set).
* After CatBoost refinement, all prediction accuracies on the testing set reach the level around 0.2 MeV.
* The model-repair coefficients for all six adopted Skyrme parameter sets are uniformly more than 80%.
* The experimental dataset contains 2457 nuclei with Z, N >= 8, with an average reported mass uncertainty of roughly 25 keV.
* The rms deviations on the training data set are approximately 0.04-0.1 MeV.

## Context

This work extends prior efforts to refine nuclear mass models using machine learning algorithms, such as Bayesian neural networks (Niu, 2018), Gaussian processes (PhysRevC.109.064322), and LightGBM (Gao, 2021), by specifically evaluating the CatBoost algorithm across multiple Skyrme-HFB mass tables. The study enables systematic comparison of how different Skyrme forces interact with machine learning refinement, and demonstrates that the optimal bare HFB parameter set may not yield the best refined model, suggesting that future work should consider the interplay between the physics model and the machine learning algorithm when selecting effective interactions.
