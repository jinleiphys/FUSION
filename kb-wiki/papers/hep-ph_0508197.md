# A note on color neutrality in NJL-type models

> By referring to the underlying physics behind the color charge neutrality condition in quark matter, we discuss how this condition should be properly imposed in NJL-type models in a phenomenologically meaningful way. In particular, we show that the standard assumption regarding the use of two color chemical potentials, chosen in a very special way, is not justified in general. When used uncritically, such an approach leads to wrong or unphysical conclusions.

## Key claim

The standard practice of imposing color neutrality in NJL-type models by requiring equal densities of red, green, and blue quarks (i.e., vanishing diagonal color charges \(n_3\) and \(n_8\) only) is not generally justified; when used uncritically, it can lead to wrong or unphysical conclusions because non-diagonal color charge components may be non-zero and must also be eliminated.

## Method

The authors analyze the color neutrality condition in NJL-type models by comparing it to the underlying QCD physics, where static gluon condensates \(\langle A^0_a \rangle\) enforce color neutrality. They demonstrate that the eight color chemical potentials \(\mu_a\) should correspond to the eight gluon field components and that the neutrality condition \(n_a = 0\) must be imposed for all eight color charges \(a = 1,\dots,8\). Using a specific \(SU(3)_c\) rotation that transforms the standard 2SC diquark condensate into a symmetric one, they show that the rotated state, while having \(n_3 = n_8 = 0\), acquires non-zero non-diagonal color charges \(n_1, n_4, n_6\), proving it is not color neutral.

## Key numbers

* For the standard 2SC phase with only red and green quarks paired, the non-vanishing color density is \(n_8 > 0\) (with \(n_r = n_g > n_b\)).
* Under a color rotation to the symmetric state (\(\Delta_1' = \Delta_2' = \Delta_3' = \Delta_3/\sqrt{3}\)), the non-diagonal color charges become \(n_1' = -n_4' = n_6' = n_8/\sqrt{3} \neq 0\), while \(n_3' = n_8' = 0\).
* The length \(|\vec n| = \sqrt{\sum n_a^2}\) of the color charge vector remains invariant under color rotations.

## Context

This work extends the earlier argument by Alford and Rajagopal (2002) that color neutrality in NJL models must be imposed by hand, and it corrects the approach used in recent papers by He (2005) and Blaschke (2005), which restricted neutrality to diagonal charges and incorrectly claimed a symmetric state was the most favored color-neutral solution. The paper clarifies that for the most common phases (2SC, CFL, uSC), the standard procedure using only \(\mu_3\) and \(\mu_8\) is sufficient, but this must be verified case by case rather than assumed.
