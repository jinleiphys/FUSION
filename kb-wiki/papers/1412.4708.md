# Alternative ansatz to wounded nucleon and binary collision scaling in high-energy nuclear collisions

> We introduce TRENTO, a new parametric initial condition model for high-energy nuclear collisions based on eikonal entropy deposition via a "reduced thickness" function. The model simultaneously describes experimental proton-proton, proton-nucleus, and nucleus-nucleus multiplicity distributions, and generates nucleus-nucleus eccentricity harmonics consistent with experimental flow constraints. In addition, the model is compatible with ultra-central uranium-uranium data unlike existing models that include binary collision terms.

# TRENTO: A Reduced Thickness Initial Condition Model for High-Energy Nuclear Collisions

## Key claim

The paper introduces TRENTO, a parametric initial condition model that deposits entropy proportional to the generalized mean of participant thickness functions, and claims it simultaneously describes multiplicity distributions across proton-proton, proton-nucleus, and nucleus-nucleus systems while being compatible with ultra-central uranium-uranium data, unlike models that include binary collision terms. The key qualifier is that the reduced thickness parameter p, gamma fluctuation parameter k, and nucleon profile are not rigorously constrained in this work, and the presented results do not necessarily represent the best-fit of the model to data.

## Method

The model constructs participant thickness functions T_A and T_B for each colliding projectile by summing over participant nucleons weighted by independent gamma-distributed random numbers, then computes entropy deposition proportional to the reduced thickness T_R = ((T_A^p + T_B^p)/2)^(1/p), the generalized mean of the two thicknesses. For composite systems (p-nucleus, nucleus-nucleus), nucleon positions are sampled from Woods-Saxon distributions, collision probabilities are calculated via eikonal overlap integrals with an effective parton-parton cross-section sigma_gg, and only nucleons that collide with at least one partner are retained as participants. The final charged-particle multiplicity is obtained by integrating the reduced thickness profile, rescaling by a normalization constant, and sampling a Poisson number for each event.

## Key numbers

- Effective proton area B = (0.6 fm)^2 used for Gaussian beam-integrated proton density
- Fluctuation parameter k values for p = +1: k = 0.8; p = 0: k = 1.4; p = -1: k = 2.2 (approximate best-fit values for each p)
- Normalization constants for p = 0: p+p norm = 19, p+Pb norm = 17, Pb+Pb norm = 16
- Normalization constants for p = -1: p+p norm = 24, p+Pb norm = 26, Pb+Pb norm = 18
- Normalization constants for p = +1: p+p norm = 9.7, p+Pb norm = 7.0, Pb+Pb norm = 13
- TRENTO with p = 0 (geometric mean) yields eccentricity ratio sqrt(epsilon_2^2)/sqrt(epsilon_3^2)^0.6 within the experimentally allowed band and similar to IP-Glasma

## Context

This work extends the wounded nucleon and two-component Monte Carlo Glauber model approach (Miller, 2007) by introducing a scale-invariant entropy deposition ansatz that excludes binary collision scaling, motivated by the failure of Glauber models to describe ultra-central uranium-uranium data from STAR (2013, 2014). The model is designed to enable systematic model-to-data comparisons (Bernhard, 2015) for constraining initial condition parameters and sQGP medium properties, and is publicly available for community use. The reduced thickness parameter p interpolates between physically different entropy production mechanisms, with p = 0 (geometric mean) showing behavior similar to the IP-Glasma dynamical model (Schenke, 2012) in eccentricity ratio tests.
