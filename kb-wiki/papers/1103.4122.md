# The First J=1+ T=0 States in a Single j Shell Configuration for Even-Even Nuclei

## Key claim
The first even-even nucleus in which J=1+ T=0 states appear within a single j shell configuration is 48Cr, but only when the configuration space includes eight nucleons (four protons and four neutrons) in the f7/2 shell. In this limited single-j model space, all T=0 to T=0 M1 matrix elements vanish identically.

## Method
The author counts the number of possible J=1+ states in the (f7/2)^4 configuration for 44Ti and the (f7/2)^8 configuration for 48Cr by enumerating all allowed proton-neutron angular momentum couplings. The number of T=0 J=1+ states is obtained by subtracting the number of J=1+ states in the neighboring odd-odd nucleus 48V (which must have isospin greater than zero) from the total number of J=1+ states in 48Cr. The M1 selection rule is derived by noting that in a single j shell the M1 operator can be replaced by g_j J, making the isoscalar M1 matrix element proportional to the total angular momentum operator, which cannot induce transitions.

## Key numbers
- In 44Ti with the (f7/2)^4 configuration, there are three J=1+ states with possible isospins 0, 1, and 2, but no T=0 J=1+ states exist.
- In 48Cr with the (f7/2)^8 configuration, there are 17 possible J=1+ states with a priori possible isospins T=0, 1, 2, 3, 4.
- In 48V (three protons and five neutrons in the f7/2 shell), there are 14 J=1+ states, all with isospin greater than zero.
- The number of T=0 J=1+ states in the (f7/2)^8 configuration is 17 minus 14, yielding three such states.
- The three J=1+ T=0 states in 48Cr have energies of 7.775 MeV (s=-1), 9.258 MeV (s=-1), and 9.037 MeV (s=+1), where s = (-1)^V with V = (v_p + v_n)/2.

## Context
This work extends the early shell model counting studies by McCullen, Bayman, and Zamick (1964) and Ginocchio and French (1963), who noted missing spin-isospin combinations in the f7/2 shell, and the more systematic state-counting methods of Talmi (2005) and Zhao and Arima (2005). The finding that all T=0 to T=0 M1 matrix elements vanish in the single j shell provides a selection rule relevant to the broader review of M1 excitations by Heyde, Von Neumann-Cosel, and Richter (2010), though the specific mode considered here has not yet been studied experimentally. The results enable future experimental searches for these J=1+ T=0 states in 48Cr and tests of the predicted M1 selection rules.
