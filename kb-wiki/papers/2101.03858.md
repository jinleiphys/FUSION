# Remarks on Static Three-Quark Potentials, String Breaking and Gauge/String Duality

> Making use of the gauge/string duality, it is possible to study some aspects of the string breaking phenomenon in the three quark system. Our results point out that the string breaking distance is not universal and depends on quark geometry. The estimates of the ratio of the string breaking distance in the three quark system to that in the quark-antiquark system would range approximately from $\frac{2}{3}$ to $1$. In addition, it is shown that there are special geometries which allow more than one breaking distance.

## Key claim

Using gauge/string duality, the string breaking distance in the three-quark system is not universal and depends on the spatial geometry of the quarks, with the ratio of the breaking distance in the three-quark system to that in the quark-antiquark system ranging approximately from 2/3 to 1, and special geometries can allow more than one breaking distance.

## Method

The author employs an effective string model on a deformed AdS5-like geometry with a scalar field (tachyon) to describe light quarks, using the Nambu-Goto action for fundamental strings, a baryon vertex action, and a boundary term for light quarks. String breaking is studied by equating the energy of the connected three-quark (QQQ) string configuration to the energy of the disconnected configuration representing a QQq meson plus a Q\bar{q} meson, using the Y-law for large quark separations. The analysis focuses on two symmetric geometries (equilateral triangle and symmetric collinear) and interpolating geometries (isosceles triangle and diquark limit).

## Key numbers

*   For the equilateral triangle geometry with parameter set L (σ = 0.450 GeV², γ = 0.176, κ = -0.102), the string breaking distance ℓ_3Q^(1) = 0.910 fm, giving a ratio ℓ_3Q^(1)/ℓ_Q\bar{Q} = 0.746.
*   For the symmetric collinear geometry with parameter set L (σ = 0.450 GeV², γ = 0.176, κ = -0.102), the string breaking distance ℓ_3Q^(1) = 1.100 fm, giving a ratio ℓ_3Q^(1)/ℓ_Q\bar{Q} = 0.902.
*   For the equilateral triangle geometry with parameter set P (σ = 0.150 GeV², γ = 0.440, κ = -0.102), the string breaking distance ℓ_3Q^(1) = 0.810 fm, giving a ratio ℓ_3Q^(1)/ℓ_Q\bar{Q} = 0.754.
*   For the symmetric collinear geometry with parameter set P (σ = 0.150 GeV², γ = 0.440, κ = -0.102), the string breaking distance ℓ_3Q^(1) = 0.934 fm, giving a ratio ℓ_3Q^(1)/ℓ_Q\bar{Q} = 0.870.
*   The string breaking distance in the Q\bar{Q} system is ℓ_Q\bar{Q} = 1.22 fm for the L parameter set and ℓ_Q\bar{Q} = 1.074 fm for the P parameter set.
*   The string tension σ = e γ s is approximately 0.215 GeV² for the L set and 0.285 GeV² for the P set.

## Context

This work extends the author's previous studies of static three-quark potentials (Andreev, 2020) and string breaking in the quark-antiquark system (Andreev, 2020) to the three-quark case, using the same effective string model that was previously fitted to lattice QCD data for pure SU(3) gauge theory. The results provide predictions for string breaking distances in baryonic systems that could be tested by future lattice simulations with dynamical quarks, and they enable the study of how the breaking distance interpolates between the diquark limit (where it equals the mesonic breaking distance) and symmetric geometries.
