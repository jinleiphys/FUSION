# Probing criticality with deep learning in relativistic heavy-ion collisions

> Systems with different interactions could develop the same critical behaviour due to the underlying symmetry and universality. Using this principle of universality, we can embed critical correlations modeled on the 3D Ising model into the simulated data of heavy-ion collisions, hiding weak signals of a few inter-particle correlations within a large particle cloud. Employing a point cloud network with dynamical edge convolution, we are able to identify events with critical fluctuations through supervised learning, and pick out a large fraction of signal particles used for decision-making in each single event.

## Key claim

Using a dynamical edge convolution network on point clouds of particle momenta, the authors can identify individual heavy-ion collision events that contain weak critical fluctuations (modeled by the 3D Ising universality class) even when only 5% or 10% of the particles carry the correlated signal, a regime where traditional scaled factorial moment (intermittency) analysis fails entirely.

## Method

The authors generate background events using the JAM hadronic transport model for Au+Au collisions at √sNN = 27 GeV. Critical fluctuations are introduced by replacing a fraction (5% or 10%) of each JAM event's particles with particles generated by the Critical Monte Carlo (CMC) model, which produces momenta following a power-law distribution f(Δp) = A Δp^{-1-ν} with ν = 1/6 (the 3D Ising universality class exponent). A graph-based neural network with two k-nearest-neighbor and dynamical edge convolution blocks is trained for two tasks: binary event classification (critical vs. background) and particle-level tagging to identify which individual particles carry the critical signal.

## Key numbers

- Testing accuracy for event classification with 10% CMC particle replacement: 97.7% using the dynamical edge convolution network, compared to 84.8% using a simple point cloud network without edge convolution.
- Testing accuracy for event classification with 5% CMC particle replacement: 92.8% using the dynamical edge convolution network, compared to 83.4% using the simple point cloud network.
- Average catching rate (fraction of CMC particles correctly tagged): 73.6% for η = 5% and 75.9% for η = 10%.
- Average tagging rate (purity of tagged particles): 94.5% for η = 5% and 95.4% for η = 10%.
- The CMC power-law distribution parameters: minimum Δp = 2 × 10^{-7} GeV/c, maximum Δp = 2 GeV/c, with exponent α = 1 + ν where ν = 1/6.
- Training set: 2 × 10^5 events; validation set: 1 × 10^3 events; test set: 2 × 10^4 events (each with a paired critical event, totaling 4.4 × 10^5 events).

## Context

This work extends the use of deep learning for QCD phase transition classification, previously applied to equation-of-state signatures by Pang (2016) and Du (2019), to the specific problem of detecting weak critical fluctuations. It directly addresses the limitation of traditional intermittency analysis (scaled factorial moments), which the authors show cannot differentiate events with 5% or 10% CMC particles from uncorrelated JAM events, consistent with the NA61/SHINE collaboration's observation that intermittency signals are weak or absent in many collision systems. The method enables event-by-event identification of critical fluctuations in heavy-ion collision data from the RHIC Beam Energy Scan program, and the tagging capability allows extraction of the specific particles carrying the critical signal for further analysis.
