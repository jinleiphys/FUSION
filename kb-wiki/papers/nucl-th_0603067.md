# Simple models for shell-model configuration densities

> We consider the secular behavior of shell-model configuration (partial) densities. When configuration densities are characterized by their moments, one often finds large third moments, which can make suitable parameterization of the secular behavior problematic. We review several parameterizations or models, and consider in depth three specific models: Cornish-Fisher, binomial, and modified Breit-Wigner distributions. Of these three the modified Breit-Wigner provides the best secular approximation to exact numerical configuration densities computed via full diagonalization from realistic interactions.

## Key claim

The modified Breit-Wigner (MBW) distribution provides the best secular approximation to exact numerical shell-model configuration densities computed via full diagonalization from realistic interactions, particularly for configurations with large third moments (asymmetries) where other models fail.

## Method

The authors computed exact configuration densities for ⁴⁴Ti in the pf-shell using the REDSTICK shell-model code with the realistic GXPF1 interaction. They compared three parameterized models, Cornish-Fisher, continuous binomial, and modified Breit-Wigner, against these exact densities by fitting each model's parameters to the low-lying moments (centroid, width, skewness, and excess) of the exact configuration densities. The quality of each model was assessed by comparing the numerical moments reproduced by each distribution against the target moments from exact diagonalization.

## Key numbers

- For configuration "Part 100" of ⁴⁴Ti (target asymmetry m⁽³⁾ = 1.02, target fourth moment m⁽⁴⁾ = 6.58): Cornish-Fisher gave m⁽³⁾ = 1.26 and m⁽⁴⁾ = 6.87; continuous binomial gave m⁽³⁾ = 0.79 and m⁽⁴⁾ = 3.72; MBW reproduced target moments exactly.
- For configuration "Part 079" (target m⁽³⁾ = -0.70, target m⁽⁴⁾ = 5.43): Cornish-Fisher gave m⁽³⁾ = -0.61 and m⁽⁴⁾ = 3.59; continuous binomial gave m⁽³⁾ = -0.62 and m⁽⁴⁾ = 3.36.
- The MBW distribution is applicable for m⁽⁴⁾ - 3 > 1.42 (m⁽³⁾)² - 0.52, a region that includes the numerically observed range for realistic interactions.
- The continuous binomial distribution cannot find solutions for |m⁽³⁾| > 0.5.

## Context

This work extends earlier studies of configuration moments by Kotlarchyk, Porile, and Shlomo (1986), who found that Gram-Charlier/Edgeworth expansions perform poorly for large asymmetries and that Cornish-Fisher distributions were superior. The authors' own prior investigation (Teran and Johnson, 2006) demonstrated that non-trivial third moments are important for realistic configuration densities. The MBW model enables practical computation of total level densities as sums of partial densities for use in Hauser-Feshbach calculations of neutron-capture rates and pre-equilibrium emission calculations, without requiring full diagonalization of the Hamiltonian.
