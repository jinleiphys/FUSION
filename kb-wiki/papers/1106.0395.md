# Spectral distribution method for neutrinoless double-beta decay nuclear transition matrix elements: Binary correlation results

> Neutrinoless double-beta decay nuclear transition matrix elements are generated by an effective two-body transition operator and it consists of Gamow-Teller like and Fermi like (also tensor) operators. Spectral distribution method for the corresponding transition strengths (squares of the transition matrix elements) involves convolution of the transition strength density generated by the non-interacting particle part of the Hamiltonian with a spreading function generated by the two-body part of the Hamiltonian. Extending the binary correlation theory for spinless embedded $k$-body ensembles to ensembles with proton-neutron degrees of freedom, we establish that the spreading function is a bivariate Gaussian for transition operators $\co(k_\co)$ that change $k_\co$ number of neutrons to $k_\co$ number of protons. Towards this end, we have derived the formulas for the fourth-order cumulants of the spreading function and calculated their values for some heavy nuclei; they are found to vary from $\sim -0.4$ to -0.1. Also for nuclei from $^{76}$Ge to $^{238}$U, the bivariate correlation coefficient is found to vary from $\sim 0.6 - 0.8$ and these values can be used as a starting point for calculating nuclear transition matrix elements using the spectral distribution method.

## Key claim

The paper establishes that the spreading function for neutrinoless double-beta decay transition strengths, generated by the two-body part of the Hamiltonian, is a bivariate Gaussian for transition operators that convert a fixed number of neutrons into an equal number of protons. This result is derived using binary correlation theory extended to ensembles with proton-neutron degrees of freedom, but the authors note that the fourth-order cumulants of the spreading function are not exactly zero, varying from approximately -0.4 to -0.1 for heavy nuclei, indicating the Gaussian approximation is close but not perfect.

## Method

The authors extend binary correlation theory, originally developed for spinless embedded k-body ensembles, to ensembles with proton-neutron degrees of freedom. They derive formulas for the fourth-order cumulants of the spreading function that enters the spectral distribution method for neutrinoless double-beta decay transition matrix elements. Using these formulas, they calculate the fourth-order cumulants and the bivariate correlation coefficient for several heavy nuclei by evaluating traces over two-orbit (proton and neutron) configurations with random two-body Hamiltonians and transition operators.

## Key numbers

- Fourth-order cumulants of the spreading function for heavy nuclei vary from approximately -0.4 to -0.1
- Bivariate correlation coefficient for nuclei from 76Ge to 238U varies from approximately 0.6 to 0.8
- The transition operator is a two-body operator with Gamow-Teller like and Fermi like components, with the tensor part contributing only up to 10% of the matrix elements (cited from Ba-09 and Ho-10)
- Parameters for the neutrino potential include: R = 1.1 A^(1/3) to 1.2 A^(1/3) fm, b = 1.003 A^(1/6) fm, and average energy E_bar = 1.12 A^(1/2) MeV

## Context

This work extends the spectral distribution method, previously applied to problems such as single particle transfer (Po-91), beta-decay rates for pre-supernovae evolution (Ko-95, Go-11), giant dipole widths (Ma-98), and parity breaking matrix elements in compound nuclei (To-01), to the calculation of neutrinoless double-beta decay nuclear transition matrix elements. The results enable future application of the spectral distribution method to neutrinoless double-beta decay, providing the bivariate correlation coefficient values as a starting point for calculating nuclear transition matrix elements without constructing large Hamiltonian matrices.
