# $^{16}$C inelastic scattering studied with microscopic coupled-channels method

> In order to test the $^{16}$C internal wave function, we perform microscopic coupled-channels (MCC) calculations of the $^{16}$C($0_1^+ \to 2_1^+$) inelastic scattering by $^{208}$Pb target at $E/A$=52.7 MeV using the antisymmetrized molecular dynamics (AMD) wave functions of $^{16}$C, and compare the calculated differential cross sections with the measured ones. The MCC calculations with AMD wave functions reproduce the experimental data fairly well, although they slightly underestimate the magnitude of the cross sections. The absolute magnitude of calculated differential cross sections is found to be sensitive to the neutron excitation strength. We prove that the MCC method is a useful tool to connect the inelastic scattering data with the internal wave functions.

## Key claim

The microscopic coupled-channels (MCC) calculations using antisymmetrized molecular dynamics (AMD) wave functions reproduce the measured differential cross sections for \(^{16}\)C(\(0_1^+ \to 2_1^+\)) inelastic scattering on a \(^{208}\)Pb target fairly well, although they slightly underestimate the magnitude of the cross sections at large angles, indicating that the AMD wave function slightly underestimates the neutron excitation strength by about 10%.

## Method

The authors performed microscopic coupled-channels (MCC) calculations for the \(^{16}\)C(\(0_1^+ \to 2_1^+\)) inelastic scattering on a \(^{208}\)Pb target at \(E/A\) = 52.7 MeV. They used antisymmetrized molecular dynamics (AMD) wave functions for \(^{16}\)C to compute the diagonal and transition densities, which were then folded with the DDM3Y effective nucleon-nucleon interaction to generate the nuclear coupling potentials. The coupled-channels equations were solved numerically with relativistic kinematics, and an imaginary potential proportional to the real nuclear potential (with strength parameter \(N_I = 1.2\)) was added.

## Key numbers

*   The incident energy is \(E/A\) = 52.7 MeV for the \(^{16}\)C beam on a \(^{208}\)Pb target.
*   The experimental \(B(E2; 2_1^+ \to 0_1^+)\) value for \(^{16}\)C is \(0.63 \pm 0.19\ e^2 {\rm fm}^4\).
*   The AMD(i) wave function (spin-orbit strength \(u_{ls}\) = 900 MeV) yields \(B(E2) = 1.9\ e^2 {\rm fm}^4\), \(M_p^{(2)} = 3.1\ {\rm fm}^2\), and \(M_n^{(2)} = 13.0\ {\rm fm}^2\).
*   The AMD(ii) wave function (\(u_{ls}\) = 1500 MeV) yields \(B(E2) = 1.4\ e^2 {\rm fm}^4\), \(M_p^{(2)} = 2.6\ {\rm fm}^2\), and \(M_n^{(2)} = 12.2\ {\rm fm}^2\).
*   The AMD(iii) wave function (\(u_{ls}\) = 2000 MeV) yields \(B(E2) = 0.9\ e^2 {\rm fm}^4\), \(M_p^{(2)} = 2.2\ {\rm fm}^2\), and \(M_n^{(2)} = 8.9\ {\rm fm}^2\).
*   The sum \(M_p^{(2)} + M_n^{(2)}\) is 16.1 fm\(^2\) for AMD(i), 14.8 fm\(^2\) for AMD(ii), and 11.1 fm\(^2\) for AMD(iii).
*   The experimental angular uncertainty used for smoothing the calculated cross sections is 0.28\(^\circ\).

## Context

This work extends the application of the microscopic coupled-channels (MCC) method, previously used for stable nuclei like \(^{6,7}\)Li, \(^9\)Be, \(^{12}\)C, and \(^{16}\)O (Sakuragi; Hirabayashi; Ito; Takashina; Katsuma), to test the internal wave function of the unstable neutron-rich nucleus \(^{16}\)C. The study directly compares MCC calculations using AMD wave functions with experimental inelastic scattering data from Elekes et al., providing a more rigorous test of the wave function than comparing only electromagnetic transition strengths like \(B(E2)\). The results enable future testing of \(^{16}\)C internal wave functions using hadronic probe data, such as \(^{16}\)C inelastic scattering on a proton target measured at RIKEN (Jin), which is more sensitive to neutron excitation.
