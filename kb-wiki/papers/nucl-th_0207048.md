# An Alternative Parameterization of R-matrix Theory

> An alternative parameterization of R-matrix theory is presented which is mathematically equivalent to the standard approach, but possesses features which simplify the fitting of experimental data. In particular there are no level shifts and no boundary-condition constants which allows the positions and partial widths of an arbitrary number levels to be easily fixed in an analysis. These alternative parameters can be converted to standard R-matrix parameters by a straightforward matrix diagonalization procedure. In addition it is possible to express the collision matrix directly in terms of the alternative parameters.

## Key claim

The paper presents an alternative parameterization of R-matrix theory that is mathematically equivalent to the standard approach but eliminates level shifts and boundary-condition constants, allowing the positions and partial widths of an arbitrary number of levels to be easily fixed in data analysis. The alternative parameters can be converted to standard R-matrix parameters by a straightforward matrix diagonalization procedure, and the collision matrix can be expressed directly in terms of the alternative parameters.

## Method

The author defines an alternative parameterization based on eigenvalues \(\tilde{E}_i\) and eigenvectors \(\tilde{\gamma}_{ic}\) of a real symmetric matrix \(\boldsymbol{\mathcal{E}}\) that incorporates the shift functions \(S_c\) and boundary-condition constants \(B_c\). The transformation from alternative to standard R-matrix parameters is accomplished by solving a generalized eigenvalue equation involving matrices \(\boldsymbol{M}\) and \(\boldsymbol{N}\) constructed from the alternative parameters, which requires a single matrix diagonalization. The inverse transformation from standard to alternative parameters requires solving a nonlinear eigenvalue problem, which the author demonstrates using Rayleigh Quotient Iteration and shows that each eigenvalue trajectory intersects the line \(\hat{E}_i = E\) exactly once.

## Key numbers

- Standard R-matrix parameters for \(J^\pi=1^-\) reactions in the \(^{16}\mathrm{O}\) system (from Azuma et al., 1994): level energies \(E_1 = -0.0451\) MeV, \(E_2 = 2.845\) MeV, \(E_3 = 11.71\) MeV; reduced width amplitudes \(\gamma_{1\alpha} = 0.0793\) MeV\(^{1/2}\), \(\gamma_{2\alpha} = 0.330\) MeV\(^{1/2}\), \(\gamma_{3\alpha} = 1.017\) MeV\(^{1/2}\) for the \(^{12}\mathrm{C}+\alpha\) channel.
- Alternative parameters derived from the standard set: \(\tilde{E}_1 = -0.0451\) MeV, \(\tilde{E}_2 = 2.400\) MeV, \(\tilde{E}_3 = 8.00\) MeV; \(\tilde{\gamma}_{1\alpha} = 0.0793\) MeV\(^{1/2}\), \(\tilde{\gamma}_{2\alpha} = 0.471\) MeV\(^{1/2}\), \(\tilde{\gamma}_{3\alpha} = 0.912\) MeV\(^{1/2}\).
- Channel radius used in the example: 6.5 fm.
- Elements of the transformation matrix \(\boldsymbol{b}\): \(b_{11} = 1.000\), \(b_{22} = 0.9781\), \(b_{33} = 0.9933\), with off-diagonal elements such as \(b_{23} = 0.2281\) and \(b_{32} = -0.1466\).

## Context

This work generalizes the alternative parameterization previously described by Angulo and Descouvemont (2000) for the single-channel case with zero boundary-condition constant, extending it to non-zero boundary-condition constants and an arbitrary number of channels. The paper also builds on concepts discussed by Barker (1971) and provides an exact method for converting between alternative and standard parameters, in contrast to the approximate iterative relation of Angulo and Descouvemont. The formulation enables direct incorporation of spectroscopic information from level compilations or shell-model calculations into R-matrix fits without the complications of level shifts, and it connects to the complex eigenvalue equation introduced by Hale, Brown, and Jarmie (1987) for locating poles of the collision matrix.
