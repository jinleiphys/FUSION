# Fast, Efficient Calculations of the Two-Body Matrix Elements of the Transition Operators for Neutrinoless Double Beta Decay

> To extract information about the neutrino properties from the study of neutrinoless double-beta (0νββ) decay one needs a precise computation of the nuclear matrix elements (NMEs) associated with this process. Approaches based on the Shell Model (ShM) are among the nuclear structure methods used for their computation. ShM better incorporates the nucleon correlations, but have to face the problem of the large model spaces and computational resources. The goal is to develop a new, fast algorithm and the associated computing code for efficient calculation of the two-body matrix elements (TBMEs) of the 0νββ decay transition operator, which are necessary to calculate the NMEs. This would allow us to extend the ShM calculations for double-beta decays to larger model spaces, of about 9-10 major harmonic oscillator shells. The improvement of our code consists in a faster calculation of the radial matrix elements. Their computation normally requires the numerical evaluation of two-dimensional integrals: one over the coordinate space and the other over the momentum space. By rearranging the expressions of the radial matrix elements, the integration over the coordinate space can be performed analytically, thus the computation reduces to sum up a small number of integrals over momentum. Our results for the NMEs are in a good agreement with similar results from literature, while we find a significant reduction of the computation time for TBMEs, by a factor of about 30, as compared with our previous code that uses two-dimensional integrals.

# Fast, Efficient Calculations of the Two-Body Matrix Elements of the Transition Operators for Neutrinoless Double Beta Decay

## Key claim

The authors present a new shell-model code that reduces the computation of two-body matrix elements (TBMEs) for neutrinoless double-beta decay from two-dimensional numerical integrals (over coordinate and momentum space) to a sum of one-dimensional momentum integrals, achieving a speedup factor of about 30 compared to their previous code, but only when using harmonic oscillator single-particle wave functions and the specific rearrangement described.

## Method

The radial matrix elements of the neutrino potential are computed by expressing the product of two harmonic oscillator wave functions as a sum over powers of r, then adding the contribution of the short-range correlation factor [1+f(r)]^2 (which introduces powers of r up to 4). The resulting integrals over the coordinate space are performed analytically, leaving only one-dimensional integrals over the momentum q of the form ∫ exp(-q^2/4μ) q^(2k+2) V_α(q) dq, where V_α(q) contains the finite nucleon size form factors and higher-order nucleon current corrections. The code is tested by computing total nuclear matrix elements for ⁴⁸Ca in the full pf model space using the GXPF1A effective interaction, and for ⁸²Se in the jj44 model space using the JUN-45 effective interaction.

## Key numbers

- Speedup factor of about 30 for TBME computation time compared to the authors' previous code (Ref. HS10, 2010)
- For ⁴⁸Ca: 94 TBMEs computed in 6.29 seconds on a single Intel Xeon X5560 CPU core, giving an average of 6.7×10⁻² seconds per TBME
- For ⁸²Se: 65 TBMEs computed in 6.30 seconds on the same machine, giving an average of 9.7×10⁻² seconds per TBME
- Total nuclear matrix element M^0ν for ⁴⁸Ca: 0.573 (with Jastrow-type SRC, FNS, and HOC included)
- Total nuclear matrix element M^0ν for ⁸²Se: 2.47 (with Jastrow-type SRC, FNS, and HOC included)
- For 8 major harmonic oscillator shells: approximately 434,000 TBMEs (GT plus F) needed; estimated 1-2 days with single-threaded code using the new algorithm

## Context

This work extends the shell-model approach of Horoi and Stoica (2010) and Caurier et al. (1995) for computing 0νββ decay nuclear matrix elements. The previous code required two-dimensional numerical integration and was intractable for calculations in 8-12 major harmonic oscillator shells. The new method enables investigation of effective transition operators in larger model spaces, as proposed by Engel and Hagen (2009) for ⁸²Se, and addresses the need to include higher-order nucleon current corrections (which reduce matrix elements by 20-30%) in large-scale shell-model calculations. The code is also compared with results from the interacting shell model (ISM) calculations of Menendez et al. (2008, 2009) and QRPA calculations of Suhonen (2007).
