# Massively parallel simulations of relativistic fluid dynamics on graphics processing units with CUDA

> Relativistic fluid dynamics is a major component in dynamical simulations of the quark-gluon plasma created in relativistic heavy-ion collisions. Simulations of the full three-dimensional dissipative dynamics of the quark-gluon plasma with fluctuating initial conditions are computationally expensive and typically require some degree of parallelization. In this paper, we present a GPU implementation of the Kurganov-Tadmor algorithm which solves the 3+1d relativistic viscous hydrodynamics equations including the effects of both bulk and shear viscosities. We demonstrate that the resulting CUDA-based GPU code is approximately two orders of magnitude faster than the corresponding serial implementation of the Kurganov-Tadmor algorithm. We validate the code using (semi-)analytic tests such as the relativistic shock-tube and Gubser flow.

## Key claim

The paper presents a GPU implementation of the Kurganov-Tadmor algorithm for 3+1d relativistic viscous hydrodynamics that achieves approximately two orders of magnitude speedup over a serial CPU implementation, but only for the specific algorithm and hardware configuration tested (Nvidia GeForce GTX 980 Ti), and the code is validated only against (semi-)analytic test problems rather than full heavy-ion collision data.

## Method

The authors implement the Kurganov-Tadmor high-resolution shock-capturing scheme on Nvidia GPUs using CUDA C to solve the 3+1d relativistic viscous hydrodynamics equations, including both bulk and shear viscosities with relaxation-type equations derived from the 14-moment approximation. The conservation laws for energy-momentum and the relaxation equations for the dissipative currents are discretized on a uniform Cartesian grid in Milne coordinates (proper time, transverse coordinates, and space-time rapidity), with the GPU handling the fully parallelizable local flux computations and time stepping. The code is validated against analytic solutions for relativistic shock-tube problems and Gubser flow.

## Key numbers

- GPU implementation achieves approximately two orders of magnitude speedup compared to a serial CPU implementation of the same Kurganov-Tadmor algorithm
- Typical running time on an Nvidia GeForce GTX 980 Ti for simulations on a 128^3 grid is 38.4 seconds per time step
- Global memory usage is 0.5 GB for a 128^3 grid
- The GeForce GTX Titan Z performs 8121.6 GFLOPS single precision and 2707.2 GFLOPS double precision, compared to 2416.58 and 1208.29 GFLOPS respectively for the Intel Xeon Phi coprocessor 7120
- Memory bandwidth is 672 GB/s for the Titan Z versus 352 GB/s for the Xeon Phi
- The specific bulk viscosity parametrization uses a peak normalization factor (zeta/s)_norm = 1, with a peak value around 1 for (zeta/s)_norm = 3

## Context

This work extends the application of GPU-accelerated computing to relativistic viscous hydrodynamics for heavy-ion collisions, following earlier CPU-based parallel simulations that relied on Beowulf clusters or supercomputers (Gale 2013, Novak 2013, Bernhard 2016). The authors note that Bayesian parameter estimation for 3+1d QGP simulations remains intractable even with CPU clusters, and the GPU approach enables faster exploration of parameter space. The code is validated against known analytic solutions (relativistic shock-tube and Gubser flow) and uses the Wuppertal-Budapest lattice QCD equation of state (Borsanyi 2010) with a bulk viscosity parametrization interpolating between lattice QGP and hadron resonance gas results (Denicol 2009).
