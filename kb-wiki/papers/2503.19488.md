# Atomic determination of the nuclear quadrupole moment $\mathrm{Q}(^{209}{\rm Bi})$ using the multi-configuration Dirac-Hartree-Fock method

> The multiconfiguration Dirac-Hartree-Fock method implemented in the Grasp2018 package was employed to calculate the magnetic dipole hyperfine interaction constants and electric field gradients of levels in the ground configuration of the neutral bismuth atom. Combining the calculated electric field gradient of the ground state with the measured electric quadrupole hyperfine interaction constant, we extracted the nuclear quadrupole moment for the $^{209}$Bi isotope, $\mathrm{Q}(^{209}\textbf{Bi}) = -422(22)$~mb. This value, together with other results obtained from atomic- and molecular-structure calculations, created the ``world average" nuclear quadrupole moment of this isotope, $\mathrm{Q}(^{209}\textbf{Bi}) = -420(17)$~mb.

## Key claim
The authors extract the nuclear quadrupole moment of $^{209}$Bi as Q = -422(22) mb using MCDHF calculations, but assign a 5% computational uncertainty to the electric field gradient (EFG) rather than the 1-2% suggested by convergence trends, because the calculated energy separations differ from NIST database values by about 5%.

## Method
The multiconfiguration Dirac-Hartree-Fock (MCDHF) method in the Grasp2018 package was used to calculate electric field gradients (EFGs) and magnetic dipole hyperfine interaction constants for levels in the $6p^3$ ground configuration of neutral bismuth. Correlation orbitals were added layer by layer in a systematic active set approach, with single and double substitutions from a multi-reference set $\{6s^26p^3, 6s^26p6d^2\}$ to capture valence-valence and core-valence correlations, followed by relativistic configuration interaction (RCI) calculations including the Breit interaction. The final EFG of the ground state was combined with the experimental electric quadrupole hyperfine interaction constant $B$ from Barzakh (2021) to extract the nuclear quadrupole moment.

## Key numbers
- Extracted nuclear quadrupole moment: Q($^{209}$Bi) = -422(22) mb, using the calculated ground-state EFG q = 3.072 a.u. and the experimental B constant from Barzakh (2021).
- The "world average" nuclear quadrupole moment from 33 theoretical results: Q($^{209}$Bi) = -420(17) mb.
- Calculated EFG of the ground state $^4S^o_{3/2}$: q = 3.072 a.u. (final value after including MR-CI, CV$_3$, and Breit corrections).
- Calculated magnetic field (MFE) of the ground state $^4S^o_{3/2}$: A$_{\text{el}}$ = -554 MHz (final value).
- Computational uncertainty assigned to EFGs: 5%, based on the 5% deviation of calculated energy separations from NIST database values (Kramida 2025).

## Context
This work resolves a longstanding atomic-molecular discrepancy in the nuclear quadrupole moment of $^{209}$Bi, which had a wide spread of results from nuclear theory, muonic, and pionic measurements as reviewed by Stone (2016) and Pyykko (2018). The result is consistent with values obtained by Skripnikov (2021) using the Fock space coupled cluster method (-418(6) mb), Wilman (2021) with semi-empirical atomic structure calculations (-446(15) mb), and Dognon (2023) from molecular structure calculations (-422(3) mb). Together with these independent calculations, the present value contributed to establishing the "world average" Q($^{209}$Bi) = -420(17) mb, which enables determination of quadrupole moments for neutron-deficient bismuth isotopes through the ratio Q($^A$Bi)/Q($^{209}$Bi) = B($^A$Bi)/B($^{209}$Bi) using the measured B constants from Barzakh (2021).
