# Identifying weak critical fluctuations of intermittency in heavy-ion collisions with topological machine learning

> Large density fluctuations of conserved charges have been proposed as a promising signature for exploring the QCD critical point in heavy-ion collisions. These fluctuations are expected to exhibit a fractal or scale-invariant behavior, which can be probed by intermittency analysis. Recent high-energy experimental studies reveal that the signal of critical fluctuations related to intermittency is very weak and thus could be easily obscured by the overwhelming background particles in the data sample. Employing a point cloud neural network with topological machine learning, we can successfully classify weak signal events from background noise by the extracted distinct topological features, and accurately determine the intermittency index for weak signal event samples.

## Key claim

The authors claim that a topological machine learning framework (TopoPointNet) can classify weak intermittency signal events from background noise and accurately determine the intermittency index, but only for signal fractions (replacement ratios) of 5% and 10% in simulated data; the method's performance at the experimentally relevant signal fraction of approximately 1-2% is not demonstrated with numerical results.

## Method

The authors generated signal events by mixing particles from the Critical Monte Carlo (CMC) model, which simulates critical fluctuations using a Levy random walk algorithm, with background events from the Ultra-relativistic Quantum Molecular Dynamics (UrQMD) model for 0%-5% most central Au+Au collisions at sqrt(s_NN)=19.6 GeV. They constructed a TopoPointNet architecture that first extracts topological features (Betti curves for beta_0 and beta_1) from the two-dimensional momentum point cloud data using Delaunay triangulation-based persistent homology, then feeds these features into a point cloud neural network (ClassifyPointNet) for binary classification of signal versus background events.

## Key numbers

*   Signal events were generated with replacement ratios lambda = 5% and lambda = 10%, using 2.56 x 10^5 signal events and UrQMD events for lambda = 5%, and 1.28 x 10^5 signal events and UrQMD events for lambda = 10%.
*   Charged particles were selected within pseudorapidity window |eta| < 0.5 and transverse momentum interval 0.2 < p_T < 2 GeV/c.
*   The CMC model parameters were set to Levy exponent mu = 1/6 and p_min/p_max = 10^{-7} for a critical system belonging to the 3D Ising universality class.
*   The second-order intermittency index phi_2 is predicted to be 2/3 for sigma condensate based on calculations from a critical equation-of-state belonging to the 3D Ising universality class.

## Context

This work extends prior intermittency analyses from the NA49, NA61/SHINE, and STAR experiments, which found that critical fluctuation signals are very weak (approximately 1-2% signal particles) and can be obscured by background. The authors build on previous applications of point cloud neural networks for identifying self-similarity (ML_mix, 2023) and on topological data analysis methods used for identifying phase transitions in the mean-field XY model (Sale, 2022) and for identifying long-range flow correlations in nuclear collisions (DTFE1, 2023). The method enables future extraction of the intermittency index from weak signal samples, potentially allowing comparison with experimental measurements at RHIC Beam Energy Scan energies.
