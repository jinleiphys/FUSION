# $Y(4260)$ as four-quark state

> We treat the $Y(4260)$ resonance as a four-quark state in the framework of the covariant confining quark model. We study two choices of the interpolating current, either the molecular-type current which effectively corresponds to the product of $D$ and $\bar D_1$ quark currents or tetraquark one. In both cases we calculate the widths of decays $Y(4260)\to Z_c(3900)+π$ and $Y(4260)\to D^{(\ast)}+\bar D^{(\ast)}$. It is found that in both approches the mode $Y\to Z^+_c + π^-$ is enhanced compared with the open charm modes. However the absolute value of the $Y\to Z^+_c + π^-$ decay width obtained in molecular picture is arguably too large. On the other hand the value obtained in tetraquark picture is reasonable.

## Key claim

The Y(4260) resonance can be described as a four-quark state, and the covariant confining quark model (CCQM) calculations favor the tetraquark picture over the molecular-type current picture, because the molecular picture yields a decay width for Y(4260) to Z_c(3900) plus pion that is arguably too large compared to the experimental total width.

## Method

The authors used the covariant confining quark model (CCQM) to compute strong decay widths of the Y(4260) resonance, treating it as a four-quark state with two different interpolating currents: a molecular-type current corresponding to a D and anti-D_1 meson pair, and a tetraquark (diquark-antidiquark) current. They calculated the matrix elements for the decays Y(4260) to Z_c(3900) plus pion and Y(4260) to open charm meson pairs (D anti-D, D* anti-D, D* anti-D*) using nonlocal generalizations of the currents and evaluating the resulting loop integrals.

## Key numbers

- Y(4260) mass: 4230 +/- 8 MeV, total width: 55 +/- 19 MeV (from PDG)
- Z_c(3900) mass: 3887.2 +/- 2.3 MeV, width: 28.2 +/- 2.6 MeV
- For the molecular-type current, Gamma(Y to Z_c+ + pi-) = 146 +/- 13 MeV
- For the tetraquark current, Gamma(Y to Z_c+ + pi-) = 5.77 +/- 0.39 MeV
- For the molecular-type current, Gamma(Y to D0 + anti-D0) = 11 +/- 2 MeV
- For the tetraquark current, Gamma(Y to D0 + anti-D0) = (0.42 +/- 0.16) x 10^{-3} MeV
- For the molecular-type current, Gamma(Y to D*0 + anti-D0) = (0.39 +/- 0.14) x 10^{-2} MeV
- For the tetraquark current, Gamma(Y to D*0 + anti-D0) = 0.32 +/- 0.09 MeV
- For the molecular-type current, Gamma(Y to D*0 + anti-D*0) = 0 MeV
- For the tetraquark current, Gamma(Y to D*0 + anti-D*0) = (0.19 +/- 0.08) x 10^{-3} MeV
- Z_c size parameter Lambda = 3.3 GeV (from prior work by Goerke et al. 2016)
- Y size parameter varied around Lambda = 3.3 +/- 0.1 GeV

## Context

This work extends the authors' previous studies of multi-quark states using the CCQM, where they treated the X(3872) as a tetraquark (Dubnicka et al. 2010, 2011) and critically examined the Z_c(3900) as both a tetraquark and a molecular state (Goerke et al. 2016). The paper addresses the long-standing puzzle of the Y(4260) resonance, which does not fit conventional charmonium assignments and has been variously interpreted as a hybrid charmonium (Zhu 2005, Kou 2005, Close 2005), a D anti-D_1 molecule (Wang 2013), or a tetraquark (Maiani 2005, Ebert 2005). The results enable future experimental tests to distinguish between the molecular and tetraquark interpretations by measuring the absolute decay width of Y(4260) to Z_c(3900) plus pion.
