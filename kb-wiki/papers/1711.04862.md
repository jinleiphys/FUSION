# Angular correlations with charmed hadrons in the Monte-Carlo model with string repulsion

> Recent experimental results revealed large elliptic flow of the charmed hadrons at LHC energies, unexpectedly similar to that of the charged pions. The mentioned measurements are often interpreted using transport models, which incorporate dissociation and recombination mechanisms for charm quarks. In this report, a modified version of the Monte Carlo model with string repulsion is used to calculate azimuthal correlations with charmed hadrons. The string repulsion mechanism may provide significant angular correlations and can be considered as an alternative to thermalization picture.

# Angular correlations with charmed hadrons in the Monte-Carlo model with string repulsion

## Key claim

The string repulsion mechanism, implemented in a Monte Carlo model where soft particle production from string hadronization receives transverse boosts from repulsive string interactions while the hard component remains unaffected, may provide significant angular correlations for charmed hadrons and can be considered as an alternative to the thermalization picture for explaining the observed elliptic flow of D mesons at LHC energies.

## Method

The authors used a modified Monte Carlo model with string repulsion, where particle spectra from string breaking in the rest frame were parametrized using exponential fits to the soft part (pt < 1 GeV/c) of identified particle spectra (pions, kaons, protons) measured by ALICE in pp collisions at sqrt(s)=2.76 TeV, with the remaining hard component from perturbative QCD subtracted and treated separately. For charmed hadrons, D meson spectra measured by ALICE in pp collisions were used to parametrize the model in the same manner, assuming c-cbar pairs can be produced in string hadronization with sufficient probability due to modified string tension from string overlapping and fusion. The hard component was explicitly not affected by the transverse boosts from string repulsion, so that in central nucleus-nucleus collisions the soft component becomes "blue-shifted" onto the hard component.

## Key numbers

- D meson elliptic flow v2 in Pb-Pb collisions at sqrt(s_NN)=2.76 and 5.02 TeV was found experimentally to be similar to that of charged pions
- Probability for a quark-gluon string to break into charm and anti-charm quark pair via the Schwinger mechanism is approximately 10^{-11}
- The model calculations for R_AA and v2 were performed for Pb-Pb collisions in centrality range 20-40%
- Experimental D meson v2 data used for comparison came from centrality range 30-50%
- The intermediate pt range where the interplay between soft and hard contributions must be correctly reproduced extends from approximately 1 to 8 GeV/c, depending on particle species

## Context

This work extends the Monte Carlo model with string repulsion previously described by Altsybeev (2016) and Altsybeev and Feofilov (2016) to include charmed hadrons, offering an alternative to transport models that incorporate dissociation and recombination mechanisms for charm quarks. The model qualitatively reproduces the mass ordering of both the nuclear modification factor R_AA and elliptic flow v2 for light flavours and D mesons, as measured by ALICE, though the increasing trend of R_AA observed in data for pt > 8 GeV/c is not captured since the hard component is not modified. The main challenge for quantitative agreement is correctly reproducing the soft-hard interplay in the intermediate pt range.
