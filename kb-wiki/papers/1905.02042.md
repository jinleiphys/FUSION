# Unified Analyses of Multiplicity Distributions and Bose-Einstein Correlations at the LHC using Double-Stochastic Distributions

> We analyze data on multiplicity distributions (MD) at Large Hadron Collider (LHC) energies using a double-negative binomial distribution (D-NBD) and double-generalized Glauber-Lachs formula (D-GGL). Moreover, we investigate the Bose-Einstein correlation (BEC) formulas based on these distributions and analyze the BEC data using the parameters obtained by analysis of MDs. From these analyses it can be inferred that the D-GGL formula performs as effectively as the D-NBD. Moreover, our results show that the parameters estimated in MD are related to those contained in the BEC formula.

## Key claim
The paper claims that the double-generalized Glauber-Lachs (D-GGL) formula performs as effectively as the double-negative binomial distribution (D-NBD) for analyzing multiplicity distributions at LHC energies, and that parameters estimated from multiplicity distributions can be related to those in Bose-Einstein correlation formulas, but this relationship depends on the choice of background subtraction method (N_I^BG vs N_II^BG).

## Method
The authors fit ATLAS multiplicity distribution data at sqrt(s) = 0.9 and 7.0 TeV (p_t > 100 MeV, |eta| < 2.5, n_ch >= 2) using both the D-NBD and the proposed D-GGL formula, where the D-GGL fixes k=2 to reflect the degree of freedom for (+-) particle ensembles. They then use the parameters obtained from these MD fits to compute Bose-Einstein correlation functions via two different background normalization schemes (N_I^BG and N_II^BG) and compare the resulting BEC predictions to ATLAS BEC data.

## Key numbers
- At sqrt(s) = 0.9 TeV, D-GGL fit yields alpha_1 = 0.41 +/- 0.05, p_1 = 0.25 +/- 0.04, <n_1> = 27.0 +/- 1.4, p_2 = 0.39 +/- 0.02, <n_2> = 10.6 +/- 0.4, with chi^2/ndf = 60.3/46
- At sqrt(s) = 7.0 TeV, D-GGL fit yields alpha_1 = 0.67 +/- 0.01, p_1 = 0.68 +/- 0.03, <n_1> = 36.7 +/- 0.3, p_2 = 0.42 +/- 0.02, <n_2> = 10.4 +/- 0.1, with chi^2/ndf = 114/80
- For BEC using Eq. (8) with D-GGL at 0.9 TeV: R_1 = 1.69 +/- 0.04 fm (Gaussian), R_2 = 1.44 +/- 0.18 fm (exponential), chi^2 = 120
- For BEC using Eq. (8) with D-GGL at 7.0 TeV: R_1 = 3.63 +/- 0.01 fm (exponential), R_2 = 1.33 +/- 0.11 fm (exponential), chi^2 = 557
- For comparison, the conventional formula CF_I at 7.0 TeV gives R = 2.1 +/- 0.0 fm (exponential), lambda = 0.62 +/- 0.01, chi^2 = 919

## Context
This work extends the authors' previous analyses using the generalized Glauber-Lachs formula (Mizoguchi 2010) and the double-NBD approach proposed by Giovannini (1998) and Ghosh (2012) to explain KNO scaling violation first observed by the UA5 Collaboration (Fuglesang 1989, Alner 1985, Ansorge 1988). The paper introduces a unified framework for simultaneously analyzing multiplicity distributions and Bose-Einstein correlations using common parameters, which had previously been treated as independent observables. This enables future studies to connect the two-particle correlation structure with the underlying event classification into soft and hard components, and the authors note that a triple-NBD extension (Biyajima 2019) shows improved agreement with BEC data.
