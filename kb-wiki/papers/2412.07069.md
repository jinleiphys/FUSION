# Sim-to-real supervised domain adaptation for radioisotope identification

> Machine learning has the potential to improve the speed and reliability of radioisotope identification using gamma spectroscopy. However, meticulously labeling an experimental dataset for training is often prohibitively expensive, while training models purely on synthetic data is risky due to the domain gap between simulated and experimental measurements. In this research, we demonstrate that supervised domain adaptation can substantially improve the performance of radioisotope identification models by transferring knowledge between synthetic and experimental data domains. We consider two domain adaptation scenarios: (1) a simulation-to-simulation adaptation, where we perform multi-label proportion estimation using simulated high-purity germanium detectors, and (2) a simulation-to-experimental adaptation, where we perform multi-class, single-label classification using measured spectra from handheld lanthanum bromide (LaBr) and sodium iodide (NaI) detectors. We begin by pretraining a spectral classifier on synthetic data using a custom transformer-based neural network. After subsequent fine-tuning on just 64 labeled experimental spectra, we achieve a test accuracy of 96% in the sim-to-real scenario with a LaBr detector, far surpassing a synthetic-only baseline model (75%) and a model trained from scratch (80%) on the same 64 spectra. Furthermore, we demonstrate that domain-adapted models learn more human-interpretable features than experiment-only baseline models. Overall, our results highlight the potential for supervised domain adaptation techniques to bridge the sim-to-real gap in radioisotope identification, enabling the development of accurate and explainable classifiers even in real-world scenarios where access to experimental data is limited.

## Key claim

Supervised domain adaptation via pretraining on synthetic data followed by fine-tuning on just 64 labeled experimental spectra yields a test accuracy of 96% for radioisotope identification with a LaBr detector, but this result is conditioned on the experimental datasets being relatively simple single-label measurements with high signal-to-noise ratio where 100% accuracy can be achieved with approximately 500 training examples.

## Method

The authors pretrained a custom transformer-based neural network (TBNN) on approximately 1.4 million synthetic gamma spectra generated using the GADRAS simulation package for handheld LaBr and NaI detectors. They then fine-tuned the pretrained model on small subsets (as few as 64 spectra) of experimentally measured spectra collected at Pacific Northwest National Laboratory using 32 different source isotopes in standardized source-shield configurations. Performance was compared against source-only models trained purely on synthetic data and target-only models trained from scratch on the same experimental subsets, using 10 repeated trials with different random initializations for uncertainty estimation.

## Key numbers

* TBNN (ours) domain-adapted model achieved test accuracy of 0.955 ± 0.025 on LaBr experimental data using only 64 fine-tuning spectra, compared to source-only accuracy of 0.745 ± 0.010 and target-only accuracy of 0.795 ± 0.047
* TBNN (ours) domain-adapted model achieved test accuracy of 0.946 ± 0.032 on NaI experimental data using 64 fine-tuning spectra, versus source-only accuracy of 0.738 ± 0.024 and target-only accuracy of 0.785 ± 0.043
* In the sim-to-sim scenario with HPGe detectors, TBNN (ours) achieved testing APE score of 0.762 ± 0.002 with 1024 fine-tuning spectra, outperforming source-only (0.652 ± 0.003) and target-only (0.554 ± 0.012)
* The experimental LaBr dataset contained 15,091 spectra and the NaI dataset contained 10,440 spectra, partitioned into 70/15/15 train/validation/test splits
* Synthetic training datasets contained 1.4 × 10^6 spectra for each detector type, generated in less than 15 minutes on a commodity laptop

## Context

This work extends prior efforts in simulation-based inference for gamma spectroscopy, including the transformer-based neural network for nuclide identification introduced by Li (2024) and the transfer learning approaches explored by Moore (2020) which achieved mixed results. The authors build on the PyRIID synthetic data generation framework developed by Morrow (2021) and the GADRAS simulation package. The results enable practical deployment of machine learning classifiers for radioisotope identification in scenarios where experimental training data is scarce, though the authors note that more complicated experimental datasets with mixed sources would be desirable for broader validation.
