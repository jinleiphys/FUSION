# GLISSANDO: GLauber Initial-State Simulation AND mOre

> GLISSANDO is a Glauber Monte-Carlo generator for early-stages of relativistic heavy-ion collisions, written in c++ and interfaced to Root. Several models are implemented: the wounded-nucleon model, the binary collisions model, the mixed model, and the model with hot-spots. Subtleties of the distribution of nucleon in the nucleus are discussed. The original geometric distribution of sources in the transverse plane can be superimposed with a statistical distribution simulating the dispersion in the generated transverse energy in each individual collision. The program generates inter alia the fixed axes (standard) and variable-axes (participant) two-dimensional profiles of the density of sources in the transverse plane and their Fourier components. These profiles can be used in further analyses of physical phenomena, such as the the jet quenching, event-by-event hydrodynamics, or analysis of the elliptic flow and its fluctuations. Characteristics of the event (multiplicities, eccentricities, Fourier coefficients, etc.) are evaluated and stored in a file for further off-line studies. A number of scripts is provided for that purpose. Supplied variants of the code can also be used for the proton-nucleus and deuteron-nucleus collisions.

## Key claim

GLISSANDO provides a publicly available, flexible Monte-Carlo generator for Glauber-like initial-state models in relativistic heavy-ion collisions, implementing wounded-nucleon, binary-collision, mixed, and hot-spot models with optional statistical superposition of transverse-energy deposition, but the specific numerical results for eccentricities and cross sections depend on the chosen model parameters (e.g., wounding profile, expulsion distance, superposition parameters) and are not universal predictions.

## Method

The code generates nucleon positions in each nucleus using Woods-Saxon distributions (with parameters adjusted to reproduce electron-scattering profiles after folding with nucleon charge form factors and accounting for center-of-mass shift and nucleon-nucleon expulsion), then determines wounded nucleons and binary collisions via hard-sphere or Gaussian wounding criteria based on the NN inelastic cross section. For each event, sources (wounded nucleons and/or binary collisions) are assigned relative deposited strengths (RDS) according to the chosen model (wounded-nucleon, binary, mixed with parameter α, or hot-spot with reduced binary cross section), optionally superimposed with Poisson or gamma distributions to simulate event-by-event fluctuations. The program computes fixed-axes (standard) and variable-axes (participant) two-dimensional density profiles and their Fourier components, storing event characteristics (multiplicities, eccentricities, Fourier coefficients) in ROOT files for offline analysis.

## Key numbers

- Default NN inelastic cross section at RHIC: σ_w = 42 mb (for √s_NN = 200 GeV); at LHC: σ_w ≃ 63 mb (√s_NN = 5500 GeV); at SPS: σ_w ≃ 32 mb (√s_NN = 19 GeV)
- Mixed model parameter α = 0.145 for collisions at √s_NN = 200 GeV, α = 0.12 for √s_NN = 19.6 GeV (from fits to particle multiplicities in Back (2004))
- Default expulsion distance d = 0.4 fm; alternative values 0 or 0.8 fm are implemented with separate parameterizations
- Typical running time: 40 s/10000 events for wounded-nucleon model, 60 s/10000 events for mixed model with Γ distribution (minimum-bias Au+Au, hard-sphere wounding, fixed-last algorithm, Intel Core 2 Duo T5200 1.6 GHz); a 500000-event run takes about 1 hour
- Example output for Au+Au at 200 GeV (100000 events, wounded-nucleon model, b_max=20 fm): A+B cross section = 6436.6 mb, N_w = 103.179 ± 105.16, fixed-axes eccentricity ε_fix_2 = 0.315127 ± 0.316701, variable-axes eccentricity ε_var_2 = 0.475002 ± 0.275256
- Hulthen distribution parameters for deuteron: a_d = 0.228 fm^{-1}, b_d = 1.118 fm^{-1}
- Nucleon charge profile width σ = 0.79/√3 fm (reproducing rms charge radius 0.79 fm)
- Hot-spot model uses σ_bin ≃ 2 mb for semi-hard binary collisions

## Context

This work provides a dedicated, publicly available Glauber Monte-Carlo package, filling a gap noted by the authors that existing codes (e.g., HIJING by Wang (1991), VENUS by Werner (1988)) are broader in scope and make modifications or comparisons more difficult. The code extends the standard Glauber approach by incorporating variable-geometry (participant) analysis as developed by Manly (2005), Alver (2006), Voloshin (2006), and Broniowski (2007), and by allowing superposition of statistical distributions (Poisson or gamma) over the geometric source distribution. The generated profiles and eccentricities can serve as input for event-by-event hydrodynamics, jet quenching studies, and analysis of elliptic flow fluctuations, enabling systematic comparisons of different initial-state models and their effects on final-state observables.
