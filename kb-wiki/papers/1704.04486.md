# Flow in small and large quark-gluon plasma droplets: the role of nucleon substructure

> We study the effects of nucleon substructure on bulk observables in proton-lead collisions at the LHC using Bayesian methodology. Substructure is added to the TRENTO parametric initial condition model using Gaussian nucleons with a variable number of Gaussian partons. We vary the number and width of these partons while recovering the desired inelastic proton-proton cross section and ensemble averaged proton density. We then run the model through a large number of minimum bias hydrodynamic simulations and measure the response of final particle production and azimuthal particle correlations to initial state properties. Once these response functions are determined, we calibrate free parameters of the model using established Bayesian methodology. We comment on the implied viability of the partonic model for describing hydrodynamic behavior in small systems.

## Key claim
The paper shows that adding nucleon substructure (multiple Gaussian partons) to the TRENTO initial condition model can describe flow in small systems, but the Bayesian calibration to proton-lead data alone cannot exclude a partonless wounded-nucleon model; the entropy deposition parameter p and nucleon width w remain unconstrained by p+Pb data.

## Method
The authors extended the TRENTO initial condition model by replacing each Gaussian nucleon with a variable number of Gaussian partons of fixed width, then determined participant nucleons via parton-parton collision probabilities tuned to reproduce the inelastic proton-proton cross section. They generated 240 parameter configurations using a space-filling algorithm, ran minimum bias hydrodynamic simulations to compute initial entropy density and eccentricity harmonics, trained a Gaussian process emulator to interpolate these quantities, and calibrated the model to ALICE p+Pb data at 5.02 TeV using Bayesian Monte Carlo methods with response functions linking initial state properties to final charged particle yields and flow cumulants.

## Key numbers
- Proton-lead collisions at center-of-mass energy sqrt(s_NN) = 5.02 TeV, using data from the ALICE experiment
- 240 unique parameter configurations sampled for the design space scaffolding
- Minimum parton width v_min = sqrt(A_min / N_partons) with A_min = 0.09 fm^2
- Parton width capped at v_max = w (the nucleon width)
- Flow cumulants v_n{2} measured for transverse momentum range 0.2 < p_T < 3.0 GeV and pseudorapidity gap Delta_eta > 1
- Preferred parton number N_part >= 10 near the upper end of the design range, with parton width roughly one-half the nucleon width

## Context
This work extends the TRENTO initial condition model (Moreland, 2014) and the Bayesian calibration framework (Bernhard, 2016) from heavy-ion collisions to small systems, addressing the inability of Gaussian proton profiles to generate anisotropic flow in proton-proton collisions. The substructure approach follows earlier work on deformed or lumpy protons (Schenke, 2014). The results enable future simultaneous calibration to both proton-lead and lead-lead systems and motivate expanded design ranges and more rigorous emulator validation.
