# A Bethe-Salpeter model for light mesons: spectra and decays

> The spectra and electroweak decay properties of light mesons are analyzed within the framework of the instantaneous Bethe-Salpeter equation. The interaction kernel comprises alternative spin-structures for a parameterization of confinement and a residual quark-antiquark interaction based on instanton effects. It is shown that only with a vector confinement the parameters can be chosen such as to yield an excellent description of the light pseudoscalar and vector mesons including weak and two photon decays. However it is found that it is not possible to reconcile this with the Regge behavior of higher lying meson states with the same parameter set.

## Key claim

The paper demonstrates that a relativistic Bethe-Salpeter model with a vector confining kernel and an instanton-induced residual interaction can quantitatively describe the masses, weak decay constants, and two-photon decay widths of light pseudoscalar and vector mesons, but only with the explicit caveat that the same parameter set cannot simultaneously reproduce the Regge behavior of higher-lying meson states.

## Method

The authors solved the instantaneous Bethe-Salpeter (Salpeter) equation as an eigenvalue problem for meson bound state masses, using a confining kernel parameterized with either scalar (1⊗1) or vector (γ⁰⊗γ₀) spin structure and a linear-plus-constant potential in coordinate space. A residual quark-antiquark interaction derived from 't Hooft's instanton effects was added, regularized with a Gaussian cutoff, and the model parameters (quark masses, confinement slope and offset, instanton coupling constants, and range) were fitted to reproduce the masses and decay observables of the low-lying pseudoscalar and vector mesons.

## Key numbers

- Pion decay constant: \( f_\pi = 92.4 \) MeV (experimental value used for comparison); model V1 achieved quantitative agreement
- Two-photon decay width for \(\pi^0 \rightarrow \gamma\gamma\): excellent agreement with experiment in model V1
- Two-photon decay width for \(\eta \rightarrow \gamma\gamma\): excellent agreement with experiment in model V1
- Two-photon decay width for \(\eta' \rightarrow \gamma\gamma\): underestimated in model V1
- Leptonic decay widths for \(\rho\), \(\omega\), and \(\phi\) mesons: in good agreement with experimental data for model V1
- For model V2 (designed for Regge trajectories), decay constants and photon decay widths disagreed by about 50% compared to experiment
- The instanton size cutoff \(\rho_c = 0.408\) fm yielded constituent quark masses \(m_n = 170\) MeV and \(m_s = 270\) MeV, with coupling constants \(g = 79\) MeV·fm³ and \(g' = 58\) MeV·fm³

## Context

This work extends the relativistic quark model framework previously developed by the same group (Münz et al., 1993) for solving the Salpeter equation numerically, and applies it to light mesons where nonrelativistic quark models (e.g., Blaizot, 1980) fail to describe pseudoscalar decay widths. The instanton-induced interaction follows the approach of 't Hooft (1976) and Shifman, Vainshtein, and Zakharov (1979), which had been used successfully in nonrelativistic models. The paper contrasts with the Nambu-Jona-Lasinio model (Nambu and Jona-Lasinio, 1961; Klevansky, 1992; Weise, 1992) which describes chiral symmetry breaking but cannot handle confinement or higher angular momentum states, and with the work of Lagae (1992) who rejected scalar confinement in the Salpeter equation. The results enable future calculations of electromagnetic transitions and the pion form factor within the same framework.
