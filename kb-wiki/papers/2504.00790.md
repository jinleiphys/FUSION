# A Novel Deep Learning Method for Detecting Nucleon-Nucleon Correlations

> This study investigates the impact of nucleon-nucleon correlations on heavy-ion collisions using the hadronic transport model SMASH in $\sqrt{s_{\rm NN}}=3$ GeV $^{197}{\rm Au}$+$^{197}{\rm Au}$ collisions. We developed an innovative Monte Carlo sampling method that incorporates both single-nucleon distributions and nucleon-nucleon correlations. By comparing three initial nuclear configurations - a standard Woods-Saxon distribution (un-corr), hard-sphere repulsion (step corr), and ab initio nucleon-nucleon correlations (nn-corr)- we revealed minimal differences in traditional observables except for ultra-central collisions. When distinguishing between un-corr and nn-corr configurations, conventional attention-based point cloud networks and multi-event mixing classifiers failed (accuracy ~50%). To resolve this, we developed a novel deep learning architecture integrating multi-event statistics and high-dimensional latent space feature correlations, achieving 60\% overall classification accuracy, which improved to 70\% for central collisions. This method enables the extraction of subtle nuclear structure signals through statistical analysis in high-dimensional latent space, offering a new paradigm for studying initial-state nuclear properties and quark-gluon plasma characteristics in heavy-ion collisions. It overcomes the limitations of traditional single-event analysis in detecting subtle initial-state differences.

## Key claim

The paper claims that conventional deep learning methods (attention-based point cloud networks and multi-event mixing classifiers) fail to distinguish between uncorrelated and ab initio correlated nuclear configurations in heavy-ion collisions (accuracy ~50%), but a novel architecture integrating multi-event statistics and high-dimensional latent space feature correlations achieves 60% overall classification accuracy, improving to 70% for central collisions, a modest but statistically significant improvement that the abstract presents as enabling extraction of subtle nuclear structure signals.

## Method

The authors generated initial nuclear configurations for 197Au nuclei using three different nucleon-nucleon correlation schemes: uncorrelated Woods-Saxon sampling (un-corr), hard-sphere repulsion with a minimum distance of 1 fm (step corr), and ab initio nucleon-nucleon correlations (nn-corr) obtained by polynomial fitting to theoretical calculations. These configurations were used as input to the SMASH hadronic transport model to simulate 197Au+197Au collisions at sqrt(s_NN)=3 GeV, producing 500,000 collision events per configuration. A deep neural network combining point cloud convolutions with multi-head self-attention layers was designed to learn both intra-event particle correlations and inter-event statistical features from the final-state hadron momentum distributions, using 50 events combined into a single sample for classification.

## Key numbers

- Collision energy: sqrt(s_NN)=3 GeV for 197Au+197Au collisions in SMASH
- Overall classification accuracy between un-corr and nn-corr configurations: 60%
- Classification accuracy for central collisions: 70%
- Conventional attention-based point cloud networks and multi-event mixing classifiers achieved accuracy: ~50% (random guessing)
- Minimum nucleon-nucleon distance for step corr configuration: 1 fm
- Number of collision events generated per configuration: 500,000
- Number of gold nuclei in each input file: 600,000
- Maximum number of particles per event used in neural network: 700
- Number of events combined per sample: 50
- Number of final-state hadron types found in collision events: 17

## Context

This work extends prior studies of nucleon-nucleon short-range correlations in nuclear structure (Hen 2016, Fomin 2011) and their implementation in heavy-ion collision simulations (Alvioli 2009, Baym 1995). It builds on previous applications of deep learning to extract nuclear structure information from heavy-ion collisions (Pang 2019, He 2021), particularly the use of point cloud networks for inverse problems. The method enables future extraction of subtle initial-state nuclear properties, such as nucleon-nucleon correlations, that are invisible to traditional single-event observables, offering a new paradigm for studying nuclear structure and quark-gluon plasma characteristics through statistical analysis in high-dimensional latent space.
