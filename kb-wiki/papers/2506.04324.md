# Unpaired Weyl fermion on an axion string in a finite lattice

> Domain wall fermions use a $2n$-dimensional spacetime defect embedded in $(2n+1)$-dimensional spacetime to realize massless lattice Dirac fermions. Recent work has extended this idea to realize a single unpaired Weyl fermion in a finite lattice. Here, we realize the same using a $2n$-dimensional string defect embedded in $(2n+2)$-dimensional spacetime on a finite lattice for $n=1$. This string is a lattice version of the continuum axion string described in Callan-Harvey. Our results are obtained in a Hamiltonian formulation in Minkowski spacetime. Extending the results to Euclidean spacetime and to $n>1$ is straightforward. This work has applications to lattice chiral gauge theories and axion cosmology.

# Unpaired Weyl fermion on an axion string in a finite lattice

## Key claim
The authors demonstrate that a single unpaired Weyl fermion can be realized on a finite lattice using a string defect embedded in (2n+2)-dimensional spacetime, specifically for n=1, by modifying the mass profile so that a vortex and anti-vortex pair merge into a singly connected defect loop. This construction works in a Hamiltonian formulation in Minkowski spacetime, with extension to Euclidean spacetime and higher dimensions stated to be straightforward.

## Method
The authors begin with a (2+1)-dimensional Wilson fermion Hamiltonian on a finite spatial lattice with periodic boundary conditions, introducing scalar and pseudo-scalar mass terms φ₁ and φ₂ that create crossed domain walls. They first modify the mass profile to produce a single vortex-anti-vortex pair (instead of the usual four defects from periodic boundary conditions), yielding a massless Dirac fermion. They then break translation invariance in the x² direction by making the vortex-anti-vortex separation depend on x², causing the two defects to merge into a singly connected vortex loop, which hosts a single unpaired Weyl fermion. The Hamiltonian is diagonalized numerically on an N³ lattice with N=28.

## Key numbers
- Lattice size: N = 28 sites in each spatial direction, with periodic boundary conditions.
- Cutoff scale for the Dirac fermion spectrum: E ~ 0.63 in units of inverse lattice spacing.
- For the Weyl fermion construction, a right-handed Weyl-like dispersion relation is observed for energies E ≲ 0.6.
- The vortex loop defect has radius r₀ = L/4, where L = N is the lattice extent.
- The separation between vortex and anti-vortex at x² = L/2 is L/2, given by D = √((2x² - L/2)(3L/2 - 2x²)).

## Context
This work extends the extra-dimension approach to lattice chiral fermions beyond domain wall fermions (Kaplan 1992) to string defects analogous to the continuum axion string of Callan and Harvey (1985). It builds on prior lattice studies of crossed domain walls (Sen 2022) and the recent demonstration of unpaired Weyl fermions using disk-shaped domain wall defects (Kaplan 2023). The construction enables potential applications to lattice chiral gauge theories and to the phenomenology of superconducting cosmic strings, where lattice-regulated axion strings may exhibit richer behavior than continuum treatments due to parameter-dependent changes in the number of localized Weyl fermions and anomaly inflow (Sen 2022).
