# Nonlocal and Mixed Condensates Tests of Gauge/String Duality

> Using a duality approach we give an example of modeling nonlocal and mixed condensates whose behavior mimics that of QCD. Quantitatively, the estimated value of the condensate parameter $m_0^2$ is approximately $0.6-0.7 GeV^2$.

## Key claim
Using a gauge/string duality based on a five-dimensional metric with a Gaussian warp factor, the paper analytically estimates the mixed condensate parameter \(m_0^2\) to be approximately \(0.6-0.7\ \text{GeV}^2\), but notes that the result is sensitive to the renormalization scheme and the normalization condition for the nonlocal condensate \(Q(r)\).

## Method
The author computes the nonlocal quark condensate \(\Psi(x_1,x_2) = \langle \bar q q \rangle Q(r)\) via the Nambu-Goto action for a string surface in a five-dimensional Euclidean background metric \(ds^2 = R^2 (h/z^2)(dt^2 + d\vec{x}^2 + dz^2)\) with \(h = e^{\mathfrak{s} z^2}\). The function \(Q(r)\) is obtained as \(Q = e^{-S}\), where \(S\) is the renormalized area of a surface bounded by a static string profile and a straight Wilson line. The parameter \(m_0^2\) is extracted from the short-distance expansion \(Q(r) = 1 - \frac{1}{16} m_0^2 r^2 + O(r^4)\), and the long-distance behavior yields an exponential decay \(Q \sim e^{-Mr}\).

## Key numbers
* The mixed condensate parameter is estimated as \(m_0^2 \approx 0.70\ \text{GeV}^2\) using \(\mathfrak{s} \approx 0.45\ \text{GeV}^2\) and \(\mathfrak{g} \approx 0.94\) from fits to the Regge trajectory of \(\rho(n)\) mesons and the Cornell potential.
* With normalization at \(r_0 = 0.2\ \text{fm}\), the estimate becomes \(m_0^2 \approx 0.60\ \text{GeV}^2\).
* The long-distance decay constant is \(M \approx 0.15\ \text{GeV}\) under the same parameters, or \(M \approx 0.42\ \text{GeV}\) if one imposes \(M = \sqrt{\sigma}\) with string tension \(\sigma = 0.18\ \text{GeV}^2\).
* The original phenomenological QCD sum-rule estimate of \(m_0^2\) is \(0.8 \pm 0.2\ \text{GeV}^2\) (Ioffe, year not given), while lattice simulations and the field correlator method yield values near \(1\ \text{GeV}^2\), and some lattice and instanton-liquid results give \(2.4-2.5\ \text{GeV}^2\).

## Context
This work extends the author’s prior holographic estimate of the gluon condensate (Andreev, 2009) to nonlocal and mixed condensates, using the same string dual with a Gaussian warp factor that had previously described the heavy quark potential (Andreev, 2006) and the Regge slopes of \(\rho(n)\) mesons. The result for \(m_0^2\) falls within the range of the original QCD sum-rule estimate by Ioffe and is compared with lattice simulations and the instanton liquid model. The approach enables analytic computation of condensate parameters in a nonperturbative framework, though the author notes that the value is scheme-dependent and that string-breaking effects and quasiclassical corrections remain open problems.
