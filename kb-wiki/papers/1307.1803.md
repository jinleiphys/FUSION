# Self-Consistent RPA from a Coupled Cluster Wave Function Perspective

> Self-Consistent RPA is rederived in a consistent way with the help of the Coupled Cluster ground state wave function truncated at the two body level. An exact killing operator for this wave function is introduced allowing for a detailed discussion of the approximation scheme. Several exactly solvable models are reanalyzed under this new perspective giving raise to a quantitative evaluation of the performances of this many body method.

## Key claim

Self-Consistent RPA (SCRPA) can be rederived using a Coupled Cluster ground state wave function truncated at the two-body level, which provides an exact killing operator for the RPA vacuum; however, the scheme is not fully variational and retains slight Pauli principle violations due to an approximation of the two-body terms in the generalized RPA destruction operator.

## Method

The authors generalize the standard RPA destruction operator by adding specific two-body terms involving particle and hole occupation operators, then show that this operator annihilates a Coupled Cluster ground state of the form |Z⟩ = e^Z|HF⟩ with Z containing two-particle-two-hole excitations. Using this wave function, all one-body and two-body correlation functions (including occupation numbers and their fluctuations) are expressed entirely in terms of the RPA amplitudes X and Y, closing the self-consistent system. The resulting RPA matrix equations retain the same mathematical structure as standard RPA but with matrix elements evaluated using the correlated ground state rather than the Hartree-Fock state.

## Key numbers

- For the two-level Lipkin model with N=4 particles, the relative error in the ratio r = sqrt(⟨J₀²⟩)/⟨-J₀⟩ reaches a maximum of about nine percent at coupling χ=1 for N=4, decreasing for larger N and becoming exact for N=2 and N→∞.
- For the three-level Lipkin model with N=4 particles, the maximum error in excitation energies at χ=1 is about 0.5 percent.
- For the two-level Lipkin model, the occupation number ⟨J₀⟩ = -N/2 / (1 + (4/N)Y²) and the pair correlation ⟨J₊²⟩ = NXY / (1 + (4/N)Y²), where Y is the RPA amplitude.

## Context

This work extends the SCRPA approach previously developed by the authors and others (Ethofer, Duke, Roepstorff, Hirsch, Storo, Jemai, Delion, Catara 1996) by providing a rigorous wave-function foundation using Coupled Cluster Theory truncated at the two-body level, which resolves the long-standing problem of how to incorporate occupation numbers and their fluctuations self-consistently. The method enables systematic improvement over standard RPA, particularly near phase transition points where the quasi-boson approximation fails, and it reproduces exact results for the two-particle limit and the infinite-matter limit. The approach is demonstrated on the two-level Lipkin model, three-level Lipkin model, and the picket fence (pairing) model, where previous SCRPA calculations (Hirsch) are shown to be equivalent to the new formalism.
