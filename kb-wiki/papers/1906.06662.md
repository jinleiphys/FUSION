# Comparison between Variational Monte Carlo and Shell Model Calculations of Neutrinoless Double Beta Decay Matrix Elements in Light Nuclei

> Benchmark comparisons between many-body methods are performed to assess the ingredients necessary for an accurate calculation of neutrinoless double beta decay matrix elements. Shell model and variational Monte Carlo (VMC) calculations are carried out for $A=10$ and $12$ nuclei. Different variational wavefunctions are used to evaluate the uncertainties in the {\it ab initio} calculations, finding fairly small differences between the VMC double beta decay matrix elements. For shell model calculations, the role of model space truncation, radial wavefunction choices, and short-range correlation are investigated and all found to be important. Based on the detailed comparisons between the VMC and shell model approaches, we conclude that accurate descriptions of neutrinoless double beta decay matrix elements require a proper treatment of both long-range and short-range correlations.

# Comparison between Variational Monte Carlo and Shell Model Calculations of Neutrinoless Double Beta Decay Matrix Elements in Light Nuclei

## Key claim

Accurate descriptions of neutrinoless double beta decay matrix elements require a proper treatment of both long-range and short-range correlations, but the study is limited to light nuclei (A=10 and 12) that are not directly relevant experimentally, serving only as a benchmark for more approximate methods applicable to heavier nuclei.

## Method

Variational Monte Carlo (VMC) calculations using the Argonne v18 two-nucleon potential and Illinois-7 three-nucleon interaction were compared with shell model calculations using the PSDMWK Hamiltonian in p-shell and psd-shell model spaces with up to 4p4h excitations. The shell model calculations examined the effects of model space truncation, choice of harmonic oscillator versus Woods-Saxon radial wave functions, and inclusion of various short-range correlation (SRC) functions including Miller-Spencer, coupled-cluster method (CCM), and cluster variational Monte Carlo (CVMC) parameterizations.

## Key numbers

- VMC-1 result for $^{10}$Be$\rightarrow^{10}$C Fermi matrix element: -1.001(40) (dimensionless)
- VMC-1 result for $^{10}$Be$\rightarrow^{10}$C Gamow-Teller matrix element: 2.273(91) (dimensionless)
- VMC-1 result for $^{12}$Be$\rightarrow^{12}$C Fermi matrix element: -0.100(4) (dimensionless)
- VMC-1 result for $^{12}$Be$\rightarrow^{12}$C Gamow-Teller matrix element: 0.257(10) (dimensionless)
- VMC-2 result for $^{12}$Be$\rightarrow^{12}$C Fermi matrix element: -0.113(5) (dimensionless)
- VMC-2 result for $^{12}$Be$\rightarrow^{12}$C Gamow-Teller matrix element: 0.274(11) (dimensionless)
- Shell model with Woods-Saxon radial functions and psd model space (no SRC) for $^{10}$Be$\rightarrow^{10}$C Fermi: -1.100, GT: 2.783
- Shell model with Woods-Saxon radial functions and psd model space (no SRC) for $^{12}$Be$\rightarrow^{12}$C Fermi: -0.198, GT: 0.570
- Miller-Spencer SRC reduces shell model NMEs by about 20% in these light nuclei
- The first node in the $^{12}$Be$\rightarrow^{12}$C normalization density appears at about 2 fm in both shell model and VMC calculations

## Context

This work extends the VMC calculations of neutrinoless double beta decay matrix elements in A=6-12 nuclei reported by Pastore (2017) by providing detailed benchmark comparisons with shell model approaches. The study addresses the well-known factor-of-two-to-three discrepancies between different theoretical methods for heavier nuclei such as $^{48}$Ca, as seen in calculations by Menendez (2008), Senkov (2013), and others using shell model, energy density functionals, QRPA (Simkovic, 2013), and the interacting boson model (Barea, 2015). The results enable improved understanding of which nuclear structure ingredients are essential when extending shell model calculations to experimentally relevant isotopes like $^{76}$Ge, $^{130}$Te, and $^{136}$Xe, where current experimental half-life limits exceed $8\cdot10^{25}$ yr, $1.5\cdot10^{25}$ yr, and $1.1\cdot10^{26}$ yr respectively.
