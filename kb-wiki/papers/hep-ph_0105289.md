# Chiral corrections to baryon properties with composite pions

> A calculational scheme is developed to evaluate chiral corrections to properties of composite baryons with composite pions. The composite baryons and pions are bound states derived from a microscopic chiral quark model. The model is amenable to standard many-body techniques such as the BCS and RPA formalisms. An effective chiral model involving only hadronic degrees of freedom is derived from the macroscopic quark model by projection onto hadron states. Chiral loops are calculated using the effective hadronic Hamiltonian. A simple microscopic confining interaction is used to illustrate the derivation of the pion-nucleon form factor and the calculation of pionic self-energy corrections to the nucleon and Delta(1232) masses.

## Key claim

The paper develops a calculational scheme for evaluating chiral corrections to baryon properties (specifically the nucleon and Delta(1232) masses) where both the baryons and the pions are composite bound states derived from the same underlying microscopic chiral quark model, rather than treating the pion as an elementary point-like particle as in earlier hybrid approaches.

## Method

The authors start from a chiral quark Hamiltonian with a confining four-fermion interaction and solve for the vacuum using a Bogoliubov-Valatin transformation (BCS formalism) to obtain the chiral angle. Pion states are constructed via the random-phase approximation (RPA) as quark-antiquark bound states with both positive- and negative-energy Salpeter components, while baryon states are obtained variationally from the BCS vacuum. An effective hadronic Hamiltonian is derived by projecting the microscopic quark Hamiltonian onto single-pion and single-baryon states, yielding a pion-baryon vertex function that incorporates the composite nature of both particles. Chiral loop corrections to the nucleon and Delta(1232) masses are then calculated using time-ordered perturbation theory with this effective Hamiltonian.

## Key numbers

* The strength of the confining potential is set to \(K_0 = 247\) MeV.
* The variational parameter for the baryon Gaussian wave function is \(\alpha_N = 1.2 K_0\) (approximately 296 MeV), and the same value is used for the Delta(1232).
* The leading nonanalytic contribution to the nucleon mass from chiral loops is given by \(M_N^{LNA} = -\frac{3}{16\pi^2 f_\pi^2} g_A^2 m_\pi^2\), matching the prediction from chiral perturbation theory.

## Context

This work extends the authors' previous studies of a chiral quark model with confinement and dynamical chiral symmetry breaking, which had been applied to meson spectroscopy (LeYaouanc et al., 1980s) and baryon structure (Bicudo et al., 1999). It differs from earlier hybrid approaches such as the chiral bag model (Brown and Rho, 1979; Vento et al., 1980; Jaffe, 1979) by treating the pion as a composite quark-antiquark bound state rather than an elementary field. The scheme enables future calculations of baryon-meson and baryon-baryon interactions using the same microscopic quark dynamics, and provides a framework for systematically including chiral loop corrections in composite hadron models.
