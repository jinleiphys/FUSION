# The $ηNN$ coupling constant

> We derive the light cone QCD sum rule for the $ηNN$ coupling constant $g_{ηNN} $. The contribution from the excited states and the continuum is subtracted cleanly through the double Borel transform with respect to the two external momenta, $p_1^2$, $p_2^2=(p-q)^2$. Our result is $α_{ηNN}=(0.3\pm 0.15)$, which favors small values used in literatures.

## Key claim

The light cone QCD sum rule calculation yields a small value for the eta-nucleon coupling constant, \(\alpha_{\eta NN} = 0.3 \pm 0.15\), which arises from a strong cancellation between the leading twist-two term and the mass-correction terms, a feature specific to the eta meson due to its relatively large mass and isoscalar nature.

## Method

The author derived a light cone QCD sum rule for the \(\eta NN\) coupling constant by analyzing a two-point correlation function with an interpolating nucleon current and an external eta meson state. The contribution from excited states and the continuum was subtracted using a double Borel transform with respect to the two external momenta. The sum rule was evaluated at the symmetric point \(u_0 = 1/2\) using eta meson distribution amplitudes up to twist four, and the result was normalized by dividing by the Ioffe mass sum rule for the nucleon to reduce uncertainty from the nucleon overlap amplitude.

## Key numbers

*   \(\alpha_{\eta NN} = 0.3 \pm 0.15\) (dimensionless), the central result for the eta-nucleon coupling constant defined as \(\alpha_{\eta NN} = g^2_{\eta NN} / 4\pi\).
*   \(g_{\eta NN} = 1.7 \pm 0.3\) (dimensionless), obtained using a quark condensate value \(a = 0.67\ \text{GeV}^3\) (corresponding to \(\langle \bar q q \rangle = -(240 \pm 10)^3\ \text{MeV}^3\)).
*   \(g_{\eta NN} = 2.1 \pm 0.3\) (dimensionless), obtained using a smaller quark condensate value \(a = 0.55\ \text{GeV}^3\) (corresponding to \(\langle \bar q q \rangle = -(225 \pm 10)^3\ \text{MeV}^3\)).
*   The Borel parameter working interval was \(0.9\ \text{GeV}^2 \leq M_B^2 \leq 1.8\ \text{GeV}^2\), with a continuum threshold \(s_0 = 2.25\ \text{GeV}^2\).
*   Input parameters include the eta decay constant \(f_\eta = 0.133 \pm 0.01\ \text{GeV}\) and the parameter \(\mu_\eta = 2.13\ \text{GeV}\) at the scale \(\mu = 1\ \text{GeV}\).

## Context

This work extends the author's previous light cone QCD sum rule calculations for the \(\pi NN(1535)\) coupling constant (Zhu, 1999) and the \(\rho NN\) and \(\omega NN\) vector meson couplings (Zhu, 1999) to the poorly known \(\eta NN\) sector. The result favors small values of \(\alpha_{\eta NN}\) (below 1), consistent with constraints from dispersion relation analyses (Grein, 1978) and chiral perturbation theory (ChPT) which require \(\alpha_{\eta NN}\) in the range 0.32–0.53, but contradicts the larger values (3–7) often used in one-boson-exchange potential fits. The calculation enables a theoretically grounded input for analyzing future eta meson photo- and electro-production experiments.
