# Multinucleon transfer mechanism in ${}^{250}\text{Cf}+{}^{232}\text{Th}$

> Production cross-sections of heavy neutron-rich isotopes are calculated by employing quantal transport description in ${}^{250}\text{Cf}+{}^{232}\text{Th}$ collisions. This quantal transport description is based on the stochastic mean-field (SMF) approach, and it provides a microscopic approach beyond time-dependent Hartree-Fock (TDHF) theory to include mean-field fluctuations. De-excitation of primary fragments is determined by employing the statistical GEMINI++ code. Calculations provide predictions for production cross-sections of neutron rich transfermium isotopes without any adjustable parameters.

## Key claim
The paper claims that production cross-sections for neutron-rich transfermium isotopes can be predicted using a quantal transport description based on the stochastic mean-field (SMF) approach without any adjustable parameters, but only for the specific system ${}^{250}\text{Cf}+{}^{232}\text{Th}$ at $E_{\text{c.m.}}=950$ MeV and only for the four specific collision geometries (tip-tip, tip-side, side-tip, side-side) that were simulated.

## Method
The authors performed time-dependent Hartree-Fock (TDHF) calculations using the SLy4d Skyrme energy density functional for ${}^{250}\text{Cf}+{}^{232}\text{Th}$ collisions at $E_{\text{c.m.}}=950$ MeV across four initial collision geometries with orbital angular momenta in steps of $40\hbar$. They then applied the stochastic mean-field (SMF) approach to derive quantal diffusion coefficients and coupled Langevin equations for neutron and proton number fluctuations, using the TDHF mean-field evolution to determine drift paths and potential energy surfaces. De-excitation of primary fragments was handled with the statistical GEMINI++ code.

## Key numbers
- Bombarding energy: $E_{\text{c.m.}}=950$ MeV for the ${}^{250}\text{Cf}+{}^{232}\text{Th}$ system
- Reduced isoscalar curvature parameters: $\beta(\text{tt})=0.005$ for tip-tip, $\beta(\text{ts})=0.009$ for tip-side, $\beta(\text{st})=0.009$ for side-tip, $\beta(\text{ss})=0.004$ for side-side geometries
- Reduced isovector curvature parameters: $\alpha(\text{tt})=0.16$ for tip-tip, $\alpha(\text{ts})=0.15$ for tip-side, $\alpha(\text{st})=0.12$ for side-tip, $\alpha(\text{ss})=0.11$ for side-side geometries
- For head-on tip-tip collision ($\ell_i=0$): final Cf-like fragment $A_1^f=230.7$, $Z_1^f=89.9$; final Th-like fragment $A_2^f=251.3$, $Z_2^f=98.1$; total kinetic energy TKE $=612.8$ MeV; total excitation energy $E^*=335.9$ MeV
- For head-on side-side collision ($\ell_i=0$): final Cf-like fragment $A_1^f=250.0$, $Z_1^f=97.4$; final Th-like fragment $A_2^f=232.0$, $Z_2^f=90.6$; TKE $=620.0$ MeV; $E^*=327.7$ MeV
- Neutron dispersion $\sigma_{NN}$ ranges from 5.6 to 9.3 across geometries for head-on collisions; proton dispersion $\sigma_{ZZ}$ ranges from 3.7 to 6.1; mass dispersion $\sigma_{AA}$ ranges from 8.5 to 14.7
- The isoscalar path makes an angle $\phi = 32^{\circ}$ with respect to the horizontal neutron axis

## Context
This work extends the TDHF-based analysis of multinucleon transfer by Kedziora and Simenel (2010), who investigated the same reaction system using only the mean-field approximation without fluctuations. The SMF approach provides a microscopic method to include fluctuations beyond TDHF, which previously could only describe the most probable dynamical path. The calculations enable predictions for production cross-sections of neutron-rich transfermium isotopes, offering a parameter-free microscopic test for phenomenological models such as the multidimensional Langevin model of Zagrebaev (2006, 2008, 2011) and the di-nuclear system model of Feng (2009, 2017).
