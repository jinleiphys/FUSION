# SimpleTensor -- a user-friendly Mathematica package for elementary tensor and differential-geometric calculations

> In this paper we present a short overview of the new Wolfram Mathematica package intended for elementary "in-basis" tensor and differential-geometric calculations. In contrast to alternatives our package is designed to be easy-to-use, short, all-purpose, and hackable. It supports tensor contractions using Einstein notation, transformations between different bases, tensor derivative operator, expansion in basis vectors and forms, exterior derivative, and interior product.

## Key claim

SimpleTensor is presented as a user-friendly, concise, and hackable Mathematica package for elementary in-basis tensor and differential-geometric calculations, deliberately sacrificing large-scale performance and specialized functionality in favor of clarity and accessibility for students and scientists.

## Method

The package provides 11 public functions operating on tensors explicitly defined by their components in coordinate systems within user-defined spaces. It supports tensor contractions using Einstein notation, basis transformations via a metric tensor, a tensor derivative operator, and differential-geometric operations including exterior derivative, wedge product, and interior product, all evaluated through the central `GetArray` command.

## Key numbers

* The package contains only 11 public functions.
* The package was released after 12 versions and 33 years of Wolfram Mathematica development.
* In the Minkowski space example, the metric is set to `DiagonalMatrix[{1, -1, -1, -1}]`.
* The Levi-Civita tensor is defined using `LeviCivitaTensor[4]`.
* The boost transformation matrix uses a velocity \(v\) with factor \(1/\sqrt{1-v^2}\).
* The trace identity \(\text{tr}(\gamma^\mu\gamma^\nu) = 4 g^{\mu\nu}\) is verified.
* The Dirac equation determinant yields a doubly-degenerate dispersion \(p_0 = \pm \sqrt{m^2 + p_1^2 + p_2^2 + p_3^2}\).

## Context

This work extends the functionality of base Mathematica, which lacks native support for tensor calculus and differential geometry, by providing an alternative to existing packages such as xAct, TensoriaCalc, grt, and Ricci (all cited in the paper). The author notes these alternatives are often outdated, hard to master, focused on General Relativity, or difficult to adapt. SimpleTensor aims to fill this gap for elementary calculations, enabling students and researchers to perform relativistic electromagnetism, spinor algebra, and basic differential-geometric computations without large complexity overhead.
