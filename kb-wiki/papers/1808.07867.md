# $e^+ e^-$ angularity distributions at NNLL$^\prime$ accuracy

> We present predictions for the $e^{+}e^{-}$ event shape angularities at NNLL$^{\prime}$ resummed and $\mathcal{O}(α_s^{2})$ matched accuracy and compare them to LEP data at center-of-mass energies $Q=91.2$ GeV and $Q=197$ GeV. We perform the resummation within the framework of Soft-Collinear Effective Theory, and make use of recent results for the two-loop angularity soft function. We determine the remaining NNLL$^{\prime}$ and $\mathcal{O}(α_s^2)$ ingredients from a fit to the EVENT2 generator, and implement a shape function with a renormalon-free gap parameter to model non-perturbative effects. Using values of the strong coupling $α_s(m_Z)$ and the universal non-perturbative shift parameter $Ω_1$ that are consistent with those obtained in previous fits to the thrust and $C$-parameter distributions, we find excellent agreement between our predictions and the LEP data for all angularities with $a\in[-1,0.5]$. This provides a robust test of the predictions of QCD, factorization, and the universal scaling of the non-perturbative shift across different angularities. Promisingly, our results indicate that current degeneracies in the $\{α_s(m_Z),Ω_1\}$ parameter space could be be alleviated upon fitting these parameters to experimental data for the angularity distributions.

## Key claim

The paper presents NNLL' resummed and O(alpha_s^2) matched predictions for e+e- angularity distributions that show excellent agreement with LEP data for angularities with a in [-1,0.5], using values of alpha_s(m_Z) and the non-perturbative shift parameter Omega_1 consistent with previous fits to thrust and C-parameter, but the authors note that current degeneracies in the {alpha_s(m_Z), Omega_1} parameter space could potentially be alleviated by fitting these parameters to angularity data rather than having definitively resolved them.

## Method

The authors perform resummation within Soft-Collinear Effective Theory (SCET) using recent two-loop angularity soft function results, determine the remaining NNLL' and O(alpha_s^2) ingredients from fits to the EVENT2 generator, and implement a shape function with a renormalon-free gap parameter to model non-perturbative effects. They extract the two-loop jet function constant c_J^2 by comparing EVENT2 output to the singular SCET prediction, using the known total cross section to determine the singular constants via the relation sigma_tot = A + r_c(1).

## Key numbers

- Center-of-mass energies: Q = 91.2 GeV and Q = 197 GeV
- Angularity parameter range studied: a in [-1, 0.5]
- Maximum four-particle angularity for a=0: tau_0^{2,max} = 0.423 (compared to three-particle maximum 0.333)
- Maximum four-particle angularity for a=-1: tau_{-1}^{2,max} = 0.219
- Maximum four-particle angularity for a=0.5: tau_{0.5}^{2,max} = 0.587
- Non-perturbative shift parameter used in test: Omega_1(R_Delta,R_Delta) = 0.2 GeV
- Strong coupling used in test: alpha_s(m_Z) = 0.118
- EVENT2 runs used: 1e11 events with cutoff 10^{-12} for O(alpha_s) and 4e10 events with cutoff 10^{-12} for O(alpha_s^2)
- Extracted r_2^{CF} values range from -0.16 (a=-1.0) to -49.9 (a=0.5)
- Extracted r_2^{CA} values range from -42.3 (a=-1.0) to 18.1 (a=0.5)
- Extracted r_2^{nf} values range from 6.76 (a=-1.0) to 4.92 (a=-0.25)

## Context

This work extends previous SCET-based resummation of thrust and C-parameter by Abbate et al. (2010, 2012) and Hoang et al. (2015) to the full family of angularities, enabled by recent two-loop soft function calculations by Bell et al. (2015, 2018). The results enable future extractions of alpha_s and Omega_1 from angularity data at a single center-of-mass energy but different a values, which could discriminate between these parameters similarly to varying Q, and the authors note that angularities for -2 <= a <= 0.5 exhibit a factor of six variance in the non-perturbative shift, equivalent to measurements between Q=35 GeV and Q=207 GeV for thrust.
