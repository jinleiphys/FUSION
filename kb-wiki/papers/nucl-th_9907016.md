# Probing the width of compound states with rotational gamma rays

> The intrinsic width of (multiparticle-multihole) compound states is an elusive quantity, of difficult direct access, as it is masked by damping mechanisms which control the collective response of nuclei. Through microscopic cranked shell model calculations, it is found that the strength function associated with two-dimensional gamma-coincidence spectra arising from rotational transitions between states lying at energies >1 MeV above the yrast line, exhibits a two-component structure controlled by the rotational (wide component) and compound (narrow component) damping width. This last component is found to be directly related to the width of the multiparticle-multihole autocorrelation function.

## Key claim

The paper argues that the intrinsic width of compound states (multiparticle-multihole configurations) can be probed indirectly through the narrow component of two-dimensional gamma-coincidence spectra from rotational transitions, provided one uses the correlational spreading width (derived from the autocorrelation function) rather than the conventional width from the averaged strength function, which is inflated by energy shifts of individual states.

## Method

Microscopic cranked shell model calculations were performed for the rare-earth nucleus 168Yb, diagonalizing a Hamiltonian consisting of the cranked Nilsson mean-field and a surface-delta residual interaction using the lowest 2000 many-particle many-hole configurations for each angular momentum and parity. The resulting compound states were used to compute zero-step strength functions and autocorrelation functions for the np-nh configurations, as well as one-step and two-step E2 strength functions for gamma transitions between compound states at different angular momenta. The relation between the narrow component width of the two-step function and the correlational spreading width was tested for different residual interactions (surface-delta, volume delta, and random two-body interaction).

## Key numbers

* The correlational spreading width extracted from the autocorrelation function for np-nh states at I=40+ is Gamma_mu^corr = 41 keV, about a factor four smaller than the width from the averaged strength function Gamma_mu^s.
* For an extended basis of 6000 np-nh states at excitation energy U ≈ 3 MeV and I=40,41, Gamma_mu^corr = 133 keV and Gamma_mu^s = 305 keV, maintaining a factor of about 2 difference.
* The narrow component width in the two-step E2 strength function satisfies Gamma_narrow ≈ (2-3) × Gamma_mu, and the intensity of the narrow component follows Inarrow ≈ 2D/Gamma_mu for Gaussian or D/Gamma_mu for Breit-Wigner distributions.
* The root-mean-square value of the surface-delta interaction two-body matrix elements is sqrt(<v_ijkl^2>) = 19 keV, with large matrix elements (|v_ijkl| > 60 keV) comprising only 2% of the total but contributing significantly to the r.m.s. value.

## Context

This work extends earlier studies of rotational damping and spectral fluctuations in cranked shell models (Matsuo 1993, 1997) and provides a theoretical framework for interpreting experimental two-dimensional gamma-coincidence spectra from the quasi-continuum of 168Yb. The results contradict the assumption that the spreading width from the averaged strength function (as defined in standard textbooks like Bohr and Mottelson) is the relevant quantity for compound state decay, showing instead that the correlational width from the autocorrelation function is physically meaningful. The paper enables experimental extraction of compound damping widths from unresolved gamma-ray spectra, building on earlier experimental hints of two-component structure (Leoni 1999) and suggesting techniques involving fluctuation analysis and higher-dimensional spectra.
