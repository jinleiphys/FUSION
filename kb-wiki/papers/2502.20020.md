# The $η^\prime N$ interaction from the $η^\prime p$ correlation function

> We evaluate for the first time the $η^\prime p$ femtoscopic correlation function to study the $η^\prime N$ interaction. We find it extremely sensitive to the value of the $η^\prime p$ scattering length, for which at present there exists only very limited information, not even knowing its sign. The measurement of this correlation function would provide much valuable information on the $η^\prime N$ interaction, which could then also be used to settle the issue of possible $η^\prime$ nucleus bound states, an issue attracting much attention in the nuclear physics community.

# The $\eta^\prime N$ interaction from the $\eta^\prime p$ correlation function

## Key claim
The $\eta^\prime p$ femtoscopic correlation function is extremely sensitive to the value of the $\eta^\prime p$ scattering length, whose sign is not yet known experimentally, and measuring this correlation function could determine both the sign and magnitude of the scattering length, thereby settling whether $\eta^\prime$ nucleus bound states exist. The effect of inelastic coupled channels on the correlation function is found to be extremely small, so the elastic channel alone provides the essential information.

## Method
The authors compute the $\eta^\prime p$ femtoscopic correlation function using the Koonin-Pratt formalism with a coupled-channel Bethe-Salpeter approach. The coupled channels considered are $K^0 \Sigma^+$, $K^+ \Sigma^0$, $K^+ \Lambda$, $\eta p$, and $\eta^\prime p$, with the interaction kernel taken from chiral perturbation theory including an additional singlet-meson coupling parameter $\alpha$. The scattering matrix $T$ is obtained by solving the Bethe-Salpeter equation with a cutoff regularization ($q_{\rm max}=630$ MeV), and the correlation function is evaluated for a Gaussian source of size $R=1$ fm, typical of $pp$ collisions.

## Key numbers
- The $\eta^\prime p$ scattering length for $\alpha = 4.00$: $a_{\eta^\prime p} = -0.406 + i\,0.007$ fm (repulsive), with $|a_{\eta^\prime p}| = 0.406$ fm.
- The $\eta^\prime p$ scattering length for $\alpha = -0.55$: $a_{\eta^\prime p} = +0.464 + i\,0.087$ fm (attractive), with $|a_{\eta^\prime p}| = 0.472$ fm.
- The $\eta^\prime p$ scattering length for $\alpha = 0.00$: $a_{\eta^\prime p} = -0.002 + i\,0.020$ fm, with $|a_{\eta^\prime p}| = 0.020$ fm.
- The experimental result from Anisovich (2018) gives $|a_{\eta^\prime p}| = (0.403 \pm 0.020 \pm 0.060)$ fm from $\gamma p \to \eta^\prime p$ analysis.
- The experimental result from Czerwinski (2014) gives $a_{\eta^\prime p} = 0^{+0.43}_{-0.43} + i\,0.37^{+0.40}_{-0.16}$ fm from $pp \to pp\eta^\prime$ at COSY-11.
- The cutoff variation $q_{\rm max} = 637 \pm 72$ MeV is used to estimate uncertainties in the correlation function.
- The inelastic channel contributions modify the correlation function by less than about 10% even with production weights as large as 10.

## Context
This work extends the femtoscopic technique, previously applied to $\eta N$ correlation functions for studying the $N^*(1535)$ resonance (Molina 2023), to the $\eta^\prime N$ system for the first time. The $\eta^\prime N$ interaction is poorly constrained: the chiral coupled-channel approach of Oset (2010) and Bruns (2019) gives repulsive scattering lengths near $-0.41$ fm, while the linear sigma model of Sakai (2013, 2016) predicts an attractive $+0.85$ fm. The present calculation shows that measuring the $\eta^\prime p$ correlation function can discriminate between these scenarios and resolve the sign ambiguity, which is essential for predicting whether $\eta^\prime$ mesic nuclei exist as bound states. The work enables experimental searches at facilities such as those analyzing $pp$ collision data from ALICE or STAR.
