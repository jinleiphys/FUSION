# Single Transverse-Spin Asymmetry and Sivers Function in Large Momentum Effective Theory

> We apply recent developments in large momentum effective theory (LaMET) to formulate a non-perturbative calculation of the single-transverse spin asymmetry in terms of the quasi transverse-momentum-dependent quark distribution functions from the so-called Sivers mechanism. When the spin asymmetry is defined as the ratio of the quark Sivers function over the spin averaged distribution, it can be directly calculated in terms of the relevant quasi distributions with the soft functions and perturbative matching kernels cancelling out. Apart from the general formula presented, we have verified the result in the small transverse distance limit at one-loop order, which reduces to a collinear expansion at twist-three level.

## Key claim

The authors formulate a non-perturbative calculation of the single transverse-spin asymmetry via the quark Sivers function using large momentum effective theory (LaMET), and they verify that the matching kernel between the quasi-Sivers function and the physical Sivers function is independent of the spin structure, reducing to the same kernel as for unpolarized distributions at one-loop order in the small transverse distance limit.

## Method

The authors define a quasi-TMDPDF for quarks in a proton moving along the +z direction, using a staple-shaped gauge link with finite length L and a subtraction factor Z_E to remove pinch-pole singularities. They derive the LaMET expansion formula relating the quasi-Sivers function to the physical Sivers function via a perturbative hard kernel H, a reduced soft function S_r, and the Collins-Soper kernel K. The one-loop calculation of the quasi-Sivers function is performed in the collinear twist-three framework using the Qiu-Sterman matrix element T_F(x,x), with the soft-gluon pole and hard-gluon pole contributions evaluated via cut diagram approximation and collinear k_{g‚ä•}-expansion.

## Context

This work extends the LaMET framework, previously applied to unpolarized TMDPDFs (Ji 2018, Ebert 2019, 2020) and k_‚ä•-even spin-dependent TMDPDFs (Ebert 2020), to the k_‚ä•-odd quark Sivers function. The matching kernel is shown to be identical to that for the unpolarized case, confirming the spin-independence argument of Ebert (2020). The one-loop result reduces to the collinear twist-three expansion at small b_‚ä•, connecting to the Qiu-Sterman matrix element formalism (Qiu 1991, 1998) and prior calculations of the light-cone Sivers function (Sun 2013, Scimemi 2019). This enables future lattice QCD calculations of the Sivers function via Euclidean quasi-distributions.
