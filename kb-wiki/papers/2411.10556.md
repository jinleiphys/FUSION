# Star Log-extended eMulation: a method for efficient computation of the Tolman-Oppenheimer-Volkoff equations

> We emulate the Tolman-Oppenheimer-Volkoff (TOV) equations, including tidal deformability, for neutron stars using a new method based upon the Dynamic Mode Decomposition (DMD). This method, which we call Star Log-extended eMulation (SLM), utilizes the underlying logarithmic behavior of the differential equations to enable accurate emulation of the nonlinear system. We show predictions for well-known equations of state (EOSs) with fixed parameters using the SLM, accurately recreating high-fidelity results while achieving a computational speed-up of $\approx 2.4 \times 10^4$. We test our parametric SLM method for a two-parameter quarkyonic EOS against high-fidelity RK4 TOV calculations and find a computational speedup of $\approx 7.0 \times 10^4$. Hence, SLM is an efficient emulator for the numerous TOV evaluations required by multi-messenger astrophysical frameworks that infer constraints on the EOS. The ability of the SLM algorithm to learn a mapping between parameters of the EOS and subsequent neutron star properties also opens up potential extensions for assisting in computationally prohibitive uncertainty quantification (UQ) for any type of EOS. The source code for the methods employed in this work is openly available in a public GitHub repository for community modification and use.

## Key claim

The Star Log-extended eMulation (SLM) method accurately emulates the coupled TOV and tidal deformability equations for neutron stars by exploiting the underlying logarithmic behavior of the differential equations, achieving computational speed-ups of approximately \(2.4 \times 10^4\) for fixed-parameter equations of state and approximately \(7.0 \times 10^4\) for a parametric two-parameter quarkyonic EOS, though the method's accuracy depends on the specific EOS and the number of training snapshots used.

## Method

The authors apply Dynamic Mode Decomposition (DMD) to snapshot matrices of the TOV solutions, first taking the logarithm of the snapshots and then extending them via Hadamard products to handle the nonlinearity of the coupled system. For the parametric version (pSLM), they perform individual SLM emulations at several training parameter sets and interpolate the resulting eigen-pairs to new test parameters using the B-GRIM interpolation method. The emulated solutions are then exponentiated to recover the original physical quantities.

## Key numbers

- Computational speed-up of approximately \(2.4 \times 10^4\) for fixed-parameter tabular EOSs using SLM, compared to the high-fidelity RK4 solver.
- Computational speed-up of approximately \(7.0 \times 10^4\) for the parametric two-parameter quarkyonic EOS using pSLM, compared to the high-fidelity RK4 solver.
- For the tabular EOS SLy4, the SLM emulated maximum mass is 2.066 \(M_\odot\) (0.04% relative error) and the corresponding radius is 10.031 km (0.01% relative error), compared to the HF values of 2.067 \(M_\odot\) and 10.033 km.
- For the tabular EOS APR, the SLM emulated maximum mass is 2.193 \(M_\odot\) (0.01% relative error) and the corresponding radius is 9.981 km (0.02% relative error), compared to the HF values of 2.193 \(M_\odot\) and 9.979 km.
- For the quarkyonic EOS with parameters \(\Lambda = 300.00\), \(\kappa = 0.26\), the pSLM emulated maximum mass is 2.658 \(M_\odot\) (0.30% relative error) and the corresponding radius is 12.642 km (0.22% relative error), compared to the HF values of 2.666 \(M_\odot\) and 12.614 km.
- Average SLM computation time for tabular EOSs is approximately \(4.6 \times 10^{-4}\) seconds, compared to the HF average of approximately 11 seconds.
- The SLM routine used 50 total snapshots for the tabular EOS emulations and chose \(\eta = 0.9999\) in the singular value selection criterion.

## Context

This work extends the application of DMD-based emulators, previously used in fluid dynamics and nuclear scattering (Drischler 2022, Bonilla 2022, Maldonado 2024), to the nonlinear TOV equations for neutron stars, which are computationally expensive to solve repeatedly in Bayesian inference frameworks for multi-messenger astrophysics (LIGO 2017, Pang 2022). The SLM method enables efficient computation of mass-radius and tidal deformability curves, making it suitable for large-scale uncertainty quantification and EOS inference frameworks that require thousands of TOV evaluations, such as those used by LIGO and NICER collaborations.
