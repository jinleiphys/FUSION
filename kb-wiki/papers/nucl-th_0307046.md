# An importance sampling algorithm for generating exact eigenstates of the nuclear Hamiltonian

> We endow a recently devised algorithm for generating exact eigensolutions of large matrices with an importance sampling, which is in control of the extent and accuracy of the truncation of their dimensions. We made several tests on typical nuclei using a correlated basis obtained from partitioning the shell model space. The sampling so implemented allows not only for a substantial reduction of the shell model space but also for an extrapolation to exact eigenvalues and E2 strengths.

## Key claim

The paper presents an importance sampling algorithm that, when combined with a correlated basis obtained from partitioning the shell model space, allows for a drastic truncation of the Hamiltonian matrix dimensions while still enabling extrapolation to exact eigenvalues and E2 strengths; the method is claimed to be effective specifically when eigenvectors are highly localized, a condition achieved through the multipartitioning approach.

## Method

The authors endow a previously developed iterative diagonalization algorithm (Andreozzi 2002) with an importance sampling procedure that selects only basis states relevant to the eigensolutions. The sampling is implemented using a similarity transformation and a decoupling condition that yields a dispersion relation, solved via the Newton method, to determine whether a new basis state should be accepted based on a threshold parameter epsilon. The shell model space is first partitioned into subspaces, and a correlated basis is generated by solving eigenvalue equations for the Hamiltonian components in each subspace, then used as input for the importance sampling.

## Key numbers

* For $^{108}$Sn, the full dimension N=17467 for the $2^+$ states; using the sampling, a truncated matrix of dimension n=1867 yielded an approximate energy of -3.120 MeV for the $2^+_1$ state, with an extrapolated value of -3.143(001) MeV compared to the exact value of -3.151 MeV.
* For $^{48}$Cr, the full dimension N=41355 for the $0^+$ states; a truncated matrix of dimension n=5739 gave an approximate energy of -32.851 MeV for the $0^+_1$ state, with an extrapolated value of -32.955(019) MeV compared to the exact value of -32.954 MeV.
* For $^{133}$Xe, the full dimension N=125756 for the $1/2^+$ states; a truncated matrix of dimension n=6763 gave an approximate energy of -11.256 MeV for the $1/2^+_1$ state, with an extrapolated value of -11.279(002) MeV.
* The $B(E2)$ strength for the $2^+_1 \rightarrow 0^+_1$ transition in $^{108}$Sn was 42.45 $e^2 fm^4$ at epsilon=1x10$^{-4}$, with an extrapolated value of 42.68(02) $e^2 fm^4$ compared to the exact value of 42.71 $e^2 fm^4$.
* The $B(E2)$ strength for the $2^+_1 \rightarrow 0^+_1$ transition in $^{48}$Cr was 226.5 $e^2 fm^4$ at epsilon=4x10$^{-5}$, with an extrapolated value of 227.8(8) $e^2 fm^4$ compared to the exact value of 228 $e^2 fm^4$.

## Context

This work extends the iterative diagonalization algorithm of Andreozzi (2002) by adding an importance sampling capability, and builds upon the multipartitioning method of Andreozzi and Porrino (2001) to generate a correlated basis. The approach contrasts with the Monte Carlo technique (Johnson 1992, Ormand 1994, White 2000) which suffers from the minus-sign problem, and with the stochastic truncation method of Horoi, Brown, and Zelevinsky (1999, 2003) which proposed a different exponential extrapolation law. The method enables the study of heavier nuclei with neutron excess, such as $^{133}$Xe, and is planned for extension to include n-particle n-hole correlated configurations for studying intruder states.
