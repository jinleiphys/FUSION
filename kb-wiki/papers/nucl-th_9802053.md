# The nucleon wave function in light-front dynamics

> The general spin structure of the relativistic nucleon wave function in the $3q$-model is found. It contains 16 spin components, in contrast to 8 ones known previously, since in a many-body system the parity conservation does not reduce the number of the components. The explicitly covariant form of the wave function automatically takes into account the relativistic spin rotations, without introducing any Melosh rotation matrices. It also reduces the calculations to the standard routine of the Dirac matrices and of the trace techniques. In examples of the proton magnetic moment and of the axial nucleon form factor, with a particular wave function, we reproduce the results of the standard approach. Calculations beyond the standard assumptions give different results.

## Key claim

The relativistic nucleon wave function in the 3-quark model has 16 spin components, not 8 as previously thought, because in a many-body system parity conservation does not reduce the number of components; the extra eight components are necessary to represent even symmetric S-wave spin structure in an arbitrary reference frame and are not a peculiarity of the covariant formulation of light-front dynamics.

## Method

The author constructs the general spin structure of the nucleon wave function in the explicitly covariant version of light-front dynamics, using the light-front plane defined by the invariant equation ω·x=0 with ω²=0. The wave function is expressed in terms of Dirac spinors and projection operators, and the 16 spin structures are found by constructing three-dimensional spin matrices that transform like vectors under rotations and coincide with Pauli matrices in the center-of-mass system. The permutation symmetry of the three-quark system is then imposed to obtain the fully symmetric nucleon wave function.

## Key numbers

- The nucleon wave function contains **16 spin components**, in contrast to the **8** previously known from Dzhibuti (1988)
- The number of components follows from 2×2×2×2=16 for the nucleon
- In the nonrelativistic limit, the wave function reduces to **8** structures (items 1-5, 8, and two n-independent structures from items 11-16 of the basis)
- For the deuteron, the n-dependent component f₅ dominates starting at **q=0.5 GeV/c**

## Context

This work extends the earlier eight-component spin structure found by Dzhibuti (1988) for the nucleon wave function in the 3-quark model, showing that parity conservation does not reduce the number of components in a many-body system as noted by Kolyb (1965). The covariant approach automatically incorporates relativistic spin rotation effects without introducing Melosh rotation matrices, and reproduces standard results for the proton magnetic moment and axial nucleon form factor when using a particular wave function. The extra eight components should appear in any light-front approach, not just the covariant formulation, and enable calculations beyond the standard assumptions that give different results.
