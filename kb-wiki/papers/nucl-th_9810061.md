# Single Particle and Collective Structure for Nuclei near Sn-132

> A new Nilsson single-particle structure is proposed for neutron-rich nuclei near $^{132}$Sn. In general, a large reduction in spin-orbit interaction is required and the neutron $N = 82$ gap persists in the new set of parameters. The ground state deformations for several isotopic chains are studied with this set and compared with the results of the standard set and with measured ones. Collective bands in two even-even, neutron-rich nuclei are calculated using the Projected Shell Model with the new set of parameters and improved agreement with existing data is found.

# Single Particle and Collective Structure for Nuclei near Sn-132

## Key claim
A new set of Nilsson model parameters (κ and μ) is proposed for neutron-rich nuclei near ¹³²Sn, requiring a large reduction in spin-orbit interaction strength compared to the standard stability-valley parameters. The new set reproduces available single-particle data and improves agreement with collective band calculations, but the authors caution that the parameterization is on firmer ground for nuclei with proton number beyond 50 and should be employed with confidence only for Z = 46–56 and N = 68–96.

## Method
The authors adjusted the Nilsson model parameters κ and μ separately for each major oscillator shell (N = 3, 4, 5) for both protons and neutrons to reproduce experimental single-particle levels in nuclei near ¹³²Sn. They then used these new parameters in Nilsson-Strutinsky-BCS calculations to obtain ground-state deformations for cadmium, tellurium, and xenon isotopic chains, and employed the Projected Shell Model (PSM) with a three-major-shell basis to calculate yrast bands for ¹³⁶Te and ¹⁴²Xe, comparing results against those obtained with the standard Nilsson parameter set.

## Key numbers
- The 3⁻ state in ¹³²Sn lies at 4.3512 MeV, compared to 2.6146 MeV in ²⁰⁸Pb.
- The g₇/₂ orbital is required by data to be about 800 keV below the d₅/₂ orbital.
- The νi₁₃/₂ orbital lies at 3.4 MeV relative to the νf₇/₂ orbital in the new parameter set.
- Deformations used in PSM calculations: ε₂ = 0.060 and ε₄ = −0.004 for ¹³⁶Te; ε₂ = 0.150 and ε₄ = −0.012 for ¹⁴²Xe.
- The separation between νi₁₃/₂ and νf₇/₂ orbitals is about 1 MeV larger from the new parameter set than from the standard set.

## Context
This work extends the Nilsson model parameterization, originally developed for stable nuclei by Nilsson and collaborators, to the neutron-rich region near the doubly magic ¹³²Sn. The authors note that Relativistic Mean Field (RMF) calculations with interactions such as NL1, NL3, and NLSH (Ring, 1996; Lalazissis et al., 1997) also predict reduced spin-orbit splittings in this region, and the new Nilsson parameters show similarities to RMF results. The new single-particle basis enables improved Projected Shell Model calculations (Hara and Sun, 1995) for collective bands, and the authors suggest future work should investigate octupole deformation effects on yrast bands in this mass region.
