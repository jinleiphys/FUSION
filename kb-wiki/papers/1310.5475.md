# GLISSANDO 2: GLauber Initial-State Simulation AND mOre..., ver. 2

> We present an extended version of GLISSANDO, a Monte-Carlo generator for Glauber-like models of the initial stage of relativistic heavy-ion collisions. The increased functionality of the code incorporates a parametrization of shape of nuclei, including light nuclei needed in the NA61 experiment, the nuclear deformation, a possibility of using correlated distributions of nucleons in nuclei read from external files, an option of overlaying distributions of produced particles dependent on the space-time rapidity, the inclusion of the core-corona effect, or the output of the source distributions that can be used in event-by-event hydrodynamics. Together with other features, such as incorporation of various variants of Glauber models, or the implementation of a realistic NN collision profile, the generator offers a realistic and practical approach to describe the early phase of the collision in 3+1 dimensions; the predictions may later be used in modeling the intermediate evolution phase, e.g., with hydrodynamics. The software is integrated with the ROOT platform. The supplied scripts compute and plot numerous features of the distributions, such as the multiplicity distributions and centrality classes, harmonic asymmetry coefficients and their correlations, forward-backward correlations, etc. The code can also be used for the proton-nucleus and deuteron-nucleus collisions.

## Key claim

GLISSANDO 2 extends the original Glauber Monte Carlo generator to include a parametrization of light nuclei needed for the NA61 experiment, nuclear deformation, correlated nucleon distributions from external files, space-time rapidity-dependent particle production, the core-corona effect, and output for event-by-event hydrodynamics, while maintaining downward compatibility with the original version.

## Method

The code implements Monte Carlo simulations of relativistic heavy-ion collisions using various Glauber model variants, including wounded nucleon, binary collision, and mixed models, with options for hard-sphere, Gaussian, or Gamma nucleon-nucleon collision profiles. Nucleon positions are generated from parametrized nuclear density distributions (harmonic oscillator shell model for light nuclei with A between 3 and 16, Woods-Saxon profiles for heavier nuclei, optionally with deformation parameters beta2 and beta4) or read from external files containing correlated distributions. The code overlays distributions of produced particles with options for Poisson, gamma, or negative binomial weighting, and can generate source distributions in 3+1 dimensions including space-time rapidity profiles for use as input to hydrodynamic evolution codes.

## Key numbers

- Light nuclei harmonic oscillator shell model parameter values: for He-4, mean squared charge radius = 2.81 fm^2 (d=0) and 2.45 fm^2 (d=0.9 fm); for C-12, 6.10 fm^2 (d=0) and 5.66 fm^2 (d=0.9 fm)
- Woods-Saxon parameters for heavy nuclei: R = (1.12 A^{1/3} - 0.86 A^{-1/3}) fm, a = 0.54 fm (d=0); R = (1.1 A^{1/3} - 0.656 A^{-1/3}) fm, a = 0.459 fm (d=0.9 fm)
- Deformation parameters for U-238: beta2 = 0.28, beta4 = 0.093; for Au-197: beta2 = -0.13, beta4 = -0.03
- Gamma NN collision profile with G=1 and omega=0.4 reproduces experimental values sigma_in = 73 mb and sigma_el = 25 mb at sqrt(s_NN) = 7 TeV
- Typical running time: 80 s/10000 events for wounded-nucleon model and 100 s/10000 events for mixed model with Gamma distribution, minimum-bias Pb+Pb collisions with hard-sphere wounding profile (Intel Xeon X5650, 2.67 GHz)
- Memory required: below 120 MB
- Rapidity distribution parameters for RHIC: eta_0 = 1, eta_m = 3.36, sigma_eta = 1.3

## Context

This work extends the original GLISSANDO code (Broniowski 2009) by adding capabilities needed for the NA61 experiment's mass-number scan, for collisions of deformed nuclei such as U+U at RHIC (Huang 2012, Iordanova 2013), and for improved modeling of LHC energies using the Gamma NN collision profile (Rybczynski 2013). The code enables event-by-event hydrodynamic simulations (Andrade 2006, Werner 2009, Petersen 2010, Holopainen 2010, Gardim 2011, Bozek 2011, Schenke 2010) and studies of forward-backward correlations (Bialas 2004, Gazdzicki 2005, Bzdak 2009, Bozek 2010), two-dimensional correlations in rapidity and azimuth (Bozek 2012), and jet quenching (Fries 2010, Rodriguez 2010).
