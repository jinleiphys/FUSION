# Bose-Einstein correlations in heavy-ion and electron-positron collisions

> We shortly review recent successes in applying Bose-Einstein interferometry in heavy ion collisions and the proceed to some model calculations for 3-dimensional Bose-Einstein correlation functions in electron-positron collisions at the Z0 pole.

# Bose-Einstein correlations in heavy-ion and electron-positron collisions

## Key claim
Bose-Einstein correlation functions in electron-positron collisions at the Z0 pole can be computed from a phase-space version of JETSET 7.4 that provides both momenta and production coordinates, but the version used produces longitudinal HBT radii up to 5 fm that are incompatible with experimental data giving about 1 fm, indicating the model disagrees with experiment at the level of two-particle correlations.

## Method
The authors used a phase-space version of JETSET 7.4 that provides both momenta and production coordinates for pions produced in e+e- collisions at the Z0 pole, considering only directly produced pions from a simple 2-jet event topology. They computed the two-particle correlation function using the classical algorithm without wave packet smearing, analyzing it in a Cartesian coordinate system with longitudinal (L) axis along the jet axis, outward (o) direction defined by transverse pair momentum, and sideward (s) direction. The reduced correlator was isolated by dividing out the ratio of single-particle spectra to extract source sizes.

## Key numbers
- The input rms width of the string used in the model is 0.78 fm, giving a sideward HBT radius R_s = 0.55 fm
- For the unreduced correlator K, the extracted HBT radii in the side direction were 0.31, 0.41, 0.46, and 0.50 fm at K_T = 0, 0.3, 0.5, and 1.0 GeV respectively
- The reduced correlator K_red in the side direction was independent of K_T and reproduced the input rms width of 0.55 fm
- Longitudinal HBT radii from the model reached up to 5 fm, while experimental data give about 1 fm
- The emission time duration Delta_tau was up to 3 fm/c at low K_T
- The transverse distance of production points from the central string axis was distributed according to a Gaussian with rms radius 0.78 fm

## Context
This work extends the authors' previous studies of Bose-Einstein correlations using the VNI event generator (Heinz et al., 2000) and compares with the standard Gaussian parametrization framework for HBT radii (Wiedemann and Heinz, 1999). The paper identifies a discrepancy between JETSET-based model predictions and experimental data from e+e- collisions at the Z0 pole, where the model produces longitudinal source sizes much larger than observed. The analysis demonstrates that dividing out the single-particle spectrum ratio is essential for extracting correct source sizes from correlation functions, particularly for small sources like those in e+e- collisions, and that this step has not been performed in existing data analyses. The work enables future improvements to event generators by constraining the space-time structure of string breaking models using two-particle correlations.
