# The proton spin sum rule chiral bag prediction, an update

> We reevaluate a quark model prediction using the new QCD evolution function calculated to the 3 loop order and conclude that this model compares favorably with the new experimental results.

## Key claim

The chiral bag model prediction for the integrated proton spin structure function \(\Gamma_p(Q^2)\) remains consistent with the new experimental results from the SMC collaboration when updated with the 3-loop QCD evolution function, provided the model's calculated flavor-singlet amplitude is identified with the renormalization-group-invariant quantity \(\Sigma_{inv}\) rather than with a low-scale matrix element.

## Method

The authors recalculate the chiral bag model prediction for \(\Gamma_p(Q^2)\) using the 3-loop QCD evolution function for the flavor-singlet axial current, with the model's input amplitudes \(F = 0.455\) and \(D = 0.795\) (giving \(F/D = 0.57\)) from their earlier chiral bag calculation that includes two-body magnetic gluon exchange spin-spin corrections. They compare the resulting model values for \(\Gamma_p\) and \(\Gamma_n\) as functions of \(\alpha(Q^2)\) against experimental data from the SMC and other polarized deep-inelastic scattering experiments, using \(\Lambda = 240\) MeV for the running coupling.

## Key numbers

- Experimental integrated proton spin structure function at \(\bar{Q}^2 = 10\) GeV\(^2/c^2\): \(\Gamma_p = 0.136 \pm 0.011 \pm 0.011\) (SMC, 1995).
- Chiral bag model flavor-singlet amplitude: \(I_{0,\text{model}} = 0.22\), corresponding to a "spin content" \(\Sigma_{\text{model}} = 0.33\) when pseudoscalar mesons are excluded from the bag interior.
- Extracted value from experiment using 3-loop QCD corrections: \(Y(\alpha(\mu^2),\alpha(Q^2)) I_0(\mu^2) = 0.194 \pm 0.073 \pm 0.073\).
- Model values for the quark spin contributions as \(Q^2 \rightarrow \infty\): \(\Delta u = 0.83\), \(\Delta d = -0.42\), \(\Delta s = -0.08\).
- Latest Ellis and Karliner analysis gives \(\Sigma(Q^2 = 10 \text{ GeV}^2) = 0.31 \pm 0.07\) (ignoring higher twist effects).

## Context

This work updates the chiral bag model predictions of Hogaasen, Myhrer, and collaborators (1990) in light of new polarized deep-inelastic scattering data from the SMC, E142, and other experiments, and incorporates the 3-loop QCD evolution function recently calculated by Larin (1993) and Kvan and N. (1993). The results are consistent with the QCD spectral sum rule analysis of Narison, Shore, and Veneziano (1994), and show that the model can simultaneously reproduce pre-EMC static baryon properties (axial charge, magnetic moments) and the new spin structure data, while also demonstrating that the extraction of \(\Delta s\) from data assuming SU(3) flavor symmetry should be treated with caution.
