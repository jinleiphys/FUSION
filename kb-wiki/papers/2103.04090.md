# Deep learning stochastic processes with QCD phase transition

> It is non-trivial to recognize phase transitions and track dynamics inside a stochastic process because of its intrinsic stochasticity. In this paper, we employ the deep learning method to classify the phase orders and predict the damping coefficient of fluctuating systems under Langevin's description. As a concrete set-up, we demonstrate this paradigm for the scalar condensation in QCD matter near the critical point, in which the order parameter of chiral phase transition can be characterized in a $1+1$-dimensional Langevin equation for $σ$ field. In a supervised learning manner, the Convolutional Neural Networks(CNNs) accurately classify the first-order phase transition and crossover based on $σ$ field configurations with fluctuations. Noise in the stochastic process does not significantly hinder the performance of the well-trained neural network for phase order recognition. For mixed dynamics with diverse dynamical parameters, we further devise and train the machine to predict the damping coefficients $η$ in a broad range. The results show that it is robust to extract the dynamics from the bumpy field configurations.

## Key claim

The paper claims that deep convolutional neural networks (CNNs) can accurately classify the order of QCD phase transitions (first-order vs. crossover) and predict the damping coefficient from stochastic Langevin dynamics, even when the input is limited to the final-stage, noisy field configurations after the phase transition has already occurred. The hedge is that performance degrades for noise magnitudes outside the training range, though the network shows some generalization.

## Method

The authors numerically solve a 1+1-dimensional Langevin equation for the scalar sigma field, which serves as the order parameter for the chiral phase transition in QCD matter near the critical point. The system is cooled along trajectories of constant baryon chemical potential (mu = 180 MeV for crossover, mu = 240 MeV for first-order) with a Hubble-like temperature drop, and the damping coefficient eta is varied from 1.0 to 5.5 fm^-1. The final 4 fm of the evolution (40 time steps by 30 spatial grid points) are recorded as 40x30 pixel images, which are fed into a deep CNN for supervised classification of phase order and regression of the damping coefficient.

## Key numbers

* Test accuracy for phase order classification reaches 99% for configurations with noise parameter B = 0.5 and B = 1.0, using a training set of 20,000 events at each of those noise levels.
* For configurations with noise parameter B = 1.5 (outside the training range), the test accuracy remains above 95%.
* For the damping coefficient regression task, the coefficient of determination R^2 reaches 93% after training on configurations with eta in the ranges (1.0-2.5) fm^-1 and (4.6-5.5) fm^-1, with 1,000 events per 0.1 fm^-1 bin.
* The network shows reasonable interpolation performance for damping coefficients in the unseen range eta ~ (2.6-4.5) fm^-1.
* A comparison LSTM network achieved only 68.7% test accuracy on the same phase classification task, versus the CNN's 99%.

## Context

This work extends the paradigm of using deep learning to identify phase transitions in heavy-ion collisions, previously applied to deterministic systems by Pang (2018) and Du (2020) using final-state particle spectra. Here the authors generalize to stochastic Langevin dynamics, using the sigma field configurations directly as input. The method enables extraction of dynamical parameters (the damping coefficient) from noisy, non-equilibrium systems, with potential applications to topological-dependent stochastic processes and to real physical observations via transfer learning.
