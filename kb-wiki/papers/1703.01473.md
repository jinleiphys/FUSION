# A new relativistic viscous hydrodynamics code and its application to the Kelvin-Helmholtz instability in high-energy heavy-ion collisions

> We construct a new relativistic viscous hydrodynamics code optimized in the Milne coordinates. We split the conservation equations into an ideal part and a viscous part, using the Strang spitting method. In the code a Riemann solver based on the two-shock approximation is utilized for the ideal part and the Piecewise Exact Solution (PES) method is applied for the viscous part. We check the validity of our numerical calculations by comparing analytical solutions, the viscous Bjorken's flow and the Israel-Stewart theory in Gubser flow regime. Using the code, we discuss possible development of the Kelvin-Helmholtz instability in high-energy heavy-ion collisions.

## Key claim

The authors construct a new relativistic viscous hydrodynamics code optimized in Milne coordinates that achieves high numerical accuracy with small numerical viscosity, enabling the study of Kelvin-Helmholtz instability development in high-energy heavy-ion collisions, but the instability discussion is limited to a simplified (x,η) plane scenario and the code's validity for viscous calculations degrades after τ=4 fm in Gubser flow tests.

## Method

The conservation equations are split into ideal and viscous parts using the Strang splitting method, with a Riemann solver based on the two-shock approximation for the ideal part and the Piecewise Exact Solution (PES) method for the viscous part. The constitutive equations for viscous tensors are further decomposed into convection, relaxation, and source-term equations, solved respectively by the upwind scheme, the PES method, and the predictor-corrector method. Space derivatives are evaluated using either the MC limiter for second-order accuracy or the piecewise parabolic method (PPM) for third-order accuracy, with the dimensional splitting method using PPM employed for solving the convection term.

## Key numbers

- Initial temperature T₀ = 300 MeV and initial proper time τ₀ = 1 fm for viscous Bjorken flow tests
- Shear viscosity to entropy density ratio η/s = 0.2 used in both Bjorken flow and Gubser flow tests
- Bulk viscosity ζ = 1000 MeV/fm² and relaxation time τ_Π = 1 fm for bulk pressure evolution test in Bjorken flow
- Space-grid size Δη = 0.1 and time-step size Δτ = 0.1 τ₀ Δη for Bjorken flow calculations
- Space-grid sizes (Δx, Δy, Δη) = (0.05 fm, 0.05 fm, 0.1) and Δτ = 0.1 Δx for Gubser flow calculations
- Relaxation time τ_η = 5η/(Ts) used in Gubser flow calculations with η/s = 0.2
- In viscous Gubser flow, deviation from semi-analytic solution appears after τ = 4 fm

## Context

This work extends the authors' previous ideal fluid code (Okamoto 2016) to include viscous effects, building on the Israel-Stewart theory framework (Israel 1979, Stewart 1977) and conformal hydrodynamics (Baier 2007). The code enables investigation of Kelvin-Helmholtz instability, previously discussed by Csernai (2012) and Romatschke (2008), with smaller numerical viscosity than earlier approaches. The PES method follows the approach of Inutsuka (2011), and the Gubser flow validation follows the semi-analytic solutions of Marrochio (2015). The code is intended for quantitative extraction of transport coefficients from high-precision experimental data at RHIC and the LHC, particularly for studying higher flow harmonics and longitudinal fluctuations.
