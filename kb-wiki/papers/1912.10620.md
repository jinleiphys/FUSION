# Time- and parity-violating effects of nuclear Schiff moment in molecules and solids

> We show that existing calculations of the interaction between nuclear Schiff moments and electrons in molecules use an inaccurate operator which gives rise to significant errors. By comparing the matrix elements of the accurate and imprecise Schiff moment operators, we calculated the correction factor as a function of the nuclear charge Z and presented corrected results for the T,P-violating interaction of the nuclear spin with the molecular axis in the TlF, RaO, PbO, TlCN, ThO, AcF molecules and in the ferroelectric solid PbTiO$_3$.

## Key claim

Existing calculations of the interaction between nuclear Schiff moments and electrons in molecules use an inaccurate operator that overestimates results by up to 20%; the authors provide correction factors as a function of nuclear charge Z and present corrected results for several molecules and a solid.

## Method

The authors compared matrix elements of the correct Schiff moment operator (derived from a finite-size nucleus potential) with those of the imprecise operator (which places the Schiff moment at the nuclear center). They calculated analytical correction factors using relativistic electron wavefunctions expanded to fourth order in Zα, and performed numerical relativistic Hartree-Fock-Dirac (HFD) calculations with random-phase approximation (RPA) corrections for atoms Yb, Tl, Pb, Ra, Ac, and Th. The correction factors were then applied to rescale published molecular interaction constants W_S.

## Key numbers

- For TlF, the published interaction constant W_S = 14000 a.u. (Hinds 1980) is corrected to 12400 a.u. using the average scaling factor r̄^sp = 1.13.
- For TlF, the published W_S = 46458 a.u. (Parpia 1997) is corrected to 41113 a.u. using r̄^sp = 1.13.
- For TlF, the published W_S = 52482 a.u. (Quiney 1998) is corrected to 46444 a.u. using r̄^sp = 1.13.
- For RaO, the published W_S = 45192 a.u. (Flambaum 2002, Kudashov 2013) is corrected to 39127 a.u. using r̄^sp = 1.16.
- For PbTiO₃, the published W_S = 30270 a.u. (Skripnikov 2016) is corrected to 26670 a.u. using r̄^sp = 1.14.
- For PbO, the published W_S = 44400 a.u. (Skripnikov 2016) is corrected to 39119 a.u. using r̄^sp = 1.14.
- For TlCN, the published W_S = 7150 a.u. (atoms7030062) is corrected to 6327 a.u. using r̄^sp = 1.13.
- For ThO, the published W_S = 45000 a.u. (Flambaum and Dzuba 2019) is corrected to 39000 a.u. using r̄^sp = 1.17.
- For AcF, the published W_S = 160000 a.u. (Flambaum and Dzuba 2019) is corrected to 140000 a.u. using r̄^sp = 1.16.
- The center-placement operator overestimates matrix elements by up to 20% for heavy nuclei; the surface-placement operator gives errors ≤ 5%.
- For Z=90 (Th), the numerical correction factor r^{sp_{1/2}}_{center} = 1.19 and r^{sp_{3/2}}_{center} = 1.14.

## Context

This work corrects a systematic error in all previous molecular calculations of Schiff moment interactions, which used the imprecise operator from Hinds (1980) and others. The correct operator was derived by Flambaum and Ginges (2002) but had not been adopted in molecular codes. The scaling factors enable immediate correction of existing results for molecules such as TlF, RaO, PbO, TlCN, ThO, AcF, and the ferroelectric solid PbTiO₃, and can be applied to future calculations without rewriting computer codes.
