# Is a consistent holographic description of excited hadrons with fixed L possible?

> We present a no-go theorem that it is impossible in general to construct a holographic description of each meson with the quark-antiquark valence degree of freedom only with fixed (conserved) quantum number L in 3+1 dimensions and that would satisfy at the same time manifest chiral symmetry in the ultraviolet and would not violate unitarity.

## Key claim
It is impossible in general to construct a holographic description of each meson (with only quark-antiquark valence degrees of freedom) that simultaneously has a fixed conserved orbital angular momentum \(L\) in 3+1 dimensions, preserves manifest chiral symmetry in the ultraviolet, and does not violate unitarity. The argument specifically targets the de Teramond–Brodsky construction, which the author claims violates either unitarity or chiral symmetry in the ultraviolet.

## Method
The author examines the matching condition at the ultraviolet boundary \(z=0\) between QCD composite operators (built from free quarks and gluons) and their duals in 5-dimensional AdS space. Using the unitary transformation from the quark-antiquark chiral basis to the standard nonrelativistic \(^{2S+1}L_J\) basis (generalizing the transformation from the two-particle relativistic helicity basis), the author shows that a given quark-antiquark composite operator cannot generally be assigned a dual with fixed nonrelativistic angular momentum \(L\). The specific example of \(\rho\)-mesons is worked out: the vector current \(\bar q \gamma^\mu \vec \tau q\) and the pseudotensor current \(\bar q \sigma^{0i} \vec \tau q\) both create \(\rho\)-mesons but have different chiral transformation properties, and each is a superposition of \(^3S_1\) and \(^3D_1\) states.

## Key numbers
- The vector current \(\bar q \gamma^\mu \vec \tau q\) transforms as \((1,0)+(0,1)\) of \(SU(2)_L \times SU(2)_R\); the pseudotensor current \(\bar q \sigma^{0i} \vec \tau q\) belongs to the \((1/2,1/2)_b\) representation.
- The unitary transformation yields: \(|(0,1)+(1,0);1\ 1^{--}\rangle = \sqrt{\frac{2}{3}}|1;{}^3S_1\rangle + \sqrt{\frac{1}{3}}|1;{}^3D_1\rangle\).
- The unitary transformation yields: \(|(1/2,1/2)_b;1\ 1^{--}\rangle = \sqrt{\frac{1}{3}}|1;{}^3S_1\rangle - \sqrt{\frac{2}{3}}|1;{}^3D_1\rangle\).

## Context
This work directly challenges the holographic description of excited hadrons proposed by de Teramond and Brodsky (2009), which claimed a description with fixed orbital angular momentum \(L\) and principal quantum number \(\sim n_r + L\). The no-go theorem extends earlier work on chiral classification of QCD operators (CJ, G1) and the unitary transformation from the chiral basis to the nonrelativistic basis (GN, L). The result implies that any bottom-up AdS/QCD construction that attempts to assign fixed \(L\) to each meson while preserving chiral symmetry in the ultraviolet must fail, and suggests that alternative approaches (e.g., allowing broken chiral symmetry or abandoning fixed \(L\)) are needed for a consistent holographic description of excited hadrons.
