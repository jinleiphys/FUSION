# Ratio of Isoscalar to Isovector Core Polarization for Magnetic Moments

> In calculations of isoscalar magnetic moments of odd-odd N=Z nuclei it was found that for medium to heavy mass nuclei large scale shell model calculations yielded results which were very close to much simpler single j shell ones. To understand this we compare isoscalar and isovector configuration mixing in first order perturbation theory using a spin dependant delata interaction.The isoscalar corrections are much smaller

## Key claim

The isoscalar magnetic moments of odd-odd N=Z nuclei approach the single-j shell (Schmidt) values for medium-to-heavy masses because the spin-exchange term in the nucleon-nucleon interaction suppresses isoscalar core-polarization corrections relative to isovector ones by about a factor of two or three, with the exact suppression depending on the spin-exchange parameter x.

## Method

The authors used first-order perturbation theory with a spin-dependent surface delta interaction V(r) = -G δ(r)(1 + x P_σ) to calculate core-polarization configuration-mixing corrections to magnetic moments of mirror pairs (⁵⁷Ni, ⁵⁷Cu) and the odd-odd nucleus ⁵⁸Cu. They fitted the strength G to the isovector magnetic moment deviation from the Schmidt value, then determined the spin-exchange parameter x by requiring the isoscalar deviation to match experiment. The ratio of isoscalar to isovector corrections was expressed analytically as Δμ(0)/Δμ(1) = (1-2x) × (g_S - g_L)_IS / (g_S - g_L)_IV.

## Key numbers

- Experimental magnetic moments: μ(⁵⁷Ni) = -0.7975 μ_N, μ(⁵⁷Cu) = 2.582(7) μ_N
- Derived isoscalar and isovector moments: μ(IS) = 0.892 μ_N, μ(IV) = 1.690 μ_N
- Schmidt values: μ(IS)_Schmidt = 0.940 μ_N, μ(IV)_Schmidt = 2.853 μ_N
- Deviations from Schmidt: Δμ(IS) = -0.0480 μ_N, Δμ(IV) = -1.163 μ_N
- Ratio of deviations Δμ(IS)/Δμ(IV) = 0.04256
- Free-nucleon g-factor differences: (g_S - g_L)_IS = 0.38, (g_S - g_L)_IV = 4.21, ratio = 0.09
- Fitted spin-exchange parameter x = 0.264 (with free-nucleon g-factors); x = 0.349 (with renormalized isovector g_S = 0.7 × free value and g_L = 0.6)
- Average absolute deviation between large-scale and single-j shell model isoscalar g-factors for nuclei heavier than ⁶Li: about 0.02 (approximately 4%)

## Context

This work extends the comparison by Yeager (2009) of large-scale shell model versus single-j shell model calculations for isoscalar magnetic g-factors of odd-odd N=Z nuclei, where it was observed that results converge for heavier nuclei. The authors use perturbation theory following Arima (1954) and Noya (1958), and the particle-hole formalism of Mavromatis (1967), to explain why isoscalar corrections are much smaller than isovector ones. The results enable future measurements of magnetic moments such as ⁴⁵V to test the predicted isovector structure, and provide a framework for understanding how spin-exchange interactions suppress isoscalar core polarization.
