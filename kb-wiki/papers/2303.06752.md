# High energy nuclear physics meets Machine Learning

> Though being seemingly disparate and with relatively new intersection, high energy nuclear physics and machine learning have already begun to merge and yield interesting results during the last few years. It's worthy to raise the profile of utilizing this novel mindset from machine learning in high energy nuclear physics, to help more interested readers see the breadth of activities around this intersection. The aim of this mini-review is to introduce to the community the current status and report an overview of applying machine learning for high energy nuclear physics, to present from different aspects and examples how scientific questions involved in high energy nuclear physics can be tackled using machine learning.

## Key claim

This mini-review surveys the current status of applying machine learning to high-energy nuclear physics, arguing that while the intersection is relatively new, ML techniques can help tackle challenges in data analysis, simulation optimization, and model calibration, but the authors emphasize that the field must go beyond mere application of existing algorithms to incorporate physical constraints and symmetries for effective physics discovery.

## Method

The paper provides a pedagogical overview of machine learning methodologies relevant to high-energy nuclear physics, including supervised learning (classification and regression), unsupervised learning, semi-supervised learning, self-supervised learning, active learning, and reinforcement learning. It then describes specific neural network architectures applicable to heavy-ion collision problems: convolutional neural networks (CNNs) for image-like data, recurrent neural networks (RNNs) for sequential data, point cloud networks for particle sets in momentum space that preserve permutation symmetry, and generative models for probability distribution learning. The review also covers optimization techniques including stochastic gradient descent, auto-differentiation, and physics-informed neural networks (PINNs).

## Key numbers

- The detectors at RHIC and the LHC can produce "petabytes of raw data per year" from high-energy nuclear collisions
- A point cloud network example uses a fully connected neural network to transform "m input features into 128 features in high-dimensional latent space" for each particle, with global max pooling or global average pooling applied to extract global information
- In binary classification, the cross-entropy loss function is defined over "m" samples in each minibatch, with the sigmoid activation function ensuring output values between 0 and 1, using 0.5 as the decision boundary

## Context

This mini-review extends the discussion of machine learning applications in nuclear physics beyond low and intermediate energy heavy-ion collisions, referencing recent work by Gao (2021), Xie (2021), Ming (2022), Wu (2022), and a mini-review by He (2023) for lower-energy regimes. It also connects to broader applications in hadron physics and lattice QCD, noting that machine learning has been applied to topics such as nuclear phase transition classification using point cloud networks (Steinheimer, 2019) and critical phenomenon searches using dynamical edge convolution (Huang, 2021). The review enables future work by providing a methodological foundation for researchers to apply ML techniques to open questions in heavy-ion collision physics, including initial state characterization, bulk properties extraction, medium effects, and hard probe analysis.
