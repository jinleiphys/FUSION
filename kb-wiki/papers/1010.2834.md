# Nucleon strangeness form factors and moments of PDF

> The calculation of the nucleon strangeness form factors from N_f=2+1 clover fermion lattice QCD is presented. Disconnected insertions are evaluated using the Z(4) stochastic method, along with unbiased subtractions from the hopping parameter expansion. We find that increasing the number of nucleon sources for each configuration improves the signal significantly. We obtain G_M^s(0) = -0.017(25)(07), which is consistent with experimental values, and has an order of magnitude smaller error. Preliminary results for the strangeness contribution to the second moment of the parton distribution function are also presented.

## Key claim
The authors report a lattice QCD calculation of the nucleon strangeness magnetic form factor at zero momentum transfer, obtaining a value consistent with experimental results but with an order of magnitude smaller statistical uncertainty. The result is conditional on the use of N_f=2+1 clover fermion configurations, the Z(4) stochastic method with hopping parameter expansion subtractions, and multiple nucleon sources per configuration to improve the signal.

## Method
The calculation uses N_f=2+1 dynamical configurations with nonperturbatively O(a) improved clover fermions and RG-improved gauge action generated by the CP-PACS/JLQCD Collaborations, on a 16^3 x 32 lattice with lattice spacing a^{-1} = 1.625 GeV. Disconnected insertions are evaluated using the Z(4) stochastic method with unbiased subtractions from the hopping parameter expansion up to order (kappa D)^4 for form factors and (kappa D)^3 for the second moment, employing 600-800 stochastic noises and 64-82 nucleon sources per configuration depending on the quark mass.

## Key numbers
- G_M^s(0) = -0.017(25)(07), where the first error is statistical and the second accounts for uncertainties in Q^2 and chiral extrapolations
- Lattice spacing a^{-1} = 1.625 GeV, with lattice size (2 fm)^3 box in physical spatial size
- Pion masses used: m_pi = 0.60, 0.70, and 0.84 GeV, corresponding to kappa_ud = 0.13825, 0.13800, and 0.13760 respectively
- 800 configurations used for kappa_ud = 0.13760, and 810 configurations for kappa_ud = 0.13800 and 0.13825
- N_noise = 600 for kappa_ud = 0.13760 and 0.13800, N_noise = 800 for kappa_ud = 0.13825
- N_src = 64 for kappa_ud = 0.13760, N_src = 82 for kappa_ud = 0.13800 and 0.13825
- Statistical errors reduced to about 50% for form factors and 70% for the second moment compared to results without hopping parameter expansion subtraction

## Context
This work extends earlier quenched lattice QCD calculations of strangeness matrix elements (Sasaki et al., 2003; Sasaki et al., 2005; Young et al., 2005) by using dynamical N_f=2+1 configurations and improved stochastic methods. The result provides a benchmark for experimental determinations of nucleon strangeness form factors and enables more precise comparisons between lattice QCD and experiments such as those from parity-violating electron scattering. The preliminary results for the second moment of the parton distribution function open the path toward full determination of strangeness contributions to nucleon structure.
