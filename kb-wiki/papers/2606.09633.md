# Effective QCD model with consistent quasi-gluon treatment : formulation and application

> The Polyakov loop enhanced Nambu-Jona-Lasinio model is reformulated in terms of the gluon quasi-particles in addition to the already existing quark quasi-particles. The formulation goes beyond the saddle point approximation for the gluon sector. The framework provides a physically consistent quasiparticle model for QCD thermodynamics. The ensuing advantages of this formulation is discussed using transport coefficients in the light quark sector.

## Key claim

The paper reformulates the Polyakov loop enhanced Nambu-Jona-Lasinio (PNJL) model by integrating a quasi-gluon description that goes beyond the saddle point approximation, claiming this provides a physically consistent quasiparticle model for QCD thermodynamics where previous quasi-gluon approaches produced unphysical negative thermodynamic quantities below the deconfinement temperature.

## Method

The authors couple a 2-flavor Nambu-Jona-Lasinio (NJL) quark model to a gluon quasi-particle model where the gluon thermodynamic potential is expressed in terms of the Polyakov loop background field. Instead of using the saddle point (mean-field) approximation for the gluon sector, they integrate the partition function over all constant configurations of the SU(3) class parameters θ₁ and θ₂, weighted by the Vandermonde determinant and the quasi-gluon thermodynamic potential, with a phenomenological volume parameter v. The chiral condensate σ is still obtained from a saddle point condition, while expectation values of Polyakov loop observables are computed via integration over the class parameters.

## Key numbers

- NJL model parameters: cutoff Λ = 0.651 GeV, bare quark mass m₀ = 0.0055 GeV, coupling G = 10.08 GeV⁻² (from Ref. PhysRevD.73.014019)
- Deconfinement temperature parameter T_d = 270 MeV used in the numerical analysis
- Volume parameter v = (0.5 T_d)⁻³, same as in the pure gauge study (Islam 2021)
- Polynomial potential parameters for the standard PNJL comparison: b₀ = 6.75, b₁ = -1.95, b₂ = 2.625, b₃ = -7.44, α₂ = 0.75, α₃ = 7.5, T_d = 0.270 GeV, κ = 0.1 (from Ghosh 2007 and PhysRevD.73.014019)
- Effective gluon mass parametrized as m_g(T)/T = α + exp[-β(T/T_d - γ)] with parameters fitted to pure gauge lattice pressure data (Giusti 2016)

## Context

This work extends the quasi-gluon matrix model approach developed for SU(3) pure gauge theory (Islam 2021) to include quark degrees of freedom via the NJL model. Previous attempts to couple quasi-gluons to chiral models (Sasaki 2012, Islam 2012, Alba 2014) suffered from unphysical negative thermodynamic quantities at temperatures below the deconfinement temperature, which the authors attribute to the saddle point approximation. The new integration-based method avoids this issue and enables explicit study of gluonic contributions to transport properties, such as shear and bulk viscosity, which is beyond the scope of standard PNJL models that use polynomial Polyakov potentials (Fukushima 2004, Megias 2004, Ghosh 2007).
