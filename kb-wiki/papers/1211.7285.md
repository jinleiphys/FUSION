# Roy-Steiner equations for pi N scattering

> Starting from hyperbolic dispersion relations for the invariant amplitudes of pion-nucleon scattering together with crossing symmetry and unitarity, one can derive a closed system of integral equations for the partial waves of both the s-channel (pi N --> pi N) and the t-channel (pi pi --> Nbar N) reaction, called Roy-Steiner equations. After giving a brief overview of the Roy-Steiner system for pi N scattering, we demonstrate that the solution of the t-channel subsystem, which represents the first step in solving the full system, can be achieved by means of Muskhelishvili-Omnès techniques. In particular, we present results for the P-waves featuring in the dispersive analysis of the electromagnetic form factors of the nucleon.

# Roy–Steiner equations for pi N scattering

## Key claim

The authors demonstrate that the t-channel subsystem of the Roy–Steiner equations for pion–nucleon scattering can be solved using Muskhelishvili–Omnès techniques, and that with increasing numbers of subtractions the solutions converge toward the results of the KH80 analysis, provided one uses KH80 values for the coupling and subtraction parameters as a starting point.

## Method

Starting from hyperbolic dispersion relations for the invariant amplitudes of pion–nucleon scattering, the authors derive a closed system of integral equations for the partial waves of both the s-channel (pi N → pi N) and t-channel (pi pi → Nbar N) reactions. They solve the t-channel subsystem as a single-channel Muskhelishvili–Omnès problem with a finite matching point, using as input pi pi phase shifts from CCL (Regge + partial-wave analysis), s-channel partial waves (l ≤ 4) from SAID for W ≤ 2.5 GeV, and a Regge model above that energy, with a pi N coupling of g^2/(4 pi) = 14.28.

## Key numbers

- The unsubtracted s-channel Roy–Steiner equations hold for W in [m_N + M_pi, 1.38 GeV] using hyperbola parameter a = -23.19 M_pi^2.
- The unsubtracted t-channel Roy–Steiner equations hold for sqrt(t) in [2 M_pi, 2.00 GeV] using a = -2.71 M_pi^2.
- The matching point for the Muskhelishvili–Omnès solutions is set at sqrt(t_m) = 0.98 GeV, above which all t-channel input is set to zero.
- The pi N coupling used as starting point is g^2/(4 pi) = 14.28, with modern analyses yielding 13.7 ± 0.2.
- The t-channel unitarity relation involves sigma^pi_t(t) = sqrt(1 - 4 M_pi^2 / t).

## Context

This work extends the Roy–Steiner equation formalism, previously applied to pi pi scattering (Roy equations, as in Ananthanarayan et al., 2001), to the pi N system. The t-channel P-wave solutions are directly relevant for the dispersive analysis of nucleon electromagnetic form factors (see e.g. Lorenz, 2012). The authors plan a self-consistent iteration between the s- and t-channel subsystems to obtain a precise low-energy pi N scattering amplitude with error estimates, improving upon the KH80 analysis (Höhler, 1983) which lacks thorough error estimates and uses older experimental data.
