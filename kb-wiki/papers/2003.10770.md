# Classifying Pole of Amplitude Using Deep Neural Network

> Most of exotic resonances observed in the past decade appear as peak structure near some threshold. These near-threshold phenomena can be interpreted as genuine resonant states or enhanced threshold cusps. Apparently, there is no straightforward way of distinguishing the two structures. In this work, we employ the strength of deep feed-forward neural network in classifying objects with almost similar features. We construct a neural network model with scattering amplitude as input and nature of pole causing the enhancement as output. The training data is generated by an S-matrix satisfying the unitarity and analyticity requirements. Using the separable potential model, we generate a validation data set to measure the network's predictive power. We find that our trained neural network model gives high accuracy when the cut-off parameter of the validation data is within $400$-$800\mbox{ MeV}$. As a final test, we use the Nijmegen partial wave and potential models for nucleon-nucleon scattering and show that the network gives the correct nature of pole.

# Classifying Pole of Amplitude Using Deep Neural Network

## Key claim

The authors demonstrate that a deep feed-forward neural network can classify whether a near-threshold enhancement in two-body scattering is caused by a bound state, virtual state, or resonance pole, but only when the cut-off parameter of the validation data lies within 400–800 MeV; outside this range the network's accuracy degrades significantly.

## Method

The authors generate training data from an S-matrix parametrized by a K-matrix satisfying unitarity and analyticity, producing cross-section inputs on a discretized center-of-mass energy axis from 0 to 100 MeV with 0.5 MeV spacing. A deep neural network with two hidden layers (100 and 50 nodes, ReLU activation) and a three-node softmax output layer is trained via stochastic gradient descent on 4 million input-output samples. Validation is performed using an exactly solvable separable potential model (Yamaguchi form factor) with both energy-independent and energy-dependent couplings, and finally tested on Nijmegen partial wave and potential models for nucleon-nucleon scattering.

## Key numbers

- Input energy range: 0 to 100 MeV with 0.5 MeV spacing, giving 200 input nodes
- Training dataset size: 4,000,000 input-output tuples (Set 1 and Set 2 each)
- Testing accuracy on Set 1: 99.7% after 1,000 epochs
- Testing accuracy on Set 2: 97.3% after 1,000 epochs
- High accuracy achieved when validation cut-off parameter is within 400–800 MeV
- Bound state binding energy range: 0.106 MeV to 42.55 MeV
- Resonance width range: 0.12 MeV to 64 MeV
- Learning rate: 0.01; batch size: 1600
- Training time for two-hidden-layer architecture: 2.6 × 10^5 seconds

## Context

This work addresses the long-standing problem of distinguishing genuine resonant states from threshold cusps in hadron spectroscopy, a challenge highlighted by discoveries such as X(3872) (Choi, 2003) and subsequent exotic hadron candidates (Olsen, 2018). Prior theoretical work by Frazer (1964), Pearce and Gibson, and Swanson (2016) showed that threshold cusps require a nearby pole to produce significant enhancement, while triangle singularities (Landau, 1959; Nakamura, 2019) can produce enhancements without nearby poles. The authors' neural network approach enables automated classification of pole nature from cross-section data, potentially aiding analysis of future experimental data where traditional methods cannot distinguish between bound, virtual, and resonance poles near threshold.
