# Eta-N Final State Interaction in incoherent Photoproduction of Eta-mesons from the Deuteron near Threshold

> An analysis of incoherent photoproduction of $η$ mesons off the deuteron for photon energies from threshold to 800 MeV is presented. The dominant contribution, the $γ$N-$η$N amplitude, is described within an isobar model. Effects of the final state interactions in the $NN$ as well as the $ηN$ systems are included employing models derived within the meson-exchange approach. It is found that their consideration is important. Specifically, due to an interference effect the influence of the $ηN$ final state interaction is enhanced in the reaction $γd \to np η$ close to threshold.

## Key claim

The eta-nucleon (eta N) final state interaction (FSI) provides an additional enhancement of the total cross section for the reaction gamma d -> n p eta near threshold, but this effect is only visible for excess energies below 40 MeV and is magnified by constructive interference with the dominant nucleon-nucleon (NN) FSI; the authors find that the available data favor moderate values of the real part of the eta N scattering length, but the analysis is limited by the theoretical uncertainty in the eta N interaction model.

## Method

The authors calculated the total cross section for incoherent photoproduction of eta mesons from the deuteron using an isobar model for the gamma N -> eta N amplitude dominated by the S11(1535) resonance, combined with the impulse approximation. They included final state interactions in both the neutron-proton (np) system, using the CD-Bonn potential for the half-shell np t-matrix, and the eta N system, using a half-shell eta N t-matrix derived from the Julich meson-baryon coupled-channels model for pi N scattering that includes the eta N channel. The cross section was evaluated as a function of photon energy from threshold to 800 MeV, and the sensitivity to the eta N scattering length was tested by replacing the full eta N t-matrix with an effective range expansion using different values of the scattering length from the literature.

## Key numbers

- The Julich meson-baryon model yields an eta N scattering length a_etaN = 0.42 + i 0.34 fm with effective range r_etaN = -2.0 + i 0.8 fm.
- A variant of the model that also fits the pi- p -> eta n cross section gives a_etaN = 0.54 + i 0.18 fm.
- The NN FSI and eta N FSI affect the cross section only for excess energies epsilon smaller than 40 MeV; for epsilon >= 50 MeV the impulse approximation alone reproduces the data.
- The eta N FSI enhances the cross section relative to the impulse approximation plus NN FSI calculation by a factor visible in the ratio plot for epsilon <= 30 MeV.
- Calculations with scattering lengths a_etaN = 0.25 + i 0.16 fm (Bennhold 1991) and a_etaN = 0.74 + i 0.27 fm (Green 1997) show that the data favor the smaller value.

## Context

This work extends a previous study by the same group (Sibirtsev et al., Phys. Rev. C 64, 024006) that calculated the gamma d -> n p eta cross section with impulse approximation and np FSI but found residual discrepancies near threshold. The present paper adds the eta N FSI using the Julich meson-baryon model (Krehl et al., Phys. Rev. C 62, 025207, 2000) and demonstrates that the eta N interaction can account for the remaining enhancement. The analysis is motivated by the longstanding uncertainty in the eta N scattering length, which varies widely across different theoretical approaches (e.g., Bhalerao 1985, Bennhold 1991, Green 1997, Kaiser 1995) and is relevant for interpreting data from other few-body reactions such as pd -> He-3 eta (Mayer 1996) and pn -> d eta (Calen 1997). The results suggest that the gamma d -> n p eta reaction can help constrain the eta N scattering length, but more precise data are needed.
