# Deep inelastic scattering and final state interaction in an exactly solvable relativistic model

> In the theory of deep inelastic scattering (DIS) the final state interaction (FSI) between the struck quark and the remnants of the target is usually assumed to be negligible in the Bjorken limit. This assumption, still awaiting a full validation within nonperturbative QCD, is investigated in a model composed by two relativistic particles, interacting via a relativistic harmonic oscillator potential, within light-cone hamiltonian dynamics. An electromagnetic current operator whose matrix elements behave properly under Poincaré transformations is adopted. It is shown that: i) the parton model is recovered, once the standard parton model assumptions are adopted; and ii) when relativistic, interacting eigenfunctions are exactly taken into account for both the initial and final states, the values of the structure functions, averaged over small, but finite intervals of the Bjorken variable $x$, coincide with the results of the parton model in the Bjorken limit.

## Key claim

In an exactly solvable relativistic two-body model with a confining harmonic oscillator potential, the parton model's neglect of final state interaction (FSI) is justified in the Bjorken limit only after the structure functions are averaged over small but finite intervals of the Bjorken variable \(x\); without this averaging, the exact structure functions are discrete sums of delta functions rather than the continuous parton-model result.

## Method

The authors constructed a light-cone Hamiltonian dynamics model of two relativistic spin-1/2 particles of equal mass \(m_0\), interacting via a relativistic harmonic oscillator potential that yields only a discrete mass spectrum (confinement). They computed the deep inelastic scattering (DIS) structure functions \(F_1\) and \(F_2\) in two ways: first using the standard parton-model impulse approximation with free final states, and second using the exact interacting eigenfunctions for both initial and final states. The calculations were performed in the Breit frame with \(\vec{P'}_\perp = \vec{q}_\perp = 0\), using an electromagnetic current operator that satisfies Poincaré covariance and current conservation.

## Key numbers

* The mass eigenvalues of the two-body system are \(M_n = 2[m_0^2 + a^2(2n+3)]^{1/2}\), where \(n = n_x + n_y + n_z\) and \(a\) is a constant with dimension GeV.
* The ground-state mass is \(m = 2[m_0^2 + 3a^2]^{1/2}\).
* In the parton model, the structure functions satisfy the Callan-Gross relation \(F_2(x) = 2x F_1(x)\) and exhibit Bjorken scaling (no \(Q\) dependence).
* The exact structure function \(F_1(x,Q)\) contains a delta function \(\delta(m^2 + \frac{Q^2(1-x)}{x} - M_n^2)\), enforcing discrete final-state masses rather than a continuous spectrum.

## Context

This work extends earlier nonrelativistic two-body studies of FSI in DIS (e.g., IKL, PSW) to a fully relativistic framework, addressing the fact that the Bjorken limit requires \(q^0 \gg m\) and cannot be treated nonrelativistically. It also goes beyond the Bethe-Salpeter approach of Gurvitz (1994), which assumed dominance of handbag diagrams, by providing an exact solution without such approximations. The result enables a rigorous check of the standard parton-model assumption that FSI is a higher-twist effect, showing that confinement and the parton model can be reconciled in the sense of distribution theory after averaging over finite \(x\) intervals.
