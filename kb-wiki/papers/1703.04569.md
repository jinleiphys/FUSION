# Laplace expansion method for the calculation of the reduced width amplitudes

> We derive the equations to calculate the reduced width amplitudes (RWA) of the different size clusters and deformed clusters without any approximation. These equations named Laplace expansion method are applicable to the nuclear models which uses the Gaussian wave packets. The advantage of the method is demonstrated by the numerical calculations of the ${}^{16}{\rm O}+α$ and ${}^{24}{\rm Mg}+α$ RWAs in $^{20}{\rm Ne}$ and $^{28}{\rm Si}$.

## Key claim

The paper derives exact equations for calculating reduced width amplitudes (RWA) of different-size clusters and deformed clusters using Laplace expansion of the Slater determinant, without the approximations required by ordinary methods that assume common oscillator parameters and spherical clusters.

## Method

The authors apply the Laplace expansion of the matrix determinant to decompose the antisymmetrized molecular dynamics (AMD) wave function of an A-body system into products of subsystem wave functions for clusters of masses C1 and C2. The RWA is then obtained by calculating three types of overlaps: the relative wave function overlap between clusters, and the overlaps of each cluster's wave function with the corresponding subsystem AMD wave functions, all of which are analytically tractable for Gaussian wave packets.

## Key numbers

- For the 16O+alpha clustering in 20Ne, the alpha spectroscopic factor S_alpha for the 0+_1 state is 0.24 (common size calculation) and 0.26 (unequal size calculation with nu_O=0.157 fm^-2, nu_alpha=0.25 fm^-2)
- For the 0+_4 state in 20Ne, S_alpha = 0.62 (common size) and 0.53 (unequal size)
- For the 1-_1 state in 20Ne, S_alpha = 0.71 (common size) and 0.63 (unequal size)
- The dimensionless decay width theta^2_alpha at channel radius a=6 fm for the 0+_4 state is 0.44 (common size) and 0.37 (unequal size)
- The dimensionless decay width theta^2_alpha at channel radius a=6 fm for the 1-_1 state is 0.49 (common size) and 0.41 (unequal size)
- The oscillator parameters used for the unequal size calculation were nu_O=0.157 fm^-2 and nu_alpha=0.25 fm^-2, while the common size calculation used nu=0.16 fm^-2 for both clusters

## Context

This work extends the theoretical framework for RWA calculations beyond the limitations of ordinary methods developed by Horiuchi (1972, 1977) and Kanada-En'yo (2003), which required SU(3) shell model wave functions with common oscillator parameters and could not accurately treat unequal-size clusters or deformed clusters. The Laplace expansion method is applicable to nuclear models using Gaussian wave packets such as AMD and the Brink-Bloch wave function (Brink, 1966), and enables calculation of RWAs for systems like 16O+alpha, 24Mg+alpha, and 6He+alpha clusters without the approximations previously needed by Kanada-En'yo (2014) or Baba (2016, 2017). The method can be extended recursively to decompose A-body wave functions into three or more subsystems.
