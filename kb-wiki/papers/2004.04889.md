# Spectral density estimation with the Gaussian Integral Transform

> The spectral density operator $\hatρ(ω)=δ(ω-\hat{H})$ plays a central role in linear response theory as its expectation value, the dynamical response function, can be used to compute scattering cross-sections. In this work, we describe a near optimal quantum algorithm providing an approximation to the spectral density with energy resolution $Δ$ and error $ε$ using $\mathcal{O}\left(\sqrt{\log\left(1/ε\right)\left(\log\left(1/Δ\right)+\log\left(1/ε\right)\right)}/Δ\right)$ operations. This is achieved without using expensive approximations to the time-evolution operator but exploiting instead qubitization to implement an approximate Gaussian Integral Transform (GIT) of the spectral density. We also describe appropriate error metrics to assess the quality of spectral function approximations more generally.

## Key claim

The paper claims a near-optimal quantum algorithm for approximating the spectral density operator that achieves an energy resolution Δ and error ε using O(√(log(1/ε)(log(1/Δ)+log(1/ε)))/Δ) operations, but this scaling holds only under the specific condition that the target accuracy parameters satisfy β_L ≤ β ≤ β_U (the "intermediate regime"), and the algorithm avoids expensive time-evolution approximations by using qubitization to implement a Gaussian Integral Transform (GIT).

## Method

The algorithm approximates the spectral density operator ρ̂(ω)=δ(ω-Ĥ) by constructing a Gaussian Integral Transform (GIT) of the response function S(ω)=⟨Ψ|ρ̂(ω)|Ψ⟩, using a kernel K_G(σ,ω,Λ) = exp(-(σ-ω)²/(2Λ²))/(√(2π)Λ). The GIT is implemented via qubitization, which uses the qubiterate unitary W_Q to compute Chebyshev polynomial expectation values ⟨Ψ_G|W_Q^k|Ψ_G⟩, and the Gaussian kernel is expanded in a truncated Chebyshev series of order L. The truncation order L is chosen to bound the approximation error, and the expectation values are estimated from repeated measurements to produce a (Σ,Δ,β,η_β)-approximation to the response function.

## Key numbers

- The GIT algorithm requires M = O((1/Δ)√(log(1/ε)log(1/(Δε)))) calls to the qubiterate unitary W_Q to achieve a (ε,Δ,ε,η_β)-approximation.
- The total number of samples required is N_S = O((1/(Δ³ε²))(log(1/ε)log(1/(Δε)))³/² log(1/η_β)).
- For the Fejer kernel method, the number of oracle calls is M = O(1/(Δε)) and samples N_S = O((1/ε²)log(1/η_β)).
- For the Time-Series Analysis (TSA) method of Somma (2019), the maximum evolution time is t_max = O((1/Δ)log(1/ε)²) and samples N_S = O((1/(Δ³ε²))log(1/ε)⁶log(1/η_β)).
- The Gaussian kernel resolution Λ must satisfy Λ ≤ Δ/√(2log(1/Σ)) to achieve Σ-accuracy with resolution Δ.

## Context

This work extends the Quantum Eigenvalue Estimation Problem (QEEP) framework of Somma (2019) by introducing a Gaussian Integral Transform approach that achieves near-optimal quantum query complexity, improving over the TSA method's logarithmic factors. The GIT method avoids the expensive time-evolution operator approximations required by Fejer kernel approaches (Roggero 2019, Roggero 2020) and instead uses qubitization (Low 2017, Low 2019) to implement the transform directly. The technique enables efficient estimation of dynamical response functions for scattering cross-sections and other linear response observables, with potential future extensions to approximate inversion schemes like the Maximum Entropy Method (Gubernatis 1991).
