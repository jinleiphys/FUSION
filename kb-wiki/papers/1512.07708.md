# Two decay paths for calculation of nuclear matrix element of neutrinoless double-beta decay using quasiparticle random-phase approximation

> It is possible to employ virtual decay paths, including two-particle transfer, to calculate the nuclear matrix element of neutrinoless double-beta decay under the closure approximation, in addition to the true double-beta path. In the quasiparticle random-phase approximation (QRPA) approach, it is necessary to introduce the product wave functions of the like-particle and proton-neutron QRPA ground states, for achieving consistency between the calculations of the true and virtual paths. Using these different paths, the problem of whether or not these two methods give equivalent nuclear matrix elements (NME) is investigated. It is found that the two results are inequivalent, resulting from the different many-body correlations included in the two QRPA methods, i.e., the use of the product wave functions alone is not sufficient. The author proposes introduction of the proton-neutron pairing interaction with an adequate strength in the double-beta-path method, which carries less many-body correlations without this supplemental interaction, for obtaining the NME equivalent to that of the two-particle-transfer-path method. The validity of the proposed modified approach is examined.

## Key claim

The paper investigates whether the nuclear matrix element (NME) for neutrinoless double-beta decay calculated via the true double-beta path and via a virtual two-particle-transfer path are equivalent within the quasiparticle random-phase approximation (QRPA) framework, and finds they are inequivalent due to different many-body correlations; the author proposes introducing a proton-neutron pairing interaction with adequate strength in the double-beta-path method to achieve equivalence.

## Method

The author extends the QRPA ground states to product wave functions of like-particle (lp) and proton-neutron (pn) QRPA ground states, then computes the NME for the decay 150Nd → 150Sm using both the double-beta path (pnQRPA intermediate states) and the two-particle-transfer path (lpQRPA intermediate states). Analytical derivation of the "cross terms" between lp and pn QRPA operators is performed, and numerical tests are carried out for the (Kπ) = (2+) component to assess the approximation of neglecting these cross terms. A contact volume proton-neutron pairing interaction with T=0 component is introduced into the pnQRPA calculations to supplement many-body correlations and bring the two path calculations into agreement.

## Key numbers

- The (Kπ) = (2+) component of M(0ν) obtained in the test calculation was 0.3712, identical within the shown order to the value obtained without the cross terms.
- Using all lpQRPA solutions with the same (Kπ) value, the corresponding NME value was 0.3710.
- The M(0ν) value obtained via the double-beta path without the pn pairing interaction was 5.324.
- The M(0ν) value obtained via the two-particle-transfer path was 3.604.
- The M(0ν) value obtained via the double-beta path with the pn pairing interaction (using g_pnpair_T=0 = -197.44 MeV fm3 for 150Nd and -200.09 MeV fm3 for 150Sm) was 3.697, which is 48% larger than the two-particle-transfer path value without the interaction but close to it after inclusion.

## Context

This work extends the author's previous modified QRPA approach (Terasaki 2013, 2015) that introduced product wave functions of lp and pn QRPA ground states and used virtual decay paths for NME calculation. It addresses the problem that the two decay paths yield inequivalent NMEs within QRPA, contrary to what would be expected for exact nuclear wave functions. The proposed solution of introducing a pn pairing interaction builds on findings by multiple groups (e.g., Mustonen and Engel 2013) that stronger pn pairing reduces M(0ν). This work enables further refinement of QRPA-based NME calculations and contributes to resolving the factor-of-2-3 discrepancy among different theoretical methods for 0νββ NMEs.
