# Hadronic decays of eta and eta-prime with coupled channels

> The hadronic decays eta -> pi pi pi, eta-prime -> pi pi pi and eta-prime -> eta pi pi are investigated within a U(3) chiral unitary approach. Final state interactions are included by deriving the effective s-wave potentials for meson meson scattering from the chiral effective Lagrangian and iterating them in a Bethe-Salpeter equation. With only a small set of parameters we are able to explain both rates and spectral shapes of these decays.

## Key claim

The authors show that a U(3) chiral unitary approach, which iterates effective s-wave potentials from the chiral Lagrangian in a Bethe-Salpeter equation, can simultaneously explain the decay rates and spectral shapes of η → πππ, η′ → πππ, and η′ → ηππ using only a small set of parameters, but the parameter choice is not unique and moderate changes from previously fitted values were necessary to achieve agreement with experimental decay rates and Dalitz distributions.

## Method

The authors derive effective s-wave potentials for meson-meson scattering from the U(3) chiral effective Lagrangian up to fourth chiral order, including π⁰-η₈-η₀ mixing from isospin breaking. They iterate these potentials in a Bethe-Salpeter equation to generate infinite meson-meson rescattering chains, accounting for final state interactions in the s-, t-, and u-channels, while restricting to s-waves and setting all momenta on-shell.

## Key numbers

- Γ(η → π⁰π⁰π⁰) = (379 ± 40) eV (experimental value)
- Γ(η → π⁰π⁺π⁻) = (274 ± 33) eV (experimental value)
- Ratio r = Γ(η → π⁰π⁰π⁰) / Γ(η → π⁰π⁺π⁻) = 1.35 ± 0.05 (experimental value)
- η′ mass = 958 MeV (used as a massive state, not treated as small)
- Chiral symmetry breaking scale Λχ = 4πfπ ≈ 1.2 GeV
- Fitted low-energy constants: β₀⁽⁰⁾ = 0.56 × 10⁻³, β₃⁽⁰⁾ = -0.3 × 10⁻³, β₅⁽⁰⁾ = 1.4 × 10⁻³, β₆⁽⁰⁾ = 0.06 × 10⁻³

## Context

This work extends the U(3) chiral unitary approach previously applied to meson-meson scattering phase shifts by Beisert and Borasoy (2001) to the hadronic decays of η and η′. It improves upon one-loop SU(3) ChPT calculations by Gasser and Leutwyler (1985) and the Khuri-Treiman equation approach by Kambor, Wiesendanger, and Wyler (1996) by providing a unified framework that also handles the η′ decays, and it enables testing whether the same low-energy constants can describe both scattering and decay data simultaneously.
