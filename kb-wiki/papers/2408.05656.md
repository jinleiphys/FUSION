# Applications of the Modified Hulthén-Kohn Method for Bound and Scattering States

> We apply the Hulthèn-Kohn method suggested by V. D. Efros [Phys. Rev. C 99, 034620 (2019)] for calculating various observables in the continuum and discrete spectrum using two-body interactions in single- and coupled-channel systems. This method is promising for many-body applications and ab initio description of nuclear reactions. We explore the convergence of phase shifts and wave functions as well as the location of S-matrix poles which enables obtaining both resonance and bound state parameters. We find that adopting wave functions from approximate bound-state solutions for the short-range components of basis wave functions leads to good convergence.

## Key claim

The modified Hulthén–Kohn method, originally suggested by Efros (2019), can be extended to calculate phase shifts, continuum wave functions, and S-matrix poles for both single- and coupled-channel two-body scattering problems, with the convergence improved by using approximate bound-state solutions for the short-range components of the basis wave functions.

## Method

The authors parameterize the scattering wave function as a sum of incoming and outgoing spherical waves (with unknown S-matrix elements) plus a finite set of short-range functions expanded in harmonic oscillator basis states. They then solve a system of linear equations obtained by projecting the Schrödinger equation onto a set of linearly-independent short-range test functions, avoiding the need for "free-free" matrix elements. For multichannel problems, they impose S-matrix symmetry by reducing the number of independent equations in successive entrance channels. S-matrix poles are located by solving det A¹(E) = 0 in the complex energy plane using contour integration.

## Key numbers

- The oscillator length parameter is defined as b = √(ħ / (m Ω)), where m is the reduced mass and Ω is the oscillator frequency.
- Kinetic energy matrix elements in the oscillator basis are given by T^l_nn = (1/2)ħΩ(2n + l + 3/2) for diagonal terms and T^l_{n-1,n} = -(1/2)ħΩ √(n(n + l + 1/2)) for off-diagonal terms.
- The relative momentum of colliding particles is k = √(2mE) / ħ.

## Context

This work extends the Efros (2019) method beyond simple model problems to realistic two-body interactions, including the Daejeon16 NN interaction, for both single- and coupled-channel scattering. It builds on the HORSE formalism (Bang, 2000; Zaitsev, 1998) and the J-matrix method (Yamani, 1975) by incorporating their oscillator-basis expansions for regularizing the irregular spherical Bessel functions. The approach addresses limitations of the SS-HORSE method (Shirokov, 2016, 2018), which loses continuum wave function information, and is intended to enable future ab initio many-body calculations of nuclear reactions within the no-core shell model framework.
