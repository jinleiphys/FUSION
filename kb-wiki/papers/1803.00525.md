# Beta Spectrum Generator: High precision allowed $β$ spectrum shapes

> Several searches for Beyond Standard Model physics rely on an accurate and highly precise theoretical description of the allowed $β$ spectrum. Following recent theoretical advances, a C++ implementation of an analytical description of the allowed beta spectrum shape was constructed. It implements all known corrections required to give a theoretical description accurate to a few parts in $10^4$. The remaining nuclear structure-sensitive input can optionally be calculated in an extreme single-particle approximation with a variety of nuclear potentials, or obtained through an interface with more state-of-the-art computations. Due to its relevance in modern neutrino physics, the corresponding (anti)neutrino spectra are readily available with appropriate radiative corrections. In the interest of user-friendliness, a graphical interface was developed in Python with a coupling to a variety of nuclear databases. We present several test cases and illustrate potential usage of the code. Our work can be used as the foundation for current and future high-precision experiments related to the beta decay process. ## Key claim

The paper presents a C++ implementation of an analytical description of the allowed beta spectrum shape that achieves a theoretical description accurate to a few parts in \(10^4\), but this precision depends on the user providing or calculating the remaining nuclear structure-sensitive input, which can optionally be obtained through an extreme single-particle approximation or interfaced with more sophisticated computations.

## Method

The authors implemented the analytical corrections described in recent theoretical work (Hayen et al., 2018) into a C++ library, with a Python graphical user interface. Nuclear matrix elements for allowed Gamow-Teller beta decay are calculated in a spherical harmonic oscillator basis, where wave functions can be computed in an extreme single-particle approximation using different nuclear potentials or provided by the user from more sophisticated many-body calculations. The code also calculates corresponding (anti)neutrino spectra with appropriate radiative corrections, and provides connections to nuclear databases through the graphical interface.

## Key numbers

- The theoretical description is accurate to a few parts in \(10^4\) (relative precision)
- Typical values for the weak magnetism parameter \(b_\text{wm}\) result in spectral slopes of 0.5% MeV\(^{-1}\)
- The weak magnetism correction needs to be known with better than 20% accuracy to allow extraction of new physics at the \(10^{-3}\) level
- The atomic exchange correction becomes negligible typically in the 30 to 200 keV range for low and high Z nuclei, respectively
- Nuclear deformation corrections increase linearly with Z and can become significant at the few \(10^{-4}\) level in the fission fragment and lead region

## Context

This work extends the formalism developed by Hayen et al. (2018), which introduced several additional corrections to bring the analytical beta spectrum shape to a relative precision of a few parts in \(10^4\), with particular focus on atomic effects such as screening and exchange processes. The code enables high-precision searches for Beyond Standard Model physics through beta decay spectrum measurements, including searches for scalar and tensor currents via the Fierz term, and can serve as the foundation for current and future high-precision experiments related to the beta decay process.
