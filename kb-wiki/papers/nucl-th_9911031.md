# Multi-pion production in the d d -> alpha X reaction

> A simple model, based on two parallel and independent N N -> d pi processes, has recently been proposed for two-pion production in the d d -> alpha X reaction. It reproduces all observed features, including the sharp peak structure in momentum distributions (the ABC effect) and the strong oscillations in the deuteron vector and tensor analyzing powers. This model is now extended to describe also four-pion production with the same basic mechanism, but with two n p -> d pi pi processes as input. The calculations of the high missing mass spectra are within about 30% of the experimental data for beam energies in the range 1.9 < T_d < 2.4 GeV.

# Multi-pion production in the dd -> alpha X reaction

## Key claim
The model based on two parallel and independent NN -> d pi processes, which successfully reproduces two-pion production in the dd -> alpha X reaction, can be extended to describe four-pion production using two np -> d pi pi processes as input, with calculations of high missing mass spectra falling within about 30% of experimental data for beam energies between 1.9 and 2.4 GeV.

## Method
The author models four-pion production as two parallel sigma meson productions (np -> sigma d), where the sigma meson serves as a convenient representation of an I_X=0 pion pair. The amplitude for np -> sigma d is parametrized as a single s-wave amplitude, with its mass dependence extracted from experimental np -> dX data by transforming laboratory momentum distributions to missing mass distributions in the center-of-mass frame. The matrix element for dd -> alpha sigma sigma is written as a product of the spin kernel and a dimensionless form factor, with the integration over meson angles performed directly and results accumulated over all possible combinations of the two sigma masses to produce the full four-pion spectrum.

## Key numbers
- Calculations of high missing mass spectra are within about 30% of experimental data for beam energies in the range 1.9 < T_d < 2.4 GeV
- The Gaussian width for energy distribution between subprocesses is sigma_E approximately 25 MeV, calculated from a Gaussian deuteron wave function
- Cross sections d sigma/d Omega for the structure attributed to the omega meson are 9, 6, and 2 nb/sr at beam energies of 1.9, 2.2, and 2.4 GeV respectively
- The relative weights for charged and neutral pions from the sigma sigma production mechanism are N(++--):N(0 0 +-):N(0 0 0 0) = 4:4:1
- The rho rho production contribution is estimated to be less than 1% of the sigma sigma cross section

## Context
This work extends the model of Gardestig, Faldt, and Wilkin (1999) which successfully described two-pion production in the dd -> alpha X reaction for beam energies 0.8 < T_d < 1.9 GeV, including the ABC effect and deuteron analyzing powers. The model addresses the long-standing difficulty in quantitatively describing the ABC effect observed in np -> dX, pd -> He-3 X, and dd -> alpha X reactions, which previous theoretical approaches by BRS, ALS76, Barry, and KE had failed to reproduce fully. The results suggest that most multi-meson production in the dd -> alpha X reaction up to 2.4 GeV can be attributed to the same double-interaction mechanism, and the author notes that further support could come from estimating gamma gamma production via two np -> d gamma processes as background to the charge-symmetry-breaking dd -> alpha pi0 reaction.
