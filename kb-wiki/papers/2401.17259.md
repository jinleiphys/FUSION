# Photon-triggered jets as probes of multi-stage jet modification

> Prompt photons are created in the early stages of heavy ion collisions and traverse the QGP medium without any interaction. Therefore, photon-triggered jets can be used to study the jet quenching in the QGP medium. In this work, photon-triggered jets are studied through different jet and jet substructure observables for different collision systems and energies using the JETSCAPE framework. Since the multistage evolution used in the JETSCAPE framework is adequate to describe a wide range of experimental observables simultaneously using the same parameter tune, we use the same parameters tuned for jet and leading hadron studies. The same isolation criteria used in the experimental analysis are used to identify prompt photons for better comparison. For the first time, high-accuracy JETSCAPE results are compared with multi-energy LHC and RHIC measurements to better understand the deviations observed in prior studies. This study highlights the importance of multistage evolution for the simultaneous description of experimental observables through different collision systems and energies using a single parameter tune.

## Key claim

The JETSCAPE multistage framework, using the same parameter tune previously applied to jet and leading hadron studies (PP19 and AA22 tunes), can simultaneously describe photon-triggered jet observables across multiple collision systems and energies without additional parameter adjustment, though a multiplication factor of 1.4 is required to match the isolated photon and di-jet correlation results from ATLAS.

## Method

The authors used the JETSCAPE 3.X framework with MATTER and LBT energy loss modules for partonic showers in Pb-Pb collisions, and MATTER alone for p-p collisions, with PYTHIA providing initial hard scattering and final-state string hadronization. Trigger photons were identified using the same isolation criteria as in ATLAS and CMS experimental analyses, applied to both modified prompt-photon events and full events. Jets were clustered using the anti-kT algorithm within FastJet, and results were compared with experimental data at sqrt(S_NN)=5.02 TeV for p-p and Pb-Pb collisions.

## Key numbers

- Gamma-jet asymmetry (X_jgamma) distributions compared for four pT-gamma intervals from 63.1 GeV to 200 GeV for both p-p and Pb-Pb collisions
- A multiplication factor of 1.4 was required to describe both p-p and Pb-Pb results for the isolated photon and di-jet correlation (x_JJgamma) distribution
- Results shown for Pb-Pb 0-10% centrality class for gamma-jet asymmetry compared with CMS data
- The zg distribution (energy imbalance of the hardest split in a gamma-triggered jet) was calculated for four different X_Jgamma regions

## Context

This work extends the JETSCAPE collaboration's previous studies using the PP19 tune (JETSCAPE, 2019) for p-p collisions and the AA22 tune (JETSCAPE, 2022) for Pb-Pb collisions, applying them without modification to photon-triggered jet observables. The study provides the first high-accuracy JETSCAPE comparison with multi-energy LHC and RHIC measurements, addressing deviations observed in prior single-stage jet evolution studies. The results enable future comparisons with ongoing experimental measurements of groomed jet substructure in photon-triggered jets and highlight the need for additional theoretical work on isolated photon and multi-jet correlations, where no current model can explain the data without a scaling factor.
