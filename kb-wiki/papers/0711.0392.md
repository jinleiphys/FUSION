# Polarized structure functions of nucleons and nuclei

> We determine the quark distributions and structure functions for both unpolarized and polarized DIS of leptons on nucleons and nuclei. The scalar and vector mean fields in the nucleus modify the motion of the quarks inside the nucleons. By taking into account this medium modification, we are able to reproduce the experimental data on the unpolarized EMC effect, and to make predictions for the polarized EMC effect. We discuss examples of nuclei where the polarized EMC effect could be measured. We finally present an extension of our model to describe fragmentation functions.

## Key claim

The authors predict that the polarized EMC effect (the ratio of nuclear to nucleon polarized structure functions) is larger than the unpolarized EMC effect for the same nuclei, based on a model where nuclear scalar and vector mean fields modify the quark wave functions inside bound nucleons.

## Method

The nucleon is described as a quark-diquark bound state using the Faddeev framework within the Nambu-Jona-Lasinio (NJL) model, with scalar and axial-vector diquark channels and an infrared cutoff in proper-time regularization. Finite nuclei are treated in an independent particle picture using Woods-Saxon scalar and vector potentials for nucleons, with depth parameters from self-consistent nuclear matter calculations; the average scalar and vector fields for quarks are obtained via the quark-diquark Faddeev equation and used in the quark propagators to compute nuclear quark distributions via convolution with nucleon light-cone momentum distributions.

## Key numbers

- Free proton quark spin sums: Δu = 0.97, Δd = -0.30, Σ = 0.67, g_A = 1.27
- For a proton bound in ¹¹B: Δu = 0.88, Δd = -0.28, Σ = 0.60, g_A = 1.16
- For a proton bound in infinite nuclear matter: Δu = 0.74, Δd = -0.25, Σ = 0.49, g_A = 0.99
- Free proton tensor charges: Δ_T u = 1.04, Δ_T d = -0.24; in nuclear matter: Δ_T u = 0.93, Δ_T d = -0.23
- The model scale Q_0 is set to 400 MeV, equal to the constituent quark mass

## Context

This work extends the approach of Cloet (2006) that explained the unpolarized EMC effect through medium modification of quark wave functions, now making predictions for the polarized EMC effect in nuclei such as ⁷Li, ¹¹B, and ²⁷Al. The results establish a link between quark physics and nuclear structure through the relation of bound-nucleon axial coupling constants to nuclear Gamow-Teller matrix elements, as discussed by Arima (1988). The extension to fragmentation functions via the Drell-Levy-Yan relation opens the possibility to compute both distribution and fragmentation functions within the same framework.
