# Constraining Bag constant for Hybrid Neutron stars

> We study the star matter properties for Hybrid equation of state (EoS) by varying the bag constant. We use the Effective-Field-Theory motivated Relativistic Mean-Field model (E-RMF) for hadron phase with recently reported FSUGarnet, G3 and IOPB-I parameter sets. The result of NL3 and NL3${ωρ}$ sets are also shown for comparison. The simple MIT Bag model is applied for the quark phase to construct the hybrid EoS. The hybrid neutron star mass and radius are calculated by varying with $B^{1/4}$ to constrain the $B^{1/4}$ values. It is found that $B^{1/4}$=130-160 MeV is suitable for explaining the quark matter in neutron stars.

## Key claim

The bag constant range \(B^{1/4} = 130\text{–}160\) MeV is suitable for explaining quark matter in hybrid neutron stars, but this conclusion is conditional on the specific hadronic parameter sets used (IOPB-I and G3) and on the requirement that the maximum mass of the hybrid star falls within the range \(2.01\pm0.04 \le M(M_\odot) \le 2.16\pm0.03\) inferred from gravitational-wave observations; the NL3 and NL3\(\omega\rho\) sets produce higher masses at the same bag values and are ruled out by the GW170817 data.

## Method

The authors construct hybrid equations of state by combining the Effective-Field-Theory motivated Relativistic Mean-Field (E-RMF) model for the hadronic phase (using the FSUGarnet, G3, IOPB-I, NL3, and NL3\(\omega\rho\) parameter sets) with the simple MIT Bag model for the quark phase, assuming \(m_u = m_d = 5.0\) MeV and \(m_s = 150\) MeV. The hadron-quark mixed phase is treated via Gibbs construction, and the Tolman-Oppenheimer-Volkoff (TOV) equations are solved to obtain the mass and radius of hybrid stars for bag constant values \(B^{1/4} = 100, 130, 160, 180,\) and \(200\) MeV.

## Key numbers

- For the G3 parameter set with \(B^{1/4} = 130\) MeV, the hybrid star maximum mass is \(2.15 M_\odot\); for \(B^{1/4} = 160\) MeV, it is \(1.88 M_\odot\).
- For the IOPB-I parameter set with \(B^{1/4} = 130\) MeV, the maximum mass is \(2.32 M_\odot\); for \(B^{1/4} = 160\) MeV, it is \(2.07 M_\odot\).
- For the NL3 parameter set with \(B^{1/4} = 130\) MeV, the maximum mass is \(2.43 M_\odot\); for \(B^{1/4} = 160\) MeV, it is \(2.08 M_\odot\).
- The radius of the canonical mass (\(1.4 M_\odot\)) for bag constants \(B^{1/4} = 130\text{–}160\) MeV is found to be in the range \(12.5\text{–}13.2\) km, consistent with the constraint \(R_{1.4} \le 13.76\) km from the GW170817 event.
- For \(B^{1/4} = 100\) MeV, the hadron-quark phase transition occurs at a density below normal nuclear density (\(\rho_0 = 0.16\) fm\(^{-3}\)), e.g., \(\rho_{MP} = 0.98\rho_0\) for NL3, which is deemed unphysical.
- The pure hadronic maximum masses are \(2.81 M_\odot\) (NL3), \(2.75 M_\odot\) (NL3\(\omega\rho\)), \(2.15 M_\odot\) (IOPB-I), and \(2.03 M_\odot\) (G3).

## Context

This work extends prior studies of hybrid neutron stars by Rather et al. (2020) that combine E-RMF hadronic models with the MIT Bag model, and it directly addresses the need to constrain the bag constant using gravitational-wave observations from GW170817 and GW190425. The results contradict the wider bag range of \(B^{1/4} = 150\text{–}180\) MeV proposed by Aziz et al. (2019) and rule out very low (100 MeV) and very high (200 MeV) bag values as incompatible with either the observed maximum mass range or the requirement that quarks appear only at supra-nuclear densities. The constrained range \(B^{1/4} = 130\text{–}160\) MeV enables future calculations of hybrid star properties (tidal deformability, cooling) that can be tested against multi-messenger observations.
