# Numerical study of computational cost of maintaining adiabaticity for long paths

> Recent work argued that the scaling of a dimensionless quantity $Q_D$ with path length is a better proxy for quantifying the scaling of the computational cost of maintaining adiabaticity than the timescale. It also conjectured that generically the scaling will be superlinear (although special cases exist in which it is linear). The quantity $Q_D$ depends only on the properties of ground states along the Hamiltonian path and the rate at which the path is followed. In this paper, we demonstrate that this conjecture holds for simple Hamiltonian systems that can be studied numerically. In particular, the systems studied exhibit the behavior that $Q_D$ grows approximately as $L \log L$ where $L$ is the path length when the threshold error is fixed.

## Key claim

The paper provides numerical evidence supporting the conjecture that the dimensionless quantity \(Q_D\), which serves as a proxy for computational cost of maintaining adiabaticity, grows superlinearly with path length \(L\) for generic Hamiltonian systems when the error threshold is fixed; the specific systems studied exhibit \(Q_D \propto L \log L\) behavior, though the authors note that special cases exist where scaling is linear.

## Method

The authors numerically solved the time-dependent Schrodinger equation for 4-dimensional Hilbert space Hamiltonians (4x4 matrices) with random, non-periodic time dependence constructed from trigonometric functions with incommensurate frequencies. They determined the scale factor \(s_c\) that yields a fixed threshold error \(\epsilon_{\th} = 0.1\) for each path length, then computed \(Q_D\) from the ground state properties along the path using the relation \(Q_D = s_c \int_0^{T_{\text{end}}} dt \; \langle \dot{g} | (H(t) - E_g(t)) | \dot{g} \rangle / \langle \dot{g} | \dot{g} \rangle\).

## Key numbers

* For four different random 4x4 Hamiltonians of the form \(H(t) = (2.1\sin(t) + \sin(\sqrt{2}t)) H_1 + (2.7\cos(t) + \cos(\sqrt{2}t)) H_2\) with threshold error \(\epsilon_{\th} = 0.1\), \(Q_D/L\) shows a secular increase with path length \(L\), consistent with superlinear scaling (Fig. 4).
* The path length for random Hamiltonians at fixed end time \(T_{\text{end}} = 10\) increases more slowly with Hilbert space dimension, following a logarithmic fit \(a \log L + b\) (Fig. 2).
* For a single random Hamiltonian, the three different proxies \(Q_{D_1}\), \(Q_{D_2}\), and \(Q_{D_{1/2}}\) show nearly identical behavior when plotted against path length (Fig. 3).

## Context

This work extends the theoretical framework of Cohen (2024), which introduced \(Q_D\) as a better proxy for computational cost than timescale and conjectured superlinear scaling with path length. The numerical results support this conjecture for simple systems, which is relevant because Cohen (2023) showed that some state preparation methods scale linearly with path length, making it important to determine whether adiabatic state preparation is competitive for long paths such as those arising in quantum field theory simulations. The paper uses random Hamiltonians to capture generic behavior, avoiding special cases like the translationally-invariant Hamiltonian \(H(t) = \exp(-i v A t / \sqrt{\langle g_0|A^2|g_0\rangle}) H_0 \exp(i v A t / \sqrt{\langle g_0|A^2|g_0\rangle})\) which yields linear scaling.
