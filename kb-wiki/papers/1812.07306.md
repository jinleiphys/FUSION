# Neutron star cooling with microscopic equations of state

> We model neutron star cooling with several microscopic nuclear equations of state based on different nucleon-nucleon interactions and three-body forces, and compatible with the recent GW170817 neutron star merger event. They all feature strong direct Urca processes. We find that all models are able to describe well the current set of cooling data for isolated neutron stars, provided that large and extended proton 1S0 gaps and no neutron 3PF2 gaps are active in the stellar matter. We then analyze the neutron star mass distributions predicted by the different models and single out the preferred ones.

## Key claim

All four microscopic equations of state (EOSs) can describe the current set of cooling data for isolated neutron stars, provided that large and extended proton 1S0 pairing gaps are active and neutron 3PF2 gaps are absent; the authors note that this conclusion depends on assuming no exotic components like hyperons or quark matter.

## Method

The authors performed neutron star cooling simulations using the NSCool code with four microscopic Brueckner-Hartree-Fock (BHF) equations of state (BOB, V18, N93, UIX) based on different nucleon-nucleon potentials and three-body forces, all compatible with the GW170817 merger event. They solved the BCS gap equations for neutron 1S0, proton 1S0, and neutron 3PF2 pairing channels using the same interactions, and computed cooling curves for a range of neutron star masses with both Fe and light-element atmospheres.

## Key numbers

- DU onset proton fraction thresholds: x_DU = 0.1331 (N93), 0.1348 (V18), 0.1357 (BOB), 0.1363 (UIX)
- DU onset densities: rho_DU = 0.30 fm^-3 (N93), 0.37 fm^-3 (V18), 0.41 fm^-3 (BOB), 0.45 fm^-3 (UIX)
- NS masses at DU onset: M_DU = 0.82 M_sun (N93), 1.01 M_sun (V18), 1.17 M_sun (UIX), 1.56 M_sun (BOB)
- Maximum NS masses: M_max = 2.13 M_sun (N93), 2.04 M_sun (UIX), 2.34 M_sun (V18), 2.51 M_sun (BOB)
- Density where p1S0 BCS gap vanishes: rho_1S0 = 0.52 fm^-3 (N93), 0.60 fm^-3 (V18), 0.59 fm^-3 (BOB), 0.70 fm^-3 (UIX)
- NS masses where p1S0 gap vanishes: M_1S0 = 1.59 M_sun (N93), 1.91 M_sun (V18), 2.23 M_sun (BOB), 1.70 M_sun (UIX)

## Context

This work extends previous cooling simulations by Wei et al. (2016, 2018) that used only one microscopic EOS, now applying the same analysis to three additional EOSs identified as compatible with the GW170817 merger event by Drago et al. (2018) and Jinbiao (2018). The results challenge the "minimal cooling paradigm" that previously excluded EOSs featuring direct Urca processes, and enable future combined analysis of neutron star mergers, radius measurements, and cooling to further constrain the nuclear equation of state, including possible exotic components like hyperons or quarks.
