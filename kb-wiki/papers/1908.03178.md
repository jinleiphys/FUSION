# Sum Rule of Femtoscopic Correlation Function

> A correlation function of two particles with small relative velocities obeys a sum rule - the momentum integral of the function is determined due to the completeness of quantum states of the particles. The original sum rule derived in 1995 suffered from a serious problem: the momentum integral was ultraviolet divergent in physically interesting cases. We resolve the problem by considering the sum rule not of a single correlation function but of a sum or difference of two appropriately chosen correlation functions. The improved sum rule is shown to work well for the exact Coulomb correlation functions. We argue that the sum rule can be used to test an accuracy and range of applicability of correlation functions computed in approximate models. The neutron-proton correlation function is discussed as an example.

## Key claim

The original sum rule of femtoscopic correlation functions (Mrowczynski, 1995) is mathematically flawed because the momentum integral diverges in physically interesting cases; the authors resolve this by considering the sum or difference of two appropriately chosen correlation functions rather than a single one, provided the ultraviolet divergence cancels.

## Method

The authors derive an improved sum rule by taking the momentum integral of the difference (or sum) of two correlation functions whose asymptotic behaviors are matched, so that the integrand falls off faster than \(q^{-3}\). They test the improved sum rule against exact Coulomb correlation functions for charged pions (both repulsive and attractive cases) and then apply it to the neutron-proton correlation function computed using the Lednicky-Lyuboshits (1981) model with asymptotic scattering wave functions. The sum rule is used to extract the deuteron formation rate and to assess the validity range of the approximate model.

## Key numbers

* For charged pions with a Gaussian source of radius \(r_0 = 3\) fm, the function \(\mathcal{S}(q_{\mathrm{max}})\) becomes indistinguishable from \(\pi^3 D_r(0)\) for \(q_{\mathrm{max}} r_0 \ge 1.5\).
* For a point-like source of \(\pi^+ \pi^-\) pairs, the sum rule after subtraction of the divergent term yields \(- 8\pi^2 \zeta(3) / a_B^3\) with \(\zeta(3) \approx 1.202\) and \(a_B\) the Bohr radius of the pair.
* The neutron-proton scattering lengths and effective ranges used are: singlet \(a^s = -23.7\) fm, \(d^s = 2.7\) fm; triplet \(a^t = 5.4\) fm, \(d^t = 1.7\) fm (McCarthy, 1968).
* The deuteron root-mean-square radius is 2.0 fm (Babenko, 2008); the Hulthén wave function parameters are \(\alpha = 0.23\) fm\(^{-1}\) and \(\beta = 1.61\) fm\(^{-1}\) (Hodgson, 1971).
* The improved sum rule for neutron-proton correlation functions is satisfied for source radii \(r_0\) larger than 2 fm when using the Hulthén wave function for the deuteron formation rate.

## Context

This work resolves a long-standing ultraviolet divergence problem in the sum rule originally derived by Mrowczynski (1995). It extends the theoretical foundation of femtoscopic correlation functions by providing a convergent integral relation that can test approximate models. The improved sum rule is demonstrated on exact Coulomb correlation functions (Maj, 2004) and on the neutron-proton correlation function computed with the Lednicky-Lyuboshits (1981) asymptotic wave function, showing that the latter is reliable only for source radii above 2 fm. The sum rule can now be applied to test the accuracy of models used in modern femtoscopic analyses of particle sources and interaction parameters (Acharya, 2019).
