# Color Octet Contribution to High p_T J/ΨProduction in pp Collisions at \sqrt s = 500 and 200 GeV at RHIC

> We compute \frac{dσ}{dp_T} of the J/ψproduction in pp collisions at RHIC at \sqrt s = 500 and 200 GeV by using both the color octet and singlet models in the framework of non-relativistic QCD. The J/ψwe compute here includes the direct J/ψfrom the partonic fusion processes and the J/ψcoming from the radiative decays of χ_J's both in the color octet and singlet channel. The high p_T J/ψproduction cross section is computed within the PHENIX detector acceptance ranges: -0.35 < η< 0.35 and 1.2 < \|η| < 2.4, the central electron and forward muon arms. It is found that the color octet contribution to J/ψproduction is dominant at RHIC energy in comparison to the color singlet contributions. We compare our results with the recent preliminary data obtained by PHENIX detector for the high p_T J/ψmeasurements. While the color singlet model fails to explain the data completely the color octet model is in agreement with the single data point above 2 GeV transverse momentum. A measurement of J/ψproduction at RHIC in the next run with better statistics will allow us to determine the validity of the color octet model of J/ψproduction at RHIC energies. This is very important because it is necessary to know the exact mechanism for J/ψproduction in pp collisions at RHIC if one is to make predictions of J/ψsuppression as a signature of quark-gluon plasma. These mechanisms also play an important role in determining the polarized spin structure function of the proton at RHIC.

## Key claim

The color octet contribution to high-\(p_T\) J/\(\psi\) production in pp collisions at RHIC is dominant over the color singlet contribution, and the color octet model is in agreement with the single PHENIX data point above 2 GeV transverse momentum, whereas the color singlet model fails to explain that data point; however, the authors caution that the comparison is not conclusive due to the preliminary nature and limited statistics of the data.

## Method

The authors computed the differential cross section \(d\sigma/dp_T\) for J/\(\psi\) production in pp collisions at \(\sqrt s = 500\) and 200 GeV using both the color singlet model and the color octet model within the non-relativistic QCD (NRQCD) framework. They included direct J/\(\psi\) from partonic fusion processes and J/\(\psi\) from radiative decays of \(\chi_J\) states in both color channels, using parton distribution functions (GRV98 NLO) and non-perturbative NRQCD matrix elements extracted from CDF data at the Tevatron. The calculation was performed within the PHENIX detector acceptance ranges: central electron arm (\(-0.35 < \eta < 0.35\)) and forward muon arms (\(1.2 < |\eta| < 2.4\)), corresponding to J/\(\psi\) rapidity ranges \(-0.5 < y < 0.5\) and \(1 < y < 2\).

## Key numbers

*   The non-perturbative matrix element combination used for the color octet model: \(\frac{<0|{\cal{O}}_8^{J/\psi}(^{3}P_0)|0>}{M_c^2} + \frac{<0|{\cal{O}}_8^{J/\psi}(^{1}S_0)|0>}{3} = (2.2 \pm 0.5) \times 10^{-2}\) GeV\(^3\).
*   The non-perturbative matrix element for the color octet model: \(<0|{\cal{O}}_8^{J/\psi}(^{3}S_1)|0> = (6.6 \pm 2.1) \times 10^{-3}\) GeV\(^3\).
*   The non-perturbative matrix element for the color octet model: \(<0|{\cal{O}}_8^{\chi_1}(^{3}S_1)|0> = (9.8 \pm 1.3) \times 10^{-3}\) GeV\(^3\).
*   For the color singlet model, the radial wave function at the origin for direct J/\(\psi\): \(|R(0)|^2 = 0.81\) GeV\(^3\), using charm quark mass \(M_c = 1.48\) GeV.
*   For the color singlet model, the derivative of the radial wave function at the origin for \(\chi_J\) processes: \(\frac{9}{2\pi}\frac{|R^\prime(0)|^2}{M_c^4} = 15\) MeV.
*   The comparison with PHENIX data was made at a single data point at \(p_T \sim 2.5\) GeV for \(\sqrt s = 200\) GeV in the rapidity range \(1 < y < 2\).

## Context

This work extends the study of J/\(\psi\) production mechanisms to RHIC energies, following the Tevatron measurements by the CDF collaboration that ruled out the color singlet model for high-\(p_T\) quarkonium production at that energy. The color singlet model had previously given a reasonable description at ISR energies. The paper enables future tests of the color octet model at RHIC with higher-statistics data, which is important because a validated J/\(\psi\) production mechanism in pp collisions is necessary for interpreting J/\(\psi\) suppression as a signature of quark-gluon plasma in Au-Au collisions, and for determining the polarized spin structure function of the proton.
