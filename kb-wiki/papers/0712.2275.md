# PLATYPUS: a code for fusion and breakup in reactions induced by weakly-bound nuclei within a classical trajectory model with stochastic breakup

> A self-contained Fortran-90 program based on a classical trajectory model with stochastic breakup is presented, which should be a powerful tool for quantifying complete and incomplete fusion, and breakup in reactions induced by weakly-bound two-body projectiles near the Coulomb barrier. The code calculates complete and incomplete fusion cross sections and their angular momentum distribution, as well as breakup observables (angle, kinetic energy and relative energy distributions).

## Key claim

The paper presents a classical trajectory model with stochastic breakup that can quantify complete fusion, incomplete fusion, and breakup observables in reactions induced by weakly-bound two-body projectiles near the Coulomb barrier, but the model is not a breakup model itself; rather it uses a stochastically sampled breakup function determined from sub-barrier breakup measurements to relate sub-barrier no-capture breakup to above-barrier incomplete and complete fusion.

## Method

The code PLATYPUS uses a three-dimensional classical dynamical reaction model combined with Monte Carlo sampling of probability-density distributions. For each partial wave of the projectile-target system, the code calculates the orbit of the bound projectile, then samples ISEEDMAX breakup events with initial conditions determined by conservation of energy, linear momentum, and angular momentum. The classical trajectories of the three bodies (two breakup fragments and target) are integrated using a fourth-order Runge-Kutta integrator, and during propagation the code determines capture of fragments by the target to classify events as complete fusion, incomplete fusion, or no-capture breakup.

## Key numbers

- Total incomplete fusion cross section (SIGMA_ICF) = 288.3 mb for the pseudo-8Be + 208Pb system at the incident energy specified in the input file
- Total complete fusion cross section (SIGMA_CF(TOT)) = 298.6 mb, comprising SIGMA_CF(NO BU) = 222.2 mb from events that survive breakup and SIGMA2_CF = 76.37 mb from complete fusion following breakup
- No-capture breakup cross section (SIGMA_BU) = 265.3 mb for the same system
- Inert projectile complete fusion cross section (SIGMA_CF(INERT PROJ)) = 652.1 mb, representing the case when the projectile cannot break up
- The code uses 1000 breakup events per partial wave (ISEEDMAX) in the example calculation
- Typical running time is about 75 minutes on a PC with 1.5 GHz processor
- RAM requirement is 10 MB

## Context

This work extends the classical trajectory approach previously applied to multi-fragmentation and incomplete fusion in heavy-ion reactions at energies well above the Coulomb barrier (Moehring, Bondorf) to the near-barrier energy regime for weakly-bound nuclei. The model has been successfully applied to interpreting fusion and breakup measurements of weakly-bound nuclei (Ramin, Huy, Santra) and isomer ratio measurements (Gasques). The code enables researchers to calculate observables that continuum-discretised coupled-channels (CDCC) models cannot provide unambiguously, such as integrated incomplete and complete fusion cross sections, and the evolution of surviving breakup fragments after incomplete fusion product formation.
