# Real-time lattice gauge theory actions: unitarity, convergence, and path integral contour deformations

> The Wilson action for Euclidean lattice gauge theory defines a positive-definite transfer matrix that corresponds to a unitary lattice gauge theory time-evolution operator if analytically continued to real time. Hoshina, Fujii, and Kikukawa (HFK) recently pointed out that applying the Wilson action discretization to continuum real-time gauge theory does not lead to this, or any other, unitary theory and proposed an alternate real-time lattice gauge theory action that does result in a unitary real-time transfer matrix. The character expansion defining the HFK action is divergent, and in this work we apply a path integral contour deformation to obtain a convergent representation for U(1) HFK path integrals suitable for numerical Monte Carlo calculations. We also introduce a class of real-time lattice gauge theory actions based on analytic continuation of the Euclidean heat-kernel action. Similar divergent sums are involved in defining these actions, but for one action in this class this divergence takes a particularly simple form, allowing construction of a path integral contour deformation that provides absolutely convergent representations for U(1) and SU(N) real-time lattice gauge theory path integrals. We perform proof-of-principle Monte Carlo calculations of real-time U(1) and SU(3) lattice gauge theory and verify that exact results for unitary time evolution of static quark-antiquark pairs in (1 + 1)D are reproduced.

## Key claim

The Wilson action discretization applied to continuum real-time gauge theory does not yield a unitary theory, contrary to what might be expected from analytic continuation of the Euclidean Wilson action; the authors demonstrate that the Hoshina-Fujii-Kikukawa (HFK) action and a newly introduced modified heat-kernel (HKbar) action can provide unitary real-time transfer matrices, but only the HKbar action admits a convergent path integral contour deformation suitable for Monte Carlo calculations in both U(1) and SU(N) gauge theories.

## Method

The authors analyze the unitarity of real-time transfer matrices for compact gauge variables by examining the kinetic integral kernels of discretized actions, showing that non-Gaussian kinetic terms break unitarity for compact variables. They introduce path integral contour deformations that depend on summation indices in the action's character expansion, rotating the kinetic term prefactor from -1 toward i while exactly cancelling contour segments related by periodicity, to obtain absolutely convergent representations for the HFK action in U(1) and the HKbar action in both U(1) and SU(N). Proof-of-principle Monte Carlo calculations are performed for real-time U(1) and SU(3) lattice gauge theory in (1+1)D, verifying that exact results for unitary time evolution of static quark-antiquark pairs are reproduced.

## Key numbers

- The real-time Wilson action for SU(2) one-plaquette model in (1+1)D is shown to be non-unitary; the ratio of transfer matrix eigenvalues I_k(-i/a) / I_l(-i/a) as a → 0 converges to 1 for k ≡ l mod 2 but does not exist for k ≡ l+1 mod 2.
- The Euclidean heat-kernel action of Menotti and Onofri (1981) is analytically continued to define real-time actions; the HKbar action is the only unitary choice for which a convergent Monte Carlo calculation can be performed in real-time SU(N) LGT.
- Proof-of-principle Monte Carlo calculations in (1+1)D reproduce exact results for unitary time evolution of static quark-antiquark pairs for both U(1) and SU(3) gauge theories.

## Context

This work extends the analysis of Hoshina, Fujii, and Kikukawa (2020), who first pointed out that the Wilson action discretization applied to continuum real-time gauge theory does not lead to a unitary theory and proposed an alternate real-time action. The authors address the divergence problem in the HFK action's character expansion by introducing path integral contour deformations, building on techniques previously applied to sign problems in finite-density systems and real-time quantum-mechanical theories reviewed by Alexandru et al. (2020). The results enable nonperturbative Monte Carlo calculations of real-time lattice gauge theory observables, which were previously inaccessible due to the sign problem from pure-phase path integral weights e^{iS_M} and the lack of convergent unitary actions.
