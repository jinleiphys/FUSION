# Time-ordered Diagrammatic Monte Carlo for atomic nuclei

> Diagrammatic Monte Carlo provides a systematically improvable framework for stochastically resumming many-body expansions to high orders through direct sampling of diagram topologies. We advance our earlier work by introducing a novel time-ordered Diagrammatic Monte Carlo algorithm for the single-particle Green's function. The algorithm is tailored to finite nuclei, formulated in discrete model spaces and applicable to arbitrary two-body interactions. The new time-ordered diagrammatic Monte Carlo algorithm is based on the on-the-fly evaluation of time-ordered Goldstone diagrams, avoiding explicit diagram enumeration and expensive frequency integration. We show the algorithm by computing ${}^{16}$O up to fifth order in a reduced model space using optimized reference state orbitals and including effective three-body forces. Benchmarking against established truncation schemes in ab initio nuclear theory demonstrates its potential to overcome the limitations of current many-body approaches.

# Time-ordered Diagrammatic Monte Carlo for atomic nuclei

## Key claim

The authors introduce a time-ordered Diagrammatic Monte Carlo (TO-DiagMC) algorithm that stochastically resums Goldstone diagrams for the single-particle Green's function in finite nuclei, demonstrating calculations up to fifth order for ¹⁶O in a reduced model space (e_max=2) with a finite regulator η=10 MeV, but note that this application should be regarded as a proof of applicability rather than a calculation aimed at achieving full convergence with respect to model-space truncation.

## Method

The algorithm samples diagram topologies, time orderings, and internal quantum numbers in a single Markov chain using Metropolis-Hastings updates, avoiding explicit diagram enumeration and frequency integration by evaluating time-ordered Goldstone diagrams on the fly. The self-energy is partitioned into frequency bins and projected onto Legendre polynomials using Gauss-Legendre quadrature, with the diagrammatic series recast as a weighted average of diagram signs. Simulations were performed for ¹⁶O using an optimized reference state (OpRS) basis derived from the self-consistent Green's function framework, with the soft chiral interaction ΔNNLO_GO(394) and a regulator η=10 MeV to control oscillations near poles.

## Key numbers

- Self-energy computed for ¹⁶O up to **fifth order** in a reduced model space with **e_max=2**
- Regulator **η=10 MeV** used to enforce convergence over the entire energy range, with verification that decreasing η to **0.5 MeV** systematically approaches the exact η=0 limit
- Self-energy shown over energy range **ω ∈ [0, 70] MeV** for optical potential applications
- Spectral function computed on energy mesh **ω ∈ [-240, 160] MeV**
- Each run employs **128 walkers** performing approximately **10¹¹ updates** in diagrammatic space
- Ground-state energy extrapolations performed using data at **η=20, 15, and 10 MeV**

## Context

This work extends earlier diagrammatic Monte Carlo proof-of-principle applications in the Richardson pairing model (Brolli, 2025) to a realistic nuclear system, building on the self-consistent Green's function framework (Barbieri, 2009; Barbieri, 2022). The TO-DiagMC approach complements recent developments combining automated diagrammatic generators (Arthuis, 2019; Arthuis, 2021) with Monte Carlo techniques for infinite nuclear matter up to fifth order (Drischler, 2019; Drischler, 2026). The results are benchmarked against the ADC(3)-D truncation scheme, the current state of the art in SCGF theory (Marino, 2026), and demonstrate that higher-order correlations beyond third order remain important. The method enables future applications to many-body perturbation theory where no regulator is required, and to larger model spaces through GPU acceleration and improved sampling weights.
