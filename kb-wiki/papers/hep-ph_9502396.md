# Confinement and scaling in deep inelastic scattering

> We show that parton confinement in the final state generates large $1/Q^2$ corrections to Bjorken scaling, thus leaving less room for the logarithmic corrections. In particular, the $x$-scaling violations at large $x$ are entirely described in terms of power corrections. For treatment of these non-perturbative effects, we derive a new expansion in powers of $1/Q^2$ for the structure function that is free of infra-red singularities and which reduces corrections to the leading term. The leading term represents scattering from an off-mass-shell parton, which keeps the same virtual mass in the final state. It is found that this quasi-free term is a function of a new variable $\bar x$, which coincides with the Bjorken variable $x$ for $Q^2\to\infty$. The two variables are very different, however, at finite $Q^2$. In particular, the variable $\bar x$ depends on the invariant mass of the spectator particles. Analysis of the data at large $x$ shows excellent scaling in the variable $\bar x$, and determines the value of the diquark mass to be close to zero. $\bar x$-scaling allows us to extract the structure function near the elastic threshold. It is found to behave as $F_2\sim (1-x)^{3.7}$. Predictions for the structure functions based on $\bar x$-scaling are made.

## Key claim

Parton confinement in the final state generates large $1/Q^2$ corrections to Bjorken scaling, and at large $x$ the scaling violations are entirely described by these power corrections rather than by logarithmic QCD corrections; the leading quasi-free term in a new expansion is a function of a modified scaling variable $\bar x$ that differs substantially from the Bjorken variable $x$ at finite $Q^2$.

## Method

The author derives a new expansion of the nucleon structure function in powers of $1/Q^2$ by expanding the full Green's function of the Bethe-Salpeter equation around the operator $h = V - G_0^{-1}(P)$ rather than around the interaction $V$ alone, which eliminates infra-red singularities from confining final-state interactions. The leading term represents scattering from an off-mass-shell parton that retains the same virtual mass in the final state, yielding a structure function that depends only on a new scaling variable $\bar x$ defined in terms of the Bjorken variable $x$, the nucleon mass $M$, the spectator mass $m_s$, and $Q^2$. The first correction term is evaluated explicitly for a linear-rising confining potential and shown to be small.

## Key numbers

- The new scaling variable $\bar x$ is given by $\bar x = (x + \sqrt{1+4M^2x^2/Q^2} - \sqrt{(1-x)^2+4m_s^2x^2/Q^2}) / (1 + \sqrt{1+4M^2x^2/Q^2})$, and for large $Q^2$ it relates to the Nachtmann variable $\xi$ as $\bar x \simeq \xi + M^2x^2/Q^2 - m_s^2x^2/[(1-x)Q^2]$.
- Analysis of BCDMS and SLAC-MIT proton structure function data at large $x$ determines the diquark mass to be close to zero, with the coefficient $C \approx 3$ (GeV)$^2$ in the spectator mass relation $m_s^2 \simeq m_d^2 + C(1-x)$.
- The extracted structure function near the elastic threshold behaves as $F_2 \sim (1-x)^{3.7}$.
- The first correction term relative to the leading term is estimated as ${\cal F}_1/(Q^2{\cal F}_0) \sim {\cal A}_1 y\Delta/Q^2$, where $\Delta = \gamma r_0$ is of order the first excitation energy, making the approximation valid for $Q^2 \gtrsim$ few (GeV/c)$^2$.

## Context

This work extends the nonrelativistic expansion method of Gersch, Rodriguez, and Smith (1973) to the relativistic case, showing that confining final-state interactions can be effectively incorporated into a modified scaling variable rather than treated as small higher-twist corrections. The approach contradicts the common assumption that logarithmic QCD corrections dominate scaling violations at moderate $Q^2$, and instead attributes the observed $Q^2$-dependence primarily to power corrections from confinement. The $\bar x$-scaling framework enables extraction of structure functions near the elastic threshold and provides predictions for structure functions at finite $Q^2$ that can be tested against new SLAC threshold-region measurements.
