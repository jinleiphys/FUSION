# On Size and Shape of the Average Meson Fields in the Semibosonized Nambu & Jona-Lasinio Model

> We consider a two-flavor Nambu \& Jona-Lasinio model in Hartree approximation involving scalar-isoscalar and pseudoscalar-isovector quark-quark interactions. Average meson fields are defined by minimizing the effective Euklidean action. The fermionic part of the action, which contains the full Dirac sea, is regularized within Schwinger's proper-time scheme. The meson fields are restricted to the chiral circle and to hedgehog configurations. The only parameter of the model is the constituent quark mass $M$ which simultaneously controls the regularization. We evaluate meson and quark fields self-consistently in dependence on the constituent quark mass. It is shown that the self-consistent fields do practically not depend on the constituent quark mass. This allows us to define a properly parameterized reference field which for physically relevant constituent masses can be used as a good approximation to the exactly calculated one. The reference field is chosen to have correct behaviour for small and large radii. To test the agreement between self-consistent and reference fields we calculate several observables like nucleon energy, mean square radius, axial-vector constant and delta-nucleon mass splitting in dependence on the constituent quark mass. The agreement is found to be very well. Figures available on request.

## Key claim

The self-consistent meson profile function \(\Theta(r)\) in the semibosonized Nambu & Jona-Lasinio model is nearly independent of the constituent quark mass \(M\) over a wide range (350 MeV to 1000 MeV), allowing a single analytic reference profile to approximate the exact numerical solution for physically relevant masses.

## Method

The authors solved the two-flavor Nambu & Jona-Lasinio model in Hartree approximation with scalar-isoscalar and pseudoscalar-isovector interactions, restricting meson fields to the chiral circle and to hedgehog configurations. The fermionic Dirac sea was regularized using Schwinger's proper-time scheme, and the profile function \(\Theta(r)\) was determined iteratively by diagonalizing the quark hamiltonian in a discrete basis within a spherical box of radius \(D = 15/M\). Self-consistent profiles were computed for 13 values of the constituent quark mass \(M\) between 350 and 1000 MeV, and compared to a reference profile derived from asymptotic expansions of the equation of motion at small and large radii.

## Key numbers

- Constituent quark mass range studied: \(350\ \mathrm{MeV} \le M \le 1000\ \mathrm{MeV}\)
- Box radius used in calculations: \(D = 15/M\)
- Maximal momentum cutoff for basis: \(K_{max} = 8 M\)
- Reference profile radius parameter: \(R = 0.42\ \mathrm{fm}\)
- Critical mass where valence level dips into Dirac sea: \(M_{dip} \approx 750\ \mathrm{MeV}\) for self-consistent profiles; shifted to \(M \approx 725\ \mathrm{MeV}\) for the reference profile
- Axial-vector coupling constant from reference profile: \(g_A \approx 0.6\) to \(0.8\) (compared to experimental value \(g_A^{exp} \approx 1.25\))
- Physically relevant constituent mass region: around \(M = 400\ \mathrm{MeV}\)

## Context

This work extends earlier self-consistent soliton solutions of the Nambu & Jona-Lasinio model for baryon number B=1 (Dyakonov et al., 1988; Wakamatsu, 1991; Meissner et al., 1991) by demonstrating that the meson profile function is nearly independent of the constituent quark mass. The reference profile provides a computationally efficient alternative to the time-consuming iterative determination of self-consistent fields, analogous to how the Woods-Saxon potential approximates nuclear mean fields. The approach enables systematic studies of NJL model properties without repeating the full self-consistent calculation for each parameter value, though the authors note that the too-small axial-vector coupling constant is a limitation shared by many chiral nucleon models.
