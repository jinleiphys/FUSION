# Reconstruction of bottomonium spectral functions in thermal QCD using Kernel Ridge Regression

> We discuss results for bottomonium at nonzero temperature obtained using NRQCD on Fastsum Generation 2L ensembles, as part of the Fastsum collaboration's programme to determine the spectrum of the bottomonium system as a function of temperature using a variety of approaches. Here we give an update on results for spectral functions obtained using Kernel Ridge Regression. We pay in particular attention to the generation of training data and introduce the notion of using lattice QCD ensembles to learn how to improve the generation of training data. A practical implementation is given.

## Key claim

Kernel Ridge Regression (KRR) can reconstruct bottomonium spectral functions from Euclidean lattice correlators, but the quality of the reconstruction depends critically on generating training data that overlaps with the actual lattice QCD correlators; the authors introduce a method to improve the training set by using the actual correlators to select representative mock spectral functions, though the resulting spectral functions show ground and first excited state masses smaller than expected.

## Method

The authors used NRQCD on anisotropic Fastsum Generation 2L ensembles with N_f=2+1 flavours of clover-improved Wilson fermions, varying temperature by changing N_tau from 16 to 128. They generated 20,000 mock spectral functions as sums of 5 Gaussian peaks, trained a KRR model on 15,000 of them, and then refined the training set by selecting the correlators closest to actual lattice QCD correlators and resampling new parameter values from Gaussian distributions around those selected points.

## Key numbers

- Lattice spacing: a_s = 0.01136(6) fm, a_tau = 5.997(34) GeV^-1, anisotropy xi = 3.453(6)
- Pion mass: m_pi = 236(2) MeV, with m_pi L = 4.36
- Pseudocritical temperature: T_pc = 164(2) MeV
- Temperature range: 47 MeV (N_tau=128) to 375 MeV (N_tau=16)
- Additive constant for bottomonium: Delta M = 7.465 GeV
- Training set size: 20,000 mock spectral functions, 15,000 used for training
- Number of Gaussian peaks per spectral function: N_p = 5, giving 15 parameters per function
- R^2 score for ground state mass: R^2_{m_0} = 0.869
- R^2 score for ground state width: 0.148
- R^2 score for ground state amplitude: 0.336
- Optimal hyper-parameters for N_tau=20: (ln(gamma), ln(lambda)) = (-1.25, -3.25)
- Optimal hyper-parameters for N_tau=128: (ln(gamma), ln(lambda)) = (-1.5, -2.25)

## Context

This work extends the Fastsum collaboration's programme of determining bottomonium spectral functions at nonzero temperature (Aarts 2010, Aarts 2011, Aarts 2014) and builds on the introduction of KRR for this purpose by Offler (2019). The key advance is using actual lattice QCD correlators to guide the generation of training data, addressing a mismatch between training and real data that was particularly severe for P-wave channels. This enables improved spectral reconstruction, though the authors note the Gaussian peak parametrisation may be a limiting factor, and they point to alternative methods presented at the same conference by Tom and Ben (2021) for comparison.
