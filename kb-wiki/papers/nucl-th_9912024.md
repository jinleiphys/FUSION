# Three-body matrix elements for calculations of mean field and exp(S) ground sate correlations

> In this document we present our approach to the computation of three-body matrix elements, based on the Urbana family of three-nucleon potentials. The calculations refer only to the necessary matrix elements needed to include the three-nucleon interaction in the manner presented in nucl-th/9912023.

## Key claim

The paper presents a computational method for evaluating three-body matrix elements of the Urbana family of three-nucleon potentials, specifically those needed for mean-field and exp(S) ground-state correlation calculations, but the authors explicitly restrict the work to the matrix elements required by the companion paper nucl-th/9912023, not to all possible three-body matrix elements.

## Method

The authors decompose the Urbana three-nucleon interaction into a two-pion exchange long-range part and a phenomenological short-range repulsion term, then express the matrix elements using Fourier transforms and expansions in harmonic oscillator basis functions. They separate the computation into density-dependent matrix elements (Section 2) and exchange matrix elements (Section 3), handling the cyclic permutations of particle indices by evaluating three separate matrix elements for each term. The radial integrals are performed using Gauss integration quadrature, with the radial functions expanded in harmonic oscillator functions whose Fourier transforms are also harmonic oscillator functions.

## Key numbers

- For the Urbana IX potential, the fitted parameters are: \(A_{2\pi} = -0.0293\) and \(U_0 = +0.0048\) (these are dimensionless parameters of the three-nucleon interaction).
- The short-range repulsion term uses \(T^2(r_{12})T^2(r_{13})\) with the radial function \(\tilde T(q) = \int r_{12}^2 dr_{12} T^2(r_{12}) j_0(q r_{12})\).
- The two-pion exchange term involves coupling constants \(K_2\) and \(K_3\) which take values 0 (for the \(\sigma\cdot\sigma\) term) or 2 (for the tensor term).

## Context

This work extends the computational framework for three-body forces in nuclear structure calculations, building on the Urbana potential series described in prior work by the Urbana group (Pieper, Wiringa, and collaborators). The method is designed to be used in conjunction with the companion paper nucl-th/9912023 for mean-field and exp(S) ground-state correlation calculations. The approach enables the inclusion of realistic three-nucleon interactions in nuclear many-body calculations beyond the three-body system, using harmonic oscillator basis expansions and Gauss integration techniques previously developed for two-body matrix elements.
