# Fragmentation Function in Non-Equilibrium QCD Using Closed-Time Path Integral Formalism

> In this paper we implement Schwinger-Keldysh closed-time path integral formalism in non-equilibrium QCD to the definition of Collins-Soper fragmentation function. We consider a high p_T parton in QCD medium at initial time t_0 with arbitrary non-equilibrium (non-isotropic) distribution function f(\vec{p}) fragmenting to hadron. We formulate parton to hadron fragmentation function in non-equilibrium QCD in the light-cone quantization formalism. It may be possible to include final state interactions with the medium via modification of the Wilson lines in this definition of the non-equilibrium fragmentation function. This may be relevant to study hadron production from quark-gluon plasma at RHIC and LHC.

## Key claim

The paper derives gauge-invariant definitions of the Collins-Soper parton-to-hadron fragmentation function for quarks and gluons in a non-equilibrium QCD medium, using the Schwinger-Keldysh closed-time path (CTP) formalism. The key generalization is that the vacuum expectation value in the standard definition is replaced by an average over an initial non-equilibrium state \(|in\rangle\), and the normalization factor acquires a factor \([1+f(\vec{k})]\) involving the arbitrary non-equilibrium distribution function \(f(\vec{k})\) of the fragmenting parton.

## Method

The author starts from the standard Collins-Soper fragmentation function in vacuum, expressed in light-cone quantization. The vacuum state \(|0\rangle\) and vacuum expectation values are replaced by the initial non-equilibrium state \(|in\rangle\) of the QCD medium, defined at an initial time \(\tau_0\) (or \(x^+_{in}=0\)). The non-equilibrium distribution function \(f(\vec{k})\) for quarks or gluons is introduced via the relation \(\langle in| a^\dagger(\vec{k}) a(\vec{k}') |in\rangle = f(\vec{k}) (2\pi)^{d-1} \delta^{(d-1)}(\vec{k}-\vec{k}')\). The derivation follows the same steps as the vacuum case but uses the modified normalization of the parton state \(\langle k|k'\rangle = (2\pi)^{d-1} \delta(k^+-k'^+) \delta^{d-2}(k_T-k'_T) [1+f(k^+,k_T)]\) to obtain the non-equilibrium fragmentation function. Gauge invariance is restored by inserting Wilson lines, as in the vacuum case.

## Key numbers

* The quark fragmentation function in non-equilibrium QCD is given by Eq. (qnf) and Eq. (qnonf), which includes a normalization factor \(1/[1+f_q(k^+,k_T)]\) where \(f_q\) is the non-equilibrium quark distribution function at initial time.
* The gluon fragmentation function in non-equilibrium QCD is given by Eq. (gnf) and Eq. (gnonf), which includes a normalization factor \(1/[1+f_g(k^+,k_T)]\) where \(f_g\) is the non-equilibrium gluon distribution function at initial time.
* The special case of equilibrium finite-temperature QCD is recovered when \(f(\vec{p}) = 1/(e^{p_0/T} \pm 1)\).
* The paper works in \(d=4-2\epsilon\) dimensions, with \(3-2\epsilon\) being the spatial dimension.

## Context

This work extends the Collins-Soper fragmentation function (Collins, 1981; Collins and Soper, 1982) from vacuum QCD to non-equilibrium QCD media, relevant for heavy-ion collisions at RHIC and LHC where the quark-gluon plasma may be out of equilibrium. It implements the Schwinger-Keldysh closed-time path formalism (Schwinger, 1961; Keldysh, 1964) to define the fragmentation function in terms of an initial non-equilibrium state \(|in\rangle\). The author notes that final-state interactions with the medium could be included via modification of the Wilson lines, similar to the treatment of the \(p_T\) distribution of parton distribution functions studied by Belitsky, Ji, and Yuan (2003). This formalism enables first-principles studies of hadron production from a non-equilibrium quark-gluon plasma.
