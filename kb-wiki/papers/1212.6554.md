# Further results on peripheral-tube model for ridge correlation

> Peripheral one-tube model has shown to be a nice tool for dynamically understanding several aspects of ridge structures in long-range two-particle correlations, observed experimentally and obtained also in our model calculations using NexSPheRIO code. Here, we study an extension of the model, to initial configurations with several peripheral tubes distributed randomly in azimuth. We show that the two-particle correlation is almost independent of the number of tubes, although the flow distribution becomes indeed strongly event dependent. In our picture, the ridge structures are causally connected not only in the longitudinal direction but also in azimuth.

## Key claim

The two-particle correlation in the peripheral-tube model is almost independent of the number of randomly distributed peripheral tubes (2, 3, or 4), even though the single-particle flow distributions become strongly event-dependent; this indicates that ridge structures are determined locally by each tube's expansion rather than by the global initial condition.

## Method

The authors extended their previous boost-invariant one-tube model to multi-tube configurations by placing 2, 3, or 4 identical Gaussian energy tubes at a fixed radial distance of 5.4 fm from the axis, with random azimuthal positions, on top of an isotropic background. They performed 3D hydrodynamic calculations for each configuration using 50 random events per case, computing the Fourier components \(v_n\) of the single-particle azimuthal distributions, the symmetry angles \(\Psi_n\), and the two-particle correlations in \(\Delta\phi\).

## Key numbers

- Tube radial position fixed at \(r_0 = 5.4\,\text{fm}\) from the axis.
- Energy distribution parametrized as \(\epsilon = 12\exp[-0.0004 r^5] + \sum_{i=1}^n \frac{34}{0.845\pi} \exp\left[-\frac{|\mathbf{r} - \mathbf{r}_i|^2}{0.845}\right]\) for \(n = 1, 2, 3, 4\) tubes.
- For the one-tube model, the Fourier coefficients are \(v_2 = 0.0569\), \(v_3 = 0.0740\), \(v_4 = 0.0479\), and \(v_5 = 0.0160\).
- For the one-tube model, \(\Psi_2 - \Psi_3 = \pi/6\) and \(\Psi_3 - \Psi_4 = \pi/12\).
- Only 50 random events were computed for each multi-tube case.

## Context

This work extends the peripheral one-tube model of Hama et al. (2012), which successfully reproduced ridge structures in long-range two-particle correlations observed in Au+Au collisions at 200A GeV using the NexSPheRIO hydrodynamic code with NEXUS fluctuating initial conditions. The multi-tube generalization shows that the ridge mechanism is local and causal, contradicting any interpretation that requires global initial-state geometry. The results enable further studies of event-by-event fluctuations and provide a unified causal description of both near-side and away-side ridges, complementing the longitudinal causality discussion of Dumitru (2008).
