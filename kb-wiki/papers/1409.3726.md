# The Hoyle state and its relatives

> The Hoyle state and other resonances in the continuum above the 3 alpha threshold in 12C are studied in a microscopic cluster model. Whereas the Hoyle state is a very sharp resonance and can be treated reasonably well in bound state approximation, the other higher lying states require a proper treatment of the continuum. The model space consists of an internal region with 3 alpha particles on a triangular grid and an external region consisting of the 8Be ground state and excited (pseudo)-states of 8Be with an additional alpha. The microscopic R-matrix method is used to match the many-body wave function to the asymptotic Coulomb behavior of bound states, Gamow states and scattering states. 8Be-alpha phase shifts are analyzed and resonance properties like radii and transition strengths are investigated.

# The Hoyle state and its relatives

## Key claim

The Hoyle state (the first excited 0⁺ state in ¹²C) can be treated reasonably well in a bound-state approximation because it is a very sharp resonance, but the other higher-lying resonances in the ¹²C continuum above the 3α threshold require a proper treatment of the continuum, which the authors implement via a microscopic R-matrix method with an internal region of three α particles on a triangular grid and an external region of ⁸Be+α configurations.

## Method

The authors use a microscopic cluster model with Brink-type wave functions for three α clusters, projected on parity and angular momentum. The internal region consists of three α particles placed on a triangular grid with grid spacing 1.75 fm, while the external region uses ⁸Be eigenstates (including the 0⁺ ground state, a second 0⁺ state, two 2⁺ states, and a 4⁺ state) combined with an additional α cluster at varying distances. The microscopic R-matrix method matches the many-body wave function to asymptotic Coulomb behavior for bound states, Gamow states, and scattering states, with the matching performed at a channel radius of 16.5 fm.

## Key numbers

- Hoyle state: resonance energy 0.29 MeV above the 3α threshold with a width of 18 eV (from Gamow boundary conditions)
- 0⁺ resonance at 4.11 MeV with width 120 keV, showing strong coupling between ⁸Be 0⁺ and 2⁺ channels
- 2⁺ resonance at 1.51 MeV (Γ = 0.32 MeV) decaying through the ⁸Be ground state
- Second 2⁺ resonance at 4.31 MeV (Γ = 0.14 MeV) decaying dominantly through the ⁸Be 2⁺ state with relative orbital angular momentum L=2
- 4⁺ resonance at 1.17 MeV with width 8 eV (member of ground state band)
- Second 4⁺ resonance at 4.06 MeV (Γ = 0.98 MeV) decaying dominantly through the ⁸Be ground state
- Hoyle state excitation energy: 7.76 MeV (Gamow state result), compared to experimental 7.65 MeV
- Hoyle state radius: real part 4.08 fm with small imaginary part (Gamow state), increasing from 3.68 fm in the smallest bound-state model space
- Monopole matrix element M(E0, 0₁⁺→0₂⁺): 6.02 + 0.01i e·fm² (Gamow), compared to experimental 5.47(9) e·fm²
- B(E2, 2₂⁺→0₁⁺): 2.11 + 1.41i e²·fm⁴ (Gamow), compared to experimental 1.57⁺⁰·¹⁴₋₀·₁₁ e²·fm⁴
- Coulomb barrier located at distances of about 10-12 fm with height about 1.5 MeV

## Context

This work extends earlier microscopic cluster model studies of the Hoyle state by Kamimura (1978, 1981) and the THSR ansatz of Funaki (2003), using a larger basis for ⁸Be wave functions and more ⁸Be pseudo-states than previous similar approaches by Descouvemont (1987) and Arai (2006). The calculations use the Volkov V2 interaction with α particle parameters from Kamimura that reproduce Hoyle state properties in agreement with experiment. The results enable future fully microscopic calculations within the fermionic molecular dynamics (FMD) approach with realistic nucleon-nucleon interactions, which could also describe β-transitions and T=1 states not accessible in an α-cluster model.
