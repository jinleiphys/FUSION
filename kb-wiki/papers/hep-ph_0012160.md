# Counting Form Factors of Twist-Two Operators

> We present a simple method to count the number of hadronic form factors based on the partial wave formalism and crossing symmetry. In particular, we show that the number of independent nucleon form factors of spin-n, twist-2 operators (the vector current and energy-momentum tensor being special examples) is n+1. These generalized form factors define the generalized (off-forward) parton distributions that have been studied extensively in the recent literature. In proving this result, we also show how the J^{PC} rules for onium states arise in the helicity formalism.

## Key claim

The paper proves that the number of independent nucleon form factors for spin-\(n\), twist-2 operators (such as the vector current and energy-momentum tensor) is exactly \(n+1\), using a counting method based on crossing symmetry and the partial wave formalism rather than direct time-reversal constraints.

## Method

The authors first note that the number of independent amplitudes is the same in all crossed channels, so they count the independent matrix elements \(\langle P\overline{P}|\hat O^{\mu_1\cdots\mu_n}| 0\rangle\) in the center-of-momentum frame. They classify the \(J^{PC}\) content of the twist-2 operator (which decomposes into angular momentum components \(J^P = 0^+, 1^-, \dots, n^{(-1)^n}\) with charge conjugation \(C = (-1)^n\)) and match these to the allowed \(J^{PC}\) states of a nucleon-antinucleon pair, enumerated using either non-relativistic orbital angular momentum or the helicity formalism. The number of independent matrix elements for each \(J^{PC}\) is then counted, and the pattern is extended to arbitrary \(n\).

## Key numbers

* For a spin-\(n\), twist-2 operator, the number of independent nucleon form factors is \(n+1\) (e.g., \(n=1\) gives 2 form factors, \(n=2\) gives 3, etc.).
* For a spinless state like the pion, the number of independent form factors for the same operators is \([n/2] + 1\).
* The vector current (\(n=1\)) has exactly 2 independent form factors for nucleons; a non-conserved vector current would have 3, but time-reversal symmetry forces the third to vanish.

## Context

This work extends the formalism of generalized (off-forward) parton distributions, which had been studied extensively in the years prior (Ji, 1997; Ji, 1998). The counting method provides a simple, general alternative to writing down all invariant form factors and imposing time-reversal symmetry directly. It enables systematic classification of form factors for any operator and hadron state, and the authors demonstrate the method by reproducing the known \(J^{PC}\) rules for onium states from the helicity formalism.
