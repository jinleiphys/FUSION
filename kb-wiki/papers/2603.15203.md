# Bridging Theory and Data: Correcting Nuclear Mass Models with Interpretable Machine Learning

> Nuclear mass prediction is one of the core issues in nuclear physics research, yet it faces the challenge of small-sample datasets with high complexity. This study introduces the Kolmogorov-Arnold Network (KAN) into the refinement of nuclear mass models, proposing an efficient and interpretable solution. By constructing the KAN-WS4 hybrid model, the prediction accuracy is significantly improved (the root mean square error is reduced from 0.3 MeV to 0.16 MeV). Furthermore, leveraging the intrinsic interpretability of KAN, feature importance analysis reveals that the proton number is the most critical factor influencing residuals, indicating potential systematic biases in proton-related terms within existing theoretical models. The method's generality is demonstrated across five mass models. This study shows that KAN provides a novel approach to small-sample, high-complexity scientific problems. Its interpretability facilitates the data-driven discovery of physical laws, promising broad applicability to key nuclear physics issues.

# Bridging Theory and Data: Correcting Nuclear Mass Models with Interpretable Machine Learning

## Key claim

The Kolmogorov-Arnold Network (KAN) hybrid model reduces the root-mean-square error of the WS4 nuclear mass model from approximately 0.3 MeV to 0.16 MeV across 2340 nuclides from AME2020, but the improvement is significantly weaker in the light nuclide region compared to medium-heavy nuclides, and the test set RMS error (0.205 MeV) is notably higher than the training set RMS error (0.157 MeV).

## Method

The authors decomposed nuclear mass into a smooth component from the WS4 theoretical model and a residual term, then trained a shallow KAN with one hidden layer to learn the residual using four input features: proton number Z, neutron number N, a pairing-related quantity P, and a shell-effect-related quantity S. The model was trained on 2340 nuclei (Z, N ≥ 8) from AME2020 with experimental uncertainties ≤ 100 keV, split 80% training and 20% test, and the same KAN framework was applied to correct five additional mass models (Bhagwat, DZ28, FRDM, HFB31, KTUY).

## Key numbers

- RMS error of KAN-WS4 hybrid model over the entire nuclide chart: 0.167 MeV
- RMS error on training set: 0.157 MeV; on test set: 0.205 MeV
- WS4 model baseline RMS error: 0.286 MeV (bare model) versus 0.286 MeV stated in abstract as approximately 0.3 MeV
- For the six corrected models, test set RMS errors after KAN correction range from 0.205 MeV (WS4) to 0.356 MeV (HFB31), with bare model errors ranging from 0.286 MeV (WS4) to 0.715 MeV (KTUY)
- Dataset size: 2340 nuclei selected from AME2020

## Context

This work extends the line of machine-learning corrections to nuclear mass models, which includes prior approaches such as PhysRevC.93.014311, PhysRevC.96.044308, and PhysRevLett.122.062502, by introducing the KAN architecture as an alternative to conventional deep learning methods that typically require larger datasets. The interpretability analysis identifying proton number Z as the most critical feature suggests systematic biases in proton-dependent terms (Coulomb energy, proton shell effects, symmetry energy) of existing macro-micro models like WS4 (Wang 2014) and FRDM (PhysRevLett.108.052501). This enables future work where KAN-learned residual patterns could be fed back to refine phenomenological formulas, and the framework could be applied to other small-sample, high-complexity nuclear physics problems such as charge radii, decay lifetimes, and fission barriers.
