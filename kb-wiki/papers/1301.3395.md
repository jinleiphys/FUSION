# Predictions for $p+$Pb Collisions at sqrt s_NN = 5 TeV

> Predictions for charged hadron, identified light hadron, quarkonium, photon, jet and gauge bosons in p+Pb collisions at sqrt s_NN = 5 TeV are compiled and compared. When test run data are available, they are compared to the model predictions.

## Key claim

This paper compiles model predictions for a wide range of observables in p+Pb collisions at sqrt s_NN = 5 TeV, comparing them to each other and to available test run data from ALICE, with the caveat that most calculations assume the proton circulates toward positive rapidity while the ALICE data use the opposite convention (Pb+p), requiring adjustments to the predictions.

## Method

The authors collected predictions from multiple theoretical approaches including Color Glass Condensate (CGC) saturation models (rcBK and IP-Sat), event generators (HIJING2.1, HIJINGBB2.0, AMPT), and collinear factorization perturbative QCD calculations. Each model group computed charged hadron distributions, identified light hadron spectra, quarkonium production, direct photon yields, jet observables, and gauge boson production at midrapidity and minimum bias, with some results at different rapidities and centralities. The predictions were compared to ALICE test run data from September 2012 where available.

## Key numbers

- Charged particle predictions are compiled for p+Pb collisions at sqrt s_NN = 5 TeV, with test run data from ALICE at the same energy
- In HIJING2.1, the Gaussian width for k_T broadening is given by <k^2_T> = [0.14 log(sqrt s/GeV) - 0.43] GeV^2/c^2, fitted to fixed-target p+A data
- HIJINGBB2.0 uses a cutoff parameter p_0(s,A) = 0.416 sqrt(s)^0.191 A^0.128 GeV/c for A+A collisions, varying from p_0 = 1.5 GeV/c at sqrt s = 20 GeV to 4.2 GeV/c at sqrt s = 5.5 TeV
- HIJING2.0 uses an energy-dependent cutoff varying from 1.7 to 3.5 GeV/c over the same energy range
- The IP-Sat model parameters from HERA fits include B_G = 4.0 GeV^-2, mu_0 = 1.17 GeV^2, A_g = 2.55, and lambda_g = 0.020
- The rcBK initial condition uses x_0 = 0.01 and Lambda = 0.241 GeV as the infrared scale

## Context

This work extends the JET Collaboration's program of systematic model comparisons from heavy-ion collisions to proton-nucleus collisions at LHC energies, providing a baseline for understanding cold nuclear matter effects before the 2013 p+Pb run. The predictions build on earlier saturation model developments by Albacete (2010) and Rezaeian (2012), event generator frameworks from Wang and Gyulassy (1991-1994), and the IP-Sat model fits to HERA data by Kowalski (2003, 2006). The compilation enables future comparisons to LHC p+Pb data to discriminate between shadowing, saturation, and multiple scattering mechanisms.
