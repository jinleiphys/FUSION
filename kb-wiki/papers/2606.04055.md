# Boson Models with Interactions of Arbitrary Order

> The paper considers quantal many-boson systems that are described by a rotationally invariant and boson-number conserving Hamiltonian. The properties of a generic model are studied which treats N bosons of p different kinds with non-zero angular momenta l_1,l_2,...,l_p, possibly augmented with a (number of) scalar s boson(s). The order k of the interaction between the bosons is arbitrary and closed formulas are given for matrix elements between N-boson states for any k if p=1 and p=2. A recursive procedure is defined for arbitrary k and p. With the expressions derived in the paper it is possible to express symbolically a Hamiltonian matrix element between N-boson states as a linear combination of k-body interaction matrix elements. More generally, the formulas allow the evaluation of matrix elements of tensor operators that are not necessarily scalar nor boson-number conserving. The numerical implementation of the formalism is discussed and illustrated with a few examples.

## Key claim

The paper presents a general formalism for computing matrix elements of rotationally invariant, boson-number-conserving Hamiltonians with interactions of arbitrary order \(k\) in systems of \(N\) bosons of \(p\) different kinds, but the closed-form expressions are only given for \(p=1\) and \(p=2\); for \(p>2\) a recursive procedure is defined rather than a closed formula.

## Method

The method uses coefficients of fractional parentage (CFPs) in a seniority basis for single-\(j\) bosons, computed via the Redmond recurrence relation and Gram–Schmidt orthonormalisation. For \(s\ell\)-IBM (one boson type plus a scalar \(s\) boson) and \(s\ell_1\ell_2\)-IBM (two boson types plus an \(s\) boson), closed-form expressions for matrix elements of \(k\)-body interactions and non-scalar operators are derived in terms of these CFPs. For more than two non-scalar boson types (\(p>2\)), a recursive procedure reduces the matrix element in \(\ell_1\dots\ell_p\)-IBM to products of matrix elements in \(\ell_i\)-IBM via angular-momentum recoupling with \(6j\) and \(9j\) symbols.

## Key numbers

* The multiplicity \(d_\ell(\upsilon,L)\) of angular momentum \(L\) for given seniority \(\upsilon\) in \(\ell\)-IBM is given by a contour integral: \(d_\ell(\upsilon,L)=\frac{i}{2\pi}\oint_{|z|=1}\frac{(z^{2L+1}-1)(z^{2\upsilon+2\ell-1}-1)\prod_{k=1}^{2\ell-2}(z^{\upsilon+k}-1)}{z^{\ell\upsilon+L+2}\prod_{k=1}^{2\ell-2}(z^{k+1}-1)}dz\).
* The seniority reduction formula for bosons (integer \(j\)) gives: \([j^{n-1}(\upsilon-1,\alpha_1J_1),j|\}j^n\upsilon\alpha J]=\bigl[\frac{\upsilon(2j-1+n+\upsilon)}{n(2j-1+2\upsilon)}\bigr]^{1/2}[j^{\upsilon-1}(\upsilon-1,\alpha_1J_1),j|\}j^\upsilon\upsilon\alpha J]\).
* The seniority reduction formula for bosons also gives: \([j^{n-1}(\upsilon+1,\alpha_1J_1),j|\}j^n\upsilon\alpha J]=(-)^{J+J_1}\bigl[\frac{(\upsilon+1)(n-\upsilon)(2J_1+1)}{n(2j+1+2\upsilon)(2J+1)}\bigr]^{1/2}[j^\upsilon(\upsilon\alpha J),j|\}j^{\upsilon+1}\upsilon+1,\alpha_1J_1]\).
* The \(s\)-boson matrix element factor is: \(\frac{\langle s^{n_s}|(s^\dag)^{m_s}(\tilde s)^{m'_s}|s^{n'_s}\rangle}{\sqrt{m_s!m'_s!}}=\bigl[\binom{n_s}{m_s}\binom{n'_s}{m'_s}\bigr]^{1/2}\delta_{n_s-m_s,n'_s-m'_s}\).

## Context

This work extends the interacting boson model (IBM) formalism, originally developed for \(s\) and \(d\) bosons by Iachello (1987) and for the vibron model by Iachello (1995), to arbitrary interaction order \(k\) and arbitrary number \(p\) of boson types with non-zero angular momenta. It generalises earlier treatments that relied on multi-\(j\) CFPs (Skouras, 1986) which become computationally prohibitive for large \(N\) or \(p\), and instead uses single-\(j\) CFPs with a recursive angular-momentum recoupling scheme. The formalism enables symbolic evaluation of Hamiltonian matrix elements for systems such as \(sdg\)-IBM (Devi, 1992), neutron–proton IBM (Arima, 1977), and isospin-invariant IBM (Elliott, 1980), and can handle operators that are not scalar or boson-number conserving.
