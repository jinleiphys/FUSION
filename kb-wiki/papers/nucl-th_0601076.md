# Long- and short-range correlations in the ab-initio no-core shell model

> In the framework of the ab-initio no-core shell model (NCSM), we describe the longitudinal-longitudinal distribution function, part of the inclusive (e,e') longitudinal response. In the two-body cluster approximation, we compute the effective operators consistent with the unitary transformation used to obtain the effective Hamiltonian. When short-range correlations are probed, the results display independence from the model space size and length scale. Long-range correlations are more difficult to model in the NCSM and they can be described only by increasing the model space or increasing the cluster size. In order to illustrate the model space independence for short-range observables, we present results for a large set of model spaces for 4He, and in 0-4hw model spaces for 12C.

## Key claim

In the two-body cluster approximation, the ab-initio no-core shell model (NCSM) can produce model-space- and frequency-independent results for the longitudinal-longitudinal distribution function at high momentum transfer (short-range correlations), but only when effective operators consistent with the unitary transformation are used; long-range correlations require larger model spaces or higher-order cluster approximations.

## Method

The authors computed the longitudinal-longitudinal distribution function ρLL(q), part of the inclusive (e,e') longitudinal response, for 4He and 12C using the NCSM with the CD-Bonn nucleon-nucleon interaction. They performed a unitary transformation to obtain effective one- and two-body operators in the two-body cluster approximation, then diagonalized the effective Hamiltonian in model spaces of varying size (up to 18ħΩ for 4He and 4ħΩ for 12C) and compared results using bare versus effective operators at different harmonic oscillator frequencies.

## Key numbers

- For 4He with Nmax=10 (10ħΩ) and ħΩ=19 MeV, the ground-state energy is -28.30 MeV; with ħΩ=28 MeV it is -27.56 MeV, compared to the exact CD-Bonn ground-state energy of -26.16 MeV.
- For q ≲ 3 fm⁻¹ in 4He, results using bare operators in the 18ħΩ model space at ħΩ=28 MeV are indistinguishable from those using effective operators.
- For q > 3 fm⁻¹ in 4He, bare-operator results deviate from renormalized values.
- For 12C, when using effective operators, all curves collapse into the same shape and agree for q ≳ 3 fm⁻¹ across different model spaces and frequencies.

## Context

This work extends the effective operator formalism previously applied to the Hamiltonian (Navratil 2003, Caurier 2002) and to general one- and two-body operators (Stetcu 2005) to the calculation of the Coulomb sum rule observable. It demonstrates that the two-body cluster approximation is sufficient for short-range correlations but not for intermediate- and long-range correlations, which require larger model spaces or higher-order cluster approximations. The results enable reliable calculations of short-range observables in heavier nuclei like 12C using small model spaces, and motivate future work including three-body forces, exchange currents, and relativistic corrections for direct comparison with experiment.
