# Entanglement entropy and Schmidt number as measures of delocalization of $α$ clusters in one-dimensional nuclear systems

> We calculated the von Neumann entanglement entropy and the Schmidt number of one dimentional (1D) cluster states and showed that these are useful measures to estimate entanglement caused by delocalization of clusters. We analyze system size dependence of these entanglement measures in the linear-chain $nα$ states given by Tohsaki-Horiuchi-Schuck-Röpke wave functions for 1D cluster gas states. We show that the Schmidt number is an almost equivalent measures to the von Neumann entanglement entropy when the delocalization of clusters occurs in the entire system but it shows different behaviors in a partially delocalized state containing localized clusters and delocalized ones. It means that the Rényi-2 entanglement entropy, which relates to the Schmidt number, is found to be almost equivalent to the von Neumann entanglement entropy for the full delocalized cluster system but it is less sensitive to the partially delocalized cluster system than the von Neumann entanglement entropy. We also propose a new entanglement measure which has a generalized form of the Schmidt number. Sensitivity of these measures of entanglement to the delocalization of clusters in low-density regions was discussed.

## Key claim

The von Neumann entanglement entropy and the Schmidt number are almost equivalent measures when cluster delocalization occurs throughout the entire system, but they show different behaviors in partially delocalized states containing both localized and delocalized clusters, with the Schmidt number being less sensitive to partial delocalization.

## Method

The author calculated the von Neumann entanglement entropy, Schmidt number, and a newly proposed generalized Schmidt number (Kγ with γ=log2) from the eigenvalues of the one-body density matrix for one-dimensional (1D) cluster states. The 1D-Tohsaki-Horiuchi-Schuck-Röpke (THSR) wave functions were used to describe linear-chain nα states (2α, 3α, 4α) and α+(2α) states, with the system size controlled by the Gaussian range parameter β. The entanglement measures and their spatial distributions were analyzed as functions of the system size.

## Key numbers

- The α-cluster size parameter b was chosen as b = 1.376 fm, same as in Suhara (2013).
- For a fully delocalized cluster state with constant occupation probability ρl = n/mV, the Schmidt number K equals mV/n and the entanglement entropy gives eS/n = mV/n.
- For a partially delocalized state with n−1 localized clusters and one delocalized cluster, in the large mV limit the Schmidt number K approaches n/(n−1), while Kγ approaches mV^(1−γ).
- A small value of ε = 0.02 fm was used to describe the 2α core in the α+(2α) wave function.

## Context

This work extends the analysis of entanglement entropy for 1D cluster states previously reported by the same author (Kanada-En'yo, 2014) and builds on the 1D-THSR wave functions introduced by Suhara (2013) for linear-chain 3α and 4α states in 12C and 16O. The study enables quantitative discrimination between fully delocalized cluster gas states and partially delocalized states where a cluster moves around a core, which is relevant to understanding α-cluster states in light nuclei such as 20Ne.
