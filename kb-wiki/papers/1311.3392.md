# Giant dipole resonance width and the universality of the Critical Temperature included Fluctuation Model

> The universality of the Critical Temperature included Fluctuation Model (CTFM) in explaining the evolution of the giant dipole resonance (GDR) width as a function of angular momentum is examined in the light of recent experimental data on $^{144}$Sm and $^{152}$Gd. We compare both the data sets with the phenomenological formula based on the CTFM and the thermal shape fluctuation model (pTSFM). The CTFM describes both the data sets reasonably well using the actual ground state GDR width ($Γ_0$) values, whereas, the pTSFM describes the $^{144}$Sm data well but is unable to explain the $^{152}$Gd data using a single value of $Γ_0$ for two excitation energies. These interesting results clearly indicate that the phenomenological CTFM can be used universally to describe the evolution of the GDR width with both angular momentum and temperature in the entire mass region. Moreover, it should provide new insights into the modification of the TSFM.

## Key claim

The Critical Temperature included Fluctuation Model (CTFM) can universally describe the evolution of the giant dipole resonance (GDR) width with both angular momentum and temperature across the entire mass region, using the actual ground state GDR width values, whereas the phenomenological thermal shape fluctuation model (pTSFM) fails for the heavier nucleus $^{152}$Gd when using a single value of the ground state width.

## Method

The authors re-analyzed experimental GDR width data for $^{144}$Sm (populated via $^{28}$Si + $^{116}$Cd at 125 and 140 MeV) and $^{152}$Gd (populated via $^{28}$Si + $^{124}$Sn at 149 and 185 MeV) by re-estimating average temperature and angular momentum values using a modified version of the statistical model code CASCADE with the Ignatyuk-Reisdorf level density prescription. They compared the data with both the pTSFM universal scaling function and the CTFM formalism, which incorporates a critical temperature below which the GDR width remains constant at the ground state value due to intrinsic GDR-induced fluctuations. For CTFM, the critical temperature was extracted by comparing the intrinsic GDR fluctuation ($\beta_{GDR}$) with the variance of deformation ($\Delta\beta$) from thermal fluctuations, and the ground state GDR width was estimated from deformation systematics and a spreading width parameterization.

## Key numbers

- For $^{144}$Sm: ground state GDR width $\Gamma_0$ = 4.4 MeV (consistent with experimental value 4.37 $\pm$ 0.15 MeV); critical temperature $T_c$ = 0.88 MeV (including shell effects) or 0.8 MeV (without shell effects); ground state deformation $\beta$ = 0.0874.
- For $^{152}$Gd: ground state GDR width $\Gamma_0$ = 5.7 MeV; critical temperature $T_c$ = 0.92 MeV (including shell effects), close to systematic value of 0.94 MeV; ground state deformation $\beta$ = 0.206.
- For $^{144}$Sm at 125 MeV: GDR widths range from 8.2 $\pm$ 0.5 MeV (J_avg = 27 $\hbar$, T_avg = 1.51 MeV) to 8.4 $\pm$ 0.5 MeV (J_avg = 41 $\hbar$, T_avg = 1.36 MeV).
- For $^{144}$Sm at 140 MeV: GDR widths range from 8.3 $\pm$ 0.4 MeV (J_avg = 41 $\hbar$, T_avg = 1.49 MeV) to 9.8 $\pm$ 0.6 MeV (J_avg = 54 $\hbar$, T_avg = 1.32 MeV).
- The pTSFM required $\Gamma_0$ = 3.8 MeV for $^{144}$Sm and two different values (3.8 and 4.8 MeV) for $^{152}$Gd at the two excitation energies.

## Context

This work extends the CTFM proposed by Pandit et al. (2012) to test its universality against recent experimental data on $^{144}$Sm from Ishikawa et al. (2013) and $^{152}$Gd from Dracoulis et al. (2010). The pTSFM, based on the thermal shape fluctuation model of Kusnezov et al. (1998), was previously found to underestimate GDR widths at higher angular momenta and low temperatures. The CTFM's success using actual ground state widths, combined with a universal correlation between average deformation and GDR width, suggests that the thermal shape fluctuation model should be modified to include the GDR-induced quadrupole moment, enabling more accurate predictions of nuclear deformation at finite temperature and angular momentum.
