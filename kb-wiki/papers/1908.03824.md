# Development and testing of an unstructured mesh method for whole plasma gyrokinetic simulations in realistic tokamak geometry

> In this work, we have formulated and implemented a mixed unstructured mesh-based finite element (FE)-Fourier decomposition scheme for gyrokinetic simulations in realistic tokamak geometry. An efficient particle positioning (particle-triangle mapping) scheme for the charge deposition and field scattering using an intermediate grid as the search index for triangles has been implemented and a significant speed-up by a factor of $\sim30$ is observed as compared with the brute force scheme for a medium-size simulation. The TRIMEG (TRIangular MEsh based Gyrokinetic) code has been developed. As an application, the ion temperature gradient (ITG) mode is simulated using the simplified gyrokinetic Vlasov-Poisson model. Our simulation and that using the ORB5 code for the DIII-D Cyclone case show reasonable agreement. As an additional application, ITG simulations using an ASDEX Upgrade equilibrium have been performed with density and temperature gradient profiles similar to the Cyclone case. Capabilities of the TRIMEG code for simulations with realistic experimental equilibria in the plasma core and in the whole plasma volume with open field lines are demonstrated.

## Key claim

The TRIMEG code demonstrates that a mixed unstructured mesh finite element-Fourier decomposition scheme can perform gyrokinetic simulations in realistic tokamak geometry including the open field line region, but the authors acknowledge that the current results use a simplified gyrokinetic Vlasov-Poisson model with several approximations (dominant terms only in equations of motion, adiabatic electrons, single toroidal harmonic, Dirichlet boundary conditions) and that more comprehensive physics models are needed for full fidelity.

## Method

The authors implemented a mixed particle-in-cell/particle-in-Fourier (PIC-PIF) scheme using unstructured triangular meshes in the (R,Z) plane with Fourier decomposition in the toroidal direction. An efficient particle positioning scheme was developed using intermediate rectangular grids as search indices to map markers to triangles, replacing the brute-force O(N*N_t) approach. The gyrokinetic Poisson equation was solved using the finite element method with PETSc, and the Runge-Kutta fourth-order integrator was used for particle pushing. Linear ion temperature gradient (ITG) mode simulations were performed using the simplified gyrokinetic Vlasov-Poisson model with adiabatic electrons.

## Key numbers

- Speed-up factor of approximately 30 for the particle positioning scheme compared to brute force for a medium-size simulation with radial grid number N_r=90 and 25.6 million markers
- Optimal speed-up values of 35.5, 35.7, and 36.3 for intermediate box sizes N_x=256, 512, and 1024 respectively
- Error in ∇·B/B well below 10^{-4} with upper limit of 10^{-3} using N_R=N_Z=129 grid points for equilibrium construction
- Convergence in growth rate achieved for radial grid number N_r≥64, marker number per triangle N/N_t>4, and time step dt≤0.5 (with numerical instability for dt>1)
- Good strong scaling demonstrated up to 1280 cores for simulations with 25.6 million markers
- DIII-D Cyclone case parameters: a/R_0=0.36, a/ρ_s=180, T_e/T_i=1, R_0/L_ref=1, κ_Ti=6.69, κ_n=2.23

## Context

This work extends the gyrokinetic simulation capability to whole plasma volume including open field line regions, building on approaches used in codes such as ORB5 (Jolliet 2007), XGC (Ku 2009), GTS (Wang 2015), and GTC/GTC-X (Nishimura 2006, Bao 2019). The unstructured mesh approach differs from ORB5 which does not include the open field line region, and the mixed PIC-PIF scheme offers an alternative to the multiple-patch structured mesh approach used in JOREK (Huysmans 2007). The benchmark against ORB5 using the DIII-D Cyclone case (Goerler 2016 intercode benchmark) shows reasonable agreement, and the demonstration with ASDEX Upgrade equilibrium (shot 34924 at 3.600s) shows capability for realistic experimental geometries. Future work requires implementing more comprehensive physics including fully nonlinear collision operators, sheath boundary conditions, neoclassical radial electric fields, and zonal flow physics.
