# Generalized coherent states satisfying the Pauli principle in a nuclear cluster model

> We propose a new basis state, which satisfies the Pauli principle in the nuclear cluster model. The basis state is defined as the generalized coherent state of the harmonic oscillator wave function using a pair of the creation operators and is orthogonal to the Pauli-forbidden states having smaller quanta. In the coherent basis state, the range parameter is changeable and controls the radial dilation. This property is utilized for the precise description of the relative motion between nuclear clusters. We show the reliability of this framework for the $2α$ system of $^8$Be in the semi-microscopic orthogonality condition model. We obtain the resonances and non-resonant continuum states of $2α$ with complex scaling. The resonance solutions and the phase shifts of the $α$-$α$ scattering agree with those using the conventional projection operator method to remove the Pauli-forbidden states. We further discuss the extension of the present framework to the multi-$α$ cluster systems using the SU(3) wave functions.

## Key claim

The paper proposes a new basis state for nuclear cluster models that automatically satisfies the Pauli principle by being orthogonal to Pauli-forbidden harmonic oscillator states, without needing the conventional projection operator method; the authors demonstrate this for the 2α system of ⁸Be but note that the method's extension to multi-α cluster systems using SU(3) wave functions is discussed rather than fully validated.

## Method

The authors construct generalized coherent states of the harmonic oscillator using the raising operator \(\hat{D}^\dagger = \hat{\vc{a}}^\dagger \cdot \hat{\vc{a}}^\dagger\), which increases the oscillator quanta by two without changing angular momentum. These coherent basis states, parameterized by a dilation parameter \(\beta\) that controls the radial range, are automatically orthogonal to all Pauli-forbidden HO states with lower quanta. The method is applied to the 2α system of ⁸Be within the semi-microscopic orthogonality condition model (OCM), using the Schmid-Wildermuth effective nucleon-nucleon interaction and complex scaling to obtain resonances and continuum states.

## Key numbers

- The HO range parameter for the α cluster single-nucleon state is \(\nu = 0.535\) fm\(^{-2}\), corresponding to a length \(b = 1.3672\) fm; the relative motion range parameter is \(\nu_{\rm rel} = 2\nu = 1.070\) fm\(^{-2}\) (\(b_{\rm rel} = 0.9667\) fm).
- The Pauli-forbidden states in the 2α system are defined by the condition of quanta \(N_{\rm PF} < 4\) for the relative motion; the coherent basis states use \(N = 4\) for \(0^+\), \(2^+\), and \(4^+\) states, and \(N = 6\) for the \(6^+\) state.
- For the \(0^+\) resonance of ⁸Be, the coherent basis method gives energy 0.294 MeV and decay width 0.014 MeV (experimental: 0.0918 MeV, \(5.57(25) \times 10^{-6}\) MeV).
- For the \(2^+\) resonance, the coherent basis method gives energy 3.01 MeV and decay width 1.65 MeV (experimental: 3.12(1) MeV, 1.513(15) MeV).
- For the \(4^+\) resonance, the coherent basis method gives energy 12.13 MeV and decay width 5.19 MeV (experimental: 11.44(15) MeV, \(\approx 3.5\) MeV).
- The bound-state test with enhanced nuclear potential (\(\delta = 0.05\) to 0.40) shows exact agreement between the coherent basis method and projection operator method for \(0^+\) and \(2^+\) energies, e.g., at \(\delta = 0.30\) both give \(-4.147\) MeV for \(0^+\) and \(-0.954\) MeV for \(2^+\).

## Context

This work extends the orthogonality condition model (OCM) approach of Saito (1968) by providing an alternative to Kukulin's projection operator method (1978) for removing Pauli-forbidden states, which becomes numerically demanding for multi-cluster systems such as 4α and 5α. The method builds on the generalized coherent state formalism of Perelomov (1986) and the symplectic Sp(3,R) algebra of Rowe (1985), and is demonstrated for the 2α system previously studied by Kamimura (1977) and Aoyama (2006). The approach enables future extension to multi-α cluster systems using SU(3) wave functions, as discussed in the paper.
