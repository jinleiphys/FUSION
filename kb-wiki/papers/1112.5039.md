# Shell Model description of the beta beta decay of 136Xe

> We study in this letter the double beta decay of 136Xe with emission of two neutrinos which has been recently measured by the EXO-200 collaboration. We use the same shell model framework, valence space, and effective interaction that we have already employed in our calculation of the nuclear matrix element (NME) of its neutrinoless double beta decay. Using the quenching factor of the Gamow-Teller operator which is needed to reproduce the very recent high resolution 136Xe (3He, t)136Cs data, we obtain a nuclear matrix element M(2nu)=0.025 MeV^(-1) compared with the experimental value M(2nu)=0.019(2) MeV^(-1).

## Key claim

The shell model calculation yields a two-neutrino double-beta decay nuclear matrix element for 136Xe of M(2nu)=0.025 MeV^(-1), which agrees with the experimental value of 0.019(2) MeV^(-1) only after using a quenching factor of q=0.45 extracted from charge-exchange reaction data, though the authors note that systematic errors in the absolute normalization of the Gamow-Teller strength from those reactions could modify the extracted quenching factor.

## Method

The authors performed large-scale shell model calculations using the valence space spanning the orbits 0g7/2, 1d5/2, 1d3/2, 2s1/2, and 0h11/2 (between N,Z=50 and N,Z=82) with the effective interaction gcn50:82, which is a renormalized G-matrix from the Bonn-C potential fitted to about 300 energy levels from approximately 90 nuclei. They computed the Gamow-Teller strength function for 136Xe and normalized the theoretical strength in the experimental energy window to the measured data from the 136Xe (3He, t)136Cs charge-exchange reaction, extracting a quenching factor q=0.45, then used this quenched operator to calculate the two-neutrino double-beta decay matrix element via the sum over intermediate 1+ states in 136Cs.

## Key numbers

- M(2nu) for 136Xe decay to 136Ba: theoretical value 0.025 MeV^(-1) compared to experimental value 0.019(2) MeV^(-1)
- Quenching factor q=0.45 extracted from normalizing theoretical B(GT-) strength to experimental data from the 136Xe (3He, t)136Cs reaction
- Excitation energy of the first 1+ state in 136Cs: 0.59 MeV, previously unknown
- With a minor variant of the interaction that improves quadrupole properties of 136Ba: q=0.48 and M(2nu)=0.021 MeV^(-1)
- Approximately 40% of the total 2nu matrix element comes from states above the experimentally studied energy window
- For 128Te decay to 128Xe: M(2nu)=0.037 MeV^(-1) using the newer quenching factor from 136Xe charge-exchange data
- For 130Te decay to 130Xe: M(2nu)=0.027 MeV^(-1) using the newer quenching factor from 136Xe charge-exchange data
- States beyond 10 MeV excitation energy above the first 1+ state in the intermediate odd-odd nucleus do not contribute to M(2nu)

## Context

This work extends the authors' previous shell model calculations of neutrinoless double-beta decay matrix elements for 136Xe (Caurier, Nowacki, Poves, 2010) to the two-neutrino decay channel, which was recently measured by the EXO-200 collaboration and confirmed by KamLAND-Zen. The calculation uses the same valence space and effective interaction as the earlier neutrinoless decay work, and incorporates new charge-exchange reaction data from Frekers (2011) to determine the Gamow-Teller quenching factor. The results enable more reliable predictions for neutrinoless double-beta decay matrix elements by validating the nuclear structure description against the measured two-neutrino decay rate.
