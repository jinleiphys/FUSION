# End-to-end complexity for simulating the Schwinger model on quantum computers

> The Schwinger model is one of the simplest gauge theories. It is known that a topological term of the model leads to the infamous sign problem in the classical Monte Carlo method. In contrast to this, recently, quantum computing in Hamiltonian formalism has gained attention. In this work, we estimate the resources needed for quantum computers to compute physical quantities that are challenging to compute on classical computers. Specifically, we propose an efficient implementation of block-encoding of the Schwinger model Hamiltonian. Considering the structure of the Hamiltonian, this block-encoding with a normalization factor of $\mathcal{O}(N^3)$ can be implemented using $\mathcal{O}(N+\log^2(N/\varepsilon))$ T gates. As an end-to-end application, we compute the vacuum persistence amplitude. As a result, we found that for a system size $N=128$ and an additive error $\varepsilon=0.01$, with an evolution time $t$ and a lattice spacing a satisfying $t/2a=10$, the vacuum persistence amplitude can be calculated using about $10^{13}$ T gates. Our results provide insights into predictions about the performance of quantum computers in the FTQC and early FTQC era, clarifying the challenges in solving meaningful problems within a realistic timeframe.

## Key claim

The authors claim that for a system size \(N=128\) and additive error \(\varepsilon=0.01\), with evolution time \(t\) and lattice spacing \(a\) satisfying \(t/2a=10\), the vacuum persistence amplitude of the Schwinger model can be calculated using about \(10^{13}\) T gates, but this estimate assumes the initial state is a simple computational basis state (Néel state) rather than the true ground state, which avoids the additional cost of ground state preparation that previous estimates in quantum chemistry and condensed matter have neglected.

## Method

The authors construct a block-encoding of the Schwinger model Hamiltonian in the formulation where Gauss's law is solved to eliminate the electric field degree of freedom, resulting in all-to-all interactions among fermions. They decompose the Hamiltonian into six terms (\(H_{XX}\), \(H_{YY}\), \(H_{Z}\), \(H_{Z\text{even}}\), \(H_{Z\text{odd}}\), and \(H_{Z^2}\)) and implement each via PREPARE and SELECT operators, using techniques such as uniform superposition state preparation, fixed-point amplitude amplification, and the Chebyshev polynomial method to square block-encoded operators. The full block-encoding is obtained by taking a linear combination of these six block-encoded terms using a split operator \(P_{\mathrm{split}}\), and the time evolution operator \(e^{-iHt}\) is implemented via quantum eigenvalue transformation.

## Key numbers

* Block-encoding normalization factor scales as \(\mathcal{O}(N^3)\), with explicit expression \(\alpha_S = w(N-1) + \frac{m}{2}N + \frac{J\theta}{2\pi} \sum_{l=1, l: \mathrm{even}}^{N-1} l + \left(\frac{J\theta}{2\pi}+\frac{J}{2}\right) \sum_{l=1, l: \mathrm{odd}}^{N-1} l + \frac{J}{8}\sum_{l=1}^{N-1} l^2\).
* T gate count for block-encoding is \(\mathcal{O}(N+\log^2(N/\varepsilon))\), with explicit formula: \(20N + 4 d\left(8\left\lceil\log_2{\left(\frac{28d\alpha_S}{\varepsilon}\right)}\right\rceil + 8 \lceil \log_2 N \rceil +2C-2\right) +312 \left\lceil \log_2 \left(\frac{546 \alpha_S}{\varepsilon}\right)\right\rceil + 124 \lceil \log_2 N \rceil +38\lceil \log_2 N' \rceil + 38\lceil \log_2 N'' \rceil + f + 78 C + 216\), where \(d\) is an odd integer satisfying \(d \geq \sqrt{2} \ln{(2\sqrt{14\alpha_S/\varepsilon})}\) and \(C \approx 10.09\).
* For system size \(N=128\), additive error \(\varepsilon=0.01\), and evolution time satisfying \(t/2a=10\), the vacuum persistence amplitude requires \(1.72 \times 10^{13}\) T gates.
* Ancilla qubit count for block-encoding is \(6\lceil\log_2 N \rceil + \max{(2\lceil \log_2 N' \rceil + \lceil \log_2 (N'-1) \rceil, 3\lceil \log_2 N'' \rceil)} + 6\), where \(N' = \lceil N/2 \rceil\) and \(N'' = \lfloor N/2 \rfloor\).

## Context

This work extends prior resource estimation for the Schwinger model using Trotterization by Nguyen et al. (2022), which achieved \(\mathcal{O}\left(N^{4+1/p} t^{1+1/p}/\varepsilon^{1/p}\right)\) cost with \(p\)-th order product formulas, and the state-of-the-art algorithm by Tong et al. (2022) that implements \(e^{-iHt}\) in time \(\widetilde{\mathcal{O}}\left(N t\ \mathrm{polylog}\left(\Lambda_0 \varepsilon^{-1}\right)\right)\) but requires more qubits and ancillary qubits for HAM-T oracles. The block-encoding approach presented here is motivated by the distinctive structure of the Schwinger model Hamiltonian, which has all-to-all interactions but only a constant number of parameters (four) independent of system size, making it amenable to efficient block-encoding techniques from Babbush et al. (2018) and von Burg et al. (2021). This work enables concrete resource estimates for early fault-tolerant quantum computers (FTQC) to compute quantities like the vacuum persistence amplitude that are classically intractable due to the sign problem in Monte Carlo methods.
