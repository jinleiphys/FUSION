# Multiple $SU(3)$ algebras in interacting boson model and shell model: Results for $(β,γ$) bands and scissors $1^+$ band

> Shell model and interacting boson model spaces admit multiple $SU^{(α)}(3)$ algebras generating the same rotational spectra but different $E2$ decay properties, depending on the phases $α$ in the quadrupole generator. In the ground ($g$) $K=0^+$ bands in nuclei this is demonstrated recently using systems with nucleons in a single oscillator shell [Kota, Sahu and Srivastava, Bulg. J. Phys. {\bf 46}, 313 (2019); Eur. Phys. J. Special Topics {\bf 229}, 2389 (2020)]. Going beyond these preliminary studies, results are presented here for $E2$ decay properties of $β$ and $γ$ bands members, as generated by multiple $SU(3)$ algebras, using $sdg$IBM and $sdgi$IBM examples. In addition, results are presented for the $E2$ and $M1$ decay properties of the levels of the $1^+$ scissors band in heavy nuclei using $sdg$IBM-2 and $sdgi$IBM-2. The scissors $1^+$ band properties are also studied using a shell model example with six protons in $(pf)$ shell and twelve neutrons in $(sdg)$ shell. These results establish that: (i) with multiple $SU(3)$ algebras, it is possible to have rotational bands with very weak $E2$ strengths among the levels where normally one expects strong strengths; (ii) $E2$ decay of the levels of $β$ and $γ$ bands to the ground band are quite different for some of the $SU^{(α)}(3)$ algebras with strong dependence on $α$; (iii) it is possible to have the scissors $1^+$ band with the $E2$ and $M1$ decay of the low-lying levels of this band to the $g$ band are strong or weak depending on $α$.

## Key claim

The paper demonstrates that multiple SU(3) algebras, distinguished by phase choices in the quadrupole generator, produce rotational bands with identical spectra but dramatically different E2 decay properties for beta, gamma, and scissors 1+ bands, with the specific claim that it is possible to have rotational bands with very weak E2 strengths where normally one expects strong strengths, depending on the phase parameter α.

## Method

The authors used analytical large-N intrinsic state methods in sdgIBM and sdgiIBM (interacting boson models) to compute B(E2) values for transitions from beta and gamma bands to the ground band, employing the Kuyucak-Morrison formalism for angular momentum projection. For the shell model, they performed deformed shell model (DSM) calculations for a system of eight protons in the sdg shell, using Hartree-Fock single-particle states with band mixing and angular momentum projection. For scissors 1+ band properties, they extended the analysis to sdgIBM-2 and sdgiIBM-2 (proton-neutron IBM) and a shell model example with six protons in the pf shell and twelve neutrons in the sdg shell.

## Key numbers

- For sdgIBM with 10 bosons, B(E2; 2_g → 0_g) = 137.05 (q₂)²b² for (α_sd,α_dg)=(+,+), but only 3.16 (q₂)²b² for (-,+)
- For sdgIBM with 10 bosons, B(E2; 2_γ → 0_g) = 0 for (+,+), 10.19 for (-,+), 39.1 for (+,-), and 89.2 (q₂)²b² for (-,-)
- For sdgIBM with 10 bosons, B(E2; 2_β → 0_g) = 0 for (+,+), 42.44 for (-,+), 22.91 for (+,-), and 2.99 (q₂)²b² for (-,-)
- For sdgiIBM with 10 bosons, B(E2; 2_g → 0_g) ranges from 902.95 (q₂)²b² for (+,+,+) to 33.47 (q₂)²b² for (+,-,-)
- For sdgiIBM with 10 bosons, B(E2; 2_γ → 0_g) ranges from 0 for (+,+,+) to 491.91 (q₂)²b² for (-,-,-)
- For sdgiIBM with 10 bosons, B(E2; 2_β → 0_g) ranges from 0 for (+,+,+) to 189.43 (q₂)²b² for (-,+,+)
- For the shell model (sdg)⁸p system, DSM results give B(E2; 2⁺_g → 0⁺_g) = 747 e²fm⁴ for (-,-) and 18 e²fm⁴ for (-,+)
- For the shell model (sdg)⁸p system, B(E2; 2⁺_γ → 0⁺_g) = 25 e²fm⁴ for (-,-) and 0.2 e²fm⁴ for (+,-)

## Context

This work extends preliminary studies by Kota, Sahu and Srivastava (2019, 2020) that demonstrated multiple SU(3) algebras in ground K=0⁺ bands, now applying the framework to beta and gamma bands in sdgIBM and sdgiIBM and to scissors 1⁺ bands in IBM-2 and shell model. The results enable future experimental searches for signatures of different SU(3) phase choices across the periodic table, and open the possibility of studying quantum phase transitions and shape coexistence using multiple SU(3) algebras, as well as order-chaos-order transitions previously investigated with multiple pairing and SU(3) algebras in sdIBM-1.
