# New QCD Sum Rules for Nucleon Axial Vector Coupling Constants

> New QCD sum rules for the nucleon axial-vector current coupling constants are derived. One set is derived from a generalized correlator of spin-1/2 interpolating fields, and the other with a mixed correlator of spin-1/2 and spin-3/2 fields. Using a recently-developed Monte-Carlo based uncertainty analysis and techniques for determining the optimal interpolating fields, the predicative ability of QCD sum rules for axial-vector current coupling constants are determined. We find the previous studies of this matrix element are based on an invalid sum rule. A combined analysis of this sum rule together with the nucleon mass sum rules yields $g_A=1.48\pm^{1.06}_{0.65}$. The relative error of approximately 60% is large compared to a 10% error for the nucleon mass obtained from the same input parameters.

# New QCD Sum Rules for Nucleon Axial Vector Coupling Constants

## Key claim
The paper derives new QCD sum rules for the nucleon axial-vector coupling constant \(g_A\) and finds that previous studies of this matrix element were based on an invalid sum rule. The combined analysis yields \(g_A = 1.48 \pm^{1.06}_{0.65}\), but the relative error of approximately 60% is large compared to a 10% error for the nucleon mass obtained from the same input parameters.

## Method
Two sets of new QCD sum rules were derived using the external field method with generalized nucleon interpolating fields. One set came from a correlator of spin-1/2 interpolating fields \(\eta_{1/2} = \eta_1 + \beta \eta_2\) (where \(\beta\) is a real parameter), and another set of 8 sum rules came from a mixed correlator of spin-1/2 and spin-3/2 fields. A Monte Carlo based uncertainty analysis was applied for the first time to a three-point function, exploring the complete QCD parameter space to determine how uncertainties in QCD input parameters propagate to phenomenological fit parameters; sum rules that did not meet criteria for OPE convergence (last term less than 10% of total OPE side) and ground-state dominance (excited state contributions no more than 50% of phenomenological side) were discarded as invalid.

## Key numbers
- The valid sum rule at the structure \(iZ_\mu \sigma^{\mu\nu} p_\nu \gamma_5\) has a Borel window from 0.91 GeV to 1.16 GeV with optimal mixing \(\beta = -1.2\).
- Combined analysis of this sum rule with nucleon mass sum rules yields \(g_A = 1.48 \pm^{1.06}_{0.65}\).
- The nucleon mass from the same input parameters is \(M_N = 0.94 \pm 0.10\) GeV.
- The pole residue is \(\tilde{\lambda}^2_{1/2} = 0.12 \pm 0.6\) GeV\(^6\).
- The transition strength is \(A_3 = 1.08 \pm 0.61\) GeV\(^{-1}\).

## Context
This work extends the QCD sum rule approach to three-point functions for the first time with a comprehensive Monte Carlo uncertainty analysis, building on the uncertainty analysis method developed by Leinweber (1995) for nucleon mass sum rules. The paper contradicts previous studies of \(g_A\) (referenced as "old" works) by showing that the sum rule they used does not have a valid Borel window, making their predictions unreliable. The results enable future analysis of other axial couplings and highlight that the large uncertainty in \(g_A\) originates from the poorly determined pole residue and the vacuum susceptibility \(\kappa_v\), suggesting that improved determination of these quantities is needed for more precise predictions.
