# Global Analysis of Nuclear PDFs

> We present a new global QCD analysis of nuclear parton distribution functions. In addition to the most commonly analyzed data sets for deep inelastic scattering of charged leptons off nuclei and Drell Yan di-lepton production, we include also measurements for neutrino-nuclei scattering as well as inclusive pion production in deuteron-gold collisions. The emerging picture is one of consistency, where universal nuclear modification factors for each parton flavor reproduce the main features of all data without any significant tension among the different sets.

## Key claim

The paper presents a new global QCD analysis of nuclear parton distribution functions (nPDFs) that, for the first time, simultaneously fits charged lepton deep inelastic scattering, Drell-Yan di-lepton production, neutrino-nuclei scattering, and inclusive pion production in deuteron-gold collisions, finding that universal nuclear modification factors for each parton flavor can reproduce all data without significant tension, but only if nuclear parton-to-hadron fragmentation functions (nFFs) are used for the pion production data, and with the caveat that the gluon modification factor \(R_g^{Au}\) exhibits only moderate nuclear corrections, in contrast to earlier extractions that used vacuum fragmentation functions and large data weights.

## Method

The authors performed a next-to-leading order (NLO) global QCD fit of nuclear modification factors \(R_i^A(x,Q_0)\) for valence quarks, sea quarks, and gluons, parametrized at an initial scale \(Q_0 = 1\ \mathrm{GeV}\) with 25 free parameters, using a Hessian method for uncertainty estimation with a tolerance criterion of \(\Delta\chi^2 = 30\). The fit included 1579 data points from charged lepton DIS (894 points), neutrino DIS (532 points), Drell-Yan di-lepton production (92 points), and inclusive pion production in deuteron-gold collisions (61 points), with statistical and systematic errors added in quadrature and no artificial weights applied to any data set.

## Key numbers

* Total \(\chi^2\) for the optimum fit: 1544.7 for 1579 data points, giving \(\chi^2/\mathrm{d.o.f.} = 0.994\).
* Charged lepton DIS: \(\chi^2 = 897.52\) for 894 data points.
* Neutrino DIS: \(\chi^2 = 488.20\) for 532 data points.
* Drell-Yan di-lepton production: \(\chi^2 = 90.72\) for 92 data points.
* Pion production in deuteron-gold collisions: \(\chi^2 = 68.26\) for 61 data points, with \(\chi^2_{dAu}/n = 1.12\) when using nuclear fragmentation functions, compared to \(\chi^2_{dAu}/n = 1.37\) when using vacuum fragmentation functions.
* The fit used an initial scale \(Q_0 = 1\ \mathrm{GeV}\) for the nPDF parametrization.

## Context

This work extends previous nPDF analyses by EPS09 (Ref. [ref:eps09]) and others (Schienbein et al., Paukkunen, Kovarik et al.) by including neutrino-nuclei scattering data from NuTeV, CDHSW, and CHORUS, as well as inclusive pion production in deuteron-gold collisions from RHIC, using a consistent NLO framework with a general mass variable flavor number scheme. The authors find that, unlike the EPS09 analysis which used vacuum fragmentation functions and a large weight for the dAu data to drive a strong gluon modification, their fit with nuclear fragmentation functions yields only moderate gluon corrections; they note that the CDHSW neutrino data at \(Q^2 < 10\ \mathrm{GeV}^2\) show a different slope than other data sets, suggesting systematic discrepancies that require further investigation. The resulting nPDFs supersede the authors' earlier work (Ref. [ref:nds]) and provide a baseline for future constraints from prompt photon and Drell-Yan di-lepton production in dAu and pPb collisions at RHIC and the LHC.
