# Instructive discussion of an effective block algorithm for baryon-baryon correlators

> We describe an approach for the efficient calculation of a large number of four-point correlation functions for various baryon-baryon (BB) channels, which are the primary quantities for studying the nuclear and hyperonic nuclear forces from lattice quantum chromodynamics. Using the four-point correlation function of a proton-$Λ$ system as a specific example, we discuss how an effective block algorithm significantly reduces the number of iterations. The effective block algorithm is applied to calculate 52 channels of the four-point correlation functions from nucleon$-$nucleon to $Ξ-Ξ$, in order to study the complete set of isospin symmetric BB interactions. The elapsed times measured for hybrid parallel computation on BlueGene/Q demonstrate that the performance of the present algorithm is reasonable for various combinations of the number of OpenMP threads and the number of MPI nodes. The numerical results are compared with the results obtained using the unified contraction algorithm for all computed sites of the 52 four-point correlators.

## Key claim

The paper describes an effective block algorithm for computing baryon-baryon four-point correlation functions that significantly reduces the number of iterations compared to naive contraction methods, but the reduction factor depends on the specific diagram and the number of quark exchanges between baryons, with the algorithm being applied specifically to isospin symmetric BB interactions from nucleon-nucleon to Xi-Xi.

## Method

The algorithm classifies Wick contraction diagrams into baryon-block pairs and uses Fast-Fourier-Transform (FFT) to evaluate the spatial convolution of these blocks, reducing the computational cost from O(L^6) to O(L^3 log L^3). For the p-Lambda system as a specific example, the authors construct baryon blocks where internal colour and spinor summations are performed prior to the FFT, with explicit indices only remaining for exchanged quark fields between the two baryons. The method is then generalised to 52 channels of four-point correlation functions covering all isospin symmetric baryon-baryon systems from nucleon-nucleon to Xi-Xi.

## Key numbers

- The reduction factor for the first diagram (no quark exchange) is (N_c! N_alpha)^2 x 2^(B - N_Lambda - N_Sigma0) / 1 = 1152
- The reduction factor for the second diagram (one u-quark exchange) is (N_c! N_alpha)^2 x 2^(B - N_Lambda - N_Sigma0) / (N_c^2) = 128
- The algorithm was applied to calculate 52 channels of four-point correlation functions from nucleon-nucleon to Xi-Xi
- The computational cost improves from O(L^6) to O(L^3 log L^3) by using FFT

## Context

This work extends the HAL QCD method for studying nuclear and hyperonic forces from lattice QCD, which was originally proposed by Ishii (2006) and developed by the HAL QCD Collaboration. The effective block algorithm differs from the unified contraction algorithm of Doi (2012) and has been previously used to calculate Lambda-N and Sigma-N potentials by Nemura (2009, 2010, 2012). The approach enables simultaneous calculation of a large number of Nambu-Bethe-Salpeter wave functions for various baryon-baryon channels in a single lattice QCD calculation, which is crucial for studying flavour symmetry breaking in isospin symmetric BB interactions.
