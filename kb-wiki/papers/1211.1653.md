# Accuracy of a new hybrid finite element method for solving a scattering Schroedinger equation

> This hybrid method (FE-DVR), introduced by Resigno and McCurdy, Phys. Rev. A 62, 032706 (2000), uses Lagrange polynomials in each partition, rather than "hat" functions or Gaussian functions. These polynomials are discrete variable representation functions, and are.orthogonal under the Gauss-Lobatto quadrature discretization approximation. Accuracy analyses of this method are performed for the case of a one dimensional Schroedinger equation with various types of local and nonlocal potentials for scattering boundary conditions. The accuracy is ascertained by comparison with a spectral Chebyshev integral equation method, accurate to 1:10^(-11). For an accuracy of the phase shift of 1:10^(-8) The FE-DVR method is found to be 100 times faster than a sixth order finite difference method (Numerov), is easy to program, and can routinely achieve an accuracy of better than 1:10^(-8) for the numerical examples studied.

# Accuracy of a new hybrid finite element method for solving a scattering Schroedinger equation

## Key claim
The FE-DVR hybrid method can routinely achieve phase shift accuracy better than 1:10^(-8) for the numerical examples studied, and is approximately 100 times faster than a sixth-order finite difference (Numerov) method for that accuracy level, but this performance holds only for the specific one-dimensional local and nonlocal potentials tested and the accuracy degrades with increasing partition size or number of Lobatto points beyond an optimal value.

## Method
The authors implement the FE-DVR method by dividing the radial interval into partitions and expanding the wave function in each partition using Lagrange polynomials defined on Gauss-Lobatto quadrature points. Continuity of the wave function and its derivative between partitions is enforced by transforming the homogeneous Galerkin equations in each partition into inhomogeneous equations of reduced dimension, using the wave function value and derivative from the previous partition as driving terms. Accuracy is assessed by comparing FE-DVR solutions for one-dimensional Schroedinger equations with local Morse and Woods-Saxon potentials and a nonlocal Perey-Buck potential against a spectral Chebyshev integral equation method (S-IEM) accurate to 1:10^(-11).

## Key numbers
- The S-IEM reference method is accurate to 1:10^(-11).
- For an accuracy of the phase shift of 1:10^(-8), the FE-DVR method is 100 times faster than a sixth-order finite difference (Numerov) method.
- The FE-DVR method requires 15 times fewer meshpoints than the Numerov method to achieve tan(delta) accuracy of approximately 10^(-8).
- For the nonlocal Perey-Buck potential, a single partition from 0 to 15 fm using 130 Lobatto grid points achieves accuracy of 10^(-8).
- For the Woods-Saxon potential with wave number k=0.5 fm^(-1), partition size 1 fm, and 20 Lobatto points per partition, the wave function error starts at 10^(-11) at small distances and increases to 10^(-10) at larger distances.
- For the Morse potential V_M, the accuracy of the integral in Eq. (B10) shows nearly exponential increase with the number N of Lobatto points per partition until N exceeds approximately 20, beyond which roundoff error accumulation dominates.
- The computing time per floating point operation in MATLAB on the described desktop system is approximately 2x10^(-8) seconds.
- For 20 partitions the computation time is 0.060 s; for 100 partitions it is 0.075 s.

## Context
This work extends the FE-DVR hybrid method introduced by Rescigno and McCurdy (2000) for quantum scattering calculations by providing a detailed accuracy analysis that was not performed in prior applications. The method is compared against a spectral Chebyshev integral equation method (S-IEM) and against the sixth-order Numerov finite difference method. The results enable reliable use of the FE-DVR method for scattering problems with local and nonlocal potentials, with the advantage that the Galerkin matrix elements of the kinetic energy operator need not be recalculated for each partition because they are the same to within a normalization factor. The authors note that a possible disadvantage is the accumulation of roundoff and algorithm errors when the number of Lagrange polynomials per partition or the number of partitions becomes very large, as would be the case for long-ranged potentials.
