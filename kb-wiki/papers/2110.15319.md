# A novel event generator for the automated simulation of neutrino scattering

> An event generation framework is presented that enables the automatic simulation of events for next-generation neutrino experiments in the Standard Model or extensions thereof. The new generator combines the calculation of the leptonic current based on an automated matrix element generator, and the computation of the hadronic current based on a state-of-the-art nuclear physics model. The approach is validated in Standard-Model simulations for electron scattering and neutrino scattering. Furthermore, the first fully-differential neutrino trident production results are shown in the quasielastic region.

## Key claim

The paper presents a new event generation framework that combines an automated matrix element generator (Comix, part of Sherpa) for the leptonic current with a state-of-the-art nuclear physics model (the correlated-basis function spectral function formalism) for the hadronic current, enabling automatic simulation of neutrino scattering events for Standard Model or beyond-Standard-Model processes; the validation is limited to the quasielastic region and does not include final state interaction effects, which the authors note would shift peak locations and broaden tails.

## Method

The leptonic current is computed using the color-dressed Berends-Giele recursive relations (Dyson-Schwinger based technique) implemented in the Comix matrix element generator within the Sherpa event generation framework, with an interface to the UFO file format for arbitrary new physics models. The hadronic current is computed in the impulse approximation using the correlated-basis function (CBF) spectral function formalism for Carbon-12, where the nuclear current is expressed as a sum of one-body currents and the nuclear final state is written as a struck nucleon plus an A-1 spectator remnant. The phase-space integration uses recursive techniques combined with the multi-channel method and the Vegas adaptive integration algorithm, and the full differential cross section is obtained by squaring the sum of leptonic and hadronic currents to automatically include interference terms.

## Key numbers

- Axial-vector coupling constant: g_A = 1.2694, with axial mass M_A = 1.0 GeV used in the dipole parameterization of the axial form factor
- Fermi momentum for Pauli blocking: k_F = 225 MeV, applied to ensure the outgoing nucleon momentum exceeds this value
- Total cross section for neutrino trident production (nu_mu + C-12 -> nu_mu e+ e- X) at 1 GeV beam energy: 3.973e-14 +/- 2.764e-17 nb, consistent with results from Ballett (2018)
- Electroweak parameters used: alpha = 1/137, G_F = 1.16637e-5 GeV^-2, M_Z = 91.1876 GeV
- Minimum opening angle between the two electrons in trident production: 5 degrees; minimum electron energy: 30 MeV; minimum angle from neutrino beam axis: 10 degrees

## Context

This work extends the automated event generation toolchain developed for LHC physics (FeynRules, UFO file format, Sherpa/Comix) to the neutrino-nucleus scattering domain, enabling systematic exploration of beyond-Standard-Model hypotheses against experimental data from DUNE, Hyper-Kamiokande, MicroBooNE, and MiniBooNE. The nuclear physics approach builds on the spectral function formalism of Benhar (1994) and Rocco (2018), using the correlated-basis function spectral function for Carbon-12, and the validation compares against electron scattering data from Sealock (1989) and prior neutrino scattering calculations by Rocco (2018). The first fully-differential neutrino trident production results in the quasielastic region demonstrate the framework's ability to handle processes beyond 2-to-2 scattering, which is relevant for distinguishing Standard Model tridents from beyond-Standard-Model explanations of the MiniBooNE excess, such as those proposed by Bertuzzo (2018).
