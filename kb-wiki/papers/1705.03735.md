# Proton structure fluctuations: constraints from HERA and applications to p+A collisions

> We constrain proton structure fluctuations at small-$x$ by comparing with the HERA diffractive vector meson production data, and find that large geometric fluctuations in the proton wave function are needed. Hydrodynamical simulations of proton-nucleus collisions including the fluctuating proton structure are found to be compatible with the momentum anisotropies measured at the LHC.

## Key claim

The authors find that large geometric fluctuations in the proton wave function are required to describe HERA diffractive vector meson production data, and that hydrodynamical simulations of proton-nucleus collisions using these fluctuating protons produce momentum anisotropies compatible with LHC measurements, in contrast to earlier calculations without geometric fluctuations that largely underestimated the harmonic flow coefficients.

## Method

The authors model proton structure fluctuations using a constituent quark inspired picture where small-x gluons are located around three valence quarks (hot spots), with the hot spot positions sampled from a Gaussian distribution of width B_qc = 3.0 GeV^-2 and each hot spot having a Gaussian density profile of width B_q = 0.3 GeV^-2. They constrain these parameters by comparing calculated coherent and incoherent diffractive J/Psi photoproduction cross sections with H1 data from HERA, then use the resulting fluctuating proton configurations as initial conditions for hydrodynamical simulations of proton-lead collisions at sqrt(s_NN) = 5.02 TeV using the IP-Glasma framework and the MUSIC hydrodynamic code with second order transport parameters, effective shear viscosity eta/s = 0.2, and temperature-dependent bulk viscosity.

## Key numbers

* Hot spot width B_q = 0.3 GeV^-2 and hot spot position spread B_qc = 3.0 GeV^-2, obtained by fitting H1 diffractive J/Psi photoproduction data
* Hydrodynamical evolution starts at tau_0 = 0.2 fm/c, with switching temperature T_switch = 155 MeV for hadronization via Cooper-Frye prescription and UrQMD cascade
* Proton-lead collisions simulated at sqrt(s_NN) = 5.02 TeV, with elliptic (v_2) and triangular (v_3) flow compared to CMS data

## Context

This work extends the IP-Glasma framework by incorporating geometric fluctuations of the proton, which were absent in earlier calculations by Schenke (2014) that largely underestimated harmonic flow coefficients. The fluctuating proton structure constrained by HERA data enables realistic hydrodynamical simulations of small collision systems (p+A and potentially p+p) at LHC energies. The necessity of large geometric fluctuations was also found to improve the description of exclusive J/Psi production in ultraperipheral heavy ion collisions at the LHC, as shown in a related paper by the same authors.
