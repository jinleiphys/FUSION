# Two-body Coulomb scattering and complex scaling

> The two-body Coulomb scattering problem is solved using the standard complex scaling method. The explicit enforcement of the scattering boundary condition is avoided. Splitting of the scattering wave function based on the Coulomb modified plane wave is considered. This decomposition leads a three-dimensional Schrödinger equation with source term. Partial wave expansion is carried out and the asymptotic form of the solution is determined. This splitting does not lead to simplification of the scattering boundary condition if complex scaling is invoked. A new splitting carried out only on partial wave level is introduced and this method is proved to be very useful. The scattered part of the wave function tends to zero at large inter-particle distance. This property permits of easy numerical solution: the scattered part of the wave function can be expanded on bound-state type basis. The new method can be applied not only for pure Coulomb potential butin the presence of short range interaction too.

## Key claim

The authors show that the standard complex scaling method can be applied to two-body Coulomb scattering without any cutoff of the long-range interaction, but only if the splitting of the wave function is performed at the partial-wave level rather than using the three-dimensional Coulomb modified plane wave splitting.

## Method

The total scattering wave function is split at the partial-wave level into a known incoming part (the Coulomb Jost solution) and an unknown scattered part, leading to a radial driven Schrödinger equation with a source term. Complex scaling is then applied by replacing the radial coordinate \(r\) with \(re^{i\theta}\), which makes the scattered part of the wave function square integrable and allows expansion on bound-state-type basis functions. The resulting differential equation is solved numerically using the finite element method with Lobatto shape functions.

## Key numbers

- Momentum \(k = 3\) and Sommerfeld parameter \(\gamma = 1/3\) are used for the pure Coulomb test case with orbital angular momentum \(l = 0\).
- The complex scaling angle \(\theta = 0.1\) radian is used in the numerical calculations.
- With 50 Lobatto shape functions per element and boundary condition imposed at \(R = 250\), the exact phase shift is reproduced with three digits accuracy.
- With 100 Lobatto shape functions per element, four digits agreement with the exact solution is achieved almost everywhere.

## Context

This work extends the standard complex scaling method, previously applicable only to short-range potentials (Nuttall and Cohen 1969), to pure Coulomb scattering without the artificial cutoff required by exterior complex scaling methods (Rescigno and McCurdy 1997, Volkov and Yakubovsky 2009). The new partial-wave splitting avoids the failure of the Coulomb modified plane wave splitting, which the authors show still contains incoming wave components after complex scaling. The method enables straightforward numerical solution of two-body Coulomb scattering and can be extended to cases where a short-range potential is added to the Coulomb interaction.
