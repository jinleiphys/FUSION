# Gamow-Teller transitions and magnetic moments using various interactions

> In a single j-shell calculation we consider the effects of several different interactions on the values of Gamow-Teller (B(GT)'s) and magnetic moments. The interactions used are MBZE, J=0 pairing, J_{max} pairing and half and half.

## Key claim

In a single j-shell calculation restricted to the f7/2 shell, the values of Gamow-Teller B(GT) and magnetic moments depend strongly on the choice of interaction (MBZE, J=0 pairing, Jmax pairing, and half-and-half), but the abstract's claim of studying "effects" glosses over the fact that care must be taken with isospin crossings and degeneracies, and that the results are obtained using bare operators with no quenching.

## Method

The authors performed shell-model calculations in the f7/2 single j-shell using four different interactions: the empirical MBZE interaction (derived from experimental two-body matrix elements of 42Sc), J=0 pairing (only the J=0 T=1 matrix element nonzero at -1), Jmax pairing (only the J=7 matrix element nonzero at -1), and half-and-half (both J=0 and J=7 matrix elements set to -1). They computed Gamow-Teller B(GT) values and magnetic moments for various nuclei in the f7/2 region (43Sc, 44Sc, 45Sc, 45Ti, 46Ti, and their decay partners) using the bare GT operator, and compared results to experimental data where available.

## Key numbers

- For the transition 42Sc (I=7+) → 42Ca (I=6+), the experimental B(GT) is 0.2699, while the theoretical value assuming (f7/2)2 configuration is 0.2743, giving a quenching factor of 0.992.
- For 43Sc (I=3.5) → 43Ca (I=2.5), B(GT) values are: MBZE = 0.1181, J=0 = 0, J=Half = 0.0592, J=7 = 0.2434, experiment = 0.0326.
- For 43Sc (I=3.5) → 43Ca (I=3.5), B(GT) values are: MBZE = 0.1682, J=0 = 0.5713, J=Half = 0.2747, J=7 = 0.0397.
- For 44Sc (I=2) → 44Ca (I=2), B(GT) values are: MBZE = 0.0505, J=0 = 0.0613, J=Half = 0.0142, J=7 = 0.0259, experiment = 0.01962.
- For 45Ti (I=3.5) → 45Sc (I=3.5), B(GT) values are: MBZE = 0.1466, J=0 = 0.1499, J=Half = 0.1732, J=7 = 5.89×10−4, experiment = 0.0980.
- For 46Ti (I=4) → 46V (I=4), B(GT) values are: MBZE = 0.0065, J=0 = 0.0166, J=Half = 0.2898, J=7 = 2.03×10−4, experiment = 0.0025.
- For 46Ti (I=1) → 46V (I=0), B(GT) values are: MBZE = 0.0789, J=0 = 0, J=Half = 0.0367, J=7 = 0.2332, experiment = 0.0196.
- Magnetic moment of 43Sc (spin 3.5): MBZE = 4.324, J=0 = 3.614, J=Half = 4.204, J=7 = 4.328, experiment = +4.62.
- Magnetic moment of 45Ti (spin 3.5): MBZE = −0.604, J=0 = −0.891, J=Half = −0.779, J=7 = −0.377, experiment = 0.095 (sign undetermined).

## Context

This work extends the single j-shell studies of Gamow-Teller transitions and magnetic moments in the f7/2 region by Escuderos et al. (experimental two-body matrix elements) and Harper and Zamick (who studied M1 transitions and seniority selection rules). The results show that the MBZE interaction, while giving values in the right ballpark, has significant deviations from experiment, indicating the need for configuration mixing beyond the single j-shell. The paper also clarifies a discrepancy with Harper and Zamick regarding monotonic versus non-monotonic behavior in 46Ti by showing that constraining isospin to be unchanged yields simpler monotonic trends, whereas allowing isospin crossings (as Harper and Zamick did by taking the lowest energy state regardless of isospin) produces non-monotonic results.
