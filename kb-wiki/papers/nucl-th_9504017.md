# Isospin-mixing corrections for fp-shell Fermi transitions

> Isospin-mixing corrections for superallowed Fermi transitions in {\it fp}-shell nuclei are computed within the framework of the shell model. The study includes three nuclei that are part of the set of nine accurately measured transitions as well as five cases that are expected to be measured in the future at radioactive-beam facilities. We also include some new calculations for $^{10}$C. With the isospin-mixing corrections applied to the nine accurately measured $ft$ values, the conserved-vector-current hypothesis and the unitarity condition of the Cabbibo-Kobayashi-Maskawa (CKM) matrix are tested.

## Key claim

Isospin-mixing corrections for superallowed Fermi transitions in fp-shell nuclei are computed using shell-model calculations, and when applied to the nine accurately measured ft values, the conserved-vector-current (CVC) hypothesis is satisfied, but the unitarity condition of the Cabbibo-Kobayashi-Maskawa (CKM) matrix is violated at the 3-4 sigma level, requiring an additional uniform correction of 0.3-0.4% that lies outside acceptable nuclear correction uncertainties.

## Method

The authors performed shell-model calculations using the FPD6 interaction for A ≤ 50, FPD6* for A=54, and both FPD6* and FPVH interactions for 58 ≤ A ≤ 74, with model-space truncations applied to all nuclei except 46V and 74Rb. The isospin-mixing correction δ_C was factored into two components: δ_IM from configuration mixing within the valence fp-shell, evaluated using an isospin-nonconserving (INC) interaction derived from Coulomb and nuclear forces, and δ_RO from radial overlap deviations, computed using both Hartree-Fock (Skyrme M* force) and Woods-Saxon single-particle wave functions.

## Key numbers

- For A < 58, δ_IM is of order 0.02-0.10%; for heavier nuclei (A ≥ 62), δ_IM can be as large as 0.4%
- For A ≤ 54, the difference between HF and WS evaluations of δ_RO is 0.02-0.2%; for A ≥ 58, this difference ranges from 0.3-0.7%
- The error-weighted average corrected Ft value for the nine accurately measured transitions is 3150.8 ± 1.2 ± 2.5 s with χ²/ν = 0.66
- Using v_us = 0.2199(17) and v_ub < 0.0075, the CKM unitarity condition is found to be 0.9956 ± (0.0008)_stat ± (0.0007)_sys
- For 54Co, the best value of δ_IM is 0.04(1)%, based on the renormalized interaction approach of Towner (1989)
- The theoretical uncertainty in δ_C for A ≤ 54 is of order 0.09%, from 0.05% for δ_IM, 0.06% for δ_RO, and 0.05% for spectator mismatch

## Context

This work extends the shell-model calculations of isospin-mixing corrections previously performed by Ormand (1989) and Towner (1977) for lighter nuclei, now applied to fp-shell transitions including 46V, 50Mn, and 54Co from the set of nine accurately measured transitions, plus five heavier cases (58Zn, 62Ga, 66As, 70Br, 74Rb) expected for future radioactive-beam measurements. The results enable tests of the CVC hypothesis and CKM matrix unitarity, and the larger corrections and greater divergence between HF and WS methods for heavier nuclei suggest that precise measurements of those cases could discriminate between the two approaches for δ_RO.
