# From nucleon-nucleon interaction matrix elements in momentum space to an operator representation

> Starting from the matrix elements of the nucleon-nucleon interaction in momentum space we present a method to derive an operator representation with a minimal set of operators that is required to provide an optimal description of the partial waves with low angular momentum. As a first application we use this method to obtain an operator representation for the Argonne potential transformed by means of the unitary correlation operator method and discuss the necessity of including momentum dependent operators. The resulting operator representation leads to the same results as the original momentum space matrix elements when applied to the two-nucleon system and various light nuclei. For applications in fermionic and antisymmetrized molecular dynamics, where an operator representation of a soft but realistic effective interaction is indispensable, a simplified version using a reduced set of operators is given.

## Key claim

The authors present a method to derive an approximate operator representation from partial-wave matrix elements of a nucleon-nucleon interaction, using a fit that prioritizes accurate reproduction of low-momentum, low-angular-momentum channels rather than exact reproduction across all momenta and partial waves, which is sufficient for describing light nuclei.

## Method

Starting from the exact partial-wave matrix elements of the UCOM-transformed Argonne V18 potential in momentum space, the authors define an ansatz operator representation with radial functions parameterized as sums of Gaussians. The unknown parameters in the ansatz are obtained by fitting the matrix elements of the ansatz to the exact matrix elements, separately for central, spin-orbit, and tensor contributions in each spin-isospin channel. The fit uses a geometrical sequence of Gaussian width parameters from 0.05 fm² to 6.4 fm², and the resulting operator representation is tested in two-nucleon and few-nucleon systems.

## Key numbers

- The UCOM(SRG) transformation uses flow parameters of α = 0.04 fm⁴ (UCOM(0.04)) and α = 0.20 fm⁴ (UCOM(0.20)).
- The Gaussian width parameters for the radial function parameterization follow a geometrical sequence: κ₁ = 0.05 fm² with factor b = 2, giving κ = {0.05, 0.1, 0.2, ..., 6.4} fm².
- The maximum width parameter of 6.4 fm² corresponds to the interaction range of a few fm; the minimum width of 0.05 fm² corresponds to momentum transfers up to roughly 1/√(0.05) ≈ 4.5 fm⁻¹.

## Context

This work extends the unitary correlation operator method (UCOM) as developed by Feldmeier (1998), Neff (2003), and Roth (2004, 2010) by providing a systematic fitting procedure to obtain simplified operator representations from momentum-space matrix elements. The method enables the use of soft realistic effective interactions in fermionic molecular dynamics (FMD) and antisymmetrized molecular dynamics (AMD), which require explicit operator representations rather than basis-dependent matrix element representations used in no-core shell model (NCSM) calculations. The approach is analogous to coarse-grained potentials (PhysRevC.88.064002) in that low-momentum matrix elements are insensitive to short-length-scale variations in the radial functions.
