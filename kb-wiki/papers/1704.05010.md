# Constructing realistic alpha cluster channels

> We present techniques that allow for $α$-cluster channels with realistic $α$-particle wave functions from No Core Shell Model calculations to be constructed. We compare results of several clustering calculations with realistic $α$ wave functions to those assuming a trivial $(0s)^4$ structure.

## Key claim

The paper presents a numerical technique for constructing alpha-cluster channels that uses realistic alpha-particle wave functions from No Core Shell Model calculations, rather than the simplified \((0s)^4\) structure, and shows that the resulting spectroscopic factors differ from those obtained with the trivial approximation.

## Method

The authors start with a realistic alpha-particle wave function from No Core Shell Model calculations, apply the Glockner-Lawson procedure to obtain a state with the center-of-mass in a harmonic oscillator \(n=0\) state, and then use sequential applications of center-of-mass creation and annihilation operators to boost the alpha particle into any desired center-of-mass harmonic oscillator quantum numbers. The spectroscopic factors for parent-to-daughter transitions are then computed using standard shell model techniques with these boosted realistic alpha wave functions.

## Key numbers

- For \(^{20}\text{Ne}\rightarrow^{16}\text{O}\), the spectroscopic factor increases from 0.755 with \(N_{\text{max}}=0\) to 0.827 with \(N_{\text{max}}=6\) (experimental value 1), using \(\hbar\Omega = 14\) MeV.
- For \(^{22}\text{Ne}\rightarrow^{18}\text{O}\), the spectroscopic factor changes from 0.481 with \(N_{\text{max}}=0\) to 0.563 with \(N_{\text{max}}=6\) (experimental value 0.37), using \(\hbar\Omega = 14\) MeV.
- For \(^{24}\text{Mg}\rightarrow^{20}\text{Ne}\), the spectroscopic factor changes from 0.411 with \(N_{\text{max}}=0\) to 0.519 with \(N_{\text{max}}=6\) (experimental value 0.66), using \(\hbar\Omega = 14\) MeV.
- For \(^{28}\text{Si}\rightarrow^{24}\text{Mg}\), the spectroscopic factor changes from 0.526 with \(N_{\text{max}}=0\) to 0.575 with \(N_{\text{max}}=6\) (experimental value 0.33), using \(\hbar\Omega = 14\) MeV.
- The weight of the \((sd)^4\) configuration in the alpha wave function boosted by 8 quanta (\(L=0\)) is 0.038 for \(N_{\text{max}}=0\) and 0.035 for \(N_{\text{max}}=4\), with \(\hbar\Omega = 20\) MeV.

## Context

This work extends the shell model approach to clustering developed in a series of earlier works (Mang 1957, Smirnov 1977, Volya 2015) by removing the assumption that the alpha particle has a trivial \((0s)^4\) structure. The new technique enables clustering calculations with realistic cluster wave functions from No Core Shell Model calculations, which is numerically efficient and does not rely on the algebraic SU(3) methods used previously. The approach can be applied to other cluster systems and allows for more accurate spectroscopic factors that better reproduce experimental data in some cases.
