# New CTEQ global analysis of quantum chromodynamics with high-precision data from the LHC

> We present the new parton distribution functions (PDFs) from the CTEQ-TEA collaboration, obtained using a wide variety of high-precision Large Hadron Collider (LHC) data, in addition to the combined HERA I+II deep-inelastic scattering data set, along with the data sets present in the CT14 global QCD analysis. New LHC measurements in single-inclusive jet production with the full rapidity coverage, as well as production of Drell-Yan pairs, top-quark pairs, and high-$p_T$ $Z$ bosons, are included to achieve the greatest sensitivity to the PDFs. The parton distributions are determined at NLO and NNLO, with each of these PDFs accompanied by error sets determined using the Hessian method. Fast PDF survey techniques, based on the Hessian representation and the Lagrange Multiplier method, are used to quantify the preference of each data set to quantities such as $α_s(m_Z)$, and the gluon and strange quark distributions. We designate the main resulting PDF set as CT18. The ATLAS 7 TeV precision $W/Z$ data are not included in CT18, due to their tension with other data sets in the global fit. Alternate PDF sets are generated including the ATLAS precision 7 TeV $W/Z$ data (CT18A), a new scale choice for low-$x$ DIS data (CT18X), or all of the above with a slightly higher choice for the charm mass (CT18Z). Theoretical calculations of standard candle cross sections at the LHC (such as the $gg$ fusion Higgs boson cross section) are presented.

## Key claim

The CTEQ-TEA collaboration presents new parton distribution functions (CT18) that incorporate high-precision LHC data alongside legacy HERA and Tevatron measurements, but the ATLAS 7 TeV precision W/Z data are excluded from the main CT18 set due to systematic tension with other data sets, and are instead provided in a separate alternative fit (CT18A).

## Method

The analysis performs global QCD fits at NLO and NNLO using the Hessian method for error determination, incorporating combined HERA I+II deep-inelastic scattering data, fixed-target experiments, Tevatron measurements, and new LHC data on inclusive jet production, Drell-Yan pairs, top-quark pairs, and high-pT Z bosons. Fast PDF survey techniques using the Hessian representation and Lagrange Multiplier method quantify the preference of each data set for quantities such as alpha_s(m_Z) and the gluon and strange quark distributions, while the CTEQ fitting code was parallelized to enable fits within hours on high-performance computing clusters.

## Key numbers

- Total data points in CT18 fit: 3681, with chi^2/N_pt = 1.17 at NNLO
- Total data points in CT18Z fit: 3493, with chi^2/N_pt = 1.19 at NNLO
- The HERA I+II inclusive DIS data set shows S_E values as high as five units or more in the effective Gaussian variable distribution
- The quality of fit to HERA data improves by about 50 units when using the x_B-dependent factorization scale mu^2_F,x = 0.8^2 (Q^2 + 0.3 GeV^2/x_B^0.3)
- PDFs assume world-average QCD coupling constant alpha_s(M_Z) = 0.118

## Context

This work updates the CT14 PDFs (Dulat, 2016) by adding LHC Run-1 data from ATLAS, CMS, and LHCb on jets, W/Z bosons, and top-quark pairs, while retaining the legacy data sets from HERA, fixed-target experiments, and the Tevatron. The CT18 analysis extends the CTEQ-TEA global QCD analysis program and provides four PDF ensembles (CT18, CT18A, CT18X, CT18Z) that enable users to assess systematic uncertainties from data tensions and scale choices, with the CT18 and CT18Z ensembles being the most dissimilar in gluon and strangeness distributions.
