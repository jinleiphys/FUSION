# Unified description of fission in fusion and spallation reactions

> We present a statistical-model description of fission, in the framework of compound-nucleus decay, which is found to simultaneously reproduce data from both heavy-ion-induced fusion reactions and proton-induced spallation reactions at around 1 GeV. For the spallation reactions, the initial compound-nucleus population is predicted by the Liège Intranuclear Cascade Model. We are able to reproduce experimental fission probabilities and fission-fragment mass distributions in both reactions types with the same parameter sets. However, no unique parameter set was obtained for the fission probability. The introduction of fission transients can be offset by an increase of the ratio of level-density parameters for the saddle-point and ground-state configurations. Changes to the finite-range fission barriers could be offset by a scaling of the Bohr-Wheeler decay width as predicted by Kramers. The parameter sets presented allow accurate prediction of fission probabilities for excitation energies up to 300 MeV and spins up to 60 \hbar.

## Key claim

The authors present a statistical-model description of fission that simultaneously reproduces data from both heavy-ion-induced fusion reactions and proton-induced spallation reactions at around 1 GeV, but they find that no unique parameter set was obtained for the fission probability because changes to the fission barriers can be offset by scaling the Bohr-Wheeler decay width as predicted by Kramers.

## Method

The authors coupled the Liege Intranuclear Cascade Model (INCL4.5) to the GEMINI++ statistical de-excitation code to describe spallation reactions, while for fusion reactions the compound-nucleus spin distribution was determined from the total fusion cross section. They adjusted the ratio of level-density parameters for the saddle-point and ground-state configurations (a_f/a_n), the fission barriers, and the scaling of the Bohr-Wheeler decay width, using either the standard Bohr-Wheeler formalism or the Lestone tilting modification, to fit experimental fission probabilities and fission-fragment mass distributions from both reaction types.

## Key numbers

* For the 1-GeV p+208Pb spallation reaction, the INCL model predicts average compound-nucleus excitation energy of about 167 MeV and average spin of about 16.5 hbar, with distributions extending up to about 650 MeV and about 50 hbar.
* The best fit to both fusion and spallation data used a_f/a_n = 1.036 with no scaling of the Bohr-Wheeler decay width (Gamma_BW x 1.00) and Sierk fission barriers.
* An equivalent fit was obtained with Gamma_BW x 2.46 and a_f/a_n = 1.00, or with Sierk fission barriers reduced by 1.0 MeV and Gamma_BW x 1.00, a_f/a_n = 1.00.
* The parameter sets allow accurate prediction of fission probabilities for excitation energies up to 300 MeV and spins up to 60 hbar.
* For the 19F+181Ta -> 200Pb fusion reaction, fits were obtained with Gamma_BW x 2.46, a_f/a_n = 1.00; Gamma_BW x 1.00, a_f/a_n = 1.036; Gamma_Lestone x 7.38, a_f/a_n = 1.00; and Gamma_Lestone x 1.00, a_f/a_n = 1.057.

## Context

This work extends the GEMINI++ statistical decay model, previously adjusted for heavy-ion fusion reactions as described in Charity (2010), to spallation reactions by coupling it with the INCL intranuclear cascade model. The simultaneous fitting of fusion and spallation data reduces parameter ambiguities compared to fitting either reaction type alone, though the ambiguity between fission barrier magnitude and the Kramers scaling factor remains unresolved. The results enable accurate predictions of fission probabilities across a wide range of excitation energies and spins, and challenge interpretations of enhanced evaporation-residue yields as evidence for transient fission, as proposed by Back (1999), by showing that an energy-dependent level-density parameter can produce similar effects.
