# Microscopic version of the Bohr-Mottelson model and its application

> The shell-model coupling scheme of the proton-neutron symplectic model (PNSM), defined by the following dynamical symmetry chain $Sp(12,R) \supset SU(1,1) \otimes SO(6) \supset U(1) \otimes SU_{pn}(3) \otimes SO(2) \supset SO(3)$, is considered. It is shown that it corresponds to a microscopic version of the Bohr-Mottelson collective model which captures the original relationships between its exactly solvable submodel limits. This variant of the PNSM provides an interesting and relevant shell-model symplectic-based framework for exploring the nuclear collective dynamics. Some simple applications of the present theory to different nuclei with various collective properties are given.

## Key claim

The paper claims that the proton-neutron symplectic model (PNSM) coupling scheme defined by the dynamical symmetry chain \(Sp(12,R) \supset SU(1,1) \otimes SO(6) \supset U(1) \otimes SU_{pn}(3) \otimes SO(2) \supset SO(3)\) provides a microscopic version of the Bohr-Mottelson collective model that preserves the original relationships between its exactly solvable submodel limits, in contrast to the one-component \(Sp(6,R)\) symplectic model which lacks an \(SO(5)\) or \(SO(6)\) structure needed to capture the \(\gamma\)-unstable Wilets-Jean dynamics.

## Method

The paper classifies nuclear shell-model states within the PNSM using the reduction chain \(Sp(12,R) \supset SU(1,1) \otimes SO(6) \supset U(1) \otimes SU_{pn}(3) \otimes SO(2) \supset SO(3)\), where \(SU(1,1)\) provides radial wave functions and \(SO(6)\) provides angular wave functions via Dragt's spherical harmonics. A dynamical symmetry Hamiltonian is constructed as a linear combination of Casimir operators of the subgroups, including a harmonic oscillator mean field, an \(SO(6)\) term, an \(SU_{pn}(3)\) term, and residual rotor terms. The Hamiltonian is diagonalized in the irreducible collective space spanned by the maximal seniority \(SO(6)\) representation for each nucleus, using no effective charges for \(B(E2)\) calculations.

## Key numbers

- For \(^{158}\)Gd, the experimental ratio \(E_{4^{+}_{1}}/E_{2^{+}_{1}} \simeq 3.26\) is reproduced using model parameters (in MeV): \(B = -0.039\), \(a = 0\), \(b = 0.247\), \(c = 0\), and \(h = -0.147\).
- For \(^{150}\)Nd, the experimental ratio \(E_{4^{+}_{1}}/E_{2^{+}_{1}} \simeq 2.93\) is reproduced using model parameters (in MeV): \(B = -0.046\), \(a = 0.015\), \(b = 0.075\), \(c = -0.000016\), and \(h = -0.074\).
- For \(^{148}\)Nd, the experimental ratio \(E_{4^{+}_{1}}/E_{2^{+}_{1}} \simeq 2.49\) is reproduced using model parameters (in MeV): \(B = -0.038\), \(a = 0.023\), \(b = 0.203\), \(c = -0.000055\), and \(h = -0.0924\).
- The microscopic \(\gamma\)-unstable Wilets-Jean limit gives a characteristic ratio \(E_{4^{+}_{1}}/E_{2^{+}_{1}} \simeq 2.67\) for ground state band energies where \(L=\upsilon\), in contrast to the phenomenological WJ model value of 2.50.
- The \(Sp(12,R)\) irreducible representations used are 0p-0h \([36]_{6}\) for \(^{158}\)Gd, 0p-0h \([24]_{6}\) for \(^{150}\)Nd, and 0p-0h \([18]_{6}\) for \(^{148}\)Nd.

## Context

This work extends the microscopic foundation of the Bohr-Mottelson model beyond the one-component \(Sp(6,R)\) symplectic model of Rowe and Rosensteel (1980), which lacked an \(SO(5)\) or \(SO(6)\) structure for describing \(\gamma\)-unstable dynamics. The approach builds on the recent embedding of the generalized Bohr-Mottelson model in the two-component proton-neutron symplectic model (PNSM) framework (Ganev, 2020) and provides a microscopic counterpart to the Algebraic Collective Model (ACM) of Rowe (2004), but with the key difference that the PNSM state space is defined by Pauli-principle-allowed \(Sp(12,R)\) irreducible representations rather than the \(O(A-1)\)-scalar subspace used in phenomenological models. The results enable future detailed calculations of nuclear collective dynamics within a fully microscopic shell-model framework that preserves the relationships between the \(\gamma\)-unstable Wilets-Jean and rigid-rotor limits.
