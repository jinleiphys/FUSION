# Tensor-optimized antisymmetrized molecular dynamics in nuclear physics

> We develop a new formalism to treat nuclear many-body systems using bare nucleon-nucleon interaction. It has become evident that the tensor interaction plays important role in nuclear many-body systems due to the role of the pion in strongly interacting system. We take the antisymmetrized molecular dynamics (AMD) as a basic framework and add a tensor correlation operator acting on the AMD wave function using the concept of the tensor-optimized shell model (TOSM). We demonstrate a systematical and straightforward formulation utilizing the Gaussian integration and differentiation method and the antisymmetrization technique to calculate all the matrix elements of the many-body Hamiltonian. We can include the three-body interaction naturally and calculate the matrix elements systematically in the progressive order of the tensor correlation operator. We call the new formalism "tensor-optimized antisymmetrized molecular dynamics".

## Key claim

The paper develops a new formalism called "tensor-optimized antisymmetrized molecular dynamics" (TOAMD) that combines the antisymmetrized molecular dynamics (AMD) framework with the tensor correlation operator concept from the tensor-optimized shell model (TOSM), enabling the use of bare nucleon-nucleon interactions in AMD. The authors claim this approach can systematically calculate matrix elements for many-body operators up to six-body using Gaussian integration and differentiation methods, but the formalism is presented as a theoretical framework with no numerical results or applications to specific nuclei in this paper.

## Method

The authors construct the TOAMD wave function as a sum of an AMD wave function (a Slater determinant of single-nucleon Gaussian wave packets with variational spin and position parameters) plus a tensor correlation operator acting on that AMD wave function. The tensor correlation operator is expressed as a sum of Gaussian functions multiplied by the tensor operator and isospin operator, with variational parameters determined by energy minimization. All interactions (kinetic energy, two-body Argonne AV14 potential with central, tensor, spin-orbit, squared angular momentum, and squared spin-orbit components, and three-body Urbana-type interactions) are expanded in Gaussian functions and written in momentum space to achieve separable forms, allowing systematic calculation of matrix elements using Gaussian integrals and co-factor matrix techniques for antisymmetrization.

## Key numbers

- The tensor interaction contributes about 80% of the entire contribution of the two-body interaction to nuclear binding energy in light nuclei, based on Green's function Monte Carlo simulations by the Argonne group (Pieper, 2001).
- The center-of-mass kinetic energy matrix element for the AMD wave function is given as 3ν/(2m), where ν is the size parameter and m is the nucleon mass.
- The single-particle kinetic energy matrix element is (1/(2m))(3ν - ν²(D_p - D_q)²) times the exponential factor, spin matrix element, and isospin matrix element.

## Context

This work extends the tensor-optimized shell model (TOSM) developed by Myo et al. (2009, 2011, 2012, 2014), which successfully described light nuclei (He, Li, Be isotopes) using bare nucleon-nucleon interactions but could not adequately reproduce alpha cluster structures. The formalism also builds upon the AMD framework developed by the Kyoto group (Kanada-En'yo, 1995, 2003, 2012), which previously relied on effective interactions rather than bare interactions. The TOAMD method aims to enable description of both shell and cluster structures in light nuclei (such as ⁸Be, ¹²C, and ¹⁶O) using bare nucleon-nucleon interactions, potentially allowing application to heavier nuclei than those accessible to Green's function Monte Carlo simulations.
