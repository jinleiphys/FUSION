# JEWEL on a (2+1)D background with applications to small systems and substructure

> High-$p_T$ jets are an important tool for characterizing the quark-gluon plasma (QGP) created in heavy-ion collisions. However, a precise understanding of the jet-medium interaction is still lacking, and the development of more sophisticated observables is needed. This work presents a tool that allows for the exploration of alternative high-$p_T$ observables in a variety of collision systems. The tool builds on the publicly available JEWEL Monte Carlo code, and allows for the evolution of a jet on any given (2+1)-dimensional background. Proof-of-concept observables are also presented, studied using the latest version of JEWEL, JEWEL-2.4.0. The simplicity of the separation of the physics of the jet from the physics of the medium (while still allowing for the usual JEWEL medium-response), allows for easy interpretation without the need for complex parameterization. Results are produced using the RIVET toolkit, which allows for transparent preservation and development of analyses that are compatible with experimental methods. The code and analysis used to produce the plots presented here are made publicly available on a Github repository, with up-to-date usage instructions. This tool is expected to be useful to the broad jet-physics community for the study of precision observables for jets.

## Key claim
This work presents a publicly available tool that extends the JEWEL Monte Carlo code to allow jet evolution on any given (2+1)-dimensional background, enabling exploration of alternative high-pT observables in a variety of collision systems. The tool's value lies in its simplicity of separating jet physics from medium physics, though the author notes that the interface does not yet account for the underlying event and cannot use two different PDFs in asymmetric systems.

## Method
The author modified JEWEL-2.4.0 by implementing a medium interface (named "medium-2D") that reads in temperature and velocity contours for up to 90 time steps from hydrodynamic simulations or arbitrary backgrounds, correctly boosts into the rest frame of each fluid cell to determine fluid density, and optionally reads binary collision locations for sampling initial hard scattering positions. Proof-of-concept simulations were performed using publicly available (2+1)D MUSIC profiles with IP-Glasma initial conditions for 0-5% most central helium-gold collisions at sqrt(s_NN)=200 GeV and lead-lead collisions at sqrt(s_NN)=2.76 TeV, running 2000 events on each of 100 (200) simulation profiles for lead-lead (helium-gold). Results were produced using the RIVET toolkit with jets clustered via FASTJET and the LundPlane extension, employing constituent subtraction and dynamical grooming techniques.

## Key numbers
- 90 time steps: maximum number of time steps for which the interface reads temperature and velocity contours
- 2000 events: number of events run on each simulation profile for both lead-lead and helium-gold systems
- 100 (200) simulation profiles: number of profiles used for lead-lead (helium-gold) respectively
- R=0.4: jet radius used for anti-kT jets in the substructure analysis
- R=0.2: subjet radius used when reclustering constituents of R=0.4 jets
- sqrt(s_NN)=0.2 TeV: center-of-mass energy for helium-gold collisions
- sqrt(s_NN)=2.76 TeV: center-of-mass energy for lead-lead collisions
- 0-5%: centrality bin used for both collision systems

## Context
This work extends the JEWEL Monte Carlo framework (Zapp 2012, 2013, 2008) by building on a medium interface originally implemented by Korinna Zapp, similar in spirit to the standard "medium-simple.f" interface but now made public. Previous hydrodynamic plugins for JEWEL have been developed to study elliptic flow and R_AA interplay (Barreto 2022, Canedo 2020, Barreto 2021), but these were not public and had not been used for jet substructure in small systems. The tool enables the community to develop precision substructure observables that are independent of yield but sensitive to the colliding system, addressing the inconsistency where low-momentum QGP signatures are observed in small systems like p-Pb and d-Au while high-pT signatures appear absent (Nagle 2018, Apolinario 2022).
