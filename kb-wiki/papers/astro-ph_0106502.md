# New calculation of radioactive secondaries in cosmic rays

> We use a new version of our numerical model for particle propagation in the Galaxy to study radioactive secondaries. For evaluation of the production cross sections we use the Los Alamos compilation of all available experimental cross sections together with calculations using the improved Cascade-Exciton Model code CEM2k. Using the radioactive secondary ratios 26Al/27Al, 36Cl/Cl, 54Mn/Mn, we show how the improved cross-section calculations together with the new propagation code allow us to better constrain the size of the CR halo.

## Key claim

Using improved cross-section calculations from the CEM2k code and the Los Alamos compilation, together with a new version of the GALPROP propagation code, the authors show that the radioactive secondary ratios 26Al/27Al, 36Cl/Cl, and 54Mn/Mn can better constrain the size of the cosmic-ray halo, but the derived limits depend on the specific isotope and include uncertainties from cross-section inaccuracies and elemental abundance measurements.

## Method

The authors used the GALPROP propagation code with a diffusive reacceleration model, solving the full nuclear reaction network on a spatially resolved grid for halo heights of 2, 4, 6, and 10 kpc. Production cross sections for isotopes of Al, Cl, and Mn were taken from the Los Alamos T16 compilation of experimental data and calculations using the Cascade-Exciton Model code CEM2k, renormalized to data where necessary. The model was adjusted to fit the B/C ratio and stable nuclei ratios, then the radioactive secondary ratios were computed and compared with ACE, Ulysses, and Voyager data.

## Key numbers

- Half-life times used: 0.87 Myr for 26Al, 0.31 Myr for 36Cl, 0.63 Myr for 54Mn
- New halo size limits derived: 26Al gives 3.5-6 kpc, 36Cl gives 4-15 kpc, 54Mn gives 3-7 kpc
- Be-derived limits: 1.5-6 kpc (from prior work)
- Preliminary conclusion from all radioactive nuclei: halo height zh = 4-6 kpc based on ACE data and Ulysses elemental abundances
- Previous result: zh = 3-7 kpc (from Moskalenko and Strong 2001a)
- Discrepancy between cross section calculations and data often exceeds a factor of 2

## Context

This work extends the authors' previous GALPROP modeling (Moskalenko and Strong 2000, 2001a) by incorporating improved cross-section calculations from the CEM2k code and the Los Alamos compilation, replacing the older Webber (1993) phenomenological approximation. The new limits are consistent with the authors' prior Be-derived halo size of 1.5-6 kpc and support their previous conclusion that the large dispersion between isotopes is mostly due to cross-section inaccuracies. The results enable better constraints on the cosmic-ray halo size and suggest future work using ratios of all isotopes of Al, Cl, and Mn to their main progenitors (e.g., 26,27Al/28Si) to remove uncertainty from elemental abundance errors.
