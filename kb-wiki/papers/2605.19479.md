# Ab initio correlations between neutrinoless and two-neutrino double-beta decays in $^{48}$Ca

> We develop a novel ab initio in-medium no-core configuration-interaction (IM-NCCI) framework for nuclear charge-exchange processes by combining the in-medium similarity renormalization group with chiral nuclear Hamiltonians, and apply it to the $2νββ$ and $0νββ$ decays of $^{48}$Ca. This framework reproduces the locations of several main resonance peaks in the Gamow-Teller (GT) strength distribution for the $^{48}\mathrm{Ca}\to{}^{48}\mathrm{Sc}$ transition. The cumulative GT strength indicates missing contributions from two-body weak currents, corresponding to an effective quenching factor of $q\simeq0.84$. Incorporating this quenching yields a $2νββ$ nuclear matrix element (NME) in excellent agreement with experiment. Applying the same framework to $0νββ$ decay, and including the contribution from short-range operators, we obtain a total NME of $M^{0ν}=1.00\text{-}2.02$. Using 34 non-implausible chiral Hamiltonians, we establish from first principles strong linear correlations between the $0νββ$ NME and the NMEs governing $2νββ$ decay and double GT transitions. Combining these correlation relations within the 95% confidence level with the experimental $2νββ$-decay data yields a constrained prediction of $M^{0ν}=1.30\text{-}1.65$. This work establishes IM-NCCI as a complementary ab initio framework for nuclear weak decays and opens a pathway toward constraining $0νββ$ NMEs in heavier candidate nuclei using experimentally accessible $2νββ$-decay data.

## Key claim

The paper claims a strong linear correlation between the neutrinoless double-beta decay nuclear matrix element and the two-neutrino double-beta decay nuclear matrix element in ⁴⁸Ca, established from first principles using 34 chiral Hamiltonians. However, this correlation is presented without including the effects of two-body currents on either matrix element, and the authors note that the magnitude of the corresponding quenching remains an open question.

## Method

The authors developed an ab initio in-medium no-core configuration-interaction (IM-NCCI) framework combining the in-medium similarity renormalization group (IMSRG) with chiral nuclear Hamiltonians. The IMSRG decouples the reference state of ⁴⁸Ca from its excitations, while the NCCI method describes the final states of ⁴⁸Sc and ⁴⁸Ti using an importance-truncated many-particle–many-hole basis up to 3p3h and 4p4h configurations. The framework was applied to compute Gamow-Teller transition strengths, the 2νββ nuclear matrix element, and the 0νββ nuclear matrix element for ⁴⁸Ca, using both the EM1.8-2.0 interaction and 34 samples of chiral Hamiltonians based on the ΔN2LO_GO(394) interaction.

## Key numbers

- Quenching factor extracted from cumulative Gamow-Teller strength: q = 0.836 (ratio of experimental to theoretical integrated strength below 25 MeV is q² = 0.836²)
- Unquenched 2νββ nuclear matrix element from IM-NCCI: M²ν_LO = 0.066–0.077 MeV⁻¹
- Quenched 2νββ nuclear matrix element (with q = 0.836): M²ν = 0.046–0.054 MeV⁻¹, compared to experimental value M²ν = 0.0425(3) MeV⁻¹ (Barabash 2020)
- Total 0νββ nuclear matrix element from IM-NCCI: M⁰ν = 1.00–2.02 (long-range contribution M⁰ν_LR = 0.60–1.14, short-range contribution M⁰ν_SR = 0.40–0.88)
- Constrained prediction for 0νββ nuclear matrix element using correlation with 2νββ data at 95% confidence level: M⁰ν = 1.30–1.65
- Pearson correlation coefficient between M⁰ν and M²ν after quality filter: R = 0.91 (R = 0.82 for the full set of 34 Hamiltonians)
- Pearson correlation coefficient between M⁰ν and double-GT matrix element: R ≈ 0.92
- Upper limit on effective Majorana neutrino mass from experimental half-life limit: ⟨m_ββ⟩ ≲ 4.1–8.6 eV

## Context

This work extends previous ab initio calculations of 0νββ nuclear matrix elements in ⁴⁸Ca using the in-medium generator coordinate method (Yao 2020), valence-space IMSRG (Belley 2021), and coupled-cluster theory (Novario 2021), which all employed the same chiral interaction EM1.8-2.0. The IM-NCCI framework enables the first systematic first-principles investigation of correlations between 0νββ and 2νββ nuclear matrix elements, building on phenomenological studies that suggested such correlations exist (Horoi 2022, Jokiniemi 2023). The results open a pathway toward constraining 0νββ nuclear matrix elements in heavier candidate nuclei using experimentally accessible 2νββ-decay data, though future work must address the quantitative treatment of two-body current contributions to both decay modes.
