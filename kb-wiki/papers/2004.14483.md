# Moravcsik's theorem on complete sets of polarization observables reexamined

> We revisit Moravcsik's theorem on the unique extraction of amplitudes from polarization observables, which has been originally published in 1985. The proof is (re-) written in a more formal and detailed way and the theorem is corrected for the special case of an odd number of amplitudes (this case was treated incorrectly in the original publication). Moravcsik's theorem, in the modified form, can be applied in principle to the extraction of an arbitrary number of $N$ helicity amplitudes. The uniqueness theorem is then applied to hadronic reactions involving particles with spin. The most basic example is Pion-Nucleon scattering ($N=2$), the first non-trivial example is pseudoscalar meson photoproduction ($N=4$) and the most technically involved case treated here is given by pseudoscalar meson electroproduction ($N=6$). The application of Moravcsik's theorem to electroproduction yields new results, which for the first time provide insights into the structure and content of complete sets for this particular process. The uniqueness-statements for the various reactions are compared and an attempt is made to recognize general patterns, which emerge under the application of the theorem.

## Key claim

Moravcsik's theorem on complete sets of polarization observables is corrected for the special case of an odd number of amplitudes, which was treated incorrectly in the original 1985 publication; the modified theorem requires a closed loop of N bilinear products containing an odd number of imaginary parts (dashed lines) and no solid lines at all, contrary to the original criterion which demanded both solid and dashed lines in each loop.

## Method

The authors reformulate Moravcsik's geometrical analog, representing each amplitude as a point and each bilinear product as a line (solid for real part, dashed for imaginary part), then prove that a closed loop of N lines containing all N amplitudes exactly once yields a unique solution if it contains an odd number of dashed lines. The proof uses consistency relations for relative phases and the discrete ambiguity transformations for cosine-type and sine-type ambiguities. The theorem is applied to Pion-Nucleon scattering (N=2), pseudoscalar meson photoproduction (N=4), and pseudoscalar meson electroproduction (N=6), with a Mathematica code developed to enumerate all possible topologies for higher N.

## Key numbers

- For N=2 (Pion-Nucleon scattering), all 4 observables are required for a complete set, consistent with the theorem requiring 2N = 4 quantities.
- For N=4 (pseudoscalar meson photoproduction), the number of fully complete combinations in a minimal closed loop is 8, following the formula 2^(N-1) = 2^3 = 8.
- For N=6 (pseudoscalar meson electroproduction), the number of fully complete combinations in a minimal closed loop is 32, following 2^(N-1) = 2^5 = 32.
- The general formula for the number of fully complete combinations in a minimal closed loop is 2^(N-1) for any number of amplitudes N.

## Context

This work revisits and corrects Moravcsik's theorem from 1985 on complete sets of polarization observables, which had received little attention despite offering a lucid graphical representation valid for arbitrary N. The theorem extends the work of Keaton and Workman (1995, 1996) and Chiang and Tabakin (1996) on discrete amplitude ambiguities, and complements Nakayama's (2018) algebraic phase-fixing procedure by providing a more computationally automatable approach for reactions with complicated spin structures like pseudoscalar meson electroproduction. The results enable systematic preselection of complete sets of observables for any hadronic reaction involving particles with spin, with the number 2N emerging as the universally required minimum number of observables.
