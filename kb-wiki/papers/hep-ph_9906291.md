# The Soft Gluon Emission Process in the Color-Octet Model for Heavy Quarkonium Production

> The Color-Octet Model has been used successfully to analyze many problems in heavy quarkonium production. We examine some of the conceptual and practical problems of the soft gluon emission process in the Color-Octet Model. We use a potential model to describe the initial and final states in the soft gluon emission process, as the emission occurs at a late stage after the production of the heavy quark pair. It is found in this model that the soft gluon M1 transition, 1S0(8)->3S1(1), dominates over the E1 transition, 3PJ(8)->3S1(1), for J/psi and psi' production. Such a dominance may help resolve the questions of isotropic polarization and color-octet matrix element universality in the Color-Octet Model.

# The Soft Gluon Emission Process in the Color-Octet Model for Heavy Quarkonium Production

## Key claim

The soft gluon M1 transition, \(^1S_0^{(8)} \rightarrow ^3S_1^{(1)}\), dominates over the E1 transition, \(^3P_J^{(8)} \rightarrow ^3S_1^{(1)}\), for \(J/\psi\) and \(\psi'\) production in the Color-Octet Model, which may help resolve the questions of isotropic polarization and color-octet matrix element universality, but this conclusion depends on using a potential model with a screened potential that accounts for string breaking due to spontaneous light quark pair production.

## Method

The author formulates the soft gluon emission process by describing both the initial color-octet state and the final color-singlet bound state using a potential model for the heavy quark-antiquark pair, with a screened potential that accounts for string breaking via spontaneous light quark pair production. The soft gluon emission matrix elements are evaluated using multipole expansion in the long-wavelength approximation, with the density of color-octet states derived from the potential model. The cross section for quarkonium production via soft gluon emission is obtained by projecting the color-octet Feynman amplitude onto nonperturbative color-octet wave functions and computing the transition matrix elements for M1 and E1 radiative transitions.

## Key numbers

- For \(J/\psi\) production at fixed-target energies: \(\langle {\cal O}_8^{J/\psi} ({}^1S_0) \rangle + \frac{7}{m_c^2} \langle {\cal O}_8^{J/\psi} ({}^3P_0) \rangle = 3.0 \times 10^{-2}~\text{GeV}^3\)
- For \(\psi'\) production at fixed-target energies: \(\langle {\cal O}_8^{\psi'} ({}^1S_0) \rangle + \frac{7}{m_c^2} \langle {\cal O}_8^{\psi'} ({}^3P_0) \rangle = 0.5 \times 10^{-2}~\text{GeV}^3\)
- For \(J/\psi\) production from CDF at \(\sqrt{s}=1.8\) TeV: \(\langle {\cal O}_8^{J/\psi} ({}^1S_0) \rangle + \frac{3}{m_c^2} \langle {\cal O}_8^{J/\psi} ({}^3P_0) \rangle = 6.6 \times 10^{-2}~\text{GeV}^3\)
- For \(\psi'\) production from CDF at \(\sqrt{s}=1.8\) TeV: \(\langle {\cal O}_8^{\psi'} ({}^1S_0) \rangle + \frac{3}{m_c^2} \langle {\cal O}_8^{\psi'} ({}^3P_0) \rangle = 1.8 \times 10^{-2}~\text{GeV}^3\)

## Context

This work extends the Color-Octet Model developed by Bodwin, Braaten, and Lepage (1995) for heavy quarkonium production, which assumes that bound states are produced via color-octet mechanisms where a heavy quark-antiquark pair neutralizes its color by emitting a soft gluon. The paper addresses unresolved questions about isotropic polarization observed in fixed-target experiments by Hei (1991), Bii (1987), and Ake (1993), and the lack of universality of color-octet matrix elements between fixed-target data from Beneke et al. (1996) and high-\(p_T\) CDF data from Cho (1996). The finding that M1 transitions dominate over E1 transitions for soft gluon emission suggests a resolution to these problems by breaking the velocity counting rule of Bodwin, Braaten, and Lepage (1995), which would otherwise predict comparable contributions from both channels.
