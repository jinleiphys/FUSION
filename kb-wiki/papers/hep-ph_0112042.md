# Testing an approximation to large-Nc QCD with a toy model

> We consider a simple model of large-Nc QCD defined by a spectrum consisting of an infinite set of equally spaced zero-width vector resonances. This model is an excellent theoretical laboratory for investigating certain approximation schemes which have been used recently in calculations of hadronic parameters, such as the Minimal Hadronic Approximation. We also comment on some of the questions concerning issues of local duality versus global duality and finite-energy sum rules.

## Key claim

The Minimal Hadronic Approximation (MHA), which replaces the infinite spectrum of narrow resonances in large-\(N_c\) QCD with a finite set of low-lying states plus a perturbative continuum, can be systematically improved by combining short-distance (OPE) and long-distance (chiral perturbation theory) constraints, but the convergence is slow at large \(Q^2\) when only chiral constraints are used, and the OPE constraints alone may be unreliable below a \(Q^2\) scale determined by the number of constraints.

## Method

The authors construct a toy model of large-\(N_c\) QCD where the vector spectral function consists of an infinite set of equally spaced zero-width resonances with masses squared \(M_n^2 = (n+1/2)\sigma^2\) (chosen to eliminate the \(1/Q^2\) term in the OPE). They then define the MHA by keeping \(m+1\) explicit narrow states with adjustable masses \(\tilde{M}_n^2\) and a continuum starting at \(s_0 = (m+1)\sigma^2\), and fix the masses by matching either OPE constraints (Eq. 3.6), chiral constraints (Eq. 3.7), or a combination of both. The exact Adler function \(\mathcal{A}(Q^2)\) is given by a Hurwitz zeta function, and its OPE and chiral expansions are derived explicitly; the MHA is tested by comparing its Adler function and the \(\tau\) decay branching ratio \(\mathcal{R}_\tau(s)\) to the exact results.

## Key numbers

* The exact Adler function in the toy model is \(\mathcal{A}(Q^2) = \frac{Q^2}{\sigma^2} \zeta\!\left(2, \frac{Q^2+M_0^2}{\sigma^2}\right)\) with \(M_0^2 = \sigma^2/2\); its OPE expansion is \(\mathcal{A}^{(\infty)}(Q^2) \to 1 - \frac{1}{12}\left(\frac{\sigma^2}{Q^2}\right)^2 + \frac{7}{240}\left(\frac{\sigma^2}{Q^2}\right)^4 - \frac{31}{1344}\left(\frac{\sigma^2}{Q^2}\right)^6 + \cdots\).
* The chiral expansion of the exact Adler function is \(\mathcal{A}^{(0)}(Q^2) \to 4.9348\left(\frac{Q^2}{\sigma^2}\right) - 16.8288\left(\frac{Q^2}{\sigma^2}\right)^2 + 48.7045\left(\frac{Q^2}{\sigma^2}\right)^3 - 128.579\left(\frac{Q^2}{\sigma^2}\right)^4 + \cdots\), convergent for \(Q^2/\sigma^2 \le 1/2\).
* For the MHA with one state plus continuum, fixing the mass using the first OPE constraint gives \(\tilde{M}_0^2 = 0.541667\,\sigma^2\) (exact \(M_0^2 = 0.5\,\sigma^2\)); using the first chiral constraint gives \(\tilde{M}_0^2 = 0.504125\,\sigma^2\).
* For the MHA with two states plus continuum, using two OPE constraints yields \(\tilde{M}_0^2 = 0.481174\,\sigma^2\) and \(\tilde{M}_1^2 = 1.56049\,\sigma^2\) (exact \(M_1^2 = 1.5\,\sigma^2\)); using one OPE and one chiral constraint yields \(\tilde{M}_0^2 = 0.499093\,\sigma^2\) and \(\tilde{M}_1^2 = 1.54257\,\sigma^2\).
* With 11 resonances in the MHA, the maximum difference between the exact and approximate Adler function is of order \(10^{-8}\), occurring at \(Q^2/\sigma^2 \simeq 4\).
* The exact \(\tau\) branching ratio in the toy model is \(\mathcal{R}_\tau(s) = \frac{1}{2} - \frac{1}{2}\left(\frac{\sigma^2}{s}\right)^3 x(1-x)(1-2x) + \frac{1}{2}\left(\frac{\sigma^2}{s}\right)^4\left[x^2(1-x)^2 - \frac{1}{16}\right]\), where \(x\) is the fractional part of \(s/\sigma^2 - 1/2\); the OPE prediction is \(\mathcal{R}_\tau^{\mathrm{OPE}}(s) = \frac{1}{2} - \frac{7}{480}\left(\frac{\sigma^2}{s}\right)^4\).

## Context

This work extends the program of testing the Minimal Hadronic Approximation (MHA) introduced by Knecht, Peris, de Rafael (1998-2001) for large-\(N_c\) QCD, by using a solvable toy model with equally spaced resonances first proposed by Bramon, Etim, Greco (1972) and later studied by Shifman (2000) in the context of local versus global duality. The model allows explicit comparison of the MHA to the exact result, showing that combining short-distance (OPE) and long-distance (chiral) constraints yields rapid convergence, and that the OPE prediction for the \(\tau\) decay sum rule oscillates around the exact result, with optimal points where the difference nearly vanishes. This enables future systematic studies of duality violations and the optimal choice of constraints in realistic QCD applications.
