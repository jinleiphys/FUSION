# Event-by-event evaluation of the prompt fission neutron spectrum from 239Pu(n, f)

> Earlier studies of 239Pu(n, f) have been extended to incident neutron energies up to 20 MeV within the framework of the event-by-event fission model FREYA, into which we have incorporated multichance fission and pre-equilibrium neutron emission. The main parameters controlling prompt fission neutron evaporation have been identified and the prompt fission neutron spectrum has been analyzed by fitting those parameters to the average neutron multiplicity nubar from ENDF-B/VII.0, including the energy-energy correlations in nubar(E) obtained by fitting to the experimental nubar data used in the ENDF-B/VII.0 evaluation. We present our results, discuss relevant tests of this new evaluation, and describe possible further improvements.

## Key claim

The authors extend the event-by-event fission model FREYA to calculate the prompt fission neutron spectrum (PFNS) for 239Pu(n,f) up to 20 MeV incident neutron energy, but the results depend on fitting key model parameters to the average neutron multiplicity nubar from ENDF-B/VII.0 rather than directly to spectral data, because the experimental spectral data are "neither sufficiently accurate nor of sufficiently consistent quality" to allow an improved PFNS evaluation.

## Method

The FREYA model simulates individual fission events from the initial compound nucleus 240Pu* through pre-fission neutron emission (including multichance fission and pre-equilibrium emission via a two-component exciton model), mass and charge partition using a three-mode Gaussian parameterization of fragment yields, assignment of fragment kinetic energies and excitation energies based on experimental TKE data and level-density parameters, and sequential neutron evaporation from each fragment using a Weisskopf spectrum. The model parameters controlling the prompt fission neutron spectrum are determined by fitting to the average neutron multiplicity nubar from ENDF-B/VII.0, including energy-energy correlations in nubar(E).

## Key numbers

* The probability for pre-equilibrium neutron emission grows approximately linearly to about 24% at 20 MeV incident neutron energy.
* The asymmetric mode displacements for first-chance fission (A0=240) are D1 = 20.05 and D2 = 14.54, increased by 1/2 for each pre-fission neutron emitted.
* The width parameters for the asymmetric modes are sigma1 = 5.6 + 0.0937*(En/MeV) + 0.034*(En/MeV)^2 and sigma2 = 2.5 + 0.11060*(En/MeV) + 0.008*(En/MeV)^2.
* The normalization parameters for the asymmetric modes use a Fermi distribution with midpoint E1 = 10.14 MeV and width E2 = 1.15 MeV, with N1^0 = 0.757 and N2^0 = 0.242.
* The width of the super-long symmetric component is sigma_L = 12, assumed constant.
* The charge dispersion is sigma_Z = 0.5, from Reisdorf (1972).
* The Hill-Wheeler transmission probability uses hbar*omega = 1 MeV for sub-barrier fission.

## Context

This work extends the earlier FREYA model development by Vogt and Randrup (2010) to higher incident neutron energies and incorporates multichance fission and pre-equilibrium neutron emission, which were not present in the original version. The approach contrasts with the Madland-Nix model used in earlier evaluations and with the GNASH calculations used in ENDF-B/VII.0 for the multichance fission probabilities. The results enable improved PFNS evaluations for applications such as criticality safety and nuclear energy, and the authors note that further improvements could come from better experimental data on fragment yields and TKE at higher incident energies.
