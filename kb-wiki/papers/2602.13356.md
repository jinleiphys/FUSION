# MFGSB (ver. 1.0): Computer code for self-consistent mean-field calculations of atomic nuclei using Gaussian expansion method

> MFGSB, a computer code for self-consistent mean-field calculations of atomic nuclei using the Gaussian expansion method, is now accessible at Chiba University Repository.

## Key claim

MFGSB is a self-consistent mean-field code for atomic nuclei that uses the Gaussian expansion method, and it is uniquely applicable to Yukawa-type effective interactions up to the tensor channel, with no additional approximation needed for the Coulomb exchange term or the two-body center-of-mass term.

## Method

The code performs Hartree-Fock (HF), HF+BCS, and Hartree-Fock-Bogolyubov (HFB) calculations under spherical, axial, or \(J_z\)-symmetry assumptions, using either an iterative diagonalization method or a conjugate gradient method. It employs a single set of Gaussian basis function parameters that are insensitive to nuclide and deformation, allowing a single interaction file to cover almost all nuclei. The code is written in FORTRAN77 and requires BLAS and LAPACK libraries.

## Key numbers

- The default value of the angular-momentum cutoff \(\ell_\mathrm{cut}=7\) requires **8 GB of memory**; increasing \(\ell_\mathrm{cut}\) by one approximately doubles the memory requirement.
- The full code distribution (zip file `mfgsb.zip`) has a size of **16.6 GB**.
- The code is accessible at Chiba University Repository under DOI: **10.20776/900123722**.

## Context

This code extends the Gaussian expansion method (GEM) for nuclear mean-field calculations, as developed in prior work by Nakada (2002, 2006, 2008) and Nakada and Ito (2025). It enables systematic self-consistent calculations across a wide range of the nuclear chart without tuning basis parameters, and is the only SCMF code that handles Yukawa interactions including the tensor channel. The code is now publicly available for the nuclear-structure community.
