# Number of states with fixed angular momentum for identical fermions and bosons

> We present in this paper empirical formulas for the number of angular momentum I states for three and four identical fermions or bosons. In the cases with large I we prove that the number of states with the same ${\cal M}$ and n but different J is identical if $I \ge (n-2)J - {1/2} (n-1)(n-2)$ for fermions and $I \ge (n-2)J$ for bosons, and that the number of states is also identical for the same ${\cal M}$ but different n and J if ${\cal M} \le $min(n, 2J+1 - n) for fermions and for ${\cal M} \le $min(n, 2J) for bosons. Here ${\cal M} =I_{max}-I$, n is the particle number, and J refers to the angular momentum of a single-particle orbit for fermions, or the spin L carried by bosons.

# Number of states with fixed angular momentum for identical fermions and bosons

## Key claim

The paper presents empirical formulas for the number of angular momentum I states for three and four identical fermions or bosons, and proves that for large I the number of states with the same M and n but different J becomes identical under specific inequalities. The authors hedge that the formulas are obtained empirically, though they have been confirmed by computer for j ≤ 999/2, l ≤ 500, or L ≤ 500, and for many randomly chosen larger J values.

## Method

The authors constructed empirical formulas for the number of states D(I) for three and four identical particles in a single-j shell (fermions with half-integer j), a single-l shell (fermions with integer l), and bosons with spin L. They verified these formulas computationally for large ranges of j, l, and L. They also proved analytically that the hierarchical structure of states obtained by successive application of the angular momentum lowering operator J- on the maximum-M state remains identical for different J values above certain thresholds, using the standard enumeration procedure based on counting Slater determinants.

## Key numbers

- For n=3 fermions with I ≤ J: D(I)_j = floor((2I+3)/6)
- For n=3 fermions with I ≥ J-1 or bosons with I ≥ J: D(I)_J = floor((I_max - I)/6) + δ_I, where δ_I = 0 if (I_max - I)%6 = 1, otherwise δ_I = 1
- For n=4 fermions in a single-j shell with I ≤ 2J (even I): D(I)_J = floor((L - I/2)/3) × (I/2 + 1) + C(I)m - δ + D(I), where L = j - 1/2, C(I) = floor(I/6) + 1, and K = floor((I+4)/12)
- For n=4 fermions with I ≥ 2J - 3: D(I)_J = 3K(K+1) - K + (K+1)(K+1) + δ_{K0} - 1, where K = floor(m/6) and m is defined from I = I_max - 2m (even I) or I = I_max - 3 - 2m (odd I)
- For n=5, the D(M)_J series (independent of J) for M = 0 through 28 are: 1, 0, 1, 1, 2, 2, 3, 3, 5, 5, 7, 7, 10, 10, 13, 14, 17, 18, 22, 23, 28, 29, 34, 36, 42, 44, 50, 53, 60, 63
- The identity condition for fermions: I ≥ (n-2)J - (1/2)(n-1)(n-2)
- The identity condition for bosons: I ≥ (n-2)J
- For M ≤ min(n, 2J+1-n) (fermions) or M ≤ min(n, 2J) (bosons): D(M)_J series are 1, 0, 1, 1, 2, 2, 4, 4, 7, 8, 12, 14, 21, 24, 34, 41, 55, 66, 88, 105, 137, 165, 210, 253, 320 for M = 0 through 24

## Context

This work extends the limited algebraic formulas previously available for counting angular momentum states, which were only known for I=0 states with 4 fermions (Ginocchio). The authors note that textbooks (Ring) typically tabulate these numbers rather than provide formulas. The study was motivated by a previous observation (Zhao) that the angular momentum 0 ground state probability of 4 fermions in a single-j shell with random interactions shows synchronous staggering with the number of I=0 states. The results enable practical computation of state counts without tedious enumeration, and the proven identities reveal universal structures that had eluded observation despite the long history of the enumeration procedure.
