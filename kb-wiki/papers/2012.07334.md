# Taming nucleon density distributions with deep neural network method

> We investigate the density distributions of finite nuclei employing a well-designed deep neural network method. We calculate the target nucleon density distributions with Skyrme density functional theories, which are used to train the networks. We find that the training with only about $10\%$ nuclei ($300-400$) is sufficient to describe the nucleon density distributions of all the nuclear chart within 2\% relative error. The relative error comes to 5\% when about 200 proton(neutron) density distributions are used for training. We obtained very similar results for different Skyrme density functional theories. Therefore the ability to train networks is weakly dependent on the theoretical model. Moreover, in the process of machine learning, there is a turning point showing the transition from the Fermi-like distribution to the realistic Skyrme distribution, which provides significant properties of convergence process.

# Taming nucleon density distributions with deep neural network method

## Key claim

A deep neural network trained on only about 10% of nuclei (300-400 nuclei, or roughly 6800 proton/neutron density distributions) can predict the nucleon density distributions of all known nuclei with a central relative error below 2%, but this accuracy is conditional on using a normalized mean squared error loss function that enforces particle-number conservation; without sufficient training data (e.g., only 2% of the dataset), the network produces only Fermi-like distributions and the second convergence to realistic Skyrme distributions does not occur.

## Method

The authors used a deep neural network with eight layers and approximately 860,000 parameters, taking as input the proton number, neutron number, and a nucleon-type flag (τ=1 for proton, τ=2 for neutron). The network was trained using a normalized mean squared error (NMSE) loss function that enforces the constraint ∫4πρ(r)r²dr = X (where X is Z or N), with target density distributions calculated from Skyrme density functional theory (SKM*-SHF and SLy4-HFB). Training employed the Adam optimizer with Keras and Tensorflow backend, running for up to 2000 epochs.

## Key numbers

- Training with about 10% of the dataset (300-400 nuclei, ~6800 density distributions) yields a central relative error (CRE) less than 2% compared to the saturation density of ~0.08 fm⁻³ (half of nuclear saturation density 0.16 fm⁻³)
- Training with about 200 proton/neutron density distributions (5% of dataset) gives a CRE between 5% and 6%
- Training with only 2% of the dataset yields a CRE less than 10%, producing a Fermi-like distribution without the second convergence (no W-point)
- The maximum error σ(r) for the 10% training set is within 0.0015 fm⁻³
- The radial range considered is from r=0 fm to r=15 fm, divided into N=150 grids for all nuclei
- The W-point (second significant convergence) appears at approximately epoch 500 for the 10% training set

## Context

This work extends the application of machine learning to nuclear structure, following earlier uses of neural networks for predicting nuclear binding energies (Neural network methods, 2018), ground-state properties from ab-initio approaches (Negoita et al., 2018), and equation-of-state estimation from neutron star observations. The authors specifically build on the Skyrme-Hartree-Fock (SHF) and Hartree-Fock-Bogoliubov (HFB) methods, using SKM*-SHF and SLy4-HFB interactions as the target models. The approach enables rapid prediction of nucleon density distributions across the entire nuclear chart without solving the full many-body problem for each nucleus, and the identified W-point provides insight into the learning process, showing a transition from simple Fermi-like parameterizations to realistic DFT distributions.
