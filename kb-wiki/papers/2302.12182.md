# From inclusive to semi-inclusive one-nucleon knockout in neutrino event generators

> In neutrino event generators, for models for neutrino and electron scattering only inclusive cross sections are implemented. When these models are used to describe a semi-inclusive cross section, the event generator attaches the hadron variables based on some assumptions. In this work we compare the nucleon kinematics given by the method used in the GENIE event generator, e.g. in the implementation of the SuSAv2 model, to a fully unfactorized calculation using the relativistic distorted wave impulse approximation (RDWIA). We focus on kinematics relevant to the $e4ν$ analysis and show that observables obtained with RDWIA differ significantly from those of the approximate method used in GENIE, the latter should be considered unrealistic.

## Key claim

The approximate method used in the GENIE event generator to attach hadron variables for semi-inclusive one-nucleon knockout produces nucleon kinematics that differ significantly from the fully unfactorized relativistic distorted wave impulse approximation (RDWIA) calculation, and the GENIE-generated observables should be considered unrealistic for the kinematics relevant to the e4nu analysis.

## Method

The authors compared the nucleon kinematics produced by the GENIE event generator's implementation of the SuSAv2 model (which uses an inclusive cross section and then attaches hadron variables based on assumptions) against a fully unfactorized RDWIA calculation. The comparison focused on kinematics relevant to the e4nu analysis, examining observables such as nucleon momentum distributions and angular distributions.

## Key numbers

* For the kinematics relevant to the e4nu analysis, the RDWIA calculation yields nucleon kinematics that differ significantly from those produced by the GENIE approximate method, though no specific numerical values for the differences are provided in the available text.
* The paper focuses on kinematics relevant to the e4nu analysis, but no explicit beam energy or target nucleus numbers are given in the truncated text.

## Context

This work extends the ongoing effort to improve neutrino event generators by moving beyond purely inclusive cross section models toward semi-inclusive descriptions. It directly compares the GENIE event generator's SuSAv2 implementation (which uses an approximate factorized approach) against the more sophisticated RDWIA framework developed by previous authors. The results indicate that current event generator treatments of semi-inclusive knockout are inadequate for the e4nu analysis, motivating the development of improved hadronization models that incorporate unfactorized dynamics.
