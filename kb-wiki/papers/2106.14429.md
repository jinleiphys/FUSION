# Angularity in DIS at next-to-next-to-leading log accuracy

> Angularity is a class of event-shape observables that can be measured in deep-inelastic scattering. With its continuous parameter $a$ one can interpolate angularity between thrust and broadening and further access beyond the region. Providing such systematic way to access various observables makes angularity attractive in analysis with event shapes. We give the definition of angularity for DIS and factorize the cross-section by using soft-collinear effective theory. The factorization is valid in a wide range of $a$ below and above thrust region but invalid in broadening limit. It contains an angularity beam function, which is new result and we give the expression at $\mathcal{O}(\as)$. We also perform large log resummation of angularity and make predictions at various values of $a$ at next-to-next-to-leading log accuracy.

## Key claim

The paper defines angularity for deep-inelastic scattering and provides a factorized cross-section using soft-collinear effective theory that is valid for a wide range of the angularity parameter \(a\) below and above the thrust region, but explicitly invalid in the broadening limit (\(a \to 1\)).

## Method

The authors define DIS angularity using beam and jet axes, express it in Lorentz-invariant form, and factorize the cross-section into hard, beam, jet, and soft functions using soft-collinear effective theory (SCET). They perform large-logarithm resummation in Laplace space at next-to-next-to-leading log (NNLL) accuracy, and compute the angularity beam function at \(\mathcal{O}(\alpha_s)\) for the first time.

## Key numbers

* The angularity parameter \(a\) can be any real number in the region \(-\infty < a < 2\); it is infrared-unsafe for \(a \ge 2\).
* The angularity reduces to thrust when \(a = 0\) and to broadening when \(a = 1\).
* The one-loop non-cusp anomalous dimensions are given as \(\gamma^H_0 = -12 C_F\), \(\gamma^S_0 = 0\), and \(\gamma^J_0 = 6 C_F\).
* The two-loop hard anomalous dimension is \(\gamma^H_1 = -2C_F\big[ \big( \frac{82}{9} - 52\zeta_3\big) C_A + (3-4\pi^2+48\zeta_3)C_F + \big(\frac{65}{9} + \pi^2\big)\beta_0\big]\).

## Context

This work extends the study of angularity from \(e^+e^-\) annihilation (Berger 2003, Hornig 2009) to deep-inelastic scattering, and builds on prior factorization results for DIS event shapes such as 1-jettiness (Kang 2013). The new angularity beam function and NNLL resummation enable future measurements at the Electron-Ion Collider and EicC, and provide an independent path to determine the strong coupling constant from DIS event shapes, potentially addressing tensions between thrust and \(C\)-parameter determinations (Abbate 2010, Hoang 2014).
