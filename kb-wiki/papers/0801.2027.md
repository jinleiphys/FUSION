# Strong Linear Correlation Between Eigenvalues and Diagonal Matrix Elements

> We investigate eigenvalues of many-body systems interacting by two-body forces as well as those of random matrices. We find a strong linear correlation between eigenvalues and diagonal matrix elements if both of them are sorted from the smaller values to larger ones. By using this linear correlation we are able to predict reasonably all eigenvalues of given shell model Hamiltonian without complicated iterations.

# Strong Linear Correlation Between Eigenvalues and Diagonal Matrix Elements

## Key claim
The authors demonstrate a strong linear correlation between exact eigenvalues and diagonal matrix elements for individual runs of the two-body random ensemble (TBRE) and random matrices, provided both sets are sorted from smallest to largest. The correlation holds with coefficients typically above 0.95, but the prediction is most accurate for eigenvalues in the medium region and shows small deviations near the smallest and largest eigenvalues.

## Method
The authors computed exact eigenvalues by full diagonalization of Hamiltonian matrices for several nuclear many-body systems: four identical fermions in a single-\(j\) shell (\(j=31/2\)), \(sd\)-boson systems, and three valence protons plus three valence neutrons in the \((2s_{1/2},1d_{3/2},0i_{11/2})\) shell, all with two-body interactions drawn from Gaussian random ensembles. They also studied random matrices with uniformly distributed and Gaussian distributed elements. For each system, they sorted both eigenvalues and diagonal matrix elements from smallest to largest, computed the linear correlation coefficient \(r\), and derived a linear fit \(E_i = A H_{ii} + B\) using the trace and sum-of-squares constraints \(\sum E_i = \sum H_{ii}\) and \(\sum E_i^2 = \sum_{i,j} H_{ij}^2\). They tested the method on a realistic shell-model calculation for \(^{24}\)Mg using the USD interaction.

## Key numbers
- Linear correlation coefficients \(r > 0.95\) for four identical fermions in a single-\(j\) shell (\(j=31/2\), \(I=20\), dimension \(D_I=29\)) across six random interaction sets.
- Linear correlation coefficients \(r > 0.95\) for \(sd\)-boson systems (boson number \(n=12\), \(I=6\), dimension \(D_I=37\)).
- For \(^{24}\)Mg with USD interactions, the correlation coefficient for \(I=0\) states (dimension \(D_I=1161\)) is larger than 0.999.
- For Gaussian random matrices of dimension \(D=500\), all correlation coefficients \(r\) are larger than 0.98.
- For matrices with uniformly distributed random elements (dimension \(D=500\)), the linear correlation is very good except near the maximum and minimum eigenvalues.
- The authors report prediction precision of 1-4% in the examples studied.

## Context
This work extends earlier efforts to obtain eigenvalues using energy centroids and spectral moments, as pursued by Ratcliff (1970s), Zuker (1980s), Vary (1990s), Papenbrock (2000s), and Yoshinaga (2000s), which were only valid at the statistical ensemble-average level. The present finding provides a simple method to predict all eigenvalues of a given shell-model Hamiltonian without full diagonalization, using only the diagonal matrix elements and the sum of squares of all matrix elements. This enables approximate eigenvalue calculations for systems where matrix dimensions exceed \(10^5\) and full diagonalization becomes impractical, particularly relevant for nuclear shell-model calculations where diagonal matrix elements are typically one to two orders of magnitude larger than off-diagonal elements.
