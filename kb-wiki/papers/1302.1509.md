# $0νββ$ and $2νββ$ nuclear matrix elements, QRPA, and isospin symmetry restoration

> Within QRPA we achieve partial restoration of the isospin symmetry and hence fulfillment of the requirement that the $2νββ$ Fermi matrix element $M^{2ν}_F$ vanishes, as it should, unlike in the previous version of the method. This is accomplished by separating the renormalization parameter $g_{pp}$ of the particle-particle proton-neutron interaction into the isovector and isoscalar parts. The isovector parameter $g_{pp}^{T=1}$ need to be chosen to be essentially equal to the pairing constant $g_{pair}$, so no new parameter is needed. For the $0νββ$ decay the Fermi matrix element $M^{0ν}_F$ is substantially reduced, while the full matrix element $M^{0ν}$ is reduced by $\approx$ 10%. We argue that this more consistent approach should be used from now on in the proton-neutron QRPA and in analogous methods.

## Key claim

Within the proton-neutron QRPA, isospin symmetry can be partially restored by separating the particle-particle renormalization parameter \(g_{pp}\) into isovector (\(g_{pp}^{T=1}\)) and isoscalar (\(g_{pp}^{T=0}\)) parts, which forces the \(2\nu\beta\beta\) Fermi matrix element \(M^{2\nu}_F\) to vanish as required by isospin conservation, unlike in previous QRPA implementations where it did not vanish.

## Method

The authors modified the standard QRPA by replacing the single renormalization parameter \(g_{pp}\) for the particle-particle proton-neutron interaction with two independent parameters: \(g_{pp}^{T=1}\) for the isovector part and \(g_{pp}^{T=0}\) for the isoscalar part. The isovector parameter was set equal to the average pairing strength \(\bar{d}\) (the average of the four pairing renormalization parameters \(d_{pp}^{(i)}, d_{pp}^{(f)}, d_{nn}^{(i)}, d_{nn}^{(f)}\)), while the isoscalar parameter was fitted to reproduce the experimentally known \(2\nu\beta\beta\) half-lives. Nuclear matrix elements for both \(2\nu\beta\beta\) and \(0\nu\beta\beta\) decays were then computed using this new parametrization, with explicit summation over virtual intermediate states (not closure approximation) and including higher-order weak currents, nucleon form factors, and short-range correlations.

## Key numbers

- For \(^{76}\)Ge with the Argonne V18 potential and \(g_A = 1.27\), the old parametrization gave \(M^{0\nu}_F = -2.632\) and total \(M^{0\nu} = 5.812\); the new parametrization gave \(M^{0\nu}_F = -1.615\) and total \(M^{0\nu} = 5.157\), a reduction of approximately 11\%.
- For \(^{136}\)Xe with the Argonne V18 potential and \(g_A = 1.27\), the old parametrization gave \(M^{0\nu}_F = -1.211\) and total \(M^{0\nu} = 2.437\); the new parametrization gave \(M^{0\nu}_F = -0.806\) and total \(M^{0\nu} = 2.177\), a reduction of approximately 11\%.
- The ratio \(\chi_F = M^{0\nu}_F / M^{0\nu}_{GT}\) changed from approximately \(-0.5\) in the old parametrization to approximately \(-0.3\) to \(-0.4\) (roughly \(-1/3\)) in the new parametrization across most nuclei.
- For \(^{48}\)Ca with the Argonne V18 potential and \(g_A = 1.27\), the new parametrization gave \(M^{0\nu} = 0.541\), close to the shell model value of 0.59 from Caurier (2008).
- The isovector parameter \(g_{pp}^{T=1}\) was found to be within about 10\% of the average pairing parameter \(\bar{d}\) for all nuclei studied, with values ranging from 0.800 to 1.038 depending on the nucleus and nucleon-nucleon potential.

## Context

This work extends the proton-neutron QRPA method for calculating \(\beta\beta\)-decay nuclear matrix elements, previously developed by Simkovic, Rodin, Faessler, and Vogel (e.g., 2008, 2009), by addressing the long-standing problem of isospin violation that caused the \(2\nu\beta\beta\) Fermi matrix element to be non-zero. The approach follows suggestions made by Rodin (2011) and is consistent with the nuclear shell model (Caurier, 2008; Horoi, 2013), where isospin is conserved by construction and yields \(\chi_F \approx -(0.2-0.3)\). The new parametrization enables more consistent QRPA calculations that can be used for interpreting \(0\nu\beta\beta\) decay experiments and planning future searches, with the total \(0\nu\beta\beta\) matrix elements reduced by approximately 10-20% compared to previous QRPA results.
