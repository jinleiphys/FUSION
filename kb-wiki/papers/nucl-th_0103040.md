# Solutions of Faddeev-Yakubovsky equations in configuration space for the 4N scattering states

> The Faddeev-Yakubovsky equations in configuration space have been solved for the four nucleon system with special interest in their scattering states. We present results concerning the structure of the first $^4$He, scattering states in different (T,S) channels including the first inelatic threshold N+3N$\to$NN+NN and the n-t cross section using realistic potentials.

## Key claim

The first excited state of \(^4\)He, experimentally a 0.5 MeV wide resonance 0.4 MeV above the p-\(^3\)H threshold, is found to be a bound state 0.26 MeV below the N+NNN threshold in the model, and its structure suggests a 3+1 "halo-nucleus" configuration rather than a breathing mode, though the open question is whether Coulomb forces can place it correctly with required accuracy.

## Method

The Faddeev-Yakubovsky equations in configuration space were solved for four-nucleon scattering states, coupling two amplitudes (K and H) expanded in tripolar harmonics. Boundary conditions of Dirichlet type were imposed, with the 1+3 elastic case using precomputed triton Faddeev amplitudes. Calculations used MT I-III, AV14, and Nijm II potentials, with some calculations including a phenomenological three-nucleon interaction (TNI) parameterized as a sum of two Yukawa terms.

## Key numbers

- \(^4\)He ground state binding energy: 30.30 MeV (MT I-III potential)
- \(^4\)He first excited state (\(^4\)He\(^*\)) binding energy: 8.79 MeV (MT I-III), corresponding to 0.26 MeV below the N+NNN threshold
- \(^4\)He\(^*\) r.m.s. radius: 4.95 fm (MT I-III)
- \(^3\)H binding energy: 8.53 MeV (MT I-III)
- n+t singlet scattering length \(a_0\): 4.31 fm (AV14), 4.31 fm (Nijm II), 4.10 fm (MT I-III)
- n+t triplet scattering length \(a_1\): 3.79 fm (AV14), 3.76 fm (Nijm II), 3.63 fm (MT I-III)
- n+t coherent scattering length \(a_c\): 3.92 fm (AV14), 3.90 fm (Nijm II), 3.75 fm (MT I-III); experimental value 3.59 ± 0.02 fm
- n+t zero-energy cross section \(\sigma(0)\): 194 fm\(^2\) (AV14), 192 fm\(^2\) (Nijm II), 177 fm\(^2\) (MT I-III); experimental value 170 ± 3 fm\(^2\)
- Imaginary part of NN+NN scattering length: \(a_I = -0.0115 \pm 0.0001\) fm (MT I-III)
- Real part of NN+NN scattering length: \(a_R = +4.91 \pm 0.02\) fm (MT I-III)
- With TNI: \(B_3 = 8.48\) MeV, \(B_4 = 29.0\) MeV, \(a_0 = 4.0\) fm, \(a_1 = 3.53\) fm, \(a_c = 3.65\) fm, \(\sigma(0) = 168\) fm\(^2\)

## Context

This work extends the Faddeev-Yakubovsky approach from the 4N bound state (previously solved by the same group and others, e.g., the Grenoble group in 1998) to the 4N continuum, including the first inelastic threshold N+3N → NN+NN. It contradicts the widely held view that the first \(^4\)He excitation is a breathing mode, instead proposing a halo structure. The results enable future calculations with Coulomb forces to test whether they can place the excited state at the correct energy, and call for more precise measurements of the n+t zero-energy cross section, possibly using the CERN neutron beam facility.
