# RHODIUM: A post-processor for BIGSTICK configuration-interaction wave functions

> RHODIUM is a postprocessing code for nuclear structure physics. It can be used to compute density matrices, spectroscopic amplitudes, and other information, from wave function and basis files created by the configuration-interaction shell-model code BIGSTICK. The source code is available at github.com/cwjsdsu/Rhodium. This manual gives detailed instructions how to use.

## Key claim

RHODIUM is a postprocessing code that extends the capabilities of the configuration-interaction shell-model code BIGSTICK by enabling operations between wave functions computed in different bases with different quantum numbers, such as different M (Jz), parity, or particle number, which BIGSTICK cannot handle due to its fixed-basis restriction.

## Method

The code reads basis (.bas) and wave function (.wfn) files generated by BIGSTICK, then performs operations including computation of one-body density matrices (both charge-conserving and charge-changing), spectroscopic amplitudes for single-nucleon addition or removal, application of one-body operators to wave functions, projection of states into bases with different truncations, and calculation of overlaps between wave functions in different Hilbert spaces. RHODIUM addresses the problem of vanishing Clebsch-Gordan coefficients that occur for M=0 states by applying the angular momentum raising operator J+ to break the symmetry and extract missing density matrix elements.

## Key numbers

* Basis dimension of 28,503 for the example 24Mg calculation with 4 valence protons and 4 valence neutrons in the sd shell
* 483,887 basis dimension for the initial 47Cr wave function in the pf valence space using the GX1A interaction
* 1,963,461 basis dimension for the final 48Cr wave function in the pf valence space
* Spectroscopic amplitude of -0.353712 for orbit 3 (0f7/2) between 47Cr state 1 (E = -83.81071 MeV, 2xJ = 5, 2xT = 0) and 48Cr state 1 (E = -99.57887 MeV, 2xJ = 0, 2xT = 0)
* Spectroscopic amplitudes of 0.697354 (orbit 1), 0.227033 (orbit 2), -0.083912 (orbit 3), and -0.061296 (orbit 4) between 47Cr state 1 and 48Cr state 2 (E = -98.79021 MeV, 2xJ = 4, 2xT = 0)

## Context

This work extends the functionality of the BIGSTICK configuration-interaction shell-model code (Johnson 2018, Johnson 2013) by providing a postprocessor that overcomes BIGSTICK's limitation of working only within a fixed M-scheme basis with fixed quantum numbers. RHODIUM enables computation of spectroscopic amplitudes for nucleon transfer reactions and charge-changing transitions such as beta decay without assuming isospin symmetry, and allows calculation of density matrix elements between states of opposite parity or with different M values. The code enables future work such as bootstrapping block Lanczos calculations in BIGSTICK by projecting states from smaller spaces into larger spaces, and provides output files compatible with other postprocessing codes for computing electromagnetic transitions and other observables.
