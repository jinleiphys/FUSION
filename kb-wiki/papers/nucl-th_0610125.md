# Source chaoticity in relativistic heavy ion collisions at SPS and RHIC

> We investigate degree of coherence of pion sources produced in relativistic heavy ion collisions using multi-particle interferometry. In order to obtain ``true'' chaoticity, lambda^true from two-pion correlation functions measured in experiments, we make a correction for long-lived resonance decay contributions. Using this lambda^true and the weight factor which are obtained from parameter fitted to two- and three-pion correlation function, we calculate a chaotic fraction epsilon and the number of coherent sources alpha for different colliding energies. The result gives constraints on the source and shows an increase of the minimum number of alpha with multiplicity, although multiplicity independent chaoticity is not excluded.

## Key claim

After correcting for long-lived resonance decay contributions using a statistical model, the "true" chaoticity parameter λ^true and the weight factor ω from two- and three-pion correlations together constrain the source structure, showing an increase in the minimum number of coherent sources α with multiplicity, although a multiplicity-independent chaoticity is not excluded.

## Method

The authors corrected experimentally measured two-pion chaoticity parameters λ^exp for long-lived resonance decay contributions using a statistical model with chemical freeze-out temperature T and baryonic chemical potential μ_B determined by χ^2 fitting to particle ratio data. They extracted the weight factor ω by fitting a simple source function (assuming instantaneous emission and spherical symmetry) simultaneously to two- and three-pion correlation functions from SPS (NA44, WA98) and RHIC (STAR) experiments. Using the partially multicoherent source model of Nakamura et al., they solved for the chaotic fraction ε and the number of coherent sources α from the corrected λ^true and ω values.

## Key numbers

- S+Pb (NA44): λ^true = 0.94 ± 0.06, ω = 0.40 ± 0.44
- Pb+Pb (NA44): λ^true = 0.98 ± 0.03, ω = 1.15 ± 0.67
- Pb+Pb (WA98): λ^true = 1.03 ± 0.04, ω = 0.78 ± 0.44
- Au+Au (STAR): λ^true = 0.93 ± 0.08, ω = 0.872 ± 0.097
- Chemical freeze-out parameters (T, μ_B): S+Pb (173, 196) MeV; Pb+Pb (161, 223) MeV; Au+Au (158, 36) MeV

## Context

This work extends earlier analyses by Nakamura et al. (2001) on partially coherent sources and by Morita et al. (2005) on resonance decay corrections using statistical models. It applies the method to multiple collision systems across SPS and RHIC energies, enabling constraints on the number of coherent sources as a function of multiplicity. The results suggest that Coulomb corrections may be over-corrected in existing data, as noted by the CERES and PHENIX collaborations, and that improved Coulomb corrections would yield smaller λ^true values and clearer systematic trends.
