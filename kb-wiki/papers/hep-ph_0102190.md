# Pion form factor in large $N_c$ QCD

> The electromagnetic form factor of the pion is obtained using a particular realization of QCD in the large $N_c$ limit, which sums up the infinite number of zero-width resonances to yield an Euler's Beta function of the Veneziano type. This form factor agrees with space-like data much better than single rho-meson dominance. A simple unitarization ansatz is illustrated, and the resulting vector spectral function in the time-like region is shown to be in reasonable agreement with the ALEPH data from threshold up to about 1.3 ${GeV}^2$.

# Pion form factor in large $N_c$ QCD

## Key claim

The electromagnetic form factor of the pion, constructed from an infinite sum of zero-width vector meson resonances in the large $N_c$ limit of QCD, agrees with space-like experimental data much better than single rho-meson dominance, provided the single free parameter $\beta$ is fitted to approximately 2.3 rather than the naive Vector Meson Dominance value of 2.

## Method

The author constructs the pion electromagnetic form factor as an Euler Beta function of the Veneziano type by choosing specific masses (equally spaced squared masses $M_n^2 = M_\rho^2 (1 + 2n)$) and coupling coefficients that sum the infinite tower of zero-width vector meson resonances predicted by large $N_c$ QCD. The single free parameter $\beta$ is fitted to space-like data, and a simple Breit-Wigner unitarization ansatz with $\Gamma_n = \gamma M_n$ (with $\gamma \simeq 0.2$ fixed by the rho-meson width) is applied to smear the poles in the time-like region.

## Key numbers

- Fitted parameter $\beta \simeq 2.3$ from least-squares fit to space-like data, yielding $\chi^2_F \simeq 1.4$ per degree of freedom, compared to $\chi^2_F \simeq 11$ for Vector Meson Dominance ($\beta = 2$).
- Predicted pion mean squared radius $<r^2_\pi> = 0.436$ fm$^2$, compared to experimental value $<r^{2}_{\pi}> = 0.439 \pm 0.008$ fm$^2$ and Vector Meson Dominance prediction $0.394$ fm$^2$.
- Predicted ratio $g_{\rho\pi\pi}/f_\rho|_{QCD_{\infty}} \simeq 1.2$, compared to experimental value $1.21 \pm 0.02$ and Vector Meson Dominance value of 1.
- Mass predictions for first two radial excitations: $M_{\rho'} \simeq 1340$ MeV and $M_{\rho''} \simeq 1720$ MeV, compared to experimental values $M_{\rho'} = 1465 \pm 25$ MeV and $M_{\rho''} = 1700 \pm 20$ MeV.
- Integrated vector spectral function from threshold to $s_0 = 1.25$ GeV$^2$: theoretical value $1.1$ GeV$^2$, experimental value $0.9$ GeV$^2$.

## Context

This work extends the connection between Veneziano's dual model (Veneziano, 1968) and large $N_c$ QCD (t Hooft, 1974; Witten, 1979) to the three-point function of the pion form factor, building on earlier dual model studies of form factors (Frampton, 1970) and two-point functions (Dominguez, 1982). The Dual-$QCD_\infty$ realization provides a specific mass spectrum and coupling scheme that goes beyond naive Vector Meson Dominance and agrees with ALEPH data on the vector spectral function up to about 1.3 GeV$^2$, after which perturbative QCD is expected to take over. This enables a systematic framework for modeling hadronic three-point functions in the large $N_c$ limit that can be unitarized without the pathologies affecting four-point and higher functions.
