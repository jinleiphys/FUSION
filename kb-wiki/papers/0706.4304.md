# Erratum: Assessment of uncertainties in QRPA $0νββ$-decay nuclear matrix elements [Nucl. Phys. A 766, 107 (2006)]

> Accepted for publication in Nucl. Phys. A, see also nucl-th/0503063

# Erratum: Assessment of uncertainties in QRPA 0νββ-decay nuclear matrix elements

## Key claim

After correcting a coding error in the treatment of short-range correlations, the neutrinoless double beta decay matrix elements are larger than previously reported, but the original claim that the chosen method of adjusting the interaction strength makes the matrix elements essentially independent of the size of the single-particle basis, the G-matrix parametrization, whether QRPA or RQRPA is used, and whether g_A is quenched or not remains valid.

## Method

The authors corrected a coding error in the treatment of short-range correlations within the quasiparticle random phase approximation (QRPA) and the renormalized QRPA (RQRPA) frameworks. They recalculated the 0νββ-decay nuclear matrix elements for a set of candidate nuclei using the same interaction-strength adjustment procedure as in the original paper, which fixes the interaction by reproducing the measured 2νββ-decay half-lives. The corrected results are presented in a revised table and figure that replace the original ones.

## Key numbers

- For ⁷⁶Ge → ⁷⁶Se with g_A = 1.25: RQRPA matrix element ⟨M'⁰ν⟩ = 3.92 (variance 0.12), QRPA matrix element = 4.51 (variance 0.17); corresponding half-life T⁰ν₁/₂ = 0.86 × 10²⁷ years for ⟨m_ββ⟩ = 50 meV
- For ⁷⁶Ge → ⁷⁶Se with g_A = 1.00: RQRPA matrix element = 3.46 (variance 0.13), QRPA matrix element = 3.83 (variance 0.14); half-life = 1.10 × 10²⁷ years
- For ¹³⁰Te → ¹³⁰Xe with g_A = 1.25: RQRPA matrix element = 2.95 (variance 0.12), QRPA matrix element = 3.26 (variance 0.12); half-life = 2.16 × 10²⁶ years
- For ¹³⁶Xe → ¹³⁶Ba with g_A = 1.25 and M_GT^exp = 0.030 MeV⁻¹: RQRPA matrix element = 1.97 (variance 0.13), QRPA matrix element = 2.11 (variance 0.11); half-life = 4.55 × 10²⁶ years
- For ¹³⁶Xe → ¹³⁶Ba with g_A = 1.25 and vanishing 2ν matrix element: RQRPA matrix element = 1.67 (variance 0.13), QRPA matrix element = 1.78 (variance 0.11); half-life = 7.00 × 10²⁶ years
- For ¹⁵⁰Nd → ¹⁵⁰Sm with g_A = 1.25: RQRPA matrix element = 4.16 (variance 0.16), QRPA matrix element = 4.74 (variance 0.20); half-life = 2.23 × 10²⁵ years
- QRPA results are approximately 10% larger than RQRPA results across the studied nuclei

## Context

This erratum corrects the numerical results of the original paper by Rodin, Faessler, Simkovic, and Vogel (2006) on uncertainties in QRPA 0νββ-decay matrix elements. The corrected matrix elements are larger than those originally reported due to the short-range correlation coding error, but the methodological conclusions remain unchanged. The results provide updated nuclear matrix elements for 0νββ-decay searches in ⁷⁶Ge, ⁸²Se, ⁹⁶Zr, ¹⁰⁰Mo, ¹¹⁶Cd, ¹²⁸Te, ¹³⁰Te, ¹³⁶Xe, and ¹⁵⁰Nd, enabling more accurate half-life predictions for experiments assuming a given effective Majorana mass ⟨m_ββ⟩.
