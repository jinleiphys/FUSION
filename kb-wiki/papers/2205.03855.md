# Nonparametric Representation of Neutron Star Equation of State Using Variational Autoencoder

> We introduce a new nonparametric representation of the neutron star (NS) equation of state (EoS) by using the variational autoencoder (VAE). As a deep neural network, the VAE is frequently used for dimensionality reduction since it can compress input data to a low-dimensional latent space using the encoder component and then reconstruct the data using the decoder component. Once a VAE is trained, one can take the decoder of the VAE as a generator. We employ 100,000 EoSs that are generated using the nonparametric representation method based on \citet{2021ApJ...919...11H} as the training set and try different settings of the neural network, then we get an EoS generator (trained VAE's decoder) with four parameters. We use the mass\textendash{}tidal-deformability data of binary neutron star (BNS) merger event GW170817, the mass\textendash{}radius data of PSR J0030+0451, PSR J0740+6620, PSR J0437-4715, and 4U 1702-429, and the nuclear constraints to perform the joint Bayesian inference. The overall results of the analysis that includes all the observations are $R_{1.4}=12.59^{+0.36}_{-0.42}\,\rm km$, $Λ_{1.4}=489^{+114}_{-110}$, and $M_{\rm max}=2.20^{+0.37}_{-0.19}\,\rm M_\odot$ ($90\%$ credible levels), where $R_{1.4}$/$Λ_{1.4}$ are the radius/tidal-deformability of a canonical $1.4\,\rm M_\odot$ NS, and $M_{\rm max}$ is the maximum mass of a non-rotating NS. The results indicate that the implementation of the VAE techniques can obtain the reasonable results, while accelerate calculation by a factor of $\sim$ 3\textendash10 or more, compared with the original method.

## Key claim

The paper introduces a variational autoencoder (VAE) to reduce the dimensionality of a nonparametric neutron star equation of state (EoS) representation from 31 parameters to 4 latent variables, while still producing posterior distributions consistent with the original high-dimensional feed-forward neural network (FFNN) method of Han (2021). The VAE approach accelerates Bayesian inference by a factor of approximately 3 to 10 or more, though the constraints at densities above about 4 times nuclear saturation density are less stringent than those from the FFNN model.

## Method

The authors first generated 100,000 EoSs using the FFNN nonparametric representation from Han (2021), where each EoS is a 128-dimensional array of squared sound speed as a function of rest-mass density. They trained a VAE with a 4-dimensional latent space on this training set, then used the trained decoder as an EoS generator with only four latent parameters. They performed joint Bayesian inference using the mass-tidal-deformability data from GW170817, mass-radius data from PSR J0030+0451, PSR J0740+6620, PSR J0437-4715, and 4U 1702-429, plus nuclear constraints, sampling the posterior with the PyMultiNest algorithm and solving the Tolman-Oppenheimer-Volkoff (TOV) equations numerically.

## Key numbers

- Radius of a canonical 1.4 solar mass neutron star: R_1.4 = 12.59^{+0.36}_{-0.42} km (90% credible level) for the full data set D_2
- Tidal deformability of a canonical 1.4 solar mass neutron star: Lambda_1.4 = 489^{+114}_{-110} (90% credible level) for the full data set D_2
- Maximum mass of a non-rotating neutron star: M_max = 2.20^{+0.37}_{-0.19} solar masses (90% credible level) for the full data set D_2
- For the reduced data set D_1 (GW170817, PSR J0030+0451, PSR J0740+6620): R_1.4 = 12.14^{+0.94}_{-0.93} km, Lambda_1.4 = 374^{+283}_{-160}, M_max = 2.26^{+0.39}_{-0.23} solar masses (all at 90% credible level)
- Computation time for data set D_2: FFNN model required 45.28 hours, VAE model required 18.22 hours on a single compute node with 128 cores
- Computation time for data set D_1: FFNN model required 3.87 hours, VAE model required 1.27 hours on a single compute node with 128 cores

## Context

This work extends the nonparametric EoS representation method of Han (2021), which used a feed-forward neural network with 31 parameters, by applying a variational autoencoder for dimensionality reduction. The results are consistent with previous constraints from studies such as those using the same observational data (Tang 2021) and other nonparametric approaches. The method enables faster Bayesian inference for neutron star EoS studies and can be further improved by incorporating constraints from chiral effective field theory and perturbative quantum chromodynamics at very low and very high densities, respectively.
