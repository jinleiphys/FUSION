# Trees and Islands -- Machine learning approach to nuclear physics

> We implement machine learning algorithms to nuclear data. These algorithms are purely data driven and generate models that are capable to capture intricate trends. Gradient boosted trees algorithm is employed to generate a trained model from existing nuclear data, which is used for prediction for data of damping parameter, shell correction energies, quadrupole deformation, pairing gaps, level densities and giant dipole resonance for large number of nuclei. We, in particular, predict level density parameter for superheavy elements which is of great current interest. The predictions made by the machine learning algorithm is found to have standard deviation from 0.00035 to 0.73.

## Key claim

Gradient boosted trees (GBT) can predict a range of nuclear observables with standard deviations between 0.00035 and 0.73, but the method is purely data-driven and does not incorporate nuclear theoretical modeling; the quality of prediction depends strongly on the number of engineered features (seven features outperform two features for all observables tested).

## Method

The author applied the Gradient Boosted Trees (GBT) algorithm to nuclear data from the RIPL library and other tabulated sources. The algorithm was trained on a randomly sampled 60% of the data (with part used for validation) and tested on the remaining 40%. Seven features were engineered: volume, surface, Coulomb interaction, mass asymmetry, pairing energies, number of protons, and number of neutrons; for temperature-dependent level densities, temperature was added as an eighth feature. Predictions were quantified by the standard deviation (root mean square difference between predicted and actual values on the test set).

## Key numbers

- Damping parameter gamma: standard deviation of 0.00035 on a test set from 290 nuclei (Z=11 to 98)
- Shell correction energy: standard deviation of 0.553 on a test set from 6735 nuclei (Z=8 to 99)
- Quadrupole deformation beta_2: standard deviation of 0.015 on a test set from 8983 nuclei (Z=8 to 136)
- Proton pairing gap: standard deviation of 0.037 on a test set from 8979 nuclei (Z=8 to 136)
- Neutron pairing gap: standard deviation of 0.023 on a test set from 8979 nuclei (Z=8 to 136)
- Level density parameter (Gilbert-Cameron model): standard deviation of 0.73 on a test set from 289 nuclei (Z=11 to 98)
- Level density parameter (semiclassical trace formula, temperature-dependent): standard deviation of 0.3 on 3000 data points from 32 nuclei
- Giant dipole resonance first peak energy: standard deviation of 0.76 on a test set from 180 nuclei
- Predicted level density parameters for superheavy elements (Z=104 to 122) range from 19.85 to 27.67 MeV^-1, with n-values (from A/n trend) of 9.70 to 11.68 for GC and 13.45 to 16.04 for STF

## Context

This work extends the application of machine learning to nuclear data beyond earlier efforts by Clark (2006) and Utama (2016), who used Support Vector Machines and Neural Networks with only two features (neutron and proton numbers) to predict nuclear mass and charge radii. The author shows that increasing the feature set to seven (including liquid-drop model terms) improves prediction accuracy across multiple observables. The results enable predictions for superheavy elements in the island of stability, where experimental data are not yet available, and suggest that machine learning can serve as a tool for global mathematical modeling of nuclei where clear-cut theory is lacking.
