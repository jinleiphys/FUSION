# Comment on "QCD factorization with multihadron fragmentation functions"

> We make several comments on the recent work in Ref.~\cite{Rogers:2024nhb} while also reaffirming and adding to the work in Ref.~\cite{Pitonyak:2023gjx}. We show that the factorization formula for $e^+e^-\to (h_1\cdots h_n)\, X$ in Ref.~\cite{Rogers:2024nhb} is equivalent to a version one can derive using the definition of a $n$-hadron fragmentation function (FF) introduced in Ref.~\cite{Pitonyak:2023gjx}. In addition, we scrutinize how to generalize the number density definition of a single-hadron FF to a $n$-hadron FF, arguing that the definition given in Ref.~\cite{Pitonyak:2023gjx} should be considered the standard one. We also emphasize that the evolution equations for dihadron FFs~(DiFFs) in Ref.~\cite{Pitonyak:2023gjx} have the same splitting functions as those for single-hadron FFs. Therefore, the DiFF (and $n$-hadron FF) definitions in Ref.~\cite{Pitonyak:2023gjx} have a natural number density interpretation and are consistent with collinear factorization using the standard hard factors and evolution kernels. Moreover, we make clear that the operator definition for the DiFF $D_1^{h_1h_2}(ξ,M_h)$ written down in Ref.~\cite{Rogers:2024nhb} agrees exactly with the one in Ref.~\cite{Pitonyak:2023gjx}. Contrary to what is implied in Ref.~\cite{Rogers:2024nhb}, this definition did not appear in the literature prior to the work in Ref.~\cite{Pitonyak:2023gjx}. There also seem to be inconsistencies in how $D_1^{h_1h_2}(ξ,M_h)$ appears in previous unpolarized cross section formulas in the literature.

## Key claim

The factorization formula for $e^+e^-\to (h_1\cdots h_n)\, X$ proposed in Rogers (2024) is equivalent to one using the $n$-hadron fragmentation function definition from Pitonyak (2023), but the latter provides an unambiguous number density interpretation in specific variables that the former lacks; the operator definition of the dihadron FF $D_1^{h_1h_2}(\xi,M_h)$ in Rogers (2024) agrees exactly with that in Pitonyak (2023) and did not appear in the literature prior to that work.

## Method

The authors compare the operator definitions of $n$-hadron fragmentation functions from Pitonyak (2023) and Rogers (2024), showing that the factorization formula in Rogers (2024) can be rewritten in terms of the Pitonyak (2023) definition $D_1^{\{h_i\}_n}(\{\xi_i\}_n,\{\vec{P}_{i\perp}\}_n)$ with the same standard hard factor. They derive the Jacobian relating the two sets of variables and demonstrate that the number density interpretation of the Pitonyak (2023) definition is confirmed through a number operator calculation using lightcone creation and annihilation operators. They also verify that the evolution equations for dihadron FFs from Pitonyak (2023) use the same splitting functions as single-hadron FFs, with only the integration measure changing depending on the specific DiFF definition.

## Key numbers

- The $n$-hadron FF $D_1^{\{h_i\}_n}(\{\xi_i\}_n,\{\vec{P}_{i\perp}\}_n)$ is defined with a prefactor $1/[4(16\pi^3)^{n-1}\xi_1\cdots\xi_n]$ times the correlator $\Delta^{\{h_i\}_n}$, while the alternative $d(\xi,-\xi\vec{k}_T,\{P_h\})$ uses a prefactor $1/(4\xi)$ times the same correlator.
- For $n=2$, the Jacobian from variables $(\xi_1,\xi_2,\vec{P}_{1\perp},\vec{P}_{2\perp})$ to $(\xi,\zeta,\xi\vec{k}_T,\vec{\widetilde{M}}_h)$ is $4\pi^3\xi(1-\zeta^2)=16\pi^3\xi_1\xi_2/\xi$, where $\vec{\widetilde{M}}_h$ has magnitude $M_h/\sqrt{32\pi^3}$.
- The evolution equation for $D_1^{h_1h_2}(\xi,M_h)$ has integration measure $d\hat{z}/\hat{z}$, while for $D_1^{h_1h_2}(\xi_1,\xi_2,\vec{R}_T)$ it is $d\hat{z}/\hat{z}^2$, both using standard unpolarized time-like splitting functions $P_{i\to i'}(\hat{z})$.
- The cross section formulas are $d\sigma/dz\,dM_h = \int_z^1 (d\hat{z}/\hat{z})\,(d\hat{\sigma}/d\hat{z})\,D_1^{h_1h_2}(z/\hat{z},M_h)$ and $d\sigma/dz_1dz_2d^2\vec{R}_T = \int_z^1 (d\hat{z}/\hat{z}^2)\,(d\hat{\sigma}/d\hat{z})\,D_1^{h_1h_2}(z_1/\hat{z},z_2/\hat{z},\vec{R}_T)$.

## Context

This comment extends the work of Pitonyak (2023) on multi-hadron fragmentation functions and responds to Rogers (2024), which proposed an alternative definition. The authors argue that the Pitonyak (2023) definition provides a clear number density interpretation in $3n$ variables $(\xi_1,\dots,\xi_n,\vec{P}_{1\perp},\dots,\vec{P}_{n\perp})$ that is consistent with parton model calculations, sum rules, and factorization, while the Rogers (2024) definition is ambiguous for general $n$. The work enables consistent phenomenological extraction of dihadron FFs from $e^+e^-$ annihilation data and clarifies the correct cross section formulas for processes like SIDIS that involve these functions.
