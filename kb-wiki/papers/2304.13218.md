# The chiral limit of a fermion-scalar $(1/2)^+$ system in covariant gauges

> The homogeneous Bethe-Salpeter equation (BSE) of a (1/2)$^+$ bound system, that has both fermionic and bosonic degrees of freedom, that we call a {\em mock nucleon}, is studied in Minkowski space, in order to analyse the chiral limit in covariant gauges. After adopting an interaction kernel built with a one-particle exchange, the $χ$-BSE is numerically solved by means of the Nakanishi integral representation and light-front projection. Noteworthy, the chiral limit induces a scale-invariance of the model and consequently generates a wealth of striking features: i) it reduces the number of non trivial Nakanishi weight functions to only one; ii) the form of the surviving weight function has a factorized dependence on the two relevant variables, compact and non-compact one; iii) the coupling constant becomes an explicit function of the real exponent governing the power-law fall-off of the non trivial Nakanishi weight function. The thorough investigation at large transverse-momentum of light-front Bethe-Salpeter amplitudes, obtained with massive constituents, provides a confirmation of the expected universal power-law fall-off, with exponents predicted by our non-perturbative framework. Finally, one can shed light on the exponents that govern the approach to the upper extremum of the longitudinal-momentum fraction distribution function of the {\em mock nucleon}, when the coupling constant varies.

## Key claim

The chiral limit of the fermion-scalar Bethe-Salpeter equation induces scale invariance, which forces the Nakanishi weight function to factorize into a power-law in the non-compact variable and a function of the compact variable, and makes the coupling constant an explicit function of the power-law exponent; this analysis is restricted to coupling constants below a gauge-dependent critical value, above which the continuous scale symmetry is broken to a discrete one (Miransky scaling).

## Method

The homogeneous Bethe-Salpeter equation for a (1/2)^+ fermion-scalar bound system (mock nucleon) with a one-vector-boson-exchange kernel was solved in Minkowski space using the Nakanishi integral representation and light-front projection. In the chiral limit (all masses set to zero), the coupled integral equations for the two Nakanishi weight functions decouple, and a factorization Ansatz of the surviving weight function as a power-law in the non-compact variable times a function of the compact variable was inserted. The resulting integral equation for the compact-variable function was discretized on a spline basis and solved as an eigenvalue problem for the coupling constant.

## Key numbers

* Critical coupling constant in Feynman gauge: α_c = 1.187, with exponent r_c = -0.455.
* Critical coupling constant in Landau gauge: α_c = 1.276, with exponent r_c = -0.522.
* Critical coupling constant varies by only ~8% across covariant gauges in the range ζ ∈ [0,1].
* For a massive mock nucleon with m_s/m_f = 2, M/̄m = 1.9, and μ/̄m = 0.15, the coupling constant is α = 0.648, giving r = -0.817.
* For the same system with μ/̄m = 0.5, the coupling constant is α = 0.898, giving r = -0.718.
* For an equal-mass system with massless vector exchange and B/̄m = 0.5, the coupling constant is α = 1.189, very close to the critical value α_c = 1.187.
* The valence parton distribution function near ξ=1 behaves as (1-ξ)^{2|r|}, yielding exponents 1.634 and 1.436 for the two massive cases above.

## Context

This work extends the analysis of the mock nucleon model previously studied in the Feynman gauge by Alvarenga Nogueira et al. (2019) to arbitrary covariant gauges and to the chiral limit. The scale-invariance-induced relation between coupling constant and power-law exponent parallels the Miransky scaling phenomenon found in gauge theories by Miransky (1984) and the Thomas-Efimov effect in non-relativistic few-body physics. The results enable quantitative predictions for the ultraviolet behavior of transverse-momentum distributions and the end-point behavior of longitudinal-momentum fraction distributions in hadronic models, and motivate future generalization to Faddeev-Bethe-Salpeter equations for nucleon phenomenology.
