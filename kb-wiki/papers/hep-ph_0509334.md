# Neutral Larkin--Ovchinnikov--Fulde--Ferrell state and chromomagnetic instability in two-flavor dense QCD

> In two-flavor dense quark matter, we describe the dynamics in the single plane wave Larkin--Ovchinnikov--Fulde--Ferrell (LOFF) state satisfying the color and electric neutrality conditions. We find that because the neutral LOFF state itself suffers from a chromomagnetic instability in the whole region where it coexists with the (gapped/gapless) two-flavor superconducting (2SC/g2SC) phases, it cannot cure this instability in those phases. This is unlike the recently revealed gluonic phase which seems to be able to resolve this problem.

## Key claim

The neutral single-plane-wave LOFF state itself suffers from a chromomagnetic instability in the entire region where it coexists with the 2SC/g2SC phases, and therefore cannot cure this instability in those phases, contrary to the gluonic phase which appears able to resolve the problem.

## Method

The authors use a Nambu-Jona-Lasinio (NJL) model with two quark flavors to study the single-plane-wave LOFF state under color and electric neutrality conditions. They derive the free energy in the hard dense loop approximation, solve the coupled gap equations for the diquark gap Δ and LOFF wavevector q together with the neutrality conditions for the electric chemical potential δμ and color chemical potential μ₈, and compute the Meissner masses for the 4-7th gluons to assess chromomagnetic stability.

## Key numbers

- The neutral LOFF solution exists only for Δ₀ values in the window 63 MeV < Δ₀ < 137 MeV (LOFF window), compared to the g2SC window 92 MeV < Δ₀ < 130 MeV.
- The neutral LOFF state develops chromomagnetic instability (imaginary Meissner masses for 4-7th gluons) for all Δ₀ > Δ₀^cr = 81 MeV.
- For μ = 300 MeV, the LOFF window is 49 MeV < Δ₀ < 121 MeV and the instability threshold is Δ₀^cr = 64 MeV; for μ = 500 MeV, the LOFF window is 74 MeV < Δ₀ < 141 MeV and Δ₀^cr = 94 MeV.
- The neutral LOFF phase is more stable than the neutral normal phase in the whole LOFF window, and more stable than the neutral g2SC/2SC phase in the g2SC window plus the narrow region 130 MeV < Δ₀ < 136 MeV.

## Context

This work extends the study of chromomagnetic instability previously found in the 2SC/g2SC phases by Hashimoto and Miransky (2005) and in the color-flavor locked phase of three-flavor QCD. The authors show that the single-plane-wave LOFF state, previously considered as a candidate to resolve this instability, fails to do so because it is itself unstable. This points toward the gluonic phase introduced by Gorbar, Hashimoto, and Miransky (2005) as a more promising resolution, and motivates further study of that phase and of multi-plane-wave LOFF states.
