# Impact of the leading-order short-range nuclear matrix element on the neutrinoless double-beta decay of medium-mass and heavy nuclei

> We evaluate the leading-order short-range nuclear matrix element for the neutrinoless double-beta ($0νββ$) decay of the nuclei most relevant for experiments, including $^{76}$Ge, $^{100}$Mo, $^{130}$Te and $^{136}$Xe. In our calculations, performed with the nuclear shell model and proton-neutron quasiparticle random-phase approximation (pnQRPA) methods, we estimate the coupling of this term by the contact charge-independence-breaking coupling of various nuclear Hamiltonians. Our results suggest a significant impact of the short-range matrix element, which is about $15\%-50\%$ and $30\%-80\%$ of the standard $0νββ$-decay long-range matrix element for the shell model and pnQRPA, respectively. Combining the full matrix elements with the results from current $0νββ$-decay experiments we find that, if both matrix elements carry the same sign, these searches move notably toward probing the inverted mass ordering of neutrino masses.

## Key claim

The short-range nuclear matrix element contributes 15%-50% (shell model) or 30%-80% (pnQRPA) of the standard long-range matrix element for neutrinoless double-beta decay in medium-mass and heavy nuclei, but this estimate depends on the unknown sign of the short-range term relative to the long-range term and on the assumed coupling constant extracted from charge-independence-breaking nuclear interactions.

## Method

The authors performed nuclear shell model (NSM) calculations for seven isotopes (48Ca, 76Ge, 82Se, 124Sn, 128Te, 130Te, 136Xe) and proton-neutron quasiparticle random-phase approximation (pnQRPA) calculations for nine isotopes (76Ge, 82Se, 96Zr, 100Mo, 116Cd, 124Sn, 128Te, 130Te, 136Xe). They estimated the short-range coupling g_nu^NN by using six different charge-independence-breaking (CIB) couplings from various nuclear Hamiltonians with Gaussian regulators, combined with Argonne and CD-Bonn short-range correlations. The full matrix elements were then combined with likelihood functions from the GERDA, KamLAND-ZEN, EXO-200, and CUORE experiments to constrain the effective Majorana mass.

## Key numbers

- Short-range matrix element is 15%-50% of the long-range matrix element for shell model calculations across all studied nuclei
- Short-range matrix element is 30%-80% of the long-range matrix element for pnQRPA calculations across all studied nuclei
- For 76Ge: long-range NME ranges 4.72-5.22 (pnQRPA) and 3.34-3.54 (NSM); short-range NME ranges 1.49-3.80 (pnQRPA) and 0.52-1.49 (NSM)
- For 136Xe: long-range NME ranges 2.53-2.80 (pnQRPA) and 2.62-2.79 (NSM); short-range NME ranges 0.76-1.95 (pnQRPA) and 0.45-1.31 (NSM)
- For 130Te: long-range NME ranges 3.46-3.89 (pnQRPA) and 3.26-3.48 (NSM); short-range NME ranges 1.18-3.05 (pnQRPA) and 0.57-1.64 (NSM)
- For 100Mo (pnQRPA only): long-range NME 3.40-3.95; short-range NME 1.66-4.26; ratio 49%-108%
- For 48Ca (NSM only): long-range NME 0.96-1.05; short-range NME 0.22-0.65; ratio 23%-62%
- Coupling values g_nu^NN used: -0.67 fm^2 (Lambda=450 MeV), -1.01 fm^2 (Lambda=550 MeV), -1.44 fm^2 (Lambda=465 MeV), -0.91 fm^2 (Lambda=465 MeV), -1.44 fm^2 (Lambda=349 MeV), -1.03 fm^2 (Lambda=349 MeV)

## Context

This work extends the short-range matrix element studies of Cirigliano (2018, 2019) from light nuclei (12Be, 48Ca) to the full set of experimentally relevant medium-mass and heavy nuclei. It uses the same strategy as Cirigliano (2019) of estimating the unknown coupling from charge-independence-breaking terms, but applies it with both shell model and pnQRPA many-body methods. The results enable more precise interpretation of current experimental limits from GERDA, KamLAND-ZEN, EXO-200, and CUORE, and show that if the short-range and long-range matrix elements have the same sign (as suggested by the recent 48Ca ab initio result of Wirth (2021)), current experiments approach the inverted neutrino mass ordering region.
