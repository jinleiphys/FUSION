# Alpha clustering and alpha-capture reaction rate from ab initio symmetry-adapted description of $^{20}$Ne

> We introduce a new framework for studying clustering and for calculating alpha partial widths using ab initio wave functions. We demonstrate the formalism for $^{20}$Ne, by calculating the overlap between the $^{16}$O$+α$ cluster configuration and states in $^{20}$Ne computed in the ab initio symmetry-adapted no-core shell model. We present spectroscopic amplitudes and spectroscopic factors, and compare those to no-core symplectic shell-model results in larger model spaces, to gain insight into the underlying physics that drives alpha-clustering. Specifically, we report on the alpha partial width of the lowest $1^-$ resonance in $^{20}$Ne, which is found to be in good agreement with experiment. We also present first no-core shell-model estimates for asymptotic normalization coefficients for the ground state, as well as for the first excited $4^{+}$ state in $^{20}$Ne that lies in a close proximity to the $^{16}$O$+α$ threshold. This outcome highlights the importance of correlations for developing cluster structures and for describing alpha widths. The widths can then be used to calculate alpha-capture reaction rates for narrow resonances of interest to astrophysics. We explore the reaction rate for the alpha-capture reaction $^{16}$O$(α,γ)^{20}$Ne at astrophysically relevant temperatures and determine its impact on simulated X-ray burst abundances.

## Key claim

The paper introduces a new ab initio framework for calculating alpha partial widths and asymptotic normalization coefficients from symmetry-adapted no-core shell model wave functions, but the demonstration assumes each cluster (alpha and oxygen-16) has a single equilibrium shape with suppressed vibrations, which simplifies the calculation to a recursive procedure rather than a fully general method.

## Method

The authors compute the overlap between the oxygen-16 plus alpha cluster configuration and states in neon-20 calculated in the ab initio symmetry-adapted no-core shell model (SA-NCSM), using the complementary nature of the symplectic Sp(3,R) basis and the cluster basis. The spectroscopic amplitude is obtained through a recursive procedure that relates excitations within a symplectic irreducible representation to excitations of the relative motion between the two clusters, with the base case defined when the symplectic bandhead matches the cluster configuration. The alpha partial width is then extracted from the spectroscopic amplitude using R-matrix theory, and the resulting width is used to compute the oxygen-16(alpha,gamma)neon-20 reaction rate for X-ray burst simulations.

## Key numbers

- The alpha partial width of the lowest 1- resonance in neon-20 at 5.79 MeV is found to be in good agreement with experiment, with no parameters adjusted in the study.
- The first no-core shell-model estimates for asymptotic normalization coefficients are presented for the ground state and the first excited 4+ state in neon-20, which lies in close proximity to the oxygen-16 plus alpha threshold.
- Using the calculated width versus the experimental width produces almost no difference in the simulated X-ray burst abundance pattern.

## Context

This work extends the long-standing relationship between the symplectic Sp(3,R) basis and the microscopic cluster basis, previously exploited by Hecht (1977, 1979) and Suzuki (1982, 1986) for describing alpha clustering in light nuclei, to a fully ab initio framework using SA-NCSM wave functions. The method enables calculation of alpha-capture reaction rates for narrow resonances of astrophysical interest, with a work in progress focusing on the oxygen-15(alpha,gamma)neon-19 reaction, which has been suggested by Cyburt (2016) to have the highest impact on X-ray burst nucleosynthesis.
