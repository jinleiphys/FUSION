# Strangeness in the proton from W+charm production and SIDIS data

> We perform a global QCD analysis of unpolarized parton distribution functions (PDFs) in the proton, including new $W +$\,charm production data from $pp$ collisions at the LHC and semi-inclusive pion and kaon production data in lepton-nucleon deep-inelastic scattering, both of which have been suggested for constraining the strange quark PDF. Compared with a baseline global fit that does not include these datasets, the new analysis reduces the uncertainty on the strange quark distribution over the range $0.01 < x < 0.3$, and provides a consistent description of processes sensitive to strangeness in the proton. Including the new datasets, the ratio of strange to nonstrange sea quark distributions is $R_s = (s+\bar s)/(\bar u+\bar d) = \{0.72^{+0.52}_{-0.34},\, 0.46^{+0.30}_{-0.20},\, 0.32^{+0.23}_{-0.15}\}$ for $x = \{ 0.01, 0.04, 0.1 \}$ at $Q^2 = 4$~GeV$^2$. The data place more stringent constraints on the strange asymmetry $s-\bar s$, which is found to be consistent with zero in this range.

## Key claim

The inclusion of both semi-inclusive deep-inelastic scattering (SIDIS) kaon production data and \(W+\)charm production data from the LHC reduces the uncertainty on the strange quark parton distribution function (PDF) over the range \(0.01 < x < 0.3\) and provides a consistent description of strangeness-sensitive processes, but the strange-to-nonstrange sea ratio \(R_s\) is found to be suppressed relative to some previous analyses of ATLAS \(W+\)charm data alone.

## Method

The authors performed a global QCD analysis of unpolarized parton distribution functions (PDFs) and fragmentation functions (FFs) using a Monte Carlo-based Bayesian inference approach, fitting nearly 6,000 data points from deep-inelastic scattering, Drell-Yan lepton-pair production, weak boson and jet production, \(W+\)charm production at the LHC, semi-inclusive deep-inelastic scattering (SIDIS) of pions and kaons from COMPASS, and single-inclusive \(e^+e^-\) annihilation (SIA). The PDFs and FFs were parameterized at the input scale \(\mu^2 = m_c^2\) using a standard template function, evolved to next-to-leading logarithmic accuracy via DGLAP evolution in the zero-mass variable flavor scheme, and the partonic cross sections were computed at next-to-leading order (NLO) in \(\alpha_s\). The analysis compared a baseline fit (excluding SIDIS and \(W+\)charm data) with scenarios adding SIDIS data alone, \(W+\)charm data alone, and the combined JAM24 fit.

## Key numbers

- The ratio of strange to nonstrange sea quark distributions is \(R_s = (s+\bar s)/(\bar u+\bar d) = \{0.72^{+0.52}_{-0.34},\, 0.46^{+0.30}_{-0.20},\, 0.32^{+0.23}_{-0.15}\}\) for \(x = \{0.01, 0.04, 0.1\}\) at \(Q^2 = 4\) GeV\(^2\).
- The analysis includes nearly 6,000 data points from various experiments.
- The input scale for PDF and FF evolution is set to the charm quark mass, \(m_c = 1.28\) GeV, with the bottom quark mass threshold at \(m_b = 4.18\) GeV.
- The strong coupling boundary condition is \(\alpha_s(M_Z) = 0.118\) at the \(Z\)-boson mass, \(M_Z = 91.18\) GeV.
- The total number of free parameters inferred from the data in the combined full analysis is 258, comprising 215 parameters for input scale PDFs and FFs and 43 free normalization parameters for datasets.

## Context

This work extends previous JAM global QCD analyses (Cocuzza 2021, Moffat 2021) by simultaneously fitting PDFs and FFs, eliminating the need to fix FFs via a reweighing procedure that inflates experimental errors. It contrasts with earlier analyses of strange quark PDFs from neutrino-induced DIS on nuclear targets (CCFR, NuTeV, CHORUS, NOMAD), which suffer from poorly understood nuclear corrections, and from inclusive \(W\) and \(Z\) boson production at the LHC (ATLAS, CMS), which gave mixed results. The combined SIDIS and \(W+\)charm data reduce uncertainties relative to the baseline and yield a suppressed \(R_s\) for \(0.02 \lesssim x \lesssim 0.2\) compared to some previous analyses of ATLAS \(W+\)charm data (ATLAS 2014), enabling more precise extractions of Standard Model parameters such as \(V_{cs}\) and \(\sin^2\theta_W\).
