# Strangeness enhancement across collision systems without a plasma

> We present novel rope hadronization results for strange hadron enhancement in pp, pPb, and PbPb collisions using PYTHIA/Angantyr at LHC energies. With the rope model for string fragmentation, we find that the strangeness and baryon enhancement has a coherent increase across all collision systems as a function of average charged central multiplicity, in qualitative agreement with LHC data. In AA collisions, we find that the baryonic yields overshoot data at high multiplicities, and we discuss how a combination of rope hadronization with other string interactions may tame this rise.

## Key claim

The rope hadronization model, implemented via the parallel frame formalism in PYTHIA/Angantyr, produces a coherent increase in strangeness and baryon yields across pp, pPb, and PbPb collisions as a function of average charged multiplicity, in qualitative agreement with ALICE data; however, in AA collisions the baryonic yields overshoot data at high multiplicities, and the authors argue that including string shoving (repulsion between overlapping strings) may tame this rise.

## Method

The authors apply the rope hadronization model, which treats overlapping colour flux tubes as higher-representation multiplets with increased effective string tension, to pp collisions at 7 TeV, pPb collisions at 5.02 TeV, and PbPb collisions at 2.76 TeV using the PYTHIA/Angantyr event generator. They use a parallel frame formalism to compute fractional overlaps between string pieces from different sub-collisions, then perform a random walk in colour space to determine the multiplet {p,q} and the corresponding effective string tension kappa_eff for each string breakup. The resulting strange hadron to pion yield ratios (K0_s, Lambda, Xi, Omega) are plotted as a function of average central charged multiplicity and compared to ALICE data from 2016.

## Key numbers

* String width assumed as R = 0.5 fm (canonical value); also tested with R = 1.0 fm for comparison.
* Effective string tension for a rope multiplet {p,q} given by kappa^{{p,q}}/kappa^{{1,0}} = (1/4)(p^2 + pq + q^2 + 3p + 3q), where kappa^{{1,0}} is the single triplet string tension.
* The reduction in rope tension when one string breaks is kappa_eff = ((2p+q+2)/4) kappa.
* For an octet rope, the string tension is 9 kappa/4.
* The analysis uses events with central charged multiplicity dN_ch/d|eta| values of approximately 25, 50, 100, and >1000 across the three collision systems.
* The (Lambda + anti-Lambda)/2K0_s ratio is shown for pp at sqrt(s) = 7 TeV as a function of average charged multiplicity.

## Context

This work extends the rope hadronization model previously applied to pp collisions (Bierlich 2014, Bierlich 2022) to pA and AA collisions using the Angantyr model (Bierlich 2018), which generalizes the Lund string model in PYTHIA to nucleus-nucleus collisions. The results contradict the conventional interpretation that strangeness enhancement necessarily signals quark-gluon plasma formation (Rafelski 1982, ALICE 2013, ALICE 2016). The paper enables future work combining rope hadronization with string shoving (Bierlich 2017, Bierlich 2020), colour reconnections, and hadronic rescattering to build a complete string-based description of all collision systems without invoking a thermalized QGP.
