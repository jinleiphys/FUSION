# The strongly intensive observable in pp collisions at LHC energies in the string fusion model

> The properties of the strongly intensive variable characterizing correlations between the number of particles in two separated rapidity interval in pp interactions at LHC energies are studied in the framework of the string fusion model. We perform the MC simulations of string distributions in the impact parameter plane to take into account the experimental conditions of pp collisions. We account the string fusion processes, leading to the formation of string clusters, embedding a finite lattice (a grid) in the impact parameter plane. As a result, we found the dependence of this variable both on the distance between the centers of the observation windows and their width for the minbias pp collisions at several initial energies. Analyzing these dependencies we can extract the important information on the properties of string clusters. We show that in pp collisions at LHC energies the string fusion effects have a significant impact on the behavior of this strongly intensive variable. The role of these effects is increasing with the initial energy and centrality of collisions. In particular, we found that the increase of this variable with initial energy takes place due to the growth of the portion of the fused string clusters in string configurations arising in pp interactions.

# The strongly intensive observable in pp collisions at LHC energies in the string fusion model

## Key claim

The strongly intensive observable Sigma(n_F, n_B) in pp collisions at LHC energies increases with initial energy because the portion of fused string clusters grows, but this observable loses its strictly intensive property when multiple cluster types are present, becoming a weighted average over cluster types with weights that depend on collision energy and centrality.

## Method

The authors performed Monte Carlo simulations of string distributions in the impact parameter plane for pp collisions, using a Gribov-Regge approach to generate the number of cut pomerons (strings) and their transverse positions. They introduced a finite transverse lattice (grid) with cell area equal to the transverse area of a single string (r_str = 0.2 fm), where strings whose centers fall in the same cell are considered fused into a string cluster. Using the generated string configurations, they calculated the weights alpha_eta (the average portion of particles from clusters of eta fused strings) and then computed Sigma(n_F, n_B) via the weighted sum over cluster types, assuming exponential two-particle correlation functions for clusters with parameters that scale with eta.

## Key numbers

- String radius used: r_str = 0.2 fm
- For pp collisions at sqrt(s) = 7 TeV: average number of initial strings <N_str> = 13.4, non-diffractive cross section sigma^{ND} = 52.5 mb (theoretical) and 52.4 mb (MC), multiplicity density at mid-rapidity dN^{ND}/dy = 5.44
- For pp collisions at sqrt(s) = 13 TeV: average number of initial strings <N_str> = 16.0, non-diffractive cross section sigma^{ND} = 56.5 mb (theoretical) and 56.6 mb (MC), multiplicity density at mid-rapidity dN^{ND}/dy = 6.03
- For pp collisions at sqrt(s) = 900 GeV: average number of initial strings <N_str> = 7.8, non-diffractive cross section sigma^{ND} = 39.9 mb, multiplicity density at mid-rapidity dN^{ND}/dy = 3.76
- For pp collisions at sqrt(s) = 60 GeV: average number of initial strings <N_str> = 4.2, non-diffractive cross section sigma^{ND} = 24.9 mb, multiplicity density at mid-rapidity dN^{ND}/dy = 2.44
- Parameters used for the cluster correlation function: correlation length y_c = 2.7, Lambda_0^eta = 0.8 (constant), with y_corr^eta = y_c / sqrt(eta)
- Single string rapidity particle density: mu_0 = 0.7

## Context

This work extends the string fusion model approach previously applied to nucleus-nucleus collisions (Biro 1984, Braun and Pajares 1993) to pp collisions at LHC energies. It builds on earlier studies of forward-backward correlations in the independent string model (Vechernin 2018, EPJA 2019) where the strongly intensive observable Sigma was shown to depend only on single-string parameters, and on the development of strongly intensive observables by Gorenstein and Gazdzicki (2011). The results enable extraction of string cluster properties (multiplicity density and pair correlation functions) from experimental data on Sigma(n_F, n_B) in pp collisions at different energies and centralities, with preliminary centrality studies reported in Belokurova (2020).
