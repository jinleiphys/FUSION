# Nuclear Structure from the In-Medium Similarity Renormalization Group

> Efforts to describe nuclear structure and dynamics from first principles have advanced significantly in recent years. Exact methods for light nuclei are now able to include continuum degrees of freedom and treat structure and reactions on the same footing, and multiple approximate, computationally efficient many-body methods have been developed that can be routinely applied for medium-mass nuclei. This has made it possible to confront modern nuclear interactions from Chiral Effective Field Theory, that are rooted in Quantum Chromodynamics with a wealth of experimental data. Here, we discuss one of these efficient new many-body methods, the In-Medium Similarity Renormalization Group (IMSRG), and its applications in modern nuclear structure theory. The IMSRG evolves the nuclear many-body Hamiltonian in second-quantized form through continuous unitary transformations that can be implemented with polynomial computational effort. Through suitably chosen generators, we drive the matrix representation of the Hamiltonian in configuration space to specific shapes, e.g., to implement a decoupling of low- and high-energy scales, or to extract energy eigenvalues for a given nucleus. We present selected results from Multireference IMSRG (MR-IMSRG) calculations of open-shell nuclei, as well as proof-of-principle applications for intrinsically deformed medium-mass nuclei. We discuss the successes and prospects of merging the (MR-)IMSRG with many-body methods ranging from Configuration Interaction to the Density Matrix Renormalization Group, with the goal of achieving an efficient simultaneous description of dynamic and static correlations in atomic nuclei.

## Key claim

The In-Medium Similarity Renormalization Group (IMSRG) provides a computationally efficient, polynomial-scaling framework for solving the nuclear many-body problem by evolving the Hamiltonian via continuous unitary transformations to decouple low- and high-energy scales, but the method remains approximate due to the truncation of induced many-body operators beyond the normal-ordered two-body level.

## Method

The IMSRG evolves the nuclear many-body Hamiltonian in second-quantized form using operator flow equations of the type dH(s)/ds = [eta(s), H(s)], where the anti-Hermitian generator eta(s) is chosen to drive the Hamiltonian toward a desired block-diagonal or band-diagonal structure in configuration space. For open-shell nuclei, the Multireference IMSRG (MR-IMSRG) extension uses correlated reference states obtained from a preceding mean-field or beyond-mean-field calculation, and the flow equations are solved after normal-ordering the Hamiltonian with respect to that reference. The method is implemented with polynomial computational effort by truncating the operator evolution to the normal-ordered two-body level, discarding induced three-body and higher-rank terms.

## Key numbers

* Ground-state calculations have been performed for nuclei across the chart up to mass number A ~ 100 using IMSRG starting from chiral NN and 3N interactions, as shown in a survey figure up to 2017.
* The chiral EFT breakdown scale Lambda_chi is traditionally considered in the range 700-1000 MeV, with the lower value corresponding to the rho meson mass.
* The typical momentum scale of a nucleon-pion system is q ~ m_pi ~ k_F ~ 140 MeV/c.
* SRG evolution of the chiral N3LO NN interaction to resolution scales lambda = 3 fm^{-1} and lambda = 2 fm^{-1} suppresses off-diagonal momentum-space matrix elements, limiting momentum transfers to |Delta q| ~ 2 fm^{-1} at the lowest scale.
* The Argonne V18 interaction has off-diagonal matrix element tails extending to relative momenta as high as |q| ~ 20 fm^{-1} in the 3S1 partial wave.

## Context

This work extends the SRG methodology previously applied to free-space nuclear interactions (Bogner 2003, Bogner 2007, Roth 2010) into an in-medium framework that directly solves the many-body problem, building on earlier ideas from Wegner (1994) and Kehrein (2006). The IMSRG approach enables ab initio calculations for medium-mass nuclei that were previously only accessible to exact methods for light systems, and it provides a pathway to merge with Configuration Interaction and Density Matrix Renormalization Group methods (White 1992) for describing both dynamic and static correlations simultaneously.
