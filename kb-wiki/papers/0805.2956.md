# Numerical Object Oriented Quantum Field Theory Calculations

> The qft++ package is a library of C++ classes that facilitate numerical (not algebraic) quantum field theory calculations. Mathematical objects such as matrices, tensors, Dirac spinors, polarization and orbital angular momentum tensors, etc. are represented as C++ objects in qft++. The package permits construction of code which closely resembles quantum field theory expressions, allowing for quick and reliable calculations.

## Key claim

The qft++ package enables numerical (not algebraic) quantum field theory calculations for tree-level expressions, with the specific caveat that no work has been done to provide a simple way of performing loop integrals.

## Method

The qft++ library implements C++ template classes for matrices, tensors, Dirac spinors, polarization vectors, and orbital angular momentum tensors, using operator overloading so that code closely resembles symbolic field theory expressions. Matrix multiplication and tensor contraction are performed numerically through nested loops, with the user never needing to manage indices explicitly. The package was designed specifically for event-based partial wave analysis (PWA), where amplitudes must be calculated for upwards of 100 million events.

## Key numbers

- The example calculations run at approximately 3 kHz on a 2.5 GHz processor, meaning about 3000 kinematic points per second.
- For the Compton scattering example, the differential cross section was calculated for a 10 MeV incident photon.
- The omega meson mass and width used in the example are 0.78256 GeV and 0.00844 GeV respectively.

## Context

This work extends the approach of algebraic manipulation packages like FeynCalc and FeynArts, and automated event generators like GRACE and CompHEP, by providing a numerical alternative specifically suited for partial wave analysis. The package has been used in several PWAs, as referenced in Williams' thesis and work by Chung and Anisovich. The formalism follows the Rarita-Schwinger approach for higher-spin particles and Zemach's covariant tensor formalism. The package enables physicists to compare or fit theoretical models to experimental data by coding expressions directly, and can also be used for numeric checks of analytic calculations or for extracting expansion coefficients when analytic solutions are not feasible.
