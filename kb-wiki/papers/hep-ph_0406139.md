# A critique of the angular momentum sum rules and a new angular momentum sum rule

> We present a study of the tensorial structure of the hadronic matrix elements of the angular momentum operators $\bm{J}$. Well known results in the literature are shown to be incorrect, and we have taken pains to derive the correct expressions in three different ways, two involving explicit physical wave packets and the third, totally independent, based upon the rotational properties of the state vectors. Surprisingly it turns out that the results are very sensitive to the type of relativistic spin state used to describe the motion of the particle i.e. whether a canonical (i.e. boost) state or a helicity state is utilized. We present results for the matrix elements of the angular momentum operators, valid in an arbitrary Lorentz frame, both for helicity states and canonical states. These results are relevant for the construction of angular momentum sum rules, relating the angular momentum of a nucleon to the spin and orbital angular momentum of its constituents. It turns out that it is necessary to distinguish carefully whether the motion of the partons is characterized via canonical or helicity spin states. Fortunately, for the simple parton model interpretation, when the proton moves along $OZ$, our results for the sum rule based upon the matrix elements of $J_z$ agree with the often used sum rule found in the literature. But for the components $J_x, J_y$ the results are different and lead to a new and very intuitive sum rule for transverse polarization.

## Key claim

The paper argues that previously published angular momentum sum rules for the nucleon (Jaffe and Manohar, 1990) are incorrect for transverse spin components, because the matrix elements of the angular momentum operators are sensitive to whether canonical (boost) states or helicity states are used to describe the particle's motion; the authors derive correct expressions and show that for a proton moving along OZ the standard sum rule for J_z remains valid, but for J_x and J_y a new sum rule emerges.

## Method

The authors derive the forward matrix elements of the angular momentum operators J_i using three independent methods: two involving explicit physical wave packets for spin-1/2 particles (one using Dirac spinors, one using field theory), and a third based solely on the rotational properties of state vectors valid for arbitrary spin. They then compute the expectation values for both canonical spin states and Jacob-Wick helicity states, and use these to construct angular momentum sum rules for the nucleon in terms of parton distributions.

## Key numbers

* For a massive spin-1/2 particle in a canonical spin state |p, s⟩, the forward matrix element is ⟨p', s|J_i|p, s⟩ = 2p^0(2π)^3 [ (1/2) s_i + i (p × ∇_p)_i ] δ^3(p' - p)
* For a spin-1/2 particle in a helicity state |p, λ⟩, the forward matrix element is ⟨p', λ'|J_i|p, λ⟩ = (2π)^3 2p_0 [ λ η_i(p) + i (p × ∇_p)_i ] δ^3(p' - p) δ_{λλ'}, where η_x = cos(φ)tan(θ/2), η_y = sin(φ)tan(θ/2), η_z = 1
* The Jaffe-Manohar expectation value for J_i is ⟨J_i⟩_{JM} = (1/(4Mp^0)) { (3p_0^2 - M^2) s_i - ((3p_0 + M)/(p_0 + M)) (p·s) p_i }, which differs from the correct result ⟨J_i⟩ = (1/2) s_i except when s = p̂
* For a proton with transverse spin vector s_T, the new sum rule is 1/2 = (1/2) Σ_{q, q̄} ∫ dx Δ_T q^a(x) + Σ_{q, q̄, G} ⟨L_{s_T}⟩^a, where Δ_T q^a(x) ≡ h^q_1(x) are the quark transversity distributions

## Context

This work corrects and extends the foundational angular momentum sum rule analysis of Jaffe and Manohar (1990), who derived sum rules relating nucleon angular momentum to parton spin and orbital contributions but whose results the authors show are incorrect for transverse components. The paper also addresses the wave-packet treatment of Shore and White (1999), which attempted to rigorize the Jaffe-Manohar approach but inherited similar flaws. The new transverse sum rule enables model tests of nucleon structure using transversity distributions and orbital angular momentum contributions, and clarifies that any parton model interpretation must specify whether canonical or helicity spin states are used for the constituents.
