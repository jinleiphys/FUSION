# Symmetry-projected variational calculations with the numerical suite TAURUS I. Variation after particle-number projection

> We present the numerical code TAURUS_vap that solves the variation after particle-number projection equations for symmetry-unrestricted real Bogoliubov quasiparticle states represented in a spherical harmonic oscillator basis. The model space considered is invariant under spatial and isospin rotations but no specific set of orbits is assumed such that the code can carry out both valence-space and no-core calculations. In addition, no number parity is assumed for the Bogoliubov quasiparticle states such that the code can be used to describe even-even, odd-even and odd-odd nuclei. The variational procedure can be performed under several simultaneous constraints on the expectation values of a variety of operators such as the multipole deformations, the pairing field or the components of the angular momentum. To demonstrate the potential and versatility of the code, we perform several example calculations using an empirical shell-model interaction as well as a chiral interaction. The ability to perform advanced variational Bogoliubov calculations offered by this code will, we hope, be beneficial to the shell model and ab initio communities.

## Key claim

The code TAURUS_vap solves the variation after particle-number projection (VAPNP) equations for real general Bogoliubov quasiparticle states in a spherical harmonic oscillator basis, enabling both valence-space and no-core calculations without assuming number parity, but the variational space is restricted to real Bogoliubov matrices and Hamiltonians that respect rotational and parity symmetries.

## Method

The code performs a variational minimization of the particle-number projected energy by solving the projected gradient equations for a Bogoliubov reference state represented in a spherical harmonic oscillator single-particle basis. The gauge integrals for particle-number projection are discretized, and the norm overlaps are computed using Pfaffian algebra while the rotated Hamiltonian kernels are evaluated via the Generalized Wick Theorem. The variational procedure can be carried out under simultaneous constraints on expectation values of multipole deformations, the pairing field, or angular momentum components.

## Key numbers

- The model space dimension is given by \( d_{\mathcal{M}} = 2 \sum_{\breve{a}}^{\mathcal{M}} (2j_a + 1) \), where the sum runs over all orbits in the model space
- The oscillator length is defined as \( b = \sqrt{\hbar / (m \omega)} \) with \( m = (m_p + m_n)/2 \)
- The radial normalization constant is \( \mathcal{N}_{n_a l_a} = \sqrt{2^{n_a + l_a + 2} n_a! / [\pi^{1/2} (2 n_a + 2 l_a + 1)!!]} \)
- The code uses a discretized projection operator with integration interval reduced to \([0, \pi]\) for the nucleon-number parity operator, improving numerical efficiency

## Context

This work extends the capabilities of existing HFB and beyond-mean-field codes (such as those by Bennaceur (2005), Carlsson (2010), Maruhn (2014), Ryssens (2015), Perez (2017), and Schunck (2017)) by providing a publicly available code that can handle general Hamiltonians from effective field theory or shell-model interactions, not just energy density functionals. The code enables advanced variational Bogoliubov calculations for the shell-model and ab initio communities, and has already been applied with a shell-model interaction by Bally (2019) and with a chiral interaction evolved through the In-Medium Similarity Renormalization Group by Yao (2020).
