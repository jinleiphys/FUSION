# Ab initio translationally invariant nucleon-nucleus optical potentials

> We combine the \textit{ab initio} symmetry-adapted no-core shell model (SA-NCSM) with the single-particle Green's function approach to construct optical potentials rooted in first principles. Specifically, we show that total cross sections and phase shifts for neutron elastic scattering from a $^4$He target with projectile energies between 0.5 and 10 MeV closely reproduce the experiment. In addition, we discuss an important new development that resolves a long-standing issue with spurious center-of-mass motion in the Green's function formalism for many-body approaches. The new development opens the path for first-principle predictions of cross sections for elastic scattering of single-nucleon projectiles, nucleon capture and deuteron breakup reactions, feasible for a broad range of open-shell spherical and deformed nuclei in the SA-NCSM approach.

## Key claim

The paper demonstrates that combining the symmetry-adapted no-core shell model (SA-NCSM) with the single-particle Green's function approach yields translationally invariant nucleon-nucleus optical potentials from first principles, and shows that for neutron elastic scattering off ^4He at projectile energies between 0.5 and 10 MeV, the calculated total cross sections and phase shifts closely reproduce experimental values, provided the long-standing issue of spurious center-of-mass motion in the Green's function formalism is resolved via the Lawson procedure.

## Method

The authors combine the SA-NCSM, which provides ab initio wavefunctions for the target nucleus and the A±1 systems, with the single-particle Green's function formalism to construct nonlocal, energy-dependent nucleon-nucleus optical potentials. The Green's function is computed in a harmonic oscillator single-particle basis using the Lehmann representation, and the optical potential is obtained by inverting the Green's function in configuration representation via V = G₀⁻¹ - G⁻¹, where G₀ is the free propagator. To remove spurious center-of-mass contamination, the Lawson procedure is applied with a Lagrange multiplier term λ_CM N_CM added to the Hamiltonian, and the translationally invariant Green's function is extracted by projecting onto states with zero center-of-mass excitations (Γ=0) and using Talmi-Moshinsky brackets to relate laboratory-frame overlaps to intrinsic-frame overlaps. Phase shifts are then evaluated using the R-matrix method with the resulting nonlocal potential.

## Key numbers

- Neutron elastic scattering from ^4He is calculated for projectile energies between 0.5 and 10 MeV.
- The Lawson multiplier λ_CM is typically set to ~50 MeV (with a range of 50–100 MeV noted as typical).
- The condition |E| < λ_CM ensures exclusion of spurious center-of-mass poles, which is always valid for the low-energy regime of applicability.
- The harmonic oscillator basis truncation is determined by N_max, the total number of harmonic oscillator excitations above the lowest-energy configuration, which defines n_max, the highest HO shell available to the A and A±1 systems.

## Context

This work extends the Green's function approach to optical potentials previously applied with the coupled-cluster method (Rotureau et al., 2017; Phys. Rev. C 98, 044625) and self-consistent Green's function method (Idini, 2019) for closed-shell nuclei, and with the no-core shell model for intermediate-energy scattering (Burrows et al., 2020; Vorabbi et al., 2022). It also builds on earlier studies of n-^4He scattering using the no-core shell model with continuum (NCSMC) (Quaglioni and Navratil, 2009; Kravvaris et al., 2020), Faddeev-Yakubovsky approaches (Lazauskas, 2018), and the single-state harmonic oscillator representation of scattering equations (Phys. Rev. C 98, 044624). The key advance is the resolution of center-of-mass spuriosity in the Green's function formalism, which enables first-principles optical potentials for open-shell spherical and deformed nuclei, and opens the path to predictions for elastic scattering, nucleon capture, and deuteron breakup reactions across a broad range of nuclei.
