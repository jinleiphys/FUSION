# Cross-Comparison of Sampling Algorithms for Pulse Profile Modeling of PSR J0740+6620

> In the last few years, NICER data has enabled mass and radius inferences for various pulsars, and thus shed light on the equation of state for dense nuclear matter. This is achieved through a technique called pulse profile modeling. The importance of the results necessitates careful validation and testing of the robustness of the inference procedure. In this paper, we investigate the effect of sampler choice for X-PSI (X-ray Pulse Simulation and Inference), an open-source package for pulse profile modeling and Bayesian statistical inference that has been used extensively for analysis of NICER data. We focus on the specific case of the high-mass pulsar PSR J0740+6620. Using synthetic data that mimics the most recently analyzed NICER and XMM-Newton data sets of PSR J0740+6620, we evaluate the parameter recovery performance, convergence, and computational cost for MultiNest's multimodal nested sampling algorithm and UltraNest's slice nested sampling algorithm. We find that both samplers perform reliably, producing accurate and unbiased parameter estimation results when analyzing simulated data. We also investigate the consequences for inference using the real data for PSR J0740+6620, finding that both samplers produce consistent credible intervals.

## Key claim

Both MultiNest's multimodal nested sampling and UltraNest's slice nested sampling produce accurate, unbiased parameter estimates and consistent credible intervals for PSR J0740+6620 when used with X-PSI, but the paper's validation is limited to the specific case of this high-mass pulsar and uses synthetic data that mimics the most recently analyzed NICER and XMM-Newton data sets.

## Method

The authors used X-PSI (X-ray Pulse Simulation and Inference) with the Oblate Schwarzschild + Doppler approximation to model energy-resolved X-ray pulses from PSR J0740+6620, employing a fully ionized hydrogen atmosphere model (NSX) and two circular uniform temperature hot regions (ST-U model). They generated ten synthetic datasets using parameter vectors randomly sampled from the prior distributions of Salmi (2024), then performed Bayesian inference with both MultiNest (using 4x10^3 live points, efficiency 0.1) and UltraNest's slice sampler (minimum 400 live points, 240 steps) to compare parameter recovery, convergence, and computational cost.

## Key numbers

- Equatorial radius from Salmi (2024) using MultiNest: 12.49^{+1.28}_{-0.88} km, with gravitational mass 2.073^{+0.069}_{-0.069} solar masses
- Equatorial radius from Dittmann (2024) using a different procedure: 12.76^{+1.49}_{-1.02} km (with radius prior upper limit set to 16 km for consistency)
- MultiNest parameter recovery test combined p-value: 0.976 (minimum individual p-value 0.258 for phi_s)
- UltraNest parameter recovery test combined p-value: 0.607 (minimum individual p-value 0.084 for log10(T_s))
- MultiNest computational cost: average 4.8k core hours per run
- UltraNest computational cost: average 45k core hours per run
- NICER dataset exposure time: 2.73 Ms of on-source exposure (2018 September 21 to 2022 April 21)

## Context

This work extends the pulse profile modeling analyses of PSR J0740+6620 by Salmi (2024) and Riley (2021), who used MultiNest with X-PSI to infer mass and radius, and responds to concerns about sampler-induced biases raised by Dittmann (2024) and Vinciguerra (2023). The cross-comparison validates that previous NICER results are robust to sampler choice, while also demonstrating that UltraNest's slice sampler provides safer uncertainty estimation and self-diagnostic convergence tools (relative jump distance and Mann-Whitney-Wilcoxon U-test) at higher computational cost, enabling future work to verify convergence without requiring multiple expensive runs.
