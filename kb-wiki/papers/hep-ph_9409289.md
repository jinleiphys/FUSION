# FORTRAN program for a numerical solution of the nonsinglet Altarelli-Parisi equation

> We investigate a numerical solution of the flavor-nonsinglet Altarelli-Parisi equation with next-to-leading-order $α_s$ corrections by using Laguerre polynomials. Expanding a structure function (or a quark distribution) and a splitting function by the Laguerre polynomials, we reduce an integrodifferential equation to a summation of finite number of Laguerre coefficients. We provide a FORTRAN program for Q$^2$ evolution of nonsinglet structure functions (F$_1$, F$_2$, and F$_3$) and nonsinglet quark distributions. This is a very effective program with typical running time of a few seconds on SUN-IPX or on VAX-4000/500. Accurate evolution results are obtained by taking approximately twenty Laguerre polynomials.

## Key claim

The paper presents a FORTRAN program (LAG2NS) that solves the flavor-nonsinglet Altarelli-Parisi equation with next-to-leading-order (NLO) corrections using Laguerre polynomials, claiming accurate evolution results with approximately twenty Laguerre polynomials and typical running times of a few seconds on SUN-IPX or VAX-4000/500, though the accuracy degrades to within 3% only in the range 0.001 < x < 0.8 and becomes worse at very small x (x < 0.001) and very large x (x > 0.9).

## Method

The authors expand the quark distribution (or structure function) and the splitting function in Laguerre polynomials, reducing the integrodifferential Altarelli-Parisi equation to a finite sum of Laguerre coefficients. The LO evolution operator is solved first, then the NLO operator is obtained from it using a relation involving the NLO splitting function and the running coupling constant. The program is written in FORTRAN 77 and uses double precision arithmetic with the modified minimal subtraction (MS-bar) renormalization scheme.

## Key numbers

- Typical running time: approximately 5 seconds on SUN-IPX and 3 seconds on VAX-4000/500 for initial distributions of the form a x^b (1-x)^c.
- Accurate results obtained by taking approximately twenty Laguerre polynomials (NMAX ~ 20).
- Accuracy within 3% in the x range 0.001 < x < 0.8 for evolved valence-quark distributions.
- Differences between the authors' NLO evolution and the HMRS-B program results are of order 0.5% in the x range 0.01 < x < 0.5, and of order 2% at large x.
- Test run parameters: F0=4.0, IORDER=2, ITYPE=4, IMORP=1, Q02=4.0 GeV^2, Q2=200.0 GeV^2, DLAM=0.19 GeV, NMAX=30, IQN=1, ILOG=2, XMIN=0.01, XMAX=1.0, NXSTEP=30.
- QCD scale parameter Lambda used in comparisons: 0.19 GeV (HMRS-B comparison) and 0.21 GeV (CCFR F3 comparison), with four flavors.
- Example initial distributions: HMRS-B valence-quark distribution [x u_v + x d_v] = 0.5469 x^0.237 (1-x)^4.07 (1+23.8x); CCFR F3 structure function x F3 = 5.976 x^0.766 (1-x)^3.101.

## Context

This work extends the earlier leading-order Laguerre method program LAG1 by Kobayashi, Konuma, and Kumano (1994) to include NLO corrections, building on the Laguerre polynomial method first investigated by Furmanski and Petronzio (1982) and further developed by Ramsey (1992). The program enables fast and accurate Q^2 evolution of nonsinglet structure functions and quark distributions for comparison with experimental data such as CDHSW neutrino data and HMRS-B parton distributions, and the authors state they will work on flavor-singlet structure functions and nuclear structure functions in the future.
