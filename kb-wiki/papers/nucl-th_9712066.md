# Optimized Discretization of Sources Imaged in Heavy-Ion Reactions

> We develop the new method of optimized discretization for imaging the relative source from two particle correlation functions. In this method, the source resolution depends on the relative particle separation and is adjusted to available data and their errors. We test the method by restoring assumed pp sources and then apply the method to pp and IMF data. In reactions below 100 MeV/nucleon, significant portions of the sources extend to large distances (r > 20 fm). The results from the imaging show the inadequacy of common Gaussian source-parametrizations. We establish a simple relation between the height of the pp correlation function and the source value at short distances, and between the height and the proton freeze-out phase-space density.

## Key claim

The paper develops an optimized discretization method for imaging the relative source function from two-particle correlation functions that adjusts the source resolution based on relative particle separation and available data errors, without requiring explicit constraints like positivity for stabilization. The authors claim that common Gaussian source parametrizations are inadequate for describing the sources imaged from heavy-ion reactions below 100 MeV/nucleon, as significant portions of the sources extend to large distances (r > 20 fm).

## Method

The authors discretize the source function into radial intervals and vary the interval boundaries to minimize the sum of relative errors of the source values, using a model source of exponential form S(r) proportional to exp(-r/R_0') with R_0' of order a few fm. The optimal set of radial boundaries is found by minimizing the sum of relative errors at fixed number of intervals N and r_0 = 0, using only the measured relative momenta and their errors (not the correlation function values themselves). Once the optimal intervals are determined, the source values are obtained by solving the matrix equation S = (K^T B K)^{-1} K^T B R^{exp}, where K is the discretized kernel matrix and B contains the inverse squared errors.

## Key numbers

- For the simulated pp source test with R_0 = 3.5 fm, the optimal discretization with N=7 intervals produced a first interval width of 2 fm and a sixth interval width of 3.6 fm
- The rms error magnitude of 0.015 was used for the simulated pp correlation function, representative of the pp data from Gonin (1990)
- The pp data from the 14N + 27Al reaction at 75 MeV/nucleon were analyzed in three total momentum intervals: 270-390, 450-780, and 840-1230 MeV/c
- The relation between the pp correlation function maximum and the source value at short distances is C_P(20 MeV/c) approximately equal to 1 + 540 fm^3 * S_P(r approaches 0), valid to within plus or minus 20% for R_0 = 2.5-6.0 fm
- The proton freeze-out phase-space density relation is given as <f(p)> = 0.0018 (GeV/c)^3 * (C_{2p}^{max} - 1) * (E_p/m) * (dN_p/dp)
- For the IMF data from central 84Kr + 197Au reactions, the minimum first interval boundary was set to r_1^{min} = 7.0 fm, with typical touching distance r_t approximately 5 fm
- The generalized chaoticity parameter lambda(20 fm) for IMF sources was 20-30% below 1, while lambda over the whole restored image (cutoff of 90 fm) was consistent with 1 within errors

## Context

This work extends the imaging methods first demonstrated by Brown and Danielewicz (1997) for extracting relative source functions from two-particle correlation functions in heavy-ion reactions. The optimized discretization method improves upon the earlier equal-interval discretization approach and addresses the stability problems that Tikhonov (1963) first discussed for inversion problems. The results enable direct comparison of imaged sources with predictions from the Boltzmann-equation transport model of Danielewicz (1995), revealing that the model describes high-momentum proton sources adequately but shows discrepancies for lower momentum intervals where significant portions of the source are missing from the imaged regions.
