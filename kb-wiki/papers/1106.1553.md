# Numerical solution of $Q^2$ evolution equations for fragmentation functions

> Semi-inclusive hadron-production processes are becoming important in high-energy hadron reactions. They are used for investigating properties of quark-hadron matters in heavy-ion collisions, for finding the origin of nucleon spin in polarized lepton-nucleon and nucleon-nucleon reactions, and possibly for finding exotic hadrons. In describing the hadron-production cross sections in high-energy reactions, fragmentation functions are essential quantities. A fragmentation function indicates the probability of producing a hadron from a parton in the leading order of the running coupling constant $α_s$. Its $Q^2$ dependence is described by the standard DGLAP (Dokshitzer-Gribov-Lipatov-Altarelli-Parisi) evolution equations, which are often used in theoretical and experimental analyses of the fragmentation functions and in calculating semi-inclusive cross sections. The DGLAP equations are complicated integro-differential equations, which cannot be solved in an analytical method. In this work, a simple method is employed for solving the evolution equations by using Gauss-Legendre quadrature for evaluating integrals, and a useful code is provided for calculating the $Q^2$ evolution of the fragmentation functions in the leading order (LO) and next-to-leading order (NLO) of $α_s$. The renormalization scheme is $\overline{MS}$ in the NLO evolution. Our evolution code is explained for using it in one's studies on the fragmentation functions.

## Key claim

The authors provide a publicly available Fortran77 code (ffevol1.0) that solves the timelike DGLAP evolution equations for fragmentation functions using a direct x-space method with Gauss-Legendre quadrature, but the accuracy of the gluon evolution depends significantly on the number of Gauss-Legendre points, especially near x=1.

## Method

The DGLAP integro-differential equations are solved by Euler's method for the derivative with respect to ln Q² and Gauss-Legendre quadrature for the x-integral, using linear-x scaling for x>0.1 and logarithmic-x scaling for x<0.1. The code provides evolution in both leading order (LO) and next-to-leading order (NLO) of α_s in the MS-bar renormalization scheme, with initial fragmentation functions supplied by the user in subroutines FF_INI and HQFF. The HKNS07 parametrization is included as an example of initial functions at Q₀²=1 GeV² for light partons and at mass-threshold values for charm and bottom.

## Key numbers

- The code uses 580 t-steps (Nt), 160 x-steps (Nx), and 32 Gauss-Legendre points (NGLI) as default parameters for evolving the HKNS07 functions
- The scale parameter Λ for Nf=4 is 0.323 GeV in the NLO HKNS07 parametrization
- Heavy-quark mass thresholds are set at m_c=1.43 GeV and m_b=4.3 GeV in the HKNS07 example
- The initial Q² scale is 1.0 GeV² and the maximum Q² is 1.558×10⁸ GeV² in the HKNS07 library example
- The code requires 130 M bytes of RAM and runs in a few seconds on an HP DL360G5-DC-X5160 computer
- The minimum x value used in the example is x_min=1×10⁻²

## Context

This work extends the brute-force Euler method for DGLAP evolution previously used by the same authors (Hirai, Kumano, 2000) by replacing Simpson's rule with Gauss-Legendre quadrature for improved accuracy. It provides a public code to fill the gap noted by the authors that no useful code was available for calculating Q² variations of fragmentation functions, unlike the situation for parton distribution functions. The code enables users to evolve fragmentation functions from hadron-model calculations (such as Bentz, 2003) or from global analyses to compare with experimental data at different Q² scales, and can be used for studies of semi-inclusive hadron production at RHIC and LHC.
