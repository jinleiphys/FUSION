# General pairing interactions and pair truncation approximations for fermions in a single-j shell

> We investigate Hamiltonians with attractive interactions between pairs of fermions coupled to angular momentum J. We show that pairs with spin J are reasonable building blocks for the low-lying states. For systems with only a J = Jmax pairing interaction, eigenvalues are found to be approximately integers for a large array of states, in particular for those with total angular momenta I le 2j. For I=0 eigenstates of four fermions in a single-j shell we show that there is only one non-zero eigenvalue. We address these observations using the nucleon pair approximation of the shell model and relate our results with a number of currently interesting problems.

# General pairing interactions and pair truncation approximations for fermions in a single-j shell

## Key claim
The authors show that attractive pairing interactions between fermions coupled to angular momentum J favor pairs with that same angular momentum J as building blocks for low-lying states, but this approximation is not perfect for all cases, for example, with J=14 pairs and four nucleons in a j=25/2 shell, the ground state angular momentum is not correctly reproduced, even though the low-lying energies are still reasonably approximated.

## Method
The authors diagonalize Hamiltonians with attractive pairing interactions between fermions coupled to angular momentum J in a single-j shell, primarily for four nucleons (n=4). They compare exact shell-model results with results from a truncated space containing only pairs with angular momentum J, using the nucleon pair approximation of the shell model. They analyze the matrix elements in a basis of two-fermion-pair states and examine the role of nine-j symbols in producing approximate integer eigenvalues.

## Key numbers
- For four nucleons in a j=31/2 shell with J=30 pairing, all eigenvalues for states with I < 22 are 0, -1, or -2 to within a precision of 0.01.
- For states with 22 ≤ I ≤ 52 in the same system, the three integer eigenvalues continue to be valid except for 7 eigenvalues that are not close to 0, -1, or -2.
- For J=6 and J=14 pairing with four nucleons, the J-pair approximation correctly reproduces the ground state angular momentum in most cases, with exceptions at 2j=19 and 2j=25 for J=14.
- The number of I=0 states for n=4 is D_0^{(j)} = [(2j+3)/6], giving 1, 1, 1, 2, 2, 2, 3, 3, 3, ... for 2j = 3, 5, 7, 9, 11, 13, 15, ...
- For n=4 and I=0, there is exactly one non-zero eigenvalue, with value E_0^{J(j)} = -N^{(0)}_{JJ;JJ}.

## Context
This work extends the seniority scheme introduced by Racah for atomic spectra and later applied in nuclear physics, where S pairs (angular momentum zero) are related to monopole pairing. It also relates to the interacting boson model (IBM) introduced by Arima and Iachello, where s and d bosons correspond to correlated nucleon pairs. The finding that only one non-zero eigenvalue exists for I=0 states of four fermions sheds light on the problem of angular momentum zero ground state dominance in many-body systems interacting by random interactions, as studied by Johnson. The results enable a simple argument for the large probability of I=0 ground states in four-fermion systems and explain the staggering of this probability with j.
