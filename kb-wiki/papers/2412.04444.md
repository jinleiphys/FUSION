# Block Lanczos algorithm for lattice QCD spectroscopy and matrix elements

> Recent work introduced a new framework for analyzing correlation functions with improved convergence and signal-to-noise properties, as well as rigorous quantification of excited-state effects, based on the Lanczos algorithm and spurious eigenvalue filtering with the Cullum-Willoughby test. Here, we extend this framework to the analysis of correlation-function matrices built from multiple interpolating operators in lattice quantum chromodynamics (QCD) by constructing an oblique generalization of the block Lanczos algorithm, as well as a new physically motivated reformulation of the Cullum-Willoughby test that generalizes to block Lanczos straightforwardly. The resulting block Lanczos method directly extends generalized eigenvalue problem (GEVP) methods, which can be viewed as applying a single iteration of block Lanczos. Block Lanczos provides qualitative and quantitative advantages over GEVP methods analogous to the benefits of Lanczos over the standard effective mass, including faster convergence to ground- and excited-state energies, explicitly computable two-sided error bounds, straightforward extraction of matrix elements of external currents, and asymptotically constant signal-to-noise. No fits or statistical inference are required. Proof-of-principle calculations are performed for noiseless mock-data examples as well as two-by-two proton correlation-function matrices in lattice QCD.

## Key claim

The block Lanczos method extends the previously introduced Lanczos framework for analyzing single correlation functions to correlation-function matrices built from multiple interpolating operators, providing qualitative and quantitative advantages over generalized eigenvalue problem (GEVP) methods analogous to the benefits of Lanczos over standard effective mass estimators, including faster convergence to ground- and excited-state energies and asymptotically constant signal-to-noise, with the caveat that these advantages are demonstrated in proof-of-principle calculations on noiseless mock data and a single two-by-two proton correlation-function matrix in lattice QCD.

## Method

The authors construct an oblique generalization of the block Lanczos algorithm that uses distinct left and right bases, deriving recursion relations on finite-dimensional correlation-function matrix elements that can be applied to noisy lattice QCD data. They reformulate the Cullum-Willoughby test for filtering spurious noise-artifact states, giving it a physical interpretation as identifying states with zero overlap with the entire set of interpolating operators, and generalize this test to the block case. The method is tested on noiseless mock-data examples and a two-by-two proton correlation-function matrix in lattice QCD, with comparisons to GEVP methods and scalar Lanczos.

## Key numbers

* For the noiseless mock-data examples, the block Lanczos method demonstrates faster convergence to ground- and excited-state energies compared to GEVP methods, with explicitly computable two-sided error bounds.
* The block Lanczos method is applied to a two-by-two proton correlation-function matrix in lattice QCD, where it extracts the spectrum and overlap factors with asymptotically constant signal-to-noise properties.
* The GEVP method is shown to coincide with a single iteration of block Lanczos, meaning block Lanczos with m=1 iterations reproduces GEVP results.

## Context

This work extends the Lanczos analysis framework introduced by Wagman (2024) and Hackett (2024) for single correlation functions to correlation-function matrices, directly generalizing GEVP methods developed by Fox (1982), Michael (1982), Luscher (1990), and Blossier (2009). The block Lanczos method provides a superset of GEVP methods with advantages including faster convergence, two-sided error bounds, and straightforward extraction of matrix elements of external currents, enabling more reliable determination of excited-state energies and matrix elements in lattice QCD calculations.
