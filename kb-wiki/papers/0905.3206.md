# Continuum Hartree-Fock-Bogoliubov theory for weakly bound deformed nuclei using coordinate-space Green's function method

> We formulate a new scheme of the Hartree-Fock-Bogoliubov mean-field theory applicable to weakly bound and pair correlated deformed nuclei using the coordinate-space Green's function technique. On the basis of a coupled-channel representation of the quasiparticle wave function expanded in terms of the partial waves, we impose the correct boundary condition of the asymptotically out-going waves on the continuum quasiparticle states. We perform numerical analysis for $^{38}$Mg to illustrate properties of the continuum quasiparticle states and the pair correlation in deformed nuclei near the neutron drip-line.

## Key claim
The paper formulates a new continuum Hartree-Fock-Bogoliubov (HFB) method for weakly bound deformed nuclei using a coordinate-space Green's function technique with correct asymptotic boundary conditions, but the numerical demonstration is limited to a simplified model using a fixed Woods-Saxon potential rather than a fully selfconsistent Hartree-Fock treatment, and the authors state they intend to clarify qualitative features rather than make precise quantitative predictions.

## Method
The authors extend the Green's function approach to the HFB equation for deformed nuclei by employing a coupled-channel representation based on partial wave expansion, imposing asymptotically out-going wave boundary conditions on continuum quasiparticle states. They construct the HFB Green's function in coordinate space using the coupled-channel formalism, then compute the generalized density matrix via contour integration in the complex energy plane. The method is applied to the neutron-rich deformed nucleus ³⁸Mg using an axially-symmetric deformed Woods-Saxon potential with a selfconsistent density-dependent delta interaction for pairing, solving the coupled-channel equations numerically with the Runge-Kutta-Nyström method.

## Key numbers
- The deformation parameter β = 0.3 is used for the Woods-Saxon potential, representing a typical value from mean-field predictions.
- The cut-off quasiparticle energy is E_cut = 60 MeV.
- The maximal magnetic quantum number Ω_max = 21/2 is adopted for truncation of quasiparticle states.
- The radial grid extends to R_max = 15 fm with an equidistant interval Δr = 0.2 fm.
- The pairing interaction parameters are v_0 = -458.4 MeV fm⁻³, η = 0.76, α = 0.59, and ρ_c = 0.08 fm⁻³, chosen to reproduce the ¹S scattering length a = -18.5 fm.
- The Gauss-Legendre quadrature uses N_θ = 60 points for angular integration and M_ζ = 100 points for the contour integral.

## Context
This work extends the continuum HFB Green's function method previously developed for spherical nuclei (Matsuo, 2001) to deformed nuclei, building on Hamamoto's (2004, 2005) analysis of quasiparticle motion in deformed Woods-Saxon potentials with correct asymptotic boundary conditions but without selfconsistent pairing. The formulation is intended as a first step toward developing a continuum quasiparticle random phase approximation (QRPA) for deformed nuclei, which the authors note can be straightforwardly formulated once the quasiparticle Green's function for deformed nuclei is constructed. The method enables proper description of exponentially decaying density tails in weakly bound deformed nuclei near the neutron drip-line, which the box boundary condition fails to reproduce.
