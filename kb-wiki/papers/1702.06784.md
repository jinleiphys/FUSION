# Analytic matrix elements with shifted correlated Gaussians

> Matrix elements between shifted correlated Gaussians of various potentials with several form-factors are calculated analytically. Analytic matrix elements are of importance for the correlated Gaussian method in quantum few-body physics.

## Key claim
The paper derives analytic matrix elements for shifted correlated Gaussians with short-range potentials, filling a gap where such analytic expressions were previously missing (Mitroy 2013). The key condition is that the form-factor must belong to the class \(f(r)=r^n e^{-\gamma r^2-\mu r}/r\) for the matrix elements to be fully analytic.

## Method
The author computes overlap, kinetic energy, and potential energy matrix elements between two shifted correlated Gaussians with parameters \((A,\mathbf{s})\) and \((A',\mathbf{s}')\). For central potentials, the matrix element reduces to a one-dimensional integral involving the radial form-factor \(f(r)\), which is evaluated analytically for screened Yukawa, Yukawa, screened Coulomb, screened exponential, exponential, and Gaussian form-factors. Tensor and spin-orbit matrix elements are then obtained by taking derivatives of the central matrix element with respect to the shift-vector components.

## Key numbers
- Overlap matrix element: \(\langle g'|g\rangle = e^{\frac14\mathbf{v}^T B^{-1}\mathbf{v}} \left(\frac{\pi^N}{\det(B)}\right)^{3/2}\), where \(B=A'+A\) and \(\mathbf{v}=\mathbf{s}'+\mathbf{s}\).
- Kinetic energy matrix element: \(\langle g'|\hat{K}|g\rangle = \left[6\,\mathrm{trace}(A'\Lambda A B^{-1}) + (\mathbf{s}'-2A'\mathbf{u})^T\Lambda(\mathbf{s}-2A\mathbf{u})\right] M\), with \(\mathbf{u}=\frac12 B^{-1}\mathbf{v}\).
- Gaussian central potential: \(\langle g'|e^{-\gamma\mathbf{r}^T w w^T\mathbf{r}}|g\rangle = e^{\frac14\mathbf{v}^T B'^{-1}\mathbf{v}} \left(\frac{\pi^N}{\det(B')}\right)^{3/2}\), where \(B'=B+\gamma w w^T\).
- Coulomb form-factor: \(J[1/r] = \frac{\mathrm{erf}(\sqrt{\beta}q)}{q}\), with \(\beta=(w^T B^{-1}w)^{-1}\) and \(q=w^T\mathbf{u}\).
- Screened Yukawa form-factor: \(J[e^{-\gamma r^2-\mu r}/r]\) given by equation (6) involving error functions and exponentials.
- Gaussian form-factor: \(J[e^{-\gamma r^2}] = \left(\frac{\beta}{\beta+\gamma}\right)^{3/2} e^{-\frac{\beta\gamma q^2}{\beta+\gamma}}\).

## Context
This work extends the analytic matrix element toolkit for the correlated Gaussian method (Mitroy 2013, Cafiero 2003, Sharkey 2010, DailyGreene 2014) by providing expressions for shifted Gaussians with short-range potentials, which were previously missing. The results enable efficient numerical optimization of variational wavefunctions in few-body nuclear and atomic systems where short-range interactions dominate, and the author suggests representing general potentials as linear combinations of Gaussians for maximum analytic simplicity.
