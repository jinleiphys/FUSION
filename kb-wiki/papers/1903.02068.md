# Odd-mass nuclei in the cluster shell model

> In this contribution, we present the cluster shell model which is analogous to the Nilsson model, but for cluster potentials. Special attention is paid to the consequences of the discrete symmetries of three alpha-particles in an equilateral triangle configuration. This configuration is characterized by a special structure of the rotational bands which can be used as a fingerprint of the underlying geometric configuration. The cluster shell model is applied to the nucleus 13C.

# Odd-mass nuclei in the cluster shell model

## Key claim

The cluster shell model (CSM), analogous to the Nilsson model but for cluster potentials, can describe odd-mass nuclei such as 13C as a system of three alpha-particles in an equilateral triangle configuration plus an additional neutron moving in the deformed field generated by the cluster, provided the discrete symmetries of the D'_{3h} double group are respected.

## Method

The authors apply the algebraic cluster model (ACM) to describe the rotation-vibration spectrum of 12C as a three-alpha-particle system with D_{3h} triangular symmetry, using a Hamiltonian with vibrational and rotational terms. For 13C, they develop the cluster shell model in which the odd neutron moves in a deformed potential obtained by convolving the alpha-particle density distribution with the nucleon-alpha interaction, plus spin-orbit and Coulomb terms. The single-particle levels are classified by irreducible representations of the double group D'_{3h} (E_{1/2}^{(+)}, E_{1/2}^{(-)}, and E_{3/2}), and rotational bands are constructed by coupling the intrinsic states to the rotational bands of the 12C core.

## Key numbers

- For 12C, the ACM predicts B(E2;2_1^+ → 0_1^+) = 8.4 e^2 fm^4 (experiment: 7.61 ± 0.42 e^2 fm^4)
- For 12C, the ACM predicts B(E3;3_1^- → 0_1^+) = 73 e^2 fm^6 (experiment: 104 ± 14 e^2 fm^6)
- For 12C, the ACM predicts quadrupole moment Q(2_1^+) = 5.2 e fm^2 (experiment: 5.3 ± 4.4 e fm^2)
- For 12C, the ACM predicts charge radius ⟨r^2⟩^{1/2}_ch = 2.389 fm (experiment: 2.468 ± 0.012 fm)
- The alpha-particle size parameter is α = 0.56 fm^{-2}, determined from the radius of the alpha-particle
- The distance parameter β = 1.74 fm, determined from the first minimum of the elastic form factor of 12C
- For 13C, the CSM predicts B(E2;3/2_1^- → 1/2_1^-) = 8.3 e^2 fm^4 (experiment: 6.4 ± 1.5 e^2 fm^4)
- For 13C, the CSM predicts B(E2;5/2_1^- → 1/2_1^-) = 5.5 e^2 fm^4 (experiment: 5.6 ± 0.4 e^2 fm^4)
- For 13C, the CSM predicts B(E3;5/2_1^+ → 1/2_1^-) = 42 e^2 fm^6 (experiment: 100 ± 40 e^2 fm^6)
- The B(EL) ratios between 13C and 12C are given by numerical factors: B(E2;3/2_1^- → 1/2_1^-) = 0.99 × B(E2;2_1^+ → 0_1^+), B(E2;5/2_1^- → 1/2_1^-) = 0.66 × B(E2;2_1^+ → 0_1^+), B(E3;5/2_1^+ → 1/2_1^-) = 0.57 × B(E3;3_1^- → 0_1^+)

## Context

This work extends the algebraic cluster model (ACM) developed by Bijker and collaborators (2000, 2002, 2014, 2017) for even-mass alpha-cluster nuclei like 8Be, 12C, and 16O to odd-mass nuclei via the cluster shell model. The approach is analogous to the Nilsson model (Nilsson) but replaces the deformed mean field with a cluster potential. The results provide evidence for triangular D_{3h} symmetry in 12C and 13C, and the predicted rotational bands and electromagnetic transitions can serve as fingerprints for identifying such geometric configurations in other light nuclei. The work enables future studies of other kα + x nuclei and tests of cluster versus shell-model descriptions of light nuclear structure.
