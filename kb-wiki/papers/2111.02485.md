# Three-dimensional Skyrme Hartree-Fock-Bogoliubov solver in coordinate-space representation

> The coordinate-space representation of the Hartree-Fock-Bogoliubov theory is the method of choice to study weakly bound nuclei whose properties are affected by the quasiparticle continuum space. To describe such systems, we developed a three-dimensional Skyrme-Hartree-Fock-Bogoliubov solver HFBFFT based on the existing, highly optimized and parallelized Skyrme-Hartree-Fock code Sky3D. The code does not impose any self-consistent spatial symmetries such as mirror inversions or parity. The underlying equations are solved in HFBFFT directly in the canonical basis using the fast Fourier transform. To remedy the problems with pairing collapse, we implemented the soft energy cutoff and pairing annealing. The convergence of HFB solutions was improved by a sub-iteration method. The Hermiticity violation of differential operators brought by Fourier-transform-based differentiation has also been solved. The accuracy and performance of HFBFFT were tested by benchmarking it against other HFB codes, both spherical and deformed, for a set of nuclei, both well-bound and weakly-bound.

## Key claim

The paper presents a new 3D coordinate-space Skyrme-HFB solver, HFBFFT, that solves the HFB equations directly in the canonical basis using fast Fourier transforms without imposing any self-consistent spatial symmetries, but the accuracy and performance are demonstrated only for a set of benchmark nuclei against other HFB codes, with the caveat that the method's advantage for weakly-bound nuclei is asserted rather than exhaustively proven.

## Method

The code HFBFFT extends the existing Skyrme-HF code Sky3D by implementing the HFB theory in the canonical basis, using fast Fourier transforms for derivatives and the Coulomb solver on a 3D Cartesian grid. To address pairing collapse, the authors implement a soft energy cutoff on the pairing-active space and a pairing annealing procedure, and they introduce a sub-iteration method in configuration space to stabilize and accelerate convergence. The Hermiticity violation of differential operators from Fourier-transform-based differentiation is resolved, and the code is benchmarked against the spherical HFB code Sky1D, the axial HFB code Sky2D, and the basis-expansion code HFBTHO.

## Key numbers

* The soft cutoff parameter for pairing-active space is set to \(\Delta\epsilon_\mathrm{cut}=15\) MeV above the Fermi energy.
* The damping parameters for the gradient iteration are \(x_0=0.45\), \(E_0=100\) MeV, and \(\tilde{h}_0=\mathrm{max}\left[\tilde{h}_\mathrm{n}(\vec{r}),\tilde{h}_\mathrm{p}(\vec{r})\right]\).
* The default linear mixing ratio for densities is \(\gamma=0.2\).
* The grid spacing is defined as \(\delta x\) with an even number of grid points \(N_x\) in each dimension, and the Fourier-space grid spacing is \(\delta k = 2\pi/(N_x\delta x)\).

## Context

This work extends the coordinate-space HFB solver family, which includes Sky1D (spherical), Sky2D (axial), and Sky3D (HF only), by adding full HFB capability in 3D without symmetry restrictions. It contrasts with basis-expansion methods like HFBTHO (Stoitsov, 2005) and HFODD (Dobaczewski, 1997), which struggle with weakly-bound nuclei due to the need for large configuration spaces. The canonical-basis HFB approach follows the method proposed by Reinhard (1997) and Tajima (2004), and the code enables future studies of exotic, weakly-bound nuclei where the quasiparticle continuum is important.
