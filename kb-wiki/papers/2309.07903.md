# Shell-model study for allowed and forbidden $β^-$ decay properties in the mass region "south" of $^{208}$Pb

> The large-scale shell-model calculations have been performed for the neutron-rich nuclei in the south region of $^{208}$Pb in the nuclear chart. The $β$-decay properties, such as the $\log ft$, average shape factor values, half-lives, and partial decay rates are calculated for these neutron-rich nuclei using recent effective interaction for the $^{208}$Pb region. These calculations have been performed without truncation in a particular model space for nuclei $N\leq 126$; additionally, particle-hole excitations are included in the case of core-breaking nuclei ($Z\leq 82, N>126$). An extensive comparison with the experimental data has been made, and spin parities of several states have been proposed.

## Key claim
Large-scale shell-model calculations, using a slightly modified effective interaction by Yuan et al. (2022), can reproduce experimental beta-decay properties (log ft, half-lives, shape factors) for neutron-rich nuclei in the south region of 208Pb, but only after applying empirically determined quenching factors separately for Gamow-Teller and each rank of first-forbidden operators, with different quenching sets needed for nuclei with N ≤ 126 versus those with N > 126.

## Method
The authors performed shell-model calculations using the KSHELL code with a model space of five proton orbitals (50 ≤ Z ≤ 82) and 13 neutron orbitals (82 ≤ N ≤ 184). For nuclei with N ≤ 126, calculations were done without truncation in the model space 50 ≤ Z ≤ 82, 82 ≤ N ≤ 126. For core-breaking nuclei with Z ≤ 82 and N > 126, particle-hole excitations were included by allowing up to one proton and one neutron excitation across the Z=82 and N=126 shell closures simultaneously. Quenching factors for Gamow-Teller and first-forbidden operators were determined by chi-squared fitting to experimental average shape factor values.

## Key numbers
- For case (i) (N ≤ 126): GT quenching factor q_A = 0.70; first-forbidden quenching factors q_w = 0.41, q_ξ'v = 1.265, q_x = 0.40, q_u = 0.43, q_z = 0.66
- For case (ii) (N > 126): GT quenching factor q_A = 0.70; first-forbidden quenching factors q_w = 0.57, q_ξ'v = 1.320, q_x = 0.54, q_u = 0.30, q_z = 0.54
- For 205Hg(1/2^-) → 205Tl(1/2_1^+): experimental log ft = 5.257(11), shell-model log ft = 5.299
- For 206Tl(0^-) → 206Pb(0_1^+): experimental log ft = 5.1775(13), shell-model log ft = 5.187
- For 204Tl(2^-) → 204Pb(0_1^+): experimental log ft = 9.7839(15), shell-model log ft = 9.791
- For 207Hg(9/2^+) → 207Tl((9/2_1^-)) at 2912.6 keV: experimental log ft = 6.3(2), shell-model log ft = 5.384

## Context
This work extends previous shell-model studies of beta decay in the lead region by Suzuki (2012), Zhi (2013), and Kumar (2021), who studied N=125 and 126 isotones, by systematically covering a wider range of neutron-rich nuclei south of 208Pb including both N ≤ 126 and N > 126 regions. The calculations use the recently modified effective interaction by Yuan et al. (2022), which was previously tested only for energy spectra and electromagnetic properties. The results enable predictions of beta-decay half-lives and spin-parity assignments for nuclei in the Terra Incognita region that are relevant for r-process nucleosynthesis around the A=195 peak, particularly for upcoming experiments at facilities such as those described by Watanabe (2015) and at CERN and RIKEN.
