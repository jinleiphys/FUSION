# Microscopic collectivity: The ridge and strangeness enhancement from string-string interactions

> We present the microscopic model for collective effects, built on string-string interactions (string shoving and rope hadronization), recently implemented in the PYTHIA 8 and DIPSY event generators. Rope hadronization is shown to give a good description of strangeness enhancement across pp, pA and AA collisions systems, while string shoving qualitatively describes the ridge observed in pp collisions.

## Key claim
String-string interactions (shoving and rope hadronization) implemented in PYTHIA 8 and DIPSY can qualitatively describe the ridge in pp collisions and strangeness enhancement across pp, pA, and AA systems, but the description is qualitative for the ridge and shows worse agreement for multi-strange baryons like Xi and Omega compared to kaons and Lambdas.

## Method
The model implements two stages of string-string interactions in the Lund string framework: string shoving, where overlapping strings generate transverse pressure via a classical force derived from a colour superconductor analogy with Gaussian field strength; and rope hadronization, where strings at high density hadronize as colour multiplets with higher effective string tension, increasing the production probability of heavier quarks. The shoving model is implemented in PYTHIA 8 with a simplistic impact-parameter model for multiparton interaction distributions, while rope hadronization is implemented in both PYTHIA 8 and DIPSY.

## Key numbers
- String tension kappa = 1 GeV/fm in the Lund string model
- Constituent quark masses: m_u approximately 0.3 GeV, m_s approximately 0.5 GeV, giving a naive strangeness suppression factor of 0.08
- String shoving operates at times tau approximately 0.6-2 fm/c, with maximal effect at tau approximately 1 fm/c
- Rope hadronization occurs at tau approximately 2 fm/c
- The force per unit length between two strings at distance d_perp is f(d_perp) = (g*kappa*d_perp/R^2)*exp(-d_perp^2/(4R^2)), where g is a strength parameter near unity and R specifies string width
- Strangeness suppression ratio rho = P_s/P_{u,d} = exp(-pi(m_s^2 - m_u^2)/kappa)

## Context
This work extends the Lund string model (Andersson 1983) and colour reconnection models (Ortiz 2013) by adding explicit string-string interactions to describe collective effects in small systems. The rope hadronization builds on the original proposal by Biro (1984) and was compared to ALICE measurements (2017) of strange particle ratios across e+e-, pp, pPb, and PbPb systems. The shoving model provides a microscopic explanation for the pp ridge observed by CMS (Khachatryan 2010). The Angantyr extension (Bierlich 2016, 2018) enables future application to pPb and PbPb collisions, while alternative approaches include QCD coherence effects (Blok 2017) and improved colour reconnection models (Christiansen 2015).
