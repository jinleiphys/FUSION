# Unified Coupled-Channels and Hauser-Feshbach Model Calculation for Nuclear Data Evaluation

> We present an overview of the coupled-channels optical model and the Hauser-Feshbach theory code CoH$_3$, which focuses on the nuclear reaction calculations in the keV to tens of MeV region with special attention to the nuclear deformation. The code consists of three major sections that undertake the one-body potential mean-field theory, the coupled-channels optical model, and the Hauser-Feshbach statistical decay. There are other complementary segments to perform the whole nuclear reaction calculations, such as the direct/semidirect radiative capture process, pre-equilibrium process, and prompt fission neutron emission.

# Unified Coupled-Channels and Hauser-Feshbach Model Calculation for Nuclear Data Evaluation

## Key claim
The CoH₃ code provides a unified framework combining coupled-channels optical model calculations with Hauser-Feshbach statistical theory, but the Engelbrecht-Weidenmüller transformation (EWT) for diagonalizing the S-matrix remains optional because it requires longer computational time when the number of coupled channels is large; when EWT is not activated, the generalized transmission coefficient approximation works well for targets that are not strongly deformed.

## Method
The CoH₃ code performs nuclear reaction calculations by first solving the one-body potential mean-field theory, then executing coupled-channels optical model calculations for deformed nuclei using rotational or vibrational models, and finally applying Hauser-Feshbach statistical decay with width fluctuation corrections. For strongly coupled channels, the Engelbrecht-Weidenmüller transformation diagonalizes the S-matrix into eigen-channel space where width fluctuation corrections are applied, then transformed back to physical space. The code also includes complementary models for pre-equilibrium processes (two-component exciton model), direct/semidirect radiative capture, and prompt fission neutron emission (Madland-Nix model).

## Key numbers
- The code consists of about 200 source files including 80 defined classes
- Calculations are performed in the keV to tens of MeV energy region
- Neutron-induced reactions on ⁵⁸Ni were calculated and compared with ENDF/B-VII.1 and JENDL-4.0 evaluated data
- Secondary particle angular distributions for neutron-induced reactions on ⁵⁸Ni were calculated at Eₙ = 3 MeV
- Elastic and inelastic scattering cross sections for ¹⁸²W were calculated comparing the EWT case (solid curves) versus generalized transmission coefficients (dashed curves), with a relatively large difference seen in the first excited state

## Context
This work extends the capabilities of statistical Hauser-Feshbach codes by implementing the Engelbrecht-Weidenmüller transformation (Engelbrecht 1973) to properly handle strongly coupled channels in deformed nuclei, which is especially important for actinide evaluations. The code CoH₃ is compared with other available HF codes such as EMPIRE (INDC0603), TALYS (Koning 2008), and CCONE (Iwamoto 2007); a recent code comparison (Capote 2017) suggested that CoH₃ inelastic scattering cross sections tend to be slightly higher than the other codes due to this difference in treatment. The code enables more accurate nuclear data evaluation for deformed nuclei by providing two methods to combine coupled-channels and Hauser-Feshbach theories, with the EWT method being more rigorous but computationally more expensive.
