# Infrared Dynamics of a Large N QCD Model, the Massless String Sector and Mesonic Spectra

> A consistency check for any UV complete model for large N QCD should be, among other things, the existence of a well-defined vector and scalar mesonic spectra. In this paper, we use our UV complete model in type IIB string theory to study the IR dynamics and use this to predict the mesonic spectra in the dual type IIA side. The advantage of this approach is two-fold: not only will this justify the consistency of the supergravity approach, but it will also give us a way to compare the IR spectra and the model with the ones proposed earlier by Sakai and Sugimoto. Interestingly, the spectra coming from the massless stringy sector are independent of the UV physics, although the massive string sector may pose certain subtleties regarding the UV contributions as well as the mappings to actual QCD. Additionally, we find that a component of the string landscape enters the picture: there are points in the landscape where the spectra can be considerably improved over the existing results in the literature. These points in the landscape in-turn also determine certain background supergravity components and fix various pathologies that eventually lead to a consistent low energy description of the theory.

## Key claim

The vector and scalar mesonic spectra derived from the massless open string sector of the UV complete type IIB model are independent of the UV physics, although the massive string sector may introduce subtleties regarding UV contributions and mappings to actual QCD; the spectra can be improved at certain points in the string landscape of flux vacua, which also fix background supergravity pathologies.

## Method

The authors use a UV complete type IIB string theory model with D3, D5, and D7 branes, whose gravitational dual is a warped resolved-deformed conifold geometry divided into three regions (IR, intermediate, UV). They perform a T-duality along the ψ direction to obtain a type IIA description, embed a probe D6-brane (representing Nf = 1 flavor) in this background, and compute the DBI action to derive the vector and scalar mesonic spectra from the massless open string sector, expanding in the small parameter δ ≡ gsM²/N up to first order.

## Key numbers

- The warp factor in Region 1 is h(r) = (27π gs N)/(4 r⁴) [1 + (3 gs M² log(r/𝒜^{2/3}))/(2π N)], where 𝒜^{2/3} is the minimal value of r.
- The free energy scales as F ∼ -T² for Tc < T < small T₀, F ∼ -T³ for Tc < T < intermediate T₀, F ∼ -T⁴[1 + (b₀₁ gsM²)/(4N) + (b₁ gsM²)/N log(T√(Nα'))] for T ≲ large T₀, and F ∼ -T⁴ for T > large T₀.
- The conformal anomaly Δ scales as 1/T² for Tc < T < small T₀, 1/T for Tc < T < intermediate T₀, (27π⁶ N α'⁴ V₅ b₁ gsM²)/(512 κ₁₀²) for T ≲ large T₀, and 0 for T > large T₀.
- The squashing parameter u appears in the internal metric via h₄ = 1 + u/sin²θ₂, with h₁ = 1, h₂ = h₃ = 1/6, and h₅ = u₀.
- The dilaton in type IIA is e^{-φ(r)} = h(r)^{1/4} r/(6 gs).

## Context

This work extends the Sakai-Sugimoto model (Sakai and Sugimoto, 2005) by using a UV complete type IIB model first proposed in FEP (2010) and its T-dual to predict mesonic spectra, enabling direct comparison with the Sakai-Sugimoto type IIA approach. It also builds on the Klebanov-Strassler model (Klebanov and Strassler, 2000) and earlier thermodynamics work (Dasgupta et al., recent papers) that showed the conformal anomaly scaling matches lattice simulations (Panero, 2009). The results enable future study of how the string landscape of flux vacua can improve holographic QCD predictions.
