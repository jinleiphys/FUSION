# Generation of random deviates for relativistic quantum-statistical distributions

> We provide an algorithm for generation of momenta (or energies) of relativistic particles according to the relativistic Bose-Einstein or Fermi-Dirac distributions. The algorithm uses rejection method with effectively selected comparison function so that the acceptance rate of the generated values is always better than 0.9. It might find its use in Monte-Carlo generators of particles from reactions in high-energy physics.

## Key claim

The paper presents an algorithm for generating random momenta according to relativistic Bose-Einstein or Fermi-Dirac distributions using a rejection method with a piecewise comparison function, claiming an acceptance rate always better than 0.9; however, this acceptance rate is demonstrated only for a specific set of test parameters and the algorithm requires numerical solution of several transcendental equations for each new set of temperature, chemical potential, and particle mass.

## Method

The algorithm uses a three-piece comparison function for the dimensionless momentum distribution \(S(x)\) (with energy pre-factor): a linear segment for small momenta, a constant segment at the mode value, and an exponential tail for large momenta. The joint points \(x_-\) and \(x_+\) are determined from the mode position \(x_m\), the left touch point \(x_l\), and the exponential touch point \(x_u\), all found by solving transcendental equations numerically. Random deviates are generated by sampling from the normalized comparison function via its inverse integral, then accepted or rejected using the ratio \(S(x)/S'(x)\).

## Key numbers

- Acceptance rate is "around 90%" for all tested parameter ranges, including bosons with \(A = 2/3\), \(M = 0\) (pions at \(k_B T = 207\) MeV) and bosons with \(A = 2\), \(M = 0.993\) (pions at \(k_B T = 69\) MeV, close to condensation).
- For fermions, acceptance rate is "around 90%" for \(A = 4.536\), \(M = 0\) (protons at \(k_B T = 207\) MeV) and \(A = 13.609\), \(M = 0.9989\) (protons at \(k_B T = 69\) MeV).
- The exponential touch point \(x_u\) is chosen where the distribution drops to \(1/e\) of its maximum value \(S_m\).
- For bosons, the second derivative of \(\ln S(x)\) may become positive only for \(A \le 3\) and \(M\) close to 1, but the crossing point \(x_c\) is always below the mode \(x_m\) (verified by scanning the parameter region).

## Context

This work extends the algorithm for relativistic Maxwellian distributions reported by Swisdak and Devroye to include quantum statistics (Bose-Einstein and Fermi-Dirac) with non-zero chemical potential. The algorithm has been implemented in the Monte Carlo event generator DRAGON for generating hadrons in high-energy nuclear collisions, and can be used in any application requiring relativistic momenta from quantum-statistical distributions.
