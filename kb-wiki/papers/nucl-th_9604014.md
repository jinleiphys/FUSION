# Pion and photon induced reactions on the nucleon in a unitary model

> We present a relativistic calculation of pion scattering, pion photoproduction and Compton scattering on the nucleon in the energy region of the \Del-resonance (upto 450 MeV photon lab energy), in a unified framework which obeys the unitarity constraint. It is found that the recent data on the cross section for nucleon Compton scattering determine accurately the parameters of the electromagnetic nucleon--\Del\ coupling. The calculated pion-photoproduction partial-wave amplitudes agree well with the recent Arndt analysis.

## Key claim

The paper presents a relativistic, unitary K-matrix calculation of pion scattering, pion photoproduction, and Compton scattering on the nucleon in the Delta-resonance region, finding that Compton scattering data accurately determine the parameters of the electromagnetic nucleon-Delta coupling, but only when the off-shell coupling parameters in the gamma N Delta vertex are also considered.

## Method

The authors employed a K-matrix approach in the combined pi N and gamma N space, constructing the K-matrix from tree-level diagrams with nucleon, Delta, and Roper resonances (with real self-energies) and t-channel meson exchanges (sigma, rho, omega, pi). The T-matrix was obtained via the unitarizing relation T = K/(1-iK), which generates the resonance widths dynamically from the on-shell iteration of the direct diagrams, and the partial-wave amplitudes were decomposed using the Jacob-Wick formalism.

## Key numbers

*   The calculation covers photon lab energies up to 450 MeV, in the region of the Delta-resonance.
*   Three parameter sets for pi N scattering are given: set #1 uses g_piNN = 12.95, chi = 0.0, g_piNDelta = 2.19, z_pi = -0.34, G_sigma = 23, g_rhoNN = 5.87, kappa_rho = 2.1; set #2 uses chi = 0.2, G_sigma = 43, g_rhoNN = 2.90; set #3 uses z_pi = -0.16, G_sigma = 28, g_rhoNN = 5.40.
*   The E2/M1 decay ratio for the Delta resonance varies from -4.47% (set #1) to -0.57% (set #3) depending on the gamma N Delta vertex parameters, while still fitting the Compton scattering data.
*   The unitarity constraint becomes important for the Compton cross section at photon energies exceeding 250 MeV.

## Context

This work extends the tree-level calculation of Pascalutsa (1995) by imposing exact unitarity via the K-matrix formalism, which was missing in that earlier approach. It also builds on the pi N scattering model of Goudsmit (1994) by including the Roper resonance and extending to higher energies. The results enable a more reliable extraction of the gamma N Delta vertex parameters from Compton scattering data, and show that Compton scattering is largely insensitive to the strong interaction channels, unlike pion photoproduction which is sensitive to the off-shell parameters in the pi N Delta vertex.
