# Strangeness Enhancement due to String Fluctuations

> We study string fragmentation in high multiplicity proton-proton collisions in a model where the string tension fluctuates. These fluctuations produce exponential pion spectra which are fitted to the transverse momentum distributions of charged particles for different multiplicities. For each multiplicity the so obtained hadronic slope parameter defines the magnitude of the string fluctuations which in turn determines the produced ratio of strange to light quarks. PYTHIA string decay simulations are used to convert each ratio of strange to light quarks to the appropriate ratio of strange hadrons to pions.

## Key claim

String tension fluctuations in high-multiplicity proton-proton collisions can explain the observed enhancement of strange hadron production, with the effect arising from a linear (rather than quadratic) mass dependence in the quark production exponent after averaging over fluctuations, and the magnitude of the enhancement is determined by fitting the charged-hadron transverse momentum spectra.

## Method

The authors model string fragmentation by letting the string tension fluctuate according to a Gaussian distribution, which converts the standard Schwinger formula's Gaussian quark transverse momentum spectrum into an exponential form. They fit this exponential spectrum to ATLAS charged-hadron transverse momentum data for four multiplicity classes in pp collisions at 8 TeV, extracting the mean string tension for each class. The resulting strange-to-light quark ratios from the fluctuating Schwinger model are then used as input parameters in PYTHIA 8.235 string decay simulations to compute the yields of strange hadrons relative to pions.

## Key numbers

* Mean string tension values extracted from fits: 0.21 GeV^2 (multiplicity 7.92), 0.22 GeV^2 (multiplicity 11.87), 0.25 GeV^2 (multiplicity 18.8), and 0.29 GeV^2 (multiplicity 31.7) for pp collisions at sqrt(s) = 8 TeV
* Resulting strange-to-light quark ratios s-bar(s)/u-bar(u): 0.217 (lowest multiplicity), 0.230, 0.257, and 0.292 (highest multiplicity)
* Light quark constituent mass set to 0.3 GeV; strange quark constituent mass fitted to 0.687 GeV to match PYTHIA default s-bar(s)/u-bar(u) = 0.217 at minimum bias
* Fit range for charged-hadron spectra: 0.5 < p_T < 1.4 GeV/c

## Context

This work extends the Schwinger model of string fragmentation (Schwinger 1951; Casher, Neuberger, and Nussinov 1978) by incorporating string tension fluctuations as suggested by Bialas (1999), providing an alternative to collective flow explanations for strangeness enhancement observed by ALICE (2017) in high-multiplicity pp collisions. The approach parallels other recent modifications of string dynamics (Fischer 2016; Bierlich 2014, 2015) and offers a mechanism for early thermalization without requiring partonic rescattering. The results enable further study of how string dynamics, including possible string merging and repulsion effects, can produce thermal-like hadron distributions in small collision systems.
