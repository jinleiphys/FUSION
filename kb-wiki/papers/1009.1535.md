# Matrix algorithms for solving (in)homogeneous bound state equations

> In the functional approach to quantum chromodynamics, the properties of hadronic bound states are accessible via covariant integral equations, e.g. the Bethe-Salpeter equations for mesons. In particular, one has to deal with linear, homogeneous integral equations which, in sophisticated model setups, use numerical representations of the solutions of other integral equations as part of their input. Analogously, inhomogeneous equations can be constructed to obtain off-shell information in addition to bound-state masses and other properties obtained from the covariant analogue to a wave function of the bound state. These can be solved very efficiently using well-known matrix algorithms for eigenvalues (in the homogeneous case) and the solution of linear systems (in the inhomogeneous case). We demonstrate this by solving the homogeneous and inhomogeneous Bethe-Salpeter equations and find, e.g. that for the calculation of the mass spectrum it is more efficient to use the inhomogeneous equation. This is valuable insight, in particular for the study of baryons in a three-quark setup and more involved systems.

## Key claim

The authors demonstrate that for calculating bound-state mass spectra from covariant integral equations, solving the inhomogeneous Bethe-Salpeter equation via matrix inversion is more efficient than solving the homogeneous eigenvalue equation, particularly when multiple eigenvalues are needed or when the system size grows (e.g., for three-quark baryon setups).

## Method

The authors discretize the homogeneous and inhomogeneous Bethe-Salpeter equations using the Nystrom (quadrature) method on the momentum variables and an orthonormal Dirac-covariant basis for the spin structure, converting the integral equations into matrix equations. They solve the homogeneous equation via eigenvalue algorithms (simple iteration and implicitly restarted Arnoldi factorization) and the inhomogeneous equation via direct iteration, exact matrix inversion, and the Bi-Conjugate Gradients stabilized (BiCGstab) algorithm, comparing the number of matrix-vector multiplications required for convergence.

## Key numbers

* The kernel matrix for the pseudoscalar-meson test case has dimensions (32, 24, 4) × (32, 24, 4), corresponding to N_q = 32 quadrature points for the radial momentum, N_z = 24 for the angular variable, and 4 Dirac covariants.
* The effective interaction parameters used are D = 0.93 GeV² and ω = 0.4 GeV, a common choice from Maris and Tandy (1999).
* The simple iteration for the homogeneous equation requires more matrix-vector multiplications per additional eigenvalue, while the Arnoldi factorization shows a more favorable scaling (demonstrated at a fixed P² = -M₀²).

## Context

This work extends the standard Dyson-Schwinger equation (DSE) approach to QCD bound states, where mesons are described by Bethe-Salpeter equations (Bethe 1951, Salpeter 1951) and baryons by covariant Faddeev equations (Faddeev 1960, Eichmann 2009). The matrix-algorithm comparison provides practical guidance for future studies of more complex systems, such as three-quark baryons, where the kernel matrix is too large to store in memory and must be recomputed on the fly, making the number of matrix-vector multiplications the critical cost factor.
