# Diffractive scattering at next-to-leading order in the dipole picture

> We discuss recent developments towards next-to-leading order (NLO) accuracy in the dipole picture. We review recent NLO results for exclusive vector meson production and diffractive structure functions, and discuss how it is becoming possible to describe both inclusive and exclusive DIS data simultaneously within the Color Glass Condensate framework at NLO accuracy. These developments will make it possible to probe the gluon saturation phenomena in detail especially when nuclear-DIS data from the EIC become available.

## Key claim

The Color Glass Condensate (CGC) framework is being extended to next-to-leading order (NLO) accuracy for diffractive processes, but the full NLO result for diffractive structure functions is not yet complete; the author reports that the finite \(q\bar q g\) contribution has been calculated, while loop corrections and gluon emission after the shockwave remain to be finalized in a separate publication.

## Method

The author reviews NLO calculations in the dipole picture for exclusive vector meson production (J/ψ and ρ) and for diffractive structure functions. For vector mesons, the NLO cross section includes the \(|q\bar q g\rangle\) Fock state and loop corrections to the photon and meson wave functions, with the meson wave function treated via perturbative corrections to the non-relativistic form or via a distribution amplitude at high \(Q^2\). For diffractive structure functions, the leading \(q\bar q g\) contribution is computed using the exact light-cone wave function for \(\gamma^* \to q\bar q g\) splitting, with the interaction described by a \(q\bar q g\)-target scattering amplitude expressed in terms of dipole amplitudes via the Fierz identity.

## Key numbers

- Coherent J/ψ production cross section as a function of center-of-mass energy \(W\): CGC results from Ref. [Mäntysaari:2022sux] describe available high-energy data, with no significant saturation-induced slowdown of the \(W\) dependence predicted.
- Differential coherent J/ψ production in photon-nucleus collisions: CGC calculations overestimate nuclear cross sections (underestimate suppression) compared to ALICE data, especially near midrapidity in LHC kinematics.
- For J/ψ production at NLO, results are shown at fixed \(W = 75\ \mathrm{GeV}\) (Fig. 3, Ref. [Mäntysaari:2022kdm]), with the first relativistic correction (\(v^2 \sim \alpha_s\)) included to obtain good description of HERA data at low \(Q^2\).
- For ρ production at NLO, results are shown at fixed \(W = 75\ \mathrm{GeV}\) (Fig. 4, Ref. [Mäntysaari:2022bsp]), with excellent description of H1 data in the high-\(Q^2\) region where the calculation is valid.

## Context

This work extends the leading-order dipole picture phenomenology that successfully described HERA structure function data (H1 Collaboration, 2015) and diffractive structure functions (Kowalski et al., 2008) using the Balitsky-Kovchegov evolution equation. The NLO results for vector meson production (Mäntysaari et al., 2022) and the partial NLO calculation for diffractive structure functions (Beuf et al., 2022) enable, for the first time, simultaneous description of inclusive and exclusive DIS data within the CGC framework at NLO accuracy. This will allow detailed probes of gluon saturation when nuclear-DIS data from the future Electron-Ion Collider become available.
