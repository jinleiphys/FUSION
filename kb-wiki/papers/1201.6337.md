# Spin and angular momentum in the nucleon

> Using the covariant spectator theory (CST), we present the results of a valence quark-diquark model calculation of the nucleon structure function f(x) measured in unpolarized deep inelastic scattering (DIS), and the structure functions g1(x) and g2(x) measured in DIS using polarized beams and targets. Parameters of the wave functions are adjusted to fit all the data. The fit fixes both the shape of the wave functions and the relative strength of each component. Two solutions are found that fit f(x) and g1(x), but only one of these gives a good description of g2(x). This fit requires the nucleon CST wave functions contain a large D-wave component (about 35%) and a small P-wave component (about 0.6%). The significance of these results is discussed.

## Key claim

Using the covariant spectator theory (CST) with a valence quark-diquark model, the authors find that fitting the unpolarized structure function \(f(x)\) and the polarized structure function \(g_1(x)\) yields two possible solutions, but only one of these also provides a good description of the transverse polarization structure function \(g_2(x)\); this successful fit requires the nucleon wave functions to contain a large D-wave component (about 35%) and a small P-wave component (about 0.6%).

## Method

The authors computed the deep inelastic scattering (DIS) structure functions \(f(x)\), \(g_1(x)\), and \(g_2(x)\) from the imaginary part of the forward handbag diagram within the covariant spectator theory (CST), using nucleon wave functions composed of S-state, P-state, and D-state quark-diquark components. The scalar parts of the wave functions were expressed as functions of the covariant variable \(\chi\), and the parameters of the wave functions (including the relative strengths \(n_P\) and \(n_D\) of the P and D components) were adjusted to fit experimental data for the proton unpolarized structure function \(f(x)\) and the polarized structure functions \(g_1(x)\) and \(g_2(x)\). The fits allowed the \(u\) and \(d\) quark distributions to differ, breaking isospin symmetry.

## Key numbers

* The successful fit requires the nucleon CST wave functions to contain a large D-wave component of about 35% and a small P-wave component of about 0.6%.
* The experimental value of the first moment of the proton polarized structure function is \(\Gamma_1^p = \int_0^1 dx\, g_{1p}^{\mathrm{exp}\pm}(x) = 0.128 \pm 0.013\).
* A naive S-state model gives a much larger value of 0.278 for \(\Gamma_1^p\) in this model, while Jaffe and Manohar (1989) gave 0.194.
* The momentum sum rule requires a gluon contribution \(N_g \simeq 0.5\) to the nucleon momentum.

## Context

This work extends the covariant spectator theory (CST) quark-diquark model, previously used to describe nucleon form factors and electromagnetic transitions (Gross 2006, Ramalho 2008-2011), to the calculation of DIS structure functions, directly addressing the proton spin crisis first reported by Ashman (1987, 1989). The authors note that the experimental value of \(\Gamma_1^p = 0.128\) is much smaller than the naive S-state expectation, and their fit shows that orbital angular momentum components (particularly a large D-wave) are essential to reproduce the data. This enables future work where the wave functions fixed by DIS will be used to compute low-\(Q^2\) pion cloud contributions and nucleon form factors, allowing the constituent quark form factors to be determined independently.
