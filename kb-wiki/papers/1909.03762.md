# Confinement of two-body systems and calculations in $d$ dimensions

> A continuous transition for a system moving in a three-dimensional (3D) space to moving in a lower-dimensional space, 2D or 1D, can be made by means of an external squeezing potential. A squeeze along one direction gives rise to a 3D to 2D transition, whereas a simultaneous squeeze along two directions produces a 3D to 1D transition, without going through an intermediate 2D configuration. In the same way, for a system moving in a 2D space, a squeezing potential along one direction produces a 2D to 1D transition. In this work we investigate the equivalence between this kind of confinement procedure and calculations without an external field, but where the dimension $d$ is taken as a parameter that changes continuously from $d=3$ to $d=1$. The practical case of an external harmonic oscillator squeezing potential acting on a two-body system is investigated in details. For the three transitions considered, 3D~$\rightarrow$~2D, 2D~$\rightarrow$~1D, and 3D~$\rightarrow$~1D, a universal connection between the harmonic oscillator parameter and the dimension $d$ is found. This relation is well established for infinitely large 3D scattering lengths of the two-body potential for 3D~$\rightarrow$~2D and 3D~$\rightarrow$~1D transitions, and for infinitely large 2D scattering length for the 2D~$\rightarrow$~1D case. For finite scattering lengths size corrections must be applied. The traditional wave functions for external squeezing potentials are shown to be uniquely related with the wave functions for specific non-integer dimension parameters, $d$.

## Key claim

The paper establishes a universal connection between the harmonic oscillator squeezing parameter and a continuously varying dimension parameter \(d\) for two-body systems confined from 3D to 2D, 2D to 1D, and 3D to 1D, but this connection is rigorously established only in the limit of infinitely large scattering lengths; for finite scattering lengths, size corrections must be applied.

## Method

The authors solve the two-body Schrodinger equation with an external harmonic oscillator squeezing potential acting along one or two spatial directions, using partial wave expansions in 3D and 2D and a Cartesian coordinate approach for the 2D to 1D case. Separately, they solve the two-body problem in \(d\) dimensions using the \(d\)-dimensional radial Schrodinger equation for \(s\)-waves, treating \(d\) as a continuous parameter. They then establish equivalence between the two methods by defining a scale parameter \(s\) that deforms the radial coordinate in the \(d\)-dimensional wave function to match the squeezed three-dimensional or two-dimensional wave function, and compute overlaps between the wave functions from both methods.

## Key numbers

* For the 3D to 2D transition, the harmonic oscillator ground state energy is \(E_{ho} = \hbar\omega/2\), and the trap potential is \(V_{trap}(r,\theta) = \frac{1}{2}\mu\omega^2 r^2 \cos^2\theta\).
* For the 3D to 1D transition with simultaneous squeezing along two directions, the harmonic oscillator energy is \(E_{ho} = \hbar\omega\) (double that of the 3D to 2D case).
* The \(d\)-dimensional centrifugal barrier for \(s\)-waves is \(\frac{1}{4}(d-3)(d-1)/r^2\), which can be written as \(\ell^*(\ell^*+1)/r^2\) with \(\ell^* = (d-3)/2\).
* The two short-distance behaviors of the radial solution are \(R_d^{(1)} \propto r^{(d-1)/2}\) (regular for all \(1 \leq d \leq 3\)) and \(R_d^{(2)} \propto r^{(3-d)/2}\) (regular only for \(d < 2\)).

## Context

This work extends the authors' previous investigation of 3D to 2D confinement (Garrido 2019) to include 2D to 1D and direct 3D to 1D transitions. It connects to prior work using harmonic oscillator traps for binary atomic collisions (Petrov 2001) and compactification methods in momentum space for three-body physics (Sanders 2018, Ross 2018, Yamashita 2015). The approach enables the numerically simpler \(d\)-dimensional method to be self-sufficient for describing confined systems, with future application to three-body systems and the Efimov effect.
