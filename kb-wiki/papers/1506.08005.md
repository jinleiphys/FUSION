# Improved parametrization of the unified model for alpha decay and alpha capture

> The updated data for the ground-state-to-ground-state alpha-transition half-lives in 401 nuclei and the alpha capture cross sections of 40Ca, 44Ca, 59Co, 208Pb and 209Bi are well described in the framework of the unified model for alpha-decay and alpha-capture. The updated values of the alpha decay half-lives, the binding energies of nuclei, the spins of parent and daughter nuclei, and the surface deformation parameters are used for the reevaluation of the model parameters. The data for the ground-state-to-ground-state alpha-decay half-lives are also well described by the empirical relationships.

## Key claim

The unified model for alpha-decay and alpha-capture (UMADAC) can describe both ground-state-to-ground-state alpha-transition half-lives in 401 nuclei and alpha capture cross sections for five target nuclei, but the model's parameters must be reevaluated using updated experimental data for half-lives, binding energies, spins, and deformation parameters to achieve this agreement.

## Method

The authors fit the parameters of the alpha-nucleus potential and the alpha-particle assault frequency within the UMADAC framework using a dataset of 401 ground-state-to-ground-state alpha transitions and alpha capture cross sections for 40Ca, 44Ca, 59Co, 208Pb, and 209Bi. The alpha-decay half-lives are calculated from the total width obtained by integrating the partial width over emission angles, using the WKB approximation for the transmission coefficient through a barrier composed of Coulomb, nuclear, and centrifugal potentials. The deformation parameters beta2 and beta4 for daughter nuclei are taken from the Hartree-Fock-Bogoliubov nuclear mass model HFB-27, and the Q-values are computed from the AME2012 atomic mass evaluation with electron binding energy corrections.

## Key numbers

- RMS error of the decimal logarithm of alpha-decay half-lives for the full dataset of 401 nuclei: delta_tot = 0.590 (SET I empirical relationship)
- RMS error for even-even nuclei (N=145): delta_e-e = 0.327 (SET I)
- RMS error for even-odd nuclei (N=107): delta_e-o = 0.683 (SET I)
- RMS error for odd-even nuclei (N=86): delta_o-e = 0.696 (SET I)
- RMS error for odd-odd nuclei (N=63): delta_o-o = 0.732 (SET I)
- RMS error for UMADAC model on full dataset: delta_tot = 0.753
- RMS error for heavy nuclei subset (N=164, A-Z>126 and Z>82) using SET III: delta_tot = 0.612
- RMS error for light nuclei subset (N=237) using SET II: delta_tot = 0.479
- Alpha-decay half-lives in the dataset span from approximately 10^-7 seconds to approximately 10^+27 seconds
- The Hartree-Fock-Bogoliubov mass model HFB-27 has a standard deviation of 0.500 MeV with respect to 2353 available mass data

## Context

This work extends the previous UMADAC analysis by Denisov and collaborators (2015), which used 344 ground-state-to-ground-state alpha transitions, by updating to 401 transitions using the NUDAT compilation and AME2012 atomic masses. The empirical relationship (SET I) achieves lower RMS errors than other empirical formulas such as those by Royer (2010), Royer (2011), and the universal decay law by Qi et al. (2009), particularly because it includes angular momentum and parity correction terms for even-odd, odd-even, and odd-odd nuclei. The improved parametrization enables more accurate predictions of alpha-decay half-lives for superheavy element studies and nuclei far from stability, where experimental data are scarce.
