# JETSCAPE v1.0 Quickstart Guide

> The JETSCAPE collaboration announced the first public release of its framework and Monte Carlo event generator at this conference, providing a unified interface and a comprehensive suite of model implementations for all stages of ultra-relativistic heavy ion collisions. This release focuses on validation of the framework and the $pp$ reference. A full manual is under development. In the mean-time, these proceedings will provide a guide for installation and simulation runs in lieu of the more traditional summary of the presentation.

## Key claim

The JETSCAPE collaboration released the first public version of its modular Monte Carlo event generator framework, but the heavy-ion portion is explicitly uncalibrated and requires Bayesian parameter tuning that is still in progress; the release focuses on validation of the framework and the proton-proton (pp) reference only.

## Method

The framework uses a task-based architecture with inter-task communication via the signal-slot paradigm and graph representation of the shower structure. For the validated pp reference, initial hard scattering is generated with PYTHIA8 (with multi-parton interaction and initial state radiation enabled), final state radiation is handled by the MATTER module with the transport coefficient qhat set to zero, and hadronization is performed by PYTHIA in either colored or colorless mode. The event generator is steered through a wrapper program and an XML configuration file, with physics modules attached using shared pointers and managed by dedicated energy loss and hadronization managers.

## Key numbers

* Charged hadron spectrum and R=0.4 anti-kT jet spectrum compared to pp data at center-of-mass energy sqrt(s)=2.76 TeV, showing agreement at the level of PYTHIA.
* The switching point between multiple energy loss modules (e.g., MATTER+MARTINI or MATTER+LBT) is set at 1.0 GeV/c^2 based on parton virtuality.
* The heavy-ion calibration process requires upwards of 30 million core hours.
* Example cross section from a pp run: sigmaGen = 8.35147e-06 with sigmaErr = 3.2343e-07 (units implicit from context, mb or similar).

## Context

This work extends the JETSCAPE collaboration's mission (formed in 2016) to create a complete, extensible, and modular event generator for ultra-relativistic heavy ion collisions, combining efforts of theorists, experimentalists, statisticians, and computer scientists. The release enables community testing and feedback on the uncalibrated heavy-ion modules while the collaboration performs Bayesian calibration of parameters. The framework uniquely supports multi-stage energy loss (Majumder 2010, Cao 2017) where distinct models like MATTER, MARTINI, LBT, and AdS/CFT-based modules are responsible only in their region of applicability, with upcoming releases planned to incorporate hadronic rescattering via SMASH (Weil 2016) and medium recoil.
