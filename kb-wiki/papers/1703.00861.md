# Parallel tempering algorithm for integration over Lefschetz thimbles

> The algorithm based on integration over Lefschetz thimbles is a promising method to resolve the sign problem for complex actions. However, this algorithm often meets a difficulty in actual Monte Carlo calculations because the configuration space is not easily explored due to the infinitely high potential barriers between different thimbles. In this paper, we propose to use the flow time of the antiholomorphic gradient flow as an auxiliary variable for the highly multimodal distribution. To illustrate this, we implement the parallel tempering method by taking the flow time as a tempering parameter. In this algorithm, we can take the maximum flow time to be sufficiently large such that the sign problem disappears there, and two separate modes are connected through configurations at small flow times. To exemplify that this algorithm does work, we investigate the (0+1)-dimensional massive Thirring model at finite density and show that our algorithm correctly reproduces the analytic results for large flow times such as T=2.

## Key claim

The paper proposes that using the flow time of the antiholomorphic gradient flow as a tempering parameter in a parallel tempering algorithm can simultaneously resolve both the sign problem and the multimodal sampling problem that plague Lefschetz thimble Monte Carlo calculations, provided the maximum flow time is taken sufficiently large to eliminate the sign problem there and intermediate flow times are sufficiently dense to allow configuration swaps.

## Method

The authors implement a parallel tempering algorithm where the flow time \(t\) of the antiholomorphic gradient flow serves as the tempering parameter. They prepare a set of replicas with flow times \(\{t_\alpha\}\) ranging from \(t_0=0\) to \(t_A=T\), where \(T\) is large enough that the sign problem disappears. For each replica, they evolve the original real variables \(x\) to complexified variables \(z_{t_\alpha}(x)\) via the flow equations and compute the effective action \(S_{\text{eff}}(x;t) = S_R(z_t(x)) - \ln|\det J_t(x)|\). The algorithm alternates between Metropolis updates within each replica and swap attempts between adjacent replicas, with the swap acceptance probability given by the ratio of the effective Boltzmann weights. The final expectation values are estimated using only configurations from the replica with the largest flow time \(T\).

## Key numbers

- The maximum flow time used in the numerical test is \(T=2\).
- The number of replicas is \(A+1 = 21\), with flow times \(t_\alpha = 0, 0.1, 0.2, \ldots, 2.0\) (equal separations of 0.1).
- The numerical test uses the \((0+1)\)-dimensional massive Thirring model with parameters \(N=8\) (lattice sites), \(m=1\), \(\mu=1.3\), and \(g^2=1/2\).
- The average acceptance rate for swaps between adjacent replicas is more than \(50\%\) for all pairs \((\alpha, \alpha+1)\).
- The chiral condensate computed with the parallel tempering algorithm at \(T=2\) agrees with the analytic result, while the result without parallel tempering at \(T=2\) shows statistically significant discrepancies.
- The absolute value of the denominator in the reweighting formula (divided by sample size) at \(T=2\) with parallel tempering is close to 1 for \(\mu \gtrsim 1.0\), indicating the sign problem is resolved, whereas at \(T=0\) it drops to near zero for \(\mu \gtrsim 1.0\).

## Context

This work extends the finite-flow-time Lefschetz thimble algorithm proposed by Alexandru et al. (2015), which used a single finite flow time to balance the sign problem and multimodality but required a priori knowledge of the optimal flow time. The parallel tempering approach removes this ambiguity by connecting configurations at large flow times (where the sign problem is absent but multimodality is severe) through configurations at small flow times (where multimodality is mild). The method is demonstrated on the \((0+1)\)-dimensional massive Thirring model at finite density, which was previously studied analytically by Pawlowski (2013, 2014) and numerically by Fujii (2015). The algorithm enables correct sampling from multiple Lefschetz thimbles, which is necessary for accurate results in models where more than one thimble contributes significantly.
