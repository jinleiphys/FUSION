# An alternative approach to the $σ$-meson-exchange in nucleon-nucleon interaction

> Through a quantitative comparative study of the properties of deuteron and nucleon-nucleon interaction with chiral quark model and quark delocalization color screening model. We show that the $σ$-meson exchange used in the chiral quark model can be replaced by quark delocalization and color screening mechanism.

## Key claim
The paper shows that the intermediate-range attraction in nucleon-nucleon interaction, traditionally attributed to σ-meson exchange in the chiral quark model, can be reproduced by the quark delocalization and color screening mechanism of the Quark Delocalization Color Screening Model (QDCSM), provided the same one-gluon-exchange and π-exchange contributions are used.

## Method
The authors compared the Salamanca chiral quark model (which includes σ-meson exchange) with the QDCSM (which replaces σ-meson exchange by quark delocalization and color screening) by calculating deuteron properties and NN scattering phase shifts using the resonating-group method (RGM). For a direct comparison, one parameter set (set 1) used identical values for baryon size b, quark-gluon coupling α_s, chiral coupling α_ch, quark mass m_u, pion mass m_π, and cutoff Λ, so that only the intermediate-range attraction mechanism differed. Two additional QDCSM parameter sets (set 2 and set 3) tested sensitivity to baryon size and gluon coupling.

## Key numbers
- Deuteron binding energy: Salamanca model gave 2.0 MeV; QDCSM gave 1.94 MeV (set 1), 2.01 MeV (set 2), 2.01 MeV (set 3).
- Deuteron root-mean-square radius: Salamanca model gave 1.96 fm; QDCSM gave 1.93 fm (set 1), 1.92 fm (set 2), 1.94 fm (set 3).
- Deuteron D-state probability: Salamanca model gave 4.86%; QDCSM gave 5.25% for all three parameter sets.
- Quark-gluon coupling constant α_s: 0.485 (Salamanca and QDCSM set 1), 0.724 (QDCSM set 2), 0.996 (QDCSM set 3).
- Baryon size parameter b: 0.518 fm (Salamanca and QDCSM set 1), 0.56 fm (QDCSM set 2), 0.60 fm (QDCSM set 3).
- Color screening parameter μ: 0.45 (QDCSM set 1), 0.57 (QDCSM set 2), 1.00 (QDCSM set 3).
- σ-meson mass in Salamanca model: 675 MeV.

## Context
This work extends the Quark Delocalization Color Screening Model (QDCSM) proposed in the 1990s, which had already been applied to NN and hyperon-nucleon scattering. It directly compares QDCSM with the Salamanca chiral quark model (Fernandez et al., 1993) to test whether σ-meson exchange can be replaced by quark delocalization and color screening. The results suggest that the σ-meson, whose existence as a fundamental particle remains controversial (Particle Data Group listings), may not be necessary for describing NN intermediate-range attraction, and the QDCSM provides a natural explanation for the similarity between molecular and nuclear forces.
