# Quadrupole properties of the eight $SU(3)$ algebras in $(sdgi)$ space

> With nucleons occupying an oscillator shell $η$, there are $2^{η/2}$ number of $SU(3)$ algebras; $η/2$ is the integer part of $η/2$. Analyzing the first non trivial situation with four $SU(3)$ algebras in $(sdg)$ space, demonstrated recently is that they generate quite different quadrupole properties though they all generate the same spectrum. More complex situation is with eight $SU(3)$ algebras in $(sdgi)$ space. In the present work, quadrupole properties generated by these eight algebras are analyzed first using the more analytically tractable interacting boson model. In addition, shell model and the closely related deformed shell model are used with three examples of nucleons in $sdgi$ space. It is found that in general six of the $SU(3)$ algebras generate prolate shape and two oblate shape. Out of all these, one of the $SU(3)$ algebra generates quite small quadrupole moments for the low-lying states.

# Quadrupole properties of the eight $SU(3)$ algebras in $(sdgi)$ space

## Key claim

The eight $SU(3)$ algebras in $(sdgi)$ space all generate identical rotational spectra, but they produce markedly different quadrupole properties: six generate prolate shapes, two generate oblate shapes, and among the prolate cases one algebra generates very small quadrupole moments for low-lying states. This pattern holds across both the interacting boson model and shell model calculations, with the caveat that the results are demonstrated for specific particle numbers and the quadrupole operator is fixed to the standard choice with all signs negative.

## Method

The authors first analyze the eight $SU(3)$ algebras analytically using the interacting boson model (IBM) in $(sdgi)$ space, deriving large-N limit formulas for quadrupole moments and B(E2) values along the ground K=0 band. They then perform shell model calculations using the Antoine code for the $(sdgi)^{6p}$ system and deformed shell model (DSM) calculations based on Hartree-Fock single-particle states for the $(sdgi)^{6p,6n}$ and $(sdgi)^{12p,6n}$ systems, using the eight different quadrupole-quadrupole Hamiltonians $H_Q^{(\alpha_{02},\alpha_{24},\alpha_{46})} = -(1/4) Q^2(\alpha_{02},\alpha_{24},\alpha_{46}) \cdot Q^2(\alpha_{02},\alpha_{24},\alpha_{46})$.

## Key numbers

- For a 15-boson system in $sdgi$IBM, $Q(2)$ values range from $-52$ (in units of $q_2$) for $(+,+,+)$ to $+22$ for $(-,-,-)$, with $(+,-,-)$ giving $-10$ (the smallest magnitude among prolate cases)
- For the $(sdgi)^{6p}$ shell model system, $Q(2^+_1)$ values (in $e\,fm^2$) are $-84$, $-51$, $-19$, $14$, $30$, $-3$, $-35$, and $-68$ for the eight algebras respectively
- For the $(sdgi)^{6p,6n}$ system, $B(E2; 2^+_1 \rightarrow 0^+_1)$ values (in $e^2\,fm^4$) range from $6482$ for $(-,-,-)$ down to $14$ for $(+,+,-)$
- For the $(sdgi)^{12p,6n}$ system, $Q(2)$ values (in $e\,fm^2$) are $-234$, $-92$, $-120$, $22$, $76$, $-66$, $-38$, and $-180$ for the eight algebras respectively
- The intrinsic quadrupole moments (in units of $b^2$) for the $(sdgi)^{6p,6n}$ system are $120$, $70$, $34$, $-16$, $-43$, $8$, $43$, and $94$ for the eight algebras

## Context

This work extends the earlier analysis of four $SU(3)$ algebras in $(sdg)$ space by the same group (Ko-19), which first demonstrated that multiple $SU(3)$ algebras generate different quadrupole properties despite identical spectra. The $(sdgi)$ case with eight algebras represents the next nontrivial complexity level in the hierarchy of multiple $SU(3)$ algebras in oscillator shells, following the well-known two-algebra case in $(sd)$ space studied by Parikh and Iachello. The results establish that the pattern observed in $(sdg)$ space is generic, and they enable future studies of quantum phase transitions using these multiple $SU(3)$ algebras, as well as investigations of $\gamma$ and $\beta$ bands and extensions to IBM-2 spaces with separate proton and neutron $SU(3)$ algebras.
