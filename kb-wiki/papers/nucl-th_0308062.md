# Wavelets in Momentum-Space Scattering Calculations

> We demonstrate that wavelet bases have features that make them advantageous for solving momentum-space scattering integral equations. Using the example of two nucleons interacting with the Malfliet-Tjon V interaction, we show it is possible to reduce the size of the matrix representation of the Lippmann-Scwhinger equation by 96% with a loss of accuracy of only a few parts in a million. We also demonstrate that wavelet methods provide an accurate means for treating the singularities that appear in the scattering integral equations.

## Key claim
Wavelet bases can reduce the matrix size of the Lippmann-Schwinger equation by 96% while losing only a few parts per million in accuracy, but this result is demonstrated only for the specific case of two nucleons interacting with the Malfliet-Tjon V potential.

## Method
The authors represented the kernel of the momentum-space K-matrix equation for nucleon-nucleon scattering in a Daubechies wavelet basis of order three. They then set the smallest 96% of the matrix elements to zero, exploiting the sparsity that arises because the wavelet representation expresses the kernel as the sum of a sparse matrix and a matrix with small norm. Singular integrals appearing in the scattering equations were handled using wavelet-based quadrature rules derived from the renormalization group equation.

## Key numbers
- Matrix representation size reduced by 96% (the smallest 96% of matrix elements in the kernel were set to zero).
- Mean square error after truncation was only a few parts in a million (no exact number given beyond "a few parts in a million").
- The test system was s-wave nucleon-nucleon scattering with the Malfliet-Tjon V potential.

## Context
This work extends the use of Daubechies wavelets from signal processing into numerical solutions of momentum-space scattering integral equations, which are essential for relativistic few-body calculations at TJNAF. Prior momentum-space approaches suffered from large dense matrices and singular integrals; the wavelet method overcomes both issues by producing sparse representations and robust quadrature. The results enable future application to realistic few-body scattering calculations where momentum-dependent rotations and nonlinear mass-energy relations are required.
