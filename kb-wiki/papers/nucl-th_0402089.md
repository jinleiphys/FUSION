# Interrelationship of Isospin and Angular Momentum

> It is noted that the simple interaction in isospin variables $a (1/4 - t(i)\cdot t(j))$, in a single $j$ shell calculation, can also be written with angular momentum variables. For the configuration $(j^2) J_A$ for even $J_A$ the isospin is one; for odd $J_A$ it is zero. Hence the above interaction can also be written as $a (1 - (-1)^{J_A})/2$. For the I=0 state of an even-even Ti isotope with $n$ neutrons, the hamiltonian matrix element of this interaction is $\bra [J'J']_0 |H| [JJ]_0\ket/a = (n+1) δ_{JJ'} - (n+1) (j^n Jj|\} j^{n+1} j) (j^n J'j|\} j^{n+1} j)$. The eigenvalues of this interaction can be found by using the isospin form of the interaction. They are $(n+1)a$ for $T = |N-Z|/2$ and zero for $T = |N-Z|/2 + 2$. One can apply this to some extent to obtain the number of pairs of nucleons with given total angular momentum $J_A$ in a given Ti isotope.

## Key claim

The paper demonstrates that a simple two-body interaction expressed in isospin variables, \(a (1/4 - t(i)\cdot t(j))\), can be rewritten in terms of angular momentum variables as \(a (1 - (-1)^{J_A})/2\) for a single \(j\) shell, and that this equivalence allows one to derive closed-form expressions for the number of neutron-proton pairs with given total angular momentum in even-even Ti isotopes, but only for the specific case where the interaction is zero for \(T=1\) states and nonzero for \(T=0\) states.

## Method

The authors perform a single \(j\) shell calculation for the \(f_{7/2}\) orbital, considering Ti isotopes with two protons and \(n\) neutrons (\(n = 2, 4, 6\) for \(^{44}\text{Ti}\), \(^{46}\text{Ti}\), \(^{48}\text{Ti}\)). They evaluate the Hamiltonian matrix for \(I=0\) states using both isospin variables and angular momentum variables, employing coefficients of fractional parentage (cfp) and unitary nine-\(j\) symbols. The eigenvalues are obtained from the isospin form, and the angular momentum form yields a separable interaction whose eigenfunctions are expressed in terms of one-particle cfp's.

## Key numbers

* For \(^{44}\text{Ti}\) (\(n=2\)), the lowest isospin state (\(T = |N-Z|/2\)) has eigenvalue \(3a\).
* For \(^{46}\text{Ti}\) (\(n=4\)), the lowest isospin state has eigenvalue \(5a\).
* For \(^{48}\text{Ti}\) (\(n=6\)), the lowest isospin state has eigenvalue \(7a\).
* For \(T = T_{\text{min}} = |N-Z|/2\), the number of even \(n\)-\(p\) pairs is \(n-1\) (e.g., 1 for \(^{44}\text{Ti}\), 3 for \(^{46}\text{Ti}\), 5 for \(^{48}\text{Ti}\)).
* For \(T = T_{\text{min}} + 2\), the number of even \(n\)-\(p\) pairs is \(2n\) (e.g., 4 for \(^{44}\text{Ti}\), 8 for \(^{46}\text{Ti}\), 12 for \(^{48}\text{Ti}\)).
* For \(T = T_{\text{min}} + 2\), the number of odd \(n\)-\(p\) pairs is 0.
* For \(^{44}\text{Ti}\), the number of \(J_A=0\) \(n\)-\(p\) pairs for \(T=T_{\text{min}}\) is 1; for \(^{46}\text{Ti}\) it is 4/5; for \(^{48}\text{Ti}\) it is 3/7.

## Context

This work extends the analysis of Moya de Guerra et al. (previous work) on neutron-proton pair counting in the single \(j\) shell, and contrasts with the more restricted results of Engel et al. (previous work) who considered only isovector pairing interactions and states with \(T = T_{\text{min}}\). The paper provides general expressions for the number of pairs with given angular momentum that hold for any interaction, enabling future studies of two-nucleon transfer reactions to test the relative importance of \(J=0\) \(T=1\) pairing versus \(J=1\) \(T=0\) pairing.
