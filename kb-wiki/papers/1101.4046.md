# Spin-aligned neutron-proton pair mode in atomic nuclei

> Shell model calculations using realistic interactions reveal that the ground and low-lying yrast states of the $N=Z$ nucleus $^{92}_{46}$Pd are mainly built upon isoscalar neutron-proton pairs each carrying the maximum angular momentum J=9 allowed by the shell $0g_{9/2}$ which is dominant in this nuclear region. This structure is different from the ones found in the ground and low-lying yrast states of all other even-even nuclei studied so far. The low-lying spectrum of excited states generated by such correlated neutron-proton pairs has two distinctive features: i) the levels are almost equidistant at low energies and ii) the transition probability $I\rightarrow I-2$ is approximately constant and strongly selective. This unique mode is shown to replace normal isovector pairing as the dominating coupling scheme in $N=Z$ nuclei approaching the doubly-magic nucleus $^{100}$Sn.

## Key claim

Shell model calculations using realistic interactions reveal that the ground and low-lying yrast states of the N=Z nucleus 92Pd are mainly built upon isoscalar neutron-proton pairs each carrying the maximum angular momentum J=9 allowed by the 0g9/2 shell, but the authors note that the microscopic origin of the equidistant pattern generated by the aligned np pairs remains unclear at present.

## Method

The authors performed nuclear shell model calculations within the 1p3/2 0f5/2 1p1/2 0g9/2 (fpg) model space using the Hamiltonian from Honma (2009), and also within the restricted 0g9/2 and 1p1/2 (pg) space with interaction matrix elements from Johnson (2001). They evaluated the pair content in the many-fermion wave functions by calculating the average number of pairs using two-particle coefficients of fractional parentage (cfp) in the isospin representation, and examined how the spectrum changes as a function of a controlling parameter δ that scales the J=9 np pair interaction matrix element.

## Key numbers

- The ground state wave function of 92Pd has 66% occupancy of the configuration ((((νπ)9⊗(νπ)9)I'=16⊗(νπ)9)I''=9⊗(νπ)9)I, with amplitude X(0+1)=0.81
- The first 2+ state has amplitude X(2+1)=0.89 for the same configuration
- For the 0f7/2, 0g9/2 and 0h11/2 shells, the normal isovector pairing coupling scheme (ν2)0⊗(π2)0 accounts for only X2Jp=Jn=0=0.51-0.62 of the ground state wave functions
- The spin-aligned np coupling scheme accounts for X2J1=J2=2j=0.92-0.95 of the ground state wave functions
- The matrix element for the Jπ=9+ np pair coupling is V9=⟨(g9/22)J=9|V|(g9/22)J=9⟩∼-2 MeV
- For 94Ag, the 7+ and 8+ T=0 states are dominated by the aligned np pair coupling with |X|≈0.97 in restricted 0g9/2 shell calculations, decreasing to |X|=0.83 for the first 9+ state

## Context

This work extends the experimental findings of Cederwall (2010) who first measured the low-lying yrast states in 92Pd and found it necessary to assume an isoscalar spin-aligned np coupling scheme. The results contradict the long-inferred dominance of J=0 isovector pairing correlations (Bohr 1975, Dean 2003) in even-even nuclei, and show that the spin-aligned np pair mode replaces normal isovector pairing as the dominating coupling scheme in N=Z nuclei approaching the doubly-magic nucleus 100Sn. The work enables future measurements of E2 transition rates to confirm this interpretation, and suggests that the spin-aligned np pair must be considered as an essential building block in nuclear structure calculations.
