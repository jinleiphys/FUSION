# A consistent formalism for the Thomas-Ehrman Level Displacement

> Usage of the Thomas-Ehrman Level Displacement formalism has been examined. Mistakes and inconsistencies are found in several papers, being repeated in subsequent works. Here, we present a complete formalism with a consistent set of definitions. Full algorithms are made available, both as a {\tt FORTRAN} source file and as a user-friendly Visual Basic executable tool, available for download on the World Wide Web.

# A consistent formalism for the Thomas-Ehrman Level Displacement

## Key claim
The paper presents a corrected and consistent Thomas-Ehrman Level Displacement (TELD) formalism, identifying that previous works used inconsistent definitions of critical parameters (partial width, reduced width, dimensionless reduced width, and dimensionless single-particle reduced width) and that several papers incorrectly wrote the level displacement as E*(n)-E*(p)=E_b-E_r instead of the correct expression E*(n)-E*(p)=E'_b-E_r, though the correct expression was used in the calculations of at least one prior work (Gorres 2006).

## Method
The authors re-derived the TELD formalism following Thomas (1952) with consistent terminology, adopting the definitions of French (1960) for reduced widths and dimensionless reduced widths. They compared exact evaluations of Coulomb and Whittaker functions using modern computer codes against the WKB approximation, and identified that the subroutine COULFG could not reproduce results from RCWFN near eta=1.70x+5.13 unless the ACCUR variable was set below 10^{-17}. The formalism was implemented as a FORTRAN source file and a Visual Basic executable tool available for download.

## Key numbers
- Channel radius defined as a_c = 1.44 x (A_{1c}^{1/3} + A_{2c}^{1/3}) fm
- Difference between WKB approximation and exact evaluation of g_{c-} in region 0.1 < x < 5.0, 0.1 < eta < 10.0 can be as large as 3% for l=1, 1% for l=2, and 0.6% for l=3
- For l=0, differences between WKB and exact evaluation can be as much as 10% when x < 0.1 and eta < 0.9, or x < 0.2 and eta < 0.5, or x < 0.5 and eta < 0.3
- The subroutine COULFG required ACCUR variable set to less than 10^{-17} to reproduce results from RCWFN near eta=1.70x+5.13
- The definition of gamma_{lambda c}^2 from French (1960) differs from that of Thomas (1952) by a factor of 3/a_c

## Context
This work extends the original Thomas-Ehrman formalism from Thomas (1952) and Ehrman (1951) by correcting inconsistencies found in applications to mirror pairs including 21Ne-21Na (Marion 1957), 20F-20Na (Lang 1986), 18O-18Ne (Wiescher 1988), 22Ne-22Mg (Ruiz 2003), and 46Ti-46Cr (Horiuchi 2002, He 2007). The paper provides a standardized computational tool that enables reliable calculation of level displacements between mirror nuclei, which is particularly important for determining proton resonance properties in proton-rich nuclei relevant to astrophysical reaction rates. The formalism enables either prediction of unknown mirror state energies from known spectroscopic factors, or extraction of spectroscopic factors from measured mirror level displacements.
