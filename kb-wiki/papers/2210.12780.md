# Independence of current components, polarization vectors, and reference frames in the light-front quark model analysis of meson decay constants

> The issue of resulting in the same physical observables with different current components, in particular from the minus current, has been challenging in the light-front quark model (LFQM) even for the computation of the two-point functions such as meson decay constants. At the level of one-body current matrix element computation, we show the uniqueness of pseudoscalar and vector meson decay constants using all available components including the minus component of the current in the LFQM consistent with the Bakamjian-Thomas construction. Regardless of the current components, the polarization vectors, and the reference frames, the meson decay constants are uniquely determined in the non-interacting constituent quark and antiquark basis while the interactions of the constituents are added to the meson mass operator in the LFQM.

## Key claim

The authors demonstrate that pseudoscalar and vector meson decay constants computed in the light-front quark model (LFQM) are uniquely determined regardless of which current component (including the minus component), polarization vector, or reference frame is used, provided the self-consistency condition \(P = p_1 + p_2\) (or equivalently \(M \to M_0\)) is imposed at the meson-quark vertex, consistent with the Bakamjian-Thomas construction for one-body current matrix elements.

## Method

The authors compute meson decay constants from one-body current matrix elements in the standard LFQM, using the non-interacting constituent quark and antiquark basis with interactions added only to the mass operator. They establish a link between the manifestly covariant Bethe-Salpeter (BS) model and the LFQM via the "Type II" substitution: \(\sqrt{2N_c}\chi(x, \mathbf{k}_\perp)/(1-x) \to \phi(x,\mathbf{k}_\perp)/\sqrt{m^2 + \mathbf{k}_\perp^2}\) and \(M \to M_0\), and verify that this substitution yields identical results for all combinations of current components (\(\gamma^\mu\), \(\sigma^{\mu\nu}\)) and polarization vectors (\(\epsilon^\mu(J_z)\)) for both pseudoscalar and vector mesons.

## Key numbers

* Pseudoscalar meson decay constant \(f_\pi = 131\) MeV, obtained with model parameters \((m, \beta) = (0.25, 0.3194)\) GeV, in agreement with experimental value \(f_\pi^{\text{Expt.}} = 130.3 \pm 0.3\) MeV.
* Vector meson decay constant \(f_\rho = 215\) MeV, in agreement with experimental value \(f_\rho^{\text{Expt.}} = 210 \pm 4\) MeV.
* Transverse vector meson decay constant \(f_\rho^T = 173\) MeV.
* The analytic forms satisfy the SU(6) symmetry relation \(f_{\rm P} + f_{\rm V}(J_z) = 2 f^T_{\rm V}(J_z)\) for each polarization vector \(J_z\).
* The difference operator \(\tilde{\mathcal{O}}^{(T)}_{\rm V} = \mathcal{O}^{(T)}_{\rm V}(J_z=1) - \mathcal{O}^{(T)}_{\rm V}(J_z=0) = \frac{2}{\mathcal{D}_0}(\mathbf{k}_\perp^2 - 2 k_z^2)\) integrates to zero, proving \(f^{(T)}_{V}(J_z=1) = f^{(T)}_{V}(J_z=0)\).

## Context

This work resolves a long-standing issue in LFQM regarding the apparent dependence of physical observables on the choice of current component, particularly the minus current, which had been discussed in the context of Fock space truncation and zero-mode contributions by Jaus (1999), Choi et al. (2003), and others. It extends the previous demonstration by Choi and Ji (2013) for specific current-polarization combinations to all possible combinations, and confirms that the self-consistency condition \(M \to M_0\) is equivalent to the Bakamjian-Thomas construction for two-body systems. This enables reliable LFQM calculations of meson decay constants without ambiguity from current component choice, and provides a foundation for extending such analyses to form factors and other hadronic observables.
