# SPARKX: A Software Package for Analyzing Relativistic Kinematics in Collision Experiments

> SPARKX is an open-source Python package developed to analyze simulation data from heavy-ion collision experiments. By offering a comprehensive suite of tools, SPARKX simplifies data analysis workflows, supports multiple formats such as OSCAR2013, and integrates seamlessly with SMASH and JETSCAPE/X-SCAPE. This paper describes SPARKX's architecture, features, and applications and demonstrates its effectiveness through detailed examples and performance benchmarks. SPARKX enhances productivity and precision in relativistic kinematics studies.

**SPARKX: A Software Package for Analyzing Relativistic Kinematics in Collision Experiments**

## Key claim
SPARKX provides a modular, rigorously tested Python framework for analyzing event generator outputs from heavy-ion collisions, but its performance benchmarks show longer execution times and higher memory usage than the C++ framework Rivet, reflecting a design trade-off prioritizing usability and flexibility over raw speed.

## Method
The package loads simulation output from SMASH and JETSCAPE/X-SCAPE using dedicated Loader and Storer classes, applies user-defined particle and event filters, and then performs analyses through separate class modules for bulk observables, anisotropic flow (using six different methods), jet finding (via a FastJet wrapper), and event characterization. The core Particle class wraps a NumPy array to minimize memory overhead while providing access to particle properties enriched with PDG data.

## Key numbers
* Test coverage stands at 90% across the codebase, enforced through automated pytest workflows.
* The package computes multi-particle transverse momentum correlations and cumulants up to the 8th order.
* The example flow analysis uses centrality bins of [0, 20, 40, 60, 100] percent for Pb-Pb collisions at sqrt(s_NN) = 17.3 GeV.
* The jet analysis example applies a transverse momentum cut of 0.1 GeV for charged particles and uses a jet radius R = 0.6 with pseudorapidity range (-2.0, 2.0) and jet pT range (10.0, infinity) GeV.

## Context
SPARKX addresses the gap between experimental frameworks like ROOT (Brun and Rademakers, 1997) and Rivet (Bierlich et al., 2020), which are C++-based and designed for experimental workflows, and the ad-hoc Python scripts often used by theorists and phenomenologists. The package has already been employed in published studies including centrality classification, particle spectra, eccentricities, anisotropic flow, and jet observables (Gotz et al., 2023; Gotz et al., 2025; Borghini et al., 2024; Andronic et al., 2025; Roch et al., 2025). Future development will focus on optimizing data loading strategies and improving parallel processing to narrow the performance gap with C++ frameworks.
