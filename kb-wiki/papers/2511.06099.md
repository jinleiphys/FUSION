# Universal modified function in conjunction with the short range correlation effect to extract the nuclear $xF_3^A$ structure function

> Parton distribution functions (PDFs) are comprehensive and not reliant on the process. They are affected by nuclear matter during nuclear scattering process. As a recent approach, in order to study the nuclear PDFs (nPDFs), the nucleon pair PDFs are utilized to describe parton distributions in the nucleon pair which are confined to a nucleus. Nucleon pair PDFs stem from nucleon-nucleon correlation which is called short range correlation (SRC) and are proportional to common nucleon PDFs. In this regard a modified universal function is constructed which provides a test for SRC in (neutrino)-nucleus scattering. In fact we can show that the modification of the structure function of nucleons bound in atomic nuclei (known as the EMC effect) are consistently accounted for within the frame work of a universal modification of nucleons in SRC pairs. In this article, based on the strategy which was introduced in Ref.\cite{nature}, we are investigating to find the universality behaviour for the ratio of nonsinglet $xF_3^A$ nuclear structure function. The numerical calculations performed within the CTEQ framework confirm the universality feature of the concerned ratio, as has been established for the ratio of the structure function $F^A_2$ in Ref. \cite{nature}. Following that we reformulate the $R_{EMC}$ nuclear weight function which relates the free and bound structure function in terms of modified universal function. It makes us a possibility to achieve $R_{EMC}$ and finally $xF_3^A$ nuclear structure function for each nucleus only by considering the specified feature of nucleus.The results are compared with the nuclear $xF_3$ structure function which are computing from bound PDFs, based on some parameterizations models. The findings, considering the SRC effect, demonstrating qualitative agreement with nCTEQ15 and EPPS21 parametrization models and also the available experimental data.

# Universal modified function in conjunction with the short range correlation effect to extract the nuclear $xF_3^A$ structure function

## Key claim

The ratio of the nonsinglet nuclear structure function $xF_3^A$ to the deuteron structure function, when modified by the short range correlation (SRC) scale factor and atomic number, exhibits a universal behavior across different nuclei in the valence quark dominated region $0.3 < x < 0.7$, consistent with the universality previously established for the $F_2^A$ structure function. This universality allows the construction of a modified weight function $R_{EMC}$ that can reproduce the nuclear $xF_3^A$ structure function for any nucleus using only the atomic number $Z$, mass number $A$, and SRC scale factor $a_2$, with results showing qualitative agreement with nCTEQ15 and EPPS21 parametrization models and available experimental data.

## Method

The authors computed the ratio $xF_3^A / xF_3^d$ for multiple nuclei (Li, He, F, C, Al, Fe, Cu, Pb, U) at $Q^2 = 10$ GeV$^2$ using nCTEQ15 nuclear PDFs via the ManeParse framework. They then constructed a modified universal function $R_f^u$ from the right-hand side of Eq. (13), which expresses the SRC pair modification in terms of the structure function ratios, atomic numbers, and SRC scale factors $a_2$. This universal function was fitted to a functional form $R_f^u(x) = a + b x + c e^{d(1-x)}$, and then used to derive the EMC weight function $R_{EMC}$ via Eq. (18), which relates the free proton structure function $xF_3^p$ to the nuclear structure function $xF_3^A$ for any nucleus.

## Key numbers

- The SRC scale factors used are $a_2^{Cr} = 4.49$, $a_2^{Fe} = 4.80$, and $a_2^{Pb} = 4.80$, quoted from CLAS data in Ref. [nature].
- The fitted parameters for the universal modified function $R_f^u(x)$ are $a = 3.60462$, $b = -0.908723$, $c = -2.82168$, $d = 0.242195$.
- The slope of the modified universal function as a function of mass number $A$ is given by $S_f^u(A) = a_u + b_u A$ with $a_u = 0.0695199$ and $b_u = 0.000178467$.
- The analysis includes experimental data from CCFR (116 initial points, 67 after kinematic cuts with $Q^2 \geq 5$ GeV$^2$ and $W^2 \geq 12.5$ GeV$^2$), NuTeV (75 original points, 59 passing cuts), CHORUS (67 points, 41 retained), and CDHSW (143 points, 96 used after cuts).
- The kinematic range covers $x = 0.0075$ to $0.75$ and $Q^2 = 0.19$ to $196.3$ GeV$^2$ across the experimental datasets.

## Context

This work extends the SRC-based universality approach previously applied to the $F_2^A$ structure function in Ref. [nature] (Weinstein et al., 2011) to the nonsinglet $xF_3^A$ structure function in neutrino-nucleus deep inelastic scattering. The linear relation between the EMC effect magnitude and the SRC scale factor, proposed by Weinstein et al. (2011) and investigated by Hen et al. (2014) and Duer et al. (2018), is here exploited to construct a universal modified function that enables prediction of nuclear structure functions for any nucleus. The results are compared with the nCTEQ15 parametrization (Kovarik et al., 2016) and EPPS21 parametrization, and the approach enables testing of QCD symmetry breaking mechanisms and distinguishing nuclear physics effects from beyond-standard-model effects in neutrino scattering experiments.
