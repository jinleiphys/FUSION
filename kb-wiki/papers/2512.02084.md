# Modifications in clusterization procedures for heavy-ion collisions: minimum spanning tree, simulated annealing, coalescence

> A new open-source cluster finding library ''Common Clusterization Library'' (CCL) is proposed to describe the clusters production when applied to the transport codes. The new library was applied to the Parton-Hadron-Quantum-Molecular Dynamics approach to be comparable with the established and well known MST and SACA implementations on the exactly same set of the events. The presented procedures were tested with rapidity density distributions and complex observables like $\langle M_{imf} \rangle$ and $\langle Z_{\text{max}} \rangle$ vs $Z_{bound}2$ at the energies of ALADiN and NA49 experiments. An improvement to the simulated annealing chain is proposed to reduce the computational time. The new stable clusters tracking algorithm for the ''lost'' cluster recovering is presented. The results of the ''box-like'' coalescence procedure and a mixed method which incorporates the coalescence algorithm steps and the negative binding energy criteria are presented. The difference between the Helium-3 found by the MST-based cluster finding procedure and the coalescence algorithm applied on the exactly same set of events and baryons is shown for the first time.

## Key claim

The paper presents a new open-source cluster finding library (CCL) that reproduces the results of established MST and SACA implementations on the same event set, while introducing modifications that reduce computational time and improve cluster stability; however, the results show that no single fixed set of parameters in the MST or simulated annealing approach is sufficient to describe complex observables across all time steps.

## Method

The CCL library implements a Disjoint Set Union (DSU)-based Minimum Spanning Tree (MST) procedure that avoids constructing the full graph for each event, a two-pass Simulated Annealing (SA) chain that first deconstructs unstable MST pre-fragments individually before recombining stable clusters, and a "box-like" coalescence procedure with sequential formation of heavier to lighter nuclei. A new stable cluster tracking algorithm ("sMST") recovers clusters that artificially dissolve during QMD propagation by checking constituent collision times and binding energy criteria across time steps. The procedures were tested on PHQMD-simulated events using rapidity density distributions and the observables ⟨M_imf⟩ and ⟨Z_max⟩ versus Z_bound2 at ALADiN and NA49 experiment energies.

## Key numbers

- MST clusterization radius r_clust = 2.5 fm used in SA parameter sets S1 and S2
- Binding energy cutoff E_cut = -4 MeV for distinguishing stable from unstable clusters in the SA chain
- SA initial temperature T_max = 1, minimum temperature T_min = 10^{-15}, with 500 steps for pass 1 and 4000 steps for pass 2
- Tracking procedure with MST parameter sets: T1 (Δr = 4.0 fm), T2 (Δr = 2.5 fm), T3 (Δr = 2.5 fm, Δp = 0.3 GeV/c)
- Application of second MST after stable cluster tracking led to approximately 3% increase in overall stable cluster multiplicity
- 50k central Pb+Pb events at E_lab = 20 AGeV with impact parameter b = 0-5 fm used for coalescence testing

## Context

This work extends the MST-based clusterization approach from Aichelin (1991) and the Simulated Annealing Clusterization Algorithm (SACA) from Puri (1998), as well as the FRIGA approach from LeFevre (2019), by providing an open-source library that is not tied to a single transport code. The new "sMST" tracking algorithm addresses the problem of artificial cluster dissolution in QMD dynamics previously discussed by Coci (2023) with the "aMST" procedure, but without requiring explicit knowledge of kinetic freeze-out time. The library enables systematic comparison of different clusterization methods on identical event sets and can be applied to future studies of hypernuclei production and heavy-ion collision dynamics across a wide energy range.
