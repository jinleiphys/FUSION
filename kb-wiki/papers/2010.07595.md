# Setting the string shoving picture in a new frame

> Based on the recent success of the \angantyr model in describing multiplicity distributions of the hadronic final state in high energy heavy ion collisions, we investigate how far one can go with a such a string-based scenario to describe also flow effects measured in such collisions. For this purpose we improve our previous so-called \textit{shoving} model, where strings that are close in space--time tend to repel each other in a way that could generate anisotropic flow, and we find that this model can indeed generate such flows in Å collisions. The flow generated is not quite enough to reproduce measurements, but we identify some short-comings in the presented implementation of the model that, when fixed, could plausibly give a more realistic amount of flow.

## Key claim

The authors improve their previous "shoving" model for string-string repulsion in heavy ion collisions and find that it can generate anisotropic flow in AA collisions, but the generated flow is not sufficient to reproduce experimental measurements; they identify specific shortcomings in the implementation that, if fixed, could plausibly produce a more realistic amount of flow.

## Method

The authors implement a new shoving model where all string pieces in an event can interact, not only in pp but also in AA and other collision systems. The force between nearby strings is applied as tiny transverse momentum pushes given directly to the produced hadrons during hadronization, rather than through additional soft gluons. The model is applied to a toy geometry for the initial partonic state of an AA collision and then to a more realistic initial state generated by the Angantyr model for AA collisions in PYTHIA8.

## Key numbers

- String tension kappa_S approximately 0.9 GeV/fm
- Hadronization proper time tau_H approximately 1.5-2 fm
- For a single string: energy density dE/dy = kappa_S / tau_H
- In the bag model estimate: string radius R approximately 1.7 fm, or sqrt(<rho^2>) approximately 1.2 fm
- In dual QCD estimate: sqrt(<rho^2>) approximately 0.95 fm
- For RHIC AuAu at 200 GeV and b=9 fm (centrality approximately 40%): dN_ch/dy approximately 150, giving dE/dy approximately 110 GeV
- For central PbPb at LHC: dN_ch/dy approximately 2000, giving dE/dy approximately 3000 particles, energy density dE/d^3 x approximately 5-6 GeV/fm^3 in string scenario
- In glasma scenario for LHC: energy density dE/d^3 x approximately 500 GeV/fm^3 at tau = 0.1 fm

## Context

This work extends the Angantyr model (Bierlich 2018) which successfully described multiplicity distributions in heavy ion collisions, and the authors' previous shoving model (Bierlich 2017) which described the near-side ridge in pp collisions. The paper contrasts the string-based approach with the thermalized quark-gluon plasma scenario using glasma initial conditions (Schenke 2010, 2012) and hydrodynamical modeling. The results suggest that while the string shoving picture can qualitatively generate flow, quantitative agreement with experimental data from ALICE and other collaborations requires further improvements to handle soft gluons from the perturbative phase in PYTHIA8.
