# Nuclear Mass Predictions Using a Neural Network with Additive Gaussian Process Regression-Optimized Activation Functions

> Nuclear masses are machine-learned as a function of proton and neutron numbers. The neural network with additive Gaussian process regression-optimized activation functions (GPR-NN) method is employed for the first time for this purpose. GPR-NN combines the advantages of both neural networks and Gaussian process regression, in that it possesses the expressive power of an NN, in principle allowing modeling any kind of dependence of nuclear mass on the features, and robustness of a linear regression with respect to overfitting. A study of the GPR-NN approach for interpolation and extrapolation in nuclear mass predictions is presented. It is found that the optimal hyperparameters for the GPR-NN approach in interpolation and extrapolation are different. If an appropriate set of hyperparameters is adopted, the GPR-NN approach can achieve good extrapolation performance for nuclear mass prediction, which could potentially help improve the mass predictions of a large number of currently experimentally unknown nuclei.

## Key claim

The GPR-NN method can achieve good extrapolation performance for nuclear mass prediction, but only if a distinct set of hyperparameters (specifically larger kernel length scale and regularization) is adopted rather than the hyperparameters that are optimal for interpolation.

## Method

The authors apply the GPR-NN (neural network with additive Gaussian process regression-optimized activation functions) method to predict nuclear mass residuals (deviations between experimental data and RCHB theoretical predictions) as a function of proton number Z and neutron number N. The method uses a single-hidden-layer NN architecture where neuron activation functions are obtained in one linear step via additive GPR, with redundant coordinates generated from a Sobol sequence. Hyperparameters (number of redundant coordinates R, kernel length scale L, and regularization parameter delta) are optimized separately for interpolation (via tenfold cross-validation on 2278 nuclei from AME2020) and for extrapolation (by optimizing predictions on test sets of neutron-rich nuclei removed from training).

## Key numbers

* Training set: 2278 nuclei from AME2020 with Z >= 8 and N >= 8, excluding masses with experimental error exceeding 100 keV
* Optimal interpolation hyperparameters: R = 3, L = 0.031, delta = 0.001, yielding rms deviation of approximately 440 keV
* Optimal extrapolation hyperparameters: R = 3, L = 0.080, delta = 0.200
* For R = 0 (no redundant coordinates), rms deviation is approximately 1000 keV
* Test sets for generalization: AME83-93 (177 nuclei), AME93-03 (404 nuclei), AME03-12 (254 nuclei), AME12-20 (108 nuclei)

## Context

This work extends prior machine-learning applications to nuclear mass prediction, including kernel ridge regression (Wu 2020), Gaussian process regression (Neufcourt 2019), and Bayesian neural network approaches (Utama 2016, Niu 2018). The GPR-NN method was previously developed for molecular potential energy surfaces (Manzhos 2023) and materials informatics (Tang 2025). The study demonstrates that hyperparameter optimization must be tailored to the prediction task (interpolation vs. extrapolation), and that the GPR-NN approach with appropriate hyperparameters can improve mass predictions for neutron-rich nuclei relevant to r-process nucleosynthesis studies.
