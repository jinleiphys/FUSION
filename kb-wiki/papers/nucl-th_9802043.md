# Electromagnetic decays of vector mesons as derived from QCD sum rules

> We apply the method of QCD sum rules in the presence of external electromagnetic fields $F_{μν}$ to the problem of the electromagnetic decays of various vector mesons, such as $ρ\toπγ$, $K^\ast\to Kγ$ and $η'\toργ$. The induced condensates obtained previously from the study of baryon magnetic moments are adopted, thereby ensuring the parameter-free nature of the present calculation. Further consistency is reinforced by invoking various QCD sum rules for the meson masses. The numerical results on the various radiative decays agree very well with the experimental data.

# Electromagnetic decays of vector mesons as derived from QCD sum rules

## Key claim
The QCD sum rule method in an external electromagnetic field, using induced condensate parameters previously determined from baryon magnetic moments, yields predictions for radiative vector meson decays that agree well with experimental data. The parameter-free nature of the calculation is conditional on adopting a specific set of susceptibility values (χ = -3.5 GeV⁻², 2κ + ξ ≈ 0) that represent an average of several prior determinations, rather than a single uniquely determined set.

## Method
The authors compute the two-point correlation function of vector and pseudoscalar currents in the presence of a constant external electromagnetic field Fμν, then equate the hadronic-level expression (containing the ρπγ coupling gρπγ) with the quark-level evaluation using quark propagators in the external field. After performing a Borel transform to suppress continuum contributions, they obtain a QCD sum rule for the coupling constant, with the dominant contribution coming from the induced quark condensate characterized by the magnetic susceptibility χ. The overlap amplitudes λρ and λπ are determined self-consistently from separate mass sum rules for the ρ and π mesons, and the method is generalized to other decays including K* → Kγ and η' → ργ with appropriate modifications for strange quark mass and mixing effects.

## Key numbers
- gρπγ = 0.59, compared to the value 0.58 extracted from the experimental partial width for ρ → πγ
- λπ = 0.17 ± 0.03 GeV², λρ = 0.17 ± 0.01 GeV² from QCD sum rules
- λK = 0.23 ± 0.03 GeV², λK* = 0.21 ± 0.02 GeV²
- λω = 0.16 ± 0.01 GeV², λφ = 0.24 ± 0.02 GeV²
- λη = 0.23 ± 0.03 GeV², λη' = 0.33 ± 0.05 GeV²
- φ-ω mixing parameter ε = 0.063 ± 0.005, varying from 0.068 to 0.058 as the Borel mass varies from 0.6 GeV² to 1.1 GeV²
- η-η' mixing angle θ = -19° ± 2°
- Predicted partial widths (in keV): ω → πγ = 720, ω → ηγ = 7.1, ρ → πγ = 65, ρ → ηγ = 48, φ → ηγ = 80, φ → η'γ = 0.67, K*⁰ → K⁰γ = 102, K*⁺ → K⁺γ = 60, η' → ωγ = 6.1, φ → πγ = 5.8
- Experimental values for comparison (in keV): ω → πγ = 716, ρ → πγ = 68, φ → πγ = 5.8, η' → ργ = 60 (used as input)
- Quark condensate susceptibility χ = -3.5 GeV⁻² adopted (average of values from -3.0 to -5.7 GeV⁻² from prior work)
- Strange quark mass ms ≈ 150 MeV used for K* decays
- ⟨0|s̄s|0⟩ ≈ 0.8 × ⟨0|ūu|0⟩ used

## Context
This work extends the QCD sum rule method in external electromagnetic fields, previously applied to baryon magnetic moments by Ioffe and Smilga (1984) and Balitsky and Yung (1984), to the radiative decays of vector mesons. It provides an alternative to the Vector Meson Dominance model (VDM) and the effective chiral Lagrangian approach of Bijnens, Bramon, and Cornet (1990), which used the Bardeen-subtracted Wess-Zumino action with symmetry-breaking corrections. The results enable parameter-free predictions for a range of decays and allow extraction of the φ-ω mixing parameter and η-η' mixing angle within the QCD sum rule framework, with the η' → ργ width used as input to determine the mixing angle.
