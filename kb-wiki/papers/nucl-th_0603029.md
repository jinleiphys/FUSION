# Kaon-nucleon scattering lengths from kaonic deuterium experiments

> The extraction of the S-wave kaon-nucleon scattering lengths a0 and a1 from a combined analysis of existing kaonic hydrogen and synthetic deuterium data has been carried out within the framework of a low-energy effective field theory. It turns out that with the present DEAR central values for the kaonic hydrogen ground-state energy and width, a solution for a0 and a1 exists only in a restricted domain of input values for the kaon-deuteron scattering length. Consequently, measuring this scattering length imposes stringent constraints on the theoretical description of the kaon-deuteron interactions at low energies.

## Key claim

The extraction of the S-wave kaon-nucleon scattering lengths a0 and a1 from combined kaonic hydrogen and synthetic deuterium data is possible only in a restricted domain of input values for the kaon-deuteron scattering length when using the DEAR central values for kaonic hydrogen; with the KpX data the solutions exist in a much larger area.

## Method

The authors used a low-energy effective field theory framework with the Fixed Center Approximation (FCA) to express the kaon-deuteron scattering length in terms of the elementary kaon-nucleon scattering lengths a0 and a1. They then performed a combined analysis of existing DEAR kaonic hydrogen data and synthetic kaonic deuterium data, solving the non-linear equations that relate the kaon-deuteron scattering length to a0 and a1, using both non-relativistic EFT deuteron wave functions and NLO EFT wave functions with non-perturbative pions.

## Key numbers

- DEAR kaonic hydrogen ground-state shift: epsilon_1s = 193 +/- 37 (stat) +/- 6 (syst) eV
- DEAR kaonic hydrogen ground-state width: Gamma_1s = 249 +/- 111 (stat) +/- 30 (syst) eV
- Estimated kaonic deuterium ground-state binding energy: E_1s^d = 10.4 keV
- Estimated kaonic deuterium decay width: Gamma_1s^d = 1.2 keV
- Product tau * E_1s^d = 8.6 for kaonic deuterium
- Kaon-deuteron scattering length calculated with Meissner-Oller input using NLO EFT wave functions (Lambda=600 MeV): A_Kd = -1.30 + i 1.67 fm
- Kaon-deuteron scattering length calculated with Borasoy et al. input using NLO EFT wave functions (Lambda=600 MeV): A_Kd = -1.56 + i 1.49 fm
- Solutions for a0 and a1 exist only when Im A_Kd is less than approximately 1 fm with DEAR input
- With DEAR input and lambda=132 MeV, using Torres et al. synthetic A_Kd = -1.34 + i 1.04 fm yields a0 = -1.27 + i 0.36 fm and a1 = 1.19 + i 0.09 fm

## Context

This work extends previous studies of kaon-deuteron scattering within potential multiple-scattering approaches (Hetherington, Torres 1986, Deloff 1999, Barrett 1999) and the chiral unitary approach of Kamalov (2000). Unlike prior work that focused on predicting the K-d scattering length from input kaon-nucleon scattering lengths, this paper performs the "reversed" analysis to determine a0 and a1 from kaonic hydrogen and deuterium data. The results show that the upcoming SIDDHARTA experiment's measurement of kaonic deuterium will impose stringent constraints on theoretical descriptions and may enable accurate determination of the scattering lengths, but only if the kaon-deuteron scattering length falls within a specific range.
