# Symmetries in the \bm{g_{9/2}} shell

> We consider symmetries which arise when two-body interaction matrix elements with isopin T=0 are set equal to a constant in a single-j -shell calculation. The nucleus ^{96} Cd is used as an example.

## Key claim

When two-body interaction matrix elements with isospin T=0 are set to a constant in a single-j-shell calculation for the g9/2 shell, a partial dynamical symmetry (PDS) emerges for T=0 states in 96Cd, but only for angular momenta that cannot occur for four identical nucleons (T=2 states); the symmetry does not hold for states with angular momenta that exist in the (g9/2)4 configuration of identical particles.

## Method

The authors performed single-j-shell calculations for the four-hole system 96Cd using the INTd two-body matrix elements, first with the full interaction and then with all T=0 matrix elements set to a constant (zero in the demonstration). They compared the resulting wave functions and energies for states with J=11 through J=16, examining the decomposition into proton and neutron angular momentum quantum numbers (Jp, Jn) to identify where these become good dual quantum numbers and where degeneracies appear.

## Key numbers

- The J=16+ state in 96Cd lies at 4.9371 MeV with the full INTd interaction, below the lowest J=15+ state at 6.2789 MeV and the lowest J=14+ state at 5.1098 MeV, explaining its isomeric nature
- With T=0 matrix elements set to zero, the J=16+ state lies at 5.6007 MeV, degenerate with the second J=14+ state (also 5.6007 MeV) and above the lowest J=14+ state at 5.3798 MeV
- For the PDS case, J=11, 13, and 14 states with (Jp,Jn)=(6,8) and (8,6) all have the same energy of 5.3798 MeV
- The J=15+ state with (Jp,Jn)=(8,8) lies at 6.2789 MeV with the full interaction and at 7.9251 MeV with T=0 matrix elements set to zero

## Context

This work extends the authors' previous studies of partial dynamical symmetries in the f7/2 shell (Zamick and Robinson, 2001) to the g9/2 shell, motivated by the experimental discovery of a J=16+ isomer in 96Cd by Nara Singh et al. (2011). The results are complementary to recent work on T=0 pairing with maximum alignment in N=Z 92Pd (Cederwall et al., 2011; Ziliani et al., 2011), which focused on lower spins (J=0,2,4,6,8), whereas the PDS discussed here occurs for J=11 and beyond. The findings enable further study of how fluctuations in T=0 matrix elements produce isomeric states in the upper half of the g9/2 shell.
