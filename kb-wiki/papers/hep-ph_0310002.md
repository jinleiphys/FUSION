# Confining Strings and High-Energy Reactions

> We present the loop-loop correlation model that allows us to compute confining flux tubes and to examine their effects and manifestations in high-energy reactions.

## Key claim

The loop-loop correlation model (LLCM) combines perturbative gluon exchange with the non-perturbative stochastic vacuum model to compute confining flux tubes, and through analytic continuation provides the S-matrix element for high-energy dipole-dipole scattering that shows clear manifestations of confining strings for reactions involving large dipoles.

## Method

The LLCM computes chromo-electric fields of a static color dipole in the fundamental and adjoint representation of SU(N_c), obtaining confining flux tubes for large dipoles that show exact Casimir scaling. Based on analytic continuation from Euclidean space, the model provides the S-matrix element S_DD for high-energy dipole-dipole scattering. This S-matrix element is then convoluted with appropriate wave functions to describe dipole-hadron scattering, allowing calculation of unintegrated gluon distributions of hadrons and photons via k_perp factorization.

## Key numbers

- The static quark-antiquark potential shows color Coulomb behavior for small source separations and confining linear rise for large source separations, in agreement with lattice results (no specific numerical values given for the potential parameters).
- Confining flux tubes for large dipoles show exact Casimir scaling in SU(N_c) (no specific numerical values given for the scaling factors).

## Context

This work extends the stochastic vacuum model of Dosch and collaborators by combining it with perturbative gluon exchange to describe both short-range and long-range behavior of the static quark-antiquark potential. The LLCM enables calculation of unintegrated gluon distributions of hadrons and photons from dipole-hadron and dipole-photon cross sections via k_perp factorization, as developed further in Shoshi (2002). The model provides a framework for connecting confining flux tube dynamics to high-energy scattering observables.
