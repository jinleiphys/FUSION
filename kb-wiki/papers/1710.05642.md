# Calm Multi-Baryon Operators

> Outstanding problems in nuclear physics require input and guidance from lattice QCD calculations of few baryons systems. However, these calculations suffer from an exponentially bad signal-to-noise problem which has prevented a controlled extrapolation to the physical point. The variational method has been applied very successfully to two-meson systems, allowing for the extraction of the two-meson states very early in Euclidean time through the use of improved single hadron operators. The sheer numerical cost of using the same techniques in two-baryon systems has been prohibitive. We present an alternate strategy which offers some of the same advantages as the variational method while being significantly less numerically expensive. We first use the Matrix Prony method to form an optimal linear combination of single baryon interpolating fields generated from the same source and different sink interpolators. Very early in Euclidean time this linear combination is numerically free of excited state contamination, so we coin it a calm baryon. This calm baryon operator is then used in the construction of the two-baryon correlation functions. To test this method, we perform calculations on the WM/JLab iso-clover gauge configurations at the SU(3) flavor symmetric point with mπ $\sim$ 800 MeV --- the same configurations we have previously used for the calculation of two-nucleon correlation functions. We observe the calm baryon removes the excited state contamination from the two-nucleon correlation function to as early a time as the single-nucleon is improved, provided non-local (displaced nucleon) sources are used. For the local two-nucleon correlation function (where both nucleons are created from the same space-time location) there is still improvement, but there is significant excited state contamination in the region the single calm baryon displays no excited state contamination.

## Key claim

The Matrix Prony method can be used to construct an optimal linear combination of single-baryon sink interpolating fields (a "calm baryon") that is numerically free of excited state contamination at early Euclidean times, and this calm baryon operator can then be used to build two-baryon correlation functions with significantly reduced excited state contamination, provided non-local (displaced nucleon) sources are used.

## Method

The authors first performed a Matrix Prony analysis of single-nucleon correlation functions constructed from point-smeared and smeared-smeared sink interpolators to determine the linear combination that isolates the ground state nucleon at the earliest possible Euclidean time. This optimized linear combination was then used to construct "calm" baryon blocks, which were employed in the unified contraction algorithms to build two-nucleon correlation functions. The method was tested on WM/JLab iso-clover gauge configurations at the SU(3) flavor symmetric point, using both local two-nucleon operators (both nucleons at the same space-time point) and non-local displaced nucleon operators at the source.

## Key numbers

- The calculation was performed at the SU(3) flavor symmetric point with mπ ∼ 800 MeV and a ∼ 0.145 fm, on a volume with L/a = 24.
- The optimized ground state Matrix Prony correlation function for the single nucleon plateaus 5-6 time slices earlier than the original smeared-smeared and point-smeared correlation functions.
- With the calm baryon method applied to non-local two-nucleon operators, the signal can be extracted approximately 1 fm earlier in Euclidean time where the signal-to-noise is exponentially larger.

## Context

This work extends the Matrix Prony method previously applied to two-nucleon correlation functions by the NPLQCD collaboration (Beane et al., 2009; Beane et al., 2011), but applies it to single-nucleon operators before constructing two-nucleon operators, rather than after. The method is substantially less expensive than the full variational method (GEVP) used successfully in mesonic systems by Dudek et al. (2012) and Wilson et al. (2014), and offers a numerically inexpensive exponential improvement over standard approaches. The calm baryon technique enables future calculations at lighter pion masses where controlled extrapolations to the physical point become feasible, and is readily applicable to multi-nucleon systems and channels with other flavors of baryons.
