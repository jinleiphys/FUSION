# Imaging Three Dimensional Two-particle Correlations for Heavy-Ion Reaction Studies

> We report an extension of the source imaging method for analyzing three-dimensional sources from three-dimensional correlations. Our technique consists of expanding the correlation data and the underlying source function in spherical harmonics and inverting the resulting system of one-dimensional integral equations. With this strategy, we can image the source function quickly, even with the finely binned data sets common in three-dimensional analyses.

## Key claim

The paper extends the one-dimensional source imaging method to three-dimensional correlations by expanding both the correlation data and the underlying source function in spherical harmonics, enabling fast inversion even with finely binned data sets. The authors note that this technique works for like-meson pairs where final-state interactions and Coulomb effects are present, but the method's applicability to other pair types (e.g., proton-proton) is mentioned only as a future possibility rather than demonstrated.

## Method

The authors convert the three-dimensional Koonin-Pratt integral equation relating the correlation function \(C_{\bf P}({\bf q})\) to the source function \(S_{\bf P}({\bf r})\) into a system of one-dimensional integral equations by expanding both functions in spherical harmonics and the kernel in Legendre polynomials. They then invert each one-dimensional equation using basis spline representations of the source with optimized discretization, following techniques from prior imaging work (Brown et al., 2002; Brown et al., 2003). The method is tested on a model correlation generated from a single-particle source consisting of a prompt Gaussian component and a long-lived exponential component, using the CRAB code to compute the correlation including Coulomb repulsion.

## Key numbers

- Test source Gaussian radii: \(R_x = R_y = R_z = 4\) fm
- Long-lived source lifetime: \(\tau = 20\) fm/c
- Fraction of pions from prompt source: \(f = 0.65\)
- Source temperature: 50 MeV
- Pairs kept for source calculation: relative momentum less than 60 MeV/c
- Total simulated pairs used: 925,260
- Correlation binning: 5 MeV wide slices along each axis
- Spherical harmonic expansion truncated at \(\ell_{\text{max}} = 2\) for the test case
- Bump volume integral flattens past \(q = 35\) MeV/c
- Angular power spectrum flattens past \(\ell = 4\)

## Context

This work extends the one-dimensional source imaging technique developed by Brown et al. (2002, 2003) to three-dimensional correlations, enabling model-independent extraction of non-Gaussian source functions from multidimensional pion correlation data. The method contrasts with standard analyses that apply ad-hoc Coulomb corrections before fitting to a Gaussian, and it builds on the Koonin-Pratt formalism (Koonin, 1977; Pratt, 1990) used in heavy-ion interferometry at facilities from MSU-NSCL to RHIC. The technique enables direct comparison of three-dimensional proton and pion sources free from final-state interaction distortions, and could be applied to other pair types such as proton-proton pairs (Lisa et al., 2005) in future work.
