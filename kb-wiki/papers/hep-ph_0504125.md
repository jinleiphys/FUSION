# Relativistic Calculation of Pentaquark Widths

> We calculate the widths of the various pentaquarks in a relativistic model in which the pentaquark is considered to be composed of a scalar diquark and a spin 1/2 triquark. We consider both positive and negative parity for the pentaquark. There is a single parameter in our model which we vary and which describes the size of the pentaquark. We obtain quite small widths for the decay Theta^(+) -> N+K^(+) and for Theta_c^0 -> P+D^{*-} consistent with the experimental situation. For the sum of the decay widths for Xi(bar)^(--) -> Xi^(-) + pi^(+) and Xi(bar)^(--) -> Sigma^(-) + K^(-) we find values of the order of 4-8 MeV for pentaquarks of the characteristic size considered in this work. (The experimental situation with respect to te observation of the Xi(bar)^(--) is somewhat uncertain at this time.) We also provide results for the decays N^(+) -> N + pi and N_s^(+) -> Lambda^(0) + K^(+). Our model of confinement plays an important role in our analysis and makes it possible to use Feynman diagrams to describe the decay of the pentaquark.

## Key claim

The authors calculate pentaquark decay widths in a relativistic diquark-triquark model and find that very small widths for the \(\Theta^+\) and \(\Theta_c^0\) can be obtained, but only for a specific range of the single free parameter \(w\) that controls the pentaquark size; the results depend on the assumed parity (positive or negative) of the pentaquark.

## Method

The pentaquark is modeled as a bound state of a scalar diquark and a spin-1/2 triquark, with a covariant confinement model that prevents unphysical decays into on-shell constituents. The decay amplitude is computed from a Feynman diagram where the triquark emits a quark that combines with the diquark to form the final-state baryon, while the final-state meson is emitted from the triquark vertex. The nucleon and triquark wave functions are taken from a fit to earlier work (Shakin et al., 2005), while the pentaquark wave function is a Gaussian whose width \(w\) is the single adjustable parameter; widths are evaluated for both positive and negative parity assignments.

## Key numbers

- For the decay \(\Theta^+ \rightarrow N + K^+\) with \(m_\Theta = 1.540\) GeV, \(m_T = 0.800\) GeV, \(m_D = 0.400\) GeV, \(m_N = 0.939\) GeV, \(m_K = 0.495\) GeV: very small widths are obtained for \(w\) in the range \(0.75\) GeV \(< w < 0.80\) GeV (negative parity) and \(0.70\) GeV \(< w < 0.90\) GeV (positive parity).
- For the decay \(\Theta_c^0 \rightarrow P + D^{*-}\) with \(m_{\Theta_c^0} = 3.099\) GeV, \(m_P = 0.939\) GeV, \(m_D = 0.400\) GeV, \(m_T = 2.00\) GeV, \(m_q = 0.350\) GeV, \(m_{D^{*-}} = 2.007\) GeV: widths of about \(3\)–\(4\) MeV are found for the positive-parity case in the range \(0.75\) GeV \(< w < 0.80\) GeV.
- For the sum of the decays \(\bar{\Xi}^{--} \rightarrow \Xi^- + \pi^-\) and \(\bar{\Xi}^{--} \rightarrow \Sigma^- + K^-\) with \(m_{\bar{\Xi}} = 1.862\) GeV, \(m_T = 0.800\) GeV, \(m_D = 0.700\) GeV, \(m_s = 0.450\) GeV, \(m_\Xi = 1.321\) GeV, \(m_\pi = 0.139\) GeV, \(m_{\Sigma^-} = 1.197\) GeV, \(m_K = 0.495\) GeV: summed widths of order \(4\)–\(8\) MeV are obtained in the region \(0.75\) GeV \(< w < 0.80\) GeV.
- The phase space factor for \(\Theta^+\) decay is \(\rho = 0.0261\) GeV, with \(|\vec{P}_N| = 0.269\) GeV.

## Context

This work extends the diquark-triquark model of pentaquarks introduced by Jaffe and Wilczek (2003) and applies a covariant confinement model previously developed by Shakin et al. (2005) for meson and nucleon structure. The calculations provide a theoretical framework for the observed narrow widths of the \(\Theta^+(1540)\) and \(\Theta_c^0\) pentaquarks, and they predict widths of 4–8 MeV for the \(\bar{\Xi}^{--}\) decays, which may help resolve the experimental controversy surrounding that state (reported by some experiments but not seen by others). The results also give predictions for the decays of \(N^+\) and \(N_s^+\) pentaquarks, which are expected to mix with ordinary nucleon resonances.
