# HFB3: an axial HFB solver with Gogny forces using a 2-center HO basis (C++/Python)

> The HFB3 program solves the axial nuclear Hartree-Fock-Bogoliubov (HFB) equations using bases formed by either one or two sets of deformed Harmonic Oscillator (HO) solutions with D1-type and D2-type Gogny effective nucleon-nucleon interactions. Using two sets of HO solutions shifted along the z-axis (2-center basis) allows to accurately describe highly elongated nuclear systems while keeping a moderate basis size, making this type of basis very convenient for the description of the nuclear fission process. For the description of odd-even and odd-odd systems, the equal-filling-approximation is used. Several observables can be calculated by the program, including the mean values of the multipole moments, nuclear radii, inertia tensors following Adiabatic Time-Dependent Hartree-Fock-Bogoliubov (ATDHFB) or Generator Coordinate Method (GCM) prescriptions, local and non-local one-body densities, local and non-local pairing densities, some fission fragment properties, etc. The program can ensure that the mean values associated with some specific operators take pre-defined values (constraints). Such constraints can be set on the usual multipole moments (for protons, neutrons or total mass). This program can be used as a monoprocess and monothreaded CLI executable, or through full-featured Python bindings (available through the Python Package Index PyPI).

## Key claim

The HFB3 program solves the axial nuclear Hartree-Fock-Bogoliubov equations using a basis formed by either one or two sets of deformed harmonic oscillator solutions, where the two-center basis option allows accurate description of highly elongated nuclear systems while keeping a moderate basis size, making it convenient for nuclear fission studies.

## Method

The program solves the HFB equations using a deformed axial harmonic oscillator basis or the union of two sets of z-shifted deformed axial harmonic oscillator bases, with constraints placed on specific multipole moments to calculate potential energy surfaces of any dimension. Self-consistent iterations via Broyden mixing or gradient descent are used to find the HFB minimum, and the equal-filling approximation is employed for odd-even and odd-odd systems. The program implements D1-type and D2-type Gogny effective nucleon-nucleon interactions, with matrix elements evaluated in either a 1-center or 2-center basis.

## Key numbers

- The basis truncation parameters N and Q define the set of quantum numbers included, with m_max found as the supremum of i such that nu(i) = (N+2)*Q^(2/3) + 1/2 - i*Q is greater than or equal to 1
- The beta_20 elongation operator is defined as beta_20 = (4*pi)/(3*r_0^2*A^(5/3))*Q_20 with r_0 = 1.2 fm
- The example calculation for 240Pu uses an optimized 2-center, 11 major HO shells basis (called a "2x11 basis") with constraints Q_10 = 0 fm and beta_20 = 1.4
- The example calculation for 16O uses an optimized 7-shells 1-center basis with constraints Q_10 = 0 fm, Q_20 = 8 fm^2, and Q_30 = 0 fm^3
- The default number of blocking trials for each isospin value in odd-system calculations is set to 4

## Context

This work extends the lineage of axial HFB solvers such as HFBTHO (Tho22) and the solvers by Berger and Robledo, providing a new implementation that supports both 1-center and 2-center harmonic oscillator bases with Gogny forces. The program enables future studies of nuclear fission through constrained HFB calculations of potential energy surfaces, and its Python bindings and open-source distribution through PyPI make it accessible for broader use in nuclear structure and reaction studies.
