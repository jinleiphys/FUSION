# Improvement Studies of an Effective Interaction for N=Z sd-shell Nuclei by Neural Networks

> The nuclear shell model is one of the successful models in theoretical understanding of nuclear structure. If a convenient effective interaction can be found between nucleons, various observables such as energies of nuclear states are accurately predicted by this model. The basic requirements for the shell model calculations are a set of single particle energies and two-body interaction matrix elements (TBME) which construct the residual interaction between nucleons. This latter could be parameterized in different ways. In this study, we have used a different approach to improve existing USD type Hamiltonians for the shell model calculations of N=Z nuclei in the A=16-40 region. After obtaining the SDNN new effective interaction, shell model calculations have been performed for all N=Z nuclei in sd shell. In which, $^{16}{O}$ doubly magic nucleus has been assumed as an inert core and active particles are distributed in the $d_{5/2}$, $s_{1/2}$ and $d_{3/2}$ single particle orbits. The rms deviations from experimental energy values are lower for the newly generated effective interaction than those obtained using the original one for the studied nuclei.

## Key claim

The authors claim that a new effective interaction (SDNN) generated by training an artificial neural network on the existing USDB Hamiltonian yields lower root-mean-square deviations from experimental energy levels for N=Z sd-shell nuclei than the original USDB interaction, but this improvement is marginal (approximately a factor of 1.1 for even-even nuclei) and does not hold for all individual nuclei (e.g., USDB gives better results for 32S).

## Method

The authors used a feed-forward artificial neural network with one input neuron, one hidden layer of 8 neurons, and one output neuron (1-8-1 architecture) to map the 63 two-body matrix elements (TBMEs) of the USDB Hamiltonian to a new set of TBMEs called SDNN. The network was trained using the Levenberg-Marquardt back-propagation algorithm with a mean square error target of about 6x10-29. Shell model calculations for all N=Z nuclei in the sd shell (A=16-40) were then performed using both the original USDB and the new SDNN interactions with the KShell code, assuming a 16O inert core and valence nucleons distributed in the d5/2, s1/2, and d3/2 orbitals.

## Key numbers

- The root mean square deviation of the new SDNN matrix elements from the original USDB matrix elements is 0.052 MeV.
- For the matrix element ⟨21|V|21⟩_{21}, the difference between SDNN and USDB reaches a maximum of -0.2768 MeV.
- For even-even N=Z nuclei, the rms deviations from experimental energy levels using USDB are: 163.917 keV (20Ne), 330.292 keV (24Mg), 244.900 keV (28Si), 156.905 keV (32S), and 121.655 keV (36Ar); using SDNN they are: 116.221 keV (20Ne), 312.307 keV (24Mg), 220.341 keV (28Si), 170.940 keV (32S), and 105.983 keV (36Ar).
- The overall rms deviation for all even-even N=Z sd-shell isotopes is 218.327 keV for USDB and 202.436 keV for SDNN.
- For all N=Z isotopes (including odd-odd), the rms deviations are 208.255 keV for USDB and 205.363 keV for SDNN.

## Context

This work extends the USD family of Hamiltonians (Wildenthal, 1984; Brown and Richter, 2006) by applying artificial neural networks, which have previously been used in nuclear physics for mass systematics (Athanassopoulos, 2004; Bayram, 2013), proton radii (Graczyk, 2011), and nuclear mass prediction (Utama, 2015). The new SDNN interaction enables improved shell model calculations for N=Z nuclei in the sd shell, though the improvement is modest and the method could be applied to other mass regions or to refine other effective interactions.
