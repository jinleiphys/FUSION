# Initial condition for the Balitsky-Kovchegov equation at next-to-leading order

> We determine the initial condition of the Balitsky-Kovchegov evolution equation at next-to-leading order (NLO) accuracy using HERA deep inelastic scattering data. Posterior distributions characterizing the initial condition are extracted using Bayesian inference. The total cross section and charm quark production data from HERA are found to provide stringent constraints on the posterior. These distributions quantify the uncertainty in the initial condition and serve as necessary input for propagating uncertainties to all NLO calculations within the Color Glass Condensate framework.

## Key claim

The authors determine the initial condition for the Balitsky-Kovchegov evolution equation at next-to-leading order accuracy by performing a Bayesian inference fit to HERA deep inelastic scattering data, but only using an approximate evolution equation (the kinematically constrained BK equation) that resums the dominant double transverse logarithms rather than the full NLO BK equation.

## Method

The authors compute deep inelastic scattering structure functions using NLO impact factors for light and heavy quarks coupled to dipole-target scattering amplitudes evolved with the kinematically constrained Balitsky-Kovchegov (KCBK) equation, which resums double transverse logarithms to approximate the full NLO BK evolution. Bayesian inference with Gaussian process emulators is employed to extract posterior distributions for the parameters of a McLerran-Venugopalan-inspired initial condition, using HERA combined total reduced cross section data and charm production data in the kinematic range 2.0 GeV^2 ≤ Q^2 < 50.0 GeV^2 and x ≤ 0.01. Two running coupling prescriptions are tested: the parent dipole prescription and the Balitsky+smallest dipole prescription.

## Key numbers

- Initial saturation scale Q_{s,0}^2 = 0.130_{-0.012}^{+0.008} GeV^2 (median with 95% credible interval) for the Balitsky+smallest dipole running coupling setup
- Initial saturation scale Q_{s,0}^2 = 0.096_{-0.014}^{+0.016} GeV^2 for the parent dipole running coupling setup
- Anomalous dimension γ = 3.08_{-0.64}^{+0.38} (Balitsky+smallest dipole) and γ = 1.67_{-0.11}^{+0.15} (parent dipole)
- Charm quark mass m_c = 1.25_{-0.03}^{+0.02} GeV (Balitsky+smallest dipole) and m_c = 1.45_{-0.05}^{+0.06} GeV (parent dipole)
- Proton transverse area σ_0/2 = 8.75_{-0.53}^{+0.72} mb (Balitsky+smallest dipole) and σ_0/2 = 20.7_{-2.0}^{+2.1} mb (parent dipole)
- Saturation scale Q_s^2 at Y = ln(1/0.01) = 0.203 GeV^2 (Balitsky+smallest dipole) and 0.208 GeV^2 (parent dipole)
- χ^2/d.o.f. for all data = 1.35 (Balitsky+smallest dipole) and 1.22 (parent dipole) at MAP values
- χ^2/d.o.f. for charm data only = 3.96 (Balitsky+smallest dipole) and 2.84 (parent dipole) at MAP values

## Context

This work extends the previous leading-order Bayesian analysis by Casuga (2023) to next-to-leading order accuracy, and improves upon earlier NLO dipole fits by Beuf (2020) and Hanninen (2022) by including both total cross section and charm production data simultaneously in the fit, accounting for correlated experimental uncertainties, and providing systematic uncertainty quantification through posterior distributions. The extracted initial condition with quantified uncertainties enables propagation of uncertainties to all NLO calculations within the Color Glass Condensate framework, which is essential for precision comparisons with future measurements at the Electron-Ion Collider and the LHC.
