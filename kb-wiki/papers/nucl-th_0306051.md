# Multi-Shell Shell Model for Heavy Nuclei

> Performing a shell model calculation for heavy nuclei has been a long-standing problem in nuclear physics. Here we propose one possible solution. The central idea of this proposal is to take the advantages of two existing models, the Projected Shell Model (PSM) and the Fermion Dynamical Symmetry Model (FDSM), to construct a multi-shell shell model. The PSM is an efficient method of coupling quasi-particle excitations to the high-spin rotational motion, whereas the FDSM contains a successful truncation scheme for the low-spin collective modes from the spherical to the well-deformed region. The new shell model is expected to describe simultaneously the single-particle and the low-lying collective excitations of all known types, yet keeping the model space tractable even for the heaviest nuclear systems.

## Key claim

The authors propose a new "Heavy Shell Model" (HSM) that combines the Projected Shell Model (PSM) and the Fermion Dynamical Symmetry Model (FDSM) to enable shell model calculations for heavy nuclei, but the paper presents only the conceptual framework and formalism without reporting any actual numerical results or test calculations.

## Method

The authors review the PSM, which uses deformed-BCS quasi-particle bases with angular-momentum and particle-number projection to describe high-spin rotational states, and the FDSM, which uses an S-D pair truncation scheme based on group theory to describe low-spin collective vibrations. They then demonstrate that the projected deformed-BCS vacuum in the PSM exhibits an SU(3) symmetry spectrum nearly identical to the FDSM SU(3) formalism, using the nucleus 168Er as an example with effective neutron number 32 and effective proton number 16. The proposed HSM would integrate these two approaches by adopting the FDSM's microscopic classification for collective vibrations while keeping the PSM's method of building shell model configuration space through projection.

## Key numbers

- The PSM calculation for 168Er used effective neutron number n_eff_nu = 32 and effective proton number n_eff_pi = 16 for comparison with the FDSM SU(3) limits
- The PSM usually includes three major shells for neutrons and protons in deformed nuclei calculations, and four major shells for superdeformed nuclei
- The monopole-pairing strength in the PSM is taken as G_M = G/A, where A is the mass number, with G adjusted to yield known odd-even mass differences
- The quadrupole-pairing strength G_Q in the PSM is assumed to be about 20% of G_M
- The full fp shell calculation of A=52 nuclei by the ANTOINE code achieved basis dimensions in excess of 100 million

## Context

This work extends the long-standing effort to perform shell model calculations for heavy nuclei, which has relied primarily on mean-field approximations as described by Ring and Schuck (1980) and Frauendorf (2001). The paper builds upon the PSM developed for high-spin states of deformed and superdeformed nuclei, and the FDSM which provides a unified description for low-spin collective excitations. The proposed HSM aims to enable simultaneous description of single-particle and low-lying collective excitations for the heaviest nuclear systems, with potential applications to nuclear astrophysics calculations of transition probabilities, spectroscopic factors, and beta-decay rates.
