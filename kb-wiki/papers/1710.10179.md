# Nucleon structure functions in the truncated moments approach

> We demonstrate advantages of the truncated Mellin moments (TMM) approach in the analysis of DIS data. We present a novel method for determination of the Bjorken sum rule (BSR) from restricted in $x$ variable experimental data. We show how to incorporate different uncertainties for each kinematic bin. We apply our analysis to recent COMPASS data.

## Key claim

The truncated Mellin moments (TMM) approach allows determination of the Bjorken sum rule (BSR) from experimental data restricted to a finite range of the Bjorken variable \(x\), using a generalized BSR that converges to the full BSR value more rapidly than the standard truncated integral, provided the first-order Taylor approximation is applied.

## Method

The authors construct a generalized truncated BSR \(\Gamma_{1;\omega}(x_0)\) by convolving the spin structure function \(g_1\) with a normalized, sign-changing weight function \(\omega(z)\) depending on parameters \(z_1, z_2, A\). This generalized moment obeys the same DGLAP evolution as \(g_1\) and approaches the full BSR \(\Gamma_1(0)\) faster than the ordinary truncated moment \(\Gamma_1(x_0)\). The full BSR is then estimated via a first-order Taylor expansion (IAPX) of \(\Gamma_{1;\omega}\) around \(x_0\). Both two-point and multi-point versions of the weight function are implemented, with the multi-point version incorporating experimental uncertainties by assigning weights inversely proportional to relative uncertainties.

## Key numbers

- The first-order approximation estimate of the full BSR from COMPASS data, using the two-point method with \(x_{\text{min}} = 0.0036\) and various \(r\) values (0.9, 0.5, 0.3, 0.1), yields \(\Gamma^{\text{IAPX}}_1(x_{\text{min}}, r) = 0.190\).
- The multi-point estimation using the same COMPASS data gives \(\Gamma^{\text{IAPX}}_1(\{x_i\}_1^n) = 0.190\).
- These results are compared to the COMPASS collaboration value \(\Gamma_1(0) = 0.192 \pm 0.007_{\text{stat}} \pm 0.015_{\text{syst}}\).
- The fit to COMPASS data at \(Q^2 = 3\ \text{GeV}^2\) is \(g_1(x) \sim x^{-0.36}(1-x)^3(1+3.9x)\).
- The input parametrization used for tests at \(Q_0^2 = 1\ \text{GeV}^2\) is \(g_1(x,Q_0^2) = N \cdot x^a (1-x)^b (1+\gamma x)\) with \(b=3\), \(\gamma=5\), and \(a=0\) or \(a=-0.4\).

## Context

This work extends the truncated Mellin moments approach introduced in the late 1990s (Kotlorz and collaborators, late 1990s) and its application to spin structure functions, including the generalized Wandzura-Wilczek relation and evolution equations for \(g_2\) (Kotlorz and collaborators, 2000s). The method enables determination of the Bjorken sum rule from data restricted to \(x \gtrsim 0.0036\), as demonstrated on COMPASS data, and can be adapted to other experiments such as those at JLab where the accessible \(x\) range is far from zero. The approach also provides a framework for incorporating bin-dependent experimental uncertainties into sum-rule extraction.
