# Uncertainties in nuclear transition matrix elements for neutrinoless $ββ$ decay II: the heavy Majorana neutrino mass mechanism

> Employing four different parametrization of the pairing plus multipolar type of effective two-body interaction and three different parametrizations of Jastrow-type of short range correlations, the uncertainties in the nuclear transition matrix elements $M_{N}^{(0ν)}$ due to the exchange of heavy Majorana neutrino for the $0^{+}\rightarrow 0^{+}$ transition of neutrinoless double beta decay of $^{94}$Zr, $^{96}$Zr, $^{98}$Mo, $^{100}$Mo, $^{104}$Ru, $^{110}$Pd, $^{128,130}$Te and $^{150}$Nd isotopes in the PHFB model are estimated to be around 25%. Excluding the nuclear transition matrix elements calculated with Miller-Spenser parametrization of Jastrow short range correlations, the uncertainties are found to be 10%-15% smaller.

## Key claim

The theoretical uncertainty in nuclear transition matrix elements for neutrinoless double beta decay via heavy Majorana neutrino exchange, calculated in the Projected-Hartree-Fock-Bogoliubov (PHFB) model, is estimated to be around 35% when including the Miller-Spenser parametrization of short range correlations, but reduces to below 20% when that parametrization is excluded.

## Method

The authors calculated nuclear transition matrix elements for the heavy Majorana neutrino exchange mechanism in the PHFB model for nine isotopes (94Zr, 96Zr, 98Mo, 100Mo, 104Ru, 110Pd, 128Te, 130Te, and 150Nd). They employed four different parametrizations of the pairing plus multipolar effective two-body interaction (PQQ1, PQQHH1, PQQ2, PQQHH2) and three different Jastrow-type short range correlation parametrizations (Miller-Spenser, CD-Bonn, and Argonne V18), generating twelve sets of matrix elements per nucleus. The statistical mean and standard deviation of these twelve values were computed to estimate the theoretical uncertainty.

## Key numbers

- For 94Zr with PQQ1 parametrization and finite size only (no SRC): matrix element = 236.9498 (dimensionless)
- For 94Zr with PQQ1 and SRC1 (Miller-Spenser): matrix element = 77.5817
- For 94Zr with PQQ1 and SRC2 (CD-Bonn): matrix element = 138.2606
- For 94Zr with PQQ1 and SRC3 (Argonne V18): matrix element = 191.3897
- For 100Mo with PQQ1 and finite size only: matrix element = 365.8004
- For 100Mo with PQQ1 and SRC1: matrix element = 122.2000
- For 100Mo with PQQ1 and SRC2: matrix element = 215.8882
- For 100Mo with PQQ1 and SRC3: matrix element = 296.9869
- For 150Nd with PQQ1 and finite size only: matrix element = 163.8037
- For 150Nd with PQQ1 and SRC1: matrix element = 55.8968
- For 150Nd with PQQ1 and SRC2: matrix element = 97.8169
- For 150Nd with PQQ1 and SRC3: matrix element = 133.6912
- Overall uncertainty including all SRC parametrizations: approximately 35%
- Uncertainty excluding Miller-Spenser SRC: smaller than 20%

## Context

This work extends the authors' previous statistical analysis of uncertainties in light Majorana neutrino exchange matrix elements (Rath et al., 2010) to the heavy Majorana neutrino mass mechanism, using the same PHFB model and interaction parametrizations. The study builds on earlier QRPA calculations by Tomoda (1991), Hirsch et al. (1996), and Simkovic et al. (2001, 2008, 2009), and addresses the need for reliable uncertainty estimates in nuclear matrix elements for neutrinoless double beta decay. The results enable more robust extraction of limits on the effective heavy Majorana neutrino mass from experimental half-life limits, and confirm that the Miller-Spenser parametrization is a major source of theoretical uncertainty, consistent with findings by Simkovic et al. (2009).
