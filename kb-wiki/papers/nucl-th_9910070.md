# Ternary configuration in the framework of inverse mean-field method

> A static scission configuration in cold ternary fission has been considered in the framework of mean field approach. The inverse scattering method is applied to solve single-particle Schroedinger equation, instead of constrained selfconsistent Hartree-Fock equations. It is shown, that it is possible to simulate one-dimensional three-center system via inverse scattering method in the approximation of reflectless single-particle potentials.

# Ternary configuration in the framework of inverse mean-field method

## Key claim
The authors show that a one-dimensional three-center nuclear system can be simulated using the inverse scattering method in the approximation of reflectless single-particle potentials, as an alternative to solving constrained self-consistent Hartree-Fock equations. This approach is presented as a tool to prepare initial states for dynamical mean-field calculations, but the authors explicitly note that the one-dimensional model is too primitive to describe real three-dimensional fragmentation.

## Method
The authors apply the inverse scattering method to solve the single-particle Schroedinger equation for a slab geometry (finite in the z coordinate, infinite and homogeneous in two transverse directions), using the approximation of reflectless potentials (R(k)=0) and symmetric potentials (U(-z)=U(z)). Instead of solving constrained Hartree-Fock equations directly, they use the Gel'fand-Levitan-Marchenko integral equation to reconstruct the single-particle potential, wave functions, and density profiles from a given set of bound state eigenvalues. The method is demonstrated on a three-level (N=3, N0=3) light model system with A approximately 20.

## Key numbers
- The model system has A approximately 20, with thickness parameter A approximately 1.0 (dimensionless, corresponding to the one-dimensional analogue of a three-dimensional system radius)
- The system uses N=3 bound states, all of which are occupied (N0=3)
- The Fermi energy e_F controls conservation of the total number of nucleons via the relation A = (6 A rho_N^2 / pi)^(1/3)

## Context
This work extends the inverse mean-field method previously developed by Kartavenko and Madsen (1987) for ground state nuclear slabs to the case of fragmented two- and three-center configurations. It addresses the lack of a developed three-center shell model, which has only seen very early steps (Sandulescu et al., 1977), in contrast to the well-developed two-center shell model (Maruhn and Greiner, 1972). The method is intended to provide a simple way to prepare initial states for dynamical mean-field calculations of cold ternary fission, which has been observed experimentally in the spontaneous fission of Cf-252 with emission of alpha particles (He-4) and Be-10.
