# Dynamic aspect of the chiral phase transition in the mode coupling theory

> We analyze the dynamic aspect of the chiral phase transition. We apply the mode coupling theory to the linear sigma model and derive the kinetic equation for the chiral phase transition. We challenge Hohenberg and Halperin's classification scheme of dynamic critical phenomena in which the dynamic universality class of the chiral phase transition has been identified with that of the antiferromagnet. We point out a crucial difference between the chiral dynamics and the antiferromagnet system. We also calculate the dynamic critical exponent for the chiral phase transition. Our result is $z=1-η/2\cong 0.98$ which is contrasted with $z=d/2=1.5$ of the antiferromagnet.

## Key claim

The authors argue that the chiral phase transition does **not** belong to the same dynamic universality class as the antiferromagnet, contrary to the classification scheme of Hohenberg and Halperin (1977) and the earlier claim by Rajagopal and Wilczek (1992). The crucial difference is that the slow mode for the chiral order parameter is a propagating "particle mode" (requiring both the meson field and its canonical momentum as slow variables), whereas the slow mode for the antiferromagnet order parameter is diffusive.

## Method

The authors apply the mode coupling theory of Kawasaki to the O(2) and O(4) linear sigma models. They derive the kinetic equation for the chiral phase transition by identifying the slow variables (meson field, canonical momentum, chiral charge density, energy density, and momentum density) and computing the frequency matrix, dissipation terms, and mode coupling terms from the microscopic commutation relations and static susceptibilities. The dynamic critical exponent is then obtained from the scaling behavior of the kinetic equation.

## Key numbers

- Dynamic critical exponent for the chiral phase transition: \(z = 1 - \eta/2 \cong 0.98\)
- Dynamic critical exponent for the antiferromagnet: \(z = d/2 = 1.5\) (where \(d = 3\) is the spatial dimensionality)
- The exponent \(z = 0.98\) is contrasted with the antiferromagnet value of \(z = 1.5\)

## Context

This work challenges the identification by Rajagopal and Wilczek (1992) that the chiral phase transition belongs to the same dynamic universality class as the Heisenberg antiferromagnet, which had been based on Hohenberg and Halperin's (1977) classification method. The authors point out that the canonical momentum conjugate to the meson field is a necessary slow variable for describing the propagating meson mode, yet it is neither an order parameter nor a conserved quantity, so the standard prescription of gathering only order parameters and conserved quantities is insufficient. This result enables a reanalysis of the dynamic critical behavior of the chiral phase transition using mode coupling theory rather than the time-dependent Ginzburg-Landau approach used previously.
