# Systematic study of proton radioactivity half-lives based on the relationship between the Skyrme-Hartree-Fock and the macroscopic quantities of nuclear matter

> In the present work, we systematically study the proton radioactivity half-lives of 33 spherical nuclei based on the relationship between the Skyrme parameters and the macroscopic quantities of nuclear matter. Using the two-potential approach with the spherical Skyrme-Hartree-Fock model, the correlation between proton radioactivity half-life and macroscopic quantities was analyzed. Moreover, we obtained a new Skyrme parameter set by fitting the two most weighted macroscopic quantities. Compared with Skyrme parameters MSL0 and the theoretical model of proton radioactivity UDLP, the theoretical proton radioactivity half-life calculated by the new Skyrme parameter set can better reproduce the experimental data.

# Systematic study of proton radioactivity half-lives based on the relationship between the Skyrme-Hartree-Fock and the macroscopic quantities of nuclear matter

## Key claim

The authors claim that a new Skyrme parameter set (MQSP), obtained by fitting only the two macroscopic quantities that most strongly influence the proton radioactivity half-life (the normal density rho_0 and the gradient coefficient G_S), yields theoretical half-lives that better reproduce experimental data than the existing MSL0 parameter set or the universal decay law for proton radioactivity (UDLP). The hedge is that this improvement is demonstrated only for 33 spherical nuclei with proton numbers 69 <= Z <= 81, and the new parameter set is specifically tailored for proton radioactivity calculations rather than for general nuclear structure.

## Method

The authors used the two-potential approach (TPA) combined with the spherical Skyrme-Hartree-Fock (SHF) model to calculate proton radioactivity half-lives. They first computed half-lives for 33 spherical proton emitters using the MSL0 Skyrme parameter set, then systematically varied each of nine macroscopic nuclear matter quantities (rho_0, E_0, K_0, m*_s,0, m*_v,0, E_sym, L, G_S, G_V) individually to determine which quantities most strongly affect the standard deviation between calculated and experimental half-lives. Based on this sensitivity analysis, they fitted a new Skyrme parameter set (MQSP) by adjusting only the two most influential quantities (rho_0 and G_S) while keeping the other seven at their MSL0 values.

## Key numbers

- Standard deviation between experimental and calculated half-lives using MSL0: Delta_MSL0 = 0.405
- Standard deviation using the new MQSP parameter set: Delta_MQSP = 0.362
- Standard deviation using UDLP: Delta_UDLP = 0.459
- Fitted value of normal density rho_0 in MQSP: 0.155 fm^{-3} (compared to 0.16 fm^{-3} in MSL0)
- Fitted value of gradient coefficient G_S in MQSP: 182 MeV fm^5 (compared to 132 MeV fm^5 in MSL0)
- The 33 studied nuclei span half-lives from approximately 10^{-6} s to 10^2 s, covering eight orders of magnitude
- The new MQSP parameter set yields t_0 = -2134.31 MeV fm^5, t_1 = 529.92 MeV fm^5, t_2 = -187.13 MeV fm^5, t_3 = 13130.28 MeV fm^{3+3alpha}

## Context

This work extends the authors' previous studies (Cheng 2020) that used TPA-SHF with 115 existing Skyrme parameter sets and found SLY7 to be most suitable for proton radioactivity. It builds on the relationship between Skyrme parameters and macroscopic nuclear matter quantities established by Chen et al. (2010, Physical Review C 82, 024321). The new MQSP parameter set is specifically designed for proton radioactivity calculations, unlike existing Skyrme sets that were constructed for ground state properties and symmetric nuclear matter saturation. This work enables future studies to use a dedicated Skyrme parameter set for proton radioactivity calculations and constrains the physically allowed ranges of rho_0 (around 0.156 fm^{-3}) and G_S (about 187 MeV fm^5) based on proton radioactivity data.
