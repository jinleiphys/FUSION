# Event rates for the scattering of weakly interacting massive particles from $^{23}$Na and $^{40}$Ar

> Detection rates for the elastic and inelastic scattering of weakly interacting massive particles (WIMP) off $^{23}$Na are calculated within the framework of Deformed Shell Model (DSM) based on Hartree-Fock states. First the spectroscopic properties like energy spectra and magnetic moments are calculated and compared with experiment. Following the good agreement for these, DSM wave functions are used for obtaining elastic and inelastic spin structure functions, nuclear structure coefficients etc. for the WIMP-$^{23}$Na scattering. Then, the event rates are also calculated with a given set of supersymmetric parameters. In the same manner, using DSM wavefunctions, nuclear structure coefficients and event rates for elastic scattering of WIMP from $^{40}$Ar are also obtained. These results for event rates and also for annual modulation will be useful for the upcoming and future WIMP detection experiments involving detectors with $^{23}$Na and $^{40}$Ar.

## Key claim

The Deformed Shell Model (DSM) calculations for WIMP scattering off ²³Na and ⁴⁰Ar produce event rates and annual modulation patterns that are useful for upcoming experiments, but the results depend on a specific set of supersymmetric parameters and the assumed detector threshold energies, with the nuclear structure coefficients showing peaks that shift to higher WIMP masses as threshold energy increases.

## Method

The authors performed Hartree-Fock calculations to generate deformed intrinsic states for ²³Na (using the USD interaction with a ¹⁶O core and sd-shell orbitals) and ⁴⁰Ar (using the sdpf-u interaction with a ¹⁶O core and seven orbitals). Angular momentum projection and band mixing were applied to obtain wave functions, which were then used to compute spin structure functions, nuclear form factors, and nuclear structure coefficients for elastic scattering (both nuclei) and inelastic scattering (²³Na only, to the 5/2⁺ state at 440 keV). Event rates were calculated by integrating these nuclear structure coefficients over WIMP velocity distributions and momentum transfer, using fixed nucleonic current parameters (f⁰_A=3.55e-2, f¹_A=5.31e-2, f⁰_S=8.02e-4, f¹_S=-0.15×f⁰_S).

## Key numbers

- For ²³Na ground state, static spin matrix elements: Ω₀ = 0.727, Ω₁ = 0.652; calculated magnetic moment = 2.393 nm (experimental = 2.218 nm)
- For ²³Na inelastic scattering to 5/2⁺ state (440 keV excitation), static spin matrix elements: Ω₀ = -0.368, Ω₁ = -0.462
- For ²³Na elastic scattering, peak event rate occurs at m_χ ≈ 30 GeV for Q_thr = 0 keV, shifting to ≈ 50 GeV for Q_thr = 10 keV
- For ⁴⁰Ar elastic scattering, peak event rate occurs at m_χ ≈ 35 GeV for Q_thr = 0 keV, shifting to ≈ 45 GeV for Q_thr = 10 keV
- Oscillator length parameter b = 1.573 fm for ²³Na, b = 1.725 fm for ⁴⁰Ar
- Local WIMP density ρ₀ = 0.3 GeV/cm³, σ₀ = 0.77 × 10⁻³⁸ cm², v₀ = 220 km/s, escape velocity v_esc = 625 km/s

## Context

This work extends the authors' previous DSM calculations for WIMP scattering from ⁷³Ge (Sahu and Kota, 2017) and from ¹²⁷I, ¹³³Cs, and ¹³³Xe (Sahu and Kota, 2020) to the detector nuclei ²³Na and ⁴⁰Ar. It complements large-scale shell-model calculations by Divari et al. (2000) and Klos et al. (2013) for ²³Na, and prior studies of ⁴⁰Ar using shell model and coupled cluster theory. The results are intended to guide experiments such as DAMA/LIBRA, ANAIS, DM-Ice, DARKSIDE-50, and DEAP-3600 in their searches for WIMP dark matter.
