# QCD Evolution by Finite Element Methods

> A simple, new method for solving for the $Q^2$ evolution of parton distributions in perturbative QCD using cubic splines is described and applied to the evolution of nonsinglet quark distributions.

## Key claim

The paper presents a new method for solving the Altarelli-Parisi evolution equations for parton distributions using cubic splines and moment inversion, claiming it is straightforward and stable; however, the method's accuracy depends on using a sufficient number of spline points (N >= 7) and a variable transformation to handle the small-x singularity.

## Method

The method numerically calculates the first N moments of a nonsinglet quark distribution using Gauss-Legendre integration with a variable transformation y = x^α (α < 1) to improve small-x behavior, then inverts the moment-weight matrix to obtain the distribution values at N integration points. The full distribution is reconstructed by cubic spline interpolation between these points, and evolution to a new Q² is performed by applying the leading-order moment evolution formula to the calculated moments before inversion.

## Key numbers

- For a test curve xq_NS(x) = √x (1-x)³, reconstruction with N=3 splines failed, while N=5 showed slight deviation; N=7 and N=10 produced curves "virtually indistinguishable" from the original.
- Leading-order evolution of the valence d quark distribution from Morfin and Tung (1990) was performed from Q₀² = 4 GeV² to Q² = 10 and 50 GeV² using 10 splines with α = 1/2.
- The method uses Gauss-Legendre integration with α = 2/3 for the test case and α = 1/2 for the MT distribution evolution.

## Context

This work extends prior approaches to solving the Altarelli-Parisi equations, which include brute-force integration (Morfin and Tung, 1990), orthogonal polynomial expansions, and Bernstein polynomial methods for next-to-leading order evolution. The finite element method offers advantages over orthogonal polynomial schemes by avoiding global oscillations near singularities, and it is faster than brute-force integration because the moment-weight matrix need only be inverted once for fitting data over a large Q² range. The method is applicable to both singlet and nonsinglet distributions at leading and next-to-leading order.
