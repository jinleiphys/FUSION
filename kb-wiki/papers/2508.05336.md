# Mode-by-mode evolution of Pb-Pb collisions at 5.02 TeV in a hybrid model

> We determine the average state and the uncorrelated modes that characterize the event-by-event fluctuations of the initial state in two typical centrality classes of Pb-Pb collisions at 5.02 TeV. We find that modes in a narrow central bin are similar to those in events at fixed vanishing impact parameter, while those in a mid-peripheral centrality class are affected by the impact-parameter variation. We study how each fluctuation mode affects observables both in the initial state and in the final state of the collisions, at the end of a state-of-the-art boost-invariant hybrid evolution with KoMPoST + MUSIC + iSS + SMASH, and show that implementing a hadronic transport cascade in such a mode-by-mode analysis with reasonable statistical noise is costly but feasible.

## Key claim

The mode decomposition of initial-state fluctuations in Pb-Pb collisions at 5.02 TeV works for centrality classes with varying impact parameter, but the modes in mid-peripheral collisions (30–40%) are significantly affected by impact-parameter variation, unlike those in central collisions (0–2.5%) which resemble events at fixed vanishing impact parameter; implementing a hadronic transport cascade (SMASH) in this mode-by-mode analysis is feasible but computationally costly due to statistical noise.

## Method

The authors generated 2^21 Monte Carlo Glauber initial energy-density profiles for each of two centrality classes (0–2.5% and 30–40%) of Pb-Pb collisions at 5.02 TeV, computed the average state and the fluctuation modes via diagonalization of the autocorrelation matrix of fluctuations, then evolved the average state plus single-mode perturbations through a hybrid model consisting of KoMPoST (pre-equilibrium), MUSIC (viscous hydrodynamics with HotQCD+SMASH equation of state), and either MUSIC freeze-out or iSS+SMASH for hadronic transport. Linear and quadratic response coefficients for initial-state characteristics and final-state observables were extracted from the mode-by-mode evolution using finite-difference formulas.

## Key numbers

- Number of events per centrality class: 2^21 (2,097,152) initial energy-density profiles
- Grid: 192 × 192 points with physical spacing 0.11 fm for initial profiles; 30 fm box with 0.1 fm spacing for dynamical evolution
- MC Glauber parameters: Woods-Saxon half-density radius R = 6.62 fm, diffusivity a = 0.546 fm, minimum nucleon distance 0.4 fm, inelastic nucleon-nucleon cross section 67.6 mb, mixing parameter α = 0.2
- Normalization K-factor: 37.53 GeV/fm² to match ALICE charged-hadron multiplicity
- Initial time τ₀ = 0.2 fm, switching to hydrodynamics at τ_hydro = 1.0 fm
- Specific shear viscosity η/s = 0.16 (constant), particlization temperature T_f.o. = 155 MeV
- Average initial state relative weight w̄ ≈ 13% for both centrality classes
- Eccentricity of average state in 0–2.5% centrality: ε₂ = 0.0174, ε₄ ≈ 10⁻⁴
- Eccentricity of average state in 30–40% centrality: ε₂ = 0.3214, ε₄ = 0.1132 (with Φ₄ = π/4)
- Relative energy content |ε̃₀|_l of mode l=0 in 30–40% centrality: nearly 12% of average state energy; modes l=2 and l=5: close to 8%

## Context

This work extends the mode decomposition framework introduced by Borghini (2022) for fixed impact parameter collisions to realistic centrality classes with varying impact parameter, using a state-of-the-art hybrid evolution (KoMPoST + MUSIC + iSS + SMASH) instead of only hydrodynamics. It enables systematic mode-by-mode studies of how specific initial-state fluctuation patterns propagate to final observables, potentially allowing more efficient Bayesian parameter estimation and better understanding of the relationship between initial geometry and final flow coefficients.
