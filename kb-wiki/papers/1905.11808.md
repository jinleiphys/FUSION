# Bayesian updating for data adjustments and multi-level uncertainty propagation within Total Monte Carlo

> In this work, a method is proposed for combining differential and integral benchmark experimental data within a Bayesian framework for nuclear data adjustments and multi-level uncertainty propagation using the Total Monte Carlo method. First, input parameters to basic nuclear physics models implemented within the state of the art nuclear reactions code, TALYS, were sampled from uniform distributions and randomly varied to produce a large set of random nuclear data files. Next, a probabilistic data assimilation was carried out by computing the likelihood function for each random nuclear data file based first on only differential experimental data (1st update) and then on integral benchmark data (2nd update). The individual likelihood functions from the two updates were then combined into a global likelihood function which was used for the selection of the final 'best' file. The proposed method has been applied for the adjustment of $^{208}$Pb in the fast neutron energy region below 20 MeV. The 'best' file from the adjustments was compared with available experimental data from the EXFOR database as well as evaluations from the major nuclear data libraries and found to compare favourably.

## Key claim

The paper proposes a Bayesian method that explicitly combines differential experimental data (from EXFOR) and integral benchmark data (from ICSBEP) into a single global likelihood function for nuclear data adjustments within the Total Monte Carlo framework, applied to n+208Pb below 20 MeV; the method assumes differential and integral data are independent and that the nuclear models used are "good enough" without correction for model defects.

## Method

Input parameters to the TALYS nuclear reaction code were sampled from uniform distributions and randomly varied to produce 2700 random nuclear data files for 208Pb. A first Bayesian update assigned file weights based on a reduced chi-square computed from selected differential experimental data (reaction channels: total, non-elastic, inelastic, (n,2n), and (n,gamma) between 5 and 20 MeV). A second Bayesian update assigned file weights based on a single integral benchmark (hmf57c1, a highly enriched uranium metallic fast benchmark) using the calculated keff versus the experimental benchmark keff. The individual likelihood functions from the two updates were multiplied to form a global likelihood function, and the file maximizing this combined weight was selected as the "best" file.

## Key numbers

* 2700 random nuclear data files were produced for 208Pb by varying 28 model parameters within assigned widths.
* After applying a threshold weight (wthres = 2.06e-09, corresponding to a critical chi-square of 40 at 95% confidence with 28 degrees of freedom), 2046 files were accepted, a 24% reduction from the initial 2700 files.
* The hmf57c1 benchmark used for the second update has an experimental benchmark keff of 1.0000 with an experimental benchmark uncertainty of 200 pcm.
* The integral benchmarks used include: PMF035 case 1 (keff = 1.0000, uncertainty 160 pcm), HMF027 case 1 (keff = 1.0000, uncertainty 250 pcm), HMF057 cases 1-5 (keff = 1.0000, uncertainties 200-400 pcm), HMF064 case 1 (keff = 0.9996, uncertainty 80 pcm), LCT010 case 1 (keff = 1.0000, uncertainty 210 pcm), and LCT010 case 20 (keff = 1.0000, uncertainty 280 pcm).
* Differential experimental data used include: (n,tot) 142 data points from Carlton (1991) and 116 points from Foster (1971); (n,inl) 46 points from Mihailescu (2008); (n,2n) 16 points from Frehaut (1980); (n,gamma) 8 points from Bergqvist (1972), 7 points from Csikai (1967), and 1 point from Drake (1971).

## Context

This work extends the Total Monte Carlo method developed by Koning (2008) by explicitly incorporating both differential and integral experimental data into a single Bayesian updating scheme, whereas prior TMC-based adjustments used either only integral benchmark data (as in the "Petten method" applied by Rochman (2012) for 239Pu and 63,65Cu) or only differential data (as in the work of Helgesson (2017) and Koning (2015)). The method enables multi-level uncertainty propagation where lower-level nuclear physics parameters are constrained by both microscopic and macroscopic measurements, and the resulting adjusted files can be tested against additional benchmarks not used in the adjustment (such as the other nine benchmarks listed in the paper).
