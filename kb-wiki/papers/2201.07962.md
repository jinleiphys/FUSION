# Event-by-event investigation of the two-particle source function in heavy-ion collisions with EPOS

> Exploring the shape of the pair-source function for particles such as pions or kaons has been an important goal of heavy-ion physics, and substantial effort has been made in order to understand the underlying physics behind the experimental observations of non-Gaussian behavior. In experiments, since no direct measurement of the source function is possible, quantum-statistical momentum correlations are utilized to gain information about the space-time geometry of the particle emitting source. Event generators, such as EPOS, however, provide direct access to the freeze-out coordinates of final state particles, and thus the source function can be constructed and investigated. The EPOS model is a sophisticated hybrid model where the initial stage evolution of the system is governed by Parton-Based Gribov-Regge theory, and subsequently a hydrodynamic evolution is utilized, followed by hadronization and hadron dynamics. EPOS has already proven to be successful in describing several different experimental observations for systems characterized by baryon chemical potential close to zero, but so far the source shape has not been explored in detail. In this paper we discuss an event-by-event analysis of the two-particle source function in $\sqrt{s_{NN}}$ = 200 GeV Au+Au collisions generated by the EPOS model. We find that when utilizing all stages of the model, Lévy-shaped distributions (unlike Gaussian distributions) provide a good description of the source shape in the individual events. Hence it is clear that it is not the event averaging that creates the non-Gaussian features in the pair distributions. Based on this observation, we determine Lévy-parameters of the source as a function of event centrality and particle momentum.

## Key claim

Using the EPOS event generator for sqrt(s_NN) = 200 GeV Au+Au collisions, the two-particle source function in individual events is well described by Levy-shaped distributions rather than Gaussian distributions, demonstrating that the non-Gaussian features observed in experimental pair-source functions are not caused by event averaging but are intrinsic to each event.

## Method

The authors performed an event-by-event analysis of the two-particle source function using the EPOS hybrid model, which combines Parton-Based Gribov-Regge theory for the initial stage with hydrodynamic evolution, hadronization, and hadron dynamics. They directly accessed the freeze-out coordinates of final-state particles from the event generator to construct the pair-source function, then fitted Levy-stable distributions to the source shape in individual events. The Levy parameters (alpha, R, and lambda) were extracted as functions of event centrality and particle momentum.

## Key numbers

- The analysis was performed for Au+Au collisions at sqrt(s_NN) = 200 GeV.
- Levy-shaped distributions (not Gaussian) provided a good description of the source shape in individual events when utilizing all stages of the EPOS model.
- The paper reports that non-Gaussian features in pair distributions are present at the single-event level, contradicting the hypothesis that event averaging creates these features.

## Context

This work extends previous studies of the pair-source function in heavy-ion collisions, where experiments have observed non-Gaussian behavior in quantum-statistical momentum correlations for pions and kaons. The EPOS model had previously been successful in describing various experimental observables for systems with baryon chemical potential near zero, but its source shape had not been explored in detail. The finding that Levy-shaped sources appear at the single-event level enables future comparisons between event generator predictions and experimental correlation measurements, and motivates further theoretical work on the physical origin of the Levy shape in the emission source.
