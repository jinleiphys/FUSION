# Short-range YN interactions in the Quark Cluster Model

> A phenomenological model for the hyperon-nucleon interactions is constructed by using the quark cluster model approach to the short-distance baryon-baryon interactions. The model contains the SU(3) symmetric meson exchange interaction at large distances and the quark-exchange short-distance interaction. The main feature of the model is that strong channel dependences of the short range repulsions due to the quark model symmetry. It is pointed out that two channels, ($I$, $S$)= (1/2, 0) and (3/2, 1), of the S-wave sigma-nucleon interactions have extremely strong repulsions at short-distances.

## Key claim

The paper constructs a phenomenological hyperon-nucleon (YN) interaction model that combines SU(3)-symmetric meson exchange at long distances with quark-exchange short-distance interactions from the quark cluster model, but the model is not yet complete and still borrows meson-baryon couplings from the Nijmegen potential rather than determining them from first principles.

## Method

The authors employed the resonating group method (RGM) with a six-quark wave function that includes full antisymmetrization among valence quarks, using a nonrelativistic quark Hamiltonian with confinement and one-gluon exchange. They added the meson-exchange part of the Nijmegen model D potential (without its hard core) to the quark cluster model equation, folding it with Gaussian quark density form factors. The five model parameters (light quark mass, strange/light quark mass ratio, confinement strength, gluon coupling constant, and quark size parameter) were fixed using baryon masses and NN scattering data, with the size parameter b=0.56 fm chosen to fit the NN 1S0 phase shift.

## Key numbers

- Light quark mass set to 383.7 MeV (one-third of the average octet baryon mass)
- Light/strange quark mass ratio fixed to 0.6 to reproduce the Lambda-Sigma mass difference
- Quark size parameter b = 0.56 fm (determined from NN 1S0 scattering data)
- Confinement strength a = 20.8 MeV/fm
- Gluon coupling constant alpha_s = 1.85
- Smallest normalization kernel eigenvalue e = 1/9 for N-Sigma (S=0, I=1/2) channel
- Smallest normalization kernel eigenvalue e = 2/9 for N-Sigma (S=1, I=3/2) channel
- Effective core radius 0.68 fm for N-Sigma (S=0, I=1/2) channel
- Effective core radius 0.67 fm for N-Sigma (S=1, I=3/2) channel
- Effective core radius 0.40 fm for N-Lambda (S=0, I=1/2) channel
- Effective core radius 0.34 fm for N-Lambda (S=1, I=1/2) channel

## Context

This work extends the quark cluster model approach previously applied successfully to NN interactions by Oka and Yazaki (1980) and the Nagoya group, applying it to hyperon-nucleon systems. The model contrasts with conventional meson-exchange potentials like the Nijmegen and Julich models, which treat short-range repulsion with hard cores and many arbitrary parameters. The results suggest that two specific Sigma-N channels have exceptionally strong repulsions due to Pauli exclusion among valence quarks, which may make bound Sigma hypernuclei implausible and provides predictions for double-strange systems (Lambda-Lambda, N-Xi, and the H dibaryon) that are not yet experimentally accessible.
