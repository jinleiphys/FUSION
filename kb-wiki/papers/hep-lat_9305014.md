# Undesirable effects of covariance matrix techniques for error analysis

> Regression with $χ^2$ constructed from the covariance matrix should not be used for some combinations of covariance matrices and fitting functions. Using the technique for unsuitable combinations can amplify systematic errors. This amplification is uncontrolled, and can produce arbitrarily inaccurate results that might not be ruled out by a $χ^2$ test. In addition, this technique can give incorrect (artificially small) errors for fit parameters. I give a test for this instability and a more robust (but computationally more intensive) method for fitting correlated data.

# Undesirable effects of covariance matrix techniques for error analysis

## Key claim
Covariance matrix regression can amplify systematic errors in an uncontrolled way and produce artificially small error estimates for fit parameters, but this instability only occurs for certain combinations of covariance matrices and fitting functions; the paper provides a test to identify when the technique is unsafe.

## Method
The author constructs a gedanken experiment with two correlated measurements of a random voltage, first without and then with a systematic error (capacitor discharge between measurements). He compares the standard covariance matrix regression approach to a more robust best-estimator technique that imposes non-negativity constraints on the linear combination coefficients. He derives a general stability condition requiring that the partial derivative of the fitted function with respect to each data point be non-negative, and shows how to compute this condition for arbitrary fitting functions.

## Key numbers
- For the unstable regression with systematic error, the fitted value V can be as low as zero while the true mean is $\overline{v}$, whereas a naive least-squares fit always yields a value between $\gamma \overline{v}$ and $\overline{v}$.
- In the limit $(\gamma-1)\sigma \gg e_1, e_2$, the regression gives $\sigma_V^2 \rightarrow (\gamma^2 e_1^2+e_2^2)/[(N-1)(\gamma-1)^2]$, which can be much smaller than $\sigma^2/(N-1)$.
- For the unstable case with $\sigma \rightarrow \infty$, $\chi^2 \rightarrow (N-1)(\overline{v}/\sigma)^2$, so fits are acceptably small when $N < (\sigma/\overline{v})^2$.
- The stability condition for fitting a constant is $\forall i: \sum_j (C^{-1})_{ij} \geq 0$.

## Context
This paper addresses a problem identified in lattice gauge theory analyses by DeGrand (1987), DeTar and Kogut (1987), and Gottlieb et al. (1988), as well as in scaled factorial moment analyses by Abreu et al. (1990) and Wosiek (1990), where covariance matrix regression produced best-fit lines falling below all data points and error bars. The work extends the standard statistical methods described in Eadie et al. (1971) by providing a stability test and a more robust alternative fitting procedure. The results enable practitioners to avoid uncontrolled systematic error amplification when fitting correlated data.
