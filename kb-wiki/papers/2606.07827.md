# Improving the Predictive Capability of the Fission Reaction Event Yield Algorithm ($\mathtt{FREYA}$)

> We determine the optimal parameters for thermal neutron-induced fission of $^{233,235}$U and $^{239,241}$Pu in the complete event fission model $\mathtt{FREYA}$. First, we revisit and, in most cases, improve the prior values determined for spontaneous fission using a genetic algorithm. Our optimization procedure is then applied to thermal neutron-induced fission, replacing the empirically-chosen parameters previously employed in $\mathtt{FREYA}$. Finally, the optimized parameter values are used to make predictions for spontaneous and thermal neutron-induced fission observables not included in the fits. This work represents the first step in a broader program to study neutron-induced fission as a function of incident neutron energy, to systematically improve the performance of $\mathtt{FREYA}$, capturing any energy-dependent trends of the physics-based $\mathtt{FREYA}$ parameters.

## Key claim

The authors determine optimal parameter sets for thermal neutron-induced fission of ²³³,²³⁵U and ²³⁹,²⁴¹Pu in the FREYA model using a genetic algorithm, replacing the previously hand-tuned or energy-independent default parameters; however, the optimization is limited to thermal incident energies and represents only the first step toward a systematic study of energy-dependent trends.

## Method

The five physics-based FREYA parameters (e₀, x, c, c_S, and dTKE) were optimized by minimizing a total χ² computed from comparisons between one-million-event FREYA simulations and experimental or evaluated data for each fissioning system. A genetic algorithm with a population of 50 individuals, uniform crossover, and adaptive mutation rates was employed, running 10 independent optimizations per isotope to form an ensemble. The optimized parameters were then used to predict observables not included in the fits, such as neutron-neutron angular correlations and prompt fission neutron spectra.

## Key numbers

- For ²⁵²Cf(sf), the genetic algorithm reproduced the simulated annealing results of Van Dyke (2019) and typically improved the parameter values; the same data sets were used with the addition of PFNS data from Gerasimenko (2002) for ²⁴⁰,²⁴²Pu(sf)
- Parameter ranges adopted from Van Dyke (2019): e₀ = [7, 12] MeV⁻¹, x = [1.0, 1.5], c = [1, 3], c_S = [0.5, 1.5], dTKE = [-5, 5] MeV
- The genetic algorithm used a gene sequence length of f = 58 bits to achieve three significant figures in optimized parameters, with population size p = 50, crossover rate c = 0.80, mutation rate m = 1/58, and g = 50 generations
- For ²⁴⁰Pu(sf), including n–n correlation data from Verbeke (2018) caused significant tension with PFNS data and was excluded from the final fit; for ²⁵²Cf(sf), adding n–n correlation data from Gagarski (2008) did not change fitted parameters within uncertainties

## Context

This work extends the spontaneous fission parameter optimization of Van Dyke (2019), which used simulated annealing, by applying a genetic algorithm to both spontaneous and thermal neutron-induced fission for the first time. The results replace the empirically-chosen, often energy-independent parameters previously used in FREYA (Verbeke, 2018) and enable systematic future studies of neutron-induced fission as a function of incident neutron energy, capturing energy-dependent trends of the physics-based parameters. The approach is designed to be parallelizable on high-performance computing platforms, unlike the serial simulated annealing method, facilitating broader energy-dependent studies.
