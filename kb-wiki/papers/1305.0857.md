# On description of the correlation between multiplicities in windows separated in azimuth and rapidity

> The forward-backward (FB) multiplicity correlations in two windows separated in rapidity and azimuth are analyzed in the framework of the model with independent identical emitters (strings). Along with the short-range contribution, originating from the correlation between particles produced by a single string, the long-range contribution, originating from the fluctuation in the number of strings, is taken into account. The connection of the FB multiplicity correlation coefficient with the two-particle correlation function and the di-hadron correlation analysis is traced. It's also shown that the direct azimuthal flow leads to the forward ridge structure in the resulting two-particle correlation function.

## Key claim

The forward-backward multiplicity correlation coefficient, when measured in two small windows separated in azimuth and rapidity, is equivalent to the standard two-particle correlation function up to a factor depending on window width, and the constant pedestal in the two-particle correlation function contains physical information on the event-by-event fluctuation in the number of emitters (strings), not just the short-range correlations from a single string.

## Method

The author analyzes forward-backward multiplicity correlations using a two-stage model with independent identical emitters (strings), where the number of strings fluctuates event-by-event and each string produces particles with its own one- and two-particle densities. The connection between the traditional FB correlation coefficient and the two-particle correlation function is derived, and the model is extended to include azimuthal dependence by introducing Fourier decomposition of the azimuthal inclusive distribution for each string configuration, with averaging over string configurations and over random reaction-plane phases.

## Key numbers

- The scaled variance of the number of strings is defined as ω_N = D_N / ⟨N⟩, where D_N = ⟨N²⟩ − ⟨N⟩²; this quantity is noted to be non-Poissonian (ω_N ≠ 1) in pp and especially AA collisions.
- The two-particle correlation function in the model with strings is given by C₂(Δη, Δϕ) = [Λ(Δη, Δϕ) + ω_N] / ⟨N⟩, where Λ is the single-string correlation function.
- The pedestal constant in C₂ is C₀ = (⟨(ρ̄ⁱ)²⟩ − ⟨ρ̄ⁱ⟩²) / ⟨ρ̄ⁱ⟩², equal to the variance of the mean multiplicity per string configuration divided by the square of the averaged multiplicity.
- Under the approximation that mean multiplicity per configuration is constant, the ridge structure becomes C₂(Δϕ) = 2 Σₙ ⟨(vⁱₙ)²⟩ cos(n Δϕ), expressed through mean-squared flows v^{ms}_n = √[⟨(vⁱₙ)²⟩].

## Context

This work extends the string model approach previously developed by the author and collaborators (PLB00, EPJC04, YF1) and connects it to the di-hadron correlation analysis used by STAR (STARdihadr09) and CMS (CMSridge10). The paper shows that the pedestal in the two-particle correlation function, emphasized in earlier work (FB13, CapKrz), carries information on fluctuations in the number of emitters, and demonstrates that direct azimuthal flow v₁ leads to a forward ridge structure even without intrinsic two-particle correlations from a single string. This enables future experimental extraction of the fluctuation magnitude of the number of emitters from the pedestal height in correlation measurements.
