# Real-time Spin Systems from Lattice Field Theory

> We construct a lattice field theory method for computing the real-time dynamics of spin systems in a thermal bath. This is done by building on previous work of Takano with Schwinger-Keldysh and functional differentiation techniques. We derive a Schwinger-Keldysh path integral for generic spin Hamiltonians, then demonstrate the method on a simple system. Our path integral has a sign problem, which generally requires exponential run time in the system size, but requires only linear storage. The latter may place this method at an advantage over exact diagonalization, which is exponential in both. Our path integral is amenable to contour deformations, a technique for reducing sign problems.

## Key claim
The paper constructs a lattice field theory method for computing real-time dynamics of spin systems in a thermal bath that, while requiring exponential runtime in system size, requires only linear storage; this may place it at an advantage over exact diagonalization, which is exponential in both runtime and storage.

## Method
The method builds on Takano's (1980s) framework using spin coherent states and a Schwinger-Keldysh path integral discretized in both space and time. The path integral is derived by expressing short-time propagators in terms of spin coherent states via a diagonalization theorem, then using functional differentiation to insert observables rather than Takano's product formula. Correlation functions are obtained by taking derivatives of the path integral with respect to classical sources and taking the time continuum limit.

## Key numbers
- The path integral reproduces exact real-time correlation functions up to errors of O(dt, dtau) that vanish in the time continuum limit.
- For a two-qubit system with beta = 3.0 j^{-1} and t_max = 10 j^{-1}, the method was demonstrated using N=5000 timeslices.
- Continuum extrapolation errors for the finest fit window {10000,12500,15000} timeslices were: Re<s_x(x,t)s_x(x,0)> = -4.0e-7, Im<s_x(x,t)s_x(x,0)> = 2.5e-7, Re<s_x(x,t)s_x(x+1,0)> = -2.2e-7, Im<s_x(x,t)s_x(x+1,0)> = 2.6e-7 (all in units of j^{-2}).

## Context
This work extends Takano's (1980s) Monte Carlo method for thermal properties of spin systems to real-time dynamics, using functional differentiation to avoid Takano's product formula that becomes prohibitively difficult for complex Hamiltonians and observables. The method provides an alternative to exact diagonalization (exponential in both storage and runtime) and to tensor networks, which are generally less efficient in more than one spatial dimension as noted by Orus (2014) and Orus (2019). The path integral is amenable to contour deformations, a technique for reducing sign problems that has been applied to other real-time problems by Alexandru et al. (2016) and Alexandru et al. (2017).
