# Elliptic flow from event-by-event hydrodynamics with fluctuating initial state

> We develop an event-by-event ideal hydrodynamical framework where initial state density fluctuations are present and where we use a similar flow-analysis method as in the experiments to make a one-to-one $v_2$ comparison with the measured data. Our studies also show that the participant plane is quite a good approximation for the event plane.

## Key claim

Event-by-event ideal hydrodynamics with fluctuating initial states can reproduce the measured centrality- and $p_T$-dependence of elliptic flow $v_2(p_T)$ at RHIC for all centrality classes from 0-5% to 30-40% up to $p_T \sim 2$ GeV, but only when the elliptic flow is calculated with respect to the event plane rather than the reaction plane; the reference plane definition is crucial for matching data.

## Method

The authors generate fluctuating initial energy-density profiles using a Monte Carlo Glauber model, distributing energy around wounded nucleons with a 2D Gaussian smearing function of width $\sigma$. They solve (2+1)-dimensional ideal hydrodynamic equations with longitudinal boost-invariance and the Laine (2006) equation of state, then compute thermal hadron spectra via the Cooper-Frye method at a decoupling temperature of 160 MeV, letting unstable hadrons decay through PYTHIA 6.4. Elliptic flow is calculated both with respect to the reaction plane and using the event-plane method with two-subevent resolution correction, mimicking experimental flow-analysis techniques.

## Key numbers

- Au+Au collisions at $\sqrt{s_{NN}} = 200$ GeV; inelastic nucleon-nucleon cross section $\sigma_{NN} = 42$ mb.
- Initial time $\tau_0 = 0.17$ fm, taken from the EKRT pQCD+saturation model (Eskola 1999).
- Decoupling temperature $T_{\rm dec} = 160$ MeV.
- Gaussian smearing width $\sigma = 0.4$ fm used for the main $v_2$ results.
- Event-by-event $v_2\{{\rm EP}\}$ reproduces measured data for all centrality classes from 0-5% to 30-40% up to $p_T \sim 2$ GeV.

## Context

This work extends earlier hydrodynamical calculations based on averaged initial conditions (e.g., Niemi 2008) which systematically underestimated $v_2(p_T)$ in central collisions. The authors show that the deficit is resolved not by fluctuations alone but by using the event-plane method (Poskanzer 1998) rather than the reaction plane, and that the participant plane is a good approximation for the event plane. This enables more reliable extraction of shear viscosity from measured $v_2$ data by clarifying the role of the reference plane in flow analyses.
