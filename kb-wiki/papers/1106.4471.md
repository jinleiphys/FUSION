# Elliptic flow from event-by-event hydrodynamics

> We present an event-by-event hydrodynamical framework which takes into account the initial density fluctuations arising from a Monte Carlo Glauber model. The elliptic flow is calculated with the event plane method and a one-to-one comparison with the measured event plane $v_2$ is made. Both the centrality- and $p_T$-dependence of the $v_2$ are remarkably well reproduced. We also find that the participant plane is a quite good approximation for the event plane.

# Elliptic flow from event-by-event hydrodynamics

## Key claim
Event-by-event hydrodynamics with initial density fluctuations from a Monte Carlo Glauber model reproduces the centrality- and $p_T$-dependence of the elliptic flow $v_2$ measured via the event plane method, but only when the reference plane definition (event plane vs. participant plane vs. reaction plane) is properly accounted for; the participant plane is a "quite good approximation" for the event plane, but the reaction plane result is close to the smooth initial-state result.

## Method
The authors generate initial energy density profiles event-by-event using a Monte Carlo Glauber model with a Woods-Saxon nucleon distribution and a 2D Gaussian smearing (width parameter $\sigma$) around each participant nucleon, then solve ideal boost-invariant hydrodynamics with an equation of state from Laine (2006) and freeze-out at $T_F = 160$ MeV. The elliptic flow $v_2$ is computed using the event plane method with the 2-subevent correction $\mathcal{R}$, and compared to results with respect to the participant plane and the reaction plane; 500 hydro runs per centrality class are performed, with each thermal hadron spectrum sampled 20 times.

## Key numbers
- Inelastic nucleon-nucleon cross section: $\sigma_{NN} = 42$ mb at $\sqrt{s_{NN}} = 200$ GeV
- Initial time: $\tau_0 = 0.17$ fm
- Freeze-out temperature: $T_F = 160$ MeV
- 500 hydro runs per centrality class, with each thermal hadron spectrum sampled 20 times
- Smooth initial states obtained by averaging over 20,000 initial energy density profiles

## Context
This work extends the event-by-event hydrodynamical framework introduced in Holopainen (2010) to address the failure of smooth initial-state hydrodynamics to reproduce the centrality dependence of event-plane $v_2$, as noted in Niemi (2008). The one-to-one comparison with PHENIX and STAR data (Afanasiev 2009, Adare 2010, Adams 2004) demonstrates that initial density fluctuations alone do not generate more elliptic flow; rather, the choice of reference plane (event plane vs. participant plane vs. reaction plane) is critical. This enables future studies to disentangle initial-state geometry from hydrodynamic response in heavy-ion collisions.
