# MCNP6 Study of Fragmentation Products from 112Sn + 112Sn and 124Sn + 124Sn at 1 GeV/nucleon

> Isotope production cross sections from 112Sn + 112Sn and 124Sn + 124Sn reactions at 1 GeV/nucleon, which were measured recently at GSI using the heavy-ion accelerator SIS18 and the Fragment Separator (FRS), have been analyzed with the latest Los Alamos Monte-Carlo transport code MCNP6 using the LAQGSM03.03 event generator. MCNP6 reproduces reasonably well all the measured cross sections. Comparison of the MCNP6 results with the measured data and with calculations by a modification of the Los Alamos version of the Quark-Gluon String Model allowing for multifragmentation processes in the framework of the Statistical Multifragmentation Model (SMM) by Botvina and coauthors, as realized in the code LAQGSM03.S1, does not suggest unambiguous evidence of a multifragmentation signature.

## Key claim

MCNP6 with the LAQGSM03.03 event generator reproduces all measured isotope production cross sections reasonably well, but comparisons with calculations that include multifragmentation via the Statistical Multifragmentation Model (SMM) do not provide unambiguous evidence of a multifragmentation signature in these reactions.

## Method

The authors used the Los Alamos Monte-Carlo transport code MCNP6 with the LAQGSM03.03 event generator to calculate isotope production cross sections for 1 GeV/nucleon 112Sn + 112Sn and 124Sn + 124Sn reactions, comparing results from sequential and parallel (MPI) MCNP6 runs as well as from LAQGSM03.03 used as a stand-alone code. They also performed calculations with a modified version, LAQGSM03.S1, which incorporates multifragmentation via the SMM for nuclei with excitation energies E/A ≥ 4 MeV/nucleon, and compared all results against GSI measurements from the SIS18 accelerator and FRS separator.

## Key numbers

- 112Sn + 112Sn and 124Sn + 124Sn reactions at 1 GeV/nucleon were studied, using data measured at GSI with the SIS18 accelerator and Fragment Separator (FRS)
- LAQGSM03.S1 calculations used a multifragmentation transition condition at excitation energy E/A = 4 MeV/nucleon (with the transition believed to occur between ~3 and ~5 MeV/nucleon)
- MCNP6 results from sequential and parallel (MPI) modes coincided with each other, and MCNP6 results using LAQGSM03.03 were practically identical to stand-alone LAQGSM03.03 calculations, with only a tiny uniform difference in absolute cross-section values due to different total reaction cross-section normalizations

## Context

This work extends validation of the MCNP6 transport code and its LAQGSM03.03 event generator to heavy-ion fragmentation reactions, addressing a gap where previous validation focused mostly on particle spectra rather than isotope-production yields. The study tests the same reactions against the multifragmentation-inclusive LAQGSM03.S1 version, following similar conclusions by the authors in a prior analysis of other heavy-ion reactions (Mashnik and Sierk, 2012) and by Mancusi et al. (2011) for 1-GeV proton-nucleus reactions. The results enable improved modeling of relativistic heavy-ion collisions for applications in nuclear physics and transport simulations.
