# Variation after projection calculations for high-spin states

> In this paper, an improvement has been made on the variation after projection (VAP) method, which is crucial in the calculations of high-spin states. It turns out that, the form of the trial VAP wave function with spin $J$ can be simplified by adopting just one projected state rather than previously adopting all $(2J+1)$ angular momentum projected states for each selected reference state, $|Φ\rangle$. The present calculations show that such simplification still minimizes the angular momentum projected energy up to a very good approximation. In this simplified VAP scheme, one can obtain almost equivalent VAP wave functions starting from different sets of the projected basis states. This clearly shows that a nuclear state can not be identified with a single intrinsic state, while in the traditional nuclear collective models, an intrinsic state is usually assigned to a well deformed rotational band.

## Key claim

The paper claims that the variation after projection (VAP) wave function for a state of spin \(J\) can be simplified to use only one angular-momentum-projected state per reference Slater determinant, rather than all \((2J+1)\) projected states, and that this simplification still minimizes the projected energy to a very good approximation. The hedge is that this equivalence holds "up to a very good approximation" and is demonstrated numerically for specific cases, not proven rigorously.

## Method

The author performs VAP calculations using Slater determinants (SDs) as reference states in a single harmonic-oscillator shell model space. The trial wave function is taken as a single \(K\)-projected state \(P^J_{MK}|\Phi\rangle\) (normalized), and its energy \(E^J(K)\) is minimized independently for each \(K\). For non-yrast states, the wave function is generalized to a sum over \(n\) SDs, each projected with the same \(K\), and the sum of the lowest \(m\) energies is minimized with two constraint terms to ensure stability. The results are compared to exact shell-model diagonalizations using the USDB and KB3 interactions.

## Key numbers

* For the yrast \(2^+\) and \(7^+\) states in \(^{24}\)Mg, all \(2J+1\) independently minimized energies \(E^J(K)\) are "almost the same" as the VAP energy \(E^J_1\) obtained from the full \((2J+1)\)-state basis; the worst overlap between any two converged wave functions \(|\Psi_{JM}(K)\rangle\) and \(|\Psi_{JM}(K')\rangle\) is "still over 0.98".
* For the lowest five \(J^\pi=8^+\) states in several \(sd\)-shell nuclei (even-even, odd-odd, and odd-mass), VAP calculations with \(K=0\) and \(K=8\) yield energies that "perfectly coincide" and overlaps \(|\langle\Psi_{JM\alpha}(K=0)|\Psi_{JM\alpha}(K=8)\rangle| \geq 0.95\) (most above 0.98).
* For yrast states in \(^{48}\)Cr up to spin \(J=12\), VAP calculations with \(K=0\) and \(K=J\) give energies and B(E2) values that are "very close" to shell-model results; the overlap \(|\langle\Psi_{JM1}(K=0)|\Psi_{JM1}(K=J)\rangle|\) is above 98% for all spins.

## Context

This work extends the VAP method of Tuya et al. (2017) and Wang et al. (2018) by showing that the \(K\)-mixing traditionally required in angular-momentum-projected VAP (as in the Hill-Wheeler equation) can be omitted without significant loss of accuracy. The simplification makes VAP calculations for high-spin states as straightforward as those for \(J=0\) states. The results contradict the traditional nuclear collective model assumption that a well-deformed rotational band can be assigned a single intrinsic state; instead, the author finds that a nuclear state cannot be identified with a single intrinsic state. The method enables future VAP calculations for arbitrary high-spin states and suggests a possible way to define an intrinsic state by minimizing the sum of projected energies of band members from a common reference state.
