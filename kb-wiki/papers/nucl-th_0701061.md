# Long-range behaviour in hyperspherical formalism

> It is shown that hyperspherical harmonics can be represented in a form typical of traditional microscopic cluster models. This allows those hyperharmonics that are responsible for long-range behaviour of valence nucleons in loosely-bound nuclei to be selected. The hyperspherical cluster model based on such hyperharmonics is tested for $^5$He using a simplified description of the $^4$He core and two versions of the Volkov potential which give $^5$He as either bound or unbound. The study has confirmed that it is possible to get a converged solution for binding energy, r.m.s. radius, overlap integral and asymptotic normalization coefficient with small number of basis functions for bound $^5$He. The values obtained of these quantities are very close to those calculated in a microscopic cluster model.

## Key claim

The paper demonstrates that a restricted subset of hyperspherical harmonics, called hyperspherical cluster harmonics (HCH), can reproduce the long-range behaviour of a weakly-bound valence nucleon in a cluster nucleus, achieving convergence for bound $^5$He with only a small number of basis functions, but only when the Majorana parameter of the Volkov potential is tuned to a non-standard value ($m=0.3$) that makes the nucleus bound.

## Method

The author constructs hyperspherical cluster harmonics by antisymmetrizing a product of a hyperspherical harmonic for the $A-1$ core with a relative angular function describing the valence nucleon, then expands the total wave function in this basis. The hyperradial coupled differential equations of the hyperspherical functions method are solved using the computer code STURMXX, with matrix elements calculated via a Laplace transform technique that connects the hyperspherical basis to oscillator shell model cluster wave functions. The method is tested on $^5$He using the Volkov V1 effective NN potential with two values of the Majorana parameter ($m=0.6$ and $m=0.3$), keeping only the lowest-order hyperspherical harmonic for the $^4$He core.

## Key numbers

- For bound $^5$He with $m=0.3$ and twelve basis functions, the binding energy converges to $E = -30.66$ MeV and the r.m.s. radius converges to $\langle r^2 \rangle^{1/2} = 1.96$ fm.
- For $K_{\max} = 9$, the HCM calculation uses only five basis states while the full HSFM calculation uses 117 basis states, with a difference in binding energy of about 1.5 MeV between the two approaches.
- The r.m.s. radius of the overlap integral $\langle ^4\text{He} \otimes n | ^5\text{He} \rangle$ converges to 3.82 fm with twelve HCH functions.
- The overlap integral $I_l(r)$ converges for $r \leq 6$ fm with twelve HCH functions, and the ratio $i I_l(r) / \kappa h_1^{(1)}(i\kappa r)$ converges at $5 \leq r \leq 7$ fm, enabling determination of the asymptotic normalization coefficient.

## Context

This work extends the hyperspherical interpolation approach proposed by B. Zh. (year not given in text) and earlier applications to $^4$He+n elastic scattering by Efros and Zh. (year not given), as well as three-body model studies of $^8$B and $^6$Li by G. (year not given). The method addresses the long-standing problem that ab-initio calculations, such as the no-core shell model overlap calculations for $^8$B by Navratil (2006) and variational Monte-Carlo overlaps by Nollett (2001), fail to reproduce correct asymptotic behaviour at distances beyond 5-6 fm, requiring artificial matching to Whittaker functions or Woods-Saxon potentials. The HCH approach enables converged solutions for binding energies, radii, overlap integrals, and asymptotic normalization coefficients with small basis sizes, and is designed to be extendable to heavier weakly-bound nuclei and to realistic NN interactions beyond the effective Volkov potential used in this test.
