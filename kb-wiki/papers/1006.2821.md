# Relation Between a Three Parameter Formula for Isotope Shifts and Staggering Parameters

> It is noted that the staggering parameters used to describe even-odd effects for isotope shifts can in some cases exhibit very rapidly varying behavior as a function of neutron number. On the other hand a three parameter formula (3P) with fixed coefficients can explain the same behaviour.

# Relation Between a Three Parameter Formula for Isotope Shifts and Staggering Parameters

## Key claim
The three parameter formula (3P) with fixed coefficients can reproduce the rapidly varying even-odd staggering behavior of isotope shifts as a function of neutron number, even though the staggering parameter gamma can exhibit large negative dips and returns to positive values that might appear mysterious without this framework.

## Method
The author derives an analytic relation between the staggering parameter gamma_n and the three parameters A, B, C of the 3P formula r^2(n) = nC + n(n-1)/2 A + [n/2]B, showing that gamma_n depends only on two independent constants y = (A+B)/2C and x = A/C. Staggering parameters are then calculated for Ar, K, Ca, and Ti isotopes using the 3P formula with parameters compiled from Blaum (2008), and compared with experimental staggering parameters obtained directly from measured isotope shifts.

## Key numbers
- For Ar isotopes using the 3P formula: gamma_0 = 0.388, gamma_2 = -0.086, gamma_4 = -3.805, gamma_6 = 2.908
- For K isotopes using the 3P formula: gamma_0 = 0.660, gamma_2 = 0.460, gamma_4 = -4.000, gamma_6 = 1.853
- For Ca isotopes using the 3P formula: gamma_0 = 0.052, gamma_2 = -1.844, gamma_4 = 3.844, gamma_6 = 1.948
- For Ti isotopes using the 3P formula: gamma_0 = -0.387, gamma_2 = -128.000, gamma_4 = 2.418, gamma_6 = 1.705
- Experimental staggering parameters for Ar: gamma_0 = 0.527, gamma_2 = -0.337, gamma_4 = -3.037, gamma_6 = 1.461 (with 3P formula used for 45Ar where no measurement exists)
- Experimental staggering parameters for K: gamma_0 = 0.274, gamma_2 = -1.385, gamma_4 = -0.174, gamma_6 = 1.278
- For argon, the 3P parameters from Blaum (2008) are A = -0.018(1), B = 0.119(13), C = 0.032(5)
- B(E2) values for even-even Ar isotopes from Robinson (2009): 130(10) e^2 fm^4 for A=38, 330(40) for A=40, 430(100) for A=42, 345(41) for A=44, and 196(39) for A=46

## Context
This work extends the three parameter formula originally developed by Zamick (1971) and Talmi (1984) for isotope shifts in a single j shell, showing that the formula also describes the staggering parameter behavior for Ar, K, Ca, and Ti isotopes. The author notes that more fundamental approaches have difficulties: Uhrer and Sorensen (1966) could not obtain odd-even staggering with their pairing plus quadrupole model, and Sagawa, Arima, and Scholten (1987) found that quadrupole couplings gave staggering that was too small for lead isotopes. The 3P formula provides a practical tool for experimentalists to analyze isotope shifts even when atomic physics parameters like the F factor are not well determined, as the staggering parameter is defined as a ratio that factors out such uncertainties.
