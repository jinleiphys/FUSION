# Clustering aspects and the shell model

> In this talk I shall discuss the clustering aspect and the shell model. I shall first discuss the $α$-cluster aspects based on the shell model calculations. Then I shall discuss the spin zero ground state dominance in the presence of random interactions and a new type of cluster structure for fermions in a single-$j$ shell in the presence of only pairing interaction with the largest multiplicity.

# Clustering aspects and the shell model

## Key claim

The paper argues that the spin-0 ground state dominance observed in two-body random ensemble (TBRE) calculations is not independent of the two-body interaction, contrary to a popular idea; instead, specific two-body matrix elements (e.g., J=0, 6, 8, 12, and 22 for j=31/2) are essential to produce a sizable P(0). The paper also claims that a new type of cluster structure emerges for fermions in a single-j shell when only the pairing interaction with the largest multiplicity is present.

## Method

The author reviews the historical connection between alpha-cluster wavefunctions and SU(3) shell model wavefunctions for light nuclei, showing that the 2-alpha condensate wavefunction reduces to the SU(3) shell model wavefunction when the width parameter epsilon is small. For the spin-0 ground state dominance problem, the author uses a single-j shell Hamiltonian with two-body random ensemble (TBRE) interactions, where the two-body matrix elements G_J are taken as Gaussian random numbers with width 1 and average 0, and computes ground state spin probabilities from 1000 runs. An empirical formula is introduced where only one G_J is set to -1 and all others to zero, and the number N_I of times each spin I appears as the ground state is counted; the predicted probability is P(I) = N_I/N where N is the number of independent two-body matrix elements. For the new cluster type, the author examines eigenvalues and wavefunctions for four fermions in a j=31/2 shell with only the J_max pairing interaction turned on.

## Key numbers

- For 4 fermions in a single-j shell with j=31/2, the total number of states is 790, of which 380 have non-zero eigenvalues when only the J_max pairing interaction is present.
- Within a precision of 10^{-2}, 308 of those 380 states have eigenvalues located at the eigenvalues of the n=3 system, and 21 states have eigenvalues at -2.
- The lowest eigenvalue for I=28 of four fermions in j=31/2 with only the J_max interaction is -2.26271186440690, which is very close to the n=3 maximum-spin eigenvalue of -267/118 = -2.26271186440678.
- For the Cohen-Kurath interaction in 8Be, the matrix element of the Majorana operator P_M is -5.77, compared to -6 for exact SU(4) symmetry; the overlap between the Cohen-Kurath ground state and the SU(4) symmetry state is 0.97.
- For 20Ne, the [4] symmetry percentage in the wavefunction is 91.8% for the 0_1 state, 92.0% for 0_2, 89.9% for 2_1, 81.9% for 2_2, 87.5% for 4_1, 83.0% for 4_2, and 80.0% for 6_1.

## Context

This talk extends the long-standing work on alpha-clustering in light nuclei, connecting the SU(3) shell model of Elliott to the Wildermuth alpha-cluster wavefunctions and the Resonating Group Method (Generator Coordinate Method). The spin-0 ground state dominance problem was discovered by Johnson, Bertsch and Dean (1998) using the two-body random ensemble, and the author's empirical formula builds on work by Bijker and by Zhao and collaborators. The new cluster structure found for the pairing interaction with largest multiplicity provides a novel classification of states by clusters of 2, 3, or 4 particles, which may have applications in understanding nuclear structure beyond the standard shell model.
