# Reweighting the quark Sivers function with STAR jet data

> The Bayesian reweighting procedure is applied for the first time to a TMD distribution, the quark Sivers function extracted from SIDIS data. By exploiting the recent published single spin asymmetry data for the inclusive jet production in $p^\uparrow p$ collisions from the STAR collaboration at RHIC, we show how such a procedure allows to incorporate the information contained in the new data set, without the need of re-fitting, and to explore a much wider $x$ region compared to SIDIS measurements. The reweighting method is also extended to the case of asymmetric errors, and the results show a significant improvement on the knowledge of the quark Sivers function.

## Key claim

The Bayesian reweighting procedure is applied for the first time to a TMD distribution, the quark Sivers function extracted from SIDIS data, and the results show that the new STAR jet data are compatible with the SIDIS extraction while significantly improving constraints at large \(x\), though the reweighted \(\chi^2_{\rm dof}\) slightly favors the GPM approach over the CGI-GPM approach.

## Method

The authors apply a Bayesian reweighting procedure to a prior set of Monte Carlo parameter sets for the quark Sivers function, originally extracted from 220 SIDIS data points. They compute new weights for each parameter set using the \(\chi^2\) of the STAR inclusive jet single-spin asymmetry data at \(\sqrt{s}=200\) and \(500\) GeV, selecting the subset with photon multiplicity \(n_\gamma > 2\). The reweighting is performed within both the generalized parton model (GPM) and the color gauge invariant generalized parton model (CGI-GPM), and asymmetric uncertainties are calculated using the median and \(2\sigma\) confidence level intervals.

## Key numbers

* The SIDIS extraction used \(N_{\rm dat}^{\rm SIDIS} = 220\) datapoints with \(\Delta\chi^2 = 11.31\) for \(N=5\) parameters at \(2\sigma\) CL.
* The STAR data cover \(x_F = 2 P_L / \sqrt{s} \in [0.1:0.6]\) at center-of-mass energies \(\sqrt{s} = 200\) GeV and \(\sqrt{s} = 500\) GeV.
* After reweighting, the combined dataset has \(N_{\rm dat}^{\rm SIDIS + jet} = 238\) datapoints.
* The Monte Carlo generation used \(N_{\rm set} = 2 \cdot 10^5\) sets via a Markov-Chain MC procedure with Metropolis-Hastings algorithm.

## Context

This work extends the Bayesian reweighting technique, previously applied to collinear PDFs by Giele (1998), Ball (2010), Sato (2013), and D'Alesio (2020), to the TMD domain for the first time. The quark Sivers function parametrization used as prior was extracted by Boglione (2018) from SIDIS data. The results demonstrate compatibility between SIDIS and inclusive jet data, and enable future global analyses that could simultaneously reweight the Sivers, transversity, and Collins functions using inclusive pion production data, as well as upcoming measurements from COMPASS, JLab, and the Electron Ion Collider.
