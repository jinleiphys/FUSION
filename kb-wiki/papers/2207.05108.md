# Neutrinoless $ββ$-decay nuclear matrix elements from two-neutrino $ββ$-decay data

> We study two-neutrino ($2νββ$) and neutrinoless double-$β$ ($0νββ$) decays in the nuclear shell model and proton-neutron quasiparticle random-phase approximation (pnQRPA) frameworks. Calculating the decay half-life of several dozens of nuclei ranging from calcium to xenon with the shell model, and of $ββ$ emitters with a wide range of proton-neutron pairing strengths in the pnQRPA, we observe good linear correlations between $2νββ$- and $0νββ$-decay nuclear matrix elements for both methods. We then combine the correlations with measured $2νββ$-decay half-lives to predict $0νββ$-decay matrix elements with theoretical uncertainties based on our systematic calculations. Our results include two-body currents and the short-range $0νββ$-decay operator.

## Key claim

The authors demonstrate that there is a good linear correlation between two-neutrino double-beta decay nuclear matrix elements and neutrinoless double-beta decay nuclear matrix elements across many nuclei, and use this correlation together with measured two-neutrino double-beta decay half-lives to predict neutrinoless double-beta decay matrix elements with theoretical uncertainties. The correlation holds for both the nuclear shell model and the proton-neutron quasiparticle random-phase approximation, and the predictions include contributions from two-body currents and the short-range neutrinoless double-beta decay operator.

## Method

The authors performed nuclear shell-model calculations for several dozen nuclei in the mass range 46 to 136 using three different configuration spaces with the KB3G, GXPF1B, GCN2850, JUN45, JJ4BB, GCN5082, and QX interactions, and proton-neutron quasiparticle random-phase approximation calculations for nine beta-beta emitters using large no-core single-particle bases in a Coulomb-corrected Woods-Saxon potential with interactions based on the Bonn-A potential. They calculated two-neutrino double-beta decay nuclear matrix elements directly and neutrinoless double-beta decay nuclear matrix elements in the closure approximation for the shell model and by summing over intermediate states for the proton-neutron quasiparticle random-phase approximation, then fitted linear correlations between the two sets of matrix elements and combined these with experimental two-neutrino double-beta decay half-lives to extract neutrinoless double-beta decay matrix elements with uncertainties.

## Key numbers

- Correlation coefficients for the linear fits between two-neutrino and neutrinoless double-beta decay nuclear matrix elements: r = 0.84 for the proton-neutron quasiparticle random-phase approximation; r = 0.86 for shell model in the pf-shell (A = 46-60), r = 0.95 for the pfg-shell (A = 72-84), and r = 0.97 for the sdg-shell (A = 124-136)
- Two-body currents reduce neutrinoless double-beta decay nuclear matrix elements by 25% to 45% in both the shell model and proton-neutron quasiparticle random-phase approximation
- The short-range neutrinoless double-beta decay operator amounts to 30% to 80% of the one-body long-range value in the proton-neutron quasiparticle random-phase approximation and 15% to 50% in the shell model
- Quenching factors used for two-neutrino double-beta decay nuclear matrix elements: q = 0.65-0.77 for A = 46-60, q = 0.55-0.64 for A = 72-84, q = 0.42-0.72 for A = 124-136 in the shell model; q = 0.79 in the proton-neutron quasiparticle random-phase approximation

## Context

This work extends previous studies that found a correlation between two-neutrino and neutrinoless double-beta decay nuclear matrix elements in calcium-48 (Horoi 2022) and a relation between their radial transition densities in all nuclei (Simkovic 2011). It also builds on earlier work using two-neutrino double-beta decay data to adjust proton-neutron quasiparticle random-phase approximation model parameters (Rodin 2003, Rodin 2005, Faessler 2008, Simkovic 2013, Simkovic 2018) and to estimate improved neutrinoless double-beta decay matrix elements (Brown 2015). The results enable more reliable predictions of neutrinoless double-beta decay nuclear matrix elements with quantified uncertainties, which are essential for interpreting experimental searches for neutrinoless double-beta decay.
