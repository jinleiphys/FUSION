# Removal of $K$-mixing in angular momentum projected nuclear wave functions

> Angular momentum projection plays a key role in studying quantum many-body systems with rotational invariance such as atomic nuclei. At a given spin $J$, one can generate $2J+1$ angular momentum projected states labeled with $-J\leq K \leq J$ from a deformed Slater determinant. Usually, a nuclear wave function with $K$-mixing can be expressed as a superposition of all these $2J+1$ projected states, where the coefficients can be obtained by solving the generalized eigenvalue equation. In this Letter, we report a new fundamental feature that the frequently discussed $K$-mixing in the angular momentum projected nuclear wave function can be safely removed. Strikingly, we found that such nuclear wave function with $K$-mixing can always be equivalently replaced by a single projected state with any given $K$. Consequently, such nuclear wave function can be significantly simplified, especially for high-spin states. This also indicates that the $K$-mixing in the angular momentum projected nuclear wave functions, adopted by many present-day nuclear models, does not carry any physical meaning, and is essentially different from that $K$-mixing caused by the Coriolis force in the cranked shell model.

## Key claim

The paper claims that any angular momentum projected nuclear wave function that includes $K$-mixing (a superposition of $2J+1$ projected states with different $K$ values) can always be equivalently replaced by a single projected state with any chosen $K$, generated from a different reference state. This equivalence is proven analytically and demonstrated numerically, with the caveat that the replacement reference state may be complicated but can in practice be a simple Slater determinant when variational optimization is performed.

## Method

The authors first provide an analytical proof using angular momentum ladder operators to show that any $K$-mixing wave function $|\Psi_{JM}\rangle = \sum_{K'} f_{K'} \hat{P}^J_{MK'}|\Phi\rangle$ can be rewritten as a single projected state $\hat{P}^J_{MK}|\phi\rangle$ from a transformed reference state $|\phi\rangle$. They then perform numerical variational calculations for several nuclei ($^{60}$Zn, $^{61}$Zn, $^{58}$Cu, $^{68}$Se, $^{26}$Mg, $^{48}$Cr) using realistic shell model interactions (GXPF1A, JUN45, USDB), where they minimize a quantity $R_K$ (the reciprocal of the squared overlap $O_K$) to find a Slater determinant $|\Phi'\rangle$ such that $\hat{P}^{J\pi}_{MK}|\Phi'\rangle$ approximates the original $K$-mixing wave function.

## Key numbers

- For $^{60}$Zn with $J^\pi=4^+$ and $16^+$ states using the GXPF1A interaction, all calculated $O_K$ overlaps between the $K$-mixing wave function and the single projected state are "very close to 1", and all energy differences are within 100 keV.
- For $^{68}$Se with $J=16$ and both parities, using the JUN45 interaction with deformation parameters $\varepsilon_2=0.45$, $\gamma=40^\circ$, and $\varepsilon_3=0.3$, all calculated $O_K$ values are "very close to 1".
- For $^{26}$Mg and $^{48}$Cr with $J^\pi=8^+$ states during variational after projection iterations, all energy differences between the $K$-mixing wave function and the single projected states are within 400 keV.

## Context

This work extends the authors' previous variational after projection (VAP) calculations (Lu et al., 2022; Lian et al., 2022, 2023) where wave functions without $K$-mixing were used and good shell model approximation was achieved. It contradicts the widespread assumption in modern nuclear models (projected shell model, VAP methods, quantum Monte Carlo approaches) that $K$-mixing is necessary for obtaining lower energies. The result enables significant simplification of nuclear wave functions, especially for high-spin states, and clarifies that $K$-mixing in angular momentum projected wave functions is fundamentally different from $K$-mixing caused by the Coriolis force in the cranked shell model (Zeng, 1990).
