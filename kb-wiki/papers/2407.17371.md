# Wave function of $^9$Be in the three-body (alpha-alpha-n)-model

> A simple analytic expression of the three-body wave function describing the system $(ααn)$ in the ground state $\frac{3}{2}^-$ of ${}^9\mathrm{Be}$ is obtained. In doing this, it is assumed that the $α$ particles interact with each other via the $S$-wave Ali-Bodmer potential including the Coulomb term, and the neutron-$α$ forces act only in the $P$-wave state. This wave function is constructed by trial and error method via solving in this way a kind of inverse problem when the two-body $αα$ potential is recovered from a postulated three-body wave function. As a result, the wave function is an exact solution of the corresponding three-body Schrödinger equation for experimentally known binding energy and for the $αα$ potential whose difference from the Ali-Bodmer one is minimized by varying the adjustable parameters which the postulated wave function depends on.

## Key claim

The paper presents a simple analytic three-body wave function for the ground state of \(^9\)Be that is an exact solution of the Schrödinger equation for the experimental binding energy, but only for an \(\alpha\alpha\) potential that is an approximation to, not identical with, the standard Ali-Bodmer potential, with the difference minimized by parameter variation.

## Method

The author postulates a parametrized analytic form for the radial part of the three-body wave function \(R(r_{\alpha\alpha}, r_n)\) for the \((\alpha\alpha n)\) system in the ground state \((3/2)^-\). Using an inverse-problem technique adapted from Belyaev et al., the \(\alpha\alpha\) potential is recovered from the postulated wave function via Eq. (Vgeneral), which integrates over all coordinates except \(r_{\alpha\alpha}\). The three adjustable parameters \(a_1, a_2, a_3\) in the wave function are then varied using the MINUIT minimization code to minimize the least-square difference between the recovered \(\alpha\alpha\) potential and the known Ali-Bodmer S-wave potential (including Coulomb term), while the \(\alpha n\) interaction is fixed to the Bang-Gignoux Woods-Saxon plus spin-orbit potential acting only in the P-wave.

## Key numbers

* Binding energy of \(^9\)Be: \(|E_b| = 1.5736\) MeV (experimental value, used as input).
* Ali-Bodmer potential parameters: \(V_R = 1050\) MeV, \(\beta_R = 0.8\) fm\(^{-1}\), \(V_A = 150\) MeV, \(\beta_A = 0.5\) fm\(^{-1}\), \(R_\alpha = 1.44\) fm.
* Bang-Gignoux \(\alpha n\) potential parameters: \(W_1 = -43.0\) MeV, \(R_1 = 2.0\) fm, \(d_1 = 0.7\) fm, \(W_2 = 40.0\) MeV\(\cdot\)fm\(^2\), \(R_2 = 1.5\) fm, \(d_2 = 0.35\) fm.
* Optimized wave function parameters: \(a_1 = 0.37441\) fm\(^{-1}\), \(a_2 = 0.070112\) fm\(^{-1}\), \(a_3 = 0.41953\) fm\(^{-1}\), normalization constant \(N = 927446.734\) fm\(^{9/2}\).
* Calculated RMS distances in \(^9\)Be: \(\sqrt{\langle r_{\alpha\alpha}^2\rangle} = 3.46\) fm; \(\sqrt{\langle r_{\mathrm{cm}-n}^2\rangle} = 4.65\) fm; \(\sqrt{\langle r_{\mathrm{cm}-\alpha}^2\rangle} = 1.83\) fm.
* Calculated charge radius: \(\sqrt{\langle r_{\mathrm{ch}}^2\rangle} = 2.48\) fm (experimental: 2.519 fm from ADNDT1, ADNDT2).
* Calculated matter radius: \(\sqrt{\langle r_{\mathrm{mat}}^2\rangle} = 2.70\) fm (experimental: \(2.50 \pm 0.01\) fm from PRL).
* Maximum of radial wave function occurs at \(r_{\alpha\alpha} = 2.4051\) fm and \(r_n = 1.4654\) fm.

## Context

This work extends the inverse-problem method of Belyaev et al. to construct a practical analytic wave function for the Borromean nucleus \(^9\)Be, which has been studied extensively via Faddeev equations (Efros, Filikhin) and other three-body methods (Casal, Itagaki, Descouvemont). Unlike those approaches, which yield wave functions not easily usable by others, the present result provides a compact closed-form expression that can be directly employed for constructing folding potentials or estimating reaction cross sections involving \(^9\)Be, such as the astrophysically important \(n(\alpha\alpha,\gamma)^9\)Be radiative capture. The method is proposed as applicable to other Borromean systems like \(^6\)He and \(^6\)Li in the \((\alpha NN)\) model, and potentially to systems with more than three bodies.
