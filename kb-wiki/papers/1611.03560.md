# A Novel Algorithm for Extracting the Parton Distribution Amplitude from the Euclidean Bethe-Salpeter Wave Function

> We propose a new numerical method to compute parton distribution amplitude(PDA) from the Euclidean Bethe-Salpeter wave function. The essential step is to extract the weight function in the Nakanishi representation of the Bethe-Salpeter wave function in Euclidean space, which is an ill-posed inversion problem, via the maximum entropy method(MEM). The Nakanishi weight function as well as the corresponding light-front PDA can be well determined. We confirm the previous works on PDA computation therein the different method has been performed.

## Key claim

The authors propose that the maximum entropy method (MEM) can reliably extract the Nakanishi weight function from Euclidean Bethe-Salpeter wave function data, thereby enabling computation of the parton distribution amplitude (PDA) without relying on parameterized forms or a limited number of Mellin moments, though they acknowledge they cannot prove the weight function is uniquely determined.

## Method

The authors first generate mock Bethe-Salpeter amplitude data from an analytical Nakanishi weight function model to validate the MEM approach. They then solve the rainbow-ladder truncated Bethe-Salpeter equation and Dyson-Schwinger equation for pseudoscalar mesons (pion and eta_c) in Euclidean space, extract the Nakanishi weight function from the numerical wave function via the maximum entropy method, and compute the leading-twist light-front PDA using the integral representation that relates the weight function to the PDA.

## Key numbers

- For the pion PDA calculation, the interaction parameters are set to omega = 0.5 GeV and D*omega = (0.87 GeV)^3, matching those in Chang (2013).
- For the eta_c PDA calculation, the interaction parameters are omega = 0.8 GeV and D*omega = (0.7 GeV)^3, yielding a leptonic decay constant of about 0.28 GeV.
- The MEM reconstruction uses a default model m(gamma,z) = m_0 * theta(Lambda - gamma) with a typical error parameter s <= 0.1 for the likelihood function.
- The analytical test model for the Nakanishi weight function is g(gamma,z) = exp[-(gamma+1)/(1-z^2)].

## Context

This work extends the Nakanishi representation approach pioneered by Chang (2013) for computing pion PDAs and the brute-force Mellin moment method of Ding (2016) for heavy mesons, providing an alternative numerical technique that avoids parameterizing the weight function or relying on a limited number of moments. The method enables direct extraction of the full PDA from Euclidean Bethe-Salpeter wave functions and could be extended to compute light-front wave functions, though the authors note limitations for cases where the weight function is not positive definite or the wave function is not monotonic.
