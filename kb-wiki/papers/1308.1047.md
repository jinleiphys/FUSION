# Study of Nuclear Effects in the Computation of the 0νββ Decay Matrix Elements

> We analyse the effects that different nuclear structure approximations associated with the short range correlations (SRC), finite nucleon size (FNS), higher order terms in the nucleon currents (HOC) and with some nuclear input parameters, have on the values of the nuclear matrix elements (NMEs) for the neutrinoless double beta (0νββ) decay. The calculations are performed with a new Shell Model(ShM) code which allows a fast computation of the two-body matrix elements of the transition operators. The treatment of SRC, FNS and HOC and include the use of quenched or unquenched values for the axial vector coupling constant produce the most important effects on the NMEs values. Equivalent effects of some of these approximations are also possible, which may lead (accidentally) to close final results. We found that the cummulative effect of all these nuclear ingredients on the calculated nuclear matrix elements NMEs is significant. Since the NMEs values are often obtained with different approximations and/or with different input parameters, a convergent view point on their inclusion/neglection and an uniformization of the calculations are needed, in order to enter in an era of precision concerning the computation of the NMEs for double beta deacay.

## Key claim

The cumulative effect of short-range correlations (SRC), finite nucleon size (FNS), higher-order currents (HOC), and the choice of quenched or unquenched axial coupling constant \(g_A\) on the neutrinoless double beta decay nuclear matrix elements (NMEs) is significant, but some combinations of approximations can accidentally produce similar final values, meaning that a uniform treatment is needed before precision calculations can be achieved.

## Method

The authors used a new Shell Model (ShM) code that computes two-body matrix elements (TBMEs) for the \(0\nu\beta\beta\) decay transition operators by analytically performing radial integrals over harmonic oscillator wave functions, leaving only a single numerical integration over neutrino momentum. They calculated NMEs for \({}^{48}\)Ca, \({}^{76}\)Ge, and \({}^{82}\)Se, gradually including SRC (via Jastrow with Miller-Spencer parameterization and coupled-cluster method with Argonne V18 and CD-Bonn forces), FNS, HOC (including the tensor term), and varying input parameters such as \(g_A\), \(r_0\), and cutoff parameters \(\Lambda_V\) and \(\Lambda_A\).

## Key numbers

* For \({}^{48}\)Ca, the bare NME \(M_b = -1.166\) decreased to \(M^{0\nu}_{\text{total}} = -0.809\) when all effects were included with CCM-CD-Bonn SRC and \(g_A = 1.25\).
* For \({}^{76}\)Ge, the bare NME \(M_b = 4.168\) decreased to \(M^{0\nu}_{\text{total}} = 3.177\) with CCM-CD-Bonn SRC and \(g_A = 1.25\).
* For \({}^{82}\)Se, the bare NME \(M_b = -3.779\) decreased to \(M^{0\nu}_{\text{total}} = -2.878\) with CCM-CD-Bonn SRC and \(g_A = 1.25\).
* The cumulative effect of all nuclear ingredients on the bare NME was 43-56% for Jastrow-MS SRC, and 24-31% for CCM-AV18 and CCM-CD-Bonn SRC.
* Neglecting HOC resulted in larger \(M^{0\nu}_{\text{total}}\) values by about 20% for \({}^{48}\)Ca and about 16% for \({}^{76}\)Ge and \({}^{82}\)Se.
* Differences between quenched (\(g_A = 1.0\)) and unquenched (\(g_A = 1.25\)) results were within 10-14%; differences between \(g_A = 1.25\) and \(g_A = 1.275\) were negligible.
* Using \(r_0 = 1.1\) fm instead of 1.2 fm gave an error of about 7%; using cutoff parameters (710, 850) MeV instead of (850, 1086) MeV gave differences up to 8%.
* The tensor component contributed 4-9% of the total NME.

## Context

This work extends previous Shell Model calculations of \(0\nu\beta\beta\) decay NMEs by the same group (Neacsu and Stoica, 2012) and by others using the Interacting Shell Model (Caurier et al., 1995; Horoi and Stoica, 2010), and it compares with results from QRPA (Simkovic et al., 2009; Suhonen, 2007), IBA-2 (Barea and Iachello, 2013), and EDF (Rodriguez and Martinez-Pinedo, 2010). The paper demonstrates that softer SRC prescriptions (CCM, UCOM) yield larger NMEs than the older Jastrow-MS method, and it calls for a standardized treatment of nuclear effects and input parameters to enable reliable comparisons between different nuclear-structure methods and to move toward precision calculations of \(0\nu\beta\beta\) decay lifetimes.
