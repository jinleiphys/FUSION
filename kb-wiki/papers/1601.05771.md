# Nonperturbative True Muonium on the Light Front with TMSWIFT

> The true muonium $(μ\barμ)$ bound state presents an interesting test of light-cone quantization techniques. In addition to exhibiting the standard problems of handling non-perturbative calculations, true muonium requires correct treatment of $e\bar{e}$ Fock-state contributions. Having previously produced a crude model of true muonium using the method of iterated resolvents, our current work has focused on the inclusion of the box diagrams to improve the cutoff-dependent issues of the model. Further, a parallel computer code, TMSWIFT, allowing for smaller numerical uncertainties, has been developed. This work focuses on the current state of these efforts to develop a model of true muonium that is testable at near-term experiments.

## Key claim

The paper presents a nonperturbative light-front model of true muonium that includes electron-positron Fock-state contributions and uses a counterterm subtraction to improve ultraviolet behavior, but the results are currently limited by the need to include the two-photon Fock state and proper renormalization before the model can produce predictions testable at near-term experiments.

## Method

The authors use Discretized Light-Cone Quantization (DLCQ) with a truncated Fock space containing seven explicit states (muon-antimuon, electron-positron, tau-antitau, photon, and each lepton pair with a photon) and fold the photon and lepton-pair-photon components into effective interactions using the method of iterated resolvents. A counterterm subtraction removes the strong cutoff dependence of the singlet S-wave states by subtracting the asymptotic limit of the problematic matrix element. The parallel code TMSWIFT solves the resulting eigenvalue problem using the SLEPc library, allowing multiple lepton flavors with independent cutoffs and discretization parameters.

## Key numbers

- True muonium ground state (1^1S_0) mass squared at α=0.3: M² = 3.8944(3) in units of m_μ from TMSWIFT, compared to instant-form value 3.9053
- True muonium ground state (1^1S_0) mass squared at α=0.01 (≈1.4α_QED): M² = 3.999900017(5) in units of m_μ from TMSWIFT, compared to instant-form value 3.999899994
- Three-flavor calculations with N_μ=N_τ=37 and N_e=71 for J_z=0 completed in 2 hours using 512 cores of the Stampede supercomputer
- Previous serial code (Mesonix) required a full day on a 3.0 GHz core for N_μ=N_e=37 to produce a single J_z spectrum

## Context

This work extends the DLCQ methods developed by Tang (1991), Krautgartner (1991), Kaluza (1991), and Trittmann (1997) to true muonium, building on the authors' earlier crude model using iterated resolvents (Lamm 2013). The inclusion of box diagrams and the counterterm subtraction improves the cutoff-dependent issues of the earlier model. The results enable future inclusion of the two-photon Fock state, which dominates singlet-state decays, and provide a foundation for applying similar techniques to QCD bound states.
