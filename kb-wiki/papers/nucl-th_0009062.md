# $πΔΔ$ coupling constant

> We calculate the $πΔΔ$ coupling $g_{π^0Δ^{++}Δ^{++}}$ using light cone QCD sum rule. Our result is $g_{π^0Δ^{++}Δ^{++}}=(11.8\pm 2.0)$.

# πΔΔ coupling constant

## Key claim

The πΔΔ coupling constant is calculated to be large (11.8 ± 2.0) using light cone QCD sum rules, but this value is only about half of the large Nc prediction (9/5 of the πNN coupling) and contradicts the chiral symmetry scheme where parity-nonchanging couplings like πΔΔ are forbidden at leading order.

## Method

The author uses light cone QCD sum rules (LCQSR) to calculate the π⁰Δ⁺⁺Δ⁺⁺ coupling constant by considering a correlator of two Δ⁺⁺ interpolating currents in the presence of a pion state. The operator product expansion is performed using pion light cone wave functions, followed by double Borel transformation to extract the double spectral density and subtract continuum contributions. The coupling is obtained by dividing the sum rule for the πΔΔ correlator by the Δ mass sum rule.

## Key numbers

- g_{π⁰Δ⁺⁺Δ⁺⁺} = 11.8 ± 2.0 (central value at M² = 1.4 GeV², s_Δ = 3.5 GeV²)
- g_{π⁰PP} (πNN coupling) = 13.2 ± 1.5 (from same LCQSR formalism for comparison)
- Large Nc prediction would give g_{π⁰Δ⁺⁺Δ⁺⁺} = (9/5) × g_{π⁰PP} ≈ 23.8
- Preliminary chiral perturbation theory extraction gave g₁ = −0.94 to −2.65 (related to g_{πΔΔ} via g₁ = F_π g_{πΔΔ} / m_Δ, with F_π = 92.4 MeV)
- Working region for Borel parameter M²: 1.2–1.7 GeV² for Δ sum rule, 1.0–1.5 GeV² for nucleon sum rule
- Continuum thresholds: s_Δ = 3.4, 3.5, 3.6 GeV²; s_N = 2.15, 2.25, 2.35 GeV²

## Context

This work extends the author's previous LCQSR calculations of πNN, πNN*, ηNN, ρNN, and ωNN couplings (Zhu, 2000) to the Δ resonance. The result contradicts the chiral symmetry quartet scheme proposed in Physical Review Letters (1999) where πΔΔ couplings are forbidden at leading order, and disagrees with quark model predictions and the large Nc relation g₁ = (9/5)g_A. The calculated value is consistent with an old phenomenological extraction from the isobar production experiment π⁻p → π⁺π⁻n near threshold (Mitaroff, 1970s). This independent theoretical determination helps clarify the ambiguous situation where chiral perturbation theory fits to πN scattering phase shifts gave a very different and negative value for g₁.
