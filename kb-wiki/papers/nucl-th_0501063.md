# The number of J=0 pairs in {44,46,48}Ti

> In the single j-shell, the configuration of an even-even Ti isotope consists of 2 protons and n neutrons. The I=0 wave function can be written in terms of D(J,Jv)'s, which are the probability amplitudes for the 2 protons to couple to J and the n neutrons to couple to Jv (where J stands for the angular momentum and v for the seniority quantum number). There are several states with isospin Tmin=|(N-Z)/2|, but only one with Tmax=Tmin+2. By demanding that the Tmax wave function be orthogonal to the Tmin ones, we obtain the following simple expressions for the number of J=0 np pairs in these Ti isotopes: For T=Tmin, number of pairs(J12=0)=2|D(00)|^2/n For T=Tmax, number of pairs(J12=0)=2n|D(00)|^2=2n(2j+1-n)/(2j+1)(n+1) For 44Ti we have also the results for even J12: number of nn pairs=number of pp pairs=number of np pairs=|D(J12,J12)|^2.

# The number of J=0 pairs in {44,46,48}Ti

## Key claim
Within the single j-shell model, the number of J=0 neutron-proton (np) pairs in even-even Ti isotopes can be expressed in simple closed form for both the minimum-isospin (Tmin) and maximum-isospin (Tmax) states, but the derivation relies on the orthogonality condition between Tmax and Tmin wave functions and applies only to the single-j configuration of 2 protons and n neutrons.

## Method
The authors construct I=0 wave functions for even-even Ti isotopes (2 protons plus n neutrons) in the single j-shell, using probability amplitudes D(J,Jv) for proton and neutron angular-momentum couplings. By imposing orthogonality between the single Tmax state and the several Tmin states, they derive a relation involving a one-particle coefficient of fractional parentage, which then yields analytic expressions for the number of J=0 np pairs.

## Key numbers
- For T = Tmin: number of J=0 np pairs = 2|D(00)|² / n
- For T = Tmax: number of J=0 np pairs = 2n|D(00)|² = 2n(2j+1-n) / [(2j+1)(n+1)]
- For ⁴⁴Ti: number of nn pairs = number of pp pairs = number of np pairs = |D(J12,J12)|² for even J12

## Context
This work extends the single-j shell treatment of pairing in Ti isotopes, providing explicit formulas for J=0 pair counts that complement earlier seniority-based approaches. The results enable quantitative comparison with shell-model calculations for ⁴⁴Ti, ⁴⁶Ti, and ⁴⁸Ti, and can be used to test the validity of the single-j approximation against more realistic interactions or experimental data.
