# Resolving the $R_{\rm pA}$ and $v_2$ puzzle of $D^0$ mesons in $p-$Pb collisions

> It has been difficult to reconcile the experimental data on the $D^0$ meson nuclear modification factor and elliptic flow in $p-$Pb collisions at LHC energies. Here we study these observables with the string melting version of a multi-phase transport model, which has been improved with the implementation of the Cronin effect (or transverse momentum broadening) and independent fragmentation for charm quarks. Using a strong Cronin effect allows us to provide the first simultaneous description of the $D^0$ meson $R_{\rm pA}$ and $v_2$ data at $p_{\rm T} \leq$ 8 GeV$/c$. The model also provides a reasonable description of the $D^0$ meson $p_{\rm T}$ spectra and the low-$p_{\rm T}$ (below $\sim$ 2 GeV$/c$) charged hadron spectra in $p+p$ and $p-$Pb collisions as well as $R_{\rm pA}$ and $v_2$ in $p-$Pb collisions. We find that both parton scatterings and the Cronin effect are important for the $D^0$ meson $R_{\rm pA}$, while parton scatterings are mostly responsible for the $D^0$ meson $v_2$. Our results indicate that it is crucial to include the Cronin effect for the simultaneous description of the $D^0$ meson $R_{\rm pA}$ and $v_2$. Since the Cronin effect is expected to grow with the system size, this work implies that the Cronin effect could also be important for heavy hadrons in large systems.

## Key claim

Using a strong Cronin effect (transverse momentum broadening) in the string melting version of the AMPT model, the authors provide the first simultaneous description of both the $D^0$ meson nuclear modification factor $R_{\rm pA}$ and elliptic flow $v_2$ in $p-$Pb collisions at LHC energies, resolving a long-standing puzzle where previous models could describe only one observable at a time.

## Method

The authors used the string melting version of the AMPT (a multi-phase transport) model, improved with the implementation of transverse momentum broadening (Cronin effect) for charm quarks and independent fragmentation as an additional hadronization channel. Initial charm quarks from the HIJING model were separated from the string melting process and entered the parton cascade after a formation time $\tau_f=E/m_T^2$, with distinct cross sections for light quarks ($\sigma_{\rm LQ}=0.5$ mb) and heavy-light quark interactions ($\sigma_{\rm HQ}=1.5$ mb). The Cronin effect was implemented by applying a transverse momentum kick $\vec{k}_T$ to each $c\bar{c}$ pair, sampled from a Gaussian distribution with width $w = w_0 \sqrt{1+(n_{\rm coll}-i)\delta}$, where $\delta$ controls the strength of the broadening beyond $p+p$ collisions.

## Key numbers

- For $p+p$ collisions at 5.02 TeV, the Gaussian width parameter $w_0 = 0.375$ GeV$/c$, yielding $\langle k_T^2 \rangle = 0.04$ GeV$^2$ for a single charm quark and $0.14$ GeV$^2$ for a $c\bar{c}$ pair
- For minimum bias $p-$Pb collisions at 5.02 TeV with $\delta=5$, the model yields $\langle k_T^2 \rangle = 3.27$ GeV$^2$ for a single charm quark and $13.0$ GeV$^2$ for a $c\bar{c}$ pair
- Default parton cross sections: $\sigma_{\rm LQ}=0.5$ mb (light quarks) and $\sigma_{\rm HQ}=1.5$ mb (heavy-light quarks), determined by fitting charged hadron $v_2$ data in $p-$Pb at 5.02 TeV and $D^0$ meson $v_2$ data in $p-$Pb at 8.16 TeV, respectively
- The simultaneous description of $D^0$ meson $R_{\rm pA}$ and $v_2$ is achieved for transverse momenta $p_{\rm T} \leq 8$ GeV$/c$
- The model with $\delta=5$ or $\delta=7$ provides reasonable description of both observables; a strong Cronin effect significantly enhances $D^0$ meson $R_{\rm pA}$ at high $p_{\rm T}$ while causing a small decrease in $v_2$

## Context

This work resolves a long-standing puzzle where previous models could describe either the heavy meson $R_{\rm pA}$ (e.g., the POWLANG model by Beraudo (2015) or pQCD calculations by Kang (2014)) or the heavy meson $v_2$ (e.g., the color glass condensate framework by Zhang (2020)), but not both simultaneously. The results demonstrate that the Cronin effect, which grows with system size, is crucial for simultaneously describing $D^0$ meson $R_{\rm pA}$ and $v_2$ in $p-$Pb collisions, and the authors suggest this effect could also be important for heavy hadrons in larger collision systems such as Pb-Pb.
