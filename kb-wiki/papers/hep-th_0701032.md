# A new representation for non--local operators and path integrals

> We derive an alternative representation for the relativistic non--local kinetic energy operator and we apply it to solve the relativistic Salpeter equation using the variational sinc collocation method. Our representation is analytical and does not depend on an expansion in terms of local operators. We have used the relativistic harmonic oscillator problem to test our formula and we have found that arbitrarily precise results are obtained, simply increasing the number of grid points. More difficult problems have also been considered, observing in all cases the convergence of the numerical results. Using these results we have also derived a new representation for the quantum mechanical Green's function and for the corresponding path integral. We have tested this representation for a free particle in a box, recovering the exact result after taking the proper limits, and we have also found that the application of the Feynman--Kac formula to our Green's function yields the correct ground state energy. Our path integral representation allows to treat hamiltonians containing non--local operators and it could provide to the community a new tool to deal with such class of problems.

# Alternative representation for non-local operators and path integrals

## Key claim

The paper derives an analytical representation for the relativistic non-local kinetic energy operator that is exact in the limit of an arbitrarily fine grid, avoiding any expansion in local operators; the representation is tested on the relativistic harmonic oscillator and yields arbitrarily precise results by increasing the number of grid points, but the method's convergence depends on the scale parameter μ and the non-relativistic expansion can converge to a wrong value when the wave function is sizeable at p = μ.

## Method

The author uses the Variational Sinc Collocation Method (VSCM) with Little Sinc Functions (LSF) to obtain a matrix representation of the relativistic kinetic energy operator √(p̂² + μ²) in coordinate space. The matrix elements are computed analytically by expanding the LSF in a Fourier series and applying the operator term-by-term, then collocating on a grid; the optimal grid scale L is determined by applying the Principle of Minimal Sensitivity (PMS) to the trace of the truncated Hamiltonian matrix. The method is extended to compute Green's functions and path integrals for Hamiltonians containing non-local operators using the Trotter product formula.

## Key numbers

- For the relativistic harmonic oscillator with μ = 30 and N = 50 grid points, the ground state energy ε − μ = 0.386266042572445193517188444455, agreeing with Lucha (2005) to all 6 digits shown there.
- For μ = 5, the non-relativistic expansion provides only four-digit precision, while the non-local representation yields (ε − μ) = 0.91531941208 with 12 correct digits for N = 50.
- For the potential V(x) = √(a² + x²) with a = μ = 1, the ground state energy (ε − μ) = 2.075870921 using N = 100.
- For the modified kinetic operator √(p̂⁴ + p̂² + μ²) with μ = a = 1, (ε − μ) = 2.724199599 using N = 100.
- For the logarithmic kinetic operator μ log[1 + p̂²/(2μ²)] with V(x) = x²/2 and μ = 10, the ground state energy ε = 20.04909725 using N = 30.

## Context

This work extends the variational sinc collocation method of Amore (2006a, 2006b) to relativistic problems, providing an alternative to the numerical matrix representation of Fulcher (1994) used with the Lagrange mesh method by Semay (2005). The paper also generalizes the approach to Green's functions and path integrals, recovering the exact free-particle result in a box and the correct ground state energy via the Feynman-Kac formula. The method enables treatment of Hamiltonians containing non-local operators, such as the spinless Salpeter equation, without resorting to non-relativistic expansions that can converge to incorrect values when the wave function is significant at momenta above the mass scale.
