# A covariant model for the nucleon spin structure

> We present the results of the covariant spectator quark model applied to the nucleon structure function $f(x)$ measured in unpolarized deep inelastic scattering, and the structure functions $g_1(x)$ and $g_2(x)$ measured in deep inelastic scattering using polarized beams and targets ($x$ is the Bjorken scaling variable). The nucleon is modeled by a valence quark-diquark structure with $S,P$ and $D$ components. The shape of the wave functions and the relative strength of each component are fixed by making fits to the deep inelastic scattering data for the structure functions $f(x)$ and $g_1(x)$. The model is then used to make predictions on the function $g_2(x)$ for the proton and neutron.

## Key claim

The covariant spectator quark model (CSQM), which includes S, P, and D orbital angular momentum components in the nucleon wave function, can describe unpolarized and polarized deep inelastic scattering data at Q² = 1 GeV², but only under the assumption that valence quarks are the relevant degrees of freedom and that gluon and sea quark effects can be neglected in a first approximation.

## Method

The nucleon is modeled as a valence quark-diquark system with S, P, and D wave components, where the quark electromagnetic structure is parametrized by Dirac and Pauli form factors. The radial wave functions and the relative strengths of the S, P, and D components are determined through a three-step fitting procedure: first to the unpolarized structure functions f_u and f_d using the MRST(02) parametrization, then to the first moments of the polarized structure functions g_1^u and g_1^d, and finally to the polarized data for Δu and Δd from the LSS(10) parametrization, all at Q² = 1 GeV². The model then predicts the spin-dependent structure function g_2(x) for the proton and neutron.

## Key numbers

- The first moment of the proton's g_1 function from experiment: Γ_1^p = 0.128 ± 0.013
- The S-state model prediction for Γ_1^p is 0.278, which overestimates the experimental value
- The fitted first moments for the quark contributions: Γ_1^u = 0.333 ± 0.039 and Γ_1^d = -0.335 ± 0.080
- The P-state and D-state mixtures in the best fit are 1% and 35% respectively
- The unpolarized structure functions satisfy ∫ dx (2xf_u + xf_d) ≈ 0.5, indicating that gluons account for the remaining 50% of the nucleon momentum

## Context

This work extends the covariant spectator quark model, previously applied to nucleon electromagnetic form factors and baryon resonances (Ramalho, 2014; Gross, 1969), into the deep inelastic scattering regime. The model addresses the long-standing "proton spin crisis" from the EMC experiments at CERN (1980s), where the quark orbital momentum contribution to the proton spin was found to be only about 30%. The results enable future predictions for g_2(x) that can be tested with more accurate data, and the model can be extended to higher Q² using DGLAP evolution equations.
