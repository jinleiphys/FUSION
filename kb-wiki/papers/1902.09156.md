# Alpha Clustering from the Quartet Model

> Alpha clustering in nuclei is considered with the quartet model (QM) where four valence nucleons (the quartet) move on the top of the core (daughter) nucleus. In the QM approach, it is assumed that the intrinsic wave function of the quartet is changed from the pure cluster configuration to the shell-model configuration when it crosses some critical radius and enters into the core nucleus. The QM approach could give not only the level scheme, the electromagnetic transition, the nuclear radius, but also the alpha-cluster formation probability. Numerical results are calculated for ${}^{20}$Ne, ${}^{44}$Ti, and ${}^{212}$Po, where a quartet moves on top of a double magic nucleus. Good agreement with experimental data and previous theoretical studies is obtained. The QM approach is a useful complement to the present phenomenological and microscopic models and could help deepen our understanding of alpha clustering across the nuclide chart.

## Key claim

The quartet model (QM) provides a unified phenomenological description of alpha clustering in alpha+closed shell nuclei, including the alpha-cluster formation probability, by assuming that the intrinsic wave function of the four valence nucleons changes abruptly from a shell-model configuration to a pure cluster configuration at a critical radius determined by the Mott density, rather than treating the parent nucleus as a pure two-body alpha-core system.

## Method

The parent nucleus is modeled as a doubly magic core plus four valence nucleons (a quartet). The quartet wave function is decomposed into a center-of-mass component and an intrinsic component; the intrinsic component is taken as a shell-model state inside a critical radius and a pure alpha-cluster state outside that radius. The center-of-mass motion is solved using a local Schrödinger equation with a Woods-Saxon-Gaussian (WSG) effective potential plus Coulomb and centrifugal terms, and the Wildermuth condition is imposed to account for Pauli blocking. The alpha-cluster formation probability is then computed as the integral of the squared center-of-mass wave function outside the critical radius.

## Key numbers

- For $^{20}$Ne, the critical radius is set to $R_\text{crit} = 1.2 R_\text{Mott}$, with $R_\text{Mott} = 3.34$ fm for $^{16}$O; the Wildermuth condition requires $G = 2n + L = 8$.
- For $^{44}$Ti, the critical radius is set to $R_\text{crit} = 1.2 R_\text{Mott}$, with $R_\text{Mott} = 4.50$ fm for $^{40}$Ca; the Wildermuth condition requires $G = 12$.
- For $^{212}$Po, the critical radius is set to $R_\text{crit} = R_\text{Mott}$, with $R_\text{Mott} = 7.74$ fm for $^{208}$Pb; the Wildermuth condition requires $G = 18$.
- The WSG potential parameters are: $V_0 = 203.3$ MeV, $a = 0.73$ fm, $\alpha = -0.478$, $\beta = 0.054$ fm$^{-2}$, with $R_D(^{20}\text{Ne}) = 3.25$ fm, $R_D(^{44}\text{Ti}) = 4.61$ fm, $R_D(^{212}\text{Po}) = 6.73$ fm.
- The Mott density for homogeneous nuclear matter is $\rho_\text{Mott} = 0.02917$ fm$^{-3}$.

## Context

This work extends the binary cluster model (Buck, 1990) by incorporating the concept of a critical radius from the quartetting wave function approach (Röpke, 2014), enabling the calculation of alpha-cluster formation probabilities that the binary cluster model cannot provide. The QM results for $^{20}$Ne and $^{44}$Ti are calibrated against antisymmetrized molecular dynamics (AMD) calculations (Kimura, 2003; Kimura, 2004), and the ground-state alpha-cluster formation probability for $^{212}$Po is compared with previous quartetting wave function approach results (Xu, 2015). The approach enables future systematic studies of alpha clustering across the nuclide chart, including excited states in heavy nuclei.
