# Relativistic corrections to the vector meson light front wave function

> We compute a light front wave function for heavy vector mesons based on long distance matrix elements constrained by decay width analyses in the Non Relativistic QCD framework. Our approach provides a systematic expansion of the wave function in quark velocity. The first relativistic correction included in our calculation is found to be significant, and crucial for a good description of the HERA exclusive $\mathrm{J}/ψ$ production data. When looking at cross section ratios between nuclear and proton targets, the wave function dependence does not cancel out exactly. In particular the fully non-relativistic limit is found not to be a reliable approximation even in this ratio. The important role of the Melosh rotation to express the rest frame wave function on the light front is illustrated.

## Key claim

The paper constructs a light front wave function for heavy vector mesons using NRQCD matrix elements, finding that the first relativistic correction (order \(v^2\)) is significant and crucial for describing HERA exclusive \(\mathrm{J}/\psi\) production data; even in cross section ratios between nuclear and proton targets, the fully non-relativistic limit is not a reliable approximation.

## Method

The authors start from the meson rest frame, where the scalar part of the wave function is expanded as a Taylor series \(\phi(\vec r) = A + B \vec r^2\) with coefficients \(A\) and \(B\) determined from NRQCD long-distance matrix elements \(\langle \mathcal{O}_1 \rangle_{\mathrm{J}/\psi}\) and \(\langle {\vec q}^{\, 2} \rangle_{\mathrm{J}/\psi}\) extracted from leptonic decay width analyses. This rest-frame wave function is transformed to the light front using a Melosh rotation, with the longitudinal momentum fraction \(z\) defined via the on-shell invariant mass \(M = \sqrt{(\kt^2 + m_{c,NR}^2)/(z(1-z))}\). The resulting light front wave function is then used in the dipole picture to compute exclusive \(\mathrm{J}/\psi\) photoproduction cross sections, employing the IPsat parametrization for the dipole-proton scattering amplitude.

## Key numbers

- \(\langle \mathcal{O}_1 \rangle_{\mathrm{J}/\psi} = 0.440^{+0.067}_{-0.055} \ \mathrm{GeV}^3\) (long-distance matrix element from decay width analysis)
- \(\langle {\vec q}^{\, 2} \rangle_{\mathrm{J}/\psi} = 0.441^{+0.140}_{-0.140} \ \mathrm{GeV}^2\) (velocity-suppressed matrix element)
- \(A = 0.213 \ \mathrm{GeV}^{3/2}\) (coefficient of the wave function at the origin, including \(\alpha_s\) correction factor)
- \(B = -0.0157 \ \mathrm{GeV}^{7/2}\) (coefficient of the \(\vec r^2\) term in the wave function expansion)
- NRQCD charm quark mass \(m_{c,NR} = 1.4 \ \mathrm{GeV}\) used in the meson and photon wave functions

## Context

This work extends the nonrelativistic treatment of vector meson wave functions (e.g., Ryskin (1992), Brodsky (1994)) by incorporating systematic relativistic corrections from NRQCD, following the matrix element extraction of Bodwin (2007). It addresses the \(\sim 30\%\) model uncertainty among phenomenological light cone wave functions noted in earlier work (Mantysaari (2017), Lappi (2013)). The resulting wave function enables more robust predictions for exclusive \(\mathrm{J}/\psi\) production at HERA, the LHC, and future facilities like the Electron Ion Collider, particularly for nuclear targets where saturation effects are probed.
