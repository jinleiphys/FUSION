# Particle Production via Strings and Baryon Stopping within a Hadronic Transport Approach

> The stopping of baryons in heavy ion collisions at beam momenta of $p_{\rm lab} = 20-160A$ GeV is lacking a quantitative description within theoretical calculations. Heavy ion reactions at these energies are experimentally explored at the Super Proton Synchrotron (SPS) and the Relativistic Heavy Ion Collider (RHIC) and will be studied at future facilities such as FAIR and NICA. Since the net baryon density is determined by the amount of stopping, this is the pre-requisiste for any investigation of other observables related to structures in the QCD phase diagram such as a first-order phase transition or a critical endpoint. In this work we employ a string model for treating hadron-hadron interactions within a hadronic transport approach (SMASH, Simulating Many Accelerated Strongly-interacting Hadrons). Free parameters of the string excitation and decay are tuned to match experimental measurements in elementary proton-proton collisions, where some mismatch in the $x_F$ distribution of protons is still present. Afterwards, the model is applied to heavy ion collisions, where the experimentally observed change of the shape of the proton rapidity spectrum from a single peak structure to a double peak structure with increasing beam energy is reproduced. Heavy ion collisions provide the opportunity to study the formation process of string fragments in terms of formation times and reduced interaction cross-sections for pre-formed hadrons. A good agreement with the measured rapidity spectra of protons and pions is achieved while insights on the fragmentation process are obtained. In the future, the presented approach can be used to create event-by-event initial conditions for hybrid calculations.

## Key claim

The paper demonstrates that a hadronic transport approach (SMASH) with a tuned string model can reproduce the experimentally observed change in the proton rapidity spectrum from a single peak to a double peak structure in heavy ion collisions at SPS energies, although some mismatch in the Feynman-x distribution of protons in elementary proton-proton collisions remains.

## Method

The authors implemented a string model for hadron-hadron interactions within the SMASH hadronic transport framework, using Pythia 8.235 for string fragmentation. Free parameters of the string excitation and decay were tuned to match experimental measurements in elementary proton-proton collisions, including the parton distribution function parameters, fragmentation function parameters, and transverse momentum width. The tuned model was then applied to heavy ion collisions, where the formation process of string fragments was studied in terms of formation times and reduced interaction cross-sections for pre-formed hadrons.

## Key numbers

- Beam momentum range studied: \(p_{\rm lab} = 20-160A\) GeV for heavy ion collisions at SPS and RHIC
- Transition region for nucleon-pion collisions: between \(\sqrt{s}=1.9\) GeV and \(2.2\) GeV
- Transition region for nucleon-nucleon collisions: between \(\sqrt{s}=4\) GeV and \(5\) GeV
- Lower \(p_T\) threshold for partonic interactions: \(1.5\) GeV
- Default value for parameter \(\beta_\mathrm{quark}\) in quark PDF: \(7.0\)
- Default value for parameter \(a_\mathrm{leading}\) in fragmentation function for leading baryons: \(0.2\)
- Default value for parameter \(b_\mathrm{leading}\) in fragmentation function for leading baryons: \(2.0\) GeV\(^{-2}\)
- Default value for parameter \(a_\mathrm{string}\) in fragmentation function for remaining hadrons: \(2.0\)

## Context

This work extends the hadronic transport approach SMASH (Weil 2016) to higher energies by incorporating a string model for particle production, addressing the long-standing problem of baryon stopping in heavy ion collisions at SPS energies that lacked quantitative theoretical description. The approach follows the same methodology as UrQMD (Bass 1998, Bleicher 1999) for soft string excitation but uses Pythia 8 for fragmentation, and it enables future hybrid calculations where the presented approach can create event-by-event initial conditions for hydrodynamic evolution, complementing efforts such as 3-fluid hydrodynamics (Ivanov 2005) and gluon saturation approaches (McLerran 2018).
