# The one loop gluon emission light cone wave function

> Light cone perturbation theory has become an essential tool to calculate cross sections for various small-$x$ dilute-dense processes such as deep inelastic scattering and forward proton-proton and proton-nucleus collisions. Here we set out to do one loop calculations in an explicit helicity basis in the four dimensional helicity scheme. As a first process we calculate light cone wave function for one gluon emission to one-loop order in Hamiltonian perturbation theory on the light front. We regulate ultraviolet divergences with transverse dimensional regularization and soft divergences with using a cut-off on longitudinal momentum. We show that when all the renormalization constants are combined, the ultraviolet divergences can be absorbed into the standard QCD running coupling constant, and give an explicit expression for the remaining finite part.

## Key claim

The authors calculate the one-loop (order \(g^3\)) correction to the light cone wave function for a single gluon emission from a quark, showing that all ultraviolet divergences can be absorbed into the standard QCD running coupling constant, and provide an explicit expression for the remaining finite part. The calculation is performed in the four-dimensional helicity (FDH) scheme with transverse dimensional regularization for UV divergences and a longitudinal momentum cutoff for soft divergences, meaning the result is scheme-dependent and the separation of divergences is specific to light cone perturbation theory.

## Method

The authors use Hamiltonian perturbation theory on the light front (light cone perturbation theory, LCPT) to compute the one-loop quark-to-quark-gluon splitting wave function. They work in an explicit helicity basis for both quarks and gluons, using the four-dimensional helicity (FDH) regularization scheme, with ultraviolet divergences regulated by performing transverse momentum integrals in \(2-2\varepsilon\) dimensions and soft divergences regulated by a cutoff \(\alpha\) on longitudinal momentum fractions. The calculation involves enumerating all \(x^+\)-ordered diagrams at one-loop order, evaluating the elementary vertices (gluon emission, gluon splitting, and instantaneous interactions) expressed in terms of helicity states and transverse momenta, and combining the results with the wave function renormalization constant \(Z_q(p^+)\) obtained from the normalization condition.

## Key numbers

- The elementary gluon emission vertex from a quark is given by \(V_{\lambda,h}^{i;j,a}(\qt,z) = \frac{-2gt^{a}_{ji}}{z\sqrt{1-z}} \left( \delta_{\lambda,h} + (1-z) \delta_{\lambda,-h}\right) \qt \cdot \epst^*_\lambda\), where \(z = k^+/p^+\) is the longitudinal momentum fraction and \(\qt = \kt - z\pt\) is the center-of-mass transverse momentum.
- The gluon splitting vertex into a quark-antiquark pair is \(A_{\lambda,h}^{a;j,i}(\qt,z) = \frac{-2gt^a_{ij}}{\sqrt{z(1-z)}} \left( z \delta_{\lambda,h} - (1-z) \delta_{\lambda,-h} \right) \qt \cdot \epst_\lambda\).

## Context

This work extends the program of one-loop calculations in light cone perturbation theory, which has been applied to NLO calculations in the color glass condensate (CGC) framework for processes such as small-\(x\) evolution equations (Balitsky 2008, Kovner 2013, Lappi 2015, Iancu 2015), inclusive DIS cross sections (Beuf 2011, Balitsky 2012), and single inclusive particle production (Chirilli 2012, Stasto 2013, Kang 2014). The explicit helicity basis and FDH scheme used here provide a simpler computational framework compared to conventional dimensional regularization approaches (e.g., Beuf 2016), and the resulting one-loop wave function enables future calculations of NLO corrections to dilute-dense scattering processes where the probe is described by a light cone wave function interacting eikonally with the target.
