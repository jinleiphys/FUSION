# Diquark properties and their role in hadrons

> Diquark correlations are important in baryons, which can be modeled as quark-diquark bound states. In addition, diquarks could play a role in non-standard hadrons such as tetraquarks and pentaquarks. Here, we obtain properties of these diquarks from the corresponding bound state equation, using a model for the effective quark-quark interaction that has proved successful in the light meson sector. Subsequently, we use the same model to estimate the masses of the lightest diquark-diquark and diquark-antidiquark states.

## Key claim

Diquark correlations in a color antitriplet configuration can be studied using the same rainbow-ladder truncation of Dyson-Schwinger equations that successfully describes light mesons, but the resulting diquark masses are more sensitive to details of the effective interaction than meson properties are.

## Method

The homogeneous Bethe-Salpeter equation for quark-quark bound states was solved in ladder truncation using a phenomenological effective interaction model previously calibrated for light mesons (Maris 1999). The color factor in the Bethe-Salpeter kernel was set to 2/3 for diquarks (compared to 4/3 for mesons), and only color antitriplet configurations were considered. The same interaction model was then applied to estimate masses of diquark-diquark and diquark-antidiquark bound states by treating the diquarks as scalar colored particles in ladder approximation.

## Key numbers

- For the parameter set with ω = 0.40 GeV and D = 0.93 GeV²: scalar ud diquark mass = 0.821 GeV, scalar qs diquark mass = 1.10 GeV, vector ud diquark mass = 1.02 GeV
- For the parameter set with ω = 0.50 GeV and D = 0.79 GeV²: scalar ud diquark mass = 0.688 GeV, scalar qs diquark mass = 0.96 GeV, vector ud diquark mass = 0.89 GeV
- The ud diquark charge radius r_ud = 0.71 fm, about 8% larger than the pion charge radius r_π = 0.66 fm
- For color-singlet scalar tetraquark states with two identical ud diquarks (mass 0.821 GeV): bound state mass = 1.068 GeV with binding energy 0.574 GeV
- For color-triplet scalar pentaquark states with two identical ud diquarks (mass 0.821 GeV): no bound state exists (X in table, forbidden by Bose statistics)

## Context

This work extends the Dyson-Schwinger/Bethe-Salpeter approach previously applied to mesons (Maris 1997, Maris 1999) to diquark correlations, which are relevant for modeling baryons as quark-diquark bound states. The results suggest that simple gluon exchange does not provide enough binding for pentaquarks in a (qq)-(qq)-(anti-q) configuration, as proposed by Jaffe (2003). The calculated diquark properties enable estimates of tetraquark and pentaquark masses, though the author notes that additional meson-like states from diquark-antidiquark configurations would mix with ordinary mesons and meson molecules, making identification difficult.
