# ${2ŒΩ}$ Double Beta Decay within the Relativistic QRPA

> We perform a self-consistent relativistic QRPA (RQRPA) calculation of $2ŒΩ\b\b$-decay based on relativistic BCS (RBCS) mean field theory results for odd-odd intermediate nuclei $^{48}$Sc, $^{76}$As, $^{82}$Br, $^{100}$Tc, $^{128}$I, and $^{130}$I. The RBCS equations that resemble the non-relativistic ones are constructed from a Dirac-Gorkov variational functional. We use the parameter set $NL1$ for the $œÉ$, $œâ$ and $œÅ$ mesons. The RQRPA equations are solved for the residual $œÄ+œÅ$ interaction by employing the same parameters used in the RBCS for the latter meson, and experimental values for the pion and nucleon. The RQRPA results for the ${2ŒΩ}\b\b$ matrix elements are similar to those obtained within the QRPA and the shell model.

## Key claim

The relativistic QRPA (RQRPA) calculation of 2ŒΩŒ≤Œ≤-decay matrix elements yields results that are of the same order of magnitude as those from the non-relativistic QRPA and the shell model, but the authors note that it is difficult to discern which calculation is better or worse given the small fraction of double Gamow-Teller strength going to the final state and the uncertainty in the quenching factor.

## Method

The authors constructed a relativistic BCS (RBCS) mean field from a Dirac-Gorkov variational functional, using the NL1 parameter set for the œÉ, œâ, and œÅ mesons. They then solved the RQRPA equations for the residual œÄ+œÅ interaction, using the same parameters for the œÅ meson as in the RBCS and experimental values for the pion and nucleon. The matrix elements were evaluated using a single set of QRPA solutions for the intermediate odd-odd nucleus, following the prescription of Cha (1983) and Krmpotic et al. (1997).

## Context

This work extends the relativistic QRPA formalism, previously applied to beta-decays and neutrino-nucleus reactions by Marketin et al. (2007) and Paar et al. (2008), to the calculation of 2ŒΩŒ≤Œ≤-decay matrix elements for the first time. The results are compared to the non-relativistic QRPA calculations of Krmpotic et al. (1994) and the shell model calculations of Caurier et al. (2011). The authors plan to apply the same RQRPA model to study neutrinoless Œ≤Œ≤-decay using the formalism of Barbero et al. (1999).
