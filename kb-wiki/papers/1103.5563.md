# The Most Negative and the Most Positive Value of the Spin Operator for a Many Fermion System

> We consider the most negative and most positive value of the spin operator for a many nucleon system. In the usual Nilsson model and in the single nucleon model this value has a narrow range.In the Nilsson model with an excited intrinsic state we can get values of the spin operator greater than one.

# The Most Negative and the Most Positive Value of the Spin Operator for a Many Fermion System

## Key claim

The range of possible expectation values of the spin operator ⟨σ⟩ for many-nucleon systems is considerably wider than the single-particle Schmidt limits (−j/(j+1) for j = L−1/2 and 1 for j = L+1/2). Values greater than one can be obtained in the Nilsson model only when more than one nucleon is unpaired, as demonstrated with an excited intrinsic state containing three unpaired particles.

## Method

The author derives a general LS-coupling formula for ⟨σ⟩ = [S(S+1)+J(J+1)−L(L+1)]/(J+1) and evaluates it for various configurations. The Nilsson model with a deformed potential is used to compute isoscalar magnetic moments and spin expectation values in both the weak deformation limit and the asymptotic limit, for mirror pairs with A=9, 21, 23, and 25. The author constructs an excited intrinsic state with three unpaired particles to demonstrate how ⟨σ⟩ can exceed one.

## Key numbers

- For a single nucleon with j = L+1/2, ⟨σ⟩ = 1; for j = L−1/2, ⟨σ⟩ = −j/(j+1)
- For A=9 (J=3/2) mirror pair ⁹Li-⁹C: ⟨σ⟩ = 1.434 (experimental)
- For A=21 (J=5/2) mirror pair ²¹F-²¹Ne: ⟨σ⟩ = 1.176 (experimental)
- For A=21 (J=3/2) mirror pair ²¹Ne-²¹Na: ⟨σ⟩ = 0.589 (experimental)
- For A=23 (J=3/2) mirror pair ²³Na-²³Mg: ⟨σ⟩ = 0.479 (experimental)
- For A=25 (J=5/2) mirror pair ²⁵Mg-²⁵Al: ⟨σ⟩ = 0.766 (experimental)
- In the asymptotic limit with three unpaired particles (Y₁₀↑, Y₁₁↑, Y₁₋₁↑): ⟨σ⟩ = 1.8
- For LS coupling with L=0, S=5/2, J=5/2: upper limit ⟨σ⟩ = 5
- For LS coupling with L=1, S=3/2, J=5/2: upper limit ⟨σ⟩ = 3
- Nilsson model deviation from Schmidt: −8.1% for A=9, −3.8% for A=21

## Context

This work extends the discussion of isoscalar magnetic moments from mirror pairs, following measurements by Kramer et al. (2011) for ²¹Mg and Matsuta et al. for A=9, which found ⟨σ⟩ values exceeding the single-particle Schmidt limits. The author clarifies that these results are not anomalous when using theoretical LS-coupling limits rather than empirical single-particle limits. The work connects to earlier studies by Wigner on supermultiplet quantum numbers, Bohr and Mottelson on the rotational model, and Ripka and Zamick on intrinsic wave functions. The results enable a more complete understanding of how deformation and multi-particle configurations affect spin expectation values in nuclear structure.
