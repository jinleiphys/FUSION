# Meson-Exchange Model for the YN Interaction

> We report on progress in the development of a microscopic model for the hyperon-nucleon interaction within the meson-exchange framework, which incorporates both one-meson as well as correlated two-meson exchange contributions. The main new feature of the model is the exchange of two correlated pions or kaons, both in the scalar-isoscalar and the vector-isovector channels. In the scalar channel it provides the main part of the intermediate-range interaction between two baryons, thereby eliminating the need for the fictitious sigma meson, which has been an unavoidable element of effective one-boson-exchange descriptions.

# Meson-Exchange Model for the YN Interaction

## Key claim
The paper reports progress on a microscopic hyperon-nucleon (YN) interaction model within the meson-exchange framework that incorporates both one-meson and correlated two-meson exchange contributions, specifically eliminating the need for a fictitious sigma meson by replacing its role with correlated two-pion and two-kaon exchange in the scalar-isoscalar channel.

## Method
The authors construct a meson-exchange model for the YN interaction that includes one-meson exchange diagrams and, as the main new feature, correlated two-meson exchange contributions from two pions or two kaons in both the scalar-isoscalar and vector-isovector channels. The correlated two-meson exchange provides the intermediate-range attraction between baryons that was previously attributed to a fictitious sigma meson in effective one-boson-exchange (OBE) descriptions.

## Context
This work extends the meson-exchange approach to the hyperon-nucleon interaction by incorporating correlated two-meson exchange, building on earlier one-boson-exchange models that required a fictitious sigma meson. The approach enables a more microscopic description of the YN interaction without introducing ad-hoc scalar mesons, and can be compared with phenomenological OBE models such as those by the Nijmegen and Jülich groups. Future work would involve applying this model to YN scattering data and hypernuclear structure calculations.
