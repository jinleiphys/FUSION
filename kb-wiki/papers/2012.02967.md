# Equivalence of generator coordinate Brink cluster model and nonlocalized cluster model and supersolidity of $α$ cluster structure in nuclei

> It is found that $α$ cluster structure has the apparently opposing dual property of crystallinity and condensation simultaneously. The mathematical equivalence of the spatially localized Brink $α$ cluster model in the generator coordinate method (GCM) and the nonlocalized cluster model (NCM), which is also called the THSR (Tohsaki-Horiuchi-Schuck-R$\ddot{\rm o}$pke) wave function based on the condensation of $α$ clusters, is shown. The latter is found to be an equivalent representation of the localized cluster model and it is a natural consequence that the many NCM (THSR) calculations reproduce the proceeding cluster model calculations using the GCM and the resonating group method (RGM). Localized cluster models, which have been successfully used for more than half a century, will continue to be very powerful. The equivalence is a manifestation of the duality of incompatible aspects: crystallinity and coherent wave nature due to condensation of $α$ clusters, i.e. the dual properties of a supersolid. The Pauli principle causes the duality. The evidence for supersolidity, the emergence of a Nambu-Goldstone mode caused by the spontaneous symmetry breaking of the global phase, is discussed

# Wiki Page: Equivalence of Generator Coordinate Brink Cluster Model and Nonlocalized Cluster Model

## Key claim
The paper demonstrates that the spatially localized Brink α cluster model in the generator coordinate method (GCM) and the nonlocalized cluster model (NCM, also called the THSR wave function) are mathematically equivalent, revealing that α cluster structure possesses the dual property of crystallinity and condensation simultaneously. The equivalence is a manifestation of supersolidity, with the Pauli principle causing the duality.

## Method
The author shows mathematical equivalence by rewriting the GCM wave function for n-α clusters using a Laplace transformation that connects the generator coordinate weight function f(R) to the NCM weight function g(μ). For the two-α cluster system of ⁸Be, the GCM wave function is expressed as an integral over the Brink wave function with a weight function f(R), which is then transformed into an integral over the NCM (THSR) wave function with weight function g(μ). The weight function g(μ) is shown to be a δ-function, proving that the GCM wave function is exactly equal to a single NCM wave function. This proof is generalized to n-α cluster systems.

## Context
This work resolves a puzzle that persisted for two decades: why NCM (THSR) calculations based on a nonlocalized, gas-like cluster picture reproduced results from the traditional localized Brink cluster model using GCM and RGM, despite the apparently opposing concepts. The paper extends the recent superfluid α cluster model work by Ohkubo (2020) that first proposed the duality interpretation. The equivalence means that localized cluster models, which have been successfully used for more than half a century as reviewed in Supplements (1972, 1980, 1998) and by Wildermuth (1977), will continue to be powerful. The finding enables understanding of very low-lying intruder 0⁺ states near α threshold energies (such as the 6.05 MeV state in ¹⁶O) as Nambu-Goldstone modes from spontaneous symmetry breaking of the global phase, analogous to rotational band states in deformed nuclei. The supersolidity interpretation connects nuclear cluster physics to recent observations of supersolids in optical lattice experiments.
