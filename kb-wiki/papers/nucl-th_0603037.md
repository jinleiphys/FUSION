# Application Of Support Vector Machines To Global Prediction Of Nuclear Properties

> Advances in statistical learning theory present the opportunity to develop statistical models of quantum many-body systems exhibiting remarkable predictive power. The potential of such ``theory-thin'' approaches is illustrated with the application of Support Vector Machines (SVMs) to global prediction of nuclear properties as functions of proton and neutron numbers $Z$ and $N$ across the nuclidic chart. Based on the principle of structural-risk minimization, SVMs learn from examples in the existing database of a given property $Y$, automatically and optimally identify a set of ``support vectors'' corresponding to representative nuclei in the training set, and approximate the mapping $(Z,N) \to Y$ in terms of these nuclei. Results are reported for nuclear masses, beta-decay lifetimes, and spins/parities of nuclear ground states. These results indicate that SVM models can match or even surpass the predictive performance of the best conventional ``theory-thick'' global models based on nuclear phenomenology.

## Key claim

Support Vector Machines (SVMs) can match or surpass the predictive performance of the best conventional "theory-thick" global nuclear models, but this claim is conditioned on the use of a carefully chosen kernel (the pA kernel, a sum of polynomial and ANOVA kernels) and on separate modeling of nuclei grouped by even-oddness classes (EE, EO, OE, OO), which introduces minimal domain knowledge.

## Method

The authors applied Support Vector Machine regression and classification to predict nuclear mass excess, beta-decay lifetimes, and ground-state spins/parities as functions of proton number Z and neutron number N. Separate SVM models were constructed for each even-oddness class (EE, EO, OE, OO) using a custom pA kernel (sum of polynomial and ANOVA kernels), with data partitioned into training, validation, and test sets (typically 80:10:10 or 90:5:5) drawn from the AME03 mass compilation and the Brookhaven NNDC database.

## Key numbers

- Overall SVM mass model (80:10:10 partition): training set rms error 0.35 MeV (1597 nuclides), validation set 0.58 MeV (200 nuclides), test set 0.71 MeV (200 nuclides)
- SVM mass model (90:5:5 partition): test set rms error 0.49 MeV (100 nuclides), compared to FRDM test set error 0.70 MeV (158 nuclides) and best MLP test set error 0.71 MeV (158 nuclides)
- For EE class (531 nuclides), SVM mass error 0.42 MeV compared to 1.5 MeV obtained by Bertsch et al.
- Average rms error of 38 Q_alpha estimates for superheavy elements (Z=110-118) is 0.82 MeV, average absolute error 0.64 MeV
- SVM beta-decay halflife model (overall): training set rms error 0.47 (672 nuclides), validation set 0.46 (83 nuclides), test set 0.53 (83 nuclides), with mean absolute errors of 0.16, 0.19, and 0.27 respectively
- SVM parity model: 94% correct on test set (averaged over even-oddness classes)
- SVM spin model: predictive quality in the 80-90 percentile range, compared to 60% accuracy for macroscopic/microscopic approach of Moller et al. on odd-A nuclei (428 correct out of 713)

## Context

This work extends the application of statistical learning methods to nuclear systematics, building on earlier neural network (MLP) studies by Clark and collaborators (2000-2005) and on the SVM methodology of Vapnik (1995, 1998). The SVM models are compared against the Finite-Range Droplet Model (FRDM) of Moller, Nix, and collaborators (1995) and the HFB2 density functional model of Pearson (2001), as well as MLP models trained by enhanced backpropagation. The results enable data-driven predictions for nuclei far from stability, relevant to radioactive ion-beam facilities and nucleosynthesis modeling, and suggest that hybrid approaches combining SVMs with traditional models (e.g., modeling residuals from FRDM) can capture 1/2 to 2/3 of the physical regularities missed by theory-thick models.
