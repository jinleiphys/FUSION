# Parametric Matrix Models

> We present a general class of machine learning algorithms called parametric matrix models. In contrast with most existing machine learning models that imitate the biology of neurons, parametric matrix models use matrix equations that emulate physical systems. Similar to how physics problems are usually solved, parametric matrix models learn the governing equations that lead to the desired outputs. Parametric matrix models can be efficiently trained from empirical data, and the equations may use algebraic, differential, or integral relations. While originally designed for scientific computing, we prove that parametric matrix models are universal function approximators that can be applied to general machine learning problems. After introducing the underlying theory, we apply parametric matrix models to a series of different challenges that show their performance for a wide range of problems. For all the challenges tested here, parametric matrix models produce accurate results within an efficient and interpretable computational framework that allows for input feature extrapolation.

## Key claim
Parametric matrix models (PMMs) are universal function approximators that learn governing equations from data using matrix equations, but the paper's claim of superior performance is qualified by the fact that the comparison uses number of trainable parameters as a proxy for efficiency rather than direct computational cost measurements, and the authors acknowledge that such a comparison would be "heavily biased in favor of methods with the benefit of decades of software optimization."

## Method
The authors define PMMs as sets of matrix equations with trainable parameters, where primary matrices (Hermitian or unitary) and secondary matrices have elements that are analytic functions of input features. Outputs are formed as bilinears of eigenvectors of primary matrices with secondary matrices, analogous to quantum mechanical observables and transition probabilities. The models are trained using gradient-based optimization, with gradients computed efficiently via first-order perturbation theory, and parameter optimization is accelerated by applying random unitary transformations to the matrices and eigenvectors.

## Key numbers
- For multivariable regression on 17 benchmarks, PMMs obtained the lowest normalized mean absolute error for 15 of the 17 benchmarks, using "an order of magnitude fewer trainable real parameters" than the corresponding Multilayer Perceptron (MLP) for each benchmark
- For the quantum computing Trotter extrapolation problem with a 10-spin Heisenberg model with DM interactions, the PMM was "more than an order of magnitude better for the ground state $E_0$" compared to MLP and polynomial interpolation at $dt=0$
- For the ALMG model with $N=1000$ particles and anharmonicity parameter $\alpha = -0.6$, a $9\times9$ PMM accurately reproduced the second-order ground-state quantum phase transition at $\xi = 0.2$
- On MNIST Digits, a pure PMM achieved 97.38% accuracy with 4990 trainable float parameters, while a ConvPMM achieved 99.10% accuracy with 129416 parameters
- On Fashion MNIST, a PMM achieved 88.58% accuracy with 16744 parameters, and a ConvPMM achieved 90.94% accuracy with 278280 parameters
- On CIFAR-10 with hybrid transfer learning using ResNet50, a PMM head achieved 83.11% accuracy with 95176 trainable parameters, compared to an FNN head with 271178 parameters achieving 83.68% accuracy

## Context
This work introduces PMMs as a new class of machine learning algorithms that differ from most existing models that imitate the biology of neurons (Hopfield 1988, Hinton 1992, LeCun 2015). PMMs extend the principles of model order reduction and reduced basis methods such as eigenvector continuation (EC) (Demol 2019, Konig 2019, Ekstrom 2019, Furnstahl 2020, Duguet 2023) and proper orthogonal decomposition (Berkooz 1993). The paper demonstrates that PMMs can outperform standard techniques including Kernel Ridge Regression, Multilayer Perceptrons, and polynomial interpolation for scientific computing problems, while also matching or exceeding the performance of efficient neural network architectures (Granmo 2019, Mazzia 2021) on image classification benchmarks. The approach enables future work on implementing PMMs using Hamiltonian learning techniques in quantum computing (Granade 2012, Wiebe 2014, Wang 2017).
