# Quantum Monte Carlo simulation in the canonical ensemble at finite temperature

> A quantum Monte Carlo method with non-local update scheme is presented. The method is based on a path-integral decomposition and a worm operator which is local in imaginary time. It generates states with a fixed number of particles and respects other exact symmetries. Observables like the equal-time Green's function can be evaluated in an efficient way. To demonstrate the versatility of the method, results for the one-dimensional Bose-Hubbard model and a nuclear pairing model are presented. Within the context of the Bose-Hubbard model the efficiency of the algorithm is discussed.

## Key claim

The paper presents a quantum Monte Carlo method that directly samples the canonical ensemble (fixed particle number) at finite temperature, using a worm operator that is local in imaginary time and commutes with the interaction part of the Hamiltonian, thereby eliminating "bounces" and achieving unit acceptance probability for all moves.

## Method

The method decomposes the evolution operator via an exact perturbative expansion in the interaction V around a diagonal part H0, and inserts a worm operator A that is local in imaginary time and commutes with V. The worm operator moves through imaginary time by drawing exponential time shifts; when it encounters an interaction vertex, it can pass through, create, or annihilate the vertex, with probabilities chosen so that all Metropolis-Hastings acceptance factors become unity. The algorithm is applied to the one-dimensional Bose-Hubbard model (with H0 = (U/2) Σ n_i(n_i-1) and V = t Σ b†_i b_j) and to a nuclear pairing model.

## Key numbers

- For the one-dimensional Bose-Hubbard model: simulations used N_s = 128 sites at density exactly one particle per site, inverse temperature β = 128 t^{-1}.
- Efficiency comparison used N_s = 32 sites, β = 32 t^{-1}, filling one boson per site, with 40 Markov chains each running 600 seconds on a Pentium III processor.
- The condensed fraction ρ_c and superfluid fraction ρ_s were computed as functions of U/t; the superfluid fraction is given by ρ_s = ⟨W²⟩ N_s² / (2 t N β).
- For diagonal configurations with E_L = E_R, the algorithm parameter R_LR takes the constant value 2φ, where φ is a small positive parameter satisfying 0 < φ < N_DD' for all diagonal configurations.

## Context

This work extends continuous-time world-line algorithms (Beard, Prokofiev) and loop algorithms (Evertz, Kawashima) from the grand-canonical ensemble to the canonical ensemble, addressing inefficiencies in prior approaches that either rejected particle-number-changing updates (Evertz, 2003) or post-selected configurations with the correct particle number (Roos, 1999). The method enables direct simulation of systems where particle conservation is essential, such as the superfluid-to-Mott transition at constant filling in the Bose-Hubbard model, mesoscopic superconducting grains (Dukelsky, 1999; von Delft, 2001), and atomic nuclei (Koonin, 1997; Rombouts, 1998). The algorithm also allows inclusion of other symmetry projections and generates configurations with different winding numbers, in contrast to the local world-line method of Hirsch (1981).
