# Cluster form factor calculation in the ab initio no-core shell model

> We derive expressions for cluster overlap integrals or channel cluster form factors for ab initio no-core shell model (NCSM) wave functions. These are used to obtain the spectroscopic factors and can serve as a starting point for the description of low-energy nuclear reactions. We consider the composite system and the target nucleus to be described in the Slater determinant (SD) harmonic oscillator (HO) basis while the projectile eigenstate to be expanded in the Jacobi coordinate HO basis. This is the most practical case. The spurious center of mass components present in the SD bases are removed exactly. The calculated cluster overlap integrals are translationally invariant. As an illustration, we present results of cluster form factor calculations for <5He|4He+n>, <5He|3H+d>, <6Li|4He+d>, <6Be|3He+3He>, <7Li|4He+3H>, <7Li|6Li+n>, <8Be|6Li+d>, <8Be|7Li+p>, <9Li|8Li+n> and <13C|12C+n>, with all the nuclei described by multi-hbar Omega NCSM wave functions.

## Key claim

The paper derives algebraic expressions for channel cluster form factors (overlap integrals) that enable spectroscopic factor calculations and provide a starting point for low-energy reaction descriptions using ab initio no-core shell model (NCSM) wave functions, but the method is limited to projectiles of up to four nucleons and requires identical harmonic oscillator frequencies for all participating nuclei (or differing by one unit for opposite parity states).

## Method

The author derives formulas for cluster overlap integrals where the composite A-nucleon system and target (A-a)-nucleon system are described in a Slater determinant harmonic oscillator basis, while the projectile eigenstate (a ≤ 4 nucleons) is expanded in a Jacobi coordinate harmonic oscillator basis. Spurious center-of-mass components present in the Slater determinant bases are removed exactly using harmonic oscillator bracket transformations, yielding translationally invariant cluster form factors. The final expressions relate the overlap integrals to matrix elements of creation operators between the composite and target eigenstates, which can be computed by shell model codes.

## Key numbers

- The center-of-mass correction factor for single-nucleon projectiles is given by (A/(A-1))^((2n+l)/2), derived from the harmonic oscillator bracket ⟨nl00l|00nll⟩_{1/(A-1)} = (-1)^l ((A-1)/A)^((2n+l)/2)
- For the ⟨⁵He|⁴He+n⟩ system, calculations used basis sizes from N_max=4 (4ħΩ) to N_max=12 (12ħΩ) with the CD-Bonn 2000 NN potential and HO frequency ħΩ=16 MeV
- The HO frequency dependence was tested over the range ħΩ=13-19 MeV for the ⟨⁵He|⁴He+n⟩ system
- For the ⟨⁵He|³H+d⟩ system, tests were performed by switching target and projectile using both the two-nucleon and three-nucleon formulas

## Context

This work extends the NCSM approach (Navratil 2000) toward describing low-energy nuclear reactions, building on earlier techniques for spurious center-of-mass removal (Tassie 1957, Dieperink 1974, Philpott 1977) and cluster overlap calculations (Smirnov 1961, Rotter 1968, Kurath 1973) that were typically limited to single major harmonic oscillator shells. The method enables calculation of spectroscopic factors and provides a foundation for a resonating group method (RGM) approach to obtain cross sections, with applications demonstrated for systems including ⁵He, ⁶Li, ⁶Be, ⁷Li, ⁸Be, ⁹Li, and ¹³C.
