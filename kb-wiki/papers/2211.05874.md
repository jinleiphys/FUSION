# Toward a description of the centrality dependence of the charge balance function in the HYDJET++ model

> Data from the Large Hadron Collider on the charge balance function in Pb+Pb collisions at center-of-mass energy 2.76~TeV per nucleon pair are analyzed and interpreted within the framework of the \hydjet++ model. This model allows us to qualitatively reproduce the experimentally observed centrality dependence of the balance function widths at relatively low transverse momentum intervals due to the different charge creation mechanisms in soft and hard processes. However, a fully adequate description of the balance function in these intervals implies an essential modification of the model by including exact charge conservation via the canonical rather than the grand canonical ensemble. A procedure is proposed for introducing charge correlations into the thermal model without changing other model parameters. With increasing transverse momenta, the default model results describe the experimental data much better because the contribution of the soft component of the model is significantly reduced in these transverse momentum intervals. In practical terms, there is a transition to a single source of charge correlations, namely, charge correlations in jets in which exact charge conservation holds at each stage.

## Key claim

The HYDJET++ model can qualitatively reproduce the experimentally observed centrality dependence of the charge balance function widths at relatively low transverse momentum intervals by enhancing the hard component contribution in peripheral collisions, but a fully adequate description requires explicit inclusion of exact charge conservation via a canonical rather than grand canonical ensemble approach, which the authors implement for the first time in such a Monte Carlo event generator.

## Method

The authors used the two-component HYDJET++ model (version 2.4) to simulate Pb+Pb collisions at 2.76 TeV per nucleon pair, separating particle production into a soft hydro-type thermal component and a hard jet component from in-medium multiparton fragmentation. They calculated the charge balance function B(Δη) and its width ⟨Δη⟩ for charged hadrons in the pseudorapidity interval |η| < 0.8, comparing default model results with ALICE data across centrality classes. To improve agreement at low transverse momenta (0.3 < pT < 1.5 GeV/c), they modified the generation procedure of soft direct hadrons by randomly removing half of all charged particles in each event and adding antiparticles with opposite charge, using Gaussian distributions in η and φ with fitted dispersions to introduce exact charge conservation locally.

## Key numbers

- Center-of-mass energy: 2.76 TeV per nucleon pair for Pb+Pb collisions
- Transverse momentum interval for low-pT analysis: 0.3 < pT < 1.5 GeV/c
- Default model parameter pT^min = 8.2 GeV/c for 0-5% centrality; modified values: 6.7 GeV/c for 20-30% centrality, 5.15 GeV/c for 50-60% centrality
- Soft component scaling factor fs: 1.0 for 0-5% centrality, 0.75 for 20-30% centrality, 0.395 for 50-60% centrality
- Fitted Gaussian dispersion in pseudorapidity for most central collisions: ση(0) = 0.35
- High transverse momentum intervals studied: (1) 2.0 < pT,assoc < 3.0 < pT,trig < 4.0 GeV/c and (2) 3.0 < pT,assoc < 8.0 < pT,trig < 15.0 GeV/c

## Context

This work extends previous studies of the charge balance function at RHIC by the STAR Collaboration (2003, 2010, 2016) and at LHC by the ALICE Collaboration (2013, 2016), where the width was observed to decrease with increasing collision centrality. The authors note that this centrality dependence is not reproduced by other event generators such as HIJING and AMPT, and previous HYDJET++ studies of the soft component alone using the FASTMC generator (Fu, 2011) showed widths close to STAR data but underestimated the balance function magnitude. The proposed modification enables future quantitative studies of charge correlations in heavy-ion collisions and can be applied to other statistical model approaches without changing single-particle spectra or other free parameters.
