# From Extraction of Nucleon Resonances to LQCD

> The intrinsic difficulties in extracting the hadron resonances from reaction data are illustrated by using several exactly soluble $ππ$ scattering models. The finite-volume Hamiltonian method is applied to predict spectra using two meson-exchange Hamiltonians of $πN$ reactions. Within a three-channel model with $πN$, $πΔ$ and $σN$ channels, we show the advantage of the finite-volume Hamiltonian method over the approach using the Lüscher formula to test Lattice QCD calculations aimed at predicting nucleon resonances. We discuss the necessary steps for using the ANL-Osaka eight-channel Hamiltonian to predict the spectra for testing the LQCD calculations for determining the excited nucleon states up to invariant mass $W= 2 $ GeV.

## Key claim

The extraction of nucleon resonances from reaction data is inherently model-dependent, and the finite-volume Hamiltonian method offers a practical advantage over the multi-channel Lüscher formula for testing Lattice QCD calculations because it requires only a single volume size to test the LQCD spectrum, rather than several different volumes.

## Method

The authors first illustrate the model dependence of resonance extraction by constructing three exactly soluble ππ scattering models (Models I-A, I-B, I-C) with different form-factor parametrizations and fitting them to synthetic data generated from one model, then to real empirical ππ data (Models II-A, II-B, II-C). They then apply the finite-volume Hamiltonian method to a three-channel model (πN, πΔ, σN) derived from the ANL-Osaka eight-channel Hamiltonian, solving the Schrödinger equation in a periodic box to predict finite-volume spectra, and compare this approach to the multi-channel Lüscher formula.

## Key numbers

- Model I-A pole position for the first resonance: 639.3 − i158.9 MeV; Model I-B: 637.8 − i159.9 MeV; Model I-C: 634.5 − i156.2 MeV.
- Model I-A pole position for the second resonance: 1000.30 − i8.89 MeV; Model I-B: 1000.14 − i8.88 MeV; Model I-C: 1000.04 − i8.83 MeV.
- When fitting empirical ππ data, Model II-A gives a pole at 523.7 − i264.6 MeV (χ²=40), Model II-B at 597.0 − i217.1 MeV (χ²=36), and Model II-C at 672.3 − i292.0 MeV (χ²=43).
- The three-channel model yields extracted resonance poles at MR = 1353.5 − i38.3 MeV for P11 and MR = 1211.9 − i52.8 MeV for P33; the P33 value is close to the SL model value of MR = 1216.4 − i50.0 MeV.
- For the three-channel model at E = 1440 MeV, six different L values are needed to apply the multi-channel Lüscher formula, whereas the finite-volume Hamiltonian method requires only one L.

## Context

This work extends the ANL-Osaka multi-channel analysis (Lee et al., ANL-Osaka) and the SL model (Sato and Lee, 1996) by demonstrating how the finite-volume Hamiltonian method (developed in Refs. ad-1, ad-2) can connect realistic reaction models to Lattice QCD calculations. It contrasts with the approach using the Lüscher formula (Lüscher, 1986, 1991) and with K-matrix methods such as those used in the JLab LQCD analysis, showing that the finite-volume Hamiltonian method requires fewer LQCD volume sizes to test resonance properties. The work enables future tests of LQCD calculations for nucleon resonances up to invariant mass W = 2 GeV using the full ANL-Osaka eight-channel Hamiltonian.
