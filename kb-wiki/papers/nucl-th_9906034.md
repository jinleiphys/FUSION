# Two-Particle Correlations from the q-Boson Viewpoint

> We propose and develop to some extent a novel approach, which allows us to effectively describe, for relativistic heavy-ion collisions, the empirically observed deviation from unity of the intercept λ(i.e. the measured value corresponding to zero relative momentum {\bf p} of two registered identical pions or kaons) of the two-particle correlation function C(p,K). The approach is based on the use of two versions of the so-called q-deformed oscillators and the corresponding picture of ideal gases of q-bosons. By these techniques the intercept λis put into direct correspondence with the deformation parameter q. For fixed deformation strength, the model predicts dependence of the intercept λon the pion pair mean momentum {\bf K}.

## Key claim

The paper proposes that the empirically observed deviation from unity of the intercept λ of the two-particle correlation function for identical pions or kaons in relativistic heavy-ion collisions can be effectively described by modeling the particles as q-deformed bosons (q-bosons), where the deformation parameter q is directly related to λ. The model predicts a dependence of λ on the pair mean momentum K for a fixed deformation strength, which is a condition not stated in the abstract.

## Method

The authors construct an ideal gas model of q-bosons using two versions of q-deformed oscillator algebras (type A and type B). For type A, the deformation parameter q is real and lies in the interval -1 ≤ q ≤ 1; for type B, q is taken as a pure phase q = exp(iθ). They compute thermal averages of the number operators and two-particle number operators for each type, using the Hamiltonian H = Σ_i ω_i N_i, where N_i is the number operator for mode i. From these averages, they derive expressions for the intercept λ = (⟨a_i^† a_i^† a_i a_i⟩ / ⟨a_i^† a_i⟩²) − 1 as a function of q, temperature T, particle mass m, and mean momentum |K|.

## Key numbers

- For type A q-bosons, the intercept is given by λ = q (e^{βω} − 1) / (e^{βω} − q²), where ω = √(m² + K²) and β = 1/T.
- In the limit T → 0 or |K| → ∞, the relation simplifies to q = λ for type A oscillators, and 2 cos θ = λ + 1 for type B oscillators.
- For pions with mass m_π = 139.57 MeV and temperature T = 120 MeV, the deformation angle θ = 24° produces a q-distribution function that lies between the Bose-Einstein and Maxwell-Boltzmann distributions.
- Two critical values of the dimensionless variable w = βω are identified: w₀ ≈ 0.481 and w′₀ ≈ 0.696, which for pions at |K| = 0 correspond to temperatures T₀ = 290.0 MeV and T′₀ = 200.5 MeV.
- For kaons, the empirical intercept λ ≈ 0.88 corresponds to a smaller deformation than for pions.

## Context

This work extends the application of q-deformed algebras to heavy-ion collision phenomenology, following earlier uses of q-deformed algebras in hadron scattering and mass sum rules by Gavrilik and others (1990s). The approach contrasts with conventional explanations for the reduced intercept, such as resonance decays studied by Heiselberg (1996) and Heinz (1996). The model enables a new method to extract the deformation parameter from two-particle correlation data and predicts a momentum dependence of λ that can be tested experimentally; the authors suggest future work on three-particle correlations and fermion correlations using the same q-techniques.
