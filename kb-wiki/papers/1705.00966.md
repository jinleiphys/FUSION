# Neutron Parton Structure And The Light-Front Spectral Function Of 3He

> Semi-inclusive deep inelastic electron scattering off polarized 3He is studied in a non-relativistic framework, using a distorted spin-dependent spectral function for 3He to take care of the final state interaction between the observed pion and the remnant. A simple procedure is shown to be valuable for the extraction of Sivers and Collins asymmetries for the neutron fromthe corresponding asymmetries for 3He. To extend this study in a relativistic framework, a novel approach for a Poincaré covariant description of nuclear dynamics is presented, based on the light-front Hamiltonian dynamics. The key quantity is the light-front spectral function, where both normalization and momentum sum rule can be satisfied at the same time. Preliminary results are discussed for an initial analysis of the role of relativity in the EMC effect in 3He, and the generalization of the procedure for the extraction of neutron asymmetries within the light-front dynamics is outlined.

# Neutron Parton Structure And The Light-Front Spectral Function Of 3He

## Key claim

Semi-inclusive deep inelastic scattering (SIDIS) off polarized 3He can be used to extract neutron Sivers and Collins asymmetries using a simple linear formula, even when final-state interactions (FSI) are included via a generalized eikonal approximation, because the FSI effects on effective polarizations and dilution factors compensate each other to a large extent. A novel Poincaré covariant light-front spectral function for 3He, which satisfies both normalization and momentum sum rules simultaneously, yields preliminary results suggesting that relativistic effects improve the description of the EMC effect compared to non-relativistic approaches.

## Method

The authors studied SIDIS off polarized 3He using a non-relativistic distorted spin-dependent spectral function that incorporates final-state interactions between the observed pion and the remnant via a generalized eikonal approximation (GEA) with a Glauber operator. They then developed a Poincaré covariant light-front spectral function within light-front Hamiltonian dynamics, using the Bakamjian-Thomas construction to embed non-relativistic nuclear phenomenology (specifically the AV18 NN interaction) into a relativistic framework, with the tensor product of a plane wave for the struck nucleon and a fully interacting intrinsic state for the spectator pair to implement macrocausality.

## Key numbers

- Effective polarization for the neutron in 3He: p_n = 0.878 (within the impulse approximation)
- Effective polarization for the proton in 3He: p_p = -0.024 (within the impulse approximation)
- FSI changes the effective polarizations p_p and p_n by 10-15%, but the products p_p * d_p and p_n * d_n remain essentially unchanged
- For the EMC effect calculation, average values used for the intrinsic momentum of the (23) pair in the three-body channel: <k_23> = 113.53 MeV for the proton and <k_23> = 91.27 MeV for the neutron
- The momentum sum rule for the light-front spectral function yields MSR = M_A / (A * m), with the calculated value MSR_calc = 0.3331 for 3He
- The SIDIS calculations were performed at initial electron energy E = 8.8 GeV and Q^2 = 5.73 (GeV/c)^2

## Context

This work extends previous non-relativistic studies of neutron structure extraction from 3He targets (Kaptari, Del Dotto, Kaptari1) by including final-state interactions through a distorted spectral function, and then generalizes to a Poincaré covariant framework using light-front Hamiltonian dynamics. The approach builds on the Bakamjian-Thomas construction (Baka) and the successful non-relativistic phenomenology of the AV18 NN interaction (AV18). The light-front spectral function enables a fully relativistic description of deep inelastic scattering, SIDIS, and deeply virtual Compton scattering off 3He, with the EMC effect calculation compared to experimental data from Seely and the approach of Sauer. The next steps include a full calculation of the EMC effect with exact three-body contributions and the introduction of FSI within the light-front framework.
