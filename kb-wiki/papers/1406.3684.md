# Elementarity of composite systems

> The "compositeness" or "elementarity" is investigated for s-wave composite states dynamically generated by energy-dependent and independent interactions. The bare mass of the corresponding fictitious elementary particle in an equivalent Yukawa model is shown to be infinite, indicating that the wave function renormalization constant Z is equal to zero. The idea can be equally applied to both resonant and bound states. In a special case of zero-energy bound states, the condition Z = 0 does not necessarily mean that the elementary particle has the infinite bare mass. We also emphasize arbitrariness in the "elementarity" leading to multiple interpretations of a physical state, which can be either a pure composite state with Z = 0 or an elementary particle with Z \ne 0. The arbitrariness is unavoidable because the renormalization constant Z is not a physical observable.

## Key claim

The paper shows that any s-wave composite state dynamically generated by an energy-dependent interaction (such as the Weinberg-Tomozawa term) can be represented by a fictitious elementary Yukawa particle with wave function renormalization constant Z = 0, but this compositeness condition is not unique: the same physical state can be interpreted either as a pure composite state with Z = 0 or as an elementary particle with Z ≠ 0, because Z is not a physical observable and depends on the chosen model.

## Method

The authors construct an equivalent Yukawa model for composite states generated by energy-dependent interactions (Weinberg-Tomozawa type) by introducing a constant shift δ in the inverse interaction kernel, then taking the limit δ → 0. They compare the scattering amplitudes of the composite model and the Yukawa model, deriving the wave function renormalization constant Z from the dressed propagator of the fictitious elementary particle. The analysis is extended to general energy-dependent interactions and to the case of zero-energy bound states, and the arbitrariness of Z is demonstrated using the sigma (σ(500)) resonance in the sigma model as an example.

## Key numbers

- The Weinberg-Tomozawa interaction is given by v(s) = -(1/f²)(s - m²) with f = 92.4 MeV and m = 140 MeV (pion mass) used in numerical examples.
- The bare mass of the fictitious elementary particle m_fic² = f²/δ + m² diverges as δ → 0, leading to Z = 0 for composite states generated by the WT interaction.
- For the sigma model example, the bare mass M₀ = 500 MeV is used for the explicit pole term in the interaction kernel v(s) = -(1/f²)(s - m²) + (x/f²)(s - m²)²/(s - M₀²).
- For the case x = 1 (sigma model), the equivalent Yukawa pole term has finite bare coupling g₀² = (s - m²)(M₀² - m²)/f², giving non-zero Z, in contrast to the x = 0 case (pure WT) where Z = 0.

## Context

This work extends the compositeness condition Z = 0 originally formulated by Lurie (1964) and Weinberg (1962, 1965) for bound states with energy-independent interactions, applying it to resonant states generated by energy-dependent interactions like the Weinberg-Tomozawa term. The paper contrasts with PhysRevC.85.015201, where the wave function renormalization constant for a composite state generated by the WT interaction was found to be non-zero, because that work used a Yukawa model with constant coupling rather than the energy-dependent coupling introduced here. The results enable a clearer understanding of the model-dependence of compositeness measures and show that the elementarity of states like the σ(500) resonance cannot be determined uniquely from experiment without specifying a model with a definite cutoff scale.
