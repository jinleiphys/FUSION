# Universal results from an alternate random matrix model for QCD with a baryon chemical potential

> We introduce a new non-Hermitian random matrix model for QCD with a baryon chemical potential. This model is a direct chiral extension of a previously studied model that interpolates between the Wigner-Dyson and Ginibre ensembles. We present exact results for all eigenvalue correlations for any number of quark flavors using the orthogonal polynomial method. We also find that the parameters of the model can be scaled to remove the effects of the chemical potential from all thermodynamic quantities until the finite density phase transition is reached. This makes the model and its extensions well suited for studying the phase diagram of QCD.

## Key claim

The paper introduces a new non-Hermitian random matrix model for QCD with a baryon chemical potential that is exactly solvable for all eigenvalue correlations, but the model's physical accuracy is limited to chemical potentials below the critical value μ_c = 1, where it shows no change in thermodynamic quantities until the finite density phase transition is reached.

## Method

The author constructs a chiral random matrix model based on the Dirac operator form, replacing the constant γ₀ matrix elements of Stephanov's model with random matrices B. Using the orthogonal polynomial method extended to the complex plane, the eigenvalue representation of the partition function is derived and solved exactly for any number of quark flavors. The quenched (N_f=0) case yields Laguerre polynomials as the orthogonal polynomials, and unquenched results are obtained through an iterative biorthogonal polynomial construction.

## Key numbers

- The model exhibits a first-order phase transition at μ = 1, where for μ > 1 and α > 0 the chiral condensate vanishes in the large N limit
- The microscopic kernel for the quenched case is given by an integral expression involving Bessel functions and the parameter μ_s² = 2Nμ²
- The condition for agreement with Akemann's model in the microscopic limit is |z_s_min|² ≫ 4 μ_s², where z_s_k = ΣV z_k
- The orthogonal polynomials are p_k⁰(z) = ((μ²-1)/(αN))^k k! L_k^ν(αN z²/(1-μ²)) with normalization h_k⁰ = π μ² (1+μ²)^{2k+ν} k! (k+ν)! / (2 (αN)^{2k+ν+2})

## Context

This work extends the random matrix approach to QCD with chemical potential, improving upon Stephanov's model (1996) which showed unphysical behavior in baryon density and could not be solved exactly for eigenvalue correlations. The model agrees with Akemann's eigenvalue-based model (2001) in the microscopic limit for sufficiently small chemical potential, but is more physically motivated as it is directly based on the Dirac operator form. The results enable future studies of the full QCD phase diagram by including temperature, and could be extended to random banded matrices as done by Guhr and Osborn (2003) for the μ=0 case.
