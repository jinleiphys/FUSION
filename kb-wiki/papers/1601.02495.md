# Nonclassical Particle Transport in the 1-D Diffusive Limit

> This paper provides numerical results that demonstrate the validity of the nonclassical diffusion approximation to the nonclassical transport equation in certain 1-D diffusive systems. This result provides a more solid foundation in which to improve this theory for relevant nuclear applications.

## Key claim

The paper provides numerical results demonstrating for the first time that the solution of the nonclassical transport equation is well-approximated by the solution of the nonclassical diffusion equation in certain 1-D diffusive random periodic systems, but only in the asymptotic limit where the system is optically thick and the scattering ratio approaches unity.

## Method

The authors perform an asymptotic analysis of the nonclassical linear Boltzmann equation in 1-D rod geometry (particles moving only in directions mu = +/- 1) with isotropic scattering and monoenergetic transport. They scale the total cross section as O(1), the absorption and source as O(epsilon^2), and the spatial derivative as O(epsilon), then derive a solvability condition that yields the nonclassical diffusion equation. They validate the theory by solving the nonclassical transport equation numerically using a first-order HLL finite volume method in pseudo-time s and space x, with source iteration, for a finite random periodic system of alternating solid and void layers of equal width.

## Key numbers

* The path-length distribution function p(s) for the 1-D random periodic system has first moment <s> = 2 and second moment <s^2> = 5 + 2e/(e-1) approximately equal to 8.1640.
* Three problems are considered with M = 10, 15, and 20, where M = 1/epsilon and the total system width is 2X = 2M (since each layer width ell = 1).
* Source iteration required 299 iterations for c = 0.99 (M=10), 579 iterations for c = 224/225 approximately 0.9956 (M=15), and 936 iterations for c = 0.9975 (M=20).
* The numerical mesh used spatial interval Delta x = 1/400 and a CFL number of 0.5 giving Delta s = 1/800, with integration cutoff at s_max = 45.

## Context

This work extends the nonclassical transport theory developed by Vasques (2014) for inhomogeneous random media with nonexponential attenuation, specifically addressing Pebble Bed reactor cores where pebble locations are spatially correlated. The paper provides the first computational validation of the asymptotic theory showing that the nonclassical diffusion equation approximates the nonclassical transport equation in the diffusive limit. This enables future improvements to the nonclassical model for nuclear reactor applications, building on the HLL finite volume method introduced by Krycki (2013).
