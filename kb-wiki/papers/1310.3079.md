# Polyakov loop, Hadron Resonance Gas Model and Thermodynamics of QCD

> We summarize recent results on the hadron resonance gas description of QCD. In particular, we apply this approach to describe the equation of state and the vacuum expectation value of the Polyakov loop in several representations. Ambiguities related to exactly which states should be included are discussed.

## Key claim

The Hadron Resonance Gas (HRG) model can describe the Polyakov loop in the fundamental representation of QCD, but only if a sufficiently complete spectrum of heavy-light hadrons (beyond the Particle Data Group listings) is included; the approach is affected by ambiguities regarding which color-singlet configurations are truly irreducible, and these ambiguities only impact non-conventional (exotic) hadron states.

## Method

The authors apply the HRG model to compute the QCD trace anomaly and the Polyakov loop expectation value in several color representations, using both the experimentally known hadron spectrum from the Particle Data Group and theoretical spectra from the Relativized Quark Model (RQM) and the MIT bag model. They also develop a Polyakov-Constituent Quark Model (PCM) with local, quantum Polyakov loop variables, expanding observables in the number of dynamical constituents (quarks, antiquarks, gluons) to compare two prescriptions for removing spurious reducible color-singlet configurations: retaining only irreducible clusters versus taking a quotient of partition functions.

## Key numbers

* The HRG model describes the trace anomaly \((\epsilon-3P)/T^4\) for temperatures \(T < 180\ \mathrm{MeV}\), as compared to lattice data from Bazavov (2009) and Borsanyi (2010).
* The cumulative number of hadrons with \(u,d,s\) quarks from the RQM follows an exponential form \(n(M) \simeq A e^{M/T_H}\) with \(A = 0.80\) and Hagedorn temperature \(T_H = 260\ \mathrm{MeV}\) in the mass range \(500\ \mathrm{MeV} < M < 2300\ \mathrm{MeV}\).
* The Polyakov loop sum rule in the fundamental representation is saturated by the RQM spectrum with one heavy quark when including states up to a cutoff \(\Delta < 5500\ \mathrm{MeV}\) (where \(\Delta = M - m_h\)); using only the 12 lowest-lying charmed meson states and 42 charmed baryon states from the PDG falls short.
* Lattice simulations predict the chiral and deconfinement transition temperatures coincide at \(T_\chi \approx T_D \approx 200\ \mathrm{MeV}\) (Borsanyi 2010, Bazavov 2009).
* In the PCM, constituent quark and gluon masses are taken as \(M_q = 300\ \mathrm{MeV}\) and \(M_g = 664\ \mathrm{MeV}\), with string tension \(\sigma = (425\ \mathrm{MeV})^2\) and a domain volume \(V_\sigma = 8\pi T^3/\sigma^3\).

## Context

This work extends the HRG description of the QCD equation of state (Hagedorn 1984, Cleymans 1999, Borsanyi 2010) to the Polyakov loop order parameter, following the sum rule approach of Megias (2012). It confronts the model with lattice data from the HISQ/tree action (Bazavov 2011) and stout action (Borsanyi 2010). The study of higher-representation Polyakov loops (adjoint, decuplet, 15-plet, 27-plet) and the ambiguity between irreducible and reducible color-singlet configurations provides a test for the existence of exotic hadrons (hybrids, pentaquarks) in the QCD spectrum, to be resolved in future work (Megias, Ruiz Arriola, Salcedo, in preparation).
