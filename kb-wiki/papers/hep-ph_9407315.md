# Convolution Model for the Structure Functions of the Nucleon

> We start from an MIT-bag model calculation which provides information about the constituent quark distributions in the nucleon. The constituent quarks, however, are themselves considered as complex objects whose partonic substructure is resolved in deep inelastic scattering. This gives rise to structure functions of the constituent quarks which, in the unpolarized case, are fitted to data at a fixed scale employing three model parameters. Using $Q^{2}$--evolution equations the data are also well described at other scales. For the spin\-dependent struc\-ture func\-tions $g_{1}^{p,n}$ we additionally have to introduce polar\-izat\-ion functions for valence and sea quarks which are determined by exploiting the $x$--dependence of the available proton data only. A negatively polarized sea in the range $x\geq 0.01$ is suggested. We are then capable of predicting the shape of the neutron structure function $g_{1}^{n}$ which turns out to be in good agreement with experiment. Finally we present an estimate for the trans\-versely polarized structure function $g_{2}$, offering the possibility of extrac\-ting the twist--3 con\-tri\-bution and rating its importance.

## Key claim

The authors claim that a convolution model, combining MIT-bag model constituent quark distributions with fitted parton substructure functions for those constituent quarks, can describe both unpolarized and polarized nucleon structure functions. The hedge is that for the spin-dependent case, a negatively polarized sea in the range \(x \geq 0.01\) is *suggested* by the data, and the sea polarization functions must be introduced ad hoc, determined only from proton data, to then predict the neutron structure function.

## Method

The authors start from an MIT-bag model calculation to obtain constituent quark distributions in the nucleon. They then introduce parton distribution functions (valence \(\phi_v\), sea \(\phi_s\), and gluon \(\phi_g\)) inside these constituent quarks, fitted to unpolarized structure function data at a reference scale \(Q_0^2 = 10\ \text{GeV}^2\) using three parameters. The \(Q^2\)-evolution of these parton distributions is handled via leading-order DGLAP evolution equations. For the polarized case, polarization functions \(\mathcal{P}_v\) and \(\mathcal{P}_s\) for valence and sea quarks are introduced, with \(\mathcal{P}_v\) constrained by the Bjorken sum rule and \(\mathcal{P}_s\) fitted to the \(x\)-dependence of proton \(g_1^p\) data alone.

## Key numbers

*   The three parameters fitted to unpolarized data at \(Q_0^2 = 10\ \text{GeV}^2\) are \(a = -0.17\), \(b = 4\), and \(A = 0.137\) for the valence and sea parton distributions \(\phi_v\) and \(\phi_s\).
*   The gluon distribution parameter at \(Q_0^2 = 10\ \text{GeV}^2\) is \(B = 1.48\), \(c = 2.37\), corresponding to \(g(x, Q_0^2) = 4.27 \frac{1}{x} (1-x)^8\).
*   The QCD parameter \(\Lambda\) is set to \(200\ \text{MeV}\).
*   The bag model gives \((g_A/g_V)_{\text{bag}} = 1.09\), which underestimates the experimental value by roughly \(12\%\).
*   The valence polarization parameter \(P_v\) at \(Q_0^2 = 10\ \text{GeV}^2\) is set to \(0.035\) to satisfy the Bjorken sum rule.

## Context

This work extends the convolution model approach of Hwa (1980) and Jaffe (1985) by applying it to both unpolarized and polarized structure functions, using the MIT-bag model of Chodos et al. (1974) as the nonperturbative input. It enables a prediction for the neutron spin structure function \(g_1^n\) and an estimate for the transverse structure function \(g_2\), offering a way to extract twist-3 contributions. The approach contrasts with earlier bag model calculations by Jaffe (1975) and Hughes (1977) which did not include the partonic substructure of constituent quarks.
