# On gap equations and color-flavor locking in cold dense QCD with three massless flavors

> The superconductivity in cold dense QCD with three massless flavors is analyzed in the framework of the Schwinger-Dyson equation. The set of two coupled gap equations for the color antitriplet, flavor antitriplet (bar{3},bar{3}) and the color sextet, flavor sextet (6,6) order parameters is derived. It is shown that the antitriplet-antitriplet gives the dominant contribution to the color-flavor locked order parameter, while the sextet-sextet is small but nonzero.

## Key claim

The color-flavor locked phase of dense QCD with three massless flavors has a dominant order parameter in the color antitriplet, flavor antitriplet channel, while the color sextet, flavor sextet contribution is small but nonzero; this conclusion holds when the interaction is mediated by perturbative one-gluon exchange, not only in the instanton model.

## Method

The authors derived a set of two coupled Schwinger-Dyson gap equations for the color antitriplet, flavor antitriplet and color sextet, flavor sextet order parameters in cold dense QCD with three massless flavors, using the hard dense loop (HDL) approximation for the gluon propagator and neglecting wave function renormalization. They introduced a new set of projectors in color-flavor space to simplify the structure, then decoupled the equations by transforming to the antitriplet-antitriplet and sextet-sextet basis. The resulting differential equations were solved approximately in the vicinity of the Fermi surface, keeping only the dominant magnetic gluon mode contributions.

## Key numbers

* The dominant antitriplet-antitriplet gap at the Fermi surface is given by \(\Delta_{0} \approx \Lambda \exp(-\pi/\nu)\) where \(\nu = \sqrt{8\alpha_{s}/9\pi}\) and \(\Lambda = 16(2\pi)^{3/2}\mu/(3\alpha_{s})^{5/2}\); this yields \(\Delta_{0} = \frac{16(2\pi)^{3/2}\mu}{(3\alpha_{s})^{5/2}} \exp\left(-\frac{3\pi^{3/2}}{2^{3/2}\sqrt{\alpha_{s}}}\right)\).
* The relation between the two independent gap parameters is \(\Delta^{-}_{1}(p_{4}) \approx 2\Delta^{-}_{2}(p_{4}) = -2\Delta^{-}_{3}(p_{4}) \approx 2\Delta^{-}_{(\bar{3},\bar{3})}(p_{4})\).
* The sextet-sextet order parameter \(\Delta^{-}_{(6,6)}\) is nonzero only in the narrow infrared region \(0 \lesssim p^{2} \lesssim \Delta_{0}^{2}\), where it is of order \(\Delta_{0}\), but its precise magnitude is unreliable in the approximation used.

## Context

This work extends the Schwinger-Dyson analysis of color superconductivity from the two-flavor case studied by Hong, Miransky, Shovkovy, and Wijewardhana (1999) and by Pisarski and Rischke (1999) to three massless flavors, confirming the color-flavor locked phase structure previously found in the instanton liquid model by Alford, Rajagopal, and Wilczek (1999). The results enable future next-to-leading order studies that would properly account for the Meissner effect, wave function renormalization, and vertex corrections, which are needed for a reliable estimate of the sextet-sextet contribution and for gauge-invariant determination of the gap magnitude.
