# Pion Boer-Mulders function using a contact interaction

> A symmetry preserving treatment of a vector $\otimes$ vector contact interaction (SCI) is used as the basis for calculations of the two pion transverse momentum dependent parton distribution functions (TMDs); namely, that for unpolarised valence degrees-of-freedom and the analogous Boer-Mulders (BM) function. Amongst other things, the analysis enables the following themes to be addressed: the quark current mass dependence of pion TMDs; the impact of the gauge link model on the positivity constraint that bounds the BM function relative to the unpolarised TMD; the equivalence of direct diagrammatic and light-front wave function TMD calculations; and the size of the BM shift. Interpreted astutely, these SCI results enable one to draw insightful pictures of pion TMDs.

## Key claim

Using a symmetry-preserving treatment of a vector⊗vector contact interaction (SCI), the pion Boer-Mulders function is nonzero only when interactions between the struck quark and the spectator are included via the gauge link; the magnitude of the effect reflects the scale of emergent hadron mass, and the function satisfies the positivity bound only when the gauge link completion uses a momentum-dependent gluon propagator rather than a momentum-independent one.

## Method

The authors compute the helicity-independent pion transverse momentum dependent parton distribution function (TMD) and the Boer-Mulders function by evaluating the quark+quark correlation function diagrammatically, using a symmetry-preserving contact interaction (SCI) for the bound-state kernel. Two gauge link completions are considered: a momentum-independent gluon propagator and a momentum-dependent one with an infrared coupling α_L = 0.97π; the results are also obtained via projection of the pion light-front wave function derived from the SCI Bethe-Salpeter amplitude.

## Key numbers

- The infrared value of QCD's process-independent effective charge used for the momentum-dependent gauge link is α_L = 0.97π, known with a precision of 4%.
- The dynamically generated mass scale M used in the TMD definitions is approximately 4 f_π, where f_π is the pion leptonic decay constant.
- The helicity-independent TMD satisfies the light-front momentum sum rule: ∫_0^1 dx ∫ d²k_⊥ 2x f_{1π}(x,k_⊥²;ζ_H) = 1 at the hadron scale ζ_H.
- The SCI pion light-front wave function has the form ψ(x,k_⊥²) = √(2N_c) / [k_⊥² + M² - x(1-x)m_π²].

## Context

This work extends previous studies of pion transverse structure using the SCI framework (Zhang 2020, Raya 2021, Gutierrez-Guerrero 2021, Cheng 2022, Xing 2022, Xing 2023) and addresses the positivity bound for the Boer-Mulders function discussed by Bacchetta (1999). The results enable comparisons with other model calculations of the pion BM function (Lu 2012, Pasquini 2014, Wang 2017, Ahmady 2019, de Paula 2020, Zhu 2023, Kou 2023) and provide a baseline for future studies using more realistic momentum-dependent interactions.
