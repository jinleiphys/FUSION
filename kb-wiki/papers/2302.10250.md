# Probing the dark sector with nuclear transition photons

> Here we present world-leading sensitivity to light ($< 170$ MeV) dark matter (DM) using beam-dump experiments. Dark sector particles produced during pion decay at accelerator beam-dumps can be detected via scattering in neutrino detectors. The decay of nuclei excited by the inelastic scattering of DM is an unexploited channel which has significantly better sensitivity than similar searches using the elastic scattering channel. We show that this channel is a powerful probe of DM by demonstrating sensitivity to the thermal relic abundance benchmark in a scalar DM dark-photon portal model. This is achieved through the use of existing data, obtained by the KARMEN experiment over two decades ago, which allow us to set world-leading constraints on this model over a wide mass range. With experimental improvements planned for the future, this technique will be able to probe the thermal relic benchmark for fermionic DM across a wide mass range.

## Key claim
The paper claims world-leading sensitivity to light dark matter (DM) below 170 MeV by using the inelastic nuclear scattering channel (detecting deexcitation photons from excited nuclei) rather than the elastic scattering channel, but this sensitivity is demonstrated specifically for a scalar DM dark-photon portal model using existing KARMEN data, with the thermal relic benchmark reached only for scalar DM (not fermionic DM) in the current analysis.

## Method
The authors compute Gamow-Teller (GT) transition strengths for carbon, argon, sodium, and iodine nuclei using the nuclear shell model code BIGSTICK, then calculate the inelastic DM-nucleus scattering cross section mediated by a dark photon. They use existing KARMEN data on the 12C(nu, nu')12C* reaction (86±15 events) as a background to set constraints on DM-induced nuclear excitations, and project sensitivities for future experiments COHERENT, CCM, and PIP2-BD by simulating DM fluxes from pion decay at beam-dump facilities and applying timing cuts to reduce neutrino backgrounds.

## Key numbers
- KARMEN observed 86±15 events from the 12C(nu_mu, nu_mu')12C* reaction with 4.65×10^22 protons-on-target (POT)
- KARMEN detector: 56 tonne liquid scintillator calorimeter at 17.7 m distance, 100° angle, 4 years runtime, 10 MeV threshold
- The GT strength for 12C is B(GT)=0.255±0.021, derived from the flux-averaged cross section
- For scalar DM, the KARMEN result probes the thermal relic benchmark over DM masses m_chi ~ 5-40 MeV
- The kink in sensitivity curves occurs at m_chi=45 MeV where pi0 decay closes (m_A' = 3m_chi = 135 MeV ≈ m_pi0)
- COHERENT NaI detector: 3.5 tonne mass, 22 m distance, 120° angle, ~few keV threshold
- CCM: 7 tonne LAr detector, 20 m distance, 90° angle, 25 keV threshold
- PIP2-BD: 100 tonne LAr detector, 15 m distance, 20 keV threshold, 5 years runtime

## Context
This work extends the search for light DM via dark photon mediators from elastic scattering channels (previously used by COHERENT, LSND, MiniBooNE, NA64, and E137) to the inelastic nuclear excitation channel, which provides higher energy signals (MeV-scale photons vs keV-scale nuclear recoils) and lower backgrounds through line searches. The approach uses the same nuclear matrix elements as neutrino neutral-current inelastic scattering calculations (Donnelly 1974, Ejiri 2019) and builds on recent GT transition calculations (inelastic2022) and experimental data (Tornow 2022). This enables using existing beam-dump neutrino experiment data (KARMEN) and planned large detectors (PIP2-BD) to probe thermal relic DM benchmarks that were previously inaccessible.
