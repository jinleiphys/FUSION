# Extracting a Short Distance Top Mass with Light Grooming

> We propose a kinematic method based on a factorization formula for precisely measuring the top quark mass $m_t$ in $pp$ collisions using boosted top jets with light soft drop grooming. By using light grooming, which is an order of magnitude less aggressive than typical grooming, we retain a universal description of the top mass scheme and decay effects, while still effectively removing soft contamination from the top jet. We give field theory results for the hadronization corrections for jets induced by a heavy top quark, showing they are described by a universal hadronic parameter that also appears for groomed light quark jets. An important phenomenological application of our results is that one can obtain $m_t$ in a short distance scheme by fitting the hadron level jet mass distributions, predicted by our factorization formula, to data or by Monte-Carlo calibration. The peaked distributions for $pp$ and $e^+e^-$ collisions are similar, up to sensitivity to underlying event which is significantly reduced by soft drop. Since soft drop implies that the $t$ and $\bar t$ jet masses each can be independently measured, the analysis enables the use of lepton+jet samples.

## Key claim

The paper proposes that a short-distance top quark mass can be extracted from boosted top jet mass distributions in \(pp\) collisions using "light" soft drop grooming with \(z_{\rm cut} \lesssim 0.01\) and \(\beta=2\), provided the grooming is weak enough to retain the top decay products and ultra-collinear radiation while still removing soft contamination, enabling a factorization-based description with controlled hadronization corrections.

## Method

The authors derive a factorization theorem in Soft-Collinear Effective Theory (SCET) for the groomed top jet mass distribution in the peak region \((M_J^2 - m_t^2)/m_t \sim \Gamma_t \ll m_t\), for boosted top quark pair production in \(pp\) collisions. They apply soft drop grooming with parameters chosen in the "light grooming" region, defined by conditions \(z_{\rm cut} \lesssim \Gamma_t/(h^{2+\beta} m_t) (p_T/m_t)^\beta\) and \(z_{\rm cut}^{1/(2+\beta)} \gg (1/2)(\Gamma_t/m_t \cdot m_t^2/p_T^2)^{1/(2+\beta)}\), which ensures the top decay products are retained while soft contamination is removed. The factorization formula includes a heavy quark jet function for the top decay dynamics, a collinear-soft function for the soft drop stopping radiation, and nonperturbative hadronization corrections described by a universal hadronic parameter.

## Key numbers

* The top width is \(\Gamma_t \simeq 1.4\) GeV.
* For \(p_T = 750\) GeV and \(h=2\), the light grooming constraints become \(z_{\rm cut} \lesssim 0.01\) and \(z_{\rm cut}^{1/4} \gg 0.066\), satisfied by \(z_{\rm cut} \simeq 0.01\).
* The soft drop modified hard scale is defined as \(Q_{\rm cut} = 2^\beta Q \tilde{z}_{\rm cut} = (2\cosh\eta_J/R_0)^\beta Q z_{\rm cut}\).
* The nonperturbative hadronization corrections for groomed top jets depend on the same parameter \(\Omega_{1q}^{\figeight}\) that appears for massless quark initiated groomed jets.

## Context

This work extends the factorization approach for top mass extraction from boosted tops in \(e^+e^-\) collisions (Fleming 2007) to the more complex \(pp\) collision environment, where soft contamination from initial state radiation and underlying event had previously prevented a similar treatment. It builds on the soft drop grooming algorithm (Dasgupta 2013, Larkoski 2014) and the factorization for groomed massless jets (Frye 2016), and uses hadronization results from Hoang (2019). The method enables extraction of a short-distance top mass from LHC data or calibration of Monte-Carlo generator mass parameters, similar to what was done for 2-Jettiness in \(e^+e^-\) collisions (Butenschoen 2016).
