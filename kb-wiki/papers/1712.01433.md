# Inclusive breakup calculations in angular momentum basis: application to $^7$Li+$^{58}$Ni

> The angular momentum basis method is introduced to solve the inclusive breakup within the model proposed by Ichimura, Austern, and Vincent [Phys. Rev. C 32, 431 (1985)]. This method is based on the geometric transformation between Jacobi coordinates, thus it is easy to corporate with particle spins. To test the validity of this partial wave expansion method, a benchmark calculation is done comparing with the one given in [Phys. Rev. C 92, 044616 (2015)]. Using the distorted-wave Born approximation (DWBA) version of IAV model, some applications to $^7$Li reactions are presented and compared with available data.

## Key claim

The paper introduces an angular momentum basis method for solving the Ichimura-Austern-Vincent (IAV) inclusive breakup model that is more efficient for projectiles with internal orbital angular momentum \(\ell \geq 1\) (such as \(^7\)Li), whereas previous implementations were restricted to \(\ell = 0\) cases (deuterons and \(^6\)Li). The method is validated by a benchmark calculation showing less than 1% numerical difference from the earlier method for \(\ell = 0\), and then applied to \(^7\)Li reactions where the nonelastic breakup (NEB) component dominates the inclusive alpha production.

## Method

The inclusive breakup cross section is separated into elastic breakup (EBU) and nonelastic breakup (NEB) components following the IAV model. The EBU part is calculated using the continuum-discretized coupled-channels (CDCC) formalism with the code FRESCO, while the NEB part is computed using the distorted-wave Born approximation (DWBA) version of the IAV model. The wave functions are expanded in an angular momentum basis that couples the orbital angular momenta and spins of the three particles (\(b\), \(x\), and target \(A\)) to total angular momentum \(J\), and a geometric transformation between Jacobi coordinate sets is performed using coefficients \(\mathcal{G}^{out\gets in}_{\alpha_{in},\alpha_{out}}(r_x r_b x)\). For the \(^7\)Li projectile, the \(\alpha + t\) cluster model is used with \(\ell = 1\) internal angular momentum, including partial waves up to \(\ell = 3\) for convergence.

## Key numbers

- The benchmark calculation comparing the new angular momentum basis method with the earlier method of Jin (2015) shows a numerical difference of less than 1% using the same input parameters.
- The \(\alpha + d\) breakup threshold of \(^6\)Li is 1.474 MeV, while the \(\alpha + t\) breakup threshold of \(^7\)Li is 2.468 MeV.
- For the \(^4\)He + \(^{58}\)Ni optical potential at 12 MeV, the parameters are: \(V = 49.5\) MeV, \(R_0 = 5.88\) fm, \(a_0 = 0.5\) fm, \(W = 11.0\) MeV, \(R_w = 5.69\) fm, \(a_w = 0.5\) fm.
- The \(^7\)Li + \(^{58}\)Ni elastic scattering data from Pfeiffer (1973) are reproduced by both CDCC and optical model calculations at multiple incident energies around the Coulomb barrier.
- The \(\ell = 3\) resonant states of the \(\alpha + t\) system included are at 4.63 MeV (\(7/2^{-}\)) and 6.68 MeV (\(5/2^{-}\)).

## Context

This work extends the IAV model, originally proposed by Ichimura, Austern, and Vincent (1985), which had been applied primarily to deuterons and \(^6\)Li (Jin 2015, Jin 2017) due to computational difficulties for \(\ell > 0\) cases. The new angular momentum basis method enables calculations for \(^7\)Li reactions, where the \(\ell = 1\) configuration of the \(\alpha + t\) cluster makes the calculation more challenging than the \(\ell = 0\) case of \(^6\)Li. The results enable future applications to knockout reactions at intermediate energies and provide a framework for interpreting large alpha yields observed in experiments with weakly bound projectiles such as \(^{6,8}\)He, \(^{6,7}\)Li, and \(^{7,9}\)Be.
