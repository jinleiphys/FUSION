# Does particle decay cause wave function collapse: An experimental test

> We describe an experimental test of whether particle decay causes wave function collapse. The test uses interference between two well separated, but coherent, sources of vector mesons. The short-lived mesons decay before their wave functions can overlap, so any interference must involve identical final states. Unlike previous tests of nonlocality, the interference involves continuous variables, momentum and position. Interference can only occur if the wave function retains amplitudes for all possible decays. The interference can be studied through the transverse momentum spectrum of the reconstructed mesons.

## Key claim

The paper proposes that observing interference in the transverse momentum spectrum of short-lived vector mesons produced at two separated sources would demonstrate that the wave function does not collapse when a particle decays, but instead retains amplitudes for all possible decay modes and angular distributions until an external measurement is made. The test is conditional on the meson lifetime being short enough that the mesons decay before their wave functions can spatially overlap, so any interference must involve the decay products.

## Method

The authors describe using ultra-peripheral heavy ion collisions at RHIC and the LHC, where a photon from the electromagnetic field of one nucleus fluctuates to a quark-antiquark pair that elastically scatters from the other nucleus, producing a vector meson. Because the two nuclei act as a two-source interferometer with a fixed phase relationship, the interference pattern can be studied through the reconstructed vector meson transverse momentum spectrum. The degree of decoherence is parameterized by a factor eta, which for short-lived mesons like the rho, omega, and phi would be near unity if collapse occurs at decay, while for the longer-lived J/psi the decoherence would be only partial.

## Key numbers

- Median impact parameter for rho and omega production: about 40 fm at RHIC, rising to 300 fm at the LHC; for J/psi: 23 fm at RHIC, about 50 fm at the LHC
- Vector meson lifetimes range from 4 x 10^-24 s for the rho to 7.5 x 10^-21 s for the J/psi
- Production rates: more than 100 rho^0 per second at RHIC, rising to 230,000 rho^0 per second at the LHC
- The STAR collaboration has observed more than 10,000 rho^0 events
- The dip in the p_T spectrum from interference occurs for p_T less than 25 MeV/c
- For J/psi at the LHC, the decay distance is approximately equal to the median impact parameter, so decoherence would be only partial if collapse occurs at decay

## Context

This work extends the EPR paradox and Bell's inequality tests (Einstein, Podolsky, and Rosen 1935; Bell 1965) from discrete variables like spin or CP violation (Apostolakis 1998) to continuous variables (momentum and position) using vector meson interferometry. Unlike previous tests using photon pairs (Horne, Shimony, and Zeilinger 1989) or kaon systems, this system is sensitive to whether particle decay itself causes wave function collapse. The authors had previously calculated the interference pattern (Klein and Nystrand 2000) and the cross sections using the Glauber approach (Klein and Nystrand 1999). The experiment enables tests of nonlocality for continuous variables and could be performed with existing data from the STAR detector at RHIC.
