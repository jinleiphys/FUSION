# An upgraded issue of the parton and hadron cascade model, PACIAE 2.2

> The parton and hadron cascade model PACIAE 2.1 (cf. Comput. Phys. Commun.184 (2013) 1476) has been upgraded to the new issue of PACIAE 2.2. By this new issue the lepton-nucleon and lepton-nucleus (inclusive) deep inelastic scatterings can also be investigated. As an example, the PACIAE 2.2 model is enabled to calculate the specific charged hadron multiplicity in the $e^-$+p and $e^-$+D semi-inclusive deep-inelastic scattering at 27.6 GeV electron beam energy. The calculated results are well comparing with the corresponding HERMES data. Additionally, the effect of model parameters αand βin the Lund string fragmentation function on the multiplicity is studied.

## Key claim

The upgraded PACIAE 2.2 model extends the previous version (PACIAE 2.1) to also handle lepton-nucleon and lepton-nucleus deep inelastic scattering, treating these processes analogously to proton-nucleon and proton-nucleus collisions but using the lepton-nucleon DIS cross section instead of the nucleon-nucleon cross section. The model reproduces HERMES data for charged hadron multiplicities in 27.6 GeV electron beam semi-inclusive DIS off proton and deuteron targets using default PYTHIA parameters, without additional tuning.

## Method

The PACIAE 2.2 model treats lepton-nucleon and lepton-nucleus DIS the same way as proton-nucleon and proton-nucleus collisions in PACIAE 2.1, but replaces the nucleon-nucleon cross section with the lepton-nucleon DIS cross section. For lepton-nucleus DIS, the incident electron is assumed to undergo at most one DIS with the struck nucleon (the one with smallest approaching distance), because the lepton-nucleon DIS total cross section is orders of magnitude smaller than the pp cross section below 1000 GeV center-of-mass energy. The model uses the PYTHIA event generator with string fragmentation switched off to produce a partonic initial state, followed by parton rescattering via 2-to-2 LO-pQCD cross sections, hadronization via the Lund string fragmentation model, and final-state hadronic rescattering.

## Key numbers

- Electron beam energy: 27.6 GeV for the e-+p and e-+D semi-inclusive deep inelastic scattering calculations compared to HERMES data.
- For e-+p DIS with default parameters (alpha=0.3, beta=0.58): DIS normalized multiplicities are pi+ = 1.339, pi- = 0.9666, K+ = 0.1177, K- = 0.04646.
- For e-+D DIS with default parameters (alpha=0.3, beta=0.58): DIS normalized multiplicities are pi+ = 1.326, pi- = 1.383, K+ = 0.1219, K- = 0.05696.
- Varying alpha from 0.2 to 0.45 (with beta fixed at 0.58) changes e-+p pi+ multiplicity from 1.331 to 1.351; varying beta from 0.39 to 0.87 (with alpha fixed at 0.3) changes e-+p pi+ multiplicity from 1.355 to 1.315.
- Running time: 1000 events for sqrt(s)=200 GeV Non Single Diffractive pp collision by PACIAE 2.2a takes 0.5 minute; 10 events for 10-40% most central Au+Au collisions at sqrt(s_NN)=200 GeV by PACIAE 2.2b takes 4 minutes, by PACIAE 2.2c takes 8 minutes.

## Context

This work upgrades the PACIAE model from version 2.1 (Sa et al., Comput. Phys. Commun. 184 (2013) 1476) to version 2.2, adding the capability to simulate lepton-nucleon and lepton-nucleus deep inelastic scattering. The approach follows earlier work using PYTHIA combined with the BUU transport model (Falter, Cassing, Gallmeister, and Mosel, 2004) for lepton-nucleus DIS, but integrates the lepton DIS treatment directly into the PACIAE cascade framework. The model enables future studies of lepton-nucleus collisions at proposed electron-ion colliders such as eRHIC, ELIC, and LHeC, and provides a tool for investigating parton distribution functions and fragmentation functions in nuclear environments.
