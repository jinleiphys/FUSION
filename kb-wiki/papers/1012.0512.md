# Relation between the $0νββ$ and $2νββ$ nuclear matrix elements revisited

> We show that the dominant Gamow-Teller part, $M^{0ν}_{GT}$, of the nuclear matrix element governing the neutrinoless $ββ$ decay is related to the matrix element $M^{2ν}_{cl}$ governing the allowed two-neutrino $ββ$ decay. That relation is revealed when these matrix elements are expressed as functions of the relative distance $r$ between the pair of neutrons that are transformed into a pair of protons in the $ββ$ decay. Analyzing this relation allows us to understand the contrasting behavior of these matrix elements when $A$ and $Z$ is changed; while $M^{0ν}_{GT}$ changes slowly and smoothly, $M^{2ν}$ has pronounced shell effects. We also discuss the possibility of phenomenological determination of the $M^{2ν}_{cl}$ and from them of the $M^{0ν}_{GT}$ values from the experimental study of the $β^{\pm}$ strength functions.

## Key claim

The dominant Gamow-Teller part of the neutrinoless double-beta decay nuclear matrix element, \(M^{0\nu}_{GT}\), is related to the closure-approximation matrix element of the two-neutrino double-beta decay, \(M^{2\nu}_{cl}\), through the relation \(C^{0\nu}_{GT}(r) = H(r,\bar{E}) \times C^{2\nu}_{cl}(r)\), where \(r\) is the relative distance between the transformed neutron pair; this relation explains why \(M^{0\nu}_{GT}\) varies slowly and smoothly with mass number \(A\) and charge \(Z\) while \(M^{2\nu}\) exhibits pronounced shell effects.

## Method

The authors used the Quasiparticle Random Phase Approximation (QRPA) with a single-particle model space of 23 states (0–5\(\hbar\omega\) oscillator shells plus \(0i_{11/2}\) and \(0i_{13/2}\) levels) and a G-matrix interaction derived from the Argonne V18 potential. They defined radial distribution functions \(C^{0\nu}_{GT}(r)\) and \(C^{2\nu}_{cl}(r)\) for the matrix elements and computed their integrals as functions of the upper integration limit \(r_0\) to separate short-range nucleon-nucleon correlations from longer-range nuclear many-body effects. The particle-particle coupling strength \(g_{pp}\) was adjusted to reproduce experimental two-neutrino double-beta decay half-lives for each nucleus.

## Key numbers

* The experimental two-neutrino double-beta decay matrix element \(|M^{2\nu}_{exp}|\) for \(^{76}\)Ge is 0.136 MeV\(^{-1}\).
* The closure matrix element \(M^{2\nu}_{cl}\) for \(^{76}\)Ge from QRPA is 0.099.
* The full neutrinoless double-beta decay matrix element without closure \(|M^{0\nu}|\) for \(^{76}\)Ge is 5.24.
* The Gamow-Teller part of the neutrinoless matrix element without closure \(|M^{0\nu}_{GT}|\) for \(^{76}\)Ge is 4.33.
* The experimental two-neutrino matrix element \(|M^{2\nu}_{exp}|\) for \(^{100}\)Mo is 0.231 MeV\(^{-1}\).
* The closure matrix element \(M^{2\nu}_{cl}\) for \(^{100}\)Mo from QRPA is -0.933.
* The full neutrinoless matrix element without closure \(|M^{0\nu}|\) for \(^{100}\)Mo is 4.99.
* The experimental two-neutrino matrix element \(|M^{2\nu}_{exp}|\) for \(^{130}\)Te is 0.033 MeV\(^{-1}\).
* The closure matrix element \(M^{2\nu}_{cl}\) for \(^{130}\)Te from QRPA is -0.464.
* The full neutrinoless matrix element without closure \(|M^{0\nu}|\) for \(^{130}\)Te is 4.07.
* The neutrino potential \(H_{GT}(r,\bar{E})\) at \(r=0\) with \(\bar{E}_{0\nu}=0\) is \(5M_A R/16\), where \(M_A = 1.09\) GeV.
* With effective axial coupling \(g^{eff}_A = 1.0\), the full matrix elements \(M'^{0\nu}\) are reduced by 20–30% compared to values with \(g_A = 1.269\).

## Context

This work extends earlier QRPA studies by Simkovic et al. (2008) and nuclear shell model calculations by Menendez et al. (2008) by introducing a radial decomposition that directly links the Gamow-Teller parts of the \(0\nu\beta\beta\) and \(2\nu\beta\beta\) matrix elements. It contradicts the early speculation by Primakoff and Rosen (1959) that the two matrix elements might be simply proportional. The relation enables future phenomenological determination of \(M^{0\nu}_{GT}\) from measured \(\beta^\pm\) strength functions and experimental \(2\nu\beta\beta\) half-lives, potentially reducing nuclear model uncertainties in interpreting neutrinoless double-beta decay experiments.
