# A new relativistic hydrodynamics code for high-energy heavy-ion collisions

> We construct a new Godunov type relativistic hydrodynamics code in Milne coordinates, using a Riemann solver based on the two-shock approximation which is stable under the existence of large shock waves. We check the correctness of the numerical algorithm by comparing numerical calculations and analytical solutions in various problems, such as shock tubes, expansion of matter into the vacuum, the Landau-Khalatnikov solution, and propagation of fluctuations around Bjorken flow and Gubser flow. We investigate the energy and momentum conservation property of our code in a test problem of longitudinal hydrodynamic expansion with an initial condition for high-energy heavy-ion collisions. We also discuss numerical viscosity in the test problems of expansion of matter into the vacuum and conservation properties. Furthermore, we discuss how the numerical stability is affected by the source terms of relativistic numerical hydrodynamics in Milne coordinates.

## Key claim

The authors construct a new Godunov-type relativistic hydrodynamics code in Milne coordinates using a Riemann solver based on the two-shock approximation, which is stable under the existence of large shock waves; the algorithm is designed to reduce numerical uncertainty when extracting the physical viscosity of the quark-gluon plasma from experimental data, but the paper only tests ideal hydrodynamics, not viscous hydrodynamics.

## Method

The code solves the relativistic ideal hydrodynamic equations in Milne coordinates using a Lagrange-step scheme with a Riemann solver based on the two-shock approximation. The Riemann problem in Milne coordinates is shown to be identical to that in Cartesian coordinates, allowing the use of Cartesian-coordinate Riemann solutions after a Lorentz boost to \(\eta=0\). The one-dimensional code in the \(\tau\)-\(\eta\) direction is extended to (3+1) dimensions using the Strang splitting method, with the \(x\) and \(y\) directions solved using the same Cartesian-coordinate algorithm from Akamatsu (2014).

## Context

This work extends the Cartesian-coordinate algorithm of Akamatsu (2014) to Milne coordinates, which are more suitable for describing the space-time evolution of high-energy heavy-ion collisions at RHIC and the LHC where approximate longitudinal boost invariance is observed. The code is designed to serve as the ideal-hydrodynamics component in a future viscous hydrodynamic solver using operator splitting, following the approach outlined in Inutsuka (2011) and Akamatsu (2014). The paper enables more precise extraction of the QGP transport coefficients by reducing numerical dissipation compared to previous hydrodynamic models.
