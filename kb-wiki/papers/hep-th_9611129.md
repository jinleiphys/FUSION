# Interlinked 3D and 4D 3-Quark Wave Functions in a Bethe-Salpeter Model

> Using the method of Green's functions within the framework of a Bethe-Salpeter formalism characterized by a pairwise $qq$ interaction with a 3D support to its kernel (expressed in a Lorentz-covariant manner), the 4D BS wave function for a system of three identical relativistic spinless quarks is reconstructed from the corresponding 3D form which satisfies a {\it fully connected} 3D BSE. This result is a 3-body generalization of a similar interconnection between the 3D and 4D 2-body wave functions that had been found earlier under identical conditions of a 3D support to the corresponding BS kernel, using the ansatz of Covariant Instaneity for the pairwise $q\bar q$ interaction. The generalization from spinless to fermion quarks is straightforward.

# Interlinked 3D and 4D 3-Quark Wave Functions in a Bethe-Salpeter Model

## Key claim

The 4D Bethe-Salpeter wave function for three identical relativistic spinless quarks can be reconstructed from the corresponding 3D form, but unlike the two-body case where the 3D-4D transition is fully reversible, the three-body case requires an additional ansatz involving a delta-function in the spectator energy variable due to the truncation of Hilbert space implied by the 3D support condition on the pairwise kernel.

## Method

Using Green's function techniques within a Bethe-Salpeter formalism with pairwise quark-quark interactions having 3D support to the kernel (expressed Lorentz-covariantly via the Covariant Instantaneity Ansatz), the author first derives the 3D-4D interconnection for the two-body (quark-antiquark) system as a reference. For the three-quark system, the 4D Bethe-Salpeter equation is reduced to a fully connected 3D form by integrating over two internal time-like momenta, and then the 4D Green's function is reconstructed from the 3D form by introducing a spectator propagation ansatz with a delta-function constraint on the time-like momentum component, with the constant determined by contour integration over the spectator propagator pole.

## Key numbers

- The 3D denominator function is given by $D({\hat q}) = 4{\hat \omega}({\hat \omega}^2 - M^2/4)$, where ${\hat \omega}^2 = {m_q}^2 + {\hat q}^2$ (quark mass $m_q$, hadron mass $M$, 3D relative momentum ${\hat q}$)
- The contour integration for the spectator propagator yields the constant $C_3 = i \pi / (M {\hat \omega}_3)$, where ${{\hat \omega}_3}^2 = {m_q}^2 + {{\hat p}_3}^2$
- The 3D BSE for the three-quark system is obtained with a factor $(2\pi)^3$ on the left-hand side, matching the two-body result $(2\pi)^3$ for the 3D Green's function equation

## Context

This work extends the author's earlier two-body 3D-4D interconnection result (Mitra, 1991) using the Covariant Instantaneity Ansatz to the three-body case, and provides a formal derivation of a semi-intuitive ansatz previously employed (Mitra, 1995) for baryon-qqq vertex functions. The 3D support ansatz for the Bethe-Salpeter kernel is contrasted with approaches using O(4)-like spectra (which disagree with observed O(3) spectra) and with parametric methods such as QCD sum rules, effective Lagrangians, and direct gaussian parametrizations. The result enables systematic calculation of hadronic transition amplitudes via 4D quark loop integrals, as already demonstrated for the proton-neutron mass difference problem.
