# Shear Viscosity at Infinite Coupling: A Field Theory Calculation

> I derive an exact integral expression for the ratio of shear viscosity over entropy density $\fracη{s}$ for the massless (critical) O(N) model at large N with quartic interactions. The calculation is set up and performed entirely from the field theory side using a non-perturbative resummation scheme that captures all contributions to leading order in large N. In 2+1d, $\fracη{s}$ is evaluated numerically at all values of the coupling. For infinite coupling, I find $\fracη{s}\simeq 0.42(1)\times N$. I show that this strong coupling result for the viscosity is universal for a large class of interacting bosonic O(N) models.

## Key claim

The paper derives an exact large-N expression for the shear viscosity over entropy density ratio η/s in the massless O(N) model with quartic interactions, finding that at infinite coupling in 2+1 dimensions the result is η/s ≃ 0.42(1) × N, and this strong-coupling value is universal for a large class of interacting bosonic O(N) models.

## Method

The calculation uses a non-perturbative resummation scheme (R0 through R4 levels) that captures all contributions to leading order in large N, building on the 2PI formalism of Aarts (2004). The shear viscosity is extracted from the zero-frequency limit of the retarded energy-momentum tensor correlator via the Kubo relation, with the pinching pole enhancement mechanism providing the leading large-N contributions. The resulting integral expression for η/s is evaluated numerically using orthogonal polynomial expansions for all values of the dimensionless coupling λ̂ = βλ.

## Key numbers

- At infinite coupling in 2+1 dimensions: η/s = 0.42(1) × N (the uncertainty is in the last digit)
- The strong-coupling result is universal for a class of bosonic O(N) models, including the φ⁶ theory with action S = ∫ dᵈX [½ ∂_μ φ_a ∂_μ φ_a + (λ/N²)(φ_a φ_a)³]
- The entropy density s is given exactly at large N by s = (N T²/(2π)) [2 m̂² ln n(m) + 6 m̂ Li₂(e^{-m̂}) + 6 Li₃(e^{-m̂})], where m̂ = βm and n(x) = (e^{βx} - 1)⁻¹
- In the strong coupling limit, the saddle point condition gives the universal solution m̂ = 2 ln((1+√5)/2)

## Context

This work extends the program of calculating transport coefficients in the O(N) model at large N, where the entropy density was previously known for all couplings (Romatschke 2019). It provides the first complete coupling-dependence for η/s in a quantum field theory without relying on holographic dualities, in contrast to the N=4 Super-Yang-Mills result η/s = 1/(4π) ≈ 0.08 (Policastro 2001). The result enables tests of hydrodynamic breakdown, validation of analytic continuation techniques used in lattice Monte-Carlo studies (Meyer 2007, Pasztor 2018), and benchmarking of approximation schemes used for QCD transport (Arnold 2000, 2003; Ghiglieri 2018).
