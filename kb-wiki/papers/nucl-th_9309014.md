# First-Forbidden beta decay: Medium enhancement of the axial charge at A=16

> Calculations are presented for four relatively strong first-forbidden $β$ decays in the region A=11-16 in order to study the very large mesonic-exchange-current enhancement of the rank-zero components. The $μ^-$ capture on $^{16}$O is considered on the same footing. The wave functions utilized include up to 4$\hbarω$ excitations. Two-body exchange-current matrix elements are calculated as well as one-body impulse approximation matrix elements. The resultant enhancement factor that multiplies the impulse-approximation axial-charge matrix element is thereby determined by comparison to experiment to be $ε_{exp}$ = 1.61 $\pm$ 0.03 from three $β^-$ decays and $μ^-$ capture on $^{16}$O, which is in excellent agreement with meson-exchange calculations in the soft-pion approximation.

## Key claim

The paper determines that the mesonic-exchange-current enhancement factor for the rank-zero axial-charge matrix element in first-forbidden beta decays around A=16 is ε_exp = 1.61 ± 0.03, in excellent agreement with soft-pion approximation calculations, but this result is conditional on using large-basis shell-model wave functions that include up to 4ℏω excitations and on the specific treatment of the 0p-1s0d gap.

## Method

The authors performed large-basis shell-model diagonalizations using the OXBASH code with model spaces including up to 4ℏω excitations for ¹⁶O and (1+3)ℏω for ¹⁶N, employing two interactions (WBP and WBN) with different treatments of the mixing interaction V²ℏω. They calculated one-body impulse-approximation matrix elements and two-body soft-pion exchange-current matrix elements for four first-forbidden β⁻ decays (¹⁶N, ¹⁵C, ¹¹Be, ¹⁶C) and for μ⁻ capture on ¹⁶O, using both harmonic-oscillator and Woods-Saxon radial wave functions.

## Key numbers

- Enhancement factor ε_exp = 1.61 ± 0.03, determined from three β⁻ decays and μ⁻ capture on ¹⁶O
- For ¹⁶N(0⁻) → ¹⁶O(0⁺) β⁻ decay: experimental matrix element M^β_0 = 58.4 ± 1.1 fm, corresponding to decay rate Λ_β = 0.485 ± 0.019 s⁻¹
- For ¹⁶O(0⁺) → ¹⁶N(0⁻) μ⁻ capture: experimental matrix element M^μ_0 = 113.5 ± 3.4 fm, corresponding to capture rate Λ_μ = 1560 ± 94 s⁻¹
- Harmonic-oscillator energy for ¹⁶O: ℏω = 13.60 MeV (from charge radius ⟨r²⟩¹/² = 2.730 fm)
- 0p-1s0d gap reduction: 3.0 MeV for WBP interaction, 3.5 MeV for WBN interaction
- Ground state of ¹⁶O with WBN interaction: 34.9% 0ℏω, 46.7% 2ℏω, 18.4% 4ℏω
- Lowest 0⁻ T=1 state in ¹⁶N with WBN: 64.5% 1ℏω, 35.5% 3ℏω

## Context

This work extends previous calculations of first-forbidden beta decay observables in heavier nuclei (A=50 by Warburton (1991), A=133-134 by Warburton (1992), and A=205-212 by Warburton (1993)) to the light A=11-16 region, where the large model spaces allow inclusion of all first-order core-polarization effects without multiplicative renormalization factors. The results confirm the soft-pion approximation predictions of Towner (1992) and enable precise tests of meson-exchange current effects in light nuclei, building on the earlier (0+2+4)ℏω model of Brown and Green (1966).
