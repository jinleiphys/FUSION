# Does Bayesian Model Averaging improve polynomial extrapolations? Two toy problems as tests

> We assess the accuracy of Bayesian polynomial extrapolations from small parameter values, x, to large values of x. We consider a set of polynomials of fixed order, intended as a proxy for a fixed-order effective field theory (EFT) description of data. We employ Bayesian Model Averaging (BMA) to combine results from different order polynomials (EFT orders). Our study considers two "toy problems" where the underlying function used to generate data sets is known. We use Bayesian parameter estimation to extract the polynomial coefficients that describe these data at low x. A "naturalness" prior is imposed on the coefficients, so that they are O(1). We Bayesian-Model-Average different polynomial degrees by weighting each according to its Bayesian evidence and compare the predictive performance of this Bayesian Model Average with that of the individual polynomials. The credibility intervals on the BMA forecast have the stated coverage properties more consistently than does the highest evidence polynomial, though BMA does not necessarily outperform every polynomial.

## Key claim

Bayesian Model Averaging (BMA) improves the calibration of credibility intervals for polynomial extrapolations compared to using the single highest-evidence polynomial, but BMA does not necessarily outperform every individual polynomial and can fail entirely when the underlying function has a pole that causes all finite-order polynomials to underpredict the true value.

## Method

The authors generated pseudodata sets from two known underlying functions, \(g_1(x) = (1/2 + \tan(\pi x/2))^2\) and \(g_2(x) = (1.3/(1.3+x))^2\), each with 10 data points uniformly spaced in \(x \in [0, 1/\pi]\) and Gaussian noise of 5% of the function value. They performed Bayesian parameter estimation for polynomial models of degree \(M = 0\) through \(M = 6\) using a Gaussian "naturalness" prior on the coefficients with width \(\sigma_a\), then marginalized over \(\sigma_a\) using an inverse-chi-squared prior. They computed the Bayesian evidence for each polynomial degree and formed a BMA prediction by weighting each polynomial's posterior predictive distribution at a target point \(x_t\) by its evidence, then assessed predictive performance using a Credibility Interval Diagnostic (CID) that measures empirical coverage probabilities.

## Key numbers

* Underlying function \(g_1\) has a pole at \(x = 1\); its Taylor expansion coefficients are all positive: \(g_1(x) = 0.25 + 1.57x + 2.47x^2 + 1.29x^3 + \ldots\)
* Underlying function \(g_2\) has a pole at \(x = -1.3\); its Taylor expansion coefficients alternate in sign: \(g_2(x) = 1 - 1.54x + 1.78x^2 - 1.87x^3 + 1.75x^4 + \ldots\)
* Pseudodata were generated with 10 points uniformly spaced in \(x \in [0, 1/\pi]\), with Gaussian noise of standard deviation \(\sigma_i = 0.05 \times g(x_i)\)
* The prior on the LEC width \(\sigma_a\) was taken as an inverse-chi-squared distribution with hyperparameters \(\nu_0 = \tau_0 = 1.5\) for all main results
* Target points for extrapolation included \(x_t = 1.2/\pi\), as shown in the BMA posterior figures

## Context

This work extends the Bayesian EFT framework developed by Schindler (2009) and Wesolowski (2016), who used the same two toy functions to study LEC extraction and naturalness priors. It directly tests BMA in an "M-open" context where the true model is not among the set of finite polynomials, following the general BMA methodology of Hoeting (1999). The paper provides a controlled test of BMA for polynomial extrapolation, complementing the lattice QCD test by Jay (2020) and the nuclear mass model applications by Kejzlar (2020) and Neufcourt (2019). The results enable practitioners to understand when BMA can improve extrapolation reliability and when it cannot compensate for fundamental limitations of the model set.
