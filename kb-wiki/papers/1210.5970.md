# Local Potential Model of the Hoyle Band in 12C

> We describe the excited 0+ state of 12C at 7.654 MeV, often called the Hoyle state, in terms of a local potential 8Be+alpha cluster model. We use a previously published prescription for the cluster-core potential to solve the Schroedinger equation to obtain wave functions for this state, and also for higher angular momentum states of the same system. We calculate energies, widths and charge radii for the resulting band of states, with particular emphasis on the recently discovered 2+ state. We examine various choices of the oscillator quantum number 2n + L = G for the cluster-core relative motion, and find that G = 6 leads to the most coherent description of the properties of the states and is consistent with recent experimental data on the L = 2 state.

## Key claim

The Hoyle state and its proposed rotational band in \(^{12}\)C can be described by a local potential \(^8\)Be+\(\alpha\) cluster model with a global quantum number \(G=6\), but only if one accepts that the model cannot describe the ground state band or the \(3^-\) state, and that the choice of \(G\) is determined *a posteriori* by comparison with experimental data rather than by oscillator shell-model considerations.

## Method

The authors solve the Schrödinger equation for a \(^8\)Be+\(\alpha\) two-body system using a deep, local nuclear potential with a previously published parametrization (fixed parameters \(v_0=54.0\) MeV, \(a=0.73\) fm, \(x=0.33\)), plus a Coulomb potential for a point cluster and uniformly charged spherical core. The potential radius \(R_0\) is fitted to reproduce the experimental energy of the Hoyle state (7.654 MeV) for each choice of global quantum number \(G=2n+L\) (with \(G=4,6,8\)). Energies and widths of resonant states are cross-checked using a bound-state code with a truncated potential, the complex-arithmetic code GAMOW, the elastic-scattering code SCAT2 (monitoring phase shifts), and a semiclassical approximation for charged-particle decay widths. Mean square charge radii are computed from the resulting wave functions using the known charge radius of the \(\alpha\) particle and a separate cluster-model description of \(^8\)Be.

## Key numbers

- Hoyle state (\(0^+\)) excitation energy: 7.654 MeV (used to fit the potential radius; experimental value).
- For \(G=6\): \(0^+\) width = 6 eV (experimental width is \(8.5 \pm 1.0\) eV).
- For \(G=6\): \(2^+\) state at \(9.61 \pm 0.36\) MeV (excitation energy) with width 360 keV.
- For \(G=6\): \(4^+\) state at \(13.71 \pm 1.24\) MeV with width 1.24 MeV.
- For \(G=6\): root-mean-square charge radius of the Hoyle state = 3.67 fm (from \(\langle r^2 \rangle = 13.457\) fm\(^2\)).
- For \(G=8\): \(2^+\) state at \(8.85 \pm 0.068\) MeV with width 68 keV; \(4^+\) state at \(11.52 \pm 0.30\) MeV with width 300 keV.
- For \(G=4\): no resonant \(2^+\) or \(4^+\) state exists.
- Potential radius \(R_0\) values: 0.8883 fm (\(G=4\)), 1.9386 fm (\(G=6\)), 2.9008 fm (\(G=8\)).

## Context

This work extends the local potential cluster model previously applied to nuclei from \(^6\)Li to \(^{242}\)Cm (Buck, Dover, and Vary, 1975) to the specific case of the Hoyle band in \(^{12}\)C, which had not been treated with this model because the ground state band of \(^{12}\)C is oblate (Vermeer, 1984) and a two-body cluster model inevitably produces prolate shapes. The model is motivated by recent experimental evidence for a \(2^+\) rotational excitation of the Hoyle state from Itoh et al. (2011), Freer et al. (2009), and Gai (2011). The results enable predictions for a possible \(4^+\) state near 14 MeV, which may correspond to a state recently reported by Freer et al. (2012) at \(13.3 \pm 0.2\) MeV, and provide a simple phenomenological benchmark against which more microscopic calculations (e.g., AMD, FMD, and 3\(\alpha\) condensate models) can be compared.
