# Microscopic understanding for the simplicity of the quadrupole moments newly observed in Cd isotopes

> The simplicity in the nuclear quadrupole moments reported recently in $^{107-129}$Cd, i.e., a linear increase of the ${11/2}^-$ quadrupole moments, is investigated microscopically with the covariant density functional theory. Using the newly developed functional PC-PK1, the quadrupole moments as well as their linear increase tendency with the neutron number are excellently reproduced without any {\it ad hoc} parameters. The core polarization is found to be very important and contributes almost half of the quadrupole moments. The simplicity of the linear increase is revealed to be due to the pairing correlation which smears out the abrupt changes induced by the single-particle shell structure.

## Key claim

The linear increase of the 11/2⁻ quadrupole moments in ¹⁰⁷–¹²⁹Cd is explained microscopically by pairing correlation smearing out abrupt single-particle shell-structure changes, rather than by the linear neutron occupation mechanism in the h₁₁/₂ shell previously assumed.

## Method

The authors performed self-consistent covariant density functional theory (CDFT) calculations using the point-coupling functional PC-PK1, solving the Dirac equation on an axially symmetric harmonic oscillator basis with 14 major shells. Pairing correlations were treated with the Bardeen-Cooper-Schrieffer (BCS) method using a zero-range δ force, with the h₁₁/₂ orbital (K = 11/2) blocked to obtain the 11/2⁻ states. The intrinsic quadrupole moments were projected to the laboratory frame using the standard relation for axially symmetric nuclei.

## Key numbers

- The calculated quadrupole moments for ¹⁰⁷–¹²⁹Cd reproduce the experimental linear increase tendency without any ad hoc parameters.
- For nuclei with N ≤ 71, the calculated quadrupole moments without pairing are slightly smaller than observed but still linear; for N ≥ 71, they deviate from both data and linearity.
- The neutron occupation number in the h₁₁/₂ shell without pairing shows an inflexion point at ¹¹⁹Cd (N = 71), where it starts increasing linearly from 1 to 11 for ¹²⁹Cd.
- The core (Z = 40) contributes almost half of the total quadrupole moments, comparable to the contribution from the eight valence protons in the g₉/₂ shell.
- A small offset term Q_const around 150 mb was applied to the calculated quadrupole moments in the shell-model comparison.

## Context

This work extends the experimental findings of Yordanov (2013), who first demonstrated that the linear behavior of quadrupole moments in Cd isotopes persists beyond a single shell. The authors contradict the linear occupation assumption proposed in that same experimental paper, showing instead that pairing correlation is the key mechanism. The covariant density functional approach with PC-PK1 provides a fully self-consistent microscopic description without effective charges, in contrast to earlier shell-model treatments that required ad hoc parameters. This enables future studies of quadrupole moments across long isotopic chains using universal density functionals.
