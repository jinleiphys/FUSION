# Neutrino-induced meson productions off nucleon at forward limit in nucleon resonance region

> We study forward neutrino-induced meson production off the nucleon in the resonance region. Our calculation is based on a dynamical coupled-channels (DCC) model that reasonably describes pi (gamma) N -> pi N, eta N, K Lambda, K Sigma data in the resonance region. We apply the PCAC hypothesis to the DCC model to relate the pi N reaction amplitude to the forward neutrino reaction amplitude. In this way, we give a prediction for nu N -> pi N, pi pi N, eta N, K Lambda, K Sigma reaction cross sections. The predicted nu N -> pi pi N, eta N, K Lambda, K Sigma cross sections are, for the first time, based on a model extensively tested by data. We compare our results with those from the Rein-Sehgal model that has been very often used in the existing Monte Carlo simulators for neutrino experiments. We find a significant difference between them.

## Key claim

The authors predict forward neutrino-induced meson production cross sections (nu N -> pi N, pi pi N, eta N, K Lambda, K Sigma) by applying the PCAC hypothesis to a dynamical coupled-channels model, but the prediction is strictly valid only at the forward limit (theta -> 0) where Q^2 = 0, and relies on the off-mass-shell extrapolation of the pion T-matrix.

## Method

The authors used a unitary dynamical coupled-channels (DCC) model that was previously fitted to approximately 20,000 data points from pi(gamma) N -> pi N, eta N, K Lambda, K Sigma reactions up to total energy W = 2.1 GeV. They applied the Partially Conserved Axial Current (PCAC) hypothesis to relate the divergence of the axial current to the pion field, thereby connecting the pi N reaction amplitude at Q^2 = 0 to the forward neutrino reaction amplitude. The resulting structure function W_2 at the forward limit was then used to compute cross sections for charged-current neutrino-induced meson production.

## Key numbers

* The DCC model was fitted to data up to total energy W = 2.1 GeV, covering the resonance region from the Delta(1232) through the 2nd and 3rd resonance regions up to W approximately 2 GeV.
* The analysis included fitting about 20,000 data points from pi(gamma) N -> pi N, eta N, K Lambda, K Sigma reactions.
* The forward limit cross section formula uses the Fermi constant G_F, the CKM matrix element V_ud, and the pion decay constant f_pi with pion mass m_pi.

## Context

This work extends the authors' previous dynamical coupled-channels model (Nakamura et al., 2011; Kamano et al., 2012) from electromagnetic and pion-induced reactions to neutrino-induced reactions. It provides the first predictions for nu N -> pi pi N, eta N, K Lambda, K Sigma cross sections based on a model extensively tested against data, in contrast to the Rein-Sehgal model which has been widely used in Monte Carlo simulators for neutrino experiments but shows significant differences from these results. This enables more precise modeling of neutrino-nucleus interactions in the resonance region for next-generation neutrino oscillation experiments targeting leptonic CP violation and mass hierarchy.
