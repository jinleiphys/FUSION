# Microscopic study of 4-alpha-particle condensation with proper treatment of resonances

> The 4-alpha condensate state for ^{16}O is discussed with the THSR (Tohsaki-Horiuchi-Schuck-Roepke) wave function which has alpha-particle condensate character. Taking into account a proper treatment of resonances, it is found that the 4-alpha THSR wave function yields a fourth 0^+ state in the continuum above the 4-alpha-breakup threshold in addition to the three 0^+ states obtained in a previous analysis. It is shown that this fourth 0^+ ((0_4^+)_{THSR}) state has an analogous structure to the Hoyle state, since it has a very dilute density and a large component of alpha+^{12}C(0_2^+) configuration. Furthermore, single-alpha motions are extracted from the microscopic 16-nucleon wave function, and the condensate fraction and momentum distribution of alpha particles are quantitatively discussed. It is found that for the (0_4^+)_{THSR} state a large alpha-particle occupation probability concentrates on a single-alpha 0S orbit and the alpha-particle momentum distribution has a delta-function-like peak at zero momentum, both indicating that the state has a strong 4-alpha condensate character. It is argued that the (0_4^+)_{THSR} state is the counterpart of the 0_6^+ state which was obtained as the 4-alpha condensate state in the previous 4-alpha OCM (Orthogonality Condition Model) calculation, and therefore is likely to correspond to the 0_6^+ state observed at 15.1 MeV.

## Key claim

The fourth 0+ state extracted from the THSR wave function with proper resonance treatment, rather than the third 0+ state as previously thought, is the true 4-alpha condensate state in 16O and likely corresponds to the experimental 0_6+ state at 15.1 MeV, contingent on the proper treatment of resonances above the 4-alpha breakup threshold.

## Method

The authors employed the THSR (Tohsaki-Horiuchi-Schuck-Roepke) wave function for 4-alpha systems and solved the Hill-Wheeler equation using 20 discrete generator coordinates. To extract the resonance state above the 4-alpha threshold, they applied a method based on diagonalizing the squared-radius operator among continuum eigenstates obtained by varying a pseudo-potential strength parameter delta, which allowed separation of the resonance component from continuum admixtures.

## Key numbers

- The (0_4+)_THSR state lies at 3.04 MeV above the 4-alpha threshold, compared to the (0_6+)_OCM state at 2.08 MeV above threshold
- The (0_4+)_THSR state has a root-mean-square radius of 6.1 fm, larger than the (0_3+)_THSR state at 4.2 fm and comparable to the (0_6+)_OCM state at 5.6 fm
- The (0_4+)_THSR state has an alpha decay width of 0.14 MeV, while the (0_3+)_THSR state has a width of 1.6 MeV
- The monopole matrix element M(E0) for the (0_4+)_THSR state is 1.2 fm^2, compared to 2.5 fm^2 for the (0_3+)_THSR state
- The ground state (0_1+)_THSR has binding energy -15.05 MeV relative to the 4-alpha threshold and r.m.s. radius 2.5 fm

## Context

This work extends the previous THSR analysis of Funaki et al. (2008) which found only three 0+ states, by properly treating resonances to reveal a fourth state. It directly confronts the 4-alpha OCM calculation of the same authors (2010) which identified the 0_6+ state at 15.1 MeV as the 4-alpha condensate. The new (0_4+)_THSR state is argued to be the counterpart of the (0_6+)_OCM state, resolving the discrepancy between the earlier THSR and OCM results and enabling a consistent description of alpha-particle condensation in 16O.
