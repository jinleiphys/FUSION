# A Three-Dimensional Treatment of the Three-Nucleon Bound State

> Recently a formalism for a direct treatment of the Faddeev equation for the three-nucleon bound state in three dimensions has been proposed. It relies on an operator representation of the Faddeev component in the momentum space and leads to a finite set of coupled equations for scalar functions which depend only on three variables. In this paper we provide further elements of this formalism and show the first numerical results for chiral NNLO nuclear forces.

## Key claim

The paper presents a three-dimensional (3D) operator-based formalism for solving the Faddeev equation for the three-nucleon (3N) bound state, avoiding the traditional partial-wave decomposition. The first numerical results are shown for chiral NNLO nuclear forces, but the formalism is designed to handle more complex chiral forces at N3LO, which are tedious to treat in partial-wave representations.

## Method

The Faddeev component of the 3N bound state is expanded in momentum space using eight scalar functions \(\phi_{tT}^{(i)}(|\vec p|,|\vec q|,\hat{\vec p}\cdot\hat{\vec q})\) multiplied by spin-momentum operators \(O_i\) acting on a spin-isospin reference state. The Faddeev equation is rewritten using the 2N potential directly (instead of the t-matrix) and a 3N force, leading to a set of coupled integral equations for the scalar functions. The equations are solved by iteration with a Lanczos-type algorithm, using cubic Hermite spline interpolation for the three-dimensional momentum variables.

## Key numbers

* The determinant of the \(C_{ki}\) coefficient matrix is \(\frac{2916}{5} |\hat{\vec p} \times \hat{\vec q}|^{12} = \frac{2916}{5} (1 - x^2)^6\), where \(x = \hat{\vec p} \cdot \hat{\vec q}\), indicating potential numerical issues when \(\hat{\vec p} \times \hat{\vec q} = 0\).
* The normalization condition for the full 3N wave function is given by \(1 = 12\pi^2 \sum_{tT} \sum_{i,j=1}^8 \int_0^\infty dp\,p^2 \int_0^\infty dq\,q^2 \int_{-1}^1 dx\, \phi_{tT}^{(i)}(p,q,x) \beta_{tT}^{(j)}(p,q,x) \tilde{C}_{ij}(x)\).
* The expectation value of the kinetic energy is \(\langle E_{kin} \rangle = 12\pi^2 \sum_{tT} \sum_{i,j=1}^8 \int_0^\infty dp\,p^2 \int_0^\infty dq\,q^2 \left( \frac{p^2}{m} + \frac{3q^2}{4m} \right) \int_{-1}^1 dx\, \phi_{tT}^{(i)}(p,q,x) \beta_{tT}^{(j)}(p,q,x) \tilde{C}_{ij}(x)\).

## Context

This work extends the 3D operator formalism for the 3N bound state previously proposed by the same group (Golak et al., 2012, Ref. [2N3N]), which itself built on earlier operator representations by Gerjuoy (Ref. [gerjuoy]) and Fachruddin et al. (Ref. [fach04]). The new formalism avoids expanding the 3N force into spin-momentum operators, simplifying the integral kernel and reducing the number of scalar coefficients compared to the earlier approach. This enables future calculations with chiral 3N forces at N3LO, which have a very rich spin and momentum structure that makes partial-wave calculations tedious (as noted for triton calculations at N3LO, Ref. [tritonn3lo]).
