# Collective neutrino oscillations with tensor networks using a time-dependent variational principle

> If a system of flavor-oscillating neutrinos is at high enough densities that neutrino-neutrino coherent forward scatterings are non-negligible, the system becomes a time-dependent many-body problem. An important and open question is whether the flavor evolution is sufficiently described by a mean-field approach or can be strongly affected by correlations arising from two-body interactions in the neutrino Hamiltonian, as measured by nontrivial quantum entanglement. Numerical computations of the time evolution of many-body quantum systems are challenging because the size of the Hilbert space scales exponentially with the number of particles N in the system. Thus, it is important to investigate approximate but beyond-mean-field numerical treatments at larger values of N. Here we investigate the efficacy of tensor network methods to calculate the time evolution of interacting neutrinos at larger values of N than are possible with conventional methods. In particular, we introduce the use of time-dependent variational principle methods to address the long-range (in momentum space) interactions of the neutrino Hamiltonian when including many distinct vacuum oscillation frequencies. We also define new error measures based upon the instantaneously conserved charge operators known for this Hamiltonian to determine validity of large-N tensor network calculations.

## Key claim

Tensor network methods using a time-dependent variational principle (TDVP) can provide significant computational speedup for simulating collective neutrino oscillations beyond mean-field theory, but the usefulness depends considerably on the number of spectral splits resulting from the initial condition; for initial conditions with fewer spectral split frequencies, the entanglement can be efficiently described by a matrix-product state representation, while for systems with two spectral splits the required basis set is much larger.

## Method

The authors model a system of flavor-oscillating neutrinos using a two-flavor, single-angle Hamiltonian with SU(2) neutrino isospin operators, evolving the many-body wave function via the time-dependent Schrodinger equation. They implement tensor network calculations using matrix-product states evolved with time-dependent variational principle methods to handle the long-range (in momentum space) interactions of the neutrino Hamiltonian, employing SWAP operations to localize the nonlocal interactions. For comparison, they also perform exact calculations using fourth-order Runge-Kutta integration and Lanczos propagation in a sparse-matrix representation for systems up to N=16 neutrinos.

## Key numbers

- Tensor network calculations (GSE-TDVP2) permit substantial speedup for larger-N calculations compared to RK4 and Lanczos methods, with computation times shown for systems up to N=16 on a single 2.5 GHz Quad-Core Intel Core i7 processor
- For the box spectrum with discrete, equally spaced vacuum oscillation frequencies, each oscillation frequency is occupied by a single neutrino with mixing angle theta = 0.161
- The single-angle coupling uses parameters: R_nu = 32.2 omega_0^-1, mu(R_nu) = 3.62 x 10^4 omega_0, and initial time/radius given by mu(r) = 5 omega_0
- For N=2 to 16, Lanczos propagation approximating the evolved state to order (delta t)^5 produces wave function coefficients with discrepancy less than 10^-3 compared to RK4 results
- The time step is prescribed to scale as delta t ~ 0.1 [mu * N/2 * (N/2 + 1) + sum_omega |omega|]^-1

## Context

This work extends previous studies of collective neutrino oscillations beyond mean-field theory, which had been limited by numerical instability of Bethe ansatz solutions for N >= 10 (Patwardhan 2019, Cervia 2019) and by exponential scaling of direct RK4 integration that could only reach N up to 16 (Patwardhan 2021). The authors build on tensor network approaches recently proposed for the nonlocal neutrino Hamiltonian using SWAP gate networks (Roggero 2021, Martin 2021), but those tests were limited to just a few different neutrino momenta using the two-beam model. This paper introduces TDVP methods to include many distinct vacuum oscillation frequencies and defines new error measures based on instantaneously conserved charge operators, enabling consistency checks for N beyond the capabilities of exact methods on modern hardware.
