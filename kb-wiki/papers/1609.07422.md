# Heavy quark fragmentation functions at next-to-leading perturbative QCD

> It is well-known that the dominant mechanism to produce hadronic bound states with large transverse momentum is fragmentation. This mechanism is described by the fragmentation functions (FFs) which are the universal and process-independent functions. Here, we review the perturbative FFs formalism as an appropriate tool for studying these hadronization processes and detail the extension of this formalism at next-to-leading order (NLO). Using the Suzuki's model, we calculate the perturbative QCD FF for a heavy quark to fragment into a S-wave heavy meson at NLO. As an example, we study the LO and NLO FFs for a charm quark to split into the S-wave $D$-meson and compare our analytic results both with experimental data and well-known phenomenological models.

## Key claim

The paper claims that next-to-leading order (NLO) perturbative QCD corrections to the Suzuki model for heavy quark fragmentation functions improve agreement with experimental data compared to leading-order (LO) results, but this improvement relies on replacing transverse momentum integrations by their average values and neglecting virtual gluon corrections, which the authors argue are small.

## Method

The authors calculate the fragmentation function for a heavy quark Q to fragment into an S-wave heavy meson M(Q\bar{q}) at NLO in perturbative QCD using the Suzuki model, which combines perturbative Feynman diagrams with a delta-function approximation for the meson bound state wave function. They compute the real gluon radiative corrections from four Feynman diagrams (Fig. 2) and replace transverse momentum integrations by their average value <k_T^2>, while arguing that virtual gluon corrections can be ignored. The LO result from a previous work (Nejad 2013) is combined with the NLO real corrections, and the total function is normalized to unity.

- Strong coupling constant α_s(m_c) = 0.38 ± 0.03, adjusted such that α_s(m_Z) = 0.1184 with m_Z = 91.18 GeV
- Average transverse momentum squared <k_T^2> = 1 GeV^2 is used as an extreme value for the numerical analysis
- QCD scale Λ ≈ 231 MeV from experimental measurements of e^-e^+ reactions (Nakamura 2010)

## Context

This work extends the Suzuki model (Suzuki 1977, 1985) for heavy quark fragmentation functions from leading order to next-to-leading order in perturbative QCD. It builds on previous LO calculations by Nejad (2013) and MoosaviNejad (2016) for c-quark fragmentation into D^0/D^+-mesons, and compares results with experimental data from BELLE and CLEO collaborations. The approach contrasts with phenomenological methods that fit free parameters to experimental data (Soleymaninia 2014), and with the pQCD approach of Braaten (1993) which integrates over the invariant mass rather than using the Suzuki model's transverse momentum averaging. The NLO results enable more accurate predictions for heavy meson production at the LHC and can be evolved to higher scales using DGLAP evolution equations.
