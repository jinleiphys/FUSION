# g1(x) and g2(x) in the Meson Cloud Model

> We calculate the spin dependent structure functions g1(x) and g2(x) of the proton and neutron. Our calculation uses the meson cloud model of nucleon structure and includes the effects of kinematic terms which mix transverse and longitudinal spin components. We find small corrections to the nucleon structure functions, however these are significant for the neutron.

## Key claim

The meson cloud model (MCM) contributions to the spin-dependent structure functions g1(x) and g2(x) are small for the proton but around 10% of the experimental data for the neutron, which is significant because these corrections have similar magnitude to higher twist components and a weak scale dependence that can mimic twist-3 contributions at low Q2.

## Method

The calculation uses the meson cloud model, where the physical nucleon is expanded as a Fock state of a bare nucleon plus baryon-meson components (primarily N → Nπ and N → Δπ). The spin-dependent structure functions are computed via convolution integrals that include both longitudinal and transverse spin projections of the cloud baryons, using time-ordered perturbation theory in the infinite momentum frame. The bare hadron structure functions are calculated using the MIT bag model with methods from the Adelaide group, and a phenomenological Δg(x) is added so that the integral of g1p(x) agrees with experiment.

## Key numbers

- MCM contributions to the neutron structure functions are around 10% of the size of the experimental data.
- The correction to g2n(x) is positive and may account for the deviation of the JLab E97-103 data around x = 0.2 and Q2 = 1 GeV2 from the Wandzura-Wilczek term.
- Leading twist contributions to g2 from the meson cloud are of the order of 10% of the structure function.

## Context

This work extends earlier MCM calculations of spin-dependent sea distributions (Δū(x), Δđ(x), s(x), Δs̄(x)) by Signal and collaborators (2005), which agreed with HERMES data, to the valence-dominated structure functions g1 and g2. The calculation incorporates the formalism of Kumano and Miyama (2000) for handling transverse spin components at finite Q2, and uses the MIT bag model approach of the Adelaide group (Boros and Thomas, 2000). The results enable more precise extraction of higher twist components from neutron structure function data, particularly for experiments like JLab E97-103.
