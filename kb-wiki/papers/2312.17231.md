# Static force from generalized Wilson loops on the lattice using the gradient flow

> The static QCD force from the lattice can be used to extract $Λ_{\overline{\textrm{MS}}}$, which determines the running of the strong coupling. Usually, this is done with a numerical derivative of the static potential. However, this introduces additional systematic uncertainties; thus, we use another observable to measure the static force directly. This observable consists of a Wilson loop with a chromoelectric field insertion. We work in the pure SU(3) gauge theory. We use gradient flow to improve the signal-to-noise ratio and to address the field insertion. We extract $Λ_{\overline{\textrm{MS}}}^{n_f=0}$ from the data by exploring different methods to perform the zero-flow-time limit. We obtain the value $\sqrt{8t_0} Λ_{\overline{\textrm{MS}}}^{n_f=0} =0.629^{+22}_{-26}$, where $t_0$ is a flow-time reference scale. We also obtain precise determinations of several scales: $r_0/r_1$, $\sqrt{8 t_0}/r_0$, $\sqrt{8 t_0}/r_1$ and we compare these to the literature. The gradient flow appears to be a promising method for calculations of Wilson loops with chromoelectric and chromomagnetic insertions in quenched and unquenched configurations

## Key claim

The paper claims that the static QCD force can be measured directly using a Wilson loop with a chromoelectric field insertion combined with gradient flow, avoiding the systematic uncertainties introduced by numerical differentiation of the static potential; however, the extraction of $\Lambda_{\overline{\textrm{MS}}}^{n_f=0}$ requires careful handling of the zero-flow-time limit and the method works reliably only when the flow radius exceeds one lattice spacing ($\sqrt{8\tau_F}>a$).

## Method

The authors performed lattice simulations in pure SU(3) gauge theory using the Wilson action on four ensembles (L20, L26, L30, L40) with different lattice spacings. They computed the static force directly from Wilson loops with a chromoelectric field insertion, applied gradient flow to improve the signal-to-noise ratio and reduce discretization effects, and extracted the renormalization factor $Z_E$ nonperturbatively by comparing the direct force measurement to the numerical derivative of the static potential. The continuum limit was taken using extrapolations linear and quadratic in $a^2/t_0$ at fixed physical distances and flow times, with Akaike information criterion averaging over fit ranges and polynomial orders.

## Key numbers

- $\sqrt{8t_0} \Lambda_{\overline{\textrm{MS}}}^{n_f=0} = 0.629^{+22}_{-26}$ (final result from the constant zero-flow-time limit method)
- $r_0/r_1 = 1.380(14)$ (ratio of reference scales in pure gauge)
- $\sqrt{8t_0}/r_0 = 0.9569(66)$ (ratio of gradient flow scale to hadronic scale)
- $\sqrt{8t_0}/r_1 = 1.325(13)$ (ratio of gradient flow scale to hadronic scale)
- String tension: $\sigma t_0 = 0.151(3)$ from Cornell fit over $r/\sqrt{t_0} \approx 1.1$ to $3.0$
- String tension in $r_0$ units: $\sigma r_0^2 = 1.345(54)$
- $Z_E = 1$ within 1% deviation for flow radii larger than one lattice spacing ($\sqrt{8\tau_F}/a > 1$)

## Context

This work extends the direct force measurement approach from Brambilla (2021) by incorporating gradient flow to handle the chromoelectric field insertion, and it provides the first direct determination of the scale ratio $r_0/r_1$ in pure gauge theory, which was previously only estimated at about 1.37 from curves in Necco (2001). The extracted ratio $\sqrt{8t_0}/r_0$ agrees within errors with previous determinations by Luscher (2010), Asakawa (2015), Francis (2015), Ce (2015), Kamata (2016), Knechtli (2017), Giusti (2018), and Dalla Brida (2019), while the ratio $\sqrt{8t_0}/r_1$ is the first direct measurement in pure gauge. The results enable future applications of gradient flow to Wilson loops with chromoelectric and chromomagnetic insertions in both quenched and unquenched configurations.
