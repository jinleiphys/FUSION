# A High-Performance Algorithm to Calculate Spin- and Parity-Dependent Nuclear Level Densities

> A new algorithm for calculating the spin- and parity-dependent shell model nuclear level densities using the moments method in the proton-neutron formalism is presented. A new, parallelized code based on this algorithm was developed and tested using up to 4000 cores for a set of nuclei from the sd-, pf-, and pf + g9/2 - model spaces. By comparing the nuclear level densities at low excitation energy for a given nucleus calculated in two model spaces, such as pf and pf + g9/2, one could estimate the ground state energy in the larger model space, which is not accessible to direct shell model calculations due to the unmanageable dimension. Examples for the ground state energies of for 64Ge and 68Se in the pf + g9/2 model space are presented.

## Key claim

The authors present a new algorithm for calculating spin- and parity-dependent nuclear level densities using the moments method in the proton-neutron formalism, which achieves near-perfect parallel scaling up to 4000 cores. The key advance is that by comparing level densities at low excitation energy between a smaller solvable model space and a larger inaccessible model space, one can estimate the ground state energy in the larger space without direct diagonalization.

## Method

The algorithm calculates fixed-spin and fixed-parity configuration centroids and widths of the nuclear shell-model Hamiltonian using trace propagation functions in a proton-neutron separated basis. The proton and neutron dimensions are computed separately and then combined via convolution to obtain the full J-dependent traces, which are used to construct finite-range Gaussian level densities for each configuration. The method was implemented in a parallelized code and tested on nuclei in the sd-, pf-, and pf+g9/2 model spaces using up to 4000 cores on the FRANKLIN supercomputer at NERSC.

## Key numbers

- For 28Si in the sd-shell with USD interaction, the ground state energy used was E_gs = -135.94 MeV, with cut-off parameter eta = 2.8
- For 52Fe and 52Cr in the pf-shell with GXPF1A interaction, the cut-off parameter was eta = 2.6
- For 64Ge in the pf+g9/2 model space, the estimated ground state energy was E_gs = -306.7 MeV (compared to -304.25 MeV in the pf-shell)
- For 68Se in the pf+g9/2 model space, the estimated ground state energy was E_gs = -356.5 MeV (compared to -353.1 MeV in the pf-shell)
- Calculation time for 68Se in pf+g9/2 (total m-scheme dimension ~10^15) was 1.03 x 10^4 seconds on a 16-core machine with 2.8 GHz CPU frequency
- Calculation time for 28Si in sd-shell (dimension ~10^6) was 0.7 seconds on the same 16-core machine
- The error factor f_err for the moments method was in the range 0.1-0.2 for optimal cut-off parameters eta between 2.5 and 3.0

## Context

This work extends the statistical spectroscopy approach of French (1983) and Wong (1986) by implementing a proton-neutron formalism that improves parallel scalability over previous methods. The algorithm enables level density calculations for nuclei like 64Ge and 68Se, which are waiting-point nuclei in the rp-process path studied by Schatz (1998) and others. The ground state energy extraction method provides an alternative to the exponential convergence method of Horoi (2004) and the projected configuration interaction method, and the resulting level densities can be compared with the Hartree-Fock-Bogoliubov plus combinatorial method of Goriely (2008).
