# Prescriptions for the scaling variable of the nucleon structure function in nuclei

# Prescriptions for the scaling variable of the nucleon structure function in nuclei

## Key claim

The authors test whether choosing an appropriate in-medium definition of the Bjorken scaling variable \(x_N\) can absorb all nuclear modifications and allow the nucleon structure function in a nucleus to be described by the free-nucleon structure function. They find that none of the tested prescriptions for \(x_N\) yields agreement with experimental EMC data, unambiguously showing that the in-medium nucleon structure function is different from that of a free nucleon.

## Method

The authors compute the isoscalar EMC ratio \(R_A(x,Q^2)\) for several nuclei using three different prescriptions for the in-medium scaling variable \(x_N\): their own earlier prescription from Ryazanov et al. (2006), the prescription of Molochkov (using the Bethe–Salpeter formalism with a different choice of integration contour), and the prescription of Pandharipande and coworkers (based on energy–momentum conservation for a bound nucleon). The calculations use the MRST 2002 NLO parton distributions for the free nucleon and the Bonn-B wave function for the deuteron, with the nuclear spectral function taken as the doorway-state spectral function rather than the ground-state spectral function.

## Key numbers

- The interaction time of deep inelastic scattering is \(2q/Q^2 = (mx)^{-1} = \frac{0.7}{x} \times 10^{-24}\) sec, which for \(x > 0.3\) is less than \(3 \times 10^{-24}\) sec, an order of magnitude shorter than the fragmentation time of hole states (20 MeV spreading width, corresponding to \(3 \times 10^{-23}\) sec).
- The three interaction regions are defined by the parton size \((mx)^{-1} = \frac{0.21}{x}\) fm: correlation region \(0 < x < 0.2\) (size exceeds inter-nucleon distance \(r_0 = 1.2\) fm), one-nucleon region \(0.2 < x < 0.8\), and competition region \(0.8 < x \sim 1\).
- The EMC ratio for \(^{56}\)Fe is calculated at \(Q^2 = 5\) GeV\(^2\) and \(Q^2 = 100\) GeV\(^2\), showing insignificant \(Q^2\) dependence.

## Context

This work extends the authors' earlier EMC calculations (Ryazanov et al., 2006) which used the doorway-state spectral function and found that assuming the free-nucleon structure function with their own \(x_N\) prescription did not agree with data. Here they test two additional prescriptions: Molochkov's Bethe–Salpeter-based definition and the prescription of Pandharipande and coworkers (based on binding-energy corrections to the energy transfer). The results contradict the hope that a suitable choice of \(x_N\) could absorb all in-medium effects, and instead indicate that the nucleon structure function itself is modified in the nuclear medium. This motivates future work to determine the actual form of the in-medium structure function \(F_{2m}\).
