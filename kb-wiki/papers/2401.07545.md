# Calculation of Fission Fragment Yields for thermal neutron reaction of $^{239}$Pu

> Fission fragment yield evaluation in the past has been done mainly by considering independent and cumulative fission yields. In addition to them, the fission fragment yields are related to various observables such as total kinetic energy, prompt fission neutron, decay heats, etc. Various improvements were carried out in the fission fragment yield evaluation of JENDL-5, however it did not consider correlations between such fission observables and fragment yields. A new evaluation including fission observables is thus required for next generation of evaluated data. We recently developed a new calculation system using CCONE code to calculate fission fragment yields. This calculation system enables us to compare various fission observables with the experimental data simultaneously. In this work, we calculated thermal-induced fission of 239Pu. We present the result of prompt fission neutrons, decay heats, and delayed neutrons

## Key claim

The paper presents a new calculation system using the CCONE code that simultaneously estimates fission fragment yields and multiple fission observables (prompt neutrons, decay heats, delayed neutrons) for thermal neutron-induced fission of 239Pu, but the abstract glosses over that the calculated neutron multiplicities show deviations from experimental data for light fragments (60 ≤ A ≤ 120) while agreeing better for heavy fragments (135 < A ≤ 150).

## Method

The pre-neutron emission mass yields (PFY) were approximated by a sum of four Gaussians with parameters determined from least-square fitting to experimental data. Charge distributions used the Zp-model with modifications to the most probable atomic number and standard deviation, and total kinetic energy (TKE) was fitted using a function originally proposed by Okumura (2018). Excitation energy distributions between light and heavy fragments were estimated using an anisothermal model, and the final fission fragment yields including spin-parity distributions were calculated with odd-even effect parameters adjusted by hand to match delayed and prompt neutron experimental data.

## Key numbers

- Pre-neutron mass yield Gaussian parameters: N1=0.21, μ1=14.44, σ1=3.593, N2=0.79, μ2=20.60, σ2=6.299
- TKE fitting parameters: p0=220.4, p1=1.050, p2=0.1303, p3=29.77
- Average TKE value from experimental data: 178 MeV
- TKE width parameters: s0=5.520, s1=5.750, s3=5512
- Odd-even effect parameters: fz=1.120, fn=1.020
- Spin distribution parameter: fs=2.756
- Scaling factor of asymptotic level density parameter: fa=0.6

## Context

This work extends the author's previous calculation system developed for thermal neutron-induced fission on 235U (Minato 2024) to 239Pu, and addresses a limitation of the JENDL-5 evaluation which did not consider correlations between fission observables and fragment yields. The new CCONE-based system enables simultaneous comparison of multiple fission observables (prompt neutrons, decay heats, delayed neutrons) with experimental data, and the author states they will calculate fission fragment yields of different systems using this code in future work.
