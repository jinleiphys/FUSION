# Bose-Einstein correlations for Levy stable source distributions

> The peak of the two-particle Bose-Einstein correlation functions has a very interesting structure. It is often believed to have a multivariate Gaussian form. We show here that for the class of stable distributions, characterized by the index of stability $0 < α\le 2$, the peak has a stretched exponential shape. The Gaussian form corresponds then to the special case of $α= 2$. We give examples for the Bose-Einstein correlation functions for univariate as well as multivariate stable distributions, and check the model against two-particle correlation data.

## Key claim

The paper shows that for the class of Lévy stable source distributions, characterized by the index of stability \(0 < \alpha \le 2\), the peak of the two-particle Bose-Einstein correlation function has a stretched exponential shape, with the Gaussian form corresponding only to the special case of \(\alpha = 2\). This result is exact for all values of the relative momentum \(q\), not an approximation valid only near \(q=0\).

## Method

The authors derive the two-particle Bose-Einstein correlation function from the Fourier transform (characteristic function) of Lévy stable source distributions, using Nolan's \(S(\alpha,\beta,\gamma,\delta;1)\) convention. For symmetric stable distributions, the correlation function takes the form \(C(q) = 1 + \exp(-|qR|^\alpha)\), which generalizes the Gaussian (\(\alpha=2\)) and exponential (\(\alpha=1\)) cases. The analysis is extended to multivariate symmetric stable distributions with ellipsoidal contour lines, and to three-particle correlations which can recover the asymmetry parameter \(\beta\) that is inaccessible from two-particle correlations alone.

## Key numbers

- The index of stability \(\alpha\) can take any value in the interval \(0 < \alpha \le 2\), with \(\alpha = 2\) corresponding to the Gaussian case and \(\alpha = 1\) to the Lorentzian (Cauchy) case.
- For symmetric stable distributions, the two-particle Bose-Einstein correlation function is \(C(q;\alpha) = 1 + \exp(-|qR|^\alpha)\) for all values of \(q\).
- For the asymmetric Lévy distribution with \(\alpha = 1/2, \beta = 1\), the correlation function is \(C(q) = 1 + \exp(-\sqrt{|qR|})\).
- Typical values of the minimum relative momentum \(Q_{min}\) are 5-10 MeV in present correlation measurements in heavy ion physics.
- The large-distance behavior of the density distribution for multivariate symmetric stable distributions scales as \(f(\mathbf{x}) \approx (\sum_{ij} R^{-2}_{ij} x_i x_j)^{-(1+\alpha)/2}\) for \(0 < \alpha < 2\).

## Context

This work extends the earlier observation in Csörgő (1999) and the appendix of Csörgő and collaborators (2001) that stable distributions play an important role in non-Gaussian Bose-Einstein correlation studies, and generalizes the Gaussian approximation that had been commonly assumed in the field. The results enable model-independent characterization of source distributions using the stretched exponential form, and show that three-particle correlations can recover the asymmetry parameter \(\beta\) that is inaccessible from two-particle correlations alone, building on the formalism of the Yano-Koonin approach and the generalized Central Limit Theorem.
