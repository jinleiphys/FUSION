# The vector boson transverse momentum distributions

> The transverse momentum dependent distributions (TMD) are an essential part of the factorization theorems in vector boson production. They are non-perturbative, double scale dependent functions that asymptotically match onto collinear parton distributions functions (PDF). Once TMD are expressed using PDF, one observes that they are sensitive to the choice and quality of PDF sets (PDF bias). A solution to this problem is found and discussed. Nevertheless the main source of error on vector boson spectra still comes from PDF uncertainty propagation.

## Key claim

The paper demonstrates that the flavor dependence of the non-perturbative part of transverse momentum dependent distributions (TMD) is essential to reduce the sensitivity of TMD extractions to the choice of collinear PDF set (the "PDF bias"), but even after this improvement, the dominant source of uncertainty on vector boson transverse momentum spectra remains the propagation of PDF uncertainties.

## Method

The study uses the public code artemide to perform global fits of Drell-Yan data, extracting the non-perturbative TMD function f^NP with a flavor-dependent ansatz that distinguishes u, d, anti-u, anti-d, and sea quark flavors (11 free parameters total). The fits are performed separately using four different PDF sets (CT18, HERA20, MSHT20, NNPDF31), each with 1000 replicas, and the experimental uncertainty is evaluated by generating pseudo-data with Gaussian noise. The perturbative inputs for TMD are at next-to-next-to-leading order (NNLO), with the cusp anomalous dimension at order alpha_s^3.

## Key numbers

- The final chi^2 per data point for each PDF set: 1.12 (MSHT20), 0.91 (HERA20), 1.21 (NNPDF31), 1.08 (CT18)
- More than 75% of PDF replicas in each set have chi^2/N_pt > 2, while the central replica describes data with chi^2/N_pt ~ 1
- The fitted parameter lambda_0 (universal for all flavors) ranges from 24 (CT18) to 339 (HERA20), with uncertainties spanning roughly 10 to 200
- The parameter c0 (normalization) ranges from 0.0261 (NNPDF31) to 0.0439 (CT18), with uncertainties of about 0.003 to 0.01
- The flavor-dependent parameters lambda_1^u range from 0.05 (CT18) to 0.28 (NNPDF31), with uncertainties typically 0.05-0.12

## Context

This work extends the SV19 extraction (Scimemi, 2019) by introducing a flavor-dependent non-perturbative TMD model to resolve the PDF bias problem identified in that earlier analysis. The study uses the zeta-prescription (Scimemi, 2018) for factorization and builds on NNLO perturbative calculations (Gehrmann, 2014; Echevarria, 2015-2016; Luo, 2019). The results enable more reliable predictions for vector boson production at LHC energies, with the authors noting that recent CMS and LHCb data at sqrt(s)=13 TeV are correctly predicted using their settings, to be shown in future publications.
