# (3+1)-dimensional dissipative relativistic fluid dynamics at non-zero net baryon density

> Heavy-ion collisions at center-of-mass energies between 1 and 100 GeV/nucleon are essential to understand the phase diagram of QCD and search for its critical point. At these energies the net baryon density of the system can be high, and simulating its evolution becomes an indispensable part of theoretical modeling. We here present the (3+1)-dimensional diffusive relativistic hydrodynamic code BEShydro which solves the equations of motion of second-order Denicol-Niemi-Molnar-Rischke (DNMR) theory, including bulk and shear viscous currents and baryon diffusion currents. BEShydro features a modular structure that allows to easily turn on and off baryon evolution and different dissipative effects and thus to study their physical effects on the dynamical evolution individually. An extensive set of test protocols for the code, including several novel tests of the precision of baryon transport that can also be used to test other such codes, is documented here and supplied as a permanent part of the code package.

## Key claim

The paper presents the (3+1)-dimensional dissipative relativistic hydrodynamic code BEShydro, which solves second-order DNMR theory equations including bulk and shear viscous stresses and baryon diffusion currents, but the authors note that they have omitted certain second-order couplings between baryon diffusion and viscous stresses, following arguments that these effects are negligible for most observables.

## Method

The code solves the conservation laws for energy, momentum, and net baryon number in Milne coordinates, coupled to relaxation-type evolution equations for the bulk viscous pressure, shear stress tensor, and baryon diffusion current derived from DNMR theory. The numerical scheme uses a Runge-Kutta Kurganov-Tadmor algorithm on a fixed computational grid, with transport coefficients obtained from kinetic theory in the relaxation time approximation for an almost massless Boltzmann gas. The code features a modular structure allowing independent activation of baryon evolution and different dissipative effects.

## Key numbers

- Memory usage: 0.57 GB for conserved variables, 0.003 GB for EOS4 tables, 0.035 GB for dynamical sources on a 121×121×121 grid
- Running time: 0.32 sec/time step on a 2D 121×121 grid, 46.08 sec/time step on a 3D 121×121×121 grid (MacBook Pro with 2.7 GHz Intel Core i5)
- Default kinematic shear viscosity: constant value of 0.2
- Pseudocritical temperature for bulk viscosity parametrization: 155 MeV
- Bulk viscosity relaxation time relation: ζ/τ_Π = 15(1/3 - c_s²)²(ε + p)
- Shear viscosity relaxation time relation: η/τ_π = (ε + p)/5
- Baryon diffusion relaxation time parametrization: τ_n = C_B/T with free parameter C_B

## Context

This work extends the CPU version of the GPU-VH code (Bazow, 2016) by adding bulk and shear viscous evolution equations alongside net baryon charge and diffusion current evolution. It is developed independently from the MUSIC code (Denicol, 2018), which shares similar capabilities, and is designed for the Beam Energy Scan program at RHIC as well as future experiments at NICA and FAIR. The code enables systematic studies of dissipative baryon transport effects in heavy-ion collisions at center-of-mass energies between 1 and 100 GeV/nucleon, where net baryon density becomes significant.
