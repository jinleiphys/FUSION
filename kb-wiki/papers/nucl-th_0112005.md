# Two-fermion bound states in the explicitly covariant Light-Front Dynamics

> We develop a general method to construct two relativistic fermions bound states with a given $J^π$, in the framework of the explicitly covariant light-front dynamics.

# Two-fermion bound states in the explicitly covariant Light-Front Dynamics

## Key claim

The paper develops a general method to construct relativistic two-fermion bound states with definite total angular momentum and parity in explicitly covariant light-front dynamics, but the angular condition (required for rotational invariance) can only be satisfied approximately when using a one-boson-exchange kernel, leading to a mass splitting between states with different light-front angular momentum projections that should be degenerate in the exact solution.

## Method

The authors solve the bound state problem for two relativistic fermions in the ladder approximation using the explicitly covariant light-front dynamics framework, where the state vector is defined on a hyperplane given by ω·x = 0 with ω² = 0. They decompose the wave functions into covariant spin structures multiplied by scalar components, constructing eigenstates of J² and A² = (n·J)² for J = 0⁺ and J = 1⁺ states. For J = 1⁺, they impose the angular condition approximately by requiring the wave function at k = 0 to be independent of the light-front orientation vector n, forming a superposition of states with different a values (a = 0 and a = 1).

## Key numbers

- For J = 1⁺ with scalar exchange (meson mass μ = 0.25, fermion mass m = 1, coupling α = 1.18), the binding energy B = 2m - M = 0.0501, with a 2% energy split between a = 0 and a = 1 states
- For two scalar particles with the same parameters, the energy split was 20%
- For two fermions with pseudoscalar exchange at α = 60, the binding energies are B_{a=0} = 0.103 and B_{a=1} = 0.0494, giving more than 100% splitting
- For J = 0⁺ with scalar exchange, the solution is stable at k_max → ∞ for coupling constant α < α_c ≈ 3.72
- The coefficients for the J = 1⁺ superposition are c₀ = 0.5817 and c₁ = 0.8134, close to 1/√3 ≈ 0.5773 and √(2/3) ≈ 0.8165 respectively

## Context

This work extends the explicitly covariant light-front dynamics approach previously applied to two scalar particles (Karmanov, Carbonell, and Mangin-Brinet, 2000) and perturbative calculations of the light-front deuteron wave function (Karmanov, Carbonell, and Jaus, 1999) to the non-perturbative treatment of two-fermion bound states. The method enables calculation of bound state masses and wave functions for scalar, pseudoscalar, vector, and pseudovector one-boson-exchange kernels, and the approximate restoration of rotational invariance through the angular condition provides a way to compare with Bethe-Salpeter solutions, as was done for the scalar case.
