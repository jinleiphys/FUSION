# Pauli-Potential and Green Function Monte-Carlo Method for Many-Fermion Systems

> The time evolution of a many-fermion system can be described by a Green's function corresponding to an effective potential, which takes anti-symmetrization of the wave function into account, called the Pauli-potential. We show that this idea can be combined with the Green's Function Monte Carlo method to accurately simulate a system of many non-relativistic fermions. The method is illustrated by the example of systems of several (2-9) fermions in a square well.

## Key claim

The paper shows that the Pauli-potential idea of Tursunov and Zhirov (1989), originally developed for Langevin-equation simulations, can be combined with the Green's Function Monte Carlo method to simulate many-fermion systems, but only with a systematic error linear in the time-step parameter Delta that must be extrapolated to zero to recover exact results.

## Method

The authors implement a modified Green's Function Monte Carlo algorithm that uses a trial harmonic oscillator potential and a guidance function constructed from a Slater determinant of harmonic-oscillator wave functions. The Pauli-potential, which approximates anti-symmetrization to leading order in the imaginary time step beta, is applied through a complex, nonlocal effective potential that acts on pairs of fermions. The algorithm samples imaginary time beta from an exponential distribution with parameter Delta, and the ground-state energy is extracted from the population size evolution using the relation E_0 = E_T + (1/Delta)(P_{n-1}/P_n - 1).

## Key numbers

* For nine spinless fermions in a square well of depth V_0 = -3.5 and radius R = 2, with Delta = 0.0005, the computed energy converges to -12.8, compared to the exact value of -11.501.
* The number of particles in the square well was varied between two and nine.
* The maximum multiplicity cutoff M_max was typically set between 5 and 20.
* The population size for the nine-fermion calculation was approximately one thousand points.

## Context

This work extends the Pauli-potential approach of Tursunov and Zhirov (1989) from the Langevin-equation framework to the more efficient Green's Function Monte Carlo method developed by Kalos and Ceperley. The modified algorithm builds on the work of KePoSh90 to handle non-positive-definite kernels. The method enables simulation of up to nine spinless fermions in a simple potential, with the key open question being whether the linear extrapolation in Delta to zero time step can be applied reliably to more realistic nuclear systems with more complex interactions.
