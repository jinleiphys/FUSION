# Importance of realistic phase space representations of initial quantum fluctuations using the stochastic mean-field approach for fermions

> In the stochastic mean-field (SMF) approach, an ensemble of initial values for a selected set of one-body observables is formed by stochastic sampling from a phase-space distribution that reproduces the initial quantum fluctuations. Independent mean-field evolutions are performed with each set of initial values followed by averaging over the resulting ensemble. This approach has been recently shown to be rather versatile and accurate in describing the correlated dynamics beyond the independent particle picture. In the original formulation of SMF, it was proposed to use a Gaussian assumption for the phase-space distribution. This assumption turns out to be rather effective when the dynamics of an initially uncorrelated state is considered, which was the case in all applications of this approach up to now. Using the Lipkin-Meshkov-Glick (LMG) model, we show that such an assumption might not be adequate if the quantum system under interest is initially correlated and presents configuration mixing between several Slater determinants. In this case, a more realistic description of the initial phase-space is necessary. We show that the SMF approach can be advantageously combined with standard methods to describe phase-space in quantum mechanics. As an illustration, the Husimi distribution function is used here to obtain a realistic representation of the phase-space of a quantum many-body system. This method greatly improves the description of initially correlated fermionic many-body states. In the LMG model, while the Gaussian approximation failed to describe these systems in all interaction strength range, the novel approach gives a perfect agreement with the exact evolution in the weak coupling regime and significantly improves the description of correlated systems in the strong coupling regime.

## Key claim

The stochastic mean-field (SMF) approach, which previously used a Gaussian assumption for the initial phase-space distribution, fails to describe the dynamics of initially correlated fermionic states that involve configuration mixing between several Slater determinants; combining SMF with the Husimi distribution function provides a realistic phase-space representation that yields perfect agreement with exact evolution in the weak coupling regime and significantly improves description in the strong coupling regime.

## Method

The authors test the SMF approach on the Lipkin-Meshkov-Glick (LMG) model with N=40 particles, comparing three predefined distribution functions (Gaussian, uniform, bimodal) against a distribution obtained from the Husimi quasiprobability function. For the Husimi-based method, stochastic angles (θ, φ) are sampled from the Husimi function using rejection sampling, then transformed into effective quasispin values {j_x, j_y, j_z} via Weyl symbols with a rescaling procedure that ensures the first and second quantum moments are exactly reproduced. Each set of initial quasispin values is evolved using the mean-field equations, and results are averaged over 10^6 events.

## Key numbers

- Number of particles: N=40, giving total quasispin j=20.
- Relative interaction strength parameter: χ = V(N-1)/ε, with weak coupling χ=0.5 and strong coupling χ=5.
- For the initial coherent state |θ₀=π/3, φ₀=π/4⟩, quantum kurtoses are β_x^(Q)=β_y^(Q)=3.01 and β_z^(Q)=2.98, close to the Gaussian value of 3.
- For the correlated state |j 0⟩, quantum kurtoses are β_x^(Q)=β_y^(Q)=3/2 - 4/[N(N+2)] ≈ 1.4976, far from the Gaussian limit.
- Number of stochastic events: 10^6.

## Context

This work extends the stochastic mean-field approach originally introduced by Ayik (2008) and previously validated for initially uncorrelated Slater determinant states in Lacroix (2012, 2013, 2014). The paper shows that the Gaussian assumption used in those earlier studies is inadequate for initially correlated states, and demonstrates that combining SMF with the Husimi distribution function, a technique used in bosonic systems such as the Bose-Hubbard model by Trimborn and Polkovnikov, can be adapted to fermionic systems. This enables accurate treatment of correlated initial states and mixed densities, correcting the overdamping of fluctuations that was previously misinterpreted as a generic defect of SMF.
