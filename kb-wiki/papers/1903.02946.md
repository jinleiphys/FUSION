# On formal scattering theory for differential Faddeev equations

> The formal scattering theory is developed for the three-particle differential Faddeev equations. The theory is realised along the same line as in the standard two-body case. The solution of the scattering problem is expressed in terms of the matrix T-operator constructed from the matrix resolvent of the differential Faddeev equations. The relationships of the matrix T-operator with elements of transition operators and Faddeev T-matrix components have been established.

## Key claim

The paper develops a formal scattering theory for the three-particle differential Faddeev equations (DFE) by constructing a matrix T-operator from the matrix resolvent of the DFE, establishing that this operator completely defines the scattering state and that its matrix elements yield elastic, rearrangement, and breakup amplitudes directly, analogous to the standard two-body case.

## Method

The author introduces a 3x3 matrix T-operator defined as T(z) = V̅ - V̅ G(z) V̅, where V̅ is the off-diagonal potential matrix and G(z) is the resolvent of the matrix operator generated by the differential Faddeev equations. Using the second resolvent identities for the matrix resolvent, the scattering state vector is expressed in terms of this T-operator, and explicit relationships are derived connecting the T-operator components to the Lovelace transition operators, the AGS transition operators, and the Faddeev T-matrix components. The asymptotic forms of the wave function components are then evaluated to extract scattering amplitudes.

## Key numbers

- The scattering amplitude for elastic and rearrangement processes is given by A_{αβ} = -2π² ⟨Φ̂_α(p̃_α)|T_{αβ}(E_β+i0)|Φ̂_β(p_β)⟩, where p̃_α = p_α y_α/y_α and p²_α = E_β - ε_α.
- The breakup amplitude is A^{(0)}_β = -2π² E^{3/4}_β Σ_α ⟨Φ̂⁻_α(k̄_α, p̄_α)|T_{αβ}(E_β+i0)|Φ̂_β(p_β)⟩, where E_β = ε_β + p²_β and the prefactor involves E^{3/4}_β.
- The asymptotic form of the breakup component U^{(0)}_{αβ} decays as exp{i√E_β X - iπ3/4} / X^{5/2} in the 6D coordinate space, where X = √(x²_α + y²_α).

## Context

This work extends the author's previous formal scattering theory for differential Faddeev equations (Yakovlev 1996), which treated the DFE as a spectral problem for a non-symmetric matrix operator and established biorthogonality and completeness of eigenfunctions. The present paper provides an alternative formulation where the matrix T-operator is constructed from transition operators rather than from the resolvent directly, establishing explicit connections to the Lovelace (1964) and AGS (1967) formulations of three-body scattering. This enables practical calculations where the well-established AGS solution method can be used to compute the U_{αβ} operators first, then the wave function components via the derived relationships.
