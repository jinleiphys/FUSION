# NCrystal : a library for thermal neutron transport

> An open source software package for modelling thermal neutron transport is presented. The code facilitates Monte Carlo-based transport simulations and focuses in the initial release on interactions in both mosaic single crystals as well as polycrystalline materials and powders. Both coherent elastic (Bragg diffraction) and incoherent or inelastic (phonon) scattering are modelled, using basic parameters of the crystal unit cell as input. Included is a data library of validated crystal definitions, standalone tools and interfaces for C++, C and Python programming languages. Interfaces for two popular simulation packages, Geant4 and McStas, are provided, enabling highly realistic simulations of typical components at neutron scattering instruments, including beam filters, monochromators, analysers, samples and detectors. All interfaces are presented in detail, along with the end-user configuration procedure which is deliberately kept user-friendly and consistent across all interfaces. An overview of the relevant neutron scattering theory is provided, and the physics modelling capabilities of the software are discussed. Particular attention is given here to the ability to load crystal structures and form factors from various sources of input, and the results are benchmarked and validated against experimental data and existing crystallographic software. Good agreements are observed.

## Key claim

NCrystal provides an open-source software library for modeling thermal neutron transport in crystalline materials, enabling Monte Carlo simulations that incorporate both coherent elastic (Bragg diffraction) and incoherent or inelastic (phonon) scattering using basic crystal unit cell parameters as input, though the paper notes that the current release does not support surface effects, non-crystalline or textured materials, or effects requiring breakdown of the Born approximation such as strong reflections in neutron guide systems.

## Method

The authors implemented a C++ library with no third-party dependencies for the core functionality, providing physics models for thermal neutron scattering in mosaic single crystals, polycrystalline materials, and powders. The code calculates scattering cross sections using the Fermi pseudopotential and the master equation for thermal neutron scattering, separating coherent and incoherent contributions, and supports loading crystal structures and form factors from various input sources. Interfaces for C++, C, Python, Geant4, and McStas are provided, along with a data library of validated crystal definitions and a unified configuration system for end-users.

## Key numbers

- The separation between scattering and absorption processes is valid at the order of 10^-4 level for thermal neutrons
- The code is released with version number 1.0.0 under the Apache License, Version 2.0
- Mosaic crystal mosaicity values typically range from a few arc seconds to a few degrees
- The absorption cross section model uses 1/v scaling, which follows from the probability of absorption being proportional to the time spent by the neutron near the nucleus

## Context

This work addresses the gap in existing Monte Carlo transport applications such as MCNP, Geant4, and FLUKA, which typically treat all materials as a free gas of unbound atoms and neglect material structure effects like Bragg diffraction, while specialized applications like McStas and VITESS lack capabilities for complex geometries and non-neutron physics. The NCrystal library enables realistic simulations of neutron scattering instrument components including beam filters, monochromators, analyzers, samples, and detectors, and was originally developed to facilitate design and optimization of neutron detectors for the European Spallation Source. The approach differs from dedicated utilities like CRIO, NJOY, and NXS which calculate various properties of thermal neutrons but do not deliver the same combination of physics models, flexible design, multiple interfaces, and user-friendly configuration.
