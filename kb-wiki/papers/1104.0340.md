# r-Java : An r-process Code and Graphical User Interface for Heavy-Element Nucleosynthesis

> We present r-Java, an r-process code for open use, that performs r-process nucleosynthesis calculations. Equipped with a simple graphical user interface, r-Java is capable of carrying out nuclear statistical equilibrium (NSE) as well as static and dynamic r-process calculations for a wide range of input parameters. In this introductory paper, we present the motivation and details behind r-Java, and results from our static and dynamic simulations. Static simulations are explored for a range of neutron irradiation and temperatures. Dynamic simulations are studied with a parameterized expansion formula. Our code generates the resulting abundance pattern based on a general entropy expression that can be applied to degenerate as well as non-degenerate matter, allowing us to track the rapid density and temperature evolution of the ejecta during the initial stages of ejecta expansion. At present, our calculations are limited to the waiting-point approximation. We encourage the nuclear astrophysics community to provide feedback on the code and related documentation, which is available for download from the website of the Quark-Nova Project: http://quarknova.ucalgary.ca/

# r-Java: An r-process Code and Graphical User Interface for Heavy-Element Nucleosynthesis

## Key claim

The authors present r-Java, an open-use r-process nucleosynthesis code with a graphical user interface, but explicitly state that all calculations are currently limited to the waiting-point approximation, with the code restrained to neutron densities below 10^32 cm^-3 because fission has not yet been included in the static simulations.

## Method

The code implements nuclear statistical equilibrium (NSE) via the Newton-Raphson method for chemical potentials, and performs static and dynamic r-process calculations using a reduced reaction network of 136 isotopic chains under the waiting-point approximation. Static simulations assume a non-expanding neutron-rich chunk near neutron drip density (approximately 4x10^11 g/cc) bombarded by constant neutron flux, while dynamic simulations use a parameterized expansion formula with user-specified density profiles and solve the coupled differential equations using a Crank-Nicholson method with implicit Backward Euler scheme for the stiff system.

## Key numbers

- Static simulations require neutron density of 10^28 cm^-3 and temperature near 2x10^9 K to produce a strong third r-process peak at A=195
- The waiting-point approximation is valid only for temperatures above 2x10^9 K and neutron densities above 10^20 cm^-3
- Dynamic simulations used initial temperature of 4x10^9 K, initial density of 1x10^11 g/cc, and electron fraction Ye=0.03 with pure iron (Z0=26) as starting composition
- For non-relativistic expansion with initial density rho0~10^13 g/cc, the initial temperature must satisfy T0 less than or approximately equal to 10^11 K
- Dynamic simulations were run for one second duration with expansion time-scales of tau=0.001 s and tau=0.1 s
- The reaction network spans nuclei with mass numbers between 50 and 300 for the beta-decay module

## Context

This work extends the tradition of r-process nucleosynthesis codes following the foundational work of Burbidge (1957) and Cameron (1957), and builds upon NSE benchmarks from Cliff and Tayler (1965) and other NSE codes. The code is designed to address the need for an openly shared, cross-platform program that can incorporate new nuclear data from upcoming rare-isotope experiments such as FRIB, and allows users to explore different dynamical environments including neutron star mergers (Freiburghaus et al., 1999) and neutrino-driven winds (Qian and Woosley, 1996). The authors note that producing the third r-process peak requires extreme values of entropy and dynamic time-scale that challenge current Type II supernova models (Qian and Wasserburg, 2008), and that neutron star mergers provide larger neutron-to-seed ratios but occur too infrequently to explain early r-process enrichment in metal-poor stars (Qian, 2000).
