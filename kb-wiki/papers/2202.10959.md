# Pion model with the Nakanishi Integral Representations

> In the present work, we describe a model for the pion based on an analytic expression for the Bethe-Salpeter (BSA) amplitude, combined with some ingredients from Lattice QCD calculations. The running quark mass function $M (p^2)$, used here, reproduces well the results of Lattice QCD calculations. The analytical form of the running quark mass function contains a single time-like pole, which implies in time-like poles of the dressed quark propagator. Such a form allows to build the weight functions, $G_i(γ,z)$, for the Nakanishi integral representation of each scalar function, $χ_{i}(k,p)$, appearing in the decomposition of the Bethe-Salpeter amplitude in terms of Dirac operators, Such scalar amplitudes can also be used to obtain the pion valence light-front wave function.

## Key claim

The paper constructs a model for the pion Bethe-Salpeter amplitude (BSA) that uses an analytic running quark mass function with a single time-like pole, which allows the explicit construction of Nakanishi weight functions for each scalar component of the BSA, but the model is limited by the use of a simplified quark wave function renormalization factor set to unity and by the presence of time-like poles in the dressed quark propagator.

## Method

The authors model the dressed quark propagator using a running mass function \(M(k^2)=m_0-m^3/(k^2-\lambda^2+i\epsilon)\) fitted to Euclidean Lattice QCD results for space-like momenta, with the quark wave function renormalization set to \(Z(k^2)=1\). They then write the pion Bethe-Salpeter amplitude using a vertex function dominated by the dressed quark mass in the chiral limit, and decompose the amplitude into four scalar functions \(\chi_i(k,p)\). Using the spectral representation of the propagator and an integral identity, they express each scalar function in the Nakanishi integral representation and obtain explicit analytic expressions for the weight functions \(G_i(\gamma,z)\) as sums over the three time-like poles of the propagator.

## Key numbers

* Running quark mass function parameters: \(m_0 = 0.014\ \text{GeV}\), \(m = 0.574\ \text{GeV}\), \(\lambda = 0.846\ \text{GeV}\) (the "IP" set).
* Dressed quark propagator pole masses: \(m_1 = 0.371\ \text{GeV}\), \(m_2 = 0.644\ \text{GeV}\), \(m_3 = 0.954\ \text{GeV}\).
* Residues for the spectral decomposition: \(D_1 = 1.4992\), \(D_2 = -0.5941\), \(D_3 = -0.09498\) (dimensionless); \(E_1 = 0.4240\ \text{GeV}\), \(E_2 = -0.3314\ \text{GeV}\), \(E_3 = -0.07864\ \text{GeV}\).
* Nakanishi weight functions \(G_i(\gamma,z)\) are computed for \(\gamma\) values of \(0.45\ \text{GeV}^2\) and \(0.75\ \text{GeV}^2\), with an arbitrary normalization \(\mathcal{N}=100\).

## Context

This work extends the approach of Clayton (2017) by using the same analytic quark propagator model to compute the Nakanishi weight functions for the pion Bethe-Salpeter amplitude, rather than only the amplitude itself. It builds on Lattice QCD results for the dressed quark mass function from Parappilly (2005) and the parametrization of Rojas (2013). The computed weight functions can be used in future work to obtain the pion valence light-front wave function and momentum distributions, as well as to study the scalar functions in the orthogonal Dirac basis.
