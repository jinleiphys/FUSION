# Novel analysis of the decays psi' -> h_c pi^0 and eta_c'-> chi_{c0} pi^0

> We show that in the transitions psi' -> h_c π^0 and eta_c' -> chi_{c0} pi^0 the contributions from charmed meson loops are highly suppressed, in contrast to various other charmonium decays. We calculate the width of the psi' -> h_c pi^0, which agrees with the recent BES-III data, and predict the width of the eta_c' -> chi_{c0} pi^0, Γ(eta_c' -> chi_{c0} pi^0) = 1.5\pm0.4 keV. A confirmation of this prediction would also provide additional support for a recent analysis of psi' -> J/psi pi^0 (eta), where loops are claimed to play a prominent role.

## Key claim

Charmed meson loop contributions are highly suppressed in the decays psi' -> h_c pi^0 and eta_c' -> chi_{c0} pi^0, in contrast to other charmonium decays such as psi' -> J/psi pi^0(eta) where loops are enhanced; this suppression allows a tree-level prediction for the eta_c' -> chi_{c0} pi^0 width that is testable at PANDA.

## Method

The authors constructed a leading-order chiral Lagrangian for the tree-level transitions between S-wave and P-wave charmonia, incorporating isospin breaking via the quark mass difference (m_d - m_u) and pi^0-eta mixing. They then applied a non-relativistic effective field theory (NREFT) to compute charmed meson loop contributions, using power counting arguments to compare loop and tree-level amplitudes, and verified the results with an effective Lagrangian approach.

## Key numbers

- Gamma(psi' -> h_c pi^0) = 0.26 +/- 0.05 keV, extracted from BES-III data on B(psi' -> pi^0 h_c) x B(h_c -> gamma eta_c) = (4.58 +/- 0.40 +/- 0.50) x 10^{-4} and B(psi' -> pi^0 h_c) = (8.4 +/- 1.3 +/- 1.0) x 10^{-4}
- Predicted Gamma(eta_c' -> chi_{c0} pi^0) = 1.5 +/- 0.4 keV (experimental and theoretical uncertainties summed in quadrature)
- Predicted branching fraction B(eta_c' -> chi_{c0} pi^0) = (1.1 +/- 0.6) x 10^{-4}, using total width Gamma(eta_c') = 14 +/- 7 MeV
- Ratio of widths Gamma(eta_c' -> chi_{c0} pi^0) / Gamma(psi' -> h_c pi^0) = 5.86 +/- 0.94
- Loop-only width for psi' -> h_c pi^0: 2.1 x 10^{-7} g_1^2 g_2'^2 keV, with g_1 in GeV^{-1/2} and g_2' in GeV^{-3/2}
- Loop-only width for eta_c' -> chi_{c0} pi^0: 1.0 x 10^{-5} g_1^2 g_2'^2 keV
- Pion momentum in psi' rest frame: q_1 = 86 MeV; in eta_c' rest frame: q_2 = 171 MeV
- Quark mass ratio r = m_u/m_d = 0.47 +/- 0.08, giving B_0(m_d - m_u) = (6.6 +/- 2.0) x 10^{-3} GeV^2

## Context

This work extends the NREFT framework developed by Guo (2009) for charmonium decays, which showed that charmed meson loops are enhanced by 1/v in psi' -> J/psi pi^0(eta) (with v ~ 0.5). The present analysis demonstrates that for S-wave to P-wave transitions, loops are suppressed by a factor q^2/(M_D^2 v^3) ~ 1/30 for psi' -> h_c pi^0 and ~ 1/10 for eta_c' -> chi_{c0} pi^0. The tree-level prediction for eta_c' -> chi_{c0} pi^0 provides a test of the NREFT approach; confirmation would support the loop-dominance analysis of psi' -> J/psi pi^0(eta). The result also resolves the discrepancy between earlier estimates by Segre (1976) using pi^0-eta mixing (5-30 keV) and the BES-III measurement (0.26 keV). The prediction is testable at the PANDA experiment at FAIR.
