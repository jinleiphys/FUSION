# Towards a Stringy Description for the $Q\bar Q q\bar q$-Quark System

> For the case of two light flavors we propose the stringy description of the system made of one heavy and one light quark-antiquark pair, with the aim of exploring the two lower-lying Born-Oppenheimer potentials as a function of a separation of the heavy quark-antiquark pair. Our analysis reveals three critical separations related to the processes of string reconnection, breaking and junction annihilation. In particular, for the ground state potential only the process of string reconnection matters. We find that a tetraquark state makes the dominant contribution to the potential of the first excited state at small separations, and this is the big difference with the $QQ\bar q\bar q$-quark system where it does so to the ground state potential. Another big difference is the emergence of the full diquark picture $[Qq][\bar Q\bar q]$ rather than the partial picture $QQ[\bar q\bar q]$ for the tetraquark state. On the other hand, the scales of string junction annihilation, below which the systems can be thought of mainly as the compact tetraquarks, are very close for both cases and become almost the same if the phenomenological rule $E_{QQ}=\frac{1}{2} E_{Q\bar Q}$ holds. The same is also true for the screening lengths whose values are in agreement with lattice QCD.

## Key claim

For the $Q\bar Q q\bar q$-quark system with two light flavors, the stringy description reveals that a tetraquark state dominates the first excited Born-Oppenheimer potential at small separations (not the ground state as in the $QQ\bar q\bar q$ system), and the full diquark picture $[Qq][\bar Q\bar q]$ emerges rather than the partial picture $QQ[\bar q\bar q]$, with three critical separations related to string reconnection, breaking, and junction annihilation.

## Method

The author constructs static string configurations in a five-dimensional deformed AdS space (soft wall model) with metric $ds^2 = e^{s r^2} (R^2/r^2)(dt^2 + d\vec x^2 + dr^2)$, using Nambu-Goto strings, baryon vertices with action $S_v = \tau_v \int dt \, e^{-2s r^2}/r$, and light quarks described by a constant tachyon field with action $S_q = T_0 R \int dt \, e^{\frac{1}{2} s r^2}/r$. The two lower-lying Born-Oppenheimer potentials are obtained by minimizing over five string configurations: two disconnected meson pairs, a connected tetraquark configuration (with two branches for small and larger heavy-quark separations), and two configurations with an additional light quark-antiquark pair.

## Key numbers

- Critical separation for string reconnection (ground state): $l_{Qq} = 0.219$ fm, defined by $E_{Q\bar Q}(l_{Qq}) + E_{q\bar q} = 2E_{Q\bar q}$
- Critical separation for diquark picture onset: $\ell(v_*) = 0.106$ fm, where $v_* = 1/12$ and $\kappa = -\frac{1}{4} e^{1/4}$
- String tension: $\sigma = 0.45$ GeV$^2$ (from Regge slope), with $\g = 0.176$ and $\n = 3.057$ (fitted to lattice string breaking distance $1.22$ fm for $u,d$ quarks)
- Ratio of Coulomb coefficients at small $\ell$: $\alpha_{Q\bar Q q\bar q} / \alpha_{Q\bar Q} = 0.939$
- Ratio of effective string tensions at small $\ell$: $\boldsymbol{\sigma}_{Q\bar Q q\bar q} / \sigma = 1.406$
- Light quark position in bulk: $r_q = 1/\sqrt{s}$ (on the soft wall), giving pion rest energy $E_{q\bar q} = 2\n\sqrt{\g\sigma}$
- Solution for light quark position: $q = s r_q^2 = 0.566$ from force balance equation

## Context

This work extends the author's previous stringy descriptions of doubly heavy quark systems (Andreev 2021, 2022) to the $Q\bar Q q\bar q$ system, complementing lattice QCD studies of four-quark systems (Alexandrou and Pochinsky 2000; Prelovsek et al. 2021) which have limited data. The analysis introduces three critical separations characterizing string reconnection, breaking, and junction annihilation, with the screening lengths agreeing with lattice QCD results. The framework enables calculation of Born-Oppenheimer potentials for exotic hadrons such as the $Z_b$ states observed by Belle (2011), and can be extended to $N_f = 2+1$ flavors.
