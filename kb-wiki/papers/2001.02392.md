# Quantum delocalization of strings with boundary action in Yang-Mills theory

> The width of the quantum delocalization of the QCD strings is investigated in effective string models beyond free Nambu-Goto approximation. We consider two Lorentzian-invariant boundary-terms in the Lüscher-Weisz string action in addition to self-interaction term equivalent to two loop order in the (NG) string action. The geometrical terms which realize the possible rigidity of the QCD string is scrutinized as well. We perform the numerical analysis on the 4-dim pure $SU(3)$ Yang-Mills lattice gauge theory at two temperature scales near deconfinement point. The comparative study with this QCD string model targets the width of the energy profile of a static quark-antiquark system for color sources separation $0.5 \le R \le 1.2$ fm. We find the inclusion of rigidity properties and symmetry effects of the boundary action into the string paradigm to reproduce a good match with the profile of the Mont-Carlo data of QCD flux-tube on this distance scale.

## Key claim

The inclusion of rigidity properties (via the Polyakov-Kleinert extrinsic curvature term) and Lorentz-invariant boundary action terms (at orders four and six derivatives) into the effective string model reproduces a good match with Monte Carlo data for the QCD flux-tube width profile at color source separations \(0.5 \le R \le 1.2\) fm, where the free Nambu-Goto approximation shows deviations near the deconfinement point.

## Method

The authors performed lattice Monte Carlo simulations of 4-dimensional pure \(SU(3)\) Yang-Mills theory at two temperature scales near the deconfinement point. They computed the mean-square width of the energy profile of a static quark-antiquark system and compared it to perturbative expansions of the Lüscher-Weisz effective string action that include two Lorentz-invariant boundary terms (couplings \(b_2\) and \(b_4\)) and a self-interaction term equivalent to two-loop order in the Nambu-Goto action, as well as the leading and next-to-leading order contributions from the Polyakov-Kleinert rigidity term.

## Key numbers

* The color source separation range studied is \(0.5 \le R \le 1.2\) fm.
* The leading-order free-string width formula (Eq. \ref{WidthLO}) contains the factor \(\frac{D-2}{2\pi\sigma_{0}}\log\left(\frac{R}{R_{0}(\zeta)}\right)\), where \(D=4\) and \(\sigma_{0}\) is the string tension.
* The leading-order rigidity correction to the width (Eq. \ref{WExt}) is \(W_{\rm{R_{(\ell o)}}}^2 = \frac{-\pi(D-2)\alpha_{r}}{24 R^2 \sigma^2 } E_2 \left( \frac{ L_{T}}{2 R}\right)\), where \(\alpha_{r}\) is the rigidity parameter and \(E_2\) is the Eisenstein series.
* The next-to-leading-order rigidity correction (Eq. \ref{Ext2loopT}) contains terms proportional to \(\frac{ \pi^3 \alpha_r (3(D-2)^2-(D-2))}{8 R^5 \sigma^3}\) and \(\frac{\pi^2 \alpha_{r}(D-2)}{8 R^4 \sigma^3}\).

## Context

This work extends the effective string description of QCD flux tubes beyond the free Nambu-Goto approximation, which had shown substantial deviations from lattice data at high temperatures close to the deconfinement point (Bakry 2010, Bakry 2012). It builds on the Lüscher-Weisz action framework (Lüscher 2004) and the Polyakov-Kleinert rigid string model (Polyakov 1986, Kleinert 1986), and follows recent calculations of boundary action contributions to the width (Bakry 2019). The results enable more precise modeling of the confining flux-tube structure at intermediate distances and high temperatures, and may inform studies of baryonic flux tubes and the string spectrum.
