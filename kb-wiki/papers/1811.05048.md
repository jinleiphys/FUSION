# Testing a best-fit hydrodynamical model using PCA

> Recently, a comprehensive Bayesian analysis was performed to simultaneously extract the values of a number of hydrodynamic parameters necessary for compatibility with a limited set of experimental data from the LHC. In this work, this best-fit model is tested against newly measured experimental flow results not included in the original work, namely the principal components of the two-particle correlation matrix in transverse momentum. The results from simulations show a good numerical agreement with data obtained by the CMS Collaboration.

## Key claim

The best-fit hydrodynamical model obtained from a previous comprehensive Bayesian analysis, which was tuned to a limited set of LHC data, shows good numerical agreement with newly measured principal components of the two-particle correlation matrix in transverse momentum from the CMS Collaboration, but the test is limited to observables not used in the original fit.

## Method

The authors performed hydrodynamic simulations using the previously determined best-fit parameters from a Bayesian analysis and computed the principal components (PCs) of the two-particle correlation matrix in transverse momentum. They then compared these simulated principal components to the corresponding experimental data published by the CMS Collaboration, evaluating the agreement between the model predictions and the new flow observables.

## Context

This work extends the validation of a hydrodynamical model that was previously calibrated through a comprehensive Bayesian analysis (the authors refer to this as "a comprehensive Bayesian analysis was performed to simultaneously extract the values of a number of hydrodynamic parameters"). The model had been fitted to a limited set of LHC data, and this paper tests its predictive power against independent flow observables from the CMS Collaboration that were not used in the original calibration. The results enable further refinement of hydrodynamic modeling for heavy-ion collisions and demonstrate the utility of principal component analysis as a tool for testing model predictions against multi-dimensional correlation data.
