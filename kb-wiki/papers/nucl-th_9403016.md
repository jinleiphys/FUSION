# Shell model on a random gaussian basis

> Pauli-projected random gaussians are used as a representation to solve the shell model equations. The elements of the representation are chosen by a variational procedure. This scheme is particularly suited to describe cluster formation and cluster decay in nuclei. It overcomes the basis-size problem of the ordinary shell model and the technical difficulties of the cluster-configuration shell model. The model reproduces the $α$-decay width of $^{212}$Po satisfactorily.

# Shell model on a random gaussian basis

## Key claim
The paper claims that a shell model using Pauli-projected random gaussian basis functions, selected by a stochastic variational procedure, can reproduce the α-decay width of ²¹²Po satisfactorily. However, the calculated width (7.2×10⁻¹⁶ MeV) is about half the experimental value (1.5×10⁻¹⁵ MeV) and about one third of the hybrid-model value, indicating the model underperforms relative to the cluster-configuration shell model it aims to simplify.

## Method
The authors replace conventional harmonic oscillator valence orbits with combinations of gaussian functions of variable size parameters, with exact Pauli projection off the ²⁰⁸Pb core orbits. Basis states are generated randomly and selected via a variational admittance test (utility testing) based on their contribution to the ground-state energy, using a modified Jacobi method for efficient eigenvalue solution. The four-particle basis for ²¹²Po is constructed in proton-neutron formalism with jj coupling, coupling two-proton and two-neutron configurations up to Jπ=8 and Jν=8.

## Key numbers
- Ground-state energy of ²¹²Po: E = -19.25 MeV (experimental: -19.35 MeV), converged at basis dimension N=397 with 118 configurations
- Pure shell model ground-state energy: -18.96 MeV; hybrid model: -19.18 MeV
- Three independent calculations with different random parameter sets (N=413, n=120; N=380, n=127) agreed within 0.04 MeV
- With reversed configuration order: -19.24 MeV, -19.28 MeV, -19.25 MeV (basis sizes nearly doubled)
- With ε=0.005 MeV: lowest energy -19.27 MeV (basis N=631, n=193)
- Spectroscopic factor: 0.011 (hybrid model: 0.023)
- α-decay width: Γ = 7.2×10⁻¹⁶ MeV (hybrid model: 2.1×10⁻¹⁵ MeV; experimental: 1.5×10⁻¹⁵ MeV)
- Main peak of spectroscopic amplitude at 8.0 fm (hybrid model: 8.2 fm)
- Size parameter interval for valence orbits: 4 fm ≤ ν⁻¹/² ≤ 14 fm
- Core oscillator size parameter: ν₀ = 0.083 fm⁻²
- Scalar strengths: Vp = 61.60 MeV (protons), Vn = 43.80 MeV (neutrons)

## Context
This work extends the cluster-configuration shell model (hybrid model) of Varga (1993) by replacing explicit cluster-model basis states with random gaussian functions, aiming to overcome the basis-size problem of the ordinary shell model and the technical difficulties of the cluster-configuration approach. The method is based on the stochastic variational method previously developed for multicluster systems (Varga, Suzuki, Lovas, 1993). The model enables simpler application to other α-decays and heavy-cluster decays, though it produces a spectroscopic factor (0.011) smaller than the hybrid model (0.023) and a decay width about half the experimental value, comparable to results from sophisticated truncation schemes (Arima, 1972; Gordana, 1990).
