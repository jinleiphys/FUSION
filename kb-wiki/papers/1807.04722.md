# Proof of Color Octet NRQCD Factorization of P-Wave Heavy Quarkonium Production at All Orders in Coupling Constant

> Recently we have proved color octet NRQCD factorization of S-wave heavy quarkonium production at all orders in coupling constant at high energy colliders in Eur. Phys. J. C76 (2016) 448. In this paper we extend this to prove color octet NRQCD factorization of P-wave heavy quarkonium production at all orders in coupling constant at high energy colliders. We find that while the color octet NRQCD S-wave non-perturbative matrix element contains two gauge-links in the adjoint representation of SU(3), the color octet NRQCD P-wave non-perturbative matrix element contains four gauge-links in the fundamental representation of SU(3).

## Key claim

The paper proves that color-octet NRQCD factorization holds for P-wave heavy quarkonium production at all orders in the coupling constant at high-energy colliders, extending a previous proof for S-wave production. The key structural difference is that the gauge-invariant color-octet NRQCD non-perturbative matrix element for P-wave contains four gauge-links in the fundamental representation of SU(3), whereas the S-wave matrix element contains two gauge-links in the adjoint representation.

## Method

The proof uses the path integral formulation of QCD combined with the background field method. The author shows that infrared divergences arising from light-like eikonal lines can be described by pure gauge fields, and then demonstrates that the non-perturbative correlation function with inserted gauge links equals the original correlation function without gauge links, establishing factorization at all orders. The gauge-invariant definition of the color-octet NRQCD P-wave non-perturbative matrix element is derived and shown to be independent of the light-like vector used to define the gauge links.

## Key numbers

- The gauge-invariant color-octet NRQCD P-wave non-perturbative matrix element contains **four** gauge-links in the fundamental representation of SU(3), compared to **two** gauge-links in the adjoint representation for the S-wave case.
- The light-like gauge link is defined as $\Phi(x)={\cal P} e^{-ig T^d \int_0^\infty dz l\cdot A^d(x+lz)}$ with $l^2=0$.
- The operator ${\overline { \nabla}}$ is defined by $\eta {\overline { \nabla}} \zeta =\eta ({ {\vec \nabla}} \zeta) -( { {\vec \nabla}}\eta) \zeta$.

## Context

This work extends the author's previous proof of color-octet NRQCD factorization for S-wave heavy quarkonium production at all orders (Nayak, 2016) to the P-wave case. It also builds on earlier NNLO diagrammatic calculations (Nayak, 2015) that first identified the need for gauge links in the non-perturbative matrix elements. The result provides a rigorous foundation for using NRQCD factorization in P-wave heavy quarkonium production at high-energy colliders, which is relevant for understanding quark-gluon plasma probes at RHIC and LHC heavy-ion colliders.
