# Low-energy behavior of $E2$ strength functions

> Electric quadrupole strength functions have been deduced from averages of a large number of $E2$ transition strengths calculated within the shell model for the nuclides $^{94}$Mo and $^{95}$Mo. These strength functions are at variance with phenomenological approximations as provided by the Reference Input Parameter Library RIPL-3 for calculations of reaction rates on the basis of the statistical model.

## Key claim

Shell-model calculations of E2 transition strengths in 94Mo and 95Mo show that the low-energy E2 strength function has a finite, bell-shaped (Gaussian-like) maximum at zero transition energy, in contrast to the unphysical pole at Eγ = 0 produced by the Lorentz-curve parametrization recommended in the RIPL-3 database.

## Method

The author performed large-scale shell-model calculations using the code RITSSCHIL with a model space of π(0f5/2, 1p3/2, 1p1/2, 0g9/2) proton orbits and ν(1p1/2, 0g9/2, 1d5/2) neutron orbits relative to a 66Ni core. For each spin (J = 0–10 in 94Mo, J = 1/2–21/2 in 95Mo), the lowest 40 states were calculated, yielding more than 36,000 E2 transitions per parity. Average B(E2) values were obtained by sorting transitions into 100 keV bins of excitation energy or transition energy, and E2 strength functions were derived using the relation f2(Eγ) = 0.80632 × 10−12 × average B(E2) × level density.

## Key numbers

- Calculated B(E2, 2+1 → 0+1) = 107 e2fm4 in 94Mo, compared to experimental value 406(9) e2fm4.
- Calculated B(E2, 4+1 → 2+1) = 49 e2fm4 in 94Mo, compared to experimental value 660(102) e2fm4 from Coulomb excitation.
- Calculated B(E2, 4+2 → 2+1) = 147 e2fm4 in 94Mo, matching experimental value 147(23) e2fm4.
- Calculated B(E2, 2+3 → 2+1) = 87 e2fm4 in 94Mo, compared to experimental value 126(76) e2fm4.
- Calculated B(E2, 3/2+1 → 5/2+1) = 202 e2fm4 in 95Mo, compared to experimental value 554(28) e2fm4.
- Calculated B(E2, 9/2+1 → 5/2+1) = 81 e2fm4 in 95Mo, compared to experimental value 291(15) e2fm4.
- Low-energy average B(E2) distributions for 94Mo positive parity approximated by Gaussian with B0 = 5.0 e2fm4, σ = 1.2 MeV; negative parity with B0 = 4.5 e2fm4, σ = 1.0 MeV.
- Low-energy average B(E2) distributions for 95Mo positive parity approximated by Gaussian with B0 = 5.0 e2fm4, σ = 1.0 MeV; negative parity with B0 = 4.0 e2fm4, σ = 1.0 MeV.
- Level density parameters: 94Mo ρ0 = 1.37 MeV−1, Tρ = 0.67 MeV; 95Mo ρ0 = 1.90 MeV−1, Tρ = 0.54 MeV.

## Context

This work extends the author's previous shell-model study of M1 strength functions in 90Zr, 94Mo, 95Mo, and 96Mo (Schwengner 2013), which explained the low-energy enhancement of dipole strength as M1 transitions between states with the π(0g9/2^2) ν(1d5/2^2) configuration. The present results contradict the phenomenological E2 strength function recommended in the RIPL-3 database (Belgya 2006), which uses a Lorentz curve with parameters Emax = 63 A−1/3 MeV, Γ = 6.11 − 0.021 A MeV, and produces an unphysical pole at Eγ = 0. The calculated E2 strength functions can be implemented in statistical reaction codes such as TALYS to test their influence on reaction rates, though the behavior beyond about 6 MeV and in nuclei farther from shell closures remains an open question.
