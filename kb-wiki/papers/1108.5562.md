# Analytical description of the Coherent State Model for near vibrational and well deformed nuclei

> Analytical formulas for the excitation energies as well as for the electric quadrupole reduced transition probabilities in the ground, beta and gamma bands were derived within the coherent state model for the near vibrational and well deformed nuclei. Numerical calculations were performed for 42 nuclei exhibiting various symmetries and therefore with specific properties. Comparison of the calculation results with the corresponding experimental data shows a good agreement. The parameters involved in the proposed model satisfy evident regularities being interpolated by smooth curves. Few of them, which fall out of the curves, are interpreted as signatures for a critical point in a specific phase transition. This is actually supported also by the figures showing the excitation energy dependence on the angular momentum. The formulas provided for energies and B(E2) values are very simple, being written in a compact form, and therefore easy to be handled to explain the new experimental data.

## Key claim
The paper derives analytical, compact formulas for excitation energies and electric quadrupole transition probabilities (B(E2)) within the Coherent State Model (CSM) for both near-vibrational and well-deformed nuclei, but the formulas are obtained via separate power-series expansions in a deformation variable \(x = d^2\) (for small \(d\)) and in \(1/x\) (for large \(d\)), meaning the results are valid only in those respective limits and not as a single unified expression.

## Method
The authors expand the exact matrix elements of the CSM Hamiltonian and E2 transition operator in power series of the deformation parameter \(x = d^2\) for small deformation (near-vibrational regime) and in power series of \(1/x\) for large deformation (well-deformed regime). The resulting analytical formulas for ground, beta, and gamma band energies and B(E2) values are then fitted to experimental data for 42 nuclei using a least-square procedure to determine the model parameters \(d, A_1, A_2, A_3, A_4, A_5, q_h, q_{anh}\).

## Key numbers
* For 13 near-vibrational nuclei, the fitted deformation parameter \(d\) ranges from 0.67610 (for \(^{126}\)Xe) to 1.69634 (for \(^{182}\)Pt), with the ratio \(E_{4_1^+}/E_{2_1^+}\) ranging from 2.194 (for \(^{152}\)Gd) to 3.083 (for \(^{188}\)Os).
* The root-mean-square deviation \(\chi\) between calculated and experimental excitation energies for these 13 nuclei ranges from 11.93490 keV (for \(^{190}\)Os) to 57.96742 keV (for \(^{196}\)Pt).
* For the Gd isotopic chain, the deformation parameter \(d\) for well-deformed isotopes is close to the rotational limit of 3.3, with \(^{154}\)Gd identified as a critical nucleus in the phase transition from SU(5) to SU(3) symmetry.
* The asymptotic limit for large deformation yields Alaga's rule for E2 transitions, with the reduced matrix element for intraband transitions in the ground band given by \(\langle\phi_{J}^{g}||Q_{2}||\phi_{J-2}^{g}\rangle = \sqrt{J/2}(q_h - q_{anh}d)\).

## Context
This work extends the authors' previous study (Raduta, Budaca, Faessler, 2011) on ground band energies to include analytical formulas for the beta and gamma bands within the Coherent State Model (CSM). It provides compact, easy-to-handle expressions that can be used to describe new experimental data for both near-vibrational and well-deformed nuclei, and it identifies nuclei where fitted parameters deviate from smooth regularities as signatures of a critical point in a phase transition, building on the symmetry-based approaches of the Interacting Boson Approximation (IBA) (Iachello, 1970s) and the Bohr-Mottelson model as generalized by Faessler and Greiner.
