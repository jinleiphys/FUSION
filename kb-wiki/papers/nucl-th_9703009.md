# The Eta-Nucleon Scattering Length and Effective Range

> The coupled eta N, pi N system is described by a K-matrix method. The parameters in this model are adjusted to get an optimal fit to pi N -> pi N, pi N -> eta N and gamma N -> eta N data in an energy range of about 100MeV each side of the eta threshold. In the notation T^(-1)+iq_(eta) = 1/a + (r_0)/2 q^2_(eta) + s q^4_(eta), q_(eta) being the momentum in the eta N center-of-mass, the resulting effective range parameters for eta N scattering are found to be a(fm) = 0.75(4)-i0.27(3), r_0(fm) = -1.50(13)-i0.24(4) and s(fm^3) = -0.10(2)-i0.01(1)

# The Eta-Nucleon Scattering Length and Effective Range

## Key claim

The eta-nucleon scattering length, effective range, and shape parameter are extracted from a coupled-channel K-matrix fit to pi N -> pi N, pi N -> eta N, and gamma N -> eta N data, but the extracted values carry uncertainties that depend on the specific parametrization of the S(1535) and S(1650) resonances and on the treatment of the two-pion-nucleon inelastic channel.

## Method

The authors constructed a two-channel K-matrix model coupling the pi N and eta N channels, with the S(1535) and S(1650) resonances included as poles and the two-pion-nucleon channel treated as an optical potential correction. The 10 free parameters of the model were fitted using Minuit to 110 data points: 23 pi N amplitudes from Arndt et al., 11 pi N -> eta N cross sections reviewed by Nefkens, and 53 gamma N -> eta N cross sections from Krusche et al., with the photoproduction amplitude treated as an energy-independent normalization parameter.

## Key numbers

- Eta-nucleon scattering length: a = 0.75(4) - i 0.27(3) fm
- Eta-nucleon effective range: r0 = -1.50(13) - i 0.24(4) fm
- Eta-nucleon shape parameter: s = -0.10(2) - i 0.01(1) fm^3
- Overall chi-squared per degree of freedom: 0.83
- Separate chi-squared per data point: pi N amplitudes 0.73, pi N -> eta N 0.75, gamma N -> eta N 0.94, and an additional set 0.60
- Threshold eta N amplitude: 0.75 + i 0.27 fm at 1485.7 MeV; at 1468.4 MeV the amplitude becomes 0.49 + i 0.10 fm; at 1500.0 MeV it becomes 0.51 + i 0.51 fm
- Predicted two-pion branching ratio for S(1535): 0.038, compared with experimental estimates of 0.05 to 0.20

## Context

This work extends earlier predictions of eta-nucleon quasi-bound states by Haider and Liu and by Li et al., and the suggestion by Wilkin that such a state is seen in the rapid slope of the pd -> He-3 eta amplitude near threshold. The extracted scattering length supports the estimates of Wilkin and of Abaev et al. (1996), and provides the improved parametrization needed for firmer calculations of eta-nucleus interactions, particularly for determining whether eta-bound states exist in few-nucleon systems.
