# Automated calculation of spin, isospin branching rules for su(4) irreps

> The open-source Python package, su4-branching, is introduced for the derivation of comprehensive spin S and isospin T branching rules for any SU(4) irreducible representation.The Wigner supermultiplet scheme in nuclear and hadronic physics is based on SU(4) symmetry. However, the community does not have easy access to practical calculations of branching rules for any irreps.Our implementation combines group-theoretical methods with a notebook interface that is easy to use. This lets researchers look at large and complicated SU(4) irreps and check their work.The software produces tables, CSV files, and visual summaries, and it has been tested against both classic and modern reference results. This work enables group-structure investigations in nuclear modeling, particle physics, and quantum chemistry.

## Key claim

The paper introduces an open-source Python package, `su4-branching`, that automates the calculation of spin (S) and isospin (T) branching rules for any SU(4) irreducible representation, but the implementation is restricted to Young diagrams with four rows (f4 = 0) and has been tested only up to 24 particles, meaning it does not claim full generality for all SU(4) irreps or arbitrarily large systems.

## Method

The package implements Racah's tensor-contraction method, which uses closed-form combinatorial formulas with greatest-integer (floor) functions to compute multiplicities N_ST(f1, f2, f3, 0) for each (S,T) pair without explicit matrix diagonalization. The algorithm evaluates differences of step functions W_ST(f, f') that count allowed SU(2) couplings, with each term corresponding to adding or removing a box in the Young diagram. The code is written in Python 3.8+ with dependencies on NumPy, SymPy, pandas, and matplotlib, and outputs results as tables, CSV files, LaTeX-compatible formats, and visual summaries.

## Key numbers

- The package has been tested up to SU(4) irreps with dimension approximately 10,000,000.
- For the (9,4,2,0) irrep, the code reproduces 31 distinct (S,T) multiplets with multiplicities ranging from 1 to 5, including states such as (S=1/2, T=1/2) with multiplicity 1 and dimension 4, up to (S=11/2, T=7/2) with multiplicity 1 and dimension 96.
- For the (8,4,2,0) irrep, the code reproduces 28 distinct (S,T) multiplets including (S=0, T=0) with multiplicity 1 and dimension 1, up to (S=5, T=3) with multiplicity 1 and dimension 77.
- For the (6,4,1,0) irrep, the code reproduces 19 distinct (S,T) multiplets including (S=1/2, T=1/2) with multiplicity 1 and dimension 4, up to (S=9/2, T=3/2) with multiplicity 1 and dimension 40.
- The sd-shell constraint limits f1(SU(4)) to less than or equal to 6, while the pf-shell constraint limits f1(SU(4)) to less than or equal to 10.
- The (9,6,3,0) irrep is cited as having dimension 4,096.

## Context

This work extends the foundational group-theoretical methods of Wigner (1937), Hecht (1969), and Racah (1949) by providing a freely available computational implementation that automates what was previously a manual or inaccessible calculation. The package enables systematic exploration of SU(4) branching rules for nuclear shell-model calculations in the sd-shell (U(6) x SU(4) classification) and pf-shell (U(10) x SU(4) classification), as well as applications in quantum chemistry and ultracold atomic gases where SU(4) symmetry appears in alkaline-earth atoms like Sr-87 and Yb-173. The code has been validated against classic tables from Quesne (1976), Patera (1981), and Pan (2023), and uses a dimensional-consistency check (sum of (2S+1)(2T+1) weighted by multiplicities must equal the SU(4) irrep dimension) as a necessary and sufficient condition for correctness.
