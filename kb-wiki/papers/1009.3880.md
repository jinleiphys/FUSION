# Many-body fits of phase-equivalent effective interactions

> In many-body theory it is often useful to renormalize short-distance, high-momentum components of an interaction via unitary transformations. Such transformations preserve the on-shell physical observables of the two-body system (mostly phase-shifts, hence unitarily-connected effective interactions are often called phase-equivalent), while modifying off-shell T-matrix elements influential in many-body systems. In this paper I lay out a general and systematic approach for controlling the off-shell behavior of an effective interaction, which can be adjusted to many-body properties, and present an application to trapped fermions at the unitary

## Key claim

The paper demonstrates that one can systematically tune the off-shell behavior of a phase-equivalent effective interaction by fitting the coefficients of generators of a unitary transformation to many-body data, while preserving two-body on-shell observables; the method is illustrated for trapped fermions at the unitary limit, but the author notes that the minimization is model-space dependent and that residual errors from induced three-body forces remain.

## Method

The author constructs a unitary transformation \(\hat{U} = \exp(\hat{A})\) in the relative two-body space, expands the anti-Hermitian operator \(\hat{A}\) in a set of generators with coefficients \(c_i\), and uses the Hellman-Feynman theorem to compute the first-order dependence of many-body eigenenergies on the \(c_i\). Minimizing the root-mean-square difference between calculated and target many-body energies yields a linear system solved by singular value decomposition, which is iterated to convergence. The method is applied to spin-1/2 fermions in a harmonic trap with a zero-range contact interaction at the unitary limit (infinite scattering length), using configuration-interaction calculations in a truncated harmonic oscillator basis.

## Key numbers

- For the bare interaction with no unitary transformation, the rms error on Set II (ground state energies for \(N=3\)–10) is 1.16 \(\hbar\Omega\).
- For the ABF regularized interaction with no transformation, the rms error on Set II is 2.32 \(\hbar\Omega\).
- Using all ten generators of the unitary transformation fitted to Set I, the rms error on Set II (extrapolation) is 0.32 \(\hbar\Omega\) for the bare starting interaction and 0.37 \(\hbar\Omega\) for the ABF starting interaction.
- Using only the single generator \(d/dr\) fitted to Set I, the rms error on Set II is 0.55 \(\hbar\Omega\) for the bare interaction and 0.58 \(\hbar\Omega\) for the ABF interaction.
- The best fit using all generators fitted directly to Set II yields rms errors of 0.28 \(\hbar\Omega\) (bare) and 0.25 \(\hbar\Omega\) (ABF).

## Context

This work generalizes earlier effective interaction methods such as Okubo-Lee-Suzuki (OLS), the unitary correlation operator method (UCOM), the similarity renormalization group (SRG), and the Alhassid-Bertsch-Fang (ABF) approach, all of which generate phase-equivalent interactions but lack an explicit many-body fitting procedure. The paper extends the phenomenological fitting of two-body matrix elements to low-energy nuclear spectra (Brussard and Glaudemans, 1977; Brown and Richter, 2006) by instead fitting the generators of a unitary transformation, and it parallels the strategy of UCOM (which fits a correlation operator to minimize errors in \(A=3\) and 4 ground states) and the INOY and JISP16 potentials (which are adjusted to both two-body data and many-body binding energies). The method enables systematic control of off-shell behavior to reduce the need for explicit three-body forces in many-body calculations, and the author notes that extending the work to nuclei and to other observables such as rms radii is under investigation.
