# A microscopic cluster model study of $^3$He+$p$ scatterings

> We calculate $^3$He+$p$ scattering phase shifts in two different microscopic cluster models, Model T and Model C, in order to show the effects of tensor force as well as $D$-wave components in the cluster wave function. Model T employs a realistic nucleon-nucleon potential and includes the $D$-wave, whereas Model C employs an effective potential in which the tensor-force effect is considered to be renormalized into the central force and includes only the $S$-wave for the cluster intrinsic motion. The $S$- and $P$-wave elastic scattering phase shifts are obtained in the \{$^3$He+$p$\}+\{$d$ + 2$p$\} coupled-channels calculation. In Model T, the $d$ + 2$p$ channel plays a significant role in producing the $P$-wave resonant phase shifts but hardly affects the $S$-wave non-resonant phase shifts. In Model C, however, the effect of the $d$ + 2$p$ channel is suppressed in both of the $S$- and $P$-wave phase shifts, suggesting that it is renormalized mostly as the $^3$He(1/2$^+$)+$p$ channel in the resonance region.

## Key claim

The paper demonstrates that the role of the $d$+$2p$ rearrangement channel in $^3$He+$p$ scattering depends critically on whether a realistic nucleon-nucleon potential with tensor force (Model T) or an effective central potential (Model C) is used, and that in Model C the importance of this channel is artificially inflated unless the $^3$He cluster wave function is described realistically rather than with a simple harmonic-oscillator function.

## Method

The authors performed microscopic resonating group method (RGM) calculations of $^3$He+$p$ elastic scattering in two models: Model T using the G3RS realistic nucleon-nucleon potential (including central, $LS$, and tensor terms) with $D$-wave components in the $^3$He and deuteron cluster wave functions, and Model C using the Minnesota effective potential (central only) with only $S$-wave cluster wave functions. Both models employed a coupled-channels framework including the $\{^3$He+$p\}$ and $\{d$+$2p\}$ configurations, with the cluster relative motion solved via the microscopic $R$-matrix method.

## Key numbers

- The $D$-wave probability $P_D$ in the $^3$He ground state is 6.2% with the limited basis set and 7.0% with the full basis set using the G3RS potential
- The $D$-wave probability in the deuteron is 5.0% with the limited basis set and 4.8% with the full basis set using the G3RS potential
- The binding energy of $^4$He in Model T is $-$24.41 MeV for the full coupled-channels calculation, only 0.88 MeV above the more precise SVM value of $-$25.29 MeV
- The $d$+$d$ channel contributes 2.1 MeV to the $^4$He binding energy in Model T, compared to only 0.03 MeV in Model C
- The $^3$He+$p$ threshold with the simple $(0s)^3$ harmonic-oscillator function is about 2.4 MeV too high compared to the $p$+$p$+$n$ three-body wave function

## Context

This work extends previous RGM studies of $^3$He+$p$ scattering by Pfitzinger et al. (2001) using realistic potentials, and by Furutani (1979) using effective interactions. The paper clarifies that the tensor force and $D$-wave components in cluster wave functions, which are known to be important for the $^4$He ground state (Suzuki, 2008) and $\alpha$+$n$ $P$-wave phase shifts (Myo, 2005), also play a crucial role in $^3$He+$p$ scattering. The findings enable more reliable microscopic cluster model calculations by demonstrating that the importance of additional rearrangement channels depends sensitively on whether the cluster intrinsic wave functions are described realistically enough to account for tensor-force effects.
