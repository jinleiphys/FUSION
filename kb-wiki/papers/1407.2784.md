# Fermion-dimer scattering using an impurity lattice Monte Carlo approach and the adiabatic projection method

> We present lattice Monte Carlo calculations of fermion-dimer scattering in the limit of zero-range interactions using the adiabatic projection method. The adiabatic projection method uses a set of initial cluster states and Euclidean time projection to give a systematically improvable description of the low-lying scattering cluster states in a finite volume. We use Lüscher's finite-volume relations to determine the s-wave, p-wave, and d-wave phase shifts. For comparison, we also compute exact lattice results using Lanczos iteration and continuum results using the Skorniakov-Ter-Martirosian equation. For our Monte Carlo calculations we use a new lattice algorithm called impurity lattice Monte Carlo. This algorithm can be viewed as a hybrid technique which incorporates elements of both worldline and auxiliary-field Monte Carlo simulations.

## Key claim

The authors demonstrate that the adiabatic projection method can be combined with Monte Carlo simulations (specifically the new impurity lattice Monte Carlo algorithm) to compute fermion-dimer scattering phase shifts, including for the first time p-wave and d-wave channels, with the caveat that the method becomes systematically more accurate as the Euclidean projection time is increased and as more initial cluster states are included.

## Method

The authors use the adiabatic projection method, constructing initial fermion-dimer cluster states parameterized by relative momentum rather than spatial separation, then projecting these states using Euclidean time evolution via the transfer matrix. They introduce the impurity lattice Monte Carlo algorithm, which treats the single spin-down particle as an impurity sampled via worldline Monte Carlo while the spin-up particles are handled using the auxiliary-field formalism, and compute finite-volume energy levels from the resulting dressed cluster states. The s-wave, p-wave, and d-wave phase shifts are then extracted using Lüscher's finite-volume relations, and the results are benchmarked against exact lattice calculations using Lanczos iteration and continuum results from the Skorniakov-Ter-Martirosian equation.

* Particle mass taken as 938.92 MeV; interaction strength C_0 tuned to deuteron energy -2.2246 MeV

## Context

This work extends prior lattice calculations of s-wave fermion-dimer scattering (PhysRevD.84.091503, PhysRevC.86.034003, Rokash 2013, Pine 2013) to include p-wave and d-wave phase shifts for the first time. The adiabatic projection method was previously demonstrated using exact sparse matrix methods by Rupak (2013) and Pine (2013); this paper provides the first application using Monte Carlo simulations. The impurity lattice Monte Carlo algorithm, developed jointly with Bour (2014), enables efficient treatment of few-impurity systems and is computationally superior to auxiliary-field Monte Carlo for this three-particle system due to speed and control over sign oscillations.
