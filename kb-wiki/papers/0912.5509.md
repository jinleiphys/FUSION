# Jet Function with a Jet Algorithm in SCET

> The jet function for the factorized cross section $e^+e^-$ into dijets is given as a function of the jet invariant mass s and with a generic jet algorithm at $\mathcal{O}(α_s)$. We demonstrate the results using the Sterman-Weinberg algorithm and show that the jet function is independent of the energy fraction $β$ of the soft radiation. The anomalous dimension has the same form with and without the cone half-angle $δ$. The dependence of the finite part of the jet function on the cone angle is given.

## Key claim

The jet function for the factorized \(e^+e^-\) dijet cross section can be computed at \(\mathcal{O}(\alpha_s)\) for any jet algorithm expressible in terms of theta functions on final-state momenta, and for the Sterman-Weinberg algorithm the jet function is independent of the soft energy fraction \(\beta\) to leading order in the power counting, provided \(\delta \sim \beta \sim \lambda\) or \(\delta \sim \lambda,\ \beta \sim \lambda^2\).

## Method

The author derives a general formalism for the jet function \(J_n^F(s)\) in SCET by inserting a constraint function \(F(a_i,\hat{p}_j)\) into the cut propagators of the inclusive jet function, then adding and subtracting the inclusive result to isolate the algorithm-dependent contribution \(\Delta J_n^F\). At one loop, the naive collinear contribution and the gluon zero-bin subtraction are computed separately using dimensional regularization, and the Sterman-Weinberg algorithm is implemented by writing the cone and energy constraints in terms of the gluon momentum fraction \(y = p_g^-/Q\) and the jet invariant mass \(s\). The full SW jet function is obtained by summing the inclusive jet function from the literature with the algorithm-dependent piece after zero-bin subtraction.

## Key numbers

* The renormalized SW jet function at \(\mathcal{O}(\alpha_s)\) in the \(\overline{\text{MS}}\) scheme is given by Eq. (4.13), with terms proportional to \(\delta(s)\), \([ \mu^2 \theta(s)/s]_+\), \([ \mu^2 \theta(s) \ln(s/\mu^2)/s]_+\), and finite contributions \(24/(4s + Q^2 \delta^2)\) for \(s < (Q\delta/2)^2\) and \(3/s + (4/s)\ln(4s/(Q^2 \delta^2))\) for \(s > (Q\delta/2)^2\).
* The anomalous dimension of the SW jet function is \(\gamma^{\rm SW}_{J_n}(s,\mu) = -(\alpha_s C_F/4\pi)\{ (8/\mu^2)[\mu^2\theta(s)/s]_+ - 6\,\delta(s) \}\), identical in form to the inclusive case.
* The algorithm-dependent contribution \(\Delta J_n^{\rm SW}\) is finite as \(\epsilon \to 0\) and contains no \(\beta\) dependence; the cancellation of \(\beta\) occurs between the naive collinear and zero-bin pieces.
* The jet function is plotted for \(\delta = 0.2\) and \(\delta = 0.15\) with \(\mu = 0.1 \times (Q/2)\), showing a continuous function whose derivative changes sign at \(s = (Q\delta/2)^2\).

## Context

This work extends the SCET factorization program for \(e^+e^-\) dijet cross sections beyond the hemisphere jet algorithm used by Fleming et al. (2007) and Hornig et al. (2009) to accommodate realistic jet algorithms like Sterman-Weinberg. The result for the anomalous dimension agrees with the consistency condition derived by Ellis et al. (2009) for exclusive multijet cross sections, while the finite part of the jet function is new. The formalism enables a full factorized cross section for \(e^+e^- \to\) dijets with SW jets once the corresponding soft function is computed, and it provides a template for incorporating other cone or cluster algorithms into SCET factorization theorems.
