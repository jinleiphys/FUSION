# Nuclear Dynamics with the Sky3D code

> A description is presented of how to use the Sky3D time-dependent Hartree-Fock code to calculate giant monopole resonances. This requires modification to the code, and a step-by-step guide of how to make the necessary modification is given. An example of how to analyse the output of the code to obtain quantities of physics interest is included. Together, the modifications and the post-processing are intended to serve as a typical example of how the code, which was designed to be extendable to particular users' needs, can be extended.

# Nuclear Dynamics with the Sky3D code

## Key claim
The paper provides a practical, step-by-step guide for extending the Sky3D time-dependent Hartree-Fock code to calculate giant monopole resonances, but this extension is presented only as a typical example of how the code can be adapted for specific user needs, not as a general or validated method for all monopole calculations.

## Method
The Sky3D code is first run in static mode (imode=1) to generate a Hartree-Fock ground state for ¹⁸O using the SkI4 Skyrme force. The external field subroutine (external.f90) is then modified to add a monopole boost operator proportional to r², alongside the existing quadrupole Q₂₀ operator, by declaring a new variable amplm and modifying the damped field application. A time-dependent calculation (imode=2) is performed with 5000 time steps of 0.2 fm/c, and the monopole response is extracted from the quadrupoles.res output file by computing the expectation value of r² from the separate x², y², z² moments for protons and neutrons.

## Key numbers
- Time step size: 0.2 fm/c for the dynamic calculation
- Number of time steps: 5000 in the dynamic run
- Monopole kick strength: amplm=0.001 (dimensionless, as set in the input file)
- Grid dimensions: 24×24×24 points with spacing dx=dy=dz=1.0 fm
- Static calculation uses 8 protons and 10 neutrons (¹⁸O)
- Revenant vibrations appear after approximately t=400 fm/c due to flux rebounding from the coordinate space box walls

## Context
This work extends the Sky3D TDHF code (Maruhn, 2014) which provides a general-purpose time-dependent density functional theory code with no symmetry restrictions, building on the established TDHF method (Simenel, 2012) for giant multipole resonance studies (Tohsaki, 2002; Stevenson, 2004; Nakatsukasa, 2005; Umar, 2005; Brink, 2006; Stevenson, 2007; Nakatsukasa, 2007; Inakura, 2009; Pardi, 2013). The modification enables users to study monopole resonances beyond the code's default quadrupole-only external field capability, and the post-processing method using spectral analysis (Almehed, 2005) allows extraction of strength functions from the time-dependent response.
