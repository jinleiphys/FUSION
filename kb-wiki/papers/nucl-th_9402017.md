# Fragmentation model and quark/hadron system

> Nuclear multifragmentation process can be viewed as a recombination of nucleons into clusters of various sizes. In a combinatorial analysis, various moments of cluster size distribution appear to be quite simple in terms of canonical partition functions. This simple model can also describe a branching phenomena and a clusterization phenomena. The possibility of applying this fragmentation model in describing a hadronization of quark-gluon system and a strangeness production is discussed.

## Key claim

The paper proposes that a combinatorial fragmentation model, originally developed for nuclear multifragmentation, can be applied to describe the hadronization of a quark-gluon system and strangeness production, but only after further studies to determine the appropriate parameters (such as the number of species, weighting parameters, and constraints) for the quark-hadron system.

## Method

The author develops a combinatorial model where a system of A nucleons (or more generally, a conserved quantity A) is partitioned into clusters of various sizes, with each partition assigned a weight based on parameters x_i and beta_i. The canonical partition function Q_A is derived from a generating function, and the model is extended to describe evolution and branching phenomena by making the weight parameters x_i(t) functions of an evolution parameter such as time or temperature, leading to dynamic equations for the moments of the cluster size distribution and for the factorial distribution of species.

## Key numbers

- For a collision involving 235U + 235U, the number of possible partitions is of the order 10^20.
- When all species have the same weight x_i = x, the canonical partition function becomes Q_A = Gamma(x + A) / Gamma(x).
- For the branching process in phi^3 theory with N = 2, x_1(t) = 1/omega(t) and x_2(t) = 1 - 1/omega(t), the solution for the multiplicity distribution is the Furry distribution: F_{A,1}^{n_k}(omega) = Gamma(A) / (Gamma(n_k) Gamma(A - n_k + 1)) * (1/omega)^{n_k} * (1 - 1/omega)^{A - n_k}, with omega(t) = e^{beta t}.

## Context

This work extends the combinatorial fragmentation model developed by Mekjian (1990) and Lee (1993) for nuclear multifragmentation, which had previously been used to fit data in nucleon-nucleus and nucleus-nucleus collisions as well as thermodynamic statistical models, Monte Carlo calculations (Bondorf), and percolation models. The paper discusses the possibility of applying this model to quark-hadron phase transitions, using pions as a simple example, but notes that further studies are needed to determine the appropriate parameters for practical application to quark-gluon systems including strangeness.
