# Holographic chiral currents in a magnetic field

> In the presence of a quark chemical potential, a magnetic field induces an axial current in the direction of the magnetic field. We compute this current in the Sakai-Sugimoto model, a holographic model which, in a certain limit, is dual to large-N_c QCD. We also compute the analogous vector current, for which an axial chemical potential is formally introduced. This vector current can potentially be observed via charge separation in heavy-ion collisions. After implementing the correct axial anomaly in the Sakai-Sugimoto model we find an axial current in accordance with previous studies and a vanishing vector current, in apparent contrast to previous weak-coupling calculations.

## Key claim

After implementing the correct (covariant) axial anomaly in the Sakai-Sugimoto model via Bardeen's counterterm, the axial current in the direction of the magnetic field agrees with previous studies, but the vector current vanishes for all magnetic fields, in apparent contrast to previous weak-coupling calculations for the chiral magnetic effect.

## Method

The authors solve the equations of motion for the gauge fields in the Sakai-Sugimoto model, a holographic top-down model dual to large-N_c QCD, in the presence of a homogeneous magnetic field B and quark chemical potentials. They compute the chiral currents from the on-shell action plus Bardeen's counterterm, separately for the confined (chirally broken) and deconfined (chirally restored) phases, and compare results obtained from the full renormalized currents versus the Yang-Mills part only.

## Key numbers

- In the deconfined (chirally symmetric) phase, the renormalized axial current is exactly \(\mathcal{J}_5 = \frac{\mu B N_c}{2\pi^2}\), with the model parameter \(M_{\rm KK}\) dropping out of this universal result.
- In the deconfined phase, the renormalized vector current is zero for all magnetic fields, while the weak-coupling result would be \(\mathcal{J} = \frac{\mu_5 B N_c}{2\pi^2}\).
- In the confined (chirally broken) phase, the renormalized axial current is given by \(\mathcal{J}_5 / \frac{\mu B N_c}{2\pi^2} = \frac{\beta\coth\beta\pi}{2\rho(\beta)}\) where \(\beta = \alpha B/M_{\rm KK}^2\) is the dimensionless magnetic field and \(\rho(\beta) \equiv \beta\coth\beta\pi + \frac{\pi\beta^2}{2\sinh^2\beta\pi}\).
- In the confined phase, the renormalized vector current is also zero for all magnetic fields.

## Context

This work extends previous holographic studies of magnetic fields in the Sakai-Sugimoto model by Bergman (2008), Thompson (2008), and Rebhan (2008), and addresses the chiral magnetic effect proposed by Kharzeev (2007) and Fukushima (2008). The authors find that the correct implementation of the covariant QED anomaly via Bardeen's counterterm yields a vanishing vector current, contradicting weak-coupling expectations; they note that a recent study by Gynther (2010) using linear response theory in a similar holographic model found the vector current to coincide with the weak-coupling result, suggesting the discrepancy may arise from how the chemical potential is defined for finite background fields.
