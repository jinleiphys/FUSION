# Core Polarization with Variable Oscillator Length Parameters for Valence Particles

> We study quadrupole and monopole core polarization using harmonic oscillator wave functions but with different length parameters for the valence particle as compared to the core. We use perturbation theory with a delta interaction. The results also hold for a density dependent delta interaction [1]. We study how the amount of core polarization varies with the distance of the valence particle from the core

## Key claim

The paper shows that the core polarization contribution to quadrupole and monopole moments depends sensitively on the ratio of the valence particle's oscillator length parameter to that of the core, with the effect enhanced when the valence particle moves closer to the core and diminished when it moves farther away, contrary to the abstract's neutral phrasing that only states the variation is studied.

## Method

The author uses first-order perturbation theory with a zero-range delta interaction (which also holds for a density-dependent delta interaction) to compute quadrupole and monopole core polarization in a simple model of a valence p3/2 neutron relative to a 4He core. Harmonic oscillator wave functions are employed with a fixed oscillator length parameter for the core (b_c) and a variable oscillator length parameter for the valence nucleon (b). The ratio of core polarization contributions for b ≠ b_c versus b = b_c is derived analytically using scaling properties of harmonic oscillator wave functions.

## Key numbers

- For the quadrupole mode, the ratio RQ(x) = x²/(0.5 + 0.5x²)^3.5, where x = b/b_c; RQ reaches a maximum of 1.394 at x = √(2/5) ≈ 0.632.
- At x = 0.9, RQ = 1.149; at x = 1, RQ = 1; at x = 1.1, RQ = 0.853; at x = 2, RQ = 0.162.
- For the monopole mode, the ratio RM(x) = (3 − 2x²)/(0.5 + 0.5x²)^3.5; at x = 0, RM = 33.941; at x = 0.9, RM = 1.957; at x = 1, RM = 1; at x = 1.1, RM = 0.409.
- The monopole ratio vanishes at x = √(3/2) ≈ 1.225, becomes negative beyond that, and reaches an extremum at x = √(5/2) ≈ 1.581 with RM = −0.282.

## Context

This work extends the standard treatment of core polarization in nuclear physics by allowing the valence particle to have a different oscillator length than the core, motivated by the fact that in neutron-rich nuclei valence neutrons can be very loosely bound and thus have reduced ability to polarize the core. The results are derived for a delta interaction and are noted to also hold for a density-dependent delta interaction (referenced as [1] in the paper). The simple analytic expressions enable quick estimates of how core polarization changes with the spatial extent of the valence orbit, which is relevant for understanding quadrupole moments and monopole shifts in exotic nuclei far from stability.
