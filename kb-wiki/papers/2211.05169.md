# Mueller's dipole wave function in QCD: emergent KNO scaling in the double logarithm limit

> We analyze Mueller's QCD dipole wave function evolution in the double logarithm approximation (DLA). Using complex analytical methods, we show that the distribution of dipole in the wave function (gluon multiplicity distribution) asymptotically satisfies the Koba-Nielsen-Olesen (KNO) scaling, with a non-trivial scaling function $f(z)$ with $z=\frac n{\bar n}$. The scaling function decays exponentially as $2(2.55)^2ze^{-\frac{z}{0.3917}}$ at large $z$, while its growth is log-normal as $e^{-\frac{1}{2}\ln^2 z}$ for small-$z$. A detailed analysis of the Fourier-Laplace transform of $f(z)$, allows for performing the inverse Fourier transform, and access the non-asymptotic bulk-region around the peak. The bulk and asymptotic results are shown to be in good agreement with the measured hadronic multiplicities in DIS, as reported by the H1 collaboration at HERA in the region of large $Q^2$. A numerical tabulation of $f(z)$ is included. Remarkably, the same scaling function is found to emerge in the resummation of double logarithms in the evolution of jets. Using the generating function approach, we show why this is the case. The absence of KNO scaling in non-critical and super-renormalizable theories is briefly discussed. We also discuss the universal character of the entanglement entropy in the KNO scaling limit, and its measurement using the emitted multiplicities in DIS and $e^+e^-$ annihilation.

## Key claim

The authors show that Mueller's QCD dipole wave function evolution in the double logarithm approximation (DLA) asymptotically satisfies Koba-Nielsen-Olesen (KNO) scaling, with a non-trivial scaling function that agrees with measured hadronic multiplicities in deep inelastic scattering (DIS) from the H1 collaboration at HERA, contrary to the simpler exponential scaling function previously believed to emerge from the diffusion limit.

## Method

The authors simplify Mueller's evolution equation for the generating functional of dipole multiplicities in the DLA limit, where emitted dipoles carry progressively smaller sizes. They derive a second-order nonlinear differential equation for the generating function, solve for the asymptotic moment sequence using recursive relations involving Bell polynomials, and obtain the KNO scaling function through complex analytic continuation of the Fourier-Laplace transform using an integral representation of the function W = ln Z.

## Key numbers

* The asymptotic moment sequence satisfies a_n → 2(0.39174)^n n n! for large n, with the constant C = 0.3917 determined by the radius of convergence r = 1/C = e^(-0.93715) = 0.391743
* The scaling function f(z) decays exponentially as 2(2.55)^2 z e^(-z/0.3917) for large z, where 2.55 = 1/0.3917
* The mean multiplicity in DLA is given by n̄(ρ) = I_0(2√ρ) - 1, where ρ = (α_s N_c/π) y ln(Q^2/Q_0^2) for fixed coupling
* The first few asymptotic moment coefficients are a_2 = 4/3, a_3 = 9/4, a_4 = 208/45, a_5 = 2425/216, a_6 = 2207/70

## Context

This work extends the analysis of Mueller's dipole evolution (Mueller, 1993, 1994) beyond the diffusion limit, which previously suggested a simple exponential KNO scaling function e^(-z). The authors show that the DLA limit produces a scaling function matching H1 collaboration data (2020) from HERA, and demonstrate that the same scaling function emerges in jet evolution through the BMS-BK correspondence (Banfi, 2002; Weigert, 2003; Marchesini, 2003; Hatta, 2013; Caron-Huot, 2015). The results enable estimation of entanglement entropy in DIS scattering and establish connections between dipole wave function evolution and DGLAP evolution in their common DLA region.
