# Eta Meson Production in NN Collisions

> Eta meson production in both proton-proton and proton-neutron collisions is investigated within a relativistic meson exchange model of hadronic interactions. It is found that the available cross section data can be described equally well by either the vector or pseudoscalar meson exchange mechanism for exciting the S_{11}(1535) resonance. It is shown that the analyzing power data can potentially be very useful in distinguishing these two scenarios for the excitaion of the S_{11}(1535) resonance.

## Key claim

The available total cross section data for eta meson production in both proton-proton and proton-neutron collisions can be described equally well by either vector meson exchange dominance or pseudoscalar (pi and eta) meson exchange as the dominant mechanism for exciting the S11(1535) resonance, but the analyzing power data may distinguish these two scenarios.

## Method

The authors employed a relativistic meson exchange model within the Distorted Wave Born Approximation, constructing the production current from nucleonic, mesonic, and resonance (S11(1535), P11(1440), D13(1520)) contributions. The NN final state interaction was calculated using the Bonn potential, while the NN initial state interaction was treated in the on-shell approximation using phase shifts from the SAID database. A separate vector meson exchange dominance model was also constructed, using gamma5-gamma_mu couplings for the rho and omega meson vertices with the S11(1535) resonance, to allow direct comparison with previous work.

## Key numbers

- The vector meson exchange dominance model uses coupling constants g_rho_NN* = -0.85 and g_omega_NN* = -1.10 for the S11(1535) resonance excitation
- The eta NN coupling constant used is g_eta_NN = 6.14, consistent with the Bonn potential
- The NN initial state interaction reduction factor for the ^3P_0 state is lambda = 0.19 near threshold, and for the ^1P_1 state is lambda = 0.27
- The nucleon form factor cutoff parameter is Lambda_N = 1.2 GeV

## Context

This work extends the debate on S11(1535) resonance excitation mechanisms in NN collisions, directly contrasting with the rho meson exchange dominance claims of Faldt and Wilkin (1997) and Gedalin et al. (1998). The authors show that pi and eta exchange can describe the data as well as the vector meson dominance scenario, contrary to the findings of Santra and Jain (1998) and Pena et al. (1999). The results enable future experimental measurements of spin observables, particularly the analyzing power, to distinguish between these competing reaction mechanisms, and provide a framework for investigating eta N interactions and possible eta NN bound states.
