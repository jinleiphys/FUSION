# Neutrino-nucleus DIS data and their consistency with nuclear PDFs

> In this talk, we discuss the compatibility of different deeply inelastic neutrino-nucleus data sets and the universal nuclear PDFs. This is an issue that has lately been investigated by different groups but the conclusions have been surprisingly contradictory. While some studies have found a good overall agreement between the nuclear PDFs and the neutrino data, others have claimed for an incompatibility. Here, we demonstrate that the independent neutrino data sets from NuTeV, CHORUS and CDHSW collaborations differ in the absolute overall normalization and that it is not possible to accurately reproduce all the data simultaneously with a single set of PDFs. Our strategy to overcome this difficulty and allow a consistent use of all neutrino data in global PDF analyses is to normalize the data by the integrated cross-sections thereby cancelling possible inaccuracies in the absolute normalization. Indeed, this brings all data to a surprisingly good mutual agreement underscoring the x-dependence of the nuclear modifications in a model-independent way. The consistency of these data with the present nuclear PDFs is verified by introducing a method to test the effect of a new data set in an existing global fit that performed a Hessian error analysis.

## Key claim

The independent neutrino-nucleus deep inelastic scattering data sets from NuTeV, CHORUS, and CDHSW collaborations differ in absolute overall normalization, making it impossible to accurately reproduce all data simultaneously with a single set of parton distribution functions (PDFs); however, normalizing each data set by its integrated cross-section brings all data into surprisingly good mutual agreement and confirms consistency with existing nuclear PDFs.

## Method

The authors compared neutrino-nucleus cross-section data from NuTeV, CHORUS, and CDHSW experiments (2136, 824, and 937 data points respectively, after cuts of Q² > 4 GeV² and W² > 12.25 GeV²) with next-to-leading order pQCD calculations using CTEQ6.6 free proton PDFs and EPS09 nuclear modifications, with SACOT heavy quark treatment and electroweak/target-mass corrections. They constructed ratios of experimental to theoretical cross-sections, then normalized each data set by its integrated cross-section per energy bin to cancel absolute normalization inaccuracies, and verified consistency using a Hessian reweighting technique that tests whether new data can be accommodated within the existing global fits.

## Key numbers

- 2136 NuTeV, 824 CHORUS, and 937 CDHSW data points remain after cuts of Q² > 4 GeV² and W² > 12.25 GeV²
- Neutrino beam energy ranges from E ~ 20 GeV up to E ~ 300 GeV
- For NuTeV data without normalization: χ²/N = 1.04 with central PDFs, EPS09 penalty = 40.3 (close to the permitted maximum of 50) and CTEQ6.6 penalty = 42.5
- For NuTeV data with normalization: χ²/N = 0.84 with central PDFs, EPS09 penalty = 13.9 and CTEQ6.6 penalty = 35.4
- When freezing CTEQ6.6 to central value (mimicking prior analyses), NuTeV data without normalization gives EPS09 penalty = 77.4, clearly above the permitted 50

## Context

This work extends the analysis of Paukkunen (2010) which first noticed beam-energy-dependent normalization fluctuations in NuTeV data, and directly contradicts the conclusions of Schienbein (2007, 2009) and Kovarik (2010) who claimed neutrino data are incompatible with charged-lepton data. The normalization procedure enables consistent inclusion of all neutrino-nucleus data in global nuclear PDF analyses, resolving a long-standing discrepancy. The Hessian reweighting technique provides a quantitative method to test new data sets against existing global fits without refitting.
