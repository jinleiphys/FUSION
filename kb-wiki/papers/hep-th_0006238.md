# Deformed Heisenberg Algebra with Reflection, Anyons and Supersymmetry of Parabosons

> Deformed Heisenberg algebra with reflection appeared in the context of Wigner's generalized quantization schemes underlying the concept of parafields and parastatistics of Green, Volkov, Greenberg and Messiah. We review the application of this algebra for the universal description of ordinary spin-$j$ and anyon fields in 2+1 dimensions, and discuss the intimate relation between parastatistics and supersymmetry.

# Deformed Heisenberg Algebra with Reflection, Anyons and Supersymmetry of Parabosons

## Key claim
The deformed Heisenberg algebra with reflection (RDHA) provides a universal algebraic framework that unifies the description of ordinary spin-\(j\) fields and anyon fields in 2+1 dimensions via first-order linear differential equations, and simultaneously reveals that purely parabosonic systems of even order \(p = 2(k+1)\) possess a hidden supersymmetry, but this supersymmetry is polynomial (nonlinear) except in the special case \(k=0\) where it reduces to conventional linear \(N=1\) supersymmetry.

## Method
The paper reviews the algebraic structure of the RDHA, defined by \([a^-, a^+] = 1 + \nu R\) with reflection operator \(R\) satisfying \(\{R, a^\pm\} = 0\) and \(R^2 = 1\), and shows that its irreducible representations correspond to parabosons of order \(p = \nu + 1\) (for \(\nu > -1\)) or to deformed parafermions (for \(\nu = -(2p+1)\)). The \(osp(1|2)\) superalgebraic structure carried by RDHA is then exploited to construct a spinor set of linear differential equations \(D_\alpha \Psi(x) = 0\) that describe both finite-dimensional (ordinary spin-\(j\)) and infinite-dimensional (anyon) representations in 2+1 dimensions, depending on the deformation parameter \(\nu\).

## Key numbers
- Paraboson order \(p\) is related to the deformation parameter by \(\nu = p - 1\), with \(p = 1, 2, \ldots\) for unitary infinite-dimensional representations.
- For \(\nu = 2k+1\) (\(k = 0, 1, \ldots\)), the paraboson order is \(p = 2(k+1)\); the corresponding supersymmetric spectrum has \(k+1\) singlet states with energies \(E = 0, 2, \ldots, 2k\).
- The Casimir operator of \(osp(1|2)\) takes the fixed value \(\mathcal{C} = \frac{1}{16}(1 - \nu^2)\).
- The spin \(s\) of the field described by the linear equations is \(s = \frac{1}{4}(1 + \nu)\); integer spin occurs at \(\nu = 4n - 1\) (\(s = n\), \(n = 1, 2, \ldots\)) and half-integer spin at \(\nu = 4n + 1\) (\(s = n + \frac{1}{2}\), \(n = 0, 1, \ldots\)).
- Finite-dimensional representations occur for \(\nu = -(2p+1)\) with \(p = 1, 2, \ldots\), giving dimension \(2p+1\).

## Context
This review extends the earlier work on the RDHA by Plyushchay and collaborators (def1, def2) and the discovery of hidden supersymmetry in purely parabosonic systems (hid). It connects the algebraic structure to the description of anyons via the infinite-component Majorana equation in 2+1 dimensions, as studied by Torres (tor) in the context of the relativistic particle with torsion. The results enable a unified treatment of ordinary spin fields and anyon fields through a single set of linear differential equations, and suggest that polynomial supersymmetry with multiple singlet states could serve as a signature of parabosonic-like quasiparticles in physical systems.
