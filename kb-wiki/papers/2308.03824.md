# Exact Neutron-Proton Wavefunctions Using the Phase Function Method

> Radial phase shifts ($δ(r)$), amplitude functions ($A(r)$), and exact wavefunctions ($u(r)$) for various uncoupled S, P, and D channels of neutron--proton scattering have been calculated using the Phase Function Method (PFM). In these calculations, inverse potentials obtained from the Morse function as the zeroth-order reference potential are employed. The parameters of the Morse potential were optimized using the comprehensive GRANADA partial wave analysis, consisting of 6713 experimental \textit{np} phase shift data points from 1950 to 2013, by minimizing the mean square error (MSE) as a cost function. The present work provides detailed radial dependence of $δ(r)$, $A(r)$, and $u(r)$ up to 5~fm for laboratory energies $E_{\ell \text{lab}} = [1, 10, 50, 100, 150, 250, 350]$~MeV. The obtained wavefunctions show excellent agreement with high-precision Nijmegen-II results, highlighting the accuracy and transparency of the PFM approach for uncoupled scattering states.

## Key claim

The Phase Function Method (PFM), using inverse potentials derived from a Morse function whose parameters are optimized against the comprehensive GRANADA partial wave analysis of 6713 experimental *np* phase shift data points, can produce radial phase shifts, amplitude functions, and exact wavefunctions for uncoupled S, P, and D channels that show excellent agreement with high-precision Nijmegen-II results, though a certain phase lag is observed for some states due to the simplicity of the Morse model.

## Method

The author solved the Riccati-type first-order differential equation of the Phase Function Method (PFM) using a fifth-order Runge-Kutta (RK-5) method to obtain radial phase shifts δ(r) for various uncoupled channels of neutron-proton scattering. Inverse potentials were constructed using the Morse function as a zeroth-order reference potential, with parameters V₀, rₘ, and aₘ optimized by minimizing the mean square error (MSE) against the GRANADA partial wave analysis dataset. From the phase shifts, the amplitude functions A(r) and exact wavefunctions u(r) were then computed for laboratory energies E_lab = [1, 10, 50, 100, 150, 250, 350] MeV up to a radial distance of 5 fm.

## Key numbers

- Morse potential parameters for the ¹S₀ state: V₀ = 70.438 MeV, rₘ = 0.901 fm, aₘ = 0.372 fm, with MSE = 0.649.
- Morse potential parameters for the ³P₀ state: V₀ = 11.579 MeV, rₘ = 1.750 fm, aₘ = 0.601 fm, with MSE = 0.049.
- Morse potential parameters for the ¹D₂ state: V₀ = 131.302 MeV, rₘ = 0.010 fm, aₘ = 0.526 fm, with MSE = 0.026.
- The value of ħ²/(2μ) used for the *np* system is 41.47 MeV fm².
- The GRANADA dataset used for optimization consists of 6713 experimental *np* phase shift data points from 1950 to 2013.
- Wavefunctions and phase shifts were computed for laboratory energies E_lab = [1, 10, 50, 100, 150, 250, 350] MeV.

## Context

This work extends the author's previous application of the Phase Function Method with the Morse potential for nucleon-nucleon scattering (Khachi, 2023) by providing the full radial dependence of phase shifts, amplitudes, and wavefunctions rather than just asymptotic phase shifts. The results are validated against the high-precision Nijmegen-II (NIJM-II) potential and the JISP model, with the wavefunctions for states such as ³P₀, ³P₁, and ¹D₂ showing excellent agreement. The author notes that a Combinatorial Data Analysis (CDA) procedure developed by their group could generate isospectral potentials to refine the results and reduce the observed phase lag, and that the methodology can be extended to other scattering systems such as n-α, p-α, and n-C.
