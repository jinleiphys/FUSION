# Prompt and non-prompt production of charm hadrons in proton-proton collisions at the Large Hadron Collider using machine learning

> In this contribution, we use machine learning (ML) based models to separate the prompt and non-prompt production of heavy flavour hadrons, such as $D^0$ and J/$ψ$, in proton-proton collisions at LHC energies. For this purpose, we use PYTHIA~8 to generate events, which provides a good qualitative agreement with experimental measurements of charm hadron production. The input features for the ML models are experimentally measurable. The prediction accuracy of the ML models used in this study reaches up to 99\%. The ML models can be useful in providing precise track-level identification, which is not possible in experiments with traditional methods. The contribution also discusses future applications of the ML models to understand the production of prompt and non-prompt heavy quark hadrons.

# Prompt and non-prompt production of charm hadrons in proton-proton collisions at the Large Hadron Collider using machine learning

## Key claim
Machine learning models can separate prompt from non-prompt charm hadrons (D⁰ and J/ψ) in proton-proton collisions with up to 99% classification accuracy, but the models show small misclassification rates (1% for non-prompt D⁰ classified as prompt, 2% for prompt J/ψ classified as non-prompt) that the authors argue do not significantly affect results due to the small ratio of non-prompt to prompt production.

## Method
The authors used PYTHIA 8 Monte Carlo event generator to simulate proton-proton collisions at LHC energies, generating D⁰ → π⁺K⁻ and J/ψ → μ⁺μ⁻ decay channels. They trained multiple machine learning models (XGBoost, LightGBM, CatBoost, and Random Forest) on experimentally measurable input features including invariant mass, transverse momentum, pseudorapidity, pseudoproper decay length, distance of closest approach, and pseudoproper time to classify charm hadrons as prompt or non-prompt based on topological properties.

## Key numbers
- Classification accuracy of the machine learning models reaches up to 99%
- For D⁰, XGBoost model misclassifies 1% of non-prompt D⁰ as prompt; Random Forest misclassification rises to 3%
- For J/ψ, XGBoost model misclassifies 2% of prompt J/ψ as non-prompt
- The non-prompt to prompt D⁰ production fraction was studied at three collision energies: √s = 13 TeV, 5.02 TeV, and 0.9 TeV
- The self-normalized yield ratios of both prompt and non-prompt D⁰ and J/ψ show a stronger-than-linear increase with self-normalized charged particle density

## Context
This work extends previous studies by Goswami (2024) and Prasad (2023) on topological separation of charm hadron production using machine learning techniques. The approach addresses the experimental limitation that traditional methods cannot provide precise track-level identification of prompt versus non-prompt charm hadrons. The results enable future measurements of charm hadron production dynamics in different topological regions, which could reduce uncertainties compared to conventional analysis methods. The authors note that the stronger-than-linear rise in self-normalized yields for prompt J/ψ is not trivially understood and may arise from biases due to event selections, as discussed in Prasad (2024).
