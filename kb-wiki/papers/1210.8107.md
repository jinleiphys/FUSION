# Determination of the Parameters of a Color Neutral 3D Color Glass Condensate Model

> We consider a version of the McLerran-Venugopalan model by Lam and Mahlon where confinement is implemented via colored noise in the infrared. This model does not assume an infinite momentum frame, hence the boosted nuclei are not infinitely thin. Instead, the nuclei have a finite extension in the longitudinal direction and therefore depend on the longitudinal coordinate. In this fully three dimensional framework an x dependence of the gluon distribution function emerges naturally. In order to fix the parameters of the model, we calculate the gluon distribution function and compare it with the JR09 parametrization of the data. We explore the parameter space of the model to attain a working framework that can be used to calculate the initial conditions in heavy ion collisions.

## Key claim

The 3dMVn model's gluon distribution function, when evolved via DGLAP to higher Q², best fits the JR09 parametrization at Q² = 100 GeV² with α_s = 0.5 and λ = 1.8 fm, but only within the restricted x-range 0.015 < x < 0.04 where the model is valid; the fit discrepancy is only 2% in that region.

## Method

The author calculated the gluon distribution function xg_A(x,Q²) for a gold nucleus (A=197) from the 3dMVn model at Q² = 0.55 GeV² for a grid of α_s (0.1 to 1.0) and λ (0.2 to 2.6 fm) values. The resulting gluon distributions were then evolved to higher Q² using the DGLAP equation at NNLO(MS-bar) via the QCDNUM17 code, with valence and sea quark inputs taken from the JR09 parametrization at the initial scale. The evolved gluon distributions were compared to the JR09 parametrization at Q² = 100 GeV² in the x-range 0.015 < x < 0.04 to find the best-fit parameters.

## Key numbers

- Best-fit parameters: α_s = 0.5 and λ = 1.8 fm.
- Discrepancy between best-fit 3dMVn model and JR09 at Q² = 100 GeV² in the fit region (0.015 < x < 0.04) is only 2%.
- Model validity range for gold: 0.035 < x < 0.060 at Q² = 0.55 GeV²; lower x limit is x ~ A^{-1/3}/(m R_0) ≈ 0.035 for gold.
- Upper x limit from weak coupling: x ≲ 0.25.
- Q² validity range: 0.16 x ≲ Q² ≲ 0.032/x (for R_0 = 1 fm).
- Correlation length λ is of order nucleon size ~ 1 fm; best-fit λ = 1.8 fm.
- Nuclear size correction estimate: R_0/R ~ A^{-1/3} ~ 0.17 for gold, implying corrections up to 17%.

## Context

This work extends the McLerran-Venugopalan model by incorporating the color-neutral 3D version of Lam and Mahlon (1999, 2000), which implements confinement via colored noise and includes longitudinal dependence, yielding an x-dependent gluon distribution. The paper fixes the model's free parameters (α_s and λ) by direct comparison with the JR09 parametrization (Jimenez-Delgado and Reya, 2009), enabling future calculations of initial energy density distributions in heavy ion collisions. The approach contrasts with event-by-event Monte Carlo methods (Schenke et al., 2010-2012) and provides an analytical framework for CGC practitioners.
