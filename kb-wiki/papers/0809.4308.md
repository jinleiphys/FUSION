# New method for extracting neutron structure functions from nuclear data

> We propose a new method for extracting neutron structure functions from inclusive structure functions of nuclei. Unlike earlier approaches, the new method is applicable to both spin-averaged and spin-dependent structure functions. We test the reliability of the method on unpolarized F_2 and polarized g_1 structure functions of the deuteron in both the nucleon resonance and deep inelastic regions. The new method is able to reproduce known input functions of almost arbitrary shape to very good accuracy with only several iterations.

## Key claim

The paper proposes a new "additive" iterative method for extracting neutron structure functions from nuclear data that, unlike the standard multiplicative smearing-factor method, remains well-defined even when the structure function changes sign (as occurs for spin-dependent polarized structure functions), and can recover resonance-region structure that has been smeared by nucleon Fermi motion.

## Method

The method treats the convolution equation relating the nuclear structure function to the nucleon structure function by splitting the smearing function into a delta-function peak plus a finite-width correction, then solving iteratively for the neutron structure function. For the spin-dependent case, the coupled equations for \(g_1\) and \(g_2\) are solved sequentially, using the diagonal \(f_{11}\) distribution as the dominant term and iterating the off-diagonal contributions. The procedure is tested by constructing simulated deuteron "data" from known input proton and neutron structure functions (using the MRST parameterization at \(Q^2 = 10\) GeV\(^2\) for deep inelastic scattering and the MAID Unitary Isobar Model at \(Q^2 = 1\) GeV\(^2\) for the resonance region), then applying the extraction and comparing with the known input.

## Key numbers

* At \(Q^2 = 10\) GeV\(^2\) in the deep inelastic region, starting from an initial guess \(F_2^{n(0)} = 0\), the extracted \(F_2^n\) is almost indistinguishable from the input after just a single iteration.
* In the resonance region at \(Q^2 = 1\) GeV\(^2\), after 5 iterations the extracted \(F_2^n\) is very close to the true result; after 30 iterations the complete resonance structures are reproduced almost exactly.
* The deuteron D-state probability for the Paris wave function used is \(\omega_d = 5.8\%\).
* The deuteron binding energy is \(\varepsilon_d = -2.2\) MeV.
* The differences between the unpolarized distribution \(f_0\) and the polarized diagonal distribution \(f_{11}\) are approximately \(0.5\%\) at the peak for \(\gamma = 1\) and \(\lesssim 2.5\%\) for \(\gamma = 2\).

## Context

This work extends the smearing-factor method of Bodek (1979) and Whitlow (1992), which was limited to positive-definite unpolarized structure functions in deep inelastic kinematics, by providing an additive iterative approach that works for spin-dependent structure functions with zeros and for the nucleon resonance region. The method is tested within the convolution framework using deuteron smearing functions from Kulagin and Melnitchouk (2007) and enables future extraction of neutron resonance structure from deuterium data, which has been absent in tests of quark-hadron duality for the neutron.
