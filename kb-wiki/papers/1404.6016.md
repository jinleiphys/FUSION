# Approximation of reduced width amplitude and application to cluster decay width

> We propose a simple method to approximately evaluate reduced width amplitude (RWA) of a two-body spinless cluster channel using the norm overlap with the Brink-Bloch cluster wave function at the channel radius. The applicability of the present approximation is tested for the $^{16}$O+$α$ channel in $^{20}$Ne as well as the $α$+$α$ channel in $^8$Be. The approximation is found to be reasonable to evaluate the RWA for states near the threshold energy and it is useful to estimate the $α$-decay width of resonance states. The approximation is also applied to $^9$Li, and the partial decay width of the $^6$He($0^+_1$)+$t$ channel is discussed.

## Key claim

The authors propose that the reduced width amplitude (RWA) for a two-body cluster channel can be approximated using the norm overlap of the total wave function with a Brink-Bloch cluster wave function at the channel radius, provided the channel radius is large enough that antisymmetrization effects between clusters are negligible. The approximation is stated to be "reasonable" for states near the threshold energy, but the authors explicitly note it is "not valid in the small a region with the strong antisymmetrization effect."

## Method

The authors calculate the norm overlap between the total A-body wave function and a spin-parity projected Brink-Bloch cluster wave function localized at inter-cluster distance S_k = a. They then convert this overlap into an approximate RWA using a Gaussian approximation for the reference inter-cluster wave function, assuming the true RWA varies slowly compared to the localized reference function. The approximation is tested by comparing with exact RWA values from cluster-GCM calculations for the 16O+α channel in 20Ne and the α+α channel in 8Be, and is also applied to AMD wave functions for 20Ne and to 9Li for the 6He(0+1)+t channel.

## Key numbers

- For the 20Ne cluster-GCM calculation: width parameter ν = 0.16 fm−2 used for both 16O and α clusters; ten Brink-Bloch wave functions with 16O-α distance S_k = 1, 2, ..., 10 fm adopted as basis
- For the AMD(VAP) calculation of 20Ne: Volkov No.2 with m = 0.66 supplemented by G3RS spin-orbit force with strength u_I = −u_II = 2400 MeV
- For the cluster-GCM calculation of 20Ne: Volkov No.2 with m = 0.62

## Context

This work extends the traditional R-matrix approach for estimating cluster decay widths, which has been applied in conventional cluster models such as the resonating group method (RGM) and generator coordinate method (GCM) for systems like α+α in 8Be and 16O+α in 20Ne (Ikeda 1977, Matsuse 1975). The proposed approximation addresses the difficulty of extracting RWA for exotic clusters (such as t, 6He, 10Be, 14C) that appear in neutron-rich nuclei, where conventional cluster models assuming simple shell-closed clusters are no longer applicable. The method enables estimation of cluster decay widths from general A-body wave functions obtained from frameworks like antisymmetrized molecular dynamics (AMD), which can describe cluster formation, breaking, and non-cluster components without assuming specific clusters a priori.
