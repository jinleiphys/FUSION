# Threshold pi^0 photo- and electro-production in a meson-exchange model

> We show that, within a meson-exchange dynamical model describing well most of the existing pion electromagnetic production data up to the second resonance region, one is also able to obtain a good agreement with the pi^0 photo- and electroproduction data near threshold. In the case of pi^0 production, the effects of final state interaction in the threshold region are nearly saturated by single charge exchange rescattering. This indicates that in ChPT, it might be sufficient to carry out the calculation just up to one-loop diagrams for threshold neutral pion production.

## Key claim

Within a meson-exchange dynamical model (the Dubna-Mainz-Taipei or DMT model) that already describes pion electromagnetic production data up to the second resonance region, good agreement with threshold pi0 photo- and electroproduction data is also obtained; the key qualifier is that final state interaction effects in the threshold region are nearly saturated by single charge exchange rescattering, implying that one-loop calculations in chiral perturbation theory (ChPT) may be sufficient for threshold neutral pion production.

## Method

The authors use the DMT dynamical model, which starts from an effective chiral Lagrangian and constructs a gamma-pi transition potential v_{gamma pi} containing Born terms plus rho- and omega-exchange in the t-channel, with resonance contributions treated via standard Breit-Wigner forms. The t-matrix for electromagnetic pion production is solved via a scattering equation t_{gamma pi} = v_{gamma pi} + v_{gamma pi} g_0 t_{pi N}, where t_{pi N} is obtained from a meson-exchange pi N model in the Bethe-Salpeter formalism. For pi0 photoproduction, the multipole E_{0+} near threshold is calculated by solving coupled channels equations with physical pion and nucleon masses, using a dipole-like off-shell form factor with Lambda = 440 MeV to ensure convergence of the principal value integral.

## Key numbers

- For gamma p -> pi0 p, the full DMT model gives Re E_{0+} = -1.00 x 10^{-3}/m_pi at threshold, compared to the experimental value of -1.33 +/- 0.11 x 10^{-3}/m_pi and the ChPT value of -1.1 x 10^{-3}/m_pi.
- The tree-level contribution to E_{0+} for gamma p -> pi0 p is -2.26 x 10^{-3}/m_pi, while the 1-loop contribution gives -1.06 x 10^{-3}/m_pi and the 2-loop contribution gives -1.01 x 10^{-3}/m_pi.
- For gamma p -> pi+ n, the full DMT model gives E_{0+} = 28.85 x 10^{-3}/m_pi, compared to the experimental value of 28.3 +/- 0.3 x 10^{-3}/m_pi and ChPT value of 28.2 +/- 0.6 x 10^{-3}/m_pi.
- For gamma n -> pi0 n, the full DMT model gives E_{0+} = 2.18 x 10^{-3}/m_pi, compared to the ChPT value of 2.13 x 10^{-3}/m_pi.
- For gamma n -> pi- p, the full DMT model gives E_{0+} = -33.31 x 10^{-3}/m_pi, compared to the experimental value of -31.8 +/- 1.9 x 10^{-3}/m_pi and ChPT value of -32.7 +/- 0.6 x 10^{-3}/m_pi.
- Replacing t_{pi N} by v_{pi N} (i.e., keeping only 1-loop rescattering) changes the full calculation result by only 5%.
- For electroproduction at Q^2 = 0.1 (GeV/c)^2, the DMT model predictions for E_{0+} and L_{0+} multipoles are compared with data from NIKHEF and Mainz.
- There is a 15% - 20% difference between DMT and ChPT predictions for the P-wave combination P_3 = 2M_{1+} + M_{1-}.

## Context

This work extends the Dubna-Mainz-Taipei (DMT) dynamical model, previously developed for pion electromagnetic production up to the second resonance region (Kamalov 1999, Sabit 2001), to the threshold region for pi0 photo- and electroproduction. It directly addresses the long-standing discrepancy between the old low-energy theorem prediction of E_{0+} = -2.4 x 10^{-3}/m_pi and experimental measurements (Beck 1990, Bergstrom) that yielded E_{0+} ~ -1.3 x 10^{-3}/m_pi, which was resolved by ChPT calculations (Bernard 1991) showing loop corrections give non-analytic terms in m_pi. The finding that single charge exchange rescattering nearly saturates final state interaction effects supports the reliability of one-loop ChPT calculations (Bernard 1996) for threshold neutral pion production, and the model enables extraction of S-wave multipoles from electroproduction data using DMT P-wave predictions rather than ChPT predictions, yielding better agreement with NIKHEF data.
