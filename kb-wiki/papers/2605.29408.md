# Large language model for unified and accurate description of multidimensional nuclear properties

> A prior-informed large language model (LLM) driven multi-task learning framework is proposed for the unified description of multiple nuclear observables. By fine-tuning the pre-trained DeepSeek-R1-1.5B model with Low-Rank Adaptation (LoRA), lightweight adapters are introduced while preserving general pre-trained parameters. Under a causal language modeling paradigm, the model is trained autoregressively on deviations between experimental and theoretical values. Significant accuracy improvements are achieved across seven observables, including charge radii, masses, binding energies, separation energies, and decay energies, with the training loss decreasing by over 98% across all tasks. This demonstrates that the LLM-based framework, through structured prior embedding, offers an efficient and shared approach for multi-task regression in fundamental nuclear properties.

## Key claim
A large language model (LLM) fine-tuned with Low-Rank Adaptation (LoRA) can simultaneously predict multiple nuclear observables with high accuracy, but only when the target physical quantities have smooth data distributions or when explicit prior information (magic numbers, shell effects) is added as input features; the model struggles with quantities like separation energies that amplify local fluctuations.

## Method
The authors fine-tuned the pre-trained DeepSeek-R1-1.5B model using LoRA, keeping the original weights frozen and training only low-rank adapter matrices. Under a causal language modeling paradigm, the model was trained autoregressively on the residuals between experimental values and theoretical predictions from the WS4 mass model, Random Forest, and LightGBM. Input features were formatted as structured text sequences containing neutron number N, proton number Z, and optionally additional physical features (M_p, M_n, δ), with the model predicting multiple output quantities simultaneously.

## Key numbers
- Single-task charge radius prediction (Type1, five features, Z ≥ 8 region): testing set RMSE of 0.0195 fm
- Single-task charge radius prediction (Type2, only N and Z, Z ≥ 8 region): testing set RMSE of 0.0204 fm
- Multi-task prediction (Type4, N and Z only, targets Q_α, Q_β, mass, BE): testing set MAE for Q_α and Q_β approximately 0.09 MeV, RMSE within 0.13 MeV; mass MAE 0.16 MeV, RMSE 0.20 MeV
- Multi-task prediction (Type5, five features, targets Q_α, Q_β, BE, S_p, S_n): testing set RMSE for Q_α and Q_β approximately 0.105 MeV; BE RMSE 0.169 MeV; S_p and S_n RMSE around 0.210 MeV
- Training loss decreased by over 98% across all multi-task objectives, and over 99% for the single-objective task

## Context
This work extends prior machine learning approaches to nuclear properties, including tree models, Bayesian analysis, neural networks, and kernel ridge regression (e.g., G15, Z16, X14, Z25), which achieved MAE of 0.1–0.3 MeV for single observables but struggled with multi-task unification. The authors specifically compare against multi-objective optimization strategies (Y09, R08, X03, X04, W05, C06, J07) and prior charge radius predictions (Y01, T02). The framework enables simultaneous high-precision prediction of multiple nuclear observables using a single model, though the authors note that more complex architectures reduce interpretability and increase computational cost.
