# Solution of the Skyrme-Hartree-Fock-Bogolyubov equations in the Cartesian deformed harmonic-oscillator basis. (VII) HFODD (v2.49t): a new version of the program

> We describe the new version (v2.49t) of the code HFODD which solves the nuclear Skyrme Hartree-Fock (HF) or Skyrme Hartree-Fock-Bogolyubov (HFB) problem by using the Cartesian deformed harmonic-oscillator basis. In the new version, we have implemented the following physics features: (i) the isospin mixing and projection, (ii) the finite temperature formalism for the HFB and HF+BCS methods, (iii) the Lipkin translational energy correction method, (iv) the calculation of the shell correction. A number of specific numerical methods have also been implemented in order to deal with large-scale multi-constraint calculations and hardware limitations: (i) the two-basis method for the HFB method, (ii) the Augmented Lagrangian Method (ALM) for multi-constraint calculations, (iii) the linear constraint method based on the approximation of the RPA matrix for multi-constraint calculations, (iv) an interface with the axial and parity-conserving Skyrme-HFB code HFBTHO, (v) the mixing of the HF or HFB matrix elements instead of the HF fields. Special care has been paid to using the code on massively parallel leadership class computers. For this purpose, the following features are now available with this version: (i) the Message Passing Interface (MPI) framework, (ii) scalable input data routines, (iii) multi-threading via OpenMP pragmas, (iv) parallel diagonalization of the HFB matrix in the simplex breaking case using the ScaLAPACK library. Finally, several little significant errors of the previous published version were corrected.

## Key claim

The paper presents version v2.49t of the HFODD code, which extends the previous version by adding isospin mixing and projection, finite-temperature formalism, Lipkin translational energy correction, and shell correction capabilities, while also implementing hybrid MPI/OpenMP parallelism for leadership-class computers; the abstract omits that the isospin projection requires a subsequent rediagonalization of the Hamiltonian in the good-isospin basis to obtain physical isospin mixing, and that the finite-temperature HFB equations are formally equivalent to the T=0 equations but with generalized density matrix and pairing tensor expressions.

## Method

The code solves the nuclear Skyrme Hartree-Fock (HF) or Skyrme Hartree-Fock-Bogolyubov (HFB) problem by expanding single-particle or single-quasiparticle wave functions in a Cartesian deformed harmonic-oscillator basis and iteratively diagonalizing the mean-field Hamiltonians or Routhians that depend non-linearly on local neutron and proton densities. New physics features implemented include: (i) one-dimensional isospin projection using the operator \(\hat{P}^T_{T_z T_z}\) followed by rediagonalization of the total Hamiltonian (including isoscalar kinetic energy, isospin-invariant Skyrme functional, and Coulomb force decomposed into isoscalar, isovector, and isotensor components) in the good-isospin basis, with an optional four-dimensional isospin and angular-momentum projection; (ii) finite-temperature formalism for HFB and HF+BCS methods, where the density matrix and pairing tensor are generalized using the Fermi function \(f_\mu = 1/(1+e^{\beta E_\mu})\); (iii) the Lipkin translational energy correction method using the operator \(\hat{K}=k\mathbf{\hat{P}}^2\); and (iv) calculation of the shell correction. Numerical methods added include the two-basis method for HFB, the Augmented Lagrangian Method for multi-constraint calculations, and parallelization via MPI, OpenMP, and ScaLAPACK.

## Key numbers

- The isospin projection uses a parameter \(\varepsilon_T = 10^{-10}\) to retain only states with \(|b_{TT_z}|^2 \geq \varepsilon_T\), which sets the limit of \(T \leq |T_z|+5\).
- The code contains 104,666 lines of which 47,059 are comments and separators.
- The code requires approximately 10 Mwords of memory to execute with typical data.
- The isospin impurity \(\alpha_C^n\) for the \(n\)-th eigenstate is defined as \(\alpha_C^n = 1 - |a^n_{TT_z}|_{\text{max}}^2\) and is given in percents.
- The Fermi matrix element for superallowed beta decay is given by \(|\langle I=0, T\approx 1, T_z = \pm 1 | \hat{T}_{\pm} | I=0, T\approx 1, T_z = 0 \rangle|^2 \equiv 2(1-\delta_C)\), where \(\delta_C\) is the isospin-breaking correction.

## Context

This work extends the series of HFODD code versions previously published by Dobaczewski et al. (2009, v2.40h) and earlier versions (v1.60r, v1.75r, v2.08i, v2.08k). The new version enables systematic studies of isospin mixing in N=Z nuclei and superallowed beta decay (as discussed by Satula et al., 2010 and 2011), finite-temperature studies of fission paths and barriers in actinide and superheavy elements (Pei et al., 2009; Sheikh et al., 2009), and large-scale calculations on massively parallel computers. The code provides an interface with the axial and parity-conserving Skyrme-HFB code HFBTHO.
