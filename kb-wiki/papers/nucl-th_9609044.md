# The Problem of Matter Stability in the Nambu--Jona-Lasinio Model

> We reinvestigate the conditions for stable matter solutions in the Nambu--Jona-Lasinio (NJL) model. In mean field approximation the NJL model can be regarded as an extension of the Walecka mean field model to include negative energy fermion states. While this extension is necessary to allow for a chiral phase transition, it was found some time ago that at the same time it destroys the wanted saturation properties of the Walecka model. We reformulate this problem in terms of the thermodynamic potential and find that there is indeed a connection between these two features. We show that the minimum of the thermodynamic potential which corresponds to stable nuclear matter in the Walecka model is shifted from a finite to zero effective fermion mass in the chiral NJL model. This shift is closely related to the chiral phase transition. Under certain conditions the shifted minima may still lead to stable matter solutions but only in the chirally restored phase. We discuss a possible interpretation of these solutions as a schematic bag model description.

## Key claim

The NJL model in mean field approximation does not allow stable matter in the chirally broken phase (massive fermions), but under certain conditions (sufficiently large bag constant and weak vector coupling) it does allow stable matter in the chirally restored phase (massless fermions), which can be interpreted as a schematic bag model description.

## Method

The author reformulates the stability problem in terms of the thermodynamic potential as a function of the effective fermion mass \(m\) and the renormalized chemical potential \(\mu_R\), rather than using the gap equation alone. For a given chemical potential \(\mu\), the self-consistent solutions correspond to minima of the thermodynamic potential; stable matter requires an additional minimum at a finite effective mass with zero pressure and chemical potential below the vacuum mass, analogous to a first-order gas-liquid phase transition. The analysis is performed at zero temperature for the NJL model with scalar and vector-isoscalar interactions, using a sharp three-momentum cutoff for regularization, and compared to the Walecka mean field model.

## Key numbers

*   Nuclear matter saturation density: \(\rho_o = 0.17\ \text{fm}^{-3}\) (used as reference).
*   For parameter set 2 (constituent quark mass \(m_{vac} = 400\ \text{MeV}\)) with \(G_V = 0\): stable quark matter appears at critical chemical potential \(\mu^* = 378.5\ \text{MeV}\), corresponding to baryon number density \(2.8\ \rho_o\).
*   For parameter set 2 with \(G_V = 0\): the vacuum thermodynamic potential minimum is at \(m = 400\ \text{MeV}\); the \(m = 0\) maximum turns into a minimum at \(\mu = 344.4\ \text{MeV}\).
*   For parameter set 2 with \(G_V = 0\): the original minimum at finite mass disappears at \(\mu = 409.3\ \text{MeV}\).
*   For parameter set 2: stable matter exists only for \(G_V < 0.3\ G_S\); for \(0.3\ G_S < G_V < 0.8\ G_S\) there is a first-order phase transition with finite transition density but no stable matter; for \(G_V > 0.8\ G_S\) the phase transition is second order.
*   For parameter set 3: stable matter exists for \(G_V < 0.6\ G_S\); no stable matter for \(G_V > 0.6\ G_S\).
*   Minimal bag constant for stable matter solutions: \(B \gsim (175\ \text{MeV})^4 \simeq 125\ \text{MeV}\ \text{fm}^{-3}\), about twice the MIT bag constant.
*   For parameter set 2 with \(G_V = 0\) and a small current mass \(m_o = 5.5\ \text{MeV}\): the effective mass inside the droplet \(m^*\) is about \(40\ \text{MeV}\).
*   For parameter set 3 with \(G_V = 0\) and \(m_o = 5.5\ \text{MeV}\): \(m^*\) is about \(20\ \text{MeV}\).
*   The \(\pi N\) sigma term used for comparison: \(\sigma_{\pi N} = 45\ \text{MeV}\).

## Context

This work extends and clarifies the earlier finding by Koch et al. (1987) that the NJL model, unlike the Walecka model, lacks saturation properties for stable matter. The author shows that the Walecka model's saturation arises from an explicitly broken symmetry (the \((m-m_N)^2\) term), and that in the chirally symmetric NJL model the corresponding minimum is shifted to \(m=0\), linking the loss of saturation to the chiral phase transition. The results enable a schematic connection to bag models (MIT bag model by DeGrand et al., 1975) and to the Guichon model (1988, 1994) where interacting MIT bags are used, suggesting a path toward a unified description of hadronic matter and the chiral phase transition.
