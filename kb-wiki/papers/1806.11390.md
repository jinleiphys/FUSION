# Single string structure and multiple string interaction effect on strange particle production in pp collisions at $\sqrt{s}=7$ TeV

> We present a systematic study on the strange particle production at the Large Hadron Collider (LHC) in proton-proton (pp) collisions at $\sqrt{s}=$ 7 TeV based on PACIAE simulations. Two different mechanisms accounting for single string structure variations and multiple string interactions are implemented in the simulations. These modifications give rise to increased effective string tension in the Lund fragmentation model and generate more strange particles in the hadronic final state. By comparing the results with a wealth of the LHC data, it is turned out that the inclusion of variable effective string tension is capable to reach an improved agreement between theory and experiment, especially on the recently observed multiplicity dependence of strangeness enhancement in pp collisions. This approach provides us a new method to understand the microscopic picture of the novel high multiplicity pp events collected at the LHC in the string fragmentation framework.

## Key claim

The paper demonstrates that the multiplicity-dependent strangeness enhancement observed in pp collisions at $\sqrt{s}=7$ TeV can be reproduced within the Lund string fragmentation framework by introducing an effective string tension that varies either string-by-string (from gluon wrinkling) or event-by-event (from multiple string interactions), or both combined, without requiring quark-gluon plasma formation.

## Method

The authors implement two phenomenological modifications to the string tension in the PACIAE Monte Carlo event generator (based on PYTHIA 6.4): a string-wise effective tension $\kappa_{eff}^{s}=\kappa_0(1-\xi)^{-\alpha}$ accounting for gluon wrinkling on individual strings, and an event-wise effective tension $\kappa_{eff}^{m}=\kappa_0(1+\frac{n_{MPI}-1}{1+p^{2}_{T\ ref}/p^{2}_0})^{r}$ accounting for multiple parton interaction density. These modify the Lund fragmentation parameters (strange-to-light quark ratio, diquark suppression, spin ratio, and transverse momentum width) via a scaling relation derived from the Schwinger tunneling probability. Simulations are compared to ALICE data on strange particle yields ($K_S^0$, $\Lambda$, $\Xi$, $\Omega$) as functions of event multiplicity in ten event classes.

## Key numbers

- $\sqrt{s}=7$ TeV: collision energy for pp collisions studied
- $\kappa_0 \approx 1$ GeV/fm: baseline pure quark-antiquark string tension
- $\alpha=3.5$: tuned parameter for single string structure effective tension
- $r=0.2$: tuned exponent for multiple string interaction effective tension
- $\beta=0.05$: fixed parameter for diquark from two-step quark pair production
- $K=0.82$: factor introduced in PACIAE for hard scattering cross sections
- Reference fragmentation parameters from Perugia2011 tune: PARJ(1)=0.087, PARJ(2)=0.19, PARJ(3)=0.95, PARJ(4)=0.043, PARJ(21)=0.33
- In event class I (0-0.95% most central), $\langle dN_{ch}/d\eta\rangle_{|\eta|<0.5}$ = 21.3$\pm$0.6 (data), 24.1 ($\kappa_0$), 22.1 ($\kappa_{eff}^{m}$), 21.5 ($\kappa_{eff}^{s}$), 19.8 ($\kappa_{eff}^{s+m}$)
- $K_S^0$ yield increases by a factor of 50% between $\kappa_0$ and $\kappa_{eff}^{s+m}$ scenarios in highest multiplicity events

## Context

This work extends the Lund string fragmentation approach to explain the multiplicity-dependent strangeness enhancement recently observed by the ALICE collaboration (2017) in pp collisions, a phenomenon previously associated with quark-gluon plasma formation in heavy-ion collisions. It builds on the color ropes mechanism implemented in DIPSY (Bierlich, 2014) and the thermodynamic string model in PYTHIA8 (Fischer, 2016), while introducing a simpler parameterization that separates single-string structure effects from inter-string interactions. The approach enables systematic study of how string tension variations affect strange particle production across different multiplicity classes, providing a microscopic picture within the string fragmentation framework without invoking deconfinement.
