# Shared Data and Algorithms for Deep Learning in Fundamental Physics

> We introduce a Python package that provides simply and unified access to a collection of datasets from fundamental physics research - including particle physics, astroparticle physics, and hadron- and nuclear physics - for supervised machine learning studies. The datasets contain hadronic top quarks, cosmic-ray induced air showers, phase transitions in hadronic matter, and generator-level histories. While public datasets from multiple fundamental physics disciplines already exist, the common interface and provided reference models simplify future work on cross-disciplinary machine learning and transfer learning in fundamental physics. We discuss the design and structure and line out how additional datasets can be submitted for inclusion. As showcase application, we present a simple yet flexible graph-based neural network architecture that can easily be applied to a wide range of supervised learning tasks. We show that our approach reaches performance close to dedicated methods on all datasets. To simplify adaptation for various problems, we provide easy-to-follow instructions on how graph-based representations of data structures, relevant for fundamental physics, can be constructed and provide code implementations for several of them. Implementations are also provided for our proposed method and all reference algorithms.

## Key claim

The paper introduces a unified Python package (pd4ml) that provides consistent access to five supervised learning datasets from particle physics, astroparticle physics, and nuclear physics, and demonstrates that a single graph-based neural network architecture can achieve performance close to dedicated methods across all these diverse physics tasks.

## Method

The authors assembled five datasets covering hadronic top quark tagging, Belle II event filtering, spinodal decomposition classification in heavy-ion collisions, equation-of-state classification in QCD matter, and air-shower maximum regression. They implemented a common Python interface for downloading, preprocessing, and loading these datasets, including automatic construction of graph representations. As a showcase, they trained a single fully-connected network and a single graph neural network architecture on all datasets and compared performance against dataset-specific reference models.

## Key numbers

- Top Tagging Landscape: 1.2M training examples, 400k test, 400k validation; 200 particles per event with 4 features per particle
- Smart Backgrounds: 157k training examples, 39k test, 84k validation; 100 particles per event with 9 features per particle; background retention rate of 5%
- Spinodal or Not: 16.3k training examples, 4k test, 8.7k validation; 20x20 pixel histograms of net baryon density
- EoS: 121k training examples, 25k test, 54k validation; 24x24 pixel histograms of pion spectra with 24 transverse momentum bins and 24 azimuthal angle bins
- Air Showers: 56k training examples, 30k test, 14k validation; 81 detector stations with 80 signal bins plus timing per station; time step width of 25 ns; detector spacing of 1500 m; observatory height of 1400 m above sea level

## Context

This work extends prior domain-specific benchmark datasets such as the Top Tagging Landscape (Kasieczka 2019) and the Smart Backgrounds classification (James 2019, CHEP2019) by unifying them under a single interface. It enables cross-disciplinary machine learning and transfer learning studies in fundamental physics, where previously groups had to develop separate data handling pipelines. The graph-based approach builds on successful applications of graph neural networks in particle physics (Qu 2019, Moreno 2019, Qasim 2019) and provides a flexible representation that can be applied to the diverse data structures encountered across physics domains.
