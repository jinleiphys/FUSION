# Solving for the Particle-Number-Projected HFB Wavefunction

> Recently we proposed a particle-number-conserving theory for nuclear pairing [Jia, Phys. Rev. C 88, 044303 (2013)] through the generalized density matrix formalism. The relevant equations were solved for the case when each single-particle level has a distinct set of quantum numbers and could only pair with its time-reversed partner (BCS-type Hamiltonian). In this work we consider the more general situation when several single-particle levels could have the same set of quantum numbers and pairing among these levels is allowed (HFB-type Hamiltonian). The pair condensate wavefunction (the HFB wavefunction projected onto good particle number) is determined by the equations of motion for density matrix operators instead of the variation principle. The theory is tested in the simple two-level model with factorizable pairing interactions and the semi-realistic model with the zero-range delta interaction.

# Solving for the Particle-Number-Projected HFB Wavefunction

## Key claim
The particle-number-conserving pairing theory based on the generalized density matrix (GDM) formalism, previously solved for BCS-type Hamiltonians, is extended to HFB-type Hamiltonians where multiple single-particle levels share the same quantum numbers and pairing among them is allowed. The pair condensate wavefunction is determined by equations of motion for density matrix operators rather than the variation principle, and the method is tested in simple and semi-realistic models.

## Method
The theory assumes the ground state of a 2N-particle system is an N-pair condensate, and introduces a unitary transformation to a canonical basis that diagonalizes the pair structure matrix. The equations of motion for the density matrix are derived using the approximation that the two-body density matrix factorizes into products of one-body density matrices in both particle-hole and particle-particle channels. The resulting equations (Eqs. 12 and 11) provide constraints that fix the pair structure matrix parameters, with the number of constraints equaling the number of independent parameters in the Hermitian block-diagonal pair-structure matrix.

## Key numbers
- Root-mean-square deviations in the two-level model (360 examples): σ_ραα = σ_ρββ = 0.0125, σ_ραβ = σ_ρβα = 0.0198, σ_καα̃ = 0.0183, σ_κββ̃ = 0.0214, σ_καβ̃ = σ_κβα̃ = 0.0384
- Average ground state energy error in the two-level model: Ē_GDM = 0.0198 (compared to average pairing correlation energy Ē_pair = 1.78)
- Single-particle energies in the delta interaction model: ε_2s1/2 = 1.16 MeV, ε_1d3/2 = 2.11 MeV, ε_1d5/2 = 0.11 MeV, ε_3s1/2 = 12.98 MeV, ε_2d5/2 = 10.37 MeV
- Pairing strength λ values chosen so that mean pairing matrix elements are realistic, with units MeV·b³

## Context
This work extends the author's previous particle-number-conserving pairing theory (Jia, Phys. Rev. C 88, 044303, 2013) from BCS-type Hamiltonians to the more general HFB-type case. The method offers a computationally cheaper alternative to the variation-after-particle-number-projection (VAPNP) approach used by Dietrich, Flocard, Sheikh, Robledo, and others, as it requires only one-body density matrices rather than two-body density matrices. The theory is tested in a two-level model with factorizable pairing and a semi-realistic model with zero-range delta interaction, and is stated to be ready for application to realistic nuclear systems.
