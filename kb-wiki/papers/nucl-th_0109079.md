# Calculation of the number of partitions with constraints on the fragment size

## Key claim
The paper introduces recursive relations that allow exact calculation of the number of integer partitions under constraints on minimum and/or maximum fragment size, but these relations are derived for the combinatorial enumeration itself rather than for any specific physical model; the practical utility is demonstrated only by reference to future work applying these counts to the Moretto charge correlation in the minimal information model.

## Method
The author derives recursive relations for the number of partitions of an integer S into M fragments (multiplicity M) with constraints on fragment size, using Ferrers diagram arguments and the concept of complementary partitions. For a maximum size constraint s_max, a three-term recurrence is given (Eqs. 4-5); for a minimum size constraint s_min, the count reduces to unconstrained partitions of S - M*s_min (Eq. 11); for simultaneous minimum and maximum constraints, the count is expressed as a sum over singly-constrained partitions (Eqs. 16-17). The relations are demonstrated graphically using Ferrers diagrams with filled and open dots.

## Key numbers
- The total number of unconstrained partitions of integer S is approximated by the Ramanujan-Hardy formula: N(S) ≈ exp(π√(2S/3)) / (4S√3).
- For partitions with minimum size s_min, the number is given by _{s_min}N(S,M) = N(S - M(s_min - 1), M), where N is the unconstrained partition number.
- For partitions with both minimum and maximum size constraints, the number satisfies _{s_min}^{s_max}N(S,M) = ^{s_max - s_min + 1}N(S - M(s_min + 1), M) and also = ^{s_max - s_min + 1}N(M(s_max + 1) - S, M).

## Context
This work extends the combinatorial techniques used in nuclear multifragmentation models, particularly the minimal information model of Engel (1984), Engel (1991), and Botvina (2000), where all partitions of a given sum are equally probable. The constrained partition numbers are directly applicable to the combinatorial procedure introduced by Moretto and collaborators (1996, 1997) for isolating rare events corresponding to fragmentation into nearly equal-size intermediate mass fragments (IMFs) supplemented by light fragments. The paper enables explicit calculation of the Moretto charge correlation in the minimal information model, which the author states will be presented in a forthcoming article (Desesquelles, 2001).
