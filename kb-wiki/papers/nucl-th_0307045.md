# Dressed vertices

> The response of a correlated nuclear system to an external field is discussed. The Bethe-Salpeter equation for the dressed vertex is solved. The kernel of the integral equation for the vertex is chosen consistently with the approximation for the self-energy. This guarantees the fulfillment of the f-sum rule for the response function.

# Dressed vertices

## Key claim

The paper presents the first solution of the Bethe-Salpeter equation for the dressed vertex in a nuclear system for general momentum and energy of an external field coupled to density, using the real-time Green's function formalism. The key condition is that the kernel of the integral equation for the vertex must be chosen consistently with the approximation for the self-energy, which guarantees fulfillment of the f-sum rule for the response function.

## Method

The author solves the self-consistent second Born approximation for the self-energy and dressed propagators in equilibrium nuclear matter, using an effective interaction. The Bethe-Salpeter equation for the dressed vertex is then solved iteratively for each given momentum and energy of the external field, using the equilibrium Green's functions. The particle-hole irreducible kernel is taken as the functional derivative of the self-energy with respect to the dressed Green's function, ensuring consistency between the self-energy and vertex approximations.

## Key numbers

- Density of the system: 0.32 fm^-3
- Temperature: 5 MeV
- Single-particle width: approximately 20-30 MeV
- At momentum transfer q = 220 MeV, the naive polarization loop with dressed propagators violates the f-sum rule by a factor of 4.9
- The numerical method is accurate for q > 100 MeV, where both the f-sum rule and the dispersion relation are satisfied

## Context

This work extends the self-consistent Green's function approach for nuclear systems, which previously included iterated second Born approximation (Danielewicz, 1984) and in-medium T-matrix calculations (Bozek, 2002; Dewulf, 2003; Frick, 2003). The paper addresses the known problem that the density response function for the electron gas in the self-consistent GW approximation violates the f-sum rule (Schindlmayr, 1998; Henning, 1999). The method enables calculation of realistic response functions with different vertices and can be used to test approximations for dressed vertices or particle-hole interactions for more complicated T-matrix self-energies in nuclear matter.
