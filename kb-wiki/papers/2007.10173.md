# Coulomb Corrections for Bose-Einstein Correlations from One- And Three-Dimensional Lévy-Type Source Functions

> In the study of femtoscopic correlations in high-energy physics, besides Bose--Einstein correlations, one has to take final-state interactions into account. Amongst them, Coulomb interactions play a prominent role in the case of charged particles. Recent measurements have shown that in heavy-ion collisions, Bose--Einstein correlations can be best described by Lévy-type sources instead of the more common Gaussian assumption. Furthermore, three-dimensional measurements have indicated that, depending on the choice of frame, a deviation from spherical symmetry observed under the assumption of Gaussian source functions persists in the case of Lévy-type sources. To clarify such three-dimensional Lévy-type correlation measurements, it is thus important to study the effect of Coulomb interactions in the case of non-spherical Lévy sources. We calculated the Coulomb correction factor numerically in the case of such a source function for assorted kinematic domains and parameter values using the Metropolis--Hastings algorithm and compared our results with previous methods to treat Coulomb interactions in the presence of Lévy sources.

## Key claim

The authors numerically calculated the Coulomb correction factor for non-spherical (three-dimensional) Lévy-type source functions, finding that a one-dimensional spherical Coulomb correction using appropriately averaged scale parameters is a sufficient approximation for both one-dimensional and three-dimensional experimental analyses, despite the source being non-spherical in the pair center-of-mass system (PCMS) even when it is spherical in the longitudinally co-moving system (LCMS).

## Method

The authors used the Metropolis-Hastings algorithm to numerically evaluate the integral of the Coulomb-modified two-particle wave function (the solution of the Schrödinger equation with repulsive Coulomb force) multiplied by a three-dimensional symmetric Lévy-type pair distribution. They transformed the Lévy source from the LCMS to the PCMS via a Lorentz boost depending on the average transverse momentum \(K_T\) and transverse mass \(m_T\), then compared the resulting three-dimensional Coulomb correction with one-dimensional spherical calculations using various averaging schemes for the scale parameters \(R_\text{out}, R_\text{side}, R_\text{long}\).

## Key numbers

* The three-dimensional Lévy source is defined with scale parameters \(R_\text{out}, R_\text{side}, R_\text{long}\) and Lévy exponent \(\alpha\); the pair distribution has modified scale parameters \(2^{1/\alpha}R_\text{out}, 2^{1/\alpha}R_\text{side}, 2^{1/\alpha}R_\text{long}\).
* The Lorentz boost from LCMS to PCMS uses \(\beta_T = K_T/m_T\) where \(m_T = \sqrt{m^2 + K_T^2}\); for a spherical source in LCMS (\(R_\text{out}=R_\text{side}=R_\text{long}=R\)), the effective PCMS scale parameters become \(R_1 = R/\sqrt{1-\beta_T^2}\), \(R_2 = R\), \(R_3 = R\).
* The quadratic average PCMS radius for a spherical LCMS source is \(R_\text{PCMS} = \sqrt{(1 - \frac{2}{3}\beta_T^2)/(1-\beta_T^2)} \, R\).
* For a non-spherical source, the recommended average PCMS radius for one-dimensional Coulomb correction is \(R_\text{PCMS} = \sqrt{\frac{1}{3}\left(R_\text{out}^2/(1-\beta_T^2) + R_\text{side}^2 + R_\text{long}^2\right)}\).
* The numerical integration used a radial cutoff \(r_\text{max}\) such that the integral of the Lévy distribution was at least 0.99 (less than 1% error).
* The three-dimensional correlation functions were evaluated at \(100^3\) points in momentum space.
* For a test case with \(K_T = 0.8\) GeV/\(c\), the one-dimensional Coulomb correction with the transformed average \(R\) agreed with the angle-averaged three-dimensional calculation for \(q > 20\) MeV/\(c\).

## Context

This work extends the treatment of Coulomb corrections for Bose-Einstein correlations from spherically symmetric Lévy sources (Csanád 2019) to three-dimensional, non-spherical Lévy sources, motivated by experimental evidence that Lévy-type sources better describe heavy-ion collision data than Gaussian sources (Adare 2017, NA61/SHINE 2023, CMS 2023). The first three-dimensional Lévy-type correlation measurement (Kurgyis 2018) highlighted the need for this methodology. The results enable experimental analyses at SPS, RHIC, and LHC to apply Coulomb corrections using the simpler one-dimensional approach with appropriately averaged scale parameters, as has already been implemented in several subsequent analyses (Lökös 2022, NA61/SHINE 2023, Mukherjee 2023, Porfy 2023, Korodi 2023, CMS 2023, Kovács 2023).
