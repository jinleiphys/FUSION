# An R-matrix package for coupled-channel problems in nuclear physics

> We present an $R$-matrix Fortran package to solve coupled-channel problems in nuclear physics. The basis functions are chosen as Lagrange functions, which permits simple calculations of the matrix elements. The main input are the coupling potentials at some nucleus-nucleus distances, specified by the program. The program provides the collision matrix and, optionally, the associated wave function. The present method deals with open and closed channels simultaneously, without numerical instability associated with closed channels. It can also solve coupled-channel problems for non-local potentials. Long-range potentials can be treated with propagation techniques, which significantly speed up the calculations. We first present an overview of the $R$-matrix theory, and of the Lagrange-mesh method. A description of the package and its installation on a UNIX machine is then provided. Finally, five typical examples are discussed.

## Key claim

The paper presents an R-matrix Fortran package that solves coupled-channel problems in nuclear physics using Lagrange functions as basis states, which allows for simple calculation of matrix elements without numerical quadrature. The method handles open and closed channels simultaneously without numerical instability, and can treat non-local potentials, though the propagation technique for long-range potentials cannot be used with non-local potentials.

## Method

The coupled-channel system is solved by dividing configuration space into an internal region (r ≤ a) where wave functions are expanded over Lagrange basis functions, and an external region where asymptotic Coulomb functions apply. The Bloch operator is introduced to ensure Hermiticity of the kinetic-energy operator over the finite interval, and the R-matrix is constructed from the inverse of the C-matrix containing kinetic energy, Bloch operator, and potential terms. For long-range potentials, the internal region can be divided into subintervals with propagation of the R-matrix from one interval to the next, reducing the computational cost from inverting a single large matrix to inverting several smaller matrices.

## Key numbers

- For single-channel calculations, running time is typically less than 1 second
- For large-scale calculations with more than 100 channels, running time may increase up to several minutes on a multi-CPU Linux machine
- The propagation technique reduces inversion computer time by approximately a factor of N_s^2, where N_s is the number of subintervals
- Units used are MeV for energies and fm for lengths, with all energies given in the center-of-mass frame

## Context

This work extends the R-matrix method originally developed by Wigner (1946) and further elaborated by Lane and Thomas (1958) and Descouvemont and Baye (2010), combining it with Lagrange functions as described by Baye (2015) for efficient matrix element calculations. Previous codes such as those by Baluja, Burke, and Morgan (1982) and Morgan (1984) were focused on atomic physics or only considered propagation of the R-matrix across several intervals. The package enables straightforward implementation of coupled-channel calculations in any nuclear physics code where coupling potentials are known, and can be applied to problems involving three-body scattering, scattering of exotic nuclei with higher-order Coulomb multipoles, or electron-atom scattering.
