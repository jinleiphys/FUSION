# The LISE package: solvers for static and time-dependent superfluid local density approximation equations in three dimensions

> Nuclear implementation of the density functional theory (DFT) is at present the only microscopic framework applicable to the whole nuclear landscape. The extension of DFT to superfluid systems in the spirit of the Kohn-Sham approach, the superfluid local density approximation (SLDA) and its extension to time-dependent situations, time-dependent superfluid local density approximation (TDSLDA), have been extensively used to describe various static and dynamical problems in nuclear physics, neutron star crust, and cold atom systems. In this paper, we present the codes that solve the static and time-dependent SLDA equations in three-dimensional coordinate space without any symmetry restriction. These codes are fully parallelized with the message passing interface (MPI) library and take advantage of graphic processing units (GPU) for accelerating execution. The dynamic codes have checkpoint/restart capabilities and for initial conditions one can use any generalized Slater determinant type of wave function. The code can describe a large number of physical problems: nuclear fission, collisions of heavy ions, the interaction of quantized vortices with nuclei in the nuclear star crust, excitation of superfluid fermion systems by time dependent external fields, quantum shock waves, domain wall generation and propagation, the dynamics of the Anderson-Bogoliubov-Higgs mode, dynamics of fragmented condensates, vortex rings dynamics, generation and dynamics of quantized vortices, their crossing and recombinations and the incipient phases of quantum turbulence.

## Key claim

The LISE package provides a fully parallelized, GPU-accelerated implementation of the static and time-dependent superfluid local density approximation (SLDA/TDSLDA) equations in three-dimensional coordinate space without any symmetry restrictions, enabling simulations of nuclear fission, heavy-ion collisions, and neutron star crust dynamics that are computationally intractable with basis-expansion methods at scission configurations.

## Method

The static code solves the HFB/SLDA equations by constructing the quasiparticle Hamiltonian matrix on a 3D spatial lattice and performing direct diagonalization using the ScaLAPACK library with a 2D block-cyclic distribution across MPI processes. The time-dependent code propagates quasiparticle wavefunctions using a predictor-modifier-corrector Adams-Bashforth-Milne algorithm with accuracy of order O(Δt)^6, evaluating spatial derivatives via fast Fourier transforms and performing all time-stepping operations on GPUs using CUDA. Both codes use MPI for distributed memory parallelism and support checkpoint/restart capabilities for long dynamical simulations.

## Key numbers

- Time integration accuracy is of order O(Δt)^6 using the Adams-Bashforth-Milne predictor-modifier-corrector algorithm
- The quasiparticle Hamiltonian dimension N = 4NxNyNz can reach O(10^6) for heavy nuclei simulations
- Example input file specifies a lattice of 24 x 24 x 48 grid points with spacing dx = dy = dz = 1.25 fm
- Example input file specifies a nucleus with N = 146 neutrons and Z = 94 protons (240Pu)
- The code has been implemented on supercomputers including Jaguar, Titan, Piz Daint, Tsubame, Summit, and Sierra

## Context

This work extends the coordinate-space HFB approach beyond earlier implementations such as EV8 (Ryssens, 2015) and Sky3D (Maruhn, 2014), which are limited to smaller numbers of single-particle wavefunctions or rely on imaginary-time evolution. The LISE package overcomes the truncation errors inherent in harmonic oscillator basis methods (Dobaczewski, 1997; Schunck, 2017) that cannot accurately describe separated fission fragments at scission and beyond. The code enables full 3D simulations of nuclear fission, heavy-ion collisions, quantized vortex dynamics in neutron star crusts, and the Anderson-Bogoliubov-Higgs mode dynamics without any symmetry restrictions.
