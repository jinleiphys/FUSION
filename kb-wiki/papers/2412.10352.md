# Toward a foundation model for heavy-ion collision experiments based on point-cloud diffusion

> A novel point cloud diffusion model for relativistic heavy-ion collisions, capable of ultra-fast generation of complete, event-by-event collision output, is introduced. When trained on UrQMD cascade simulations, the model generates realistic collision event output containing 26 distinct hadron species, as a list of particle momentum vectors along with their particle ID. From solving inverse problems to accelerating model calculations or detector simulations, the model can be a promising general purpose tool for heavy-ion collisions beneficial to both theoretical studies and experimental applications.

## Key claim

The paper introduces HEIDi, a point-cloud diffusion model that generates complete heavy-ion collision events as lists of particle momentum vectors and hadron types, but the results are demonstrated only for a single fixed condition (Au-Au at 10A GeV with impact parameter b=1 fm) and trained on only 18000 UrQMD events, with the authors explicitly stating that extensive reliability tests are still needed before the model can be applied to physics analyses.

## Method

The authors trained a conditional diffusion probabilistic model on UrQMD cascade simulations of Au-Au collisions at 10A GeV, where each event is represented as a point cloud of 1084 particles (with padding for lower-multiplicity events) containing 26 hadron species. The model uses a PointNet-based encoder and normalizing flow decoder to learn a latent vector encoding global event features, which conditions the reverse diffusion process that iteratively denoises random samples into realistic particle point clouds. The generation process takes approximately 30 milliseconds per event on an Nvidia A100 GPU, compared to about 3 seconds for the UrQMD cascade mode.

## Key numbers

- Training dataset: 18000 UrQMD events of Au-Au collisions at 10A GeV with impact parameter b = 1 fm
- Each event output contains 1084 particles (fixed number, with padding for lower-multiplicity events)
- The model generates 26 distinct hadron species, comprising more than 99% of particles produced at this energy
- Generation speed: approximately 30 milliseconds per event on an Nvidia A100 GPU with 40 GB memory
- UrQMD cascade mode takes about 3 seconds per event, representing a speedup of two orders of magnitude
- The model reproduces multiplicity distributions spanning four orders of magnitude, from abundant particles to exponentially less likely high-momentum particles
- Deviation appears at low transverse momentum below pT < 400 MeV, where HEIDi generates an excess of low-momentum particles

## Context

This work extends the application of diffusion-based generative models from particle physics tasks like jet and calorimeter simulation (Mikuni 2022, Buhmann 2023) to the more complex problem of complete heavy-ion event generation with over a thousand particles and dozens of hadron species. It contrasts with prior machine learning emulators for heavy-ion collisions (Bernhard 2016, Nijs 2020) that require training separate models for each observable, and with concurrent generative approaches (Sun 2024, Torbunov 2024) that produce only histogram or image representations rather than complete particle-level event data. The HEIDi framework enables future applications including comprehensive Bayesian analysis, gradient-based parameter estimation directly on raw experimental data, and potential extension to varying centralities, beam energies, collision systems, and detector effects.
