# A simple method to compute excitation energies in variational many-body calculations

> We propose a simple, easy to implement, variant of the EXCITED method for variational many-body calculations for excited states. We apply this method to the Hybrid Multideterminant method(HMD). We test this method with relatively few Slater determinants by comparing the results with exact shell model calculations for ${}^{56}Ni$ using the $fpd6$ interaction. We obtain very good agreement with the exact results.

## Key claim

The paper proposes a simpler alternative to the EXCITED method for computing excited states in variational many-body calculations, where the key simplification is that the new Centroid HMD (CHMD) method minimizes a weighted average of eigenvalues from a single generalized eigenvalue problem rather than requiring sequential Gram-Schmidt orthogonalization for each excited state.

## Method

The CHMD method constructs a basis of Slater determinants and solves the generalized eigenvalue problem \(h g = o g E\) to obtain all eigenvalues \(E_k\). The quantity minimized during the variational procedure is a weighted average \(F = \sum_k w_k E_k / \sum w_k\) with positive weights \(w_k\), typically taken for only the first few low-lying states. In the limiting case where only one weight is non-zero at a time, the method simulates the EXCITED approach by sequentially adding Slater determinants for each eigenstate while keeping previously determined determinants fixed, with orthogonality automatically enforced by the diagonalization.

## Key numbers

- For \({}^{56}\)Ni with the \(fpd6\) interaction, exact shell model energies are: \(E(0^+_1) = -203.198\) MeV, \(E(0^+_2) = -198.723\) MeV, \(E(0^+_3) = -198.204\) MeV, and \(E(2^+_1) = -200.190\) MeV.
- With 15 Slater determinants for the ground state only (\(w_1=1, w_2=w_3=0\)): \(E(0^+_1) = -203.154\) MeV (44 keV above exact), \(E(0^+_2) = -197.632\) MeV (1.1 MeV above exact), \(E(0^+_3) = -193.330\) MeV (4.9 MeV above exact).
- After adding 15 more determinants for the first excited state (\(w_1=0, w_2=1, w_3=0\)): \(E(0^+_2) = -198.597\) MeV (126 keV above exact).
- After adding 15 more determinants for the second excited state (\(w_1=w_2=0, w_3=1\)): \(E(0^+_3) = -198.147\) MeV (57 keV above exact).
- With 45 total Slater determinants plus 5 additional for \(0^+_1\) and 5 for \(0^+_2\): \(E(0^+_1) = -203.166\) MeV (32 keV discrepancy), \(E(0^+_2) = -198.630\) MeV (93 keV discrepancy), \(E(0^+_3) = -198.132\) MeV (72 keV discrepancy).
- With 50 Slater determinants and balanced weights (\(w_1=3, w_2=2.5, w_3=1\)): \(E(0^+_1) = -203.167\) MeV, \(E(0^+_2) = -198.637\) MeV, \(E(0^+_3) = -198.130\) MeV.
- For the \(2^+\) states with 50 Slater determinants: \(E(2^+_1) = -200.122\) MeV (70 keV above the shell model value).

## Context

This work extends the Hybrid Multideterminant (HMD) method (Puddu, 2001) and the EXCITED-MAD-VAMPIR approach (Schmid, 1987) by providing a simpler implementation that requires only about 170 lines of FORTRAN 77 code modification for the ground-state program. The method enables accurate computation of non-yrast states without repeating calculations when higher accuracy is desired, and was tested against exact shell model diagonalizations for \({}^{56}\)Ni (refs. 3 and 4 in the paper).
