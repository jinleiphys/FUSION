# Two-neutrino double beta decay of deformed nuclei within QRPA with realistic interaction

> A method to implement realistic nucleon-nucleon residual interaction (Bonn CD force) based on the Brueckner G-matrix into the Quasiparticle Random Phase Approximation (QRPA) for deformed nuclei is formulated. The two-neutrino double decay for ground state to ground state transitions $^{76}$Ge$ \to ^{76}$Se and $^{150}$Nd $\to ^{150}$Sm is calculated along with the Gamow-Teller strength distributions. The effect of deformation on the observables is studied.

# Two-neutrino double beta decay of deformed nuclei within QRPA with realistic interaction

## Key claim

The authors implement a realistic nucleon-nucleon residual interaction (Bonn CD force via Brueckner G-matrix) into the Quasiparticle Random Phase Approximation (QRPA) for deformed nuclei, but the adiabatic (strong-coupling) approximation used for the wave functions is formally valid only for large deformations and may fail for the weakly deformed nuclei like ⁷⁶Ge and ⁷⁶Se, where the weak-coupling limit would be more appropriate.

## Method

The deformed axially symmetric Woods-Saxon single-particle wave functions are expanded over spherical harmonic oscillator basis states, allowing the spherical G-matrix elements (from the Bonn CD force) to be transformed into the deformed basis. The QRPA equations are solved in the intrinsic frame for each K-projection (K=0,±1), and the laboratory-frame 1⁺ states are constructed using the Bohr-Mottelson strong-coupling ansatz. The 2νββ-decay matrix elements M²ν_GT are calculated by summing over intermediate states from both initial and final nuclei, including overlaps between the two sets of QRPA phonons and BCS vacua.

## Key numbers

- For ⁷⁶Ge → ⁷⁶Se with deformation β₂=0.10 (⁷⁶Ge) and 0.16 (⁷⁶Se), the fitted particle-particle strength parameter is g_pp=0.99 (case I) and 0.97 (case II) using the realistic Bonn CD force.
- For ¹⁵⁰Nd → ¹⁵⁰Sm with experimental deformations β₂=0.37 (¹⁵⁰Nd) and 0.23 (¹⁵⁰Sm), the fitted g_pp=0.78 (case I) and 0.65 (case II); with Möller deformations β₂=0.24 (¹⁵⁰Nd) and 0.21 (¹⁵⁰Sm), the fitted g_pp=1.35 (case I) and 1.32 (case II).
- The Ikeda sum rule is satisfied to 94-98% across all calculations, with values ranging from 94.1% (¹⁵⁰Nd, experimental deformation) to 98.2% (⁷⁶Se, spherical limit).
- The particle-hole strength parameter is fixed at g_ph=1.15 for all nuclei, reproducing the experimental Gamow-Teller giant resonance position in ⁷⁶Ge.

## Context

This work extends previous QRPA calculations for deformed nuclei that used schematic separable forces (Gese et al., 2000; Rodin et al., 2006; Alvarez-Rodriguez et al., 2005) by implementing a realistic Brueckner G-matrix interaction derived from the Bonn CD potential. It is a necessary first step toward calculating the neutrinoless double beta decay (0νββ) matrix element for strongly deformed candidates like ¹⁵⁰Nd, which has a phase-space factor about 33 times larger than ⁷⁶Ge but whose deformation has previously only been treated in severely truncated models like the pseudo-SU(3) approach (Hirsch et al., 1995) or the Projected Hartree-Fock-Bogoliubov method (Hirsch et al., 2008). The spherical QRPA calculations of Rodin et al. (2006) for 0νββ matrix elements used the same realistic interaction but assumed spherical symmetry, so this work enables future comparison of deformation effects on the 0νββ matrix element.
