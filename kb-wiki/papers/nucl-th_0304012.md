# Merging the CEM2K and LAQGSM Codes with GEM2 to Describe Fission and Light-fragment Production

> We present the current status of the improved Cascade-Exciton Model (CEM) code CEM2k and of the Los Alamos version of the Quark-Gluon String Model code LAQGSM. To describe fission and light-fragment (heavier than He4) production, both CEM2k and LAQGSM have been merged with the GEM2 code of Furihata. We present some results on proton- and deuteron-induced spallation, fission, and fragmentation reactions predicted by these extended versions of CEM2k and LAQGSM. We show that merging CEM2k and LAQGSM with GEM2 allows us to describe many fission and fragmentation reactions in addition to the spallation reactions which are already relatively well described. Nevertheless, the standard version of GEM2 does not provide a completely satisfactory description of complex particle spectra, heavy-fragment emission, and spallation yields, and is not yet a reliable tool for applications. We conclude that we may choose to use a model similar to the GEM2 approach in our codes, but it must be significantly extended and further improved. We observe that it is not sufficient to analyze only A and Z distributions of the product yields when evaluating this type of model, as is often done in the literature; instead it is important to study all the separate isotopic yields as well as the spectra of light particles and fragments.

## Key claim

Merging the CEM2k and LAQGSM codes with the GEM2 evaporation/fission code allows description of many fission and fragmentation reactions in addition to spallation, but the standard GEM2 does not provide a completely satisfactory description of complex particle spectra, heavy-fragment emission, and spallation yields, and is not yet a reliable tool for applications without significant extension and further improvement.

## Method

The authors merged the GEM2 code of Furihata with their existing CEM2k (Cascade-Exciton Model) and LAQGSM (Los Alamos Quark-Gluon String Model) codes, using GEM2 to handle evaporation of up to 66 types of particles and fragments and fission after the cascade and preequilibrium stages calculated by CEM2k or LAQGSM. They benchmarked the merged codes against proton- and deuteron-induced spallation, fission, and fragmentation reactions, particularly GSI measurements on p+Pb and d+Pb reactions in inverse kinematics, and compared results both with and without preequilibrium particle emission. To obtain correct fission cross sections, they adjusted the level-density parameter a_f in GEM2 by factors of 1.06 (with preequilibrium) and 0.947 (without preequilibrium) for nuclei with Z ≤ 88.

## Key numbers

- For p(1 GeV) + Pb, the level-density parameter a_f was increased by a factor of 1.06 relative to the original RAL/GEM2 value when including preequilibrium emission (a_f^CEM / a_f^RAL = 1.06), and decreased to 0.947 when omitting preequilibrium emission (a_f^CEM / a_f^RAL = 0.947)
- The GEM2 code considers evaporation of 66 types of particles and fragments, including isotopes from n, p, d, t, He-3,4,6,8 through Mg-22-28
- For asymmetric fission, the mass of one fragment is selected from a Gaussian distribution with mean A_f = 140 and width σ_M = 6.5
- The charge distribution of fission fragments uses σ_Z = 0.75 (Furihata's value) rather than the original Atchison value of σ_Z = 2.0
- For p(100 MeV) + U-238, the merged CEM2k+GEM2 gave a fission cross section in agreement with data using default GEM2 parameters without adjustment

## Context

This work extends the authors' previous development of CEM2k and LAQGSM codes, which originally described only spallation reactions without fission fragment production, by merging with Furihata's GEM2 code (2000-2002) to handle fission and light-fragment emission. The approach contrasts with the use of Atchison's fission model in MCNPX transport code, and the authors find that GEM2 parameters fitted for use with Bertini or ISABEL intranuclear cascade models (without preequilibrium) require adjustment when coupled with CEM2k or LAQGSM. The work enables improved modeling of reactions relevant to Accelerator Transmutation of nuclear Wastes, Accelerator Production of Tritium, Rare Isotope Accelerator, and Proton Radiography projects, but the authors conclude that integral A and Z distributions are insufficient for model validation and that separate isotopic yields and particle spectra must be studied.
