# LUCIAE 3.0: A new version of a computer program for Firecracker Model and rescattering in relativistic heavy-ion collisions

> LUCIAE is a Monte Carlo program that, connected to FRITIOF, implements both the Firecracker Model (FCM), a possible mechanism for collective multi-gluon emission from the colour fields of interacting strings, and the reinteraction of the final state hadrons in relativistic heavy ion collisions. This paper includes a brief presentation of the dynamics of LUCIAE with an emphasis on the new features in this version, as well as a description of the program.

## Key claim

LUCIAE 3.0 is a Monte Carlo program that implements both the Firecracker Model (FCM) for collective multi-gluon emission from interacting colour strings and the rescattering of final-state hadrons in relativistic heavy-ion collisions, with the aim of testing whether observed collective effects can be explained without invoking quark-gluon plasma formation.

## Method

The program connects to FRITIOF7.02, JETSET7.4, PYTHIA5.5, and ARIADNE4.02 to simulate nucleus-nucleus collisions. When multiple excited strings are close together, the Firecracker Model allows them to emit gluons using their common total energy density, producing enhanced high-transverse-momentum gluons. The rescattering module initializes produced hadrons in space-time within the nuclear overlap region, then simulates their subsequent elastic and inelastic collisions using a minimum-distance criterion and cross-section-based selection, with the new version including many more inelastic channels and their reverse reactions.

## Key numbers

- The effective string tension is parametrized as κ_eff = κ_0 (1-ξ)^{-α}, with α = 3.5 and √s_0 = 0.8 GeV determined from hadron-hadron data
- Default JETSET parameters retuned for pp at 200 GeV/c: PARJ(1)=0.046, PARJ(2)=0.2, PARJ(3)=0.3, PARJ(21)=0.32 (defaults were 0.1, 0.3, 0.4, 0.37 respectively)
- Typical running times on an HP station: 0.6 minutes/event for ³²S+³²S at p_lab = 200 A GeV/c; 9 minutes/event for ²⁰⁷Pb+²⁰⁷Pb at p_lab = 158 A GeV/c; 80 minutes/event for ¹⁹⁷Au+¹⁹⁷Au at √s = 200 GeV
- Minimum distance between spectator nucleons set to 0.5 fm
- Formation time of a hadron at its rest frame: PARAM(7) = 0.5 fm
- Default cross sections: σ_tot(NN) = 40.0 mb, σ_tot(πN) = 25.0 mb, σ_tot(KN) = 35.0 mb, σ_tot(ππ) = 10.0 mb
- Default π+N → K+Y cross section: VFR(20) = 0.35 mb
- Default π+N → Δ+π cross section: VFR(21) = 3.0 mb
- Default N+N → Δ+N cross section: VFR(24) = 6.0 mb
- Default π+N → ρ+N cross section: VFR(22) = 1.0 mb
- Default π+π → K+K cross section: PARAM(5) = 3.0 mb
- Default N N annihilation cross section: PARAM(13) = 28.0 mb
- Default Λ N annihilation cross section: PARAM(14) = 5.6 mb
- Default nucleon density: PARAM(11) = 0.16/fm³
- Default minimal CMS energy for starting Firecracker Model: VFR(19) = 6.5 GeV
- Default size of firecracker cluster when KFR(18)=0: VFR(18) = 0.35 GeV/c

## Context

This work extends the earlier LUCIAE 2.0 (Sa Ben-Hao and Tai An, 1995) by adding more inelastic rescattering channels including those for Ξ and Ω⁻ production, implementing p and Λ annihilation in nuclear matter, and including the effect of firecracker gluons on string fragmentation through an effective string tension. The program is intended to provide a conventional-physics baseline for interpreting collective effects observed in relativistic heavy-ion collisions at AGS, CERN, and RHIC energies, complementing models that assume quark-gluon plasma formation.
