# Transverse momentum dependent distributions in $e^+e^-$ and semi-inclusive deep-inelastic scattering using jets

> The extraction of transverse momentum dependent distributions (TMDs) in semi-inclusive deep inelastic scattering (SIDIS) is complicated by the presence of both initial- and final-state nonperturbative physics. We recently proposed measuring jets (instead of hadrons) as a solution, showing that for the Winner-Take-All jet axis the same factorization formulae valid for hadrons applied to jets of arbitrary size. This amounts to simply replacing TMD fragmentation functions by our TMD jet functions. In this paper we present the calculation of these jet functions at one loop. We obtain phenomenological results for $e^+e^- \to$ dijet (Belle II, LEP) and SIDIS (HERA, EIC) with a jet, building on the arTeMiDe code. Surprisingly, we find that the limit of large jet radius describes the full $R$ results extremely well, and we extract the two-loop jet function in this limit using Event2, allowing us to achieve N$^3$LL accuracy. We demonstrate the perturbative convergence of our predictions and explore the kinematic dependence of the cross section. Finally, we investigate the sensitivity to nonperturbative physics, demonstrating that jets are a promising probe of proton structure.

## Key claim

The authors claim that using the Winner-Take-All (WTA) jet axis allows the same TMD factorization formulae valid for hadrons to apply to jets of arbitrary size, simply replacing TMD fragmentation functions with perturbatively calculable TMD jet functions. The key hedge is that this holds specifically for the WTA axis, not for the standard jet axis, and that the factorization is independent of the jet radius R only in the limit where the angular decorrelation θ is much smaller than R.

## Method

The authors calculate the one-loop TMD jet functions for quarks in both momentum space and impact-parameter space using dimensional regularization and the modified δ-regulator for rapidity divergences. They implement these jet functions into the arTeMiDe code to obtain phenomenological predictions for e+e- → dijet (at Belle II and LEP energies) and SIDIS (at HERA and EIC kinematics), and extract the two-loop jet function in the large-R limit using Event2 to achieve N3LL accuracy.

## Key numbers

- For e+e- → dijet at √s = 91.2 GeV (LEP), the cross section is computed as a function of qT from 0.1 GeV to 10 GeV, with the large-R approximation valid down to R ≈ 0.1
- For SIDIS at EIC kinematics with Q = 10 GeV, x = 0.1, and z = 0.5, predictions are shown for qT from 0.1 GeV to 5 GeV
- The one-loop quark jet function in impact-parameter space is given by J_q^(1)(z, b, E R, μ, ζ) = δ(1-z) [C_F (L_b^2/2 - γ_0^q L_b + G_q(ERb)) + ...] where L_b = ln(μ^2 b^2/4e^{-2γ_E}) and γ_0^q = 3C_F/2
- The two-loop cusp anomalous dimension used for N3LL resummation is A^(3) = (245/6 - 134ζ_2/3 + 22ζ_3/3 + 88ζ_4/3) C_A C_F + ...

## Context

This work extends the authors' previous proposal (Gutierrez-Reyes et al., 2018) to use jets instead of hadrons for TMD measurements, providing the first complete one-loop calculation of the TMD jet functions and full phenomenological analysis. It builds on the arTeMiDe code framework (Scimemi, 2017; Scimemi, 2018) and the ζ-prescription for TMD evolution. The approach contrasts with studies using the standard jet axis (Neill et al., 2016; Neill et al., 2018) which require more complicated factorization with non-global logarithms, and with hadron-based TMD extractions that suffer from nonperturbative fragmentation functions. The results enable direct extraction of TMD PDFs from SIDIS data using jets, with the nonperturbative corrections to the jet function being universal and fit-able from e+e- data, making this a promising probe of proton structure at the EIC.
