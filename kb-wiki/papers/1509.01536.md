# Preserving Local Gauge Invariance with t-Channel Regge Exchange

> Considering single-meson photo- and electroproduction off a baryon, it is shown how to restore local gauge invariance that was broken by replacing standard Feynman-type meson exchange in the t-channel by exchange of a Regge trajectory. This is achieved by constructing a contact current whose four-divergence cancels the gauge-invariance-violating contributions resulting from all states above the base state on the Regge trajectory. To illustrate the procedure, modifications necessary for the process $γ+p \to K^+ + Σ^{*0}$ are discussed in some detail. We also provide the general expression for the contact current for an arbitrary reaction.

## Key claim

The paper shows that local gauge invariance broken by replacing standard Feynman-type t-channel meson exchange with Regge trajectory exchange can be restored by constructing a contact current whose four-divergence cancels the gauge-invariance-violating contributions from all states above the base state on the Regge trajectory, but only if the Reggeization is applied consistently across all elements of the production current, not just the t-channel term.

## Method

The authors start from the field-theoretical formalism of Haberzettl (1997) which expresses the photoproduction current as a sum of s-channel, u-channel, t-channel, and interaction-current contributions, with local gauge invariance enforced by generalized Ward-Takahashi identities (gWTI). They show that replacing the t-channel meson propagator and form factor by a Regge trajectory propagator violates the gWTI because higher-lying states on the trajectory couple with the same electromagnetic current as the base state instead of with transverse transition currents. The restoration is achieved by also Reggeizing the vertex in the interaction current, replacing the standard t-channel vertex \(F_t\) with the Reggeized vertex \(\mathcal{R}F\) everywhere, and constructing a minimal contact current that satisfies the modified four-divergence condition \(k_\mu \mathcal{M}_\text{IC}^\mu = Q_m \mathcal{R}F + Q_{b'} F_u - F_s Q_b\).

## Key numbers

* The Regge residual function \(\mathcal{F}_t(t) = (t-M_m^2)\mathcal{P}_m(t)\) is normalized to unity at the base pole: \(\mathcal{F}_t(M_i^2) = 1\) for both pseudoscalar (\(i=0\)) and vector (\(i=1\)) trajectories.
* The Regge trajectory slope parameters are defined as \(\alpha_i(t) = \alpha_i' (t-M_i^2)\) for \(i=0,1\), with the scale parameter typically chosen as \(s_\text{scale} = 1\) GeV.
* For the example reaction \(\gamma + p \to K^+ + \Sigma^{*0}\), the minimal bare contact current is \(m_c^{\nu\mu} = -e_K g \, g^{\nu\mu}\), where \(e_K = e\) is the fundamental charge unit.
* The form factors are normalized as \(f_s(M_N^2)=1\) and \(f_t(M_K^2)=1\) at the proton mass \(M_N\) and kaon mass \(M_K\), respectively.

## Context

This work extends the hybrid Regge-Feynman approach developed by Guidal, Laget, and Vanderhaeghen (1997) and applied by many groups (e.g., Corthalis 2006, 2007; He 2014; Wang 2014), which used the residual function of the Regge base state as a common multiplicative factor to restore global gauge invariance but lacked a dynamical foundation. The present method provides a rigorous field-theoretical restoration of local gauge invariance based on generalized Ward-Takahashi identities from Haberzettl (1997, 2006), enabling consistent Reggeization of t-channel exchanges in single-meson photoproduction while preserving the full off-shell gauge structure needed for embedding in larger processes.
