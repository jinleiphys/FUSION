# Decomposition of angular momentum projected nuclear wave function

> Angular momentum projection is a basic technique in constructing nuclear wave functions with good spins. Traditionally, a projected nuclear wave function is expressed in terms of the bases built by performing the angular momentum projection directly on reference states for the whole nuclear system. Alternatively, one can construct nuclear wave function with another kind of projected bases, called as the coupled projected bases, which are generated by first performing the angular momentum projections on the reference states for neutrons and protons, respectively, then coupling the neutron projected states with the proton ones via Clebsch-Gordon coefficients. In the present work, we derive a new identity, which provides a decomposition of the conventional angular momentum projected nuclear wave function in terms of the coupled projected bases. This decomposition offers direct insight into the underlying structure of nuclear states. To show this point, we present the decompositions of variation after projection shell model (VAPSM) wave functions for the ground states in some $sd$ shell nuclei. It is interesting to see that even for the ground states in even-even nuclei, the nucleons are not fully paired. Finally, we demonstrate that the VAPSM wave function can be further improved by adopting the coupled projected bases.

## Key claim

The paper derives a new identity that decomposes a conventional angular-momentum projected nuclear wave function into coupled projected bases built from separate neutron and proton angular momentum projections, revealing that even in ground states of even-even nuclei the nucleons are not fully paired.

## Method

The authors derive an operator identity expressing the total angular momentum projection operator \(P^J_{MK}\) as a sum over products of separate neutron and proton projection operators coupled via Clebsch-Gordan coefficients. They then apply this decomposition to variation after projection shell model (VAPSM) wave functions for \(sd\) shell nuclei and for the heavy deformed nucleus \(^{166}\)Dy, calculating the weights \(C(J_\pi,J_\nu)\) of each coupled angular momentum component. Finally, they propose a generalized eigenvalue equation using the coupled projected bases as a variational space to improve the VAPSM wave function.

## Key numbers

* For the ground state of \(^{20}\)Ne, the \(C(J_\pi=2,J_\nu=2)\) component is larger than the \(C(J_\pi=0,J_\nu=0)\) component, indicating the ground state is not dominated by fully paired nucleons.
* For the ground \(0^+\) state in \(^{166}\)Dy, the sum of all \(C(J^-_\pi,J^-_\nu)\) values is about 0.042%, and the \(C(J_\pi=0,J_\nu=0)\) component is only about 3.2%, while components with \(J_\pi=J_\nu=2,4,6\) each exceed 15%.
* For the \(J^\pi=0^+\) state in \(^{166}\)Dy, the calculated components with \(J_\pi\leq20\) and \(J_\nu\leq20\) cover 99.9998% of the wave function; for the \(2^+\) state they cover 99.999%, and for the \(10^+\) state they cover 99.94%.

## Context

This work extends the earlier decomposition approach of Otsuka (1993), which assumed axial intrinsic states with \(K=0\) for even-even nuclei, to a general formalism applicable to any even-even, odd-mass, or odd-odd nucleus with arbitrary reference states. The coupled projected bases are analogous to those used in studies of the scissors mode by Sun (1998) and Lv (2022). The decomposition enables direct insight into the microscopic structure of nuclear states and provides a path to systematically improve VAPSM calculations by treating the coefficients of the coupled projected bases as free variational parameters.
