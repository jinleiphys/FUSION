# Best linear unbiased estimation of the nuclear masses

> This paper presents methods to provide an optimal evaluation of the nuclear masses. The techniques used for this purpose come from data assimilation that allows combining, in an optimal and consistent way, information coming from experiment and from numerical model. Using all the available information, it leads to improve not only masses evaluations, but also to decrease uncertainties. Each newly evaluated mass value is associated with some accuracy that is sensibly reduced with respect to the values given in tables, especially in the case of the less well-known masses. In this paper, we first introduce a useful tool of data assimilation, the Best Linear Unbiased Estimation (BLUE). This BLUE method is applied to nuclear mass tables and some results of improvement are shown.

## Key claim

The Best Linear Unbiased Estimation (BLUE) method from data assimilation can produce an improved set of nuclear mass excess values with reduced uncertainties compared to tabulated experimental values, but the improvement is largest for nuclei far from the stability valley where experimental accuracy is poorest, while stable nuclei see negligible changes (around 10⁻¹⁰%).

## Method

The authors applied the BLUE data assimilation technique to combine experimental mass excess values from the Audi tables (treated as observations) with theoretical mass excess values from the Finite-Range Droplet Model of Moller et al. (treated as the background). The observation error covariance matrix was taken as diagonal using the experimental uncertainties, and the background error covariance matrix was approximated as diagonal with a single global variance σ²_b = 0.652904 MeV², estimated from the mean square difference between model and experiment over all nuclei with Z ≥ 8. The analysis was performed using the standard BLUE formula x^a = x^b + K(y − Hx^b) with the gain matrix K = BH^T(HBH^T + R)⁻¹, where the observation operator H reduced to the identity since both datasets are in the same space.

## Key numbers

- Background error standard deviation: σ_b = 0.808024 MeV, estimated from the mean square difference between model and experimental mass excess over all nuclei with Z ≥ 8
- Mean difference between model and experiment: −0.058326 MeV, indicating near-unbiased background
- Correlation between background error and measurement error: 0.1998
- Relative modification of mass excess values ranges from 10⁻¹⁰% (for well-known stable nuclei) up to 80% (for poorly known nuclei far from stability)
- Accuracy improvement (reduction of uncertainty) reaches up to roughly 70% for some nuclei

## Context

This work extends data assimilation techniques, previously applied in meteorology (e.g., ERA-40 re-analysis by Uppala et al.) and nuclear core neutronic activity evaluation (Massart, 2007; Bouriquet et al., 2010), to the problem of nuclear mass evaluation. It builds on prior efforts to combine experimental information (Audi et al., 2003) with nuclear mass models such as the Finite-Range Droplet Model (Moller et al., 1995) and the Finite-Range Liquid-Drop Model (Myers and Swiatecki, 1966; Bethe, 1971). The method enables production of an improved mass data set with reduced uncertainties, and the authors suggest that refining the background error covariance matrix could allow forecasting masses of yet unknown nuclei.
